当您添加，更改或删除DnD™操作时， GML Live Preview将更新，以向您显示在“幕后”创建的实际代码。实时预览中的代码无法直接进行编辑，但是您可以选择部分并将其复制并粘贴到GML脚本或代码操作中（例如）。

如果您确定要DnD™转换为代码，则可以通过单击鼠标右键再次完成此操作。在具有操作的任何事件工作区中，然后选择“转换为GML 。

首次执行此操作时，系统会显示警告消息，提示您这是单向转换，因为您可以将动作转换为代码，但是以后不能再将其转换回单个动作。单击此处的“确定”将为您执行转换。
