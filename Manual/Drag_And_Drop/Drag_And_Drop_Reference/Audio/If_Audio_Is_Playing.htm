如果音频正在播放

此操作可用于检查您的游戏中当前是否正在播放任何声音资源的实例。您从资产资源管理器中提供要检查的声音资源，该函数将返回 true if it is playing and false如果不是，尽管如果您选中“ not”标志，则相反，该函数将返回 true如果声音没有播放并且 false否则。
