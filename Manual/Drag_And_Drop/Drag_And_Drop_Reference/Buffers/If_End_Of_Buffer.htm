If End Of Buffer

这个动作可以用来检查读/写位置是否已经到达缓冲区的末端。你提供由动作Create Buffer返回的唯一的缓冲区ID值，如果读/写位置已经到达缓冲区中所有可用数据的末端，动作将返回true，如果没有，则返回false。你可以标记 "not "来使动作检查读/写位置是否在缓冲区的末端。

请注意，要把动作添加到 "如果 "块中，应该把它们放到动作的一侧，如下图所示。
