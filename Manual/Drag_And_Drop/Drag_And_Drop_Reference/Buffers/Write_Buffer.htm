写缓冲区

此操作可用于将数据写入缓冲区。您提供要写入的唯一缓冲区ID（由操作Create Buffer返回），然后提供要写入的值，该值可以是实数或字符串，但是请注意，所有值都存储为字符串，因此当您从缓冲区（使用Read Buffer ），您可能需要使用数据类型操作之一将其从字符串转换回原始值。还要记住，缓冲区是按顺序读取和写入的，因此写入缓冲区将使“寻找”位置沿一个方向移动，这意味着要写入各种数据，您只需简单地将此操作称为所需次数即可。
