获取视图变量

该操作允许您获取与给定查看端口相关的某些内置变量的值。选择要检索的查看端口变量，然后将查看端口提供给目标（从0到7），最后将目标变量提供给保存返回值的目标变量（可以将其标记为临时局部变量）。视口是将用于绘制给定摄像机视图（在房间编辑器中设置）的屏幕区域，下表概述了您可以通过这种方式获取的不同变量：

多变的描述
Camera返回的值将是分配给端口的摄像机的唯一ID值（有关更多信息， 请参见此处）。
Visibility这将返回值 true或者 false在哪里 true视口是可见的（即：它被绘制到屏幕上），并且 false意思是什么也没画。
View Port X Coordinate这是游戏窗口中视口的X位置。
View Port Y Coordinate这是游戏窗口中视口的Y位置。
View Port Width这是视口的宽度（以像素为单位）。
View Port Height这是视口的高度（以像素为单位）。
View Port Surface ID这将返回分配给视口的表面的唯一ID值，如果未分配任何表面，则返回-1。
