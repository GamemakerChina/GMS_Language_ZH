设置视图变量

此操作允许您设置某些内置变量，这些变量与视口如何显示房间的摄像机视线有关。选择要编辑的视口变量，然后将视口指定给目标（从0到7），最后为变量指定新值。视口是将用于绘制给定摄像机视图（在房间编辑器中设置）的屏幕区域，下表概述了可以以此方式设置的不同变量：

多变的描述
Camera这是用于设置视口以显示不同的“摄影机视图”。您需要提供要更改为的摄像机的唯一ID值（有关更多信息， 请参见此处）。
Visibility这是查看端口的可见性切换，期望值为 true或者 false在哪里设置 true将使视口可见（即：它将绘制到屏幕上），并且值为 false不会在屏幕上画任何东西。
View Port X Coordinate这是游戏窗口中视口的X位置。请注意，虽然只有在多个视图端口处于活动状态时才真正需要此端口（例如，创建分屏效果），并且如果仅在一个端口处于活动状态时使用，则将获得意外的结果。
View Port Y Coordinate这是游戏窗口中视口的Y位置。请注意，虽然只有在多个视图端口处于活动状态时才真正需要此端口（例如，创建分屏效果），并且如果仅在一个端口处于活动状态时使用，则将获得意外的结果。
View Port Width这是视口的宽度（以像素为单位）。视口的宽度（如果活动的视口超过一个，则视口的宽度为准）定义了游戏开始时游戏窗口或背景画布的宽度，因此在游戏开始后更改此值不会对游戏产生明显影响。游戏窗口大小，除非与GML函数一起调用 window_set_size() 。另请注意，如果端口大小大于或小于分配给摄像机的端口大小，则摄像机视图将按比例缩小或放大以适合。
View Port Height这是视口的高度（以像素为单位）。视口的高度（或活动视口以上的组合视口）的高度定义了游戏开始时游戏窗口或背景画布的高度，因此在游戏开始后更改此值不会对游戏窗口大小，除非与GML函数一起调用 window_set_size() 。还要注意，如果端口大小大于或小于分配给摄像机视图的端口大小，则摄像机视图将按比例缩小或放大以适合。
View Port Surface ID使用此变量，您可以设置给定视口的内容以绘制到表面上，您提供的值是用于将视口绘制到的表面的值。有关更多信息，请参见此处。
