如果放置任何物体

此操作用于检查并基于调用实例的碰撞蒙版查看是否有任何对象的实例放置在某个位置。您指定位置，该位置可以是绝对位置，也可以是相对于实例的位置，操作将返回 true如果有碰撞或 false如果没有。您还可以检查“ Not”标志，以检查在给定位置是否没有碰撞，然后动作将返回 true如果没有发现碰撞 false否则。冲突是根据调用实例的碰撞蒙版与该位置处任何实例的碰撞蒙版重叠（就像它是“放置”在该位置处的）重叠计算的。

重要的！碰撞只会为那些具有有效碰撞蒙版的实例进行注册，即：将一个sprite分配给 sprite_index，或分配给 mask_index。如果无论是在碰撞的情况下，有没有碰撞面具那么碰撞不会被检测到，无论什么样的情况下在时间画画。
