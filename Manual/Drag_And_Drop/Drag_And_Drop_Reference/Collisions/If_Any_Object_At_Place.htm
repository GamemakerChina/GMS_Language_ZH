如果有任何对象在地方

这个动作用于检查并查看是否有任何对象的实例放置在基于调用实例的碰撞掩码的位置上。你给出位置，可以是一个绝对位置，也可以是一个相对于实例的位置。 如果有碰撞，动作将返回true，如果没有则返回false。你也可以检查 "Not "标志，以检查在给定位置是否有碰撞，如果没有发现碰撞，动作将返回true，否则返回false。碰撞的计算是基于调用实例的碰撞掩码与该位置的任何实例的碰撞掩码相重叠（就像它被 "放置 "在该位置一样）。

重要的是!碰撞将只登记那些具有有效碰撞掩码的实例，即：它们有一个分配给sprite_index的精灵，或一个分配给mask_index的精灵。如果碰撞中的任何一个实例 碰撞中的任何一个实例没有碰撞掩码，那么这个碰撞将不会被检测到，不管这个实例当时正在画什么。
