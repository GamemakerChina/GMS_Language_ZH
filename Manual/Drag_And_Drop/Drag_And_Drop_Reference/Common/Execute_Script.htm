执行脚本

此操作可用于调用脚本函数或用户定义的函数。通常，此操作仅用于旧式支持，不应在当前或新项目中真正使用。要使用该操作，您需要给脚本资产调用-或给用户定义的函数变量名称调用-然后提供它需要的任何参数，即：任何值（字符串，整数，资源索引等）您希望输入到脚本或函数中以便可以使用它们的代码。您可以根据需要通过单击加号图标添加其他参数。在动作旁边。

某些脚本和用户定义的函数还可以返回一个值，以供以后在操作块列表中使用，因此您可以指定应存储此返回值的目标变量。这可以是以前创建的，也可以是新的临时本地文件（如果选中“ Temp ”复选框）。如果不需要，则目标字段和参数字段都可以保留为空白。

请注意，通常最好使用Action Function Call操作来调用用户定义的函数，并且通常不需要使用此操作或直接在代码中调用脚本资产。
