函数调用

此操作可用于调用任何GML函数或任何用户定义的函数。给函数调用，然后提供它需要的任何其他参数，即：您希望输入到函数中以便可以使用它们的任何值（字符串，整数，资源索引等）。您可以通过单击加号图标来添加其他参数输入在操作旁边，您应确保您的参数计数与正在使用的函数的计数匹配（即：如果该函数需要3个参数，则应提供3个参数）。某些函数还可以返回一个值，以供以后在操作块列表中使用，因此，您还可以指定应存储此返回值的目标变量，该变量可以是以前创建的，也可以是新的临时本地变量（如果选中“温度”复选框）。如果不需要，则目标字段和参数字段都可以保留为空白。
