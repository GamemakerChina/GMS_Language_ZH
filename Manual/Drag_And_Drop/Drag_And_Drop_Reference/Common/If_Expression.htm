如果表达

此操作用于检查表达式是否为 true或者 false，您可以在此处检查任何逻辑表达式，只要它的计算结果为一个或另一个值即可。例如，可以使用以下任何表达式：

x < (room_width / 2)
a == b
++val >= 100

如果您标记“ Not”参数，则上面的表达式将是取反的，例如“== “变成”not=也请注意，表达式不限于数值，您可以检查字符串，指针，资源ID或函数可以返回或使用的其他任何内容（有关更多信息，请参见数据类型部分），但是您不能比较两个不同的数据类型而不会出现错误-例如，如果您尝试在表达式中使用字符串和整数，则会出现错误。有关表达式的更多信息，请参见此处。

重要说明：由于浮点精度问题，可能会返回以检查两个值是否完全相等 false，因为一个可能正好是1，而另一个可能是1.00000000000001。可以通过在检查之前使用“十进制到整数”操作或使用“大于（>）”或“小于（<）”表达式来避免这种情况。
