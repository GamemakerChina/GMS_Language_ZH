If Expression

这个动作用于检查一个表达式是真还是假，你可以在这里检查任何逻辑表达式，只要它评估为一个或另一个值。例如，可以使用以下任何一个表达式。

x < (room_width / 2)
a == b
++val >= 100

如果你标记了 "非 "参数，那么上述表达式将被否定，例如"=="变成"not="。还要注意的是，表达式并不局限于数值，你可以检查字符串、指针、资源ID或其他任何函数可以返回或使用的东西（更多信息见数据类型一节），但你不能在不得到错误的情况下比较两种不同的数据类型--例如，如果你试图在表达式中使用一个字符串和一个整数，你会得到一个错误。关于表达式的更多信息，请看这里。

重要提示：由于浮点精度问题，检查两个值是否完全相等可能会返回错误，因为一个值可能正好是1，而另一个可能是1.00000000000001。 另一个可能是1.00000000000001。这可以通过在检查前使用小数到整数的操作或使用 "大于（>）"或 "小于（<）"表达式来避免。
