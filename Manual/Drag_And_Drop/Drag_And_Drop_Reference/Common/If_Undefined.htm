如果未定义

此操作用于检查变量或表达式是否返回“ undefined”值，如果是，则此操作将返回 true如果不是，它将返回 false。出于多种原因，可能会返回“未定义”，并且未定义的值可能会导致项目崩溃或严重错误，因此您可以使用它来检查并在发生之前捕获它。如果标记“ Not”参数，那么您正在检查值是否不是“ undefined”。

现在，如果“如果”的计算结果为 true，而放到其他位置的所有操作都将在“ if”块之后执行。请注意，您可以通过单击加号图标在同一操作中“堆叠”未定义的检查在动作旁边，并提供另一个变量进行检查。这将检查所有给定的变量是否在一次检查中都未定义，并且如果存在则仅执行添加的操作代码。
