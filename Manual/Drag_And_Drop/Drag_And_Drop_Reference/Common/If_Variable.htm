如果可变

此操作用于使用特定表达式对照给定值检查给定变量的状态。 “ if”语句将始终返回 true或者 false并且用于检查的表达式可以是以下任意一种：

  • 等于-变量和值都完全相等
  • 小于-变量小于值
  • 大于-变量大于值
  • 小于或等于-变量小于或等于值
  • 大于或等于-变量大于或等于值

如果标记“ Not”自变量，则上面的表达式将被否定，例如，“等于”变为“不等于”。还要注意，变量和值参数都不限于数值，并且可以是字符串，指针，资源ID或函数可以返回或使用的其他任何东西（有关更多信息，请参见数据类型部分），但是您不能在没有错误的情况下比较两种不同的数据类型-例如，如果变量包含一个字符串，并且您尝试检查它是否等于数字，则将出现语法错误。

重要说明：由于浮点精度问题，可能会返回以检查两个值是否完全相等 false，因为一个可能正好是1，而另一个可能是1.00000000000001。通过在检查之前使用“十进制到整数”操作或使用“大于”或“小于”表达式，可以避免这种情况。
