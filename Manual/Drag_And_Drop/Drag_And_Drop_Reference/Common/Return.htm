返回

该动作在创建动作块功能时使用，用于将值返回给调用脚本的实例。返回值可以是变量可以保存的任何值（例如字符串，数字值，指针或资源ID），也可以是变量本身。请注意，当我们在函数中调用return动作时，该函数将退出-返回给定的值-并且返回后将不会再调用其他动作，尽管如果您在调用该函数后在事件中有动作，这些动作仍然可以运行。
