设置全局变量

此操作用于为全局变量分配值（在过程中创建值）或用于更改现有全局变量的值。全局变量是“不”属于特定实例而是整个项目的变量，因此可以被任何实例，脚本或代码访问和更改。有关全局变量的更多高级信息，请参见此处。

给出全局变量的名称，然后给出值，可以根据需要将其相对于原始值添加。如果您提供一个新的变量名（即：以前在整个游戏中从未分配过的变量），则这将创建新的全局变量并为其分配您给定的值，而如果您给出一个现有的全局变量名，它将只是将值更改为新值。请注意，选中Relative会将值添加到现有的全局变量值（使用负数减去），还可以使用表达式s和先前创建的变量作为值。

值得注意的是，当我们谈论“值”时，我们不仅指数值，因为变量可以是字符串，指针，资源ID或函数可以返回或使用的其他任何东西。如果是字符串，则检查Relative会将值与以前保存的变量连接起来，因此，如果您的变量是“ Hello”，并且使用相对检查为它分配了“ World”，则最终变量值将是“ Hello World” ”。但是，必须注意确保变量值都属于相同的数据类型，因为尝试添加例如整数和字符串会导致编译器错误。

您还可以通过单击加号图标在同一操作中添加其他变量在操作旁边，并给出另一个全局变量名称和值。有关变量的更高级信息，请参见“变量和变量范围”部分。
