清晰的数据结构

该操作将清除给定数据结构中的所有值。您必须首先选择要清除的数据结构的类型，然后指定数据结构的索引（索引是最初创建数据结构时返回并存储在变量中的值）。除网格外，已清除的数据结构被认为是“空”的（即：它没有任何条目）。对于网格，此操作将简单地将所有网格单元清除为0。

注意：这不会从内存中释放数据结构，只会清除所有信息。要释放分配给数据结构的内存，必须使用动作“释放数据结构” 。
