获取网格值

通过此操作，您可以将网格数据结构中的特定单元格设置为一个值。您提供保存网格索引的变量（由动作Create Grid返回），然后给出要设置的单元格的x和y位置（列和行），最后是该单元格的值。您还必须设置一个目标变量（可以将其标记为临时局部变量）来存储返回的单元格值，并且该值可以是任何有效的数据类型。请注意，单元格的x和y位置必须是网格边界内的整数。
