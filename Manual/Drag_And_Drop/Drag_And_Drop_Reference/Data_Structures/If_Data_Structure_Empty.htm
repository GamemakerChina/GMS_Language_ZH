如果数据结构为空

通过此操作，您可以检查以查看给定的数据结构是否为空。首先，您需要选择要检查的数据结构类型（列表，地图或堆栈-无网格，因为它们甚至不能为空），然后提供包含数据结构索引的变量（由）返回操作创建列表，创建地图和创建堆栈）。动作将返回 true如果数据结构为空，并且 false如果不是这样。请注意，您可以在操作中标记“ not”，以通过检查数据结构是否为空来检查数据结构是否确实保留了数据。
