如果数据结构存在

通过此操作，您可以检查变量以查看其是否为数据结构建立索引。首先，您需要选择要检查的数据结构类型（列表，地图，网格或堆栈），然后提供包含数据结构索引的变量（由操作Create List ， Create Map返回） ，创建网格和创建堆栈）。动作将返回 true如果数据结构存在，并且 false如果没有。请注意，您可以在操作中标记“ not”，以检查变量是否不包含数据结构索引。
