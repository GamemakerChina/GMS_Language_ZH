插入清单

通过此操作，您可以恢复给定列表中某项的索引位置。您提供存储列表索引的变量（由动作Create List返回）和要在列表中查找索引位置的值，以及一个目标变量，用于存储该项目返回的索引位置（该变量可以标记为在脚本或事件结束之前使用的临时局部变量。将值插入列表时，要插入的给定索引上方的所有索引都会上移一个，因此，如果列表中包含4个项目（索引位置0到3），并将新项目放在索引位置2 ，前一个项目2将成为项目3，而前一个项目3将成为项目4。请注意，如果您要查找的值在列表中不存在，则将没有列表索引位置要返回，因此返回值将为-1。
