设置地图值

通过此操作，您可以将键/值对添加到地图数据结构中。您提供保存地图索引的变量（由动作Create Map返回），然后提供一个“键”（它是地图中值的标识符），然后提供要与该键关联的“值” 。两个键都可以是整数值（例如0、200、36712等）或字符串（例如“ Name”，“ Shield”等...），但是值可以是任何有效的数据类型。
