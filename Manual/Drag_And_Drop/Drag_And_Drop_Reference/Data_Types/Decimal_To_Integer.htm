小数到整数

此操作将采用一个十进制数字并将其转换为整数。您提供要转换的值，然后选择将其变为整数值应进行的舍入类型。可用的舍入类型为：

  • 舍入：取一个十进制数字，并将其向上或向下舍入为最接近的整数。在特殊情况下，所提供的数字正好是整数的一半（例如1.5、17.5，-2.5等），该数字将四舍五入为最接近的偶数，例如2.5将四舍五入为2，而3.5会四舍五入为4。这种类型的舍入称为bankers舍入，并进行大量迭代，或者使用浮点数学时，与传统的“大于0.5时进行舍入，否则向下舍入”相比，舍入在统计上更好。 “ 方法。
  • 小区：这将需要一个十进制值，并四舍五入它为最接近的整数。不同于轮（以上）这并没有考虑到的值的小数部分，所以像5.0000001许多仍然被调高至6值得注意的是，如果你使用的是随机行为获取随机数生成一个随机十进制值，然后调用此操作将其四舍五入，则可能会得到意外的结果。随机操作可能会返回0，并且在其上使用“ Ceil”不会给出1而是0，因此，如果您获得0到9之间的随机小数并使用此函数，则可能希望得到1到10之间的整数，但是在现实中，您可能会得到0到10之间的整数。这是一个很小的可能性，但是在使用此操作时应予以考虑。
  • 楼层：这将需要一个十进制值，并四舍五入下来到最接近的整数值。与四舍五入（以上）不同，这不考虑值的小数部分，因此，像5.9999999这样的数字仍将下限降至5。

选择舍入类型后，必须提供要舍入的数字（可以是变量），然后提供目标变量以将新的整数值返回到（可以将其标记为临时局部变量）。
