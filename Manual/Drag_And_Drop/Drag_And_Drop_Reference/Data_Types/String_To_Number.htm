字符串转数字

此操作可以将字符串转换为数字。您提供要更改的字符串，然后提供目标变量以将实数返回给该变量（可以将该变量标记为临时局部变量）。请注意，如果字符串中包含数字和小数点以外的任何字符，则不会进行转换，尽管如果字符串以数字开头，则将转换这些字符，并排除字符串的其余部分。重要的是要注意，当将小数传递给该操作时，返回的字符串将仅四舍五入到小数点后两位。下表是输入值以及您期望返回的值的表

细绳返回值
"123" 123
"123 Hello World" 123
"Hello World" 0（默认情况下，如果无法转换字符串，则将返回0）
"Hello World 123" 0
"123.45678" 123.46
