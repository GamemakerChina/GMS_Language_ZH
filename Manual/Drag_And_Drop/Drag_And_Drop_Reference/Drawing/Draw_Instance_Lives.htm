绘制实例生命

此动作将绘制一个精灵，以表示实例当前具有的每个生命。绘制的数字取自本地实例变量的当前值 lives，这是一个特殊变量，如果您使用任何需要它的操作，它将在实例中自动创建。您可以指定精灵来绘制生命和堆栈顺序，可以是行（水平，从左到右），也可以是列（垂直，从上到下），以及位置。该位置可以是房间内的绝对位置，也可以是相对于执行绘图的实例的位置的绝对位置，并且图像之间的间距将基于子画面的宽度或高度。请注意，这只是绘制一个静态图像-给定Sprite的初始单个图像（第0帧），所有其他帧都将被忽略，通过更改实例变量（例如 image_xscale或者 image_blend）。字幕文本将在lives值之前绘制，如果是文本，则应使用“”换行，并且两者都将使用当前的绘制字体，颜色和对齐方式进行格式化。

注意：此操作仅在各种Draw Events中使用，如果在其他地方使用则不会绘制任何内容。
