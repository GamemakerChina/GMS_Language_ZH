绘制实例分数

通过此操作，您可以绘制实例的得分值。当您GameMaker Studio 2使用任何比分操作时，调用它的实例将具有一个新的实例范围变量，称为 score添加。然后可以像检查其他任何变量一样检查和设置此变量。在这种情况下，变量将与给定的标题文本一起绘制到屏幕上的给定位置。字幕文本将在得分值之前绘制，如果是文本，则应使用“”换行，并且两者都将使用当前的绘制字体，颜色和对齐方式进行格式化。

注意：此操作仅在各种Draw Events中使用，如果在其他地方使用则不会绘制任何内容。
