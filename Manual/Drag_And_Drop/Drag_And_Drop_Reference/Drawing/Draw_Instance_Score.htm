绘图实例得分

通过这个动作，你可以画出实例的分数值。当你在GameMaker Studio 2中使用任何得分动作时，调用它的实例将被添加一个新的实例范围变量，名为score。然后，这个变量可以被检查 和设置，就像其他变量一样。在这个动作的情况下，这个变量将被绘制在屏幕上的指定位置，同时还有你想提供的任何标题文本。标题文本将被画在分数值的前面，如果是文本，它应该被包起来。 如果是文本的话，应该用""来包裹，并且两者都将使用当前绘制的字体、颜色和对齐方式进行格式化。

注意：这个动作只在各种抽奖活动中使用，如果在其他地方使用，则不会抽到任何东西。
