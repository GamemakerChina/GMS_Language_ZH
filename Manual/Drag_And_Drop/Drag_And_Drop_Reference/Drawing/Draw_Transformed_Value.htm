提取转换价值

通过此操作，您可以在屏幕上绘制字符串标题和/或变量或值。 caption参数是您希望在值之前添加的字符串，并且可以根据需要将其保留为空白，并且该值可以是变量，固定值或其他字符串-在这种情况下，应将其括在引号中，并如果需要，也可以留空。然后，您可以提供绘制字符串的位置，可以是绝对位置，也可以是相对于实例绘制文本的位置。此外，您可以设置文本的比例，沿着水平或垂直轴拉伸文本，以及旋转角度（旋转文本的角度为逆时针方向，因此将其设置为90°会将其向左旋转-90 °右）。缩放是通过给定的系数完成的，因此0.5的缩放比例将是文本宽度的一半，而2的缩放比例则将文本的宽度增加一倍，其中1是不缩放的默认值。请注意，文本在绘制时的外观将取决于所使用的字体，所使用的颜色和对齐方式。可以在绘制之前更改所有这些内容，如下例所示。

注意：此操作仅在各种Draw Events中使用，如果在其他地方使用则不会绘制任何内容。
