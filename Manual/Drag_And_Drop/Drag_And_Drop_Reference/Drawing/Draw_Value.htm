通过此操作，您可以在屏幕上绘制字符串标题和/或变量或值。 caption参数是您希望在值之前添加的字符串，并且可以根据需要将其保留为空白，但如果希望包含文本，则应将其用引号引起来，并且该值可以是变量，固定值或其他值字符串-在这种情况下，也应将其括在引号中-但不能留空。然后，您可以提供绘制字符串的位置，可以是绝对位置，也可以是相对于实例绘制文本的位置。请注意，文本在绘制时的外观将取决于所使用的字体，所使用的颜色和对齐方式。可以在绘制之前更改所有这些内容，如下例所示。

注意：此操作仅在各种Draw Events中使用，如果在其他地方使用则不会绘制任何内容。
