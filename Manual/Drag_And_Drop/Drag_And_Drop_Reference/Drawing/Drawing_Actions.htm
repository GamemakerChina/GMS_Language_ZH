绘图动作库

在“绘图”动作库中，您可以找到绘制精灵，文本或形状以及设置某些绘图属性所需的动作。这些操作中的大多数仅用于对象的各种绘图事件，如果在绘图事件之外使用，则可能不起作用。 Set动作是一个例外，可以将其添加到任何事件中，并且此后将影响所有实例的所有图形。

重要的是要注意，如果您将任何动作添加到对象的主Draw事件中，那么GameMaker Studio 2使用Draw之类的动作绘制，否则它将不会绘制已分配给实例的精灵。自我。基本上，仅当Draw事件中没有GameMaker Studio 2才会默认绘制分配给实例的任何精灵。

画自我
提款价值
提取转换后的价值
画精灵
绘制精灵变形
绘制堆叠的精灵
画矩形
绘制渐变矩形
画椭圆
绘制渐变椭圆
画线
绘制健康栏
绘制实例分数
绘制实例运行状况
绘制实例生命
设置绘制颜色
获取绘图颜色
设置绘制Alpha
获取绘图Alpha
设定字型
获取绘图字体
设定文字对齐方式
获取文本对齐
