设置绘制颜色

通过此操作，您可以设置颜色以用于任何形状（矩形/椭圆/线）或随后绘制的任何文本。请注意，不必在每个平局动作之前都调用此函数，而只能在游戏开始时设置一次。例如，如果您所有的文本都将变为黄色，则可以在游戏第一个房间的“创建事件”中调用此操作，然后所有文本（和形状）将从此开始被绘制为黄色。但是，如果您需要使用多种不同的颜色，则需要在需要不同颜色的每个文本或形状之前调用此名称。

该操作的底部具有一个复选框，用于“使用alpha” 。这样一来，颜色的Alpha分量也可以用于设置绘制Alpha（与使用Set Draw Alpha设置相同。如果您不希望包含Alpha分量而只具有纯色，则可以将其禁用（默认情况下启用）。
