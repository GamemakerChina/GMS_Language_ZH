设定文字对齐方式

此操作将为所有后续绘制文本操作设置字体对齐方式。您可以将水平对齐方式设置为左，中或右，将垂直对齐方式设置为上，中或下，并且文本将相对于绘制动作的x / y位置对齐，如下图所示：

文字水平向左对齐
文本与中心水平对齐
文字水平向右对齐
文本与顶部垂直对齐
文字与中间垂直对齐
文字与底部垂直对齐

请注意，这将设置对齐方式以用于所有后续的绘画文本调用，例如，可以在游戏开始时调用一次（无论如何，不仅是绘画事件），而且所有绘画文本操作都将使用设置的对齐方式。但是，如果您需要多个绘图文本操作以不同方式对齐，则需要在希望以不同方式对齐的每个绘图文本操作之前调用此操作。
