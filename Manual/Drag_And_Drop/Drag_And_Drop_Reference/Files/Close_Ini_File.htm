关闭ini文件

此操作将关闭当前打开的Ini文件。完成对任何打开的Ini文件的读取或写入操作后，应立即调用此操作（使用“打开Ini File ”操作打开Ini文件）。如果在使用“写入Ini文件”操作之后不使用此操作，则不会将任何内容写入磁盘，因为文件信息会保留在内存中，直到调用此操作为止，此时将从内存中取出该文件并将其写入磁盘上的文件。另请注意，如果您尝试打开Ini文件而不先关闭另一个文件（或相同文件），则会收到错误消息。
