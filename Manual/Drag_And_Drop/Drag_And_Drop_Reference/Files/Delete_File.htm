删除文件

通过此操作，您可以删除以前创建的文件。您提供要删除的文件的文件名（作为字符串）。请注意，您不能删除游戏捆绑包中包含的任何文件，只能删除使用“关闭Ini”或“保存缓冲区”操作创建的文件（有关更多信息，请参见文件系统部分）。
