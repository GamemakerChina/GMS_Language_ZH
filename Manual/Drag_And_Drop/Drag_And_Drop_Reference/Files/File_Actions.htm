文件操作库

文件操作处理两种不同的文件类型-缓冲文件和Ini文件-并且还具有某些通用文件操作来重命名或复制现有文件。

通过从以前创建的缓冲区中保存数据来创建缓冲区文件，然后可以在以后的任何时候再次将这些数据加载到缓冲区中。您可以在这里找到有关缓冲区的更多信息： Buffer Actions 。

Ini文件是小型，轻便的文件，与大多数平台兼容。它们非常适合存储小信息，例如界面首选项，本地高分，级别数据等，并且非常易于使用。不必先创建Ini文件即可使用这些操作，并且您可以从不存在的Ini文件中读取内容，您将简单地得到 default返回值（您指定的值），但是我们建议您至少创建一个“基本” ini文件以便在使用操作之前打开和修改。只需调用Open Ini File操作然后再单击Close Ini File即可创建此基本ini文件，因为关闭该文件会将其写入磁盘，也可以在资源浏览器的“包含的文件”中包含一个。如果您使用资产浏览器中包含的文件作为基本Ini，则还应该阅读手册中有关文件系统如何工作的部分。

注意：对于具有本地化版本的游戏，从外部文件中读取带重音的字母非常重要。这意味着您应该首先以UTF8格式创建ini文件，然后将其GameMaker Studio 2 ，以便在运行游戏时将其导出并代替由ini创建的默认ANSI格式ini文件使用。以前未提供任何功能时，该功能将起作用。这样，您可以在保持所有重音和非罗马字母的情况下正确地对其进行读写。
