如果文件存在

通过此操作，您可以检查是否存在具有给定名称的文件。您选择要检查的文件（作为字符串，包括扩展名），函数将返回 true如果存在，并且 false如果没有。请注意，这将返回 true如果该文件作为附带文件存在于游戏包中，或者该文件是否在游戏运行时被保存了（有关更多信息，请参见文件系统部分）。
