加载缓冲区

通过这个动作，你可以加载一个之前用动作Save Buffer保存的缓冲区。这个动作将创建一个新的缓冲区，将数据文件加载到其中，并返回唯一的ID值，以识别缓冲区中的 变量。你提供要加载的文件的名称（作为一个字符串）和一个变量来存储返回的缓冲区ID。注意，如果你将变量标记为临时的本地变量，你需要确保在动作脚本或事件结束前删除该函数创建的缓冲区。 否则，当变量被丢弃时，你将失去对它的引用并得到一个内存泄漏。
