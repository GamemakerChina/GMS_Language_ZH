打开ini文件

通过此操作，您可以打开一个Ini文件进行读取和/或写入。如果ini文件并不在您检查存在的位置， GameMaker Studio 2可以创建一个，但只有当你向其中写入数据。如果仅从Ini文件中读取信息，则将返回read函数的默认值，但实际上不会创建Ini文件。
