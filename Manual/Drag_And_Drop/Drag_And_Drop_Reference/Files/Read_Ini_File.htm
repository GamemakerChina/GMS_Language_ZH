读取Ini文件

通过此操作，您可以从Ini文件中读取一段数据。您必须事先使用“打开Ini文件”操作打开Ini文件以进行读取/写入，然后通过此操作指定要读取的数据类型（实数或用“”括起来的字符串），以及节标题和该节中的键以从中获取数据。您还可以提供默认值，该值可以是字符串的实数，并且类型应与操作中设置的“类型”一致。如果发生以下任何情况，则此操作将返回此默认操作：

  • 您打开的Ini文件不存在
  • 您引用的节标题不存在
  • 您尝试从中获取数据的密钥不存在

在上述任何一种情况下，您将获得返回的默认值，而不是Ini文件中的值。返回的任何值都将存储在您指定的目标变量中，如果将其标记为临时局部变量，也可以在该时刻创建。

重要的！尝试使用“打开Ini文件”操作从以前未打开的Ini文件中读取数据会导致严重错误并导致游戏崩溃。
