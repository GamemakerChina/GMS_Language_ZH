保存缓冲区

通过这个动作你可以保存一个包含缓冲区数据的文件。你提供唯一的缓冲区ID（当你用Create Buffer创建缓冲区时返回的），并给出一个文件名（作为一个字符串。 缓冲区将被写入该文件，以便再次使用加载缓冲区加载。
