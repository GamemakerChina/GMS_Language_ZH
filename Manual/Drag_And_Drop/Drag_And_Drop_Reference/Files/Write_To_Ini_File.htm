写入Ini文件

通过此操作，您可以将数字值或字符串写入Ini文件。您必须事先使用“打开Ini文件”操作打开Ini文件以进行读/写操作，然后通过此操作指定要写入的数据类型（实数或用“”括起来的字符串），以及部分标题，该部分中的键以及该键的值。为了帮助可视化Ini文件的结构，请参见以下图像：

如您所见，Ini文件分为多个部分-每个部分都有其自己的标题名称-然后每个部分随后都被分为各种键/值对（非常类似于Map Data Structure ）。请注意，您可以使用任何文本处理软件（例如Windows Notepad）自己创建Ini文件，然后将它们作为包含文件包含在其中，以在游戏中进行读取和写入。

重要的！尝试使用“打开Ini文件”操作从以前未打开的Ini文件中读取数据会导致严重错误并使游戏崩溃。

重要的！这只会将数据写入内存，最后将数据写入磁盘上的文件，您必须调用Close Ini File ，否则数据将不会保存。
