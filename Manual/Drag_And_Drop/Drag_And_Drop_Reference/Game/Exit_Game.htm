退出游戏

通过此操作，您可以结束游戏，并触发“其他游戏结束事件” 。请注意，这不会在调用操作的那一刻发生，而是在当前事件或脚本结束时发生，因此在调用此操作之后您所执行的任何操作仍将继续运行，直到到达事件/脚本的结尾为止。

请注意，此操作具有以下限制：

  • iOS目标上的错误，因为它是针对其市场条件的。
  • 在HTML5和Windows UWP目标上调用时，它将无效。
  • 它将在Windows （包括为Steam ）， Android ， Ubuntu （ Linux ）和Mac上运行。
