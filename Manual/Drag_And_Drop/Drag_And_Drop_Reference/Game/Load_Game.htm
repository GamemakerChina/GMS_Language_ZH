加载游戏

此操作将加载以前使用“保存游戏”操作保存的游戏。您提供要加载文件的文件名（作为字符串，包括扩展名），游戏将被加载并从保存的点继续运行。请注意，这并非设计为通用的保存/加载系统，尝试加载从项目的前一次运行中保存的游戏可能会产生错误（尤其是如果您使用过诸如“数据结构”或“粒子”之类的东西），因此应该仅尝试加载在单个游戏进行过程中保存的游戏（用于检查点之类的东西）。还要注意的是，文件不会被加载，直到当前事件或脚本的末尾，因此负载动作被称为后的任何行动仍将进行。有关加载保存的游戏数据的更全面的方法，请参见“文件操作” 。
