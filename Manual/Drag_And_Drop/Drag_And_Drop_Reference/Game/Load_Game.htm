加载游戏

这个动作将加载一个先前用 "保存游戏"动作保存的游戏。你给出要加载的文件名（作为一个字符串，包括扩展名），游戏将被加载并继续运行。 从保存点继续运行。请注意，这不是一个通用的保存/加载系统，试图加载一个从以前的项目运行中保存的游戏可能会出现错误（特别是如果你使用了像数据结构或粒子这样的东西），因此，你应该只尝试加载在一个游戏过程中保存的游戏（对于像检查点这样的东西）。还要注意的是，在当前事件或脚本结束之前，文件是不会被加载的，所以在调用加载动作之后的任何动作仍将被执行。关于加载保存的游戏数据的更全面的方法，见文件动作。
