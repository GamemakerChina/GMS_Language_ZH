重新开始游戏

通过此操作，您可以重新启动游戏。这类似于第一次运行游戏，因此将触发“其他游戏开始事件”以及“其他游戏结束事件” 。但是，应该注意的是，除非明确编码，否则不会重新初始化全局变量-这意味着，如果您未在游戏的第一个房间中定义和设置它们，它们将保留其先前的值-以及数据结构并且粒子也不会被释放，因此应该在调用“重新启动游戏”动作之前或在“游戏结束事件”中完成。另请注意，游戏不会在调用该动作的那一刻重新启动，而是在放置该动作的脚本或事件结束时重新启动，因此，如果在此动作之后放置了任何动作，它们仍会被调用，并且游戏在块的末尾重新启动。
