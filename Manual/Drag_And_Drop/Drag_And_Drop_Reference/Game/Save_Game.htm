保存游戏

此操作将把游戏基本保存到给定文件中。您提供一个文件名（作为字符串，包括扩展名），该操作将在当前游戏状态之外执行有限的保存。了解这一行动的目的是采取当前的游戏状态的“快照”是很重要的，并且是使用更多的检查站系统不是万能保存系统，因为它不保存任何动态的资源（如数据结构或粒子）。因此，应非常小心地使用该操作，以免发生错误或内存泄漏。有关保存游戏数据的更全面的方法，请参见“文件操作” 。
