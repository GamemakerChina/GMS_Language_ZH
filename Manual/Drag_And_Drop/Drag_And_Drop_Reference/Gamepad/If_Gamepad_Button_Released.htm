如果释放游戏手柄按钮

此操作检查何时为给定的游戏手柄索引释放了游戏手柄按钮。它将返回 true在释放按钮的单个游戏中打勾，然后 false在其他所有时间。如果启用 not改性剂，这一行动将得到扭转，并检查按钮没有被释放，则返回 false在游戏滴答声中已发布 true在其他所有时间。

如果选中的按钮是模拟按钮，则只有在按钮值低于使用“设置游戏手柄按钮阈值”操作设置的阈值时，才能检测到释放。
