设置游戏手柄轴死区

此动作可用于设置游戏手柄杆轴的“死区”。您指定要设置的游戏手柄索引，然后将值设置为0到1，并且如果输入量小于给定值，则游戏手柄杆轴被认为是0。请注意，这是一个全局设置，将影响棒的所有轴都已连接。

请注意，这不会修改Get Gamepad Axis（获取游戏手柄轴）操作的输出，该操作将始终返回-1和0、0和1的值。例如，将死区设置为0.2将意味着向右推动操纵杆只会开始如果原始轴值超过0.2，则返回0-1中的值-因此，当原始值是0.2时，返回值将为0，当原始值是0.5时，返回值将为0.375，或者如果原始值值为0.9，则返回值为0.875。
