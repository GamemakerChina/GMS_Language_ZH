设置游戏手柄按钮阈值

此操作可用于设置游戏手柄模拟按钮的“阈值”。您指定要设置的游戏手柄索引，然后将其值设置为0到1，并且如果模拟按钮输入量小于给定值，则游戏手柄按钮将被视为0。请注意，这是一个全局设置，将影响所有连接游戏手柄的模拟按钮。该值将在给定游戏手柄的所有按下，按下和释放检查中使用，但将被“获取游戏手柄触发器”操作忽略。
