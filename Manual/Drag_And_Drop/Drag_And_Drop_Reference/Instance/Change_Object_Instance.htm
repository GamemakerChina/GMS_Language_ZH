更改对象实例

通过此操作，您可以将一个实例更改为另一个实例。您需要指定将其实例更改为另一个实例的对象，该操作将更改该实例，并执行初始实例的Destroy事件和Clean事件以及新实例的Create事件。请注意，如果您未指定方向或速度等，则新实例将“继承”这些值。例如，如果初始实例具有 image_angle（旋转）为180，以每游戏跳动2px的速度移动，并且 image_speed（动画速度）为0.5，则除非在新实例的Create事件中明确设置，否则更改为的实例将保持这些值。

值得注意的是，更改实例意味着您不能在下一个游戏滴答之前对该实例执行任何进一步的操作，尤其是尝试访问变量等...，因为这将导致错误。基本上，您可以更改实例，但实际上直到当前步骤结束它才可用，因此可以直接访问它包含的任何变量（例如，调用 inst_changed.x） 不管用。
