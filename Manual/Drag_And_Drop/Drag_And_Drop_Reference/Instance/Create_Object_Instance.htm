创建对象实例

通过此操作，您可以从资产浏览器创建任何对象的实例。您提供对象的名称和要在其中创建对象的房间内的x / y位置，以及要在其上创建实例的层的名称（它必须是Instance Layer ）。您可以勾选“ relative”标志，以相对于运行动作代码块的实例定位实例。每当您在游戏中创建实例时，都会为其分配一个唯一的ID值，然后您可以使用该ID值在后续操作中标识该实例，因此，该操作允许您定义一个“目标”变量来保存该ID值（如下面的示例）。请注意，可以将目标标记为临时（局部）变量，该变量将为操作创建，并可以在所有后续操作中使用。
