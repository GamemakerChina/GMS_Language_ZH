如果实例存在

这个动作可以用来检查游戏室中是否存在一个特定对象的实例，或者一个特定实例本身是否存在。你从资产资源管理器中给出对象，如果房间中存在任何实例，那么该动作将返回true，否则将返回false。注意，你可以在这里使用一个变量，为特定的实例提供唯一的ID（例如由动作Create Object Instance返回的），或者甚至是实例ID 值，甚至是房间编辑器在你添加实例时返回的实例ID。

如果你标记了 "不 "的参数，那么该动作将检查是否没有实例存在，如果没有发现，将返回真，如果甚至有一个存在，将返回假。

请注意，要把动作添加到 "如果 "块中，应该把它们放到动作的一侧，如下图所示。

现在，如果 "if "评估为真，这些动作将被运行，而在其他地方放弃的任何动作将在 "if "块之后执行。
