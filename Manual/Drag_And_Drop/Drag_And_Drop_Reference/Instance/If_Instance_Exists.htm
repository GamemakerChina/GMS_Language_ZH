如果实例存在

此操作可用于检查游戏室中是否存在特定对象的实例，或者特定实例本身是否存在。您从资产浏览器提供对象，如果房间中存在任何实例，则操作将返回 true否则它将返回 false。请注意，您可以在此处使用具有特定实例唯一ID的变量（例如，由动作Create Object Instance返回），或者甚至添加房间实例时由房间编辑器返回的实例ID值。

如果标记“ Not”参数，则该操作将检查是否不存在实例，如果找不到，它将返回 true如果存在，它将返回 false。
