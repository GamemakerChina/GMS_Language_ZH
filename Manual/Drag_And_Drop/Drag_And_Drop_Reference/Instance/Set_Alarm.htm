设定闹铃倒数

该操作块将给定的警报设置为倒计时，并且当警报达到零时，将触发关联的警报事件。您提供的警报编号为0到11，然后提供将警报设置为的值。所使用的值是触发警报事件之前要通过的“游戏步骤”数，您可以将其设置为相对于当前警报值（对当前值添加或减去）。请注意，该值必须是整数值，您可以将其设置为-1以停止警报。请注意，当警报倒计时时，事件在到达0时运行，但是警报倒计时到-1以表明事件已结束。将警报设置为0不会运行该事件，因为它随后将向下计数到-1，因此要立即运行警报，应将其设置为1。请注意，您可以通过在同一操作中单击加号图标在动作旁边。
