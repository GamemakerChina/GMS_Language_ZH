设定动画速度

该操作块设置了 image_speed（动画速度）分配给实例的精灵。您可以在Sprite Editor中设置理想的每秒帧数，然后可以使用此值对其进行修改。默认的动画速度设置为1，这意味着游戏将尝试维持您为精灵定义的每秒帧数，但是您可以使用大于1的值来更快地进行动画处理（可能会跳过帧），或者十进制值以使动画变慢（将在各个步骤中显示帧），甚至为负值以反转动画的速度。
