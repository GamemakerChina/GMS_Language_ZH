设置动画速度

这个动作块设置分配给实例的精灵的image_speed（动画速度）。你可以在Sprite Editor中设置理想的每秒帧数，然后你可以用这个值 来修改它。默认的动画速度被设置为1，这意味着游戏将试图保持你为精灵定义的每秒帧数，但你可以使用大于1的值来加速动画制作（帧数可能被跳过）。 或者用小数点的值来使动画速度变慢（帧数将在不同的步骤中显示），甚至用负值来扭转动画的速度。
