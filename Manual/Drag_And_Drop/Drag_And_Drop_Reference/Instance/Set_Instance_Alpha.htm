设置实例Alpha

该操作块设置了 image_alpha实例的值。图像alpha是用于绘制分配给该实例的任何sprite的“透明度”值，其值介于0和1之间，其中0将是完全透明的，而1将是完全不透明的。请注意，如果子画面具有透明或半透明像素，则它们的透明度将按数量缩放。因此，如果您在子画面中具有约50％的透明像素并将实例alpha设置为0.5，则该透明像素的最终alpha将是25％。您可以将此值设置为相对值

请注意，为使此变量的更改可见，实例应没有绘制事件（因此GameMaker Studio 2将默认绘制精灵）或使用Draw Self绘制 行动。
