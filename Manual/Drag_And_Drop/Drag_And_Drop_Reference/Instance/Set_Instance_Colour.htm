设置实例颜色

该操作块设置了 image_blend实例精灵与“混合”的颜色。默认值为十六进制代码$FFFFFFFF （它表示使用ARGB格式的白色），这将在不进行混合和使用全alpha的情况下绘制精灵。任何其他值（包括内部颜色常量，例如 c_red， 或者 c_aqua）将混合指定的颜色与原始子画面以及设置 image_alpha组件的实例，但只有当使用阿尔法复选框被选中。这将覆盖以前使用“设置实例Alpha ”操作设置的所有Alpha值。

请注意，为使此操作中的更改可见，实例应没有绘制事件（因此GameMaker Studio 2将默认绘制精灵）或使用“绘制自我”操作绘制。同样要注意的是，您应该尝试限制HTML5平台上的混合（除非使用WebGL），因为每个混合精灵必须单独缓存，因此拥有许多混合精灵可能会对性能产生不利影响。
