设置实例旋转

该操作块设置了 image_angle（旋转）实例。角度以度为单位，右侧为0°，向上为90°，左侧为180°，向下为270°。将此变量设置为0以重置实例，这意味着它将绘制在Sprite编辑器中定义的所有分配给Sprite的Sprite。请注意，为使此操作中的更改可见，实例应没有绘制事件（因此GameMaker Studio 2将默认绘制精灵）或使用“绘制自我”操作绘制。
