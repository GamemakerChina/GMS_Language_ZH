设置实例比例

该操作块设置了 image_xscale和 image_yscale实例的值。这里的输入值是修饰符，将应用于分配给实例的精灵，比例为1（默认值）表示无比例缩放（1：1），较小的值将按比例缩小（例如0.5，则一半精灵的宽度），更大的值将按比例放大。您可以使用负值来翻转精灵并缩放它，除非所使用的值正好是-1（在这种情况下，精灵只是围绕其原点翻转或镜像而没有缩放）。您可以选择设置相对于当前值的水平或垂直比例，在这种情况下，您将要从当前比例值中添加或减去新值。

请注意，更改这些值还将更改实例检测冲突的方式，除非您单独提供了一个 mask_index（碰撞遮罩）在“对象编辑器”中。请注意，要使此变量的更改可见，实例应没有绘制事件（因此GameMaker Studio 2将默认绘制精灵）或使用“绘制自我”操作绘制。
