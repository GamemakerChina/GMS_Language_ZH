设置精灵

该操作块设置了 sprite_index将实例从资源浏览器转移到另一个精灵。选择要更改为的精灵，然后提供要显示的图像（动画帧）。请注意，如果您不希望在设置帧后让精灵进行动画处理，则需要使用“设置动画速度”操作并将其设置为0，或者如果您希望精灵继续进行动画处理，则将其值设置为使用 image_index内置变量而不是变量。

更改精灵也将更改实例的碰撞蒙版，除非您还指定了一个单独的 mask_index在对象编辑器中。请注意，为使此操作中的更改可见，实例应没有绘制事件（因此GameMaker Studio 2将默认绘制精灵）或使用“绘制自我”操作绘制。
