如果健康

此操作用于检查实例变量的值 health使用特定的表达式。您提供要检查的表达式类型和检查当前值的值 health对，“ if”语句将始终返回 true或者 false取决于使用的表达式和值。可用的表达式为：

  • 等于-变量和值都完全相等
  • 小于-变量小于值
  • 大于-变量大于值
  • 小于或等于-变量小于或等于值
  • 大于或等于-变量大于或等于值

如果您标记Not参数，则上面的表达式将是取反的，例如，“等于”变为“不等于”，因此您将检查是否 health值不等于给定值。

重要说明：由于浮点精度问题，可能会返回以检查两个值是否完全相等 false，因为一个可能正好是1，而另一个可能是1.00000000000001。通过在检查之前使用“十进制到整数”操作或使用“大于”或“小于”表达式，可以避免这种情况。
