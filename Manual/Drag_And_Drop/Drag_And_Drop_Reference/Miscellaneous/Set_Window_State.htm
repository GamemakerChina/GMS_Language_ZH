设定视窗状态

通过此操作，您可以将游戏窗口的状态设置为全屏或窗口化。这里有三个选项：

  • 切换-选中此选项后，每次调用该操作时，它将在全屏状态和窗口状态之间切换窗口
  • 全屏-这将使窗口从窗口状态变为全屏
  • 窗口化-如果在全屏模式下，这将使窗口返回到窗口化状态

请注意，游戏在全屏状态下的外观取决于目标平台的“游戏选项-图形”设置，因为如果游戏没有某种机制来适应不同的宽高比和分辨率，则可以进行拉伸（如果游戏选项设置为Full Scale ），或者您可以添加黑条（如果游戏选项设置为Keep Aspect Ratio ）。通常，如果没有合适的分辨率来处理不同的分辨率，则应针对要编译到的目标平台使用“保持纵横比”。
