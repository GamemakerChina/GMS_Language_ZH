如果按下

通过此操作，您可以轮询键盘以查看当前是否按住了任何键。动作将返回 true按住键盘键的每一步，或者 false否则，尽管如果您单击 not修改器，然后您可以查看该键是否没有帮助，即：该操作将返回 true虽然没有按键按下，false如果有。如果您只需要检查单个按键，则使用“如果按下按键”操作。
