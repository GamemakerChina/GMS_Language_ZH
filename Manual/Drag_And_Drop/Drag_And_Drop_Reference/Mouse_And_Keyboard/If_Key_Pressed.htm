如果按键

此操作检查是否初次按下键盘键。它将返回 true在单个游戏步骤（框架）上，首先按下键，然后 false在其他所有时间。如果启用 not改性剂，这一行动将得到扭转，并检查钥匙没有被按下时，返回 false在已被按下的游戏步骤（框架）上，true在其他所有时间。如果您需要检查是否按住了一个键，而不仅仅是初始按下事件，请使用If If Down动作。
