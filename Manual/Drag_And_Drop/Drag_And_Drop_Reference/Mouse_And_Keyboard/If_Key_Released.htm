如果密钥已释放

此操作检查何时释放了键盘键。它将返回 true在释放键的单个游戏步骤（框架）上 false在其他所有时间。如果启用 not改性剂，这一行动将得到扭转，并检查钥匙尚未释放，则返回 false在已发布的游戏步骤（框架）上 true在其他所有时间。
