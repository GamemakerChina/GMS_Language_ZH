如果钥匙被释放

这个动作检查键盘按键是否已经被释放。它将在按键被释放的单一游戏步骤（帧）中返回 "true"，在所有其他时间返回 "false"。如果你启用not修改器，这个动作将被 反过来，检查键盘是否被释放，在它被释放的游戏步骤（帧）上返回false，在其他时间返回true。

请注意，要把动作添加到 "如果 "块中，应该把它们放到动作的一侧，如下图所示。

如果 "if "评估为真，这些动作现在将被运行。 而在其他地方放弃的任何行动将在 "if "块之后执行。
