如果鼠标按下

通过此操作，您可以轮询鼠标以查看所选按钮当前是否处于按下状态。动作将返回 true按住按钮的每一步，或者 false否则，尽管如果您单击 not修改器，然后您可以检查按钮是否未按住，即：该操作将返回 true当没有按钮按下时 false如果有。如果只需要检查一个按钮，则使用“如果按下了鼠标”操作。
