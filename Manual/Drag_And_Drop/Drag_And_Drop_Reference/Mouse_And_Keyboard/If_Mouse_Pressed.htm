如果按下鼠标

此操作检查是否初次按下鼠标按钮。它将返回 true在单个游戏步骤（框架）上，首先按下按钮，然后 false在其他所有时间。如果启用 not修改器，此操作将被撤消，并检查按钮是否未被按下，返回 false在已被按下的游戏步骤（框架）上，true在其他所有时间。.如果您需要检查是否按住了一个按钮，而不仅是最初的按下事件，那么请使用If If Mouse Down动作。
