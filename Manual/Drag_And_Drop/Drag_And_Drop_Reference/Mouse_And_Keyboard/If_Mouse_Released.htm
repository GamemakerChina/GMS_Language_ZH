如果鼠标被释放

这个动作检查鼠标按钮何时被释放。它将在鼠标按钮被释放的单一游戏步骤（帧）中返回 "true"，在所有其他时间返回 "false"。如果你启用not修改器，这个动作 将被反转，并检查鼠标按钮是否被释放，在鼠标按钮被释放的游戏步骤（帧）上返回false，而在所有其他时间返回true。

请注意，要把动作添加到 "如果 "块中，应该把它们放到动作的一侧，如下图所示。

现在，如果 "if "评估为真，这些动作将被运行，而在其他地方放弃的任何动作将在 "if "块之后执行。
