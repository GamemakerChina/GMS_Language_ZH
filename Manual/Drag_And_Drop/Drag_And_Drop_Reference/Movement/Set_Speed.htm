设定速度

此操作用于设置实例的移动速度。速度计算为实例在单个游戏帧（步骤）中必须移动的像素数。默认情况下，速度值将添加到方向矢量-因此，如果将速度设置为2且方向设置为45°，则实例将在每个游戏帧上向右移动2个像素-但您也可以点击 direction字段，然后选择“水平”或“垂直”速度。

选择其中任何一个将仅针对水平或垂直矢量设置速度。请注意，您可以将值标记为相对于当前速度，这将与实例的实际速度相加或相减，还可以提供负数，这将使其与当前速度相反方向值。
