透明粒子系统

通过此操作，您可以将给定的粒子系统清除为默认状态，删除所有发射器（也无需调用“销毁粒子发射器”操作，因为这会自动执行）并从屏幕上清除所有粒子。您提供了要清除的系统的唯一ID值，该值是通过创建粒子系统操作创建系统时返回的。使用此操作时必须小心，因为它会破坏与粒子系统关联的所有发射器-这意味着房间中使用发射器的任何实例都可能会出错，因为该发射器不再存在，因此请确保该操作是仅在不打算从该发射器创建任何新粒子时调用（有关粒子发射器的更多信息，请参见创建粒子发射器）。

注意：此操作只会清除房间中可见的粒子，但不会清除粒子属性，也不会从内存中删除它们。为此，您应该使用动作Destroy Particle Type 。
