设置发射器区域

此动作用于设置粒子系统内发射器的区域和形状。您提供的粒子系统ID，该发射器属于（由动作返回的创建粒子系统，然后将实际发射器ID（作为返回由动作）创建粒子发射器），然后给一形状用于发射器采取内给定边界框值。可能的发射器形状为：

要定义形状，您需要为将要放置给定形状的“边界框”提供坐标（对于矩形，椭圆形和菱形），或者为起点和终点分别指定第一个坐标和第二个坐标（在的情况下）。所有从发射器破裂或射出的粒子都将在此边界框内的形状范围内创建。您还可以使用以下选项设置粒子的分布：

  • 线性：将在给定形状内的任意位置随机创建粒子。
  • 高斯（Gaussian） ：粒子将主要在形状的中间生成，沿边缘的数量较少。
  • 逆高斯：将主要沿形状的边缘创建粒子，而在中间较少。
