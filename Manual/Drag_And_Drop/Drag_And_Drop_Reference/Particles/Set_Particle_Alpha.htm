设置粒子Alpha

此操作将设置创建粒子时的Alpha值以及整个生命周期中的Alpha值。首先，指定粒子类型ID值（该值应与使用“创建粒子类型”操作创建粒子时返回的值相同），然后可以设置要使用的alpha值。

您可以选择三个Alpha来使用，粒子将在粒子的整个生命周期内在它们之间混合。例如，如果某个粒子的alpha值具有0.5、1和0，并且生命周期为60个游戏帧，则在创建它（帧0）时，它将为0.5 alpha，然后它将朝1的完整alpha渐变（第30帧），然后逐渐消失，使其以0的alpha值不可见（第60帧）。
