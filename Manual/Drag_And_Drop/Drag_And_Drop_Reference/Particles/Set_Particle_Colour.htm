设置粒子颜色

此操作将设置创建精灵时及其整个生命周期内要使用的一种或多种颜色。首先指定粒子类型ID值（该值应与使用“创建粒子类型”操作创建粒子时返回的值相同），然后可以设置要使用的颜色。

您可以选择三种颜色来使用，并且精灵会在粒子的整个生命周期内在它们之间混合。例如，如果某个粒子具有红色，绿色和黄色，并且生命周期为60个游戏框架，则在创建它（第0帧）时，它将着色为红色，然后在其寿命的一半时向绿色褪色（第30帧），然后在其寿命结束时逐渐淡黄色（第60帧）。如果使用自定义精灵而不是粒子形状，则白色以外的任何颜色都会与所使用的精灵混合。

注意：不使用颜色的alpha分量，仅使用颜色选择器中的RGB分量。要设置Alpha，请使用“设置粒子Alpha”操作。
