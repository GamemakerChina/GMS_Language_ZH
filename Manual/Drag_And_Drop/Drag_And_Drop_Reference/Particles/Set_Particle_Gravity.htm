设置粒子重力

此动作将模拟重力对给定类型的所有粒子的影响。首先指定粒子类型ID值（该值应与使用“创建粒子类型”操作创建粒子时使用的值相同），然后可以设置重力和方向。力是一个累积值，将在每个游戏帧上添加到速度矢量上，因此重力1将在每个游戏帧上添加一个速度值，而方向值只是重力应“拉动”方向的方向粒子。请注意，对于力和方向，您都可以使用负值，并且在GameMaker Studio 2该方向的计算方法为：右侧0°，顶部90°，左侧180°，底部270°。
