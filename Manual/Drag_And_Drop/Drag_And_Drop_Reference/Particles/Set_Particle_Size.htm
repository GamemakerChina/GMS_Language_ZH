设定粒径

此操作用于设置粒径，该粒径可以是静态的，也可以设置为随时间增加/减少。您为该类型提供唯一的ID值以设置大小（通过创建粒子类型操作创建类型时返回），然后提供一系列初始起始大小，然后是增加量，最后是“摆动”。创建粒子的大小为标量值，其中1与基本粒子图像的大小相同，0.5将为基本图像的大小的一半，值为2则为两倍。如果最小和最大大小不相同，则创建的任何粒子的初始大小将是它们之间的随机值。

增加粒子大小的选项使您可以输入一个值，该值将添加到粒子寿命的每个游戏帧的基本标量值上。因此，将此值设置为例如0.1将在每个游戏帧的比例值上添加0.1。请注意，您也可以在此处使用负值，以使粒子图像也按比例缩小。请注意，接受的最小比例尺减量为-0.01，最小增量为0.01。

最后，您可以设置“摆动”选项。这是一个将在0和您给定的值之间随机生成的值，然后在粒子寿命的每一步中将其加或减到大小上，从而导致绘制的图像大小发生波动。显然，较大的值比较小的值更明显，并且该值甚至可以是负数，最大范围在-20到20之间。
