设定粒子速度

此操作用于设置粒子速度，该速度可以是静态的，也可以设置为随时间增加/减少。您可以为类型提供唯一的ID值，以设置大小（通过创建粒子类型操作创建类型时返回），然后提供一系列初始起始速度，然后是增加量，最后是“摆动”。粒子创建的速度以每个游戏帧的像素数计算，因此最小设置2和最大设置5将发出粒子移动每个游戏帧2到5像素之间的任何位置。

增加粒子速度的选项使您能够输入一个值，该值将被添加到粒子寿命的每个游戏帧的基本速度值上。因此，将此值设置为例如0.1将使每个游戏帧的速度增加0.1。请注意，您也可以在此处使用负值，以使粒子也变慢，尽管速度永远不会低于0。请注意，可接受的最小速度减量为-0.01，最小增量为0.01。

最后，您可以设置“摆动”选项。这是一个将在0和您给定的值之间随机生成的值，然后在粒子寿命的每一步中将其加或减到速度上，从而导致绘制图像的速度波动。显然，较大的值比较小的值更明显，并且该值甚至可以是负数，最大范围在-20到20之间。
