设置粒子精灵

此动作可用于设置绘制时粒子可以使用的精灵。您可以为类型提供唯一的ID值，以设置精灵（使用动作“创建粒子类型”创建类型时返回）和要使用的精灵资源。如果子画面具有子图像，则粒子将以在图像编辑器中为子画面设置的速度通过帧进行动画处理。还要注意，如果您设置粒子的颜色值（使用“设置粒子颜色”操作），则该颜色将与使用的精灵混合，就像在实例上设置图像混合会将颜色与实例精灵混合（使用白色（无融合）。

注：如果您使用此动作，那么你不应该使用的动作设定颗粒形状，你只能有一个精灵或形状，但不能同时使用。
