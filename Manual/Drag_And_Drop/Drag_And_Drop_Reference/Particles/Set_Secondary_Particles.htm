设置二次粒子

此操作允许您将第二种粒子类型与主粒子一起使用。给出主要粒子类型ID值（该值应与使用“创建粒子类型”操作创建粒子时使用的值相同），然后可以选择让它在存在的每个游戏帧中发射辅助粒子类型，并/或在每个主要粒子寿命的尽头。您必须为“中学”指定另一个粒子ID值，然后在“计数”中指定要创建的数字。

如果选择的事件为“每步”，则屏幕上存在主要粒子的每个游戏帧都会发出给定数量的辅助粒子-例如，您的火箭可能带有“火焰”粒子，并且每个火焰粒子都发射一个次要的“烟雾”粒子。如果将事件设置为“死亡”，则仅当主要类型的粒子达到其使用寿命时才发射辅助粒子-例如，您可能有一个“爆炸”粒子发出“尘埃”生命尽头的粒子。

重要的是要注意，您不能使用与主要粒子相同的粒子类型ID，也不能使用与次要粒子相同的粒子类型ID，并且使用此操作时会很快遇到性能问题，因为以这种方式很容易快速创建数千个粒子-因此请注意限制主要粒子和次要粒子类型生成的粒子的寿命和数量。还要注意，您可以将辅助粒子分配给任何粒子类型，因此可以创建“粒子链”，其中一个粒子在每个步骤中创建另一个粒子，然后当该粒子达到其使用寿命尽头时又创建另一个粒子，依此类推。 ，以这种方式链接粒子时应格外小心，以确保不会影响性能。您也可以针对单个粒子类型两次调用此操作，并使其一步步流过并在死亡时爆发。
