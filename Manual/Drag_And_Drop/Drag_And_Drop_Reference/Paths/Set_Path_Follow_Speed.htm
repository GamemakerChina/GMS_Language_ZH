设置路径跟随速度

这个动作将设置一个实例跟随路径的速度，单位是每一游戏帧的像素（注意，你必须调用Start Following Path动作为实例指定一个路径，这样才有 任何效果）。)你可以使用负值来表示实例应该反向跟随路径，值得注意的是，你在这里设置的速度可以被路径编辑器中为每个路径点设置的速度修改器所修改，所以如果你 将路径速度设置为2，而路径中的一个点的速度修改器为200%，则该点的实际像素/游戏帧速度将为4。
