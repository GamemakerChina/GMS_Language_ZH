设定路径跟随速度

此操作将设置实例遵循路径的速度（以每游戏帧像素为单位）（请注意，您必须调用“开始跟随路径”操作才能为该实例分配路径，以使其生效）。您可以使用负值来表示实例应沿相反的方向走，并且值得注意的是，您在此处设置的速度可以通过在路径编辑器中为每个路径点设置的速度修改器进行修改，因此，如果您设置了路径速度到2，并且路径中的一个点具有200％的速度修改器，该点处每游戏帧速度的实际像素为4。
