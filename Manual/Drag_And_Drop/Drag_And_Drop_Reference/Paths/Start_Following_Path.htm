开始遵循路径

这个动作将启动一个实例沿着在资产浏览器中创建的路径前进。然后你可以用这个动作来告诉你的实例要遵循哪条路径，遵循路径的速度（以每一游戏帧的像素为单位），当它到达终点时如何 当它到达路径的终点时如何行动，以及是跟随绝对还是相对的路径位置。在设置速度时，你可以使用负值来表示实例应该反向跟随路径，值得注意的是 你在这里设置的速度可以被路径编辑器中为每个路径点设置的速度修改器修改，所以如果你把路径速度设置为2，而路径中的一个点的速度修改器为200%，那么这个点的实际速度将是 每个游戏帧4个像素。

一旦你设置了初始路径速度，你就可以从以下选项中设置实例到达路径终点时的行为方式（注意，当实例到达路径终点时，这也会触发一个其他--路径结束事件）。

  • 停止：结束路径，在最后一个路径点上停止实例。
  • 重新开始。从起点继续前进，如果在路径编辑器中没有标记为关闭的路径，则再次跳到起点位置。
  • 循环。从当前位置继续，再次运行该路径。
  • 反转。沿着路径再次倒退（通过逆转路径跟随速度实现），当它到达路径的起点时，它将再次逆转，等等。

最后，你可以将路径标记为绝对值 或不标记。默认情况下，实例将从其当前位置（相对位置）开始跟随路径，但是如果你标记了绝对复选框，那么它将开始跟随 确切地按照你在路径编辑器中设计和放置的路径（绝对位置）。下面的图片说明了这一点。
