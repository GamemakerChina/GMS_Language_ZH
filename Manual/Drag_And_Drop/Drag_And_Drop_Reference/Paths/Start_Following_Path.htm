开始跟随路径

此操作将沿着在资源浏览器中创建的路径启动实例。然后，您将使用此操作告诉实例要遵循的路径，遵循的速度（以每个游戏帧的像素为单位），到达路径末端时的行为方式以及遵循绝对路径还是相对路径路径位置。设置速度时，您可以使用负值来表示实例应反向执行路径，并且值得注意的是，您在此处设置的速度可以通过在路径编辑器中为每个路径点设置的速度修改器进行修改，因此如果将路径速度设置为2，并且路径中的一个点的速度修改器为200％，则该点的实际速度为每游戏帧4个像素。

设置初始路径速度后，您可以通过以下选项设置实例到达路径末尾时的行为方式（请注意，当实例到达路径末尾时，也会触发其他-路径末尾事件）：

  • 停止：结束路径，在最后一个路径点停止实例。
  • 重新启动：从起点继续路径，如果在路径编辑器中未将路径标记为已关闭，则再次跳至起点。
  • 循环：从当前位置继续以再次运行路径。
  • 反向：再次沿路径向后移动（通过使路径跟随速度反向来实现），当到达路径的起点时，它将再次反向，依此类推。

最后，您可以将路径标记为绝对路径。默认情况下，实例将从其当前位置（相对位置）开始沿路径移动，但是如果标记“绝对”复选框，则实例将完全按照您设计并放置在路径编辑器中的路径（绝对位置）开始沿路径移动。下图说明了这一点：
