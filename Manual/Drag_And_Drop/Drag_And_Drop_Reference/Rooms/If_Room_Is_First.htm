如果房间是第一

此操作将检查当前房间是否是“房间管理器”中列出的第一个房间。动作将返回 true如果当前房间是房间管理器中的第一个房间，并且 false如果不是这样。您可以标记“否”复选框，以将其变成“如果房间不是第一位的”。
