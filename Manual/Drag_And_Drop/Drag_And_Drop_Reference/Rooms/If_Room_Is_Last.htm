如果房间是最后一个

这个动作将检查当前房间是否是房间管理器中列出的最后一个房间。如果当前房间是房间管理器中的最后一个，该动作将返回true，如果不是，则返回false。你可以标记不是复选框，把它变成 "如果房间不是最后一个"。

请注意，要把动作添加到 "如果 "块中，应该把它们放到动作的一侧，如下图所示。

如果 "if "评估为 "true"，这些动作现在将被运行，而在其他地方放弃的任何动作 将在 "if "块之后执行。
