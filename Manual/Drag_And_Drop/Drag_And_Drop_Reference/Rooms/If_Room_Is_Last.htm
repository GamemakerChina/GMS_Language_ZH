如果房间是最后一个

此操作将检查当前房间是否是“房间管理器”中列出的最后一个房间。动作将返回 true如果当前房间是房间管理器中的最后一个房间，并且 false如果不是这样。您可以标记“否”复选框，以将其变成“如果房间不是最后一个”。
