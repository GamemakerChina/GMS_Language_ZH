创建序列元素

通过此操作，您可以从资产浏览器创建任何序列的序列元素。您提供了序列的名称以及要在其中创建序列的房间内的x / y位置，以及要在其上创建的层（该层必须是资产层）。您可以勾选“ relative”标志，以相对于运行动作代码块的实例定位序列。每当您以这种方式在游戏中创建序列时，都会为其赋予唯一的元素ID值，然后您可以使用该ID值在后续操作中标识该序列，因此，该操作允许您定义一个“目标”变量来保存此ID值（如下例所示）。请注意，可以将目标标记为临时（本地）变量，该变量将为该操作创建，并且可以在该操作所在的事件的所有后续操作中使用。
