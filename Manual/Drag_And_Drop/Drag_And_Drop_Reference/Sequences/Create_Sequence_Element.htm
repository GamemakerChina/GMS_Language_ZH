创建序列元素

通过这个动作，你可以在资产浏览器中创建任何序列的序列元素。你给出序列的名称和在房间内要创建的X/Y位置，以及要创建的层（这个层必须是资产层 是一个资产层）。)你可以勾选 "相对 "标志来定位序列与运行动作代码块的实例的相对位置。每当你以这种方式在游戏中创建一个序列，它就会被赋予一个独特的元素ID值 你可以用它在进一步的动作中识别这个序列，因此这个动作允许你定义一个 "目标 "变量来保持这个ID值（如下图所示）。请注意，目标变量可以被标记为一个临时（本地）变量。 变量，它将为该动作创建，并可用于该动作所在事件的所有后续动作中。
