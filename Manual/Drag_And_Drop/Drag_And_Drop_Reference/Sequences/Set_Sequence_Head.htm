设置序列头

通过这个动作，你可以把房间里给定的序列元素的播放头的位置设置为指定的帧。你提供序列元素的ID（由动作Create Sequence返回）和 以帧为单位的新播放头位置。关于播放头的信息，请参见《使用Dope Sheet》。

注意，位置是在 帧 ，如果你设置的值大于总帧数（或小于0），那么实际的最终播放头的位置将取决于已经选择的序列播放的类型。 遵循这些规则。

  • 没有循环。播放头将夹在循环的末端（如果给定的值是负的，则夹在开头）。
  • 循环播放。播放头将回到序列的开头，再加上额外的帧，例如：如果你在一个30帧的序列上将播放头设置为第45帧的位置，实际播放头的位置最终将是15。如果你给 负值，那么序列的播放头将以相反的方向绕行。
  • 乒乓式循环。播放头将前进到最后，然后返回超过总帧长的额外帧数，反转播放方向，例如。如果序列是30帧，你把播放头设置为 如果序列是30帧，你把播放头设置为40，最后的播放头位置将是20，方向将是相反的（往下到0）。注意，你应该 ，千万不要 ，为序列的乒乓播放提供一个负值，因为它可能产生未定义的 结果。

以这种方式设置磁头位置不会停止播放，除非暂停，否则序列将从新的播放磁头位置继续。
