设置序列头

通过此操作，您可以将房间中给定序列元素的播放头位置设置为指定的帧。您提供序列元素ID（由动作Create Sequence返回）和新的播放头位置（以帧为单位）。有关播放头的信息，请参见“使用Dope Sheet 。

请注意，该位置在 镜框 并且如果您设置的值大于总帧数（或小于0），则实际的最终播放头位置将取决于以下选定的序列播放类型：

  • 无循环：播放头将固定在循环的末尾（如果给定的值为负数，则开始循环）
  • 循环播放：播放头将返回到序列的开头加上额外的帧，例如：如果将30帧序列上的播放头设置为第45帧的位置，则实际的播放头位置最终将为15。负值，则序列播放头将反方向环绕。
  • 乒乓循环：播放头将前进到最后，然后返回超出整个帧长度的多余帧数，从而反转播放方向，例如：如果序列为30帧，并且将播放头设置为40 ，最终的播放头位置将为20，并且方向将反转（下降至0）。请注意，您应该 绝不 为序列的乒乓播放提供负值，因为它可能会产生不确定的结果。

以这种方式设置磁头位置不会停止播放，除非暂停，否则序列将从新的播放头位置继续进行。
