设置序列位置

通过这个动作，你可以为房间里给定的序列元素设置位置。你提供序列元素的ID（由动作Create Sequence返回），然后是新的X和Y位置。如果你 勾选 "相对 "复选框，那么给定的X和Y的值将被添加到当前的序列位置，否则它们将是一个绝对位置。
