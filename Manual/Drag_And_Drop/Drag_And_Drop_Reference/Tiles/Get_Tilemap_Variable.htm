获取Tilemap变量

通过此操作，您可以检索图层上图块地图元素的给定数量的变量中的任何一个。在房间编辑器中创建图块层时，该层将包含一个图块地图元素，然后使用图块集中的图块填充该图块元素。此图块地图元素具有与之关联的某些值，例如偏移位置，正在使用的图块集，宽度，高度等。您可以获得的值的完整列表为：

  • X坐标：沿着x轴在已放置切片地图元素的图层上的位置。
  • Y坐标：放置了瓦片地图元素的图层上沿y轴的位置。
  • 列：图块地图元素的宽度（定义为图块单元的列数）。
  • 行：图块地图元素的高度（定义为图块单元的行数）。
  • 平铺宽度：单个平铺单元的宽度。
  • 平铺高度：单个平铺单元的高度。
  • Tile Set Resource ：已分配给图块地图元素进行绘制的图块集资源。
  • 当前帧：正在为动画图块绘制的当前帧。
  • 遮罩：图块贴图元素的遮罩数据。
  • 全局蒙版：所有图块贴图元素的全局蒙版数据。

使用此操作时，您将提供图层名称（在“房间编辑器”中定义的字符串）来获取切片地图元素数据，然后提供您要检索的数据类型（如上面的列表所示）。然后，返回的值将存储在目标变量中，该目标变量可以是以前创建的，也可以是新的临时变量（如果选中“临时”（Temp）复选框）。请注意，您可以通过单击加号图标来检索其他值。在操作旁边，选择另一个变量并提供另一个变量来存储返回的值。
