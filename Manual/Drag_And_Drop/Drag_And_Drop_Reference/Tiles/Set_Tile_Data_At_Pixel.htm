将图块数据设置为像素

通过此操作，您可以为在图块地图图层上特定位置找到的图块单元设置图块数据。切片数据只是一个反映切片索引以及翻转，镜像和旋转值以及您使用过的任何自定义切片蒙版的值。通过此操作，您首先使用房间编辑器中定义的图层名称（字符串）提供要定位的图层，然后使用房间中的x和y轴为其设置图块数据。注意，将为给定位置所在的图块地图“单元”设置图块数据。例如，如果您的图块设置图块是16x16，则图块地图图层将图块放置在16x16像素的单元格中，因此，如果您使用此操作定位房间位置（8，8），则将在瓦片地图的像元（0，0）。

注意：必须在房间编辑器中将所选图层定义为“平铺地图图层”，否则可能会出错。

动作的最后一个参数是瓦片数据本身。通常，您可以使用“获取像素的平铺数据”操作来检索平铺地图单元的平铺数据，然后使用“设置平铺数据变换”操作对其进行操作，然后再使用此操作对其进行设置。有关图块数据的更多信息，请参见图块贴图函数GML部分。
