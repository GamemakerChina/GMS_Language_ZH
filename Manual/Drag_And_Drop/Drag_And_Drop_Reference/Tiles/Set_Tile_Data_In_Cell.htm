在单元格中设置图块数据

通过此操作，您可以在图块地图图层上设置图块单元的图块数据。切片数据只是一个反映切片索引以及翻转，镜像和旋转值以及您使用过的任何自定义切片蒙版的值。通过此操作，您首先使用房间编辑器中定义的图层名称（字符串）提供要定位的图层，然后使用图块地图图层上的列和行为其设置图块数据。 “单元格”是图块地图网格上容纳要为其设置数据的图块的区域，因此，例如，如果图块是16x16，而房间是1024x768，则图块地图将具有64列和48行。

注意：必须在房间编辑器中将所选图层定义为“平铺地图图层”，否则可能会出错。

动作的最后一个参数是瓦片数据本身。通常，您将使用“在单元格中获取切片数据”操作检索切片地图单元的切片数据，然后使用“设置切片数据变换”操作对其进行操作，然后再通过此操作对其进行设置。有关图块数据的更多信息，请参见图块贴图函数GML部分。
