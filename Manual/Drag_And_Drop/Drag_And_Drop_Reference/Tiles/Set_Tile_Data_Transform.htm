设置瓦片数据转换

通过这个动作，你可以改变瓦片数据中的三个瓦片变换之一。你提供保存瓦片数据的变量，然后选择要改变的变换。点击 "应用 "复选框将应用该变换，取消复选框将删除该变换。 取消勾选则会将其删除。瓦片数据是一个简单的值，它反映了瓦片的索引以及所应用的变换和你所使用的任何自定义瓦片掩码，这个动作允许你改变其中的三个值。 翻转、镜像和旋转。请注意，你通常要先用"单元格内获取磁贴数据"或 "像素处获取磁贴数据"来获取磁贴数据，然后应用必要的 变换，然后使用Set Tile Data In Cell或Set Tile Data At Pixel再次在瓷砖贴图中设置瓷砖，因为这个操作只是设置一个值，但并不应用于它所取的瓷砖。 关于瓦片数据的更多信息，请参见GML中的瓦片地图功能部分。
