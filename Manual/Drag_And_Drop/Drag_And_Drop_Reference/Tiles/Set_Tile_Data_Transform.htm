设置图块数据转换

通过此操作，您可以更改图块数据中的三个图块变换之一。您提供用于保存图块的图块数据的变量，然后选择要更改的变换。单击“应用”复选框将应用该转换，取消选中该转换将其删除。切片数据只是一个反映切片索引，应用的变换以及已使用的任何自定义切片蒙版的值，该操作允许您更改其中三个值：翻转，镜像和旋转。请注意，通常您通常要先使用“获取单元格中的切片数据”或“获取像素处的切片数据”来获取切片数据，然后应用所需的变换，然后使用“设置单元格中的切片数据”再次在切片图中设置切片或将图块数据设置为像素，因为此操作仅会设置一个值，而不会将其应用于从中获取的图块。有关图块数据的更多信息，请参见图块贴图函数GML部分。
