在像素处设置瓦片索引

通过这个动作，你可以将一个给定图层上的瓷砖设置为一个新的索引。图层是由给定的图层名称（一个字符串，在房间编辑器中用来命名图层）指定的，然后你给出一个x和y的位置来获得要设置的瓷砖。x和 y位置可以被标记为相对于调用实例的位置，被设置的瓦片将是给定位置在房间内的瓦片，使用的瓦片集将是在房间编辑器中分配给该层的。 编辑。最后，你要提供瓷砖的索引来设置瓷砖（瓷砖的索引从左到右，从上到下，0在左上角）。注意你可以随时将瓷砖索引设置为0来清除瓷砖。
