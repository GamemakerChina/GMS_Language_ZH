将图块索引设置为像素

通过此操作，您可以将给定图层上的图块设置为新索引。从给定的图层名称（字符串，用于在“房间编辑器”中命名该图层）指定图层，然后指定x和y位置以设置图块。可以将x和y位置标记为相对于调用实例的位置，并且正在设置的图块将是给定位置在房间内的那个，而所使用的图块集合将是分配给该层的那个在房间编辑器中。最后，您可以提供图块索引以将图块设置为（图块从左到右从上到下索引，左上角为0）。请注意，您始终可以将图块索引设置为0来清除图块。
