在单元格中设置图块索引

通过此操作，您可以将给定图层上的图块设置为新索引。从给定的层名称（字符串，用于在“房间编辑器”中命名该层）指定该层，然后在该层中指定单元格的行和列以获取要设置的图块。最后，您可以提供图块索引以将图块设置为（图块从左到右从上到下索引，左上角为0）。请注意，您始终可以将图块索引设置为0来清除图块。
