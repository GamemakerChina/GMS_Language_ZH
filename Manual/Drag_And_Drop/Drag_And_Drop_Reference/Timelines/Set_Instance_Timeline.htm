设置实例时间轴

该操作将资产浏览器中的时间轴分配给实例。您给时间轴分配时间，然后可以选择在到达最后一刻时使时间轴循环（运行最后一刻之后的下一帧将再次循环回到时间轴的第一刻），以及是否是否立即启动时间轴。如果您不立即启动时间轴，则可以使用操作“设置时间轴状态”在以后在另一个事件或脚本中启动它。
