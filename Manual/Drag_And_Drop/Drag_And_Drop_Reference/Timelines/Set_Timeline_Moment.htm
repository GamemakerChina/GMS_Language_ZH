设置时间轴时刻

此操作可用于将时间轴从当前时刻跳过到新时刻。时间线通过不同的“时刻”前进，每个时刻通常占用一个游戏框架（例如对象的“阶梯事件”）。时间轴一旦开始，它会一直持续到到达终点为止，直到结束时，它会循环回到时间轴的第一时刻或完全停止。但是，可以使用此操作修改此线性时间线方法，以随时设置不同的时刻，允许您创建自己的时间线循环，或在某些情况下跳过时间线的各个部分，等等。只需将时刻设置为跳到时间轴内，下一个游戏帧就会运行该时刻（请注意，您需要使用“设置实例时间轴”动作为实例分配时间轴）。
