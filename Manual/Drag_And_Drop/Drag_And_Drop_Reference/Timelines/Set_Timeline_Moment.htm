设置时间轴时刻

这个动作被用来将时间线从当前时刻跳到一个新的时刻。时间线通过不同的 "时刻 "向前移动，每个时刻通常占用一个游戏帧（就像一个物体的步骤事件）。一旦一条时间线开始了 它将通过各个时刻前进，直到到达终点，这时它将循环回到时间线的第一个时刻或完全停止。然而，时间线的这种线性方法可以通过这个动作进行修改，以便在任何时候设置一个不同的 瞬间，允许你创建你自己的时间线循环，或在特定情况下跳过时间线的部分，等等...。有了这个动作，你只需在时间线中设置要跳转的时刻，下一个游戏帧就会运行 该时刻（注意，你需要使用动作Set Instance Timeline为实例分配一个时间线）。
