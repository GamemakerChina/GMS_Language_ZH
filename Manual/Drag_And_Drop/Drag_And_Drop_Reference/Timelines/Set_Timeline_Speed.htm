设定时间轴速度

此操作可用于设置时间线在其包含的瞬间移动的速度。通常，在每个游戏帧中，您都希望时间轴上的位置增加1，但是您可以通过将此操作设置为其他值来更改此数量。您选择的值将被视为乘数，因此，例如，值2将使时间轴以两倍的速度运行，而值0.5将使其以一半速度运行。此刻的意思是，如果该值大于1，则在同一游戏框架内可能会发生数个瞬间（不会跳过任何瞬间），并且如果速度小于1，则在经过一瞬间后，直到下一分钟运行之前，都不会发生任何帧-例如，如果将时间轴速度设置为0.1，则您将在一个游戏帧上获得一个时刻，然后在没有任何时刻的情况下获得9个游戏帧，在第十个游戏帧上获得下一刻将运行。

如果您设置时间轴以开始使用“设置实例时间轴”或“设置时间轴状态”操作，但未设置速度，则该时间轴将默认为每个游戏帧1个瞬间。
