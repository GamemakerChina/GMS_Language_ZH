设置时间表状态

此操作可用于启动，停止，暂停或恢复时间轴。必须首先使用“设置实例时间线”操作为实例设置了时间线，然后您可以随时使用此操作从以下选项中更改其运行状态

选项描述
开始启动分配给实例的时间轴（如果时间轴已暂停，则会将开始位置再次设置为时刻0）。
恢复恢复已暂停的时间轴（如果该时间轴已停止但未暂停，则将不会执行任何操作）。
已暂停暂停分配给实例的时间轴（在时间轴中暂停的时刻不会随着游戏帧而前进）。
停止停止分配给实例的时间轴。
