设置时间线状态

这个动作可以用来启动、停止、暂停或恢复一条时间线。时间线必须首先使用动作Set Instance Timeline为实例设置，然后你可以在任何时候使用这个动作来 从以下选项中改变其运行状态

选项 描述
开始 启动分配给实例的时间线（如果时间线被暂停，这将再次将起始位置设置为时刻0）。
简历 恢复一个暂停的时间线（如果时间线已经停止，没有暂停，这将不会做任何事情）。
暂停使用 暂停分配给实例的时间线（当暂停时，时间线中的时刻将不会随着游戏帧的推进而推进）。
停止 停止分配给该实例的时间线。
