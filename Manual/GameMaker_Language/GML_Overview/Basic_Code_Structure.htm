基本代码结构

代码以块形式编写，典型的代码块由一组称为语句s的指令组成，然后由GameMaker Studio 2解释并用于使游戏中发生某些事情。这种“事情”可以简单地将2与2相加得到4，也可以复杂到当敌人的生命值低于一定值时逃走。程序的实际结构可能会有所不同，具体取决于所使用的功能，但分解为基本内容后，它看起来像这样：

<statement>;
<statement>;
...

语句应以';'分隔符号，以防止变量声明出错并保持代码整洁，它们可以由变量声明，表达式s和对特定函数的调用组成。您也可以使用大括号将语句“分组”为一个块 {}这样它们可以一起运行，如以下条件示例所示：

if (<conditional>)
{
<statement>;
<statement>;
...
}

注意： GameMaker Language也将接受 begin  和 end  而不是大括号 {}，尽管通常这不是最常用的方法：

if (<conditional>)
begin
<statement>;
<statement>;
...
end
