基本代码结构

代码是以块为单位编写的，一个典型的代码块由一组指令组成，称为语句，然后由GameMaker Studio解释 2解释，并用于在你的游戏中发生一些事情。这个 "东西 "可以是简单的2和2相加得到4，也可以是复杂的当敌人的健康状况低于某个值时使其逃跑。程序的实际结构 的实际结构可以有很大的不同，这取决于它所使用的功能，但从基本的角度来看，它看起来就像这样。

<statement>;
<statement>。
...

语句应该用'；'符号隔开，以防止变量声明的错误，并保持代码的干净整洁，它们可以由变量声明组成。 表达式和对特定函数的调用。你也可以用大括号{}将语句 "分组"，作为一个块，使它们一起运行，比如下面的例子 条件性例子。

如果（<条件>）
{
<statement>。
<statement>;
...
}

注意：GameMaker语言也会接受begin 和end ，而不是大括号{}，尽管这通常不是最常见的方式。
