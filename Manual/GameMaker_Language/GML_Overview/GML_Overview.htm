GameMaker Language概述

本手册的此部分包含了解和使用GameMaker Studio 2脚本语言GML所需的所有信息。这种语言的runtime功能可用于创建游戏，并从“对象编辑器”添加到对象中。通常，当您选择创建新的GameMaker Language Project时，尽管它也可以与DnD™ （ Drag and Drop ） 。在下面，您可以看到在事件中打开代码编辑器的对象的典型图像：

每个事件在编辑器中都有其自己的选项卡，您可以随时在其中添加，编辑或删除代码（有关事件的更多信息，请参见Object Events ）。代码本身必须具有基本结构，并且可以包含资源索引，变量，函数，表达式，关键字等。所有这些都在以下各节中进行说明。 DnD™进行编程或转换的新手，建议您从“基本代码结构”上的页面开始，然后通读本节中的所有其他页面，并在GameMaker Studio 2本身中测试每个代码。
