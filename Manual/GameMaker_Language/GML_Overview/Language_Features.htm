语言特征

GameMaker Language （ GML ）具有许多功能，这些功能是用该语言编写的所有程序的基本结构的一部分。这些中的每一个都将在您多次编写的任何代码中使用，因此在开始对项目进行编程之前，您应该了解它们的含义和工作方式。

  • if / else和条件运算符
  • repeat
  • while
  • do / until
  • for
  • switch
  • break
  • continue
  • exit
  • with
  • return
  • begin / end
  • try / catch / finally
  • throw
  • new
  • delete
