语言特色

GameMaker语言（GML）有许多特性，它们是所有用该语言编写的程序的基本结构的一部分。每个功能都会在你写的任何代码中多次使用，所以你应该知道它们是什么以及它们如何 在开始为一个项目编程之前，你应该知道它们是什么以及如何工作。

  • if / else和条件运算符
  • 重复
  • 而
  • 做/直到
  • 为
  • 开关
  • 突破
  • 继续
  • 退出
  • 与
  • 返回
  • 开始/结束
  • 尝试/捕捉/最后
  • 抛出
  • 新
  • 删除
