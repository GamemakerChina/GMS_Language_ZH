begin / end

GameMaker Language允许您使用关键字 begin和 end而不是更常见的花括号 {}创建代码块时。下面的代码示例显示了它是如何工作的：

if (!visible)
    begin
    exit;
    end
while (place_meeting(x, y))
    begin
    x -= lengthdir_x(1, direction - 180);
    y -= lengthdir_y(1, direction - 180);
    end

请注意，使用这些关键字不是很典型，并且作为语言的一部分提供给遗留支持，而不是其他任何支持，并且将来可能会不推荐使用它们。
