exit

这 exit语句具有以下语法：

exit;

exit只需结束当前脚本功能的执行，method ， 或者event。请注意，根据范围，此处的用法略有不同：

  • 如果您使用 exit在脚本函数或方法中，它将仅退出函数并返回到调用该函数的代码
  • 如果您使用 exit在对象中的代码块中，即使之后有各种单独的代码块，它也会退出整个事件 exit已被调用。

在事件中使用时，exit通常用于避免实例在满足（或不满足）特定条件时运行任何其他代码。下面的代码概述了在冲突事件中如何使用它的示例，尽管它可以在任何事件中使用。

if (!visible)
    {
    exit;
    }
while (place_meeting(x, y))
    {
    x -= lengthdir_x(1, direction - 180);
    y -= lengthdir_y(1, direction - 180);
    }

上面的代码检查一个变量，如果它解析为 true，然后退出代码块，否则继续运行其余代码。

注意：它不会结束游戏的执行。为此，您需要使用功能game_end（） 。
