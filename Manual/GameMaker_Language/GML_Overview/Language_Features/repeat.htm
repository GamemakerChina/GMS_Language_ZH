repeat

GameMaker Language具有多种执行循环的方式，即：使一个或多个语句在其自身上迭代一定次数。其中最简单的是 repeat语句，其形式为：

repeat (<expression>)
    {
    <statement>;
    <statement>;
    ...
    }

和 repeat给定的语句重复表达式的舍入值指示的次数。例如，以下代码在随机位置创建五个球：

repeat (5)
    {
    instance_create_layer(random(400), random(400), "Instances", obj_ball);
    }

这对于避免多次键入相同的代码，使用数组或通过多个操作计数等非常有用。他们放在大括号内 {}。例如：

var _x = 32;
repeat (global.p_lives)
    {
    draw_sprite(spr_heart, 0, _x, 32);
    _x += sprite_get_width(spr_heart);
    }

上面的例子将重复然而，许多在大括号中的语句迭代S中的“生命”全局变量，并且每次迭代绘制心脏精灵在 _x位置，然后根据心脏精灵的宽度沿一点移动该位置。

什么时候应该使用 repeat环形？任何时候您想要在一个或多个语句上重复执行固定的次数，而无需任何特定的次数来维护迭代次数。

值得注意的是，您可以使用特殊 break和 continue重复循环中的语句。使用 break将立即退出循环并在循环应完成后继续执行事件或函数中的任何代码，例如：

var i = 0;
var temp = 0;
repeat (10)
    {
    temp += array[i];
    if (temp > max_total)
        {
        break;
        }
    else
        {
        i += 1;
        }
    }

上面的代码循环遍历10个数组值，并将它们添加到局部变量中。如果局部变量的总和大于给定值 max_total，然后使用break终止循环，否则循环将继续。

使用示例 continue在一个 repeat循环将是：

repeat(10)
    {   
    var _x = random(room_width);
    var _y = random(room_height);
    if (instance_position(_x, y, obj_Enemy)
        {
        continue;
        }
    instance_create_layer(_x, _y, "Instances", obj_Enemy);
    }

此代码将重复10次，生成一个随机的房间位置，然后检查对象的实例是否 obj_Enemy存在于那个位置。如果是这样，则使用终止当前循环迭代 continue然后开始新的迭代，如果没有，则该对象的实例 obj_Enemy在随机位置创建。
