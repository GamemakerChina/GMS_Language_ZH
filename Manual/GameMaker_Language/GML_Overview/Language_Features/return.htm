return

这 return语句具有以下语法：

return (<expression>)

您只能使用 return脚本函数和方法中的语句，它用于从函数中返回一个值，以在进一步的代码或函数调用中使用。应该注意的是，函数的执行在 return语句，表示之后的任何代码 return已被调用将无法运行。这是一个简短的示例脚本，称为 sqr_calc它计算传递给它的任何值的平方，如果传递的参数不是实数，它会包含错误捕获：

/// @function            sqr_calc(val);
/// @param {real}  val   The value to calculate the square of
/// @description         Calculate the square of the given value

function sqr_calc(value)
    {
    if !is_real(value)
        {
        return 0;
        }
    else
        {
        return (value * value);
        }
    }

要从一段代码中调用函数，只需使用与调用runtime函数时相同的方式即可使用-即，在函数名中用括号括起参数值。因此，上面的函数将这样调用：

if keyboard_check_pressed(vk_enter)
    {
    val = scr_sqr(amount);
    }

这里的变量 val将为0（因为变量 amount不是实数）或 value * value。
