抛出

throw 运行时语句允许你生成自己的运行时异常，结束游戏并显示错误信息，使用以下语法。

抛出（<表达式>）。

使用的表达式可以是一个值或一个字符串或任何其他数据类型，然后这将产生一个异常错误--默认情况下--显示在屏幕上，关闭错误信息后游戏将结束。比如说调用这个。

抛出（"Hello World！"）。

将导致显示以下未处理的异常错误。

然而，你可以接管这个错误信息，并使用你自己的处理程序代码，通过调用 函数 exception_unhandled_handler().这个运行时函数允许你提供一个自定义的方法，当你的游戏中出现任何未处理的异常时，这个方法将被调用。
