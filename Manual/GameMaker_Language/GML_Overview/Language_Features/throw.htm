throw

这 throwruntime语句允许您runtime异常，结束游戏并显示错误消息：

throw (<expression>);

使用的表达式可以是值，字符串或任何其他数据类型，这将生成异常错误，默认情况下，该错误显示在屏幕上，并且在关闭错误消息时，游戏将结束。例如调用此：

throw ("Hello World!");
