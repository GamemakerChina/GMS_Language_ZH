结构禁止变量

创建结构时，可以使用某些内置变量作为成员变量名称，例如：

mystruct = {
object_index : obj_Player,
speed : 3,
image_blend : c_red
}

但是，只能以这种方式使用本地和实例范围的内置变量，并且将值分配为常规结构成员变量。禁止使用全局作用域内置变量，这会导致游戏问题。在下面，您可以找到这些变量的完整列表，以便您知道要避免的变量。
