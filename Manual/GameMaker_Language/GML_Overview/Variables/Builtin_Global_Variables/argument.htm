argument

该变量保存一个数组，该数组与只读变量一起使用argument_count在脚本函数或方法中。每个数组位置都保存有该函数的输入值，并且专门与可变参数函数一起使用。

请注意，还有一系列独立的全局范围变量，这些变量也可以在用户定义的函数中用于引用不同的输入参数： argument0 ， argument1 ， argument2等...直至argument15。

注意：不再严格要求此功能，并且为遗留支持提供的其他功能最多。所有用户定义的函数都将接受变量参数，并且默认情况下，传递给不属于命名输入的函数的任何参数都将初始化为undefined ，可以使用is_undefined（）函数进行检查。

句法：

argument[n]
argument1
argument2
...
argument15

返回值：

Value (can be of any data type supplied to the function)

例子：

function my_array()
{
var i, arg;
for (i = 0; i < 5; i += 1;)
   {
   if argument_count > i
      {
      arg[i] = argument[i]
      }
   else
      {
      arg[i] = -1;
      }
   }
}

上面的代码使用 argument_count变量以及 argument数组以初始化变量，并且是如何允许用户定义的函数接受可变数量的参数的示例。

返回：全局变量
