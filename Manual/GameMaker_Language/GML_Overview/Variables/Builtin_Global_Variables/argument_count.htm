argument_count

此只读变量返回传递给脚本函数或脚本的“参数”的数量。method 。通常与参数数组（argument[0...15] ），以允许给定函数使用各种输入参数。

注意：不再严格要求此功能，并且为遗留支持提供的其他功能最多。所有用户定义的函数都将接受变量参数，并且默认情况下，传递给不属于命名输入的函数的任何参数都将初始化为undefined ，可以使用is_undefined（）函数进行检查。

句法：

argument_count;

返回值：

Integer

例子：

function my_array()
{
var i, arg;
for (i = 0; i < 5; i += 1;)
   {
   if argument_count > i
      {
      arg[i] = argument[i]
      }
   else
      {
      arg[i] = -1;
      }
   }
}

上面的代码使用 argument_count变量以及 argument数组以初始化变量，并且是如何允许用户定义的函数接受可变数量的参数的示例。

返回：全局变量
