async_load

此变量的作用域是全局变量，在异步事件中使用时用于保留DS映射，在其他所有时间都使用-1。 DS映射的实际内容将取决于异步事件回调的类型以及用于触发事件的函数，因此请参阅这些事件的各个页面，以获取所有可能的DS映射内容的完整详细信息。

句法：

async_load;

返回值：

Real (integer value)

扩展示例：

sprite =  sprite_add("player_5.png", 16, true, true, 0, 0);

在游戏中的事件中将调用上述代码，以从包含的文件中添加一个精灵。然后，这将触发图像加载异步事件，在该事件中解析 async_load地图：

if ds_map_find_value(async_load, "id") == sprite
    {
    if (ds_map_find_value(async_load, "status") >= 0)
        {
        sprite_index = sprite;
        }
    }

上面的代码将首先检查 async_load映射，然后检查回调的状态。如果该值大于或等于0（表示成功发送信号），则来自回调的结果将用于将实例的精灵索引设置为新加载的图像。
