health

此变量是全局变量，用于保存通常用于播放器运行状况的数值。此变量仅设计为支持GameMaker版本中的旧项目，不应在新项目中使用，因为将来可能不推荐使用。

句法：

health;

返回值：

Real (single precision floating point value)

例子

if (health <= 0)
    {
    global.state = "Game Over";
    instance_destroy();
    }

上面的代码检查 health变量，如果它小于或等于0，则设置一个全局变量，并销毁实例。

返回：全局变量
