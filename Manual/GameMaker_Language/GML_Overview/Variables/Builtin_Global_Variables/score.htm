score

此变量是全局变量，用于保存通常用于玩家得分的数值。此变量仅设计为支持GameMaker版本中的旧项目，不应在新项目中使用，因为将来可能不推荐使用。

句法：

score;

返回值：

Real (single precision floating point value)

例子

switch (object_index)
    {
    case obj_Enemy_Fighter:
        score += 10;
        break;
    case obj_Enemy_Mage:
        score += 25;
        break;
    case obj_Enemy_Boss:
        score += 100;
        break;
    }
instance_destroy();

上面的代码使用以下命令检查运行代码的实例的对象索引switch，然后将不同的金额添加到 score变量取决于它是什么对象。

返回：全局变量
