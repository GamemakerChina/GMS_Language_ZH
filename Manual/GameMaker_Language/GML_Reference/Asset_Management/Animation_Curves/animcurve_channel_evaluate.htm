animcurve_channel_evaluate

这个函数可以用来从一个通道结构中获取特定时间点的值。你提供通道的结构指针（如函数返回的 animcurve_get_channel(), 或从函数中的动画曲线结构中返回的结构指针 animcurve_get())和要评估的曲线通道的 "x"（时间）位置。这个位置应该是在0到 1之间，函数将返回该位置的曲线值，如果提供的通道结构无效，则返回0。

语法。

animcurve_channel_evaluate(channel_struct, posx)。

争论 描述
channel_struct 用于评估通道的结构指针。
姿势 要检查的时间位置（从0到1）。

返回。

真实的

例子。

var _channel = animcurve_get_channel(ac_AlphaCurve, 0);
var _val = animcurve_channel_evaluate(_channel, sin(current_time/1000));
image_alpha = _val;

上面的代码得到了动画曲线资产 "ac_AlphaCurve "的通道结构。然后它使用返回的评估值来设置实例的图像alpha。
