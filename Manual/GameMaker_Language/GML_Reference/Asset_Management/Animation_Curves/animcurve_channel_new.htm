animcurve_channel_new

此功能可用于创建新的动画曲线通道结构。通道结构具有以下变量：

变量名称数据类型描述
姓名细绳这是频道的名称。
类型持续的这将是常量之一 animcurvetype_linear-用于点之间的线性插值-或 animcurvetype_catmullrom-使用catmull-rom插值法在点之间进行“平滑”插值。
迭代整数如果通道使用catmull-rom（“平滑”）插值，则可保存为曲线的每个段生成的点数（请注意，这些额外的点在函数内部，仅用于runtime计算）。如果通道使用线性插值，则该值将仍然存在，但可以忽略，因为它与曲线的插值方式无关。
点数数组指针这是一个数组，其中数组中的每个项目都是一个点结构。


默认情况下，当您创建新的通道结构时，“名称”变量将为空字符串，“类型”将为 animcurvetype_linear并且“迭代次数”将为16。所有这些变量都可以设置为所需的值（请注意，“迭代次数”值对线性曲线类型没有影响）。创建之后，您需要向曲线添加点，这是通过向“点”数组添加不同的点结构来完成的。可以使用函数创建这些点结构 animcurve_point_new()，并保留以下变量：

变量名称数据类型描述
posx真实的点的时间位置（从0到1标准化）。
价值真实的点的值。

句法：

animcurve_channel_new();

返回值：

Struct Reference

例子：

my_curve = animcurve_create();
my_curve.name = "My_Curve";
var _channels = array_create(1);
_channels[0] = animcurve_channel_new();
_channels[0].name = "alpha";
_channels[0].type = animcurvetype_catmullrom;
_channels[0].iterations = 8;
var _points = array_create(3);
_points[0] = animcurve_point_new();
_points[0].posx = 0;
_points[0].value = 0;
_points[1] = animcurve_point_new();
_points[1].posx = 0.5;
_points[1].value = 1;
_points[2] = animcurve_point_new();
_points[2].posx = 1;
_points[2].value = 0;
_channels[0].points = _points;
my_curve.channels = _channels;

上面的代码创建了一个新的动画曲线结构，并将其存储在变量“ my_curve”中。然后用名称和通道数组填充此结构。通道阵列包含一个具有三个点的通道。
