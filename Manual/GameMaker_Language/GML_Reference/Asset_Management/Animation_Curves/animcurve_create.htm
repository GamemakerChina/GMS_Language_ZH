animcurve_create

此函数将创建一个空的动画曲线结构，以供您填充通道数据。该函数将返回一个指向该结构的指针，然后将其用于向动画曲线添加通道和其他数据。您也可以将其提供给以下功能 animcurve_get()以便稍后从曲线中获取数据。

使用此功能时，创建的结构中将没有数据，要使用该结构，您应该添加至少一个通道，并且该通道中应添加点。要添加频道，请参见功能 animcurve_channel_new()，并将点添加到通道，请参见功能 animcurve_point_new()。另外，您可以通过设置“名称”变量来给动画曲线结构命名（如下面的示例代码所示）。添加到新动画曲线的通道应作为一个数组添加，其中数组中的每个项目都引用一个通道结构，通道中的点也应作为一个数组添加，其中每个数组项目都引用一个点结构。

请注意，以这种方式创建的动画曲线在不再需要时必须删除，因为它们会占用内存空间，否则可能导致内存泄漏，减慢速度并最终导致游戏崩溃。您可以在不再需要此功能的情况下使用该功能删除动画曲线 animcurve_destroy()。您不需要清理曲线中的任何通道或点数据，因为当破坏曲线本身时，垃圾收集器会自动完成此操作。

句法：

animcurve_create();

返回值：

Struct Reference

例子：

my_curve = animcurve_create();
my_curve.name = "My_Curve";
var _channels = array_create(1);
_channels[0] = animcurve_channel_new();
_channels[0].name = "alpha";
_channels[0].type = animcurvetype_catmullrom;
_channels[0].iterations = 8;
var _points = array_create(3);
_points[0] = animcurve_point_new();
_points[0].posx = 0;
_points[0].value = 0;
_points[1] = animcurve_point_new();
_points[1].posx = 0.5;
_points[1].value = 1;
_points[2] = animcurve_point_new();
_points[2].posx = 1;
_points[2].value = 0;
_channels[0].points = _points;
my_curve.channels = _channels;

上面的代码创建了一个新的动画曲线结构，并将其存储在变量“ my_curve”中。然后用名称和通道数组填充此结构。通道阵列包含一个具有三个点的通道。
