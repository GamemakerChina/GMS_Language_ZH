animcurve_get

此函数将返回一个结构，其中包含给定动画曲线的所有数据。您提供动画曲线资产ID（在“资源浏览器”中定义），该函数将返回具有以下变量的结构：

变量名称数据类型描述
姓名细绳这是动画曲线的名称。
频道数组指针这是一个数组，其中数组中的每个项目都是一个通道结构。


以这种方式访问动画曲线时，您将获得一个数组，其中每个数组项都是一个结构，其中包含与曲线中的通道有关的数据。通道结构将具有以下变量：

变量名称数据类型描述
姓名细绳这是频道的名称。
类型持续的这将是常量之一 animcurvetype_linear用于点之间的线性插值，或 animcurvetype_catmullrom使用catmull-rom插值法在点之间进行“平滑”插值。
迭代整数如果通道使用catmull-rom（“平滑”）插值，则可保存为曲线的每个段生成的点数（请注意，这些额外的点在函数内部，仅用于runtime计算）。如果通道使用线性插值，则该值将仍然存在，但可以忽略，因为它与曲线的插值方式无关。
点数数组指针这是一个数组，其中数组中的每个项目都是一个点结构。


与通道一样，单个通道上的点也作为结构存储在数组中，其中数组中的每个项目都是单个点结构。点结构具有以下变量：

变量名称数据类型描述
posx真实的点的时间位置（从0到1标准化）。
价值真实的点的值。


注意，如果函数失败（即：给定的动画曲线ID不存在），则函数将返回-1。

句法：

animcurve_get(curve_id);

争论描述
curve_id要获取的动画曲线的资产浏览器ID（索引）。

返回值：

Struct or -1

例子：

var _curve = animcurve_get(ac_ButtonTween);
var _channel = _curve.channels[0];
if _channel.type != animcurvetype_linear
    {
    _channel.type = animcurvetype_linear;
    }

上面的代码检索曲线资产“ ac_ButtonTween”的动画曲线结构，然后检查第一个通道的曲线类型，如果它不是线性的，则将其设置为线性。
