animcurve_get_channel

此函数将返回包含动画曲线资产中指定通道的通道数据的结构。您提供动画曲线ID或结构以及通道名称或索引，该函数将返回具有以下格式的结构：

变量名称数据类型描述
姓名细绳这是频道的名称。
类型持续的这将是常量之一 animcurvetype_linear用于点之间的线性插值，或 animcurvetype_catmullrom使用catmull-rom插值法在点之间进行“平滑”插值。
迭代整数如果通道使用catmull-rom（“平滑”）插值，则可保存为曲线的每个段生成的点数（请注意，这些额外的点在函数内部，仅用于runtime计算）。如果通道使用线性插值，则该值将仍然存在，但可以忽略，因为它与曲线的插值方式无关。
点数数组指针这是一个数组，其中数组中的每个项目都是一个点结构。


动画曲线ID是在资源浏览器中定义的动画曲线的名称，或者是函数返回的结构指针 animcurve_create()。通道名称是一个字符串，它引用在“ Animation Curve”资产中定义的通道，或者您可以提供一个从0到n的索引值，其中n是曲线资产中的最后一个通道（例如：if动画曲线有4个通道，它们的索引从0到3）。请注意，传递索引值比传递通道名称所需的处理更少。如果函数失败（即：不存在具有给定名称或索引的通道），则会出现错误。

单个通道上的点作为结构存储在数组中，其中数组中的每个项目都是单个点结构。点结构具有以下变量：

变量名称数据类型描述
posx真实的点的时间位置（从0到1标准化）。
价值真实的点的值。

句法：

animcurve_get_channel(curve_struct_or_id, channel_name_or_index);

争论描述
curve_struct_or_id动画曲线到目标的ID或结构指针
channel_name_or_index通道名称（字符串）或通道索引（整数）。

返回值：

Struct

例子：

var _channeldata = animcurve_get_channel(ac_ButtonTween, 0);
var _points = _channeldata.points;
for (var i = 0; i < array_length(_points); ++i;)
    {
    _points[i].value += 1;
    }

上面的代码在曲线资产“ ac_ButtonTween”中检索通道0的通道结构，然后循环遍历通道曲线上的点并将其值加一。
