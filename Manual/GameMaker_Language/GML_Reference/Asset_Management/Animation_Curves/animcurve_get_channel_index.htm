animcurve_get_channel_index

这个函数将返回任何给定动画曲线通道的索引值。你可以提供动画曲线ID或结构体，其中曲线ID是在资产浏览器中定义的动画曲线的名称，或结构体指针，如函数返回的那样 的结构指针。 animcurve_create().然后你提供通道的名称，作为一个字符串，该函数将返回与该通道相关的索引值。请注意，如果该曲线或通道不 不存在，那么你会得到一个错误。

语法。

animcurve_get_channel_index(curve_struct_or_id, channel_name)。

争论 描述
曲线_结构_或id 目标动画曲线的ID或结构指针
channel_name 通道名称（一个字符串）。

返回。

结构

例子。

var _channelindex = animcurve_get_channel_index(ac_ButtonTween, "x_pos")
var _channeldata = animcurve_get_channel(ac_ButtonTween, _channelindex);
var _points = _channeldata.point;
for (var i = 0; i < array_length(_points); ++i;)
{
_points[i].value += 1;
}

上面的代码检索了曲线资产 "ac_ButtonTween "中名为 "x_pos "的通道结构，然后循环浏览通道曲线上的点，并为它们的值加一。
