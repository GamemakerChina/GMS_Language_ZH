animcurve_get_channel_index

此函数将返回任何给定动画曲线通道的索引值。您提供了动画曲线ID或结构，其中曲线ID是在资源浏览器中定义的动画曲线的名称，或者是函数返回的结构指针 animcurve_create()。然后，您以字符串形式提供通道的名称，然后该函数将返回与该通道关联的索引值。请注意，如果曲线或通道不存在，则会出现错误。

句法：

animcurve_get_channel_index(curve_struct_or_id, channel_name);

争论描述
curve_struct_or_id动画曲线到目标的ID或结构指针
channel_name频道名称（字符串）。

返回值：

Struct

例子：

var _channelindex = animcurve_get_channel_index(ac_ButtonTween, "x_pos")
var _channeldata = animcurve_get_channel(ac_ButtonTween, _channelindex);
var _points = _channeldata.points;
for (var i = 0; i < array_length(_points); ++i;)
    {
    _points[i].value += 1;
    }

上面的代码检索曲线资产“ ac_ButtonTween”中名为“ x_pos”的通道的通道结构，然后循环遍历通道曲线上的点并将其值加一。
