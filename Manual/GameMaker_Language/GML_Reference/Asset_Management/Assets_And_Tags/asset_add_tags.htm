asset_add_tags

使用此功能，您可以从资产浏览器向任何资产添加一个或多个标签字符串。您可以提供资产名称（作为字符串）或其资产索引，以及单个标签字符串或其中每个项目均为单个标签字符串的数组。如果提供资产索引值，那么您将需要提供可选的资产类型参数（常量），因为不同类型的资产可以具有相同的索引，即使它们不能具有相同的名称。下表列出了可用的资产类型：

持续的描述
asset_object给定名称是指对象。
asset_sprite给定名称是指sprite 。
asset_sound给定的名称是指声音。
asset_room给定名称是指room。
asset_tiles给定名称是指图块集。
asset_path给定名称是指路径。
asset_script给定名称是指script。
asset_font给定的名称是指字体。
asset_timeline给定名称是指时间线。
asset_ shader给定名称是shader 。
asset_animationcurve给定名称是指“动画曲线” 。
asset_sequence给定名称是指Sequence 。

如果函数成功添加标签，它将返回 true否则它将返回 false。

句法：

asset_add_tags(name_or_index, tags, [asset_type]);

争论描述
name_or_index资产名称（字符串）或其索引值（整数）。
tags单个资产标签字符串或具有各种资产标签的数组。
[asset_type]选修的！设置标签的资产类型，仅在为第一个参数提供索引时使用。

返回值：

Boolean

例子：

var _a = array_create(3);
_a[0] = "enemy";
_a[1] = "all_levels";
_a[2] = "boss";
asset_add_tags(obj_Enemy_Boss_Parent, _a, asset_object);

上面的代码将创建一个标签数组，然后将它们添加到给定的对象中。

返回：资产和标签
