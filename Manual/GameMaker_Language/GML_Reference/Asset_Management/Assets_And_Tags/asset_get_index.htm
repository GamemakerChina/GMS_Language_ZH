asset_get_index

您可以使用此功能从游戏资产的名称获取游戏资产的唯一标识索引。如果找不到资产，则该函数将返回值-1，否则它将返回要检查的资产的唯一索引ID。然后，您可以像使用其他任何索引值一样将此ID用于其他函数中，例如 sprite_index或者 path_index， 例如。请注意，尽管此功能可用于从字符串中引用资产（请参见下面的示例），但在使用资产之前，应始终确保该资产存在，否则您可能会遇到使游戏崩溃的错误。

句法：

asset_get_index(name);

争论描述
name要获取（字符串）索引的游戏资产的名称。

返回值：

Asset Index (Integer)

例子：

var obj = asset_get_index("obj_Enemy_" + string(global.Level));
if obj > -1
    {
    instance_create_layer(random(room_width), random(room_height), obj, "Enemy_Layer");
    }

上面的代码将从字符串中获取对象索引，如果该索引存在，则在游戏中创建该对象的实例。

返回：资产和标签
