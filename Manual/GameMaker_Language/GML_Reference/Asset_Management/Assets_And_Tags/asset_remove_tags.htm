资产_删除标签

通过这个函数，你可以从资产浏览器中删除任何资产的一个或多个标签字符串。你可以提供资产名称（字符串）或其资产索引，以及一个单一的标签字符串或一个数组，其中每一项都是一个标签字符串。 如果你提供一个资产索引值，那么你将需要提供可选的资产类型参数（一个常数），因为不同类型的资产可以有相同的索引，尽管它们不能有相同的名称。可用的资产类型列在 在下面的表格中。

恒定 描述
资产_对象 给定的名称指的是一个对象。
资产_石膏 给定的名称指的是一个精灵。
资产_声音 这个名字指的是一种声音。
资产室 这个名字指的是一个房间。
资产 给定的名称指的是一个瓦片组。
资产路径 给定的名称指的是一个路径。
资产脚本 给定的名称指的是一个脚本。
资产字体 该名称指的是一种字体。
资产时间线 这个名字指的是一条时间线。
资产着色器 这个名字指的是一个 着色器.
资产_动画曲线 该名称指的是一个动画曲线。
资产序列 给定的名称指的是一个Sequence。

如果该函数成功地移除标签，它将返回true，否则将返回false。

语法。

asset_remove_tags(name_or_index, tags, [asset_type])。

争论 描述
名称或索引 资产的名称（一个字符串）或其索引值（一个整数）。
标签 一个单一的资产标签字符串或一个包含各种资产标签的数组。
[资产类型] OPTIONAL!要移除标签的资产类型，只在为第一个参数提供索引时使用。

返回。

布尔型

例子。

var _a = array_create(3);
_a[0] = " enemy";
_a[1] = "all_levels";
_a[2] = "boss";
asset_remove_tags(obj_Enemy_Boss_Parent, _a, asset_object);

上面的代码将创建一个标签数组，然后从给定对象中删除它们。

返回。资产和标签
