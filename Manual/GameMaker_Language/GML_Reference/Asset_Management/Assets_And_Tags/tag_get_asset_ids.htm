tag_get_asset_ids

通过这个函数，你可以获得所有给定类型的、有给定标签的资产。你可以提供一个单一的标签（字符串）或者一个数组，其中数组中的每一项都是一个标签（字符串），以及要检查的资产类型。 资产的类型应该是以下常量之一。

恒定 描述
资产_对象 给定的名称指的是一个对象。
资产_石膏 给定的名称指的是一个精灵。
资产_声音 这个名字指的是一种声音。
资产室 这个名字指的是一个房间。
资产 给定的名称指的是一个瓦片组。
资产路径 给定的名称指的是一个路径。
资产脚本 给定的名称指的是一个脚本。
资产字体 该名称指的是一种字体。
资产时间线 这个名字指的是一条时间线。
资产着色器 这个名字指的是一个 着色器.
资产_动画曲线 该名称指的是一个动画曲线。
资产序列 给定的名称指的是一个Sequence。

该函数将返回一个数组，数组中的每一项都是一个资产索引值。如果没有该类型的资产具有给定的标签，将返回一个空数组。

语法。

tag_get_asset_ids(tags, asset_type)。

争论 描述
标签 一个单一的资产标签字符串或一个包含各种资产标签的数组。
资产类型 一个资产类型的常数（上面列出）。

返回。

阵列

例子。

var _paths = tag_get_asset_ids("enemy", asset_path);
var _num = irandom(array_length(_paths) - 1);
path_start(_paths[_num], 1, path_action_reverse, false) 。

上面的代码使用标签 "敌人 "找到所有带有该标签的路径资产，然后随机选择一个并将其分配给运行该代码的实例。

返回。资产和标签
