tag_get_asset_ids

使用此功能，您可以获取给定类型的所有资产，并为其分配了给定标签。您可以提供单个标签（作为字符串）或数组，其中数组中的每个项目都是标签（作为字符串），以及要检查的资产类型。资产类型应为以下常量之一：

持续的描述
asset_object给定名称是指对象。
asset_sprite给定名称是指sprite 。
asset_sound给定的名称是指声音。
asset_room给定名称是指room。
asset_tiles给定名称是指图块集。
asset_path给定名称是指路径。
asset_script给定名称是指script。
asset_font给定的名称是指字体。
asset_timeline给定名称是指时间线。
asset_ shader给定名称是shader 。
asset_animationcurve给定名称是指“动画曲线” 。
asset_sequence给定名称是指Sequence 。

该函数将返回一个数组，其中数组中的每个项目都是单个资产索引值。如果没有具有给定标签的类型的资产，则将返回一个空数组。

句法：

tag_get_asset_ids(tags, asset_type);

争论描述
tags单个资产标签字符串或具有各种资产标签的数组。
asset_type资产类型常量（上面列出）

返回值：

Array

例子：

var _paths = tag_get_asset_ids("enemy", asset_path);
var _num = irandom(array_length(_paths) - 1);
path_start(_paths[_num], 1, path_action_reverse, false);

上面的代码在随机选择一个并将其分配给运行该代码的实例之前，使用标签“敌人”来查找具有该标签的所有路径资产。

返回：资产和标签
