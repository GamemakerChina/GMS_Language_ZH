tag_get_assets

使用此功能，您可以检索已分配给定标签或标签的所有资产的名称。您可以提供一个标签字符串或一个数组，其中数组中的每个项目都是标签字符串。该函数将返回一个数组，其中每个条目都是具有给定标签的资产的名称（作为字符串）。如果您需要资产的唯一索引，则可以使用该功能 asset_get_index()以及返回的名称。如果不存在带有给定标签的资产，或者存在错误（例如：给定标签不存在），则将返回一个空数组。

句法：

tag_get_assets(tags);

争论描述
tags单个资产标签字符串或具有各种资产标签的数组。

返回值：

Array

例子：

backgrounds = ds_list_create();
var _assets = tag_get_assets("background");
for (var i = 0; i< array_length(_assets); ++i;)
    {
    if asset_get_type(_assets[i]) == asset_sprite
        {
        ds_list_add(backgrounds, asset_get_index(_assets[i]));
        }
    }

上面的代码创建一个列表，然后检索带有标签“ background”的所有资产的名称。它遍历返回的名称数组，检查其中是否有一个精灵资源，如果是，则将该资产的唯一索引值添加到列表中以备将来使用。

返回：资产和标签
