audio_get_recorder_count

此功能将返回游戏当前可用的音频记录源（如麦克风等）的数量。因此，如果您的返回值为例如4，那么您将在源0、1、2和3上输入音频，其中一个值就是您使用该函数开始记录的值 audio_start_recording()。当人们将麦克风或其他输入源插入设备时，该值可以随时更改。请注意，您可以使用该功能 audio_get_recorder_info()获取有关连接的每个设备的信息。

重要的！在HTML5目标上使用时，此函数将始终返回0。

句法：

audio_get_recorder_count();

返回值：

Integer

例子：

if audio_get_recorder_count() > 0
   {
   channel_index = audio_start_recording(0);
   }

上面的代码检查可用的录音机，如果有1个或多个，则从第一个索引的记录（源0）开始记录。
