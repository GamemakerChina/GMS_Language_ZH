音频发射器_存在

这个函数返回一个音频发射器是否存在（真）或不存在（假）。注意，如果你搜索的索引之前没有被初始化，这个函数将导致一个错误，因为它正在搜索不存在的索引。

语法。

audio_emitter_exists(index)。

争论 描述
指数 要检查的发射器的索引。

返回。

布尔型

例子。

if audio_emitter_exists(s_emit)
{
audio_play_sound_on(s_emit, snd_Explode, false, 1);
}
else
{
s_emit = audio_emitter_create();
audio_play_sound_on(s_emit, snd_Explode, false, 1);
}

上面的代码检查一个发射器是否存在，索引在（先前初始化的）变量 "s_emit "中。如果存在，就通过它播放声音，如果不存在，就创建它，然后播放声音。

返回。音频发射器
