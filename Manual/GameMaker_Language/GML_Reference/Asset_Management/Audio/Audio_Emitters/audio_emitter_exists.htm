audio_emitter_exists

此函数返回是否存在音频发射器（true ） 或不 （false ）。请注意，如果您搜索的索引先前尚未初始化，则此功能将在搜索不存在的索引时导致错误。

句法：

audio_emitter_exists(index);

争论描述
index发射器的索引以检查是否存在。

返回值：

Boolean

例子：

if audio_emitter_exists(s_emit)
   {
   audio_play_sound_on(s_emit, snd_Explode, false, 1);
   }
else
   {
   s_emit = audio_emitter_create();
   audio_play_sound_on(s_emit, snd_Explode, false, 1);
   }

上面的代码检查是否存在一个发射器，该发射器在（先前初始化的）变量“ s_emit”中索引。如果是，则播放声音，如果没有，则创建声音，然后播放声音。
