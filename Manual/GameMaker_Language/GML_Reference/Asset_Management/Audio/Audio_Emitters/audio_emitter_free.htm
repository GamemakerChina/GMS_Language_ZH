audio_emitter_free

使用此功能，您可以从内存中删除给定的发射器。每当不打算在房间或游戏中进一步使用发射器时，都应始终执行此操作，即：在实例的Destroy Event或Room End Event中，否则您可能会因内存泄漏而减慢速度并最终导致游戏崩溃。

句法：

audio_emitter_free(emitter);

争论描述
emitter发射器释放的索引。

返回值：

N/A

例子：

if lives = 0
   {
   audio_emitter_free(s_emit);
   room_goto(rm_Menu);
   }

上面的代码检查全局变量“ lives”的值，如果它返回0，它将破坏在变量“ s_emit”中索引的发射器，然后转到在变量“ rm_Menu”中索引的房间。
