audio_emitter_gain

此功能设置声音的最大增益（音量）。声音的感知音量可以根据衰减值和相对于收听者的位置而变化，但是通过使用此功能设置增益，总音量将永远不会超过指定的增益值。下图说明了当衰减大于0时增益如何影响发射器的体积：

此功能将改变正在播放的声音的音量，以及通过给定发射器播放的所有后续声音。请注意，在某些平台上，您可以获得大于1的增益，尽管将值1视为“全音量”，并且更大的值可能会导致音频削波或失真。

注意：最终音量也会受到函数audio_master_gain（）设置的全局音频增益的影响。

句法：

audio_emitter_gain(emitter, gain);

争论描述
emitter要更改的发射器的索引。
gain最大增益（默认为1）。

返回值：

N/A

例子：

if up
   {
   gain += 0.05;
   if gain >= 1 up = false;
   }
else
   {
   gain += 0.05;
   if gain <= 0 up = true;
   }
audio_emitter_gain(s_emit, gain);

上面的代码将变量“ gain”设置为不同的值，然后使用相同的变量来设置在变量“ s_emit”中索引的发射器的增益。
