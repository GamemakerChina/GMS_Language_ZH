audio_emitter_get_gain

此函数返回为给定音频发射器设置的当前增益（音量），通常在0到1之间，其中0为静音，而1为全音量。请注意，在某些平台上，您可以将增益设置为大于1，尽管将值1视为“全音量”，任何更大的值可能会引入音频剪辑。

句法：

audio_emitter_get_gain(emitter);

争论描述
emitter要使用的发射器的索引。

返回值：

Real

例子：

if audio_emitter_get_gain(emitter_player) < 1
   {
   audio_emitter_gain(emitter_player, 1);
   }

上面的代码检查给定发射器的当前增益，如果小于1，则将其设置为1。
