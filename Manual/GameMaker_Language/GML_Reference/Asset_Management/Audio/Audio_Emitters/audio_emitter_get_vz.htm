音频_发射器_get_vz

该函数返回给定音频发射器沿Z轴的当前速度。

语法。

audio_emitter_get_vz(emitter)。

争论 描述
发射器 要使用的发射器的索引。

返回。

真实的

例子。

if audio_emitter_get_vz(emitter_player) != 0
{
audio_emitter_velocity(emitter_player, 0, 0, 0);
}

上面的代码检查某个发射器的当前z速度，如果不等于0，就把它设为0。
