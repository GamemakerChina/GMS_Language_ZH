audio_emitter_get_x

此函数返回给定音频发射器的当前x位置。

句法：

audio_emitter_get_x(emitter);

争论描述
emitter要使用的发射器的索引。

返回值：

Real

例子：

if audio_emitter_get_x(emitter_player) != x
   {
   audio_emitter_position(emitter_player, x, y, 0);
   }

上面的代码检查给定发射器的当前x位置，如果它不等于实例x位置，则将其设置为实例位置。
