audio_emitter_pitch

此功能可用于更改从给定发射器发出的所有声音的音高。这是一个音高倍增器，因为输入值将当前音高乘以该量，因此默认值1不改变音高，而小于1的值将降低音高，大于1的值将提高音高。最好对此功能使用较小的增量，因为无论如何都不会听到小于0或大于5的任何值。

注意：如果声音正在通过发射器循环播放，除非停止并再次循环播放声音，否则不会检测到音高变化！

句法：

audio_emitter_pitch(emitter, pitch);

争论描述
emitter要更改的发射器的索引。
pitch音高倍数（默认为1）。

返回值：

N/A

例子：

switch (gear)
   {
   case 1: audio_emitter_pitch(s_emit, 0.8); break;
   case 2: audio_emitter_pitch(s_emit, 0.9); break;
   case 3: audio_emitter_pitch(s_emit, 0.95); break;
   case 4: audio_emitter_pitch(s_emit, 1); break;
   case 5: audio_emitter_pitch(s_emit, 1.2); break;
   }

上面的代码将基于变量“ gear”的值来更改从变量“ s_emit”中索引的发射器播放的音频的音高。
