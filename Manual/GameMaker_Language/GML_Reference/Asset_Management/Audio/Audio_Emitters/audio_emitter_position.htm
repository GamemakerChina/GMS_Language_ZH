音频发射器位置

通过这个函数，你可以改变一个音频发射器在三维音频空间中的位置。这个位置将以不同的方式影响声音，取决于听众在音频空间中的位置 也是如此（默认位置是（0, 0, 0）。参见 audio_listener_position()获取更多信息），因此，例如，如果发射器的位置被设置为（100, 0, 0），而 当前监听者在(200, 0, 0)，从发射器流出来的音频会出现在音频区域的左边。下面的图片显示了发射器及其与监听者的相对位置的可视化表示。

语法。

audio_emitter_position(emitter, x, y, z);

争论 描述
发射器 要改变的发射器的索引。
x X的位置。
y y的位置。
z z的位置。

返回。

不适用

例子。

如果速度>0
{
audio_emitter_position(s_emit, x, y, 0);
}

上面的代码检查实例的速度是否超过0，如果是，它就会将变量 "s_emit "中索引的音频发射器更新到当前的x/y位置。
