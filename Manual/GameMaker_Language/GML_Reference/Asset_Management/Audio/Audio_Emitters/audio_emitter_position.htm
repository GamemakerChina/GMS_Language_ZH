audio_emitter_position

使用此功能，您可以更改3D音频空间中音频发射器的位置。该位置也会根据收听者在音频空间中的位置而以不同的方式影响声音（默认位置是（0，0，0）。请参见 audio_listener_position()有关更多信息），例如，如果将发射器的位置设置为（100，0，0），而当前的侦听器位于（200，0，0），则从发射器流式传输的音频将显示在屏幕的左侧。音频场。下图显示了发射器及其相对于侦听器的相对位置的直观表示：

句法：

audio_emitter_position(emitter, x, y, z);

争论描述
emitter要更改的发射器的索引。
x x位置。
y y位置。
z z位置。

返回值：

N/A

例子：

if speed > 0
   {
   audio_emitter_position(s_emit, x, y, 0);
   }

上面的代码检查实例速度是否超过0，是否将其更新为变量“ s_emit”中索引的音频发射器到当前x / y位置。
