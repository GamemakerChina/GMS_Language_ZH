audio_group_load

此功能会将标记为属于给定音频组的所有声音加载到内存中。该函数将返回 true如果启动加载并且 false如果组ID无效，或者已经将其标识为要加载。该功能是异步的，因此您的游戏将在后台加载音频时继续运行。这意味着它还将触发一个异步加载/保存事件，以通知您整个组已被加载到内存中，并且现在可以使用声音了。

句法：

audio_group_load(groupID);

争论描述
groupID要加载的音频组的索引（在“音频组”窗口中定义）

返回值：

Boolean

例子：

if !audio_group_is_loaded(audiogroup_level1)
   {
   audio_group_load(audiogroup_level1);
   }

上面的代码检查是否已加载音频组，如果未加载，则将其加载。

返回：音频组
