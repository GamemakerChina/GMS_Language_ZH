audio_group_name

此函数将返回一个字符串，其中包含用于显示或检查的给定音频组的名称。当您在游戏选项定义音频组，你给它一个独特的“名字”，这确实是作为一个ID价值的组使用一个常数。此函数所做的全部工作就是获取ID，并返回您提供的ID名称的字符串。

句法：

audio_group_name(groupID);

争论描述
groupID要检查的音频组的索引值常数（在“音频组”窗口中定义）

返回值：

String

例子：

var name= audio_group_name(audiogroup_level1);
draw_text(32, 32, "Now Playing Group: " + name);

上面的代码检索给定音频组常量的名称，并将其显示在屏幕上。

返回：音频组
