audio_group_set_gain

使用此功能，您可以在给定的时间内淡入或淡出一组声音，也可以用来立即设置该组的增益。时间以毫秒为单位，该功能要求您输入到该时间结束时该组要达到的最终增益水平。该增益可以在0（静音）和1（完整音量）之间，并且比例是线性的，因此0.5的值将是一半音量。要立即更改增益，只需将time参数设置为0。请注意，在某些平台上，您可以将增益设置为大于1，尽管将值1视为“最大音量”，而更大的值可能会引入音频剪辑。

句法：

audio_group_set_gain(groupID, volume, time);

争论描述
groupID要停止的音频组的索引（在“音频组”窗口中定义）
volume组音量的最终值。
time增益变化的长度，以毫秒为单位。

返回值：

N/A

例子：

if keyboard_check_pressed(vk_space)
   {
   audio_group_set_gain(audiogroup1, 0, 5000);
   }

上面的代码检查“ space”键，然后在5秒钟内将“ audiogroup1”的所有音频衰减为0。

返回：音频组
