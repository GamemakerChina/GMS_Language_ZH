音频组卸载

这个函数将把所有被标记为属于给定音频组的声音卸载到内存中。如果卸载已经开始，该函数将返回true，如果 组的ID无效，或者已经被标记为卸载。注意，当这个函数被调用时，当前正在播放的任何音频将被停止。

语法。

audio_group_unload(groupID)。

争论 描述
groupID 要卸载的音频组的索引（如在音频组窗口中定义）。

返回。

布尔型

例子。

if audio_group_is_loaded(audiogroup_level1)
{
audio_group_unload(audiogroup_level1)。
}

上面的代码检查是否加载了一个音频组，如果加载了，就把它卸载。
