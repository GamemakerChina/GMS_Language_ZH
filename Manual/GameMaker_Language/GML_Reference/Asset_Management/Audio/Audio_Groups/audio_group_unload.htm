audio_group_unload

此功能会将标记为属于给定音频组的所有声音卸载到内存中。该函数将返回 true如果开始卸载并且 false如果组ID无效，或者已经将其标识为要卸载。请注意，调用此功能时当前正在播放的任何音频都将停止。

句法：

audio_group_unload(groupID);

争论描述
groupID要卸载的音频组的索引（在“音频组”窗口中定义）

返回值：

Boolean

例子：

if audio_group_is_loaded(audiogroup_level1)
   {
   audio_group_unload(audiogroup_level1);
   }

上面的代码检查音频组是否已加载以及是否将其卸载。

返回：音频组
