audio_get_listener_info

此函数将创建一个DS映射，并使用给定侦听器的信息填充它。

注意：您应负责销毁返回的产品 DS Map使用适当的功能。

DS映射将包含以下键：

  • “ name ”-侦听器的名称，以字符串形式，其中“ default”是大多数目标平台上的标准侦听器名称
  • “ mask ”-侦听器的位掩码
  • “ index ”-侦听器的唯一索引值

可以使用mask值设置声音或发射器一次从多个听众中播放，只需按位 or“ |”生成声音的遮罩（请参见下面的示例代码），而索引用于使用诸如 audio_listener_set_position()。

句法：

audio_get_listener_info(num);

争论描述
num要为其获取数据的侦听器编号。

返回值：

ds_map

例子：

var num = audio_get_listener_count();
var mask = 0; for(var i = 0; i < num; ++i;)
   {
   var info = audio_get_listener_info(i);
   var m = audio_listener_get_data(info[? "mask"]);
   mask = mask | m;
   ds_map_destroy(info);
   }
audio_set_listener_mask(mask);

上面的代码检查可用侦听器的数量，然后循环遍历它们以获取其掩码位，然后将其组合以创建应用于全局侦听器的单个位掩码。

返回：音频听众
