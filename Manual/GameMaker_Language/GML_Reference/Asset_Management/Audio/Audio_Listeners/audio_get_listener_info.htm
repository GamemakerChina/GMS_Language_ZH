音频_get_listener_info

这个函数将创建一个DS地图，并为其填充给定监听器的信息。

注意： 你要负责使用适当的功能销毁返回的DS地图。

DS地图将包含以下键。

  • " name"- 倾听者的名字，是一个字符串，"default "是大多数目标平台上的标准倾听者名字。
  • " mask"- 倾听者的位掩码。
  • "index" - 听众的唯一索引值。

掩码值可以用来设置一个声音或发射器同时从多个听众处播放，只需使用位法或"|"来生成声音的掩码（见下面的示例代码），而索引则用来设置属性 如位置或速度，使用函数如 audio_listener_set_position().

语法。

audio_get_listener_info(num)。

争论 描述
数值 要获取数据的听众号码。

返回。

ds_map

例子。

var num = audio_get_listener_count();
var mask = 0; for(var i = 0; i < num; ++i;)
{
var info = audio_get_listener_info(i);
var m = audio_listener_get_data(info[? "mask"]);
mask = mask | m;
ds_map_destroy(info);
}
audio_set_listener_mask(mask)。

上面的代码检查可用的监听器的数量，然后在它们之间循环，得到它们的掩码位，然后将它们组合起来，创建一个单一的位掩码，应用于全局监听器。

返回。音频收听者
