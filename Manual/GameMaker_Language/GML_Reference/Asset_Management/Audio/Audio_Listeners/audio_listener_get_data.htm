音频监听者获取数据

这个函数将创建一个DS地图，并用给定监听器的位置、速度和方向值来填充它。默认监听器的索引是0，但你可以使用函数 audio_get_listener_info()来获取目标平台的不同指数。如果你提供了一个不正确的监听器索引，那么这个函数将返回-1。

注意： 你要负责使用适当的功能销毁返回的 DS地图 。

DS地图将包含以下键。

  • " x" - 听众的X位置
  • " y" - 听众的y位置
  • "z" - 听众的z位置
  • " vx" - 听众的X轴速度
  • " vy" - 听众的Y轴速度
  • " vz" - 听众的Z轴速度
  • "lookat_x" - 倾听者的观察向量的X分量。
  • "lookat_y" - 倾听者的观察向量的Y分量。
  • "lookat_z" - 倾听者的观察向量的Z分量。
  • "up_x"- 倾听者向上的向量的x分量。
  • "up_y"- 倾听者向上的矢量的y分量。
  • "up_z"- 听众向上的矢量的z分量。

语法。

audio_listener_get_data(index)。

争论 描述
指数 要获取数据的监听器（默认为0）。

返回。

ds_map

例子。

var num = audio_get_listener_count();
for(var i = 0; i < num; ++i;)
{
var info = audio_get_listener_info(i);
var data = audio_listener_get_data(info[? "index"]);
if data[? "x"] != 0
{
audio_listener_set_position(info[? "index"], 0, 0, 0);
}
ds_map_destroy(info)。
ds_map_destroy(data)。
}

上面的代码检查可用的监听器的数量，然后在它们之间循环，如果它们的X位置不是0，就将它们的位置设置为0, 0, 0。
