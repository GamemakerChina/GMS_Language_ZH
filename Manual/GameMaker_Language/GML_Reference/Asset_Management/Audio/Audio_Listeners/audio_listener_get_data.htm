audio_listener_get_data

此功能将创建一个DS映射，并使用给定侦听器的位置，速度和方向值填充它。默认的侦听器索引为0，但是您可以使用该函数 audio_get_listener_info()获得适用于目标平台的不同索引。如果您提供了错误的侦听器索引，则该函数将返回-1。

注意：您有责任使用适当的功能销毁返回的DS地图。

DS映射将包含以下键：

  • “ x ”-侦听器的x位置
  • “ y ”-侦听器的y位置
  • “ z ”-侦听器的z位置
  • “ vx ”-侦听器的x轴速度
  • “ vy ”-侦听器的y轴速度
  • “ vz ”-侦听器的z轴速度
  • “ lookat_x ”-侦听器的向量外观的x分量
  • “ lookat_y ”-侦听器的看向向量的y分量
  • “ lookat_z ”-侦听器的看向向量的z分量
  • “ up_x ”-侦听器的up矢量的x分量
  • “ up_y ”-侦听器的up矢量的y分量
  • “ up_z ”-侦听器的up矢量的z分量

句法：

audio_listener_get_data(index);

争论描述
index用于获取数据的侦听器（默认为0）。

返回值：

ds_map

例子：

var num = audio_get_listener_count();
for(var i = 0; i < num; ++i;)
   {
   var info = audio_get_listener_info(i);
   var data = audio_listener_get_data(info[? "index"]);
   if data[? "x"] != 0
      {
      audio_listener_set_position(info[? "index"], 0, 0, 0);
      }
   ds_map_destroy(info);
   ds_map_destroy(data);
   }

上面的代码检查可用的侦听器的数量，然后遍历它们，如果它们的x位置不为0，则将其位置设置为0、0、0。

返回：音频听众
