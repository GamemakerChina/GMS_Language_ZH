audio_listener_position

使用此功能，您可以更改3D音频空间中听众的位置。下图示例显示了侦听器在音频空间中的默认位置：

注意：如果您有多个侦听器，则应使用该功能 audio_listener_set_position().

如您所见，默认位置为（0，0，0），但是通常您会使用此功能在游戏中将播放器对象与侦听器一起移动，从而改变播放器听到发射器创建的音频的方式，例如，在自上而下游戏的下图中，玩家实例设置了侦听器，当玩家在关卡中移动时，它将导致来自各个发射器的音频“改变”：

句法：

audio_listener_position(x, y, z);

争论描述
x侦听器的x位置（默认为0）。
y侦听器的y位置（默认为0）。
z侦听器的z位置（默认为0）。

返回值：

N/A

例子：

if speed > 0
   {
   audio_listener_position(x, y, 0);
   }

上面的代码检查播放器实例的速度是否超过0，并且是否将其更新为当前x / y位置的音频侦听器。

返回：音频听众
