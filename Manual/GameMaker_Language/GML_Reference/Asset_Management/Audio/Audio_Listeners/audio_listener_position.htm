音频听众位置

通过这个功能，你可以改变听众在三维音频空间中的位置。下面的示例图片显示了听众在音频空间中的默认位置。

注意： 如果你有多个监听器，你应该使用函数audio_listener_set_position()。

正如你所看到的，默认位置是(0, 0, 0)，但你通常会使用这个函数在你的游戏中随玩家对象移动监听器，从而改变玩家听到发射器产生的音频的方式。 例如，在下面这个自上而下的游戏的图片中，玩家实例设置了监听器，当玩家在关卡中移动时，会导致来自各种发射器的音频 "改变"。

语法。

audio_listener_position(x, y, z);

争论 描述
x 听众的X位置（默认为0）。
y 倾听者的y位置（默认为0）。
z 倾听者的z位置（默认为0）。

返回。

不适用

例子。

如果速度>0
{
audio_listener_position(x, y, 0);
}

上面的代码检查播放器实例的速度是否超过0，如果是，它就把音频监听器更新到当前的X/Y位置。
