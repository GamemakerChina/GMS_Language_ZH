audio_listener_set_position

使用此功能，您可以在3D音频空间中更改给定收听者的位置。默认的侦听器索引为0，但是您可以使用该函数 audio_get_listener_info()获得适用于目标平台的不同索引。下图示例显示了侦听器在音频空间中的默认位置：

如您所见，默认位置为（0，0，0），但是通常您会使用此功能在游戏中将播放器对象与侦听器一起移动，从而改变播放器听到发射器创建的音频的方式，例如，在自上而下游戏的下图中，玩家实例设置了侦听器，当玩家在关卡中移动时，它将导致来自各个发射器的音频“改变”：

句法：

audio_listener_set_position(index, x, y, z);

争论描述
index用于获取数据的侦听器（默认为0）。
x侦听器的x位置（默认为0）。
y侦听器的y位置（默认为0）。
z侦听器的z位置（默认为0）。

返回值：

N/A

例子：

var _vmat = camera_get_view_mat(view_camera[0]);
audio_listener_set_position(global.Player_Listener, _vmat[0], _vmat[1], _vmat[2]);
audio_listener_set_orientation(info[? "index"], _vmat[3], _vmat[4], _vmat[5], _vmat[6], _vmat[7], _vmat[8]);

上面的代码检索摄像机视图[0]的视图矩阵，然后使用它为具有存储在全局变量“ Player_Listener”中的ID的侦听器设置音频侦听器的位置和方向。

返回：音频听众
