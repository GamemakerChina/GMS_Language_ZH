audio_listener_velocity

此功能可用于赋予听众多普勒效果，并基于从给定的相对x，y和z位置解析出的矢量模拟音频运动。如果侦听器本身永远不会移动，或者该移动不是恒定运动，则通常不需要设置这些值，但是，例如，如果您正在制作滚动游戏，其中玩家具有恒定的下限，顶部运动和敌人恒定的上下运动，您可以设置侦听器和发射器的速度（对于发射器，您可以使用 audio_emitter_velocity()）转换为适当的向量，以在它们经过播放器实例时模拟正确的多普勒效果。

注意：如果您有多个侦听器，则应使用函数audio_listener_set_velocity（） 。

下图显示了如何为上述示例游戏进行设置：

句法：

audio_listener_velocity(vx, vy, vz);

争论描述
vx侦听器的x速度分量（默认为0）。
vy侦听器的y速度分量（默认为0）。
vz侦听器的z速度分量（默认为0）。

返回值：

N/A

例子：

if speed > 0
   {
   audio_listener_velocity(abs(hspeed), abs(vspeed), 0);
   }

上面的代码检查播放器实例速度是否超过0，如果超过0，则使用适当的绝对值 hspeed和 vspeed设置收听者速度的组件。

返回：音频听众
