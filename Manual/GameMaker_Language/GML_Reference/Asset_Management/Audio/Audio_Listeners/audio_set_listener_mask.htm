音频_set_listener_mask

当在一个系统上使用多个听众时，你可以为一个声音设置位掩码，让它只从标记的听众那里听到。但是，你也可以用这个功能设置全局掩码，所有正常播放的声音都会从这个掩码标记的听众那里听到。 所有正常播放的声音都会被这个掩码标记的听众听到，而不需要为每个声音单独设置掩码。

语法。

audio_set_listener_mask(mask)。

争论 描述
面罩 要为听众设置的比特掩码数据

返回。

不适用

例子。

var num = audio_get_listener_count();
var mask = 0; for(var i = 0; i < num; ++i;)
{
var info = audio_get_listener_info(i);
var m = audio_listener_get_data(info[? "mask"]);
mask = mask | m;
ds_map_destroy(info);
}
audio_set_listener_mask(mask)。

上面的代码检查可用的监听器的数量，然后在它们之间循环，得到它们的掩码位，然后将它们组合起来，创建一个单一的位掩码，应用于全局监听器。

返回。音频收听者
