audio_pause_sync_group

如果正在播放给定的同步组，则该函数将暂停该组，其中组索引是使用该功能创建组时返回的值 audio_create_sync_group()。这不会停止声音，并发出 audio_resume_sync_group()，将从重新暂停的相同位置开始播放。

注意：此功能不适用于HTML5目标平台。

句法：

audio_pause_sync_group(group_index);

争论描述
group_index要暂停的组索引。

返回值：

N/A

例子：

if (keyboard_check_pressed((ord)"P"))
   {
   global.Pause = !global.Pause
   if global.Pause
      {
      audio_pause_sync_group(sg);
      }
   else
      {
      audio_resume_sync_group(sg);
      }
   }

上面的代码检查是否按下了“ P”键，如果检测到按键，则会切换“ global.Pause”变量，然后检查它是否暂停或恢复索引在变量“ sg”中的同步组。
