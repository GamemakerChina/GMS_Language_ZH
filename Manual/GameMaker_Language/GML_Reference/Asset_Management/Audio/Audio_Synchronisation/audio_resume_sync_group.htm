音频恢复_同步组

这个函数将恢复给定的同步组，如果它正在播放并且之前已经暂停了（使用函数 音频同步组(audio_pause_sync_group)).组的索引是你用函数创建组时返回的值。 时返回的值。 audio_create_sync_group().

注意：此功能不适用于HTML5目标平台。

语法。

audio_resume_sync_group(group_index);

争论 描述
组_索引 该小组指数恢复。

返回。

不适用

例子。

如果(keyboard_check_pressed((ord) "P"))
{
global.Pause = !global.Pause
if global.Pause
{
audio_pause_sync_group(sg);
}
else
{
audio_resume_sync_group(sg);
}
}

上面的代码检查是否有 "P "键被按下，如果检测到，它将切换 "global.Pause "变量，然后检查它以暂停或恢复变量 "sg "中索引的同步组。

返回。音频同步化
