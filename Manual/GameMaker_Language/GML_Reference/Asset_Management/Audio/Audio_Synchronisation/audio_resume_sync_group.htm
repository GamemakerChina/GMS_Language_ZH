audio_resume_sync_group

如果正在播放且先前已暂停的给定同步组，此功能将恢复它（使用该功能 audio_pause_sync_group）。组索引是使用函数创建组时返回的值 audio_create_sync_group()。

注意：此功能不适用于HTML5目标平台。

句法：

audio_resume_sync_group(group_index);

争论描述
group_index要恢复的组索引。

返回值：

N/A

例子：

if (keyboard_check_pressed((ord)"P"))
   {
   global.Pause = !global.Pause
   if global.Pause
      {
      audio_pause_sync_group(sg);
      }
   else
      {
      audio_resume_sync_group(sg);
      }
   }

上面的代码检查是否按下了“ P”键，如果检测到按键，则会切换“ global.Pause”变量，然后检查它是否暂停或恢复索引在变量“ sg”中的同步组。
