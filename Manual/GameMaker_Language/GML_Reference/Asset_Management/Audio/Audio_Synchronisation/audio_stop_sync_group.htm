audio_stop_sync_group

如果正在播放给定的同步组，此功能将停止该同步组，其中组索引是使用该功能创建组时返回的值 audio_create_sync_group()。

注意：此功能不适用于HTML5目标平台。

句法：

audio_stop_sync_group(group_index);

争论描述
group_index停止播放的组索引。

返回值：

N/A

例子：

if mouse_check_button_pressed(mb_left)
   {
   audio_stop_sync_group(sg);
   }

上面的代码检查是否有鼠标单击，如果检测到单击，则会停止索引在变量“ sg”中的同步组。
