audio_sync_group_get_track_pos

此函数返回给定同步组的当前播放位置。组索引是使用函数创建组时返回的值 audio_create_sync_group()，返回值是曲目播放的时间（以秒为单位）。

注意：此功能不适用于HTML5目标平台。

句法：

audio_sync_group_get_track_pos(group_index);

争论描述
group_index要获取位置的组索引。

返回值：

Real

例子：

var real_secs = audio_sync_group_get_track_pos(sg);
var secs = real_secs mod 60;
var mins = string(real_secs div 60);
if (secs > 9)
   {
   secs = string(secs);
   }
else
   {
   secs = "0" + string(secs);
   }
draw_text(32, 32, "Time = " + mins + ":" + secs);

上面的代码获取在变量“ sg”中索引的同步组的时间位置，然后使用此值在屏幕上绘制播放的时间。

背面：音频同步
