audio_sync_group_is_playing

此功能可用于检查同步组中是否正在播放任何音频。您需要提供函数返回的同步组ID audio_create_sync_group() 。

注意：此功能不适用于HTML5目标平台。

句法：

audio_sync_group_is_playing(group_index);

争论描述
group_index要检查的组索引。

返回值：

N/A

例子：

if audio_sync_group_is_playing(group_one)
   {
   audio_stop_sync_group(group_one);
   }

上面的代码检查同步组是否 group_one当前正在播放，如果正在播放，则该组已停止。
