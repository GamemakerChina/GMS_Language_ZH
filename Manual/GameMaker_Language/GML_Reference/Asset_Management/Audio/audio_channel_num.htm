audio_channel_num

GameMaker Studio 2有多少个音频通道可用于播放音频。这基本上是指您指定一次可以播放的同时声音的数量，如果数量超过该数量，则将停止优先级较低的声音，以释放优先级较高的声音的通道。您可以使用此功能针对设备优化游戏，因为音频通道数越少越好，但请记住，如果一次播放多个声音，这也会导致某些声音的明显中断。

句法：

audio_channel_num(num);

争论描述
num可用音频通道数（默认为128）。

返回值：

N/A

例子：

switch (os_browser)
   {
   case browser_not_a_browser:
      switch (os_type)
         {
         case os_windows:
         case os_macos:
            audio_channel_num(200);
            break;
         default:
            audio_channel_num(64);
            break;
         }
      break;
   default:
      audio_channel_num(16);
      break;
   }

上面的代码检查游戏运行的平台，并更改可用声音通道的数量以提高性能。
