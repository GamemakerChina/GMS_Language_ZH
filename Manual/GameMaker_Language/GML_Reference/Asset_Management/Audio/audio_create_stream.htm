audio_create_stream

使用此功能，您可以创建一个新的声音索引，然后将其用于常规音频功能中，以直接从外部OGG文件源流式传输音频。该函数需要文件名（例如，可以是包含文件），并将返回新的声音索引以供使用。请注意，在不再需要声音之后，应调用该函数 audio_destroy_stream()使用声音索引将其从内存中删除，否则可能会导致内存泄漏，从而减慢速度并最终导致游戏崩溃。

注意：此功能不适用于HTML5目标平台。

句法：

audio_create_stream(filename);

争论描述
filename要从中流音频的文件（仅OGG）。

返回值：

Sound Index

例子：

snd = audio_create_stream("Music/Track1.ogg");
audio_play_sound(snd, 0, true);

上面的代码在给定文件的变量“ snd”中创建一个新的声音索引，然后播放该声音。
