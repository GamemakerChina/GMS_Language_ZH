audio_exists

此函数返回声音是否存在（true ） 或不 （false ）。请注意，如果您搜索的索引先前尚未初始化，则此功能将在搜索不存在的资产索引时导致错误。要检查的声音可以是声音的单个实例（使用时可以将正在播放的单个声音的索引存储在变量中）。audio_play_sound()或者 audio_play_sound_at()功能）或健全资产。

句法：

audio_exists(index);

争论描述
index要检查声音是否存在的索引。

返回值：

Boolean

例子：

if audio_exists(global.Music)
   {
   audio_play_sound(global.Music, 0, true);
   }

上面的代码检查是否存在声音。如果已设置，则设置为循环播放。
