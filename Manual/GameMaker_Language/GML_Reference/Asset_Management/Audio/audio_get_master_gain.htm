audio_get_master_gain

使用此功能，您可以获取特定听众的所有声音和音乐的整体音量的绝对值。默认的侦听器索引为0，但是您可以使用该函数 audio_get_listener_info()获得适用于目标平台的不同索引。返回的增益值基于从0（静音）到1（最大音量）的线性比例。请注意，在某些平台上，您可以将增益设置为大于1，尽管将值1视为“全音量”，任何更大的值可能会引入音频剪辑。

句法：

audio_get_master_gain(listenerIndex);

争论描述
listenerIndex要获得收益的监听器的索引。

返回值：

Real

例子：

var num = audio_get_listener_count();
for( var i = 0; i < num; ++i;)
   {
   var info = audio_get_listener_info(i);
   var ind = info[? "index"];
   if audio_get_master_gain(ind) != 1
      {
      audio_set_master_gain(info[? "index"], 1);
      }
   ds_map_destroy(info);
   }

上面的代码循环遍历了可用的侦听器，检查它们的增益是否为1，如果不是，则将每个增益设置为1。
