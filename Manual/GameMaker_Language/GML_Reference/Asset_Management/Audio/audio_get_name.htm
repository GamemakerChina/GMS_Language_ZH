audio_get_name

此函数将以字符串形式返回给定音频资源的名称。 “索引”值可以是资源本身的值（如在资源浏览器中看到的），也可以是您使用以下方式播放声音时给出的唯一ID值：audio_play_sound() 。请注意，返回的字符串是不一样的资源ID，并且不能用于访问资源本身，所以应该只用于显示或错误检查。

句法：

audio_get_name(index);

争论描述
index要检查的声音的索引。

返回值：

String

例子：

var snd = audio_play_sound(choose(snd_One, snd_Two, snd_Three), 0, false);
var name = audio_get_name(snd);
show_debug_message("Sound = " + name);

上面的代码播放从三种不同的声音资源中选择的随机声音，然后显示调试消息及其名称。
