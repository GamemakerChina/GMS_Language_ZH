audio_get_type

GameMaker Studio 2添加音频时，可以将其流式传输或存储在内存中。如果您需要知道给定的声音索引是否用于流音频，则可以使用此函数，该函数将为流返回1，为内存中的声音返回0，如果有任何错误或索引未指向a则返回-1。有效的声音资源。

句法：

audio_get_type(index);

争论描述
index要检查的声音的索引。

返回值：

Real

例子：

type = audio_get_type(snd_Music_1);

上面的代码检查在变量“ snd_Music_1”中索引的音频的类型，并将返回的值存储在变量“ type”中。
