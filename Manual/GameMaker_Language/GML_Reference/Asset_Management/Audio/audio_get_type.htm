音频_get_type

当向GameMaker Studio 2添加音频时，它可以是流式的，也可以是内存的。如果你需要知道一个给定的声音索引是否为流式音频，你可以 使用这个函数，如果有任何错误或索引没有指向有效的声音资源，它将返回1表示流式，0表示内存中的声音，而-1则表示流式。

语法。

audio_get_type(index)。

争论 描述
指数 要检查的声音的索引。

返回。

真实的

例子。

type = audio_get_type(snd_Music_1);

上述代码检查变量 "snd_Music_1 "中索引的音频类型，并将返回值存储在变量 "type "中。

返回。音频
