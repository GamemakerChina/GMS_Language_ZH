audio_is_paused

此功能将检查给定的声音，以查看当前是否已暂停。声音可以是声音的单个实例（使用 audio_play_sound()或者 audio_play_sound_at()函数）或声音资产，在这种情况下，将检查给定声音的所有实例，如果其中任何一个实例被暂停，该函数将返回 true否则它将返回 false。

句法：

audio_is_paused(index);

争论描述
index要检查的声音的索引。

返回值：

Boolean

例子：

if audio_is_paused(snd_Waterfall)
   {
   audio_resume_sound(snd_Waterfall);
   }

上面的代码检查以查看在变量“ snd_Waterfall”中索引的声音当前是否已暂停以及是否返回 true然后将恢复声音的播放。
