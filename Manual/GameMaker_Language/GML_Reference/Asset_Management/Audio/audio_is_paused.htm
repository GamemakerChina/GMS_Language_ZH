暂停的音频

这个函数将检查给定的声音，看它目前是否暂停了。这个声音可以是一个单一的声音实例（当使用了 "Audio_play_sound_at "后，正在播放的单个声音的索引可以存储在一个变量中。 audio_play_sound()或 audio_play_sound_at()函数）或一个声音资产，在这种情况下，给定的声音的所有实例将被检查，如果其中任何一个暂停，该函数将返回true，否则 它将返回false。

语法。

audio_is_paused(index)。

争论 描述
指数 要检查的声音的索引。

返回。

布尔型

例子。

if audio_is_paused(snd_Waterfall)
{
audio_resume_sound(snd_Waterfall);
}

上面的代码检查变量 "snd_Waterfall "中索引的声音目前是否暂停，如果它返回真，那么声音的播放就会恢复。
