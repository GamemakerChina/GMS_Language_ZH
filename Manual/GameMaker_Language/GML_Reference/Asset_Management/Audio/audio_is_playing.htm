audio_is_playing

此功能将检查给定的声音，查看当前是否正在播放。声音可以是声音的单个实例（使用 audio_play_sound()或者 audio_play_sound_at()函数）或声音资产，在这种情况下，将检查给定声音的所有实例，如果其中任何一个正在播放，则函数将返回 true否则它将返回 false。请注意，此函数仍将返回 true如果之前使用暂停了检查的声音 audio_pause_sound()功能。

句法：

audio_is_playing(index);

争论描述
index要检查的声音的索引。

返回值：

Boolean

例子：

if !audio_is_playing(snd_Waterfall)
   {
   audio_play_sound_at(snd_Waterfall, x, y, 0, 300, true, 1);
   }

上面的代码检查是否正在播放变量“ snd_Waterfall”中索引的声音以及是否返回 false那么声音将在其房间位置循环播放，衰减距离为300，优先级较低。
