audio_master_gain

使用此功能，您可以设置所有声音和音乐的整体音量的绝对值。它基于从0（无声）到大于0的任何值的线性比例，尽管通常您会认为最大音量为1。可以使用大于1的任何值，但取决于所使用的声音和所编译的平台，播放声音时可能会失真或削波。此功能将影响游戏中播放的所有声音和音乐的相对音量。

句法：

audio_master_gain(gain);

争论描述
gain全局卷的值（0到1）。

返回值：

N/A

例子：

if keyboard_check(vk_up)
   {
   if vol < 1 vol += 0.05;
   audio_master_gain(vol);
   }
if keyboard_check(vk_down)
   {
   if vol > 0 vol -= 0.05;
   audio_master_gain(vol);
   }

上面的代码检查向上和向下箭头键的按键，然后按增大或减小变量“ vol”。然后，此变量用于设置游戏中所有声音和音乐的全局增益。
