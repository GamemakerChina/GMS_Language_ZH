音频暂停_所有

通过这个功能，你可以暂停当前正在播放的所有声音。

语法。

audio_pause_all()。

返回。

不适用

例子。

if keyboard_check_pressed(ord("P"))
{
global.Pause = !global.Pause;
if global.Pause
{
audio_pause_all()。
}
else
{
audio_resume_all()。
}
}

上面的代码检查键盘上的 "P "键是否被按下，如果检测到，它将全局变量 "Pause "设置为真或假，然后暂停所有的声音，或重新启动所有先前的 暂停的声音。
