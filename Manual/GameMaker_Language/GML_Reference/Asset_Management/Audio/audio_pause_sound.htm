audio_pause_sound

使用此功能，您可以暂停当前正在播放的任何声音。声音可以是声音的单个实例（使用 audio_play_sound()或者 audio_play_sound_at()功能）或声音资产，在这种情况下，给定声音的所有实例都会暂停。

句法：

audio_pause_sound(index);

争论描述
index要暂停的声音的索引。

返回值：

N/A

例子：

if keyboard_check_pressed(ord("P"))
   {
   global.Pause = !global.Pause;
   if global.Pause
      {
      audio_pause_sound(snd_Waterfall);
      }
   else
      {
      audio_resume_sound(snd_Waterfall);
      }
   }

上面的代码检查是否按下了键盘键“ P”，如果检测到按下，则会将全局变量“ Pause”设置为 true或false，然后暂停在变量“ snd_Waterfall”中索引的声音，或者从暂停状态恢复声音。
