音频_暂停_声音

通过这个功能，你可以暂停当前正在播放的任何声音。这个声音可以是一个单一的声音实例（正在播放的单个声音的索引可以存储在一个变量中，当使用 audio_play_sound()或 audio_play_sound_at()函数时，单个被播放的声音的索引可以存储在一个变量中）或一个声音资产，在这种情况下，所有给定的声音实例将被暂停。

语法。

audio_pause_sound(index)。

争论 描述
指数 要暂停的声音的索引。

返回。

不适用

例子。

if keyboard_check_pressed(ord("P"))
{
global.Pause = !global.Pause;
if global.Pause
{
audio_pause_sound(snd_Waterfall)。
}
else
{
audio_resume_sound(snd_Waterfall)。
}
}

上面的代码检查是否有键盘键 "P "被按下，如果检测到，它将全局变量 "Pause "设置为真或假，然后要么暂停变量 "snd_Waterfall "中的声音索引。 暂停，或者从暂停状态恢复声音。
