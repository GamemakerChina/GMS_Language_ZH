audio_resume_all

使用此功能，您可以恢复以前暂停的所有声音。

句法：

audio_resume_all();

返回值：

N/A

例子：

if keyboard_check_pressed(ord("P"))
   {
   global.Pause = !global.Pause;
   if global.Pause
      {
      audio_pause_all();
      }
   else
      {
      audio_resume_all();
      }
   }

上面的代码检查是否按下了键盘键“ P”，如果检测到按下，则会将全局变量“ Pause”设置为 true或false，然后暂停所有声音或重新启动所有以前暂停的声音。
