audio_set_master_gain

使用此功能，您可以为特定的聆听者设置所有声音和音乐的整体音量的绝对值。默认的侦听器索引为0，但是您可以使用该函数 audio_get_listener_info()获得适用于目标平台的不同索引。增益值基于从0（静音）到1（最大音量）的线性比例，并且将影响游戏中通过该听众播放的所有声音和音乐的相对音量。

句法：

audio_set_master_gain(listenerIndex, gain);

争论描述
listenerIndex用于设置增益的监听器的索引。
gain全局卷的值（0到1）。

返回值：

N/A

例子：

var num = audio_get_listener_count();
for( var i = 0; i < num; i++;)
   {
   var info = audio_get_listener_info(i);
   audio_set_master_gain(info[? "index"], 0.75);
   ds_map_destroy(info);
   }

上面的代码遍历可用的侦听器，然后将其主增益设置为0.75。
