audio_sound_gain

使用此功能，您可以在给定的时间长度内淡入或淡出声音，或者可以用来立即设置声音增益。时间以毫秒为单位，并且该功能要求您输入最终的增益级别，以使声音在该时间结束时达到。该增益可以在0（无声）和大于0的任何值之间，尽管通常您会将最大音量视为1。可以使用大于1的任何值，但是根据所用声音和所编译的平台的不同，您可以播放声音时失真或削波。请注意，增益比例是线性的，要立即更改增益，只需将time参数设置为0。

如果索引是声音资源，则此功能将影响房间中当前正在播放的声音的所有实例，并且最终音量将是将播放所有其他声音实例的音量。但是，如果您使用了从类似函数返回的索引 audio_play_sound()它只会影响声音的一个实例。

句法：

audio_sound_gain(index, volume, time);

争论描述
index要为其设置增益的声音的索引。
volume音乐音量的值。
time增益变化的长度，以毫秒为单位。

返回值：

N/A

例子：

if val
   {
   var snd = audio_play_sound(snd_fountain);
   audio_sound_gain(snd, 0, 0);
   audio_sound_gain(snd, 1, 5000);
   }

上面的代码检查变量，如果返回 true然后它将将要播放的声音的索引分配给局部变量“ snd”。然后使用此变量将特定声音的音量减小到0，并在5秒钟内逐渐减小到最大音量。
