audio_sound_get_gain

此函数将返回给定声音的当前增益值。声音可以是从正在播放的单个声音的索引中引用的一种，使用该声音时已存储在变量中。audio_play_sound()或者 audio_play_sound_at()功能或资产浏览器中的实际声音资产。增益通常计算为0到1之间的值，但是在某些平台上，您可以将增益大于1，尽管将值1视为“最大音量”，任何更大的值都会引入音频剪辑。

句法：

audio_sound_get_gain(index);

争论描述
index要获得增益的声音的索引。

返回值：

Real

例子：

if audio_sound_get_gain(snd_Music) != 1
   {
   audio_sound_gain(snd_Music, 1, 0);
   }

如果代码的增益不等于1，则上述代码将更改从索引为“ snd_Music”的声音播放的音频的增益。
