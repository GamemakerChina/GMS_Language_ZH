audio_sound_get_pitch

此功能可用于获取给定声音的音调。声音可以是从正在播放的单个声音的索引中引用的一种，使用该声音时已存储在变量中。audio_play_sound()或者 audio_play_sound_at()功能或资产浏览器中的实际声音资产。

句法：

audio_sound_get_pitch(index);

争论描述
index要获取音高的声音的索引。

返回值：

Real

例子：

if audio_sound_get_pitch(snd_Explode) != 1
   {
   audio_sound_pitch(snd_Explode, 1);
   }

如果代码的音高不等于1，则上述代码将更改从索引为“ snd_Explode”的声音播放的音频的音高。
