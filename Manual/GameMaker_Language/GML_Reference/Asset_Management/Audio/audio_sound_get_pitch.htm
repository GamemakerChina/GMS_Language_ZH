音频_sound_get_pitch

这个函数可以用来获取一个给定的声音的音高。这个声音可以是一个从正在播放的单个声音的索引中引用的声音，这个索引已经被储存在一个变量中，当使用 audio_play_sound()或 audio_play_sound_at()函数时，已经存储在一个变量中，或者是来自资产浏览器的一个实际的声音资产。

语法。

audio_sound_get_pitch(index);

争论 描述
指数 要得到声音的音高的索引。

返回。

真实的

例子。

if audio_sound_get_pitch(snd_Explode) != 1
{
audio_sound_pitch(snd_Explode, 1);
}

上面的代码将改变从索引为 "snd_Explode "的声音中播放的音频的音调，如果它的音调不等于1。
