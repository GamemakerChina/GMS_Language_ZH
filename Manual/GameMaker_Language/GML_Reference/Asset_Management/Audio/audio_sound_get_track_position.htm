音频_声音_get_track_position

这个函数将得到声音文件中声音的位置（以秒为单位），以便播放。声音只能是一个声音的单个实例（当使用 audio_play_sound()或 audio_play_sound_at()函数时，单个被播放的声音的索引可以存储在一个变量中）。)

语法。

audio_sound_get_track_position(index)。

争论 描述
指数 要获取声音的播放位置的索引。

返回。

真实的

例子。

if audio_sound_get_track_position(global.Music) != 0
{
audio_sound_set_track_position(global.Music, 0);
}

上面的代码检查一个轨道以获得它的起始位置，如果它不是0秒，它就把它设置为0秒。
