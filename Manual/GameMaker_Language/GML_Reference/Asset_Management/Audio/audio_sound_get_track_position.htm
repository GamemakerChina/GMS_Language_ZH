audio_sound_get_track_position

此功能将获取声音文件在声音文件中的位置（以秒为单位）。声音只能是声音的一个实例（使用时，正在播放的单个声音的索引可以存储在变量中）audio_play_sound()或者 audio_play_sound_at()功能）。

句法：

audio_sound_get_track_position(index);

争论描述
index获得播放位置的声音的索引。

返回值：

Real

例子：

if audio_sound_get_track_position(global.Music) != 0
   {
   audio_sound_set_track_position(global.Music, 0);
   }

上面的代码检查轨道以获取其开始位置，如果不是0秒，则将其设置为0秒。
