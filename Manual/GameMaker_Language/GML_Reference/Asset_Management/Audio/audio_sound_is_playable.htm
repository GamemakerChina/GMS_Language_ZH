audio_sound_is_playable

此功能可用于检查当前是否可以播放给定的声音索引。由于在HTML5目标平台上处理流式和非流式声音回放的方式不同，因此需要这样做，并且该方法将返回 true是否可以播放声音，以及 false如果不能。请注意，在HTML5以外的所有其他平台上，该函数将始终返回true。

句法：

audio_sound_is_playable(index);

争论描述
index要检查的声音的索引。

返回值：

Real

例子：

if audio_sound_get_pitch(snd_Explode) != 1
   {
   audio_sound_pitch(snd_Explode, 1);
   }

如果代码的音高不等于1，则上述代码将更改从索引为“ snd_Explode”的声音播放的音频的音高。
