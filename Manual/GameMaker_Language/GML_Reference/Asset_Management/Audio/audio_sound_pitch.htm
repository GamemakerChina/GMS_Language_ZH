audio_sound_pitch

此功能可用于更改给定声音的音高。声音可以是从正在播放的单个声音的索引中引用的一种，使用该声音时已存储在变量中。audio_play_sound()或者 audio_play_sound_at()功能或资产浏览器中的实际声音资产。如果它是正在播放的声音的索引，则仅该实例将被更改，但是，当使用资产浏览器中的声音资产时，该声音的所有其他正在播放的实例将被更改。

pitch参数是一个螺距乘数，因为输入值将当前螺距乘以该值，因此默认值1是螺距不变，而小于1的值将降低螺距，大于1的值将增大螺距。沥青。最好对此功能使用较小的增量，因为无论如何都不会听到小于0或大于5的任何值。值得注意的是，允许的总音高变化被钳制为（1/256）-256个八度，因此任何高于或低于此值的值都不会被记录。

注意：上面给出的限制值是GameMaker Studio 2尝试将范围限制到的值，但是并不能在所有目标平台上保证该值。 iOS在（1/256）-8，因此您可能需要在每个目标平台上进行实验，并使用不同版本的声音资源，如果需要更高或更低的八度音阶值，则将每个版本进行预移位。

句法：

audio_sound_pitch(index, pitch);

争论描述
index要更改的声音的索引。
pitch音高倍数（默认为1）。

返回值：

N/A

例子：

var s_engine = audio_play_sound(snd_CarEngine, 10, false);
switch (gear)
   {
   case 1: audio_sound_pitch(s_engine, 0.8); break;
   case 2: audio_sound_pitch(s_engine, 0.9); break;
   case 3: audio_sound_pitch(s_engine, 0.95); break;
   case 4: audio_sound_pitch(s_engine, 1); break;
   case 5: audio_sound_pitch(s_engine, 1.2); break;
   }

上面的代码将根据变量“ gear”的值更改从变量“ s_engine”中索引的声音播放的音频的音高。
