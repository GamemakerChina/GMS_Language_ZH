audio_sound_set_track_position

此功能将在声音文件中设置播放声音的位置（以秒为单位）。声音可以是声音的单个实例（使用 audio_play_sound()或者 audio_play_sound_at()功能）或健全资产。如果您使用的是唯一的声音ID，则位置将立即更改为给定的新时间，如果它是资产浏览器资产，则给定声音的所有其他播放将在新时间开始。如果给定的值大于给定的“索引”值的声音长度，则该函数将忽略任何值。

句法：

audio_sound_set_track_position(index, time);

争论描述
index要更改的声音的索引。
time起点设置的时间（以秒为单位）。

返回值：

N/A

例子：

var snd = audio_play_sound(snd_MainTrack, 0, false);
audio_sound_set_track_position(snd, 32);

上面的代码播放声音，然后使用返回的ID值将声音的开始位置设置为32秒。

背面：音频
