audio_stop_sound

如果正在播放给定声音，此功能将停止。声音可以是声音的单个实例（使用 audio_play_sound()或者 audio_play_sound_at()功能）或声音资产，在这种情况下，将停止给定声音的所有实例。

句法：

audio_stop_sound(index);

争论描述
index声音停止的索引。

返回值：

N/A

例子：

if !global.SFX
   {
   audio_stop_sound(snd_Waterfall);
   }
else
   {
   audio_play_sound_at(snd_Waterfall, x, y, 0, 100, 300, 1, true, 1);
   }

上面的代码检查全局变量“ SFX”，并返回 false，它将停止当前正在播放的变量“ snd_Waterfall”中索引的声音，如果返回 true，它将循环播放声音。
