audio_system_is_available

此功能可用于检查并查看音频系统是否已初始化，或者音频上下文是否正在运行。在所有平台上，初始化音频引擎后，游戏开始后，此功能将立即返回true， HTML5 target除外。在HTML5上，音频上下文状态可以随时更改，具体取决于用户输入，所使用的浏览器及其版本，因此可以使用此功能检查并查看是否可以播放音频。如果函数返回 false（即：音频上下文状态未在运行），那么只有未流的声音可以播放（但不能保证，您应该假定无法播放任何音频），而如果返回 true然后所有音频都会播放。

句法：

audio_system_is_available();

返回值：

Boolean

例子：

if audio_system_is_available()
    {
    if audio_is_paused(global.Music)
        {
        audio_resume_sound(global.Music)
        }
    else
        {
        if !audio_is_playing(global.Music)
            {
            global.Music = audio_play_sound(snd_Music, 0, true);
            }
        }
    }
else
    {
    if audio_is_playing(global.Music)
        {
        audio_pause_sound(global.Music);
        }
    }

上面的代码将暂停/取消暂停音轨，具体取决于音频系统是否已初始化且可用。
