font_add_enable_aa

这个函数可以用来启用或禁用用.NET技术添加的字体的抗锯齿（AA）。 font_add().这个函数需要在你添加任何字体之前被调用，将其设置为 设置为true将启用AA，设置为false将禁用AA。默认情况下，AA是启用的。请注意，这对在调用该函数之前已经添加的字体没有影响。 调用前已经添加的字体没有影响，而且该函数只需要在添加字体时调用一次，而不是在使用字体的每一次绘制/步骤中调用。

语法。

font_add_enable_aa（启用）。

争论 描述
启用 是否启用（true）或禁用（false）添加字体的AA。

返回。

不适用

例子。

if !font_add_get_enable_aa()
{
font_add_enable_aa(true)。
}

上面的代码检查添加字体的抗锯齿状态，如果没有启用，我们就启用它。

返回。字体
