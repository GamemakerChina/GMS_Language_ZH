font_add_enable_aa

此功能可用于启用或禁用使用以下命令添加的字体的抗锯齿（AA）font_add() 。在添加任何字体并将其设置为之前，需要先调用此函数。true将启用AA，并将其设置为 false将禁用它。默认情况下，AA是启用的。请注意，这将对调用该函数之前添加的字体没有影响，并且仅在添加字体时才调用该函数一次，而不是使用该字体的每个绘制/步骤。

句法：

font_add_enable_aa(enable);

争论描述
enable是否启用（true ）或禁用（false ）AA，用于添加字体。

返回值：

N/A

例子：

if !font_add_get_enable_aa()
    {
    font_add_enable_aa(true);
    }

上面的代码检查添加字体的抗锯齿状态，如果未启用，则启用它。
