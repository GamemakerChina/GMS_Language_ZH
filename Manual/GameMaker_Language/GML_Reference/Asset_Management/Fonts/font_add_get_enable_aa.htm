font_add_get_enable_aa

该函数可用于检查使用.NET技术添加的字体是否启用了抗锯齿（AA）。 font_add().如果启用了AA，该函数将返回true，如果没有，则返回false。 则返回true。请注意，AA在默认情况下是启用的，但你可以使用函数改变添加字体的AA状态。 font_add_enable_aa()来改变添加的字体的AA状态，只要在添加字体之前调用该函数 字体之前调用该函数。

语法。

font_add_get_enable_aa()。

返回。

布尔型

例子。

if !font_add_get_enable_aa()
{
font_add_enable_aa(true)。
}

上面的代码检查添加字体的抗锯齿状态，如果没有启用，我们就启用它。

返回。字体
