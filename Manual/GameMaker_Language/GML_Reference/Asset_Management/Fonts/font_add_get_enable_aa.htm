font_add_get_enable_aa

此功能可用于检查是否对使用以下命令添加的字体启用了抗锯齿（AA）font_add() 。该函数将返回 true如果启用了AA，并且 false如果不是这样。请注意，默认情况下会启用AA，但是您可以使用函数更改添加字体的AA状态 font_add_enable_aa()，只要在添加字体之前调用它即可。

句法：

font_add_get_enable_aa();

返回值：

Boolean

例子：

if !font_add_get_enable_aa()
    {
    font_add_enable_aa(true);
    }

上面的代码检查添加字体的抗锯齿状态，如果未启用，则启用它。
