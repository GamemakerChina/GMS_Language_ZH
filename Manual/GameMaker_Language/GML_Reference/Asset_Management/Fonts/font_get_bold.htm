font_get_bold

使用此功能，您可以检查任何字体资源以查看其是否具有粗体标志。如果这样做，函数将返回 true，否则它将返回 false。

句法：

font_get_bold(ind);

争论描述
ind要检查的字体索引。

返回值：

Boolean

例子：

if font_get_bold(fnt_Main)
   {
   draw_set_font(fnt_Main);
   }

这会将活动图形字体设置为 fnt_Main如果在其字体属性中将其设置为粗体。
