font_get_fontname

使用此功能，您可以获得所选字体资产的实际系统名称。此函数返回一个字符串而不是一个索引，返回的名称将取决于所使用的字体。 GameMaker Studio 2资产浏览器中拥有一个名为“ fnt_Main”的字体资产，并且该字体本身可能是Windows系统字体Arial 。在这种情况下，该函数将返回“ Arial”，因为这是字体的系统名称。如果您需要资产浏览器中显示的名称，则应使用 font_get_name()。

句法：

font_get_fontname(ind);

争论描述
ind要检查的字体索引。

返回值：

String

例子：

fnt_Name = font_get_fontname(font0);

上面的代码将获取字体资源的系统名称，并将其作为字符串存储在变量“ fnt_Name”中。
