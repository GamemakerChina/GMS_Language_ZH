font_get_italic

使用此功能，您可以检查任何字体资产以查看其是否带有斜体标记。如果这样做，函数将返回 true，否则它将返回 false。

句法：

font_get_italic(ind);

争论描述
ind要检查的字体索引。

返回值：

Boolean

例子：

if font_get_italic(fnt_Main)
   {
   draw_set_font(fnt_Main);
   }

这会将活动图形字体设置为 fnt_Main如果在其字体属性中将其设置为斜体。
