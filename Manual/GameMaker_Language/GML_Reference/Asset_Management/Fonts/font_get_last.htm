font_get_last

GameMaker Studio 2定义字体时，可以定义要包括的字符范围。这是因为字体（实际上出于法律原因）实际上并未包含在您的游戏中，而是在纹理页面中包含了字体的图像，这就是您的游戏将使用的图像（就像其他任何图形资源一样）。这意味着您将需要使使用的字符数保持最少，并仅指定游戏所需的字符范围，以使纹理内存尽可能最佳。此功能可用于查找将字体资源添加到游戏中时使用的最后一个字符（作为UTF8值）。

句法：

font_get_last(ind);

争论描述
ind要检查的字体索引。

返回值：

Real

例子：

num = font_get_last(fnt_Main);

上面的代码将存储“ fnt_Main”中索引的字体的字体范围的最后一个字母的ASCII值。
