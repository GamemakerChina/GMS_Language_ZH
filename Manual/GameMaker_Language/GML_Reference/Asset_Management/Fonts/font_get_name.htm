font_get_name

这个函数可以用来获取字体的名称（作为一个字符串），当它作为资产被添加到GameMaker Studio 2资产浏览器时，它被赋予了这个名称。请注意，这只是一个 字符串，不能用来直接引用字体--为此，你需要字体索引。然而，你可以使用这个字符串来获得字体索引，使用返回的字符串和 函数 asset_get_index().

语法。

font_get_name(ind)。

争论 描述
ind 要检查的字体的索引。

返回。

字符串

例子。

fnt_Name = font_get_name(font0)。

上面的代码将获得一个字体资源的名称，因为它出现在资产浏览器中，并将它作为一个字符串存储在变量 "fnt_Name "中。

返回。字体
