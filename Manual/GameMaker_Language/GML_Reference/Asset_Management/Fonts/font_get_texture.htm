font_get_texture

该函数为字体纹理页面返回一个特殊的指针。然后，可以在其他绘图函数中使用此值，特别是在使用图元以及Shader函数时，在一般绘图中可以使用此值。您可以使用其他方法获取有关返回的纹理页面的更多信息。texture_在这里找到功能。

句法：

font_get_texture(font);

争论描述
font要使用的字体的索引。

返回值：

Pointer (texture page ID)

例子：

tex = font_get_texture(fnt_Main);

上面的代码将获取索引为“ fnt_Main”的字体的纹理指针。
