font_get_uvs

该函数返回一个阵列与用于纹理页面上的字体纹理UV坐标，使用以下值阵列中的填充：

  • [0] =左
  • [1] =顶部
  • [2] =正确
  • [3] =底部

然后，可以在其他绘图函数中使用此值，特别是在使用图元以及Shader函数时，在一般绘图中可以使用此值。

句法：

font_get_uvs(font);

争论描述
font要使用的字体的索引。

返回值：

Array

例子：

var tex = font_get_uvs(fnt_Main);
tex_left = tex[0];
tex_top = tex[1];
tex_right = tex[2];
tex_left = tex[3];

上面的代码将给定背景的UV坐标存储在本地数组中，然后将值分配给实例变量。
