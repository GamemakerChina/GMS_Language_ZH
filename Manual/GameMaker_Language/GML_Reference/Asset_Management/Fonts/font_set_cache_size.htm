font_set_cache_size

在执行图像混合和着色时，HTML5无法以可执行文件执行的方式动态地执行此操作。因此， GameMaker Studio 2必须临时保存图像的混合副本，并在需要时将其加载。此函数设置在覆盖旧字体之前可以缓存给定字体的混合副本的数量。预设值为4。

注意：对于精灵字体，您应该对精灵使用等效功能，sprite_set_cache_size() 。

句法：

font_set_cache_size(ind, max);

争论描述
ind字体的索引以更改其缓存大小。
max可以存储的字体的最大缓存副本数。

返回值：

N/A

例子：

font_set_cache_size(fnt_MainMenu, 2);

这会将设置在变量“ fnt_MainMenu”中的字体的字体缓存设置为2个副本。
