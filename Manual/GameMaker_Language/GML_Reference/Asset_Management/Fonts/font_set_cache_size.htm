字体_set_cache_size

在进行图像混合和着色时，HTML5 无法以可执行文件的方式动态地进行。因此，GameMaker Studio 2必须临时保存图像的混合副本，并在需要时加载 他们在需要时载入。这个函数设置了在旧的字体被覆盖之前可以缓存多少个混合的副本。默认值是4。

注意： 对于精灵字体，你应该使用精灵的对应函数。 sprite_set_cache_size().

语法。

font_set_cache_size(ind, max)。

争论 描述
ind 要改变缓存大小的字体的索引。
最大 可以存储的字体的最大缓存拷贝数。

返回。

不适用

例子。

font_set_cache_size(fnt_MainMenu, 2)。

这将把变量 "fnt_MainMenu "中索引的字体的字体缓存设置为2份。

返回。字体
