font_texture_page_size

使用此内置变量时，可用于获取或设置纹理页面大小 font_add()。使用该功能添加字体后， GameMaker Studio 2将创建所需字形的纹理页面缓存，大小直至此变量定义的大小。过程如下：

  • 您使用的每个字符都会从字体缓存到纹理页面中
  • 当纹理页面已满时（因为您使用大字符或许多其他字符）， GameMaker Studio 2将从缓存中删除字符，并将其替换为所需的新字符
  • GameMaker Studio 2试图不要使纹理太大，因为不是所有设备都可以处理大纹理，并且它试图不要使纹理太小，因为删除和重新添加字符到纹理会占用CPU时间并导致游戏运行慢慢地

如果您认为需要的纹理页面尺寸大于或小于GameMaker Studio 2使用的默认1024px，则可以使用此内置变量设置值。如果您不使用 font_add()功能，然后设置此设置将不会影响您的游戏性能，因为IDE中添加的字体会缓存在常规纹理页面上（您可以从目标平台的“字体编辑器”和“游戏选项”中进行控制）。

句法：

font_texture_page_size;

返回值：

Real (maximum width/height in pixels)

例子：

font_texture_page_size = 512;
newfont = font_add("Arial", 24, true, true, 32, 128);

上面的代码将字体缓存纹理页面设置为最大大小为512px，然后向游戏添加新字体。
