实例_激活_层

用这个函数你可以激活一个之前被停用的层。你需要提供图层ID，它可以是代码编辑器中写的图层名称（作为一个字符串），也可以是实际的图层ID值，如图所示 所返回的实际层ID值。 layer_create()而该层上所有停用的实例将再次被激活。请注意，如果你停用了一个有持久性实例的层，你将需要再次激活该层。 需要在改变房间之前用这个函数再次激活该层，否则任何持久化的实例将不会被带入并被丢弃。还需要注意的是，激活并不是瞬间的，一个已经被激活的实例 的实例在调用该函数的事件结束前不会被认为是激活的。

语法。

instance_activate_layer(layer_id)。

争论 描述
layer_id 要使用的层ID字符串（或值）。

返回。

不适用

例子。

instance_activate_all();
var _vx = camera_get_view_x(view_camera[0]);
var _vy = camera_get_view_y(view_camera[0]);
instance_deactivate_region(view_xview[0] - 64, view_yview - 64, _vx + 128, _vy + 128, false, false);
instance_activate_layer("Player_Layer") 。

上面的代码激活了房间内的所有实例，然后停用了那些在当前摄像机视角限制之外的实例，但 "Player_Layer "层上的实例除外，它们在最后又被重新激活了。
