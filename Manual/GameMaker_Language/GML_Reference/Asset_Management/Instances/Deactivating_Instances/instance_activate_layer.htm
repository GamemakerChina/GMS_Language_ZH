instance_activate_layer

使用此功能，您可以激活先前已停用的图层。您需要提供图层ID ，该ID可以是代码编辑器中编写的图层名称（作为字符串），也可以是由图层返回的实际图层ID值。layer_create()并且该层上所有停用的实例将再次激活。请注意，如果您已停用具有持久性实例的图层，则需要在更改房间之前使用此功能再次激活该层，否则任何持久性实例将不会被保留并被丢弃。还要注意，激活不是瞬时的，以这种方式激活的实例在调用函数的事件结束之前不会被认为是激活的。

句法：

instance_activate_layer(layer_id);

争论描述
layer_id要使用的图层ID字符串（或值）

返回值：

N/A

例子：

instance_activate_all();
var _vx = camera_get_view_x(view_camera[0]);
var _vy = camera_get_view_y(view_camera[0]);
instance_deactivate_region(view_xview[0] - 64, view_yview - 64, _vx + 128, _vy + 128, false, false);
instance_activate_layer("Player_Layer");

上面的代码激活了房间内的所有实例，然后停用了当前摄像机视图范围之外的实例，但“ Player_Layer”层上的实例将在最后再次被激活。

返回：停用实例
