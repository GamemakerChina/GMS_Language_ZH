实例_激活_区域

有了这个功能，你可以在房间里定义一个区域来激活以前被停用的实例。这个区域可以被标记为 "内部 "或 "外部"，如下图所示。

你可以在上面的图片中看到，"苹果 "实例总是处于活动状态 因为，即使精灵本身没有与该区域重叠，但其边界框却与之重叠。所以，当实例的边界框与区域重叠时，它们被认为是在指定的区域内，而碰撞掩码的状态（精确的 或不）不被考虑。请注意，激活并不是瞬间完成的，以这种方式激活的实例在调用该函数的事件结束前不会被认为是激活的。

语法。

instance_activate_region(left, top, width, height, inside)。

争论 描述
左边 要激活的矩形区域左边的X坐标。
顶部 要激活的矩形区域顶部的y坐标。
宽度 要激活的区域的宽度。
高度 要激活的区域的高度。
内幕 是否激活区域内部（true）或外部（false）的实例。

返回。

不适用

例子。

instance_deactivate_all(true);
var _vx = camera_get_view_x(view_camera[0]);
var _vy = camera_get_view_y(view_camera[0]);
var _vw = camera_get_view_width(view_camera[0]);
var _vh = camera_get_view_height(view_camera[0]);
instance_activate_region(_vx - 64, _vy - 64, _vw + 128, _vh + 128, false）。)

上面的代码停用了所有的实例，除了正在运行代码的那个，然后激活了房间内的一个区域。
