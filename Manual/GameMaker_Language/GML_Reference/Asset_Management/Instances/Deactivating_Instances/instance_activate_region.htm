instance_activate_region

使用此功能，您可以在房间内定义一个区域来激活先前已停用的实例。可以将这个区域标记为“内部”或“外部”，如下图所示：

您可以在上图中看到“ apple”实例始终处于活动状态，因为即使Sprite自身不与区域重叠，边界框也会激活。因此，当实例的边界框与实例重叠时，它们被视为在指定的区域内，并且不考虑碰撞蒙版的状态（精确与否）。请注意，激活不是瞬时的，并且以这种方式激活的实例在调用函数的事件结束之前不会被视为处于激活状态。

句法：

instance_activate_region(left, top, width, height, inside);

争论描述
left矩形区域左侧的x坐标激活。
top要激活的矩形区域顶部的y坐标。
width要激活的区域的宽度。
height要激活的区域的高度。
inside是激活区域内部（true）还是外部（false）的实例。

返回值：

N/A

例子：

instance_deactivate_all(true);
var _vx = camera_get_view_x(view_camera[0]);
var _vy = camera_get_view_y(view_camera[0]);
var _vw = camera_get_view_width(view_camera[0]);
var _vh = camera_get_view_height(view_camera[0]);
instance_activate_region(_vx - 64, _vy - 64, _vw + 128, _vh + 128, false);

上面的代码将停用除正在运行该代码的实例之外的所有实例，然后激活房间中的一个区域。

返回：停用实例
