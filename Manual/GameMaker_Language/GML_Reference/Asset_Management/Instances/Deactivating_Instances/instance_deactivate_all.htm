实例_停用_所有

这个函数将停用所有在代码运行时在房间里的实例。如果 "notme "标志被设置为false，这可能包括运行代码的实例，但通常你会希望该实例 是活动的，在这种情况下，"notme "标志应该被设置为真。请注意，停用并不是瞬间完成的，以这种方式停用的实例在调用该函数的事件结束之前不会被认为是不活动的。 中调用该函数，才会被认为是不活动的。

注意：如果你在房间启动时停用一个实例（即：从房间创建代码，或从房间内一个实例的创建事件），所有从房间编辑器放置在房间内的实例在被停用之前仍然会运行它们的创建事件。

警告：停用启用了物理的实例不会停止它们的灯具在物理模拟中的互动。为此，你应该设置它们的 phy_active变量 为真或假，因为你激活/停用了这些实例。

语法。

instance_deactivate_all(notme)。

争论 描述
不是我 是否保持调用实例的激活（true）或不激活（false）。

返回。

不适用

例子。

instance_deactivate_all(true);
var _vx = camera_get_view_x(view_camera[0]);
var _vy = camera_get_view_y(view_camera[0]);
var _vw = camera_get_view_width(view_camera[0]);
var _vh = camera_get_view_height(view_camera[0]);
instance_activate_region(_vx - 64, _vy - 64, _vw + 128, _vh + 128, false）。)

上面的代码停用了所有的实例，除了正在运行代码的那个，然后激活了房间内的一个区域。
