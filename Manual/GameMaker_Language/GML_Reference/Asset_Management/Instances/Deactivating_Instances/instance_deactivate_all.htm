instance_deactivate_all

该功能将在代码运行时停用房间中的所有实例。如果将“ notme”标志设置为，则可能包括运行代码的实例 false，但通常您希望该实例处于活动状态，在这种情况下，应将“ notme”标志设置为 true。请注意，停用不是瞬时的，以这种方式停用的实例在调用函数的事件结束之前不会被认为是非活动的。

注意：如果您在房间启动时停用了一个实例（即：从房间创建代码，或从房间中实例的实例创建事件），则从房间编辑器放置在房间中的所有实例仍将运行其创建事件在停用之前。

警告：停用启用了物理的实例将不会阻止其夹具在物理模拟中进行交互。为此，您应该将其phy_active变量设置为 true或者 false当您激活/停用实例时。

句法：

instance_deactivate_all(notme);

争论描述
notme是否使调用实例保持激活状态（真）或不激活（假）。

返回值：

N/A

例子：

instance_deactivate_all(true);
var _vx = camera_get_view_x(view_camera[0]);
var _vy = camera_get_view_y(view_camera[0]);
var _vw = camera_get_view_width(view_camera[0]);
var _vh = camera_get_view_height(view_camera[0]);
instance_activate_region(_vx - 64, _vy - 64, _vw + 128, _vh + 128, false);

上面的代码将停用除正在运行该代码的实例之外的所有实例，然后激活房间中的一个区域。

返回：停用实例
