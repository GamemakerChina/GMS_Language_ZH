instance_deactivate_layer

使用此功能，您可以停用分配给特定图层的所有实例。您需要提供图层ID ，该ID可以是代码编辑器中编写的图层名称（作为字符串），也可以是由图层返回的实际图层ID值。layer_create() ，请注意，您只能使用此功能停用实例图层。请注意，如果您已停用包含标记为Persistent的对象实例的图层，则需要使用函数再次重新激活该图层 instance_activate_layer()更改房间之前，否则该层上的任何持久实例将不会被保留并被丢弃。还要注意，停用不是瞬时的，以这种方式停用的实例在调用函数的事件结束之前不会被认为是非活动的。

注意：如果您在房间启动时停用了一个实例（即：从房间创建代码，或从房间中实例的实例创建事件），则从房间编辑器放置在房间中的所有实例仍将运行其创建事件在停用之前。

警告：停用启用了物理的实例将不会阻止其夹具在物理模拟中进行交互。为此，您应该将其phy_active变量设置为 true或者 false当您激活/停用实例时。

句法：

instance_deactivate_layer(obj);

争论描述
layer要使用的图层ID字符串（或值）

返回值：

N/A

例子：

instance_deactivate_layer("Enemy Layer");
var _vx = camera_get_view_x(view_camera[0]);
var _vy = camera_get_view_y(view_camera[0]);
var _vw = camera_get_view_width(view_camera[0]);
var _vh = camera_get_view_height(view_camera[0]);
instance_activate_region(_vx - 64, _vy - 64, _vw + 128, _vh + 128, false);

上面的代码停用了分配给“ Enemy_Layer”层的所有实例，然后激活了房间内的一个区域。

返回：停用实例
