实例变量

当你创建一个新对象时，它将带有某些已经初始化为默认值的变量。这些变量我们称之为实例变量，因为它们将为你在游戏室中放置的每个对象实例创建，而且 一旦一个实例被创建，这些变量的值对该实例来说是唯一的，而且只有该实例。这些变量中的一些你会经常使用，比如x和y，而其他的则需要较少的时间，这取决于你想做什么 但总的来说，它们是非常有用的，在可能的情况下，你应该寻求使用它们，而不是创建你自己的自定义变量。

大多数实例变量可以被设置，也可以被读取，允许你仅仅通过调整某个变量的值来改变一个实例的属性和行为--例如，你可以阻止一个实例的绘制，仅仅通过设置 visible内置变量为false，就可以阻止一个实例的绘制。下面你可以找到不同的变量，它们被初始化为你游戏中所有对象的所有实例。

重要的是!如果一个实例是在一个序列中，那么这些变量中的一些--如image_xscale /image_yscale / image_angle /x /y - 将会被覆盖，当序列 开始播放后更新每一步时，这些变量将被覆盖。
