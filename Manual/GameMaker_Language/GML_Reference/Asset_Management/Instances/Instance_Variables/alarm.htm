alarm

此一维数组用于获取实例可能具有的任何警报的当前值，也可以用于设置这些警报。每个对象实例中内置了十二个警报，每个警报都有自己的警报event该变量达到0时将运行。

警报在每个步骤开始倒计数，之后开始步骤事件，但主要步骤事件前，任何事情之前呈现的一步。应该注意的是，警报到达0时（尽管已触发事件）并没有结束，因为下一步它会降为-1，因此，如果出于任何原因需要停止警报，则应设置其数组值为-1而不是0。警报时间以游戏步长计算，值30为30步，60为60步，依此类推...

注意：没有任何动作或代码的警报将不会运行。但是，即使只有注释而没有代码，警报也会倒计时。

您可以直接设置警报数组（也可以直接使用该数组直接检索当前警报值），但是在某些情况下，这不是最合适的方法，因此您也可以使用该功能 alarm_set()设置闹钟和功能 alarm_get()获取警报的值。

句法：

alarm[val];

返回值：

Integer (-1 if the alarm isn't running)

例子：

if canshoot
    {
    if keyboard_check_pressed(vk_space)
        {
        canshoot = false;
        alarm[0] = room_speed;
        instance_create_layer(x, y, "Bullets", obj_Bullet);
        }
    }

上面的代码先检查一个变量，然后检查它是否为真，然后检查按键是否正确，最后将变量设置为false，设置警报（将其再次设置为true）并创建实例。
