direction

GameMaker Studio 2所有实例都具有某些“内置”属性，您可以使用这些属性并将其设置为控制它们的外观和行为。Direction是这些属性之一，当实例具有一个 speed请注意， GameMaker Studio 2中的方向通常计算为：0°是右，90°是上，180°是左，270°是下，gravity和gravity_direction变量可以修改 direction在游戏中使用它们时的价值。

句法：

direction;

返回值：

Real (single precision floating point value)

例子：

if keyboard_check(vk_left) direction += 5;
if keyboard_check(vk_right) direction -= 5;

上面的代码将基于按下哪个键（左或右）来更改实例的移动方向。

返回：实例变量
