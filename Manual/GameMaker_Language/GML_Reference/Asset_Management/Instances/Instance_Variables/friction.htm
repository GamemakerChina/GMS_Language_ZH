friction

Friction是实例可具有的“内置”属性之一，可用于在一段时间后降低实例速度 speed不为零。它的工作原理很简单，只需从每一步的速度中减去一个量，直到对象的速度为0，那么如果将摩擦设置为例如0.1，并且实例的速度为1（每步1像素），它将减速并在经过10步后停止。同样要注意的是，摩擦力在正向和负向速度上均等地施加，最终结果始终是对象在给定时间后的速度为0。

句法：

friction;

返回值：

Real (single precision floating point value)

例子：

if abs(speed) > 0
    {
    friction = 0.05;
    }
else
    {
    friction = 0;
    }

上面的代码仅在实例的绝对速度大于0时才适用摩擦。

返回：实例变量
