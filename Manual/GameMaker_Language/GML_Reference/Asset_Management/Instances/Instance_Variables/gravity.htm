gravity

gravity是所有实例具有的“内置”变量之一，设置后将在 gravity_direction的实例，同时影响实例 speed 和 direction。请注意，重力是累积力，如果您选择不限制最终速度，则重力会加速对象，通常会将此变量设置为较小的十进制值（例如0.01）。如果您设定 gravity设置为0，则不会对实例施加重力（这是默认值）。

句法：

gravity;

返回值：

Real (single precision floating point value)

例子：

if !place_meeting(x, y + 1, obj_Ground)
    {
    gravity = 0.01;
    }
else
    {
    gravity = 0;
    }

上面的代码仅在实例未在其下方找到“ obj_Ground”的任何实例的情况下才适用重力。

返回：实例变量
