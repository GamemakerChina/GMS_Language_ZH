gravity_direction

gravity_direction是所有实例具有的“内置”属性之一，可用于设置实例的移动方向 gravity大于0。请注意， GameMaker Studio 2中的方向通常计算为0°右，90°上，180°左和270°下。

句法：

gravity_direction;

返回值：

Real (single precision floating point value)

例子：

if place_meeting(x, y, obj_switch)
    {
    gravity_direction += 180;
    }

如果玩家对象遇到“ obj_switch”的实例，则上面的代码将更改gravity_direction。

返回：实例变量
