重力方向

gravity_direction是所有实例都有的 "内置 "属性之一，可以用来设置运动的方向，当实例的值大于0时。 重力注意，GameMaker Studio 2中的方向通常计算为0°为右，90°为上，180°为左，270°为下。

语法。

重力方向。

返回。

实数（单精度浮点值

例子。

if place_meeting(x, y, obj_switch)
{
gravity_direction += 180;
}

如果玩家对象遇到 "obj_switch "的一个实例，上述代码将改变重力方向。
