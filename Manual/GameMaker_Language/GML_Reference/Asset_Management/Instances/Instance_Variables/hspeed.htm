hspeed

hspeed是所有实例具有的“内置”属性之一，并定义了实例的水平移动速度（沿x轴）（以像素为单位）。所以，hspeed 3表示每步向右（+ x）移动3个像素，hspeed -3表示每步向左（-x）移动3个像素。

请注意，如果您设置了 speed和/或 direction变量然后 hspeed值将自动更新以反映这些更改，同样，更改 hspeed值也会影响 speed和 direction价值观。另请注意gravity，gravity_direction和friction在游戏中使用它们时，都可以修改此变量的值。

句法：

hspeed;

返回值：

Real (single precision floating point value)

例子：

if keyboard_check(vk_left) hspeed = -5;
if keyboard_check(vk_right) hspeed = 5;

上面的代码将根据按下的键来更改水平速度。

返回：实例变量
