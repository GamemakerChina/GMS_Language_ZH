身份证

这个只读的变量保存着实例的唯一识别号码。你所创建的每一个实例--无论是通过代码还是通过在房间编辑器中把它们添加到房间中--都会被赋予一个编号，这个编号在内部用于识别这个实例 而变量 id 是你可以用来引用它的。当一个实例被创建时， id 也会被返回（并且可以被存储在一个变量中）。 instance_create_layer()或 instance_create_depth() ，以及其他实例函数来创建实例时，也会返回（并可以存储在变量中）。

注意：这个变量的值与实例在房间编辑器中被赋予的标识符不一样，也与 instance_id数组 不同，后者包含了当前所有活动实例的所有ID。

语法。

id;

返回。

实数（整数--实例ID值）。

例子。

for(var i = 0; i < instance_count; i++;)
{
if instance_id[i] != id
{
instance_id[i].scr += 5;
}
}

上面的代码为房间里的每个活动实例的 "scr "变量增加了5，除了运行代码的那个。它是通过循环浏览所有的活动实例来实现的（使用 "instance_id "数组来返回每个活动实例的 ID），并将它们与内置的 "id "变量进行比较，后者是运行代码的原始实例的ID。
