id

此只读变量保存实例的唯一标识号。您创建的每个实例（无论是通过代码还是通过将其添加到“房间编辑器”中的一个房间）都被赋予一个数字，该数字在内部用于标识该实例和变量 id是您可以用来引用它的内容。这 id使用创建实例时，也会返回（并且可以存储在变量中）instance_create_layer()或者 instance_create_depth()，以及其他实例功能。

注：此变量的值是不相同的是，实例在房间编辑给定的，也是不同的标识符INSTANCE_ID包含所有当前活动的实例的所有ids数组。

句法：

id;

返回值：

Real (integer - instance ID value)

例子：

for(var i = 0; i < instance_count; i++;)
    {
    if instance_id[i] != id
        {
        instance_id[i].scr += 5;
        }
    }

上面的代码将房间中每个活动实例的“ scr”变量加5，但运行该实例的实例除外。它通过遍历所有活动实例（使用“ instance_id”数组返回每个活动实例ID）并将它们与内置的“ id”变量（即运行代码的原始实例的ID）进行比较来实现此目的。

返回：实例变量
