solid

可以通过对象编辑器中的对象属性或通过更改此内置变量的值来将实例标记为实体。如果 solid被设定为 true然后，当检测到碰撞时，碰撞实例将自动返回到碰撞前一步中的位置（然后在碰撞事件中运行任何代码或动作）。如果设置为 false，所有定位都必须通过碰撞事件来处理。

句法：

solid;

返回值：

Boolean

例子：

if other.solid = true
    {
    x = xprevious;
    y = yprevious;
    }

上面的代码将检查冲突中“其他”实例的实体标志，以及是否将其设置为 true将该实例移回其先前位置。

返回：实例变量
