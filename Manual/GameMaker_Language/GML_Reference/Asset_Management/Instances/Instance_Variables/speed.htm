speed

GameMaker Studio 2所有实例都具有某些“内置”属性，您可以使用这些属性并将其设置为控制它们的外观和行为。speed是这些属性之一，并定义实例每步将移动多少像素。不像 hspeed和 vspeed，speed没有与之相关的方向，因为这是由 direction实例的值，但可以为负值，在这种情况下，实例的移动方向将与实例设置的方向相反 direction功能（即：direction -180°）。请注意， speed和/或 direction，还会修改 hspeed和 vspeed变量，以及gravity，gravity_direction和friction在游戏中使用它们时，都可以修改此变量的值。

句法：

speed;

返回值：

Real (single precision floating point value)

例子：

if keyboard_check(vk_up) speed = 2;
if keyboard_check(vk_left) direction += 5;
if keyboard_check(vk_right) direction -= 5;

上面的代码将使用箭头键设置实例的速度和方向。

返回：实例变量
