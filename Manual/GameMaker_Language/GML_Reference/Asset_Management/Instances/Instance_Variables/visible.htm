可见

通过将这个变量设置为true（可见）或false（不可见），可以将一个实例标记为可见或不可见。它的作用是告诉GameMaker Studio 2跳过这个实例的绘制事件，所以使用时必须注意 因为这意味着在实例不可见时，放置在绘制事件中的代码将不会被运行。还要注意的是，如果该实例被分配到的图层被标记为不可见，将这个变量设置为 true将不会有任何影响，直到该层本身也被标记为可见。

语法。

可见。

返回。

布尔型

例子。

if other.visible = true
{
x = xprevious;
y = yprevious;
}

上述代码将在碰撞事件中检查 "另一个 "实例的可见标志，如果它被设置为真，则将该实例移回其先前的位置。
