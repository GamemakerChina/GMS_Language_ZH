visible

实例可以标记为 visible或通过将此变量设置为 true（可见）或 false（无形的）。这是通过告诉GameMaker Studio 2跳过此实例的draw事件而起作用的，因此在使用此实例时必须格外小心，因为这意味着在实例不可见时不会运行放置在draw事件中的代码。另请注意，如果layer实例分配给的标记为不可见，将此变量设置为 true在图层本身也被标记为可见之前，该选项将无效。

句法：

visible;

返回值：

Boolean

例子：

if other.visible = true
    {
    x = xprevious;
    y = yprevious;
    }

上面的代码将在碰撞事件中检查“其他”实例的可见标志，以及是否将其设置为 true将该实例移回其先前位置。

返回：实例变量
