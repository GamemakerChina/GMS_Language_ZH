vspeed

vspeed是所有实例具有的“内置”属性之一，并定义了实例的垂直移动速度（沿y轴）（以像素为单位）。所以，一个 vspeed3表示每步移动到底部（+ y）的3个像素，并且 vspeed-3表示每步移动到顶部（-y）3个像素。

请注意，如果您设置了 speed和/或 direction变量然后 vspeed值将自动更新以反映这些更改，同样，更改 vspeed值也会影响 speed和 direction价值观。另请注意gravity，gravity_direction和friction在游戏中使用它们时，都可以修改此变量的值。

句法：

vspeed;

返回值：

Real (single precision floating point value)

例子：

if keyboard_check(vk_up) vspeed = -5;
if keyboard_check(vk_down) vspeed = 5;

上面的代码将根据按下的键来更改垂直速度。

返回：实例变量
