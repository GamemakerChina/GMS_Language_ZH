x

实例的x值是当前房间中的水平位置，以像素为单位。该值可以是0，可以是正数或减号，其中0是房间的左侧，向右移动增加x，向左移动减小x（x的负值表示实例已超出房间左侧）。您甚至可以给xa实际值，例如12.345，但由于不能获得0.345像素，因此实例可能似乎没有移动，尽管这些值将由GameMaker Studio 2在内部维护和使用，并且是完全有效的。

通过设置x（和y）值，您可以使对象在房间中跳转到您选择的位置，或者您可以对它们进行少量加减以产生运动的错觉，而无需实际使用内置的 speed和 direction功能。在制作游戏时，应考虑到这一点，因为一种类型的动作通常比另一种类型更适合一种特定的游戏方式。

句法：

x;

返回值：

Real (single precision floating point value)

例子：

if keyboard_check(vk_left) x += 5;
if keyboard_check(vk_right) x -= 5;

上面的代码将使用箭头键在x位置加或减，从而产生运动的错觉。

返回：实例变量
