xprevious

这个内置变量返回实例的前一个X位置。这个变量将在begin step事件开始前被设置，但它也可以通过代码在任何时候被设置，这意味着你可以给它你自己的自定义值 如果有必要的话。

语法。

以前的。

返回。

实数（单精度浮点值

例子。

if x != xprevious || y != yprevious
{
moved = true;
}

上面的代码检查xprevious和yprevious值与当前x和y值之间是否有差异。如果有，它将一个变量设置为真。
