y

实例的y值是当前房间中的垂直位置，以像素为单位。该值可以是0，可以是正数或负数，其中0是房间的顶部，向下移动y，向上移动减小y（y的负值表示实例已超出房间顶部）。您甚至可以给定真实值，例如12.345，但由于无法获得0.345像素，因此实例可能似乎没有移动，尽管这些值将由GameMaker Studio 2在内部维护和使用，并且完全有效。

通过设置y（和x）值，您可以使对象在房间中跳转到您选择的位置，或者您可以对它们进行少量加减以产生运动的错觉，而无需实际使用内置的 speed和 direction功能。在制作游戏时，应考虑到这一点，因为一种类型的动作通常比另一种类型更适合一种特定的游戏方式。

句法：

y;

返回值：

Real (single precision floating point value)

例子：

if keyboard_check(vk_down) y += 5;
if keyboard_check(vk_up) y -= 5;

上面的代码将使用箭头键在y位置加或减，从而产生运动的错觉。

返回：实例变量
