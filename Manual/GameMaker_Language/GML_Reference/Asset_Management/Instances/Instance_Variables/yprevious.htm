yprevious

此内置变量返回实例的先前y位置。该变量将在开始步骤事件开始之前设置为当前x位置，但是也可以随时通过代码设置，这意味着您可以在需要时给它自己的自定义值。

句法：

yprevious;

返回值：

Real (single precision floating point value)

例子：

if x != xprevious || y != yprevious
    {
    moved = true;
    }

上面的代码检查是否有任何区别 xprevious和 yprevious值以及当前的x和y值。如果存在，则将变量设置为true。

返回：实例变量
下一个：id
