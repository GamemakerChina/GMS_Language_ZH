以前的

这个内置变量返回实例的前一个y位置。这个变量会在begin step事件开始前被设置为当前的x位置，但它也可以在任何时候通过代码来设置，这意味着你可以 赋予它你自己的自定义值，如果有必要的话。

语法。

前一个。

返回。

实数（单精度浮点值

例子。

if x != xprevious || y != yprevious
{
moved = true;
}

上面的代码检查xprevious和yprevious值与当前x和y值之间是否有差异。如果有，它将一个变量设置为真。
