ystart

首次在房间中创建实例时，此变量存储实例的初始y位置。这不是只读变量，可以设置和读取。

句法：

ystart;

返回值：

Real (single precision floating point value)

例子：

if place_meeting(x, y, obj_spike)
    {
    score -= 100;
    x = xstart;
    y = ystart;
    }

上面的代码将检查是否与“ obj_spike”实例发生冲突，如果存在冲突，则从得分中扣除100，并将实例移回其起始位置。

返回：实例变量
下一个：x
