alarm_set

此功能可用于设置警报。您提供的警报编号为0到11，然后提供将警报设置为的值。该值必须是整数值，您可以将其设置为-1以停止警报（非整数值将四舍五入为最接近的整数）。这是直接设置警报阵列的另一种方法。

句法：

alarm_set(index, value);

争论描述
index设置的告警索引，取值范围为0〜11。
value警报设置的值（整数）。

返回值：

N/A

例子：

for (var i = 0; i < 12; i++)
   {
   if alarm_get(i) > 0 alarm_set(i, -1);
   }

上面的代码检查调用实例中的所有警报，如果警报大于0，则会将其设置为-1，从而阻止它们进一步递减计数。
