例子_拷贝

通过这个函数，你可以 "克隆 "一个实例，因为这将创建一个新版本的实例，在其相同位置运行代码。参数 "perf "用于指示这个新实例是否执行创建事件。 这个函数返回 id的参数，然后可以将其存储在一个变量中，或者用来访问该实例。

注意：如果你选择不执行创建事件，如果实例依赖于该事件中初始化的任何变量，你可能遇到错误。

语法。

instance_copy(perf)。

争论 描述
香水 是否执行新实例的创建事件（true）或不执行（false）。

返回。

真实（实例ID值

例子。

var inst = instance_number(object_index);
if inst < 10
{
instance_copy(true)。
}

上面的代码创建了一个局部变量，用它来存储房间里运行代码的对象的实例数量。如果这个数字小于10，那么这个实例就会复制一个自己。

返回:实例
