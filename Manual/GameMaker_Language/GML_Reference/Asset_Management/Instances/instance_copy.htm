instance_copy

使用此功能，您可以“克隆”实例，因为这将创建在相同位置运行代码的实例的新版本。 “ perf”参数用于指示此新实例是否执行create事件。此函数返回 id新实例的实例，然后可以将其存储在变量中或用于访问该实例。

注意：如果您选择不执行创建事件，则如果实例依赖于此事件中初始化的任何变量，则可能会遇到错误。

句法：

instance_copy(perf);

争论描述
perf是否执行新实例的Create事件（真）（否）。

返回值：

Real (instance ID value)

例子：

var inst = instance_number(object_index);
if inst < 10
    {
    instance_copy(true);
    }

上面的代码创建一个局部变量，并使用它来存储在房间中运行该代码的对象的实例数。如果该数字小于10，则该实例将对其自身进行复制。

返回：实例
