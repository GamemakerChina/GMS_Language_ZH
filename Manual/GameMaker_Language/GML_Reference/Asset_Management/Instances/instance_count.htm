instance_count

使用此只读变量，您可以获取房间中所有活动实例的计数。这将包括运行代码的实例，但不包括已使用了禁用的情况下，例如停用功能。请注意，此变量只会为您提供在步骤开始时的实例数，因此在步骤开始后对房间中的实例所做的任何更改都不会被考虑。

句法：

instance_count;

返回值：

Real

例子：

if (instance_count < 100)
    {
    var dif = 100 - instance_count;
    while (--dif > 0)
        {
        instance_create_layer(random(room_width), random(room_height), "Effects", obj_Star);
        }
    }

上面的代码将创建对象“ obj_Star”的多个实例，直到实例总数达到100。

返回：实例
