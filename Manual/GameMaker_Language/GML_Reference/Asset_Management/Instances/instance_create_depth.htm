instance_create_depth

使用此功能，您可以在房间内任何给定的点并以指定的深度创建指定对象的新实例。深度可以是任何值，其中，越接近相机的物体，绘制的深度越小，深度越大，距离越远，因此，深度为-200的实例将被绘制为深度为+300的实例（例如）。

请注意，此功能实际上将为实例创建一个房间层，因为所有实例都必须位于房间中的一层上，但是由于这是一个托管层（即：不是通过代码或在房间中创建的一层，而是GameMaker Studio 2已自动创建的一个），您将无法访问它，并且 layer实例变量将返回-1。

重要的！最小和最大图层深度为-16000到16000。放置在该范围之外的深度上的任何内容都不会绘制，尽管所有事件仍将正常进行。

此函数返回 id新实例的实例，然后可以将其存储在变量中或用于访问该实例。请注意，在继续调用该函数的事件的代码或操作之前，该函数还将调用正在创建的实例的Create Event。

句法：

instance_create_depth(x, y, depth, obj);

争论描述
x将在其上创建对象的x位置
y将在以下位置创建对象的y位置
depth将创建的实例分配给的深度
obj创建对象实例的对象的对象索引

返回值：

Real (instance ID value)

例子：

var inst = instance_create_depth(x, y, -10000, obj_Bullet);
with (inst)
    {
    speed = other.shoot_speed;
    direction = other.image_angle;
    }

上面的代码创建对象“ obj_Bullet”的新实例，并将实例ID存储在变量中。然后，该变量用于分配 speed和 direction到新实例。

返回：实例
