实例_创建_深度

通过这个函数，你可以在房间的任何给定点和指定的深度创建一个指定对象的新实例。深度可以是任何数值，深度越小，物体将被画得离摄像机越 "近"，深度越大，物体就越远。 深度越大越远，所以深度为-200的实例将被绘制在深度为+300的实例之上（例如）。

注意，这个函数实际上将为实例创建一个房间层，因为所有的实例都必须在房间的一个层上，但由于这是一个被管理的层（即：不是你通过代码或在房间中创建的，而是一个 GameMaker Studio 2自动创建的），你不能访问它，并且 层实例变量将返回-1。

重要的是!有一个最小和最大的层深度为-16000到16000。任何放置在这个范围之外的深度的东西都不会被画出来，尽管所有的事件仍将正常运行。

这个函数返回 id，然后可以将其存储在一个变量中或用于访问该实例。请注意，在继续执行调用该函数的事件的代码或动作之前，该函数也将调用被创建的实例的创建事件。

语法。

instance_create_depth(x, y, depth, obj)。

争论 描述
x 该对象将被创建的X位置
y 该对象将被创建在的y位置
深度 将创建的实例分配到的深度
obj 要创建一个实例的对象的索引

返回。

真实（实例ID值

例子。

var inst = instance_create_depth(x, y, -10000, obj_Bullet);
with (inst)
{
speed = other.shoot_speed;
direction = other.image_angle;
}

上面的代码创建了一个新的对象 "obj_Bullet "的实例，并将实例ID存储在一个变量中。然后这个变量被用来给新的实例分配速度和方向。
