instance_create_layer

使用此功能，您可以在房间内和指定层上的任何给定点创建指定对象的新实例。可以使用图层ID值（由函数返回）来标识图层 layer_create()）或房间编辑器中定义的图层名称（例如字符串，例如“ instance_layer”）。

此函数返回 id新实例的实例，然后可以将其存储在变量中或用于访问该实例。请注意，在继续调用该函数的事件的代码或操作之前，该函数还将调用正在创建的实例的Create Event。

重要的！最小和最大图层深度为-16000到16000。超出该范围的图层上的任何内容都不会绘制，尽管所有事件仍将正常进行。

句法：

instance_create_layer(x, y, layer_id, obj);

争论描述
x将在其上创建对象的x位置
y将在以下位置创建对象的y位置
layer_id将创建的实例分配给的层ID（或名称）
obj创建对象实例的对象的对象索引

返回值：

Real (instance ID value)

例子：

var inst = instance_create_layer(x, y, "instance_layer", obj_Bullet);
with (inst)
    {
    speed = other.shoot_speed;
    direction = other.image_angle;
    }

上面的代码创建对象“ obj_Bullet”的新实例，并将实例ID存储在变量中。然后，该变量用于分配 speed和 direction到新实例。

返回：实例
