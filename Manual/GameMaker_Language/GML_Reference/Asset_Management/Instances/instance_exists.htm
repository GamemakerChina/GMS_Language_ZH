实例_存在

这个函数有两种使用方法，取决于你想检查什么。你可以给它一个object_index来检查，在这种情况下，如果当前房间里存在任何指定对象的活动实例，这个函数将返回true。 或者你也可以给它一个实例ID，在这种情况下，如果这个特定的实例存在并且在当前房间里是活动的，这个函数将返回true。

请注意，这个功能不考虑那些已经用实例停用功能停用的实例。

语法。

instance_exists(obj)。

争论 描述
obj 要检查是否存在的对象或实例。

返回。

布尔型

例子。

if !instance_exists(obj_Enemy)
{
score += 200;
room_goto(rm_hiscores);
}

上面的代码检查是否有任何对象 "obj_Enemy "的实例存在，如果没有，它将添加到变量 "score "并改变房间。
