instance_exists

根据您要检查的内容，可以两种方式使用此功能。您可以给它提供一个object_index进行检查，在这种情况下，此函数将返回 true如果当前房间中存在指定对象的任何活动实例，或者您还可以为其提供实例ID，则此函数将返回 true如果该特定实例存在并且在当前房间中处于活动状态。

请注意，此功能未考虑使用实例停用功能已停用的那些实例。

句法：

instance_exists(obj);

争论描述
obj要检查其是否存在的对象或实例。

返回值：

Boolean

例子：

if !instance_exists(obj_Enemy)
    {
    score += 200;
    room_goto(rm_hiscores);
    }

上面的代码检查是否存在对象“ obj_Enemy”的任何实例，如果不存在，则将其添加到变量“ score”并更改空间。

返回：实例
