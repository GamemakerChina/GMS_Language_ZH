instance_find

所有实例都有唯一的标识符（id ），可在游戏运行时用来修改和操作它们，但是您可能并不总是知道特定实例的ID是什么，因此此功能可以为您提供帮助，因为您可以使用它遍历所有实例来查找它们你需要。您指定要查找其实例的对象和一个数字，如果实例列表中该位置有一个实例，则该函数将返回该实例的ID，如果没有，则返回special关键字。 noone 。您也可以使用关键字 all遍历一个房间中的所有实例以及一个父对象以遍历属于该父/子层次结构的所有实例，甚至可以指定一个实例（如果有 id），以检查它是否确实存在于当前房间中。请注意，由于实例是以任意方式排序的，因此此函数检查实例的方式没有特定的顺序，任何实例都可以位于任何位置。

此函数中“ n”的最大值为

  • 对于关键字 all：instance_count - 1
  • 对于对象索引：instance_number(OBJ) - 1

句法：

instance_find(obj, n);

争论描述
obj查找第n个实例的对象
n要查找的实例号。

返回值：

Real (either an instance ID value or the keyword noone)

例子：

for (var i = 0; i < instance_number(obj_Enemy); ++i;)
    {
    enemy[i] = instance_find(obj_Enemy,i);
    }

上面的代码将使用 for循环访问“ obj_Enemy”的所有实例，并将其ID存储在数组“ enemy []”中。

返回：实例
