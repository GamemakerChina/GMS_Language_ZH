instance_furthest

此函数将检查给定对象的所有实例，以查看距给定x / y点最远的实例。所有检查都将从给定的x / y位置到指定对象实例的原点（x / y位置）。如果不存在该对象的实例，则该函数将返回关键字noone，但如果有实例，它将返回 id找到的实例数。请注意，如果运行代码的实例与被检查的对象具有相同的对象索引，则该对象将包含在检查中（如果调用实例也是父对象的子对象，则包括检查父对象）。

句法：

instance_furthest(x, y, obj);

争论描述
x检查实例距离的x位置。
y检查距离远的实例的y位置。
obj要检查其实例的对象。

返回值：

Real (either an instance ID value or the keyword noone)

例子：

var inst;
inst = instance_furthest(x, y, object_index);
if inst != id
    {
    draw_line(x, y, inst.x, inst.y);
    }

上面的代码将找到与运行该代码的实例共享同一对象索引的最远实例，并将id存储在变量中。然后检查此变量以查看它是否与调用实例的ID相同，如果不相同，则在两个实例之间绘制一条线。

返回：实例
