instance_id

该只读数组包含所有 idŞ房间内的每个活跃实例。这意味着，如果您使用了任何实例取消激活功能，则那些已被取消激活的实例将不包含在此数组中（如果您以前使用过该数组中的值，则现在将返回关键字noone）。

句法：

instance_id[num]);

返回值：

Real (instance ID value)

例子：

for (var i = 0; i < instance_count; i ++;)
    {
    with (instance_id[i]) speed += 0.1;
    }

上面的代码将遍历房间内的所有实例，并将其速度增加0.1。

返回：实例
