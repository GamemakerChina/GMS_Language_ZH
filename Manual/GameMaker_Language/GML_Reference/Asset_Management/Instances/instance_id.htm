instance_id

这个只读 数组保存了房间内每个活动 实例的所有ID。这意味着，如果你使用了任何一个实例停用功能，那些已经停用的实例将不包括在这个数组中（如果你以前使用过这个数组的值，现在它将返回关键字noone）。

语法。

instance_id[num]）。)

返回。

真实（实例ID值

例子。

for (var i = 0; i < instance_count; i ++;)
{
with (instance_id[i]) speed += 0.1;
}

上面的代码将循环浏览房间内的所有实例，并将其速度增加0.1。
