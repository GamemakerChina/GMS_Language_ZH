instance_id_get

使用此功能，您可以从当前活动实例列表中获取任何实例的唯一ID值。您在实例列表中提供索引以获取ID，该函数将返回要存储在变量中的值。

句法：

instance_id_get(index);

争论描述
index实例列表中的索引从0-（实例数-1）开始。

返回值：

Instance ID value (real)

例子：

for (var i = 0; i < instance_count; ++i;)
    {
    var temp_id = instance_id_get(i);
    with (temp_id)
        {
        speed += 0.1;
        }
    }

上面的代码将遍历房间内的所有实例，并将其速度增加0.1。

返回：实例
