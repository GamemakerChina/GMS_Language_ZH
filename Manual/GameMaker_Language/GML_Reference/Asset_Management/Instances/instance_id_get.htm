instance_id_get

通过这个函数，你可以从当前活动的实例列表中获得任何实例的唯一ID值。你给实例列表中的索引来获取ID，该函数将返回该值以存储在一个变量中。

语法。

instance_id_get(index)。

争论 描述
指数 实例列表中的索引，从0-（实例数-1）。

返回。

实例ID值（实数

例子。

for (var i = 0; i < instance_count; ++i;)
{
var temp_id = instance_id_get(i);
with (temp_id)
{
speed += 0.1;
}
}

上面的代码将循环浏览房间内的所有实例，并将其速度增加0.1。

返回:实例
