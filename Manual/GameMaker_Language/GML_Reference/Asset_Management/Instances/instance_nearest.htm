instance_nearest

此函数将检查给定对象的所有实例，以查看最接近给定x / y点的实例。所有检查都将从给定的x / y位置到指定对象实例的原点（x / y位置）。如果不存在该对象的实例，则该函数将返回关键字noone，但如果有实例，它将返回 id找到的实例数。请注意，如果运行代码的实例与被检查的对象具有相同的对象索引，则该对象将包含在检查中（如果调用实例也是父对象的子对象，则包括检查父对象）。

句法：

instance_nearest(x, y, obj);

争论描述
x要检查的x位置。
y要检查的y位置。
obj要检查其实例的对象。

返回值：

Real (either an instance ID value or the keyword noone)

例子：

var inst, xx;
xx = x;
x -= 10000;
inst = instance_nearest(xx, y, object_index);
if inst != id
    {
    draw_line(x, y, inst.x, inst.y);
    }
x += 10000;

上面的代码将当前实例移动10000像素，然后检查其先前位置以找到同一对象类型的最近实例。如果该实例本身是实体，它只会做回到原位置，但是如果发现的实例不同，它将在两者之间画一条线。

返回：实例
