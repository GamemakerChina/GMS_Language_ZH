通过这个函数，你可以查出房间里存在多少个指定对象的活动实例。当使用这个函数检查时，如果该对象是一个父 对象，那么所有子对象也将包括在返回值中，并且 还要注意，那些已经用实例停用函数停用的实例将不包括在这个检查中。

语法。

instance_number(obj)。

争论 描述
obj 对象的实例总数。

返回。

真实的

例子。

如果 instance_number(object_index) < 50
{
instance_create_layer(random(room_width), random(room_height), "Instances", object_index);
}

上面的代码将检查创建的实例的数量，这些实例的形式与当前实例相同，然后如果少于50个，就在房间内的一个随机位置创建另一个。
