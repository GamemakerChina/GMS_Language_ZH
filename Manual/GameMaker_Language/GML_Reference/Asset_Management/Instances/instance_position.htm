instance_position

使用此功能，您可以检查与另一个实例或对象的所有实例发生碰撞的位置。使用此功能时，您正在检查房间中的单个实例或对象。如果该实例已检查精确的冲突，则将针对该实例的边界框或针对该实例的掩码进行此检查，并将返回唯一的实例 id。如果不需要冲突实例的ID，则应考虑使用 position_meeting()反而。此功能还接受特殊关键字all和 other并返回关键字 noone如果没有发生冲突，或者如果确实发生冲突，则找到实例的唯一ID值。

句法：

instance_position( x, y, obj );

争论描述
x检查实例的x位置。
y检查实例的y位置。
obj要检查其实例的对象。

返回值：

Real (either an instance ID value or the keyword noone)

例子：

var inst;
inst = instance_position(mouse_x, mouse_y, obj_Pause_Button);
if (inst != noone)
    {
    with (inst) image_index=1;
    instance_create_layer(room_width / 2, 0, "Controllers", obj_Menu);
    }

上面的代码将检查与鼠标位置处的“ obj_Pause_Button”实例的冲突，如果存在冲突，则它将使用返回的ID将其image_index设置为新值，然后再创建对象“ obj_Menu”的新实例”。
