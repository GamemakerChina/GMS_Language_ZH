实例_位置

用这个函数你可以检查一个位置是否与另一个实例或一个物体的所有实例发生碰撞。当你使用这个函数时，你是在检查房间里的一个实例或一个物体的单个点。这个检查将针对实例的边界 框，或者如果该实例有精确的碰撞检查，则针对该实例的掩码进行检查，并将返回唯一的实例 id.如果你不需要碰撞的实例的id，你应该考虑使用 应该考虑使用 position_meeting()来代替。这个函数也接受特殊的关键字 所有和 other ，如果没有发生碰撞，将返回关键字 noone ，如果发生了碰撞，将返回找到的实例的唯一ID值。

语法。

instance_position( x, y, obj )。

争论 描述
x 检查实例的X位置。
y 检查实例的y位置。
obj 要检查的对象的实例。

返回。

真实（要么是一个实例ID值，要么是关键词noone）。

例子。

var inst;
inst = instance_position(mouse_x, mouse_y, obj_Pause_Button);
if (inst != noone)
{
with (inst) image_index=1;
instance_create_layer(room_width / 2, 0, "Controllers", obj_Menu);
}

上面的代码将检查在鼠标位置是否与 "obj_Pause_Button "的实例发生碰撞，如果有的话，它将使用返回的id将其image_index设置为一个新的值，然后创建一个新的对象实例 "obj_Menu"。
