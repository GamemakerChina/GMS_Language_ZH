实例_位置_列表

通过这个函数，你可以检查一个位置是否与另一个物体的所有实例发生碰撞。当你使用这个函数时，你是在检查房间里的一个点是否与一个物体的所有实例相撞，而且检查将针对这些实例的边界框进行。 实例进行检查，如果该对象有精确碰撞检查，则针对实例的掩码进行检查。

该函数将填充一个（先前创建的）DS列表，其中包括被认为是在该列表中的所有对象实例的唯一值。 id的唯一值，这些对象的所有实例都被认为是与调用的实例发生了碰撞。 碰撞的对象的所有实例的唯一值，并且你可以选择根据被检查的位置到被发现有碰撞的实例的原点的距离来排列列表。请注意，该函数也接受特殊关键字 GML_Overview/Instance_Keywords.htm">全部，在这种情况下，所有发现有碰撞的实例都会被列出。该函数返回发现的实例的数量，如果没有发现，则返回0。

语法。

instance_position_list(x, y, obj, list, ordered)。

争论 描述
x 检查实例的X位置。
y 检查实例的y位置。
obj 要检查的对象的实例。
列表 用来存储碰撞实例的ID的DS列表。
有序 列表是否应按距离排序（true）或不排序（false）。

返回。

Int (发现有碰撞的实例数量)

例子。

var _list = ds_list_create();
var _num = instance_position_list(mouse_x, mouse_y, all, _list, false);
if _num > 0
{
for (var i = 0; i < _num; ++i;)
{
instance_destroy(_list[| i]);
}
}
ds_list_destroy(_list)。

上面的代码将检查与鼠标位置上发现的所有实例的碰撞情况。这些将被添加到一个DS列表中，然后通过循环来销毁每个碰撞的实例。
