instance_position_list

使用此功能，您可以检查与另一个对象的所有实例发生碰撞的位置。使用此功能时，您正在检查对象中所有实例在房间中的单个点，如果该对象已检查精确的碰撞，则将根据实例的边界框或实例的蒙版进行检查。

该函数将使用唯一的字段填充（先前创建的） DS列表 id对象的所有实例的值都被视为与调用实例发生冲突，并且您可以选择根据从检查位置到发现实例发生碰撞的原点之间的距离对列表进行排序。请注意，该函数还接受特殊关键字GML _Overview / Instance_Keywords.htm“>all ，在这种情况下，将列出所有发现冲突的实例。该函数返回找到的实例数，如果找不到则返回0。

句法：

instance_position_list(x, y, obj, list, ordered);

争论描述
x检查实例的x位置。
y检查实例的y位置。
obj要检查其实例的对象。
list用于存储冲突实例ID的DS列表。
ordered该列表是否应按距离排序（true ） 或不 （false ）。

返回值：

Int (The number of instances found to be in collision)

例子：

var _list = ds_list_create();
var _num = instance_position_list(mouse_x, mouse_y, all, _list, false);
if _num > 0
    {
    for (var i = 0; i < _num; ++i;)
        {
        instance_destroy(_list[| i]);
        }
    }
ds_list_destroy(_list);

上面的代码将检查与在鼠标位置找到的所有实例是否冲突。这些实例将添加到DS列表中，然后循环遍历以破坏发生冲突的每个实例。
