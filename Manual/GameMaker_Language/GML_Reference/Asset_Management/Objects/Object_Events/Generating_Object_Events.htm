生成对象事件

GameMaker Studio 2完全是事件驱动的，所有的行动都是由某些事件发生的结果，这些事件是按照特定的顺序或由游戏中的特定事件触发的。有许多不同的事件。 每个事件都有特定的任务或时间，其中一些甚至被分割成独立的 "子 "事件。通常情况下，当GameMaker Studio 2检测到游戏中的触发器时，就会运行这些事件，或者是每一个游戏的tick 就像步骤和绘图事件一样），但你也可以在一段代码中对当前房间的一个实例应用一个事件。

存在以下处理生成事件的函数。

  • 事件_继承的
  • 事件_执行
  • 事件_表演对象
  • 事件_用户

还有一些只读变量，可以与上述函数一起使用，这对调试特别有用。

  • 事件_行动
  • 事件_编号
  • 事件_object
  • 事件类型

有关事件的更多信息，请参见 "对象事件"部分。
