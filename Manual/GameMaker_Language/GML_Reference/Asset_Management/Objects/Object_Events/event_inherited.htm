事件_继承的

这个函数将调用实例的父对象的当前事件。通常情况下，当一个实例有一个父对象时，它会自动继承所有与父对象相同的事件，但是如果（例如）你的父对象有一个创建事件，并且 你给你的子对象添加一个事件，子对象的所有实例将运行你添加的新的创建事件，而不是父对象中的事件。如果你需要同时使用父对象的事件和子对象的事件 的时候，你应该使用这个函数，因为它将运行父对象事件，然后再继续执行子事件包含的其他代码或动作。

语法。

event_inherited()。

返回。

不适用

例子。

event_inherited();
switch (sprite_index)
{
case spr_Enemy_1: dmg += 2; break;
case spr_Enemy_4: dmg -= 1; break;
case spr_Enemy_10: dmg += 10; break;
}

上面的代码调用了继承的父事件（其中我们初始化了变量 "dmg" ），然后继续修改 "dmg" 变量。如果没有为运行这段代码的实例指定父级 运行这段代码的实例，我们会得到一个 "未知变量 "的错误，因为 dmg 还没有被定义。

返回。生成对象事件
