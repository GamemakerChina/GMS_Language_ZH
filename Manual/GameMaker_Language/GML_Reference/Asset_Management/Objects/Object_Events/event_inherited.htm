event_inherited

该函数将调用实例的父对象的当前事件。通常，当实例具有父对象时，它会自动继承与父对象相同的所有事件，但是，如果（例如）您的父对象具有create事件，并且您向子对象添加了一个事件，则子对象的所有实例都会运行您添加的新create事件，而不运行父对象中的事件。如果您需要同时使用相同类型的父对象事件和子对象事件，则应使用此函数，因为它将继续运行父对象事件，然后继续执行子事件包含的其余代码或操作。

句法：

event_inherited();

返回值：

N/A

例子：

event_inherited();
switch (sprite_index)
    {
    case spr_Enemy_1: dmg += 2; break;
    case spr_Enemy_4: dmg -= 1; break;
    case spr_Enemy_10: dmg +=10; break;
    }

上面的代码调用了继承的父事件（我们在其中初始化变量“dmg “），然后继续修改”dmg如果没有为运行此代码的实例指定父项，则将出现“未知变量”错误，如下所示： dmg尚未定义。

返回：生成对象事件
