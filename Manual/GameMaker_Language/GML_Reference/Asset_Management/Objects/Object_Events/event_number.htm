事件_编号

这个只读变量返回当前被调用的事件的编号，其中的编号实际上是指事件的 "子事件"，即：对于步骤事件，事件编号可以是常数ev_step_normal, ev_step_begin, 或ev_step_end.关于特定子事件可用的常数的完整列表，请参见 event_perform()，如果你需要知道主事件本身，你应该 检查 事件类型.

语法。

event_number;

返回。

真实的

例子。

switch (event_number)
{
case ev_step_normal: show_debug_message("Step event!"); break;
case ev_game_start: show_debug_message("Game Start"); break;
case ev_room_start: show_debug_message("Room Start!"); break;
}

上述代码可以在脚本中调用，用于显示调试信息，告知你当前正在触发哪个事件。
