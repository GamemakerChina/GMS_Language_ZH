event_number

此只读变量返回当前正在调用的事件的编号，其中该编号实际上是指该事件的“子事件”，即：对于step事件，事件编号可以是常量ev_step_normal ， ev_step_begin中的任何一个。或ev_step_end 。有关可用于特定子事件的常量的完整列表，请参见 event_perform()，如果您需要了解主要事件本身，则应检查 event_type。

句法：

event_number;

返回值：

Real

例子：

switch (event_number)
    {
    case ev_step_normal: show_debug_message("Step event!"); break;
    case ev_game_start: show_debug_message("Game Start""); break;
    case ev_room_start: show_debug_message("Room Start!"); break;
    }

上面的代码可以从脚本中调用，并用于显示调试消息，通知您当前正在触发哪个事件。

返回：生成对象事件
