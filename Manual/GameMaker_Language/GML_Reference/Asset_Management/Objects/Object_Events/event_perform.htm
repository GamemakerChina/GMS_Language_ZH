event_perform

此函数将为运行代码的实例执行带有指定参数的指定事件。这里有许多选项可以完全模拟所有可能的事件，但是请注意，这实际上只是执行事件中的所有代码，游戏不会修改任何内容以使其手动触发，例如，如果您选择执行键盘按下事件，事件将被触发，但相关按键将不会被识别为已按下。或者，如果您执行警报事件，警报计数器将不会设置为-1，而是继续递减计数。

句法：

event_perform(type, numb);

争论描述
type要执行的事件的类型。见下文。
numb特定事件常量或值（如果需要）（否则，只需使用0）。

返回值：

N/A

扩展示例


为了帮助理解如何使用此功能，请查看以下几个示例：

event_perform(ev_keypress, ord("W"));

这将执行与“按下键盘检查”的“ W”键关联的事件（实际上不会产生键盘按下）。

event_perform(ev_step, ev_step_begin);

这将执行开始步骤事件（如果从任何步骤事件中调用，将导致开始步骤事件代码运行两次）。

event_perform(ev_create, 0);

这将执行实例的Create事件。
