事件_表演对象

这个函数的工作原理与 event_perform()除了这次你可以指定来自另一个对象的事件。这里有许多选项，可以完全模拟所有可能的事件。 可能的事件，但请注意，这只是执行事件中的所有代码，游戏不会修改任何东西来使其手动触发，例如，如果你选择执行一个键盘按压事件，该事件将被触发 但相关的键不会被识别为已被按下。或者如果你执行一个警报事件，警报计数器不会被设置为-1，而是继续倒数。你可以在这个函数的页面上找到这个函数所需要的常量的完整列表。 所需的常量的完整列表，请参见该函数的页面 event_perform().

注意：用此函数调用的事件中的动作适用于当前实例，而不是给定对象的实例。

语法。

event_perform_object(obj, type, numb)。

争论 描述
obj 应该触发其事件的对象。
类型 要执行的事件的类型。见下文。
哑巴 具体事件，如果有必要的话（否则就用0）。

返回。

不适用

例子。

event_perform_object(obj_Player, ev_keypress, ord("W"))。

这将执行与当前实例中的对象 "obj_Player "的键盘检查>W键相关的事件。

返回。生成对象事件
