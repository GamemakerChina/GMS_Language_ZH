event_perform_object

该功能与 event_perform()除了这次您可以指定另一个对象的事件。这里有许多选项可以完全模拟所有可能的事件，但是请注意，这实际上只是执行事件中的所有代码，并且游戏不会修改任何内容以使其手动触发，例如，如果您选择执行键盘按下事件，事件将被触发，但相关按键将不会被识别为已按下。或者，如果您执行警报事件，警报计数器将不会设置为-1，而是继续递减计数。您可以在函数页面上找到该函数所需的可用常数的完整列表。event_perform() 。

注意：使用此函数调用的事件中的操作将应用于当前实例，而不应用于给定对象的实例。

句法：

event_perform_object(obj, type, numb);

争论描述
obj应该触发其事件的对象。
type要执行的事件的类型。见下文。
numb如果需要，则为特定事件（否则，仅使用0）。

返回值：

N/A

例子：

event_perform_object(obj_Player, ev_keypress, ord("W"));

这将在当前实例中执行与对象“ obj_Player”中与“按下键盘检查> W键”关联的事件。

返回：生成对象事件
