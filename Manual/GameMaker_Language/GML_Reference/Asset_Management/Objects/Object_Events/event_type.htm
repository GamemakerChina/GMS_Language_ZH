event_type

此只读变量返回当前正在执行的事件的类型，它可以是下列常量之一：

  • ev_create
  • ev_destroy
  • ev_step
  • ev_alarm
  • ev_keyboard
  • ev_keypress
  • ev_keyrelease
  • ev_mouse
  • ev_collision
  • ev_other
  • ev_gesture
  • ev_draw


有关与不同事件相关的常量的完整列表，请参见 event_perform()，如果您需要事件号（即子事件），则应检查 event_number。

句法：

event_type;

返回值：

Real

例子：

show_debug_message("Current Event = " + string(event_type));

上面的代码将显示一条调试消息，其中包含当前正在执行的事件类型。

返回：生成对象事件
