object_exists

此函数返回是否存在具有指定索引的对象。请注意，这将检查资产浏览器中是否存在对象，而不是游戏室中是否存在对象。为此，您应该使用功能 instance_exists()。

句法：

object_exists(obj);

争论描述
obj要检查的对象的索引。

返回值：

Boolean

例子：

if object_exists(obj_temp)
    {
    with (obj_Temp) scr_add_event();
    }

上面的代码检查变量“ obj_temp”以确保它存储了对象的索引，然后，如果这样做，则使该对象运行脚本。
