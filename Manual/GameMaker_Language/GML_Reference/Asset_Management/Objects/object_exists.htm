对象_存在

这个函数返回一个具有指定索引的对象是否存在。注意，这个函数检查对象是否存在于资产浏览器中，而不是实际存在于游戏室中。为此，你应该使用函数 instance_exists().

语法。

object_exists(obj)。

争论 描述
obj 要检查的对象的索引。

返回。

布尔型

例子。

if object_exists(obj_temp)
{
with (obj_Temp) scr_add_event();
}

上面的代码检查变量 "obj_temp"，以确保它存储了一个对象的索引，然后，如果它存储了索引，它就会使这个对象运行一个脚本。
