对象_获取掩码

这个函数将告诉你，你要检查的对象是否有一个掩码索引，如果有，它将返回该掩码的索引，如果没有，则返回-1。请注意，这不是一个实例函数!你可以有一个对象 而同一对象的一个实例可以有一个掩码，反之亦然，或者它们甚至可以有不同的掩码。你可以使用 "实例 "变量来设置单个实例的掩码索引。 mask_index 实例变量来设置单个实例的掩码索引。

语法。

object_get_mask(obj)。

争论 描述
obj 要检查的对象的索引

返回。

真正的

例子。

if (mask_index != object_get_mask(object_index))
{
mask_index = object_get_mask(object_index);
}

上面的例子将检查实例的掩码索引与实例的object_index的掩码。如果它们不一样，那么它将为实例分配与对象索引的面具相同的面具。
