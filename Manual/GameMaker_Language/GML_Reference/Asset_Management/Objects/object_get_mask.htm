object_get_mask

此函数将告诉您所检查的对象是否具有掩码索引，如果存在，它将返回该掩码的索引；如果没有，则返回-1。请注意，这不是实例函数！您可以拥有一个没有遮罩的对象，而同一对象的实例可以有一个，反之亦然，甚至它们可以有不同的遮罩。您可以使用设置单个实例的遮罩索引 mask_index实例变量。

句法：

object_get_mask(obj);

争论描述
obj要检查的对象的索引

返回值：

real

例子：

if (mask_index != object_get_mask(object_index))
    {
    mask_index = object_get_mask(object_index);
    }

上面的示例将对照实例的object_index的掩码检查实例的掩码索引。如果它们不相同，则它将为对象分配与对象索引所具有的相同的掩码。
