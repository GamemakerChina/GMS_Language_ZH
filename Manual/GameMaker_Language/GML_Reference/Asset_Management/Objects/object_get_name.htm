对象的名称（object_get_name

这个函数将返回指定对象的名称，即一个字符串。这个名字是在GameMaker Studio 2主窗口的资产浏览器中为该对象指定的名字。请注意，这只是一个字符串 不能用于直接引用对象 - 为此你需要对象的索引。然而，你可以使用这个字符串来获取对象的索引，使用返回的字符串和函数 asset_get_index().

语法。

object_get_name(obj)。

争论 描述
obj 要检查的对象的索引。

返回。

字符串

例子。

str = object_get_name(object_index)。

上面的代码将获得运行代码的实例的对象索引名称，并将返回值存储在变量 "str "中。
