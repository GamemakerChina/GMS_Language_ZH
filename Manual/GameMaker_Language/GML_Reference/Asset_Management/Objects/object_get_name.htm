object_get_name

此函数将名称作为指定对象的字符串返回。 GameMaker Studio 2主窗口的“资源浏览器”中为对象指定的名称。请注意，这只是一个字符串，不能用于直接引用对象-因为您将需要对象索引。但是，您可以使用此字符串使用返回的字符串和函数来获取对象索引 asset_get_index()。

句法：

object_get_name(obj);

争论描述
obj要检查的对象的索引。

返回值：

String

例子：

str = object_get_name(object_index);

上面的代码将获取运行该代码的实例的对象索引的名称，并将返回值存储在变量“ str”中。
