object_get_parent

此函数将为您提供已分配给指定对象的任何父对象的object_index，否则返回-100以表明该对象没有分配父对象，或者返回-1（如果被检查的对象不存在）。有关父级的更多信息，请参见“对象编辑器”部分。

句法：

object_get_parent(obj);

争论描述
obj要检查的对象的索引

返回值：

Real

例子：

par = object_get_parent(object_index);

上面的代码将获取运行该代码的实例的对象索引的父级，并将返回值存储在变量“ par”中。

背面：物件
