对象_获取_持久性

这个函数将告诉你，你正在检查的对象是否被标记为 "持久性"。一个持久的对象是会导致它的任何实例从一个房间带到另一个房间，除非它们被明确销毁。 请注意，这不是一个实例函数!所以，你可以有一个持久化的对象和同一个对象的非持久化的实例，反之亦然。你可以使用下面的函数来设置一个单独的实例的持久化标志 持久性实例 变量来设置单个实例的持久化标志。

语法。

object_get_persistent(obj)。

争论 描述
obj 要检查的对象的索引。

返回。

布尔型

例子。

if (!persistent) && (object_get_persistent(object_index))
{
persistent = true;
}

上面的代码将检查运行中的实例，看它是否是持久化的，以及检查实例的对象索引，看它是否被标记为持久化。如果该实例不是持久化的，但其对象索引被标记为 的时候，它将把该实例的 "persistent "设置为 "true"。

返回。物体
