object_get_persistent

此功能将告诉您所检查的对象是否已标记为“持久”。持久对象是一个会导致它的任何实例在一个房间到另一个房间之间传播的对象，除非它们被明确破坏。请注意，这不是实例函数！因此，您可以拥有一个持久对象和同一对象的一个非持久实例，反之亦然。您可以使用设置单个实例的持久性标志 persistent实例变量。

句法：

object_get_persistent(obj);

争论描述
obj要检查的对象的索引。

返回值：

Boolean

例子：

if (!persistent) && (object_get_persistent(object_index))
    {
    persistent = true;
    }

上面的代码将检查运行该实例的实例是否具有持久性，并检查该实例的对象索引以查看其是否被标记为持久性。如果实例不是持久性的，但对象索引被标记为持久性，它将为该实例将“ persistent”设置为true。
