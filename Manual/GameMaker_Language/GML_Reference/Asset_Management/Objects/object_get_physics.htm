object_get_physics

此功能将告诉您所检查的对象是否已标记为“启用了物理”-在这种情况下，它将返回 true，-或不-在这种情况下，它将返回 false。

句法：

object_get_physics(obj);

争论描述
obj要检查的对象的索引。

返回值：

Boolean

例子：

if object_get_physics(object_index)
    {
    phy_active = true;
    }

上面的代码将检查运行它的实例，以查看从其创建的对象是否启用了物理功能，如果是，则激活该实例的物理仿真。
