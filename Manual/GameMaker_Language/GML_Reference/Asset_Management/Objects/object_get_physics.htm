对象_get_physics

这个函数将告诉你，你正在检查的对象是否被标记为 "物理学启用"--在这种情况下，它将返回 "true"，或者没有--在这种情况下它将返回 "false"。

语法。

object_get_physics(obj)。

争论 描述
obj 要检查的对象的索引。

返回。

布尔型

例子。

如果 object_get_physics(object_index)
{
phy_active = true;
}

上面的代码将检查运行它的实例，看它所创建的对象是否启用了物理学，如果是，它就会激活实例的物理学模拟。

返回。物体
