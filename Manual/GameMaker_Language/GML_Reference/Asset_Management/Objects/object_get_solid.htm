object_get_solid

此功能将告诉您正在检查的对象是否已标记为“实体”。当使用传统的碰撞系统（即：物理世界已关闭）时，实体会生成特殊的碰撞事件。请注意，这不是实例函数！因此，您可以有一个实体对象和一个相同对象的普通实例，反之亦然。您可以使用 solid实例变量。

句法：

object_get_solid(obj);

争论描述
obj要检查的对象的索引。

返回值：

Boolean

例子：

if (!solid) && (object_get_solid(object_index))
    {
    solid = true;
    }

上面的代码将检查运行它的实例是否可靠，以及检查实例的对象索引以查看其是否被标记为可靠。如果实例不是实体，但对象索引被标记为实体，则它将对该实例的“ solid”设置为true。
