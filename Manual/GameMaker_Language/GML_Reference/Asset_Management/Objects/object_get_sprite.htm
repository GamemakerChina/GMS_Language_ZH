object_get_sprite

该函数将告诉您所检查的对象是否具有子画面，如果存在，它将返回该子画面的索引；如果没有，则返回-1。请注意，这不是实例函数！您可以拥有一个没有子画面的对象，而同一对象的实例可以有一个子画面，反之亦然，甚至它们可以具有不同的子画面。您可以使用来设置单个实例精灵 sprite_index实例变量。

句法：

object_get_sprite(obj);

争论描述
obj要检查的对象的索引

返回值：

Real (sprite asset ID)

例子：

var _spr = object_get_sprite(object_index);
if (sprite_index != _spr)
    {
    sprite_index = _spr;
    }

上面的示例将检查 sprite_index实例针对精灵的 object_index实例的如果它们不相同，则它将为对象分配与对象索引相同的精灵。
