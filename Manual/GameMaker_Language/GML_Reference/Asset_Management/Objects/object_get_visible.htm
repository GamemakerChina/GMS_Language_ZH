object_get_visible

该函数将告诉您所检查的对象是否已标记为“可见”（运行其draw事件）（不运行其draw事件）。请注意，这不是实例函数！因此，您可以拥有一个可见的对象以及该对象的不可见实例，反之亦然。您可以使用 visible实例变量。

句法：

object_get_visible(obj);

争论描述
obj要检查的对象的索引。

返回值：

Boolean

例子：

if (!visible) && (object_get_visible(object_index))
    {
    visible = true;
    }

上面的代码将检查运行该实例的实例是否可见，以及检查实例的对象索引以查看其是否标记为可见。如果实例不可见，但对象索引标记为打开，则它将对该实例的“ visible”设置为true。
