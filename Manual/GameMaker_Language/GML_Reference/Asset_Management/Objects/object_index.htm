对象_索引

这个只读变量返回实例所创建的对象的索引。这与对象名称不同，后者是一个字符串，可以用 object_get_name(), 因为这个函数返回索引号，它是GameMaker Studio 2在创建时分配给每个对象的唯一值。

语法。

object_index;

返回。

实数（整数 - 资源索引值）。

例子。

obj_name = object_get_name(object_index)。

上面的代码将使用object_index来查找当前实例所创建的对象的名称。

返回。物体
