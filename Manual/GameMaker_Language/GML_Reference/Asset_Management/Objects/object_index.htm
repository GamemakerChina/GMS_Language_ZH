object_index

该只读变量返回创建实例所基于的对象的索引。这是不一样的对象名，它是一个字符串，并且可以使用被发现 object_get_name()，因为此函数返回索引号，索引号是GameMaker Studio 2在创建时分配给每个对象的唯一值。

句法：

object_index;

返回值：

Real (integer - resource index value)

例子：

obj_name = object_get_name(object_index);

上面的代码将使用 object_index查找创建当前实例的对象的名称。
