object_set_mask

使用此功能，您可以设置特定对象的mask_index（有关蒙版的更多信息，请参见“对象编辑器” ）。这意味着在更改遮罩之后创建的该对象的所有实例都将使用此新的遮罩创建，而房间中已有的实例可能不会受到影响。可以将其设置为-1以删除蒙版精灵，因此默认使用为对象的精灵定义的蒙版（如果未选择任何精灵，则不设置任何蒙版）。

请注意，这不是实例函数！您可以使用设置单个实例的掩码索引 mask_index变量，因此有十个实例，它们的创建对象均具有不同的蒙版，即使使用此功能更改了对象的蒙版索引，当前房间中的所有实例也将保持原样，并且仅调用函数后创建的实例将从新的掩码开始。

句法：

object_set_mask(index, spr);

争论描述
index要更改的对象的索引。
spr用作对象蒙版的新精灵。

返回值：

N/A

例子：

if (global.level == 10)
    {
    object_set_mask(obj_Platform, spr_mask_10);
    }

上面的代码检查全局变量的值，然后更改对象“ obj_Platform”的掩码索引（如果它等于10）。
