object_set_solid

使用此功能，您可以将特定对象标记为实体或不实体（有关“ solid”标志的更多信息，请参见“对象编辑器”部分）。这意味着在更改实体后创建的该对象的所有实例都将以此新状态创建，而房间中已经存在的实例可能不会受到影响。

请注意，这不是实例函数！您可以使用 solid变量，因此即使有问题的对象在IDE中的标志设置为false，也有十个实例都标记为实体，即使您使用此功能将对象的实体标志更改为true，当前房间中的所有实例会保持原样，并且只有在调用函数后创建的实例才会以solid标记为true开头。

句法：

object_set_solid( index, solid );

争论描述
index要更改的对象的索引。
solid对象的新实体（true = solid，false = not solid）。

返回值：

N/A

例子：

if !object_get_solid(obj_Block)
    {
    object_set_solid(obj_Block, true);
    }

上面的代码检查对象“ obj_Block”的实体标志，如果它为false，则将其更改为标记为true。
