object_set_sprite

使用此功能可以设置特定对象的精灵索引。这意味着在更改sprite_index之后创建的该对象的所有实例都将使用此新的sprite_index创建，而房间中已经存在的实例可能不会受到影响。

请注意，这不是实例函数！您可以使用设置单个实例的精灵索引 sprite_index变量，因此有十个实例，它们的创建对象都具有不同的精灵，即使使用此功能更改了对象的精灵索引，当前房间中的所有实例也将保持原样，并且仅调用函数后创建的实例将以新的精灵开始。

句法：

object_set_sprite( index, spr );

争论描述
index要更改的对象的索引。
spr分配给对象的精灵。

返回值：

N/A

例子：

switch (room)
    {
    case rm_start: object_set_sprite(obj_Player, spr_uniform); break;
    case rm_middle: object_set_sprite(obj_Player, spr_swimsuit); break;
    case rm_end: object_set_sprite(obj_Player, spr_casual); break;
    }
instance_create_layer(32, 32, "Instances", obj_Player);

上面的代码将根据运行代码的实例当前所在的房间将对象“ obj_Player”的子画面索引设置为不同的值。然后，将创建“ obj_Player”的实例。
