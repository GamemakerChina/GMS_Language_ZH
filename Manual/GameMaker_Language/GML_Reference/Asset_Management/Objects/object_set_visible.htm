object_set_visible

使用此功能，您可以将特定对象标记为可见或不可见（有关“ visible”标记的更多信息，请参见“对象编辑器”部分）。这意味着将在更改可见性之后创建的该对象的所有实例都将以此新状态创建，而房间中已经存在的实例可能不会受到影响。

请注意，这不是实例函数！您可以使用设置单个实例的可见标志 visible变量，因此有十个实例都被标记为可见，即使该对象在IDE中的标志设置为false，即使您使用此功能将对象的可见标志更改为true，当前房间中的所有实例也将被标记为可见。会保持原样，并且只有在调用函数后创建的实例才会以visible开头，标记为true。

句法：

object_set_visible( index, vis );

争论描述
index要更改的对象的索引。
vis对象的新可见性（true =可见，false =不可见）。

返回值：

N/A

例子：

if !object_get_visible(obj_Block)
    {
    object_set_visible(obj_Block, true);
    }

上面的代码检查对象“ obj_Block”的可见标志，如果为false，则将其更改为标记为true。
