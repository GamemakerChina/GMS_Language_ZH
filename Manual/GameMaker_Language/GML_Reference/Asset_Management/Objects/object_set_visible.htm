对象_set_visible

通过这个功能，你可以将一个特定的对象标记为可见或不可见（关于 "可见 "标记的更多信息，请参见 "对象编辑器"部分）。这意味着，在改变了可见性之后创建的这个对象的所有实例 的所有实例都将以这个新的状态被创建，而已经在房间里的实例可能不受影响。

请注意，这不是一个实例函数!你可以使用变量来设置单个实例的可见标志。 可见变量来设置单个实例的可见标志，这样就可以将十个实例全部标记为可见，即使该对象在IDE中的标志被设置为false。 即使该对象在IDE中的标志被设置为false，即使你使用这个函数将该对象的可见标志改为true，目前在房间中的所有实例仍将保持原样，只有在调用该函数后创建的实例才会从可见标志为true开始。 调用该函数后创建的实例才会在开始时将visible标志设为true。

语法。

object_set_visible( index, vis )。

争论 描述
指数 要改变的对象的索引。
视觉 对象的新可见度（true=可见，false=不可见）。

返回。

不适用

例子。

if !object_get_visible(obj_Block)
{
object_set_visible(obj_Block, true);
}

上面的代码检查对象 "obj_Block "的可见标志，如果它是假的，那么它就被改为标志为真。

返回。物体
