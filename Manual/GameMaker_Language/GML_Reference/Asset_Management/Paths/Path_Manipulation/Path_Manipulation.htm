路径操纵

在GameMaker Studio 2中，你不仅仅局限于使用路径编辑器制作的路径。有许多函数可以用来 在运行时操纵和改变这些路径，或者甚至创建新的路径，然后改变和操纵这些路径。

以下是用于改变路径的功能。

  • 路径_改变点
  • 路径_插入点
  • 路径_delete_point
  • 路径_清除点
  • 路径_附加
  • 路径_指定
  • 路径_删除
  • 路径_重复
  • 路径_翻转
  • 路径_mirror
  • 路径_反转
  • 路径_旋转
  • 路径_重新缩放
  • 路径_封闭
  • 路径_设置类型
  • 路径_设置精度
  • 路径_移位
