path_add

使用此功能，您可以在GameMaker Studio 2创建路径，而无需使用路径编辑器。此函数将返回路径的索引，该索引应存储在变量中，并从那时起用作该路径的引用。请注意，创建的路径是空的，即：它没有定义点，因此您将不得不使用其他可用功能将点添加到路径或使用MP网格生成路径。一旦使用完路径，或希望创建一个新路径并将其索引存储在同一变量中，则应首先使用以下命令删除旧路径：path_delete以防止可能导致游戏崩溃的内存泄漏。

句法：

path_add();

返回值：

Real

例子：

global.newpath = path_add();

这将创建一个新路径并将其索引分配给global.newpath。

返回：路径操纵
