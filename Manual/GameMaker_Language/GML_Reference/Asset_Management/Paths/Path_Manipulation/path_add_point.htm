路径_添加点

通过这个函数，你可以在指定的路径上添加一个点，并设置其速度系数。这个点被添加到路径的末端，不管这个点的位置如何，系数都等于 path_speed的百分比。 下面的实例在到达路径中的这一点时实际走的百分比。如果你想把一个路径点放在其他位置，而不是终点，你应该使用 path_insert_point().

如果在资产浏览器的路径上使用，请注意该函数将改变实际的资产，因此将永久地影响游戏中所有实例对该路径的使用，从使用该函数的那一刻起直到游戏结束。 游戏结束。如果这不是你所需要的，那么你应该使用一个类似 path_duplicate() 的函数，先创建一个路径的副本，然后在复制的资产上调用这个函数（不要 不要忘记在资产不再需要时调用 path_delete() ）。)

语法。

path_add_point(index, x, y, speed)。

争论 描述
指数 要添加点的路径的索引。
x 新点的X坐标。
y 新点的Y坐标。
速度 该点的速度系数（默认值为100）。

返回。

不适用

例子。

my_path = path_add();
for (var i = 0; i <= 360; i += 36;)
{
path_add_point(my_path, x + lengthdir_x(50, i), y + lengthdir_y(50, i), 100);
}

上述代码将在运行代码的实例的x/y位置周围创建一个圆形路径。
