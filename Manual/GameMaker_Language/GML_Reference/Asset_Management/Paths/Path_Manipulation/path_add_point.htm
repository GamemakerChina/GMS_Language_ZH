path_add_point

使用此功能，您可以将点添加到指定的路径并设置其速度系数。不论该点的位置如何，此点都将添加到路径的末尾，并且该系数等于该点的百分比 path_speed当以下实例到达路径中的这一点时，它实际到达的位置。如果希望将路径点放置在并非终点的其他位置，则应使用 path_insert_point()。

如果在从资源浏览器的路径上使用该路径，请注意，该功能将更改实际资产，因此从使用该功能起到游戏结束之间，将永久影响游戏中所有实例使用该路径的方式。如果这不是您所需要的，则应使用类似 path_duplicate()首先创建路径的副本，然后在重复的资产上调用此函数（请不要忘记调用 path_delete()不再需要该资产时）。

句法：

path_add_point(index, x, y, speed);

争论描述
index要添加点的路径的索引。
x新点的x坐标。
y新点的y坐标。
speed点的速度因数（默认值为100）。

返回值：

N/A

例子：

my_path = path_add();
for (var i = 0; i <= 360; i += 36;)
    {
    path_add_point(my_path, x + lengthdir_x(50, i), y + lengthdir_y(50, i), 100);
    }

上面的代码将在运行该代码的实例的x / y位置周围创建一个圆形路径。

返回：路径操纵
