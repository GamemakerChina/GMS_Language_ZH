path_append

使用此代码，您可以将一条路径附加到另一条路径上，有效地将它们连接在一起，并且要使此路径有效，必须先存在两条路径。通常，您可能希望在使用以下命令创建的路径上使用此功能 path_add()，因为如果您在路径资产上使用它，则从使用该功能到游戏结束，它将永久影响游戏中实例的路径。

注意：这不会删除从游戏追加的路径。它仍然在那里，只有它现在没有任何点，如果您不再希望使用它，则应使用path_delete（）函数将其删除。

句法：

path_append(index, path);

争论描述
index路径的新索引。
path将添加到“索引”的路径的索引。

返回值：

N/A

例子：

path_append(mypath, mppath);
path_delete(mppath);

上面的代码将在变量“ mppath”中索引的路径数据附加到在变量“ mypath”中索引的路径，然后从内存中删除“ mppath”。

返回：路径操纵
