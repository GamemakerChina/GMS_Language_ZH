path_assign

使用此功能，您可以将路径数据从一个路径复制到另一路径。复制到的路径将首先被清除（如果有任何路径点），并被复制的路径完全覆盖。在此过程中，不会删除任何一条路径，结果是两条路径，具有两个不同的索引，但是形式和点完全相同。通常，您可能希望在使用以下命令创建的路径上使用此功能 path_add()，因为如果您在路径资产上使用它，则从使用该功能到游戏结束，它将永久影响游戏中所有实例的路径。

句法：

path_assign(index, path);

争论描述
index要覆盖的路径的索引。
path将覆盖“索引”的路径的索引。

返回值：

N/A

例子：

mypath = path_add();
path_assign(mypath, choose(path_1, path_2, path_3));

上面的代码将在变量“ mypath”中创建一个新的空路径，然后从三个可用路径资源之一中复制路径数据。

返回：路径操纵
