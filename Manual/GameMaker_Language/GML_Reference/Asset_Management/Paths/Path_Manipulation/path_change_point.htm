path_change_point

使用此功能，您可以更改先前为路径定义的任何点的位置和/或速度因数（该路径可以在路径编辑器中创建，也可以通过使用 path_add()）。如果在从资源浏览器的路径上使用该路径，请注意，该功能将更改实际资产，因此从使用该功能起到游戏结束之间，将永久影响游戏中所有实例使用该路径的方式。如果这不是您所需要的，则应使用类似 path_duplicate()首先创建路径的副本，然后在重复的资产上调用此函数（请不要忘记调用 path_delete()不再需要该资产时）。

句法：

path_change_point(index, n, x, y, speed);

争论描述
index更改点的路径的索引。
n更改属性的定义点。
x点的新x坐标（相对于路径）。
y点的新y坐标（相对于路径）。
speed点的新速度因子。

返回值：

N/A

例子：

for (var i = 0; i < path_get_number(path0); ++i;)
    {
    var px = path_get_point_x(pth_Patrol, i) + 64 - random(128);
    var py = path_get_point_y(pth_Patrol, i) + 64 - random(128);
    path_change_point(pth_Patrol, i, px, py, 100);
    }

上面的代码循环遍历索引为“ path0”的路径中的所有点，并将所有点重新定位到128x128像素区域内的随机位置。

返回：路径操纵
