path_clear_points

使用此功能，您可以从路径中删除所有点定义，从而有效地创建“空”路径。这不会删除路径，因为您应该使用该路径 path_delete()，但是请注意，此功能会更改实际的路径资产，因此从使用该功能到游戏结束，将永久影响游戏中所有实例使用路径的方式。如果这不是您所需要的，则应使用类似 path_duplicate()首先创建路径的副本，然后在重复的资产上调用此函数（请不要忘记调用 path_delete()不再需要该资产时）。

句法：

path_clear_points(index);

争论描述
index要清除的路径的索引。

返回值：

N/A

例子：

if path_get_number(mypath) > 0
    {
    path_clear_points(mypath);
    }

上面的代码检查是否在变量“ mypath”中索引的路径上有任何点，如果有，它将清除路径。

返回：路径操纵
