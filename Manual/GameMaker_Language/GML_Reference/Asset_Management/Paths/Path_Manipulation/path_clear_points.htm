路径_清除点

通过这个函数，你可以从一个路径中删除所有的点定义，有效地形成一个 "空 "路径。这并不是删除路径，你应该使用 path_delete(), 但是需要注意的是，这个函数改变了实际的路径资产，所以会永久影响游戏中所有实例对路径的使用，从使用这个函数的那一刻起直到游戏结束。如果这 不是你所需要的，那么你应该使用一个类似于 path_duplicate() 的函数来先创建一个路径的副本，然后在复制的资产上调用这个函数（不要忘记在资产上调用 path_delete() ，当它不再需要时）。)

语法。

path_clear_points(index)。

争论 描述
指数 要清除的路径的索引。

返回。

不适用

例子。

如果path_get_number(mypath) > 0
{
path_clear_points(mypath)。
}

上述代码检查变量 "mypath "中索引的路径上是否有任何点，如果有，则清除路径。
