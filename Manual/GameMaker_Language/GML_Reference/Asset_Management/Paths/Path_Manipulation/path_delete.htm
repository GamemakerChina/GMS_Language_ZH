path_delete

您可以使用此代码从内存中删除路径。如果此路径是使用以下方法动态创建的 path_add()，包含路径索引的变量将不再有效，因为该路径已不存在，并且如果使用路径编辑器创建了该路径，则由于要永久删除该路径，因此无法在整个游戏中访问该路径。

句法：

path_delete(index);

争论描述
index要删除的路径的索引。

返回值：

N/A

例子：

var t_path = path_add();
if mp_grid_path(grid, t_path, x, y, obj_Player.x, obj_Player.y, 1)
    {
    path_assign(mypath, t_path);
    }
path_delete(t_path);

上面的代码将创建路径并将其索引存储在局部变量中。然后，该路径用于存储 mp_grid_path()生成的路径（如果成功找到目标路径），则将其分配给在“ mypath”中索引的路径。最后，“ t_path”被删除。

返回：路径操纵
