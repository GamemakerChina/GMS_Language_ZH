路径_delete_point

通过这个函数你可以从指定的路径中移除一个点。路径点从0开始编号，所以第一个点总是第0个点，最后一个路径点将是 path_get_number()- 1.如果你 调用这个函数，那么删除的点将是永久性的，所有分配给路径的实例在将来都会缺少被删除的点。如果这不是你所需要的，那么你应该使用 一个函数，比如 path_duplicate() ，先创建一个路径的副本，然后在复制的资产上调用这个函数（不要忘记在不再需要的时候调用 path_delete() ）。

语法。

path_delete_point(ind, n)。

争论 描述
指数 要删除一个点的路径的索引。
n 要删除的定义点。

返回。

不适用

例子。

var _col = true;
while (_col)
{
var _num = path_get_number(mypath) - 1;
if (_num > 2)
{
var _x1 = path_get_point_x(my_path, _num);
var _y1 = path_get_point_y(my_path, _num);
var _x2 = path_get_point_x(my_path, _num -1);
var _y2 = path_get_point_y(my_path, _num -1);
if collision_line(_x1, _y1, _x2, _y2, obj_Wall, true, false)
{
path_delete_point(mypath, _num);
_num -= 1;
}
else _col = false;
}
else _col = false;
}

上面的代码生成一个循环，从末端开始检查倒退的点，看是否与给定的对象发生碰撞。如果发现有碰撞，就删除一个路径点，直到没有碰撞，或者路径只有2个点。
