path_duplicate

此函数采用路径并将其复制到新路径。在该过程中将创建新路径，并返回索引以在使用此新路径的所有其他调用中使用。

句法：

path_duplicate(index);

争论描述
index要复制的现有路径的索引。

返回值：

Real

例子：

mypath = path_duplicate(choose(pth_1, pth_2, pth_3, pth_4));

上面的代码选择四个路径资源之一并将其复制，将新路径的索引存储在变量“ mypath”中。

返回：路径操纵
