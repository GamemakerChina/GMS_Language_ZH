path_flip

此功能获取所有路径点，并沿水平轴翻转它们。此功能会更改实际的路径资产，因此从使用该功能到游戏结束，将永久影响游戏中所有实例使用路径的方式。如果这不是您所需要的，则应使用像 path_duplicate()首先创建路径的副本，然后在重复的资产上调用此函数（请不要忘记调用 path_delete()不再需要该资产时）。

句法：

path_flip(index);

争论描述
index翻转路径的索引。

返回值：

N/A

例子：

path_flip(mypath);

这将沿水平轴翻转索引在变量“ mypath”中的路径。

返回：路径操纵
