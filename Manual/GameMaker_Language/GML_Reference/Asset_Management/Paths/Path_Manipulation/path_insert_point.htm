path_insert_point

使用此功能，您可以将新点插入路径（该路径可以在路径编辑器中创建，也可以通过使用 path_add()）。该点将被添加到函数中指定的点“ n”之前的路径中。

句法：

path_insert_point(index, n, x, y, speed);

争论描述
index要插入点的路径的索引。
n插入新点之前的定义点。
x新点的x坐标（相对于路径）。
y新点的y坐标（相对于路径）。
speed点的速度因数。

返回值：

N/A

例子：

path_insert_point(mypath, 0, 50, 50, 100);

这将以100％的速度因子在变量“ mypath”中索引的路径的最开始处插入一个点（50,50）。

返回：路径操纵
