路径_mirror

这个函数将所有的路径点沿纵轴进行镜像。这个函数改变了实际的路径资产，因此将永久地影响游戏中所有实例对路径的使用，从使用该函数的那一刻起直到游戏结束。 被使用，直到游戏结束。如果这不是你所需要的，那么你应该使用一个类似 path_duplicate() 的函数，先创建一个路径的副本，然后在 然后在复制的资产上调用这个函数（不要忘记在资产不再需要时调用 path_delete() ）。

语法。

path_mirror(index)。

争论 描述
指数 镜像路径的索引。

返回。

不适用

例子。

path_mirror(mypath)。

这将沿纵轴反映变量 "mypath "中索引的路径。
