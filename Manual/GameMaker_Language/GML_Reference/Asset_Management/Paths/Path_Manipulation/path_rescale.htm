路径_重新缩放

这个函数可以用来沿纵轴和横轴重新调整给定路径的比例，基本上是将每个路径点移动到路径中心周围与这个比例相应的新位置。这个函数改变了实际的路径资产。 所以会永久影响游戏中所有实例对路径的使用，从使用该函数的那一刻起直到游戏结束。如果这不是你所需要的，那么你应该使用一个类似 path_duplicate() 的函数，先创建一个路径的副本，然后在复制的资产上调用这个函数（不要忘记在不再需要的时候在资产上调用 path_delete() ）。

语法。

path_rescale(index, xscale, yscale)。

争论 描述
指数 要缩放的路径的索引。
xscale 当前的水平比例要乘以什么。默认比例是1。
yscale 当前的垂直比例要乘以什么。默认比例是1。

返回。

不适用

例子。

path_rescale(pth_AI, 3, 3)。

这将把给定的路径重新放大到原来的三倍。
