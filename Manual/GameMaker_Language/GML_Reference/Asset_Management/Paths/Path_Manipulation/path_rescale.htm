path_rescale

此功能可用于沿垂直轴和水平轴（或其中一个）重新缩放给定路径，基本上将每个路径点移动到围绕路径中心的与该比例对应的新位置。此功能会更改实际的路径资产，因此从使用该功能到游戏结束之间，将永久影响游戏中所有实例使用该路径的方式。如果这不是您所需要的，则应使用类似 path_duplicate()首先创建路径的副本，然后在重复的资产上调用此函数（请不要忘记调用 path_delete()不再需要该资产时）。

句法：

path_rescale(index, xscale, yscale);

争论描述
index缩放路径的索引。
xscale用什么乘以当前水平比例尺。默认比例为1。
yscale用什么乘以当前垂直比例尺。默认比例为1。

返回值：

N/A

例子：

path_rescale(pth_AI, 3, 3);

这会将给定路径重新缩放为其原始大小的三倍。
