path_reverse

使用此功能，您可以颠倒各个路径点的编号顺序，例如，如果路径有5个点，则点0将变为点4，点1将变为点3，点2不会更改。点的实际位置保持不变，仅更改了处理顺序。此功能会更改实际的路径资产，因此从使用该功能到游戏结束之间，将永久影响游戏中所有实例使用该路径的方式。如果这不是您所需要的，则应使用类似 path_duplicate()首先创建路径的副本，然后在重复的资产上调用此函数（请不要忘记调用 path_delete()不再需要该资产时）。

句法：

path_reverse(index);

争论描述
index更改路径的索引。

返回值：

N/A

例子：

path_reverse(pth_AI);

这将颠倒处理给定路径上所有点的顺序。

返回：路径操纵
