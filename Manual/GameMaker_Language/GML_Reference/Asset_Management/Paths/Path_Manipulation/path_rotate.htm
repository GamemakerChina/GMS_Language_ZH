path_rotate

您可以使用此功能绕其中心旋转给定路径。请记住，在GameMaker Studio 2 （除非您使用物理学），角度是逆时针计算的，因此将路径旋转90度会向左旋转路径。此功能会更改实际的路径资源，因此从使用该功能到游戏结束之间，将永久影响游戏中所有实例使用该路径的方式。如果这不是您所需要的，则应使用类似 path_duplicate()首先创建路径的副本，然后在重复的资产上调用此函数（请不要忘记调用 path_delete()不再需要该资产时）。

注意：使用path_start()功能，您可以使用 path_orientation变量也可以更改路径旋转。

句法：

path_rotate(index, angle);

争论描述
index翻转路径的索引。
angle旋转角度。

返回值：

N/A

例子：

path_rotate(pth_Patrol, 90);

这将逆时针旋转给定的90度。

返回：路径操纵
