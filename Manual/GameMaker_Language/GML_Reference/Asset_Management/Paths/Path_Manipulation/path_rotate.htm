路径_旋转

你可以用这个函数将一个给定的路径绕其中心旋转。记住，在GameMaker Studio 2中（除非你使用物理学），角度是逆时针计算的，所以将路径旋转90度会 将路径向左旋转。这个函数改变了实际的路径资源，因此将永久地影响游戏中所有实例对路径的使用，从使用该函数的那一刻起直到游戏结束。 游戏结束。如果这不是你所需要的，那么你应该使用一个类似 path_duplicate() 的函数，先创建一个路径的副本，然后在复制的资产上调用这个函数（不要 不要忘记在资产不再需要时调用 path_delete() ）。)

注意：一旦一个路径被分配给一个实例，使用 path_start()函数分配给一个实例后，你可以使用 path_orientation 变量 来改变路径的旋转。

语法。

path_rotate(index, angle)。

争论 描述
指数 要翻转的路径的索引。
角度 旋转的角度度。

返回。

不适用

例子。

path_rotate(pth_Patrol, 90);

这将使给定的东西逆时针旋转90度。
