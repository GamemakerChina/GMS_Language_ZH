path_set_closed

此功能可用于将给定路径标记为打开（假）或关闭（真）。闭合路径的起点连接到其终点，形成一个环路，而开放路径的起点和终点是确定的，未连接的。此功能会更改实际的路径资产，因此从使用该功能到游戏结束，将永久影响游戏中所有实例使用路径的方式。如果这不是您所需要的，则应使用像 path_duplicate()首先创建路径的副本，然后在重复的资产上调用此函数（请不要忘记调用 path_delete()不再需要该资产时）。

句法：

path_set_closed(index, closed);

争论描述
index更改路径的索引。
closed路径是否关闭（true）或否（false）。

返回值：

N/A

例子：

path_set_closed(pth_Patrol, true);

这会将给定路径设置为封闭路径。

返回：路径操纵
