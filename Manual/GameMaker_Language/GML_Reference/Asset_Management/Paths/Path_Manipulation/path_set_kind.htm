path_set_kind

此功能可用于设置希望指定资源为的路径类型。这可以是直线路径（设置为0），也可以是平滑路径（设置为1），在这种情况下，也必须考虑路径精度（也可以使用 path_set_precision()）。此功能会更改实际的路径资源，因此从使用该功能到游戏结束，将永久影响游戏中所有实例使用路径的方式。如果这不是您所需要的，则应使用像 path_duplicate()首先创建路径的副本，然后在重复的资产上调用此函数（请不要忘记调用 path_delete()不再需要该资产时）。

句法：

path_set_kind(index, val);

争论描述
index更改路径的索引。
val路径的种类，0表示直线，1表示平滑。

返回值：

N/A

例子：

path_set_kind(pth_Patrol, true);

这会将给定路径设置为“平滑”路径。

返回：路径操纵
