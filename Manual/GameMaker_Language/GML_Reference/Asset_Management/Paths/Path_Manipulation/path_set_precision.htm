path_set_precision

使用此功能，您可以更改路径的“平滑度”。该值必须在1到8之间，较低的值将创建直的边缘，且点之间的曲线更锐利，而较高的值将使点变圆并使路径更“弯曲”。请注意，如果未在路径编辑器中或使用该功能将路径设置为平滑，则该功能将无可见效果。path_set_kind() 。此功能会更改实际的路径资产，因此从使用该功能到游戏结束之间，将永久影响游戏中所有实例使用该路径的方式。如果这不是您所需要的，则应使用类似 path_duplicate()首先创建路径的副本，然后在重复的资产上调用此函数（请不要忘记调用 path_delete()不再需要该资产时）。

句法：

path_set_precision(index, prec);

争论描述
index更改路径的索引。
prec路径的精度。必须是1到8之间的整数。

返回值：

N/A

例子：

path_set_precision(pth_Patrol, 2);

这会将给定路径的精度设置为2。

返回：路径操纵
