路径_移位

通过这个函数，你可以沿水平和垂直轴移动一个路径。请注意，如果跟随被移动的路径的实例相对地（而不是绝对地）跟随它，这在游戏中不会有明显的效果，因为路径的 的位置会被忽略掉。这个函数改变了实际的路径资产，因此将永久地影响游戏中所有实例对路径的使用，从使用这个函数的那一刻起直到游戏结束。如果这不是 不是你所需要的，那么你应该使用一个类似 path_duplicate() 的函数，先创建一个路径的副本，然后在复制的资产上调用这个函数（不要忘记调用 path_delete()在资产不再需要的时候调用这个函数）。)

语法。

path_shift(index, xshift, yshift)。

争论 描述
指数 要改变的路径的索引。
变化 移動路徑的水平像素數。负数=左，正数=右。
yshift 移除路径的垂直像素数。负的=向上，正的=向下。

返回。

不适用

例子。

my_path = path_duplicate(pth_AI);
path_shift(my_path, 50, 50);
path_start(my_path, 4, path_action_reverse, true) 。

上面的代码复制了一个路径资产，并将其ID值存储在一个变量中。然后，这个变量被用来转移复制的路径位置，然后实例启动路径。
