path_shift

使用此功能，您可以沿水平和垂直轴移动路径。请注意，如果移动路径之后的实例相对（而不是绝对）跟随它，因为此时忽略了路径的位置，则这在游戏中不会产生可见效果。此功能会更改实际的路径资产，因此从使用该功能到游戏结束之间，将永久影响游戏中所有实例使用该路径的方式。如果这不是您所需要的，则应使用类似 path_duplicate()首先创建路径的副本，然后在重复的资产上调用此函数（不要再在不再需要时在资产上调用path_delete（））。

句法：

path_shift(index, xshift, yshift);

争论描述
index更改路径的索引。
xshift转移路径的水平像素数。负=左，正=右。
yshift要移动路径的垂直像素数。负=向上，正=向下。

返回值：

N/A

例子：

my_path = path_duplicate(pth_AI);
path_shift(my_path, 50, 50);
path_start(my_path, 4, path_action_reverse, true);

上面的代码复制了路径资产，并将其ID值存储在变量中。然后，此变量用于移动重复的路径位置，然后实例启动路径。

返回：路径操纵
