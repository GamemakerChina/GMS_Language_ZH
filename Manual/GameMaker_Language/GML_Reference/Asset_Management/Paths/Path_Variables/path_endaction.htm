path_endaction

当实例到达当前路径的末端时，该变量可用于获取或更改实例的反应。通常，使用 path_start()但您可能希望根据游戏中的任意事件来更改此行为。可用值使用以下常量表示：

持续的描述
path_action_stop停止前进
path_action_restart从起始位置继续，如果路径未闭合，则跳至起始位置
path_action_continue从当前位置重新开始路径
path_action_reverse反转路径的速度（反向运行路径）

句法：

path_endaction;

返回值：

Constant (see description)

例子：

if (path_endaction == path_action_stop)
    {
    path_endaction = path_action_reverse;
    }

上面的代码将检查路径结束动作，如果将其设置为停止，则结束动作将更改为反向。

返回：路径变量
