path_index

GameMaker Studio 2所有资源都有一个唯一的标识号，并且 path_index是一个只读变量，用于保存已使用分配给实例的给定路径资源的索引值。 path_start()功能。如果实例未分配路径，则变量将设置为-1。

句法：

path_index;

返回值：

integer (Path ID)

例子：

if (path_index == -1)
    {
    path_start(pth_enemy3, 4, path_action_reverse, 0);
    }

上面的代码检查是否已将路径分配给实例，如果没有分配，它将启动新路径（将其分配给实例）。path_index同时）。

返回：路径变量
