path_orientation

该变量保存该函数时已分配给实例的路径的当前方向 path_start()被称为。创建路径时，其方向默认为0度，但是您可以使用此值将此值设置为任何值。请记住，在GameMaker Studio 2 （除非您使用物理学），角度是逆时针计算的，因此将路径方向设置为90°会使路径向左旋转。

句法：

path_orientation;

返回值：

Real

例子：

mypath = path_duplicate(choose(path_1, path_2, path_3, path_4));
path_start(path, 4, path_action_reverse, 0);
path_orientation = 90;

上面的代码将随机的预制路径资源复制到变量“ mypath”中。然后开始新的路径并将其旋转90°。

返回：路径变量
