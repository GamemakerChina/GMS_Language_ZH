路径_方向

这个变量保存了调用函数 path_start() 时分配给该实例的路径的当前方向。创建路径时，其方向是默认的 0度，但你可以用这个变量将这个值设置为你想要的任何东西。记住，在GameMaker Studio 2中（除非你使用物理学），角度是逆时针计算的，所以将路径方向设置为 将会把路径向左旋转。

语法。

path_orientation;

返回。

真实的

例子。

mypath = path_duplicate(choose(path_1, path_2, path_3, path_4));
path_start(path, 4, path_action_reverse, 0);
path_orientation = 90;

上面的代码复制了一个随机的、预制的路径资产到变量 "mypath"。然后这个新的路径被启动并旋转90°。
