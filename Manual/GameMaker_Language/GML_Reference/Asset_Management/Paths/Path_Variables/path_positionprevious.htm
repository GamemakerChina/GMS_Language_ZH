path_positionprevious

在上一步中，该变量可用于获取或设置实例沿其当前路径的位置，并且是介于0和1之间的归一化值，即：0是路径的开始位置，而1是结束位置。它类似于 xprevious和 yprevious变量如何工作，仅特定于路径。它对某些事情很有用，例如在出现问题时暂时停止路径跟随器（请参见下面的示例代码）。

句法：

path_positionprevious;

返回值：

Real

例子：

var _x = x + lengthdir_x(16, direction);
var _y = y + lengthdir_y(16, direction);
if collision_circle(xx, yy, 16, obj_Player, false,true)
    {
    path_position = path_positionprevious;
    }

上面的代码检查实例前面的区域是否与对象“ obj_Player”发生冲突，如果存在冲突，则将实例设置回其在当前路径上的先前位置，该位置由函数path_start（）分配。

返回：路径变量
