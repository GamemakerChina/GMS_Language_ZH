path_get_kind

路径可以是平滑的，也可以是直线的（平滑路径会计算围绕定义点的弯曲路径，而直线路径只是从一个点到另一个点的直线）。此功能可用于确定给定路径是否平滑（true ） 或不 （false ）。

句法：

path_get_kind(index);

争论描述
index要检查的路径的索引。

返回值：

Boolean

例子：

path_kind = path_get_kind(pth_Patrol);

这会将“ path_kind”变量设置为 true或者 false给定路径索引取决于路径的类型。
