路径

路径可以是平滑的，也可以是笔直的（平滑的路径围绕定义点计算出一条弯曲的路径，而笔直的路径只是从一个点直达另一个点）。这个 函数可以用来找出给定的路径是否光滑（真）或不光滑（假）。

语法。

path_get_kind(index)。

争论 描述
指数 要检查的路径的索引。

返回。

布尔型

例子。

path_kind = path_get_kind(pth_Patrol)。

这将把 "path_kind "变量设置为true或false ，取决于给定的路径索引是哪种路径。

返回。路径
