路径的长度（path_get_length

你可以用这个函数得到一个路径的精确长度，单位是像素。这不是一个从点到点的近似长度，而是一个沿着路径形状的精确长度，即使路径是平滑的，具有 高的弯曲精度。

语法。

path_get_length(index)。

争论 描述
指数 要测量的路径的索引。

返回。

真实的

例子。

path_len = path_get_length(pth_AI)。

这将把 "path_len "设置为 "pth_AI "中索引的路径的总长度，单位为像素。

返回。路径
