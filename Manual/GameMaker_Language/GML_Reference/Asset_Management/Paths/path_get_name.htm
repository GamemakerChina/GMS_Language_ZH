path_get_name

这个函数将返回被引用的路径的名称，是一个字符串。这个名字是在编辑器中用来定义路径的，或者（如果路径是通过代码函数创建的），它将返回一个格式为" _newpathXX "的字符串，其中" _newpathXX "是生成路径的编号，从0开始递增。 _newpathXX"，其中 "XX"是生成路径的编号，从0开始，每创建一个新路径就递增一个。请注意，这只是一个 字符串，不能用来直接引用路径--为此，你需要路径索引ID。但是，你可以使用这个字符串来获取路径索引ID，使用返回的字符串和 和函数 asset_get_index().

语法。

path_get_name(index)。

争论 描述
指数 要检查的路径的索引。

返回。

字符串

例子。

path_name = path_get_name（path_array[0]）。

这将把 "path_name "设置为给定数组中位置为0的路径索引的名称。

返回。路径
