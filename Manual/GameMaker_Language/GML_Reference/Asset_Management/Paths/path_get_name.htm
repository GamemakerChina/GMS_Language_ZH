path_get_name

此函数将返回引用为字符串的路径的名称。该名称是用于在编辑器中定义路径的名称，或者（如果该路径是通过代码功能创建的）名称将返回格式为“ _newpathXX ”的字符串，其中“XX “是生成的路径号，从0开始，每次创建新路径时递增1。请注意，这只是一个字符串，不能用于直接引用路径-为此，您将需要路径索引ID ，但是，您可以使用此字符串使用返回的字符串和函数来获取路径索引ID。asset_get_index() 。

句法：

path_get_name(index);

争论描述
index要检查的路径的索引。

返回值：

String

例子：

path_name = path_get_name(path_array[0]);

这会将“ path_name”设置为在给定数组中位置0处索引的路径的名称。
