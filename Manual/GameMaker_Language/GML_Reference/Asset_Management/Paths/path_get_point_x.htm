path_get_point_x

这个函数将返回你为你索引的路径输入的点的X位置（房间坐标）。如果该点不在路径的范围内（例如：一个路径有8个点，而你要求的是10号点的X位置），那么将返回一个值 将会被返回0。

语法。

path_get_point_x(index, n)。

争论 描述
指数 要检查的路径的索引。
n 点的身份来检查。

返回。

真实的

例子。

如果(path_position == 1)
{
var _num = path_get_number(pth_Patrol);
var _pos = floor(random(_num));
x = path_get_point_x(pth_Patrol, _pos);
y = path_get_point_y(pth_Patrol, _pos) ;
path_position = (1 / _num) * _pos;
}

上面的代码将检查一个实例是否在路径的末端。如果是的话，它将在路径上随机选择一个点，并将实例移动到该点。

返回。路径
