path_get_point_y

此函数将返回您为索引路径输入的点的y位置（在房间坐标中）。如果该点在路径范围之外（即：路径有8个点，并且您要求点10的y位置），则将返回值0。

句法：

path_get_point_y(index, n);

争论描述
index要检查的路径的索引。
n要检查的点身份。

返回值：

Real

例子：

if (path_position == 1)
    {
    var _num = path_get_number(pth_Patrol);
    var _pos = floor(random(_num));
    x = path_get_point_x(pth_Patrol, _pos);
    y = path_get_point_y(pth_Patrol, _pos);
    path_position = (1 / _num) * _pos;
    }

上面的代码将检查实例是否在路径的末尾。如果是，它将在路径上选择一个随机点并将实例移动到该点。
