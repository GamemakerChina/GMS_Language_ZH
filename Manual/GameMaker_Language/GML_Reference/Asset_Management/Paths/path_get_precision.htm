path_get_precision

该函数以给定路径“平滑”的精度返回，并且将是1到8之间的整数值。尽管您可以获取（并设置）直线路径的该值，但它不会对实例使用该路径，因为仅当路径种类设置为“平滑”时才相关。

句法：

path_get_precision(index);

争论描述
index要检查的路径的索引。

返回值：

Real

例子：

if (path_get_kind(pth_Patrol))
    {
    if (path_get_precision(pth_Patrol)) != 8
        {
        path_set_precision(pth_Patrol, 8);
        }
    }

上面的代码检查在“ pth_Patrol”中索引的路径以查看其类型，如果平滑，则检查平滑的精度。如果小于8，则将其设置为8。
