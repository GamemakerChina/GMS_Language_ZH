path_get_speed

此函数返回路径上任何给定位置的速度因数。该位置应为0到1之间的值（您可以使用 path_position，例如，如果您需要实例的当前位置），则0为开始位置，1为结束位置。返回的值将是实际 path_speed 正在设置。例如，在只有两个点0和1的路径上，其中点0速度因数为100，点1速度因数为0，该函数 path_get_speed(path, 0.5)将返回50。

句法：

path_get_speed(ind, pos);

争论描述
index要检查的路径的索引。
pos通过路径检查多远。在0（开始）和1（结束）之间，可以是十进制。

返回值：

Real

例子：

speed = path_get_speed(pth_Patrol, 0.5) / 20;

这会将调用实例的速度设置为该点的速度因子，该点恰好位于给定路径的一半处，除以20。因此，如果找到的速度因子是默认值100，它将调用实例的速度设置为5。
