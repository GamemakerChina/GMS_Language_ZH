path_get_x

使用此功能，您可以获取任何给定路径上位置的x坐标。 0是路径的起点，1是路径的终点，介于两者之间的所有内容都等于穿过路径的那一段。这不必是路径的定义点，它可以在路径上的任何位置。

句法：

path_get_x(ind, pos);

争论描述
index要检查的路径的索引。
pos通过路径检查多远。在0（开始）和1（结束）之间。

返回值：

Real

例子：

x = path_get_x(pth_WallWalk, 0.5);

这会将调用实例的x设置为该点的x坐标，该点恰好位于给定路径的中途。
