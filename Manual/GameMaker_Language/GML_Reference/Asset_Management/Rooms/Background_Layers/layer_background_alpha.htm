layer_background_alpha

此功能控制背景精灵的Alpha（透明度）。您提供背景元素ID（使用 layer_background_create()或使用该功能时 layer_background_get_id()），然后设置要使用的alpha值。 Alpha可以在0（完全透明）到1（完全不透明）之间，背景元素的默认alpha值为1。请注意，如果背景元素分配给的图层不可见-或元素本身已经制成不可见-在再次使图层或元素可见之前，此功能不会有任何区别。

句法：

layer_background_alpha(background_element_id, alpha);

争论描述
background_element_id要更改的背景元素的唯一ID值
alpha背景精灵的Alpha，从0到1（默认值为1）

返回值：

N/A

例子：

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
layer_background_alpha(back_id, random(1));

上面的代码获取分配给“ Background_sky”层的背景的ID值，然后将其alpha设置为介于0和1之间的随机值。
