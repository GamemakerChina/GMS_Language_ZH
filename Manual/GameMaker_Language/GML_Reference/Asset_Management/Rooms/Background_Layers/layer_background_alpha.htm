层_背景_alpha

这个函数控制背景精灵的alpha（透明度）。你给背景元素的ID（当你使用 layer_background_create()或当 函数的时候得到的），然后设置阿尔法值。 layer_background_get_id()时得到的），然后设置要使用的alpha值。α值可以在0（完全透明）和1（完全不透明）之间，背景元素的默认α值是1。 元素的默认alpha值为1。请注意，如果背景元素被分配到的图层不可见--或者元素本身已经变得不可见--你将不会看到这个函数的任何区别，直到该图层或元素再次变得可见。 再一次。

语法。

layer_background_alpha(background_element_id, alpha)。

争论 描述
背景_元素_id 要改变的背景元素的唯一ID值
阿尔法 背景精灵的alpha，从0到1（默认为1）。

返回。

不适用

例子。

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
layer_background_alpha(back_id, random(1)) 。

上面的代码得到了分配给图层 "Background_sky "的背景的ID值，然后将其alpha值设置为0到1之间的随机值。
