layer_background_create

使用此功能，您可以将精灵资源分配给图层，以用作项目中的背景。您提供图层ID（使用 layer_create()）或图层名称（作为字符串-会对性能产生影响）和子画面索引（这是该子画面的名称，如资源浏览器中所示），并将其添加到图层中。该函数返回元素的唯一ID值，然后可将其用于背景的其他图层函数中。

句法：

layer_background_create(layer_id, sprite)

争论描述
layer_id要定位的图层的唯一ID值（或图层名称为字符串）
sprite要使用的精灵索引

返回值：

Background element ID (Real)

例子：

global.back_layer = layer_create(10000);
global.back_trees = layer_background_create(global.back_layer, spr_Trees);

上面的代码创建一个新层，然后向其添加一个新的背景元素，并将一个精灵设置为所使用的背景图像。
