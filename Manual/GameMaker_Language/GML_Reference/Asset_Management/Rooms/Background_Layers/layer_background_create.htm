创建图层_背景

通过这个函数，你可以把一个精灵资源分配给一个图层，在你的项目中作为背景使用。你提供图层的ID（当你使用 layer_create()) 或图层名称（作为一个字符串--这将对性能产生影响）和一个精灵索引（这将是资产浏览器中显示的精灵的名称），它将被添加到图层中。该函数返回该元素的唯一ID值。 然后可以在进一步的图层函数中用于背景。

语法。

layer_background_create(layer_id, sprite)。

争论 描述
layer_id 目标层的唯一ID值（或作为字符串的层名）。
精灵 要使用的精灵索引

返回。

背景元素ID（真实）。

例子。

global.back_layer = layer_create(10000);
global.back_trees = layer_background_create（global.back_layer, spr_Trees）。

上面的代码创建了一个新的图层，然后给它添加了一个新的背景元素，设置了一个精灵作为使用的背景图片。
