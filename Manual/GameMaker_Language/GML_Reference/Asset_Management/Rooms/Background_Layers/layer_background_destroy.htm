layer_background_destroy

此功能将破坏给定的背景元素。您提供背景ID（使用以下代码创建背景ID layer_background_create()或与 layer_get_background_id()），则将其删除。请注意，这并不从中取出层，只有背景，如果背景是一个已经在房间编辑器被添加，那么下一次你离开房间，然后返回，背景将再次重现。但是，如果房间是持久性的，除非再次关闭房间持久性，否则背景将被删除。

句法：

layer_background_destroy(background_element_id)

争论描述
background_element_id要销毁的背景的唯一ID值

返回值：

N/A

例子：

var lay_id = layer_get_id("Background_trees");
var bck_id = layer_background_get_id("Forrest");
if layer_background_exists(lay_id, bck_id)
    {
    layer_background_destroy(bck_id);
    }

上面的代码检查“ Background_trees”层，以查看给定的背景元素是否存在，如果存在，则将其销毁（但不破坏该层）。
