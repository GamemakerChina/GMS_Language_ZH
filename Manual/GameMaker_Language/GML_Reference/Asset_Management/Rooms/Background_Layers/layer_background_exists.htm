layer_background_exists

您可以使用此功能来检查并查看任何给定图层上是否存在背景元素。您提供图层ID（使用 layer_create()）或图层名称（作为字符串-这将对性能产生影响）和背景元素ID（使用以下代码创建背景元素时会获得该ID）layer_background_create()或使用该功能时 layer_background_get_id()），该函数将返回 true如果元素存在，或者 false如果没有。

注意：此函数在当前目标房间的范围内起作用-默认情况下，该函数所在的房间-可以使用函数layer_set_target_room（）进行设置。

句法：

layer_background_exists(layer_id, background_element_id)

争论描述
layer_id要定位的图层的唯一ID值（或图层名称为字符串）
background_element_id要检查的背景元素的唯一ID值

返回值：

Boolean

例子：

var lay_id = layer_get_id("Background_trees");
if layer_background_exists(lay_id, global.TreesBackground)
    {
    layer_background_destroy(lay_id, global.TreesBackground);
    }

上面的代码检查“ Background_trees”层，以查看给定的背景元素是否存在，如果存在，则将其销毁（但不破坏该层）。
