层_背景_存在

你可以使用这个函数来检查并查看任何给定的图层上是否存在一个背景元素。你可以提供图层的ID（你在创建图层时使用 layer_create())或 图层名称（作为一个字符串--这将对性能有影响）和背景元素ID（当你使用 layer_background_create()或当你使用 函数 layer_background_get_id())，如果该元素存在，该函数将返回true，如果不存在，则返回false。

注意：该函数在当前目标房间的范围内工作--默认为调用该函数的房间--可以使用函数设置 layer_set_target_room().

语法。

layer_background_exists(layer_id, background_element_id)

争论 描述
layer_id 目标层的唯一ID值（或作为字符串的层名）。
背景_元素_id 要检查的背景元素的唯一ID值

返回。

布尔型

例子。

var lay_id = layer_get_id("Background_trees");
if layer_background_exists(lay_id, global.TreesBackground)
{
layer_background_destroy(lay_id, global.TreesBackground);
}

上面的代码检查图层 "Background_trees"，看给定的背景元素是否存在，如果存在，那么它就被销毁（但不是图层）。
