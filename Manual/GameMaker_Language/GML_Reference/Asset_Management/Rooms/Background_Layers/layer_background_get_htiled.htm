layer_background_get_htiled

此功能可用于确定背景元素是否水平平铺。您提供背景元素ID（使用 layer_background_create()或使用该功能时 layer_background_get_id()），该函数将返回 true如果元素当前是平铺的，或者 false如果不是这样。

句法：

layer_background_get_htiled(background_element_id);

争论描述
background_element_id从中获取信息的背景元素的唯一ID值

返回值：

Boolean

例子：

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
if !layer_background_get_htiled(back_id)
    {
    layer_background_htiled(back_id, true);
    }

上面的代码将获取名为“ Background_sky”的图层的图层ID，然后使用该ID获取该图层上背景元素的ID。然后使用该ID来检查元素是否在水平轴上平铺，如果不是，则将其设置为水平。

背面：背景图层
