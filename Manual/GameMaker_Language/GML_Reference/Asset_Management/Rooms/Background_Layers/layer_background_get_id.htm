layer_background_get_id

此功能可用于检索图层上背景元素的唯一ID值。您提供图层ID（将图层名称与 layer_get_id()），该函数将返回与图层上的背景元素关联的ID值。请注意，此功能是专门为与IDE中添加的背景一起使用而设计的，就好像您使用该功能通过代码将背景添加到图层中一样 layer_background_create()，那么它将返回添加的背景元素的唯一ID。

句法：

layer_background_get_id(layer_id)

争论描述
layer_id要定位的图层的唯一ID值

返回值：

Background element ID value (Real)

例子：

var lay_id = layer_get_id("Background_trees");
var back_id = layer_background_get_id(lay_id);
layer_background_sprite(back_id, bck_Trees_Winter);

上面的代码将获取名为“ Background_trees”的图层的图层ID，然后使用该ID获取该图层上背景元素的ID。然后使用该ID更改元素精灵。

背面：背景图层
