层_背景_获取速度

这个函数可以用来获取背景元素的当前速度倍率值。你给背景元素的ID（当你用 "创建 "背景元素时，你会得到这个ID。 layer_background_create() 或使用函数 layer_background_get_id()时得到的），该函数将返回代表用于动画化精灵的速度乘数的实际值。默认值是1。

语法。

layer_background_get_speed(background_element_id)。

争论 描述
背景_元素_id 获取信息的背景元素的唯一ID值。

返回。

真实的

例子。

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
if layer_background_get_speed(back_id) > 0
{
layer_background_speed(back_id, 0);
}

上面的代码将获得名为 "Background_sky "的层的ID，然后用它来获得该层上的背景元素的ID。这个ID然后被用来检查该元素的动画速度，如果它大于 0，它就被设置为0。
