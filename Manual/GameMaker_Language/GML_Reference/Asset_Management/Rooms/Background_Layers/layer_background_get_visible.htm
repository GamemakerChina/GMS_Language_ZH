layer_background_get_visible

此功能可用于获取背景元素的可见状态。您提供背景元素ID（使用 layer_background_create()或使用该功能时 layer_background_get_id()），该函数将返回 true如果该元素当前可见，或者 false如果不是这样。请注意，这个返回值不受该元素是在层是否可见或不可见。

句法：

layer_background_get_visible(background_element_id);

争论描述
background_element_id从中获取信息的背景元素的唯一ID值

返回值：

Boolean

例子：

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
if layer_background_get_visible(back_id)
    {
    layer_background_visible(back_id, false);
    }

上面的代码将获取名为“ Background_sky”的图层的图层ID，然后使用该ID获取该图层上背景元素的ID。然后，此ID用于检查元素的可见性，如果可见，则将此属性设置为 false。
