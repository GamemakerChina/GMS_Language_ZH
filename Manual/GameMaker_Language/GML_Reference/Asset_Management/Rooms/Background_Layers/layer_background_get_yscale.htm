layer_background_get_yscale

此函数可用于获取背景元素的当前比例乘数。您提供背景元素ID（使用 layer_background_create()或使用该功能时 layer_background_get_id()），该函数将返回表示用于绘制子画面的比例乘数的实数值。预设值为1。

句法：

layer_background_get_yscale(background_element_id);

争论描述
background_element_id从中获取信息的背景元素的唯一ID值

返回值：

Real

例子：

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
if layer_background_get_yscale(back_id) != -1
    {
    layer_background_yscale(back_id, -1);
    }

上面的代码将获取名为“ Background_sky”的图层的图层ID，然后使用该ID获取该图层上背景元素的ID。然后，此ID用于检查元素的y比例尺，如果它不是-1，则将其设置为-1。
