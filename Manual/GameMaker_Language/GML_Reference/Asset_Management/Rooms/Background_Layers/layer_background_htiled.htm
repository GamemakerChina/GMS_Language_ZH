layer_background_htiled

使用此功能，您可以更改是否应水平平铺图层上给定的背景元素。您提供背景元素ID（使用 layer_background_create()或使用该功能时 layer_background_get_id()），然后设置平铺值。如果设置为 true那么该元素将在整个房间中水平平铺显示，当设置为 false，该元素将不会平铺。此功能仅适用于2D项目，使用3D相机投影时将无法正常使用。

句法：

layer_background_htiled(background_element_id, tiled);

争论描述
background_element_id要更改的背景元素的唯一ID值
visible水平拼贴切换，可以是 true或者 false

返回值：

N/A

例子：

var back = layer_background_get_id(layer);
if !layer_background_get_htiled(back) || !layer_background_get_vtiled(back)
    {
    layer_background_htiled(back, true);
    layer_background_vtiled(back, true);
    }

上面的代码将检查分配给运行该代码的实例所在的图层的背景元素，如果未在任一方向上进行平铺，则会将其设置为垂直和水平平铺。
