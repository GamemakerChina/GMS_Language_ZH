层_背景_索引

这个函数可以用来设置有多个子图像的背景精灵的图像索引。你给背景元素的ID（当你用 "创建 "背景元素时，你会得到这个ID。 layer_background_create() 或当你使用函数 layer_background_get_id()时得到的），然后设置要使用的图像索引。如果你设置了一个超出子图像范围的值，那么图像索引将循环往复。

语法。

layer_background_index(background_element_id, image_index)。

争论 描述
背景_元素_id 要改变的背景元素的唯一ID值
指数 用作背景的图像索引

返回。

不适用

例子。

var lay_id = layer_get_id（"Background_trees"）;
var back_id = layer_background_get_id（lay_id）;
layer_background_index（back_id, 1）。

上面的代码将得到名为 "Background_trees "的层的ID，然后用它来得到该层上的背景元素的ID。这个ID然后被用来改变元素图像的索引。
