图层_背景_速度

这个函数可以用来设置一个被分配给背景元素的动画精灵的速度。你给背景元素的ID（你在用 "创建背景元素 "或 "使用该函数 "时得到）。 layer_background_create() 或当你使用函数 layer_background_get_id()时得到的），然后设置动画速度。如果IDE将精灵的动画速度设定为每游戏帧计算，那么 给出的速度值是一个乘数，1是默认值，将其设置为0.5将使动画速度减半，而设置为2将使其加倍。然而，如果IDE将用于计算精灵的动画速度设置为 为每秒帧数，那么给出的速度值就是每秒帧数的字面值，因此将其设置为5，例如，将设置动画在游戏运行的每一秒钟循环5帧。如果背景元素没有附加精灵，或者 使用的精灵没有子图像，这个函数就没有效果。

语法。

layer_background_speed(background_element_id, speed)。

争论 描述
背景_元素_id 要改变的背景元素的唯一ID值
速度 对元素使用的速度乘数/值

返回。

不适用

例子。

var lay_id = layer_get_id("Background_water");
var back_id = layer_background_get_id(lay_id);
layer_background_speed(back_id, 0.5);

上面的代码将得到名为 "Background_water "的层的ID，然后用它来得到该层上的背景元素的ID。这个ID然后被用来改变元素的图像速度。
