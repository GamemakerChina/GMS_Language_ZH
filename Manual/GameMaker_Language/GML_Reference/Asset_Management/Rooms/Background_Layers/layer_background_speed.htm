layer_background_speed

此功能可用于设置已分配给背景元素的动画小精灵的速度。您提供背景元素ID（使用 layer_background_create()或使用该功能时 layer_background_get_id()），然后设置动画速度。如果IDE已将用于计算的精灵的动画速度设置为“每游戏帧数”，则给定的速度值是一个乘数，默认值为1，将其设置为0.5将是设置动画速度的一半。它到2会使它加倍。但是，如果IDE已将用于计算的精灵的动画速度设置为“每秒帧数”，则给定的速度值是每秒的文字帧值，因此将其设置为5，例如，会将动画设置为循环播放游戏每秒运行5帧。如果背景元素没有附加的精灵或使用的精灵没有子图像，则此功能无效。

句法：

layer_background_speed(background_element_id, speed);

争论描述
background_element_id要更改的背景元素的唯一ID值
speed用于元素的速度乘数/值

返回值：

N/A

例子：

var lay_id = layer_get_id("Background_water");
var back_id = layer_background_get_id(lay_id);
layer_background_speed(back_id, 0.5);

上面的代码将获取名为“ Background_water”的图层的图层ID，然后使用该ID获取该图层上背景元素的ID。然后使用该ID更改元素图像速度。

背面：背景图层
