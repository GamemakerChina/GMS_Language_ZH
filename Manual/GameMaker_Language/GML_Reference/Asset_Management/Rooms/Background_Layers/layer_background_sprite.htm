layer_background_sprite

使用此功能，您可以设置背景元素的精灵索引。您提供背景元素ID（使用 layer_background_create()或使用该功能时 layer_background_get_id()），然后指定要使用的Sprite索引。背景元素图像将替换为新的精灵。如果您将值设为-1，则该元素将没有分配任何图片（但仍将存在，并且可以在以后再次分配一份图片）。

句法：

layer_background_sprite(background_element_id, sprite_index)

争论描述
background_element_id要更改的背景元素的唯一ID值
sprite_index用于背景元素的精灵的精灵索引

返回值：

N/A

例子：

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
if layer_background_get_sprite(back_id) != spr_Clouds
    {
    layer_background_sprite(back_id, spr_Clouds);
    }

上面的代码将获取名为“ Background_sky”的图层的图层ID，然后使用该ID获取该图层上背景元素的ID。然后使用此ID检查分配给元素的精灵，如果尚未将其设置为精灵“ spr_Clouds”。
