层_背景_斯普利特

使用这个函数你可以设置背景元素的精灵索引。你要提供背景元素的ID（当你用以下方法创建一个背景元素时，你会得到 layer_background_create()或 时得到的），然后给一个sprite index。 layer_background_get_id()时得到的），然后给一个要使用的精灵索引。背景元素的图像将被替换成新的精灵。如果你给了一个-1的值，该元素 将没有被分配精灵（但仍然存在，以后可以再分配一个精灵）。

语法。

layer_background_sprite(background_element_id, sprite_index)。

争论 描述
背景_元素_id 要改变的背景元素的唯一ID值
魉_指数 用作背景元素的精灵的精灵索引

返回。

不适用

例子。

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
if layer_background_get_sprite(back_id) != spr_Clouds
{
layer_background_sprite(back_id, spr_Clouds);
}

上面的代码将获得名为 "Background_sky "的图层的ID，然后用它来获得该图层上背景元素的ID。然后，这个ID被用来检查分配给该元素的精灵，如果不是的话，就将其设置为精灵 "spr_Clouds" 如果它还不是的话。
