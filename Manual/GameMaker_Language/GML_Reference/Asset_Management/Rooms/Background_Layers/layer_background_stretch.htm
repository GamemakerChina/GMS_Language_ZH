层_背景_拉伸

使用这个函数，你可以切换背景元素的精灵，使其伸展以适应房间，或保持1:1的分辨率。你可以提供背景元素的ID（当你用 "D "创建一个背景元素时，你会得到这个ID。 layer_background_create() 或使用函数 layer_background_get_id())，然后将拉伸参数设置为真或假。当设置为真时，元素精灵将被拉伸以适应 当设置为 "true "时，元素精灵将被拉伸以适应房间（根据精灵和房间的尺寸放大或缩小），而当设置为"false"时，元素精灵将以其原始分辨率显示。

语法。

layer_background_stretch(background_element_id, stretch)

争论 描述
背景_元素_id 要改变的背景元素的唯一ID值
伸展 拉伸切换，可以是真或假

返回。

不适用

例子。

if mouse_check_button_pressed(mb_left)
{
var back = layer_background_get_id(layer);
if layer_background_get_stretch(back)
{
layer_background_stretch(back, false);
}
else
{
layer_background_stretch(back, true);
}
}

上面的代码检查鼠标按键是否被按下，如果检测到，它将切换分配给运行该代码的实例所在层的背景元素精灵的拉伸。
