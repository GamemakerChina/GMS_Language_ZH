layer_background_stretch

使用此功能，您可以切换背景元素精灵以拉伸以适合房间或保持分辨率为1：1。您提供背景元素ID（使用 layer_background_create()或使用该功能时 layer_background_get_id()），然后将Stretch参数设置为 true或者 false。设置为 true元素精灵将被拉伸以适合房间（根据精灵和房间的大小放大或缩小），并在设置为时 false，元素精灵将以其原始分辨率显示。

句法：

layer_background_stretch(background_element_id, stretch)

争论描述
background_element_id要更改的背景元素的唯一ID值
stretch拉伸切换，可以是 true或者 false

返回值：

N/A

例子：

if mouse_check_button_pressed(mb_left)
    {
    var back = layer_background_get_id(layer);
    if layer_background_get_stretch(back)
        {
        layer_background_stretch(back, false);
        }
    else
        {
        layer_background_stretch(back, true);
        }
    }

上面的代码检查是否按下了鼠标按钮，如果检测到按下鼠标，则将切换分配给运行代码的实例所在的图层的背景元素精灵的拉伸。
