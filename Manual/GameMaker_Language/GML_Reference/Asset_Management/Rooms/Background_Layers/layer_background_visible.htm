层_背景_可见

使用这个函数你可以切换一个背景元素的可见性。你提供背景元素的ID（当你使用 layer_background_create()或当 函数的时候得到的），然后设置可见参数。 layer_background_get_id())，然后将visible参数设置为true或false。当设置为true时，该元素将被显示，而当设置为false时。 元素将不被显示。注意，这取决于层的可见性，即使背景元素被标记为可见，如果它所在的层被标记为不可见，它也不会被绘制。

语法。

layer_background_visible(background_element_id, visible)

争论 描述
背景_元素_id 要改变的背景元素的唯一ID值
可见 可见性切换，可以是真或假

返回。

不适用

例子。

if mouse_check_button_pressed(mb_left)
{
var back = layer_background_get_id(layer);
if layer_background_get_visible(back)
{
layer_background_visible(back, false);
}
else
{
layer_background_visible(back, true);
}
}

上面的代码检查鼠标按键是否被按下，如果检测到，它将切换分配给运行该代码的实例所在层的背景元素的可见性。
