layer_background_visible

使用此功能可以切换背景元素的可见性。您提供背景元素ID（使用 layer_background_create()或使用该功能时 layer_background_get_id()），然后将visible参数设置为 true或者 false。设置为 true将显示该元素，并将其设置为 false，则不会显示该元素。请注意，这取决于图层的可见性，即使将背景元素标记为可见，但将其所在的图层标记为不可见也不会绘制。

句法：

layer_background_visible(background_element_id, visible)

争论描述
background_element_id要更改的背景元素的唯一ID值
visible可见性切换，可以是 true或者 false

返回值：

N/A

例子：

if mouse_check_button_pressed(mb_left)
    {
    var back = layer_background_get_id(layer);
    if layer_background_get_visible(back)
        {
        layer_background_visible(back, false);
        }
    else
        {
        layer_background_visible(back, true);
        }
    }

上面的代码检查是否有鼠标按下，如果检测到按下鼠标，则会切换分配给运行代码的实例所在的图层的背景元素的背景可见性。
