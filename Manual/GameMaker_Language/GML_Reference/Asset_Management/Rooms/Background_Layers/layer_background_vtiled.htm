层_后台_Vtiled

使用这个函数，你可以改变一个图层上给定的背景元素是否应该被垂直平铺。你给背景元素的ID（当你用 "创建一个背景元素 "时，你会得到这个ID。 layer_background_create() 或当你使用函数 layer_background_get_id()得到的），然后设置平铺值。如果设置为"true"，那么该元素将在房间里垂直平铺，而当设置为 "false"时。 元素将不会被平铺。这个函数只适用于2D项目，当使用3D摄像机投影时，将不能正常工作。

语法。

layer_background_vtiled(background_element_id, tiled)。

争论 描述
背景_元素_id 要改变的背景元素的唯一ID值
可见 垂直平铺的切换，可以是真或假。

返回。

不适用

例子。

var back = layer_background_get_id(layer);
if !layer_background_get_htiled(back) || !layer_background_get_vtiled(back)
{
layer_background_htiled（back, true）;
layer_background_vtiled（back, true）;
}

上述代码将检查分配给运行代码的实例所在层的背景元素，如果它没有在任何一个方向上平铺，它将把它设置为垂直和水平平铺。
