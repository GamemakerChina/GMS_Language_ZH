layer_add_instance

此功能可用于将给定实例从当前所在的层移动到另一层。您提供图层ID（使用 layer_create()）或图层名称（作为字符串-这将对性能产生影响）以及在图层之间移动的实例的实例ID。例如，假设您的播放器位于比另一个图层更深的图层上，并且您希望它出现在两者之间的图层后面。您可以使用此功能将播放器从前景层切换到背景层，并使它显示在要绘制的其他层的后面。

注意：此功能不能用于将新实例添加到图层。您必须先前已经创建了实例，并将其ID存储到变量中。

注意：此功能仅在当前房间的范围内有效，并且在使用该功能更改范围后不能使用layer_set_target_room()。

句法：

layer_add_instance(layer_id, instance_id)

争论描述
layer_id要定位的实例图层的唯一ID值（或图层名称作为字符串）
instance_id实例的唯一实例ID值，以移至目标层

返回值：

N/A

例子：

var near = instance_nearest(x, y, obj_Tree);
var layer_id = layer_get_id("Instances Front");
layer_add_instance(layer_id, near);

上面的代码将首先获取到给定x / y位置的最近实例的索引，并将其存储在局部变量中。然后，它为名为“ Instances Front”的层获取唯一的实例层ID，并将找到的实例移到该层。
