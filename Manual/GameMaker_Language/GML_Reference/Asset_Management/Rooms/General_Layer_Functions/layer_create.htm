创建层

这个函数用于在当前房间内创建一个新层。你提供新层所占的深度，该函数返回唯一的层ID值，然后可以用来在进一步的函数中访问该层。你还可以 提供一个可选的 "名称 "参数，这将使你能够给该层一个特定的名称（作为一个字符串），你可以使用这个名称字符串来代替使用层ID来访问该层，但应该注意的是，使用 名称字符串会比只使用ID值对性能产生更大的影响。如果你不指定图层的名称，那么创建的图层将被分配一个格式为"_layer_XXX "的名称，其中 "XXX" 是一个十六进制值，用于给图层一个唯一的名字。

当给定一个深度值时，该值离摄像机越 "远 "就越高，离摄像机越 "近 "就越低，所以如果你有三个深度为-100、0、100的层，这些层将按以下顺序绘制 100, 0, -100，所以 "顶层"（即最接近摄像机视图，所以画在其他东西上面）将是深度为-100的那一层。

下图显示了深度对层的作用示意图。

值得注意的是，虽然你可以给一个深度指定一个以上的图层，但你没有办法保证同一深度的图层会以任何给定的顺序被绘制，因此最好只给任何给定深度指定一个图层。还要注意的是，最小和最大的图层深度是-16000到16000，超出这些深度的东西将不会被渲染。如果你需要一个超出这个范围的深度 那么你将需要使用函数 layer_force_draw_depth().

语法。

layer_create(depth, [name])。

争论 描述
深度 该层必须创建的深度。
名称 给予该层的名称（一个字符串--可选）。

返回。

图层ID（真实）。

例子。

global.Bullet_Layer = layer_create(-100);

上述代码将创建一个深度值为-100的新图层，并将其ID值分配给一个全局变量。
