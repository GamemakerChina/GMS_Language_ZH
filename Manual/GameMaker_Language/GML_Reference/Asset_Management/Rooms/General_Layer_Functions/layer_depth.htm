layer_depth

此功能可用于更改给定图层的深度，更改将其内容呈现到屏幕的顺序。您提供图层ID（使用 layer_create()）或图层名称（作为字符串-这将对性能产生影响），然后提供新的深度以将其设置为（从-16000到16000的整数值）。图层深度的定义是：距离相机“越远”，图层深度越高；如果距离相机“越近”，则图层深度越低，因此，如果您具有三个深度为-100、0、100的图层，则图层的绘制顺序为100， 0，-100，这样“顶层”图层（即最接近相机视图并在其他所有图层上绘制的图层）将是-100深度的图层。

下图显示了深度如何用于图层的示意图：

请注意，您可以使用函数随时检查图层的深度 layer_get_depth()。另请注意，最小和最大图层深度为-16000到16000，并且不会渲染这些深度之外的任何内容。如果您需要的深度不在该范围内，则需要使用该功能 layer_force_draw_depth()。

句法：

layer_depth(layer_id, depth)

争论描述
layer_id设置深度的图层的唯一ID值（或图层名称为字符串）
depth图层的新深度

返回值：

N/A

例子：

if layer_get_depth(layer) != -100
   {
   layer_depth(layer, -100);
   }

上面的代码获取运行代码的实例所在的图层的深度，如果不是-100，则将深度设置为-100。
