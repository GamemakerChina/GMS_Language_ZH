layer_destroy

此功能将破坏给定的层。您提供图层ID（使用 layer_create()）或图层名称（作为字符串-会对性能产生影响），并将其从当前房间中删除。如果该图层是在房间编辑器中设计的图层，则下次您离开该房间然后返回时，将使用原始内容再次创建该图层，但是如果该房间是永久性的，则除非删除该图层，否则房间持久性再次关闭。当您以这种方式销毁图层时，其所有内容也将被删除，因此背景或瓷砖地图等的所有参考ID将不再有效，分配给该图层的所有实例都将被销毁（执行其销毁操作）事件在同一时间，如果他们有一个，还有清理事件）。

句法：

layer_destroy(layer_id)

争论描述
layer_id要销毁的图层的唯一ID值（或图层名称为字符串）

返回值：

N/A

例子：

if !instance_exists(obj_Bullet_Parent)
   {
   layer_destroy(global.Bullet_Layer);
   }

上面的代码将检查对象“ obj_Bullet_Parent”的任何实例是否存在，如果不存在，则它将销毁具有存储在全局变量中的ID的图层。
