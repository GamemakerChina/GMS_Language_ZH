层_销毁

这个函数将销毁给定的图层。你可以提供图层的ID（你在创建图层时得到的ID是用 layer_create())或图层名称(作为一个字符串--这将对性能产生影响)，这 将从当前房间中删除它。如果该层是在房间编辑器中设计的，那么当你下次离开房间并返回时，该层将以原来的内容重新创建。 如果房间是持久的，该层将被删除，除非房间的持久性被再次关闭。当你以这种方式销毁一个图层时，它的所有内容也将被删除，所以任何背景或瓷砖地图等的参考ID将不再有效，任何分配给图层的实例 分配给该层的任何实例都将被销毁（同时执行它们的销毁事件，如果它们有一个的话，以及清理事件）。

语法。

layer_destroy(layer_id)

争论 描述
layer_id 要销毁的层的唯一ID值（或作为字符串的层名）。

返回。

不适用

例子。

if !instance_exists(obj_Bullet_Parent)
{
layer_destroy(global.Bullet_Layer);
}

上面的代码将检查 "obj_Bullet_Parent "对象的任何实例是否存在，如果不存在，它将用存储在全局变量中的ID销毁该层。
