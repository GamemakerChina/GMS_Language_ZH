layer_destroy_instances

此功能可用于销毁分配给给定层的所有实例。您提供图层ID（使用 layer_create()）或图层名称（作为字符串-这会对性能产生影响），然后该图层上的所有实例将从游戏中删除，并触发其Destroy和Clean Up事件。

句法：

layer_destroy_instances(layer_id)

争论描述
layer_id设置深度的图层的唯一ID值（或图层名称为字符串）

返回值：

Real (Unique ID value for an instance)

例子：

if global.game_over
    {
    layer_destroy_instances(layer);
    }

上面的代码将检查一个全局变量，如果它是 true那么位于调用实例层上的所有实例都将被销毁（包括调用实例）。

返回：常规层功能
