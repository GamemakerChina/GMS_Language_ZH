层_销毁物质

这个函数可以用来销毁分配给给定层的所有实例。你可以提供图层的ID（你在创建图层时使用 layer_create())或者层的名字（作为一个字符串--这 会对性能产生影响），然后所有在该层上的实例将被从游戏中移除，触发它们的Destroy和Clean Up事件。

语法。

layer_destroy_instances(layer_id)

争论 描述
layer_id 要设置深度的图层的唯一ID值（或作为字符串的图层名称）。

返回。

真实（一个实例的唯一ID值

例子。

如果 global.game_over
{
layer_destroy_instances(layer);
}

上面的代码将检查一个全局变量，如果它是真的，那么所有在调用实例层上的实例都将被销毁（包括调用实例）。

返回。一般层的功能
