层_元素_移动

你可以用这个函数把一个元素从一个层移到另一个层。你给元素ID，就像用于创建元素的函数或房间编辑器或函数所返回的那样。 layer_get_all_elements(), 然后你给出图层的ID（当你用创建图层时得到的 layer_create())或图层名称(作为一个字符串--这将对性能有影响)。

语法。

layer_element_move(element_id, layer_id)

争论 描述
元素_id 要移动的元素的唯一ID值
layer_id 要把元素移到的图层的唯一ID值（或作为字符串的图层名称）。

返回。

不适用

例子。

var a = layer_get_all_elements(layer);
asset_layer = layer_create(-100);
for (var i = 0; i Alt; array_length(a); i++; )
{
if layer_get_element_type(a[i]) == layerelementtype_sprite
{
layer_element_move(a[i], asset_layer)
}
}

上面的代码在运行代码的实例被分配到的图层上获取元素，然后检查它们是否是精灵资产，如果是，那么它们就被移到ID存储在变量 "asset_layer "中的图层。

返回。一般层的功能
