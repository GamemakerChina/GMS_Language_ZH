层_存在

这个函数可以用来检查给定的图层是否存在。你可以提供图层的ID（当你使用 layer_create())或层的名称（作为一个字符串--这将对性能产生影响 影响），如果它存在，该函数将返回一个布尔值：true；如果不存在，则返回false。

注意：这个函数在当前目标房间的范围内工作--默认情况下是调用函数的房间--可以用函数设置 layer_set_target_room().

语法。

layer_exists(layer_name)

争论 描述
层名 图层的名称（一个字符串或ID值）。

返回。

布尔型

例子。

if !layer_exists(global.tileLayer)
{
global.tileLayer = layer_create(1000);
}

上面的代码将检查存储在全局变量中的图层是否实际存在，如果不存在，那么就创建它。
