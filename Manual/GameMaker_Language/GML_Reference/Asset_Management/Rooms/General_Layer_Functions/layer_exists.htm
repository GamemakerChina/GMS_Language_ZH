layer_exists

此功能可用于检查给定图层是否存在。您提供图层ID（使用 layer_create()）或图层名称（作为字符串-这将对性能产生影响），并且该函数将返回布尔值 true如果存在或 false如果没有。

注意：此功能在当前目标房间的范围内起作用-默认情况下，该功能所在的房间-可以使用该功能进行设置 layer_set_target_room()。

句法：

layer_exists(layer_name)

争论描述
layer_name图层名称（字符串或ID值）

返回值：

Boolean

例子：

if !layer_exists(global.tileLayer)
   {
   global.tileLayer = layer_create(1000);
   }

上面的代码将检查存储在全局变量中的图层是否实际存在，如果不存在则创建该图层。

返回：常规层功能
