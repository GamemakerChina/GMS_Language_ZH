layer_force_draw_depth

此功能强制在指定的z深度处绘制所有图层。这不会更改图层的渲染顺序，并且仍将按深度顺序进行绘制，而只是更改使用的z值。通常，您不必担心这一点，但是如果您的图层深度不在合法范围内（-16000至16000），则不会渲染它们，因此您可以将Z深度强制为合理的值-例如0-并且它们都会很好地呈现。请注意，这通常仅适用于GameMaker版本中的旧项目，在这些项目中您绘制的深度可能高于或低于允许的图层范围。

句法：

layer_force_draw_depth(force, depth)

争论描述
force是否启用（设置为 true或禁用（设置为 falseZ深度强迫
depth新的Z深度

返回值：

N/A

例子：

if !layer_is_draw_depth_forced()
   {
   layer_force_draw_depth(true, 0);
   }

上面的代码检查是否强制Z层深度，如果不是，则将Z深度设置为0并启用它。
