layer_get_all

此函数将返回一个数组，其中填充了房间中每一层的唯一ID值。

句法：

layer_get_all()

返回值：

Array (1D, populated with Layer IDs)

例子：

var a = layer_get_all();
for (var i = 0; i < array_length(a); i++;)
   {
   layer_destroy(a[i]);
   }

上面的代码检索房间中的所有图层，并将其ID值添加到数组中。然后解析此数组以销毁房间层。

返回：常规层功能
