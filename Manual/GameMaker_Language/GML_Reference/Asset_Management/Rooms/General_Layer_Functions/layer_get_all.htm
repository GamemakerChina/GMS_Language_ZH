层数_获取所有

这个函数将返回一个数组，其中包含房间中每个层的唯一ID值。

语法。

layer_get_all()

返回。

数组（1D，用图层ID填充）。

例子。

var a = layer_get_all();
for (var i = 0; i < array_length(a); i++;)
{
layer_destroy(a[i])。
}

上面的代码检索了一个房间里的所有层，并将它们的ID值添加到一个数组中。然后，这个数组被解析为销毁或房间的层。

返回。一般层的功能
