layer_get_all_elements

您可以使用此功能获取给定图层的元素ID。您提供图层ID（使用 layer_create()）或图层名称（作为字符串-会对性能产生影响），该函数将返回ID数组，其中数组中的每个条目都是该图层上元素的唯一ID。例如，如果图层是资产图层，则将使用分配给该图层的每个Sprite资产的ID值填充数组。请注意，使用代码处理图层意味着您可以将不同的元素类型分配给同一图层-例如，您可以将Sprite资产与实例一起使用-在这种情况下，您可以使用函数 layer_get_element_type()获取与ID相关的元素的类型。

句法：

layer_get_all_elements(layer_id)

争论描述
layer_id从中获取元素的图层的唯一ID值（或图层名称作为字符串）

返回值：

Array (1D, populated with Element IDs)

例子：

var a = layer_get_all_elements(layer);
for (var i = 0; i < array_length(a); i++;)
   {
   if layer_get_element_type(a[i]) == layerelementtype_sprite
      {
      layer_sprite_destroy(a[i])
      }
   }

上面的代码获取分配给运行代码的实例层的所有实例元素的ID。然后，代码检查以查看是否有任何返回的元素是sprite资产，然后将它们销毁。

返回：常规层功能
