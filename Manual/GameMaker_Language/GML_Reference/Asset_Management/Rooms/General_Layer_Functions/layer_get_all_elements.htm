层级_获取所有元素

你可以使用这个函数来获取给定层的元素ID。你可以提供图层的ID（当你创建图层的时候，你可以用 layer_create())或图层名称(作为一个字符串--这将会对性能产生影响)，该函数将获得图层ID。 性能的影响），该函数将返回一个ID数组，数组中的每一个条目都是该层中一个元素的唯一ID。例如，如果该层是一个资产层，数组 将被填充为分配给该层的每个精灵资产的ID值。注意，使用代码来处理图层意味着你可以把不同的元素类型分配给同一个图层--所以你可以有精灵资产和实例。 例如--在这种情况下，你可以使用函数 layer_get_element_type()来获取ID所涉及的元素类型。

语法。

layer_get_all_elements(layer_id)

争论 描述
layer_id 要获取元素的图层的唯一ID值（或作为字符串的图层名称）。

返回。

数组（1D，用元素ID填充）。

例子。

var a = layer_get_all_elements(layer);
for (var i = 0; i < array_length(a); i++; )
{
if layer_get_element_type(a[i]) == layerelementtype_sprite
{
layer_sprite_destroy(a[i])
}
}

上面的代码获得分配给运行代码的实例层的所有实例元素的ID。然后，代码检查是否有返回的元素是精灵资产，如果是，则将其销毁。
