layer_get_element_layer

您可以使用此函数获取给定元素所在的图层ID。您提供唯一的元素ID值（例如，由创建元素的函数返回或从房间编辑器返回），该函数将返回在其上找到该元素的图层的唯一ID。如果给定的元素ID无效，则该函数将返回-1。

句法：

layer_get_element_layer(element_id)

争论描述
element_id要获取图层的元素的唯一ID值

返回值：

Layer ID or -1 if the element is invalid (ie: does not exist)

例子：

element = layer_get_element_layer(asset_1);

上面的代码获取具有存储在变量“ asset_1”中的ID的元素的层ID，并将其存储在变量中。

返回：常规层功能
