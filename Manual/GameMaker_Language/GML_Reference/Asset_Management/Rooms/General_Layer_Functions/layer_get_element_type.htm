layer_get_element_type

您可以使用此函数获取给定元素的元素类型。您提供唯一的元素ID值（例如，从创建元素的函数返回或从房间编辑器返回），该函数将返回以下常量之一（如果元素不存在或ID值为，则返回-1）。错误）：

持续的描述
layerelementtype_background元素是背景。
layerelementtype_instance元素是一个实例。
layerelementtype_sprite该元素是一个精灵资产。
layerelementtype_tilemap元素是一个tilemap。
layerelementtype_particlesystem该元素是一个粒子系统。
layerelementtype_tile元素是旧的背景图块（仅对从GameMaker版本导入的项目有效）。
layerelementtype_sequence元素是序列资产。

请注意，当您为一个图层分配了多种不同的元素类型时，此功能最有用，并且可以使用该功能获取任何给定图层上所有元素的列表。layer_get_all_elements() 。

句法：

layer_get_element_type(element_id)

争论描述
layer要获取类型的元素的唯一ID值

返回值：

Constant (see table, above) or -1 if element does not exist or is invalid

例子：

var a = layer_get_all_elements(layer);
for (var i = 0; i < array_length(a); i++;)
   {
   if layer_get_element_type(a[i]) == layerelementtype_sprite
      {
      layer_sprite_destroy(a[i])
      }
   }

上面的代码获取分配给运行代码的实例层的所有实例元素的ID。然后，代码检查以查看是否有任何返回的元素是sprite资产，然后将它们销毁。
