层级_get_element_type

你可以使用这个函数来获取给定元素的元素类型。你提供唯一的元素ID值（例如，从创建元素的函数或从房间编辑器返回），该函数将返回以下之一 常数（如果元素不存在或ID值错误，则返回-1）。

恒定 描述
层次元素类型_背景 该元素是一个背景。
层次元素类型_instance 该元素是一个实例。
铺设元素类型_sprite 该元素是一个精灵资产。
图层元素类型_tilemap 该元素是一个瓦片图。
分层元素类型_粒子系统 该元素是一个粒子系统。
铺设元素类型_瓷砖 元素是一个传统的背景瓦片（这只适用于从以前的版本中导入的项目）。 游戏制作者).
层次元素类型_序列 该元素是一个序列资产。

请注意，当你有多个不同的元素类型分配给一个图层时，这个函数是最有用的，你可以使用函数获得任何给定图层上所有元素的列表 layer_get_all_elements().

语法。

layer_get_element_type(element_id)

争论 描述
层 元素的唯一ID值，以获得其类型。

返回。

常数（见上表），如果元素不存在或无效则为-1

例子。

var a = layer_get_all_elements(layer);
for (var i = 0; i < array_length(a); i++; )
{
if layer_get_element_type(a[i]) == layerelementtype_sprite
{
layer_sprite_destroy(a[i])
}
}

上面的代码获得分配给运行代码的实例层的所有实例元素的ID。然后，代码检查是否有返回的元素是精灵资产，如果是，则将其销毁。

返回。一般层的功能
