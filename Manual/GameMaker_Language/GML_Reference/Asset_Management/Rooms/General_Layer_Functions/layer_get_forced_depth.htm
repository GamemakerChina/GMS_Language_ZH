layer_is_draw_depth_forced

此功能可用于检索为房间内的渲染层设置的Z深度。看 layer_force_draw_depth()欲获得更多信息。

句法：

layer_get_forced_depth()

返回值：

Real

例子：

if layer_get_forced_depth() != 0
   {
   layer_force_draw_depth(true, 0);
   }

上面的代码检查是否将Z层深度强制设置为0值，如果不是，则将Z层深度设置为0。

返回：常规层功能
