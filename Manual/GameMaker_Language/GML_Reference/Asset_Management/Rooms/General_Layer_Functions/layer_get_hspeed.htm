layer_get_hspeed

您可以使用此功能来检索当前范围内房间内图层的水平速度（以每个游戏帧为单位的像素）。您提供图层ID（使用 layer_create()）或图层名称（作为字符串-会对性能产生影响），并且该函数返回水平速度的实数，其中正值在右侧，而负值在左侧。默认值为0（除非在房间编辑器中设置）。

句法：

layer_get_hspeed(layer_id)

争论描述
layer_id从中获取水平速度的图层的唯一ID值

返回值：

N/A

例子：

var lay_id = layer_get_id("Sprites");
if layer_get_hspeed(lay_id) != 0 || layer_get_vspeed(lay_id) != 0
    {
    layer_hspeed(lay_id, 0);
    layer_vspeed(lay_id, 0);
    }

上面的代码检查给定图层的水平和垂直速度，如果未同时将其设置为0，则将其设置为0。

返回：常规层功能
