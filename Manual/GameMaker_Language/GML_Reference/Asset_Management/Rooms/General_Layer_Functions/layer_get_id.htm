layer_get_id

此函数可用于获取给定层的唯一ID值。在IDE中，所有图层都有一个名称和类型，并且要通过代码进行编辑或更改，必须提供图层ID值。此函数用于通过使用图层的名称（字符串）（如在IDE中编写的）来检索此ID。如果您使用函数通过代码创建新图层 layer_create()那么该函数将返回唯一的ID值（动态创建的图层不会获得名称）。请注意，如果提供当前房间中不存在的图层的名称，则将收到错误消息，并且项目将崩溃。

句法：

layer_get_id(layer_name)

争论描述
layer_name图层名称（字符串）

返回值：

Layer ID Value (Real), or -1 if the layer specified doesn't exist

例子：

var near = instance_nearest(x, y, obj_Tree);
var layer_id = layer_get_id("Instances Front");
layer_add_instance(layer_id, near);

上面的代码将首先获取到给定x / y位置的最近实例的索引，并将其存储在局部变量中。然后，它为名为“ Instances Front”的层获取唯一的实例层ID，并将找到的实例移到该层。
