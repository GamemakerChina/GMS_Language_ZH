层数_获取_ID

这个函数可以用来获取一个给定图层的唯一ID值。在IDE中，所有图层都有一个名称和一个类型，为了能够通过代码编辑或改变它们，你必须给出图层的ID值。这个函数是用来检索 这个ID是通过使用层的名称（一个字符串）（如IDE中写的）来检索的。如果你通过代码创建一个新的层，使用函数 layer_create()那么该函数将返回唯一的ID值，而不是 (动态创建的图层不会得到名字)。请注意，如果你给了一个在当前房间里不存在的层的名字，那么你会得到一个错误，项目会崩溃。

语法。

layer_get_id(layer_name)

争论 描述
层名 该层的名称（一个字符串）。

返回。

层ID值（Real），如果指定的层不存在，则为-1。

例子。

var near = instance_nearest(x, y, obj_Tree);
var layer_id = layer_get_id("Instances Front");
layer_add_instance(layer_id, near) 。

上面的代码首先会得到离给定X/Y位置最近的实例的索引，并将其存储在一个局部变量中。然后获取名为 "Instances Front "的层的唯一实例层ID，并将找到的实例移动到 该层上。

返回。一般层的功能
