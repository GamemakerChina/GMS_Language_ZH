layer_get_id_at_depth

您可以使用此功能获取分配了特定深度的所有图层的ID。您指定要检查的深度，函数将根据给定深度是否有任何层返回一个包含1个或多个条目的数组。如果在给定深度处没有图层，则数组将在[0]位置具有值为-1的单个条目，但是，如果在深度处存在图层，则将在数组中创建一个条目以用于找到的每个图层-条目值将是图层的唯一ID值。

句法：

layer_get_id_at_depth(depth)

争论描述
depth检查和检索图层ID的深度。

返回值：

Array

例子：

var a = layer_get_id_at_depth(0);
if a[0] != -1
    {
    for (var i = 0; i < array_length(a); i++;)
        {
        layer_destroy(a[i]);
        }
    }

上面的代码检索深度为0的图层的数据。检查该深度是否存在任何图层，如果存在，则对返回的数组进行解析，并破坏每个找到的图层。

返回：常规层功能
