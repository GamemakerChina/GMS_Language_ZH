层数_获取深度_ID

你可以使用这个函数来获得所有分配给特定深度的图层的ID。你给出要检查的深度，该函数将返回一个有1个或多个条目的数组，这取决于是否存在 在给定深度的任何图层或没有。如果在给定的深度没有图层，那么数组将在[0]位置有一个值为-1的条目，但是，如果在该深度有图层，那么将在数组中为每个 的条目--该条目值将是一个层的唯一ID值。

语法。

层数_获取_深度(depth)。

争论 描述
深度 检查和检索图层ID的深度。

返回。

阵列

例子。

var a = layer_get_id_at_depth(0);
if a[0] != -1
{
for (var i = 0; i < array_length(a); i++;)
{
layer_destroy(a[i])。
}
}

上面的代码检索深度为0的层的数据，检查是否有任何层存在于这个深度，如果有，那么返回的数组被解析，找到的每个层都被销毁。
