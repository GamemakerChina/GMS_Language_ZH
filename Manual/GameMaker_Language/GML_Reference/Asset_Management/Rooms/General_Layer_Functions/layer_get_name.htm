layer_get_name

您可以使用此功能获取给定图层的名称。您提供唯一的图层ID值，并且如果该图层是在房间编辑器中创建的命名图层之一，则该函数将返回带有图层名称的字符串。如果该层不是房间编辑器之一（即：它是使用 layer_create()），则将返回一个空字符串。

句法：

layer_get_name(layer_id)

争论描述
layer_id图层名称的唯一ID值

返回值：

String

例子：

var a = layer_get_all();
var layer_list = ds_list_create(); for (var i = 0; i Alt; array_length(a); i++;)
   {
   if layer_get_name(a[i]) != ""
      {
      ds_list_add(layer_list, a[i])
      }
   }

上面的代码获取房间中所有图层的ID，然后循环遍历它们以检查是否有任何命名的图层。如果它们是ID，则将其ID添加到列表中。

返回：常规层功能
