层的名称

你可以使用这个函数来获取给定图层的名称。你提供唯一的图层ID值，如果该图层是在房间编辑器中创建的命名图层之一，那么该函数将返回一个带有该图层名称的字符串。如果该图层 不是房间编辑器中的一个（即：它是用 layer_create())，那么将返回一个空字符串。

语法。

layer_get_name(layer_id)

争论 描述
layer_id 要获得该层名称的唯一ID值

返回。

字符串

例子。

var a = layer_get_all();
var layer_list = ds_list_create(); for (var i = 0; i Alt; array_length(a); i++; )
{
if layer_get_name(a[i]) !"="
{
ds_list_add(layer_list, a[i])
}
}

上面的代码得到了房间里所有图层的ID，然后在它们之间循环，检查是否有命名的图层。如果它们是，那么它们的ID就被添加到一个列表中。

返回。一般层的功能
