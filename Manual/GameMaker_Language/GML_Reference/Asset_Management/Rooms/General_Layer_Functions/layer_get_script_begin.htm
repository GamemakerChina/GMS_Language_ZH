layer_get_script_begin

您提供图层ID（使用 layer_create()）或图层名称（作为字符串-这将对性能产生影响），并且此函数将返回分配给在该层的渲染开始时运行的函数的脚本函数索引，如果没有函数，则它将返回-1被安排了。您可以使用以下命令将脚本功能分配给图层layer_script_begin()和layer_script_end（） 。

句法：

layer_get_script_begin(layer_id);

争论描述
layer_id要定位的图层的唯一ID值（或图层名称为字符串）

返回值：

Script Function Index (Real) or -1 if no script is assigned

例子：

if layer_get_script_begin(layer) == -1
   {
   layer_script_begin(layer, scr_SetShaderValues);
   }

上面的代码将检查运行该代码的实例的图层是否分配了脚本功能，如果没有分配脚本功能，则该图层将被分配。

返回：常规层功能
