层级_get_script_begin

你提供图层ID（当你使用 layer_create())或图层名称（作为一个字符串--这将对性能产生影响），这个函数将返回指定在该图层渲染开始时运行的函数的脚本函数索引，如果没有指定函数，它将返回-1。你可以用layer_script_begin() ，并将脚本函数分配给一个图层。 layer_script_end().

语法。

layer_get_script_begin(layer_id)。

争论 描述
layer_id 目标层的唯一ID值（或作为字符串的层名）。

返回。

脚本功能索引（Real），如果没有分配脚本，则为-1

例子。

if layer_get_script_begin(layer) == -1
{
layer_script_begin(layer, scr_SetShaderValues);
}

上面的代码将检查运行代码的实例所在的层是否有分配给它的脚本函数，如果没有，就分配一个。

返回。一般层的功能
