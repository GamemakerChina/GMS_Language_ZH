layer_get_ shader

此功能可用于检查给定的图层是否分配shader您提供图层ID（使用 layer_create()）或层名称（作为一个字符串-这将影响性能），并且该函数将返回的shader的的索引shader分配，或-1，如果没有shader被分配。

句法：

layer_get_shader(layer_id)

争论描述
layer_id要定位的图层的唯一ID值（或图层名称为字符串）

返回值：

Shader Index (or -1 if no shader is assigned)

例子：

if layer_get_shader(layer) == -1
   {
   layer_shader(layer, shd_Sepia);
   }

上面的代码将检查运行该代码的实例的图层是否shader ，如果没有，则分配该着色器。

返回：常规层功能
