图层_get_shader

这个函数可以用来检查给定的图层是否有一个着色器分配给它。你可以提供图层的ID（当你使用 layer_create())或图层名称（作为一个字符串--这将对性能产生影响）。 有性能上的影响），该函数将返回指定的着色器索引，如果没有指定着色器，则返回-1。

语法。

layer_get_shader(layer_id)

争论 描述
layer_id 目标层的唯一ID值（或作为字符串的层名）。

返回。

着色器索引（如果没有分配着色器，则为-1）。

例子。

if layer_get_shader(layer) == -1
{
layer_shader(layer, shd_Sepia);
}

上面的代码将检查运行代码的实例所在的层是否有分配给它的着色器，如果没有，就分配一个。
