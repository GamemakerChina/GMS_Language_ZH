layer_get_target_room

此功能将返回图层功能所针对的当前房间。看 layer_set_target_room()欲获得更多信息。

句法：

layer_get_target_room()

返回值：

Room ID (real)

例子：

if layer_get_target_room() != room
   {
   layer_reset_target_room();
   }

上面的代码检查当前目标房间，如果不是当前房间，则重置层目标房间。
