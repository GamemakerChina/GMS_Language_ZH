layer_get_visible

使用此功能，您可以检查图层是否可见。您提供图层ID（使用 layer_create()）或图层名称（作为字符串-这会对性能产生影响），并且该函数将返回 true如果可见，并且 false否则。

句法：

layer_get_visible(layer_id)

争论描述
layer_id要定位的图层的唯一ID值（或图层名称为字符串）

返回值：

Boolean

例子：

var lay_id = layer_get_id("Instances");
if layer_get_visible(lay_id)
   {
   layer_set_visible(lay_id, false);
   }
else
   {
   layer_set_visible(lay_id, true);
   }

上面的代码在房间编辑器中获取名为“实例”的图层的ID值，然后使用该ID检查该图层是否可见，并根据返回的值切换图层的可见性。
