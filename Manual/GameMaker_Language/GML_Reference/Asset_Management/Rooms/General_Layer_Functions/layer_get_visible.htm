层数_获取可见度

用这个函数你可以检查一个图层是否可见。你可以提供图层的ID（当你使用 layer_create())或图层名称（作为一个字符串--这将对性能有影响 影响），如果它是可见的，该函数将返回true，否则返回false。

语法。

layer_get_visible(layer_id)

争论 描述
layer_id 目标层的唯一ID值（或作为字符串的层名）。

返回。

布尔型

例子。

var lay_id = layer_get_id("Instances");
if layer_get_visible(lay_id)
{
layer_set_visible(lay_id, false);
}
else
{
layer_set_visible(lay_id, true);
}

上面的代码在房间编辑器中获得名为 "Instances "的图层的ID值，然后使用该ID检查该图层是否可见，根据返回值切换该图层的可见性。

返回。一般层的功能
