layer_get_vspeed

您可以使用此功能来检索当前范围内房间内图层的垂直速度（以每个游戏帧为单位的像素）。您提供图层ID（使用 layer_create()）或图层名称（作为字符串-这会对性能产生影响），并且该函数返回垂直速度的实数，其中正值向下，负值向上。默认值为0（除非在房间编辑器中设置）。

句法：

layer_get_vspeed(layer_id)

争论描述
layer_id图层的唯一ID值，用于从中获取垂直速度

返回值：

N/A

例子：

var lay_id = layer_get_id("Sprites");
if layer_get_hspeed(lay_id) != 0 || layer_get_vspeed(lay_id) != 0
    {
    layer_hspeed(lay_id, 0);
    layer_vspeed(lay_id, 0);
    }

上面的代码检查给定图层的水平和垂直速度，如果未同时将其设置为0，则将其设置为0。
