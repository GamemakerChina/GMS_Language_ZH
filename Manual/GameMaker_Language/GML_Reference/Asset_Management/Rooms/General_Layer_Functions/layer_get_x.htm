层数_获取_x

你可以使用这个函数来检索图层在当前范围内的X位置。你可以提供图层的ID（当你创建图层的时候，你可以用 layer_create())或图层名称(作为 字符串--这将对性能产生影响），该函数返回该层的X位置的实数，相对于房间的(0,0)位置。默认值是0。

语法。

layer_get_x(layer_id)

争论 描述
layer_id 要获得该层的X位置的唯一ID值。

返回。

真实的

例子。

var lay_id = layer_get_id("Sprites");
if layer_get_x(lay_id) != 0 || layer_get_y(lay_id) != 0
{
layer_x(lay_id, 0);
layer_y(lay_id, 0);
}

上面的代码检查给定的图层位置，如果它没有被设置为（0，0），那么它就被设置为该位置。

返回。一般层的功能
