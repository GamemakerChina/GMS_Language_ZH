layer_get_x

您可以使用此功能来检索当前范围内的房间中图层的x位置。您提供图层ID（使用 layer_create()）或图层名称（作为字符串-这会对性能产生影响），并且该函数将返回该图层x位置（相对于房间（0,0）位置）的实数。默认值为0。

句法：

layer_get_x(layer_id)

争论描述
layer_id获取x位置的图层的唯一ID值

返回值：

Real

例子：

var lay_id = layer_get_id("Sprites");
if layer_get_x(lay_id) != 0 || layer_get_y(lay_id) != 0
    {
    layer_x(lay_id, 0);
    layer_y(lay_id, 0);
    }

上面的代码检查给定的图层位置，如果未将其设置为（0，0），则将其设置为该位置。

返回：常规层功能
