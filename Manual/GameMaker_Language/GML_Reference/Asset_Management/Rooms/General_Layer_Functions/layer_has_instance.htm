layer_has_instance

此功能可用于检查给定实例当前是否已分配给给定图层。您提供图层ID（使用 layer_create()）或图层名称（作为字符串-这会对性能产生影响）以及要检查的实例的实例ID。您也可以给 object_index（即：资源浏览器中对象的名称），该函数将检查该对象的任何实例是否在给定的层上。该函数将返回 true如果实例在图层上，并且 false如果不是这样。

句法：

layer_has_instance(layer_id, instance_id)

争论描述
layer_id要定位的图层的唯一ID值（或图层名称为字符串）
instance_id要检查的唯一实例ID或实例的对象索引

返回值：

Boolean

例子：

if !layer_has_instance(global.Bullet_Layer, obj_Bullet_Parent)
   {
   layer_destroy(global.Bullet_Layer);
   }

上面的代码将检查给定的图层是否包含对象“ obj_Bullet_Parent”的任何实例，如果没有，则它将破坏该图层。

返回：常规层功能
