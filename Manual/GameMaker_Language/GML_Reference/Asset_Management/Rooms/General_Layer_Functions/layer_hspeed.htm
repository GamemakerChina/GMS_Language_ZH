layer_hspeed

您可以使用此功能设置当前范围内房间内图层的水平速度（每游戏帧以像素为单位）。您提供图层ID（使用 layer_create()）或图层名称（作为字符串-这会影响性能）和要设置的速度值，其中正值在右侧，而负值在左侧。

句法：

layer_hspeed(layer_id, hspd)

争论描述
layer_id图层的唯一ID值，用于设置水平速度
hspd要设置的水平速度（每游戏帧以像素为单位）

返回值：

N/A

例子：

var lay_id = layer_get_id("Sprites");
if layer_get_hspeed(lay_id) != 0 || layer_get_vspeed(lay_id) != 0
    {
    layer_hspeed(lay_id, 0);
    layer_vspeed(lay_id, 0);
    }

上面的代码检查给定图层的水平和垂直速度，如果未同时将其设置为0，则将其设置为0。

返回：常规层功能
