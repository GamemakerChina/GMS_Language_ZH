layer_is_draw_depth_forced

此功能可用于检查并查看是否启用了强制Z深度以渲染项目中的图层。看 layer_force_draw_depth()欲获得更多信息。

句法：

layer_is_draw_depth_forced()

返回值：

Boolean

例子：

if !layer_is_draw_depth_forced()
   {
   layer_force_draw_depth(true, 0);
   }

上面的代码检查是否强制Z层深度，如果不是，则将Z深度设置为0并启用它。

返回：常规层功能
