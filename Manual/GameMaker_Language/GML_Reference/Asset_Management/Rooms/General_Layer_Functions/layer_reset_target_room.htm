layer_get_target_room

此功能用于将图层目标重置为当前房间。见功能 layer_set_target_room()了解更多信息。

句法：

layer_reset_target_room()

返回值：

N/A

例子：

layer_set_target_room(rm_Game);
var l = layer_get_id("SpriteAssets");
repeat(50)
   {
   layer_sprite_create(l, random(room_get_width(rm_Game)), random(room_get_height(rm_Game)), spr_Trees);
   }
layer_reset_target_room();

上面的代码将目标房间设置为房间“ rm_Game”，然后获取该房间中名为“ SpriteAssets”的图层的图层ID。然后，在将图层目标重置为当前房间之前，可以使用该图层ID向该图层添加50个随机Sprite资产。

返回：常规层功能
