layer_set_target_room

当您调用此功能时，您是在告诉GameMaker Studio 2所有其他图层功能都应应用于给定的房间。通过这种方式，您可以在不是当前房间的房间中以程序方式更改或生成图层和图层内容。完成向房间中添加图层或图层元素后，调用该函数 layer_reset_target_room()重置房间目标（或使用房间参数-1再次调用此函数）。请注意，此功能只能在当前房间以外的其他房间上使用，并且可以将图层和图层元素添加/删除到当前正在运行的房间之外的其他房间。

警告！在定位另一个房间时，可以使用所有常规图层功能，但不能使用instance_create_layer（）或instance_create_depth（）创建实例，图层功能layer_add_instance（）也将不可用。

句法：

layer_set_target_room(room)

争论描述
room所有其他层功能的目标空间

返回值：

N/A

例子：

layer_set_target_room(rm_Game);
var l = layer_get_id("SpriteAssets");
repeat(50)
    {
    var _x = random(room_get_width(rm_Game));
    var _y = random(room_get_height(rm_Game));
    layer_sprite_create(l, _x, _y, spr_Trees);
    }
layer_reset_target_room();

上面的代码将目标房间设置为房间“ rm_Game”，然后获取该房间中名为“ SpriteAssets”的图层的图层ID。然后，在将图层目标重置为当前房间之前，可以使用该图层ID向该图层添加50个随机Sprite资产。

返回：常规层功能
