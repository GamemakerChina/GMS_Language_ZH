图层_着色器

通过这个函数，你可以给任何给定的图层分配一个着色器资源，然后该图层就会使用该着色器进行渲染。你可以提供图层的唯一ID值或图层的名称（作为一个字符串--这将对性能产生影响 以及要使用的着色器的ID。该着色器必须先前在资产浏览器中创建，然后将着色器索引（着色器资源的名称）传递给该函数。如果指定的图层有实例添加到它，那么 着色器将被应用于该实例使用的所有绘制事件--例如，如果该实例有一个Draw GUI Begin事件，那么着色器将被自动应用到它。着色器还将影响在该层上绘制的任何其他图形元素 在该层上绘制的任何其他图形元素，如精灵资产或瓷砖地图。

请注意，该函数不是为了在任何绘制事件或步骤事件中调用，而是只需要在房间创建代码中或在实例的创建事件 /房间开始事件中调用。

语法。

layer_shader(layer_id,shader)。

争论 描述
layer_id 目标层的唯一ID值（或作为字符串的层名）。
着色器 要分配给该层的着色器索引。

返回。

不适用

例子。

var lay_id = layer_get_id("Instances");
layer_shader(lay_id, shd_Sepia);

上述代码将把着色器资源 "shd_Sepia "分配给所有绘图的给定层。
