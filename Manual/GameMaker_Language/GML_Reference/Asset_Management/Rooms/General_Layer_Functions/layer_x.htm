layer_x

您可以使用此功能设置图层在当前范围内的房间中的x位置。您提供图层ID（使用 layer_create()）或图层名称（作为字符串-这会对性能产生影响），并且该函数会将图层沿房间的水平轴移动给定像素数。

句法：

layer_x(layer_id, x)

争论描述
layer_id图层的唯一ID值，用于设置x的位置
x在房间中将图层设置为的x位置

返回值：

N/A

例子：

var lay_id = layer_get_id("Sprites");
if layer_get_x(lay_id) != 0 || layer_get_y(lay_id) != 0
    {
    layer_x(lay_id, 0);
    layer_y(lay_id, 0);
    }

上面的代码检查给定的图层位置，如果未将其设置为（0，0），则将其设置为该位置。

返回：常规层功能
