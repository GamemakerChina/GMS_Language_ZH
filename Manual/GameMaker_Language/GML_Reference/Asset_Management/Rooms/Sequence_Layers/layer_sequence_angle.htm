layer_sequence_angle

使用此功能，您可以设置给定序列元素的旋转角度。您提供的序列元素ID为 layer_sequence_create()或通过图层元素功能之一以及要设置的新角度，该序列将绕其原点旋转到新位置。在GameMaker中，角度的计算方法是向右0º，然后逆时针旋转-因此，向上90º，向左180º，向下270º-序列的默认角度为0º。

请注意，您不应更改 image_xscale\ image_yscale \ image_angle \ x \ y使用此函数的序列中任何实例的变量，因为一旦调用函数后序列更新，它们将被覆盖。

句法：

layer_sequence_angle(sequence_element_id, angle)

争论描述
sequence_element_id要定位的序列元素的唯一ID值
angle将序列元素旋转到的新角度

返回值：

N/A

例子：

if current_angle < 90
    {
    current_angle += 1;
    layer_sequence_angle(my_seq, current_angle);
    }

上面的代码检查了 current_angle变量，如果小于90，则将其相加，然后使用该值设置变量中引用的序列元素的角度 my_seq。
