layer_sequence_create

使用此功能，您可以在给定层上创建序列资产的实例。您提供的层ID可以是层名称的字符串（如在房间编辑器中定义的那样），也可以是函数返回的唯一层ID。layer_get_id() ，以及在房间中创建序列的X和Y位置，最后是要创建的序列ID。序列ID是您在资源浏览器中为序列定义的名称常量。该函数将返回序列元素的唯一ID，然后可以在序列的所有其他层函数中使用该ID，或者可以使用该函数来检索序列实例结构 layer_sequence_get_instance()。

句法：

layer_sequence_create(layer_id, x, y, sequence_id)

争论描述
layer_id要定位的实例层的唯一ID值
x在房间中的x位置以创建序列
y在房间中的y位置以创建序列
sequence_id在资产浏览器中定义的要创建的序列ID

返回值：

Integer (element ID)

例子：

var _s = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
layer_sequence_pause(_s);

上面的代码在“背景”层上创建了一个新序列，然后将其暂停。

背面：序列层
