layer_sequence_exists

使用此功能，您可以检查给定层上是否存在序列元素。您提供的层ID可以是层名称的字符串（如在房间编辑器中定义的那样），也可以是函数返回的唯一层ID。layer_get_id()以及序列元素ID-由 layer_sequence_create()或通过图层元素功能之一-它将返回 true如果给定元素存在或 false否则。

句法：

layer_sequence_exists(layer_id, sequence_element_id)

争论描述
layer_ID要检查的图层的唯一ID或名称
sequence_element_id要定位的序列元素的唯一ID值

返回值：

Boolean

例子：

if !layer_sequence_exists("Background", my_seq)
    {
    my_seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
    layer_sequence_pause(my_seq);
    }

上面的代码检查给定的sequence元素是否存在，如果不存在，则在给定的层上创建一个新序列，然后暂停它。

背面：序列层
