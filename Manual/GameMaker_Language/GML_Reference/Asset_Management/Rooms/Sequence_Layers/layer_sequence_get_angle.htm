layer_sequence_get_angle

使用此功能，您可以提供序列元素ID-由 layer_sequence_create()或通过图层元素功能之一-它将返回游戏室中序列元素的当前角度。请注意，角度以度为单位返回，右为0º，向上为90º，向左为180º，向下为270º。

句法：

layer_sequence_get_angle(sequence_element_id)

争论描述
sequence_element_id要定位的序列元素的唯一ID值

返回值：

Real

例子：

var _ang = layer_sequence_get_angle(title_sequence)
if _ang > 0
    {
    _ang -= 1;
    layer_sequence_angle(title_sequence, _ang);
    }

上面的代码使用存储在变量“ title_sequence”中的ID检索序列元素的当前角度，如果它不为0，则从当前角度中减去1。
