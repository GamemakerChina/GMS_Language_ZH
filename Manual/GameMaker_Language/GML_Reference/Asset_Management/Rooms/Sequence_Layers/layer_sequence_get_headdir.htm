layer_sequence_get_headdir

使用此功能，您可以提供序列元素ID-由 layer_sequence_create()或通过图层元素功能之一-它将返回当前播放头方向，该方向将是下面列出的常数之一。

句法：

layer_sequence_get_headdir(sequence_element_id)

争论描述
sequence_element_id要定位的序列元素的唯一ID值

返回值：

Constant

持续的描述价值
seq_dir_right顺序是从左到右以递增顺序播放帧1
seq_dir_left顺序是从右到左递减播放帧-1

例子：

if layer_sequence_get_headdir(title_sequence) != seq_dir_left
    {
    layer_sequence_headdir(title_sequence, seq_dir_left);
    }

上面的代码检查存储在变量“ title_sequence”中的序列元素ID的当前播放头方向，以及是否未设置为 seq_dir_left，将其设置为此值。

背面：序列层
