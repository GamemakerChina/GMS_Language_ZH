层_序列_get_headdir

在这个函数中，你提供了序列元素的ID--如由 layer_sequence_create()或某个层元素函数返回的，它将返回当前播放头的方向，它将是下面列出的常数之一。

语法。

layer_sequence_get_headdir(sequence_element_id)。

争论 描述
sequence_element_id 目标序列元素的唯一ID值

返回。

恒定

恒定 描述 价值
顺序排列_右方 该序列以从左到右的增量顺序播放帧数 1
顺序排列_dir_left 该序列是按从右到左的递减顺序播放帧的 -1

例子。

if layer_sequence_get_headdir(title_sequence) != seq_dir_left
{
layer_sequence_headdir(title_sequence, seq_dir_left);
}

上面的代码检查了存储在变量 "title_sequence "中的序列元素ID的当前播放方向，如果它没有被设置为seq_dir_left，它就被设置为这个值。

返回。序列层数
