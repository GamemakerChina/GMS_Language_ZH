层_序列_get_instance

通过这个函数，你可以提供序列元素的ID--如由 layer_sequence_create()或某个层元素函数返回的，它将返回序列实例 结构。你可以在这个页面上找到更多关于序列实例结构的格式。

语法。

layer_sequence_get_instance(sequence_element_id)。

争论 描述
sequence_element_id 目标序列元素的唯一ID值

返回。

结构

例子。

var _seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
var _struct = layer_sequence_get_instance(_seq);
_struct.speedScale = 0.5;

上面的代码创建了一个新的序列元素，然后检索其序列实例ID。然后，这被用来改变序列实例的播放速度刻度属性。

返回。序列层数
