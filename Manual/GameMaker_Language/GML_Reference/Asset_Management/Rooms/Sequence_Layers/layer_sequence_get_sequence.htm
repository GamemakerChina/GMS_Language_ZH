layer_sequence_get_sequence

使用此功能，您可以提供序列元素ID-由 layer_sequence_create()或通过layer元素功能之一-它将返回序列struct 。此函数绕过了先获取序列实例struct的需要，并允许您直接访问序列数据。您可以从“序列函数”部分中找到有关序列结构格式的更多信息。

句法：

layer_sequence_get_sequence(sequence_element_id)

争论描述
sequence_element_id要定位的序列元素的唯一ID值

返回值：

struct

例子：

var _seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
var _struct = layer_sequence_get_sequence(_seq);
_struct.playbackSpeedType = spritespeed_framespersecond; _struct.playbackSpeed = 30;

上面的代码创建了一个新的sequence元素，然后检索了sequence数据结构。然后用于设置播放速度和序列类型。
