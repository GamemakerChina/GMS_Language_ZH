层_序列_获取_序列

通过这个函数，你可以提供序列元素ID--如由 layer_sequence_create() 或层元素函数之一返回的序列元素ID，它将返回序列对象结构。这个函数绕过了首先获得序列实例结构的需要，允许你直接访问序列数据。你可以找到更多 关于序列对象结构的格式。

语法。

layer_sequence_get_sequence(sequence_element_id)。

争论 描述
sequence_element_id 目标序列元素的唯一ID值

返回。

结构

例子。

var _seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
var _struct = layer_sequence_get_sequence(_seq);
_struct.playingSpeedType = spritespeed_framespersecond;
_struct.playingSpeed = 30;

上面的代码创建了一个新的序列元素，然后检索其序列数据结构。然后用它来改变序列的播放速度和类型。

返回。序列层数
