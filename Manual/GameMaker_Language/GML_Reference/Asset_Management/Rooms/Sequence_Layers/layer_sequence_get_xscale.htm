layer_sequence_get_xscale

使用此功能，您可以提供序列元素ID-由 layer_sequence_create()或通过图层元素功能之一-它将沿着游戏室中序列元素的X轴返回当前比例。

句法：

layer_sequence_get_xscale(sequence_element_id)

争论描述
sequence_element_id要定位的序列元素的唯一ID值

返回值：

Real

例子：

var _xs = layer_sequence_get_xscale(title_sequence)
if _xs < 1
    {
    _xs += 0.01;
    layer_sequence_xscale(title_sequence, _xs);
    }

上面的代码沿ID为“ title_sequence”存储的序列元素的X轴检索当前比例尺，如果它小于1，则将0.01添加到当前X比例尺。

背面：序列层
