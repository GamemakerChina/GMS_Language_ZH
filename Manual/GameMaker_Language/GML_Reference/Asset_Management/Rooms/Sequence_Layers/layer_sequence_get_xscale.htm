层_序列_get_xscale

在这个函数中，你提供了序列元素的ID--如由 layer_sequence_create()或某个图层元素函数返回的序列元素ID，它将返回游戏室中序列元素沿X轴的当前比例。

语法。

layer_sequence_get_xscale(sequence_element_id)

争论 描述
sequence_element_id 目标序列元素的唯一ID值

返回。

真实的

例子。

var _xs = layer_sequence_get_xscale(title_sequence)
if _xs < 1
{
_xs += 0.01;
layer_sequence_xscale(title_sequence, _xs);
}

上面的代码沿X轴检索序列元素的当前比例，其ID存储在变量 "title_sequence "中，如果它小于1，则在当前X比例上添加0.01。

返回。序列层数
