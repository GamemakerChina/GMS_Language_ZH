层_序列_get_y

在这个函数中，你提供了序列元素的ID--如由 layer_sequence_create()或某一图层元素函数返回的，它将返回该序列在游戏室中的当前Y位置。

语法。

layer_sequence_get_y(sequence_element_id)。

争论 描述
sequence_element_id 目标序列元素的唯一ID值

返回。

真实的

例子。

if layer_sequence_get_y(title_sequence) != room_height / 2
{
layer_sequence_y(title_sequence, room_height / 2);
}

上面的代码检查存储在变量 "title_sequence "中的序列元素ID的Y位置，如果它没有被设置为房间高度的一半，它就被设置为这个值。

返回。序列层数
