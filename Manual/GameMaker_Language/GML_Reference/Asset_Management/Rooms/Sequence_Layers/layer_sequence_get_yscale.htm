层_序列_得到_yscale

通过这个函数，你可以提供序列元素的ID--如由 layer_sequence_create()或某个图层元素函数返回的序列元素ID，它将返回游戏室中序列元素沿Y轴的当前比例。

语法。

layer_sequence_get_yscale(sequence_element_id)。

争论 描述
sequence_element_id 目标序列元素的唯一ID值

返回。

真实的

例子。

var _ys = layer_sequence_get_yscale(title_sequence)
if _ys < 1
{
_ys += 0.01;
layer_sequence_yscale(title_sequence, _ys);
}

上面的代码检索序列元素沿Y轴的当前比例，其ID存储在变量 "title_sequence "中，如果它小于1，则将0.01添加到当前的Y比例中。

返回。序列层数
