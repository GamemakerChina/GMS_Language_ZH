层_序列_headdir

通过这个函数，你可以为给定的序列播放头设置方向。你提供序列元素的ID，如由 layer_sequence_create() 或某个层元素函数返回的序列元素ID，然后给出播放头的方向，它应该是下列常数之一。

恒定 描述 价值
顺序排列_右方 该序列将按照从左到右的增量顺序播放帧数 1
顺序排列_dir_left 该序列将按照从右到左的递减顺序播放帧数 -1

语法。

layer_sequence_headdir(sequence_element_id, direction)。

争论 描述
sequence_element_id 目标序列元素的唯一ID值
方向 播放头的方向，一个常数，上面列出了

返回。

不适用

例子。

var _seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
layer_sequence_headdir(_seq, seq_dir_left) 。

上面的代码创建了一个新的序列，并将其ID存储在一个局部变量中。然后这个ID被用来设置播放头的方向，以减少帧数（从右到左播放）。

返回。序列层数
