layer_sequence_headdir

使用此功能，您可以设置给定序列播放头的方向。您提供的序列元素ID为 layer_sequence_create()或通过图层元素函数之一，然后给出播放头方向，该方向应为以下常数之一：

持续的描述价值
seq_dir_right该序列将从左到右以递增顺序播放帧1
seq_dir_left该序列将按从右到左的降序顺序播放帧-1

句法：

layer_sequence_headdir(sequence_element_id, direction)

争论描述
sequence_element_id要定位的序列元素的唯一ID值
direction上面列出的播放头方向（一个常数）

返回值：

N/A

例子：

var _seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
layer_sequence_headdir(_seq, seq_dir_left);

上面的代码创建一个新序列并将其ID存储在本地变量中。然后使用此ID将播放头方向设置为递减帧（从右向左播放）。

背面：序列层
