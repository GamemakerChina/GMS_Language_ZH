layer_sequence_headpos

使用此功能，可以将序列元素的播放头位置设置为特定帧。您提供的序列元素ID为 layer_sequence_create()或通过图层元素功能之一以及要设置的新位置。请注意，该位置以帧为单位，并且如果您设置的值大于帧总数（或小于0），则实际的最终播放头位置将取决于已选择的序列播放类型，请遵循以下规则：

  • 无循环：播放头将固定在循环的末尾（如果给定的值为负数，则开始循环）
  • 循环播放：播放头将返回到序列的开头加上额外的帧，例如：如果将30帧序列上的播放头设置为第45帧的位置，则实际的播放头位置最终将为15。负值，则序列播放头将反方向环绕。
  • 乒乓循环：播放头将前进到最后，然后返回超出整个帧长度的多余帧数，从而反转播放方向，例如：如果序列为30帧，并且将播放头设置为40 ，最终的播放头位置将为20，并且方向将反转（下降至0）。请注意，不要为序列的乒乓播放提供负值，因为它可能会产生不确定的结果。

以这种方式设置磁头位置不会停止播放，除非暂停，否则序列将从新的播放头位置继续进行。

句法：

layer_sequence_headpos(sequence_element_id, position)

争论描述
sequence_element_id要定位的序列元素的唯一ID值
position序列中将播放头位置设置为的位置（以帧为单位）（可以是十进制值）

返回值：

N/A

例子：

var a = layer_get_all_elements(layer);
for (var i = 0; i < array_length(a); i++;)
    {
    if layer_get_element_type(a[i]) == layerelementtype_sequence
        {
        layer_sequence_headpos(a[i], 0)
        }
    }

上面的代码获取分配给运行该代码的实例层的所有元素的ID。然后，代码检查一下是否有返回的元素是序列资产，以及是否将它们的播放头位置设置为0。
