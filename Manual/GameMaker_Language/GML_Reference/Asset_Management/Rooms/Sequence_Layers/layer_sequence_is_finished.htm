层_序列_是_完成的

通过这个函数，你提供了序列元素的ID--如由 layer_sequence_create()或某个层元素函数返回的，它将检查序列是否完成了播放，如果是则返回真，如果不是则返回假。注意，这只适用于当序列在播放模式下没有被设置为循环或乒乓时。

语法。

layer_sequence_is_finished(sequence_element_id)。

争论 描述
sequence_element_id 目标序列元素的唯一ID值

返回。

布尔型

例子。

if layer_sequence_is_finished(title_sequence) != 0
{
alarm[0] = room_speed * 3;
layer_sequence_play(title_sequence);
}

上述代码检查存储在变量 "title_sequence "中的序列元素ID是否已经完成，如果是，则再次开始播放，并设置一个警报。

返回。序列层数
