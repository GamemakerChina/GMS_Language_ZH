layer_sequence_is_finished

使用此功能，您可以提供序列元素ID-由 layer_sequence_create()或通过图层元素功能之一-它将检查序列是否已完成播放，返回 true如果是，并且 false如果不是这样。请注意，这仅适用于在播放模式下序列未设置为循环或乒乓的情况。

句法：

layer_sequence_is_finished(sequence_element_id)

争论描述
sequence_element_id要定位的序列元素的唯一ID值

返回值：

Boolean

例子：

if layer_sequence_is_finished(title_sequence) != 0
    {
    alarm[0] = room_speed * 3;
    layer_sequence_play(title_sequence);
    }

上面的代码检查存储在变量“ title_sequence”中的序列元素ID是否已完成，如果已启动，则它将再次播放并设置警报。
