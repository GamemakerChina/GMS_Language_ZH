layer_sequence_is_paused

使用此功能，您可以提供序列元素ID-由 layer_sequence_create()或通过图层元素功能之一-它将检查序列当前是否已暂停，返回 true如果已暂停，并且 false如果不是这样。

句法：

layer_sequence_is_paused(sequence_element_id)

争论描述
sequence_element_id要定位的序列元素的唯一ID值

返回值：

Boolean

例子：

if layer_sequence_is_paused(title_sequence) != 0
    {
    layer_sequence_play(title_sequence);
    }

上面的代码检查存储在变量“ title_sequence”中的序列元素ID是否暂停，如果是，则开始播放。
