layer_sequence_pause

使用此功能，您可以暂停给定序列的播放。您提供的序列元素ID为 layer_sequence_create()或使用图层元素功能之一，该功能将暂停音序，直到您再次使用该功能开始播放 layer_sequence_play()。

句法：

layer_sequence_pause(sequence_element_id)

争论描述
sequence_element_id要定位的序列元素的唯一ID值

返回值：

N/A

例子：

var _seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
if global.Pause
    {
    layer_sequence_pause(_seq);
    }

上面的代码创建一个新序列并将其ID存储在本地变量中。然后，它检查游戏是否暂停，如果暂停，则暂停序列的播放。
