层_序列_速度刻度

通过这个函数，你可以改变给定序列的播放速度。你提供序列元素的ID，如由 layer_sequence_create()或一个层元素函数返回的序列元素ID，然后提供一个速度刻度值。这个值是一个乘数，其中1是默认的播放速度，小于1的值会减慢播放速度，大于1的值会加快播放速度，例如：一个0.5的值是一半的播放速度，而 值为2将是双倍的播放速度。

语法。

layer_sequence_speedscale(sequence_element_id, speedscale)

争论 描述
sequence_element_id 目标序列元素的唯一ID值
速度刻度 要使用的速度刻度（一个乘数），默认为1

返回。

不适用

例子。

var _seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
layer_sequence_speedscale(_seq, 0.75);

上面的代码在调用实例的层上创建了一个新的序列，然后将其速度刻度值设置为0.75（四分之三的默认播放速度）。

返回。序列层数
