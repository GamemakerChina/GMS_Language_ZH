layer_sequence_speedscale

使用此功能，您可以更改给定序列的播放速度。您提供的序列元素ID为 layer_sequence_create()或通过图层元素功能之一，然后是速度比例值。该值是一个乘数，其中1是默认播放速度，小于1的值将减慢播放速度，大于1的值将加快播放速度，例如：0.5的值表示播放速度的一半，而2的值表示播放速度的一半。是两倍的播放速度。

句法：

layer_sequence_speedscale(sequence_element_id, speedscale)

争论描述
sequence_element_id要定位的序列元素的唯一ID值
speedscale要使用的速度标度（乘数），默认为1

返回值：

N/A

例子：

var _seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
layer_sequence_speedscale(_seq, 0.75);

上面的代码在调用实例的层上创建一个新序列，然后将其速度标度值设置为0.75（四分之三的默认播放速度）。
