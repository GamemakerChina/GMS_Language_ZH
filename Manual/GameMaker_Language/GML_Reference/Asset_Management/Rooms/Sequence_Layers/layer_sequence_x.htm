层_序列_x

通过这个函数，你可以为给定的序列元素设置房间沿X（水平）轴的位置。你提供的序列元素ID是由 layer_sequence_create()或由一个 所返回的序列元素ID，以及要设置的X位置，序列将被移动到新的位置。

语法。

layer_sequence_x(sequence_element_id, pos_x)

争论 描述
sequence_element_id 目标序列元素的唯一ID值
pos_x 将序列元素移动到的X位置

返回。

不适用

例子。

if layer_sequence_exists(my_seq)
{
layer_sequence_x(my_seq, x);
layer_sequence_y(my_seq, y);
}

上述代码检查变量 "my_seq "中引用的序列元素是否存在，如果存在，则将x/y位置设置为运行该代码的实例的位置。

返回。序列层数
