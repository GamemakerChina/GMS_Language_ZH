layer_sequence_y

使用此功能，可以为给定的序列元素设置沿房间的Y（垂直）轴的位置。您提供的序列元素ID为 layer_sequence_create()或通过图层元素功能之一以及要设置的Y位置将序列移动到新位置。

句法：

layer_sequence_y(sequence_element_id, pos_y)

争论描述
sequence_element_id要定位的序列元素的唯一ID值
pos_y将序列元素移至的Y位置

返回值：

N/A

例子：

if layer_sequence_exists(my_seq)
    {
    layer_sequence_x(my_seq, x);
    layer_sequence_y(my_seq, y);
    }

上面的代码检查是否存在变量“ my_seq”中引用的序列元素，如果存在，则将x / y位置设置为运行代码的实例的位置。
