层_序列_yscale

通过这个函数，你可以为给定的序列元素设置Y刻度。你提供的序列元素ID是由 layer_sequence_create()返回的序列元素ID，或者通过一个图层元素函数，以及在Y轴上设置的新比例，序列将被这个数量所缩放。1的比例表示没有比例（1:1），较小的值将缩小比例（例如，0.5，将是序列宽度的一半），较大的值将扩大比例 负值将使序列围绕其原点翻转并缩放，除非使用的值正好是-1（在这种情况下，序列只是围绕其原点翻转，没有缩放）。

非常重要的一点是，对包含任何使用旋转的实例的序列元素应用不均匀的缩放（例如：X轴缩放3，Y轴缩放2），可能会导致实例绘制、碰撞、剔除和许多其他问题。 基本上，如果你的序列依赖于任何实例的属性，那么我们不建议你将不均匀的缩放与实例的旋转相结合。

还要注意的是，你不应该改变 image_xscale \ image_yscale \ image_angle \ x \ y 变量。 实例的变量，因为在调用该函数后，一旦序列更新，它们就会被覆盖。

语法。

layer_sequence_yscale(sequence_element_id, xscale)

争论 描述
sequence_element_id 目标序列元素的唯一ID值
yscale 应用于序列元素的新的Y轴刻度值

返回。

不适用

例子。

如果seq_scale < 2
{
seq_scale += 0.01;
layer_sequence_xscale(my_seq, seq_scale);
layer_sequence_yscale(my_seq, seq_scale);
}

上面的代码检查了 seq_scale 变量中的数值，如果它小于2，那么它就把它加进去，然后用这个数值来设置变量 my_seq 中引用的序列元素的X和Y刻度。

返回。序列层数
