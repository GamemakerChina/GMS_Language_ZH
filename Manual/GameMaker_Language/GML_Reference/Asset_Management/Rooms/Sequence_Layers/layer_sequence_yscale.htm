layer_sequence_yscale

使用此功能，您可以为给定的序列元素设置Y比例尺。您提供的序列元素ID为 layer_sequence_create()或通过图层元素功能之一以及要在Y轴上设置的新比例尺来缩放，序列将以此数量进行比例尺缩放。比例尺为1表示无比例缩放（1：1），较小的值将按比例缩小（例如，0.5，将为序列宽度的一半），较大的值将按比例缩放，而负值将围绕序列的原点和比例翻转除非使用的值正好是-1（在这种情况下，序列只是围绕其原点翻转而没有缩放比例）。

请务必注意，对包含使用旋转的任何实例的序列元素应用不均匀的缩放比例（例如，将X轴缩放3，将Y轴缩放2），可能会导致实例绘制，碰撞，剔除和还有很多其他的事情。基本上，如果您的序列依赖于任何实例属性，那么我们不建议您将不均匀缩放与实例旋转结合在一起。

另请注意，您不应更改 image_xscale\ image_yscale \ image_angle \ x \ y使用此函数的序列中任何实例的变量，因为一旦调用函数后序列更新，它们将被覆盖。

句法：

layer_sequence_yscale(sequence_element_id, xscale)

争论描述
sequence_element_id要定位的序列元素的唯一ID值
yscale应用于序列元素的新Y轴比例值

返回值：

N/A

例子：

if seq_scale < 2
    {
    seq_scale += 0.01;
    layer_sequence_xscale(my_seq, seq_scale);
    layer_sequence_yscale(my_seq, seq_scale);
    }

上面的代码检查了 seq_scale变量，如果小于2，则将其加起来，然后使用该值来设置变量中引用的序列元素的X和Y标度 my_seq。
