layer_sprite_alpha

此功能控制资产层上精灵的Alpha（透明度）。您指定精灵元素ID（使用以下代码创建精灵元素时会获得该ID：layer_sprite_create()或使用该功能时 layer_sprite_get_id()），然后设置要使用的alpha值。 Alpha可以在0（完全透明）和1（完全不透明）之间，并且sprite元素的默认alpha值为1。请注意，如果sprite元素所指定的图层不可见-或元素本身已经制成不可见-在再次使图层或元素可见之前，此功能不会有任何区别。

句法：

layer_sprite_alpha(sprite_element_id, alpha);

争论描述
sprite_element_id要更改的Sprite元素的唯一ID值
alpha Sprite Sprite的Alpha，范围从0到1（默认值为1）

返回值：

N/A

例子：

var lay_id = layer_get_id("Asset_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
layer_sprite_alpha(spr_id, random(1));

上面的代码获取分配给图层“ Asset_sky”的名为“ Clouds”的精灵资产的ID值，然后将其alpha设置为0到1之间的随机值。

背面：精灵图层
