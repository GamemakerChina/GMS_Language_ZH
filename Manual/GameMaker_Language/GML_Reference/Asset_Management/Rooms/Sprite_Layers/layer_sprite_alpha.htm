层_sprite_alpha

这个函数控制资产层上的精灵的alpha（透明度）。你给精灵元素的ID（当你用 "A "创建一个精灵元素时，你会得到它。 layer_sprite_create()或在你使用 该函数 layer_sprite_get_id()时得到的），然后设置要使用的alpha值。α值可以在0（完全透明）和1（完全不透明）之间，精灵元素的默认α值为1。注意 如果精灵元素被分配到的图层是不可见的--或者元素本身已经变成不可见的--你将不会看到这个函数的任何区别，直到该图层或元素再次变成可见。

语法。

layer_sprite_alpha(sprite_element_id, alpha)。

争论 描述
萌芽_元素_ID 要改变的精灵元素的唯一ID值
阿尔法 小精灵的alpha，从0到1（默认为1）。

返回。

不适用

例子。

var lay_id = layer_get_id("Asset_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
layer_sprite_alpha( spr_id, random(1));

上面的代码得到了分配给 "Asset_sky "层的名为 "Clouds "的精灵资产的ID值，然后将其alpha设置为0到1之间的随机值。

返回。精灵层
