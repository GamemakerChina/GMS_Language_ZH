layer_sprite_angle

使用此功能，可以更改图层上给定精灵元素的角度。您指定精灵元素ID（使用以下代码创建精灵元素时会获得该ID：layer_sprite_create()或使用该功能时 layer_sprite_get_id()），然后沿逆时针方向从0到359设置角度值，其中0在右边，90在上面，180在左边，270在下面。如果您设置的值大于360，则将其循环以使其在0-359范围内。

句法：

layer_sprite_angle(sprite_element_id, angle);

争论描述
sprite_element_id要更改的Sprite元素的唯一ID值
angle精灵的角度（默认为0）

返回值：

N/A

例子：

var asset_sprite = layer_sprite_get_id(layer, "gfc_Trees");
if layer_sprite_get_angle(asset_sprite) != 0
   {
   layer_sprite_angle(asset_sprite, 0);
   }

上面的代码将检查指定运行该代码的实例所在的图层的sprite元素，如果其角度不为0，则将其设置为0。

背面：精灵图层
