层级_sprite_angle

使用这个函数，你可以改变一个层上给定的精灵元素的角度。你给精灵元素的ID（当你用 "创建 "精灵元素时，你会得到它。 layer_sprite_create()或当你 使用该函数时得到的 layer_sprite_get_id())，然后设置角度值，从0到359逆时针，其中0是向右，90是向顶，180是向左，270是向底。如果你设置的 大于360的值，它将被循环使用，以使其在0-359的范围内。

语法。

layer_sprite_angle(sprite_element_id, angle)。

争论 描述
萌芽_元素_ID 要改变的精灵元素的唯一ID值
角度 精灵的角度（默认为0

返回。

不适用

例子。

var asset_sprite = layer_sprite_get_id(layer, "gfc_Trees");
if layer_sprite_get_angle(asset_sprite) != 0
{
layer_sprite_angle(asset_sprite, 0);
}

上述代码将检查分配给运行该代码的实例所在层的精灵元素，如果它的角度不是0，它将把它设置为0。

返回。精灵层
