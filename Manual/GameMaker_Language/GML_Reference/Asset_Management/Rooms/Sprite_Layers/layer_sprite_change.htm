layer_sprite_change

使用此功能，您可以更改分配给图层上给定Sprite元素的Sprite资源。您指定精灵元素ID（使用以下代码创建精灵元素时会获得该ID：layer_sprite_create()或使用该功能时 layer_sprite_get_id()），然后提供一个精灵索引元素资产将被更改。请注意，如果子画面具有子图像，那么它也将进行动画处理（可以使用 layer_sprite_index()和 layer_sprite_speed()功能）。请注意，您可以将值-1分配为新的精灵索引，尽管该元素仍然存在并且以后仍可以更改，但不会显示任何精灵。

句法：

layer_sprite_change(sprite_element_id, sprite_index)

争论描述
sprite_element_id要更改的Sprite元素的唯一ID值
sprite_index要使用的新精灵索引

返回值：

N/A

例子：

var s = layer_sprite_get_id(layer, global.Asset_sprite);
if layer_sprite_get_sprite(s) != spr_Nighttime
    {
    layer_sprite_change(s, spr_nighttime);
    }

上面的代码使用存储在全局变量中的ID检查元素的精灵索引 global.Asset_sprite如果不是“ spr_Nighttime”，则将该子画面分配给该元素。
