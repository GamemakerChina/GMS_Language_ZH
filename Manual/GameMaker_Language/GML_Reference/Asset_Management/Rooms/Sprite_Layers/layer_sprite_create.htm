layer_sprite_create

使用此功能，您可以将精灵资源分配给要在项目中使用的图层。您提供图层ID（使用 layer_create()或与 layer_get_id()），房间内的位置以及子画面索引（这是该子画面的名称，如资源浏览器中所示），并将其添加到图层中。该函数返回元素的唯一ID值，然后可将其用于子画面的其他图层函数中。

句法：

layer_sprite_create(layer_id, x, y, sprite)

争论描述
layer_id要定位的图层的唯一ID值
x使用的x位置
y y位置
sprite要使用的精灵索引

返回值：

Sprite element ID (Real)

例子：

global.asset_layer = layer_create(10000);
for (var i = 0; i< 10; i++;)
    {
    var _x = random(room_width);
    var _y = room_height - 100;
    global.asset_spr_trees[i] = layer_sprite_create(global.asset_layer, _x, _y, spr_Trees);
    }

上面的代码创建一个新层，然后向其添加10个新的sprite元素，并将每个元素的ID存储到数组中。
