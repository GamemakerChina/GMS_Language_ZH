layer_sprite_destroy

此函数将破坏给定的sprite元素。您提供精灵ID（使用以下代码创建精灵时会获得该ID：layer_sprite_create()或与 layer_get_sprite_id()），则将其删除。请注意，这并不从中取出层，只精灵，如果精灵是一个已经在房间编辑器被添加，那么下一次你离开房间，然后返回，精灵将再次重现。但是，如果房间是持久性的，除非再次关闭房间持久性，否则将删除精灵。

句法：

layer_sprite_destroy(sprite_element_id)

争论描述
sprite_element_id要销毁的精灵的唯一ID值

返回值：

N/A

例子：

var lay_id = layer_get_id("Asset_Trees");
if layer_sprite_exists(lay_id, global.Asset_spr_trees)
    {
    layer_sprite_destroy(lay_id);
    }

上面的代码检查“ Asset_Trees”层，以查看给定的sprite元素是否存在，如果存在，则将其销毁（但不破坏该层）。
