破坏层_破坏层

这个函数将销毁给定的精灵元素。你要提供精灵的ID（当你用 "创建 "精灵时，你会得到这个ID。 layer_sprite_create()或使用图层ID与 layer_get_sprite_id()) 时得到的），这个函数将删除它。请注意，这不会删除该层，只是删除其中的精灵，如果该精灵是在房间编辑器中添加的，那么下次你离开房间然后返回时，该精灵将被重新创建。 再次创建。然而，如果房间是持久的，精灵将被删除，除非房间的持久性被再次关闭。

语法。

layer_sprite_destroy(sprite_element_id)。

争论 描述
萌芽_元素_ID 要销毁的精灵的唯一ID值

返回。

不适用

例子。

var lay_id = layer_get_id("Asset_Trees");
if layer_sprite_exists(lay_id, global.Asset_spr_trees)
{
layer_sprite_destroy(lay_id)。
}

上面的代码检查图层 "Asset_Trees"，看给定的精灵元素是否存在，如果存在，那么它就会被销毁（但不是图层）。
