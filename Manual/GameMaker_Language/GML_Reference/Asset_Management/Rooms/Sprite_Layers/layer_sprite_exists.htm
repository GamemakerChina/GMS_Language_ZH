layer_sprite_exists

您可以使用此函数检查并查看任何给定图层上是否存在sprite元素。您提供图层ID（使用 layer_create()）或图层名称（作为字符串-会对性能产生影响）和精灵元素ID（在使用来创建精灵元素时获得的ID）layer_sprite_create()或使用该功能时 layer_sprite_get_id()），该函数将返回 true如果元素存在，或者 false如果没有。

注意：此函数在当前目标房间的范围内起作用-默认情况下，该函数所在的房间-可以使用函数layer_set_target_room（）进行设置。

句法：

layer_sprite_exists(layer_id, sprite_element_id)

争论描述
layer_id要定位的图层的唯一ID值（或图层名称为字符串）
sprite_element_id要检查的Sprite元素的唯一ID值

返回值：

Boolean

例子：

var lay_id = layer_get_id("Asset_Trees");
if layer_sprite_exists(lay_id, global.TreesSprites)
    {
    layer_sprite_destroy(lay_id, global.TreesSprites);
    }

上面的代码检查“ Asset_trees”层，以查看给定的sprite元素是否存在，如果存在，则将其销毁（但不破坏该层）。
