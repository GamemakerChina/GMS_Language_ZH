层数_栅栏_获取阿尔法

这个函数可以用来获取精灵元素的alpha值。你给精灵元素的ID（当你用 "创建 "精灵元素时，你会得到这个ID。 layer_sprite_create()或在使用函数 layer_sprite_get_id())，该函数将返回一个介于0（完全透明）和1（完全不透明）之间的值。

语法。

layer_sprite_get_alpha(sprite_element_id)。

争论 描述
萌芽_元素_ID 要获取信息的精灵元素的唯一ID值。

返回。

实数（从0到1

例子。

var lay_id = layer_get_id("sprite_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
if layer_sprite_get_alpha( spr_id) < 0.1
{
layer_sprite_destroy(spr_id)。
}

上面的代码将得到名为 "sprite_sky "的图层的ID，然后用它来得到该图层上精灵元素的ID。这个ID然后被用来检查元素的alpha，如果它小于0.1，那么该层元素就被 销毁。

返回。精灵层
