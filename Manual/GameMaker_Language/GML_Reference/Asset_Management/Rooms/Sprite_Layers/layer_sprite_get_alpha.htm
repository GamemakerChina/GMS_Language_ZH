layer_sprite_get_alpha

此函数可用于获取sprite元素的alpha值。您指定精灵元素ID（使用以下代码创建精灵元素时会获得该ID：layer_sprite_create()或使用该功能时 layer_sprite_get_id()），该函数将返回介于0（完全透明）和1（完全不透明）之间的值。

句法：

layer_sprite_get_alpha(sprite_element_id);

争论描述
sprite_element_id用于从中获取信息的sprite元素的唯一ID值

返回值：

Real (from 0 to 1)

例子：

var lay_id = layer_get_id("sprite_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
if layer_sprite_get_alpha(spr_id) < 0.1
    {
    layer_sprite_destroy(spr_id);
    }

上面的代码将获取名为“ sprite_sky”的图层的图层ID，然后使用该ID获取该图层上的sprite元素的ID。然后，此ID用于检查元素的alpha，如果它小于0.1，则将破坏图层元素。

背面：精灵图层
