layer_sprite_get_angle

此函数可用于获取子元素的当前角度。您指定精灵元素ID（使用以下代码创建精灵元素时会获得该ID：layer_sprite_create()或使用该功能时 layer_sprite_get_id()），该函数将返回该角度的实际值。该值将在0到360之间，其中0表示右，90向上，180左和270向下。

句法：

layer_sprite_get_angle(sprite_element_id);

争论描述
sprite_element_id用于从中获取信息的sprite元素的唯一ID值

返回值：

Real

例子：

var lay_id = layer_get_id("sprite_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
if layer_sprite_get_angle(spr_id) != 0
    {
    layer_sprite_yscale(spr_id, 0);
    }

上面的代码将获取名为“ sprite_sky”的图层的图层ID，然后使用该ID获取该图层上的sprite元素的ID。然后使用该ID来检查角度，如果该角度不是0，则将其设置为0。

背面：精灵图层
