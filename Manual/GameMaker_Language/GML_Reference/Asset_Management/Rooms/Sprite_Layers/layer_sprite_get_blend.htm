layer_sprite_get_blend

此功能可用于获取精灵元素的混合颜色。您指定精灵元素ID（使用以下代码创建精灵元素时会获得该ID：layer_sprite_create()或使用该功能时 layer_sprite_get_id()），该函数将返回代表所分配颜色的实数值。

句法：

layer_sprite_get_blend(sprite_element_id);

争论描述
sprite_element_id用于从中获取信息的sprite元素的唯一ID值

返回值：

Colour Value (Real)

例子：

var lay_id = layer_get_id("sprite_sky");
var back_id = layer_sprite_get_id(lay_id, "Clouds");
if layer_sprite_get_blend(back_id) == c_white
    {
    layer_sprite_blend(back_id, make_colour_rgb(random(255), random(255), 255));
    }

上面的代码将获取名为“ sprite_sky”的图层的图层ID，然后使用该ID获取该图层上的sprite元素的ID。然后使用此ID检查元素混合颜色以及它是否等于常量 c_white，然后将元素混合设置为随机颜色。

背面：精灵图层
