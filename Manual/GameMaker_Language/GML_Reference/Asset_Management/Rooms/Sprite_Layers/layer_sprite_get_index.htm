layer_sprite_get_index

此函数可用于获取sprite元素的当前图像索引值。您指定精灵元素ID（使用以下代码创建精灵元素时会获得该ID：layer_sprite_create()或使用该功能时 layer_sprite_get_id()），该函数将返回代表该精灵显示的图像索引的实数值。如果精灵元素不存在或没有为其分配有效的精灵，则该函数将返回-1。

句法：

layer_sprite_get_index(sprite_element_id);

争论描述
sprite_element_id用于从中获取信息的sprite元素的唯一ID值

返回值：

Real (the current sprite image index or -1)

例子：

var lay_id = layer_get_id("sprite_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
if layer_sprite_get_index(spr_id) < 4
    {
    layer_sprite_index(spr_id, 4);
    }

上面的代码将获取名为“ sprite_sky”的图层的图层ID，然后使用该ID获取该图层上的sprite元素的ID。然后使用此ID来检查元素的图像索引是否小于4，如果是，则将其设置为4。

背面：精灵图层
