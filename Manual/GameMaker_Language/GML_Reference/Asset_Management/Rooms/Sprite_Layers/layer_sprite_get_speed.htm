层_栅栏_获取速度

这个函数可以用来获取精灵元素的当前速度乘数值。你给出精灵元素的ID（你在用 "创建 "精灵元素时得到的ID）。 layer_sprite_create()或当你使用 函数时得到的），该函数将返回实数。 layer_sprite_get_id()时得到的），该函数将返回实际值，该值代表用于为精灵制作动画的速度乘数。默认值是1。

语法。

layer_sprite_get_speed(sprite_element_id)。

争论 描述
萌芽_元素_ID 要获取信息的精灵元素的唯一ID值。

返回。

真实的

例子。

var lay_id = layer_get_id("sprite_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
if layer_sprite_get_speed( spr_id) > 0
{
layer_sprite_speed(spr_id, 0);
}

上面的代码将获得名为 "sprite_sky "的层的ID，然后用它来获得该层上的精灵元素的ID。这个ID然后被用来检查该元素的动画速度，如果它大于0，就被 设置为0。

返回。精灵层
