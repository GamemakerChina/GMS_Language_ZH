层级_亚军_获取亚军

这个函数可以用来获取当前精灵元素的精灵索引。你给出精灵元素的ID（你在用 "创建 "精灵元素时得到的ID）。 layer_sprite_create()或在你使用 该函数 layer_sprite_get_id())，该函数将返回一个实值，代表正在显示的精灵索引。如果该元素没有指定精灵，该函数将返回-1。

语法。

layer_sprite_get_sprite(sprite_element_id)。

争论 描述
萌芽_元素_ID 要获取信息的精灵元素的唯一ID值。

返回。

Sprite指数（实数）或-1

例子。

var lay_id = layer_get_id("sprite_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
if layer_sprite_get_sprite( spr_id) != spr_Clouds
{
layer_sprite_change(spr_id, spr_Clouds);
}

上面的代码将得到名为 "sprite_sky "的图层的ID，然后用它来得到该图层上的精灵元素的ID。然后，这个ID被用来检查分配给该元素的精灵，将其设置为精灵 "sprite_Clouds" 如果它还不是的话。

返回。精灵层
