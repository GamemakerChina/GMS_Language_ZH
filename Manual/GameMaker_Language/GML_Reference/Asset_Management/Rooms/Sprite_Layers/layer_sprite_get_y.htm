layer_sprite_get_y

此函数可用于获取房间中精灵元素的y位置。您指定精灵元素ID（使用以下代码创建精灵元素时会获得该ID：layer_sprite_create()或使用该功能时 layer_sprite_get_id()），该函数将返回y位置值。

句法：

layer_sprite_get_y(sprite_element_id);

争论描述
sprite_element_id用于从中获取信息的sprite元素的唯一ID值

返回值：

Real

例子：

var lay_id = layer_get_id("sprite_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
if layer_sprite_get_y(spr_id) < 0
    {
    layer_sprite_y(spr_id, room_height);
    }

上面的代码将获取名为“ sprite_sky”的图层的图层ID，然后使用该ID获取该图层上的sprite元素的ID。然后，此ID用于检查元素y位置，如果它小于0，则将图层元素移动到其他y位置。

背面：精灵图层
