层数_栅栏_索引

这个函数可以用来设置一个精灵资产的图像索引，它在一个图层上有多个子图像。你给精灵元素的ID（你在用 "创建 "精灵元素时得到的ID），或者在使用该函数时得到的ID。 layer_sprite_create() 或当你使用函数 layer_sprite_get_id()时得到的），然后设置要使用的图像索引。如果你设置了一个超出子图像范围的值，那么图像索引将循环使用。

语法。

layer_sprite_index(sprite_element_id, image_index)。

争论 描述
萌芽_元素_ID 要设置的精灵元素的唯一ID值
指数 用于精灵的图像索引

返回。

不适用

例子。

var lay_id = layer_get_id("sprite_trees");
var spr_id = layer_sprite_get_id(lay_id, "gfc_trees");
layer_sprite_index(spr_id, 1);

上面的代码将得到名为 "sprite_trees "的层的ID，然后用它来得到该层上给定的精灵元素的ID。这个ID然后被用来改变元素的图像索引。
