layer_sprite_index

此功能可用于设置在一个图层上具有多个子图像的Sprite资产的图像索引。您指定精灵元素ID（使用以下代码创建精灵元素时会获得该ID：layer_sprite_create()或使用该功能时 layer_sprite_get_id()），然后设置要使用的图片索引。如果您设置的值超出了子图像范围，则图像索引将循环。

句法：

layer_sprite_index(sprite_element_id, image_index);

争论描述
sprite_element_id要设置的Sprite元素的唯一ID值
index用于精灵的图片索引

返回值：

N/A

例子：

var lay_id = layer_get_id("sprite_trees");
var spr_id = layer_sprite_get_id(lay_id, "gfc_trees");
layer_sprite_index(spr_id, 1);

上面的代码将获取名为“ sprite_trees”的图层的图层ID，然后使用该ID获取该图层上给定sprite元素的ID。然后使用此ID更改元素图像索引。

背面：精灵图层
