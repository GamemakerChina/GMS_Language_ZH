layer_sprite_speed

此功能可用于设置已分配给sprite元素的动画图像的速度。您指定精灵元素ID（使用以下代码创建精灵元素时会获得该ID：layer_sprite_create()或使用该功能时 layer_sprite_get_id()），然后设置动画速度。如果IDE将用于计算的精灵的动画速度设置为“每游戏帧数”，则给定的速度值是一个乘数，默认值为1，将其设置为0.5将是设置动画速度的一半。达到1.5会使它翻倍。但是，如果IDE已将用于计算的精灵的动画速度设置为“每秒帧数”，则给定的速度值是每秒的文字帧值，因此将其设置为5，例如，会将动画设置为循环播放游戏每秒运行5帧。如果使用的子画面没有子图像，则此功能无效。

句法：

layer_sprite_speed(sprite_element_id, speed);

争论描述
sprite_element_id要更改的Sprite元素的唯一ID值
speed用于元素的速度乘数/值

返回值：

N/A

例子：

var lay_id = layer_get_id("Assets_water");
var spr_id = layer_sprite_get_id(lay_id, "gfc_water");
layer_sprite_speed(spr_id, 0.5);

上面的代码将获取名为“ Assets_water”的图层的图层ID，然后使用该ID获取该图层上给定sprite元素的ID。然后使用该ID更改元素图像速度。

背面：精灵图层
