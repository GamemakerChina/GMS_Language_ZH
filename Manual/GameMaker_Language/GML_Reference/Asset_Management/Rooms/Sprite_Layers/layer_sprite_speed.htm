层速度

这个函数可以用来设置分配给一个精灵元素的动画图像的速度。你给精灵元素的ID（你在用 "创建精灵元素 "时得到的ID），或在使用该函数时得到的ID。 layer_sprite_create() 或在使用函数 layer_sprite_get_id()获得），然后设置动画速度。如果IDE将精灵的动画速度设置为每游戏帧计算，那么给定的速度值就是一个乘数。 值是一个乘数，1是默认值，设置为0.5将使动画速度减半，而设置为1.5将使动画速度加倍。但是，如果IDE将用于精灵的动画速度设定为 每秒帧数，那么给出的速度值就是每秒帧数的字面值，因此将其设置为5，例如，将设置动画在游戏运行的每一秒钟循环5帧。如果使用的精灵没有子图像，这个函数将 没有任何作用。

语法。

layer_sprite_speed(sprite_element_id, speed)。

争论 描述
萌芽_元素_ID 要改变的精灵元素的唯一ID值
速度 对元素使用的速度乘数/值

返回。

不适用

例子。

var lay_id = layer_get_id("Assets_water");
var spr_id = layer_sprite_get_id(lay_id, "gfc_water");
layer_sprite_speed(spr_id, 0.5);

上面的代码将获得名为 "Assets_water "的图层的ID，然后用它来获得该图层上给定精灵元素的ID。这个ID然后被用来改变元素的图像速度。

返回。精灵层
