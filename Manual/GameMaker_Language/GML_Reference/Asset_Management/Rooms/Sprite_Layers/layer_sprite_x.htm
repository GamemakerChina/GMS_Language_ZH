层数__x

这个函数控制层上资产精灵元素的房间沿X轴的位置。你给精灵元素的ID（当你用 "创建 "精灵元素时，你会得到这个ID）。 layer_sprite_create() 或当你使用函数 layer_sprite_get_id())，然后设置要使用的X值（基于房间坐标）。

语法。

layer_sprite_x(sprite_element_id, x)。

争论 描述
萌芽_元素_ID 要改变的精灵元素的唯一ID值
x 魍魉的X位置

返回。

不适用

例子。

var lay_id = layer_get_id("Asset_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
layer_sprite_x( spr_id, irandom(room_width));

上述代码获得分配给 "Asset_sky "层的精灵资产 "Clouds "的ID值，然后将其X位置设置为0和房间宽度之间的一个随机值。

返回。精灵层
