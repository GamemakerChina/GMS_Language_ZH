layer_sprite_xscale

使用此功能，可以更改是否应沿x轴缩放图层上给定的sprite元素。您指定精灵元素ID（使用以下代码创建精灵元素时会获得该ID：layer_sprite_create()或使用该功能时 layer_sprite_get_id()），然后设置比例值。比例尺为1表示不按比例缩放（1：1），较小的值将按比例缩小（例如0.5，将是所使用精灵的宽度的一半），较大的值将按比例缩放，而负值将翻转精灵并对其进行缩放除非使用的值正好为-1（在这种情况下，使用的精灵只是围绕其原点位置从右向左翻转而没有缩放比例）。

句法：

layer_sprite_xscale(sprite_element_id, xscale);

争论描述
sprite_element_id要更改的Sprite元素的唯一ID值
xscale xscale值（默认为1）

返回值：

N/A

例子：

var asset_sprite = layer_sprite_get_id(layer, "gfc_Trees");
if layer_sprite_get_xscale(asset_sprite) != 1 || layer_sprite_get_yscale(asset_sprite) != 1
    {
    layer_sprite_xscale(asset_sprite, 1);
    layer_sprite_yscale(asset_sprite, 1);
    }

上面的代码将检查指定运行该代码的实例所在的图层的sprite元素，如果在任一方向上对其进行缩放，则会将x轴比例和y轴比例均设置为1。

背面：精灵图层
