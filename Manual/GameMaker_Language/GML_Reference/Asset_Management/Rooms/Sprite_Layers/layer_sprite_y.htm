layer_sprite_y

此功能控制图层上资产精灵元素沿房间y轴的位置。您指定精灵元素ID（使用以下代码创建精灵元素时会获得该ID：layer_sprite_create()或使用该功能时 layer_sprite_get_id()），然后设置要使用的y值（基于房间坐标）。

句法：

layer_sprite_y(sprite_element_id, y);

争论描述
sprite_element_id要更改的Sprite元素的唯一ID值
y精灵的y位置

返回值：

N/A

例子：

var lay_id = layer_get_id("Asset_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
layer_sprite_y(spr_id, 50 + irandom(room_height - 200));

上面的代码获取分配给图层“ Asset_sky”的精灵资产“ Clouds”的ID值，然后将其y位置设置为50到房间高度减去200之间的随机值。
