层数_间距_比例

使用这个函数，你可以改变一个图层上给定的精灵元素是否应该沿Y轴缩放。你给出精灵元素的ID（你在用 "创建 "精灵元素时得到的ID），或者在使用该函数时得到的ID。 layer_sprite_create() 或使用函数 layer_sprite_get_id()时得到的），然后设置缩放值。1表示不缩放（1:1），较小的值将被缩小（例如0.5，将是所使用的精灵的高度的一半），较大的值将被缩小。 0.5），较大的值将向上缩放，负值将镜像精灵并缩放，除非使用的值正好是-1（在这种情况下，使用的精灵只是在其（0，0）位置从上到下镜像，没有缩放）。

语法。

layer_sprite_yscale(sprite_element_id, yscale)。

争论 描述
萌芽_元素_ID 要改变的精灵元素的唯一ID值
yscale yscale值（默认为1

返回。

不适用

例子。

var asset_sprite = layer_sprite_get_id(layer, "gfc_Trees");
if layer_sprite_get_xscale(asset_sprite) != 1 || layer_sprite_get_yscale(asset_sprite) != 1
{
layer_sprite_xscale(asset_sprite, 1);
layer_sprite_yscale(asset_sprite, 1);
}

上述代码将检查分配给运行该代码的实例所在层的精灵元素，如果它在任何一个方向上被缩放，它将把X轴刻度和Y轴刻度都设置为1。

返回。精灵层
