平铺地图图层

GameMaker Studio 2房间编辑器允许您使用图层将瓷砖添加到任何给定的房间中。顾名思义，图块地图图层只是图块集合图块的集合，这些图块都被认为在房间内处于相同深度，并且通过堆叠图层，您可以使某些事物在其他事物之上或之下绘制。您还可以通过代码控制图层的某些方面，添加或删除内容，或更改图层的某些属性或在运行时包含的内容。请注意，在会议室编辑器中，每层仅限于一个图块图（ Tile Layer ），但是在代码中使用它们时，您可以将多个图块图分配给单个图层，每张图都有自己的唯一ID和属性，因此：

  • 瓦片集是用于创建瓦片图层和瓦片地图的资源
  • 磁贴层是您在“房间编辑器”中创建的，用于向其中添加磁贴集
  • 平铺地图就是您所谓的平铺集合，这些平铺集合可以在房间编辑器中或通过代码作为该图层上的单个元素添加到图层中

除了上面列出的用于设置图块贴图的功能之外，您还可以使用许多函数来处理单个图块的数据。瓦片数据的“ blob”由瓦片索引加上一些额外的位组成，以显示瓦片是否已旋转，翻转或镜像。您也可以在图块数据中添加自己的位掩码，尽管这仅适用于高级用户，因为您将屏蔽为图块索引保留的位-实际上，有19位保留给图块索引，但是如果只使用其中的8个，则剩下的11个位可用于创建自定义掩码。然后可以使用此蒙版并在代码中签入以创建自定义图块冲突或其他任何内容。但是，您通常不需要创建自己的图块数据Blob，而可以使用函数 tilemap_get()要获取切片数据，则可以使用以下功能对其进行操作，然后使用 tilemap_set()。
