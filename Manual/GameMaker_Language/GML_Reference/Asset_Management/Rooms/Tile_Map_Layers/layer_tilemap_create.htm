layer_tilemap_create

使用此功能，您可以将图块集资源分配给要在项目中用作图块贴图的图层。您提供图层ID（使用 layer_create()）或图层名称（作为字符串-这会对性能产生影响），然后是将图块地图添加到房间的初始（x，y）位置，要使用的图块集资源，然后是单元格中的图块地图（即：宽度20和高度10将创建一个包含200个单元格的图块地图，该单元格的宽度为20个单元格，高度为10个单元格），单元格本身的大小由所选的图块集定义。值得注意的是，您无法将图块放置在图块地图内的负位置，因此所有图块必须放置在0到宽度-1、0到高度-1的单元格空间内。

句法：

layer_tilemap_create(layer_id, x, y, tile set, width, height)

争论描述
layer_id要定位的图层的唯一ID值（或图层名称为字符串）
x房间内瓷砖地图的x位置
y房间中瓷砖地图的y位置
tile set要使用的图块集索引
width宽度图块图（在单元格中）
height平铺地图的高度（以单元格为单位）

返回值：

tile map element ID (Real)

例子：

global.back_layer = layer_create(10000);
global.back_tiles = layer_tilemap_create(global.back_layer, 0, 0, tmap_Trees, 16, 32);

上面的代码创建一个新层，然后向其添加一个新的图块地图元素，将图块地图位置设置为（0,0）以及要使用的图块设置以及图块地图的宽度和高度。

背面：平铺地图图层
