layer_tilemap_destroy

此功能将破坏给定的图块地图元素。您提供平铺地图ID（使用以下代码创建平铺地图时会获得该ID：layer_tilemap_create()或与 layer_get_tilemap_id()），则将其删除。请注意，这并不从中取出层，只有瓦片地图，如果瓷砖地图是一个已经在房间编辑器被添加，那么下一次你离开房间，然后返回，瓦片地图将被重新创建再次。但是，如果房间是持久性的，除非再次关闭房间持久性，否则将删除切片地图。

句法：

layer_tilemap_destroy(tilemap_element_id)

争论描述
tilemap_element_id要销毁的图块地图的唯一ID值

返回值：

N/A

例子：

var lay_id = layer_get_id("Tiles_trees");
var tile_id = layer_tilemap_get_id(lay_id);
if layer_tilemap_exists(lay_id, tile_id)
    {
    layer_tilemap_destroy(tile_id);
    }

上面的代码检查“ Tiles_trees”层，以查看给定的图块地图元素是否存在，如果存在，则将其销毁（但不破坏该层）。

背面：平铺地图图层
