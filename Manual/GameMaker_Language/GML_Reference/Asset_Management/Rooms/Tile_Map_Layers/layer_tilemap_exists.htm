层_tilemap_exists

你可以使用这个函数来检查并查看在任何给定的图层上是否存在瓦片贴图元素。你可以提供图层的ID（当你用 "创建 "图层时，你会得到这个ID。 layer_create())或 图层名称（作为一个字符串--这将对性能产生影响）和瓦片贴图元素ID（当你使用 layer_tilemap_create()或当你使用函数 layer_tilemap_get_id())，如果该元素存在，该函数将返回true，如果不存在，则返回false。

注意：该函数在当前目标房间的范围内工作--默认为调用该函数的房间--可以用函数设置 layer_set_target_room().

语法。

layer_tilemap_exists(layer_id, tilemap_element_id)。

争论 描述
layer_id 目标层的唯一ID值（或作为字符串的层名）。
tilemap_element_id 要检查的瓦片地图元素的唯一ID值

返回。

布尔型

例子。

var lay_id = layer_get_id("tilemap_trees");
if layer_tilemap_exists(lay_id, global.Treestilemap)
{
layer_tilemap_destroy(lay_id, global.Treestilemap);
}

上面的代码检查图层 "tilemap_trees"，看给定的瓦片地图元素是否存在，如果存在，那么它就被销毁（但不包括图层）。
