layer_tilemap_get_id

此功能可用于检索图层上图块地图元素的唯一ID值。您提供图层ID（使用 layer_create()）或图层名称（作为字符串-这将对性能产生影响），并且该函数将返回与图层上的平铺地图元素关联的ID值。请注意，此功能是专为与IDE中添加的切片图一起使用而设计的，就好像您使用该功能通过代码将切片图添加到图层上一样 layer_tilemap_create()，那么它将返回添加的图块地图元素的唯一ID。如果给定的地图ID不正确或地图不存在，则该函数将返回-1。

句法：

layer_tilemap_get_id(layer_id)

争论描述
layer_id要定位的图层的唯一ID值（或图层名称为字符串）

返回值：

tile map element ID value (Real) or -1

例子：

var lay_id = layer_get_id("tilemap_trees");
var tile_id = layer_tilemap_get_id(lay_id);
layer_tilemap_destroy(tile_id);

上面的代码将获取名为“ tilemap_trees”的图层的图层ID，然后使用该ID获取该图层上的瓦片地图元素的ID。然后使用该ID从图层中删除图块地图。

背面：平铺地图图层
