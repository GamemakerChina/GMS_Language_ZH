砖块_获取_空

这个函数可以用来检查一个给定的瓦片数据集是否为空瓦片。你给的是瓦片数据，可以用函数来检索 tilemap_get()检索，如果瓦片是空的，该函数将返回true，如果有一个瓦片索引，则返回false。

语法。

tile_get_empty(tiledata)。

争论 描述
平铺数据 要检查的瓦片数据

返回。

布尔型

例子。

var lay_id = layer_get_id（"Tiles_sky"）;
var map_id = layer_tilemap_get_id（lay_id）;
for (var i = 0; i < tilemap_get_width(map_id); i++;)
{
for (var j = 0; j < tilemap_get_height(map_id); j++;)
{
var data = tilemap_get(map_id, i, j);
if !tile_get_empty(data)
{
data = tile_set_empty(data)
tilemap_set(map_id, data, i, j);
}
}
}

上面的代码从给定的图层获得瓦片地图ID，然后继续检查地图上的每一个瓦片单元，看它是否有数据。如果有，该瓦片就被设置为空。
