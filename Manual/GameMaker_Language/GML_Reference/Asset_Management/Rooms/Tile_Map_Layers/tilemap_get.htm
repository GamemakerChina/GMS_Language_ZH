tilemap_get

使用此功能，您可以从图块地图元素的单元格中检索图块数据。您提供了平铺地图元素ID（使用 layer_tilemap_create()或使用该功能时 layer_tilemap_get_id()），以及单元格x和y的位置，该函数将返回图块数据“ blob”。此数据本质上是一个位掩码，其中包含切片索引，翻转/旋转/镜像布尔值以及已应用的任何掩码数据（请参见 tilemap_set_mask()（有关详细信息），然后可以在 tile_*更改图块属性的功能。请注意，我们使用的x / y像元位置是根据图块地图网格的大小而不是在房间中的位置使用的，因此“ cell_x”是从0到图块地图宽度的值，而“ cell_y”是从0为平铺地图高度。如果您需要获取特定房间位置的瓷砖数据，则需要使用该功能 tilemap_get_at_pixel()。

重要的！如果图块地图中的图块未更改（即：它们没有旋转或翻转等），则图块集合数据的返回值将完全等于图块集合上图块的索引。因此，您可以使用图块集合中索引1处的一个图块来创建图块的“碰撞图”（例如），然后使用此功能检查1或0（空图块）以计算碰撞。

句法：

tilemap_get(tilemap_element_id, cell_x, cell_y);

争论描述
tilemap_element_id瓦片地图元素的唯一ID值，用于获取的瓦片数据
x_cell沿x轴获取单元格数据的单元格
y_cell沿y轴获取像元数据的单元格

返回值：

Real (-1 if there is an error)

例子：

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var data = tilemap_get(map_id, 0, 0);
data = tile_set_flip(data, true);
tilemap_set(map_id, data, 0, 0);

上面的代码获取“ Tiles_Sky”层上图块地图的ID，然后使用该ID从单元格（0，0）处的图块获取数据。然后，翻转此图块数据，然后再用于在图块地图上设置图块。
