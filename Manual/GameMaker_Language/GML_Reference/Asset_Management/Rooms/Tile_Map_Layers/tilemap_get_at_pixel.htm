tilemap_get_at_pixel

使用此功能，您可以从图块地图元素的位置（房间内）检索图块数据。您提供了平铺地图元素ID（使用 layer_tilemap_create()或使用该功能时 layer_tilemap_get_id()）以及房间中的x和y位置以获取图块数据，然后该函数将返回图块数据“ blob”。此数据本质上是一个位掩码，其中包含切片索引，翻转/旋转/镜像布尔值以及已应用的任何掩码数据（请参见 tilemap_set_mask()（有关详细信息），然后可以在 tile_*更改图块属性的功能。如果您需要从特定的图块单元获取图块数据，则应使用该函数 tilemap_get()反而。

重要的！如果图块地图中的图块未更改（即：它们没有旋转或翻转等），则图块集合数据的返回值将完全等于图块集合上图块的索引。因此，您可以使用图块集合中索引1处的一个图块来创建图块的“碰撞图”（例如），然后使用此功能检查1或0（空图块）以计算碰撞。

句法：

tilemap_get_at_pixel(tilemap_element_id, x, y);

争论描述
tilemap_element_id瓦片地图元素的唯一ID值，用于获取的瓦片数据
x沿x轴获取瓦片数据的位置（以房间像素为单位）
y沿y轴的位置（以房间像素为单位）以获取图块数据

返回值：

Real (-1 if there is an error)

例子：

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var data = tilemap_get_at_pixel(map_id, 64, 128);
data = tile_set_flip(data, true);
tilemap_set_at_pixel(map_id, data, 64, 128);

上面的代码获取图层“ Tiles_Sky”上的切片地图“ Clouds”的ID，然后使用该ID从位置（64，128）的切片中获取数据。然后，翻转此图块数据，然后再用于在图块地图上设置图块。
