tilemap_get_cell_y_at_pixel

使用此功能，您可以通过给出房间内的相对y轴位置来检索单个图块地图单元格的y轴位置。您提供了平铺地图元素ID（使用 layer_tilemap_create()或使用该功能时 layer_tilemap_get_id()），以及房间内的x和y位置，该函数将返回该点在图块地图中该单元格的y位置。请注意，如果该值在图块地图区域之外，并且没有可用的单元格，则它将返回-1。

句法：

tilemap_get_cell_y_at_pixel(tilemap_element_id, x, y);

争论描述
tilemap_element_id瓦片地图元素的唯一ID值，用于获取的单元格y位置
x房间内用于获取单元格的x位置
y房间中用于获取单元格的y位置

返回值：

Integer (y-axis cell position or -1 if there is an error)

例子：

var lay_id = layer_get_id("Tiles_Walls");
var map_id = layer_tilemap_get_id(lay_id);
var _x = tilemap_get_cell_x_at_pixel(map_id, mouse_x, mouse_y);
var _y = tilemap_get_cell_y_at_pixel(map_id, mouse_x, mouse_y);
tiledata = tilemap_get(map_id, _x, _y);

上面的代码使用检索到的图块地图ID从一组房间坐标中获取单元位置，然后将在其中找到的任何图块的数据存储在实例变量中。

背面：平铺地图图层
