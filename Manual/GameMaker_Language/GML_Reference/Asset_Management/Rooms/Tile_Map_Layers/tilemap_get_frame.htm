瓷砖地图_获取框架

由于瓦片可以被动画化，有时知道当前正在绘制的是哪一帧并作出相应的反应是很有用的，所以用这个函数你可以检索给定瓦片贴图的当前帧索引。你给瓦片贴图元素ID（当你创建一个瓦片贴图时，你会得到这个ID）。 当你用这个函数创建一个瓷砖贴图元素时，你会得到这个ID。 layer_tilemap_create()或使用函数 layer_tilemap_get_id()时得到的），然后该函数将返回 帧索引。

语法。

tilemap_get_frame(tilemap_element_id)。

争论 描述
tilemap_element_id 瓦片贴图元素的唯一ID值，以获得其帧索引。

返回。

实数（介于0（包括）和动画的最大帧数（不包括）之间）。

例子。

var lay_id = layer_get_id("Tiles_Traps");
var map_id = layer_tilemap_get_id(lay_id);
if tilemap_get_frame(map_id) >= 2 && tilemap_get_frame(map_id) < 4
{
global.activated = true;
}
else
{
global.activated = false;
}

上面的代码检查了给定图层上给定瓷砖图的当前动画帧，并根据返回值设置一个全局变量。
