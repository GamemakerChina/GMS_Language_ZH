tilemap_get_mask

此功能可用于检索给定图块映射的位掩码值。您提供了平铺地图元素ID（使用 layer_tilemap_create()或使用该功能时 layer_tilemap_get_id()），函数将返回当前的掩码值；如果有错误，则返回-1；如果未指定掩码，则返回0。有关图块位掩码的更多信息，请参见函数 tilemap_set_mask()。

句法：

tilemap_get_mask(tilemap_element_id);

争论描述
tilemap_element_id瓦片地图元素的唯一ID值，用于获取蒙版

返回值：

Real (0 for no mask, -1 for an error)

例子：

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var mask = tilemap_get_mask(map_id);
var new_mask = tile_mirror | tile_flip | tile_rotate | 255;
if mask != new_mask
    {
    tilemap_set_mask(map_id, new_mask);
    }

上面的代码从给定的图层获取切片地图ID，然后检查与其关联的掩码值。如果它与变量“ new_mask”中定义的值不同，则将其设置为该值。

背面：平铺地图图层
