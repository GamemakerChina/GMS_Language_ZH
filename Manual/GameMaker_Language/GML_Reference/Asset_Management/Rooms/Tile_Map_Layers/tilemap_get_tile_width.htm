tilemap_get_tile_width

使用此功能，您可以检索图块地图元素的每个图块单元的宽度（以像素为单位）。您提供了平铺地图元素ID（使用 layer_tilemap_create()或使用该功能时 layer_tilemap_get_id()），该函数将返回图块设置的单元格宽度。

句法：

tilemap_get_tile_width(tilemap_element_id);

争论描述
tilemap_element_id瓦片地图元素的唯一ID值，用于获取以下元素的瓦片单元格宽度

返回值：

Real

例子：

var lay_id = layer_get_id("Tiles_Walls");
var map_id = layer_tilemap_get_id(lay_id);
global.snap_x = tilemap_get_tile_width(map_id);
global.snap_y = tilemap_get_tile_height(map_id);

上面的代码使用检索到的图块地图ID来获取图块单元的宽度和高度，并使用它们来设置两个全局变量。

背面：平铺地图图层
