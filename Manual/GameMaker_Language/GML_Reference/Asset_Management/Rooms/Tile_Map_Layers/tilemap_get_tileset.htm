tilemap_get_tileset

使用此功能，您可以检索分配给图层上给定图块地图元素的图块集资源的索引值。您提供了平铺地图元素ID（使用 layer_tilemap_create()或使用该功能时 layer_tilemap_get_id()），该函数将返回图块设置索引值。

句法：

tilemap_get_tileset(tilemap_element_id);

争论描述
tilemap_element_id用于从中获取图块集的图块地图元素的唯一ID值

返回值：

Tile Set index value (Real)

例子：

var lay_id = layer_get_id("Tiles_trees");
var map_id = layer_tilemap_get_id(lay_id);
if tilemap_get_tileset(map_id) != ts_Nighttime
    {
    tilemap_tileset(map_id, ts_Nighttime);
    }

上面的代码检查分配给图层“ Tiles_trees”的当前图块集，如果不是“ ts_Nighttime”，则将该图块集分配给图块地图。

背面：平铺地图图层
