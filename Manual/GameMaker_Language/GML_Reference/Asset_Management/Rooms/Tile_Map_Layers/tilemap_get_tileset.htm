tilemap_get_tileset

使用这个函数，你可以检索分配给一个图层上的特定瓷砖贴图元素的瓷砖集资源的索引值。你给瓦片贴图元素ID（当你用 "创建 "瓦片贴图元素时，你会得到这个ID）。 layer_tilemap_create() 或者当你使用函数 layer_tilemap_get_id()时得到的），该函数将返回瓦片集的索引值。

语法。

tilemap_get_tileset(tilemap_element_id)。

争论 描述
tilemap_element_id 瓷砖地图元素的唯一ID值，用于获取瓷砖组。

返回。

瓦片设置索引值（Real

例子。

var lay_id = layer_get_id("Tiles_trees");
var map_id = layer_tilemap_get_id(lay_id);
if tilemap_get_tileset(map_id) != ts_Nighttime
{
tilemap_tileset(map_id, ts_Nighttime);
}

上面的代码检查当前分配给 "Tiles_trees "层的瓦片集，如果它不是 "ts_Nighttime"，那么该瓦片集就被分配给瓦片图。
