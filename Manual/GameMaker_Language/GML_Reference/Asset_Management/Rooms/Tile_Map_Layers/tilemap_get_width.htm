tilemap_get_width

使用此功能，您可以检索图块地图元素的宽度（以单元为单位）。您提供了平铺地图元素ID（使用 layer_tilemap_create()或使用该功能时 layer_tilemap_get_id()），该函数将返回宽度（单元格宽度）。

句法：

tilemap_get_width(tilemap_element_id);

争论描述
tilemap_element_id瓦片地图元素的唯一ID值，用于获取

返回值：

Integer

例子：

var lay_id = layer_get_id("Tiles_Walls");
var map_id = layer_tilemap_get_id(lay_id);
var _w = tilemap_get_width(map_id);
var _h = tilemap_get_height(map_id);
total_tiles = (_w * _h);

上面的代码使用检索到的图块地图ID来获取图块地图的图块宽度和高度，然后使用这些值来设置实例变量。

背面：平铺地图图层
