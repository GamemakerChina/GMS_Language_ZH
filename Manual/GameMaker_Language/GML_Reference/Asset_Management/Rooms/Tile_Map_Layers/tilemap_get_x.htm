tilemap_get_x

使用此功能，您可以检索平铺地图元素的x位置（在房间内）。您提供了平铺地图元素ID（使用 layer_tilemap_create()或使用该功能时 layer_tilemap_get_id()），该函数将返回x轴位置。

句法：

tilemap_get_x(tilemap_element_id);

争论描述
tilemap_element_id瓦片地图元素的唯一ID值，用于获取x的位置

返回值：

Real

例子：

var lay_id = layer_get_id("Tiles_Walls");
var map_id = layer_tilemap_get_id(lay_id);
var _x = tilemap_get_x(map_id);
var _y = tilemap_get_y(map_id);
tilemap_x(map_id, _x + 10); tilemap_y(map_id, _y + 10);

上面的代码使用检索到的图块地图ID来获取图块地图的图块x和y位置，然后使用这些值将其移动。

背面：平铺地图图层
