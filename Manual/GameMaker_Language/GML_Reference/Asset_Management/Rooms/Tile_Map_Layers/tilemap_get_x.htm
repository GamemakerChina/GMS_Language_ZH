瓦片图_获取_x

使用这个函数，你可以检索到瓷砖贴图元素的X位置（在房间内）。你给瓦片贴图元素的ID（当你用 "创建 "瓦片贴图元素时，你会得到这个ID。 layer_tilemap_create() 或者当你使用函数 layer_tilemap_get_id())，函数将返回X轴的位置。

语法。

tilemap_get_x(tilemap_element_id)。

争论 描述
tilemap_element_id 瓷砖地图元素的唯一ID值，以获得其X位置。

返回。

真实的

例子。

var lay_id = layer_get_id("Tiles_Walls");
var map_id = layer_tilemap_get_id(lay_id);
var _x = tilemap_get_x(map_id);
var _y = tilemap_get_y(map_id);
tilemap_x(map_id, _x + 10) ; tilemap_y(map_id, _y + 10);

上面的代码使用检索到的瓦片地图ID来获得瓦片地图的x和y位置，然后使用这些值来移动它。
