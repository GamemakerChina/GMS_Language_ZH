瓷砖图案_设置

这个函数可以用来将图层上的瓦片贴图元素中的任何单元（网格方块）设置为一个新的瓦片。你给瓦片贴图元素ID（当你用 "创建 "瓦片贴图元素时，你会得到这个ID）。 layer_tilemap_create() 或当你使用函数 layer_tilemap_get_id()时得到的），然后是要设置的瓦片数据以及在瓦片地图中的位置。你可以使用函数检索瓦片数据 tilemap_get() 检索瓦片数据，然后在使用此函数设置单元格之前使用tile_函数来改变它。如果瓦片被成功设置，该函数将返回true；如果有问题，没有被设置，则返回false。

语法。

tilemap_set(tilemap_element_id, tiledata, xcell, ycell)

争论 描述
tilemap_element_id 要改变的瓦片地图元素的唯一ID值
平铺数据 要设置的瓦片数据
xcell 要沿X轴设置的单元格（网格）位置
ycell 要沿Y轴设置的单元格（网格）位置

返回。

布尔型

例子。

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var data = tilemap_get(map_id, 0, 0);
data = tile_set_flip(data, true);
tilemap_set(map_id, data, 0, 0);

上面的代码得到了 "Tiles_Sky "层上的瓷砖地图的ID，然后用它来获取单元格（0，0）处的瓷砖数据。这个瓦片数据在被用于再次设置瓦片地图上的瓦片之前被翻转。
