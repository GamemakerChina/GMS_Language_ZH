tilemap_set_at_pixel

此功能可用于通过使用图块的实际位置在房间内更改将图层上图块地图元素内的单元格设置为新图块。您提供了平铺地图元素ID（使用 layer_tilemap_create()或使用该功能时 layer_tilemap_get_id()），然后设置图块数据以及房间内的位置。您可以使用函数检索图块数据 tilemap_get_at_pixel()然后使用 tile_使用此功能进行更改之前，请先进行更改。该函数将返回 true如果磁贴设置成功，并且 false如果有问题，但没有解决。

句法：

tilemap_set_at_pixel(tilemap_element_id, tiledata, x, y)

争论描述
tilemap_element_id要更改的图块地图元素的唯一ID值
tiledata要设置的图块设置数据
xcell x位置（在房间中）
ycell y位置（在房间中）

返回值：

Boolean

例子：

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var data = tilemap_get_at_pixel(map_id, 4, 4);
data = tile_set_flip(data, true);
tilemap_set_at_pixel(map_id, data, 4, 4);

上面的代码获取“ Tiles_Sky”层上图块地图的ID，然后使用该ID从位置（4，4）上的图块获取数据。然后，翻转此图块数据，然后再用于在图块地图上设置图块。

背面：平铺地图图层
