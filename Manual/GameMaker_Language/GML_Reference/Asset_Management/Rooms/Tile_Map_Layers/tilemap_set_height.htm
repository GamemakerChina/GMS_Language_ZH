tilemap_set_height

此功能可用于调整图块地图元素的大小。您提供了平铺地图元素ID（使用 layer_tilemap_create()或使用该功能时 layer_tilemap_get_id()），以及图块单元格中新的图块高度。

句法：

tilemap_set_height(tilemap_element_id, height)

争论描述
tilemap_element_id瓦片地图元素的唯一ID值，用于设置高度
height高度值（以“单元格”为单位）

返回值：

N/A

例子：

var lay_id = layer_get_id("Tiles_Walls");
var map_id = layer_tilemap_get_id(lay_id);
if tilemap_get_height(map_id) != room_height div 16
    {
    tilemap_set_height(map_id, room_height div 16);
    }

上面的代码检查特定图块贴图的宽度，如果尺寸不正确，则设置宽度。
