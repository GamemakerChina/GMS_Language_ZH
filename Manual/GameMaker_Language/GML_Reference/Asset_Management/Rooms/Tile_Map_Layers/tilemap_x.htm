瓦片图_x

这个函数控制层上资产瓦片地图元素的房间沿X轴的位置。你给瓦片贴图元素ID（当你用 "创建 "瓦片贴图元素时，你会得到这个ID。 layer_tilemap_create() 或者当你使用函数 layer_tilemap_get_id())，然后设置要使用的x值(基于房间坐标)。

语法。

tilemap_x(tilemap_element_id, x)。

争论 描述
tilemap_element_id 要改变的瓦片地图元素的唯一ID值
x 瓦片图的X位置

返回。

不适用

例子。

var lay_id = layer_get_id("Asset_sky");
var map_id = layer_tilemap_get_id(lay_id);
tilemap_x(map_id, irandom(room_width)) 。

上面的代码得到分配给 "Asset_sky "层的瓦片地图资产的ID值，然后将其X位置设置为0和房间宽度之间的一个随机值。
