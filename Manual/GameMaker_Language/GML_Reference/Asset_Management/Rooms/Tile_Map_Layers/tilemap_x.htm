tilemap_x

此功能控制图层上资产图块地图元素沿房间x轴的位置。您提供了平铺地图元素ID（使用 layer_tilemap_create()或使用该功能时 layer_tilemap_get_id()），然后设置要使用的x值（基于房间坐标）。

句法：

tilemap_x(tilemap_element_id, x);

争论描述
tilemap_element_id要更改的图块地图元素的唯一ID值
x平铺地图的x位置

返回值：

N/A

例子：

var lay_id = layer_get_id("Asset_sky");
var map_id = layer_tilemap_get_id(lay_id);
tilemap_x(map_id, irandom(room_width));

上面的代码获取分配给图层“ Asset_sky”的瓦片地图资产的ID值，然后将其x位置设置为介于0和房间宽度之间的随机值。

背面：平铺地图图层
