room

此变量保存游戏正在运行的当前房间的房间索引。这不是只读变量，但是更改此值不会更改当前房间的索引值，而是更改房间以匹配将变量设置为的索引。这样做时要小心，好像将变量更改为的索引无效时，游戏将抛出错误并关闭。通常，使用更好的做法 room_goto()换房间。

句法：

room;

返回值：

Real (unique index value)

例子：

if room_next(room) != -1
    {
    room_goto(room_next);
    }

上面的代码将检查下一个房间是否存在，如果存在，它将进入该房间。

返回：房间
