房间

这个变量持有你的游戏正在运行的当前房间的索引。这不是一个只读变量，但是改变这个变量不会改变当前房间的索引值，而是改变房间，使其与你设置的索引相匹配。 你设置的变量的索引。在这样做的时候应该小心，因为如果你改变变量的索引是无效的，游戏将抛出一个错误并关闭。一般来说，使用 room_goto() 来改变房间。

注意：房间ID不是基于它们在资产浏览器或房间管理器中的顺序，因此你应该避免直接提供一个数字值。取而代之的是，使用房间常数，用于 你想引用的资产的房间常量（这只是它的名字）或通过一个函数来检索它。

语法。

房间。

返回。

真实（唯一的索引值）。

例子。

if (room == rm_level1)
{
audio_play_sound(snd_level1, 1, 1);
}

上面的代码将检查当前房间是否是第一层，在这种情况下，播放该层的音乐。
