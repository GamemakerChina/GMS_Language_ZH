room_exists

使用此功能，您可以检查并查看指定的房间是否存在。此函数使用房间索引（实数）而不是房间名称（字符串）。使用此功能动态创建房间时，此功能最有用 room_add()，但您也可以使用只读变量 room_first和 room_last或功能 room_next()和 room_previous()以获得特定的房间索引，或提供一个变量，该变量已存储游戏中其他任何房间的索引。

句法：

room_exists(index);

争论描述
index要检查的房间的索引。

返回值：

Boolean

例子：

if room_exists(global.rm[0])
    {
    room_goto(global.rm[0]);
    }

上面的代码检查是否在数组“ global.rm []”中索引的房间存在，如果存在，则转到该房间。

返回：房间
