房间_存在

通过这个函数，你可以检查并查看你指定的房间是否存在。这个函数接受房间索引（实数）而不是房间名称（字符串）。这个函数在你动态地创建房间时最有用。 使用函数 room_add()来动态创建房间时，这个函数最为有用，但你也可以使用只读变量 room_first和 room_last或函数 room_next() 和 room_previous()来获得一个特定的房间索引，或者提供一个存储了你游戏中任何其他房间索引的变量。

语法。

room_exists(index)。

争论 描述
指数 要检查的房间的索引。

返回。

布尔型

例子。

if room_exists(global.rm[0])
{
room_goto(global.rm[0])。
}

上面的代码检查 "global.rm[]"数组中索引的房间是否存在，如果存在，它就去那个房间。

返回。房间
