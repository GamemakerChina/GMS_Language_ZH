room_get_camera

使用此功能，您可以获取分配给当前房间以外的特定视图的摄像机的唯一索引ID。您提供了要使用的空间，要使用的视口（从0到7），该函数将返回摄像机索引。

句法：

room_get_camera(rm, vind);

争论描述
rm房间的索引以获取视图摄像机
vind用于获取摄像机的视口的索引

返回值：

Real (Camera Index)

例子：

var cam = room_get_camera(rm_Game, 0);
if cam != global.MainCam
    {
    room_set_camera(rm_Game, 0, global.MainCam);
    }

上面的代码在新创建的房间中分配了一个摄像机来查看端口[0]。
