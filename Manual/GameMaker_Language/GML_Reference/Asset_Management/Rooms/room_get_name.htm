room_get_name

此函数可用于以字符串形式返回指定房间的名称。请注意，这只是一个字符串，不能用于直接引用房间-因为您将需要房间索引。但是，您可以使用此字符串通过返回的字符串和函数来获取房间索引 asset_get_index()。

句法：

room_get_name(index);

争论描述
index要检查名称的房间的索引。

返回值：

String

例子：

var roomname = room_get_name(room);
draw_text(32, 32, roomname);

上面的代码将获取当前房间的名称并将其绘制在屏幕上。

返回：房间
