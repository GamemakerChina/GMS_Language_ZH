房间名称（Room_get_name

这个函数可以用来返回指定房间的名称，是一个字符串。请注意，这只是一个字符串，不能用来直接引用房间--为此，你需要房间的索引。然而，你可以使用这个 字符串来获取房间的索引，使用返回的字符串和函数 asset_get_index().

语法。

room_get_name(index)。

争论 描述
指数 要检查的房间名称的索引。

返回。

字符串

例子。

var roomname = room_get_name(room);
draw_text(32, 32, roomname);

上面的代码将获得当前房间的名称并在屏幕上画出来。

返回。房间
