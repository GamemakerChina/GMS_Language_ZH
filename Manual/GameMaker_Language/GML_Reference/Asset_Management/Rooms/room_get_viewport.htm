房间里的视口

通过这个函数，你可以检索到当前房间以外的一个视口的详细信息。你给出房间的ID和要检索的视口的索引（从0到7），这个函数将返回一个包含5个索引的数组，其中。

  • [0] = visible (布尔值：真/假)
  • [1] = X位置（实际）。
  • [2] = Y位置（实数）。
  • [3] = 宽度（实数
  • [4] = 高度（实数

语法。

room_get_viewport(rm, vind)。

争论 描述
rm 要获取视口数据的房间的索引
vind 要获取的视图端口的索引

返回。

数组（5个元素：visible, x, y, width, height）。

例子。

v_vals = room_get_viewport(rm_Game, 0);
if v_vals[0] == false
{
room_set_view(rm_Game, true, v_vals[1], v_vals[2], v_vals[3], v_vals[4);
}

上面的代码检索给定房间的视图端口数据，然后检查该端口是否被标记为可见。如果不是，则对视图端口数据进行设置，使其可见。

返回。房间
