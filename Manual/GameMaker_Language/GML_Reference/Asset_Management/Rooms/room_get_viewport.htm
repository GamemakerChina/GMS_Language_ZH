room_get_viewport

使用此功能，您可以在当前房间以外的房间中检索视口的详细信息。您提供房间ID和要检索的查看端口的索引（从0到7），该函数将返回5个索引的数组，其中：

  • [0] =可见（布尔值：true /false ）
  • [1] = x位置（真实）
  • [2] = y位置（真实）
  • [3] =宽度（真实）
  • [4] =高度（真实）

句法：

room_get_viewport(rm, vind);

争论描述
rm从中获取视口数据的房间索引
vind要获取的查看端口的索引

返回值：

Array (5 elements: visible, x, y, width, height)

例子：

v_vals = room_get_viewport(rm_Game, 0);
if v_vals[0] == false
    {
    room_set_view(rm_Game, true, v_vals[1], v_vals[2], v_vals[3], v_vals[4]);
    }

上面的代码检索给定房间的查看端口数据，然后检查该端口是否标记为可见。如果不是，则将视口数据设置为可见。

返回：房间
