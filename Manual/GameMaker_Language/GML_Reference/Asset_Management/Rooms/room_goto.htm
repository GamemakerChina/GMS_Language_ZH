房间_goto

这个函数允许你进入你的游戏项目中的任何房间，无论是用代码还是在资产浏览器中创建的。你提供房间的索引（存储在房间名称的变量中，或者作为一个变量从函数中返回 room_add()). 注意，在调用该函数的事件结束之前，房间不会发生变化，所以如果在同一事件中，调用该函数后的任何代码仍将运行。需要注意的是，任何创建实例的代码在调用这个函数后 后调用，将不会创建实例，所以应该在改变房间之前调用。这个函数也会触发房间结束事件。

注意：房间ID不是基于它们在资产浏览器或房间管理器中的顺序，因此你应该避免直接提供一个数字值。取而代之的是，使用房间常数，用于 你想引用的资产的房间常量（这只是它的名字）或通过一个函数来检索它。

语法。

room_goto(index)。

争论 描述
指数 要去的房间的索引。

返回。

不适用

例子。

switch (global.level)
{
case 0: room_goto(rm_level1); break;
case 1: room_goto(rm_level2); break;
case 2: room_goto(rm_level3); break;
}

上面的代码将检查一个全局变量，并根据其持有的值改变房间。

返回。房间
