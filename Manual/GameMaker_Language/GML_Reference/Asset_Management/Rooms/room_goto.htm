room_goto

此功能允许您进入游戏项目中的任何房间，无论是使用代码还是在资源浏览器中创建的。您提供唯一的房间ID值（存储在房间名称的变量中，或者作为从函数返回的变量存储）room_add() ）。请注意，直到调用该函数的事件结束之前，房间才会改变，因此，如果在同一事件中，调用此函数之后的任何代码仍将运行。需要注意的是，任何在此之后创建实例的代码都不会创建该实例，因此应在更衣室之前调用它。此功能还将触发“房间结束”事件。

句法：

room_goto(index);

争论描述
index房间的索引。

返回值：

N/A

例子：

switch (global.rm)
    {
    case 0: room_goto(rm_level1); break;
    case 1: room_goto(rm_level2); break;
    case 2: room_goto(rm_level3); break;
    }

上面的代码将检查一个全局变量，并根据其持有的值更改房间。

返回：房间
