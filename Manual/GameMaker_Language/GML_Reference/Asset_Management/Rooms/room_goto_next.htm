room_goto_next

使用此功能，您可以使游戏进入编译游戏时房间管理器中列出的下一个游戏。如果这个房间不存在，将引发错误，游戏将被迫关闭。请注意，直到调用该函数的事件结束之前，房间才会改变，因此，如果在同一事件中，调用此函数之后的任何代码仍将运行。需要注意的是，任何在此之后创建实例的代码都不会创建该实例，因此应在更衣室之前调用它。此功能还将触发“房间结束”事件。

句法：

room_goto_next();

返回值：

N/A

例子：

if room_exists(room_next(room))
    {
    room_goto_next();
    }

上面的代码将检查当前房间之后是否还有另一个房间，如果有，它将转到该房间。

返回：房间
