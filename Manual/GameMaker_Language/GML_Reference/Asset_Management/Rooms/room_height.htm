room_height

此变量保存当前房间的高度（以像素为单位）。您可以随时更改此变量以更改房间的高度，并且更改将应用于房间的底部，因为原点被视为左上角。因此，例如，如果房间的高度为480px，而您将其设置为640px，则房间将向下扩展，并在底部增加了180px。

句法：

room_height;

返回值：

Real

例子：

if bbox_bottom > room_height
   {
   y += room_height - bbox_bottom;
   }

上面的代码检查当前实例的sprite边界框是否大于房间的高度，如果是，则向上移动实例。

返回：房间
