room_instance_add

使用此功能，您可以将实例添加到当前房间以外的任何房间中，并且可以将该实例添加到该房间中的任何位置。该函数返回唯一 id实例的实例，然后可以在进一步的函数调用中使用该实例来设置实例的属性等，但前提是游戏已经进入指定的房间。如果要在当前房间中创建实例，则应使用该功能 instance_create_layer()。请注意，在资源浏览器中创建的会议室资产上调用此函数会将实例永久添加到会议室中，甚至调用 game_restart()不会将房间恢复到其原始状态（仅结束游戏并再次打开它会再次使房间恢复其原始状态）。

句法：

room_instance_add(index, x, y, obj);

争论描述
index要向其添加对象实例的房间的索引。
x新实例的x位置。
y新实例的y位置。
obj要添加实例的对象。

返回值：

Instance ID

例子：

global.rm = room_add();
room_assign(rm_Base, global.rm);
room_instance_add(global.rm, 100, 100, obj_Player);

上面的代码将为游戏添加一个新房间，然后将索引为“ rm_Base”的房间的内容复制到其中。然后，它将在此新房间的位置100,100处添加对象“ obj_player”的实例。

返回：房间
