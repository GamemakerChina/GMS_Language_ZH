room_last

此只读变量返回游戏中最后一个房间的索引（这是由房间在“房间管理器”中的显示顺序而不是它们的创建顺序定义的）。请注意，此变量将无法识别或考虑使用来动态添加的会议室room_add()或者room_duplicate()。

句法：

room_last;

返回值：

Index

例子：

if keyboard_check_pressed(ord("Q"))
    {
    room_goto(room_last);
    }

上面的代码检查是否已按下某个键，如果已按下，则转到游戏的最后一个房间。

返回：房间
下一个：room_next
