最后一个房间

这个只读变量返回游戏中最后一个房间的索引（这是由房间管理器中房间出现的顺序决定的，而不是由房间创建的顺序决定的）。 而不是它们被创建的顺序）。)请注意，这个变量不会识别或考虑到使用room_add() 或room_duplicate() 动态添加的房间。

语法。

room_last;

返回。

索引

例子。

if keyboard_check_pressed(ord("Q"))
{
room_goto(room_last);
}

上面的代码检查是否有一个键被按下，如果有，就会进入游戏的最后一个房间。
