room_next

使用此功能，您可以在将房间输入功能之后检索房间的索引。例如，您可以使用变量 room以获取当前房间的索引，然后使用此功能查找其后的房间，如“房间管理器”中所列。如果输入的房间之后没有空间，则返回-1。请注意，此功能将无法识别或不考虑使用来动态添加的房间room_add()或者room_duplicate()。

句法：

room_next(numb);

争论描述
numb要获得下一个索引的房间的索引。

返回值：

Index

例子：

if room_next(room) != -1
    {
    room_goto_next());
    }

上面的代码将检查下一个房间是否存在，如果存在，它将进入该房间。

返回：房间
