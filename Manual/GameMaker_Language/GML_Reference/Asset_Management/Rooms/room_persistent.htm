持续性的房间

这个变量可以用来获取和设置当前房间的持久化标志。如果设置为"true"，房间将被认为是持久的，在这种情况下，每次你离开房间并再次回来时，该房间内的实例的状态 将被保持。然而，如果它被标记为false，每次你回到房间时，它将被重置为初始状态。你应该注意，一个持久化的房间比一个普通的房间使用更多的内存，所以不建议 不建议在你的游戏中拥有太多这样的房间。

语法。

room_persistent。

返回。

布尔型

例子。

如果生命<1
{
room_persistent = false;
room_goto(rm_start) 。
}

上面的代码检查变量 "lives"，如果它小于1，它将把房间的持久性设置为假，然后更换房间。
