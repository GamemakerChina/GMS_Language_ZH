room_persistent

此变量可用于获取和设置当前房间的持久性标志。如果设置为 true该房间被认为是永久性的，在这种情况下，每次离开该房间并再次返回时，该房间内实例的状态将保持不变。但是，如果将其标记为 false，每次您返回房间时，它都会重置为初始状态。您应该注意，持久性空间使用的内存要比普通空间多得多，因此建议您不要在游戏中使用过多的内存。

句法：

room_persistent;

返回值：

Boolean

例子：

if lives < 1
    {
    room_persistent = false;
    room_goto(rm_start);
    }

上面的代码检查变量“ lives”，如果它小于1，它将把房间持久性设置为 false然后换房间。

返回：房间
