room_set_camera

使用此功能，您可以将摄像机分配到当前房间以外的其他特定视口。您提供房间索引，视图索引（从0到7），然后提供要使用的摄像机的索引。

句法：

room_set_camera(rm, vind, camera);

争论描述
rm设置观察摄像机的房间索引
vind视口的索引，用于将摄像机分配给
camera要分配的摄像机索引

返回值：

N/A

例子：

global.myroom = room_add();
room_set_camera(global.myroom, 0, global.MainCam);

上面的代码在新创建的房间中分配了一个摄像机来查看端口[0]。

返回：房间
