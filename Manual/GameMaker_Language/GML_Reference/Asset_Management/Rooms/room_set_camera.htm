房间_设置摄像机

通过这个功能，你可以把摄像机分配到当前房间以外的特定视口。你提供房间索引、视图索引（从0到7），然后是要使用的摄像机的索引。

语法。

room_set_camera(rm, vind, camera)。

争论 描述
rm 要设置摄像机的房间的索引。
vind 要分配给摄像机的视口的索引。
摄像机 要分配的摄像机的索引

返回。

不适用

例子。

global.myroom = room_add();
room_set_camera(global.myroom, 0, global.MainCam);

上面的代码将一个新创建的房间里的摄像机分配到视口[0]。
