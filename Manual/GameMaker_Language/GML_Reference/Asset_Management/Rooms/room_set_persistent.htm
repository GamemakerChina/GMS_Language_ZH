room_set_persistent

使用此功能，您可以更改（或设置）游戏中除当前房间以外的任何房间的持久性。具有持久性的房间标记为 true如果玩家离开然后返回，并且持久性被标记为，它将保持该房间内所有实例的状态。false每次都会将其重置为初始状态。您应该注意，持久性空间使用的内存要比普通空间多得多，因此建议您不要在游戏中使用过多的内存。

注意：如果房间已被访问，此功能将不起作用关掉坚持！一旦访问了持久性房间，该持久性房间就会保存在内存中，要关闭持久性，您应该转到该房间并将room_persistent变量设置为 false然后再次离开房间。

句法：

room_set_persistent(index, val);

争论描述
index要设置持久性的房间的索引。
val房间是否应保持不变（true ） 或不 （false ）。

返回值：

N/A

例子：

global.myroom = room_add();
room_set_width(global.myroom, 640);
room_set_height(global.myroom, 480);
room_set_persistent(global.myroom, true);

这将创建一个新房间并将其索引存储在变量“ global.myroom”中。然后，将其宽度设置为640像素，将其高度设置为480像素，将其标题设置为“游戏室”，并将其持久性设置为“真”。

返回：房间
