room_set_view_enabled

使用此功能，您可以启用（true ）或禁用（false ）您游戏中除当前房间外的任何房间的景象。

句法：

room_set_view_enabled(index, val);

争论描述
index要设置的房间的索引。
val是否启用（true ）或禁用（false ）在给定房间的景色。

返回值：

N/A

例子：

global.myroom = room_add();
room_set_view_enabled(global.myroom, true);

这将启用索引在“ global.myroom”中的房间中的视图。

返回：房间
