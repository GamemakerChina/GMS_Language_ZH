启用的房间_设置视图

通过这个功能，你可以启用（真）或禁用（假）你游戏中任何房间的视图，除了当前房间。

语法。

room_set_view_enabled(index, val)。

争论 描述
指数 要设置的房间的索引。
价值 是否启用（true）或禁用（false）给定房间的视图。

返回。

不适用

例子。

global.myroom = room_add();
room_set_view_enabled(global.myroom, true);

这将使在 "global.myroom "中索引的房间的视图成为可能。

返回。房间
