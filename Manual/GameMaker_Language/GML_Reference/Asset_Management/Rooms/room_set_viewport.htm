room_set_viewport

使用此功能，可以为游戏中除当前房间以外的任何房间设置视口属性。您提供房间索引以在其中设置查看端口，查看端口索引（从0到7），然后提供查看端口是否应在房间开始时可见（设置为 true使端口可见，并且 false否则）。之后，设置x和y坐标（对应于视口的左上位置），然后设置宽度和高度（以像素为单位）。

句法：

room_set_viewport(rm, vind, vis, xport, yport, wport, hport);

争论描述
rm要设置的房间的索引
vind要设置的查看端口的索引
vis视口的可见性（true可见 false is invisible）
xport房间中视口的x位置
yport房间中视口的y位置
wport视口的宽度（以像素为单位）
hport视口的高度（以像素为单位）

返回值：

N/A

例子：

global.myroom = room_add();
room_set_width(global.myroom, 640);
room_set_height(global.myroom, 480);
room_set_viewport(global.myroom, 0, true, 0, 0, 640, 480);

这将在变量“ global.myroom”中索引的房间中设置查看端口[0]属性。

返回：房间
