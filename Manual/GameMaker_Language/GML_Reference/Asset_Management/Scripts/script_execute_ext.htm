脚本_执行_ext

这个函数的工作原理类似于函数script_execute()，只是你可以提供一个包含被调用的函数/脚本所需参数的数组。你还可以向该函数提供两个可选的参数，以指定在该函数中的偏移量。 还可以向该函数提供两个可选的参数，以指定从数组中获取参数的偏移量，以及从数组中使用的参数数量（这必须是array_length - offset的最大值）。

语法。

script_execute(scr, array_args, [offset], [num_args)。

争论 描述
擦洗 你想调用的函数/脚本的名称
array_args 包含函数/脚本参数的数组
[偏移] OPTIONAL!参数数组中的偏移量
[num_args] OPTIONAL!的参数数量（从偏移量开始）。

返回。

将取决于被调用的函数/脚本的返回值

例子。

script_execute_ext(move_inst, move_array, floor(random(4)), 1)。

上面的示例代码将使用script_execute_ext来调用给定的函数，提供一个包含四个参数的数组，但只随机使用其中的一个参数。

注意：此功能不能与内置功能一起使用。
