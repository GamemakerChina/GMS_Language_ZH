script_execute_ext

该函数的工作方式与script_execute（）类似，只是您可以提供一个数组，其中包含要调用的函数/脚本所需的参数。您还可以向函数提供两个可选参数，以指定要从中获取参数的数组中的偏移量，以及要从数组中使用的参数数量（这必须是array_length-offset的最大值）。

句法：

script_execute(scr, array_args, [offset], [num_args]);

争论描述
scr您要调用的函数/脚本的名称
array_args包含函数/脚本参数的数组
[offset]选修的！参数数组中的偏移量
[num_args]选修的！ （从偏移量开始）自变量的数量

返回值：

Will depend on the return value from the function/script being called

例子：

script_execute_ext(move_inst, move_array, floor(random(4)), 1);

上面的示例代码将使用 script_execute_ext调用给定函数，提供四个参数的数组，但只能随机使用其中一个。

注意：此功能不能与内置功能一起使用。
