script_exists

该函数将返回 true或者 false取决于是否存在具有给定索引的脚本或脚本函数。请注意，这不是字符串，而是资产名称，该资产名称包含每个脚本的唯一索引（将在IDE中显示）或脚本函数名称变量（如脚本资产中定义的那样）（请注意，这不会为方法变量工作）。有关脚本的更多信息，请参见脚本编辑器。

句法：

script_exists(scr);

争论描述
scr您要检查的脚本的名称。

返回值：

Boolean

例子：

script[0] = -1;
script[1] = AI_Left;
script[2] = AI_Right;
var script_num = choose(0, 1, 2);
if script_exists(script[script_num])
    {
    script_execute(script[script_num]);
    }

上面的示例将两个脚本函数和一个值添加到数组中，然后继续获取随机数，并使用该随机数选择要运行的脚本函数，除非选择了-1，在这种情况下将不会发生任何事情。
