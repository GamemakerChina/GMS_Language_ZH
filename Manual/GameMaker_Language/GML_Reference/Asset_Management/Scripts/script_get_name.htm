脚本名称

这个函数将返回指定脚本的名称，即字符串。这个名字是在GameMaker Studio 2主窗口的资产浏览器中为脚本指定的名字。 关于脚本的更多信息，请参见《脚本编辑器》。

语法。

script_get_name(scr)。

争论 描述
擦洗 你想得到的脚本名称的索引。

返回。

字符串

例子。

scr_name = script_get_name(Help_File)。

上面的示例代码将在变量 "scr_name "中存储指定脚本索引的名称。

返回。脚本
