script_get_name

该函数将名称作为指定脚本的字符串返回。 GameMaker Studio 2主窗口的“资源浏览器”中为脚本指定的名称。有关脚本的更多信息，请参见脚本编辑器。

句法：

script_get_name(scr);

争论描述
scr您要获取名称的脚本的索引。

返回值：

String

例子：

scr_name = script_get_name(Help_File);

上面的示例代码将指示的脚本索引的名称存储在变量“ scr_name”中。
