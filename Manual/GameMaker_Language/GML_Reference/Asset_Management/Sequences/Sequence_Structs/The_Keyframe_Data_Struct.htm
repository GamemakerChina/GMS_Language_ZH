Keyframe数据结构

在keyframe结构时，还需要定义将与之关联的数据。它由不同的通道组成，并且可以根据要为其keyframe数据的轨道的类型，为每个通道分配特定类型的数据。这些属性是：

  • channel ：这是keyframe数据应应用到的通道。它是一个从0开始的正整数值，值得注意的是，在为“位置”或“比例” keyframes创建参数轨迹时，则需要使用非常特定的通道值。它们是：通道0是X位置或X刻度，通道1是Y位置或Y刻度。
  • spriteIndex ：用于轨道的Sprite资产的索引。此属性仅适用于以下类型的轨道 seqtracktype_graphic您可以获取或设置它。
  • soundIndex ：用于轨道的音频资产的索引。此属性仅适用于以下类型的轨道 seqtracktype_audio您可以获取或设置它。
  • playbackMode ：声音的播放模式。此属性仅适用于以下类型的轨道 seqtracktype_audio您可以获取或设置它。该属性应为以下常量之一：
持续的描述价值
seqaudiokey_loop播放时声音会循环播放。 0
seqaudiokey_oneshot声音只会播放一次，然后停止。 1

  • curve ：此属性需要动画曲线结构（有关更多信息，请参见此处），并且仅适用于该类型的轨道 seqtracktype_real。如果未使用曲线结构，则此属性的值为-1。
  • value ：此属性只是与keyframe数据通道关联的值，并且仅可用于以下类型的轨道 seqtracktype_real当没有 curve提供了struct。
  • colour ：此属性返回（或要求，如果已设置） keyframe颜色值的数组，格式为[A，R，G，B]。仅适用于以下类型的轨道 seqtracktype_colour。请注意，每个分量的值应表示为0到1，其中0对应于十六进制值＃00，1对应于十六进制值#FF（如序列中颜色轨道的颜色选择器所示，0-255编辑）。
  • sequence ：此属性将返回（或要求，如果已设置）序列对象结构，并且仅可用于该类型的轨道 seqtracktype_sequence。
  • objectIndex ：此属性将返回（或要求，如果已设置）对象索引，并且仅可用于以下类型的轨道 seqtracktype_instance。
  • events ：此属性允许访问与keyframe数据结构关联的事件和广播消息。您可以获取或设置此属性，获取时将返回一个字符串数组，设置时应指定一个字符串数组。有关事件的更多信息，请参见“序列事件和时刻”部分。此属性仅适用于以下类型的轨道 seqtracktype_message。
  • event ：此属性将返回（或可以设置为）method keyframe数据结构相关联。如果未指定任何方法，或者您希望删除该方法，则该属性应为-1。此属性仅适用于以下类型的轨道 seqtracktype_moment。
