in_sequence

这是一个内置变量，它是为游戏中对象的每个实例创建的实例变量的一部分。当实例是序列的一部分时，此变量将返回 true，如果不是，它将返回 false。这是一个只读变量，无法设置。

句法：

in_sequence

返回值：

Boolean

例子：

if (!in_sequence)
    {
    instance_destroy()
    }

上面的代码检查 in_sequence变量，如果是 false（该实例不是序列的一部分），则它将销毁该实例。
