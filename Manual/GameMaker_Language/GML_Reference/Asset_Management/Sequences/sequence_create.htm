序列_创建

通过这个函数，你可以创建一个新的序列对象，然后向其添加轨道。该函数返回一个结构，然后你可以访问该结构来设置你所创建的新序列。这个结构的内容 该结构的内容在本页面上有详细介绍。然后，序列对象结构可以用来在房间层上使用函数创建序列的实例 layer_sequence_create(). 请注意，当以这种方式创建序列对象时，你应该在不再需要它们时通过调用函数 sequence_destroy()当它们不再需要的时候，你应该再次调用函数来删除它们，否则你会有一个内存泄漏，这 否则，你会有一个内存泄漏，这可能会减缓并最终导致你的游戏崩溃。

语法。

sequence_create()。

返回。

结构

例子。

myseq = sequence_create();
myseq.length = 120;
myseq.loopmode = seqplay_pingpong;

上述代码创建了一个新的序列对象结构，并设置其播放长度和循环模式。

返回。序列
