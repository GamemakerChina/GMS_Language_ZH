sequence_create

使用此功能，您可以创建一个新的序列对象，然后可以向其添加轨道。该函数返回一个结构，您可以访问该结构以设置您创建的新序列。此结构的内容在此页面上有详细说明。然后，可以使用函数使用序列对象struct在房间层上创建序列的实例 layer_sequence_create()。请注意，以这种方式创建序列对象时，应通过调用函数再次将其删除 sequence_destroy()当不再需要它们时，否则您将发生内存泄漏，这会减慢速度并最终导致游戏崩溃。

句法：

sequence_create();

返回值：

Struct

例子：

myseq = sequence_create();
myseq.length = 120;
myseq.loopmode = seqplay_pingpong;

上面的代码创建一个新的序列对象结构，并设置其播放长度和循环模式。
