sequence_destroy

使用此功能，您可以销毁已动态创建的序列对象。您可以提供序列对象struct（由函数返回 sequence_create()）或序列ID（由函数返回）layer_sequence_get_sequence()或来自序列实例的struct属性 sequence）。每当不再需要动态创建的序列来释放与其关联的内存时，都应使用此功能。

句法：

sequence_destroy(sequence_struct_or_id);

争论描述
sequence_struct_or_id要破坏的序列对象struct或ID

返回值：

N/A

例子：

if sequence_exists(my_seq)
    {
    sequence_destroy(my_seq);
    }

上面的代码检查给定的序列对象是否存在以及是否存在。
