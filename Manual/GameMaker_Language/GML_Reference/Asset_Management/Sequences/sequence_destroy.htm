序列_销毁

通过这个函数，你可以销毁一个已经动态创建的序列对象。你可以提供序列对象的结构（如函数所返回的 sequence_create())或序列ID(如函数返回的 的序列ID（由该函数 layer_sequence_get_sequence()或从序列实例结构属性sequence 中返回）。)这个函数应该在动态创建的 序列不再需要释放与之相关的内存时，就应该使用这个函数。

语法。

sequence_destroy(sequence_struct_or_id)。

争论 描述
序列结构_or_id 要销毁的序列对象结构或ID

返回。

不适用

例子。

if sequence_exists(my_seq)
{
sequence_destroy(my_seq)。
}

上面的代码检查给定的序列对象是否存在，如果存在，就销毁它。

返回。序列
