sequence_get_objects

使用此功能，您可以检索具有在给定序列中创建的实例的所有对象索引的数组。您可以提供序列对象struct（由函数返回 sequence_create()或者 sequence_get()）或序列ID（由函数返回）layer_sequence_get_sequence()或来自序列实例的struct属性 sequence），该函数将返回一个数组，其中数组中的每个项目都是 object_index用于序列创建实例所使用的不同对象。

句法：

sequence_get_objects(sequence_struct_or_id);

争论描述
sequence_struct_or_id从中获取对象的序列对象struct或ID。

返回值：

Array

例子：

var _seq = sequence_get(seq_Logo);
obj_array = sequence_get_objects(_seq);

上面的代码获取序列对象的结构，然后检索它使用的对象并将数组存储在变量中。

背面：序列
