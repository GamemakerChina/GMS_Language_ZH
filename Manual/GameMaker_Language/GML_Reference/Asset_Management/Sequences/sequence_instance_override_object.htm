序列_实例_覆盖_对象

通过这个函数，你可以用另一个对象覆盖（替换）一个序列中使用的所有对象实例。你提供序列实例的结构ID（如在房间中创建序列实例时返回的，或通过使用房间层的一个 函数返回的），以及你想覆盖的对象的对象索引（如资产浏览器中定义的）。最后，你给一个对象索引或一个实例ID，作为 覆盖序列的对象（提供一个实例ID将简单地使用该实例创建的对象作为覆盖对象）。请注意，这只能在序列实例（而不是序列对象）上进行，而且必须在序列开始播放之前进行 而且必须在序列开始播放之前完成，否则它将无法工作。

语法。

sequence_instance_override_object(sequence_instance_struct, object_id, instance_or_object_id)。

争论 描述
序列_实例_结构 要修改的序列实例结构。
object_id 要覆盖的序列中的对象的索引。
instance_or_object_id 用来覆盖序列对象的对象索引或实例ID。

返回。

不适用

例子。

var _seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
var _seq_inst = layer_sequence_get_instance(_seq); sequence_instance_override_object(_seq_inst, obj_Trees_Winter, obj_Trees_Summer) 。

上面的代码在给定的图层上创建了一个新的序列实例，然后对其进行修改，使 "obj_Trees_Winter "对象的所有实例被 "obj_Trees_Summer "对象的实例取代。

返回。序列
