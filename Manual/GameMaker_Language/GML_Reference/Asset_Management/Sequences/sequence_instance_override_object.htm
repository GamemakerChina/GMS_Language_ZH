sequence_instance_override_object

使用此功能，您可以将一个序列中使用的对象的所有实例替换为另一个对象。您提供序列实例的结构ID（在房间中创建序列实例或使用房间层函数之一返回时返回的值-参见此处），以及对象的对象索引（在资产浏览器中定义）您要覆盖的内容。最后，您提供一个对象索引或一个实例ID来用作要覆盖序列的对象（提供实例ID将仅使用创建实例的对象作为覆盖对象）。请注意，这只能在序列实例上执行（不能在序列对象上执行），并且必须在序列开始播放之前完成，否则它将无法正常工作。

句法：

sequence_instance_override_object(sequence_instance_struct, object_id, instance_or_object_id);

争论描述
sequence_instance_struct要修改的序列实例结构。
object_id要覆盖的序列中对象的对象索引。
instance_or_object_id用于覆盖序列对象的对象索引或实例ID。

返回值：

N/A

例子：

var _seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
var _seq_inst = layer_sequence_get_instance(_seq); sequence_instance_override_object(_seq_inst, obj_Trees_Winter, obj_Trees_Summer);

上面的代码在给定的层上创建了一个新的序列实例，然后对其进行了修改，以使对象“ obj_Trees_Winter”的所有实例都被对象“ obj_Trees_Summer”的实例替换。
