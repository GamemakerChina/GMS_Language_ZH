shader _current

此函数将返回shader的索引ID值，或者如果不使用shader ，则将返回-1。

句法：

shader_current();

返回值：

Real (ID value of the shader, or -1 if no shader)

例子：

if (shader_current() == -1)
    {
    shader_set(sh_warp)
    }

上面的代码将检查当前的shader是什么，如果返回-1（不shader ），则设置shader
