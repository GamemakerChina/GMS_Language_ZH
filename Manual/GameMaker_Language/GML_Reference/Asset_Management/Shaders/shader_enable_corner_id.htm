shader_enable_corner_id

通过这个函数，你可以为所有正在使用的着色器设置一个全局状态，当启用时，着色器会从输入的颜色值中 "窃取 "两个比特。第一个是红色颜色值的低位，第二个是蓝色颜色值的低位。 第二位是蓝色颜色值的低位。这些值可以在着色器中被恢复，以计算出你正在处理的顶点（即：哪个角）。

语法。

shader_enable_corner_id(enable)。

争论 描述
启用 启用（真）或禁用（假）此功能。

返回。

不适用

例子。

shader_enable_corner_id(true)。

上面的代码将使所有着色器的角落ID使用颜色位。

返回。 着色器
