shader _enable_corner_id

使用此功能，可以为shaders设置全局状态，启用该状态后， shader将从输入颜色值中“窃取” 2位。第一个来自红色值的低位，第二个来自蓝色值的低位。 shader恢复这些值，以计算出要处理的顶点（即：哪个角）。

句法：

shader_enable_corner_id(enable);

争论描述
enable使能够 （true ）或禁用（false ）此功能。

返回值：

N/A

例子：

shader_enable_corner_id(true);

上面的代码将允许对所有shaders的角落ID使用颜色位。
