shader _get_name

使用此功能，您可以检索shader资源的名称。 shader提供唯一的ID值以获取其名称，该函数将资源名称作为字符串返回。

句法：

shader_get_name(shader);

争论描述
shader要获取其名称shader的索引。

返回值：

String

例子：

var _shader = shader_current();
var _name = shader_get_name(_shader);
draw_text(32, 32, "Debug - Currently Rendering = " + _name);

上面的代码将获取给定shader的名称并将其绘制到屏幕上。
