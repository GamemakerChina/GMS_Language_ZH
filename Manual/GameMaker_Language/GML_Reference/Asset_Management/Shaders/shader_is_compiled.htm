shader_is_compiled

这个函数将在运行时检查一个着色器，以确保它已经被成功编译了。如果它已经被编译，那么该函数将返回true，否则将返回false。这个函数应该 在游戏开始时使用，以确保运行你的游戏的平台已经成功地编译了任何使用的着色器（特别是在Windows上，一些计算机可能使用DX9的着色器级别2.0，而不是使用着色器级别3.0的更高版本）。 3.0).

如果你的着色器 没有被编译，而你调用了 shader_set()游戏就会崩溃，所以当你在使用 简单的GLSL ES着色器以外的东西时，都值得进行某种检查。

语法。

shader_is_compiled(shader)。

争论 描述
着色器 要检查的着色器。

返回。

不适用

例子。

global.GFX = false;
if(shader_is_compiled(sh_glass) && shader_is_compiled(sh_warp) )
{
global.GFX = true;
}

上面的代码将设置一个全局变量为false，然后如果被检查的两个着色器都编译正确，它将被设置为true。
