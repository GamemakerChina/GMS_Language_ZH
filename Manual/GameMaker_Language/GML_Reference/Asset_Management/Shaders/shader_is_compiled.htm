shader _is_compiled

此函数将shader ，以确保已成功编译它。如果有，则函数返回 true否则返回 false。在游戏开始时应使用此功能，以确保运行游戏的平台已成功编译了所有shaders （尤其是在Windows ，其中某些计算机可能使用具有Shader Level 2.0的shader level 3.0的更高版本）。

如果您的shader编译时没有和你打电话 shader_set()游戏会崩溃，因此当您使用简单的GLSL ES shaders之外的其他任何东西时，值得进行某种检查。

句法：

shader_is_compiled(shader);

争论描述
shader要检查的shader

返回值：

N/A

例子：

global.GFX = false;
if (shader_is_compiled(sh_glass) && shader_is_compiled(sh_warp))
    {
    global.GFX = true;
    }

上面的代码将全局变量设置为 false，然后如果两个shaders均已正确编译，则将其设置为 true。
