着色器_设置

通过这个函数，你可以将绘图目标设置为给定的着色器，所有进一步的绘图将使用该着色器完成。你可以用函数来结束着色器的使用 shader_reset().

注意：你只能在一个实例的绘制事件中设置着色器。

语法。

shader_set(shader)。

争论 描述
着色器 要使用的着色器的索引。

返回。

不适用

例子。

shader_set(shader_Glass);
draw_self();
shader_reset()。

上面的代码将设置一个用于绘制的着色器顶部，然后用它来绘制当前用于实例的精灵。

返回。 着色器
