shader

使用此功能，您可以将绘制目标设置为给定的shader并使用该着色器完成所有进一步的绘制。您可以通过功能shader shader_reset() 。

注意：您只能在实例的Draw Event shader

句法：

shader_set(shader);

争论描述
shader要使用shader的索引。

返回值：

N/A

例子：

shader_set(shader_Glass);
draw_self();
shader_reset();

上面的代码将设置一个shader顶部以用于绘制，然后使用它绘制用于实例的当前精灵。
