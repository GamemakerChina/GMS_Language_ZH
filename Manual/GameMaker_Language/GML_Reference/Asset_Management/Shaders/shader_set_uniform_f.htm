shader _set_uniform_f

使用此功能，您可以设置shader常数的值。您以前必须已经使用函数获得了常量的“句柄”shader_get_uniform() ，您将必须知道将正确数量的浮点值传递给它的常数类型，即：如果您有一个 vec2您将需要将两个值传递给该函数。

上述规则的例外是在处理32位颜色值时。您最多可以将shader ，它们会自动为您转换为vec4值0-1。但是，为此必须使用shader常量名称 in_Colour，用于单色或 in_Colour0，in_Colour1等...直至 in_Colour7用于多个输入值。以下示例显示了shader本身的外观：

    //Single colour
    attribute vec2 in_Position;
    attribute vec4 in_Colour;
    attribute vec2 in_TextureCoord;

    //Multiple colours
    attribute vec2 in_Position;
    attribute vec4 in_Colour0;
    attribute vec4 in_Colour1;
    attribute vec2 in_TextureCoord;

还要注意，如果要使用4字节颜色格式进行压缩法线等，那么您还必须使用与上面所示相同的“ in_Colour”格式。

注意：必须在调用函数shader _set（）之后和调用shader _reset（）之前设置所有统一。

句法：

shader_set_uniform_f(handle, value1 [, value2, value3, value4]);

争论描述
handle要设置shader常量的句柄。
value1... value4 shader常数设置为的浮点值（或多个值）。

返回值：

N/A

例子：

shader_set(shader_Glass);
shader_params = shader_get_uniform(shader_glass, "u_vParams");
shader_set_uniform_f(shader_params, 0.1, 0.8, 0.25);
draw_self();
shader_reset();

上面的代码将获取shader常量“ u_vParams”（a vec3 ），然后将该常数设置为给定值。
