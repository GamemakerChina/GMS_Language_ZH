shader _set_uniform_f_array

使用此功能，您可以设置shader常量以保存值的数组。您以前必须已经使用函数获得了常量的“句柄”shader_get_uniform() ，并且您必须先前已初始化数组。

注意：必须在调用函数shader _set（）之后和调用shader _reset（）之前设置所有统一。

句法：

shader_set_uniform_f_array(handle, array);

争论描述
handle要设置shader常量的句柄。
array先前初始化的浮点值数组。

返回值：

N/A

例子：

shader_set(shader_Glass);
tex_array[0] = 0.5;
tex_array[1] = 0.1;
tex_array[2] = 0.25;
shader_params = shader_get_uniform(shader_glass, "u_vParams");
shader_set_uniform_f_array(shader_params, tex_array);
draw_self();
shader_reset();

上面的代码将获取shader常量“ u_vParams”的句柄，然后将该常量设置为给定的数组。
