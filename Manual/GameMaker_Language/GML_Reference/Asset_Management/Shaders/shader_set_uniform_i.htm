shader _set_uniform_i

使用此功能，您可以设置shader常数的值。您以前必须已经使用函数获得了常量的“句柄”shader_get_uniform() ，您将必须知道将正确数量的整数值传递给它的常数类型，即：如果您有一个 vec2您将需要将两个值传递给该函数。

注意：必须在调用函数shader _set（）之后和调用shader _reset（）之前设置所有统一。

句法：

shader_set_uniform_i(handle, value1 [, value2, value3, value4]);

争论描述
handle要设置shader常量的句柄。
value1... value4 shader常数设置为的整数值。

返回值：

N/A

例子：

shader_set(shader_Glass);
shader_params = shader_get_uniform(shader_glass, "u_vParams");
shader_set_uniform_i(shader_params, 0, 65, 255);
draw_self();
shader_reset();

上面的代码将获取shader常量“ u_vParams”（a vec3 ），然后将该常数设置为给定的整数值。
