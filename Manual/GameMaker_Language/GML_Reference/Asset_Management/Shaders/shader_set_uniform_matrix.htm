shader _set_uniform_matrix

shader常数的值（或多个值）设置为当前的变换矩阵（使用Matrix Functions设置）。您以前必须已经使用函数获得了常量的“句柄”shader_get_uniform() 。

注意：必须在调用函数shader _set（）之后和调用shader _reset（）之前设置所有统一。

句法：

shader_set_uniform_matrix(handle);

争论描述
handle要设置shader常量的句柄。

返回值：

N/A

例子：

shader_set(shader_Glass);
shader_matrix = shader_get_uniform(shader_glass, "u_vMatrix");
shader_set_uniform_matrix(shader_matrix);
draw_self();
shader_reset();

上面的代码将获取shader常量“ u_vMatrix”的句柄，然后将该常量设置为当前的变换矩阵。
