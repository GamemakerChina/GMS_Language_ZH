shader_set_uniform_matrix

通过这个函数，你可以将着色器常量的值（或数值）设置为当前的变换矩阵（如使用矩阵函数设置的）。你必须事先通过函数获得 常量的 "句柄"，使用函数 shader_get_uniform().

注意：所有的制服都必须在调用函数 shader_set()之后，并在调用 shader_reset().

语法。

shader_set_uniform_matrix(handle)。

争论 描述
处理 要设置的着色器常量的句柄。

返回。

不适用

例子。

shader_set(shader_Glass);
shader_matrix =shader_get_uniform(shader_glass, "u_vMatrix");
shader_set_uniform_matrix(shader_matrix) 。
draw_self()。
shader_reset()。

上面的代码将获得着色器常量 "u_vMatrix "的句柄，然后将该常量设置为当前的变换矩阵。

返回。 着色器
