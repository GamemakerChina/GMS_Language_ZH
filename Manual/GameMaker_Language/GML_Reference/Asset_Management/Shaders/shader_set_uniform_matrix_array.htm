shader_set_uniform_matrix_array

通过这个函数，你可以设置一个着色器常量来保存一个矩阵值的数组。你必须事先用函数获取常量的 "句柄"。 shader_get_uniform(), 而且你必须事先将数组初始化为一个浮点值数组，其中每一组16个浮点是一个4x4矩阵。

注意：所有的制服都必须在调用函数 shader_set()之后，并在调用 shader_reset().

语法。

shader_set_uniform_matrix_array(handle, array)。

争论 描述
处理 要设置的着色器常量的句柄。
阵列 一个先前初始化的浮点数值的数组。

返回。

不适用

例子。

shader_set(shader_Glass);
shader_params =shader_get_uniform(shader_tint, "cMatrixArray");
shader_set_uniform_matrix_array(shader_params, matrix_array);
draw_self() 。
shader_reset()。

上面的代码将获得着色器常量 "cMatrixArray "的句柄，然后将该常量设置为给定的阵列。
