九个切片功能

除了Sprite Editor 中的 Nine Slice 部分之外。 GameMaker Studio 2提供了在运行时为精灵设置Nine Slice属性的功能。 提供了在运行时为精灵设置九宫格属性的函数。在阅读这些函数之前，建议先阅读这一页，以了解Nine Slice技术的情况。

以下函数可用于在运行时检索和改变精灵资产的Nine Slice属性。

  • 创建精灵_nineslice_create
  • 幽灵_获取_九宫格
  • 魉的故事

这些函数使用了一个Nine Slice结构，你可以在本页阅读该结构的内容。
