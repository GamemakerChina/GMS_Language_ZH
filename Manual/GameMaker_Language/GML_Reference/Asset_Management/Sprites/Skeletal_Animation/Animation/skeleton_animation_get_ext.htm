skeleton_animation_get_ext

一个骨骼动画精灵可以具有各种动画集，并且可以为这些集合指定不同的轨道，以便您可以“混合和匹配”动画集。此函数将返回给定轨道号当前使用的动画集的名称（由该函数设置）skeleton_animation_set_ext ）。

句法：

skeleton_animation_get_ext(track);

争论描述
track获得动画集名称的轨道号。

返回值：

String

例子：

if skeleton_animation_get_ext(1) != "bodyfight"
    {
    skeleton_animation_set_ext("bodyfight", 1);
    }

上面的代码会将第1轨使用的动画集更改为“ bodyfight”（如果没有）。
