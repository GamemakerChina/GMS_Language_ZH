骨架_动画_获取帧数

这个函数可以用来检索任何给定的骨架动画的帧数。你提供骨架动画的名称（以字符串形式，在制作动画的程序中定义，或通过使用函数返回 骨架_动画_获取返回，而函数则以整数形式返回其拥有的帧数。如果指定的动画不存在，该函数将返回0。

语法。

skeleton_animation_get_frames(anim_name)。

争论 描述
anim_name 要获取的动画帧的名称。

返回。

整数

例子。

var num = skeleton_animation_get_frames(skeleton_animation_get());
image_index = num -1;
image_speed = 0;

上面的代码将得到动画的帧数，然后将精灵设置为最后一帧，并停止动画。

返回。动画
