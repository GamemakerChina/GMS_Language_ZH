skeleton_animation_get_frames

此功能可用于检索任何给定的骨骼动画具有的帧数。您可以提供骨骼动画名称（作为字符串，如用于制作动画的程序中所定义，或通过使用函数返回的名称）skeleton_animation_get ，并且该函数以整数值返回它具有的帧。如果指定的动画不存在，该函数将返回0。

句法：

skeleton_animation_get_frames(anim_name);

争论描述
anim_name要获取帧的动画名称。

返回值：

Integer

例子：

var num = skeleton_animation_get_frames(skeleton_animation_get());
image_index = num -1;
image_speed = 0;

上面的代码将获取动画中的帧数，然后将精灵设置为最后一帧并停止动画。
