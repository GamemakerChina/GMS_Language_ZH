骨架_动画_设置

一旦你给你的实例分配了一个骨架动画精灵，你就可以用这个函数定义它在任何时候应该使用哪一个动画集。当你创建你的精灵时（在Spine中），你将定义多个动画 集，并对它们进行适当的命名。这些名字是你用来告诉GameMaker Studio 2在你的游戏中任何时候使用哪个动画集的。例如，如果你的精灵是一个平台游戏，那么你可能有 一个 "跳跃 "组、一个 "运行 "组和一个 "空闲 "组都包含在一个精灵中，你可以使用这个函数在游戏中切换它们。

请注意，以这种方式设置一个新的动画会将 图像_索引到0，新的动画集将从其动画循环的起点开始。如果这不是 的行为，那么你可以用 "混合 "值来设置一个 skeleton_animation_mix()来设置一个 "混合 "值，当你设置一个新的动画时，它将与之前的动画集平滑地插值。你还应该确保 确保在每个动画的开始有一个关键帧，以防止数据从以前的动画中被保留下来，除非你打算使用独立的动画轨道。

语法。

skeleton_animation_set(animname)。

争论 描述
动画名称 要使用的动画集的名称（一个字符串）。

返回。

不适用

例子。

if keyboard_check_pressed(vk_space)
{
if skeleton_animation_get() != "jump"
{
skeleton_animation_set("jump");
}
}

上面的代码会在按下空格键时将正在使用的动画集改为 "跳跃 "集，但前提是当前正在使用的动画集不是 "跳跃"。

返回。动画
