skeleton_animation_set

将骨骼动画精灵指定给实例后，便可以使用此功能定义在任何给定时间应使用的动画集。在Spine中创建精灵时，您将为不同的状态定义多个动画集，并为每个动画分别命名。这些名称是用来告诉GameMaker Studio 2在游戏中随时使用的动画集的名称。例如，如果您的精灵是用于平台游戏的，则可能包含一个“跳跃”集，一个“运行”集和一个“空闲”集，它们都包含在一个精灵中，您可以使用此设置在游戏中进行切换功能。

请注意，以这种方式设置新动画将重置 image_index设置为0，则从其动画循环的开头开始新的动画集。如果这不是所需的行为，则可以使用以下命令设置“混合”值 skeleton_animation_mix()设置新的动画时，它将平滑地与先前的动画进行插值。除非计划使用单独的动画轨道，否则还应确保keyframe ，以防止数据从以前的动画中保留下来。

句法：

skeleton_animation_set(animname);

争论描述
animname要使用的动画名称（字符串）。

返回值：

N/A

例子：

if keyboard_check_pressed(vk_space)
    {
    if skeleton_animation_get() != "jump"
        {
        skeleton_animation_set("jump");
        }
    }

上面的代码将在按下空格键时将正在使用的动画集更改为“跳转”集，但前提是当前使用的动画集尚未“跳转”。
