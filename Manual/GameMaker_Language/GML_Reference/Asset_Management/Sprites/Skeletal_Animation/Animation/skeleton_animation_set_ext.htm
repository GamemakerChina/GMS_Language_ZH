skeleton_animation_set_ext

为骨骼动画精灵设置动画集时，会在内部为其分配轨道0。但是，您可以将更多的集合分配给其他轨道（1和更高的轨道），这样，您的精灵就可以同时播放两个（或更多）动画。请注意，应该在设计动画程序时牢记这一点，以使磁道0中的“基本”动画仅具有keyframes ，而下一个磁道具有其他keyframes例如，如果您有一个“行走”动画，并且其keyframes仅影响精灵的腿，则可以有一组头部动画和手臂动画，可以分别在轨道1和2中进行设置，从而使您可以基本动作并在玩游戏时实时进行调整。

句法：

skeleton_animation_set_ext(animname, track);

争论描述
animname要使用的动画名称（字符串）。
track设置要使用的动画的轨道。

返回值：

N/A

例子：

if mouse_check_button(mb_left)
    {
    skeleton_animation_set_ext("bodyfight", 1);
    }
else
    {
    skeleton_animation_set_ext("bodywalk", 1);
    }

上面的代码将在按下鼠标左键时将轨道1所使用的动画集更改为“ bodyfight”集或“ bodywalk”集。
