骨架_动画_设置框架

这个函数将把分配给给定轨道的动画设置为你提供的帧。帧索引应该是一个介于0和动画的帧数之间的实数（你可以用函数 skeleton_animation_get_frames（骨架动画）。). 如果你提供的数值超出了这个范围，动画将被循环播放，以弥补其中的差异，例如，如果动画有5个帧，而你将帧数设置为8，实际显示的帧数将是3。

语法。

skeleton_animation_set_frame(track, index)。

争论 描述
追踪 要设置帧的动画轨道。
指数 动画帧的新索引。

返回。

不适用

例子。

if global.pause
{
skeleton_animation_set_frame(0, -image_speed);
}

上面的代码将在全局 "pause "变量为真时，将轨道0中的动画的帧索引设置为负的图像速度（基本上是暂停动画）。

返回。动画
