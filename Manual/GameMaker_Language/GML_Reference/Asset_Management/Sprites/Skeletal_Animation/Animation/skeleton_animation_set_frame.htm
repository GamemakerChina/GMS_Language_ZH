skeleton_animation_set_frame

此功能会将分配给给定轨道的动画设置为您提供的帧。帧索引应该是介于0和动画的帧数之间的实数值（您可以使用函数获得 skeleton_animation_get_frames）。如果提供的值超出此范围，则动画将循环播放以补偿差异，例如，如果动画有5帧并将帧设置为8，则实际显示的帧将为3。

句法：

skeleton_animation_set_frame(track, index);

争论描述
track要设置帧的动画轨道。
index动画帧的新索引。

返回值：

N/A

例子：

if global.pause
    {
    skeleton_animation_set_frame(0, -image_speed);
    }

当全局“ pause”变量为时，上面的代码会将轨道0中动画的帧索引设置为负图像速度。true （基本上是暂停动画）。
