骷髅头_附件_创建

当你创建你的骨架动画精灵时，你可以指定附件槽和附件，以放入这些附件。这些是简单的图像，与动画无关，但当它们被连接到 会随着它们所连接的骨骼一起移动。通常你会在你的动画程序（Spine）中指定这些附件，但有了这个函数，你可以在运行时使用游戏中的精灵资产创建自己的附件。 给附件一个名字（字符串），然后设置要使用的sprite_index和image_index，以及x和y原点（可以与sprite属性中定义的不同），你还可以设置任何变换 你还可以设置任何你希望在附加时应用到图像的变换。如果成功创建了附件，该函数将返回1；如果没有成功创建（你提供了一个无效的精灵索引，或者基础精灵不是一个Spine 精灵），那么它将返回-1。

语法。

skeleton_attachment_create（name, sprite, ind, xorigin, yorigin, xscale, yscale, rot）。

争论 描述
名称 要创建的附件的名称（字符串）。
精灵 用于获取附件图像的sprite_index。
ind 用于获取附件图片的image_index。
xorigin 正在使用的图像的X原点。
原产地 正在使用的图像的Y原点。
xscale 图像的水平缩放比例，作为一个乘数。1=正常缩放，0.5是一半等等...
yscale 图像的垂直缩放，作为一个乘数。1=正常缩放，0.5是一半等等...
腐烂 图像的旋转。0=正常，90=逆时针旋转90度等。

返回。

如果成功则为1，不成功则为-1

例子。

skeleton_attachment_create("sword", spr_Weapons, 0, 0, 80, 1, 1, 90);
skeleton_attachment_create("killer", spr_Weapons, 1, 0, 45, 1, 1, 90);
skeleton_attachment_create("crossbow", spr_Weapons, 0, 10, 30, 1, 1, 0);
skeleton_attachment_set("slot_leftHand", choose("word", "killer", "crossbow"));

上述代码将检查当前分配给名为 "slot_leftHand "的插槽的附件名称，如果返回一个空字符串，就会附加一个新的精灵。
