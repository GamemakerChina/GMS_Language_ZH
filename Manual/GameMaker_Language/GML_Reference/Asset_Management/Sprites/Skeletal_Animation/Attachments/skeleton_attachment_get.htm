skeleton_attachment_get

骨骼动画精灵可能还会添加其他精灵作为附件，这些精灵会添加到命名插槽（在动画程序中创建附件插槽时会给出名称），它们会与当前精灵的动画一起绘制。使用此功能，您可以获取当前分配的精灵的给定插槽的附件名称（作为字符串）。请注意，附加的精灵通过在Spine中分配的名称字符串或您调用时的名称字符串进行引用 skeleton_attachment_create()。

句法：

skeleton_attachment_get(slot);

争论描述
slot要获取附件的插槽名称（字符串）。

返回值：

String

例子：

if skeleton_attachment_get("slot_leftHand") == ""
    {
    skeleton_attachment_set("slot_leftHand", choose("sword", "spear", "knife"));
    }

上面的代码将检查名为“ slot_leftHand”的插槽当前分配的附件名称，如果返回空字符串，则将附加一个新的精灵。
