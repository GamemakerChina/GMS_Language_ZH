骨架_附着物_获取

一个骨架动画精灵可以添加其他精灵作为附件，这些精灵被添加到一个命名的槽中（名称是在你的动画程序中创建附件槽时给出的），它们将与当前精灵的动画一起被绘制。 的动画。通过这个函数，你可以得到当前指定精灵的指定槽的附件名称（字符串）。请注意，附件精灵是通过在Spine中分配的名称字符串来引用的，或者在你调用 当你调用 skeleton_attachment_create().

语法。

skeleton_attachment_get(slot)。

争论 描述
槽 要获取附件的槽名（字符串）。

返回。

字符串

例子。

if skeleton_attachment_get("slot_leftHand") == ""
{
skeleton_attachment_set("slot_leftHand", choose("word", "spear", "killer"));
}

上述代码将检查当前分配给名为 "slot_leftHand "的插槽的附件名称，如果返回一个空字符串，就会附加一个新的精灵。
