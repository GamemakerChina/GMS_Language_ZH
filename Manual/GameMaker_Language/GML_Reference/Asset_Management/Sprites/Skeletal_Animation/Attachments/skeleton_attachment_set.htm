skeleton_attachment_set

骨骼动画精灵可以添加其他图像作为附件，并将这些图像添加到命名插槽（在动画程序中创建附件插槽时给出的名称），它们将与当前精灵的动画一起绘制。使用此功能，您可以将附件设置到给定的插槽，并在其中指定插槽和附件的名称（作为字符串）。这些名称由使用的动画程序定义，或者在调用时（对于附件）定义 skeleton_attachment_create()。

请注意，您还可以传递一个 sprite_indexin作为附件，则将使用该精灵，也可以使用-1从插槽中删除附件。当您将sprite索引作为参数传递时，它将使用sprite的名称作为用于命名插槽的字符串来创建附件插槽（因此使用 spr_sword，例如，将创建一个附件插槽“ spr_sword”），并且该插槽将使用子画面的第一个图像索引（0）以及其x和y原点偏移量，比例为（1,1）并旋转0。

句法：

skeleton_attachment_set(slot, attachment);

争论描述
slot要获取附件的插槽名称（字符串）。
attachment附件图像的名称（作为字符串或sprite_index）。

返回值：

String

例子：

if skeleton_attachment_get("slot_leftHand") == ""
    {
    skeleton_attachment_set("slot_leftHand", choose("sword", "spear", "knife"));
    }

上面的代码将检查名为“ slot_leftHand”的插槽当前分配的附件名称，如果返回空字符串，则将附加一个新的精灵。
