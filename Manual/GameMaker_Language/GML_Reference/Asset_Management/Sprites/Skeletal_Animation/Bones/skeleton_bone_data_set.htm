skeleton_bone_data_set

您的骨骼动画由许多“骨骼”组成，您将在动画程序中定义这些骨骼并为其指定名称，并且该函数可随时用于为命名骨骼设置某些数据。请注意，此数据是指骨骼的默认姿势，而不是正在绘制的当前姿势（为此使用功能 skeleton_bone_state_set()），并且必须从先前创建的DS映射中进行设置，该映射应具有以下键及其等效值：

  • “ x”：骨骼相对于父骨骼的局部x位置。
  • “ y”：骨骼相对于父骨骼的局部y位置。
  • “角度”：骨骼相对于父骨骼的局部旋转。
  • “ xscale”：骨骼的局部水平比例。
  • “ yscale”：骨骼的局部垂直比例。
  • “ parent”：父骨骼的名称（字符串）。

句法：

skeleton_bone_data_set(bone, map);

争论描述
bone骨骼的名称（以字符串形式）。
map （先前创建的）DS映射，用于存储骨骼数据。

返回值：

N/A

例子：

var bone_map = ds_map_create();
skeleton_bone_data_get("head", bone_map);
ds_map_replace(bone_map, "parent", "body");
skeleton_bone_data_set("head", bone_map);
ds_map_destroy(bone_map);

上面的代码创建了一个DS映射，然后使用名为“ head”的骨骼中的数据填充它。然后，它将地图中的“父”骨骼键替换为新值，并使用新的数据集再次设置“头部”骨骼。
