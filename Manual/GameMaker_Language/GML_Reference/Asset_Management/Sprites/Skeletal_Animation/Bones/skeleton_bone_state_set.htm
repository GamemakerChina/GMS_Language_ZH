skeleton_bone_state_set

您的骨骼动画由许多“骨骼”组成，您将在动画程序中定义这些骨骼并为其指定名称，并且该函数可随时用于为命名骨骼设置某些数据。请注意，此数据将根据所使用的动画集设置骨骼的当前姿势，并且该功能需要先前创建的DS贴图，该贴图应具有以下关键点及其所需的值：

  • “ x”：骨骼相对于父骨骼的局部x位置。
  • “ y”：骨骼相对于父骨骼的局部y位置。
  • “角度”：骨骼相对于父骨骼的局部旋转。
  • “ xscale”：骨骼的局部水平比例。
  • “ yscale”：骨骼的局部垂直比例。
  • “ parent”：父骨骼的名称（字符串）。

提供此功能是为了使您可以“拦截”动画数据并在将其绘制到屏幕上之前对其进行修改，因此，您希望在“其他-动画更新”事件中使用它，因为此事件在“绘制”之前触发大事记

请注意，您可以使用通过地图填写的同一张地图 skeleton_bone_state_get()函数，即使它包含其他“世界”键和值。由于这些都是只读值，因此使用此功能进行设置不会有任何影响。

句法：

skeleton_bone_state_set(bone, map);

争论描述
bone骨骼的名称（以字符串形式）。
map （先前创建的）DS映射，用于存储骨骼数据。

返回值：

N/A

例子：

var map = ds_map_create();
skeleton_bone_state_get("head", map);
var xx = ds_map_find_value(map, "worldX");
var yy = ds_map_find_value(map, "worldY");
var deltax = mouse_x - (x + xx);
var deltay = mouse_y - (y + yy);
var angle = -radtodeg(arctan2(deltay, deltax));
ds_map_replace(map, "angle", angle);
skeleton_bone_state_set("head", map);
ds_map_destroy(map);

上面的代码创建了一个DS映射，然后使用名为“ head”的骨骼中的数据填充它。然后，提取骨骼的世界位置，并使用该数据设置骨骼的“角度”以跟踪游戏中的鼠标位置。
