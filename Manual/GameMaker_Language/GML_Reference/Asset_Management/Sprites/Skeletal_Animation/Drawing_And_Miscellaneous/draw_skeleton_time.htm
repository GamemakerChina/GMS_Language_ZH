draw_skeleton_time

此功能将在动画中的特定时间使用给定的皮肤绘制给定的动画。时间值应介于0（动画的开始）和结束时间之间，您可以使用函数找到该时间 skeleton_animation_get_duration()。您可以将时间值设置为高于动画总持续时间的值，并且动画将循环回到开头，但是随着累积时间变大，您将冒失去浮点精度的风险。

句法：

draw_skeleton_time(sprite, animname, skinname, time, x, y, xscale, yscale, rot, colour)

争论描述
sprite要绘制的精灵的索引。
animname要从中获取帧的动画的名称（一个字符串）。
skinname用于绘制的皮肤名称（字符串）。
time绘制动画的时间范围（从0到结束时长，以秒为单位）。
x绘制精灵的位置的x坐标。
y绘制子画面的y坐标。
xscale Sprite的水平缩放比例，作为乘数：1 =正常缩放比例，0.5等于一半，依此类推...
yscale Sprite的垂直缩放比例，作为乘数：1 =正常缩放比例，0.5等于一半，依此类推...
rot精灵的旋转。 0 =正常，90 =逆时针旋转90度，依此类推。
colour混合精灵的颜色。

返回值：

N/A

例子：

var time += delta_time / 1000000;
var d = skeleton_animation_get_duration("walk");
if time > d time -= d;
draw_skeleton_time(sprite_index, "walk", "skin1", time, x, y, image_xscale, image_yscale, image_angle, c_white);

上面的代码将使用delta-time绘制给定的骨骼动画精灵，以设置要绘制的帧。
