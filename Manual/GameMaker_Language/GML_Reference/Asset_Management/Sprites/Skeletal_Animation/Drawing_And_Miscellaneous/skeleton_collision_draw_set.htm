skeleton_collision_draw_set

使用此功能，您可以开启（true ）或关闭（false ）绘制实例正在使用的当前骨架动画精灵的碰撞数据。如果启用此选项，则边框和精确的碰撞蒙版将作为精灵周围的轮廓绘制。

句法：

skeleton_collision_draw_set(flag);

争论描述
flag设置 true打开绘图，以及 false将其关闭。

返回值：

N/A

例子：

if debug_mode == true
    {
    skeleton_collision_draw_set(true);
    }

上面的代码检查游戏是否在调试模式下运行，如果是，它将切换要显示的实例的骨骼碰撞数据。
