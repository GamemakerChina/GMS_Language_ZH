骨架_获取边界

这个函数将返回一个与当前分配的骨架动画精灵的任何给定界线盒相关的数值数组。你要提供边界盒的索引号来获取其细节（你可以使用函数来检索精灵的边界盒总数）。 你可以使用函数检索精灵的边界盒总数 skeleton_get_num_bounds()），该函数将返回一个包含以下元素的数组。

  • [0] - 包围盒上的点的数量（一个实数）。
  • [1] - 包围盒的名称（一个字符串）。
  • [2] - 第一个点的X位置
  • [3] - 第一个点的y位置
  • [等...] - 进一步的X/Y位置数据，直到包围盒上的点的数量。

语法。

skeleton_get_bounds(index)。

争论 描述
指数 要获得的包围盒索引。

返回。

数组（最少2个元素：numPoints, name [, xPos, yPos, etc..]）。

例子。

var num = skeleton_get_num_bounds();
var yy = 60;
for(var i = 0; i < num; i++;)
{
var b_info = skeleton_get_bounds(i);
if b_info[0] > 0
{
var data = b_info[1] + " :";
for(var j = 0; j < b_info[0]; j++;)
{
data += " ( " + string(b_info[(j * 2) + 2])+ ", " + string(b_info[(j * 2) + 2 + 1])+ ")";
}
draw_text(20, yy, data);
yy += 20;
}
}

上述代码将循环浏览与当前分配的精灵相关的每个边界框，然后将这些数据作为字符串画到屏幕上。
