skeleton_get_bounds

此函数将为当前分配的骨架动画精灵返回与任何给定边界框关联的值的数组。您可以提供边界框的索引号以获取详细信息（您可以使用函数检索子画面的边界框总数 skeleton_get_num_bounds()），该函数将返回包含以下元素的数组：

  • [0]-边界框上的点数（实数）
  • [1]-边界框的名称（字符串）
  • [2]-第一点的x位置
  • [3]-第一点的y位置
  • [etc...]-进一步的x / y位置数据，直到边界框上的点数

句法：

skeleton_get_bounds(index);

争论描述
index要获取其值的边界框索引。

返回值：

Array (minimum 2 elements: numPoints, name [, xPos, yPos, etc...])

例子：

var num = skeleton_get_num_bounds();
var yy = 60;
for(var i = 0; i < num; i++;)
    {
    var b_info = skeleton_get_bounds(i);
    if b_info[0] > 0
        {
        var data = b_info[1] + ":";
        for(var j = 0; j < b_info[0]; j++;)
            {
            data += " (" + string(b_info[(j * 2) + 2]) + ", " + string(b_info[(j * 2) + 2 + 1]) + ")";
            }
        draw_text(20, yy, data);
        yy += 20;
        }
    }

上面的代码将循环遍历与当前分配的精灵相关的每个边界框，然后将该数据作为字符串绘制到屏幕上。
