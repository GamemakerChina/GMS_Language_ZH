骨架的最小值

这个函数将返回一个数组，其中包含分配给一个骨架动画精灵的所有单个边界盒的总边界盒值。返回的数组将有4个元素。

  • [0] - 所有界线盒的最小X位置
  • [1] - 所有界线盒的最小Y位置
  • [2] - 所有界线盒的最大X位置
  • [3] - 所有界线盒的最大Y位置

语法。

skeleton_get_minmax()。

返回。

数组（4个元素：xMin, yMin, xMax, yMax）。

例子。

var frame = skeleton_animation_get_frame(0);
if frame > 0 && frame < 2
{
var box = skeleton_get_minmax();
var ww = (box[2] - box[0]) / 2;
var hh = (box[3] - box[1]) / 2;
part_particles_create( global.p_sys, box[0] + ww, box[1] + hh, global.Stars, 10);
}

上述代码将检查分配给0号轨道的动画的当前帧，然后从精灵的总边界框中间的一个点爆出一些粒子。
