skeleton_get_minmax

此函数将返回一个数组，其中包含分配给骨架动画精灵的所有单个边界框的总边界框值。返回的数组将包含4个元素：

  • [0]-所有边界框的最小x位置
  • [1]-所有边界框的最小y位置
  • [2]-所有边界框的最大x位置
  • [3]-所有边界框的最大y位置

句法：

skeleton_get_minmax();

返回值：

Array (4 elements: xMin, yMin, xMax, yMax)

例子：

var frame = skeleton_animation_get_frame(0);
if frame > 0 && frame < 2
    {
    var box = skeleton_get_minmax();
    var ww = (box[2] - box[0]) / 2;
    var hh = (box[3] - box[1]) / 2;
    part_particles_create(global.p_sys, box[0] + ww, box[1] + hh, global.Stars, 10);
    }

上面的代码将检查分配给磁道0的动画的当前帧，然后从该精灵的整个包围盒中间的某个点处爆炸一些粒子。
