skeleton_get_num_bounds

此函数将返回与分配给运行代码的实例的骨架动画精灵关联的边界框的数量。然后可以将其与功能一起使用 skeleton_get_bounds()检索有关每个边界框的数据。

句法：

skeleton_get_num_bounds();

返回值：

Integer

例子：

var num = skeleton_get_num_bounds();
var yy = 60;
for(var i = 0; i < num; i++;)
    {
    var b_info = skeleton_get_bounds(i);
    if b_info[0] > 0
        {
        var data = b_info[1] + ":";
        for(var j = 0; j < b_info[0]; j++;)
            {
            data += " (" + string(b_info[(j * 2) + 2]) + ", " + string(b_info[(j * 2) + 2 + 1]) + ")";
            }
        draw_text(20, yy, data);
        yy += 20;
        }
    }

上面的代码将循环遍历与当前分配的精灵相关的每个边界框，然后将该数据作为字符串绘制到屏幕上。
