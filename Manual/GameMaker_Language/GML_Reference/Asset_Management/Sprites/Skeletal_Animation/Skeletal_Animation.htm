骨骼动画精灵

本节中提供的功能仅适用于从骨骼动画文件（特别是Spine导出的JSON文件）导入的精灵，可用于获取有关游戏中动画资产的信息，以及在动画中设置某些属性。这些函数可以与常规Sprite函数和变量一起使用，从而允许您（例如）使用这些特殊函数混合两个骨骼动画，同时使用常规Sprite实例变量设置图像比例（有关Sprite实例变量的更多信息）看到这里）。

注意：有关导入使用Spine制作的骨骼动画精灵的更多信息，请参见“导入非位图精灵”一节。

您可以从以下各节中找到有关这些精灵的功能的更多信息：

淡黑色支持

此功能允许将Spine Sprite插槽的深色区域与浅色区域设置为不同的颜色（这是Spine IDE功能，请参见此处的“着色黑色”部分以获取更多详细信息）。 GameMaker Studio 2使用此功能，在绘制使用该功能的Spine子画面时，需要使用自定义shader此shader包含一个名为“gm_SpineTintBlackColour ”，转轮会使用当前的淡黑色填充，自动从Spine数据中检索。 shader如下所示：

顶点Shader （与默认的直通顶点shader ）

attribute vec3 in_Position; // (x,y,z)
attribute vec4 in_Colour; // (r,g,b,a)
attribute vec2 in_TextureCoord; // (u,v)

varying vec2 v_vTexcoord;
varying vec4 v_vColour;

void main()
{
vec4 object_space_pos = vec4( in_Position.x, in_Position.y, in_Position.z, 1.0);
gl_Position = gm_Matrices[MATRIX_WORLD_VIEW_PROJECTION] * object_space_pos;
v_vColour = in_Colour;
v_vTexcoord = in_TextureCoord;
}

片段Shader ：

varying vec2 v_vTexcoord;
varying vec4 v_vColour;

uniform vec4 gm_SpineTintBlackColour; // This is the uniform containing the tint-black colour

void main()
{
vec4 tb = gm_SpineTintBlackColour;
vec4 texcol = texture2D( gm_BaseTexture, v_vTexcoord );
vec4 outcol;
outcol.rgb = v_vColour.rgb * texcol.rgb;
outcol.rgb += tb.rgb * ((tb.a * (texcol.a - 1.0)) + (1.0 - texcol.rgb)); // This line performs the tint-black blending logic
outcol.a = v_vColour.a * texcol.a;
gl_FragColor = outcol;
}

您可以通过首先调用shader ，然后绘制精灵，然后重置shader ，如下所示：

shader_set(shd_spine_tint_black);
draw_self();
shader_reset();
