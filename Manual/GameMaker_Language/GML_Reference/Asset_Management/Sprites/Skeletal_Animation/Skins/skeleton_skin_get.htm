骷髅头_皮肤_获取

使用骨架动画精灵，你可以为动画分配单独的纹理（称为 "皮肤"），从而用一个动画做多个不同的事情。这个函数将返回当前分配给你的实例所使用的骨骼动画精灵的皮肤名称（以字符串形式）。 分配给你的实例正在使用的骨架动画精灵的皮肤名称（字符串）。返回的名称是你在你的动画程序中创建精灵时设置的。

语法。

skeleton_skin_get()。

返回。

字符串

例子。

if skeleton_skin_get() == "skin_Enemy1"
{
skeleton_skin_set(choose("skin_Enemy1", "skin_Enemy2", "skin_Enemy3");
}

上述代码将检查当前分配给动画的皮肤，如果它是 "skin_Enemy1"，它将从三个选项中选择并设置一个新的皮肤。
