skeleton_skin_get

使用骨骼动画精灵，可以为动画分配单独的纹理（称为“皮肤”），从而将一个动画用于多种不同的事物。此函数将返回当前分配给实例正在使用的骨骼动画精灵的皮肤名称（以字符串形式）。返回的名称是您在动画程序中创建精灵时设置的名称。

句法：

skeleton_skin_get();

返回值：

String

例子：

if skeleton_skin_get() == "skin_Enemy1"
    {
    skeleton_skin_set(choose("skin_Enemy1", "skin_Enemy2", "skin_Enemy3");
    }

上面的代码将检查动画当前分配的外观，如果它是“ skin_Enemy1”，它将从三个选项之一中选择并设置一个新的外观。
