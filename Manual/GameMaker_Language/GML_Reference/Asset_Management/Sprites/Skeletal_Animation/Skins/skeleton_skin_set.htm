skeleton_skin_set

可以使用各种纹理（称为“皮肤”）绘制骨骼动画精灵，从而允许您将一个动画用于多种不同目的。例如，您可能具有通用的行走动画，并且您的NPC角色，敌人等可以在游戏中简单地通过改变绘制皮肤的方式来使用它。这些皮肤需要事先在动画程序中创建并命名，并将此名称传递给函数（作为字符串）以设置精灵以在绘制时使用该皮肤。

句法：

skeleton_skin_set(skinname);

争论描述
skinname要使用的皮肤名称（字符串）。

返回值：

N/A

例子：

if skeleton_skin_get() == "skin_Enemy1"
    {
    skeleton_skin_set(choose("skin_Enemy1", "skin_Enemy2", "skin_Enemy3");
    }

上面的代码将检查动画当前分配的外观，如果它是“ skin_Enemy1”，它将从三个选项之一中选择并设置一个新的外观。
