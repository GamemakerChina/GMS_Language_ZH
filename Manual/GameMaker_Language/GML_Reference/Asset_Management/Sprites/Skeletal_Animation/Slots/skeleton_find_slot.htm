skeleton_find_slot

使用此功能，您可以在与当前实例关联的Spine子画面中找到哪些插槽位于指定的空间位置。您创建一个DS列表，并提供其ID以及一个x / y位置以进行检查，并且该列表将使用该精灵的每个可用附件插槽（包括任何附件修改）的名称字符串进行填充。请注意，列表始终从最上方的插槽开始以降序排列。

句法：

skeleton_find_slot(x, y, list);

争论描述
x要检查的房间中的x位置。
y要检查的房间中的y位置。
list用DS映射填充的DS列表的ID。

返回值：

N/A

例子：

slot_list = ds_list_create();
skeleton_find_slot(mouse_x, mouse_y, slot_list);

上面的代码创建一个DS列表，然后使用在房间中鼠标位置找到的插槽名称填充它。
