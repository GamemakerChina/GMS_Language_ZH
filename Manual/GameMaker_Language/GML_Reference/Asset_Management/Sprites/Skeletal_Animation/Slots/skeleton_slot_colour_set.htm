skeleton_slot_colour_set

使用此功能，您可以设置附件插槽的颜色和Alpha，以便将使用这些混合值绘制所有附着到其上的精灵。请记住，实例精灵可以具有混合颜色和alpha设置（image_blend和 image_alpha）以及附件（请参见功能）skeleton_attachment_create_colour() ），因此为该插槽分配的附件精灵将具有的最终颜色和Alpha将是所有这些值的组合。

句法：

skeleton_slot_colour_set(slot, colour, alpha);

争论描述
slot要设置的插槽名称，一个字符串
colour要设置的颜色，可以是整数，常数或十六进制值。
alpha设置为0到1的Alpha。

返回值：

N/A

例子：

slot_list = ds_list_create();
skeleton_find_slot(mouse_x, mouse_y, slot_list);
if !ds_list_empty(slot_list)
{
for (var i = 0; i < ds_list_size(slot_list); ++i;)
    {
    if skeleton_slot_colour_get(slot_list[| i]) != c_white
        {
        skeleton_slot_colour_set(slot_list[| i], c_white, 1);
        }
    }
}

上面的代码创建一个DS列表，然后使用在房间中鼠标位置找到的插槽名称填充它。然后，它循环浏览插槽列表，并将找到的插槽的颜色重置为白色（如果颜色不是白色，则alpha值为1）。
