骨架_slot_data

通过这个函数，你可以用一个动画所包含的每个可用的附件槽的数据来填充一个（预先创建的）DS列表。这些数据会以DS地图的形式返回，其中包含以下键/值对。

  • "名称"。插槽的名称。
  • "骨头"。骨头的名称。
  • "附件"。如果使用了附件，则是附件的名称；如果没有指定附件，则是"（无）"。

每个键的值将是字符串，然后可以用于这些类型的精灵的其他骨架附件函数。请注意，创建的DS地图不会被销毁，所以你需要循环浏览DS列表，并销毁每一个 创建的地图。

语法。

skeleton_slot_data(sprite, list)。

争论 描述
精灵 要获取数据的Spine骨骼动画的精灵索引。
列表 要用DS地图填充的DS列表的ID。

返回。

不适用

例子。

var list = ds_list_create();
var open = true;
slot_name = "";
skeleton_slot_data(sprite_index, list);
for (var i = 0; i < ds_list_size(list); i++; )
{
var map = list[| i];
if open
{
if map[? "attachment"] == "(none)"
{
open = false;
slot_name = map[? "name"];
}
}
ds_map_destroy(map)。
}
ds_list_destroy(list)。

上面的代码创建了一个DS列表，然后用实例精灵的插槽数据来填充它。然后对这些数据进行解析，提取出带有插槽数据的单个DS地图。然后检查是否有一个空槽，如果有，则将变量 "slot_name "在DS地图被销毁之前被分配给空槽的名字。最后我们销毁DS列表，因为它不再需要了。

返回。槽位
