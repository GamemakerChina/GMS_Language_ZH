幽灵_get_box_mode

这个函数将返回当前用于计算边界盒的 "模式"。你提供要检查的精灵的精灵索引，该函数将返回下面所示的常数之一。

语法。

sprite_get_box_mode(ind)。

争论 描述
ind 要检查的精灵的索引。

返回

恒定

恒定 描述
bboxmode_automatic 自动--将根据精灵的公差设置，自动计算边界框。
bboxmode_fullimage 全图--包围盒将被设置为使用精灵的全部宽度和高度，而不考虑公差和 "空 "像素。
bboxmode_manual 手动--包围盒已被手动设置为用户定义的值（可以在精灵编辑器中，或使用函数 sprite_set_box())

例子。

if sprite_get_box_mode(sprite_index) != 0
{
sprite_set_box_mode(sprite_index, bboxmode_automatic);
}

上面检查了当前精灵的bbox模式，如果它不是自动的，那么就把它设置成这个值。
