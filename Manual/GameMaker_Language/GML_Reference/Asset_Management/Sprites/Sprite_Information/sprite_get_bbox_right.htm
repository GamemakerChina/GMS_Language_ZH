魉_获得_both_right

该函数返回精灵边界框右边的相对位置。这个值是以基本精灵资产的左上角（0,0）为基础的相对值，它与在精灵编辑器中的 碰撞遮罩属性中找到的值。下面的图片显示了它是如何计算的。

语法。

sprite_get_box_right(ind)。

争论 描述
ind 要检查的精灵的索引。

返回

真实的

例子。

var ww, hh;
ww = sprite_get_box_left(sprite_index) - sprite_get_box_right(sprite_index);
hh = sprite_get_box_bottom(sprite_index) - sprite_get_box_top(sprite_index) 。

上面的代码根据相对边界框的侧边位置来计算碰撞掩码的宽度和高度。
