sprite_get_height

使用此功能，您可以找到没有变换的基本Sprite资产的高度（以像素为单位）。

句法：

sprite_get_height(index);

争论描述
index要查找高度的子画面的索引。

退货

Real

例子：

if sprite_height != sprite_get_height(sprite_index)
    {
    image_yscale = 1;
    }

上面的代码按当前实例的原样检查精灵的高度，如果该精灵与原始基本精灵之间存在差异，它将重置y轴比例。
