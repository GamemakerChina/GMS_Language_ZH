sprite_get_name

该函数将名称作为指定子画面的字符串返回。 GameMaker Studio 2主窗口的资源树中为精灵指定的名称。请注意，这只是一个字符串，不能用于直接引用sprite，因为您将需要sprite索引。但是，您可以使用此字符串以及返回的字符串和函数来获取子画面索引 asset_get_index()。

句法：

sprite_get_name(index);

争论描述
index要获取名称的子画面的索引。

退货

String

例子：

str = sprite_get_name(sprite_index);

上面的代码将获取运行该代码的实例的Sprite索引的名称，并将返回值存储在变量“ str”中。
