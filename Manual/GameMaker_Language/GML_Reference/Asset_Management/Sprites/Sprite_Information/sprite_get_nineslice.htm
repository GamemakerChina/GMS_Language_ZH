sprite_get_nineslice

此函数返回一个Sprite的Nine Slice结构，包含在Sprite Editor中设置或使用sprite_set_nineslice（）设置的所有Nine Slice属性。此结构的内容在此页面上有详细说明。

如果所提供的Sprite没有为其分配Nine Slice结构，则将为该Sprite创建一个具有默认Nine Slice属性的新结构并返回。

更改此结构中的任何值都将修改原始精灵的“九个切片”属性，从而影响以后使用该精灵进行的任何绘制调用。

句法：

sprite_get_nineslice(ind);

争论描述
ind从其检索九个切片结构的精灵的索引

返回值：

Struct (or -1 if the sprite doesn't exist)

例子：

var _box_nineslice = sprite_get_nineslice(spr_box_0);

_box_nineslice.enabled = true;
_box_nineslice.left = 10;
_box_nineslice.right = 10;
_box_nineslice.top = 10;
_box_nineslice.bottom = 10;

上面的代码从Sprite检索Nine Slice结构，为其启用Nine Slicing并设置引导偏移。
