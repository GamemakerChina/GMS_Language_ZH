sprite_get_number

一个子画面必须至少具有一个子图像，并且此函数将返回它具有多少个子图像。请记住，一个精灵子图像从0开始编号，因此，如果此函数返回3（被检查的精灵具有3个子图像），它们将被编号为0、1和2。

句法：

sprite_get_number(index);

争论描述
index子画面的索引。

退货

Integer

例子：

image_index = floor(random(sprite_get_number(sprite_index)));

上面的代码将根据子画面具有的子图像数将图像索引设置为随机帧。
