幽灵_获取速度_类型

这个函数可以用来检索精灵编辑器中为精灵资源定义的精灵动画类型。返回的值将是以下常数之一。

恒定 描述
spritespeed_framespersecond 该精灵被定义为每秒一帧的动画。
spritespeed_framespergameframe 该精灵被定义为在每一游戏帧的动画。


以下两个例子说明了在计算精灵的动画速度时，这可能会产生不同的效果。

  • 如果你有一个每秒绘制1帧的精灵，并将图像速度设置为0.5，它将以每秒0.5帧的速度绘制。如果你的游戏帧率是每秒60帧，那么该精灵将在每120个游戏帧中绘制1帧。
  • 如果你有一个精灵，每个游戏帧绘制1帧，并将图像速度设置为0.5，它将在每个游戏帧绘制0.5帧。如果你的游戏帧率是每秒60帧，那么该精灵将在每60个游戏帧中绘制30帧。

你可以用函数找出用于精灵动画的速度值。 sprite_get_speed()来了解精灵动画所使用的速度值，还可以用函数来设置动画的速度和类型。 sprite_set_speed().

语法。

sprite_get_speed_type(index)

争论 描述
指数 获取精灵的动画类型的索引。

返回。

常数(见上文)

例子。

s_speed = sprite_get_speed(sprite_index);
s_type = sprite_get_speed_type(sprite_index) 。

上面的代码获得了精灵的速度和精灵的动画类型，并将它们存储在实例变量中，以供将来使用。
