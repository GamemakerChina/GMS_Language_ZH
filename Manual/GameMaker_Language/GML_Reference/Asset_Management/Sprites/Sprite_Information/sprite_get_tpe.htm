幽灵_获取_TPE

这个函数将返回一个特殊的值，它与给定的精灵和子图像在你的游戏使用的纹理页面上的位置有关。通过这种方式，你可以将该值传递给扩展程序，以便在编译的游戏之外使用，有效地绕过了 GameMaker Studio 2的绘图功能，并允许在DOM内容中使用精灵，然后可以在包含游戏画布的窗口中的任何地方绘制。这个功能对于那些希望在GameMaker Studio 2之外创建按钮和其他交互式媒体的人来说是特别有意义的。 和其他交互式媒体的人特别感兴趣，因为他们希望在GameMaker Studio 2画布元素之外创建按钮和其他交互式媒体，在他们的主页面上使用函数 clickable_add().

注意：该功能仅适用于HTML5。

语法。

sprite_get_tpe(sprite, index)。

争论 描述
精灵 要找到纹理页入口的精灵的索引。
指数 该精灵的图像索引。

真实的

例子。

home_but = clickable_add(32, 32, sprite_get_tpe(spr_MS_Home, 0), "http://macsweeney_games.com", "_blank", "width=700, height=500, menubar=0, toolbar=0, scrollbars=0") 。

上面的代码在游戏画布运行的页面的位置（32，32）创建了一个DOM按钮。该按钮使用从纹理页面引用的精灵 "spr_MS_Home"，当点击该按钮时，将为指定的URL打开一个新窗口，并为该窗口定义属性。 当点击按钮时，将为指定的URL打开一个新窗口，并为该窗口定义属性。
