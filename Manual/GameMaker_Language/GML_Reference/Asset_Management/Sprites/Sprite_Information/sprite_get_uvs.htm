sprite_get_uvs

此函数返回一个带有UV坐标和其他数据的数组，该数组包含纹理页面上精灵子图像的纹理。该函数返回一个包含以下8个元素的数组：

  • [0] =左
  • [1] =顶部
  • [2] =正确
  • [3] =底部
  • [4] =资产编译器从子画面左侧修剪的像素数量
  • [5] =资产编译器已从精灵顶部修剪的像素数量
  • [6] =已保存到纹理页面的原始精灵宽度的归一化像素数据百分比
  • [7] =已保存到纹理页面的原始精灵高度的像素数据的标准化百分比

然后，可以将此数组用于其他绘图函数，特别是在3D中工作或使用2D基本体函数时，以及在使用Shader函数时。

注意：此功能不适用于矢量精灵或骨架动画精灵。

句法：

sprite_get_uvs(sprite, subimage);

争论描述
sprite要使用的精灵的索引。
subimage要使用的子画面的子图像。

退货

Array (1D, 8 elements)

例子：

var tex = sprite_get_uvs(sprite, 0);
tex_left = tex[0];
tex_top = tex[1];
tex_right = tex[2];
tex_bottom = tex[3];

上面的代码将给定精灵的UV坐标存储在本地数组中，然后将值分配给实例变量。

返回： Sprite信息
