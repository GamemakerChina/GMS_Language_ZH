sprite_get_width

使用此功能，您可以找到没有变换的基本资源精灵的宽度（以像素为单位）。

句法：

sprite_get_width(index);

争论描述
index要找到宽度的子画面的索引。

退货

Real

例子：

if sprite_width != sprite_get_width(sprite_index)
    {
    image_xscale = 1;
    }

上面的代码检查当前实例中精灵的宽度，如果该宽度与原始基本精灵之间存在差异，它将重置x轴比例。
