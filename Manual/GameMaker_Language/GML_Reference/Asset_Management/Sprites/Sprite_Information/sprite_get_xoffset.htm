幽灵_获取x偏移

当你在精灵编辑器中定义一个精灵时，你需要为该精灵设置原点。这是精灵在与一个实例一起使用时将被 "连接 "或 "绘制 "的点。这个函数返回精灵的X轴的相对 原点的X轴的相对偏移，以左上角为0,0的位置为基础，+x为右，-x为左。下面的图片说明了这一点。

语法。

sprite_get_xoffset(index)。

争论 描述
指数 要寻找精灵的xoffset的索引。

返回

真实的

例子。

if x - sprite_get_xoffset(sprite_index) < 0
{
x = sprite_get_xoffset(sprite_index);
}

上面的代码将确保在房间内根据精灵保持一个实例（所以精灵总是可见的）。
