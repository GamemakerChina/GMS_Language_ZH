sprite_get_xoffset

在Sprite编辑器中定义Sprite时，需要设置该Sprite的原点。这是与实例一起使用时将“附加”或“绘制”精灵的位置。该函数基于左上角在0,0位置返回原点x轴的相对偏移量，其中+ x右边，-x左边。下图说明了这一点：

句法：

sprite_get_xoffset(index);

争论描述
index要查找其xoffset的子画面的索引。

退货

Real

例子：

if x - sprite_get_xoffset(sprite_index) < 0
    {
    x = sprite_get_xoffset(sprite_index);
    }

上面的代码将确保根据Sprite在房间内维护一个实例（因此Sprite始终可见）。
