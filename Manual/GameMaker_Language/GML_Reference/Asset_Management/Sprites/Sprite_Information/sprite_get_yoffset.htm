幽灵_获取yoffset

当你在精灵编辑器中定义一个精灵时，你需要为该精灵设置原点。这是精灵在与一个实例一起使用时将被 "连接 "或 "绘制 "的点。这个函数返回 原点的Y轴的相对偏移量，以左上角的0,0位置为基础，+y表示向下，-y表示向上。下面的图片说明了这一点。

语法。

sprite_get_yoffset(index)。

争论 描述
指数 要寻找精灵的偏移量的索引。

返回

真实的

例子。

if y - sprite_get_yoffset(sprite_index) < 0
{
y = sprite_get_yoffset(sprite_index);
}

上面的代码将确保在房间内根据精灵保持一个实例（所以精灵总是可见的）。
