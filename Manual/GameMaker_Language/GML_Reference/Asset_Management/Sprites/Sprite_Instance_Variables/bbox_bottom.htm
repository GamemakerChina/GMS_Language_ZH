bbox_bottom

这个只读变量返回实例的边界框底部的y位置（在房间内），其中边界框是由实例的遮罩的最大宽度和高度定义的（如 sprite_index 或 mask_index 设置的）。)即使一个精灵有一个精确的碰撞遮罩，边界框也是存在的，并被用于某些事情。 的，所以你可以用这个变量来找到它。请注意，当实例没有分配精灵时，返回的值将与实例的Y位置相同。

语法。

bbox_bottom;

返回。

实数（整数

例子。

如果bbox_bottom > room_height
{
y = room_height - sprite_yoffset;
}

上面的代码检查实例的边界框是否在房间外面，如果是，就把实例设置到一个新的位置。
