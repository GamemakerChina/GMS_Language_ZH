bbox_left

此只读变量返回实例左边界框内的位置（沿x轴），其中边界框由实例的蒙版的最大宽度和高度（由sprite_index或通过 mask_index）。即使精灵具有精确的碰撞蒙版，也存在边界框并将其用于某些事物，因此您可以使用此变量进行查找。请注意，当实例未分配精灵时，返回的值将与实例X的位置相同。

句法：

bbox_left;

返回值：

Real (integer)

例子：

if bbox_left < 0
    {
    x = sprite_xoffset;
    }

上面的代码检查实例的边界框是否在房间外面，如果是，它将实例设置到新位置。

返回： Sprite实例变量
