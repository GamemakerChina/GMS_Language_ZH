image_alpha

此变量用于获取或设置精灵的Alpha值。 Alpha始终计算为介于0和1之间的值，其中0完全透明，而1完全不透明。请注意，要使此变量的更改可见，实例应该没有绘制事件（因此GameMaker Studio 2将默认绘制精灵）或使用扩展的绘制功能之一（例如 draw_self() 或者 draw_sprite_ext()。

句法：

image_alpha;

返回值：

Real (between 0 and 1 - single precision floating point value)

例子：

image_alpha = clamp(image_alpha - 0.01, 0, 1);

上面的代码将缓慢减小image_alpha，直到达到0。

返回： Sprite实例变量
