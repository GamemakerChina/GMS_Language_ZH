Image_alpha

这个变量用于获取或设置精灵的Alpha值。阿尔法值总是被计算为0和1之间的值，其中0是完全透明的，1是完全不透明的。请注意，要使这个变量的变化可见，该 实例应该没有绘制事件（因此GameMaker Studio 2将默认绘制精灵），或者使用某个扩展的绘制函数来绘制，例如 draw_self()或 draw_sprite_ext().

语法。

image_alpha。

返回。

实数（在0和1之间--单精度浮点值）。

例子。

image_alpha = clamp（image_alpha - 0.01, 0, 1）。

上面的代码会慢慢减少image_alpha，直到它达到0。
