image_angle

此值设置精灵的角度（旋转），以度为单位，右侧为0º，向上为90º，左侧为180º，向下为270º。将此变量设置为0以重置要在精灵编辑器中定义的要绘制的精灵。请注意，要使此变量的更改可见，实例应该没有绘制事件（因此GameMaker Studio 2将默认绘制精灵）或使用扩展的绘制功能之一（例如 draw_self()或者 draw_sprite_ext()。

句法：

image_angle;

返回值：

Real (single precision floating point value)

例子：

image_angle = point_direction(x, y, mouse_x, mouse_y);

上面的代码将旋转实例的精灵，使其始终指向鼠标位置。

返回： Sprite实例变量
