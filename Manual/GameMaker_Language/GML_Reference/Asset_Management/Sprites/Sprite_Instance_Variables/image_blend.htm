图像混合

这个变量控制实例精灵的 "着色"，默认值是-1（但也可以是c_white）。任何其他值（包括内部颜色常量，如c_red或c_aqua）都会将指定的颜色与原始精灵混合。 颜色与原始精灵混合。请注意，为了使这个变量的变化可见，该实例应该没有绘制事件（因此GameMaker Studio 2将默认绘制精灵），或者使用扩展的绘制 函数绘制，例如 draw_self()或 draw_sprite_ext().

请注意，你应该尽量限制HTML5平台上的混合（除非使用WebGL），因为每个混合的精灵都必须被单独缓存，因此有许多混合的精灵可能会对性能产生不利影响（你也可以使用以下函数设置缓存大小 函数 sprite_set_cache_size()).

语法。

image_blend;

返回。

实数（单精度浮点值

例子。

image_blend = make_colour_hsv(255, 255, random(255))。

上述代码将把为该实例定义的精灵的image_blend设置为随机颜色。
