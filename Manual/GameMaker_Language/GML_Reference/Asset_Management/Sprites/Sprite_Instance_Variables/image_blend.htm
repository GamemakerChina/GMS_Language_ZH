image_blend

此变量控制实例精灵的“着色”，默认值为-1（但也可以是 c_white）。任何其他值（包括内部颜色常量，例如 c_red， 或者 c_aqua）会将指定的颜色与原始精灵混合。请注意，要使此变量的更改可见，实例应该没有绘制事件（因此GameMaker Studio 2将默认绘制精灵）或使用扩展的绘制功能之一（例如 draw_self()或者 draw_sprite_ext()。

请注意，您应该尝试限制HTML5平台上的混合（除非使用WebGL），因为每个混合精灵必须单独缓存，因此拥有多个混合精灵可能会对性能产生不利影响（您也可以使用函数设置缓存大小 sprite_set_cache_size()）。

句法：

image_blend;

返回值：

Real (single precision floating point value)

例子：

image_blend = make_colour_hsv(255, 255, random(255));

上面的代码会将为实例定义的精灵的image_blend设置为随机颜色。
