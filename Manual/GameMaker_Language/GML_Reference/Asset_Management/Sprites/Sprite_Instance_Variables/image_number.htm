image_number

此只读变量可用于获取已分配给实例的子画面中子图像的数量（如果您需要除指定给实例的子画面以外的子画面的子图像数，则应使用 sprite_get_number()）。请注意，例如，当有1个子图像时，此变量将返回“ 1”，但该子图像的image_index为0。

句法：

image_number;

返回值：

Real (single precision floating point value)

例子：

if image_speed > 0
    {
    if image_index > image_number - 1 instance_destroy();
    }

上面的代码检查sprite是否正在设置动画，如果是，则检查当前image_index是否大于子图像的数量，是否破坏实例。

返回： Sprite实例变量
