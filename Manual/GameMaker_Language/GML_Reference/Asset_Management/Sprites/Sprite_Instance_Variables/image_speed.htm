image_speed

此变量确定GameMaker Studio 2在当前实例子画面的子图像之间循环的速度。给定的速度值是一个乘数，默认值为1，将其设置为0.5将使动画速度提高一半（如在Sprite Editor或Image Editor中设置的速度），而将其设置为2则将使动画速度加倍。如果使用的精灵没有子图像，则此变量无效。

句法：

image_speed;

返回值：

Real (single precision floating point value)

例子：

with (instance_create_layer(x, y, "Effects", obj_Explosion))
    {
    image_speed = 0.5;
    }

上面的代码创建对象“ obj_Explosion”的实例，并将其image_speed设置为0.5（有效地将运行速度减半）。

返回： Sprite实例变量
