图像_xscale

这个值设置应用于已分配给当前实例的精灵的水平缩放比例。1表示不缩放（1:1），较小的数值将缩小（例如0.5，将是精灵宽度的一半），较大的数值 将向上缩放，负值将翻转精灵并缩放，除非使用的值正好是-1（在这种情况下，精灵只是围绕其原点翻转，没有缩放）。

语法。

image_xscale;

返回。

实数（单精度浮点值

例子。

如果image_xscale < 5
{
image_xscale += 0.2;
image_yscale = image_xscale;
}
else
{
instance_create_layer(x, y, "Effects", obj_Explosion);
instance_destroy();
}

上面的代码对精灵进行了缩放，然后一旦它被缩放到原始大小的5倍，就会创建另一个对象的新实例并销毁该实例。
