image_yscale

此值设置应用于已分配给当前实例的精灵的垂直缩放比例（沿y轴）。比例尺为1表示无比例缩放（1：1），较小的值将按比例缩小（例如0.5，将是精灵的高度的一半），较大的值将按比例缩放，负值将镜像该精灵并对其进行缩放，除非所使用的值正好为-1（在这种情况下，子画面仅沿y轴镜像，没有缩放比例）。

句法：

image_yscale;

返回值：

Real (single precision floating point value)

例子：

if image_xscale < 5
    {
    image_xscale += 0.2;
    image_yscale = image_xscale;
    }
else
    {
    instance_create_layer(x, y, "Effects", obj_Explosion);
    instance_destroy();
    }

上面的代码将Sprite缩放，然后将其缩放到其原始大小的5倍后，将创建另一个对象的新实例，并销毁该实例。

返回： Sprite实例变量
