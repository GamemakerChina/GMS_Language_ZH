mask_index

GameMaker Studio 2定义一个对象时，可以为其分配一个用于碰撞的蒙版，而不是与定义的精灵对应的一个蒙版。此变量可用于查找sprite_index的蒙版（如果未分配精灵，则它将返回-1）或将实例的蒙版设置为选定的精灵。设置遮罩索引意味着您可以为例如具有不规则形状的实例创建一个精灵，但是为其提供从另一个精灵获得的圆形碰撞蒙版。

句法：

mask_index;

返回值：

Real (integer - resource index value)

例子：

mask_index = spr_Round;

上面的代码将实例的掩码设置为精灵“ spr_Round”的掩码。

返回： Sprite实例变量
