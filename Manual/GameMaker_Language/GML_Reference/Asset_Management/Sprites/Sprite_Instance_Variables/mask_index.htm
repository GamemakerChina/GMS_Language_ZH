掩码_索引

当你在GameMaker Studio 2中定义一个对象时，你可以给它分配一个掩码，用于碰撞，而不是与定义的精灵相对应的那个。这个变量可以用来查找 sprite_index 的掩码（如果没有指定精灵，它将返回-1），或者将一个实例的掩码设置为选定的精灵。设置掩码索引意味着你可以有，例如，一个不规则形状的实例的精灵，但却给它 一个圆形的碰撞遮罩，这个遮罩是由不同的精灵获得的。

语法。

mask_index;

返回。

实数（整数 - 资源索引值）。

例子。

mask_index = spr_Round;

上面的代码将实例的面具设置为精灵 "spr_Round "的面具。
