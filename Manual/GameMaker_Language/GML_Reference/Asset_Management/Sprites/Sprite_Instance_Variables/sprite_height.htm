画面高度

这个只读变量返回已经分配给实例的精灵的高度。这个高度是以像素为单位返回的，并且取决于 image_yscale. 如果你需要未缩放的高度，你应该使用 sprite_get_height().

语法。

萌萌哒_高度。

返回。

实数（单精度浮点值

例子。

if sprite_height != sprite_get_height(sprite_index)
{
image_yscale = 1;
}

上述代码将分配给运行代码的实例的精灵的高度与精灵资源的高度进行核对，如果不一致，就将image_yscale重置为1。
