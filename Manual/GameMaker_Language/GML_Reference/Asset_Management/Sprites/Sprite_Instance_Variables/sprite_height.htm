sprite_height

此只读变量返回已分配给实例的子画面的高度。此高度以像素为单位返回，并且取决于 image_yscale。如果需要不按比例缩放的高度，则应使用 sprite_get_height()。

句法：

sprite_height;

返回值：

Real (single precision floating point value)

例子：

if sprite_height != sprite_get_height(sprite_index)
    {
    image_yscale = 1;
    }

上面的代码会根据Sprite资源的高度检查分配给运行该代码的实例的Sprite的高度，如果不相同，则会重置 image_yscale至1

返回： Sprite实例变量
