sprite_index

此变量返回实例的当前Sprite的索引，如果实例没有与之关联的Sprite，则返回-1。您可以更改它，以通过从资源树中为其指定一个精灵的名称或使用在其中索引了外部加载的精灵的变量来为该实例赋予不同的精灵。更改子画面不会更改当前可见帧的索引，因此，如果更改第3帧上的子画面，则将在该帧可见的情况下绘制新的子画面（假定它具有相同的帧数）。

句法：

sprite_index;

返回值：

Real (integer - resource index value)

例子：

with (obj_Check)
    {
    if !collision_line(x, y, other.x, other.y, obj_Wall, false, true)
        {
        sprite_index = spr_spotted;
        }
    else
        {
        sprite_index = spr_clear;
        }
    }

上面的代码将遍历“ obj_Check”的所有实例，以检查它们与运行该代码的实例之间是否存在冲突线。这些实例的精灵将根据碰撞线的返回值（true或false）进行更改。

返回： Sprite实例变量
