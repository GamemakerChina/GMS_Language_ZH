魉的宽度

这个只读变量返回分配给该实例的精灵的宽度。这个宽度是以像素为单位返回的，取决于 Image_xscale.如果你需要未缩放的宽度 你应该使用 sprite_get_width().

语法。

萌芽宽度。

返回。

实数（单精度浮点值

例子。

if sprite_width != sprite_get_width(sprite_index)
{
image_xscale = 1;
}

上述代码将分配给运行代码的实例的精灵的宽度与精灵资源的宽度进行对比，如果不一样，就将image_xscale重置为1。
