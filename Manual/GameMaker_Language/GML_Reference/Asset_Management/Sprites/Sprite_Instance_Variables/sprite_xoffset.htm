魉_xoffset

这个只读变量返回已经分配给实例的精灵的本地xoffset（精灵编辑器中定义的原点的x分量）。这个xoffset是以像素为单位返回的，取决于 image_xscale, 所以（例如）如果一个32x32的正方形精灵被定义为x原点在16，当该精灵被缩放为2时，这个变量将返回32。如果你需要未缩放的xoffset，你应该使用 sprite_get_xoffset().

语法。

幽灵_xoffset。

返回。

实数（单精度浮点值

例子。

如果x - sprite_xoffset < 0
{
x = sprite_xoffset;
}

上面的代码检查实例的精灵的左边缘是否在房间外面，如果是，它就移动实例，使整个精灵可见。
