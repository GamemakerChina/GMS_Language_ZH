sprite_xoffset

此只读变量返回已分配给实例的子画面的局部xoffset（在子画面编辑器中定义的原点的x分量）。该xoffset以像素为单位返回，并且取决于 image_xscale，这样（例如），如果将一个32x32方形精灵定义为x的原点为16，当该精灵按2缩放时，此变量将返回32。sprite_get_xoffset() 。

句法：

sprite_xoffset;

返回值：

Real (single precision floating point value)

例子：

if x - sprite_xoffset < 0
    {
    x = sprite_xoffset;
    }

上面的代码检查实例精灵的左边缘是否在房间外面，如果是，它将移动实例，以便整个精灵可见。

返回： Sprite实例变量
