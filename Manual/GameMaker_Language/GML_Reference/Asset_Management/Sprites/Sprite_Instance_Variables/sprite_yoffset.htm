sprite_yoffset

此只读变量返回已分配给实例的子画面的局部yoffset（在子画面编辑器中定义的原点的y分量）。此yoffset以像素为单位返回，并且取决于 image_yscale，这样（例如），如果将一个32x32方形精灵定义为y的原点为16，当该精灵按2缩放时，此变量将返回32。如果需要未缩放的yoffset，则应使用 sprite_get_yoffset()。

句法：

sprite_yoffset;

返回值：

Real (single precision floating point value)

例子：

if y - sprite_yoffset < 0
    {
    y = sprite_yoffset;
    }

上面的代码检查实例精灵的顶部边缘是否在房间外部，是否移动实例使整个精灵可见。

返回： Sprite实例变量
