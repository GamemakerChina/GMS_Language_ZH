鼠标左键

这个只读变量返回分配给该实例的精灵的局部偏移量（在精灵编辑器中定义的原点的y分量）。这个yoffset以像素为单位返回，并取决于 图像的比例, 所以（例如）如果一个32x32的正方形精灵被定义为y原点为16，当该精灵被缩放为2时，这个变量将返回32。如果你需要未缩放的yoffset，你应该使用 sprite_get_yoffset().

语法。

魉_yoffset。

返回。

实数（单精度浮点值

例子。

if y - sprite_yoffset < 0
{
y = sprite_yoffset;
}

上面的代码检查实例的精灵的顶部边缘是否在房间外面，如果是，就移动实例，使整个精灵可见。
