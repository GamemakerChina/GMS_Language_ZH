删除精灵

这个函数将从游戏中删除一个精灵，释放为其保留的任何内存。这是一个非常重要的函数，当你需要从外部来源创建和改变精灵时（比如从一个文件中加载一个精灵，用 sprite_add() ，或用 sprite_duplicate() 来复制一个精灵），并且应该总是用来删除那些游戏不再需要的资产。 游戏不再需要的资产，或者在为某一变量重新分配另一资产之前清除该变量中的索引资产。这不能用来删除作为资产浏览器中资产的一部分而包含在游戏中的精灵。

语法。

sprite_delete(index)。

争论 描述
指数 要删除的精灵的索引。

返回

不适用

例子。

var spr = sprite_create_from_surface(application_surface, 0, 0, 32, 32, false, false, 16, 16);
sprite_merge(sprite_Player, spr);
sprite_delete(sprite);

上面的代码创建了一个局部变量，然后存储了从应用程序表面创建的精灵的索引。然后这个精灵与变量 "spr_Player "中索引的资产合并，再从内存中删除。
