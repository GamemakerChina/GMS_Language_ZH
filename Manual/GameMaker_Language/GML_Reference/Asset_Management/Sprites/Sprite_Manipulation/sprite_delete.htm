sprite_delete

此功能将从游戏中删除一个精灵，释放为其保留的所有内存。在您需要从外部源创建和更改精灵（例如使用以下命令从文件加载精灵）时，这是一项非常重要的功能： sprite_add() ，或使用复制副本 sprite_duplicate()），并且应始终用于删除游戏不再需要的那些资产，或在将另一资产重新分配给该变量之前从变量中清除索引资产。这不能用于删除资产浏览器中作为资产一部分而包含在游戏中的精灵。

句法：

sprite_delete(index);

争论描述
index要删除的精灵的索引。

退货

N/A

例子：

var spr = sprite_create_from_surface(application_surface, 0, 0, 32, 32, false, false, 16, 16);
sprite_merge(spr_Player, spr);
sprite_delete(spr);

上面的代码创建一个局部变量，然后存储从应用程序表面创建的Sprite的索引。然后，将此精灵与在变量“ spr_Player”中索引的资产合并，然后再次从内存中删除。
