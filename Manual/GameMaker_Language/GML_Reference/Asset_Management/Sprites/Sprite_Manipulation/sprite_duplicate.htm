sprite_duplicate

此函数将返回一个新创建的精灵的索引，该索引是一个输入的副本（副本），作为“ index”参数。如果函数失败，则返回-1。必须先使用此功能从原始资产复制任何精灵，然后才能对其进行任何转换。例如，如果您想更改一个精灵的边界框，或设置另一个精灵的Alpha，则必须首先对其进行复制，然后对复制的Sprite执行操作并使用它。创建后，复制的精灵将放置在其自己的唯一纹理页面上，这意味着复制多个精灵将创建多个纹理页面并影响性能，因此仅在必要时使用此功能。

注意GameMaker Studio 2复制精灵时，必须记住当不再需要它时（使用sprite_delete（） ）再次将其删除，否则存在内存泄漏的风险，这会减慢速度并最终导致游戏崩溃。

句法：

sprite_duplicate(index);

争论描述
index要复制的精灵的索引。

退货

Real (unique sprite ID value)

例子：

new_spr = sprite_duplicate(sprite_index)

上面的代码复制了当前用作实例的精灵索引的精灵，并将此新精灵的索引存储在变量“ new_spr”中。
