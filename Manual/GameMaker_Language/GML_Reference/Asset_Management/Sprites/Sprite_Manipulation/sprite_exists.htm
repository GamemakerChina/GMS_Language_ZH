幽灵_存在

该函数返回正在运行的项目中是否存在具有指定索引的精灵。

语法。

sprite_exists(index)。

争论 描述
指数 要检查的精灵的索引。

返回

不适用

例子。

if sprite_exists(spr_array[0])
{
sprite_index = spr_array[0];
}
else
{
sprite_index = spr_BaseSprite;
}

上面的代码检查一个数组，看它是否包含一个有效的精灵索引，如果是，就把这个精灵分配给实例，如果不是，就从包含的资源中分配一个精灵。
