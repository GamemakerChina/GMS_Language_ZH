sprite_exists

此函数返回正在运行的项目中是否存在具有指定索引的子画面。

句法：

sprite_exists(index);

争论描述
index要检查的精灵的索引。

退货

N/A

例子：

if sprite_exists(spr_array[0])
    {
    sprite_index = spr_array[0];
    }
else
    {
    sprite_index = spr_BaseSprite;
    }

上面的代码检查一个数组，以查看它是否包含有效的sprite索引，如果是，则将其分配给实例，但如果不是，则从包含的资源中分配一个sprite。
