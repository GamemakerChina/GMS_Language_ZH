创建精灵_nineslice_create

九宫格切片是一种用于缩放矩形图像以保留细节的技术，使其在缩放后保持原有的形态。关于九宫格切片的信息，请参见本页面。

该函数用于创建一个九宫格结构，该结构可以被修改，然后应用于精灵。该结构的内容详见本页面。

你可以将这个结构存储在一个变量中，修改其属性，并使用 sprite_set_nineslice() ，将其应用于任何精灵。

语法。

sprite_nineslice_create()。

返回。

结构

例子。

var _nineslice =sprite_nineslice_create();

_nineslice.enabled = true;
_nineslice.left = 10;
_nineslice.right = 10;
_nineslice.top = 10;
_nineslice.bottom = 10;

sprite_set_nineslice( spr_box_0, _nineslice) 。

上面的代码创建了一个新的Nine Slice结构，将其启用并为导引器设置了偏移量。然后将该结构应用于精灵，改变其Nine Slice属性。
