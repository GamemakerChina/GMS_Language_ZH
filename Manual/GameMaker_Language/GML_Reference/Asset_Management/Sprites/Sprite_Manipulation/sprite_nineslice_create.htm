sprite_nineslice_create

九级切片是一种用于缩放矩形图像以保留细节的技术，使矩形图像在缩放后仍保持其原始形状。有关九片的信息，请参见此页面。

此函数用于创建九个切片结构，可以对其进行修改然后将其应用于精灵。此结构的内容在此页面上有详细说明。

您可以将该结构存储在变量中，修改其属性，然后使用以下命令将其应用于任何精灵 sprite_set_nineslice()。

句法：

sprite_nineslice_create();

返回值：

Struct

例子：

var _nineslice = sprite_nineslice_create();

_nineslice.enabled = true;
_nineslice.left = 10;
_nineslice.right = 10;
_nineslice.top = 10;
_nineslice.bottom = 10;

sprite_set_nineslice(spr_box_0, _nineslice);

上面的代码创建一个新的Nine Slice结构，启用它并设置参考线的偏移量。然后将结构应用于精灵，更改其“九片”属性。
