sprite_prefetch_multi

此功能可用于预取（放入纹理内存中）包含给定精灵的许多纹理页面。您提供一个数组，其中填充了精灵索引（在创建精灵资源时定义），并且它们所在的纹理页面将被加载到内存中。请注意，该函数将返回 -1如果所选资源不支持预取，或者目标平台为HTML5，否则它将返回 0如果一切正常。

注意：由于大多数设备上的纹理都已上载到纹理内存中，因此性能会受到影响，因此建议您在关卡开始时在任何必需的图形上调用sprite_prefetch_multi（）以避免出现停顿。

句法：

sprite_prefetch_multi

争论描述
array带有精灵索引的数组以获取

返回值：

Int (-1 or 0)

例子：

spr_a[0] = spr_Player_Aura1;
spr_a[1] = spr_Player_Aura2;
spr_a[2] = spr_Player_Aura3;
spr_a[3] = spr_Player_Aura4;
sprite_prefetch_multi(spr_a);

上面的代码创建一个数组，其中每个元素都包含一个精灵索引。然后使用该数组将这些精灵纹理放置到内存中。
