萌芽_预取_多

这个函数可以用来预取（放置到纹理内存中）一些包含给定精灵的纹理页。你提供一个数组，里面有精灵的索引（如你在创建精灵资产时定义的），它们所在的纹理页将被载入内存。 它们所在的纹理页将被加载到内存中。请注意，如果所选资源不支持预取，或者目标平台是HTML5，则该函数将返回-1；如果一切正常，则返回0。 所有工作都正常。

注意：在大多数设备上，当纹理被上传到纹理内存时，会有一个性能上的冲击，所以建议你调用 sprite_prefetch_multi()在 任何需要的图形，以避免任何停顿。

语法。

萌芽_预取_多

争论 描述
阵列 包含要获取的精灵索引的数组

返回。

英特(-1或0)

例子。

spr_a[0] = spr_Player_Aura1;
spr_a[1] = spr_Player_Aura2;
spr_a[2] = spr_Player_Aura3;
spr_a[3] = spr_Player_Aura4;
sprite_prefetch_multi(spr_a) 。

上述代码创建了一个数组，其中每个元素持有一个精灵索引。然后，这个数组被用来将这些精灵的纹理放入内存。
