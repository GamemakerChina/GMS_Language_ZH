幽灵_保存

该函数可用于将精灵的任何子图像保存到光盘中，给它指定文件名。该精灵必须是在运行时 添加的（你不能保存通过IDE添加的精灵），文件必须以*.png为扩展名保存。 文件必须以*.png为扩展名。

注意：根据所选择的目标平台，你可以从哪里保存和加载文件是有限制的。更多信息请参见文件处理。

语法。

sprite_save(ind, subimg, fname)。

争论 描述
ind 要保存的精灵的索引。
亚img 要保存的精灵的子图像。
fname 保存的精灵的文件名。

返回

不适用

例子。

var surf, spr_custom;
surf = surface_create(32, 32);
surface_set_target(surface);
draw_clear_alpha(c_black, 0);
draw_sprite(spr_Body, 0, 0, 0);
draw_sprite(spr_Clothes, 0, 0, 0);
draw_sprite(spr_Hair, 0, 0, 0);
spr_custom = sprite_create_from_surface(surface, 0, 0, 32, 32, true, true, 16, 16);
surface_reset_target() 。
surface_free(surface)。
sprite_save(spr_custom, 0, "Player_Custom_Sprite.png");
sprite_delete(spr_Custom);

上面的代码创建了一个表面，并将其索引存储在本地变量 "surf "中。然后，它以该表面为目标，将其清除，并在上面绘制几个精灵。它从曲面上绘制的合成图像中创建一个新的精灵 并将其索引分配给本地变量 "spr_Custom"，然后释放表面所使用的内存。最后，生成的精灵被保存到一个文件中，然后也从内存中删除。
