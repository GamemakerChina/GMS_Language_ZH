sprite_set_bbox

此功能可用于设置子画面的边界框值。您提供要使用的精灵索引，然后提供边界框位置的左，上，右和下值。这些位置是绝对值，其中（0，0）位置对应于该精灵的左上角，而不管该精灵的偏移量，该精灵可能具有的任何“空”像素还是在精灵中绘制的位置房间。

注意：此功能会影响精灵资源，因此具有该精灵的所有其他实例将具有相同的边界框。

句法：

sprite_set_bbox(ind, left, top, right, bottom);

争论描述
ind设置边框的精灵的索引。
left边界框的左侧
top边界框的顶部。
right边界框的右侧
bottom边界框的底部。

N/A

例子：

if sprite_get_bbox_mode(sprite_index) == 2
    {
    left = irandom(sprite_width / 2);
    right = irandom((sprite_width / 2) + irandom(sprite_width / 2));
    top = irandom(sprite_height / 2);
    bottom = irandom((sprite_height / 2) + irandom(sprite_height / 2));
    sprite_set_bbox(sprite_index, left, top, right, bottom);
    }

上面的代码将检查分配给的精灵的边框模式 sprite_index，如果设置为 manual那么它将更改其边界框。
