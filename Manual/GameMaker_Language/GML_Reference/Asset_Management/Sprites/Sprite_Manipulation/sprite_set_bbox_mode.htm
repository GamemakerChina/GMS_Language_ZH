sprite_set_bbox_mode

此功能可用于设置精灵的边框模式。您提供了精灵索引和要使用的模式，它应该是以下常量之一：

持续的描述
bboxmode_automatic自动-边界框将根据子画面的公差设置自动计算
bboxmode_fullimage全图-边界框将设置为使用精灵的整个宽度和高度，而不考虑容差和“空”像素
bboxmode_manual手动-手动将边界框设置为用户定义的值，这需要使用该功能 sprite_set_bbox()

注意：此功能会影响子画面资源，因此具有该子画面的所有其他实例将具有相同的边界框模式。

句法：

sprite_set_bbox_mode(ind, mode);

争论描述
ind更改模式的精灵的索引。
mode要设置的模式（一个常量）。

退货

N/A

例子：

if sprite_get_bbox_mode(sprite_index) != 0
    {
    sprite_set_bbox_mode(sprite_index, bboxmode_automatic);
    }

上面检查当前精灵的bbox模式，如果不是 automatic，然后将其设置为该值。
