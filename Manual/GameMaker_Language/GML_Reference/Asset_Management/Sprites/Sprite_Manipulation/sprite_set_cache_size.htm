幽灵_set_cache_size

在执行图像混合时，HTML5 无法以可执行文件的方式动态地执行它。因此，GameMaker Studio 2必须临时保存图像的混合副本，并将其载入。这个函数设置了多少个混合的副本 在旧的副本被覆盖之前，可以缓存多少个给定精灵的混合副本。默认为4。这适用于精灵的所有子图像。

语法。

sprite_set_cache_size(ind, max)。

争论 描述
ind 要改变缓存大小的精灵的索引。
最大 可以存储的精灵的最大缓存份数。

返回

不适用

例子。

sprite_set_cache_size(sprite0, 2)。

这将把 sprite0的所有子图像的精灵缓存设置为2份。
