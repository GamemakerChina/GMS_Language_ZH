sprite_set_cache_size_ext

在执行图像混合时，HTML5无法以可执行文件执行的方式动态地执行此操作。因此， GameMaker Studio 2必须临时保存图像的混合副本并将其加载。此功能设置在覆盖旧精灵之前，可以缓存给定精灵的多少个混合副本。默认值为4。这将应用于精灵的一个给定的子图像。

句法：

sprite_set_cache_size_ext(ind, index, max);

争论描述
ind Sprite的索引，用于更改其缓存大小。
index ind的子映像更改其缓存大小。
max可以存储的sprite的最大缓存副本数。

退货

N/A

例子：

sprite_set_cache_size_ext(sprite0, 0, 2);

这会将sprite0的第一个子图像的sprite缓存设置为2个副本。
