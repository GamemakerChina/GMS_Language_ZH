sprite_set_nineslice

此函数用于将Nine Slice结构应用于子画面。您提供要应用的精灵索引和九个切片结构，可以使用以下方法检索sprite_nineslice_create()或者sprite_get_nineslice() 。

注：此功能会影响精灵的资产，以便您使用此功能的任何变化都会影响使用这个精灵的所有实例和所有未来实例了。

句法：

sprite_set_nineslice(ind, nineslice);

争论描述
ind要修改的精灵的索引
nineslice九片式结构适用

返回值：

N/A

例子：

var _nineslice = sprite_nineslice_create();

_nineslice.enabled = true;
_nineslice.left = 10;
_nineslice.right = 10;
_nineslice.top = 10;
_nineslice.bottom = 10;

sprite_set_nineslice(spr_box_0, _nineslice);

上面的代码创建一个新的Nine Slice结构，启用它并设置参考线的偏移量。然后将结构应用于精灵，更改其“九片”属性。
