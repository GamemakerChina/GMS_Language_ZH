幽灵的位置

这个函数可以用来设置精灵的X和Y原点，并以精灵的左上角（0,0）位置为基础取相对值。下面的图片说明了这一点。

注意：这个函数会影响精灵资产 ，因此，所有带有这个精灵的进一步实例都会有相同的偏移。

语法。

sprite_set_offset(ind, xoff, yoff)。

争论 描述
ind 要改变偏移量的精灵的索引。
xoff 原点的X位置。
yoff 原点的y位置。

返回

不适用

例子。

sprite_assign(spr_Custom, spr_Base);
sprite_set_offset(spr_Custom, sprite_get_xoffset(spr_Base), sprite_get_yoffset(spr_Base) )。

上述代码将 "spr_Base "中索引的精灵分配给 "spr_Custom "中索引的精灵，然后使用 "spr_Base "的x和y偏移值来设置新精灵的原点。
