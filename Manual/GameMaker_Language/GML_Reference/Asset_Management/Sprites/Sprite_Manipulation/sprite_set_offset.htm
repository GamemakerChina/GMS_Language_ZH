sprite_set_offset

此函数可用于设置子画面的x和y原点，并基于（0,0）位置作为子画面的左上角获取相对值。下图说明了这一点：

注意：此功能会影响子画面资源，因此具有该子画面的所有其他实例将具有相同的偏移量。

句法：

sprite_set_offset(ind, xoff, yoff);

争论描述
ind Sprite的索引，以更改其偏移量。
xoff原点的x位置。
yoff原点的y位置。

退货

N/A

例子：

sprite_assign(spr_Custom, spr_Base);
sprite_set_offset(spr_Custom, sprite_get_xoffset(spr_Base), sprite_get_yoffset(spr_Base));

上面的代码将在“ spr_Base”中索引的精灵分配给在“ spr_Custom”中索引的精灵，然后使用“ spr_Base”的x和y偏移值设置新的精灵的原点。
