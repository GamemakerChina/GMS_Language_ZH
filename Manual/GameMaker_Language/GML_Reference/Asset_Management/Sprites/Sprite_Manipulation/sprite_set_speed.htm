sprite_set_speed

此功能可用于设置基本动画的速度和类型，与精灵资源相同。这通常是在Sprite Editor中设置的，但有时可能需要您手动为给定的sprite资源设置此设置。您需要提供要设置的精灵索引，动画速度和要应用的动画速度类型。动画速度类型可以是下列常数之一：

持续的描述
spritespeed_framespersecond用动画以每秒帧为单位定义精灵。
spritespeed_framespergameframe该精灵是用动画定义的，以每游戏帧为单位。


选择用于计算动画的速度类型非常重要，因为这两个选项之间存在巨大差异。以下两个示例说明了这种差异：

  • 如果有，吸引每秒1帧子画面和图像的速度设定为0.5，将绘制以每秒0.5帧。如果您的游戏帧率为每秒60帧，则精灵将每120个游戏帧绘制1帧。
  • 如果您有一个精灵，每游戏帧绘制1帧并将图像速度设置为0.5，则它将每游戏帧绘制0.5帧。如果您的游戏帧率为每秒60帧，则精灵将每60个游戏帧绘制30帧。

您可以使用函数找出用于精灵动画的速度值 sprite_get_speed()，您可以使用来获得动画速度类型 sprite_get_speed_type()。

重要的！此功能会影响精灵的资源，以便您使用此功能的任何更改将影响使用精灵所有实例和所有未来实例了。

句法：

sprite_set_speed(index, speed, type)

争论描述
index精灵的索引，以获取的动画类型

返回值：

Constant (see above)

例子：

sprite_set_speed(sprite_index, 10, spritespeed_framespersecond);

上面的代码将分配给实例的精灵的基本动画速度设置为每秒10帧。
