幽灵_设置速度

这个函数可以用来设置精灵资源的基本动画速度和类型。这通常是在 "精灵编辑器"中设置的，但在某些情况下，你可能需要 为一个给定的精灵资源手动设置这个。你需要提供要设置的精灵索引、动画速度和要应用的动画速度类型。动画速度类型可以是以下任何一个常数。

恒定 描述
spritespeed_framespersecond 该精灵被定义为每秒一帧的动画。
spritespeed_framespergameframe 该精灵被定义为在每一游戏帧的动画。


选择用于计算动画的速度类型是非常重要的，因为两个选项之间有很大的区别。下面的两个例子说明了这种区别。

  • 如果你有一个每秒绘制1帧的精灵，并将图像速度设置为0.5，它将以每秒0.5帧的速度绘制。如果你的游戏帧率是每秒60帧，那么这个精灵将在每120个游戏帧中绘制1帧 游戏帧数。
  • 如果你有一个精灵，每游戏帧画1帧，并将图像速度设置为0.5，它将每游戏帧画0.5帧。如果你的游戏帧率是每秒60帧，那么这个精灵将在每60个游戏帧中绘制30帧。 每60个游戏帧将绘制30帧。

你可以用函数找出用于精灵动画的速度值。 sprite_get_speed()来了解精灵动画的速度值，你还可以用 sprite_get_speed_type().

重要的是!这个函数会影响到精灵资源，所以你用这个函数做的任何改变都会影响到所有正在使用这个精灵的实例，以及所有未来的 实例也会受到影响。

语法。

sprite_set_speed(index, speed, type)

争论 描述
指数 获取精灵的动画类型的索引。

返回。

常数(见上文)

例子。

sprite_set_speed(sprite_index, 10, spritespeed_framespersecond)。

上面的代码将分配给实例的精灵的基本动画速度设置为每秒10帧。
