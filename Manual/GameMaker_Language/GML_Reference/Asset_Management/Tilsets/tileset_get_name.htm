tileset_get_name

此函数将名称作为指定图块集的字符串返回。 GameMaker Studio 2主窗口的“资源浏览器”中为图块集指定的名称。请注意，这只是一个字符串，不能用于直接引用图块集-因为您将需要图块集索引。但是，您可以使用此字符串使用返回的字符串和函数来获取图块集索引 asset_get_index()。

句法：

tileset_get_name(index);

争论描述
index要获取其名称的图块的索引。

退货

String

例子：

var _l = layer_get_id("tilemap_trees");
var _m = layer_tilemap_get_id(_l);
var _t = tilemap_tileset(_m);
tileset_name = tileset_get_name(_t);

上面的代码将获取给定图层的图块集索引的名称，并将返回字符串存储在变量中 tileset_name。
