tileset_get_name

这个函数将返回指定的瓷砖组的名称，是一个字符串。这个名字是在GameMaker Studio 2主窗口的资产浏览器中为瓦片组指定的名字。请注意，这只是一个字符串 不能用来直接引用瓦片组 - 为此你需要瓦片组的索引。然而，你可以使用这个字符串来获得瓷砖组的索引，使用返回的字符串和函数 asset_get_index().

语法。

tileset_get_name(index)。

争论 描述
指数 要获得的瓦片集的名称的索引。

返回

字符串

例子。

var _l = layer_get_id("tilemap_trees");
var _m = layer_tilemap_get_id(_l);
var _t = tilemap_tileset(_m);
tileset_name = tileset_get_name(_t) 。

上面的代码将获得给定图层的瓷砖组索引名称，将返回的字符串存储在变量tileset_name中。

返回。瓷砖套装
