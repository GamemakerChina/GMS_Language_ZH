tileset_get_texture

这个函数为瓦片集纹理页返回一个特殊的指针。这个值可以在其他绘制函数中使用，特别是在2D原始函数中，以及在 以及 着色器函数中使用。你可以使用这里找到的不同的texture_函数获得更多关于返回的纹理页的信息。

语法。

tileset_get_texture(tileset)。

争论 描述
瓷砖集 要使用的瓦片组的索引。

返回

指针（纹理页ID）。

例子。


var tex;
tex = tileset_get_texture(spr_Wall, 0);
draw_primitive_begin_texture(pr_trianglestrip, tex);
draw_vertex_texture(0, 0, 0, 0);
draw_vertex_texture(480, 0, 1, 0);
draw_vertex_texture(480, 640, 1, 1) 640, 0, 1);
draw_primitive_end()。

上面的代码将绘制一个4个顶点的三角形条状物，其纹理保存在 "tex "变量中。

返回。瓷砖套装
