tileset_get_texture

此函数为平铺集纹理页面返回一个特殊的指针。然后可以将此值用于其他绘图函数，尤其是2D图元函数以及Shader函数。您可以使用其他方法获取有关返回的纹理页面的更多信息。texture_在这里找到功能。

句法：

tileset_get_texture(tileset);

争论描述
tileset要使用的图块的索引。

退货

Pointer (texture page ID)

例子：

var tex;
tex = tileset_get_texture(spr_Wall, 0);
draw_primitive_begin_texture(pr_trianglestrip, tex);
draw_vertex_texture(0, 0, 0, 0);
draw_vertex_texture(480, 0, 1, 0);
draw_vertex_texture(480, 640, 1, 1);
draw_vertex_texture(0, 640, 0, 1);
draw_primitive_end();

上面的代码将绘制一个4顶点三角带，其纹理保留在“ tex”变量中。
