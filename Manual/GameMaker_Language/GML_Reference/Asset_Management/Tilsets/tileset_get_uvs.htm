tileset_get_uvs

这个函数返回一个数组，其中包含给定瓦片的纹理页上设置的UV坐标和其他数据。该函数返回一个包含以下8个元素的数组。

  • [0] = 左
  • [1] = 顶部
  • [2] = 对
  • [3]=底部
  • [4] = 资产编译器从瓦片组左侧修剪的像素数量
  • [5] = 资产编译器从瓦片组顶面修剪的像素数量
  • [6] = 已保存到纹理页的原始瓦片组宽度的像素数据的正常化百分比
  • [7] = 已保存到纹理页的原始瓦片组高度的像素数据的归一化百分比

然后，这个数组可以被用于其他的绘制函数，特别是在3D工作或使用2D原始函数时，以及在使用 着色器函数时。

语法。

tileset_get_uvs(tileset)。

争论 描述
瓷砖集 要使用的瓦片组的索引。

返回

数组 (1D, 8个元素)

例子。

var tex = tileset_get_uvs(tl_Grass);
tex_left = tex[0];
tex_top = tex[1];
tex_right = tex[2];
tex_bottom = tex[3];

上面的代码将把给定瓦片组的UV坐标存储在一个本地数组中，然后把数值分配给实例变量。
