tileset_get_uvs

此函数返回一个数组，该数组包含在纹理页面上设置的给定图块的纹理的UV坐标和其他数据。该函数返回一个包含以下8个元素的数组：

  • [0] =左
  • [1] =顶部
  • [2] =正确
  • [3] =底部
  • [4] =资产编译器从图块集左侧修剪的像素数量
  • [5] =资产编译器从图块设置的顶部修剪的像素数量
  • [6] =已保存到纹理页面的原始图块集宽度中的像素数据的归一化百分比
  • [7] =已保存到纹理页面的原始图块设置高度上的像素数据的标准化百分比

然后，可以将此数组用于其他绘图函数，特别是在3D中工作或使用2D基本体函数时，以及在使用Shader函数时。

句法：

tileset_get_uvs(tileset);

争论描述
tileset要使用的图块的索引。

退货

Array (1D, 8 elements)

例子：

var tex = tileset_get_uvs(tl_Grass);
tex_left = tex[0];
tex_top = tex[1];
tex_right = tex[2];
tex_bottom = tex[3];

上面的代码将给定图块集的UV坐标存储在本地数组中，然后将值分配给实例变量。
