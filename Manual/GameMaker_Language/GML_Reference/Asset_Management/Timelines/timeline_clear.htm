timeline_clear

使用此功能，您可以清除“时刻”的特定时间线，删除该时间线的所有代码和操作并将其留空。

句法：

timeline_clear(ind);

争论描述
ind要清除的时间线的索引。

返回值：

N/A

例子：

if timeline_position > 200
    {
    timeline_clear(global.tl);
    timeline_index = -1;
    }

上面的代码将清除经过特定时刻后所有时刻的变量“ global.tl”索引的指定时间线。
