时间线_删除

通过这个函数，你可以从你的游戏中删除任何时间线。如果这个时间线是在GameMaker Studio 2资产浏览器中创建的，请注意，它将从游戏中完全删除，在使用这个函数后试图再次调用索引的 时间线会导致错误。如果时间线是用函数动态创建的，那么这个函数应该用 Timeline_add()函数动态创建的，那么这个函数应该在不再需要时间线的时候使用 如果时间线是用函数动态创建的，那么这个函数应该在不再需要时间线的时候使用，以防止任何内存泄漏，从而使你的游戏变慢（并最终崩溃）。

语法。

Timeline_delete(ind)。

争论 描述
ind 要删除的时间线的索引。

返回。

不适用

例子。

if timeline_exists(global.tl)
{
timeline_delete(global.tl);
}

上述代码检查一个时间线是否存在并被索引在全局变量 "tl "中，如果是，它就会删除该时间线。

返回。时间轴
