timeline_delete

使用此功能，您可以从游戏中删除任何时间线。如果此时间轴是在GameMaker Studio 2资源浏览器中创建的，请注意，该时间轴已从游戏中完全删除，并且在使用此功能后尝试再次调用索引的时间轴将导致错误。如果时间轴是使用 timeline_add()函数，则该函数应在不再需要时间线以防止任何可能导致游戏变慢（并最终崩溃）的内存泄漏时使用。

句法：

timeline_delete(ind);

争论描述
ind要删除的时间线的索引。

返回值：

N/A

例子：

if timeline_exists(global.tl)
    {
    timeline_delete(global.tl);
    }

上面的代码检查时间线是否存在，并在全局变量“ tl”中建立索引，如果存在，则将其删除。
