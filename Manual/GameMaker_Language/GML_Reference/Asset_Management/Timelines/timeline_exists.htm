timeline_exists

使用此功能，您可以检查并查看时间表是否存在（返回 true）否（返回 false）。当使用 timeline_add()函数，但应注意，如果搜索未初始化的变量（否则将被分配给时间线的索引），则会引发错误。

句法：

timeline_exists(ind);

争论描述
ind要检查的时间线的索引。

返回值：

Boolean

例子：

if timeline_exists(global.tl)
    {
    timeline_delete(global.tl);
    }

上面的代码检查时间线是否存在，并在全局变量“ tl”中建立索引，如果存在，则将其删除。
