Timeline_get_name

这个函数可以用来获取一个时间线的字符串名称。如果该时间线是用 Timeline_add()函数动态创建的，返回的名称将具有 格式 "__newtimelineN" 其中 " N" 是时间线的编号（从 0 开始）。请注意，这只是一个字符串，不能用来直接引用时间线--为此你需要时间线的索引。 为此，你需要时间线的索引。然而，你可以使用这个字符串来获得时间线的索引，使用返回的字符串和函数 asset_get_index().

语法。

Timeline_get_name(ind)。

争论 描述
ind 要检查的时间线名称的索引。

返回。

字符串

例子。

var _str = timeline_get_name(timeline_index);
show_debug_message("Current timeline = " + _str);

上述代码获取当前分配的时间线的名称并将其输出到控制台。

返回。时间轴
