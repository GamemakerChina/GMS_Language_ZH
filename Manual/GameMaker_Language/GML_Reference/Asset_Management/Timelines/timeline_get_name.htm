timeline_get_name

此函数可用于获取时间线的名称作为字符串。如果时间轴是使用 timeline_add()函数，返回的名称格式为“ __newtimeline N ”，其中“ N ”是时间线的编号（从0开始）。请注意，这只是一个字符串，不能用于直接引用时间轴-因为您将需要时间轴索引。但是，您可以使用此字符串通过返回的字符串和函数来获取时间轴索引 asset_get_index()。

句法：

timeline_get_name(ind);

争论描述
ind要检查名称的时间线的索引。

返回值：

String

例子：

var _str = timeline_get_name(timeline_index);
show_debug_message("Current timeline = " + _str);

上面的代码获取当前分配的时间轴的名称，并将其输出到控制台。
