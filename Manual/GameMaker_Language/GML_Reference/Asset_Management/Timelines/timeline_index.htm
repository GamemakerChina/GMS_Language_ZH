timeline_index

该变量保存当前与实例关联的时间线的索引。您可以将其设置为特定时间线以使用该时间线，也可以将其设置为-1以停止使用实例的时间线（如果未为该实例定义时间线，则也返回-1）。请注意，这不启动的时间线-对于使用变量 timeline_running。

句法：

timeline_index;

返回值：

Real (integer - resource index value)

例子：

if timeline_exists(global.tl)
    {
    timeline_index = global.tl;
    }

上面的代码将为运行代码的实例分配一个在变量“ global.tl”中索引的时间线（如果该时间线存在）。
