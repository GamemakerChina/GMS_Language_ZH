timeline_loop

无论时间线是循环的（true）还是不是循环的（false），此变量都将返回。您可以更改此变量以打开或关闭循环，并且它以负的时间线速度工作（如果时间线位置低于0，它将在最后定义的时刻重新开始）。

句法：

timeline_loop;

返回值：

Boolean

例子：

if !timeline_loop
    {
    timeline_loop = true;
    }

上面的代码检查当前分配给实例的时间轴是否循环，如果不循环，则将其设置为true。
