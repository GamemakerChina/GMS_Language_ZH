timeline_max_moment

此函数将返回时间轴执行其最终动作的时刻的值。因此，如果您的时间轴上有相距20个矩的8个不同动作，则此函数将返回该时间轴上最后一个动作的矩值（即160）。此函数是测试时间轴是否通过了时间间隔的好方法跑步时的最后一个活动时刻，因为时间轴位置将继续增加游戏的每个步骤，而不管是否还有其他动作。

句法：

timeline_max_moment(ind);

争论描述
ind要获取最后时刻的时间轴索引。

返回值：

N/A

例子：

if timeline_position > timeline_max_moment(timeline_index)
    {
    timeline_running = false;
    }

上面的代码将对照最大活动时刻检查当前时间轴位置，如果时间轴较大，则将停止时间轴。
