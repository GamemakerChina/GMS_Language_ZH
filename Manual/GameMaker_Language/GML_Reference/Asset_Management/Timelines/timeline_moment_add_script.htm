timeline_moment_add_script

使用此功能，您可以在该时间线内的任何给定“时刻”向时间线动态添加脚本功能，其中“时刻”相当于一个游戏滴答（或步进）。这样，您可以使用 timeline_add()在任何时候都可以发挥作用并添加不同的行为，或者简单地使用新行为修改先前创建的时间轴资源。请注意，该函数在使用此函数时不需要任何其他参数，并且如果使用它来修改资产浏览器中存在的时间轴资产，则使用该时间轴的所有实例都会受到影响，并且更改将持续到结束游戏（调用 game_restart()也不会重置更改）。

句法：

timeline_moment_add_script(ind, step, script);

争论描述
ind要添加时刻的时间线索引。
step要添加的时刻（步骤）。
script要添加到此刻的脚本函数的索引。

返回值：

N/A

例子：

global.tl = timeline_add();
var i = room_speed * 60;
repeat(3)
    {
    timeline_moment_add_script(global.tl, i, choose(Attack_1, Attack_2, Attack_3);
    i += room_speed * 60;
    }

上面的代码将创建一个新的时间轴，并将其索引存储在变量“ global.tl”中。然后它将以时间间隔一分钟随机地向时间线添加三个脚本。
