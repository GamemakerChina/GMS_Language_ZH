timeline_position

该变量保存时间线当前所在的当前位置（时刻）。您可以更改此值以跳过时间线的一部分，或重复部分或从头开始重新开始时间线。

句法：

timeline_position;

返回值：

Real (single precision floating point value)

例子：

if !timeline_running
    {
    timeline_position = 0;
    timeline_running = true;
    }

上面的代码将检查实例是否正在运行时间线，如果不是，则将分配的时间线重置为从第一时间开始，然后再启动它。
