timeline_running

该变量保存分配的时间线的当前状态，并将返回 true如果正在运行，则为false；否则为false。您也可以将此变量设置为 true或者 false随时启动和停止时间线。应该注意的是，停止的时间线不会重置，因此在稍后的时间再次启动它将从停止时的确切时间开始。

句法：

timeline_running;

返回值：

Boolean

例子：

if !timeline_running
    {
    timeline_position = 0;
    timeline_running = true;
    }

上面的代码将检查实例是否正在运行时间线，如果不是，则将分配的时间线重置为从第一时间开始，然后再启动它。
