timeline_size

使用此功能，您可以获取时间轴的活动时刻总数（“活动”时刻是已DnD™的时刻）。当使用 timeline_moment_add_script()和 timeline_moment_clear()功能，因为您可以检查给定的时间轴是否具有正确的活动时刻数或根本没有活动时刻。

句法：

timeline_size(ind);

争论描述
ind时间轴的索引从中获取信息。

返回值：

N/A

例子：

if timeline_size(global.tl) == 0
    {
    timeline_moment_add_script(global.tl, room_speed + irandom(room_speed), spawn_enemy);
    }

上面的代码检查给定的时间轴大小，如果它返回0（即：时间轴没有活动的时刻），它将在全局变量“ tl”中索引的时间轴内的随机时刻添加一个脚本函数。
