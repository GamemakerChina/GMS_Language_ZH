时间线_大小

通过这个函数，你可以得到一个时间线的活动时刻总数（一个 "活动 "时刻是指有代码或DnD™添加到其中的时刻）。这在创建动态时间线时很方便，使用 Timeline_moment_add_script()和 Timeline_moment_clear()函数创建动态时间线时，这很方便，因为你可以检查一个给定的时间线是否有正确数量的活动时刻或根本没有。

语法。

Timeline_size(ind)。

争论 描述
ind 获取信息的时间线的索引。

返回。

不适用

例子。

如果timeline_size(global.tl) == 0
{
timeline_moment_add_script(global.tl, room_speed + irandom(room_speed), spawn_enemy);
}

上面的代码检查给定的时间线大小，如果它返回0（即：时间线没有活动的时刻），它将添加一个脚本函数，在全局变量 "tl "中索引的时间线的随机时刻使用。

返回。时间轴
