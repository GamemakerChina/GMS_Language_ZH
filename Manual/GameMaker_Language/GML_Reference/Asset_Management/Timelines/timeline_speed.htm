timeline_speed

通常，在每个步骤中，时间线中的位置都会增加1，但是您可以通过将此变量设置为其他值来更改此值。您可以使用实数（例如0.5或2.4），如果该值大于1，则在同一时间步中可能会发生几秒钟（它们将按照时间线中定义的相同顺序执行，因此不会跳过任何操作）。

句法：

timeline_speed;

返回值：

Real (single precision floating point value)

例子：

if !timeline_running
    {
    timeline_running = true;
    timeline_speed = 0.5;
    }

上面的代码将检查并查看实例是否正在运行其关联的时间轴，如果没有运行，它将以半速启动时间轴。
