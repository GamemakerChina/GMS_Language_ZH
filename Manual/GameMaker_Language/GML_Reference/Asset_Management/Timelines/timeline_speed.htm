时间线_速度

通常情况下，每走一步，时间线中的位置就会增加1，然而你可以通过将这个变量设置为不同的值来改变这个数量。你可以使用实数（比如0.5，或者2.4），如果数值大于1，那么在同一时间步长内可以有多个时刻发生。 瞬间可以在同一时间步长内发生（它们都将按照为时间线定义的相同顺序执行，所以没有任何动作会被跳过）。

语法。

Timeline_speed;

返回。

实数（单精度浮点值

例子。

if !timeline_running
{
timeline_running = true;
timeline_speed = 0.5;
}

上述代码将检查并查看实例是否正在运行其相关的时间线，如果没有，它将启动时间线以半速运行。
