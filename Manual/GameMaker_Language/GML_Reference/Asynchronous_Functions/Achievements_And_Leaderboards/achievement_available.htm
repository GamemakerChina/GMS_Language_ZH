achievement_available

该函数将返回 true如果用户当前已连接到互联网，并且所选排行榜和成就系统可用，则它将返回 false。

句法：

achievement_available()

返回值：

Boolean

例子：

if achievement_available()
    {
    achievement_post("ach_Level1", 100);
    }

上面的代码将检查排行榜服务是否可用，如果可用，则将其发布。

返回：成就和排行榜
