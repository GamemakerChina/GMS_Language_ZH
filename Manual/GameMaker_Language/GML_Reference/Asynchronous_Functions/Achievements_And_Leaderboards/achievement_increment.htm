achievement_increment

游戏可以具有没有完成值的成就，因此您可以使用此功能将这些成就增加给定的数量。在Android ，您以字符串形式发送成就ID（这是设置成就时分配的唯一成就ID），而在所有其他平台上，您将定义的成就名称作为字符串提供，然后输入要增加的实际成就值。

该函数将触发社交异步事件，其中返回的 async_loadDS映射将具有以下键/值对：

  • “ type ”-这是已触发的事件的类型，将是字符串“achievement_increment_result ”。
  • “ id ”-这是事件的ID，对于此函数，它将返回GML常量 GooglePlayServices_EVENT_ID_INCREMENT_ACHIEVEMENT。
  • “ Achievement_id ”-这是已增加的唯一ID。
  • “ status ”-如果成功增加成就，则为1；如果请求失败，则为0
  • “ complete ”-如果成就已按此增量完成，则为1；如果请求失败，则为0

句法：

achievement_increment(name, value)

争论描述
name成就的名称/ ID。
value以增量形式发送的值。

返回值：

N/A

例子：

achievement_increment(global.Achievement[0], score);

上面的代码使用存储在全局数组中的ID将当前玩家得分添加到成就中。

返回：成就和排行榜
