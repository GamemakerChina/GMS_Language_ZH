achievement_load_friends

此功能将向服务器发送请求，以获取有关所有已登录用户好友的信息，并触发包含以下内容的回调社交异步事件：async_load 映射填充了相关的键/值对。这个的ID键 DS Map用于标识正确的回调（对于任何给定的异步事件，可以有多个触发器函数），并将其与常量配对 achievement_friends_info以及每个朋友的许多其他键/值对。该地图的确切内容如下所示：

注意：此功能仅适用于iOS 。

  • “ id ”-对于此功能，应为 achievement_friends_info
  • “ FriendN ”-朋友的名字，其中“ N”是一个整数值，对应于他们在朋友列表中的位置。
  • “ FriendidN ”-朋友的唯一用户ID，“ N”。

句法：

achievement_load_friends()

返回值：

N/A

扩展示例：

在玩家使用以下方式登录其游戏帐户后，可能会调用以下代码 achievement_login()获取所有用户朋友的列表：

achievement_friends_info();

这将发出有关用户朋友信息的请求，并生成带有特殊信息的异步回调 async_loadDS地图包含以下数据：

var ident = ds_map_find_value(async_load, "id");
if ident == achievement_friends_info
    {
    var numfriends = ds_map_find_value(async_load, "numfriends");
    global.numfriends = numfriends;
    for(var i=0; i < numfriends; i++;)
        {
        global.friendname[i] = ds_map_find_value(async_load, "Friend" + string(i));
        global.friendid[i] = ds_map_find_value(async_load, "Friendid" + string(i));
        achievement_get_pic(global.friendid[i]);
        }
    }

上面的代码检查返回的 DS Map在社交异步事件中，如果其“ id”与要检查的常数匹配，则它将循环遍历映射，将所有不同的值存储在多个数组中，并为映射中的每个条目请求一个图像（请参见 achievement_get_pic()）。
