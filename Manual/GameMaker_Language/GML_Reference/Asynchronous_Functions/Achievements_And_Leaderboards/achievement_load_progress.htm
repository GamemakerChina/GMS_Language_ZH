achievement_load_progress

此功能将向服务器发送请求，以获取有关所有可用成就的信息。它将触发一个回调社交异步事件，其中包含 async_load映射填充了相关的键/值对。这个的ID键 DS Map用于标识正确的回调（对于任何给定的异步事件，可以有多个触发器函数），并将其与常量配对 achievement_achievement_info以及每个玩家的许多其他键/值对。该地图的确切内容如下所示：

注意：此功能仅适用于iOS 。

  • “ id ”-对于此功能，应为 achievement_achievement_info。
  • “ numentries ”-列表中的成就数量。
  • “ AchN ”-成就的名称，其中“ N”是一个整数值，对应于它在条目列表中的位置。
  • “ AchCompN ”-成就“ N”的完成程度，以0到100（字符串）之间的百分比值表示。

句法：

achievement_load_progress()

返回值：

N/A

扩展示例：

iOS设备上时，以下代码将要求实现进度：

if os_type == os_ios
    {
    achievement_load_progress();
    }

这将发出有关成就信息的请求，并生成带有特殊信息的异步回调 async_loadDS地图包含以下数据：

var ident = ds_map_find_value(async_load, "id");
if ident == achievement_achievement_info
    {
    var numentries = ds_map_find_value(async_load, "numentries");
    for(var i = 0; i < numentries; i++;)
        {
        ach_id[i] = ds_map_find_value(async_load, "Ach" + string(i));
        comp[i] = ds_map_find_value(async_load, "AchComp" + string(i));
        }
    }

上面的代码检查返回的 DS Map在“社交异步事件”中，如果其“ id”与要检查的常量匹配，则它将循环遍历地图中每个成就的条目，并存储其名称和完成百分比。
