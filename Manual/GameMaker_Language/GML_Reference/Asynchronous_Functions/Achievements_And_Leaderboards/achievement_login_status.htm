achievement_login_status

该功能目前仅对Android平台上的Google Play有用。它将检查用户是否已经登录Google服务，在这种情况下，它将返回 true，还是不，返回 false。如果返回 true无需致电 achievement_login()功能。

句法：

achievement_login_status();

返回值：

Boolean

例子：

if !achievement_login_status()
    {
    achievement_login();
    }

上面的代码将使您登录到所选的排行榜和成就服务。

返回：成就和排行榜
