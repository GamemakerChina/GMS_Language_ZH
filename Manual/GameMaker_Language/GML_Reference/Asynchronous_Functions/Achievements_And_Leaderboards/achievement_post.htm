成就_帖子

你可以使用这个功能将你的成就发送到所选择的排行榜和成就服务。在安卓系统中，你需要把成就的ID作为一个字符串（这是你设置成就时分配的唯一成就ID）。 而在所有其他平台上，你需要提供一个字符串的成就名称，然后给出你为获得该成就而完成的百分比（0-无，100-完成）。该函数将触发一个社交异步事件，返回的 async_loadDS地图将有以下键/值对。

  • " type"- 这是已经触发的事件的类型，对于这个函数来说，它将是字符串" achievement_post_result"。
  • " id" - 这是事件的ID，对于这个函数，它将返回一个GML常量 GooglePlayServices_EVENT_ID_POST_ACHIEVEMENT。
  • "achievement_id "--这是被更新的成就的唯一ID。
  • "status" - 如果成就被成功更新，这将是1，如果请求失败，则是0。

语法。

achievement_post(成绩，百分比)

争论 描述
成就 成就的名称/ID。
百分比 成绩完成的百分比。

返回。

不适用

例子。

if achievement_available()
{
achievement_post(global.Achievement[0], 100);
}

上述代码检查所选择的排行榜和成就服务是否可用，如果可用，则发布一个成就。
