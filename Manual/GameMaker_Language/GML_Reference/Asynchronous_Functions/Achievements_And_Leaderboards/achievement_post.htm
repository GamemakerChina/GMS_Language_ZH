achievement_post

您可以使用此功能将成就发送到所选的排行榜和成就服务。在Android您需要将成就的ID作为字符串（这是设置成就时分配的唯一成就ID），而在所有其他平台上，您需要将成就名称作为字符串提供，然后输入您已完成取得成就的百分比（0-无，100-已完成）。该函数将触发社交异步事件，其中返回的 async_loadDS映射将具有以下键/值对：

  • “ type ”-这是已触发的事件的类型，将是字符串“achievement_post_result ”。
  • “ id ”-这是事件的ID，对于此函数，它将返回GML常量 GooglePlayServices_EVENT_ID_POST_ACHIEVEMENT。
  • “ Achievement_id ”-这是已更新的唯一ID。
  • “ status ”-如果成功更新了成就，则为1；如果请求失败，则为0

句法：

achievement_post(achievement, percent)

争论描述
achievement成就的名称/ ID。
percent完成的成就百分比。

返回值：

N/A

例子：

if achievement_available()
    {
    achievement_post(global.Achievement[0], 100);
}

上面的代码检查以查看所选的排行榜和成就服务是否可用以及是否发布了成就。
