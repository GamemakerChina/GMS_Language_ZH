achievement_reset

此功能会将所有成就重置为游戏的初始值。此功能是作为调试功能提供的，建议您不要允许最终用户在游戏中进行此操作。

句法：

achievement_reset()

返回值：

N/A

例子：

if debug_mode
    {
    if keyboard_check_pressed(vk_space)
        {
        achievement_reset()
        }
    }

上面的代码检查游戏是否处于调试模式，如果按下“空格”将重置所有成就。

返回：成就和排行榜
