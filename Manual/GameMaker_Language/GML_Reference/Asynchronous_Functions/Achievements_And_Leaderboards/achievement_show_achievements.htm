achievement_show_achievements

此功能将打开所选平台的成就页面。请注意，这是一个异步函数，即：在显示成就页面时，您的游戏将继续在后台运行。因此，在使用此功能时应格外小心，并确保暂停游戏或仅将其显示在不会干扰游戏玩法的游戏区域中。

句法：

achievement_show_achievements()

返回值：

N/A

例子：

if achievement_available()
    {
    global.Pause = true;
    instance_create_layer(0, 0, "Controllers", obj_Pause);
    achievement_show_achievements();
    }

上面的代码检查调用的函数以在屏幕上显示成就之前，检查所选排行榜和成就服务是否可用，是否设置了全局变量并创建了控制器“暂停”实例。

返回：成就和排行榜
