节省云

制作游戏时，通常需要将有关游戏状态的信息存储在某种类型的文件中，但是将这种信息存储在设备上并非总是最佳选择，因为如果玩家先删除游戏然后重新安装，则这是最佳选择。信息可能会丢失。为避免这种情况，您可以使用各种不同的云服务，这些服务通过Internet提供数据存储，以便随时进行检索和修改。您应该注意，此功能仅限于每个游戏一个单一的数据“ blob”，因此，每次将新的保存内容发送到云服务时（无论是字符串还是文件），它将覆盖以前存储的所有数据。

注意：当前只有Android目标使用这些功能，并且您需要Android游戏选项的社交区域中选中启用Google Cloud Saving复选框。这将提示您下载Google Services扩展程序，这是Android上的云保存功能正常运行所必需的。有关Cloud Saving的更多信息，请参见以下帮助台文章： Android Cloud Saving

GameMaker Studio 2通过一些与异步云事件配合使用的简单功能来支持此功能。这些功能在以下页面中说明：

  • cloud_synchronise
  • cloud_string_save
  • cloud_file_save
