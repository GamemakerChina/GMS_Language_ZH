cloud_string_save

此函数会将字符串提交给所选的云服务以进行存储。该函数将返回唯一的id值，然后该ID值应在适当的异步事件中使用，以标识作为“回叫”从云服务返回的DS映射。该字符串应包含您需要为游戏保存的所有信息，因为您只能将一个“数据blob”存储到云中，并且再次运行此函数将覆盖以前存储的所有值（使用 cloud_file_save()功能）。描述应该是描述保存的简短信息，例如：“ Level2，Stage2”。

有关返回的异步数据的更多信息，请参见函数 cloud_synchronise()。

句法：

cloud_string_save(string, description);

争论描述
string要上传的数据字符串。
description所存储数据的简要说明。

返回值：

Real

例子：

var t_str = "";
for (var i = 0; i < 10; i++;)
    {
    t_str += string(global.Highscore[i]) + "|"
    }
save_check = cloud_string_save(t_str, "Current Highscores");
var file = file_text_open_write("Highscores.txt");
file_text_write_string(file, t_str);
file_text_close(file);

上面的代码根据存储在全局数组“ Highscores”中的值创建一个字符串，然后将该字符串写入云服务进行存储，以及将其写入文件进行本地存储。

返回：云存储
