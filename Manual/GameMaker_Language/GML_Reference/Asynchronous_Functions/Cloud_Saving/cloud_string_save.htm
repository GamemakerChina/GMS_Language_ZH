云端字符串保存

该函数将提交一个字符串到所选择的云服务进行存储。该函数将返回一个唯一的ID值，然后应在适当的异步事件中使用，以识别作为 "回调 "返回的DS地图。 从云服务返回的DS地图。这个字符串应该包含你需要为你的游戏保存的所有信息，因为你只能向云端存储一个单一的 "数据块"，再次运行这个函数将覆盖任何先前存储的 值（正如使用 cloud_file_save()函数）。)描述应该是描述保存的信息的短字符串，例如："Level2, Stage2"。

关于返回的异步数据的进一步信息，请参见函数 cloud_synchronise().

语法。

cloud_string_save(string, description)。

争论 描述
绳子 要上传的数据字符串。
描述 对被存储的数据的简要描述。

返回。

真实的

例子。

var t_str = "";
for (var i = 0; i < 10; i++;)
{
t_str += string(global.Highscore[i]) + "|"
}
save_check = cloud_string_save(t_str, "Current Highscores");
var file = file_text_open_write("Highscores.txt");
file_text_write_string(file, t_str);
file_text_close(file) 。

上述代码从存储在全局数组 "Highscores "中的数值创建了一个字符串，然后将这个字符串写入云服务进行存储，同时也将其写入文件进行本地存储。
