对话

当显示对话框窗口时停止游戏时，大多数现代计算机和移动设备都不喜欢它。他们倾向于将其解释为跑步者已经停止，并且可能会认为这是错误并过早关闭了该应用。为防止这种情况， GameMaker Studio 2具有异步对话功能，使您可以请求某些形式的用户反馈或输入，但不会在任何时候阻止游戏运行者，这意味着-被调用时-游戏将在后台正常运行，而用户响应显示的对话框窗口。用户响应后，将调用对话框异步事件，以便您处理结果。

注意：这意味着如果在显示此对话框时玩家可能会输掉游戏，被杀等……那么您应该采取步骤来创建一个暂停系统，否则将是一个令人不快的用户经验。

以下是所有可用的异步对话框功能的列表：

  • get_login_async
  • get_string_async
  • get_integer_async
  • show_message_async
  • show_question_async
  • shop_leave_rating
