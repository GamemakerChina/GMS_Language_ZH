get_integer_async

此功能将打开一个窗口并显示消息以及供用户输入值的空间（该值将包含提供的默认值开头）。这是一个异步函数，因此GameMaker Studio 2不阻止它正在对在等待答案的运行，而是在设备上运行的事件作为正常保持。一旦用户输入了字符串并按下“ Okay”按钮，就会触发一个异步Dialog事件，该事件仅在该事件的持续时间内将DS映射存储在变量中 async_load。

该地图将包含三个键，“id ”，“status “， 和 ”value “。”id “是函数调用时返回的值，“状态”可以是 true按下“确定”按钮，或者 false如果消息已取消（在适用的情况下，因为并非所有目标平台都允许取消消息）。最后 ”value “将返回用户作为字符串输入的整数（如果未提供输入，将返回一个空字符串“”）。

句法：

get_integer_async(string, default);

争论描述
String对话框中显示的消息。
default默认值。

返回值：

Real

扩展示例：

显示消息的对象的左键按下事件（例如）将具有以下代码：

msg = get_integer_async("How old are you?", 0);

上面将显示一条消息，要求用户输入字符串并按“确定”。函数id存储在变量“ msg”中，并将在异步Dialogs事件中使用，如下所示：

var i_d = ds_map_find_value(async_load, "id");
if i_d == msg
    {
    if ds_map_find_value(async_load, "status")
        {
        global.Age = ds_map_find_value(async_load, "value");
        }
    }

上面的代码检查“id “返回的键 DS Map对照存储在变量“ msg”中的值。如果它们相同，则检查是否按下了“确定”（而不是关闭/取消了窗口）并返回 true然后，它根据返回的整数设置全局变量。
