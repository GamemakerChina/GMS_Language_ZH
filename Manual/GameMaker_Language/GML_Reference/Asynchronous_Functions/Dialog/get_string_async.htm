get_string_async

这个函数打开一个窗口，显示信息以及一个空间，供用户输入一个字符串（其中将包含提供的默认字符串开始）。这是一个异步函数，因此GameMaker Studio 2在等待答案时不会阻止正在运行的设备。 它正在运行的设备在等待答案，而是继续正常地运行事件。一旦用户输入了他们的字符串并按下了 "Okay "按钮，就会 触发一个异步的Dialog事件 。 触发，仅在该事件的持续时间内，将有一个DS映射存储在变量 异步_加载.

这个映射将包含三个键，"id"、"status "和"result"。"id "是函数被调用时返回的值，"status "将 是 "好的 "按钮被按下的真值，或者是 "假的"，如果消息被取消（在适用的情况下，不是所有的目标平台都允许消息被取消）。最后，"result "将返回用户输入的字符串 如果没有提供，则返回一个空字符串""。

注意。这个功能 只用于调试和测试，不应该在已发布的游戏中使用。为此，你应该创建一个自定义的用户界面来接收玩家的输入，这样你就可以完全控制对话框的外观和行为。

语法。

get_string_async(string, default);
