淘宝网上的评价

这个功能打开一个依赖操作系统的对话框，你可以要求用户对某一特定页面发表评价或评论。你可以定义出现在对话中的文本，以及你希望出现在两个按钮上的文本和必须张贴评论的URL 在那里发布评论。

注意：该功能仅适用于安卓和iOS目标。

重要的是!在 iOS，当使用TestFlight时，在开发过程中会显示提示，但你将无法真正做任何事情。一旦你的应用进入App Store，这个按钮就会激活，用户可以选择 写评论或留下评价。注意，如果你通过TestFlight发布游戏，这个窗口甚至可能不会显示。

语法。

shop_leave_rating(text, yes_string, no_string, url)

争论 描述
文本 出现在对话框中的文本。
yes_string 出现在 "是 "按钮上的文本。
拒绝 出现在 "不 "按钮上的文本。
网址 必须发送评论的完整URL。

返回。

不适用

例子。

如果定时器<=0
{
shop_leave_rating("感谢你玩游戏!为什么不留下评论呢?", "好的!", "今天不行!", "http://MacSweeney/comments");
timer = 10000000;
}
else
{
timer -= 1;
}

上面的代码将要求用户在变量 "计时器 "倒计时到0的时候留下评论。

返回。对话
