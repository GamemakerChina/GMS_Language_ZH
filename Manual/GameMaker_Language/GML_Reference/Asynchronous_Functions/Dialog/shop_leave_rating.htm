shop_leave_rating

此功能将打开一个与OS相关的对话框，您可以在其中要求用户在特定页面上发布评级或评论。您可以定义将在对话框中显示的文本，以及希望在两个按钮上显示的文本以及必须在其中张贴评论的URL。

注意：此功能仅适用于Android和iOS目标。

重要的！在iOS ，使用TestFlight时，在开发中会显示提示，但您实际上将无法执行任何操作。您的应用程序进入App Store后，此按钮将变为活动状态，并为用户提供撰写评论或留下评分的选项。请注意，如果您通过TestFlight分发游戏，则该窗口甚至可能不会显示。

句法：

shop_leave_rating(text, yes_string, no_string, url)

争论描述
text对话框上显示的文本。
yes_string显示在“是”按钮上的文本。
no_string显示在“否”按钮上的文本。
url必须发送评论的完整URL。

返回值：

N/A

例子：

if timer <= 0
    {
    shop_leave_rating("Thanks for playing the game! Why not leave a comment?", "Okay!", "Not Today!", "http://MacSweeney/comments");
    timer = 10000000;
    }
else
    {
    timer -= 1;
    }

上面的代码将要求用户在变量“ timer”倒计数到0时发表评论。

返回：对话框
