show_message_async

此功能将打开一个窗口，并向用户显示您在功能中定义的消息。这是一个异步函数，因此GameMaker Studio 2不阻止它正在对在等待答案的运行，而是在设备上运行的事件作为正常保持。一旦用户按下“ Okay”按钮，就会触发一个异步Dialog事件，该事件仅在该事件的持续时间内将ds_map存储在变量中 async_load。

该映射将包含两个键“ id”和“ status”，其中“ id”是函数在调用时返回的值，“ status”是 true按下“确定”按钮，或者 false如果消息已取消（在可用的情况下，并非所有目标平台都允许取消对话）。

句法：

show_message_async(string);

争论描述
String显示给用户的消息。

返回值：

Real

扩展示例：

显示消息的对象的左键按下事件（例如）将具有以下代码：

msg = show_message_async("You will now be taken to the store");

上面将显示带有给定字符串的消息。消息ID存储在变量“ msg”中，并将在异步Dialogs事件中使用，如下所示：

var i_d, stat;
i_d = ds_map_find_value(async_load, "id");
if i_d == msg
    {
    if ds_map_find_value(async_load, "status")
        {
        url_open("https://play.google.com/store");
        }
    }

上面的代码检查返回的“ id”键 DS Map对照存储在变量“ msg”中的值。如果它们相同，则检查是否按下了“确定”按钮（而不是关闭/取消了窗口），并且是否返回 true它打开一个URL。
