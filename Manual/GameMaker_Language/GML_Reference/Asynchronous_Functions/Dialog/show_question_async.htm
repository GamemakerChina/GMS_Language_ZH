show_question_async

该功能将打开一个窗口，并向用户显示您在功能中定义的问题。这是一个异步函数，因此GameMaker Studio 2不阻止它正在对在等待答案的运行，而是在设备上运行的事件作为正常保持。该函数具有两个按钮，分别显示“是”和“否”，并且一旦用户按下一个按钮，就会触发一个异步对话框事件，该事件仅在该事件的持续时间内将DS映射存储在变量中 async_load。

该映射将包含两个键“ id ”和“ status ”。 “ id”是函数在调用时返回的值，而“ status”将是 true或者 false分别为“是”和“否”。

句法：

show_question_async(string);

争论描述
String向用户提出的问题。

返回值：

Real

扩展示例：

显示消息的对象的左键按下事件（例如）将具有以下代码：

msg = show_question_async("Do you want to buy some armour for " + string(armour[0, 5]) + "coins?");

上面将显示给定字符串的问题，要求用户按“是”或“否”。函数ID存储在变量“ msg”中，并将在“异步对话框”事件中使用，如下所示：

var i_d, stat;
i_d = ds_map_find_value(async_load, "id");
if i_d == msg
    {
    if ds_map_find_value(async_load, "status")
        {
        coins -= armour[0,5];
        global.protection += armour[0,0];
        }
    }

上面的代码检查返回的“ id”键 DS Map对照存储在变量“ msg”中的值。如果它们相同，则检查两个按钮之一是否被按下，是否返回 true然后它将从变量中扣除一个值，并将一个值也添加到全局变量中。
