Facebook

Facebook是互联网上最重要的社交媒体网站之一，许多游戏和开发人员目前都有自己的页面来发布游戏和媒体。 GameMaker Studio 2具有扩展名，可让您与Facebook连接并执行其API允许的某些操作和任务。建议您在开始使用扩展程序之前先访问Facebook开发人员页面，以深入了解Facebook Graph的工作方式，而可用的调用和权限对于您充分利用这些功能至关重要。

您可以从以下链接中找到如何为所有可用平台设置和使用Facebook扩展程序：

  • Android ， iOS和HTML5：集成Facebook

以下是仅对旧版Facebook API有效的过时功能。因此，您将无法再在项目中使用这些功能，并且如果您导入了包含这些功能的旧项目，它们将在此处列出。对于当前的Facebook功能，您需要在目标平台的相应游戏选项中激活Facebook API ，这将提示您下载并安装扩展程序。该扩展名带有一个 *.pdf格式为“迷你手册”，解释如何使用它以及显示所有功能定义和示例。所有提到旧的 facebook_*()功能将需要替换（和更新）以使用扩展名 fb_*()等价物。

下面列出的功能用于将您的游戏与Facebook集成：

  • facebook_init
  • facebook_login
  • facebook_request_publish_permissions
  • facebook_request_read_permissions
  • facebook_check_permission
  • facebook_status
  • facebook_accesstoken
  • facebook_user_id
  • facebook_graph_request
  • facebook_dialog
  • facebook_send_invite
  • facebook_post_message
  • facebook_logout
  • facebook_launch_offerwall
