HTTP

本节列出了GameMaker Studio 2中的所有不同的异步HTTP函数。这些函数将在所有拥有它的实例中产生一个异步HTTP事件。

  • http_request
  • http_get
  • http_get_file
  • http_post_string

请注意，由于跨域安全问题，上述 http_* 功能可能无法发挥预期作用。这意味着对你的服务器的请求或从跨域加载资源的尝试会被阻止，并可能出现空白结果或404错误。 被阻止，并可能出现返回空白结果或404错误。绕过这个问题的方法之一是有一些服务器端的PHP允许某些域访问你的服务器（这也是一种保护你的资源的方法，并阻止不在PHP允许范围内的服务器 这也是一种保护你的资源和阻止不在PHP允许列表中的服务器的方法）。)

下面是一个例子，你可以为此使用的代码。

$http_origin = $_SERVER['HTTP_ORIGIN'];
if ($http_origin == "http://127.0.0.1:51268" )
{
header("Access-Control-Allow-Origin: *");
}

对于确定或设置其跨源类型很重要的图像加载请求，存在以下功能。

  • http_get_request_crossorigin
  • http_set_request_crossorigin

在大多数情况下，应该不需要使用这些功能，但如果游戏存储在一个安全的服务器上--某些资产可能需要基本认证才能访问，并且在加载时产生安全错误--将图像请求的跨源类型设置为 "使用认证 "可能是必要的（与默认的 "匿名 "设置相反）。 类型的图像请求设置为 "使用认证 "可能是必要的（而不是默认的 "匿名 "设置）。
