HTTP

GameMaker Studio 2可用的所有不同的异步HTTP功能。这些函数将在所有具有该事件的实例中生成一个异步HTTP事件：

  • http_request
  • http_get
  • http_get_file
  • http_post_string

请注意以上 http_*由于跨域安全问题，这些功能可能无法按预期运行。这意味着对您的服务器的请求或尝试从多个域加载资源的请求均被阻止，并且可能会返回空白结果或404错误。解决该问题的方法之一是拥有一些服务器端PHP，该PHP允许某些域访问您的服务器（这也是一种保护资源并阻止PHP允许列表中未包含的服务器的方法）。

以下是可用于此目的的代码示例：

$http_origin = $_SERVER['HTTP_ORIGIN'];
if ($http_origin == "http://127.0.0.1:51268")
   {
   header("Access-Control-Allow-Origin: *");
   }

对于确定或设置其跨域类型很重要的图像加载请求，存在以下功能：

  • http_get_request_crossorigin
  • http_set_request_crossorigin

在大多数情况下，不需要使用这些功能，但是如果游戏存储在受保护的服务器上-某些资产可能需要访问基本身份验证，并且在加载时会产生安全错误-设置图像请求的跨域类型可能需要“使用凭据”（与默认的“匿名”设置相反）。
