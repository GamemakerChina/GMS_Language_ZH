buffer_base64_encode

此函数会将给定缓冲区中的数据转换为以base64格式编码的字符串。这是一种常用的编码方案，通常用于需要以文本形式在互联网上存储或传输的任何媒体，并使输出无法被人眼识别。要使用此功能，您需要指定一个已经创建的缓冲区，偏移值（缓冲区中要开始编码的点）以及要编码的缓冲区内存的大小（以字节为单位）。

句法：

buffer_base64_encode(buffer, offset, size);

争论描述
buffer要使用的缓冲区的索引。
offset数据偏移值。
size缓冲区的大小。

返回值：

String

例子：

var b_str = buffer_base64_encode(buff, 0, buffer_get_size(buff))

上面的代码将对存储在变量“ buff”索引的缓冲区中的完整数据进行编码，并将返回的字符串存储在局部变量“ b_str”中。

返回：缓冲区
