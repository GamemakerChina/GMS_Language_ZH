缓冲_压缩

通过这个函数，你可以用zlib压缩法来压缩部分（或全部）缓冲区。你提供要压缩的缓冲区的ID（如由 buffer_create()), 要使用的缓冲区内的偏移量（字节），以及要压缩的缓冲区数据的大小（也是字节）。该函数将为压缩的缓冲区返回一个新的缓冲区ID值，如果由于任何原因失败，则返回一个小于0的值。这个函数 不会改变原来的缓冲区。

语法。

buffer_compress(buffer, offset, size)。

争论 描述
缓冲区 要压缩的缓冲区的索引。
补偿 要压缩的缓冲区内的偏移量（字节数）。
尺寸 要压缩的缓冲区的大小（以字节为单位）。

返回。

Int (缓冲区ID值)

例子。

var srcBuff = buffer_create(1024, buffer_grow, 1);
buffer_write(srcBuff, global.DataString);
var cmpBuff = buffer_compress(srcBuff, 0, buffer_tell(srcBuff));
buffer_save(cmpBuff, "Player_Save.sav");
buffer_delete(srcBuff);
buffer_delete(cmpBuff)。

上面的代码将创建一个缓冲区，然后用一个字符串的数据填充它。然后，这个缓冲区被压缩并保存，源缓冲区和压缩缓冲区都被删除。
