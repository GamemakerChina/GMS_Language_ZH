buffer_compress

使用此功能，您可以使用zlib compression压缩缓冲区的一部分（或全部）。您提供要压缩的缓冲区的ID（由 buffer_create()），要使用的缓冲区内的偏移量（以字节为单位）以及要压缩的缓冲区数据的大小（也以字节为单位）。该函数将为压缩缓冲区返回一个新的缓冲区ID值，如果由于任何原因失败，则返回小于0的值。此功能不会更改原始缓冲区。

句法：

buffer_compress(buffer, offset, size);

争论描述
buffer要压缩的缓冲区的索引。
offset要压缩的缓冲区内的偏移量（以字节为单位）。
size要压缩的缓冲区的大小（以字节为单位）。

返回值：

Int (Buffer ID value)

例子：

var srcBuff = buffer_create(1024, buffer_grow, 1);
buffer_write(srcBuff, global.DataString);
var cmpBuff = buffer_compress(srcBuff, 0, buffer_tell(srcBuff));
buffer_save(cmpBuff, "Player_Save.sav");
buffer_delete(srcBuff);
buffer_delete(cmpBuff);

上面的代码将创建一个缓冲区，然后使用字符串中的数据填充它。然后将该缓冲区压缩并保存，同时删除源缓冲区和压缩缓冲区
