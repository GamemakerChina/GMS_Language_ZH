buffer_copy

此功能可用于将存储在一个缓冲区中的数据段（或全部）复制到另一个缓冲区。使用两个缓冲区并从一个缓冲区复制到另一个缓冲区时，两个缓冲区必须事先使用 buffer_create()函数（例如），您可以为要从源缓冲区复制的数据的起点（相对于缓冲区的起点）指定一个数据偏移量（以字节为单位），以及另一个数据偏移量以定义要定位的位置将数据复制到目标缓冲区中。

注意：您不能复制到相同的缓冲区。

句法：

buffer_copy(src_buffer, src_offset, size, dest_buffer, dest_offset);

争论描述
src_buffer要从中复制的缓冲区的索引。
src_offset开始复制的数据偏移量（以字节为单位）。
size要复制的数据大小（以字节为单位）。
dest_buffer要复制到的缓冲区的索引。
dest_offset将数据复制到的偏移位置（以字节为单位）。

返回值：

N/A

例子：

buffer_copy(buff1, 0, 2048, buff2, 2048);

上面的代码将复制存储在变量“ buff1”中索引的缓冲区中的数据，然后将其粘贴到变量“ buff2”中索引的缓冲区中，但偏移量为2048个字节。

返回：缓冲区
