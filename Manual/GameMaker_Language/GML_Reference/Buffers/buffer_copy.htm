缓冲区拷贝

这个函数可以用来将一个缓冲区中存储的数据的一部分（或全部）复制到另一个缓冲区。当使用两个缓冲区并从一个缓冲区复制到另一个缓冲区时，两个缓冲区必须事先用 buffer_create() 函数（例如）创建的，你可以指定一个数据偏移量（以字节为单位），作为要从源缓冲区复制的数据相对于缓冲区开始的起点，以及另一个数据偏移量，以定义将数据复制到 在目标缓冲区中的位置。

注意：你不能复制到同一个缓冲区。

语法。

buffer_copy(src_buffer, src_offset, size, dest_buffer, dest_offset)。

争论 描述
src_buffer 要复制的缓冲区的索引。
src_offset 要开始复制的数据偏移量（以字节为单位）。
尺寸 要复制的数据的大小（字节）。
缓冲区（Dest_buffer） 要复制到的缓冲区的索引。
目的地_offset 要复制数据的偏移位置（以字节为单位）。

返回。

不适用

例子。

buffer_copy(buff1, 0, 2048, buff2, 2048)。

上述代码将复制存储在变量 "buff1 "索引的缓冲区中的数据，然后将其粘贴到变量 "buff2 "索引的缓冲区中，但偏移2048字节。
