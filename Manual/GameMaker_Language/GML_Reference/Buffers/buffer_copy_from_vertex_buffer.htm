buffer_copy_from_vertex_buffer

此功能可用于将存储在一个顶点缓冲区中的一些（或全部）顶点数据复制到先前创建的常规缓冲区中。使用此功能从顶点缓冲区复制到常规缓冲区时，必须事先创建两个缓冲区（使用 vertex_create_buffer()和 buffer_create()功能）。您可以指定要复制到缓冲区中的顶点数据的范围，其中起始顶点可以在0到顶点数量-1之间的任何位置，并且可以指定从该点开始复制的顶点数量。您可以使用该功能 vertex_get_number()在顶点缓冲区中获取存储的顶点总数。最后，您提供缓冲区索引以将顶点数据复制到其中，并提供数据偏移量以定义将顶点数据复制到目标缓冲区中的位置。

句法：

buffer_copy_from_vertex_buffer(vertex_buffer, start_vertex, num_vertices, dest_buffer, dest_offset);

争论描述
vertex_buffer要从中复制的顶点缓冲区的索引。
start_vertex起始顶点。
num_vertices要使用的顶点总数。
dest_buffer要复制到的缓冲区的索引。
dest_offset将数据复制到的偏移位置（以字节为单位）。

返回值：

N/A

例子：

var v_num = vertex_get_number(model_buff); buffer_copy_from_vertex_buffer(model_buffer, 0, v_num - 1, player_buffer, 0);

上面的代码将复制存储在变量“ model_buffer”中索引的顶点缓冲区中的顶点数据，然后将其粘贴到变量“ player_buffer”中索引的缓冲区中。

返回：缓冲区
