buffer_crc32

这个函数将从一个缓冲区获取输入数据并返回一个crc32校验和散列。你指定要使用的缓冲区ID，然后指定一个开始的偏移值（字节），然后指定要散列的区域的大小（同样以字节为单位），然后 该函数将为该区域返回一个32位的整数值。

语法。

buffer_crc32(buffer, offset, size)。

争论 描述
缓冲区 要使用的缓冲区的索引。
补偿 数据偏移值。
尺寸 缓冲区的大小。

返回。

国际 32

例子。

check_val = buffer_crc32(buff, 0, buffer_get_size(buff))。

上述代码将为存储在由变量 "buff "索引的缓冲区中的全部数据创建一个crc32校验和哈希值，并将返回的整数哈希值存储在变量 "check_val "中。
