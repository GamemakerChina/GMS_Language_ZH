buffer_crc32

此函数将从缓冲区获取输入数据，并返回crc32校验和哈希。您指定要使用的缓冲区的缓冲区ID，然后是开始位置的偏移值（以字节为单位），然后是要哈希的区域的大小（再次以字节为单位），该函数将返回32位整数值对于那个地区。

句法：

buffer_crc32(buffer, offset, size);

争论描述
buffer要使用的缓冲区的索引。
offset数据偏移值。
size缓冲区的大小。

返回值：

Int 32

例子：

check_val = buffer_crc32(buff, 0, buffer_get_size(buff));

上面的代码将为存储在变量“ buff”索引的缓冲区中的完整数据创建一个crc32校验和哈希值，并将返回的整数哈希值存储在变量“ check_val”中。

返回：缓冲区
