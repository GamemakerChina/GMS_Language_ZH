缓冲区解压

通过这个函数，你可以用zlib压缩法解压一个先前压缩的缓冲区。你提供要解压的缓冲区ID，该函数将返回一个新的缓冲区ID，其中包含未压缩的 数据。如果解压失败（例如，你提供的是一个没有被压缩的缓冲区），那么这个函数将返回一个小于0的值。

语法。

buffer_decompress(buffer)。

争论 描述
缓冲区 要解压的缓冲区的索引。

返回。

缓冲区ID

例子。

var cmpBuff = buffer_load("Player_Save.sav");
var srcBuff = buffer_decompress(cmpBuff);
global.DataString = buffer_read(srcBuff, buffer_string) 。

上面的代码将首先加载一个保存的缓冲区，然后解压，最后从解压的缓冲区中读取字符串数据到一个全局变量中。
