buffer_decompress

使用此功能，您可以使用zlib compression解压缩先前压缩的缓冲区。您提供要解压缩的缓冲区ID，该函数将返回一个包含未压缩数据的新缓冲区ID。如果解压缩失败（例如，您提供的缓冲区尚未压缩），则该函数将返回小于0的值。

句法：

buffer_decompress(buffer);

争论描述
buffer要解压缩的缓冲区的索引。

返回值：

Buffer ID

例子：

var cmpBuff = buffer_load("Player_Save.sav");
var srcBuff = buffer_decompress(cmpBuff);
global.DataString = buffer_read(srcBuff, buffer_string);

上面的代码将首先加载已保存的缓冲区，然后对其进行解压缩，最后将字符串数据从解压缩的缓冲区读取到全局变量中。

返回：缓冲区
