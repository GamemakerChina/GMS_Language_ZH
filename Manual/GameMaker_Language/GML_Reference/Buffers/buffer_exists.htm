buffer_exists

此函数可用于检查变量以查看其是否具有有效的缓冲区ID值。如果这样做，函数将返回 true否则它将返回 false。

句法：

buffer_exists(buffer)

争论描述
buffer要检查的缓冲区的索引。

返回值：

Boolean

例子：

if buffer_exists(buff)
    {
    buffer_delete(buff);
    }

上面的代码检查变量“buff ”保留一个缓冲区ID，如果有，则删除该缓冲区。

返回：缓冲区
