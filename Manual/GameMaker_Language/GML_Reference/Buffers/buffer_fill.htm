缓冲区填充

这个函数可以用来用一个给定的数据类型和值来填充一个先前创建的缓冲区。你填充的数据必须与这个函数的 "类型 "参数一致，也就是说，你不能试图填充一个 字符串而使用无符号16位整数类型，例如。这些类型常量与那些被 buffer_read()和 buffer_write()函数所使用的。"size" 是你希望填充的缓冲区的大小（以字节为单位），而偏移量是从缓冲区的起始点开始填充的偏移值（也以字节为单位）。

语法。

buffer_fill(buffer, offset, type, value, size)。

争论 描述
缓冲区 要填充的缓冲区的索引。
补偿 数据偏移值（以字节为单位）。
类型 要写入缓冲区的数据类型（见这里的常数列表）。
价值 要写的数据。
尺寸 你希望填充的缓冲区的大小（以字节为单位）。

返回。

不适用

例子。

map_buffer = buffer_create(16384, buffer_fixed, 0);
buffer_fill(map_buffer, 0, buffer_u16, 0, 16384) 。

上面的代码找到了存储在变量 "buff "中的id的缓冲区的起点，并向其写入了一系列有符号的16位整数值。
