buffer_fill

此函数可用于使用给定的数据类型和值填充先前创建的缓冲区。用来填充缓冲区的数据必须与该函数的“ type”参数一致，这意味着，例如，您不能尝试填充字符串并使用无符号16位整数类型。类型常量与 buffer_read()和 buffer_write()功能。 “大小”是您希望填充的缓冲区的大小（以字节为单位），而偏移量是从缓冲区的开头开始填充的偏移量值（也以字节为单位）。

句法：

buffer_fill(buffer, offset, type, value, size);

争论描述
buffer要填充的缓冲区的索引。
offset数据偏移值（以字节为单位）。
type要写入缓冲区的数据类型（请参见此处的常量列表）。
value要写入的数据。
size您希望填充的缓冲区大小（以字节为单位）。

返回值：

N/A

例子：

map_buffer = buffer_create(16384, buffer_fixed, 0);
buffer_fill(map_buffer, 0, buffer_u16, 0, 16384);

上面的代码使用存储在变量“ buff”中的id查找缓冲区的开始，然后向其写入一系列带符号的16位整数值。

返回：缓冲区
