buffer_get_address

使用此功能，您可以获得指向原始对齐缓冲区地址的指针。这主要用于扩展名，因为您可以将此值传递给扩展名，从而允许它们访问缓冲区数据。请注意，从扩展名访问时，您不能对缓冲区做一些事情：

  • 您无法释放内存。
  • 您无法调整内存大小。
  • 您不能写超出缓冲区地址加缓冲区大小的字样（可以使用 buffer_get_size()函数），否则您将获得超出范围的错误。

句法：

buffer_get_address(buffer);

争论描述
buffer要使用的缓冲区的索引。

返回值：

pointer

例子：

var b_address = buffer_get_address(buff_model);
var b_end_address = b_address + buffer_get_size(buff_model);

上面的代码将获取在变量“ buff_model”中索引的缓冲区的内存地址，然后使用以下命令获取缓冲区末尾的内存地址。buffer_get_size()函数，将两个值都存储在局部变量中以备将来使用。

返回：缓冲区
