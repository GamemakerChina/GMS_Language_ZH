buffer_get_size

用这个函数你可以得到给定的缓冲区的大小，单位是字节。

语法。

buffer_get_size(index)。

争论 描述
指数 缓冲区的索引，以获得其大小。

返回。

真实的

例子。

var _size = buffer_get_size(player_data);
var _temp = buffer_create(_size, buffer_fixed, 0);

上述代码将创建一个新的缓冲区，并将其索引存储在本地变量"_temp "中，这个新缓冲区的大小与变量 "player_data "中索引的缓冲区相同。
