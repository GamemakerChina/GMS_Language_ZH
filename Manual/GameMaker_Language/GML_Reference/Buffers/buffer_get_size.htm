buffer_get_size

使用此功能，您可以获取给定缓冲区的大小（以字节为单位）。

句法：

buffer_get_size(index);

争论描述
index要获取的缓冲区的索引大小。

返回值：

Real

例子：

var _size = buffer_get_size(player_data);
var _temp = buffer_create(_size, buffer_fixed, 0);

上面的代码将创建一个新缓冲区，并将其索引存储在局部变量“ _temp”中，此新缓冲区的大小与在变量“ player_data”中索引的缓冲区的大小相同。

返回：缓冲区
