buffer_get_type

通过这个函数，你可以检索到被缓冲区ID值引用的缓冲区类型。返回值将是以下常数之一。

恒定 描述
缓冲区_固定 一个固定大小的缓冲区。
缓冲区_增长 随着数据的增加，缓冲区将动态地 "增长"。
缓冲区_包装 一个缓冲区，数据将被 "包裹 "起来。当被添加的数据达到缓冲区大小的极限时，被覆盖的数据将被放回缓冲区的起点，然后从该点继续写。
缓冲区_快速 特殊的 "剥离 "缓冲区，读/写速度极快。只能与buffer_u8 数据类型一起使用，并且必须是1字节对齐的。
buffer_vbuffer 一个顶点缓冲区

语法。

buffer_get_type(buffer)。

争论 描述
缓冲区 要检查的缓冲区的索引。

返回。

恒定

例子。

type = buffer_get_type(buff)。

上面的代码将从变量 "buff "中索引的值中获得缓冲区的类型，并将其存储在一个变量中。
