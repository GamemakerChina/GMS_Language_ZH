buffer_load_ext

此功能将加载以前使用 buffer_save()功能进入已经创建的缓冲区。您提供要加载的先前创建的缓冲区的id，然后提供要加载的已保存的缓冲区文件，最后是要加载数据的缓冲区起始位置的偏移量（以字节为单位）。

句法：

buffer_load_ext(buffer, filename, offset);

争论描述
buffer要加载到的缓冲区的索引。
filename要加载的文件的名称。
offset要加载到的缓冲区中的偏移量（以字节为单位）。

返回值：

N/A

例子：

var pos = buffer_seek(player_buffer, buffer_seek_end, 0);
buffer_load(player_buffer, "Data_Save.sav", pos);

上面的代码将首先获取在变量“ player_buffer”中索引的缓冲区末端的位置，然后将数据从给定的位置加载到该位置（请注意，此示例仅适用于“ grow”或“ wrap”缓冲区类型）。

返回：缓冲区
