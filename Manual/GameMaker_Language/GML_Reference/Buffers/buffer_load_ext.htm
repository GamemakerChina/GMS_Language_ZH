buffer_load_ext

这个函数将把之前用函数保存的缓冲区数据加载到已经创建的 buffer_save()函数加载到一个已经创建的缓冲区。你要输入先前创建的缓冲区的ID，然后输入保存的 要加载的缓冲区文件，最后是你希望加载数据的缓冲区起点的偏移量（字节数）。

语法。

buffer_load_ext(buffer, filename, offset)。

争论 描述
缓冲区 要载入的缓冲区的索引。
文件名 要加载的文件的名称。
补偿 缓冲区内要加载的偏移量（以字节为单位）。

返回。

不适用

例子。

var pos = buffer_seek(player_buffer, buffer_seek_end, 0);
buffer_load(player_buffer, "Data_Save.sav", pos) 。

上面的代码将首先获得在变量 "player_buffer "中索引的缓冲区末端的位置，然后从给定的位置加载数据（注意，这个例子只适用于 "grow "或 "wrap" 缓冲区类型）。)
