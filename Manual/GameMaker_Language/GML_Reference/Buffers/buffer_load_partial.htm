buffer_load_partial

此功能将加载以前使用 buffer_save()功能进入已经创建的缓冲区。您提供要加载到的先前创建的缓冲区的id，然后提供要加载的已保存的缓冲区文件，然后提供要从中加载数据的缓冲区的起始偏移量（以字节为单位）。以下参数用于设置缓冲区数据距要加载的初始偏移点的长度（以字节为单位）以及将数据加载到缓冲区中的偏移点的长度（再次以字节为单位）。

句法：

buffer_load_partial(buffer, filename, offset, src_len, dest_offset);

争论描述
buffer要加载的缓冲区的索引。
filename要加载的文件的名称。
offset要加载到的缓冲区中的偏移量（以字节为单位）。
src_len缓冲区要加载的部分的长度（以字节为单位）。
dest_offset开始将部分数据放入新缓冲区的偏移量（以字节为单位）。

返回值：

Real

例子：

buff = buffer_create(256, buffer_grow, 1);
var _file = "save.dat";
var _so = 6;
var _sl = 5;
var _do= 0;
buffer_load_partial(buff, _file, _so, _sl, _do);

上面的代码将创建一个新的“增长”缓冲区，然后将保存在文件“ save.dat”中的部分数据加载到其中。

返回：缓冲区
