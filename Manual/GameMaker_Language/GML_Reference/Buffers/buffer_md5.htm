buffer_md5

在密码学中，MD5（消息摘要算法5）是具有128位哈希值的广泛使用的密码哈希函数，已在各种安全应用程序中使用。它还通常用于检查数据的完整性。该函数将从给定缓冲区（或缓冲区的一部分）中获取输入数据，并返回该数据唯一的32个字符的十六进制MD5哈希值。这样，您可以生成一个安全密钥，该密钥可以存储并用于检查发送到（例如）外部服务器（或从外部服务器接收）的信息的完整性。使用此功能将其应用于缓冲区时，必须指定要使用的缓冲区的缓冲区ID，然后指定起始位置的偏移值（以字节为单位），然后指定要哈希的区域的大小（以字节为单位）。

句法：

buffer_md5(buffer, offset, size);

争论描述
buffer要使用的缓冲区的索引。
offset数据偏移值。
size缓冲区的大小。

返回值：

String

例子：

check_string = buffer_md5(buff, 0, buffer_get_size(buff));

上面的代码将为存储在变量“ buff”索引的缓冲区中的完整数据创建一个md5哈希，并将返回的哈希存储在变量“ check_string”中。

返回：缓冲区
