buffer_peek

随着 buffer_read()函数，您可以在当前“搜索”位置从给定的缓冲区读取数据，读取的每条数据都通过读取或写入的字节在该位置前进。但是，您可能有必要在不更改当前搜索位置的情况下读取给定的数据，这就是您将使用此功能的时候。您只需为函数提供一个缓冲区id，然后为该缓冲区中要读取的偏移量位置（从缓冲区开始），以及要读取的数据类型。

注意：对于读取的数据使用错误的数据类型将导致错误的值！

如果函数成功执行，则将根据数据类型返回一个值（Real，string，int等），而失败将仅返回0。

句法：

buffer_peek(buffer, offset, type);

争论描述
buffer要使用的缓冲区的索引。
offset缓冲区中要从中读取给定数据的偏移位置（以字节为单位）。
type数据的类型，它是从缓冲区中读取（见常数的列表在这里）。

返回值：

Real/String or 0 if it fails.

例子：

var red = buffer_peek(buff, 1, buffer_u8);
var green = buffer_peek(buff, 2, buffer_u8);
var blue = buffer_peek(buff, 3, buffer_u8);
image_blend = make_colour_rgb(red, green, blue);

上面的代码将从变量“ buff”中索引的缓冲区中的三个不同位置获取三个值，然后使用这些值设置实例的图像混合。

返回：缓冲区
