buffer_poke

随着 buffer_write()函数，您可以将数据写入当前“搜索”位置的给定缓冲区，每条数据通过写入或读取字节来使位置前进。但是，可能有必要在不更改当前搜索位置的情况下更改给定的数据，这就是使用该功能的时候。您只需为函数提供一个缓冲区索引，然后为该缓冲区中要写入的缓冲区起始位置（以字节为单位）的偏移位置，以及要写入的数据类型和值。

句法：

buffer_poke(buffer, offset, type, value);

争论描述
buffer要使用的缓冲区的索引。
offset缓冲区中要写入给定数据的偏移位置（以字节为单位）。
type要写入缓冲区的数据类型（请参见此处的常量列表）。
value根据指定的类型添加到缓冲区的数据。

返回值：

N/A

例子：

buffer_poke(buff, 3, buffer_u8, colour_get_blue(image_blend));

上面的代码将用于图像混合的颜色的蓝色分量值添加到在变量“ buff”中索引的缓冲区中，在缓冲区的第三位置，并作为无符号的8位值。

返回：缓冲区
