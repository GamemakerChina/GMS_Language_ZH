缓冲区_poke

使用该 buffer_write()函数，你可以在当前的 "寻找 "位置向给定的缓冲区写入数据，每一个数据都会在这个位置上提前写入或读取。但是 可能需要你改变一个给定的数据，而不想改变当前的寻址位置，这时你就会使用这个函数。你只需向该函数提供一个缓冲区的索引，然后提供从缓冲区开始的偏移位置（字节数）。 缓冲区开始的偏移位置（以字节为单位），以及要写入的数据类型和值。

语法。

buffer_poke(buffer, offset, type, value)。

争论 描述
缓冲区 要使用的缓冲区的索引。
补偿 在缓冲区内的偏移位置（以字节为单位），将给定的数据写入。
类型 要写入缓冲区的数据类型（见这里的常数列表）。
价值 要添加到缓冲区的数据，符合指定的类型。

返回。

不适用

例子。

buffer_poke(buff, 3, buffer_u8, color_get_blue(image_blend))。

上述代码将把用于图像混合的颜色的蓝色分量值添加到变量 "buff "中索引的缓冲区中，在缓冲区的第三个位置，是一个无符号的8位数值。
