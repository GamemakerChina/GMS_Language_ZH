buffer_resize

使用此功能，您可以将给定缓冲区的大小调整为您指定的大小（以字节为单位）。

句法：

buffer_resize(buffer, newsize);

争论描述
buffer更改缓冲区大小的索引。
newsize缓冲区的新大小（以字节为单位）。

返回值：

N/A

例子：

if (buffer_get_size(buff) < 16384)
    {
    buffer_resize(buff, 16384);
    }

上面的代码将检查在变量“ buff”中索引的缓冲区的大小，如果小于给定值，则将调整缓冲区的大小。

返回：缓冲区
