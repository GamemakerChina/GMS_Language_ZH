缓冲区_保存

通过这个函数，你可以将缓冲区的内容保存到一个文件中，准备用下面的函数读回内存。 buffer_load()函数读回内存。

注意：在HTML5上，使用这个函数时，缓冲区的内容将被保存为base64编码的字符串。

语法。

buffer_save(buffer, filename)。

争论 描述
缓冲区 要保存的缓冲区的索引。
文件名 要保存为的文件的名称。

返回。

不适用

例子。

buffer_save(buff, "Player_Save.sav")。

将缓冲区的当前内容与存储在变量 "buff "中的id一起保存到一个文件中。
