buffer_save

使用此功能，您可以将缓冲区的内容保存到文件中，以备使用 buffer_load()功能。

注意：在HTML5上，使用此功能时，缓冲区的内容将另存为base64编码的字符串。

句法：

buffer_save(buffer, filename);

争论描述
buffer要保存的缓冲区的索引。
filename另存为的文件名。

返回值：

N/A

例子：

buffer_save(buff, "Player_Save.sav");

将具有变量“ buff”中存储的id的缓冲区的当前内容保存到文件中。
