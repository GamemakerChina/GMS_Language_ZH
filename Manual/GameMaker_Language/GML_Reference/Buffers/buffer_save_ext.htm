buffer_save_ext

使用此功能，您可以将缓冲区的部分内容保存到文件中，以备使用 buffer_load()功能。 “偏移量”定义了缓冲区中要保存的起始位置（以字节为单位），“大小”是从该偏移量开始要保存的缓冲区的大小（也以字节为单位）。

句法：

buffer_save_ext(buffer, filename, offset, size);

争论描述
buffer要保存的缓冲区的索引。
filename另存为的文件名。
offset缓冲区中要保存的偏移量（以字节为单位）。
size要保存的缓冲区的大小（以字节为单位）。

返回值：

N/A

例子：

buffer_save_ext(buff, "Player_Save.sav", 0, 16384);

将缓冲区的当前内容的一部分以及存储在变量“ buff”中的id保存到文件中。
