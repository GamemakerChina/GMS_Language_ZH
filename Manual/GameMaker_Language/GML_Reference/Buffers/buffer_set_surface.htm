buffer_set_surface

使用此功能，您可以将信息从缓冲区写入给定的表面。要读取的缓冲区和要写入的表面都必须事先创建，并且可以在缓冲区中提供一个偏移量以开始读取。请注意，读取将始终从缓冲区的开头加上偏移值开始，而不是从当前查找位置加上偏移值开始。

注意：此功能已从GameMaker Studio更新2.3.1开始更改。以前的版本使用常量 buffer_surface_copy，现已弃用。

句法：

buffer_set_surface(buffer, surface, offset);

争论描述
buffer要使用的缓冲区的索引。
surface要使用的表面的索引。
offset数据偏移值。

返回值：

N/A

例子：

buffer_set_surface(buff, application_surface, 0);

上面的代码将把存储在变量“ buff”中索引的缓冲区中的所有数据无偏移地复制到应用程序表面。

返回：缓冲区
