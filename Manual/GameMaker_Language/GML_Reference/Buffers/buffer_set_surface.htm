buffer_set_surface

通过这个函数，你可以把一个缓冲区的信息写到一个给定的表面。被读取的缓冲区和被写入的表面都必须是以前创建的，你可以提供一个缓冲区的偏移量来开始读取。 注意，读的时候总是从缓冲区的开始加上偏移值开始，而不是从当前的寻址位置加上偏移值开始。

注意：从GameMaker Studio更新2.3.1开始，这个功能已经改变。以前的版本使用常量 buffer_surface_copy ，现在已经废弃了。

语法。

buffer_set_surface(buffer, surface, offset)。

争论 描述
缓冲区 要使用的缓冲区的索引。
表面 要使用的表面的索引。
补偿 数据偏移值。

返回。

不适用

例子。

buffer_set_surface(buff, application_surface, 0)。

上述代码将把存储在变量 "buff "索引的缓冲区中的所有数据复制到应用程序表面，没有偏移。
