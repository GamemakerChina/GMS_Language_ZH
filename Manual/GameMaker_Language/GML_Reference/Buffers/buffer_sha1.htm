buffer_sha1

在加密技术中，SHA-1是由美国国家安全局设计的加密哈希函数，已在多种广泛使用的应用程序和协议中使用，例如流行的Git ，用于检查文件更改。此函数将从缓冲区获取输入数据，并以ASCII格式返回160位消息摘要。这样，您可以生成一个安全密钥，该密钥可以存储并用于检查发送到（例如）外部服务器（或从外部服务器接收）的信息的完整性。使用此功能将其应用于缓冲区时，必须指定要使用的缓冲区的缓冲区ID，然后指定起始位置的偏移值（以字节为单位），然后指定要哈希的区域的大小（以字节为单位）。

句法：

buffer_sha1(buffer, offset, size);

争论描述
buffer要使用的缓冲区的索引。
offset数据偏移值。
size缓冲区的大小。

返回值：

String

例子：

check_string = buffer_sha1(buff, 0, buffer_get_size(buff));

上面的代码将为变量“ buff”索引的缓冲区中存储的完整数据创建sha1哈希，并将返回的哈希存储在变量“ check_string”中。

返回：缓冲区
