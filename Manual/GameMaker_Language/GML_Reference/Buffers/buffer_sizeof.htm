buffer_sizeof

此函数将返回任何给定数据常量（在此处列出）的大小（以字节为单位）。

句法：

buffer_sizeof(type);

争论描述
type要检查的数据类型（请参见此处的常量列表）。

返回值：

N/A

例子：

var b = 12 * buffer_sizeof(buffer_u8);
buff = buffer_create(b, buffer_fixed, 1);

上面的代码首先通过将无符号的8位数据类型乘以12来计算要创建的缓冲区的大小（因为我们将使用该缓冲区保存12个数据），然后使用该值将固定的缓冲区设置为正确的尺寸。

返回：缓冲区
