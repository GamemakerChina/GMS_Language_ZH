buffer_tell

使用以下命令将数据读取或写入缓冲区时 buffer_read()或者 buffer_write()当前“搜索”位置以写入或读取的字节为单位前进，并且使用此功能，您可以获取当前“搜索”位置以用于其他缓冲区功能。例如，如果您将缓冲区对齐设置为4个字节，并且您写了一个大小为1个字节的数据，则执行 buffer_tell()，您将获得1的返回值。但是，如果您写入另一段数据（大小也为1字节），请 buffer_tell()，因为对齐方式已将数据“填充”到该位置，您将获得5的返回值。

句法：

buffer_tell(buffer);

争论描述
buffer要使用的缓冲区的索引。

返回值：

Real

例子：

var pos = buffer_tell(buff); buffer_seek(buff, buffer_seek_start, 0);
val[0] = buffer_read(buff, buffer_S16);
val[1] = buffer_read(buff, buffer_S16);
val[2] = buffer_read(buff, buffer_S16);
buffer_seek(buff, buffer_seek_start, pos);

上面的代码将在变量“ buff”中索引到本地变量“ pos”的缓冲区中存储当前搜索位置。然后将缓冲区搜寻位置设置为缓冲区的开始，并将三段数据读入数组，最后将缓冲区搜寻位置重新设置为之前的位置。
