camera_apply

此功能会将给定的摄像机设置应用于立即渲染的当前摄像机视图。虽然您可以使用特定的 camera_set_*功能，新的设置将在下一个游戏帧之前不会用于渲染，但是使用此功能，您可以立即进行更改。如果使用多个视图，则应在Draw Event中使用该函数，并且可以使用变量 view_current选择性地仅将给定相机应用于当前正在渲染的相机视图。请注意，这不会更改相机ID值，只会更改该相机的设置，并且如果您在默认相机上使用此功能（即：当没有视图相机处于活动状态时， GameMaker Studio 2通过绘制房间的内部设置覆盖了下一个游戏框架。

句法：

camera_apply(camera_id)

争论描述
camera_id创建摄像机时返回的唯一摄像机ID值。

返回值：

N/A

例子：

if view_current == 0
    {
    camera_apply(cutscene_cam);
    }

上面的代码（在Draw事件中）检查以查看当前正在渲染哪个摄影机视图，如果它是摄影机view [0]，则将变量“ cutscene_cam”引用的摄影机设置应用于该摄影机视图。
