camera_create

此功能将创建一个“空”摄像机，即：未定义视图或投影设置的摄像机。创建摄像机后，您便可以使用不同的矩阵功能来设置视图和投影矩阵。您也可以使用此功能创建相机，然后使用其他 camera_set_*设置视图的功能，但是请注意，您必须至少提供一个位置和视图大小，否则如果将其应用于视口，则生成的相机将无效并给出奇怪的结果（如果未设置跟随对象，则边框或移动速度，这些值将在“房间编辑器”中设置为用于查看摄像机的默认值）。

请注意，相机就像所有动态资源一样，在创建时会占用内存。这意味着您需要注意不要不必要地创建相机，并通过调用函数销毁不需要的相机 camera_destroy()。这也意味着，如果将摄像机分配给 global变量（它们也将在分配给实例或局部变量时仍然存在，但是您在更改空间后将无法访问它们，因此会导致内存泄漏），以便您可以（例如）创建一个新摄像机，将其分配给全局变量，然后在每个房间的开始处设置所需的视图和投影矩阵以及您需要使用的视图端口。

句法：

camera_create();

返回值：

Real (unique camera ID value)

例子：

view_camera[0] = camera_create();
var viewmat = matrix_build_lookat(640, 240, -10, 640, 240, 0, 0, 1, 0);
var projmat = matrix_build_projection_ortho(640, 480, 1.0, 32000.0);
camera_set_view_mat(view_camera[0], viewmat);
camera_set_proj_mat(view_camera[0], projmat);

上面的代码创建了一个新相机，并将其ID分配给view [0]。然后创建视图和投影矩阵，并将其存储在局部变量中，然后再用于设置摄像机。
