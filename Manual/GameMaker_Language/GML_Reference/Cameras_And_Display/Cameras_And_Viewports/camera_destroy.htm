camera_destroy

使用此功能，您可以销毁任何相机。调用该函数时，您将提供唯一的摄像机ID值，该值是从 camera_create_*功能或从 view_camera如果使用“房间编辑器”来设置视图端口和视图，则为数组。你不应该破坏当前分配到一个可见视图，除非你分配一个新的相机在相同的步骤是视摄像头，并且你应该总是破坏你已经通过代码创建的相机时，你的游戏不再需要为防止内存泄漏，如果在“房间编辑器”中分配了默认摄像头，也可以销毁默认摄像头，但是应为视口分配一个新摄像头（或将其禁用），否则会得到奇怪的结果。

句法：

camera_destroy(camera_id)

争论描述
camera_id创建摄像机时返回的唯一摄像机ID值。

返回值：

Real (unique camera ID value)

例子：

camera_destroy(view_camera[0]);
view_camera[0] = camera_create_view(0, 0, 640, 480, 0, obj_Player, 5, 5, -1, -1);

上面的代码将销毁当前分配给查看端口[0]的摄像机，然后创建一个新摄像机并将其ID分配给该端口。
