camera_get_active

此功能可用于检索当前活动摄像机的唯一摄像机ID值。

句法：

camera_get_active()

返回值：

Camera index (real).

例子：

var active = camera_get_active();
if active != view_camera[0]
    {
    view_camera[0] = active;
    }

上面的代码获取活动摄像机的摄像机ID，并将端口[0]的查看摄像机设置为使用它。
