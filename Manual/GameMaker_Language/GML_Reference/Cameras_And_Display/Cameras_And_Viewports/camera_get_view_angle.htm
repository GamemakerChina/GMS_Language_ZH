camera_get_view_angle

此功能可用于检索给定摄像机的角度。返回值将在0到360之间。请注意，此功能仅对使用 camera_create_view()或用于在房间编辑器中添加的内容。

句法：

camera_get_view_angle(camera_id)

争论描述
camera_id创建摄像机时返回的唯一摄像机ID值

返回值：

Real

例子：

var ang = camera_get_view_angle(view_camera[0]);
if ang != 0
    {
    camera_set_view_angle(view_camera[0], 0);
    }

上面的代码检索分配给查看端口[0]的摄像机的角度，然后检查它是否与给定值匹配。如果没有，则将观察摄像机设置为该角度。
