摄像头_get_view_border_y

此函数可用于检索给定摄像机的对象/实例下面沿y轴（垂直边界）的边界值。返回值的单位是像素。

语法。

camera_get_view_border_y(camera_id)

争论 描述
摄像机_ID 创建摄像机时返回的唯一的摄像机ID值

返回。

真实的

例子。

var xb = camera_get_view_border_x(view_camera[0]);
var yb = camera_get_view_border_y(view_camera[0]);
if xb != 200 || yb != 200
{
camera_set_view_border(view_camera[0], 200, 200);
}

上面的代码检索分配给view port[0]的摄像机的xborder和yborder值，然后检查它是否与给定的值一致。如果不匹配，那么视图摄像机将被设置为给定值。
