camera_get_view_mat

此功能可用于检索视图矩阵。该函数返回矩阵ID值，然后可将其用于其他矩阵函数或设置另一台摄像机的视图矩阵（使用 camera_set_view_mat()）。请注意，此功能通常用于使用 camera_create()，但也可以用于在房间编辑器（或通过 camera_create_view()）-在这种情况下，返回的矩阵仅在第一次在视口中使用给定的摄像机后才有效。

句法：

camera_get_view_mat(camera_id)

争论描述
camera_id创建摄像机时返回的唯一摄像机ID值

返回值：

Matrix ID (real)

例子：

mat = camera_get_view_mat(camera_view[0]);

上面的代码获取分配给查看端口[0]的摄像机的视图矩阵。
