摄像机_get_view_speed_x

这个函数可以用来检索给定摄像机沿x轴的移动速度（水平移动）。返回值的单位是每游戏帧的像素。

语法。

camera_get_view_speed_x(camera_id)

争论 描述
摄像机_ID 创建摄像机时返回的唯一摄像机ID值

返回。

真实的

例子。

var xs = camera_get_view_speed_x(view_camera[0]);
var ys = camera_get_view_speed_y(view_camera[0]);
if xs != 5| ys != 5
{
camera_set_view_speed(view_camera[0], 5, 5);
}

上面的代码检索分配给view port[0]的摄像机的xspeed和yspeed，然后检查它是否与给定的值一致。如果不匹配，那么视图摄像机将被设置为该速度。
