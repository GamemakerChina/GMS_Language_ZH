camera_get_view_speed_x

此功能可用于检索给定摄像机沿x轴的移动速度（水平移动）。返回值将以每个游戏帧为像素。

句法：

camera_get_view_speed_x(camera_id)

争论描述
camera_id创建摄像机时返回的唯一摄像机ID值

返回值：

Real

例子：

var xs = camera_get_view_speed_x(view_camera[0]);
var ys = camera_get_view_speed_y(view_camera[0]);
if xs != 5 || ys != 5
    {
    camera_set_view_speed(view_camera[0], 5, 5);
    }

上面的代码检索分配给查看端口[0]的摄像机的xspeed和yspeed，然后检查它是否与给定值匹配。如果没有，则将观察摄像机设置为该速度。

背面：摄像头和查看端口
