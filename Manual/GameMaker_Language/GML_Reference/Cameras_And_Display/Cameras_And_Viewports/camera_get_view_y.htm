camera_get_view_y

此功能可用于检索给定摄像机的视图的y位置。请注意，此功能仅对使用 camera_create_view()或用于在房间编辑器中添加的内容。

句法：

camera_get_view_y(camera_id)

争论描述
camera_id创建摄像机时返回的唯一摄像机ID值

返回值：

Real

例子：

var vx = camera_get_view_x(view_camera[0]);
var vy = camera_get_view_y(view_camera[0]);
draw_text(vx + 5, vy + 5, "SCORE" + string(score));

上面的代码检索分配给查看端口[0]的摄像机的位置，然后相对于该位置绘制文本。
