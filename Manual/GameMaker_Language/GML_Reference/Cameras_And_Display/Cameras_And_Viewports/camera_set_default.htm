camera_set_default

此功能可用于将默认摄像机设置为使用您以前使用以下方式之一创建的自定义摄像机：camera_create()功能。当您创建一个没有活动观察端口或观察摄像机GameMaker Studio 2仍会使用摄像机来显示游戏中的动作。该摄像机称为默认摄像机，可以像其他任何摄像机一样进行设置和操作（甚至销毁）。

句法：

camera_set_default(camera_id)

争论描述
camera_id创建摄像机时返回的唯一摄像机ID值

返回值：

N/A

例子：

var new_cam = camera_create_view(0, 0, 640, 480, 0, obj_Player, 5, 5, -1, -1);
camera_set_default(new_cam);

上面的代码创建一个新相机，然后将其分配为要使用的默认相机。
