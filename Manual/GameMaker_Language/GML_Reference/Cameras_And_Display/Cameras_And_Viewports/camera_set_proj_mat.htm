camera_set_proj_mat

此功能将设置给定摄像机的投影矩阵。您提供唯一的相机ID值（由其他 camera_create()函数）和要应用的视图矩阵。您可以从矩阵函数一节中找到有关创建投影矩阵的更多信息，特别是 matrix_build_projection_perspective()和 matrix_build_projection_ortho() 。请注意，如果您的相机执行自动对象跟踪-即：它是使用以下方法创建的 camera_create_view()对象索引/实例ID不为-1，或者您正在设置“房间编辑器”中定义的摄像机，或者您正在设置默认摄像机-那么此矩阵将在下一个游戏帧被覆盖。

句法：

camera_set_proj_mat(camera_id, matrix)

争论描述
camera_id创建摄像机时返回的唯一摄像机ID值。
matrix创建矩阵时返回的唯一矩阵ID。

返回值：

N/A

例子：

view_camera[0] = camera_create();
var viewmat = matrix_build_lookat(640, 240, -10, 640, 240, 0, 0, 1, 0);
var projmat = matrix_build_projection_ortho(640, 480, 1.0, 32000.0);
camera_set_view_mat(view_camera[0], viewmat);
camera_set_proj_mat(view_camera[0], projmat);

上面的代码创建了一个新摄像机，并将其分配给查看端口[0]。然后，此摄像机将设置其视图和投影矩阵。

背面：摄像头和查看端口
