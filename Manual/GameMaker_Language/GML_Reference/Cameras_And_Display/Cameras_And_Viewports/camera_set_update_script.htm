camera_set_update_script

此功能可用于设置脚本功能，该功能将在摄像机分配给可见和活动视口的每个游戏帧中调用。您提供唯一的相机ID值（由其他 camera_create函数）和要调用的函数的ID。附加到摄像机的功能和摄像机视图的实际渲染的顺序如下：

  • 所有可见和活动视口的摄像机的更新功能称为
  • 然后针对每个视图：
    • 相机针对该视图的开始功能称为
    • 绘制事件针对该视图执行
    • 相机的结束功能称为
  • 移至下一个活动的可见视图并重复...

句法：

camera_set_update_script(camera_id, script)

争论描述
camera_id创建摄像机时返回的唯一摄像机ID值。
script运行每个游戏框架的脚本功能

返回值：

N/A

例子：

camera_set_update_script(view_camera[0], updateCamera);

上面的代码为分配给查看端口[0]的摄像机设置更新功能。

背面：摄像头和查看端口
