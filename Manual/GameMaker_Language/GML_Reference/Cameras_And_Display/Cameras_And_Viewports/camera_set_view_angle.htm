camera_set_view_angle

您可以使用此功能更新房间内观察摄像机的角度。您提供唯一的相机ID值（由其他 camera_create()功能），然后提供您希望将相机视图设置为的角度。默认值为0°，正值将逆时针旋转摄像机，即：将值设置为90将使摄像机向左旋转90°。

句法：

camera_set_view_angle(camera_id, angle)

争论描述
camera_id创建摄像机时返回的唯一摄像机ID值
angle将相机视图设置为的新角度

返回值：

N/A

例子：

camera_set_view_angle(view_camera[0], point_direction(x, y, mouse_x, mouse_y));

上面的代码将为分配给查看端口[0]的摄像机设置查看摄像机角度。

背面：摄像头和查看端口
