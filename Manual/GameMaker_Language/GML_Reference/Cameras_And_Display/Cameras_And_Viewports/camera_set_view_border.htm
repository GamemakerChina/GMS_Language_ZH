camera_set_view_border

您可以使用此功能在房间内设置摄像机的跟随边框。您提供唯一的相机ID值（由其他 camera_create函数），然后给出x边界大小和y边界大小（水平和垂直）。这些值仅在为观察摄像机分配了要跟随的实例目标时使用（在“房间编辑器”中或使用函数 camera_set_view_target()），并且与实例在视图更新其要跟随的位置之前需要离开视图的边界有多远。

句法：

camera_set_view_border(camera_id, x_border, y_border)

争论描述
camera_id创建摄像机时返回的唯一摄像机ID值。
x_border观察相机沿水平轴的边框（以像素为单位）
y_border观察相机沿垂直轴的边框（以像素为单位）

返回值：

N/A

例子：

camera_set_view_border(view_camera[0], 64, 64);

上面的代码将为分配给查看端口[0]的摄像机设置查看摄像机边框。

背面：摄像头和查看端口
