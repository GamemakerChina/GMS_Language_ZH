摄像机_set_view_border

你可以使用这个函数来设置房间内视图摄像机的跟随边界。你可以给出唯一的摄像机ID值（由不同的摄像机_创建函数返回），然后给出X边界尺寸和Y边界尺寸（水平 和垂直）。)这些值只在摄像机被分配了一个要跟踪的实例目标时使用（无论是在房间编辑器中还是使用函数 camera_set_view_target())，并且涉及到 实例离视图边界有多远，视图才会更新其位置来跟随。

语法。

camera_set_view_border(camera_id, x_border, y_border)

争论 描述
摄像机_ID 创建摄像机时返回的唯一摄像机ID值。
x_border 摄像机沿水平轴的边界（单位：像素）。
y_border 视图摄像机沿垂直轴的边界（像素）。

返回。

不适用

例子。

camera_set_view_border（view_camera[0], 64, 64）。

上面的代码将为分配给view port[0]的摄像机设置视图摄像机边界。
