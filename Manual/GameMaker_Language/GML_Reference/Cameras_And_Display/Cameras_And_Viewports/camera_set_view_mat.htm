摄像机_set_view_mat

此函数将为一个给定的摄像机设置视图矩阵。你要给出唯一的摄像机ID值（由不同的 Camera_create()函数返回的）和要应用的视图矩阵。你可以从矩阵函数一节中了解到 更多关于创建视图矩阵的信息，特别是在矩阵函数一节中。 matrix_build_lookat(). 请注意，如果你的摄像机能自动跟踪物体，即：它是用 Camera_create_view()创建的，其对象索引/实例ID不是-1，或者你正在设置一个在房间编辑器中定义的摄像机，或者你正在设置一个在房间编辑器中定义的摄像机。 房间编辑器中定义的摄像机，或者你正在设置默认摄像机 - 那么这个矩阵将在下一个游戏帧中被覆盖。

语法。

camera_set_view_mat(camera_id, matrix)。

争论 描述
摄像机_ID 创建摄像机时返回的唯一摄像机ID值。
基体 你创建矩阵时返回的唯一的矩阵ID。

返回。

不适用

例子。

view_camera[0] = camera_create();
var viewmat = matrix_build_lookat(640, 240, -10, 640, 240, 0, 0, 1, 0);
var projmat = matrix_build_projection_ortho(640, 480, 1.0, 32000.0);
camera_set_view_mat(view_camera[0], viewmat);
camera_set_proj_mat(view_camera[0], projmat）。)

上面的代码创建了一个新的摄像机并将其分配给view port[0]。然后这个相机的视图和投影矩阵被设置。
