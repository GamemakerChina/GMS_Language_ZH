摄像机_设置视图位置

你可以用这个函数来更新摄像机视图在房间内的位置。你给出唯一的摄像机ID值（由不同的 摄像机创建()函数返回的），然后给出要设置摄像机的x和y位置 来设置摄像机的位置。

语法。

camera_set_view_pos(camera_id, x, y)

争论 描述
摄像机_ID 创建摄像机时返回的唯一摄像机ID值。
x 视图所处的X位置（在房间里）。
y 视图的Y位置（在房间里）。

返回。

不适用

例子。

camera_set_view_pos(view_camera[0], x - (view_wport[0] / 2), y - (view_hport[0] / 2);

上面的代码将为分配给view port[0]的摄像机设置视图摄像机的位置。
