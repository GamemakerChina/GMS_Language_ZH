camera_set_view_pos

您可以使用此功能更新房间内摄像机视图的位置。您提供唯一的相机ID值（由其他 camera_create()功能），然后指定x和y位置以将相机设置为该位置。

句法：

camera_set_view_pos(camera_id, x, y)

争论描述
camera_id创建摄像机时返回的唯一摄像机ID值。
x将视图放置在（房间中）的x位置
y将视图放置在（房间中）的y位置

返回值：

N/A

例子：

camera_set_view_pos(view_camera[0], x - (view_wport[0] / 2), y - (view_hport[0] / 2));

上面的代码将为分配给查看端口[0]的摄像机设置查看摄像机的位置。

背面：摄像头和查看端口
