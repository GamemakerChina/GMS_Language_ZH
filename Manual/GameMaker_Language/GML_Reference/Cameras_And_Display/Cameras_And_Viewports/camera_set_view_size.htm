camera_set_view_size

您可以使用此功能更新房间内观察摄像机的尺寸。您提供唯一的相机ID值（由其他 camera_create()功能），然后指定要设置相机的宽度和高度（以像素为单位）。

句法：

camera_set_view_size(camera_id, width, height)

争论描述
camera_id创建摄像机时返回的唯一摄像机ID值。
width相机视图的宽度（以像素为单位）
height相机视图的高度（以像素为单位）

返回值：

N/A

例子：

camera_set_view_size(view_camera[0], view_wport[0], view_hport[0]);

上面的代码将为分配给查看端口[0]的摄像机设置查看摄像机的大小。

背面：摄像头和查看端口
