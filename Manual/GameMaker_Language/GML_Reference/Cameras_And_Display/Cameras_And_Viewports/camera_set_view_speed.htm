camera_set_view_speed

您可以使用此功能更新房间内观察摄像机的速度。您提供唯一的相机ID值（由其他 camera_create()函数），然后给出设置为遵循给定实例时应移动的x和y（水平和垂直）速度。速度以每步像素数计算，可以将其设置为“ -1”以使摄像机立即移动，但是如果未将摄像机设置为跟随任何实例，则此处设置的值将没有可见效果。

句法：

camera_set_view_speed(camera_id, xspeed, yspeed)

争论描述
camera_id创建摄像机时返回的唯一摄像机ID值。
xspeed视图在水平（x）轴上移动的速度（每游戏帧的像素数）
yspeed视图在垂直（y）轴上移动的速度（每游戏帧的像素数）

返回值：

N/A

例子：

camera_set_view_speed(view_camera[0], 5, 5);

上面的代码将为分配给查看端口[0]的摄像机设置查看摄像机的速度。

背面：摄像头和查看端口
