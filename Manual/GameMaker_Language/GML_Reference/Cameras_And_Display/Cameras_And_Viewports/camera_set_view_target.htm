camera_set_view_target

您可以使用此功能设置房间内观察摄像机的跟随目标。您提供唯一的相机ID值（由其他 camera_create()功能），然后提供您希望将摄像机视图设置为的目标实例或对象ID。请注意，如果您选择一个对象ID，并且房间中存在该对象的一个以上实例， GameMaker Studio 2将无法知道您希望跟随哪个实例，因此可能是其中的任何一个。

句法：

camera_set_view_target(camera_id, instance_id/object_id)

争论描述
camera_id创建摄像机时返回的唯一摄像机ID值
instance_id/object_id具有要跟随的摄影机目标的实例或对象

返回值：

N/A

例子：

camera_set_view_target(view_camera[0], id);

上面的代码会将分配给查看端口[0]的摄像机的查看摄像机目标设置为运行该代码的实例的目标。
