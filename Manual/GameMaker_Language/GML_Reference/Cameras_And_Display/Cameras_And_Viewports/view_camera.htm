view_camera

该数组保存分配给给定查看端口的唯一摄像机ID，可以将其设置为新摄像机或读取以获取当前摄像机，如果未分配摄像机，则返回-1。您最多可以在一个房间中使用8个视口（阵列值0到7），只需将变量设置为摄像机ID值即可将摄像机分配给其中的任何一个（由函数返回）camera_create()或者 camera_create_view()）。如果您已通过“房间编辑器”将摄像机设置为查看端口，则此变量将保存该端口的摄像机ID。

句法：

view_camera[0 ... 7];

返回值：

Camera index or -1 if no camera is assigned

例子：

if view_camera[0] == -1
    {
    view_camera[0] = camera_create_view(0, 0, room_width, room_height);
    }

上面的代码检查查看端口[0]是否连接了摄像机，如果没有，则创建并设置一个摄像机。
