view_current

此只读变量仅在Draw Event中有效，并返回正在渲染的当前视口。当您有各种视图时，返回值将在draw事件期间更改，因为draw事件将为每个视图端口连续调用一次。因此，例如当您在游戏室中使用视图端口[0]和视图端口[1]时，所有实例的draw事件将运行两次，每个端口运行一次，使用此变量，您可以检查当前正在绘制什么视图。通常，仅当房间中同时显示多个查看端口时，才用于将特定细节呈现给单个端口。请参见下面的示例代码。

句法：

view_current;

返回值：

Integer (view port index from 0 to 7)

例子：

if view_current == 0
    {
    var xx = camera_get_view_x(view_camera[0]);
    var yy = camera_get_view_y(view_camera[0]);
    draw_text(xx + 32, yy + 32, "Player 1");
    }
else
    {
    var xx = camera_get_view_x(view_camera[1]);
    var yy = camera_get_view_y(view_camera[1]);
    draw_text(xx + 32, yy + 32, "Player 2");
    }

上面的代码检查以查看当前正在绘制哪个视图，然后根据返回值为每个视图绘制不同的文本。
