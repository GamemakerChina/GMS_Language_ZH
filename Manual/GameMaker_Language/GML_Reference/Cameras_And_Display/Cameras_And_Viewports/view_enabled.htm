view_enabled

此变量控制是否启用房间内可见的任何查看端口。如果将查看端口设置为可见，然后禁用此选项，则整个房间将被缩放到与窗口大小成比例的屏幕，而不是通过查看端口绘制不同的摄像机。

句法：

view_enabled;

返回值：

Boolean (true: enabled, false: disabled)

例子：

if !view_enabled
    {
    view_visible[0] = true;
    view_enabled = true;
    }

上面的代码检查是否为房间启用了视图，如果没有启用，则确保查看端口[0]是可见的，然后为房间启用视图。
