view_get_hport

此功能可用于检索给定视口的高度。

句法：

view_get_hport(view_port)

争论描述
view_port目标的视口（0-7）

返回值：

Real

例子：

if view_get_hport(0) != (display_get_height () / 2)
    {
    view_set_hport(0, display_get_height() / 2);
    }

上面的代码将检查视口[0]的高度，如果它不是显示高度的一半，则将其设置为该值。
