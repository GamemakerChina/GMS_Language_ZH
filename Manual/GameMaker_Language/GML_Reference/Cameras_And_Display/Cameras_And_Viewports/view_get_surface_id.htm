view_get_surface_id

此函数可用于检索分配给给定视口的表面的唯一ID值（如果未分配表面，则将返回-1）。

句法：

view_get_surface_id(view_port)

争论描述
view_port目标的视口（0-7）

返回值：

Real

例子：

if view_get_surface_id(0) == -1
    {
    view_set_surface_id(0, global.vSurf);
    }

上面的代码将检查是否已将表面分配给视口[0]，如果尚未分配，则将分配一个视口。

背面：摄像头和查看端口
