view_get_visible

此功能可用于检查给定视口的可见性。该函数将返回 true如果可见，并且 false如果不是这样。

句法：

view_get_visible(view_port)

争论描述
view_port目标的视口（0-7）

返回值：

Boolean

例子：

if !view_get_visible(0)
    {
    view_set_visible(0, true);
    }

上面的代码将检查查看端口[0]是否可见，如果不可见，则将其设置为可见。
