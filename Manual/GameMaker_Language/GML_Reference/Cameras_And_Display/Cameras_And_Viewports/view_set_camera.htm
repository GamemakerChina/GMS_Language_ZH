view_set_camera

此功能会将摄像机设置到特定的视口。您给视口设置（从0到7），并提供唯一的摄像机ID值（由 camera_create()功能或使用时 view_get_camera()）。如果将值-1作为摄像机ID，则表示您正在从视口中移除摄像头，并请注意，如果该视口已启用且可见，则可能会得到一些不可预测的结果。

句法：

view_set_camera(view_port, camera_id)

争论描述
view_port目标的视口（0-7）
camera_id创建摄像机时返回的唯一摄像机ID值

返回值：

N/A

例子：

var cam = camera_create_view(0, 0, 640, 480, 0, obj_Player, 5, 5, -1, -1);
view_set_camera(0, cam);

上面的代码将创建一个新相机，然后为其分配查看端口[0]。

背面：摄像头和查看端口
