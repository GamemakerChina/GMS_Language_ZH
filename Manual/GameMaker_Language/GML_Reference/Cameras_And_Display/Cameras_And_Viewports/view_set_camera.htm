视图_设置摄影机

这个函数将设置一个摄像机到一个特定的视口。你要给出要设置的视口（从0到7），并提供唯一的相机ID值（如由 camera_create()函数返回的，或者当你使用 view_get_camera()). 如果你给出的摄像机ID值为-1，那么你将从视口中删除一个摄像机，注意，如果该视口是启用和可见的，你可能会得到一些不可预知的结果。

语法。

view_set_camera(view_port, camera_id)

争论 描述
视图_端口 目标视图端口（0-7）。
摄像机_ID 创建摄像机时返回的唯一摄像机ID值

返回。

不适用

例子。

var cam = camera_create_view（0, 0, 640, 480, 0, obj_Player, 5, 5, -1, -1）;
view_set_camera（0, cam）。

上面的代码将创建一个新的摄像机，然后将其分配给view port[0]。
