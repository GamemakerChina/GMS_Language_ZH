查看_set_surface_id

通过这个变量，你可以设置视口的内容绘制到曲面上。在处理曲面时，经常需要将整个屏幕的可见区域都捕捉到曲面上，因此你会用 这个函数。这意味着在所选端口视图中显示的所有内容现在都将被绘制到指定的曲面上，而该视图端口的内容将不再显示，这意味着你需要要么。

  • 启用一个新的视图并只在该视图中绘制曲面（用 视图_当前来检查哪个视图正在被绘制）。)
  • 在实例的Draw GUI事件中绘制表面，因为GUI层是独立于视图的。

当使用这个函数时，你要给出视图端口的索引（从0到7）和一个表面的索引（要么是 application_surface或由函数返回的唯一索引值 surface_create()) 或者，如果之前已经分配了一个曲面，而你想移除它，那么就给一个-1的值。 关于将视口设置为一个曲面的进一步信息，也可以参见变量 view_surface_id.

语法。

view_set_surface_id(view_port, surf)

争论 描述
视图_端口 目标视图端口（0-7）。

返回。

真实的

例子。

如果view_get_surface_id(0) == -1
{
view_set_surface_id(0, global.vSurf);
}

上面的代码将检查是否有一个表面被分配到视图端口[0]，如果没有，就会分配一个。
