view_set_surface_id

使用此变量，您可以设置视口的内容以绘制到表面上。使用曲面时，通常需要将屏幕的整个可见区域捕获到曲面上，因此您可以使用此功能将其分配给视口。这意味着所选端口视图中显示的所有内容现在都将绘制到指定的表面，并且该视图端口的内容将不再显示，这意味着您需要：

  • 启用新视图并仅在该视图中绘制表面（使用 view_current检查正在绘制哪个视图）
  • 由于GUI层独立于视图，因此在实例的Draw GUI事件中绘制表面。

使用此功能时，您可以提供视口索引（从0到7）和表面索引（可以是 application_surface或函数返回的唯一索引值 surface_create()），或者，如果先前已分配了曲面并且您要删除该曲面，则值为-1。有关将视口设置为表面的更多信息，另请参见变量 view_surface_id。

句法：

view_set_surface_id(view_port, surf)

争论描述
view_port目标的视口（0-7）

返回值：

Real

例子：

if view_get_surface_id(0) == -1
    {
    view_set_surface_id(0, global.vSurf);
    }

上面的代码将检查是否已将表面分配给视口[0]，如果尚未分配，则将分配一个视口。
