view_set_wport

此功能可用于设置给定视口的宽度。您给视口索引（从0到7）和要使用的新宽度。

句法：

view_set_wport(view_port, w)

争论描述
view_port目标的视口（0-7）
w新宽度（以像素为单位）

返回值：

Real

例子：

if view_get_wport(0) != (display_get_width () / 2)
    {
    view_set_wport(0, display_get_width() / 2);
    }

上面的代码将检查视口[0]的宽度，如果它不是显示宽度的一半，则将其设置为该值。
