view_visible

此变量可用于确定当前是否可以查看特定的查看端口。您还可以通过将该变量设置为来设置此变量，以有效地“打开”或“关闭”视图。true （可见）或 false（无形的）。请注意，即使您将查看端口设置为可见，如果未启用查看端口（使用内置变量，view_enabled或在“房间编辑器”中启用它们），则不会将它们绘制到屏幕上。

句法：

view_visible[0 ... 7];

返回值：

Boolean (true: enabled, false: disabled)

例子：

if !view_visible[0]
    {
    view_visible[0] = true;
    }

上面的代码检查view [0]是否可见，如果不可见，则将其设置为可见。
