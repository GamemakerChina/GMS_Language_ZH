查看_可见

这个变量可以用来了解某个特定的视图端口当前是否可见。你也可以通过将这个变量的值设置为true（可见）或false（不可见）来有效地 "打开 "或 "关闭 "一个视图。请注意，即使你把一个视口设置为可见，如果视口没有被启用（使用内置变量 view_enabled或在房间编辑器中启用它们），那么它们将不会被绘制到 屏幕上。

语法。

view_visible[0 ... 7]。

返回。

布尔型（真：启用，假：禁用）。

例子。

if !view_visible[0]
{
view_visible[0] = true;
}

上面的代码检查view[0]是否可见，如果不可见，则将其设置为可见。
