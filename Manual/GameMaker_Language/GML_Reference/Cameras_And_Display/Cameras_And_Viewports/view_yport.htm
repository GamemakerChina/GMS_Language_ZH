view_yport

使用此内置数组，您可以获取或设置给定视口的y位置。视图端口是屏幕上绘制视图的区域，您一次最多可以激活8个端口（该数组的值从0到7（含0和7），以提供8个端口）。 GameMaker Studio 2的默认设置是游戏窗口（或背景画布）与房间的大小相同，但是，当您激活视口和摄像机时，此行为会更改，并且所有视口的边框总大小会改变用来。因此，如果您在两个不同的位置具有两个不同的视口，则它们所覆盖的总面积将定义游戏窗口的大小。下图说明了这一点：

如果只有一个查看端口，则设置端口的x或y值可能会产生一些有趣的效果，但是通常不实用（请参见下图），因此最好将x和y位置保持为（0,0） 。

句法：

view_yport[0 .. 7];

返回值：

Real

例子：

view_xport[0] = 0;
view_yport[0] = 0;

上面的代码将视口[0]的位置重置为显示器的（0,0）位置（左上角）。
