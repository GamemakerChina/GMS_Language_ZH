游戏窗口

你创建的游戏发生在一个窗口中（即使是全屏时），这个窗口有很多属性，比如位置、大小、是否全屏等等。这些细节通常是根据房间的大小和视图端口自动设置的。 启用，并结合 目标平台 的游戏选项的设置，但你可以在游戏过程中使用本页面列出的功能来改变它们。

注意：这些功能是针对 窗口, Ubuntu(Linux)。 macOS 和HTML5目标模块，在其他任何设备上可能无法使用。

下面的图片说明了所有一些最普通的窗口功能是如何相互联系和相互作用的。
