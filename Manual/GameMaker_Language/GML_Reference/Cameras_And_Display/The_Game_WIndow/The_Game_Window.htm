游戏窗口

您创建的游戏在一个窗口中进行（即使是全屏显示），并且该窗口具有许多属性，例如位置，大小，是否为全屏显示等。这些详细信息通常会根据房间大小自动为您设置并结合目标平台的“游戏选项”中的设置查看启用的端口，但是您可以在游戏过程中使用此页面上列出的功能来更改端口。

注意：这些功能仅适用于Windows ， Ubuntu （ Linux ）， macOS和HTML5目标模块，可能不适用于任何其他设备。
