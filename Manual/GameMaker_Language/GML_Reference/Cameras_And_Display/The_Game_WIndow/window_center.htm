window_center

使用此功能，当目标模块是Windows ， Ubuntu （ Linux ）或macOS ，您可以将游戏窗口居中显示；如果目标模块是HTML5，则可以在浏览器中居中显示。此功能对任何其他设备均无效。

注意：如果您已经调整了游戏窗口的大小（例如，从全屏切换到窗口化），并且希望将新窗口居中放置，则应在警报[0]中至少比窗口大小晚一步调用此功能。更改，否则它将无法正常工作。

句法：

window_center();

返回值：

N/A

例子：

if !window_get_fullscreen()
    {
    window_center();
    }

如果上面的代码不在全屏模式下，它将使游戏窗口居中显示。
