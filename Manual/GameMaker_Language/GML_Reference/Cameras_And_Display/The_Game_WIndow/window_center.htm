窗口中心

有了这个功能，当目标模块是Windows、Ubuntu（Linux）或macOS时，你可以将游戏窗口在显示器中居中，如果目标模块是HTML5，你可以在浏览器中居中。这个功能对其他任何设备都没有影响。

注意：如果你已经调整了游戏窗口的大小（例如，从全屏切换到窗口化），并希望将新窗口居中，这个函数应该在比窗口大小变化至少晚一步的警报[0]中调用，否则 否则它将无法正常工作。

语法。

window_center()。

返回。

不适用

例子。

if !window_get_fullscreen()
{
window_center()。
}

如果不是全屏模式，上述代码将使游戏窗口在显示屏内居中。
