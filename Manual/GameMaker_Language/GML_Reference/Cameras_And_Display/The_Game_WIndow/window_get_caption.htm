window_get_caption

此函数返回窗口的标题（这是显示在窗口顶部，图标旁边的文本），默认情况下，它显示您当前所在房间的标题。

句法：

window_get_caption();

返回值：

String

例子：

if window_get_caption() != ""
    {
    window_set_caption("");
    }

上面的代码将检查Windows字幕以查看是否有文字，如果有，则将其设置为空字符串，这样就不会显示任何字幕。

返回：游戏窗口
