window_get_cursor

使用此功能，您可以获取当前在游戏窗口中使用的光标，其返回值是下面列出的任何常量之一（要设置使用的当前光标，可以使用该功能 window_set_cursor()这也需要这些常量）：

持续的光标
cr_none
cr_default
cr_arrow
cr_cross
cr_beam
cr_size_nesw
cr_size_ns
cr_size_nwse
cr_size_we
cr_uparrow
cr_hourglass
cr_drag
cr_appstart
cr_handpoint
cr_size_all

句法：

window_get_cursor();

返回值：

Constant

例子：

if mouse_check_button_pressed(mb_left)
    {
    if window_get_cursor() != cr_drag window_set_cursor(cr_drag);
    }

如果已按下鼠标左键，并且之前尚未更改，则上述代码会将窗口光标更改为标准窗口拖动光标。

返回：游戏窗口
