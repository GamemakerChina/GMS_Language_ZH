window_handle

使用此功能，您可以获得内部Windows id值（HWND，一个指针）。此功能实际上仅对需要 window handle以Windows API（返回的指针应转换为字符串，然后在C ++中将其转换为HWND）。下表显示了受支持的平台以及它们返回的内容：

平台退货
Windows窗口HWND
macOS NSWindow类
Ubuntu （ Linux ） XWindow句柄
HTML5画布编号

句法：

window_handle();

返回值：

Pointer

例子：

win_id = windows_handle();

上面的代码会将游戏窗口ID存储在一个变量中。

返回：游戏窗口
