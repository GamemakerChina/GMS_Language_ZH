窗口_手柄

通过这个函数你可以得到Windows内部的id值（HWND，一个指针）。这个函数实际上只对那些需要窗口句柄以在DLL代码中调用WindowsAPI的扩展编写者有用（返回的指针应被铸成一个字符串，然后在C++中只需将其铸成一个HWND）。 返回的指针应该被转换成一个字符串，然后在C++中把它转换成一个HWND）。)下表显示了支持的平台以及它们返回的内容。

平台 返回
窗户 窗口HWND
macOS NSWindow类
Ubuntu(Linux) XWindow手柄
HTML5 帆布的ID

语法。

window_handle()。

返回。

指针

例子。

win_id = windows_handle()。

上面的代码将在一个变量中存储游戏窗口的ID。

返回。游戏窗口
