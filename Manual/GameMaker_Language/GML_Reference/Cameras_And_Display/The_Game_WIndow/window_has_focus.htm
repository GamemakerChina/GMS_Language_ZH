window_has_focus

使用此功能，您可以轮询窗口（或选项卡）状态，如果失去焦点，该功能将返回 false否则它将返回 true。在大多数情况下，您可以简单地使用 os_is_paused()函数可以对此进行测试，但是在某些非常特殊的情况下（例如Chrome Apps上的游戏），该函数不会触发，在这种情况下，您应该改用此函数。

注意：此功能仅对HTML5， Windows和MacOS有效。

句法：

window_has_focus();

返回值：

Boolean

例子：

if !window_has_focus()
    {
    scr_PauseGame();
    }

上面的代码将检查游戏窗口是否在焦点上，以及函数是否返回 false脚本将被调用。

返回：游戏窗口
