window_mouse_set

使用此功能，您可以更改或设置鼠标在游戏窗口中的位置，例如，这对于FPS游戏很有用。该功能仅在游戏处于焦点状态时才能使用，并且使用alt + Tab键可以解锁鼠标。

注意：有关常规鼠标功能，请参见“鼠标输入”部分

句法：

window_mouse_set(x, y);

争论描述
x窗口中的x坐标。
y窗口中的y坐标。

返回值：

N/A

例子：

window_mouse_set(window_get_width() / 2, window_get_height() / 2);

上面的代码将鼠标置于游戏窗口的中央。

返回：游戏窗口
