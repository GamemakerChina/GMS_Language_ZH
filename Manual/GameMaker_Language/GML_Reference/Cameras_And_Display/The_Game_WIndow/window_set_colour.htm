window_set_colour

此功能可以设置游戏窗口的背景色。此颜色表示将用于游戏窗口中未被任何视图占用的那些区域的颜色。下图说明了这一点：

上图有两个视图，两个视图端口，每个视图端口绘制在不同的位置。这会拉伸游戏窗口以容纳两个端口，并使用窗口颜色为未显示视图的背景着色。

句法：

window_set_colour(colour);

争论描述
colour设置区域的颜色。

返回值：

N/A

例子：

if window_get_colour() != c_black
    {
    window_set_colour(c_black);
    }

上面的代码将检查窗口颜色，以查看是否将其设置为黑色，如果不是，则将其设置为黑色。

返回：游戏窗口
