window_set_cursor

使用此功能，您可以将游戏窗口的光标设置为下面列出的任何一个常量（要找到正在使用的当前光标，可以使用该功能 window_get_cursor()还将返回以下常量之一）：

持续的光标
cr_none
cr_default
cr_arrow
cr_cross
cr_beam
cr_size_nesw
cr_size_ns
cr_size_nwse
cr_size_we
cr_uparrow
cr_hourglass
cr_drag
cr_appstart
cr_handpoint
cr_size_all

句法：

window_set_cursor(cursor);

争论描述
cursor要为游戏窗口设置的光标。

返回值：

N/A

例子：

if mouse_check_button_pressed(mb_left)
    {
    window_set_cursor(cr_drag);
    }

如果按下了鼠标左键，以上代码会将窗口光标更改为标准窗口拖动光标。

返回：游戏窗口
