窗口_设置为fullscreen

通过这个功能，你可以将游戏窗口设置为全屏（真）或不全屏（假）。请注意，对于 MacOS目标，你必须取消勾选 "全屏启动 "选项，并勾选游戏选项中的 "允许玩家调整游戏窗口大小 "选项。 允许玩家调整游戏窗口的大小 "选项，否则这个功能会失败。还要注意的是，这个功能在HTML5上是不工作的，除非是作为一个 "可点击 "的 回调（更多细节见这里）。

语法。

window_set_fullscreen(full)。

争论 描述
全程 是否将屏幕设置为全屏（true）或不设置（false）。

返回。

不适用

例子。

if mouse_check_button_pressed(mb_left)
{
if window_get_fullscreen()
{
window_set_fullscreen(false);
}
else
{
window_set_fullscreen(true);
}
}

上面的代码检查鼠标按键是否被按下，如果还没有被按下，则将窗口设置为全屏，如果已经被按下，则将其设置为窗口。
