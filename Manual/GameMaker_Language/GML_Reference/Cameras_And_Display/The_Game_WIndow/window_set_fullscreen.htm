window_set_fullscreen

使用此功能，您可以将游戏窗口设置为全屏（true ） 或不 （false ）。请注意，对于macOS目标，您必须取消选中“游戏选项”中的“全屏启动”选项，并选中“允许玩家调整游戏窗口大小”选项，否则此功能将失败。还要注意，除非添加为“ clickable”回调（请参见此处以获取更多详细信息），否则此功能将不适用于HTML5。

句法：

window_set_fullscreen(full);

争论描述
full是否将屏幕设置为全屏（真）（否）。

返回值：

N/A

例子：

if mouse_check_button_pressed(mb_left)
    {
    if window_get_fullscreen()
        {
        window_set_fullscreen(false);
        }
    else
        {
        window_set_fullscreen(true);
        }
    }

上面的代码检查是否按下了鼠标按钮，然后将窗口设置为全屏（如果尚未设置），或者将其设置为窗口（如果已设置）。

返回：游戏窗口
