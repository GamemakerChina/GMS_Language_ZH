window_set_max_height

此功能可用于设置游戏的最大窗口高度。如果在目标平台的“游戏选项”中启用了“窗口调整大小”选项，则玩家可以将游戏窗口调整为所需的任何大小，但是通过使用此功能，您可以将最大高度限制为指定的大小。如果您希望返回到默认行为（即：无最小值），则使用值-1。

注意：此功能仅在Windows ， macOS和Linux平台上可用。

句法：

window_set_max_height(height);

争论描述
height游戏窗口的最大高度（以像素为单位）。

返回值：

N/A

例子：

window_set_min_width(640);
window_set_min_height(480);
window_set_max_width(1280);
window_set_max_height(960);

上面的代码将设置游戏窗口的最小和最大宽度和高度。
