window_set_position

使用此功能，您可以将游戏窗口设置为显示器内的特定位置（在macOS ， Linux （ Ubuntu ）和Windows ）或浏览器（HTML5）中。

注意：如果您的HTML5游戏使用自定义 index.html并且将画布设置为固定位置，则此功能对窗口位置无效。

句法：

window_set_position(x, y);

争论描述
x窗口位置的x坐标。
y窗口位置的y坐标。

返回值：

N/A

例子：

window_set_position(0, 0);

上面的代码会将游戏窗口定位在浏览器或显示器的左上角（取决于所使用的目标模块）。

返回：游戏窗口
