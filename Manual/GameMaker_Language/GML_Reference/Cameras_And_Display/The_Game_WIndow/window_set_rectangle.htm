window_set_rectangle

有了这个功能，你可以设置游戏窗口的位置浏览器（HTML5）或显示器（内Windows ， Ubuntu （ Linux ）或macOS ），并设置窗口的规模太大。有关窗口位置和窗口大小的更多信息，请参见 window_set_position()和 window_set_size()。

句法：

window_set_rectangle(x, y, w, h);

争论描述
x窗口的新x坐标。
y窗口的新y坐标。
w窗口的新宽度。
h窗口的新高度。

返回值：

N/A

例子：

window_set_rectangle(0, 0, display_get_width(), display_get_height());

上面的代码将游戏窗口设置为占据整个显示区域（浏览器还是屏幕，取决于目标平台）。

返回：游戏窗口
