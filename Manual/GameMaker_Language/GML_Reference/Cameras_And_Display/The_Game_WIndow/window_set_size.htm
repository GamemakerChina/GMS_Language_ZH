window_set_size

使用此功能，您可以更改游戏窗口的大小。如果您使用此功能更改游戏窗口的大小，则窗口的内容将按与背景画布的1：1比例进行缩放，以保持图像的清晰和清晰（尽管如果窗口是其他窗口则按比例缩放）超过房间或视图的大小）。例如，如果您有一个200 x 150像素的窗口，并且播放器通过拖动边框来拉伸窗口，则最终的窗口大小可能是400 x 300像素，但绘图画布仍是STILL 200 x 150，这会产生模糊的图像因为像素比例现在是1：2但是，如果使用此功能设置窗口大小，即使图像被缩放以适应新的窗口大小，也将保持1：1的比例。

句法：

window_set_size( w, h );

争论描述
w窗口的新宽度。
h窗口的新高度。

返回值：

N/A

例子：

if mouse_check_button_pressed(mb_left)
    {
    global.scale += 1;
    window_set_size(200 * global.scale, 150 * global.scale);
    }

上面的代码将在按下鼠标按钮时根据变量“ global.scale”的值更改窗口的大小。

返回：游戏窗口
