window_view_mouse_get_y

此功能将返回鼠标相对于所选视图的y位置。

注意：有关常规鼠标功能，请参见“鼠标输入”部分

句法：

window_view_mouse_get_y( id );

争论描述
id与鼠标位置进行比较的视图的ID。

返回值：

Real

例子：

if mouse_check_button_pressed(mb_left)
    {
    var xx, yy;
    xx = window_view_mouse_get_x(0);
    yy = window_view_mouse_get_y(0);
    if xx > 0 && xx < 32 && yy > 0 && yy < 32
        {
        b_press[0] = true;
        }
    }

上面的代码将检查是否有鼠标按钮被按下，如果是，则获取相对于view [0]的鼠标位置，并对其进行比较以查看是否应将变量设置为true。

返回：游戏窗口
