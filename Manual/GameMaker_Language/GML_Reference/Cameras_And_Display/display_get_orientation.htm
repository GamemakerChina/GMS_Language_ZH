display_get_orientation

此函数将返回两个常量之一， GameMaker Studio 2必须告诉您运行游戏的设备是处于横向还是纵向模式（请参见下表）。请注意，此功能在HTML5目标模块中使用时可能无法正确检测设备的方向。但是，可以通过使用以下脚本轻松模仿：

return (browser_width < browser_height);

它将返回 true用于肖像和 false用于景观。

句法：

display_get_orientation()

返回值：

Constant

持续的描述
display_landscape水平握住设备，即：最长的边缘是从左到右，菜单按钮在右边。
display_landscape_flipped如上所述，只有菜单按钮在左侧。
display_portrait垂直握住设备，即：最长的边缘从上到下，菜单按钮在底部。
display_portrait_flipped如上所述，仅菜单按钮位于顶部。

例子：

if display_get_orientation() == display_landscape
    {
    global.Config = 0;
    }
else
    {
    global.Config = 1;
    }

上面的代码检查设备的方向，并根据返回的值设置全局变量。
