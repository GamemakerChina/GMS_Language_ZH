display_get_orientation

这个函数将返回GameMaker Studio 2的两个常数之一，以告诉你运行游戏的设备是以横向还是纵向模式持有的（见下表）。请注意，这个函数在HTML5目标模块中使用时可能无法正确检测设备的方向。 在HTML5目标模块中使用时，这个函数可能无法正确检测设备的方向。然而，通过使用下面的脚本可以很容易地模仿这一点。

返回（browser_width < browser_height）。

如果是纵向，它将返回true，如果是横向，则返回false。

语法。

display_get_orientation()

返回。

恒定

恒定 描述
显示_景观 设备被水平持有，即：最长的边缘从左到右，而菜单按钮在右边。
显示_景观_翻转 和上面一样，只是现在菜单按钮在左边。
显示_肖像 设备被垂直地拿着，即：最长的边缘是从上到下，而菜单按钮在底部。
显示_肖像_翻转 如上所述，只是现在菜单按钮在顶部。

例子。

如果 display_get_orientation() == display_landscape
{
global.Config = 0;
}
else
{
global.Config = 1;
}

上面的代码检查设备的方向，并根据返回的值设置一个全局变量。
