display_get_timing_method

此功能可用于检索用于渲染游戏的计时方法。该方法可以是下面列出的常数之一。

有关不同计时方法的更多信息，请参见 display_set_timing_method()。

句法：

display_get_timing_method();

返回值：

Constant

持续的描述
tm_sleep睡眠余量值是主要的计时方法
tm_countvsyncs Vsync计时是主要的计时方法（所有受支持平台的默认设置）

例子：

if display_get_timing_method() != tm_sleep
{
display_set_timing_method(tm_sleep);
if display_get_sleep_margin() != 20
    {
    display_set_sleep_margin(20);
    }
}

上面的代码将检查计时方法，然后将其设置为 tm_sleep它将被设置为该值，睡眠裕度将被设置为20。
