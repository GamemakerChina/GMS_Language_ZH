display_mouse_set

使用此功能，您可以更改或设置鼠标在游戏显示器中的位置，例如，这对于FPS游戏很有用。该功能仅在游戏焦点对准并使用时才起作用 ALT + Tab将解锁鼠标。

句法：

display_mouse_set(x, y);

争论描述
x显示屏中的x坐标。
y显示屏中的y坐标。

返回值：

N/A

例子：

display_mouse_set(display_get_width() / 2, display_get_height() / 2);

上面的代码将鼠标置于游戏显示器的中心。
