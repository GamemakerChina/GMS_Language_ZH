display_mouse_set

有了这个功能，你可以改变或设置鼠标在游戏显示中的位置，这对FPS游戏等很有用。该功能只在游戏处于焦点状态时起作用，使用ALT+Tab会解锁鼠标。

语法。

display_mouse_set(x, y)。

争论 描述
x 显示器中的X坐标。
y 显示器中的y坐标。

返回。

不适用

例子。

display_mouse_set(display_get_width() / 2, display_get_height() / 2)。

上述代码将使鼠标在游戏显示中居中。
