display_set_gui_maximise

此功能可用于最大化GUI层，并将其设置为相对于屏幕尺寸（而不是默认应用程序表面位置和尺寸）进行缩放和偏移。默认情况下，GUI层的应用表面分辨率为1：1，并且也在表面的（0,0）位置绘制。但是，这并不总是您想要的，因此您可以使用此功能将其设置为相对于显示或游戏窗口的绝对（0,0）位置绘制。您为此函数提供的参数取决于您希望它在GUI层上产生的效果。只需不带任何参数的函数调用，即可将GUI图层设置为在屏幕或游戏窗口的（0,0）位置绘制，其宽度和高度将缩放以适合整个区域：

display_set_gui_maximise();

但是，您可以为GUI层设置缩放比例，并且宽度和高度将按该比例缩放。请记住，GUI层始终适合显示或游戏窗口或应用程序表面的大小，因此将此值设置为1以外的任何值基本上都会沿宽度和高度缩放像素数。如果您的显示器为1024x768，并且将缩放比例设置为0.5，则GUI层将是显示器的一半，有效地将像素大小增加了一倍：

display_set_gui_maximise(0.5, 0.5);

以这种方式设置值还将绘制位置设置为显示或游戏窗口的（0,0），因此您可以提供偏移值以“移动”（0,0）位置：

var pos = application_get_position();
display_set_gui_maximise(0.5, 0.5, pos[0], pos[1]);

最后，可以通过将比例因子设置为-1来使用此功能重置GUI层。这将再次将GUI层设置为1：1比例，并将绘制位置设置为应用程序表面（而不是显示器或窗口）的（0,0）位置。

display_set_gui_maximise(-1, -1);

句法：

display_set_gui_maximise(xscale, yscale, xoffset, yoffset);

争论描述
xscale <可选>水平缩放因子（使用-1重置为默认值）。
yscale <Optional>垂直缩放因子（使用-1重置为默认值）。
xoffset <可选>图纸的x偏移位置。
yoffset <可选>图纸的y偏移位置。

返回值：

N/A

例子：

display_set_gui_size(display_get_width() / 2, display_get_height() / 2);
display_set_gui_maximise(2, 2, 0, 0);

上面的代码将把draw GUI事件锁定到给定的宽度和高度，并使用该比例缩放所有组件以适合显示。
