display_set_gui_size

此功能仅会影响“绘制GUI事件”中绘制的内容，并且可用于为该事件中的所有GUI组件设置特定的宽度和高度，并且无论使用什么大小的显示或窗口来显示GUI，它都会缩放以正确适合。 GUI的位置（即（0,0）位置）将取决于您是否使用了该功能 display_set_gui_maximise()或不。如果不使用此功能（默认设置），则GUI层的（0,0）点将位于与应用程序表面（绘制游戏的区域）和宽度相同的位置。高度将被拉伸以适合该表面，以便在应用程序表面为512px时使用此功能将GUI宽度设置为256px将拉伸该256个像素以适合，有效地使像素大两倍。对于那些在全局“游戏设置”中设置了宽高比校正的游戏来说，这是理想的选择。

如果您使用过 display_set_gui_maximise()功能，则GUI层的（0,0）位置将默认为游戏窗口或显示的左上角（在全屏或设备上），缩放比例由该功能设置，宽度和高度由这个设定。您可以随时通过使用-1作为宽度和高度来调用此函数来将GUI层重置为默认配置-这会将（0，0）位置设置到应用程序表面，并且宽度和高度将重置为表面也1：1。

句法：

display_set_gui_size(width, height);

争论描述
width GUI的宽度
height GUI的高度

返回值：

N/A

例子：

display_set_gui_size(768, 1024);

上面的代码将把draw GUI事件锁定到给定的宽度和高度，并使用该比例缩放所有组件以适合应用程序表面或显示。
