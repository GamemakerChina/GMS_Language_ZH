display_set_sleep_margin

此功能可用于设置用于游戏渲染时间的睡眠裕度值，并且需要毫秒值。默认值如下：

平台毫秒
Windows 10
macOS 10
Ubuntu 10
HTML5 10
Android 4
iOS 4
Windows UWP 10
的Xbox 10
PS4 10
Nintendo Switch 10

请注意，即使将计时方法设置为使用vsync计时，睡眠裕度也会对游戏渲染产生一定影响，因此在设置此值时应格外小心。有关显示时间的更多信息，请参见 display_set_timing_method()。

注意：除了“睡眠裕量”之外，您还可以通过在runtime调整线程调度程序的分辨率Windows上的性能和功耗。

句法：

display_set_sleep_margin(milliseconds);

争论描述
milliseconds睡眠余量的毫秒数

返回值：

N/A

例子：

if display_get_timing_method() != tm_sleep
{
display_set_timing_method(tm_sleep);
if display_get_sleep_margin() != 20
    {
    display_set_sleep_margin(20);
    }
}

上面的代码将检查计时方法，然后将其设置为 tm_sleep它将被设置为该值，睡眠裕度将被设置为20。
