display_set_ui_visibility

这个功能可以用来显示或隐藏系统的用户界面。 安卓系统 的系统UI。该函数要求你提供一个或多个系统标志的整数值。当使用一个以上的标志时，这些标志需要用位法的 "or "合并，如下面的例子所示。你可以在这里找到一个安卓系统标志的列表。

语法。

display_set_ui_visibility(flags)。

争论 旗帜
旗帜 要使用的安卓系统标志（作为一个整数值）。

返回。

不适用

例子。

var flags = 1024|4096;
display_set_ui_visibility(flags);

上述代码将使用安卓系统标志 "SYSTEM_UI_FLAG_IMMERSIVE_STICKY "和 "SYSTEM_UI_FLAG_LAYOUT_FULLSCREEN "来设置显示屏的可见性。
