display_set_ui_visibility

此功能可以用来显示或隐藏在系统UI Android而已。该函数要求您提供一个或多个系统标志作为整数值。当使用多个标志时，需要使用按位“或”合并这些标志，如以下示例所示。您可以在此处Android系统标志的列表。

句法：

display_set_ui_visibility(flags);

争论标志
flags Android SYSTEM标志使用（作为整数值）。

返回值：

N/A

例子：

var flags = 1024|4096;
display_set_ui_visibility(flags);

上面的代码将使用Android系统标志“ SYSTEM_UI_FLAG_IMMERSIVE_STICKY”和“ SYSTEM_UI_FLAG_LAYOUT_FULLSCREEN”来设置显示的可见性。
