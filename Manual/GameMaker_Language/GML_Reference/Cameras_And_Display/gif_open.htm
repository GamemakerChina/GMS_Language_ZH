gif_open

使用此功能，您可以创建一个空的GIF格式图像，准备向其中添加数据。调用该函数时，您需要指定GIF的宽度和高度（以像素为单位，并且除了可用内存外，没有大小上限），并且该函数将返回用于标识礼物的唯一ID值。后续函数，如果无法初始化gif（例如，宽度/高度对于可用内存太大），则它将返回-1。您还可以指定一个可选参数来设置GIF的“透明颜色”。这是RGB值（无alpha分量），在将任何颜色添加到该颜色之前，它将清除gif。如果未提供明确的颜色参数，则将使用默认的黑色。

请注意，使用此功能时，您必须调用 gif_save()在打开另一个文件进行录制之前结束GIF的创建（因此对于每个 gif_open()必须有一个伴随 gif_save()）。

句法：

gif_open(width, height, [clear_colour]);

争论描述
width创建的gif的宽度
height创建的gif的高度
[clear_colour] [可选]清除gif的颜色（RGB）

返回值：

Integer

例子：

if save_gif == true
{
if count == 0
    {
    gif_image = gif_open(room_width, room_height);
    }
else if count < 30
    {
    gif_add_surface(gif_image, application_surface, 6/100);
    }
else
    {
    gif_save(gif_image, "GameCapture.gif");
    count = 0;
    save_gif = false;
    }
count++;
}

上面的代码将创建一个从应用程序表面获取30帧的GIF图像文件，然后将其保存。
