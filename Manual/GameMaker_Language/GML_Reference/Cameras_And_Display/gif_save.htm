gif_save

使用此功能，您可以保存GIF动画。您提供GIF索引（由函数返回）gif_open() ）以及用于保存的文件名。请注意， GameMaker Studio 2不会自动附加 .gif文件扩展名，因此如果希望将保存的文件标识为GIF，则应将其作为文件名字符串的一部分。使用通用884调色板将创建的GIF托盘化（有关更多信息， 请参见此处）。

请注意，如果函数成功并且正确保存了GIF，则它将返回0，否则将返回-1。

句法：

gif_save(gif_index, fname);

争论描述
gif_index要保存的gif的ID
fname用于gif的文件名

返回值：

Integer

例子：

if save_gif == true
{
if count == 0
    {
    gif_image = gif_open(room_width, room_height);
    }
else if count < 30
    {
    gif_add_surface(gif_image, application_surface, 6/100);
    }
else
    {
    gif_save(gif_image, "GameCapture.gif");
    count = 0;
    save_gif = false;
    }
count++;
}

上面的代码将创建一个从应用程序表面获取30帧的GIF图像文件，然后将其保存。
