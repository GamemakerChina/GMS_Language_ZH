gif_save_buffer

使用此功能，您可以保存GIF动画。您提供GIF索引（由函数返回）gif_open() ），该函数将返回一个1字节对齐的GIF数据增长缓冲区。请注意，最终的GIF数据将已经使用Universal 884调色板进行了托盘化（有关更多信息， 请参见此处）。

句法：

gif_save_buffer(gif_index);

争论描述
gif_index要保存的gif的ID

返回值：

Integer (Buffer ID)

例子：

if save_gif == true
{
if count == 0
    {
    gif_image = gif_open(room_width, room_height);
    }
else if count < 30
    {
    gif_add_surface(gif_image, application_surface, 6/100);
    }
else
    {
    global.capture_buff = gif_save_buffer(gif_image);
    count = 0;
    save_gif = false;
    }
count++;
}

上面的代码将创建一个从应用程序表面获取30帧的GIF图像文件，然后将其保存到缓冲区中。
