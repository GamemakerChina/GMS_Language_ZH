screen_save

使用此功能，您可以保存当前在游戏窗口中显示的游戏屏幕截图。该文件将保存到游戏的工作目录中，该目录在Windows中位于 <drive>:/Users/YOURUSERNAME/AppData/Local/[Game Name]/在Mac上是 ~/Library/Application Support/[Game Name]/。捕获的图像将是游戏应用程序表面的最终渲染，或者（如果已禁用应用程序表面）它将是整个显示器或窗口的图像。重要的是要注意，该函数旨在在Draw GUI End Event中调用，该事件应在所有平台上提供一致的结果。在任何其他情况下使用它可能无法按预期方式工作，并且可能在不同目标甚至设备上产生不同的结果。如果需要对保存的内容进行更多控制，则应使用该功能 screen_save_part()。

注意：此功能不适用于HTML5目标。

句法：

screen_save(fname);

争论描述
fname屏幕截图的文件名要保存

返回值：

N/A

例子：

if keyboard_check_pressed(ord("S"))
    {
    screen_save(working_directory + "Screens\Screen_" + string(num++) + ".png")
    }

当用户按下“ S”键时，上面的代码将截取游戏的屏幕截图。此屏幕快照将保存到特定目录，并使用变量编号，然后将其递增。
