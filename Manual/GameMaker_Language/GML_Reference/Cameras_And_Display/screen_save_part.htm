screen_save_part

使用此功能，您可以保存游戏部分当前在游戏显示屏中显示的屏幕截图。选择坐标基于显示器（如果不是全屏，则为窗口）的绝对大小，因此（0,0）始终是屏幕（或窗口）的左上角。下图显示了此功能的工作方式：

该文件将保存到游戏的工作目录中。重要的是要注意，该函数旨在在Draw GUI End Event中调用，该事件应在所有平台上提供一致的结果。在任何其他情况下使用它可能无法按预期方式工作，并且可能在不同目标甚至设备上产生不同的结果。

注意：此功能不适用于HTML5目标。

句法：

screen_save_part(fname,x,y,w,h);

争论描述
fname屏幕截图的文件名要保存
x要捕获的屏幕区域的左上角的x坐标
y要捕获的屏幕区域的左上角的y坐标
w屏幕捕获区域的宽度
h屏幕要捕获的区域的高度

返回值：

N/A

例子：

if keyboard_check_pressed(ord("S"))
    {
    screen_save_part("Screens\Screen_" + string(num++) + ".png", 0, 0, 100, 100)
    }

当用户按下“ S”键时，上面的代码截取了游戏屏幕一部分的屏幕截图，该屏幕截图为100像素见方。此屏幕快照将保存到特定的文件夹中，并使用一个变量编号，然后将其递增。
