ds_grid_add_disk

此函数可用于将给定值（实数或字符串）添加到在网格的定义的磁盘区域内找到的单元格的所有值。要添加的值必须与网格单元格中保存的值具有相同的类型，即：您不能将字符串添加为实数，反之亦然，对于字符串，这相当于串联。

句法：

ds_grid_add_disk(index, xm, ym, r, val);

争论描述
index网格的索引。
xm磁盘在网格上的x位置。
ym磁盘在网格上的y位置。
r网格上磁盘的半径。
val要添加到磁盘内单元的值。

返回值：

N/A

例子：

ds_grid_add_disk(grid, 7, 6, 5, 2)

这将对变量“ grid”引用的DS网格的已定义磁盘区域内的单元格中保存的所有值加2。
