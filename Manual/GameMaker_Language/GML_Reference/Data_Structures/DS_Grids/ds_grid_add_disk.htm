ds_grid_add_disk

这个函数可以用来将一个给定的值（实数或字符串）添加到一个网格的定义盘区内的所有单元格的值中。要添加的值必须与网格单元中的值具有相同的类型，即：你不能将一个字符串添加到一个实数，反之亦然。 到实数，反之亦然，对于字符串来说，这相当于连接。

语法。

ds_grid_add_disk（index, xm, ym, r, val）。

争论 描述
指数 网格的索引。
xm 磁盘在网格上的X位置。
ym 磁盘在网格上的y位置。
r 网格上的圆盘半径。
价值 要添加到磁盘中的单元格的值。

返回。

不适用

例子。

ds_grid_add_disk(grid, 7, 6, 5, 2)

这将在变量 "grid "所引用的DS网格的定义磁盘区域内的单元格中的所有数值上加2。
