ds_grid_add_grid_region

此功能可用于将在网格源区域内找到的所有像元的所有值添加到目标网格内的值，如下所示：

注意：您也可以在同一网格上使用此功能，以将网格一个区域中的值添加到另一个区域中存储的值（请参见下面的代码示例）。

句法：

ds_grid_add_grid_region(index, source, x1, y1, x2, y2, xpos, ypos);

争论描述
index目标网格的索引。
source源网格的索引。
x1要从源网格复制的单元格区域的左侧位置。
y1要从源网格复制的单元格区域的顶部位置。
x2要从源网格复制的单元格区域的正确位置。
y2要从源网格复制的单元格区域的底部位置。
xpos目标网格上要向其添加源区域的x位置。
ypos目标网格上要向其添加源区域的y位置。

返回值：

N/A

例子：

ds_grid_add_grid_region(grid, grid, 0, 0, 1, 5, 2, 0)

上面的代码会将变量“ grid”中索引的DS网格的单元区域从（0,0）复制到（1,5），并将它们从同一DS网格的位置（2,0）添加到单元中。
