ds_grid_add_region

此函数可用于将给定值（实数或字符串）添加到在网格的定义区域内找到的所有像元值。要添加的值必须与网格单元格中保存的值具有相同的类型，即：您不能将字符串添加为实数，反之亦然，对于字符串，这相当于串联。

句法：

ds_grid_add_region(index, x1, y1, x2, y2, val);

争论描述
index网格的索引。
x1网格中区域左侧的x位置。
y1网格中区域顶部的y位置。
x2网格中区域右侧的x位置。
y2网格中区域底部的y位置。
val要添加到区域单元格的值。

返回值：

N/A

例子：

ds_grid_add_region(grid, 2, 4, 5, 5, ".")

这将添加“。”变量“ grid”引用的DS网格定义区域内的单元格中保存的所有字符串。
