ds_grid_clear

这个函数可以用来清除一个给定的DS网格到一个特定的值。然后，网格内的所有单元格将包含这个值，它可以是一个实数或一个字符串。下面的图片说明了它是如何工作的。

语法。

ds_grid_clear(index, val)。

争论 描述
指数 这个要清除的网格的索引。
价值 所有网格单元的新值。

返回。

不适用

例子。

ds_grid_resize(global.Grid, room_width / 32, room_height / 32);
ds_grid_clear(global.Grid, -1)

上述代码将调整全局变量 "Grid "中索引的DS网格的大小，然后清空它，使每个单元格的数值为-1。
