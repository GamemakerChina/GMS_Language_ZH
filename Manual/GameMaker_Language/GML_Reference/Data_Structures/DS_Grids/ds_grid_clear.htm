ds_grid_clear

此功能可用于将给定的DS网格清除为特定值。然后，网格内的所有单元格将包含该值，该值可以是实数或字符串。下图说明了它是如何工作的：

句法：

ds_grid_clear(index, val);

争论描述
index此索引的网格要清除。
val所有网格单元的新值。

返回值：

N/A

例子：

ds_grid_resize(global.Grid, room_width / 32, room_height / 32);
ds_grid_clear(global.Grid, -1)

上面的代码将调整在全局变量“ Grid”中索引的DS网格的大小，然后清除它，以使每个单元格都保持值为-1。
