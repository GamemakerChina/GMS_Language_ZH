ds_grid_create

使用此功能，您可以创建具有指定单元格宽度和高度的新DS网格数据结构。此函数返回一个ID，该ID必须在处理此DS网格的所有其他函数中使用。

重要的！创建数据结构时，用于标识它的索引值是一个从0开始的整数值。这意味着不同类型的数据结构可以具有相同的索引值，因此如果有疑问，应在使用ds_exists（）函数之前访问它们。另请注意，索引已被重新使用，因此之后新创建的索引结构可能会使用已破坏的数据结构索引值。

句法：

ds_grid_create(w, h);

争论描述
w要创建的网格的宽度。
h要创建的网格的高度。

返回值：

Real

例子：

mygrid = ds_grid_create(10, 10)

这将创建一个网格，网格高10个单元，宽10个单元。
