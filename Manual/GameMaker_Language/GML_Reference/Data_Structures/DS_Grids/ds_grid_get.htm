ds_grid_get

此函数可用于从给定DS网格内的任何单元格获取值（实数或字符串）。如果将无效的网格坐标传递给该函数，则返回的值将是 undefined并且错误将显示在输出窗口中。

句法：

ds_grid_get(index, x, y);

争论描述
index网格的索引。
x要查找其值的单元格的x位置。
y要查找其值的单元格的y位置。

返回值：

Real or String

例子：

var xx = irandom(ds_grid_width(grid) - 1);
var yy = irandom(ds_grid_height(grid) - 1);
val = ds_grid_get(grid, xx, yy)

上面的代码从变量“ grid”中索引的DS网格中选择一个随机单元，并将其值存储在变量“ val”中。
