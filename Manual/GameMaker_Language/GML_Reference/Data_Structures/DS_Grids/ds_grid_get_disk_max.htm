ds_grid_get_disk_max

此功能可用于查找在网格的定义的磁盘区域内找到的所有单元的最大值，如下图所示：

句法：

ds_grid_get_disk_max(index, xm, ym, r);

争论描述
index网格的索引。
xm磁盘中心在网格中的x位置。
ym磁盘中心在网格中的y位置。
r网格中磁盘的半径。

返回值：

Real or String

例子：

val = ds_grid_get_disk_max(grid, 5, 5, 2)

上面的代码会将变量“ val”设置为在变量“ grid”中索引的DS网格的给定磁盘中包含的最大值。
