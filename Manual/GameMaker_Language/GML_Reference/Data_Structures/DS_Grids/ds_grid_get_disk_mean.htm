ds_grid_get_disk_mean

这个函数可以用来寻找在网格的定义盘区内发现的所有单元格的平均值（所有单元格的值加在一起，然后除以组成盘区的单元格总数），如下图所示。

语法。

ds_grid_get_disk_mean（index, xm, ym, r）。

争论 描述
指数 网格的索引。
xm 磁盘在网格上的X位置。
ym 磁盘在网格上的y位置。
r 网格上的圆盘半径。

返回。

实数或字符串

例子。

val = ds_grid_get_disk_mean(grid, 5, 5, 2)

上述代码将把变量 "val "设置为包含在变量 "grid "中索引的DS网格的给定盘中的平均值。
