ds_grid_get_mean

该函数可用于查找在网格的定义区域内找到的所有像元的平均值（所有像元值相加，然后除以构成该区域的像元总数），如下图所示：

句法：

ds_grid_get_mean(index, x1, y1, x2, y2);

争论描述
index网格的索引。
x1该区域的左单元格列。
y1该区域的顶部单元格行。
x2该区域的右单元格列。
y2该区域的底部单元格行。

返回值：

Real or String

例子：

val = ds_grid_get_mean(grid, 0, 0, 5, 5)

上面的代码会将变量“ val”设置为包含在变量“ grid”中索引的DS网格给定区域内的平均值。
