ds_grid_get_sum

这个函数可以用来把一个网格的定义区域内的所有单元格的所有数值加在一起，如下图所示。

语法。

ds_grid_get_sum（index, x1, y1, x2, y2）。

争论 描述
指数 网格的索引。
x1 区域的左边单元格列。
y1 区域的顶部单元格行。
x2 该区域的右侧单元格列。
y2 该区域的底部单元格行。

返回。

真实的

例子。

val = ds_grid_get_sum(grid, 0, 0, 5, 5)

上面的代码将把变量 "val "设置为包含在变量 "grid "索引的DS网格给定区域内的所有数值之和。
