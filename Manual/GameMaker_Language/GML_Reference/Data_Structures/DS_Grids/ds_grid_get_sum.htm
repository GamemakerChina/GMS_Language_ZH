ds_grid_get_sum

可以使用此函数将在网格的定义区域内找到的所有单元格的所有值相加，如下图所示：

句法：

ds_grid_get_sum(index, x1, y1, x2, y2);

争论描述
index网格的索引。
x1该区域的左单元格列。
y1该区域的顶部单元格行。
x2该区域的右单元格列。
y2该区域的底部单元格行。

返回值：

Real

例子：

val = ds_grid_get_sum(grid, 0, 0, 5, 5)

上面的代码会将变量“ val”设置为在变量“ grid”中索引的DS网格给定区域内包含的所有值的总和。
