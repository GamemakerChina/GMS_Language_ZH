ds_grid_multiply

此函数会将给定网格单元的值乘以指定的数量。

注意：此功能仅适用于实数，不适用于字符串。

句法：

ds_grid_multiply(index, x, y, val);

争论描述
index网格的索引。
x单元格在网格中的x位置。
y单元格在网格中的y位置。
val要与单元格相乘的值。

返回值：

N/A

例子：

ds_grid_multiply(mygrid, 5, 5, 2)

上面的代码会将存储在给定DS网格单元格中的值乘以2。
