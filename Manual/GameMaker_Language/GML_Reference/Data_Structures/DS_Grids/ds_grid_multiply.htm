ds_grid_multiply

这个函数将把一个给定的网格单元的值乘以指定的数量。

注意：这个函数只对实数起作用，对字符串不起作用。

语法。

ds_grid_multiply（index, x, y, val）。

争论 描述
指数 网格的索引。
x 该单元格在网格中的X位置。
y 该单元格在网格中的y位置。
价值 要与该单元格相乘的值。

返回。

不适用

例子。

ds_grid_multiply(mygrid, 5, 5, 2)

上述代码将把存储在给定的DS网格单元的值乘以2。
