ds_grid_multiply_disk

此函数将获取DS网格的给定圆盘形区域中的所有值，并将每个值乘以给定的数量。

注意：此功能仅适用于实数，不适用于字符串。

句法：

ds_grid_multiply_disk(index, xm, ym, r, val);

争论描述
index网格的索引。
xm磁盘在网格上的x位置。
ym磁盘在网格上的y位置。
r网格上磁盘的半径。
val与磁盘中的单元格相乘的值。

返回值：

N/A

例子：

ds_grid_multiply_disk(mygrid, 5, 5, 5, 2)

上面的代码将采用在圆网格区域内找到的所有值，并将每个值乘以2。
