ds_grid_multiply_grid_region

使用此功能，您可以在给定的DS网格中定义一个区域，然后取这些值并将它们与在同一DS网格或另一个（先前已创建）的单独区域中找到的值相乘。原始区域将保持不变，而与之相乘的区域现在将为每个单元格存储新值。

句法：

ds_grid_multiply_grid_region(index, source, x1, y1, x2, y2, xpos, ypos);

争论描述
index目标网格的索引。
source源网格的索引。
x1要从源网格复制的单元格区域的左侧位置。
y1要从源网格复制的单元格区域的顶部位置。
x2要从源网格复制的单元格区域的正确位置。
y2要从源网格复制的单元格区域的底部位置。
xpos目标网格上与源区域相乘的x位置。
ypos目标网格上与源区域相乘的y位置。

返回值：

N/A

例子：

ds_grid_multiply_grid_region(mygrid, mygrid, 0, 0, 5, 5, 0, 0)

这将采用DS网格“ mygrid”中从（0,0）到（5,5）的单元格区域，并将它们与同一DS网格位置（0,0）的单元格相乘。
