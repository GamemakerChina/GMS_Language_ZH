ds_grid_multiply_region

使用此功能，您可以指定网格中的一个区域，在该区域中将每个单元格值乘以给定的数量。

注意：此功能仅适用于实数，不适用于字符串。

句法：

ds_grid_multiply_region(index, x1, y1, x2, y2, val);

争论描述
index网格的索引。
x1网格中区域左侧的x位置。
y1网格中区域顶部的y位置。
x2网格中区域右侧的x位置。
y2网格中区域底部的y位置。
val与区域单元相乘的值。

返回值：

N/A

例子：

ds_grid_multiply_region( mygrid, 5, 5, 10, 10, 2 )

上面的代码将采用在定义的矩形网格区域内找到的所有值，并将每个值乘以2。
