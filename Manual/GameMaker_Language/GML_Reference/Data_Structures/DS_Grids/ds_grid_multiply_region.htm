ds_grid_multiply_region

通过这个函数，你可以指定网格的一个区域，让每个单元格的值乘以一个给定的数量。

注意：这个函数只对实数起作用，对字符串不起作用。

语法。

ds_grid_multiply_region（index, x1, y1, x2, y2, val）。

争论 描述
指数 网格的索引。
x1 该区域在网格中的左边的X位置。
y1 该区域顶部在网格中的y位置。
x2 该区域的右侧在网格中的X位置。
y2 该区域底部在网格中的Y位置。
价值 要与区域单元格相乘的值。

返回。

不适用

例子。

ds_grid_multiply_region( mygrid, 5, 5, 10, 10, 2 )

上面的代码将把在定义的矩形网格区域内发现的所有数值，逐一乘以2。
