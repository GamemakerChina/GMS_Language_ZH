ds_grid_read

该函数可用于转换该函数先前创建的字符串 ds_grid_write()回到DS网格。 DS网格必须先前已创建（请参见下面的示例）。请注意，如果数据结构是使用GameMaker版本创建的，则应添加可选参数“旧版”，并将其设置为 true因为此版本的字符串格式已更改。

句法：

ds_grid_read(index, string [, legacy]);

争论描述
index要读取的网格的索引。
string要读入DS网格的字符串。
legacy ( optional )可以是 true或者 false或完全省略。

返回值：

N/A

例子：

grid = ds_grid_create(room_width div 32, room_height div 32);
ini_open("Save.ini");
ds_grid_read(grid, ini_read_string("Save", "0", ""));
ini_close()

上面的代码根据房间的大小创建一个DS网格（每个32x32正方形像素代表一个网格），然后从ini文件中将先前保存的一组网格数据读取到新的DS网格中。
