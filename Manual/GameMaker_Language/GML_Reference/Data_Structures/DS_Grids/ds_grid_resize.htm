ds_grid_resize

通过这个函数，你可以调整给定的DS网格的大小，使其具有不同的宽度和/或高度。如果网格的大小大于当前的网格，那么新的单元格将有一个0的基值，如果大小小于当前的网格，那么单元格中的值不再是新的大小，将丢失。 将会丢失。所有其他单元格将不被触动。

语法。

ds_grid_resize(index, w, h)。

争论 描述
指数 这个要调整大小的网格的索引。
w 网格的新宽度。
h 网格的新高度。

返回。

不适用

例子。

ds_grid_resize(global.Grid, room_width / 32, room_height / 32);
ds_grid_clear(global.Grid, -1)

上述代码将调整全局变量 "Grid "中索引的DS网格的大小，然后清除它，使每个单元格的数值为-1。
