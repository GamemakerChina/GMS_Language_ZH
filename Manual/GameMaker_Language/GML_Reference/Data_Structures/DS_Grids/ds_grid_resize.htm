ds_grid_resize

使用此功能，您可以调整给定DS网格的大小，使其具有不同的宽度和/或高度。如果网格大小大于当前网格，则新单元格的基本值为0；如果网格大小较小，则保存在不再属于新大小的单元格中的值将丢失。所有其他单元将保持不变。

句法：

ds_grid_resize(index, w, h);

争论描述
index要调整网格大小的索引。
w网格的新宽度。
h网格的新高度。

返回值：

N/A

例子：

ds_grid_resize(global.Grid, room_width / 32, room_height / 32);
ds_grid_clear(global.Grid, -1)

上面的代码将调整在全局变量“ Grid”中索引的DS网格的大小，然后清除它，以使每个单元格都保持值为-1。
