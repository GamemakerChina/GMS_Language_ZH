ds_grid_set

这个函数可以用来将给定的DS网格内的某个单元格设置为任何数值，可以是实数或字符串。下面的图片说明了这一点。

语法。

ds_grid_set(index, x, y, value)。

争论 描述
指数 这个网格的索引。
x 要设置的单元格的X位置。
y 要设置的单元格的y位置。
价值 用来设置单元格的值。

返回。

不适用

例子。

grid = ds_grid_create(5, 5);
var i = 0;
var j = 0;
repeat (ds_grid_width(grid) )
{
repeat (ds_grid_height(grid))
{
ds_grid_set(grid, i, j, irandom(9));
j += 1;
}
j = 0;
i += 1;
}

上面的代码创建了一个网格，并将其索引存储在变量 "grid "中。然后，它用0到9的随机整数来填充这个网格。
