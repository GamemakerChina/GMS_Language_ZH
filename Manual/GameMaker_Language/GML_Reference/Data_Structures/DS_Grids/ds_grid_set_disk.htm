ds_grid_set_disk

使用此功能可以将网格的圆形区域设置为某个值。您需要提供一个起始网格单元（作为x和y轴坐标），以及要设置的磁盘半径和希望设置的磁盘值，如下图所示：

句法：

ds_grid_set_disk(index, xm, ym, r, val);

争论描述
index网格的索引。
xm磁盘在网格上的x位置。
ym磁盘在网格上的y位置。
r网格上磁盘的半径。
val要与磁盘中的单元格一起设置的值。

返回值：

N/A

例子：

ds_grid_set_disk(grid, ds_grid_width(grid) div 2, ds_grid_height(grid) div 2, 5, -4)

上面的代码将在变量“ grid”中索引的DS网格中将半径为5个像元的圆形区域设置为-4。
