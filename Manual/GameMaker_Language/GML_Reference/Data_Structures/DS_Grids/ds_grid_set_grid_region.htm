ds_grid_set_grid_region

这个函数可以用来将一个矩形区域的网格单元的内容从一个（先前定义的）DS网格复制到另一个，也可以用来从同一个网格内复制一个区域。下面的图片说明了这两种方式 这个函数的两种使用方式。

语法。

ds_grid_set_grid_region（index, source, x1, y1, x2, y2, xpos, ypos）。

争论 描述
指数 目标网格的索引。
来源 源网格的索引。
x1 要从源网格复制的单元格区域的左边位置。
y1 要从源网格复制的单元格区域的顶部位置。
x2 要从源网格复制的单元格区域的右边位置。
y2 要从源网格复制的单元格区域的底部位置。
xpos 复制源区域到目标网格上的X位置。
ypos 复制源区域到目标网格上的y位置。

返回。

不适用

例子。

ds_grid_set_grid_region(grid, t_grid, 0, 0, 5, 5, 10, 10)

这将复制变量 "t_grid "中索引的网格中从（0,0）到（5,5）的单元格区域，并将它们复制到变量 "grid "中索引的网格的（10,10）位置。
