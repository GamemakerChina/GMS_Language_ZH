ds_grid_set_grid_region

此功能可用于将网格单元的矩形区域的内容从一个（先前定义的）DS网格复制到另一个，或者可用于从同一网格内复制区域。下图说明了可以使用此功能的两种方式：

句法：

ds_grid_set_grid_region(index, source, x1, y1, x2, y2, xpos, ypos);

争论描述
index目标网格的索引。
source源网格的索引。
x1要从源网格复制的单元格区域的左侧位置。
y1要从源网格复制的单元格区域的顶部位置。
x2要从源网格复制的单元格区域的正确位置。
y2要从源网格复制的单元格区域的底部位置。
xpos目标网格上将源区域复制到的x位置。
ypos将目标区域复制到的目标网格上的y位置。

返回值：

N/A

例子：

ds_grid_set_grid_region(grid, t_grid, 0, 0, 5, 5, 10, 10)

这会将单元格的区域从变量“ t_grid”中索引的网格的（0,0）复制到（5,5），并将它们复制到变量“ grid”中索引的网格中的位置（10,10）。
