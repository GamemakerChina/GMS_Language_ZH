ds_grid_set_region

该函数可用于将给定网格的矩形区域设置为指定值（可以是实数或字符串），如下图所示：

句法：

ds_grid_set_region(index, x1, y1, x2, y2, val);

争论描述
index网格的索引。
x1网格中区域左侧的x位置。
y1网格中区域顶部的y位置。
x2网格中区域右侧的x位置。
y2网格中区域底部的y位置。
val设置区域单元格的值。

返回值：

N/A

例子：

ds_grid_set_region(grid, 5, 5, 10, 10, 99)

这会将在变量“ grid”中索引的网格区域内的所有像元从（5,5）设置为（10,10）到99。
