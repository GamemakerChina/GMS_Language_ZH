ds_grid_set_region

这个函数可以用来将一个给定网格的矩形区域设置为一个指定的值（可以是一个实数或一个字符串），如下图所示。

语法。

ds_grid_set_region（index, x1, y1, x2, y2, val）。

争论 描述
指数 网格的索引。
x1 该区域在网格中的左边的X位置。
y1 该区域顶部在网格中的y位置。
x2 该区域的右侧在网格中的X位置。
y2 该区域底部在网格中的Y位置。
价值 将区域单元格设置为的值。

返回。

不适用

例子。

ds_grid_set_region(grid, 5, 5, 10, 10, 99)

这将把变量 "grid "中索引的从（5,5）到（10,10）的网格区域内的所有单元格设置为99。
