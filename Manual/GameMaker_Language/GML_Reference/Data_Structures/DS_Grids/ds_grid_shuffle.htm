ds_grid_shuffle

此功能可用于随机化网格内所有像元中所有值的位置。

注意： GameMaker Studio 2每次都会生成相同的初始随机种子，因此使调试代码变得容易得多，因此，每次重新运行游戏时，此功能会将网格单元重新排列到相同位置。为了避免这种行为，请使用 randomise()在游戏开始时。这仅在测试和调试游戏时适用，因为最终的可执行程序包不会显示此行为，并且每次播放都是随机的。

句法：

ds_grid_shuffle(index);

争论描述
index要改组的网格的索引。

返回值：

N/A

例子：

ds_grid_shuffle(grid)

这将采用在变量“ grid”中索引的DS网格中的所有值，并对其进行随机排序以获得不同的顺序。
