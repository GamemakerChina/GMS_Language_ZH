ds_grid_sort

这个功能可以用来根据给定列的值对DS网格进行排序（就像你在操作系统的文件资源管理器中按日期、大小等对文件进行排序一样）。下面的图片显示了一个例子。

语法。

ds_grid_sort(index, column, ascending)。

争论 描述
指数 要排序的网格的索引。
栏目 用于对行进行排序的列
升序 是从低到高排序（真），还是从高到低排序（假）。

返回。

不适用

例子。

ds_grid_sort(grid, 3, false)

这将把DS网格中的所有数值以变量 "grid "为索引，并根据在网格的第3列中发现的数值进行排序（如上图所示）。
