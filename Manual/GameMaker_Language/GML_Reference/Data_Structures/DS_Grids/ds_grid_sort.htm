ds_grid_sort

此功能可用于根据给定列中的值对DS网格进行排序（就像您在OS文件浏览器中按日期，大小等对文件进行排序一样）。下图显示了一个示例：

句法：

ds_grid_sort(index, column, ascending);

争论描述
index要排序的网格的索引。
column用于对行进行排序的列
ascending是否从最低到最高排序（true ），或从最高到最低（false ）。

返回值：

N/A

例子：

ds_grid_sort(grid, 3, false)

这将采用在变量“ grid”中索引的DS网格中的所有值，并根据在网格第3列中找到的值对它们进行排序（如上图所示）。
