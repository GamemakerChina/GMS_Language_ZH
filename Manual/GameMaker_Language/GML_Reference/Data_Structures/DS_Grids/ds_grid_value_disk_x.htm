ds_grid_value_disk_x

通过这个函数，你可以得到被搜索的值的X坐标（在给定的网格盘状区域内）。你给出DS网格的索引（如由 ds_grid_create()返回的），以及圆盘中心单元的x/y位置 盘的中心单元。然后，在提供要搜索的值之前，你给出要搜索的中心单元周围的半径（整数值）。如果被搜索的值不存在，那么该函数将返回-1。

语法。

ds_grid_value_disk_x（index, xm, ym, r, val）。

争论 描述
指数 网格的索引。
xm 磁盘在网格上的X位置。
ym 磁盘在网格上的y位置。
r 网格上的圆盘半径。
价值 要找到的价值。

返回。

真实的

例子。

if ds_grid_value_disk_exists(grid, 5, 5, 5, val)
{
xpos = ds_grid_value_disk_x(grid, 5, 5, 5, val);
ypos = ds_grid_value_disk_y(grid, 5, 5, 5, val);
}

上面的代码在磁盘区域内检查ds_grid的特定值。如果找到了，它就将该值的x和y位置存储在两个变量中，供以后使用。
