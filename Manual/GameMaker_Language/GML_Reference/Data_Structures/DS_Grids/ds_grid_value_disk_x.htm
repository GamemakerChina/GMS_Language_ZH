ds_grid_value_disk_x

使用此功能，您可以获取要搜索的值的x坐标（在给定的网格盘形区域内）。您给出DS网格索引（由 ds_grid_create()）以及磁盘中心单元的x / y位置。然后，在提供要搜索的值之前，先给出要搜索的中心像元周围的半径（作为整数值）。如果要搜索的值不存在，则该函数将返回-1。

句法：

ds_grid_value_disk_x(index, xm, ym, r, val);

争论描述
index网格的索引。
xm磁盘在网格上的x位置。
ym磁盘在网格上的y位置。
r网格上磁盘的半径。
val寻找的价值。

返回值：

Real

例子：

if ds_grid_value_disk_exists(grid, 5, 5, 5, val)
   {
   xpos = ds_grid_value_disk_x(grid, 5, 5, 5, val);
   ypos = ds_grid_value_disk_y(grid, 5, 5, 5, val);
   }

上面的代码检查ds_grid中磁盘区域内的特定值。如果找到它，则将值的x和y位置存储在两个变量中，以备后用。
