ds_grid_value_exists

使用此功能，您可以检查给定DS网格的矩形区域内是否存在特定值（实数或字符串）。如果存在，函数将返回 true否则它将返回 false。

句法：

ds_grid_value_exists(index, x1, y1, x2, y2, val);

争论描述
index网格的索引。
x1网格中区域左侧的x位置。
y1网格中区域顶部的y位置。
x2网格中区域右侧的x位置。
y2网格中区域底部的y位置。
val寻找的价值。

返回值：

Boolean

例子：

if ds_grid_value_exists(grid, 0, 1, 5, 6, val)
   {
   xpos = ds_grid_value_x(grid, 0, 1, 5, 6, val);
   ypos = ds_grid_value_y(grid, 0, 1, 5, 6, val);
   }

上面的代码检查一个矩形区域内特定值的DS网格。如果找到它，则将值的x和y位置存储在两个变量中，以备后用。
