ds_grid_value_x

使用此功能，您可以获取要搜索的值的x坐标（在给定的矩形网格区域内）。您给出DS网格索引（由 ds_grid_create()）以及要搜索的网格区域的左上角和右下角的x / y位置，然后再提供要搜索的值。如果要搜索的值不存在，则该函数将返回-1。

句法：

ds_grid_value_x(index, x1, y1, x2, y2, val);

争论描述
index网格的索引。
x1网格中区域左侧的x位置，从0到（网格宽度-1）。
y1网格中区域顶部的y位置，从0到（网格高度-1）。
x2网格中区域右侧的x位置，从0到（网格宽度-1）。
y2网格中区域底部的y位置，从0到（网格高度-1）。
val寻找的价值。

返回值：

Real

例子：

if ds_grid_value_exists(grid, 0, 1, 5, 6, val)
   {
   xpos = ds_grid_value_x(grid, 0, 1, 5, 6, val);
   ypos = ds_grid_value_y(grid, 0, 1, 5, 6, val);
   }

上面的代码检查ds_grid中矩形区域内的特定值。如果找到它，则将值的x和y位置存储在两个变量中，以备后用。
