ds_grid_width

此函数将返回给定网格的宽度。该值是网格沿x轴具有的像元数，并且始终是整数，如下图所示：

句法：

ds_grid_width(index);

争论描述
index这个索引找到网格的宽度。

返回值：

Real

例子：

for (var i = 0; i < ds_grid_width(grid); ++i;)
   {
   for (var j = 0; j < ds_grid_height(grid); ++j;)
      {
      if (ds_grid_get(grid, i, j) == 1)
         {
         instance_create_layer(i * 32, j * 32, "Walls", obj_Wall);
         }
      }
   }

上面的代码将循环遍历在变量“ grid”中索引的DS网格，如果在任何特定单元格中找到的值等于1，则它将在房间内的适当位置创建“ obj_Wall”的实例。
