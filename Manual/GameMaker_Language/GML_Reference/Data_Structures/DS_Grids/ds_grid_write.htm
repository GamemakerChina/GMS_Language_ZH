ds_grid_write

这个函数可以用来将给定的ds_grid转换为一个字符串，然后可以将其存储在一个外部文件中（例如）。你可以使用函数将这个函数返回的字符串读回ds_grid中去。 ds_grid_read().

注意： 返回的字符串不是人类可读的字符串，而是一个数据结构内容的转储。

语法。

ds_grid_write(index)。

争论 描述
指数 要写的网格的索引。

返回。

字符串

例子。

ini_open("Save.ini");
ini_write_string("Save", "0", ds_grid_write(mygrid));
ini_close()

上面的代码将打开一个ini文件（如果它不存在，就创建它），然后把给定的ds_grid作为一个字符串写入该文件中。
