ds_list_add

此函数可用于将新值（实数或字符串）添加到列表中，并将最后添加。该函数可以使用其他可选参数（根据需要任意多个），从而允许您在单个调用中将多个值连续添加到列表中。

句法：

ds_list_add(id, val1 [, val2, ... max_val]);

争论描述
id要添加到的列表的ID。
val要添加到列表中的值。
[val2,... max_val]要添加到列表的可选值。

返回值：

N/A

例子：

ds_list_add(sc_list, score);

上面的代码会将存储在“ score”变量中的值添加到在变量“ sc_list”中索引的列表中。

返回： DS列表
