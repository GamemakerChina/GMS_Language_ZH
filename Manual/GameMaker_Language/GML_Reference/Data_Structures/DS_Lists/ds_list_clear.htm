ds_list_clear

使用此功能，您可以清除给定列表数据结构中的所有数据。这不破坏数据结构（你应该使用 ds_list_destroy()）只会清除其中的所有数据，并使列表为空（大小为零）。请注意，清除列表将取消引用存储在其中的任何数据结构，从而导致内存泄漏，因此您需要在清除列表之前先遍历该列表并手动销毁所有数据结构项，以防止出现这种情况。唯一不需要这样做的时间是将列表中的任何项目标记为另一个DS列表或DS映射时，在这种情况下，这些项目将被销毁（不清除！），并且当父项被删除时它们的内存会自动清除。已清除。

句法：

ds_list_clear(id);

争论描述
id要清除的数据结构的ID。

返回值：

N/A

例子：

if (count == 15) && (!ds_list_empty(command_list))
    {
    ds_list_clear(command_list);
    alarm[0] = room_speed;
    ai_count = 0;
    }

上面的代码检查变量是否已达到特定值，并清除变量“ command_list”中索引的DS列表，设置警报，并将变量重置为0。

返回： DS列表
