ds_list_copy

使用此功能，您可以将一个列表的内容复制到另一个列表中。这两个列表必须先前已创建，并且如果要复制到的列表中已包含信息，则将首先清除此列表。最终结果是两个包含相同信息的独立列表。

句法：

ds_list_copy( id, source );

争论描述
id要复制到的列表的ID。
source要从中复制列表的ID。

返回值：

N/A

例子：

if !ds_list_empty(main_list)
    {
    old_list = ds_list_create();
    ds_list_copy(old_list, main_list);
    ds_list_clear(main_list);
    }

上面的代码将检查DS列表以查看其是否为空。如果不为空，则将其复制到另一个DS列表（先前已创建）中，然后清除原始列表。

返回： DS列表
