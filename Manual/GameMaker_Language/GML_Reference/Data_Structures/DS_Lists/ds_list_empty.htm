ds_list_empty

使用此功能，您可以检查给定的DS列表以查看其是否为空（返回 true）否（返回 false）。

句法：

ds_list_empty(id);

争论描述
id要检查的数据结构的ID。

返回值：

Boolean

例子：

if (count == 15( && (!ds_list_empty(command_list))
    {
    ds_list_clear(command_list);
    alarm[0] = room_speed;
    count = 0;
    }

上面的代码检查变量是否已达到特定值，并清除变量“ command_list”中索引的DS列表，设置警报，并将变量重置为0。

返回： DS列表
