ds_list_find_index

使用此功能，您可以检查给定列表中的某个值，并将返回该值在列表中的位置。请注意，如果列表中有多个条目具有相同的值，则可能会返回其中任何一个的位置，如果该值不存在，则将返回-1。请注意，该值可以是一个数组，您可以使用函数进行检查 is_array()。

句法：

ds_list_find_index(id, val);

争论描述
id要使用的列表的ID。
val寻找的价值。

返回值：

Real

例子：

pos = ds_list_find_index(list, "Player1");

上面的代码检查变量“ list”中索引的列表的值“ Player1”，并将返回的位置存储在变量“ pos”中。

返回： DS列表
