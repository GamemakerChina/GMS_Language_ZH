ds_list_find_value

使用此功能，您可以检查给定的列表位置，并将返回列表中该位置所保存的值。请注意，如果您提供的位置超出给定列表大小（即：10值列表中的位置11），则该函数可能会返回 undefined或0。这是因为在创建列表时，内部将列表中的前几个条目设置为0，以最大程度地减少最初向列表中添加项目时的性能问题（尽管 ds_list_size()函数仍将在新创建的列表上返回0）。如果您希望确保列表在创建时“真正”为空，则应致电 ds_list_clear()创建列表后，这意味着为未填充的列表位置返回的所有值都将是 undefined，然后可以使用 is_undefined()功能。

句法：

ds_list_find_value(id, pos);

争论描述
id要使用的列表的ID。
pos要查看的位置，其中0对应于列表的最开始，而最终位置为ds_list_size（id）-1。

返回值：

Real, String or Undefined

例子：

val = ds_list_find_value(list, ds_list_size(list) - 1);

上面的代码检查列表中最后位置变量“ list”中索引的列表，并将返回值存储在变量“ val”中。
