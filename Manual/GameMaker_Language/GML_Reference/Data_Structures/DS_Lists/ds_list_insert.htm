ds_list_insert

该函数将给定值添加到给定位置的列表中。如果列表在给定位置之后包含更多值，则它们的位置将上移一个以腾出空间，使列表大一。

句法：

ds_list_insert(id, pos, val);

争论描述
id要添加到的列表的ID。
pos要添加值的位置，其中0对应于列表的最开始，而最终位置是ds_list_size（id）-1。
val要添加到列表中的值。

返回值：

N/A

例子：

ds_list_insert(list, 9, score);

上面的代码会将变量“ score”中存储的值添加到变量“ list”索引的列表的第10位（列表从0开始计数，因此十个值的列表从0到9编号）。

返回： DS列表
