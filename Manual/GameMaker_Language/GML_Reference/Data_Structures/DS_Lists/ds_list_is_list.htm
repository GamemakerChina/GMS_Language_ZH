ds_list_is_list

通过这个函数，你可以检查在一个DS列表中的给定位置是否存储有另一个DS列表。如果给定的位置包含一个DS列表的ID，那么这个函数将返回true，否则将返回false。

语法。

ds_list_is_list(id, pos)。

争论 描述
身份证 要检查的列表的ID。
姿势 要检查的列表中的位置。

返回。

布尔型

例子。

var size = ds_list_size(ships) ;
for (var i = 0; i < size; i++;)
{
if ds_list_is_list(ships, i)
{
ds_list_destroy(i);
}
}
ds_list_destroy(ships)。

上面的代码在一个DS列表中循环，并检查是否有条目包含其他列表的ID。如果有，就销毁这些列表，然后在循环结束后销毁主列表。

返回。DS列表
