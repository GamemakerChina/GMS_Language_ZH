ds_list_is_map

使用此功能，您可以检查DS地图是否存储在DS列表中的给定位置。如果给定位置包含DS地图ID，则该函数将返回 true否则它将返回 false。

句法：

ds_list_is_map(id, pos);

争论描述
id要检查的列表的ID。
pos列表中要检查的位置。

返回值：

Boolean

例子：

var size = ds_list_size(ships) ;
for (var i = 0; i < size; i++;)
    {
    if ds_list_is_map(ships, i)
        {
        ds_map_destroy(i);
        }
    }
ds_list_destroy(ships);

上面的代码遍历了DS列表，并检查是否有任何条目包含地图ID。如果它们这样做，则将销毁这些映射，然后在循环结束后销毁主列表。

返回： DS列表
