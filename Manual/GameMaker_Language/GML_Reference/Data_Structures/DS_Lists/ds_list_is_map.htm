ds_list_is_map

通过这个函数，你可以检查DS列表中的给定位置是否存储了一张DS地图。如果给定的位置包含一个DS地图ID，那么该函数将返回true，否则它将返回 false。

语法。

ds_list_is_map(id, pos)。

争论 描述
身份证 要检查的列表的ID。
姿势 要检查的列表中的位置。

返回。

布尔型

例子。

var size = ds_list_size(ships) ;
for (var i = 0; i < size; i++;)
{
if ds_list_is_map(ships, i)
{
ds_map_destroy(i)。
}
}
ds_list_destroy(ships)。

上面的代码在DS列表中循环，检查是否有条目包含地图ID。如果有，那么这些地图就会被销毁，然后主列表在循环结束后被销毁。

返回。DS列表
