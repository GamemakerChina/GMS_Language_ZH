ds_list_mark_as_map

此功能将“标记”（或“标记”）先前创建的DS列表中的给定位置以保存DS地图。编码JSON字符串时可以使用此功能（可以使用 json_encode()），但在嵌套数据结构时也很有用，因为当父DS列表被销毁时，以此方式标记的项目将自动进行垃圾回收（销毁）。这意味着您在销毁“父”清单之前不必手动浏览清单内容并单独销毁标记的数据结构。但是，如果您分别删除列表位置，则其中包含的数据结构将不会被垃圾回收，并且在删除父列表位置之前，应调用相应的DS地图销毁函数。另请注意，如果您调用该函数 ds_list_clear()在清单上，清除清单后，所有标记为地图的物品也将被销毁。

还值得注意的是，如果已标记的位置在列表内更改，则“标记”将随之移动，因此，如果您在列表中标记了位置3（例如），然后在其下插入2个其他项目，它会移动到位置5，仍会标记为地图。

句法：

ds_list_mark_as_map(id, pos);

争论描述
id要标记的列表的ID。
pos列表中要标记的位置。

返回值：

Real (integer - ID of the DS map if successful or -1 if it fails)

例子：

var sub_map = ds_map_create();
ds_map_add(sub_map, "player", player_array);
ds_map_add(sub_map, "enemy", enemy_array);
ds_list_add(j_list, sub_map);
ds_list_mark_as_map(j_list, 0);

上面的代码创建一个DS映射，然后用两个键填充每个键，每个键包含一个值数组。然后将此地图添加到给定的DS列表中，并在此位置“标记”，以便以后可以正确编码。

返回： DS列表
