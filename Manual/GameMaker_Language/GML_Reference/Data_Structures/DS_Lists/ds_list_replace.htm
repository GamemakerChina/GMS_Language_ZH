ds_list_replace

这个函数将用另一个值替换给定位置的值。

语法。

ds_list_replace(id, pos, val)。

争论 描述
身份证 要改变的列表的ID。
姿势 替换值的位置，其中0对应于列表的最开始，最终位置是ds_list_size(id)-1。
价值 替换给定值的新值。

返回。

不适用

例子。

ds_list_replace(n_list, 3, name)。

上述代码将把存储在列表中第3位的数值替换为存储在变量 "name "中的数值。

返回。DS列表
