ds_list_set

此功能可用于设置先前添加的列表条目。您提供列表ID（创建列表时返回的列表），列表中要设置的位置以及要设置的值。请注意，如果要设置的条目超出列表的范围（即，您设置了列表条目20，但当前列表仅包含10个条目），则列表将被填充到给定位置，并且每个条目将被设置为0。此功能与使用DS列表访问器的功能相同。

句法：

ds_list_set(id, pos, val);

争论描述
id要添加到的列表的ID。
pos列表中要设置的位置。
[val2,... max_val]在列表中设置的值。

返回值：

N/A

例子：

for (var i = 0; i < ds_list_size(list); i++;)
    {
    ds_list_set(list, i, -1);
    }

上面的代码会将存储在“ score”变量中的值添加到在变量“ sc_list”中索引的列表中。

返回： DS列表
