ds_list_shuffle

使用此功能，您可以重新整理列表，这会将所有组件值从最初添加到列表的位置重新排列到随机位置。

注意GameMaker Studio 2每次都会生成相同的初始随机种子，因此使调试代码变得容易得多，因此，每次重新运行游戏时，此功能会将列表项重新排列到相同位置。为了避免这种行为，请在游戏开始时使用randomise（）。这仅在测试和调试游戏时适用，因为最终的可执行程序包不会显示此行为，并且每次播放都是随机的。

句法：

ds_list_shuffle(id);

争论描述
id要随机播放的列表的ID。

返回值：

N/A

例子：

if restart
    {
    ds_list_shuffle(card_list);
    }

如果将变量“ restart”标记为，则上面的代码将对在变量“ card_list”中索引的列表进行混洗 true。

返回： DS列表
