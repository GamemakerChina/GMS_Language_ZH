ds_list_sort

使用此功能，您可以按升序或降序对列表中的所有值进行排序。如果列表中包含字符串，则将根据英语26字母字母表将它们按字母顺序排序。

句法：

ds_list_sort(id, ascend);

争论描述
id要排序的列表的ID。
ascend值是否应递增（true ）或降序（false ） 命令。

返回值：

N/A

例子：

if newgame
    {
    ds_list_sort(name_list, true);
    }

如果将变量“ newgame”标记为以下代码，则上面的代码将对变量“ name_list”中索引的列表进行排序 true。

返回： DS列表
