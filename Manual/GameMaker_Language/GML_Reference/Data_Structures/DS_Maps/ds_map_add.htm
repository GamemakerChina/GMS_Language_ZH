ds_map_add

此函数应用于将键/值对的集合添加到指定的DS映射中。您可以检查此功能以查看其是否成功（它将返回 true成功或 false否则），因为如果DS映射中已经存在相同的键，或者您指定了不存在的DS映射作为要添加到的映射的ID，则此操作可能会失败。您提供的键和和值可以由数据类型的任意组合组成，因此可以接受以下所有（以及更多）（尽管实际上，您通常会使用字符串作为键）：

ds_map_add(map, 5, undefined);
ds_map_add(map, "level", 1);
ds_map_add(map, 89.6, "hello world");
ds_map_add(map, 5, buffer_get_address(buff));

您也可以使用访问器“? “， 如下所示：

map[? 5] = undefined;
map[? "level"] = 1;
map[? 89.6] = "hello world";
map[? 5] = buffer_get_address(buff);

注意GameMaker Studio 2其他数据结构不同，此键不会转到DS地图的开头（也不是结尾），而只会进入DS地图的某个地方。

句法：

ds_map_add(id, key, val);

争论描述
id要添加到的地图的ID。
key要添加的值的键。
val要添加到地图的值。

返回值：

Boolean

例子：

inventory = ds_map_create();
ds_map_add(inventory, "hp potion", 1);
ds_map_add(inventory, "gold", 100);

这将创建一个新地图，将其ID分配给变量“库存”。然后，它将两个新键添加到地图上：“ hp potion”和“ gold”，并将其初始值设置为1和100。
