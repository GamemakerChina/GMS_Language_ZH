ds_map_add

这个函数应该被用来将键/值对的集合添加到指定的DS地图中。你可以检查这个函数是否成功（成功时将返回true，否则返回false）。 因为如果DS地图中已经存在相同的键，或者你指定了一个不存在的DS地图作为要添加的地图的ID，它可能会失败。你提供的键和值可以由任何数据类型的组合组成，所以以下所有的--以及更多的--都可以接受。 都是可以接受的（尽管在实践中，你最常使用一个字符串作为键）。

ds_map_add(map, 5, undefined);
ds_map_add(map, "level", 1);
ds_map_add(map, 89.6, "hello world");
ds_map_add(map, 5, buffer_get_address(buff)) 。

你也可以使用访问器"？"添加到地图中，如下图所示。

map[? 5] = undefined;
map[? "level"] = 1;
map[? 89.6] = "hello world";
map[? 5] = buffer_get_address(buff) 。

注意：与GameMaker Studio 2的其他数据结构不同，这个键不会进入DS地图的起点（也不会进入终点），而是直接进入DS地图的某个地方。

语法。

ds_map_add(id, key, val)。

争论 描述
身份证 要加入的地图的ID。
钥匙 要添加的值的键。
价值 要添加到地图上的值。

返回。

布尔型

例子。

inventory = ds_map_create();
ds_map_add(invent, "hp potion", 1);
ds_map_add(invent, "gold", 100);

这将创建一个新的地图，把它的ID分配给变量 "库存"。然后，它为地图添加两个新的键，"HP药水 "和 "黄金"，并将它们的初始值设置为1和100。

返回。DS地图
