ds_map_add_list

使用此功能，您可以将（先前创建的） DS列表分配给给定DS映射中的键。此功能旨在创建JSON兼容映射，然后使用对其进行编码 json_encode()并且仅应与该功能结合使用。如果DS映射以这种方式添加了列表，则销毁父映射也会销毁所包含的列表并释放其内存，然后调用 ds_map_clear()父地图上的也会删除并清除所有标记的列表。

句法：

ds_map_add_list(id, key, value)

争论描述
id要使用的地图的ID。
key添加列表的键。
value要添加的列表的ID。

返回值：

N/A

例子：

var j_list = ds_list_create();
ds_list_add(j_list, health);
ds_list_add(j_list, lives);
ds_list_add(j_list, score);
var j_map = ds_map_create();
ds_map_add_list(j_map, "list", j_list);
var j = json_encode(j_map);
ds_map_destroy(j_map);

上面的代码将创建一个列表，并使用全局变量的各种值填充它。然后将此列表“嵌套”在DS映射中，然后在销毁该映射，将其及其包含的任何列表从内存中删除之前，将该映射编码为JSON字符串。
