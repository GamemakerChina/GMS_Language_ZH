ds_map_add_map

使用此功能，您可以将（先前创建的）DS映射分配给给定DS映射内的键。此功能旨在创建JSON兼容映射，然后使用对其进行编码 json_encode()并且仅应与该功能结合使用。如果DS映射以这种方式添加了另一个映射，则销毁父映射也会销毁所包含的映射并释放其内存，然后调用 ds_map_clear()父地图上的也会破坏并清理所有标记的地图。

句法：

ds_map_add_map(id, key, value)

争论描述
id要使用的地图的ID。
key添加的地图的关键。
value要添加的地图的ID。

返回值：

N/A

例子：

var j_map = ds_map_create();
var j_list = ds_list_create();
var sub_map = ds_map_create();
ds_map_add_list(sub_map, "list", j_list);
ds_map_add(sub_map, "array", j_array);
ds_map_add_map(j_map, "map", sub_map);
var j = json_encode(j_map);
ds_map_destroy(j_map);

上面的代码将创建两个DS映射，然后在将其中一个列表和一个数组填充之前，将其添加到第二个DS映射中，然后将其编码为JSON字符串。然后销毁该映射以从内存中删除该映射及其包含的任何其他映射或列表。
