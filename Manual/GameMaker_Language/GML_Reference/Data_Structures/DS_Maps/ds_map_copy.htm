ds_map_copy

您可以使用此功能将一张地图的内容复制到您先前使用创建的另一张地图中 ds_map_create()。如果要复制到的DS映射不为空，则此功能将在复制之前先将其清除。通过此过程，原始DS映射保持不变。

句法：

ds_map_copy(id, source);

争论描述
id您要复制到的地图的ID
source您是从复制图的ID

返回值：

N/A

例子：

inventory_2 = ds_map_create();
ds_map_copy(inventory_2, inventory_1);

上面的代码将创建一个新地图，并将其分配给变量“ inventory_2”。然后，它将在变量“ inventory_1”中索引的DS地图的内容复制到此新地图。
