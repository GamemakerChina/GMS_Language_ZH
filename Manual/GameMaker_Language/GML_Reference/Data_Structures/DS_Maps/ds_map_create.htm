ds_map_create

此函数用于创建一个新的空DS映射，并将返回其ID，然后将其ID用于访问所有其他DS映射函数中的数据结构。

重要的！创建数据结构时，用于标识它的索引值是一个从0开始的整数值。这意味着不同类型的数据结构可以具有相同的索引值，因此如果有疑问，应在使用ds_exists（）函数之前访问它们。另请注意，索引已被重新使用，因此之后新创建的索引结构可能会使用已破坏的数据结构索引值。

句法：

ds_map_create();

返回值：

Integer (DS map ID)

例子：

inventory = ds_map_create();

上面的代码将创建一个新的空DS映射，并将其ID索引存储在变量“库存”中。
