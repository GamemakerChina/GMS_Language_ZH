ds_map_exists

该函数将返回 true如果指定的密钥存在于（先前创建的） DS映射中，并且 false如果没有。

句法：

ds_map_exists(id, key);

争论描述
id要检查的数据结构的ID
key检查的钥匙

返回值：

Boolean

例子：

if !ds_map_exists(inventory, "potions")
   {
   ds_map_add(inventory, "potions", 1);
   }

上面的代码将检查在变量“库存”中索引的DS地图中是否存在键“药水”，如果不存在，则会将其添加到地图中。
