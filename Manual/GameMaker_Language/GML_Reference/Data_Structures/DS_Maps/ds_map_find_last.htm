ds_map_find_last

此函数返回存储在DS映射中的最后一个键。这不是添加顺序中的最后一个键！ DS映射不是以线性形式存储的，因为它使用DS列表，所以所有这些操作就是找到计算机存储的最后一个键。如果您必须遍历DS映射以查找某些内容，这将很有用，但应尽可能避免这样做，因为它可能会很慢。

句法：

ds_map_find_last(id);

争论描述
id要使用的地图的ID。

返回值：

Real or String

例子：

var size = ds_map_size(inventory);
var key = ds_map_find_last(inventory);
for (var i = size; i > 0; i--;)
    {
    if (key != "gold")
        {
        key = ds_map_find_previous(inventory, key);
        }
    else break;
    }

上面的代码创建了一些临时变量，然后获取DS地图大小，并找到计算机在地图中存储的最后一个键。然后使用 for循环以遍历DS映射以查找键值“ gold”。如果找到它，它将打破循环。

背面： DS地图
