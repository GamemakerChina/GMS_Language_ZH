ds_map_find_next

此函数返回该函数中指定的键之后的下一个存储在DS映射中的键。如果您必须遍历DS映射以查找某些内容，这将很有用，但应尽可能避免这样做，因为它可能会很慢。如果不存在这样的键，则该函数将返回 <undefined>。您应该始终使用 is_undefined()功能。

句法：

ds_map_find_next(id, key);

争论描述
id要使用的地图的ID。
key找到下一个的关键。

返回值：

Real, string or undefined

例子：

var size = ds_map_size(inventory);
var key = ds_map_find_first(inventory);
for (var i = 0; i < size; i++;)
    {
    if (key != "gold")
        {
        key = ds_map_find_next(inventory, key);
        }
    else break;
    }

上面的代码创建了一些临时变量，然后获取DS地图大小，并找到计算机在地图中存储的第一个键。然后使用 for循环遍历DS映射以查找键值“ gold”。如果找到它，它将打破循环。
