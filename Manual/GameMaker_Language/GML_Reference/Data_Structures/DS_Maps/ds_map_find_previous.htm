ds_map_find_previous

这个函数返回存储在DS地图中的前一个键，即函数中指定的那个键。如果你必须在DS地图中反复寻找，这可能很有用，但应尽可能避免，因为它可能很慢。如果 不存在这样的键，那么该函数将返回未定义。你应该总是用 is_undefined()函数来检查。

语法。

ds_map_find_previous(id, key)。

争论 描述
身份证 要使用的地图的ID。
钥匙 关键是要找到前一个到。

返回。

实数、字符串或未定义

例子。

var size = ds_map_size(inventory) - 1;
var key = ds_map_find_last(inventory);
for (var i = size; i > 0; i--; )
{
if (key != "gold")
{
key = ds_map_find_previous(inventory, key);
}
else break;
}

上面的代码创建了一些临时变量，然后得到ds_map的大小，并找到计算机存储在map中的最后一个键。然后，它使用for循环遍历ds_map，寻找键值 "gold"。 如果它找到了，它就跳出循环。

返回。DS地图
