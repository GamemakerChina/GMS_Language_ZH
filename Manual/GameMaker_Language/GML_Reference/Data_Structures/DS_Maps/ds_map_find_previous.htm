ds_map_find_previous

该函数返回存储在DS映射中的前一个键，该键在该功能中指定的键之前。如果您必须遍历DS映射以查找某些内容，这将很有用，但应尽可能避免这样做，因为它可能会很慢。如果不存在这样的键，则该函数将返回 <undefined>。您应该始终使用 is_undefined()功能。

句法：

ds_map_find_previous(id, key);

争论描述
id要使用的地图的ID。
key找到上一个的关键。

返回值：

Real, string or undefined

例子：

var size = ds_map_size(inventory) - 1;
var key = ds_map_find_last(inventory);
for (var i = size; i > 0; i--;)
    {
    if (key != "gold")
        {
        key = ds_map_find_previous(inventory, key);
        }
    else break;
    }

上面的代码创建了一些临时变量，然后获取ds_map的大小，并找到计算机在地图中存储的最后一个键。然后使用 for循环遍历ds_map以查找键值“ gold”。如果找到它，它将打破循环。
