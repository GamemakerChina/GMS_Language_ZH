ds_map_find_value

通过这个函数，你可以从一个指定的键中获取值。该函数的输入值是要使用的（先前创建的）DS地图和要检查的键。

注意：如果没有这样的键存在，那么该函数将返回未定义。你应该总是使用 is_undefined() 函数

语法。

ds_map_find_value(id, key)

争论 描述
身份证 要使用的地图的ID。
钥匙 找到的关键。

返回。

实数、字符串或未定义

例子。

amount = ds_map_find_value(inventory, "food");

或者，使用地图访问器"？"。

数量=库存[? "食物"]。

上面的代码将获得键 "食物 "的值，并将其存储在变量 "金额 "中。

返回。DS地图
