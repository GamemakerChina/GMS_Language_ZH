ds_map_find_value

使用此功能，您可以从指定的键中获取值。函数的输入值是要使用的（先前创建的）DS映射和要检查的键。

注意：如果不存在这样的键，则该函数将返回 <undefined>。您应该始终使用is_undefined（）函数进行检查

句法：

ds_map_find_value(id, key)

争论描述
id要使用的地图的ID。
key找到的关键。

返回值：

Real, string or undefined

例子：

amount = ds_map_find_value(inventory, "food");

或者，使用地图访问器“？”：

amount = inventory[? "food"];

上面的代码将获取键“ food”的值并将其存储在变量“ amount”中。
