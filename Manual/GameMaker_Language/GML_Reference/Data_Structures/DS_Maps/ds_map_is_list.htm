ds_map_is_list

使用此功能，您可以检查是否在给定的地图密钥中存储了DS列表。如果给定键包含DS列表ID，则该函数将返回 true否则它将返回 false。

句法：

ds_map_is_list(id, key)

争论描述
id要使用的ds_map的ID。
key检查的关键。

返回值：

Boolean

例子：

var size = ds_map_size(inventory) ;
var key = ds_map_find_first(inventory);
for (var i = 0; i < size; i++;)
    {
    if ds_map_is_list(inventory, key)
        {
        ds_list_destroy(key);
        }
    key = ds_map_find_next(inventory);
    }
ds_map_destroy(inventory);

上面的代码遍历了DS映射，并检查其中是否有用于DS列表的键。如果是，则销毁DS列表，并在循环结束时销毁DS映射。
