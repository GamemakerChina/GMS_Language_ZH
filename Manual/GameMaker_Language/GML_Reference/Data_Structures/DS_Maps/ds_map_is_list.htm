ds_map_is_list

通过这个函数，你可以检查是否有一个DS列表存储在给定的地图键中。如果给定的键包含一个DS列表的ID，那么这个函数将返回true，否则将返回false。

语法。

ds_map_is_list(id, key)

争论 描述
身份证 要使用的ds_map的id。
钥匙 检查的关键。

返回。

布尔型

例子。

var size = ds_map_size(inventory) ;
var key = ds_map_find_first(inventory);
for (var i = 0; i < size; i++;)
{
if ds_map_is_list(inventory, key)
{
ds_list_destroy(key)。
}
key = ds_map_find_next(inventory);
}
ds_map_destroy(inventory)。

上面的代码在一个DS地图中循环，并检查其中是否有键是用于DS列表的。如果是，那么DS列表就会被销毁，并且在循环结束时，DS地图也会被销毁。

返回。DS地图
