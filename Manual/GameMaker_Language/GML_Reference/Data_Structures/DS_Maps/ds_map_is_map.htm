ds_map_is_map

使用此功能，您可以检查DS映射是否存储在给定的映射密钥中。如果给定键包含DS映射ID，则该函数将返回 true否则它将返回 false。

句法：

ds_map_is_map(id, key)

争论描述
id要使用的ds_map的ID。
key更换的钥匙。

返回值：

Boolean

例子：

var size = ds_map_size(inventory) ;
var key = ds_map_find_first(inventory);
for (var i = 0; i < size; i++;)
    {
    if ds_map_is_map(inventory, key)
        {
        ds_map_destroy(key);
        }
    key = ds_map_find_next(inventory);
    }
ds_map_destroy(inventory);

上面的代码遍历了DS映射，并检查其中是否有其他键用于其他DS映射。如果是，则销毁存储的DS映射，并且在循环结束时也销毁主DS映射。
