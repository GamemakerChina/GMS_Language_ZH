ds_map_is_map

通过这个函数，你可以检查一个DS地图是否被存储在给定的地图键中。如果给定的键包含一个DS地图的ID，那么这个函数将返回true，否则将返回false。

语法。

ds_map_is_map(id, key)

争论 描述
身份证 要使用的ds_map的id。
钥匙 替换的关键。

返回。

布尔型

例子。

var size = ds_map_size(inventory) ;
var key = ds_map_find_first(inventory);
for (var i = 0; i < size; i++;)
{
if ds_map_is_map(inventory, key)
{
ds_map_destroy(key)。
}
key = ds_map_find_next(inventory);
}
ds_map_destroy(inventory)。

上面的代码在一个DS地图中循环，并检查其中是否有任何键是用于其他DS地图。如果是，那么存储的DS地图就会被销毁，在循环的最后，主DS地图也会被销毁。

返回。DS地图
