ds_map_keys_to_array

使用此功能，您可以检索DS映射包含的所有键。您提供DS地图ID，以从中获取密钥（由返回ds_map_create()），该函数将返回GML _Overview / Arrays.htm“>数组，其中数组中的每个条目都是DS映射中的键。该函数具有可选的第二个参数，您可以在其中提供已创建的数组，如果将映射键数据附加到数组中的任何现有数据上，请注意该函数将直接修改提供的数组，但也会返回对其的引用（如果未提供，则返回对新数组的引用）。

句法：

ds_map_keys_to_array(id, [array])

争论描述
id要使用的地图的ID。
[array]选修的！将DS映射键附加到的数组。

返回值：

Array

例子：

map_keys = ds_map_keys_to_array(inventory);

上面的代码检索DS映射的键，然后将它们存储为数组以备将来使用。
