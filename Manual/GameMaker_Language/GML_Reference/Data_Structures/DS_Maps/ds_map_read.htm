ds_map_read

该函数将使用该函数先前创建的字符串 ds_map_write()然后将其读入先前创建的DS地图。如果要读取字符串的映射包含任何键/值对，则在重新构建保存的映射之前，将首先清除这些键/值对。请注意，如果数据结构是使用GameMaker Studio 2的早期版本创建的，则应添加可选参数“旧版”，将其设置为 true因为此版本的字符串格式已更改。

句法：

ds_map_read(id, str [, legacy]);

争论描述
id读取字符串的数据结构的ID
str要读取的字符串
legacy ( optional )可以是 true或者 false或完全省略。

返回值：

N/A

例子：

inventory = ds_map_create();
ini_open("map.ini");
var t_string = ini_read_string("Saved", "0", "");
if (t_string != "")
   {
   ds_map_read(inventory, t_string);
   }
ini_close();

上面的代码创建了一个新的DS映射，并将其ID索引存储在变量“库存”中。然后，它打开一个ini文件，并从该文件中读取一个字符串到临时变量“ t_string”中。最后，它检查以确保字符串有效（不是默认的ini值““”），如果正确，则在再次关闭ini之前将字符串读取到新创建的DS映射中。
