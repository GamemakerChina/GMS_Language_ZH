ds_map_replace

使用此功能，您可以在DS映射中更改给定键的值。您向地图提供索引（由函数返回）ds_map_create() ），然后是要替换的键（字符串或整数），以及要用来替换键值的值。如果给定的键不存在，则将为您创建该键，如果不存在，则将用新值替换当前值。该函数将返回 true如果键存在并且值已替换，并且 false如果键不存在，并且使用该值创建了一个新键。

句法：

ds_map_replace( id, key, val );

争论描述
id要更改的地图的ID。
key具有应由新密钥替换的值的密钥
val用以下值替换给定值的新值

返回值：

Boolean

例子：

ds_map_replace(inventory, "torso", 55);

上面的代码在DS映射中查找键“ torso”，找到键（如果不存在则创建键），将当前值替换为指定的值。

背面： DS地图
