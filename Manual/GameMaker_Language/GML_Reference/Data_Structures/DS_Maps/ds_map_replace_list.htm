ds_map_replace_list

使用此功能，您可以将存储在给定“键”中的DS列表替换为先前创建的另一个列表。此功能旨在创建JSON兼容映射，然后使用对其进行编码 json_encode()并且仅应与该功能结合使用。

句法：

ds_map_replace_list(id, key, value)

争论描述
id要使用的地图的ID。
key更换的钥匙。
value ds_list的ID，用于替换先前存储在给定密钥中的ID。

返回值：

N/A

例子：

var j_list = ds_list_create();
ds_list_add(j_list, health);
ds_list_add(j_list, lives);
ds_list_add(j_list, score);
ds_map_replace_list(j_map, "list", j_list);
var j = json_encode(j_map);
ds_list_destroy(j_list);

上面的代码将创建一个 DS List并使用各种全局变量的值填充它，然后替换之前存储在列表中的列表 DS Map“ j_map”。
