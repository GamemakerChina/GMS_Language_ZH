ds_map_replace_map

使用此功能，您可以替换 DS Map已与先前创建的另一个映射一起存储在给定的“键”中。此功能旨在创建JSON兼容映射，然后使用对其进行编码 json_encode()并且仅应与该功能结合使用。

句法：

ds_map_replace_map(id, key, value)

争论描述
id要使用的ds_map的ID。
key更换的钥匙。
value ds_map的ID，用于替换先前存储在给定密钥中的ID。

返回值：

N/A

例子：

var temp_map = ds_map_create();
ds_map_add_list(temp_map, "list", j_list);
ds_map_add(temp_map, "array", j_array);
ds_map_replace_map(j_map, "maps", temp_map);
var j = json_encode(j_map);
ds_map_destroy(temp_map);

上面的代码将创建一个 DS Map并用一个数组和一个 DS List在替换之前存储在地图中的地图之前 DS Map“ j_map”。
