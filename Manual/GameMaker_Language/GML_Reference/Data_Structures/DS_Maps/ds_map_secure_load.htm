ds_map_secure_load

此功能将从给定文件中加载先前保存的DS地图。此文件必须先前已使用 ds_map_secure_save()函数，并且在加载后，该函数将返回已从加载的数据创建的DS地图的索引。此DS映射索引应存储在变量中，并用于对该映射的所有其他函数调用。请注意，如果要加载的DS地图与一个数组一起保存为键值之一，则该数组将在加载时转换为DS列表。

重要的！安全保存的文件的功能之一是将其锁定在创建该文件的设备上，因此您无法将保存在一个设备上的文件加载到在另一个设备上运行的项目中。

句法：

ds_map_secure_load(filename);

争论描述
filename要从中加载地图数据的文件名。

返回值：

Integer (DS map index)

例子：

p_map = ds_map_secure_load("p_data.dat");

上面的代码将加载安全保存的DS映射，并将其索引值存储在变量中，以备将来使用。
