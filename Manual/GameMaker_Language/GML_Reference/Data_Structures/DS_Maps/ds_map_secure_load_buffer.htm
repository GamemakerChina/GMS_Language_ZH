ds_map_secure_load_buffer

此功能将从缓冲区加载安全保存的DS映射。您必须事先将缓冲区加载到内存中（使用 buffer_load()），然后将其传递给此函数将返回填充有缓冲区内容的DS映射。请注意，缓冲区必须已使用函数创建 ds_map_secure_save_buffer()为了使其正常工作，还请注意，如果要加载的DS映射包含一个数组，则加载时它将转换为DS列表。

重要的！安全保存的文件的功能之一是将其锁定在创建该文件的设备上，因此您无法将保存在一个设备上的文件加载到在另一个设备上运行的项目中。

句法：

ds_map_secure_load_buffer(buffer);

争论描述
filename要从中加载地图数据的缓冲区的缓冲区ID。

返回值：

Integer (DS map index)

例子：

var buff = buffer_load("save.dat");
map = ds_map_secure_load_buffer(buff);
buffer_delete(buff);

上面的代码将从缓冲区加载安全保存的DS映射，并将其索引值存储在变量中，以备将来使用。
