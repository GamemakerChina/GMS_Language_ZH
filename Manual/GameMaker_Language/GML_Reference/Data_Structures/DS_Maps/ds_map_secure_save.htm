ds_map_secure_save

此功能会将给定DS映射的内容保存到与创建该设备的设备链接的文件（这意味着如果传输到任何其他设备上将无法读取该文件）。文件本身几乎可以具有任何扩展名（例如，*.dat ，*.json ，*.bin等等），并将其混淆并存储到目标平台上的本地存储中。然后，您可以使用函数重新加载ds_map ds_map_secure_load() 。请注意，如果要保存的DS映射包含一个数组，则保存时该数组将转换为DS列表。

重要的！安全保存的文件的功能之一是将其锁定在创建该文件的设备上，因此您无法将保存在一个设备上的文件加载到在另一个设备上运行的项目中。

句法：

ds_map_secure_save(map, filename);

争论描述
map要使用的数据结构的ID
filename要将地图保存到的文件名

返回值：

Boolean

例子：

ds_map_secure_save(purchase_map, "p_data.dat");

上面的代码会将在变量“ p_data”中索引的DS映射保存到给定文件中，以供以后检索。
