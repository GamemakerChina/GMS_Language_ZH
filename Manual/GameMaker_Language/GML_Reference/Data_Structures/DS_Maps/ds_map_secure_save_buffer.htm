ds_map_secure_save_buffer

此功能会将先前创建的DS映射保存到缓冲区。您提供DS地图ID值（由函数返回）ds_map_create() ）和要写入的缓冲区的ID（由函数返回）buffer_create() ）。请注意，如果要保存的DS映射包含一个数组，则保存时它将转换为DS列表。

重要的！安全保存的DS映射文件只能加载在创建该文件的设备上，如果尝试加载保存在其他设备上的文件，则该文件将不起作用。

句法：

ds_map_secure_save_buffer(filename);

争论描述
id DS地图ID值。
buffer缓冲区ID值。

返回值：

Integer (DS map index)

例子：

buff = buffer_create(128,buffer_grow,4);
var map = ds_map_create();
ds_map_add(map,"bob","ajob");
ds_map_add(map,"money",10);
ds_map_secure_save_buffer(map, buff);
ds_map_destroy(map);

上面的代码将创建一个缓冲区和一个DS映射，然后使用一些值填充该映射并将其写入缓冲区，然后再删除该映射。
