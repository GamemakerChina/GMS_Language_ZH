ds_map_secure_save_buffer

这个函数将保存一个先前创建的DS地图到一个缓冲区。你提供DS地图的ID值（由函数返回 ds_map_create()返回的）和要写入的缓冲区的ID（由函数 buffer_create()).注意，如果被保存的DS地图包含一个数组，那么在保存时将被转换为DS列表。

重要的是!安全保存的DS地图文件只能在创建它的设备上加载，如果你试图加载保存在不同设备上的文件，那么它将无法工作。

语法。

ds_map_secure_save_buffer(filename)。

争论 描述
身份证 DS地图ID值。
缓冲区 缓冲区的ID值。

返回。

整数（DS地图索引

例子。

buff = buffer_create(128,buffer_grow,4);
var map = ds_map_create();
ds_map_add(map, "bob", "ajob");
ds_map_add(map, "money",10);
ds_map_secure_save_buffer（map, buff）;
ds_map_destroy（map）。

上面的代码将创建一个缓冲区和一个DS地图，然后用一些值填充地图，并在删除地图之前将其写入缓冲区。

返回。DS地图
