ds_map_set

使用此功能，您可以在给定的DS映射中设置键的值。您提供DS地图ID值（由函数返回）ds_map_create() ），然后提供您要设置的键和设置值。键可以是整数或字符串，如果给定的键不存在，则会为您创建并设置为值。此功能与使用DS映射访问器设置/创建映射键/值对相同。该函数不返回任何内容，因此，如果需要检查是否已替换键值或已创建新的键，则应使用该函数 ds_map_replace()。

句法：

ds_map_set(id, key, value)

争论描述
id要使用的地图的ID。
key设置的关键。
value设置键的值。

返回值：

N/A

例子：

if is_undefined(ds_map_find_value(map, "score"))
    {
    ds_map_set(map, "score", 0);
    }

上面的代码将检查给定密钥是否存在，如果不存在，则创建并设置它。
