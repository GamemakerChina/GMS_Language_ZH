ds_map_values_to_array

使用此功能，您可以检索DS映射为映射中的每个键包含的所有值。您提供DS地图ID，以从中获取密钥（由返回ds_map_create()），该函数将返回一个数组，其中数组中的每个条目都是DS映射中的值（例如，如果该映射中有10个键，则将返回一个包含10个条目的数组，其中每个条目都对应于一个钥匙）。该函数具有一个可选的第二个参数，您可以在其中提供已创建的数组，在这种情况下，映射键值数据将附加到该数组中的任何现有数据上。请注意，该函数将修改直接提供的数组，但也会返回对其的引用（如果未提供，则返回对新数组的引用）。

句法：

ds_map_values_to_array(id, [array])

争论描述
id要使用的地图的ID。
[array]选修的！将DS映射值附加到的数组。

返回值：

Array

例子：

var _values = ds_map_values_to_array(lvl_score);
var _total = 0;
for (var i = 0; i < array_lengh(values); ++i;)
    {
    _total += _values[i];
    }
draw_text(32, 32, "Total score for all levels = " + string(_total"));

上面的代码从DS映射中检索了映射中每个键的所有值，并将它们存储在作为数组的局部变量中。然后，它遍历数组并将每个值添加到局部变量，然后该局部变量用于显示所有值的总和。
