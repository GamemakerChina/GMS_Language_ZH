ds_map_write

此功能会将指定地图的DS地图数据转换为字符串格式，然后可以将其写入*.ini或*.txt文件中，以便于存储。然后可以使用以下命令将该字符串读回到新的ds_map中 ds_map_read()。

注意：返回的字符串不是人类可读的字符串，而是数据结构内容的转储。

句法：

ds_map_write(id);

争论描述
id要使用的数据结构的ID

返回值：

String

例子：

ini_open("map.ini");
var t_string;
t_string = ds_map_write(inventory);
ini_write_string("Saved", "0", t_string);
ini_close();

上面的代码打开一个准备写入的ini文件。然后使用 ds_map_write()生成存储在临时变量“ t_string”中的字符串。最后，它在关闭该字符串之前将其写入ini文件。

背面： DS地图
