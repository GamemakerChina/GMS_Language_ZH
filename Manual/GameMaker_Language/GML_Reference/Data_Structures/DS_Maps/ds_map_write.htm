ds_map_write

这个函数将把指定地图的DS地图数据变成字符串格式，然后可以写到*.ini或*.txt文件中，方便存储。这个字符串以后可以用以下方法读回一个新的ds_map中 ds_map_read().

注意：返回的字符串不是人类可读的字符串，而是一个数据结构内容的转储。

语法。

ds_map_write(id)。

争论 描述
身份证 要使用的数据结构的ID

返回。

字符串

例子。

ini_open("map.ini");
var t_string;
t_string = ds_map_write( inventory);
ini_write_string("Saved", "0", t_string);
ini_close();

上面的代码打开了一个准备被写入的ini文件。然后它使用ds_map_write()生成一个字符串，存储在临时变量 "t_string "中。最后，在关闭ini文件之前，它把这个字符串写到ini文件中。

返回。DS地图
