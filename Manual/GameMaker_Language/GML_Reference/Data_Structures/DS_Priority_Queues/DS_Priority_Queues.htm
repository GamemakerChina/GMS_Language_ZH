DS优先级队列

DS优先级队列是一种特殊的数据结构，类似于 DS queue，只有在这里，值队列才按已分配值的优先级（总是实数）排序。这种结构对于构建排行榜或信息列表非常有用，其中每个条目的重要性都可以加权。

注：与所有的动态资源，数据结构占用内存，当不再需要以防止内存泄漏，这将减缓，并最终崩溃的游戏，所以应该总是被销毁。

注意：如果需要检查数据结构是否存在，可以使用ds_exists（）函数。

存在处理DS优先级队列的以下功能：

  • ds_priority_create
  • ds_priority_destroy
  • ds_priority_clear
  • ds_priority_empty
  • ds_priority_size
  • ds_priority_add
  • ds_priority_change_priority
  • ds_priority_delete_max
  • ds_priority_delete_min
  • ds_priority_delete_value
  • ds_priority_find_max
  • ds_priority_find_min
  • ds_priority_find_priority
  • ds_priority_copy
  • ds_priority_read
  • ds_priority_write
