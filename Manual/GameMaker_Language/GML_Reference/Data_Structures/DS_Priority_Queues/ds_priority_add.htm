ds_priority_add

使用此功能，您可以将一个值（实数或字符串）添加到优先级队列，同时为其分配一个优先级值。

句法：

ds_priority_add(id, val, priority);

争论描述
id要添加到的优先级队列的ID。
val要添加到优先级队列的值。
priority要添加的值的优先级。

返回值：

N/A

例子：

ds_priority_add(ai_priority, AI_Search, 5);

上面的代码将脚本函数添加到在变量“ ai_priority”中索引的优先级队列，并为其分配优先级5。

返回： DS优先级队列
