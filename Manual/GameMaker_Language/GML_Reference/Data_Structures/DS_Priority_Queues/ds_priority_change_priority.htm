ds_priority_change_priority

此函数将采用给定的值，并在引用的优先级队列中更改其优先级。

句法：

ds_priority_change_priority(id, val, priority);

争论描述
id要更改的优先级队列的ID。
val更改优先级的值。
priority值的新优先级。

返回值：

N/A

例子：

if (global.Game_Time < 1000)
    {
    ds_priority_change(ai_priority, AI_Search, 1);
    }

上面的代码检查全局变量，如果它小于某个特定值，则它将更改优先级队列中保存的脚本函数索引的优先级。

返回： DS优先级队列
