ds_priority_copy

这个函数可以用来把一个优先级队列的内容复制到另一个队列中。请注意，这不会从原优先级队列中删除内容，也不会破坏原优先级队列。当使用这个函数时，被复制的优先级 被复制的队列必须是以前创建的，如果它在复制之前包含任何项目，那么这些项目将被首先清除（意味着这些信息将丢失）。

语法。

ds_priority_copy(id, source)。

争论 描述
身份证 要复制到的优先级队列的ID。
来源 要复制的优先级队列的ID。

返回。

不适用

例子。

with (instance_create_layer(x, y, "Enemies", obj_Enemy))
{
p_queue = ds_priority_create（）;
ds_priority_copy(p_queue, other.p_queue);
}

上述函数创建了一个新的实例，然后在该实例中创建了一个新的DS优先级队列，并将运行该代码块的实例中的优先级队列内容复制到新创建的实例优先级队列中。
