ds_priority_copy

此功能可用于将一个优先级队列的内容复制到另一个优先级队列。请注意，这不会从原来的优先级队列中移除的内容，也不会破坏原有的优先级队列。使用此功能时，必须事先创建要复制到的优先级队列，并且如果该优先级队列在复制之前包含任何项目，则将首先清除这些项目（这意味着该信息将丢失）。

句法：

ds_priority_copy(id, source);

争论描述
id要复制到的优先级队列的ID。
source要从中复制的优先级队列的ID。

返回值：

N/A

例子：

with (instance_create_layer(x, y, "Enemies", obj_Enemy))
    {
    p_queue = ds_priority_create();
    ds_priority_copy(p_queue, other.p_queue);
    }

上面的函数创建一个新实例，然后在该实例中创建一个新的DS优先级队列，并将运行代码块的实例中优先级队列的内容复制到新创建的实例优先级队列中。

返回： DS优先级队列
