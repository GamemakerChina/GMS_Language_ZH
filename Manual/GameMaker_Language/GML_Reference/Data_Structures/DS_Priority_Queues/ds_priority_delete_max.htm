ds_priority_delete_max

此函数将返回队列中具有最高优先级的值，然后从数据结构中删除该值（和优先级）。如果多个值具有相同的优先级，则可以以任何顺序返回它们中的任何一个，但是具有相同优先级的所有其他值仍将在队列中。

句法：

ds_priority_delete_max(id);

争论描述
id要使用的优先级队列的ID。

返回值：

Data type stored in the priority

例子：

if ai_move
    {
    script_execute(ds_priority_delete_max(ai_priority));
    }

上面的代码检查一个实例变量，如果它返回 true它将执行在优先级队列中索引的具有最高优先级值的脚本，然后从队列中删除该脚本。

返回： DS优先级队列
