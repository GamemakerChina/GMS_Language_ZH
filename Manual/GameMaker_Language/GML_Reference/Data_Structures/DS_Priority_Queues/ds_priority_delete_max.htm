ds_priority_delete_max

这个函数将返回在队列中具有最高优先级的值，然后从数据结构中删除该值（和优先级）。如果有一个以上的值具有相同的优先级，那么它们中的任何一个都可以按任何顺序被返回，但是所有具有相同优先级的 其他具有相同优先级的值仍将在队列中。

语法。

ds_priority_delete_max(id)。

争论 描述
身份证 要使用的优先级队列的ID。

返回。

存储在优先权中的数据类型

例子。

如果 ai_move
{
script_execute(ds_priority_delete_max(ai_priority))。
}

上面的代码检查一个实例变量，如果它返回真，它将执行一个在优先级队列中索引的、具有最高优先级值的脚本，然后从队列中删除该脚本。
