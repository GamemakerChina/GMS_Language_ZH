ds_priority_delete_value

该函数将简单地从索引优先级队列中删除给定值及其优先级。

句法：

ds_priority_delete_value(id,val);

争论描述
id要使用的优先级队列的ID。
val要从优先级队列中删除的值。

返回值：

N/A

例子：

if (ai_move == false)
    {
    ds_priority_delete_value(ai_priority, AI_Move);
    }

上面的代码检查一个实例变量，如果它返回 false它将从优先级队列中删除索引脚本功能。

返回： DS优先级队列
