ds_priority_delete_value

这个函数将简单地从索引的优先级队列中删除给定的值，以及它的优先级。

语法。

ds_priority_delete_value(id,val)。

争论 描述
身份证 要使用的优先级队列的ID。
价值 要从优先级队列中删除的值。

返回。

不适用

例子。

如果(ai_move == false)
{
ds_priority_delete_value(ai_priority, AI_Move);
}

上面的代码检查了一个实例变量，如果它返回false，它将从优先级队列中删除索引的脚本函数。
