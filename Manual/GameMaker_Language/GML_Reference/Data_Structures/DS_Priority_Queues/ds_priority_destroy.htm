ds_priority_destroy

此函数将从内存中删除给定的优先级队列数据结构，释放其正在使用的资源，并删除其包含的所有值。当您完成使用DS优先级队列时，应始终使用此功能，以防止内存泄漏会减慢游戏并使其崩溃。

重要的！创建数据结构时，用于标识它的索引值是一个从0开始的整数值。这意味着不同类型的数据结构可以具有相同的索引值，因此如果有疑问，您应该使用 ds_exists()功能，然后再访问它们。还要注意，索引已被重新使用，因此新创建的索引可能会被破坏后的数据结构索引值使用，因此我们建议始终将破坏DS索引的变量设置为-1。

句法：

ds_priority_destroy(id);

争论描述
id要删除的数据结构的ID。

返回值：

N/A

例子：

if (lives == 0)
    {
    ds_priority_destroy(AI_queue);
    AI_queue = -1;
    room_goto(rm_Menu);
    }

上面的代码将检查内置全局变量“ lives”的值，如果该值为0，它将销毁在变量“ AI_queue”中索引的DS优先级队列，然后更改房间。

返回： DS优先级队列
