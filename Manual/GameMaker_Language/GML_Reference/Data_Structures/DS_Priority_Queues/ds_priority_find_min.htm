ds_priority_find_min

使用此功能，您可以找到优先级最低的存储在优先级队列中的值，如果多个值具有相同的优先级，则可以任何顺序返回它们中的任何一个。但是，不像 ds_priority_delete_min()，此功能不会从队列中删除该值。

句法：

ds_priority_find_min(id);

争论描述
id要使用的优先级队列的ID。

返回值：

Real or String

例子：

if ai_move
    {
    script_execute(ds_priority_find_min(ai_priority));
    }

上面的代码检查一个实例变量，如果它返回 true它将执行优先级值最低的优先级队列中索引的脚本功能。

返回： DS优先级队列
