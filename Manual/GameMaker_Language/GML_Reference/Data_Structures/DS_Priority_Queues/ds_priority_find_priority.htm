ds_priority_find_priority

通过这个函数，你可以检索到任何给定值的优先级。如果该值不存在于优先级队列中，那么将返回未定义。

语法。

ds_priority_find_priority(id, val)。

争论 描述
身份证 要使用的优先级队列的ID。
价值 要找到的优先级的值。

返回。

真实或未定义

例子。

p = ds_priority_find_priority(ai_priority, "intelligence")。

上述代码将把返回的给定值的优先级存储在实例变量 "p "中。
