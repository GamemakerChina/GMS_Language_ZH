ds_priority_find_priority

使用此功能，您可以检索任何给定值的优先级。如果该值在优先级队列中不存在，则 undefined将被退回。

句法：

ds_priority_find_priority(id, val);

争论描述
id要使用的优先级队列的ID。
val查找优先级的值。

返回值：

Real or undefined

例子：

p = ds_priority_find_priority(ai_priority, "intelligence");

上面的代码将给定值返回的优先级存储在实例变量“ p”中。

返回： DS优先级队列
