ds_priority_size

这个函数将返回优先级队列的 "大小"，即：其中已被优先处理的项目的数量。

语法。

ds_priority_size(id)。

争论 描述
身份证 要检查的数据结构的ID。

返回。

真实的

例子。

if !ds_priority_empty(control_priority)
{
num = ds_priority_size(control_priority);
}

上面的代码检查一个DS优先级队列，看它是否为空，如果不是，就得到它所包含的项目数，并将该值存入一个变量。
