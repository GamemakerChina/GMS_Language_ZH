ds_priority_size

此函数将返回优先级队列的“大小”，即：已在其中进行优先级排序的项目数。

句法：

ds_priority_size(id);

争论描述
id要检查的数据结构的ID。

返回值：

Real

例子：

if !ds_priority_empty(control_priority)
    {
    num = ds_priority_size(control_priority);
    }

上面的代码检查DS优先级队列以查看其是否为空，如果不是，则获取包含的项数并将该值存储在变量中。

返回： DS优先级队列
