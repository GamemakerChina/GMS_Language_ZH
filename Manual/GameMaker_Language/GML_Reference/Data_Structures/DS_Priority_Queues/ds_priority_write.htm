ds_priority_write

这个函数返回一个字符串，然后可以用这个字符串存储或传输到另一个数据结构中。 ds_priority_read()函数来存储或转移到另一个数据结构中。

注意：返回的字符串不是人类可读的字符串，而是一个数据结构内容的转储。

语法。

ds_priority_write(id)。

争论 描述
身份证 要检查的数据结构的ID。

返回。

字符串

例子。

var str;
ini_open("save.ini");
str =ds_priority_write(p_queue);
ini_write_string("P_Queues", "0", str);
ds_priority_clear(p_queue) 。
ini_close()。

上面的代码打开了一个ini文件，然后写了一个字符串，其中包含了存储在DS优先级队列中的信息，该队列的索引是变量 "p_queue"。然后，优先级队列被清除，ini文件被关闭。
