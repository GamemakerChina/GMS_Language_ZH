ds_priority_write

该函数返回一个字符串，然后可以使用 ds_priority_read()功能。

注意：返回的字符串不是人类可读的字符串，而是数据结构内容的转储

句法：

ds_priority_write(id);

争论描述
id要检查的数据结构的ID。

返回值：

String

例子：

var str;
ini_open("save.ini");
str =ds_priority_write(p_queue);
ini_write_string("P_Queues", "0", str);
ds_priority_clear(p_queue);
ini_close();

上面的代码打开一个ini文件，然后写入一个字符串，该字符串包含在变量“ p_queue”中索引的DS优先级队列中存储的信息。然后清除优先级队列并关闭ini文件。

返回： DS优先级队列
