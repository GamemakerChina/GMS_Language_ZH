ds_queue_clear

使用此功能，您可以清除给定队列数据结构中的所有数据。这不破坏数据结构（你应该使用 ds_queue_destroy()），它只会清除其中的所有数据并返回一个空队列。

句法：

ds_queue_clear(id);

争论描述
id要清除的数据结构的ID。

返回值：

N/A

例子：

if (count = 15) && (!ds_queue_empty(command_queue))
    {
    ds_queue_clear(command_queue);
    alarm[0] = room_speed;
    ai_count = 0;
    }

上面的代码检查变量是否已达到特定值，并清除变量“ command_queue”中索引的DS队列，设置警报，并将变量重置为0。

返回： DS队列
