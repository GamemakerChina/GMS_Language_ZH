ds_queue_clear

通过这个函数你可以清除给定队列数据结构中的所有数据。这并不破坏数据结构（为此你应该使用 ds_queue_destroy())，它只是清除了其中的所有数据，并且 返回一个空队列。

语法。

ds_queue_clear(id)。

争论 描述
身份证 要清除的数据结构的ID。

返回。

不适用

例子。

if (count = 15) && (!ds_queue_empty(command_queue))
{
ds_queue_clear(command_queue)。
alarm[0] = room_speed;
ai_count = 0;
}

上面的代码检查一个变量，看它是否达到了一个特定的值，如果达到了，就清除变量 "command_queue "中索引的DS队列，设置一个警报，并将变量重置为0。
