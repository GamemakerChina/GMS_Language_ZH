ds_queue_copy

此功能可用于将一个队列的内容复制到另一个队列。请注意，这不会从原来的队列中删除的内容，也不会破坏原有的队列。使用此功能时，必须事先创建要复制到的队列，并且如果复制之前包含任何项目，则将首先清除这些项目（这意味着该信息将丢失）。

句法：

ds_queue_copy(id, source);

争论描述
id NEW队列的ID。
source要复制到ID的原始队列。

返回值：

N/A

例子：

with (instance_create_layer(x, y, "Enemies", obj_Enemy))
    {
    queue = ds_queue_create();
    ds_queue_copy(queue, other.queue);
    }

上面的函数创建一个新实例，然后在该实例中创建一个新的DS队列，并将运行代码块的实例中的队列内容复制到新创建的实例队列中。

返回： DS队列
