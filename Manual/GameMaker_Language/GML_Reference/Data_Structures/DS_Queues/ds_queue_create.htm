ds_queue_create

该函数将创建一个新的队列数据结构并返回索引值。该值应存储在变量中，并在与队列有关的所有其他函数调用中使用。

重要的！创建数据结构时，用于标识它的索引值是一个从0开始的整数值。这意味着不同类型的数据结构可以具有相同的索引值，因此如果有疑问，应在使用ds_exists（）函数之前访问它们。另请注意，索引已被重新使用，因此之后新创建的索引结构可能会使用已破坏的数据结构索引值。

句法：

ds_queue_create();

返回值：

Real

例子：

queue = ds_queue_create();

这将创建一个新队列，并将其索引ID分配给实例变量“ queue”。

返回： DS队列
