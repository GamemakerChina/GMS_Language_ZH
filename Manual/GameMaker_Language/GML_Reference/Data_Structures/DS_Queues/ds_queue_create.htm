ds_queue_create

这个函数将创建一个新的队列数据结构并返回索引值。这个值应该存储在一个变量中，并在所有与队列有关的后续函数调用中使用。

重要的是!当你创建一个数据结构时，识别它的索引值是一个从0开始的整数值。 这意味着不同类型的数据结构可以有相同的索引值，所以如果有疑问，你应该使用 ds_exists()函数来访问它们。还要注意的是，索引是可以重复使用的，所以一个被破坏的数据结构的索引值可能会被之后新创建的数据结构使用。

语法。

ds_queue_create()。

返回。

真实的

例子。

queue = ds_queue_create()。

这将创建一个新队列，并将其索引ID分配给实例变量 "queue"。
