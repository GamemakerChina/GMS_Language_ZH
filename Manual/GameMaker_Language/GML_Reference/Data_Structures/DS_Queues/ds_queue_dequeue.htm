ds_queue_dequeue

此功能将出列的头值关DS队列，从队列中移除，并返回到被存储在一个变量的值。如果队列为空，则该函数将返回常量 undefined，否则它将返回队列中包含的实数或字符串值。

句法：

ds_queue_dequeue(id);

争论描述
id要从中出队的队列的ID。

返回值：

Data type value stored in the queue or undefined

例子：

if !ds_queue_empty(move_queue)
    {
    var xx = ds_queue_dequeue(move_queue);
    var yy = ds_queue_dequeue(move_queue);
    move_towards_point(xx, yy, 4);
    }

上面的代码检查在变量“ move_queue”中索引的DS队列，看是否为空，如果不是，则将其从队列头中取出两个值，并使用它们来设置移动方向。

返回： DS队列
