ds_queue_empty

使用此功能，您可以检查给定的DS队列以查看其是否为空（返回 true）否（返回 false）。

句法：

ds_queue_empty(id);

争论描述
id要检查的数据结构的ID。

返回值：

Boolean

例子：

if (count == 15) && (!ds_queue_empty(command_queue))
    {
    ds_queue_clear(command_queue);
    alarm[0] = room_speed;
    ai_count = 0;
    }

上面的代码检查变量是否已达到特定值，并清除变量“ command_queue”中索引的DS队列，设置警报，并将变量重置为0。

返回： DS队列
