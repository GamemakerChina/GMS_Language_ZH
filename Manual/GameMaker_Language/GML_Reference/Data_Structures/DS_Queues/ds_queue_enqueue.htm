ds_queue_enqueue

此函数会将值（实数或字符串）添加到DS队列的尾部。该函数可以再使用14个可选参数（总共进行15个可能的加法运算），使您可以在单个调用中将多个值连续添加到队列的尾部。

句法：

ds_queue_enqueue(id, val [, val2, ... val15]);

争论描述
id要添加到的队列的ID。
val要添加到队列中的值。
[val2,... val15]要添加到队列的可选值。

返回值：

N/A

例子：

move_queue = ds_queue_create();
ds_queue_enqueue(move_queue, x + 200);
ds_queue_enqueue(move_queue, y);
ds_queue_enqueue(move_queue, x + 200);
ds_queue_enqueue(move_queue, y + 200);
ds_queue_enqueue(move_queue, x);
ds_queue_enqueue(move_queue, y + 200);
ds_queue_enqueue(move_queue, x);
ds_queue_enqueue(move_queue, y);

上面的代码创建了一个新的DS队列，并将其索引存储在变量“ move_queue”中。然后，它将多个值推送到队列中以备将来使用。

返回： DS队列
