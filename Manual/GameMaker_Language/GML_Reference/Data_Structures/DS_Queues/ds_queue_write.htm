ds_queue_write

该函数返回一个字符串，然后可以使用 ds_queue_read()功能。

注意：返回的字符串不是人类可读的字符串，而是数据结构内容的转储

句法：

ds_queue_write(id);

争论描述
id要写入的数据结构的ID。

返回值：

String

例子：

ini_open("save.ini");
var str =ds_queue_write(queue);
ini_write_string("Queues", "0", str);
ds_queue_clear(queue);
ini_close();

上面的代码打开一个ini文件，然后写一个字符串，其中包含存储在DS队列中的信息，该信息在变量“ queue”中索引。然后清除队列并关闭ini文件。

返回： DS队列
