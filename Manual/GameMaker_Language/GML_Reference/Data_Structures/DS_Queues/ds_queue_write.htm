ds_queue_write

这个函数返回一个字符串，然后可以用这个字符串存储或转移到另一个数据结构中。 ds_queue_read()函数来存储或转移到另一个数据结构中。

注意：返回的字符串不是人类可读的字符串，而是一个数据结构内容的转储。

语法。

ds_queue_write(id)。

争论 描述
身份证 要写入的数据结构的ID。

返回。

字符串

例子。

ini_open("save.ini");
var str =ds_queue_write(queue);
ini_write_string("queues", "0", str);
ds_queue_clear(queue);
ini_close()。

上面的代码打开了一个ini文件，然后写了一个字符串，其中包含存储在变量 "queue "索引的DS队列中的信息。然后队列被清除，ini文件被关闭。
