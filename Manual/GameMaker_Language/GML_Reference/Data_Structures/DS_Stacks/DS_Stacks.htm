一个DS堆栈是一个所谓的后进先出（LIFO）数据结构，类似于DS队列。你可以把数值推到堆栈中，然后通过从堆栈中弹出的方式再次删除它们，其中最近推到堆栈中的数值 最近被推入堆栈的值是第一个被再次弹出的值（就像一摞硬币，你添加的每一个硬币都要先被移除，然后才能拿到下面的其他硬币）。堆栈经常被用于 当有中断需要处理时，或当有递归函数时，甚至当为你的游戏构建一个基本的人工智能时，堆栈经常被使用。

注意：与所有的动态资源一样，数据结构会占用内存，所以当不再需要时，应该总是销毁，以防止内存泄漏，这将减缓并最终使你的游戏崩溃。

注意：如果你需要检查一个数据结构是否存在，你可以使用 ds_exists()函数。
