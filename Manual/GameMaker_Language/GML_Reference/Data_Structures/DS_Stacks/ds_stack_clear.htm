ds_stack_clear

使用此功能，您可以清除给定堆栈数据结构中的所有数据。这不破坏数据结构（你应该使用 ds_stack_destroy()），它只会清除其中的所有数据并返回一个空堆栈。

句法：

ds_stack_clear(id);

争论描述
id要清除的数据结构的ID。

返回值：

N/A

例子：

if (ai_count = 15) && (!ds_stack_empty(AI_stack))
    {
    ds_stack_clear(AI_stack);
    alarm[0] = room_speed;
    ai_count = 0;
    }

上面的代码检查变量是否已达到特定值，以及是否清除了在变量“ AI_stack”中索引的DS堆栈，设置了警报，并将变量重置为0。
