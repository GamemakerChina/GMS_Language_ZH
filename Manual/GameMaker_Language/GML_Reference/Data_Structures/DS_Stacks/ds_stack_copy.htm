ds_stack_copy

此功能可用于将一个堆栈的内容复制到另一个堆栈中。请注意，这不会从原来的堆栈中删除的内容，也不会破坏原有的堆栈。使用此功能时，必须事先创建要复制到的堆栈，并且如果复制之前包含任何项目，则将首先清除这些项目（这意味着此信息将丢失）。

句法：

ds_stack_copy(id, source);

争论描述
id新堆栈的ID。
source要复制到id的原始堆栈。

返回值：

N/A

例子：

with (instance_create_layer(x, y, "Enemies", obj_Enemy))
    {
    stack = ds_stack_create();
    ds_stack_copy(stack, other.stack);
    }

上面的函数创建一个新实例，然后在该实例中创建一个新的DS堆栈，并将运行代码块的实例中的堆栈内容复制到新创建的实例堆栈中。
