ds_stack_empty

通过这个函数，你可以检查给定的DS堆栈，看它是否为空（返回true）或不为空（返回false）。

语法。

ds_stack_empty(id)。

争论 描述
身份证 要检查的数据结构的ID。

返回。

布尔型

例子。

if (ai_count == 15) && (!ds_stack_empty(AI_stack))
{
ds_stack_clear(AI_stack);
alarm[0] = room_speed;
ai_count = 0;
}

上面的代码检查一个变量，看它是否达到了一个特定的值，如果达到了，它就清除变量 "AI_stack "中索引的DS栈，设置一个警报，并将变量重置为0。
