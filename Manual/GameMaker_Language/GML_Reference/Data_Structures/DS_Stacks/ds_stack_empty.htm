ds_stack_empty

使用此功能，您可以检查给定的DS堆栈是否为空（返回 true）否（返回 false）。

句法：

ds_stack_empty(id);

争论描述
id要检查的数据结构的ID。

返回值：

Boolean

例子：

if (ai_count == 15) && (!ds_stack_empty(AI_stack))
    {
    ds_stack_clear(AI_stack);
    alarm[0] = room_speed;
    ai_count = 0;
    }

上面的代码检查变量是否已达到特定值，以及是否清除了在变量“ AI_stack”中索引的DS堆栈，设置了警报，并将变量重置为0。
