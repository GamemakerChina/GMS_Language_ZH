ds_stack_pop

此函数将从DS堆栈弹出最高值，将其从堆栈中删除，然后返回要存储在变量中的值。如果堆栈为空，则该函数将返回常量 undefined，否则它将返回堆栈中包含的实数或字符串值。

句法：

ds_stack_pop(id);

争论描述
id要弹出的数据结构的ID。

返回值：

Data type value that is stored in the stack or undefined

例子：

if !ds_stack_empty(move_stack)
    {
    var xx = ds_stack_pop(move_stack);
    var yy = ds_stack_pop(move_stack);
    move_towards_point(xx, yy, 4);
    }

上面的代码检查在变量“ move_stack”中索引的DS堆栈是否为空，如果不是，则从堆栈中弹出顶部的两个值，并使用它们来设置移动方向。
