ds_stack_push

此函数会将（可以是任何数据类型的）值压入（添加）到堆栈的顶部。该函数可以再包含14个可选参数（总共进行15个可能的加法运算），使您可以在单个调用中将多个值连续推入堆栈。

句法：

ds_stack_push(id, val [, val2, ... val15]);

争论描述
id要推送到的数据结构的ID。
val压入堆栈的值。
[val2,... val13]要添加到堆栈的可选值。

返回值：

N/A

例子：

move_stack = ds_stack_create();
ds_stack_push(move_stack, x, y, x, y + 200, x + 200, y + 200, x +200, y);

上面的代码创建了一个新的DS堆栈，并将其索引存储在变量“ move_stack”中。然后，它将多个值压入堆栈以供将来使用。
