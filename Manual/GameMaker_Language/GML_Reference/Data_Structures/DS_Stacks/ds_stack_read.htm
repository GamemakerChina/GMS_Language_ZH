ds_stack_read

使用此功能，您可以重新创建一个已保存的DS堆栈（以前已使用 ds_stack_write()）。您必须首先创建一个新的DS堆栈以将字符串读入其中，如果DS堆栈已存在并已存储信息，则将在读取之前将其清除。在为游戏创建保存/加载机制时，此功能至关重要。请注意，如果数据结构是使用GameMaker版本创建的，则应添加可选参数“旧版”，并将其设置为 true因为此版本的字符串格式已更改。

句法：

ds_stack_read(id, str [, legacy] );

争论描述
id要读取的数据结构的ID。
str要读取的字符串。
legacy ( optional )可以是 true或者 false或完全省略。

返回值：

N/A

例子：

stack = ds_stack_create();
ini_open("save.ini");
var str = ini_read_string("Stacks", "0", "");
if (str != "")
    {
    ds_stack_read(stack, str);
    }
ini_close();

上面的代码创建一个堆栈，并将索引存储在变量“ stack”中。然后打开 ini文件并从中读取一个字符串，检查以确保该字符串首先不作为空返回。然后将此字符串读入新创建的DS堆栈中。
