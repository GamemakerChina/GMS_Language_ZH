ds_stack_top

该函数将只读取堆栈的第一个值（即“在顶部”的值）。它不会弹出该值，表示将来仍可以通过此函数或 ds_stack_pop()。如果堆栈为空，则该函数将返回常量 undefined，否则它将返回堆栈中包含的实数或字符串值。

句法：

ds_stack_top(id);

争论描述
id要读取的数据结构的ID。

返回值：

Data type value stored in the stack or undefined

例子：

num = ds_stack_top(control_stack);

上面的代码将从变量“ control_stack”中索引的堆栈中读取值，并将返回值存储在变量“ num”中。
