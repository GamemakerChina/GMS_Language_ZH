ds_stack_write

该函数返回一个字符串，然后可以使用 ds_stack_read()功能。

注意：返回的字符串不是人类可读的字符串，而是数据结构内容的转储

句法：

ds_stack_write(id);

争论描述
id要写入的数据结构的ID。

返回值：

String

例子：

ini_open("save.ini");
var str = ds_stack_write(stack);
ini_write_string("Stacks", "0", str);
ds_stack_clear(stack);
ini_close();

上面的代码打开一个ini文件，然后写一个字符串，其中包含在变量“ stack”中索引的DS堆栈中存储的信息。然后清除堆栈并关闭ini文件。
