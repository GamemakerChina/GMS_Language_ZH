ds_exists

通过这个函数，你可以检查一个给定类型的数据结构是否存在。你提供 "索引 "值（在一个变量中持有）和DS "类型"，它可以是下面列出的任何一个常量，该函数将返回 true如果数据结构存在，否则为false。

恒定 描述
ds_type_map 一个地图数据结构
ds_type_list 一个列表数据结构
ds_type_stack 一个堆栈数据结构
ds_type_grid 一个网格数据结构
ds_type_queue 一个队列数据结构
ds_type_priority 一个优先权数据结构

语法。

ds_exists(ind, type)。

争论 描述
ind 用于检查数据结构的变量索引
类型 要检查的数据结构类型（见下面的常量列表）。

返回。

布尔型

例子。

if !ds_exists(ai_grid, ds_type_grid)
{
ai_grid = ds_grid_create（room_width / 32, room_height / 32）;
}

上面的代码检查（先前初始化的）变量 "ai_grid"，看它是否索引了一个DS网格类型的数据结构，如果没有，就创建一个，并将其索引存入变量。
