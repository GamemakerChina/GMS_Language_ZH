ds_exists

使用此功能，您可以检查给定类型的数据结构是否存在。您提供“索引”值（保留在变量中）和DS“类型”，可以是下面列出的任何常量，该函数将返回 true如果数据结构存在，并且 false否则。

持续的描述
ds_type_map地图数据结构
ds_type_list列表数据结构
ds_type_stack堆栈数据结构
ds_type_grid网格数据结构
ds_type_queue队列数据结构
ds_type_priority优先级数据结构

句法：

ds_exists(ind, type);

争论描述
ind用于检查数据结构的变量索引
type要检查的数据结构类型（请参见下面的常量列表）

返回值：

Boolean

例子：

if !ds_exists(ai_grid, ds_type_grid)
    {
    ai_grid = ds_grid_create(room_width / 32, room_height / 32);
    }

上面的代码检查（先前初始化的）变量“ ai_grid”以查看是否为DS网格类型数据结构建立索引，如果没有，则创建一个并将其索引存储在变量中。

返回：数据结构
