fps

在GameMaker Studio 2 ，有两种主要方法可用来判断游戏的运行速度。这 game speed（在“游戏选项”中指定）和 fps（每秒帧数）。这些值通常会引起混淆，但基本上一个是GameMaker Studio 2在一秒钟内应该完成的游戏步骤数，而另一个（fps）是GameMaker Studio 2在一个时间内实际完成的CPU步骤数。第二到房间速度本身的最大值。要获得真实的fps，即每个游戏步骤的实际cpu步骤数，请使用 fps_real多变的。

此只读变量将当前fps作为整数值返回。请注意，该功能只会在游戏的每个步骤中更新一次，因此似乎从一个值“跳”到另一个值，但这是很正常的。

句法：

fps

返回值：

Real

例子：

if debug_mode
   {
   draw_text(32, 32, "FPS = " + string(fps));
   }

上面的代码将检查游戏是否处于调试模式，如果是，它将在屏幕上显示当前fps。

返回：调试
