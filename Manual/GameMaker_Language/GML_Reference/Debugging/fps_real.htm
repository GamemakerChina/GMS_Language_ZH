fps_real

在GameMaker Studio 2 ，有两种主要方法可用来判断游戏的运行速度。这 game speed（在“游戏选项”中指定）和 fps（每秒帧数）。这些值通常会造成混淆，但是基本上一个是GameMaker Studio 2在一秒钟内应该完成的游戏步骤数（游戏速度），而另一个是GameMaker Studio 2实际上在一秒内完成的CPU步骤数。秒（实际fps），该值通常比游戏速度高得多，但随着游戏变得更加复杂并使用更多处理能力来维持设定的房间速度，该值将下降。

此只读变量将当前fps作为整数值返回。请注意，该功能只会在游戏的每个步骤中更新一次，因此似乎从一个值“跳”到另一个值，但这是很正常的。

注意：在HTML5目标上，此变量将返回监视器刷新率的倍数，因为GameMaker Studio 2必须依靠浏览器进行计时和发送。

句法：

fps_real

返回值：

Real

例子：

if debug_mode
   {
   draw_text(32, 32, "FPS = " + string(fps_real));
   }

上面的代码将检查游戏是否处于调试模式，如果是，它将在屏幕上显示当前的真实fps。

返回：调试
