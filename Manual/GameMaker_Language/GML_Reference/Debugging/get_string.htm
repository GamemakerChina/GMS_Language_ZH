get_string

这将创建一个弹出窗口，显示一个标准消息，带有一个标记为“确定”的按钮，提示用户输入字符串。如果没有已经进入该函数将返回输入字符串，或默认值。

注意：此功能仅在桌面目标上供调试使用，但在所有其他目标上已弃用。

句法：

get_string(str, def);

争论描述
str在弹出消息中显示的字符串。
def文本框中的默认字符串。

返回值：

String

例子：

global.test_name = get_string("Test highscore name:", "Anonymous");

上面的代码将提示用户输入一个名称，然后将其存储在全局变量“ test_name”中。如果未输入任何内容，而用户仅按“确定”，则将返回默认值“匿名”。

返回：调试
