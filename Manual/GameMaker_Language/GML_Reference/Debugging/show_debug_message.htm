show_debug_message

此函数将创建一个自定义调试消息，该消息将在runtime显示在编译器窗口中。请注意，该消息可以是字符串，也可以是实数，但是如果您同时需要这两个信息，则必须先使用 string()函数（请参见下面的示例），如果该数字有两个以上的小数位，则应使用 string_format()以默认显示它们，小数点后四舍五入到最接近的两位小数（因此“ 1.2468”在输出窗口中将显示为“ 1.25”）。

当在“调试模式”下运行游戏时，此功能显示的调试消息将显示在IDE底部的“编译器输出”窗口以及调试器的“图形视图”中。如果您只想在“调试模式”下查看消息，则可能应该使用 debug_event()反而。

句法：

show_debug_message(string);

争论描述
string要显示的自定义调试消息。

返回值：

N/A

例子：

if !instance_exists(obj_Exit)
   {
   show_debug_message("Exit not created!");
   show_debug_message("Error Num: " + string(global.error));
   game_end();
   }

上面的代码检查实例是否存在，如果不存在，则在编译窗口中显示调试消息，然后游戏结束。
