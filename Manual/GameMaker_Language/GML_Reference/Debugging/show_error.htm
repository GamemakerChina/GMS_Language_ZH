show_error

这个函数将显示一个自定义的字符串作为错误信息，如果有必要，还可以选择事后中止游戏。

注意：该功能仅用于调试。

语法。

show_error(str, abort)。

争论 描述
弦外之音 弹出信息中要显示的字符串。
舍弃 是否中止（真）或不中止（假）。

返回。

不适用

例子。

if (room != rm_Dungeon)
{
show_error("Error: Went to wrong area. Aborting game.", true);
}

上述代码将检查玩家所在的房间是否是 "rm_Dungeon"，如果不是，它将显示一个错误信息并迫使用户中止游戏。

返回。调试
