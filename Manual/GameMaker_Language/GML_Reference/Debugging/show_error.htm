show_error

此函数将显示一个自定义字符串作为错误消息，并在需要时提供选项以中止游戏。

注：此功能仅用于调试使用。

句法：

show_error(str, abort);

争论描述
str在弹出消息中显示的字符串。
abort是否中止（是）（真）或否（假）。

返回值：

N/A

例子：

if (room != rm_Dungeon)
    {
    show_error("Error: Went to wrong area. Aborting game.", true);
    }

上面的代码将检查玩家所在的房间是否为“ rm_Dungeon”，如果不是，它将显示错误消息并强制用户中止游戏。

返回：调试
