显示信息

该函数创建一个弹出式消息框，显示给定的字符串和一个标有 "Ok "的按钮来关闭它。

注意：该函数仅在桌面目标和UWP上用于调试，但在所有其他目标上被弃用。

语法。

show_message(str)。

争论 描述
弦外之音 弹出信息中要显示的字符串。

返回。

不适用

例子。

var tot = 0;
for (var i = 0; i < 10; i += 1)
{
tot += inv[i];
}
show_message("Total = " + string(tot))。

上面的代码将循环浏览存储在数组 "inv "中的数值，并将它们添加到变量 "tot "中，然后显示一个包含总数的消息。

返回。调试
