show_message

此函数创建一个弹出消息框，其中显示代码中定义的字符串以及标记为“确定”的按钮以将其关闭。

注意：此功能仅在桌面目标和UWP上供调试使用，但在所有其他目标上已弃用。

句法：

show_message(str);

争论描述
str在弹出消息中显示的字符串。

返回值：

N/A

例子：

var tot = 0;
for (var i = 0; i < 10; i += 1)
   {
   tot += inv[i];
   }
show_message("Total = " + string(tot));

上面的代码将遍历存储在数组“ inv”中的值，并将它们添加到变量“ tot”，然后显示带有总计的消息。
