show_question

此功能创建一个带有两个按钮的弹出消息框，分别是“是”和“否”。它返回 true或者 false取决于用户按下两个按钮中的哪一个。

注意：此功能仅在桌面目标上供调试使用，但在所有其他目标上已弃用。

句法：

show_question(str);

争论描述
str要在弹出问题中显示的字符串。

返回值：

Boolean

例子：

if (score > 500) && debug_mode
   {
   if show_question("Continue to next room?")
      {
      room_goto(rm_Level2);
      }
   else game_end();
   }

上面的代码将检查得分，如果得分超过500，它将询问用户是否希望继续，如果单击“是”按钮，它将进入另一个房间，但是如果“否”按钮是选择它将结束游戏。

返回：调试
