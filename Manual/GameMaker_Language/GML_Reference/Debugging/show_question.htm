显示_问题

这个函数创建了一个弹出式消息框，有 "是 "和 "否 "两个按钮。它返回真或假，取决于用户按下这两个按钮中的哪一个。

注意：这个函数只在桌面目标上用于调试，但在所有其他目标上已被废弃。

语法。

show_question(str)。

争论 描述
弦外之音 弹出问题中要显示的字符串。

返回。

布尔型

例子。

if (score > 500) && debug_mode
{
if show_question("Continue to next room?")
{
room_goto(rm_Level2);
}
else game_end();
}

上述代码将检查分数，如果超过500分，它将询问用户是否愿意继续，如果点击 "是 "按钮，将进入另一个房间，但如果选择 "否 "按钮，将结束游戏。

返回。调试
