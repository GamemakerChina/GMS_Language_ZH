基本形式

GameMaker Studio 2具有用于绘制不同形式和形状的完整功能集合。这些函数仅应在实例的draw事件中使用（您可以在其他地方使用它们，但看不到任何东西！），并创建纯图形效果，这意味着它们不会生成碰撞事件，因为它们只能完成通过已分配给实例的精灵或碰撞遮罩。所有这些功能均受当前绘制颜色，alpha和混合模式的影响（请参见“颜色”部分）。

重要的！线条，点和表单始终以屏幕分辨率以1：1绘制，而不管它们被绘制的事件如何。

重要的！这些功能将破坏顶点批处理并增加游戏中的纹理交换次数，因此在单个并条机中多次调用其中的任何一个都可能导致性能问题。

存在以下用于绘制基本形状的绘制功能。请注意，这些函数调用缓慢，仅作为基本绘图工具提供：

  • draw_arrow
  • draw_circle
  • draw_circle_colour
  • draw_ellipse
  • draw_ellipse_colour
  • draw_line
  • draw_line_colour
  • draw_line_width
  • draw_line_width_colour
  • draw_point
  • draw_point_colour
  • draw_rectangle
  • draw_rectangle_colour
  • draw_roundrect
  • draw_roundrect_colour
  • draw_roundrect_ext
  • draw_roundrect_colour_ext
  • draw_triangle
  • draw_triangle_colour

GameMaker Studio 2中还包含以下功能，用于设置绘制精度并绘制一些非常具体的内容：

  • draw_set_circle_precision
  • draw_button
  • draw_healthbar
  • draw_path
