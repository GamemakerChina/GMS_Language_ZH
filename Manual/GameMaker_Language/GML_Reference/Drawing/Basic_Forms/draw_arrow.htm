绘制_箭头

这个函数将从点（x1,y1）到点（x2,y2）画一个箭头。箭头的茎是沿着这些点画的，实际的箭头是在末端画的，其中箭头的大小由参数 "size "定义，并且 被计算为箭杆的一部分，因此端点总是与x2,y2定义的位置对齐。箭头的宽度是按照长度的比例自动计算的。

注意：如果你想用着色器来画一个形状，你应该知道，大多数着色器都希望有以下输入：顶点、纹理、颜色。然而，当使用这个函数时，只有顶点和颜色数据被传入，所以 着色器可能不会画出任何东西（或者画出一些东西但不正确）。如果你需要以这种方式绘制形状，那么着色器的定制应该考虑到这一点。

语法。

draw_arrow(x1, y1, x2, y2, size)。

争论 描述
x1 线条起点的X坐标。
y1 线条起点的y坐标。
x2 线条末端的X坐标（箭头的终点）。
y2 线条末端的y坐标（箭头的终点）。
尺寸 箭头的长度，单位是像素。

返回。

不适用

例子。

draw_arrow(x, y, mouse_x, mouse_y, 10)。

上面的代码将从实例的位置到鼠标的位置画一个箭头，箭头的长度为10像素。
