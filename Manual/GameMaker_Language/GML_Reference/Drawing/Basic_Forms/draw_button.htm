绘制按钮

这个函数将使用当前选择的绘制颜色和阿尔法绘制一个非常简单的矩形 "按钮"，其中向上参数定义了斜边效果的外观，如下图所示。

注意：如果你想用着色器来画一个形状，你应该知道大多数着色器希望有以下输入：顶点、纹理、颜色。 以下输入：顶点、纹理、颜色。然而，当使用这个函数时，只有顶点和颜色数据被传入，因此着色器可能不会画出任何东西（或者画出一些东西但不正确）。如果你需要以这种方式来绘制形状，那么 着色器的定制应该考虑到这一点。

语法。

draw_button(x1, y1, x2, y2, up);

争论 描述
x1 按钮左边的X坐标
y1 按钮顶部的Y坐标
x2 按钮右侧的X坐标
y2 按钮底部的y坐标
了 按钮是向上（真）还是向下（假）。

返回。

不适用

例子。

draw_button(100, 100, 200, 150, !mouse_check_button(mb_left)) 。

这将画出一个按钮，如果按住鼠标左键，这个按钮就会显示被按下。
