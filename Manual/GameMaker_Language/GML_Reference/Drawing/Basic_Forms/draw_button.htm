draw_button

此函数将使用当前选择的绘制颜色和Alpha绘制一个非常简单的矩形“按钮”，其中up参数定义斜面边缘效果的外观，如下图所示：

注意：如果要使用shader绘制形状，则应注意，大多数shaders需要以下输入：顶点，纹理，颜色。但是，使用此功能时，仅传递顶点和颜色数据，因此shader可能不会绘制任何内容（或绘制某些内容，但绘制不正确）。如果需要以这种方式绘制形状，则应考虑到自定义shader

句法：

draw_button(x1, y1, x2, y2, up);

争论描述
x1按钮左侧的x坐标
y1按钮顶部的y坐标
x2按钮右侧的x坐标
y2按钮底部的y坐标
up按钮是否向上（true ）或下（false ）

返回值：

N/A

例子：

draw_button(100, 100, 200, 150, !mouse_check_button(mb_left));

这将绘制一个按钮，如果按住鼠标左键，该按钮将显示为按下状态。
