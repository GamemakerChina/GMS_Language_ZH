draw_circle

使用此功能，您可以绘制圆的轮廓或实心的圆。您可以使用函数定义图纸的精确度 draw_set_circle_precision()。

注意：如果要使用shader绘制形状，则应注意，大多数shaders需要以下输入：顶点，纹理，颜色。但是，使用此功能时，仅传递顶点和颜色数据，因此shader可能不会绘制任何内容（或绘制某些内容，但绘制不正确）。如果需要以这种方式绘制形状，则应考虑到自定义shader

句法：

draw_circle(x, y, r, outline);

争论描述
x圆心的x坐标。
y圆心的y坐标。
r圆的半径（从中心到边缘的长度）
outline圆是否被绘制填充（false ）或一像素宽的轮廓（true ）。

返回值：

N/A

例子：

draw_set_colour(c_white);
draw_circle(100, 100, 50, true);

这将绘制一个半径为50像素的1像素宽的白色圆圈轮廓。
