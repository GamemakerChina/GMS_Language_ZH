绘制圆圈

通过这个函数，你可以画一个圆的轮廓或一个填充圆。你可以用函数定义绘制的精确程度 draw_set_circle_precision().

注意：如果你想用着色器来画一个形状，你应该知道，大多数着色器都希望有以下输入：顶点、纹理、颜色。然而，当使用这个函数时，只有顶点和颜色数据被传入，所以 着色器可能不会画出任何东西（或者画出一些东西但不正确）。如果你需要以这种方式绘制形状，那么着色器的定制应该考虑到这一点。

语法。

draw_circle(x, y, r, outline);

争论 描述
x 圆心的X坐标。
y 圆心的Y坐标。
r 圆的半径（从圆心到圆边的长度）。
纲要 圆圈是被填充（false）还是作为一个像素宽的轮廓（true）来绘制。

返回。

不适用

例子。

draw_set_colour(c_white);
draw_circle(100, 100, 50, true);

这将画出一个一像素宽的白色圆形轮廓，半径为50像素。
