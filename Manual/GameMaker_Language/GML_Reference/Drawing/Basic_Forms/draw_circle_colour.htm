draw_circle_colour

使用此功能，您可以绘制圆的轮廓或实心的圆，如果已填充，则可以定义内部和外部填充颜色。如果这些颜色不相同，则您将获得一种到另一种的渐变效果，并且颜色设置将超过使用该功能设置的基本颜色 draw_set_colour()。您可以使用函数定义图纸的精确度 draw_set_circle_precision()。

注意：如果要使用shader绘制形状，则应注意，大多数shaders需要以下输入：顶点，纹理，颜色。但是，使用此功能时，仅传递顶点和颜色数据，因此shader可能不会绘制任何内容（或绘制某些内容，但绘制不正确）。如果需要以这种方式绘制形状，则应考虑到自定义shader

句法：

draw_circle_colour(x, y, r, col1, col2, outline);

争论描述
x圆心的x坐标。
y圆心的y坐标。
r圆的半径（从中心到边缘的距离），以像素为单位。
col1圆心的颜色。
col2圆边缘的颜色。
outline圆圈是否为轮廓（true ） 或不 （false ）。如果为true，则col1不相关。

返回值：

N/A

例子：

draw_circle_colour(x, y, 100, c_white, c_black, false);

这将绘制一个填充圆，其中心位于执行实例的x和y位置，半径为100像素，从中心的白色到外部的黑色。
