draw_ellipse

使用此功能，您可以通过定义矩形区域来绘制椭圆的轮廓或填充的椭圆，然后将其创建为适合的椭圆。您可以使用函数定义图纸的精确度 draw_set_circle_precision()。

注意：如果要使用shader绘制形状，则应注意，大多数shaders需要以下输入：顶点，纹理，颜色。但是，使用此功能时，仅传递顶点和颜色数据，因此shader可能不会绘制任何内容（或绘制某些内容，但绘制不正确）。如果需要以这种方式绘制形状，则应考虑到自定义shader

句法：

draw_ellipse(x1, y1, x2, y2, outline);

争论描述
x1椭圆左侧的x坐标。
y1椭圆顶部的y坐标。
x2椭圆右边的x坐标。
y2椭圆底部的y坐标。
outline椭圆是绘制为实心（false）还是绘制为一个像素宽的轮廓（true）。

返回值：

N/A

例子：

draw_set_colour(c_white);
draw_ellipse(100, 100, 300, 200, false);

这将在定义的矩形区域内绘制一个填充的椭圆。
