draw_ellipse_colour

使用此功能，您可以通过定义矩形区域来绘制椭圆的轮廓或填充的椭圆，然后将其创建为适合的椭圆。如果已填充，则可以定义内部和外部填充颜色。如果这些颜色不相同，则您将获得一种到另一种的渐变效果（颜色设置将覆盖使用该功能设置的基本颜色 draw_set_colour()）。您可以使用函数定义图纸的精确度 draw_set_circle_precision()。

注意：如果要使用shader绘制形状，则应注意，大多数shaders需要以下输入：顶点，纹理，颜色。但是，使用此功能时，仅传递顶点和颜色数据，因此shader可能不会绘制任何内容（或绘制某些内容，但绘制不正确）。如果需要以这种方式绘制形状，则应考虑到自定义shader

句法：

draw_ellipse_colour(x1, y1, x2, y2, col1, col2, outline);

争论描述
x1椭圆左侧的x坐标。
y1椭圆顶部的y坐标。
x2椭圆右边的x坐标。
y2椭圆底部的y坐标。
col1中心的颜色。
col2边缘的颜色。
outline椭圆是轮廓（true）还是填充（false）。

返回值：

N/A

例子：

draw_ellipse_colour(50, 50, 200, 100, c_black, c_white, false);

这将绘制一个填充的，水平拉伸的椭圆，其中心为黑色，边缘为白色。
