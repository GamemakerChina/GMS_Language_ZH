draw_healthbar

使用此功能，您可以绘制彩色条以显示恒定值。尽管该函数使用单词“ healthbar”，但只要显示的数量是介于0到100之间的百分比值（可以使用公式（CURRENT_Value / MAXIMUM_value ）* 100 ），因此，例如，您可以使用它来显示力量，生命值，法力，时间或其他可从条形显示中受益的内容。

您可以设置各种不同的东西来改变条形的外观，如下图所示（请注意，此处使用的颜色是 c_green为最大值和 c_red为最小值，但您可以使用所需的颜色）：

注意：如果要使用shader绘制形状，则应注意，大多数shaders需要以下输入：顶点，纹理，颜色。但是，使用此功能时，仅传递顶点和颜色数据，因此shader可能不会绘制任何内容（或绘制某些内容，但绘制不正确）。如果需要以这种方式绘制形状，则应考虑到自定义shader

句法：

draw_healthbar(x1, y1, x2, y2, amount, backcol, mincol, maxcol, direction, showback, showborder);

争论描述
x1运行状况栏左侧的x坐标
y1健康栏顶部的y坐标
x2健康栏右侧的x坐标
y2保健栏底部的y坐标
amount定义总体健康状况的变量（0到100之间）
backcol栏背景的颜色
mincol不健康时条的颜色（0）
maxcol处于完全健康状态时条的颜色（100）
direction条是“锚定的”（0 =左，1 =右，2 =顶部，3 =底部）
showback是否绘制了酒吧的背景（true ） 或不 （false ）。如果为false，则忽略backcol参数。
showborder栏的元素是否具有1像素宽的黑色边框（true ） 或不 （false ）。

返回值：

N/A

例子：

var pc;
pc = (hp / max_hp) * 100;
draw_healthbar(100, 100, 500, 200, pc, c_black, c_red, c_lime, 0, true, true)

上面的代码使用从变量“ hp”和“ hp_max”获得的百分比值绘制标准的红色/绿色健康栏。
