draw_line_colour

使用此功能，您可以绘制一条1像素宽的线，并在第一个点的颜色1和第二个点的颜色2之间混合颜色。颜色设置将覆盖使用该功能设置的基本颜色 draw_set_colour()。请注意，由于各种支持的平台之间的差异，所绘制的线条可能需要使用不同的值（x，y上为+/- 1）来绘制所需的尺寸。

注意：如果要使用shader绘制形状，则应注意，大多数shaders需要以下输入：顶点，纹理，颜色。但是，使用此功能时，仅传递顶点和颜色数据，因此shader可能不会绘制任何内容（或绘制某些内容，但绘制不正确）。如果需要以这种方式绘制形状，则应考虑到自定义shader

句法：

draw_line_colour(x1, y1, x2, y2, col1, col2);

争论描述
x1行首的x坐标。
y1行首的y坐标。
x2行尾的x坐标。
y2行尾的y坐标。
col1行开始的颜色。
col2行尾的颜色。

返回值：

N/A

例子：

draw_line_colour(50, 50, 300, 50, c_red, c_blue);

这将绘制一条从（50,50）到（300,50）的水平线，并具有从红色到蓝色的平滑渐变。
