draw_path

使用此功能，您可以使GameMaker Studio 2在屏幕上绘制路径。路径将被绘制为一条简单的线，并且可以相对于调用实例，也可以相对于在路径编辑器中或通过代码创建的绝对位置。在调试动态路径时（例如，为具有 mp_grid_path()功能）。

注意：如果要使用shader绘制形状，则应注意，大多数shaders需要以下输入：顶点，纹理，颜色。但是，使用此功能时，仅传递顶点和颜色数据，因此shader可能不会绘制任何内容（或绘制某些内容，但绘制不正确）。如果需要以这种方式绘制形状，则应考虑到自定义shader

句法：

draw_path(path, x, y, absolute);

争论描述
path画的路
x绘制路径的x坐标
y绘制路径的y坐标
absolute是否在绝对位置绘制路径（true ）或相对位置（false ）

返回值：

N/A

例子：

if mp_grid_path(grid, path, x, y, obj_Player.x, obj_Player.y, 1)
    {
    draw_path(path, x, y, false);
    }

上面的代码将使用 mp_grid_path函数生成路径并将其存储在变量“ path”中。如果成功创建了路径，则将其绘制在屏幕上相对于运行代码的实例的位置。
