draw_point_colour

使用此功能，您可以使用定义的颜色在屏幕上的任何位置绘制单个像素。颜色设置将覆盖使用该功能设置的基本颜色 draw_set_colour()。

注意：如果要使用shader绘制形状，则应注意，大多数shaders需要以下输入：顶点，纹理，颜色。但是，使用此功能时，仅传递顶点和颜色数据，因此shader可能不会绘制任何内容（或绘制某些内容，但绘制不正确）。如果需要以这种方式绘制形状，则应考虑到自定义shader

句法：

draw_point_colour(x, y, col1);

争论描述
x点的x坐标。
y点的y坐标。
col1点的颜色。

返回值：

N/A

例子：

draw_point_colour(50, 50, c_red);

这将在（50,50）处绘制一个红色像素。
