draw_rectangle_colour

使用此功能，您可以通过定义（x1，y1）位置为左上角和（x2，y2）位置为右下角的区域来绘制矩形或实心矩形的轮廓。如果已填充，则可以为每个角点定义单独的颜色，如果这些颜色不相同，则将获得一种渐变效果（颜色设置将超过使用该功能设置的基本颜色）draw_set_colour() ）。请注意，由于各种支持平台的差异，所绘制的矩形可能需要使用不同的值（在x，y或宽度或高度上为+/- 1）进行绘制，并具有所需的尺寸。

注意：如果要使用shader绘制形状，则应注意，大多数shaders需要以下输入：顶点，纹理，颜色。但是，使用此功能时，仅传递顶点和颜色数据，因此shader可能不会绘制任何内容（或绘制某些内容，但绘制不正确）。如果需要以这种方式绘制形状，则应考虑到自定义shader

句法：

draw_rectangle_colour(x1, y1, x2, y2, col1, col2, col3, col4, outline);

争论描述
x1矩形左侧的x坐标。
y1矩形顶部的y坐标。
x2矩形右侧的x坐标。
y2矩形底部的y坐标。
col1左上角的颜色。
col2右上角的颜色。
col3右下角的颜色。
col4左下角的颜色。
outline矩形是轮廓（true）还是填充（false）。

返回值：

N/A

例子：

draw_rectangle_colour(50, 50, 200, 200, c_black, c_white, c_black, c_white, false);

这将在圆角之间绘制一个平滑的黑色/白色渐变的填充正方形。
