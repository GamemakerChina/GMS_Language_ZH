绘画_圆角_颜色

用这个函数你可以画一个圆角矩形的轮廓，或者画一个填充的圆角矩形，其中（x1,y1）的位置是左上角，（x2,y2）的位置是右下角。如果它是填充的，你可以定义中心和边缘的各个 的颜色，如果这些颜色不一样，你会得到一个从一个到另一个的渐变效果（颜色设置将覆盖用函数设置的基础颜色 draw_set_colour())。 你可以用函数定义角的绘制的精确程度。 draw_set_circle_precision()你可以用函数定义角的绘制精度，但角总是以固定的半径绘制。如果你需要改变角的半径 你应该使用函数 draw_roundrect_colour_ext().请注意，被绘制的矩形可能需要不同的值（X、Y、宽度或高度上的+/-1）来绘制所需的尺寸。 由于各种支持的平台之间的差异，被绘制的矩形可能需要不同的值（在x、y、或宽度或高度上的+/-1）来绘制所需的尺寸。

注意：如果你想用着色器来画一个形状，你应该知道，大多数着色器都希望有以下输入：顶点、纹理、颜色。然而，当使用这个函数时，只有顶点和颜色数据被传入，所以 着色器可能不会画出任何东西（或者画出一些东西但不正确）。如果你需要以这种方式绘制形状，那么着色器的定制应该考虑到这一点。

语法。

draw_roundrect_colour(x1, y1, x2, y2, col1, col2, outline)。

争论 描述
x1 圆角矩形左边的X坐标。
y1 圆角矩形顶部的y坐标。
x2 圆角矩形右边的X坐标。
y2 圆角矩形底部的y坐标。
col1 中间的颜色。
col2 外边缘的颜色。
纲要 矩形是否为轮廓（真）或填充（假）。

返回。

不适用

例子。

draw_roundrect_colour(50, 50, 200, 200, c_black, c_white, false)。

这将画出一个有圆角的填充的正方形，并从中心到边缘有一个平滑的黑/白渐变。
