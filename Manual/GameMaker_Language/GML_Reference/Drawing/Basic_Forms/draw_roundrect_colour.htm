draw_roundrect_colour

使用此功能，您可以绘制圆角矩形或填充圆角矩形的轮廓，其中（x1，y1）位置是左上角，（x2，y2）位置是右下角。如果已填充，则可以定义中心和边缘的各个颜色，如果这些颜色不相同，则将获得从一种到另一种的渐变效果（颜色设置将覆盖使用功能 draw_set_colour()）。您可以使用函数定义拐角的绘制精度 draw_set_circle_precision()，但拐角始终以固定的半径绘制。如果需要更改拐角半径，则应使用该功能 draw_roundrect_colour_ext()。请注意，由于各种支持平台的差异，所绘制的矩形可能需要使用不同的值（在x，y或宽度或高度上为+/- 1）进行绘制，并具有所需的尺寸。

注意：如果要使用shader绘制形状，则应注意，大多数shaders需要以下输入：顶点，纹理，颜色。但是，使用此功能时，仅传递顶点和颜色数据，因此shader可能不会绘制任何内容（或绘制某些内容，但绘制不正确）。如果需要以这种方式绘制形状，则应考虑到自定义shader

句法：

draw_roundrect_colour(x1, y1, x2, y2, col1, col2, outline);

争论描述
x1圆角矩形左侧的x坐标。
y1圆角矩形顶部的y坐标。
x2圆角矩形右边的x坐标。
y2圆角矩形底部的y坐标。
col1中心色。
col2外边缘颜色。
outline矩形是轮廓（true）还是填充（false）。

返回值：

N/A

例子：

draw_roundrect_colour(50, 50, 200, 200, c_black, c_white, false);

这将绘制一个填充的正方形，该正方形具有圆角，并且从中心到边缘具有平滑的黑白渐变。
