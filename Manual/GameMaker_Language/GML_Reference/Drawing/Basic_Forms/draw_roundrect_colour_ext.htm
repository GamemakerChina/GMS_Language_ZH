draw_roundrect_colour_ext

使用此功能，您可以绘制圆角矩形或填充圆角矩形的轮廓，其中（x1，y1）位置是左上角，（x2，y2）位置是右下角。如果填充了矩形，则将使用颜色参数生成从中心到边缘的颜色渐变，其中颜色1是中心颜色，颜色2是边缘颜色。您还必须提供x和y轴的半径值（以像素为单位），并且拐角将被这些量舍入。您可以使用函数定义转角的绘制精度 draw_set_circle_precision()。请注意，由于各种支持平台的差异，所绘制的矩形可能需要使用不同的值（在x，y或宽度或高度上为+/- 1）进行绘制，并具有所需的尺寸。

注意：如果要使用shader绘制形状，则应注意，大多数shaders需要以下输入：顶点，纹理，颜色。但是，使用此功能时，仅传递顶点和颜色数据，因此shader可能不会绘制任何内容（或绘制某些内容，但绘制不正确）。如果需要以这种方式绘制形状，则应考虑到自定义shader

句法：

draw_roundrect_colour_ext(x1, y1, x2, y2, xrad, yrad, col1, col2, outline);

争论描述
x1圆角矩形左侧的x坐标。
y1圆角矩形顶部的y坐标。
x2圆角矩形右边的x坐标。
y2圆角矩形底部的y坐标。
xrad从矩形角沿x轴的曲线半径。
yrad从矩形角沿y轴的曲线半径。
col1中心色。
col2外边缘颜色。
outline矩形是轮廓（true）还是填充（false）。

返回值：

N/A

例子：

var dist = point_distance(x, y, mouse_x, mouse_y) / 10;
var col = make_colour_hsv(clamp(dist, 0, 255), 255, 255);
draw_roundrect_colour_ext(x - 50, y - 50, x + 50, y + 50, dist, dist, col, c_white, 0);

这将绘制一个带有圆角的实心正方形，其半径由绘制实例到鼠标的距离定义。此值也用于计算中间混合色。
