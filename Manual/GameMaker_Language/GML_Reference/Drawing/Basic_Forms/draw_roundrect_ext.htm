draw_roundrect_ext

使用此功能，您可以绘制圆角矩形或填充圆角矩形的轮廓，其中（x1，y1）位置是左上角，（x2，y2）位置是右下角。您还必须提供x和y轴的半径值（以像素为单位），并且拐角将被这些量舍入。您可以使用函数定义转角的绘制精度 draw_set_circle_precision()。请注意，由于各种支持平台的差异，所绘制的矩形可能需要使用不同的值（在x，y或宽度或高度上为+/- 1）进行绘制，并具有所需的尺寸。

注意：如果要使用shader绘制形状，则应注意，大多数shaders需要以下输入：顶点，纹理，颜色。但是，使用此功能时，仅传递顶点和颜色数据，因此shader可能不会绘制任何内容（或绘制某些内容，但绘制不正确）。如果需要以这种方式绘制形状，则应考虑到自定义shader

句法：

draw_roundrect_ext(x1, y1, x2, y2, xrad, yrad, outline);

争论描述
x1圆角矩形的左上角的x坐标（就好像它是一个普通矩形一样）。
y1圆角矩形左上角的y坐标（就好像它是一个普通矩形一样）。
x2圆角矩形右下角的x坐标（就好像是一个普通矩形）。
y2圆角矩形右下角的y坐标（就好像它是一个普通矩形一样）。
xrad从矩形角沿x轴的曲线半径。
yrad从矩形角沿y轴的曲线半径。
outline绘制的圆角矩形是填充（false）还是一像素宽的轮廓（true）。

返回值：

N/A

例子：

var dist = point_distance(x, y, mouse_x, mouse_y) / 10;
draw_set_colour(c_lime);
draw_roundrect_ext(x - 50, y - 50, x + 50, y + 50, dist, dist, 0);

这将绘制一个填充的圆角矩形，并根据鼠标相对于绘制实例的位置将圆角四舍五入。
