draw_set_circle_precision

为了在绘制圆形时优化性能， GameMaker Studio 2基本上绘制了一个多边形，并带有足够多的边使其显示为圆形。但是，根据所需的圆圈大小，您可能会发现更改此值可以帮助提高游戏性能或使圆圈看起来更好。您可以输入此函数的精度值是圆形多边形的边数，并且该值可以是4到64之间的任何数字，但必须是可以被4整除的数字，默认值为24。

注意：如果要使用shader绘制形状，则应注意，大多数shaders需要以下输入：顶点，纹理，颜色。但是，使用此功能时，仅传递顶点和颜色数据，因此shader可能不会绘制任何内容（或绘制某些内容，但绘制不正确）。如果需要以这种方式绘制形状，则应考虑到自定义shader

句法：

draw_set_circle_precision(precision);

争论描述
precision调用此函数后绘制的所有圆的精度。

返回值：

N/A

例子：

draw_set_circle_precision(12);

上面的代码将使用12面多边形绘制所有圆。
