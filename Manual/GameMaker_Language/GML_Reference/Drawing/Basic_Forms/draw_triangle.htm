draw_triangle

使用此功能，您可以绘制三角形的轮廓或实心三角形。

注意：如果要使用shader绘制形状，则应注意，大多数shaders都需要以下输入：顶点，纹理，颜色。但是，使用此功能时，仅传递顶点和颜色数据，因此shader可能不会绘制任何内容（或绘制某些内容，但绘制不正确）。如果需要以这种方式绘制形状，则应考虑到自定义shader

句法：

draw_triangle(x1, y1, x2, y2, x3, y3, outline);

争论描述
x1三角形第一个角的x坐标。
y1三角形第一个角的y坐标。
x2三角形第二个角的x坐标。
y2三角形第二个角的y坐标。
x3三角形第三个角的x坐标。
y3三角形第三个角的y坐标。
outline是将三角形绘制为实心（false）还是绘制为一个像素宽的轮廓（true）。

返回值：

N/A

例子：

draw_set_colour(c_aqua);
draw_triangle(50, 50, 200, 50, 50, 200, 0);

这将绘制一个填充的海蓝宝石色等腰直角三角形，其第一个角为（50,50），第二个角为（200,50），第三个角为（50,200）。
