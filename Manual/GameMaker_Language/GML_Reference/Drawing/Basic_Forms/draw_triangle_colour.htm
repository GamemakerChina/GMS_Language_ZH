draw_triangle_colour

使用此功能，您可以绘制三角形的轮廓或实心三角形。如果已填充，则可以为每个角点定义单独的颜色，如果这些颜色不相同，则将获得一种渐变效果（颜色设置将超过使用该功能设置的基本颜色）draw_set_colour() ）。

注意：如果要使用shader绘制形状，则应注意，大多数shaders需要以下输入：顶点，纹理，颜色。但是，使用此功能时，仅传递顶点和颜色数据，因此shader可能不会绘制任何内容（或绘制某些内容，但绘制不正确）。如果需要以这种方式绘制形状，则应考虑到自定义shader

句法：

draw_triangle_colour(x1, y1, x2, y2, x3, y3, col1, col2, col3, outline);

争论描述
x1三角形第一个角的x坐标。
y1三角形第一个角的y坐标。
x2三角形第二个角的x坐标。
y2三角形第二个角的y坐标。
x3三角形第三个角的x坐标。
y3三角形第三个角的y坐标。
col1第一个角的颜色。
col2第二个角的颜色。
col3第三个角的颜色。
outline三角形是轮廓（true）还是填充（false）。

返回值：

N/A

例子：

draw_triangle_colour(200, 200, 300, 200, 200, 300, c_red, c_blue, c_blue, false);

这将绘制一个填充的等腰直角三角形，红色为直角，其他两个角为蓝色。
