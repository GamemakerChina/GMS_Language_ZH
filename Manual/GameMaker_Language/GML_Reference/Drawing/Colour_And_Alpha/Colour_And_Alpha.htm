颜色和阿尔法

GameMaker Studio 2具有全面的功能选择，可以操纵屏幕上所画内容的颜色和 alpha（关于使用特殊混合模式和其他 PGU 特定的图像混合 关于使用特殊混合模式和其他PGU特定功能的混合图像，请参见GPU控制部分。使用这些函数，你可以把颜色分成它们的组成部分，创建新的颜色，并操纵绘制alpha来改变你的游戏中不同事物的绘制方式。 来改变你的游戏中不同的东西如何被绘制在一起。除了这些函数之外，还有一些常数可以作为这些（和其他）函数的基础颜色。

请注意，你也可以事先用"$"符号从它们的十六进制值中创建颜色，这些值也可以在颜色函数中使用，以获得成分色调或用于混合等。这些颜色的十六进制值 被分成三部分，格式为蓝/绿/红，因此，例如，要定义一个紫色，你可以使用。

col = $983c95

其中98是蓝色成分，3c是绿色成分，95是红色成分。

  • 颜色_get_red
  • 颜色_获取色调
  • 颜色_获取饱和度
  • color_get_value
  • 绘画_获取像素
  • draw_getpixel_ext
  • 绘画_获取颜色
  • 绘画_获取阿尔法（draw_get_alpha

这些函数可以用来从颜色成分的原始输入值中创建颜色。

  • make_colour_hsv
  • make_colour_rgb
  • 合并颜色

以下函数可用于设置在屏幕上绘图的各种不同选项，包括阿尔法、颜色和混合。

  • 绘画_清空
  • 绘画_清空_alpha
  • draw_set_alpha
  • 绘画_设置颜色
