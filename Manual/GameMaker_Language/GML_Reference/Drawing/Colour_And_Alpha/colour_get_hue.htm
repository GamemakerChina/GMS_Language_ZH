colour_get_hue

此函数将返回给定颜色的色相。这是“纯”色调，它是用于定义颜色的色调，饱和度和值（亮度）方法的一部分。下图说明了此值如何对应于颜色的HSV刻度：

句法：

colour_get_hue(col);

争论描述
col检查颜色

返回值：

Int

例子：

col = merge_colour(colour_get_hue(c_teal), c_white, 0.5);

上面的代码获取用于使颜色常量“ c_teal”不变的色相，然后将其与白色以50:50的比例合并，并将所得颜色存储在变量“ col”中。
