colour_get_value

此函数将返回给定颜色的值（亮度）。这是混合到最终颜色中的“光”量，并且是用于定义颜色的色相，饱和度和值方法的一部分。下图说明了此值如何对应于颜色的HSV刻度：

句法：

colour_get_value(col);

争论描述
col检查颜色

返回值：

Int

例子：

col = make_colour_hsv(random(255), 255, colour_get_value(c_teal));

上面的代码获取用于使颜色常数“ c_teal”不变的值，然后使用该值将具有相同发光度的随机颜色设置为将颜色存储在变量“ col”中。
