draw_clear_alpha

此功能可用于清除整个屏幕具有给定色和目标alpha组件设置为您所设置的值-这个功能没有做任何的勾兑，因为它的作品，但任何后续的混合操作可以被设置为使用您设置的目标Alpha。这仅用于实例的draw事件中（如果在任何其他事件中使用，则不会显示），并且在新创建表面时清除表面也非常有用。

值得注意的是，当使用HTML5模块时，您可以使用此功能使画布背景透明，这将允许在宿主HTML上运行的游戏中看到宿主HTML中使用的任何图像。要实现此效果，您必须首先进入房间编辑器并从“房间编辑器”中删除所有“背景层”，然后取消选中“视图属性”中的“清除视图背景”选项。最后，您将需要制作一个新对象，将其放置在游戏的第一个房间中，并为其分配很高的深度（例如10000）。然后在Pre Draw事件中使用此功能将画布alpha设置为所需的颜色和透明度。

句法：

draw_clear_alpha(col, alpha);

争论描述
col清除屏幕的颜色
alpha清除屏幕的颜色的透明度

返回值：

N/A

例子：

draw_clear_alpha(c_red, 0.5);

这将以红色显示50％（0.5）透明度的透明屏幕。
