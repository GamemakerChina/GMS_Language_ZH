绘画_清晰_阿尔法

这个函数可以用来用一个给定的颜色清除整个屏幕，目标的alpha分量被设置为你所设置的值--这个函数在工作时不做任何混合，但任何后续的混合操作可以设置为使用你所设置的目标alpha。 但任何后续的混合操作都可以被设置为使用你所设置的目标alpha。这个功能只在实例的绘制事件中使用（如果在其他事件中使用，它将不会显示），当新创建的表面被清除时，它也会非常有用。

值得注意的是，当使用HTML5模块时，你可以使用这个函数使画布背景透明，这将允许在主机HTML中使用的任何图像在游戏运行时被看到。为了达到这个效果 你必须首先进入房间编辑器，删除房间编辑器中的任何背景层，然后取消勾选 "视图属性 "中的 "清除视图背景"选项。 最后，你需要制作一个新的对象，把它放在游戏的第一个房间里，并给它指定一个非常高的深度（例如，10000）。然后在预绘制事件中使用这个函数将画布的alpha设置为你希望的颜色和透明度。 你希望它具有的颜色和透明度。

语法。

draw_clear_alpha(col, alpha)。

争论 描述
殖民地 清理屏幕的颜色
阿尔法 将被清除的屏幕的颜色的透明度

返回。

不适用

例子。

draw_clear_alpha(c_red, 0.5);

这将以50%（0.5）的透明度清除屏幕上的红色。
