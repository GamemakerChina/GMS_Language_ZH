draw_get_colour

此函数返回当前的绘制颜色，该颜色用于绘制表单，文本，图元和无纹理的3D模型。可以使用 draw_set_colour()功能。

句法：

draw_get_colour()

返回值：

Real

例子：

if draw_get_colour() != c_black
   {
   draw_set_colour(c_black);
   }

上面的代码首先检查当前的绘制颜色，然后将其设置为 c_black它设置为 c_black。
