绘画_获取颜色

这个函数返回当前的绘制颜色，用于绘制表格、文本、基元和无纹理的三维模型。这可以通过 draw_set_colour()函数来设置。

语法。

draw_get_colour()

返回。

真实的

例子。

如果 draw_get_colour() != c_black
{
draw_set_colour(c_black);
}

上面的代码首先检查当前的绘图颜色，然后如果它没有被设置为c_black，就将它设置为c_black。
