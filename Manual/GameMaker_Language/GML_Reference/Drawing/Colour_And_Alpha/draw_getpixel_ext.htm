draw_getpixel_ext

使用此功能，您可以获得正在绘制到当前渲染目标的任何像素的完整abgr 32bit值。这意味着结果将取决于调用函数的事件以及所使用的目标表面。

注意：此功能会严重影响性能，因此仅在绝对必要时使用。

句法：

draw_getpixel_ext(x, y);

争论描述
x要检查的像素的x坐标
y要检查的像素的y坐标

返回值：

Real

例子：

col = draw_getpixel_ext(mouse_x, mouse_y);
alpha = (col >> 24) & 255;
blue = (col >> 16) & 255;
green = (col >> 8) & 255;
red = col & 255;

上面的代码将在鼠标的位置获取32位颜色值，然后将其拆分为其分量值，并将其存储在变量中。
