draw_set_alpha

使用此功能，您可以设置游戏的基本绘制Alpha。该值可以设置为0到1，其中0完全透明，1完全不透明（默认值），并且将影响所有进一步的绘制，包括背景，精灵，字体，图元和3D。

请注意，如果您要绘制任何资产并提供Alpha参数作为函数的一部分（例如，当使用 draw_sprite_ext()），则此处设置的alpha值将被覆盖，因此使用 draw_sprite_ext()alpha参数为1时，无论将alpha设置为什么，都将使精灵完全不透明 draw_set_alpha()（当使用 draw_self()并设置 image_alpha实例）。另请注意，当实例为默认绘图时（即：它具有精灵但没有Draw事件），则 image_alpha变量将被限制为带有 draw_set_alpha()，这样 image_alpha为1且绘制alpha为0.5时将以0.4绘制精灵。image_alpha的0.25和3.5的绘制alpha将以0.25 alpha绘制精灵。

句法：

draw_set_alpha(alpha);

争论描述
alpha要设置的Alpha（介于0和1之间）

返回值：

N/A

例子：

draw_set_alpha(0.5);
draw_set_colour(c_black);
draw_text(x+5, y+5, "LEVEL 1");
draw_set_alpha(1);
draw_set_colour(c_white);
draw_text(x, y, "LEVEL 1");

上面的代码将在指定位置绘制一些文本，并通过修改的绘制Alpha和颜色创建阴影效果。
