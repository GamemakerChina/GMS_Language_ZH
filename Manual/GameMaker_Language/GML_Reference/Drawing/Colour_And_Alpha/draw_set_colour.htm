draw_set_colour

使用此功能，您可以设置游戏的基本绘画颜色。该值将在适当的时候影响所有进一步的绘制，包括字体，表单，基元和3D。如果在绘制这些资产中的任何一个时更改了其自己的颜色值，则该值将被忽略。

句法：

draw_set_colour(col);

争论描述
col要设置用于绘制的颜色。

返回值：

N/A

例子：

draw_set_alpha(0.5);
draw_set_colour(c_black);
draw_text(x+5, y+5, "LEVEL 1");
draw_set_alpha(1);
draw_set_colour(c_white);
draw_text(x, y, "LEVEL 1");

上面的代码将在指定位置绘制一些文本，并通过修改的绘制Alpha和颜色创建阴影效果。
