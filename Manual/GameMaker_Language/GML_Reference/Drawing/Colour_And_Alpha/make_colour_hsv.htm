make_colour_hsv

GameMaker Studio 2提供了此功能（以及其他功能），以允许用户制作自己的颜色。此特定功能包含三个组成部分，即色调，饱和度和值（也称为“发光度”），以创建所需的颜色。这些值被视为介于0到255之间，因此您可以使用此颜色制作16,777,216（256 * 256 * 256）种颜色！在下面，您可以查看这些组件分离后的外观的图像：

左图是该功能各个组成部分的分解图，右图是这些组件的变化如何影响最终颜色的图示。如您所见，即使不更改色相（它保持为0，红色），我们也可以使用其他两个设置来显着改变颜色，并通过饱和度使色调从灰色（0）变为纯色（255）。 ，并使用该值将其从较暗的（0）变为较浅的（255）。

句法：

make_colour_hsv(hue, sat, val);

争论描述
hue颜色的色调
sat颜色的饱和度
val颜色有多深

返回值：

Integer

例子：

col = make_colour_hsv(100, 145, 255);

上面的代码使用该函数创建颜色并将其值存储在变量“ col”中，以备后用。
