make_colour_rgb

GameMaker Studio 2提供了此功能（以及其他功能），以允许用户制作自己的颜色。此特定功能包含三个组成部分，即您要制作的颜色的红色，绿色和蓝色部分。这些值被视为介于0到255之间，因此您可以使用此颜色制作16,777,216（256 * 256 * 256）种颜色！在下面，您可以查看这些组件分离后的外观的图像：

左图是该功能各个组成部分的分解图，右图是这些组件的变化如何影响最终颜色的图示。

句法：

make_colour_rgb(red, green, blue);

争论描述
red颜色的红色成分
green颜色的绿色成分
blue颜色的蓝色成分

返回值：

Int

例子：

col = make_colour_rgb(100, 145, 255);

上面的代码使用该函数创建颜色并将其值存储在变量“ col”中，以备后用。
