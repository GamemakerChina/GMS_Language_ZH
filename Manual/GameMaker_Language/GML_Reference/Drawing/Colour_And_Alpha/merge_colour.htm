merge_colour

使用此功能，您可以选择两种颜色，然后将它们合并在一起以形成新的颜色。可以通过更改“ amount”参数来定义每种组成色的数量，其中值0将返回第一种颜色（col1），值1将返回第二种颜色（col2），并且介于两者之间的值将返回相应的混合。例如，值为0.5将相等地混合两种颜色。下图通过将红色和蓝色融合在一起来说明其工作原理：

句法：

merge_colour(col1, col2, amount);

争论描述
col1合并的第一种颜色
col2合并的第二种颜色
amount每种颜色应合并多少。例如，0将返回col1，1将返回col2，而0.5将相等地返回两种颜色的合并

返回值：

Int

例子：

col = merge_colour(c_lime, c_orange, 0.5);

上面的代码使用该函数通过将石灰和橙色50/50合并来创建颜色，然后将其值存储在变量“ col”中以备后用。
