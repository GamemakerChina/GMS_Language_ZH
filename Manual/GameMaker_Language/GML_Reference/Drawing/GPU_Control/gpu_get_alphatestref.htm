gpu_get_alphatestref

您可以使用此功能查找Alpha测试参考的当前值（默认值为0，但可以使用 gpu_set_alphatestref()将此值设置为其他值）。

句法：

gpu_get_alphatestref();

返回值：

Real (from 0 - 255)

例子：

if gpu_get_alphatestenable()
    {
    if gpu_get_alphatestref() < 254
        {
        gpu_set_alphatestref(254);
        }
    }

上面的代码检查是否启用了alpha测试，如果已启用，则将检查当前的alpha测试参考值，并设置它是否小于254。
