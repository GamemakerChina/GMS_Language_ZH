gpu_get_blendenable

此功能可用于检索alpha混合状态。如果返回 true然后启用了alpha混合，如果返回 false它被禁用。默认情况下，此功能处于启用状态，因此函数将返回 true

句法：

gpu_get_blendenable();

返回值：

Boolean

例子：

if gpu_get_blendenable() == false
    {
    gpu_set_blendenable(true);
    }

上面的代码检查alpha混合的状态，如果已禁用，则再次启用。
