gpu_get_blendmode

此功能可用于检索用于绘图的当前混合模式。返回值将是以下常量之一（默认值为 bm_normal）：

持续的描述扩展混合模式
bm_normal普通混合（默认混合模式）。 (bm_src_alpha, bm_inv_src_alpha )
bm_add添加剂混合。添加亮区的光度值。 (bm_src_alpha, bm_one )
bm_subtract减法混合。减去亮区的光度值。 (bm_zero, bm_inv_src_colour )
bm_max最大混合。类似于添加剂混合。 (bm_src_alpha, bm_inv_src_colour )

句法：

gpu_get_blendmode();

返回值：

Constant (see above for constants)

例子：

if gpu_get_blendmode() != bm_normal
    {
    gpu_set_blendmode(bm_normal);
    }

上面的代码获取当前的混合模式，如果不是 bm_normal设置为该常数。
