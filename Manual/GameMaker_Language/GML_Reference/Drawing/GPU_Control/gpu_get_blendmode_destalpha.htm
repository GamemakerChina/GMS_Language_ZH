gpu_get_blendmode_destalpha

此功能可用于检索当前目标扩展混合模式的alpha因子。返回的值将是以下常量之一（“ s”表示从源获取的值，而“ d”表示从目标获取的值），并且在绘制时仅使用“ A”组件：

持续的混合因子（红色，绿色，蓝色，Alpha）
bm_zero （0，0，0，0）
bm_one （1、1、1、1）
bm_src_colour （Rs，Gs，Bs，As）
bm_inv_src_colour （1-Rs，1-Gs，1-Bs，1-As）
bm_src_alpha （砷，砷，砷，砷）
bm_inv_src_alpha （1-As，1-As，1-As，1-As）
bm_dest_alpha （广告，广告，广告，广告）
bm_inv_dest_alpha （1个广告，1个广告，1个广告，1个广告）
bm_dest_colour （Rd，Gd，Bd，Ad）
bm_inv_dest_colour （1-Rd，1-Gd，1-Bd，1-Ad）
bm_src_alpha_sat （f，f，f，1），其中f = min（As，1-Ad）

句法：

gpu_get_blendmode_destalpha();

返回值：

Constant (see above table)

例子：

var bm;
bm[0] = gpu_get_blendmode_srcalpha();
bm[1] = gpu_get_blendmode_destalpha();
gpu_set_blendmode_ext_sepalpha(bm_inv_src_alpha, bm_inv_dest_colour, bm[0], bm[1]);

上面的代码创建一个本地数组，并获取alpha组件的当前源和目标混合因子。然后，此数组用于控制混合因子的RGB分量。
