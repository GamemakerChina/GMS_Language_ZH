gpu_get_blendmode_ext

此功能可用于检索用于绘图的当前扩展混合模式。该函数返回一个2元素的1D数组，其中包含以下元素：

  • [0] =源混合模式（默认为 bm_src_alpha）
  • [1] =目标混合模式（默认为 bm_inv_src_alpha）

数组的值将是以下常量之一（“ s”表示从源获取的值，而“ d”表示从目标获取的值）：

持续的混合因子（红色，绿色，蓝色，Alpha）
bm_zero （0，0，0，0）
bm_one （1、1、1、1）
bm_src_colour （Rs，Gs，Bs，As）
bm_inv_src_colour （1-Rs，1-Gs，1-Bs，1-As）
bm_src_alpha （砷，砷，砷，砷）
bm_inv_src_alpha （1-As，1-As，1-As，1-As）
bm_dest_alpha （广告，广告，广告，广告）
bm_inv_dest_alpha （1个广告，1个广告，1个广告，1个广告）
bm_dest_colour （Rd，Gd，Bd，Ad）
bm_inv_dest_colour （1-Rd，1-Gd，1-Bd，1-Ad）
bm_src_alpha_sat （f，f，f，1），其中f = min（As，1-Ad）

请注意，您可以更改这些值并将完整的数组传递给 gpu_set_blendmode_ext()作为单个参数。

句法：

gpu_get_blendmode_ext();

返回值：

2 element 1D Array (see above for constants)

例子：

var bm = gpu_get_blendmode_ext();
bm[0] = bm_src_alpha;
gpu_set_blendmode_ext(bm);

上面的代码获取当前的扩展混合模式，修改源，然后再次设置扩展混合模式。
