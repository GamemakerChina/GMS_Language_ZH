gpu_get_colourwriteenable

此功能可用于检索当前的颜色写入启用值。该函数返回一个4元素的1D数组，其中包含以下元素：true （启用）或 false（已禁用）。默认情况下，所有颜色书写设置为 true：

  • [0] =启用/禁用红色通道
  • [1] =启用/禁用绿色通道
  • [2] =启用/禁用蓝色通道
  • [3] =启用/禁用Alpha通道

句法：

gpu_get_colorwriteenable();

返回值：

4 element 1D array (See above for details)

例子：

var cw = gpu_get_colorwriteenable();
cw[3] = false;
gpu_set_colorwriteenable(cw);

上面的代码获取当前的颜色写入值，然后将alpha分量设置为 false。
