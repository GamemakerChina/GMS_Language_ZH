gpu_get_cullmode

此功能可用于检索背面剔除模式。返回值将是以下常量之一（默认值为 cull_noculling）：

持续的描述
cull_noculling不会淘汰
cull_clockwise所有顺时针三角形将被剔除
cull_counterclockwise所有逆时针三角形将被剔除

句法：

gpu_get_cullmode();

返回值：

Constant (see above for constants)

例子：

if gpu_get_cullmode() != cull_clockwise
    {
    gpu_set_cullmode(cull_clockwise);
    }

上面的代码获取当前的剔除模式，如果没有 cull_clockwise设置为该常数。
