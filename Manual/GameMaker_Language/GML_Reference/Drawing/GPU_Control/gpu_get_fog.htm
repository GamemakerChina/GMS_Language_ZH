gpu_get_fog

此功能可用于检索雾设置。该函数返回一个包含以下信息的4元素一维数组：

  • [0] =启用切换（布尔值，或者 true或者 false）， 默认 false
  • [1] =颜色（真实），默认 c_black
  • [2] =起始距离（实际），默认为0
  • [3] =结束距离（实际），默认为1

请注意，您可以更改这些值并将完整的数组传递给 gpu_set_fog()作为单个参数。

句法：

gpu_get_fog();

返回值：

4 element 1D Array (see above for array elements)

例子：

var fog_a = gpu_get_fog();
fog_a[1] = c_red;
gpu_set_fog(fog_a);

上面的代码获取当前的雾设置，然后将数组的color元素设置为 c_red然后使用更改后的数组再次设置雾。
