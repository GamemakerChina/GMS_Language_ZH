gpu_get_fog

这个函数可以用来检索雾的设置。该函数返回一个4个元素的一维数组，包含以下信息。

  • [0] = 启用的切换（一个布尔值，要么是真，要么是假），默认为假。
  • [1] = 颜色（实际），默认为c_black
  • [2] = 启动距离（实数），默认为0
  • [3] = 末端距离（实数），默认为1

请注意，你可以改变这些值，并将完整的数组作为一个参数传递给 gpu_set_fog()函数作为一个参数。

语法。

gpu_get_fog()。

返回。

4个元素的一维数组（见上面的数组元素）。

例子。

var fog_a = gpu_get_fog();
fog_a[1] = c_red;
gpu_set_fog(fog_a);

上面的代码得到了当前的雾气设置，然后将数组中的颜色元素设置为c_red，然后使用改变后的数组再次设置雾气。
