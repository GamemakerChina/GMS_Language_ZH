gpu_get_texrepeat

使用此功能，您可以检查是否启用了纹理重复（返回 true）否（返回 false）。有关纹理重复的更多信息，请参见函数 gpu_set_texrepeat()。

句法：

gpu_get_texrepeat();

返回值：

Boolean

例子：

if gpu_get_texrepeat()
    {
    gpu_set_texrepeat(false);
    }
else
    {
    gpu_set_texrepeat(true);
    }

上面的代码检查纹理重复是打开还是关闭，并相应地进行切换。
