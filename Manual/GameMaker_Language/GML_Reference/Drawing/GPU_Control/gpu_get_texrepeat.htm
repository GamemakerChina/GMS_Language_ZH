gpu_get_texrepeat

通过这个函数，你可以检查是否启用了纹理重复（返回true）或没有（返回false）。关于纹理重复的更多信息，见函数 gpu_set_texrepeat().

语法。

gpu_get_texrepeat()。

返回。

布尔型

例子。

如果gpu_get_texrepeat()
{
gpu_set_texrepeat(false);
}
else
{
gpu_set_texrepeat(true);
}

上面的代码检查纹理重复是否打开或关闭，并进行相应的切换。
