gpu_get_zfunc

此功能可用于检索z比较模式。返回的值将是以下常量之一（默认值为 cmpfunc_lessequal）：

  • cmpfunc_never
  • cmpfunc_less
  • cmpfunc_equal
  • cmpfunc_lessequal
  • cmpfunc_greater
  • cmpfunc_notequal
  • cmpfunc_greaterequal
  • cmpfunc_always

句法：

gpu_get_zfunc();

返回值：

Constant (see table above)

例子：

if gpu_get_zfunc() != cmpfunc_greater
    {
    gpu_set_zfunc(cmpfunc_greater);
    }

上面的代码检查z-testing方法是否设置为 cmpfunc_greater如果不是，则将其设置为该常数。
