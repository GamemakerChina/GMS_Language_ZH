gpu_get_ztestenable

此函数可用于检索是否启用z测试（该函数返回 true）或否（函数返回 false）。默认值为禁用z-testing，因此该函数将返回 false。

句法：

gpu_get_ztestenable();

返回值：

Boolean

例子：

if gpu_get_ztestenable() == false
    {
    gpu_set_ztestenable(true);
    }

上面的代码检查z测试是否启用，如果禁用，则再次启用。
