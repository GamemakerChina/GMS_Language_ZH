gpu_get_zwriteenable

此函数可用于检索是否启用z写入（该函数返回 true）或否（函数返回 false）。默认值是启用z写入，因此该函数将返回 true。

句法：

gpu_get_zwriteenable();

返回值：

Boolean

例子：

if gpu_get_zwriteenable() == false
    {
    gpu_set_zwriteenable(true);
    }

上面的代码检查是否启用了z写入，如果禁用了z写入，则再次启用它。
