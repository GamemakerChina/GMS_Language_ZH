gpu_push_state

使用此功能，您可以将当前GPU状态推入堆栈以供以后使用。如果您要“保存”当前GPU状态（通常将诸如混合模式，Alpha编写，剔除等操作都推入堆栈），然后使用不同的设置绘制然后重置将GPU堆栈恢复到之前的状态（通过调用 gpu_pop_state()）。

句法：

gpu_push_state();

返回值：

N/A

例子：

gpu_push_state();
gpu_set_blendmode(bm_add);
gpu_set_blendenable(false);
gpu_set_cullmode(true);
with (obj_Effect_Parent)
    {
    draw_self();
    }
gpu_pop_state();

上面的代码将“保存”堆栈上的当前GPU状态，然后更改某些GPU设置并绘制一组实例，然后再将GPU状态重置为之前的状态。
