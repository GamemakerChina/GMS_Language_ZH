gpu_set_alphatestenable

此功能将为您的游戏启用或禁用Alpha测试（默认情况下，此功能处于禁用状态）。通过打开Alpha测试，您可以使用伴随功能 gpu_set_alphatestref()设置将所有Alpha值都设置为0的“截止”值。下图显示了打开或关闭Alpha测试时可以看到的差异：

注意：此功能可能会对iOS和Android设备的性能产生负面影响。

句法：

gpu_set_alphatestenable(enable);

争论描述
enable启用或禁用Alpha测试（true /false ）

返回值：

N/A

例子：

if !gpu_get_alphatestenable()
   {
   gpu_set_alphatestenable(true);
   gpu_set_alphatestref(128);
   }

上面的代码将检查是否启用了alpha测试，如果未启用，将打开alpha测试并将测试阈值设置为128（仅绘制alpha大于0.5的像素）。
