gpu_set_alphatestref

此功能将设置Alpha测试的参考值（使用 gpu_set_alphatestenable()）。这是“截止”阈值，在该阈值下不会绘制具有alpha的像素。

基本上，如果您将此值设置为0，并且启用了线性插值（在“游戏选项”中或使用 gpu_set_texfilter()），仍然会在重叠对象的边缘出现视觉错误。这是因为您可以使用几乎完全透明的区域来遮挡后面绘制的内容，但是如果将此参考值设置为254（即低于最大值），则可以保证仅绘制100％不透明的像素。您也可以关闭纹理插值，通常可以使用默认值而不会出现问题，也完全不需要使用此功能，但是在很大程度上取决于源图稿。

注意：这是一项高级绘图功能，因此，如果不确定是否需要它，或者不完全知道它的功能，最好不要使用它，因为它会严重破坏游戏的绘制方式。

句法：

gpu_set_alphatestref(val);

争论描述
val将Alpha测试的参考值设置为0到255（默认值为0）

返回值：

N/A

例子：

if !gpu_get_alphatestenable()
    {
    gpu_set_alphatestenable(true);
    gpu_set_alphatestref(128);
    }

上面的代码将检查是否启用了alpha测试，如果未启用，将打开alpha测试并将测试阈值设置为128（仅绘制alpha大于0.5的像素）。
