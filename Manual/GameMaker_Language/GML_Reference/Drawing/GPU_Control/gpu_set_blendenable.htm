gpu_set_blendenable

此功能可用于打开和关闭alpha混合。基本上，如果您将此设置为 false，所有绘制的图像都将被100％不透明绘制，这意味着精灵或背景的任何透明或半透明区域都将以纯色显示。最好尽可能消除alpha混合（尤其是在为移动设备开发时），因为这样可以大大提高绘制速度。

句法：

gpu_set_blendenable(enable);

争论描述
enable启用或禁用Alpha混合值（true或者 false）。

返回值：

N/A

例子：

gpu_set_blendenable(false);
draw_sprite(spr_Background, 0, 0, 0);
gpu_set_blendenable(true);

上面的代码关闭了alpha混合以绘制背景精灵，然后再次将其重新打开以继续绘制。
