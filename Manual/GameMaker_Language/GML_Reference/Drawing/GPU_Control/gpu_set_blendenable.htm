gpu_set_blendenable

这个函数可以用来切换alpha混合的开启和关闭。基本上，如果你把它设置为false，所有正在绘制的图像将被绘制为100%不透明的，这意味着精灵或背景的任何透明或半透明区域将 都会以纯色显示。尽可能关闭阿尔法混合是个好主意（特别是在为移动设备开发时），因为这将大大增加绘制速度。

语法。

gpu_set_blendenable(enable)。

争论 描述
启用 启用或禁用alpha混合值（真或假）。

返回。

不适用

例子。

gpu_set_blendenable(false);
draw_sprite(spr_Background, 0, 0, 0);
gpu_set_blendenable(true) 。

上面的代码在绘制背景精灵时关闭了阿尔法混合，然后又重新打开，继续绘制。
