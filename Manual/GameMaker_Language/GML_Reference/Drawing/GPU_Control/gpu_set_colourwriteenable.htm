gpu_set_colourwriteenable

使用此功能，您可以打开或关闭所有进一步绘制的颜色通道和Alpha通道。例如，您可以使用此功能创建仅Alpha通道的表面（在使用表面之前关闭RGB书写），或在绘制到屏幕时创建特殊效果。每个组件的默认值为 true，可以提供为四个唯一参数，也可以提供为4个元素的1D数组，其中包含以下元素：true （启用）或 false（已禁用）：

  • [0] =启用/禁用红色通道
  • [1] =启用/禁用绿色通道
  • [2] =启用/禁用蓝色通道
  • [3] =启用/禁用Alpha通道

句法：

gpu_set_colorwriteenable(red, green, blue, alpha);

争论描述
red启用/禁用红色通道
green启用/禁用绿色通道
blue启用/禁用蓝色通道
alpha启用/禁用Alpha通道

返回值：

N/A

例子：

var cw = gpu_get_colorwriteenable();
cw[3] = false;
gpu_set_colorwriteenable(cw);

上面的代码获取当前的颜色写入值，然后将alpha分量设置为 false。
