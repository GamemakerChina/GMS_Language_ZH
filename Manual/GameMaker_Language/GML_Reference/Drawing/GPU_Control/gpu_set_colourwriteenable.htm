gpu_set_colourwriteenable

通过这个功能，你可以打开或关闭颜色通道和阿尔法通道，以便进一步绘制。例如，你可以用这个功能来创建只有阿尔法通道的表面（在使用表面之前关闭RGB书写），或者在向屏幕绘图时创建特殊效果。 在屏幕上绘图时创造特殊效果。每个组件的默认值是true，可以作为四个唯一的参数提供，也可以作为一个4个元素的1D数组提供，其中的元素是 真（启用）或假（禁用）。

  • [0] = 红色通道启用/禁用
  • [1] = 绿色通道启用/禁用
  • [2] = 启用/停用蓝色通道
  • [3] = 启用/停用Alpha通道

语法。

gpu_set_colorwriteenable(red, green, blue, alpha)。

争论 描述
红色 启用/禁用红色通道
绿色 启用/禁用绿色通道
蓝色的 启用/禁用蓝色通道
阿尔法 启用/禁用阿尔法通道

返回。

不适用

例子。

var cw = gpu_get_colorwriteenable();
cw[3] = false;
gpu_set_colorwriteenable(cw);

上面的代码得到了当前的颜色写入值，然后将alpha组件设置为false。
