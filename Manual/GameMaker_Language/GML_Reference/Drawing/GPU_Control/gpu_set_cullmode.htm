gpu_set_cullmode

此功能可用于设置背面剔除模式。多边形（原始）具有正面和背面。据说前侧是按逆时针顺序定义顶点的那一侧。通常情况下，都绘制了两个侧面，但是如果您制作一个封闭的形状（如金字塔形），那么这将浪费处理能力，因为永远无法看到三角形的背面。在这些情况下，您可以启用背面剔除，因为这样可以节省大约一半的绘制时间，尽管这意味着您必须以正确的方式定义多边形以防止出现问题。

使用的剔除模式值必须是以下常量之一（默认值为 cull_noculling）：

持续的描述
cull_noculling不会淘汰
cull_clockwise所有顺时针三角形将被剔除
cull_counterclockwise所有逆时针三角形将被剔除

句法：

gpu_set_cullmode(cullmode);

争论描述
cullmode使用的剔除模式（请参见上面的列表）

返回值：

N/A

例子：

if gpu_get_cullmode() != cull_clockwise
    {
    gpu_set_cullmode(cull_clockwise);
    }

上面的代码获取当前的剔除模式，如果没有 cull_clockwise设置为该常数。
