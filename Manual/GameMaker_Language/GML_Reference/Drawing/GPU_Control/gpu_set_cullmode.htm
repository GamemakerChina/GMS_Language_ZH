gpu_set_cullmode

此函数可用于设置背面裁剪模式。一个多边形（基元）有一个正面和一个背面。正面被称为是顶点按逆时针顺序定义的那一面。通常情况下，两个面 都会被画出来，但如果你做的是一个封闭的形状（如金字塔），那么这就是在浪费处理能力，因为三角形的背面永远不会被看到。在这种情况下，你可以打开背面剔除，因为这可以节省大约一半的 绘图时间的一半，尽管这意味着你必须以正确的方式定义你的多边形，以防止出现问题。

使用的剔除模式值必须是以下常数之一（默认值是cull_noculling）。

恒定 描述
cull_noculling 将不进行杀戮
顺时针方向删减 所有顺时针方向的三角形都将被剔除
逆时针剔除 所有逆时针方向的三角形都将被剔除

语法。

gpu_set_cullmode(cullmode)。

争论 描述
诅咒模式（Cullmode 使用的剔除模式（见上面的列表）。

返回。

不适用

例子。

如果gpu_get_cullmode() != cull_clockwise
{
gpu_set_cullmode(cull_clockwise);
}

上面的代码得到了当前的剔除模式，如果它不是cull_clockwise，就会被设置为该常量。
