gpu_set_fog

此功能可用于启用或禁用雾化绘制。雾可以用于3D游戏中，使远处的实例看起来模糊甚至消失，这有助于营造气氛并掩盖您不在绘制远处实例的事实。您设置是否启用（true ）或禁用（false ），雾用于混合的颜色，以及开始和结束绘制的距离。请注意，起雾点指示相对于起雾的视图相机的半径，而起雾点指示将完全不可见的半径（也是相对的）实例。

该函数可以采用四个单独的参数（如上所示）或具有以下结构的数组（以下示例代码显示了此方法）：

  • [0] =启用切换（布尔值，或者 true或者 false）， 默认 false
  • [1] =颜色（真实），默认 c_black
  • [2] =起始距离（实际），默认为0
  • [3] =结束距离（实际），默认为1

句法：

gpu_set_fog(enable, colour, start, end);

争论描述
enable启用或禁用雾
colour雾色
start从以下位置开始施加雾的距离
end结束雾的距离

返回值：

N/A

例子：

var fog_a = gpu_get_fog();
fog_a[1] = c_red;
gpu_set_fog(fog_a);

上面的代码获取当前的雾设置，然后将数组的color元素设置为 c_red然后使用更改后的数组再次设置雾。
