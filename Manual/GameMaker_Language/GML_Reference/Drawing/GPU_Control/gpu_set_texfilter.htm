gpu_set_texfilter

这个函数可以用来设置游戏屏幕上绘制的所有图像的纹理过滤（线性插值）。当这个功能被启用时（true），所有的纹理在绘制时都会被平滑（这包括精灵，因为它们也被认为是纹理）。 这意味着，当缩放或移动时，如果不是1:1的像素比例，就会出现各种像素的 "污点"，这可能会使图像看起来很模糊，这取决于使用的艺术风格。如果这个功能被禁用（假），那么图像 将在缩放或移动时基于最近的像素绘制，这可能导致 "块状 "图像。默认值是false，这也可以在全局游戏选项中针对各个目标平台进行更改。

注意：这个设置将覆盖任何纹理阶段插值的设置，该设置是为着色器使用函数 gpu_set_texfilter_ext().

注意：在HTML5目标上，这个功能只有在启用WebGL的情况下才能发挥作用。

语法。

gpu_set_texfilter(enable)。

争论 描述
启用 启用或禁用纹理过滤（true/false）。

返回。

不适用

例子。

如果gpu_get_texfilter()
{
gpu_set_texfilter(false);
}
else
{
gpu_set_texfilter(true);
}

上述代码检查纹理过滤是否开启或关闭，并进行相应的切换。
