gpu_set_texfilter

此功能可用于设置游戏屏幕上绘制的所有图像的纹理过滤（线性插值）。启用后（true ）所有纹理在绘制时都会被平滑化（这也包括精灵，因为它们也被视为纹理），这意味着在缩放或移动时，如果不存在1：1像素比率，则各个像素之间都会出现“模糊”现象，图像根据使用的艺术风格而显得模糊。如果此功能被禁用（false ），则在缩放或移动时会根据最近的像素绘制图像，这可能会导致“块状”图像。默认值为 false，也可以在各个目标平台的“全局游戏选项”中进行更改。

注意：此设置将覆盖使用该功能shaders设置的任何纹理阶段插值 gpu_set_texfilter_ext()。

注意：在HTML5目标上，此功能仅在启用WebGL的情况下起作用。

句法：

gpu_set_texfilter(enable);

争论描述
enable启用或禁用纹理过滤（true /false ）

返回值：

N/A

例子：

if gpu_get_texfilter()
    {
    gpu_set_texfilter(false);
    }
else
    {
    gpu_set_texfilter(true);
    }

上面的代码检查纹理过滤是打开还是关闭，并相应地进行切换。
