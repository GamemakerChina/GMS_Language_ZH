gpu_set_texfilter_ext

在GameMaker Studio 2 Shaders ，此功能可用于为单个采样器“插槽”设置线性插值。启用后（true ）采样器纹理将被平滑化，如果禁用该选项（false ），然后将根据最近的像素绘制图像。默认值为您的游戏的“全局游戏选项”设置的值，或者是使用函数设置的默认值 gpu_set_texfilter()。

注意：调用函数时，此设置将被设置的值覆盖 gpu_set_texfilter()。

注意：在HTML5目标上，此功能仅在启用WebGL的情况下起作用。

句法：

gpu_set_texfilter_ext(sampler_id, enable);

争论描述
sampler_id shader的采样器ID。
enable启用或禁用纹理过滤（true /false ）

返回值：

N/A

例子：

var s_tex = shader_get_sampler_index(shader_glass, "s_NoiseSampler");
if gpu_get_texfilter_ext(s_tex)
    {
    gpu_set_texfilter_ext(s_tex, false);
    }
else
    {
    gpu_set_texfilter_ext(s_tex, true);
    }

上面的代码检查是否为特定采样器ID（存储在局部变量中）打开或关闭纹理过滤，并相应地进行切换。
