gpu_set_texrepeat

此功能可用于向GameMaker Studio 2指示是否应重复纹理（true ） 或不 （false ）用于模型和图元。通常，纹理的宽度和高度在0到1的范围内，但是如果默认指定大于1的值，则不会绘制其余部分（整个纹理将以“空”部分绘制，该部分对应于1的百分比，即因此，如果纹理宽度或高度的值为2，则将在一半的空间中绘制纹理，而另一半则为空白）。但是，通过使用此功能将repeat设置为true，可以在所有模型和图元上自动重复纹理。这些功能的可能用例是在3D中重复纹理，但为了使其起作用而不从其余纹理页面提取图像，需要将使用的精灵标记为“单独的纹理页面”在Sprite编辑器中。

注意：始终绘制精灵和背景，而不会重复，因此绘制精灵或背景后，此值将重置为 false。此默认行为不会影响纹理阶段设置（即：调用函数会，但内部设置为false不会）。

注意：使用此功能时， shaders设置的所有纹理阶段重复设置 gpu_set_texrepeat_ext()。

句法：

gpu_set_texrepeat(enable);

争论描述
enable启用或禁用纹理过滤（true /false ）

返回值：

N/A

例子：

if gpu_get_texrepeat()
    {
    gpu_set_texrepeat(false);
    }
else
    {
    gpu_set_texrepeat(true);
    }

上面的代码检查纹理重复是打开还是关闭，并相应地进行切换。
