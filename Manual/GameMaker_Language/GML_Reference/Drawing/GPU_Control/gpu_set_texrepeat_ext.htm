gpu_set_texrepeat_ext

此功能可用于设置在GameMaker Studio 2 Shaders时单个采样器“插槽”是否重复给定纹理。设置为 true如果uv坐标超出0-1范围，则将重复纹理，而设置为 false将意味着不再重复。这些功能的可能用例是在3D中重复纹理，但为了使其起作用而不从其余纹理页面提取图像，需要将使用的精灵标记为“单独的纹理页面”在Sprite编辑器中。

注意：调用函数gpu_set_texrepeat（）时，此设置将被设置的值覆盖。

句法：

gpu_set_texrepeat_ext(sampler_id, enable);

争论描述
sampler_id shader的采样器ID。
enable启用或禁用纹理过滤（true /false ）

返回值：

N/A

例子：

var s_tex = shader_get_sampler_index(shader_glass, "s_NoiseSampler");
if gpu_get_texfilter_ext(s_tex)
    {
    gpu_set_texfilter_ext(s_tex, false);
    }
else
    {
    gpu_set_texfilter_ext(s_tex, true);
    }

上面的代码检查是否为特定采样器ID（存储在局部变量中）打开或关闭纹理过滤，并相应地进行切换。
