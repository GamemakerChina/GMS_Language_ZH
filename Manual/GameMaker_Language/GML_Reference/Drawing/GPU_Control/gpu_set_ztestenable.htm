gpu_set_ztestenable

这个函数可以用来开启或关闭Z-buffer测试，影响事物的绘制方式（一般来说，只有在处理3D项目时才会用到）。基本上，在默认情况下，当z-testing关闭时，你有两个对象在同一空间绘制。 两个对象都会被渲染，而不管其中一个是否会过度绘制另一个，从而导致不必要的绘制调用。如果你把它打开，那么Z-缓冲区就会被测试，看一个对象是否 "可见"，如果不可见就不绘制。 不可见。请注意，这是默认行为，但你可以通过改变用于z-buffer测试的比较类型来改变它（见函数 gpu_set_zfunc().

默认情况下，z-buffer测试是关闭的（false）。

语法。

gpu_set_ztestenable(enable)。

争论 描述
启用 启用或禁用z-buffer测试（真或假）。

返回。

不适用

例子。

gpu_set_ztestenable(true);
draw_sprite(spr_Background, 0, 0, 0);
gpu_set_ztestenable(false) 。

上面的代码开启了z-buffer测试，以绘制一个背景精灵，然后再把它关掉，继续绘制。
