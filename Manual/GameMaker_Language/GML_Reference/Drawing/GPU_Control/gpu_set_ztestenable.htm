gpu_set_ztestenable

此功能可用于打开或关闭z缓冲区测试，从而影响绘制方式（通常仅在使用3D项目时使用）。本质上，默认情况下，当z测试关闭且您有两个对象绘制到同一空间时，将渲染两个对象，而不管一个对象是否会过度绘制另一个对象，从而导致不必要的绘制调用。如果将其打开，则将测试z缓冲区以查看对象是否“可见”，如果不是则不绘制。请注意，这是默认行为，但是您可以通过更改用于z缓冲区测试的比较类型来更改此设置（请参见函数 gpu_set_zfunc()。

默认情况下，z缓冲区测试处于关闭状态（false ）。

句法：

gpu_set_ztestenable(enable);

争论描述
enable启用或禁用z缓冲区测试（true或者 false）。

返回值：

N/A

例子：

gpu_set_ztestenable(true);
draw_sprite(spr_Background, 0, 0, 0);
gpu_set_ztestenable(false);

上面的代码打开z缓冲区测试以绘制背景精灵，然后再次将其关闭以继续绘制。
