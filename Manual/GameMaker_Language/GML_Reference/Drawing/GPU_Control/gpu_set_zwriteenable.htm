gpu_set_zwriteenable

此功能可用于开启（true ）和关闭（false ）使用z缓冲区进行深度排序。 GameMaker Studio 2渲染实例时，将生成的像素的深度（其z坐标）存储在z缓冲区中，然后，如果必须在同一像素处渲染场景的另一个实例，则将两个深度进行比较（z测试-请参阅 gpu_set_ztestenable()），然后将与观察者相对的一个“更近的对象”保存到z缓冲区中，以替换旧的缓冲区。但是，这并不总是您希望发生的事情，有时您可能希望在其他所有物体上绘制一些东西，无论其z值是多少。在这些情况下，您可以使用此功能来关闭对z缓冲区的写入，从而强制将要绘制的内容覆盖到其他所有内容上。请注意，此处给出的说明适用于默认的z检验比较模式，但可以更改

默认值为启用z缓冲（true ）。

句法：

gpu_set_zwriteenable(enable);

争论描述
enable使用的比较模式（请参见上面的列表）

返回值：

N/A

例子：

gpu_set_zwriteenable(false);
with (obj_Menu_Containers)
    {
    draw_self();
    }
gpu_set_zwriteenable(true);

上面的代码关闭了z缓冲区写入，绘制了许多对象，然后再次将其重新打开以继续绘制。
