draw_light_define_ambient

此功能用于控制场景的环境光，即使没有定义任何点光源或定向光源，也就是场景中的环境光。它实际上是场景的整体颜色和亮度（或暗度）。默认颜色是 c_black。

句法：

draw_light_define_ambient(col);

争论描述
col使用的颜色（常数，实数或十六进制值）。

返回值：

N/A

例子：

draw_light_define_ambient(c_white);

上面的代码将环境照明定义为白色。
