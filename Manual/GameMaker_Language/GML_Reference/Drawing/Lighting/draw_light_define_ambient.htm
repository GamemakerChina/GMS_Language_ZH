绘画_光线_定义_环境

这个功能是用来控制场景的环境光，也就是在没有定义任何点或定向光源的情况下，场景中的光线。它实际上是一个场景的整体颜色和亮度（或暗度）。 默认的颜色是c_black。

语法。

draw_light_define_ambient(col)。

争论 描述
殖民地 要使用的颜色（可以是常数、实数或十六进制值）。

返回。

不适用

例子。

draw_light_define_ambient(c_white)。

上述代码将定义环境照明为白色。
