draw_light_define_direction

此功能用于定义定向光，其中方向和强度由为x，y和z单位向量输入的值设置（表示它们必须在-1和1之间，其中1或-1是全强度，且0为无强度）。还可以给灯指定颜色，这也将影响光的感知强度，因为某些颜色比其他颜色“更暗”。您还必须给光源一个索引号，该索引号将在其他功能中使用以引用它。

注意：只有8个硬件指示灯可用，因此一次只能启用8个定义的指示灯（尽管可以定义更多）。

句法：

draw_light_define_direction(ind, x, y, z, col);

争论描述
ind指示灯的索引号（任意）
x光向量的x分量
y光向量的y分量
z光向量的z分量
col灯光使用的颜色（常数，实数或十六进制值）。

返回值：

N/A

例子：

draw_set_lighting(true);
draw_light_define_direction(1, 0, 1, 0, c_white);
draw_light_enable(1, true);

上面的代码将启用整个场景的照明，然后在房间空间中定义白色定向光，然后最后打开该光。
