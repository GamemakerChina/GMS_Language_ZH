绘画_光的定义_方向

这个函数用于定义一个定向光，方向和强度由x、y和z单位向量的输入值设定（意味着它们必须在-1和1之间，其中1或-1是全强度，而0是无强度）。灯光也可以被赋予一个颜色。 灯也可以被赋予一个颜色，这也会影响到光的感知强度，因为某些颜色看起来比其他颜色 "更暗"。你还必须给灯光一个索引号，这个索引号将在其他函数中用来参考 它。

注意：只有8个硬件灯可用，所以在任何时候都只能启用8个定义的灯（尽管可以定义更多）。

语法。

draw_light_define_direction(ind, x, y, z, col)。

争论 描述
ind 灯的索引号（任意）。
x 光矢量的X分量
y 光矢量的Y分量
z 光矢量的Z分量
殖民地 用于灯光的颜色（可以是常数、实数或十六进制值）。

返回。

不适用

例子。

draw_set_lighting(true);
draw_light_define_direction(1, 0, 1, 0, c_white);
draw_light_enable(1, true) 。

上述代码将启用整个场景的照明，然后在房间空间中定义一个白色的定向灯，最后将该灯打开。
