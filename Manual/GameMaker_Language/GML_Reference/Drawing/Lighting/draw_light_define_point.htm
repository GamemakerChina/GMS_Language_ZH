draw_light_define_point

此功能用于定义位置光，您可以在其中定义光的x，y和z位置，光范围及其颜色（这也会影响光的感知强度，因为某些颜色比其他颜色“更暗” ）。您还必须给光源一个索引号，该索引号将在其他功能中使用以引用它。

注意：只有8个硬件指示灯可用，因此一次只能启用8个定义的指示灯（尽管可以定义更多）。

句法：

draw_light_define_point(ind, x, y, z, range, col);

争论描述
ind指示灯的索引号（任意）
x灯的x位置
y灯的y位置
z灯光的z位置
range光线范围（以像素为单位）
col灯光使用的颜色（常数，实数或十六进制值）。

返回值：

N/A

例子：

draw_set_lighting(true);
draw_light_define_point(1, 200, 123, 50, 2000, c_white);
draw_light_enable(1, true);

上面的代码将启用整个场景的照明，然后在房间空间的特定点定义白光，然后最后打开该光。
