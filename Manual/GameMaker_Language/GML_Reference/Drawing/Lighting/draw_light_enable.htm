draw_light_enable

这个功能是用来启用一个定义的灯。当你定义一个位置灯或方向灯时，你必须给它分配一个索引号，然后由这个函数用来打开或关闭该灯。默认是禁用（假）。

注意：只有8个硬件灯可用，所以在任何时候都只能启用8个定义的灯（尽管可以定义更多）。

语法。

draw_light_enable(ind, enable)。

争论 描述
ind 灯的索引号（从0到7）。
启用 启用或禁用所有照明（真或假）。

返回。

不适用

例子。

draw_set_lighting(true);
draw_light_define_direction(1, 0, 1, 0, c_white);
draw_light_enable(1, true) 。

上述代码将启用整个场景的照明，然后在房间空间中定义一个白色的定向灯，最后将该灯打开。
