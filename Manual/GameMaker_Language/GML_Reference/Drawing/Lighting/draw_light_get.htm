draw_light_get

此函数将获取指定的灯光参数，该参数为具有以下6个元素的数组-

  • [0] =启用/禁用（true /false ）
  • [1] =灯光类型（请参见下面的常量）
  • [2] = x位置
  • [3] = y位置
  • [4] = z位置
  • [5] =灯光半径（仅用于点光源，对于定向光源将为0）
  • [6] =浅色（实数）

光源类型可以是以下两个常数之一：

持续的描述
lighttype_dir灯是定向灯
lighttype_point灯是点灯

句法：

draw_light_get(ind);

争论描述
ind指示灯的索引号（从0到7）

返回值：

Real

例子：

light_a = draw_light_get(1);
if light_a[5] < 200
    {
    light_a[5] += 5;
    draw_light_define_point(1, 200, 123, 50, light_a[5], c_white);
    }

上面的代码将获得用于定义索引为“ 1”的光源的值，然后检查半径，如果半径小于200，则将其增大，并将光源半径设置为新值。
