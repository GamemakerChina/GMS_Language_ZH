gpu_get_tex_max_aniso

使用此功能时，您可以获取当前最大的各向异性水平 tf_anisotropic过滤器模式（请参阅 gpu_get_tex_mip_filter()欲获得更多信息）。返回值的范围是1到16。

句法：

gpu_get_tex_max_aniso();

返回值：

Real (default: 16)

例子：

if gpu_get_tex_max_aniso() != 8
    {
    gpu_set_tex_max_aniso(8);
    }

上面的代码将检查当前的最大各向异性过滤级别，如果不是8，则将其设置为8。

背面： Mipmapping
