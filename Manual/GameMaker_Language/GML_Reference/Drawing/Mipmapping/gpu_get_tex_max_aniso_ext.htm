gpu_get_tex_max_aniso_ext

通过这个函数，你可以在使用tf_anisotropic滤镜模式时获得当前最大的各向异性水平（见 gpu_get_tex_mip_filter()获取更多信息）时，可以获得当前各向异性的最大水平。你可以 提供着色器采样器的索引值（如函数所返回的 shader_get_sampler_index()的索引值，该函数将返回一个范围在1到16之间的值。 16.

语法。

gpu_get_tex_max_aniso_ext(sampler_index)。

争论 描述
采样器_索引 要获得的着色器采样器的索引

返回。

实数（默认：16

例子。

var _sampleIndex =shader_get_sampler_index(shd_Glass, "s_Background");
var _spriteTex = sprite_get_texture(sprite_index, 0);
shader_set(shd_Glass) 。
if gpu_get_tex_max_aniso_ext(_sampleIndex) != 8
{
gpu_set_tex_max_aniso_ext(_sampleIndex, 8);
}
texture_set_stage(_sampleIndex , _spriteTex);
shader_reset()。

上述代码将给定着色器纹理采样器的最大各向异性水平设置为8，如果它还没有被设置为8的话。

返回。Mipmapping
