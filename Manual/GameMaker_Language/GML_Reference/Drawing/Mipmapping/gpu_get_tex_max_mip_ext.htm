gpu_get_tex_max_mip_ext

使用此功能，您可以获得当前设置的最大mipmap级别，该级别将用于给定的shader器采样器。 shader器采样器提供索引值（由函数返回）shader_get_sampler_index() ，该函数将返回当前的最大mipmap级别，其中0表示最高分辨率，1表示使用第一个mipmap，2表示第二个，以此类推。渲染纹理，例如，将纹理页面边框设置为8px，然后将最大mipmap级别设置为3，将确保在更大的渲染距离下不会出现任何渗色问题。

句法：

gpu_get_tex_max_mip_ext(sampler_index);

争论描述
sampler_index shader器采样器的索引

返回值：

Real (default: 16)

例子：

var _sampleIndex = shader_get_sampler_index(shd_Glass, "s_Background");
var _spriteTex = sprite_get_texture(sprite_index, 0);
shader_set(shd_Glass);
if gpu_get_tex_max_mip_ext(_sampleIndex) != 4
    {
    gpu_set_tex_max_mip_ext(_sampleIndex, 4);
    }
texture_set_stage(_sampleIndex , _spriteTex);
shader_reset();

上面的代码将给定的shader纹理采样器的最大mipmap级别设置为4（如果尚未将其设置为4）。
