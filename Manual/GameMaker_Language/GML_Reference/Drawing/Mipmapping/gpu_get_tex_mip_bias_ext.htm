gpu_get_tex_mip_bias_ext

使用此功能，您可以检索给定shader器采样器的mipmap偏差值。 shader器采样器提供索引值（由函数返回）shader_get_sampler_index() ，并且该函数将返回无偏差的0.0值，或者更大的值，其中1等于第一个mipmap，2等于第二个mipmap，依此类推...这将控制mip映射的交换速率，通常会使shader纹理越模糊，值越高，被查看的“距离”越大。请注意，这也可能返回负值，在这种情况下， shader纹理在距离越远的地方越锐利，则该值越低。

句法：

gpu_get_tex_mip_bias_ext(sampler_index);

争论描述
sampler_index shader器采样器的索引

返回值：

Real (default: 0)

例子：

var _sampleIndex = shader_get_sampler_index(shd_Glass, "s_Background");
var _spriteTex = sprite_get_texture(sprite_index, 0);
shader_set(shd_Glass);
if gpu_get_tex_mip_bias_ext(_sampleIndex) != 0
    {
    gpu_set_tex_mip_bias_ext(_sampleIndex, 0);
    }
texture_set_stage(_sampleIndex , _spriteTex);
shader_reset();

上面的代码将给定的shader纹理采样器的mip过滤器偏差设置为0（如果尚未将其设置为0）。

背面： Mipmapping
