gpu_get_tex_mip_enable

通过这个函数，你可以得到mipmapping是否被关闭，是否对所有的东西都打开，或者只对Texture Group Manager中选择的纹理组打开。该函数将返回 以下所列的常数之一，默认设置为mip_markedonly。

语法。

gpu_get_tex_mip_enable（）。

返回。

恒定。

恒定 描述
ǞǞǞ Mipmapping被禁用。
姆普_安 所有纹理的Mipmapping都已启用。
mip_markedonly 对于在纹理组选项中启用了Mipmapping的纹理（默认），Mipmapping被启用。

例子。

如果gpu_get_tex_mip_enable != mip_on
{
gpu_set_tex_mip_enable(mip_on);
}

上面的代码将检查是否启用了mipmapping，如果没有，它将启用它。

返回。Mipmapping
