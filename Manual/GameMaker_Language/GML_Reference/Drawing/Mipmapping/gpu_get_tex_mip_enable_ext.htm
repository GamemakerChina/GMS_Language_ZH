gpu_get_tex_mip_enable_ext

使用此功能，您可以获取是否关闭了mipmapping，是否为所有内容打开了mipmapping或仅为shader器采样器上的Texture Group Manager中选择的纹理组打开了。 shader器采样器提供索引值（由函数返回）shader_get_sampler_index() ，该函数将返回下面列出的常量之一。

句法：

gpu_get_tex_mip_enable_ext(sampler_index);

争论描述
sampler_index shader器采样器的索引

返回值：

Constant:

持续的描述
mip_off Mipmapping已禁用。
mip_on对所有纹理启用了Mipmapping。
mip_markedonly已在“纹理组”选项中启用了纹理的纹理启用了Mipmapping。

例子：

var _sampleIndex = shader_get_sampler_index(shd_Glass, "s_Background");
var _spriteTex = sprite_get_texture(sprite_index, 0);
shader_set(shd_Glass);
if gpu_get_tex_mip_enable_ext(_sampleIndex) != mip_on
    {
    gpu_set_tex_mip_enable_ext(_sampleIndex, mip_on);
    }
texture_set_stage(_sampleIndex , _spriteTex);
shader_reset();

shader纹理采样器，则上面的代码将启用mipmapping。
