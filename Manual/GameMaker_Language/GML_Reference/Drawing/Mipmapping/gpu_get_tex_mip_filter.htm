gpu_get_tex_mip_filter

使用此功能，您可以获取当前的Mip过滤器模式。该函数返回的模式值将是下面列出的常数之一。

句法：

gpu_get_tex_mip_filter();

返回值：

Constant (listed below):

持续的描述
tf_point这意味着禁用了Mipmap级别之间的混合，这可能会导致可见的纹理过渡，但性能最佳。
tf_linear这意味着启用了mipmap级别之间的混合（也称为三线性过滤），可以平滑纹理过渡，但对性能的影响较小。
tf_anisotropic这意味着启用了各向异性过滤，这可以大大提高纹理过渡质量，并可以减少其他过滤模式下可见的模糊，但对性能的影响最高。

例子：

if keyboard_check(ck_enter)
    {
    switch(gpu_get_tex_mip_filter())
        {
        case tf_point: gpu_set_tex_mip_filter(tf_linear); break;
        case tf_linear: gpu_set_tex_mip_filter(tf_anisotropic); break;
        case tf_anisotropic: gpu_set_tex_mip_filter(tf_point); break;
        }
    }

上面的代码检查键盘，如果按下了指定的键，它将获得当前的Mip过滤器并将值切换到下一个，在不同模式之间循环。
