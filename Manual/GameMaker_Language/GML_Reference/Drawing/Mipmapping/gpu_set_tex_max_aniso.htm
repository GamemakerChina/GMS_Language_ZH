gpu_set_tex_max_aniso

当使用tf_anisotropicfilter模式时，通过这个函数你可以设置当前最大的各向异性水平（更多信息见 gpu_get_tex_mip_filter()获取更多信息）。)输入值的范围必须在 在1和16之间。

语法。

gpu_set_tex_max_aniso（maxaniso）。

争论 描述
ǞǞǞ 各向异性过滤的最大级别（默认：16）。

返回。

不适用

例子。

if gpu_get_tex_max_aniso() != 8
{
gpu_set_tex_max_aniso(8);
}

上述代码将检查当前的最大各向异性过滤水平，如果不是8，就设置为8。

返回。Mipmapping
