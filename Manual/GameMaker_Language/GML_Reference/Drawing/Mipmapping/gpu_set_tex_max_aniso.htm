gpu_set_tex_max_aniso

使用此功能时，您可以设置当前的最大各向异性水平。tf_anisotropic过滤器模式（请参阅 gpu_get_tex_mip_filter()欲获得更多信息）。输入值的范围必须在1到16之间。

句法：

gpu_set_tex_max_aniso(maxaniso);

争论描述
maxaniso各向异性过滤的最大级别（默认值：16）

返回值：

N/A

例子：

if gpu_get_tex_max_aniso() != 8
    {
    gpu_set_tex_max_aniso(8);
    }

上面的代码将检查当前的最大各向异性过滤级别，如果不是8，则将其设置为8。

背面： Mipmapping
