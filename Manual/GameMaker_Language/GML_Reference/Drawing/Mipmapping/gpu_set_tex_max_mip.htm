gpu_set_tex_max_mip

使用此功能，您可以设置要使用的当前设置的最大mipmap级别，其中0表示最高分辨率，1表示使用第一个mipmap，2表示第二个mipmap，等等。这对于避免在渲染纹理时产生流血的伪影很有用，例如，将纹理页面边框设置为8px，然后将最大mipmap级别设置为3，将确保在较大的渲染距离上不会出现任何渗色问题。

句法：

gpu_set_tex_max_mip(maxmip);

争论描述
maxmip最大mipmap级别（默认值：16）

返回值：

N/A

例子：

if gpu_get_tex_max_mip() != 4
    {
    gpu_set_tex_max_mip(4);
    }

上面的代码将检查当前的最大mipmap级别，如果不是4，则将其设置为4。
