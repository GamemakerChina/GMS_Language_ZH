gpu_set_tex_min_mip

使用此功能，您可以设置当前使用的最小mipmap级别，其中0表示最高分辨率，1表示使用第一个mipmap，2表示第二个mipmap等。

句法：

gpu_set_tex_min_mip(minmip);

争论描述
minmip使用的最低Mipmap级别

返回值：

N/A

例子：

if gpu_get_tex_min_mip() != 0
    {
    gpu_set_tex_min_mip(0);
    }

上面的代码将检查当前的最小mipmap级别，如果它不为0，则将其设置为0。
