gpu_set_tex_mip_bias

通过这个函数你可以设置mipmap的偏置值，0等于没有偏置，1等于第一个mipmap，2等于第二个mipmap等等。这控制了mip map交换的速度，通常会使渲染的纹理更加模糊。 值越高，被观看的 "距离 "越大。请注意，这个函数也可以取负值，在这种情况下，值越低，渲染的纹理在更大的距离上会更清晰。

语法。

gpu_set_tex_mip_bias(bias)。

争论 描述
偏见 要使用的mipmap偏置值（默认：0

返回。

不适用

例子。

if gpu_get_tex_mip_bias() != 0
{
gpu_set_tex_mip_bias(0);
}

上面的代码将检查当前的mipmap偏差，如果它不是0，就设置为0。

返回。Mipmapping
