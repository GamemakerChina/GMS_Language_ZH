gpu_set_tex_mip_bias

使用此功能，您可以设置mipmap偏差值，其中0等于无偏差，1等于第一个mipmap，2等于第二个mipmap，依此类推...这将控制mip映射的交换速率，通常会渲染的纹理越模糊，值越高，被查看的“距离”越大。请注意，此函数也可以采用负值，在这种情况下，渲染的纹理在距离越远越锐利，值越低。

句法：

gpu_set_tex_mip_bias(bias);

争论描述
bias要使用的mipmap偏差值（默认值：0）

返回值：

N/A

例子：

if gpu_get_tex_mip_bias() != 0
    {
    gpu_set_tex_mip_bias(0);
    }

上面的代码将检查当前的mipmap偏差，如果不为0，则将其设置为0。
