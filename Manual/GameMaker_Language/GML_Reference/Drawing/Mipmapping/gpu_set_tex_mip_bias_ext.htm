gpu_set_tex_mip_bias_ext

使用此功能，可以为给定的shader器采样器设置mipmap偏差值。 shader器采样器提供索引值（由函数返回）shader_get_sampler_index() ，然后是偏移值，其中0表示无偏移，1等于第一个mipmap，2等于第二个mipmap，以此类推...这样可以控制交换mip贴图的速率，通常会使shader纹理模糊得更高值越大，所看到的“距离”越大。请注意，此函数也可以采用负值，在这种情况shader纹理在更大的距离上将越清晰。

句法：

gpu_set_tex_mip_bias_ext(sampler_index, bias);

争论描述
sampler_index shader器采样器的索引
bias要使用的mipmap偏差值（默认值：0）

返回值：

N/A

例子：

var _sampleIndex = shader_get_sampler_index(shd_Glass, "s_Background");
var _spriteTex = sprite_get_texture(sprite_index, 0);
shader_set(shd_Glass);
if gpu_get_tex_mip_bias_ext(_sampleIndex) != 0
    {
    gpu_set_tex_mip_bias_ext(_sampleIndex, 0);
    }
texture_set_stage(_sampleIndex , _spriteTex);
shader_reset();

上面的代码将给定的shader纹理采样器的mip过滤器偏差设置为0（如果尚未将其设置为0）。
