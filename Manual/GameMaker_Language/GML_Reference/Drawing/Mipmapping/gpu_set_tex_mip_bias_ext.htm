gpu_set_tex_mip_bias_ext

通过这个函数，你可以为一个给定的着色器采样器设置mipmap偏置值。你提供着色器采样器的索引值（由函数 shader_get_sampler_index(), 然后是偏置值，其中0表示没有偏置，1表示第一个mipmap，2表示第二个mipmap等等。这控制了mip贴图的交换速度，一般来说，值越大，观看的 "距离 "越大，着色器的纹理就越模糊。 观察的 "距离 "越大。请注意，这个函数也可以取负值，在这种情况下，数值越低，着色器纹理就会在更大的距离上变得更清晰。

语法。

gpu_set_tex_mip_bias_ext(sampler_index, bias)。

争论 描述
采样器_索引 要获得的着色器采样器的索引
偏见 要使用的mipmap偏置值（默认：0

返回。

不适用

例子。

var _sampleIndex =shader_get_sampler_index(shd_Glass, "s_Background");
var _spriteTex = sprite_get_texture(sprite_index, 0);
shader_set(shd_Glass) 。
if gpu_get_tex_mip_bias_ext(_sampleIndex) != 0
{
gpu_set_tex_mip_bias_ext(_sampleIndex, 0);
}
texture_set_stage(_sampleIndex , _spriteTex);
shader_reset()。

上面的代码为给定的着色器纹理取样器将mip过滤器的偏差设置为0，如果它还没有被设置为0。

返回。Mipmapping
