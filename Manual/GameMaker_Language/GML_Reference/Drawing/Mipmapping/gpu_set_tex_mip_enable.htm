gpu_set_tex_mip_enable

使用此功能，您可以更改是关闭mipmapping，为所有内容打开mipmapping还是仅为在Texture Group Manager中选择的纹理组打开。该函数需要下列常量之一。

持续的描述
mip_off Mipmapping已禁用。
mip_on对所有纹理启用了Mipmapping。
mip_markedonly对于在“纹理组”选项中启用了纹理的纹理，启用了Mipmapping（默认）。

句法：

gpu_set_tex_mip_enable(setting);

争论描述
setting mipmap设置（一个常量，默认为：mip_markedonly ）

返回值：

N/A

例子：

if gpu_get_tex_mip_enable != mip_on
    {
    gpu_set_tex_mip_enable(mip_on);
    }

上面的代码将检查是否启用了mipmapping，如果未启用，则将启用它。
