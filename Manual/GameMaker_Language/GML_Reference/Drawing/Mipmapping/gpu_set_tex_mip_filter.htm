gpu_set_tex_mip_filter

使用此功能，您可以将当前的Mip过滤器模式设置为支持的三种类型之一。您可以提供一个常数，该常数指的是您需要的mip过滤模式，您可以在其中选择点过滤（默认设置，表示不过滤），线性过滤（也称为三线性过滤）或各向异性过滤（请参见常数表）以下）。

持续的描述
tf_point这将禁用mipmap级别之间的过滤，这可能导致可见的纹理过渡，但性能最佳。
tf_linear这将启用mipmap级别之间的线性过滤（也称为三线性过滤），这可以平滑纹理过渡，但对性能的影响较小。
tf_anisotropic这将启用mipmap级别之间的各向异性过滤，这可以大大提高纹理过渡质量，并可以减少其他过滤模式下可见的模糊，但其性能最高。

句法：

gpu_get_tex_mip_filter(filter);

争论描述
filter要使用的MIP过滤器模式（一个常量，默认为：tf_point ）。

返回值：

N/A

例子：

if keyboard_check(ck_enter)
    {
    switch(gpu_get_tex_mip_filter())
        {
        case tf_point: gpu_set_tex_mip_filter(tf_linear); break;
        case tf_linear: gpu_set_tex_mip_filter(tf_anisotropic); break;
        case tf_anisotropic: gpu_set_tex_mip_filter(tf_point); break;
        }
    }

上面的代码检查键盘，如果按下了指定的键，它将获得当前的Mip过滤器并将值切换到下一个，在不同模式之间循环。
