part_emitter_create

必须使用此函数来创建新的发射器并将其分配给给定的粒子系统。该函数将返回新发射器的索引值，该索引值必须存储在变量中，并在引用该发射器的所有其他函数中使用，并且在不再用于防止内存泄漏时必须销毁发射器本身（可以实现此目的）使用特定的发射器破坏功能或破坏发射器所属的整个粒子系统）。

句法：

part_emitter_create(ps);

争论描述
ps在其中创建发射器的粒子系统。

返回值：

Real

例子：

p_emit1 = part_emitter_create(Sname);

这将创建一个新的粒子发射器并将其索引存储在变量“ p_emit”中。

背面：粒子发射器
