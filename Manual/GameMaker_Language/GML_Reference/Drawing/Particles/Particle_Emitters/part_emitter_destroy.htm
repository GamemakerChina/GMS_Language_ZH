part_emitter_destroy

该函数将删除从给定的系统中的指定发射器和从存储器清除（这也将与由所述给定的发射器正在生产停止的任何颗粒，但它不从房间内排出它们）。当系统不再需要给定的发射器以防止内存泄漏和错误时，应始终调用此函数。

句法：

part_emitter_destroy( ps, ind );

争论描述
ps破坏发射器的粒子系统。
ind要销毁的发射器的索引。

返回值：

N/A

例子：

if part_emitter_exists(global.Sname, p_emit)
    {
    part_emitter_destroy(global.Sname, p_emit1);
    }

上面的代码将检查给定粒子系统中是否存在在变量“ p_emit”中索引的粒子发射器，如果确实存在，则将其销毁。

背面：粒子发射器
