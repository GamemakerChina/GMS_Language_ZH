辐射器的全部内容

这个函数将从给定的系统中删除所有定义的发射器，并从内存中清除它们（这也将停止任何由给定的发射器产生的粒子，但它不会从房间中删除它们）。这个函数应该总是在 始终要在系统不再需要发射器的时候调用，以防止内存泄漏和错误。

语法。

part_emitter_destroy_all( ps )。

争论 描述
诗句 销毁所有发射器的粒子系统。

返回。

不适用

例子。

if lives = 0
{
part_emitter_destroy_all(global.Sname);
room_goto(rm_Menu)。
}

上面的代码检查了内置的全局变量 "lives"，如果它是0，它就会破坏所有的粒子发射器，然后改变房间。
