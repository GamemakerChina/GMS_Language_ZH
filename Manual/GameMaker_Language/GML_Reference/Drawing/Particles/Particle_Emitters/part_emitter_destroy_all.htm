part_emitter_destroy_all

此功能会从给定的系统中删除所有定义的发射器和从内存中清除它们（这也从给定的发射器正在生产停止任何粒子，但它并没有从房间中删除）。当系统不再需要发射器以防止内存泄漏和错误时，应始终调用此函数。

句法：

part_emitter_destroy_all( ps );

争论描述
ps摧毁所有发射器的粒子系统。

返回值：

N/A

例子：

if lives = 0
    {
    part_emitter_destroy_all(global.Sname);
    room_goto(rm_Menu);
    }

上面的代码检查内置的全局变量“生命”，如果它是0，它将破坏所有粒子发射器，然后改变空间。
