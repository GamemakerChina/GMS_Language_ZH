part_emitter_exists

使用此功能，您可以检查给定的粒子发射器索引是否存在于给定的系统中。请注意，如果检查的变量是未初始化的变量（否则将向粒子发射器分配其索引），这将引发错误。

句法：

part_emitter_exists(ps, ind);

争论描述
ps用于检查发射器的粒子系统。
ind要搜索的发射器的索引。

返回值：

Boolean

例子：

if part_emitter_exists(sname, p_emit1)
    {
    part_emitter_burst(sname, p_emit1, part_1, 30);
    }

上面的代码将检查在变量“ permit”中索引的发射器，如果存在，它将从中发射30个粒子。
