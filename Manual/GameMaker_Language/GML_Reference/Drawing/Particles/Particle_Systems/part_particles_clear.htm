part_particles_clear

使用此功能，您可以从房间清除系统当前创建的所有粒子。它不会重置或删除粒子类型本身，而不会重置或删除它们的视觉表示，并且如果您有任何对象从发射器流传输粒子，这些粒子将消失，但在调用此代码后将在下一步骤再次出现。

句法：

part_particles_clear(ind);

争论描述
ind粒子系统的索引。

返回值：

N/A

例子：

if (lives <= 0)
    {
    part_particles_clear(global.Sname);
    room_goto(rm_intro);
    }

上面的代码将检查变量“ lives”的值，如果该值等于0，它将清除系统中的所有粒子，然后更改空间。
