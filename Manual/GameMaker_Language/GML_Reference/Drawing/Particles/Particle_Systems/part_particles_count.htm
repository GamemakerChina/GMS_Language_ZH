part_particles_count

使用此功能，您可以检查粒子系统当前是否在房间中创建了任何粒子，并且还会返回它们的数量。

句法：

part_particles_count(ind);

争论描述
ind粒子系统的索引。

返回值：

Real

例子：

if (part_particles_count(Sname) == 0)
    {
    part_system_destroy(Sname);
    instance_destroy();
    }

上面的代码将检查在变量“ Sname”中索引的局部粒子系统中的粒子数，如果不存在，它将破坏系统，然后破坏实例。
