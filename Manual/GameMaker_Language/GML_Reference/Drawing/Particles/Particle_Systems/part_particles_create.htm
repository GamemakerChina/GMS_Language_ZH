part_particles_create

此功能非常适合不需要粒子发射器提供的任何功能的效果（例如，从导弹中产生烟雾或简单的爆炸效果），因为它使您可以快速轻松地在粒子发射器的任何位置创建粒子。游戏室。请注意，您必须先创建粒子系统和粒子类型，然后才能使用此功能。

句法：

part_particles_create(ind, x, y, parttype, number);

争论描述
ind粒子系统的索引。
x在何处创建粒子的x坐标。
y在何处创建粒子的y坐标。
parttype要创建的粒子的索引（类型）。
number要创建的粒子数。

返回值：

N/A

例子：

if mouse_check_button(mb_left)
    {
    part_particles_create(sname, mouse_x, mouse_y, p_CursorEffect, 5);
    }

上面的代码检查鼠标按钮是否被按下以及是否返回 true它在鼠标位置生成5个粒子。
