part_particles_create

此功能非常适合不需要粒子发射器提供的任何功能的效果（例如，从导弹中产生烟雾或简单的爆炸效果），因为它使您可以快速轻松地在粒子发射器的任何位置创建粒子。游戏室。您也可以在使用此功能创建粒子时对其进行“动态着色”，并且该颜色将覆盖粒子的预定义颜色，但不会在粒子的整个生命周期中混合该颜色。请注意，您必须先创建粒子系统和粒子类型，然后才能使用此功能。

句法：

part_particles_create_colour(ind, x, y, parttype, colour, number);

争论描述
ind粒子系统的索引。
x在何处创建粒子的x坐标。
y在何处创建粒子的y坐标。
parttype要创建的粒子的索引（类型）。
colour粒子的颜色混合。
number要创建的粒子数。

返回值：

N/A

例子：

if (speed > 1)
    {
    var _c = choose(c_aqua, c_lime, c_fuschia, c_orange);
    part_particles_create_colour(sname, x, y, p_Smoke, _c, 1);
    }

上面的代码将在具有代码实例的速度大于1的每一步中生成单个粒子。这些颗粒将是随机的颜色。
