part_system_automatic_draw

此功能可用于关闭粒子系统的图形，以便看不到对系统所做的任何更新（自动或其他）。这是纯粹的视觉选项，当设置为 false您将无法看到未绘制的粒子，但它们仍然存在，并且像往常一样会改变位置，颜色等。关闭自动绘图功能后，您可以使用以下功能明确命令GameMaker Studio 2绘制粒子系统的当前状态 part_system_drawit()，并且如果将此功能设置为 true再次，您可以重新打开自动绘图。

要注意的一件事是，如果您使用的是函数创建的简单效果 effect_create_above()或者 effect_create_below()然后您可以将0（对于下面的效果）或1（对于上面的效果）的值用作粒子系统索引，因此也可以为它们切换自动绘制（这也将为DnD™粒子效果切换绘制）。

句法：

part_system_automatic_draw(ind, automatic);

争论描述
ind要更改的粒子系统的索引。
automatic自动绘图是否启用（true），否则（false）。

返回值：

N/A

例子：

part_system_automatic_draw(global.Sname, false);

上面的代码关闭了在全局变量“ Sname”中索引的粒子系统的自动绘图。
