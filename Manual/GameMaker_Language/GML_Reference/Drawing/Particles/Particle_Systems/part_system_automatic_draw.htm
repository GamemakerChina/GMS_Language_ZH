栏目组_系统_自动画图

这个函数可以用来关闭粒子系统的绘制，这样就不会看到对该系统所做的任何更新（自动或其他）。这是一个纯粹的视觉选项，当设置为false时，你将无法看到 粒子没有被绘制出来，但是它们仍然存在，并且正在改变位置、颜色等......就像它们正常情况下那样。当自动绘制关闭时，你可以明确命令GameMaker Studio 2绘制粒子的当前状态。 系统的当前状态，使用函数 part_system_drawit()，如果你再把这个函数设置为"真"，你就可以把自动绘制功能重新打开。

有一点需要注意的是，如果你使用由函数创建的简单效果 效果_创建_上方()或 effect_create_below()创建的简单效果，那么你可以使用 0（用于下面的效果）或1（用于上面的效果）作为粒子系统的索引，因此也可以切换这些粒子的自动绘制（这也可以用来切换DnD™粒子效果的绘制）。

语法。

part_system_automatic_draw(ind, automatic)。

争论 描述
ind 要改变的粒子系统的索引。
自动 自动绘图是否开启（真）或不开启（假）。

返回。

不适用

例子。

part_system_automatic_draw(global.Sname, false)。

上述代码关闭了全局变量 "Sname "中索引的粒子系统的自动绘图。
