part_system_automatic_update

此功能控制GameMaker Studio 2是否应自动更新粒子系统，通常您无需将此功能用作默认值。true （启用自动更新）是您希望发生的事情。但是，对于特殊效果或暂停游戏，您可以将其设置为 false除非使用该功能明确命令，否则它将防止在给定的粒子系统上执行任何更新 part_system_update()或者您再次使用此功能打开自动更新。没有更新意味着在该系统中创建的粒子将不再更改位置，颜色或任何其他参数，并且发射器也将停止工作，从而在将自动更新设置为的确切点处“冻结”系统。false 。

要注意的一件事是，如果您使用的是函数创建的简单效果 effect_create_above()或者 effect_create_below()然后您可以将0（对于下面的效果）或1（对于上面的效果）的值用作粒子系统索引，因此也可以切换这些值的自动更新（这也将切换DnD™粒子效果的更新）。

句法：

part_system_automatic_update(ind, automatic);

争论描述
ind要更改的粒子系统的索引。
automatic是否启用自动更新（true ）或否（错误）。

返回值：

N/A

例子：

if global.Pause
    {
    part_system_automatic_update(global.Sname, false);
    }

如果全局变量“ Paused”为true，则以上代码将关闭粒子更新。
