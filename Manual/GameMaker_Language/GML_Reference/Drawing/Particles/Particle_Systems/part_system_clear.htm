part_system_clear

使用此功能，您可以将索引系统清除为默认状态，删除所有发射器并重置深度和位置（如果已更改）。请小心使用此功能，就像使用该系统关联的发射器进行实例设置或猝发或执行其他任何操作一样，除非使用 part_emitter_exists()功能检查。也无需致电 part_emitter_destroy()功能，因为它也自动照顾。

注意：此功能将清除房间中可见的粒子，但不会清除粒子属性，也不会从内存中删除它们。为此，您应该使用part_type_clear（）和part_type_destroy（）函数。

句法：

part_system_clear(ind);

争论描述
ind要清除的粒子系统的索引。

返回值：

N/A

例子：

part_system_clear(global.Sname);

上面的代码将清除全局变量“ Sname”中索引的粒子系统为其默认状态。
