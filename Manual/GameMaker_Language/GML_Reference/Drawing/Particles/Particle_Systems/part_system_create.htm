part_system_create

此函数用于创建新的粒子系统，并将返回唯一的索引号，该索引号应在与该系统有关的所有其他函数中存储和使用。系统将被分配一个托管层，并将其深度设置为0。用户无法访问托管层，并且仅当使用深度而不是层时，托管层才用于内部管理。通常您会使用该功能 part_system_create_layer()而不是这个。

句法：

part_system_create();

返回值：

Real

例子：

global.Sname = part_system_create();

这将创建一个新的粒子系统，并将索引存储在全局变量“ Sname”中。
