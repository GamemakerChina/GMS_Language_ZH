部分系统深度

通过这个功能，你可以设置粒子系统的绘制深度，就像你可以设置房间内不同层的渲染深度一样，低绘制深度意味着它将出现在所有绘制深度较高的东西之上 低的绘制深度意味着它将出现在所有绘制深度较高的东西的上面，而高的绘制深度则将它放在所有绘制深度较低的东西的下面。

语法。

part_system_depth( ind, depth )。

争论 描述
ind 要改变的粒子系统的索引。
深度 设置粒子系统的深度。

返回。

不适用

例子。

global.Sname = part_system_create();
part_system_depth(global.Sname, -1000 ) 。

上述代码将创建一个粒子系统，并将其索引存储在全局变量 "Sname "中。然后这个系统被赋予-1000的低深度，这意味着它将出现在所有具有更高绘制深度的事物之上。
