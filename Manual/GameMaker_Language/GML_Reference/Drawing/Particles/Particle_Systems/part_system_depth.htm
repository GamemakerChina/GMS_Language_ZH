part_system_depth

使用此功能，您可以设置粒子系统的绘制深度，与设置房间内不同层的渲染深度相同，其中较低的绘制深度意味着它将显示在所有具有较高绘制深度的事物之上深度和较高的绘制深度，将其放置在具有较低绘制深度的所有内容下方。

句法：

part_system_depth( ind, depth );

争论描述
ind要更改的粒子系统的索引。
depth设置粒子系统的深度。

返回值：

N/A

例子：

global.Sname = part_system_create();
part_system_depth(global.Sname, -1000 );

上面的代码将创建一个粒子系统并将其索引存储在全局变量“ Sname”中。然后为该系统提供-1000的较低深度，这意味着它将以较高的绘制深度出现在所有内容之上。

背面：粒子系统
