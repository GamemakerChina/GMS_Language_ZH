part_system_destroy

使用此功能，您可以销毁给定的粒子系统并将其从内存中删除。当不再需要该系统时（例如，在房间的尽头或实例的销毁事件），应始终调用此方法，否则最终可能会在较晚的房间中出现粒子，并且无法移除它们内存泄漏最终将导致游戏崩溃。

注意：此功能还将销毁并删除可能已创建并与要销毁的系统相关联的所有发射器。

句法：

part_system_destroy(ind);

争论描述
ind要破坏的粒子系统的索引。

返回值：

N/A

例子：

if (global.player_hp <= 0)
    {
    part_system_destroy(p_sys);
    room_goto_next();
    }

上面的代码检查全局变量值是否小于或等于零，如果为零，则销毁给定变量中引用的粒子系统，然后转到下一个房间。
