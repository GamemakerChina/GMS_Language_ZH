销毁系统

通过这个函数，你可以销毁一个给定的粒子系统并将其从内存中移除。这个函数应该总是在不再需要这个系统的时候被调用，比如在一个房间结束的时候，或者在一个实例的销毁事件中，否则你会发现 粒子出现在以后的房间里，并且没有办法将它们删除，以及出现内存泄漏，这最终会使你的游戏崩溃。

注意：这个函数也将销毁和删除任何可能已经创建并与被销毁系统相关的发射器。

语法。

part_system_destroy(ind)。

争论 描述
ind 要销毁的粒子系统的索引。

返回。

不适用

例子。

if (global.player_hp <= 0)
{
part_system_destroy(p_sys);
room_goto_next() ;
}

上面的代码检查全局变量的值是否小于或等于0，如果是，那么它就会销毁给定变量中引用的粒子系统，然后进入下一个房间。
