栏目中的 "系统 "和 "绘图 "的顺序

通过这个函数，你可以设置在屏幕上创建粒子时的绘制方式。默认的系统使用旧的>新的外观（该函数被设置为"true"），旧的粒子被绘制在比新的粒子更高的深度，因此出现在新的粒子的 "下面"。 但通过将这个函数设置为 "假"，你可以颠倒这个顺序，让新粒子画得更高，因此出现在老粒子的 "下面"。下面的图片说明了这一点。 左边的图片为默认值true，右边的图片为false。

注意：当用加法混合模式绘制粒子时，这个功能的效果可能不一定明显。

语法。

part_system_draw_order(ind, oldtonew)。

争论 描述
ind 要改变的粒子系统的索引。
鏂囩歞ean 旧的粒子是否应该被画在新的粒子后面（真）或反过来（假）。

返回。

不适用

例子。

mysystem = part_system_create();
part_system_draw_order(mysystem, true);

这将创建一个新的粒子系统，其索引为 "mysystem"，然后它设置粒子，将较新的粒子画在较旧的粒子之上。
