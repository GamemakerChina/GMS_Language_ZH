part_system_draw_order

使用此功能，可以设置在屏幕上创建粒子时绘制粒子的方式。默认系统使用旧的>新外观（该功能设置为 true），旧粒子的绘制深度要比新粒子的深度大，因此在新粒子“下方”出现，但是将此功能设置为 false您可以颠倒此顺序并将新粒子绘制得更高，因此出现在旧粒子“下方”。下图说明了这一点，左图是的默认值。true右边的图像是 false：

注意：使用加性混合模式绘制粒子时，此功能的效果可能并不总是很明显。

句法：

part_system_draw_order(ind, oldtonew);

争论描述
ind要更改的粒子系统的索引。
oldtonew是否应将旧粒子吸引到较新的粒子之后（true），反之亦然（false）。

返回值：

N/A

例子：

mysystem = part_system_create();
part_system_draw_order(mysystem, true);

这将创建一个索引为“mysystem ”，然后设置粒子以在较旧的粒子上方绘制较新的粒子。
