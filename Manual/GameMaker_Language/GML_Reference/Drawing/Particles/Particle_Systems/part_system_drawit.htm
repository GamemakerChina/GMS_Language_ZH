part_system_update

使用此功能，您可以强制粒子系统在屏幕上绘制所有当前粒子。如果 part_system_automatic_draw()如果将其关闭，则此功能将在实例的绘制事件中使用时显示粒子，或者可以在将绘制目标设置为曲面（使用 surface_set_target()），将系统中的粒子吸引到该表面。

句法：

part_system_drawit(ind);

争论描述
ind绘制粒子的粒子系统的索引。

返回值：

N/A

例子：

if surface_exists(surf)
    {
    surface_set_target(surf);
    part_system_drawit(global.psys);
    surface_reset_target();
    }

上面的代码检查是否存在在变量“ surf”中索引的曲面，如果确实存在，则将绘图目标设置为该曲面，绘制具有存储在全局变量中的ID的粒子系统，然后重置绘图目标，以便所有普通图形再次出现在屏幕上。
