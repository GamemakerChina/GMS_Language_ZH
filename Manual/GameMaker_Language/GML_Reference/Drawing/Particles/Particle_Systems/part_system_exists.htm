part_system_exists

使用此功能，您可以检查游戏中是否存在给定索引的给定粒子系统。请注意，如果要检查的变量是未初始化的变量（否则将为粒子系统分配其索引），这将引发错误。

句法：

part_system_exists(ind);

争论描述
ind要检查的粒子系统的索引。

返回值：

Boolean

例子：

if !part_system_exists(global.Sname)
    {
    global.Sname = part_system_create();
    }

上面的代码检查全局变量中引用的粒子系统是否存在，如果不存在，则创建该粒子系统。
