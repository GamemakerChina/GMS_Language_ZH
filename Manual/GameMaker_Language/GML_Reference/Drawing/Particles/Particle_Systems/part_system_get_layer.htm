部分系统_获取图层

这个函数可以用来为给定的粒子系统检索唯一的层ID值。你提供粒子系统的ID值（由函数返回 part_system_create_layer()返回的），然后该 函数将返回该层的ID值。

语法。

part_system_get_layer(ind)。

争论 描述
ind 粒子系统的ID值，以获得该层的ID。

返回。

实数（唯一的层ID值，如果是内部管理的层则为0

例子。

如果（part_system_get_layer(global.p_sys) != "effects_layer"）
{
part_system_layer(global.p_sys, "effects_layer";
}

上面的代码将检查一个粒子系统，看它在哪个层上，如果它不在给定的层上，它将被改变。
