part_system_layer

此功能可用于将粒子系统从其当前层切换到新的层。您提供粒子系统ID值（由函数返回）part_system_create_layer() ），然后是唯一的图层ID（由函数返回）layer_create()或用作字符串的层的名称（例如，在房间编辑器中定义的“ instance_layer”），系统将移至新层。

句法：

part_system_layer(ps, layer);

争论描述
ps粒子系统ID
layer新图层ID

返回值：

N/A

例子：

if (part_system_get_layer(global.p_sys) != "effects_layer")
    {
    part_system_layer(global.p_sys, "effects_layer";
    }

上面的代码将检查粒子系统，以查看其位于哪个层上，如果不在给定层上，则将对其进行更改。
