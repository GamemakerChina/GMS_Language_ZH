part_system_position

使用此功能，您可以相对于房间的（0,0）位置设置粒子系统的基本位置，这意味着现在将相对于新位置绘制与此系统有关的所有其他粒子功能。默认情况下，此位置始终为（0,0），但是在某些非常特殊的情况下，您可能希望将其更改为其他位置。

注意：此功能将更改粒子系统中的所有内容，因此，如果在位置（100,100）处有发射器，然后将粒子系统位置设置为（0,100），则发射器现在将在（100,200）处绘制。如果您移动系统然后创建发射器，也会发生同样的情况，即使您在（100,100）处创建了发射器，也会在（100,200）处绘制它。

句法：

part_system_position(ind, x, y);

争论描述
ind要更改的粒子系统的索引。
x粒子系统的新x坐标。
y粒子系统的新y坐标。

返回值：

N/A

例子：

if mouse_check_button_pressed(mb_left)
    {
    part_system_position(global.Sname, mouse_x, mouse_y);
    }

上面的代码将检查是否按下了鼠标按钮，如果检测到按下该按钮，则在全局变量“ Sname”中索引的粒子系统将移动到鼠标的x / y位置
