part_system_update

此功能将使给定的粒子系统在游戏时间中前进一个步骤，并且可以在关闭自动更新时用于模拟粒子系统的行为，也可以用于一步一步将粒子系统推进到特定点。例如，如果您有一个用于雪的粒子系统，并且您希望它看起来像是从玩家进入房间那一刻起就开始下雪，那么您可能希望在创建事件中将雪粒子前进几秒钟一个实例，以便第一次绘制它们时，屏幕上到处都有粒子（下面的代码示例显示了这一点）。当通过该功能关闭系统的自动更新时，也可以使用此功能 part_system_automatic_update()移动系统。

句法：

part_system_update(ind);

争论描述
ind要更新的粒子系统的索引。

返回值：

N/A

例子：

repeat (room_speed * 3)
    {
    part_system_update(global.RainSys);
    }

上面的代码将使在全局变量“ RainSys”中索引的粒子系统前进3秒钟。
