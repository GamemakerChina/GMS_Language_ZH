粒子类型

粒子是可以通过在游戏中使用粒子系统来创造的图形效果。你用这些函数定义的粒子可以在你创建的任何系统中使用，通常的程序是在游戏开始时一次定义所有的粒子 在游戏开始时定义一次，并将其索引值存储在全局变量中。然后，这些全局变量可以在整个游戏过程中的任何时候用于爆裂、发射或创建粒子。你还可以根据需要创建、使用和销毁粒子 如果内存是一个问题，或者如果你希望有许多不同的类型，但只在游戏中的特定点上。无论你选择什么，粒子都是一种非常通用的资源。

粒子需要被创建，并将其索引存储在一个变量中，以便所有进一步的粒子例程能够正确访问它们。以下函数用于创建、检查、清除和销毁粒子类型。

  • 部分_类型_存在
  • 创建零件类型
  • 毁坏的部分_类型
  • 栏目组_type_clear

一旦你创建了你的粒子类型，你应该给它一个形状。这可以是14个内置的形状之一，这些形状是带有阿尔法混合的64x64像素的精灵，或者你可以使用你自己的精灵，这些精灵可以是动画的、静态的或者使用 随机子图像。下面的函数定义了每个粒子的视觉基本形状。

  • 形状_类型_形状
  • 部分_类型_斯普利特

粒子也可以有一个方向和运动。与大多数粒子函数一样，速度、方向和方位被解析为一个随机值，从一个给定的范围内选择，可以用以下函数来设置。
