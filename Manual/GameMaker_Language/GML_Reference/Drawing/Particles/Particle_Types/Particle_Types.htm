粒子类型

粒子是可以通过在游戏中使用粒子系统创建的图形效果。使用这些功能定义的粒子可以在您创建的任何系统中使用，通常的过程是在游戏开始时一次定义所有粒子并将其索引值存储在全局变量中。这些全局变量然后可以在整个游戏中的任何时候用于爆发，发射或创建粒子。如果存在内存问题，或者您希望有许多不同的类型，但仅在游戏的特定点上，还可以根据需要创建，使用然后销毁粒子。无论您选择什么，粒子都是非常多用途的资源！

需要创建粒子并将其索引存储在变量中，以便所有其他粒子例程都可以正确访问它们。以下功能用于创建，检查，清除和销毁粒子类型：

  • part_type_exists
  • part_type_create
  • part_type_destroy
  • part_type_clear

一旦创建了粒子类型，就应该给它一个形状。这可以是14种内置形状中的一种，这些形状是具有alpha混合功能的64x64像素精灵，或者您可以使用自己的精灵，可以动画，静态或使用随机子图像。以下功能定义了每个粒子的视觉基础形状：

  • part_type_shape
  • part_type_sprite

下图显示了可以选择的14种内置粒子类型：

为粒子类型（默认形状或您自己的形状）选择了精灵后，您可以指示它的大小和比例。可以将大小指定为一个范围，并且将使用从给定范围内选择的大小来创建所有粒子。该比例是在添加任何尺寸变化之前，粒子的x和y轴比例的“基本”值，并且使用以下函数来控制这些值：

  • part_type_size
  • part_type_scale

粒子也可以具有方向和运动。与大多数粒子函数一样，速度，方向和方向被解析为一个从给定范围内选择的随机值，可以使用以下函数进行设置：

  • part_type_speed
  • part_type_direction
  • part_type_gravity
  • part_type_orientation

整理好粒子的样式和运动后，即可设置其颜色和Alpha值，这两个值都可以设置为粒子整个生命周期中的绝对值，也可以设置为从彼此之间逐渐形成。请注意，还有一个特殊功能允许您设置粒子及其周围环境的混合，它既可以是普通的也可以是叠加的，并且会产生一些奇妙的效果-控制颜色，alpha和混合的功能是：

  • part_type_colour_mix
  • part_type_colour_rgb
  • part_type_colour_hsv
  • part_type_colour1
  • part_type_colour2
  • part_type_colour3
  • part_type_alpha1
  • part_type_alpha2
  • part_type_alpha3
  • part_type_blend

粒子在屏幕上存在的时间有限，这称为“寿命”。粒子寿命是分步测量的（与以前的功能类似），它表示为一个值范围，从该值可以给粒子指定的寿命。还有两个有趣的功能，可用于使粒子生成其他粒子，无论是在每个步骤中还是在它们“死亡”时（即：从屏幕上移除），但在使用这些功能时要非常小心，因为它们会快速创建数千个粒子并使游戏崩溃！设置寿命和步进或导致死亡的函数如下：

  • part_type_life
  • part_type_step
  • part_type_death

值得注意的是，由于可以更改和调整大量参数，因此通过代码创建粒子通常是一个费力且耗时的过程。因此，建议您四处搜寻并掌握某种类型的粒子设计器（或自己动手做！），以免产生粒子。
