part_type_alpha1

此函数用于将粒子类型设置为在每个单个粒子的整个生命周期内具有单个alpha值（透明度），并且可以从0（透明）到1（不透明）。

句法：

part_type_alpha1(ind, alpha1);

争论描述
ind要更改的粒子类型的索引。
alpha1粒子的Alpha。

返回值：

N/A

例子：

part_type_alpha1(global.Snow_Part, 0.5);

上面的代码会将所有在全局变量“ Snow_Part”中索引的粒子类型创建的粒子的alpha值设置为0.5（半透明）。
