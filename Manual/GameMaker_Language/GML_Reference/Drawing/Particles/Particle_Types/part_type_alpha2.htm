栏目组_类型_alpha2

这个函数可以用来为每个创建的给定类型的粒子设置一个alpha值（透明度）梯度。第一个alpha值是所有粒子开始时的值，第二个alpha值是粒子结束时的值。 并且在粒子的生命周期中，alpha将发生从一个到另一个的平滑梯度变化。这可以从0（透明）到1（不透明）。

语法。

part_type_alpha2(ind, alpha1, alpha2)。

争论 描述
ind 要改变的粒子类型的索引。
α1 粒子的起始α。
阿尔法2 粒子的末端α。

返回。

不适用

例子。

part_type_alpha2(global.Snow_Part, 0, 1)。

上述代码将设置所有创建的粒子类型的粒子在全局变量 "Snow_Part "中的索引，使其具有0的alpha值（透明），然后在其生命周期结束时淡化为1的alpha（不透明）。
