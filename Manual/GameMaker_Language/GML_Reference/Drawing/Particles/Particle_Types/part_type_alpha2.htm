part_type_alpha2

此功能可用于为给定类型创建的每个粒子设置alpha值（透明度）渐变。第一个alpha是所有粒子的起点，第二个alpha是该粒子的终点，并且alpha在整个粒子生命周期中（从一个到另一个）都将发生平滑的渐变变化。可以是0（透明）到1（不透明）。

句法：

part_type_alpha2(ind, alpha1, alpha2);

争论描述
ind要更改的粒子类型的索引。
alpha1粒子的起始Alpha。
alpha2粒子的末端alpha。

返回值：

N/A

例子：

part_type_alpha2(global.Snow_Part, 0, 1);

上面的代码将所有在全局变量“ Snow_Part”中索引的粒子类型创建的粒子的alpha值设置为0（透明），然后在其生命周期结束时淡入以使其alpha值为1（不透明）。
