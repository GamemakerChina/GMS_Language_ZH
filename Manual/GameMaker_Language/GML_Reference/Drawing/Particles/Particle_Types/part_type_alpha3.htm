part_type_alpha3

此函数可用于为给定类型创建的每个粒子设置三个alpha（透明度）值梯度。第一个阿尔法是所有粒子的起点，第二个阿尔法是将混合至其寿命一半的粒子，第三个阿尔法是粒子的终点。在整个粒子生命周期中，从一个粒子到另一个粒子，整个alpha都会发生平滑的梯度变化。

句法：

part_type_alpha3(ind, alpha1, alpha2, alpha3);

争论描述
ind要更改的粒子类型的索引。
alpha1粒子的起始Alpha。
alpha2粒子的中点alpha。
alpha3粒子的结尾alpha。

返回值：

N/A

例子：

part_type_alpha3( part_Health, 0.5, 1, 0);

上面的代码将使所有在变量“ part_Health”中索引类型创建的粒子在其生命周期内具有从0.5到1到0的alpha混合。
