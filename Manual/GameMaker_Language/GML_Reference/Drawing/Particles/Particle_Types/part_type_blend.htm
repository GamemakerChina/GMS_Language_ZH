part_type_blend

使用此功能，您可以使用附加混合模式绘制粒子（true ） 或不 （false ）。附加混合是一种特殊的混合模式，该模式会在每个粒子重叠时增加其亮度值，以便当它们重叠时，浅色将逐渐变亮（直到它们显示为白色），而深色则变得越来越透明，而黑色几乎是不可见的。此功能可以提供一些美丽的粒子效果，但不一定总是必需的。例如，烟雾痕迹将使添加剂混合掉以保持灰色效果，但是火焰效果可能会使其燃烧，以使粒子更透明和更明亮。

句法：

part_type_blend(ind, additive);

争论描述
ind要更改的粒子类型的索引。
additive是否应该以加性方式（true）或以正常方式（false）混合粒子。

返回值：

N/A

例子：

part_type_blend(part_Fire, true);

上面的代码将所有在变量“ part_Fire”中索引的粒子类型创建的粒子设置为具有附加混合模式。
