part_type_clear

使用此功能，您可以“重置”粒子，将与该粒子相关的每个功能的所有值（寿命，颜色，alpha，方向等）恢复为其默认值。请注意，此功能不会从屏幕上删除房间中当前可见的任何微粒，因为您应该使用 part_particles_clear()。

句法：

part_type_clear(ind);

争论描述
ind要清除的粒子类型的索引。

返回值：

N/A

例子：

part_type_clear(global.explode_part);

上面的代码将清除全局变量“ explode_part”中索引的粒子类型为其默认值。
