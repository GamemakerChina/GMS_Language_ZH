部件_类型_颜色1

该函数用于将一个粒子类型设置为在每个单独的粒子寿命的总时间内为单一颜色。

语法。

part_type_colour1(ind, color1)。

争论 描述
ind 要改变的粒子类型的索引。
颜色1 制作粒子类型的单一颜色。

返回。

不适用

例子。

part_type_colour1(global.Snow_Part, c_white)。

上述代码将把所有以全局变量 "Snow_Part "为索引的粒子类型创建的粒子设置为仅为白色。
