part_type_colour1

此功能用于将粒子类型设置为每种粒子的整个生命周期内的单一颜色。

句法：

part_type_colour1(ind, colour1);

争论描述
ind要更改的粒子类型的索引。
colour1成为粒子类型的单一颜色。

返回值：

N/A

例子：

part_type_colour1(global.Snow_Part, c_white);

上面的代码会将所有在全局变量“ Snow_Part”中索引的粒子类型创建的粒子设置为仅白色。
