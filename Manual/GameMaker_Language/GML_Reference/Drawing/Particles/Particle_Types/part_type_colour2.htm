部件_类型_颜色2

这个函数可以用来为每个创建的给定类型的粒子设置一个双色渐变。第一种颜色是所有粒子开始时的颜色，第二种颜色是粒子结束时的颜色。 在粒子的一生中，颜色将从一种颜色变为另一种颜色，并发生平滑的梯度变化。

语法。

part_type_colour2(ind, color1, color2)。

争论 描述
ind 要改变的粒子类型的索引。
颜色1 粒子开始时的颜色。
颜色2 粒子结束时的颜色。

返回。

不适用

例子。

part_type_colour2( part_Health, c_red, c_white)。

上述代码将使所有以 "part_Health "变量索引的类型创建的粒子在其生命周期内有一个从红色到白色的混合颜色。
