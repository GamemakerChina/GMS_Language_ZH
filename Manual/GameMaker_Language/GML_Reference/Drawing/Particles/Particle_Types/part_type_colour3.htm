part_type_colour3

此功能可用于为给定类型创建的每个粒子设置三个颜色渐变。第一种颜色是所有粒子开始使用的颜色，第二种颜色是将混合到其寿命中途的一种颜色，第三种颜色是粒子将使用其结束的颜色。在整个粒子寿命中，从一种颜色到另一种颜色，颜色之间将发生平滑的渐变变化。

句法：

part_type_colour3( ind, colour1, colour2, colour3 );

争论描述
ind要更改的粒子类型的索引。
colour1粒子将开始的颜色。
colour2粒子的颜色将处于其寿命的一半。
colour3粒子将结束的颜色。

返回值：

N/A

例子：

part_type_colour3( part_Health, c_red, c_white, c_maroon);

上面的代码将使所有在变量“ part_Health”中索引类型创建的粒子在其生命周期内具有从红色到白色到栗色的颜色混合。
