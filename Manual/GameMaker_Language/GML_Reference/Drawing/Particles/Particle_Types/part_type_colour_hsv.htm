part_type_colour_hsv

使用此功能，可以为给定类型的所有粒子设置色相，饱和度和值范围。您可以为这三个组件的每一个提供最小值和最大值，并且根据给定的参数范围，创建的粒子将具有随机颜色。这样，您可以创建具有相同色相但饱和度不同，或具有不同色相但值（发光度）相同等的粒子。所有值都必须在0到255之间。

句法：

part_type_colour_hsv(ind, hmin, hmax, smin, smax, vmin, vmax);

争论描述
ind要更改的粒子类型的索引。
hmin最终颜色的色相成分可以是最小值。
hmax最终颜色的色相分量可以最大。
smin最终颜色的饱和度分量可以是最小值。
smax最终颜色的饱和度分量可以最大。
vmin最终颜色值的最小值可以是。
vmax最终颜色值的最大值可以是。

返回值：

N/A

例子：

part_type_hsv(global.Stars, 0, 255, 0, 255, 255, 255 );

上面的代码将全局变量“ Stars”中索引的粒子类型发射的每个粒子设置为具有不同的颜色和饱和度，但具有相同的值（发光度）。
