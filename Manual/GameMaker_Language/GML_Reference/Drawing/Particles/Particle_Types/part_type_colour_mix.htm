part_type_colour_mix

使用此功能，您可以将给定的粒子类型设置为两种颜色的随机混合。

句法：

part_type_colour_mix(ind, colour1, colour2);

争论描述
ind要更改的粒子类型的索引。
colour1混合将采用的第一种颜色。
colour2混合将采用的第二种颜色。

返回值：

N/A

例子：

part_type_colour_mix(global.P_Damage, c_red, c_yellow);

上面的代码将为全局变量“ P_Damage”中索引的粒子类型的每个粒子发射的颜色设置为红色和黄色之间的随机混合。

背面：粒子类型
