栏目组_类型_颜色_rgb

通过这个函数，你可以为所有创建的给定类型的粒子设置红色、绿色和蓝色的混合颜色。你为这三种成分中的每一种提供一个最小值和一个最大值，创建的粒子将根据给定的参数范围拥有一个随机的颜色 的随机颜色。所有数值必须在0到255之间。

语法。

part_type_colour_rgb(ind, rmin, rmax, gmin, gmax, bmin, bmax p>

争论 描述
ind 要改变的粒子类型的索引。
姆指 最终颜色的红色成分可以达到的最小值。
最大值 最终颜色的红色成分可以达到的最大值。
gmin 最终颜色的绿色成分可以达到的最小值。
平均值 最终颜色的绿色成分可以达到的最大值。
bmin 最终颜色的蓝色成分可以达到的最小值。
最大值 最终颜色的蓝色成分可以达到的最大值。

返回。

不适用

例子。

part_type_rgb(global.Blood_Part, 0, 255, 0, 0, 0, 0)。

上述代码将全局变量 "Blood_Part "中索引的粒子类型所发射的每个粒子设置为只有不同的红色。
