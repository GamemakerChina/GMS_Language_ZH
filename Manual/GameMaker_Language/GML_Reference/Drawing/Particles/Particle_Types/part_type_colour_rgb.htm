part_type_colour_rgb

使用此功能，可以为给定类型创建的所有粒子设置红色，绿色和蓝色的混合。您可以为这三个组件的每一个提供最小值和最大值，并且根据给定的参数范围，创建的粒子将具有随机颜色。所有值必须在0到255之间。

句法：

part_type_colour_rgb(ind, rmin, rmax, gmin, gmax, bmin, bmax p>

争论描述
ind要更改的粒子类型的索引。
rmin最终颜色的红色分量可以达到的最小值。
rmax最终颜色的红色分量最大。
gmin最终颜色的绿色分量最小。
gmax最终颜色的绿色分量可以达到的最大值。
bmin最终颜色的蓝色分量最小。
bmax最终颜色的蓝色分量最大。

返回值：

N/A

例子：

part_type_rgb(global.Blood_Part, 0, 255, 0, 0, 0, 0);

上面的代码将在全局变量“ Blood_Part”中索引的粒子类型发射的每个粒子设置为仅不同的红色阴影。
