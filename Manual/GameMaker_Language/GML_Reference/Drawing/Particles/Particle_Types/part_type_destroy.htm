毁坏的部分_类型

通过这个函数，你可以从游戏中移除指定的粒子类型。当你使用这个函数时，所有给定类型的粒子将从房间中消失，而粒子本身也会从内存中删除，所以这个函数应该只在 所以这个函数应该只在你不再需要这个粒子的时候使用。

语法。

part_type_destroy(ind)。

争论 描述
ind 要销毁的粒子类型的索引。

返回。

不适用

例子。

if part_particles_count(smoke_sys) = 0
{
part_type_destroy(smoke_part);
part_system_destroy(smoke_sys);
instance_destroy()。
}
else alarm[0] = 1;

上面的代码检查一个粒子系统，看当前房间里是否有任何粒子可见，如果没有，粒子、系统和实例就会被销毁。
