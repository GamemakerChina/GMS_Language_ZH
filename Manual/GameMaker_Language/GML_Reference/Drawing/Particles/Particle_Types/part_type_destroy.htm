part_type_destroy

使用此功能，您可以从游戏中删除指定的粒子类型。使用此功能时，给定类型的所有粒子将从房间中消失，并且粒子本身将从内存中删除，因此仅当不再需要粒子时才应使用此功能。

句法：

part_type_destroy(ind);

争论描述
ind要破坏的粒子类型的索引。

返回值：

N/A

例子：

if part_particles_count(smoke_sys) = 0
   {
   part_type_destroy(smoke_part);
   part_system_destroy(smoke_sys);
   instance_destroy();
   }
else alarm[0] = 1;

上面的代码检查粒子系统，以查看房间中当前是否可见任何粒子，如果没有，粒子，系统和实例将被销毁。
