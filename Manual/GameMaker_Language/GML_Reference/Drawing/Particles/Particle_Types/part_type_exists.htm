part_type_exists

使用此功能，您可以检查游戏中是否存在给定索引的给定粒子类型。请注意，如果要检查的变量是未初始化的变量（否则将为粒子类型分配其索引），这将引发错误。

句法：

part_type_exists(ind);

争论描述
ind要检查的粒子类型的索引。

返回值：

Boolean

例子：

if !part_type_exists(global.p1)
    {
    global.p1 = part_type_create();
    }

上面的代码检查全局变量“ p1”是否存储粒子类型的索引，如果不存储，则创建一个。
