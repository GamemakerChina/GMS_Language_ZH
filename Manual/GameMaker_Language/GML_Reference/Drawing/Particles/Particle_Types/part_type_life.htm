部分_类型_生命

这是管理每个指定类型的单个粒子在屏幕上保持多久的函数。和其他粒子函数一样，你提供一个最小值和一个最大值（以步数为单位），每个粒子的寿命将是一个随机数 在指定范围内的随机数。要使所有的粒子具有相同的寿命，请将这两个值设置为相同。

语法。

part_type_life(ind, life_min, life_max)。

争论 描述
ind 要改变的粒子类型的索引。
寿命_最小 颗粒的最小寿命。
寿命_最大值 颗粒的最大寿命。

返回。

不适用

例子。

part_type_life(part_Flare, room_speed, room_speed * 2)。

上述代码将根据当前的室速，把所有以 "part_Flare "变量为索引的粒子类型创建的粒子设定为具有1到2秒的寿命。
