part_type_shape

使用此功能，可以设置要用于粒子类型的精灵形状，可以接受以下常量：

持续的描述
pt_shape_pixel 1x1像素。 （这是默认设置。）
pt_shape_disk实心圆。
pt_shape_square一个实心的正方形。
pt_shape_line 8像素宽的水平线。
pt_shape_star五点填充恒星。
pt_shape_circle一个3像素的轮廓圆。
pt_shape_ring向内发光的圆圈（看起来像气泡）。
pt_shape_sphere一个带有向外发光的圆，中间是实心，向外发光。
pt_shape_flare刺耳的发光点（看起来像是夜晚的真实星星）。
pt_shape_spark火花效果-就像具有多个点渐渐消失的星星一样。
pt_shape_explosion方形烟雾云需要多种颜色才能像爆炸一样。
pt_shape_cloud薄云需要扩大规模，并且需要多个粒子才能像云。
pt_shape_smoke爆炸效果的平滑版本。使用多个创建烟雾云。
pt_shape_snow通用雪花形状。

句法：

part_type_shape(ind, shape);

争论描述
ind要设置的粒子类型的索引。
shape选择的形状。

返回值：

N/A

例子：

global.p1 = part_type_create();
part_type_shape(global.p1, pt_shape_square);
part_type_size(global.p1, 1, 3, 0, 0);
part_type_scale(global.p1, 1, 1);
part_type_colour1(global.p1, c_white);
part_type_alpha2(global.p1, 1, 0);
part_type_speed(global.p1, 2, 4, 0, 0);
part_type_direction(global.p1, 0, 180, 0, 0);
part_type_gravity(global.p1, 0.20, 270);
part_type_orientation(global.p1, 0, 0, 0, 0, 1);
part_type_blend(global.p1, 1);
part_type_life(global.p1, 15, 60);

上面的代码将索引在全局变量“ p1”中的粒子类型设置为正方形，然后设置其他各种粒子属性。
