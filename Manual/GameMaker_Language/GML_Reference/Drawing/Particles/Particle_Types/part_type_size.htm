part_type_size

此函数用于确定粒子在创建时的大小，也可以用于使粒子在其整个生命周期中增大或减小。最小和最大尺寸值默认为1（表示尺寸与后 part_type_scale()比例尺已设置），如果将它们设置为相同的值，则所有粒子的初始尺寸都将相同，但是如果设置为不同的值，则每个粒子的起始尺寸都将在该范围内，例如最小为0.5，最大为2，将创建所有粒子，其大小在基本比例的一半到两倍之间。

您还可以设置一个增量值，该增量值将使粒子精灵在粒子的整个生命周期内增长（如果为正数）或缩小（如果为负数），并在每个步骤中将给定的量添加到粒子大小中。该值可以是最小值（+/-）0.01。

最后，您可以设置“摆动”因子。该值将随机增加或减少到粒子寿命每一步的大小。显然，较大的值比较小的值更明显，并且该值甚至可以是负数，最大范围在-20到20之间。

句法：

part_type_size(ind, size_min, size_max, size_incr, size_wiggle);

争论描述
ind要更改的粒子类型的索引。
size_min粒子可以开始的最小尺寸。
size_max粒子可以开始的最大尺寸。
size_incr每步粒子应增加或减少多少。
size_wiggle每步粒度应随机添加或减去多少。

返回值：

N/A

例子：

part_type_shape(particle1, pt_shape_explosion);
part_type_size(particle1, 2, 2, 0.10, 0.01);
part_type_scale(particle1, 1, 1);
part_type_colour2(particle1, 16777215, 12632256);
part_type_alpha3(particle1, 0, 0.20, 0);
part_type_speed(particle1, 0.20, 0.20, 0, 0);
part_type_direction(particle1, 0, 359, 0, 20);
part_type_orientation(particle1, 0, 359, 0, 0, 1);
part_type_blend(particle1, 0);
part_type_life(particle1, 40, 60);

上面的代码将设置各种粒子值，包括将大小设置为每个粒子创建时大小的两倍，然后将大小每步增加0.01，同时摆动0.1。
