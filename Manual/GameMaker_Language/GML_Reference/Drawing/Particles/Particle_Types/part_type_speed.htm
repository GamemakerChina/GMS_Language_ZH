part_type_speed

此功能用于确定粒子创建时的速度，也可以用于使粒子在其整个生命周期中增加或减少速度。最小和最大速度值默认为1，这意味着粒子将每步移动1个像素，如果将它们设置为相同的值，则所有粒子的初始速度将相同，但是如果设置为不同的值，每个粒子将以给定范围内的随机速度开始。例如，如果您的最小值为3，最大值为6，则将以每步3到6个像素的速度创建所有粒子。

您还可以设置一个增量值，该增量值将使粒子在其整个生命周期中运动得更快（如果为正数）或运动得更慢（如果为负数），并在每一步中将给定的量添加到粒子速度中。该值可以是最小值（+/-）0.01。

最后，您可以设置“摆动”因子。该值将随机增加或减少到粒子寿命每一步的速度。显然，较大的值比较小的值更明显，并且该值甚至可以是负数，最大范围在-20到20之间。

句法：

part_type_speed(ind, speed_min, speed_max, speed_incr, speed_wiggle);

争论描述
ind要更改的粒子类型的索引。
speed_min粒子可以开始的最低速度。
speed_max粒子可以开始的最大速度。
speed_incr每步应增加或减小多少粒子速度。
speed_wiggle每步应从粒子速度中随机增加或减少多少。

返回值：

N/A

例子：

part_type_shape(particle2, pt_shape_flare);
part_type_size(particle2, 0.01, 0.50, 0.01, 20);
part_type_scale(particle2, 0.10, 0.10);
part_type_colour3(particle2, 4210816, 4227327, 8240895);
part_type_alpha3(particle2, 0.20, 0.80, 0);
part_type_speed(particle2, 5, 10, -0.20, 20);
part_type_direction(particle2, 0, 359, -1, 20);
part_type_gravity(particle2, 0, 270);
part_type_orientation(particle2, 0, 359, 0, 20, 1);
part_type_blend(particle2, 1);
part_type_life(particle2, 1, 20);

上面的代码将设置各种粒子值，包括将最小速度设置为5并将最大速度设置为10的速度，并且速度每步降低-0.2的值。速度还将以每步0到20之间的随机值摆动。
