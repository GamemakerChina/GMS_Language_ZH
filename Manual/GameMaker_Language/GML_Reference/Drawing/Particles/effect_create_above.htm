效果_创建_上方

通过这个函数，你可以在你的房间的所有实例之上创建一个简单的效果（它实际上是在-100000的深度创建的）。如果效果是ef_rain或ef_snow以外的东西，那么你可以定义一个x/y位置来创建效果。 效果，大小可以是0、1或2的值，其中0为小，1为中，2为大。

值得注意的是，这些效果可以通过使用函数来切换它们的绘制，以及暂停它们的绘制。 part_system_automatic_draw()和 part_system_automatic_update()粒子系统索引的适当值（其中0代表下面的效果，1代表上面的效果）。

不同粒子种类的可用常数是：。

恒定 例子 描述
雲 一种产生不同大小的随机云粒子的效果
ef_ellipse 一种创造扩大椭圆的效果
ef_explosion 一种创造不断扩大的消逝的爆炸的效果
ef_firework 一种产生多个小颗粒以产生烟花爆炸的效果
ef_flare 一个产生辉煌点的效果，耀眼的光芒和渐渐消失。
ef_rain 产生从屏幕顶部下来的雨滴的效果。
ef_ring 一种产生膨胀和消退的圆圈的效果
ef_smoke 一种产生小股烟雾的效果
ef_smokeup 一个产生烟羽的效果，在屏幕上升起。
冯小刚 一个产生多个雪粒落下屏幕的效果。
ef_spark 产生小火花的一种效果
冯小刚 一种产生星形粒子的效果

语法。

effect_create_above(kind, x, y, size, color)。

争论 描述
种类 效果的种类（使用 上面列出的常数之一）。
x 如果相关，效果的X定位。
y 如果相关，效果的y定位。
尺寸 效果的大小。
颜色 效果的颜色。

返回。

不适用

例子。

if health <= 0
{
effect_create_above(ef_explosion, x, y, 1, c_yellow);
instance_destroy();
}

上面的代码将在实例上方创建一个中等的、黄色的爆炸，如果 "健康 "变量小于或等于0，则摧毁它。
