基本体和顶点格式

图元是一种使用点定义屏幕上显示内容的图形类型。它们可以由单个点，线或三角形组成，可以着色，具有alpha混合甚至具有纹理，这意味着它们可以用于在2D游戏中创建一些非常令人兴奋的效果！例如，您可以使用多个三角形图元创建一个矩形，然后用一个标志精灵对它进行纹理处理，并在实例的draw事件中使该标志“随风摇曳”，并通过简单地修改形成点的方式使所使用的精灵变形原始的。

警告！除非您在“游戏选项”中启用了OpenGL，否则这些功能不适用于HTML5模块。

有关绘图图元的信息可以在以下页面上找到：

  • draw_primitive_begin
  • draw_primitive_begin_texture
  • draw_primitive_end
  • draw_vertex
  • draw_vertex_colour
  • draw_vertex_texture
  • draw_vertex_texture_colour

GameMaker Studio 2允许您定义自己的顶点格式，从中可以创建自己的自定义图元。这可以大大加快shader操作速度，也可以用于扩展其功能并产生令人惊讶的效果。您可以在以下各节中找到有关此信息：

  • vertex_format_begin
  • vertex_format_add_colour
  • vertex_format_add_position
  • vertex_format_add_position_3d
  • vertex_format_add_texcoord
  • vertex_format_add_normal
  • vertex_format_add_custom
  • vertex_format_end
  • vertex_format_delete

一旦创建了顶点格式（请参见以上部分），便可以使用它来使用以下顶点缓冲区函数来构建要绘制的图元：

  • vertex_create_buffer
  • vertex_create_buffer_ext
  • vertex_create_buffer_from_buffer
  • vertex_create_buffer_from_buffer_ext
  • vertex_get_buffer_size
  • vertex_get_number
  • vertex_delete_buffer
  • vertex_begin
  • vertex_colour
  • vertex_normal
  • vertex_position
  • vertex_position_3d
  • vertex_argb
  • vertex_texcoord
  • vertex_float1
  • vertex_float2
  • vertex_float3
  • vertex_float4
  • vertex_ubyte4
  • vertex_end
  • vertex_freeze
  • vertex_submit

重要的！生成的图元应遵循使用“顶点格式格式”功能设置的格式，因此，如果仅使用位置数据定义了顶点格式，则没有必要使用颜色数据来构建图元。您应注意，向正在构建的图元添加属性的顺序是由创建顶点格式时添加这些属性的顺序定义的，因此，如果您已使用顺序位置，颜色和纹理定义了矢量格式坐标，则必须将这些属性以相同的顺序添加到正在构建的图元中，否则会出现错误。
