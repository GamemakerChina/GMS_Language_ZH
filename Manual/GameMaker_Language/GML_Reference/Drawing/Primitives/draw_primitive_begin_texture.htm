draw_primitive_begin_texture

在定义带纹理的图元的顶点之前，必须先调用此函数。您必须提供要使用的原语类型（请参阅 draw_primitive_begin()（有关更多信息）和要使用的纹理的ID （可以是Sprite或背景图片资源）。资产ID可以从函数中获取 sprite_get_texture()，例如（使用-1表示没有纹理）。

注意：要重复纹理，其大小必须是2的幂，即：32x32、128x128等。

句法：

draw_primitive_begin_texture(kind, tex)

争论描述
kind您将要绘制的原始类型。
tex用于基本体的纹理。

返回值：

N/A

例子：

draw_set_colour(c_white);
var tex = sprite_get_texture(spr_Background, 0);
draw_primitive_begin_texture(pr_trianglestrip, tex);
draw_vertex_texture(0, 0, 0, 0);
draw_vertex_texture(640, 0, 1, 0);
draw_vertex_texture(0, 480, 0, 1);
draw_vertex_texture(640, 480, 1, 1);
draw_primitive_end();

上面的代码将绘制4个顶点三角带（制作一个矩形），并在“ tex”变量中保留纹理，整个纹理将用于覆盖完整的图元。

返回：基元和顶点格式
