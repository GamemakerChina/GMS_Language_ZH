绘制顶点颜色

此函数为一个基元定义了一个顶点的位置，它有自己的颜色和 alpha 设置。基元的最终外观取决于所选择的要绘制的基元类型以及向其添加顶点的顺序（详见 draw_primitive_begin() 更多信息），而具有不同颜色和阿尔法的顶点会从一个顶点平滑地融合到另一个顶点。要结束并绘制基元，您必须调用 draw_primitive_end().

语法。

draw_vertex_colour(x, y, col, alpha)

争论 描述
x 顶点的X坐标。
y 顶点的Y坐标。
殖民地 绘制此顶点的颜色。
阿尔法 绘制此顶点的α值（0-1）。

返回。

不适用

例子。

draw_primitive_begin(pr_trianglelist);
draw_vertex_colour(100, 100, c_blue, 0.1);
draw_vertex_colour(100, 200, c_red, 0.1);
draw_vertex_colour(150, 150, c_green, 1);
draw_primitive_end()。

上面的代码将画出一个半透明的三角形，每个顶点的颜色都不同。
