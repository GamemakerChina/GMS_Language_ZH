draw_vertex_colour

此函数使用其自己的颜色和Alpha设置来定义图元的顶点位置。基本体的最终外观将取决于选择绘制的基本体类型以及向其添加顶点的顺序（请参见 draw_primitive_begin()有关更多信息），则具有不同颜色和Alpha的顶点将平滑地从一个过渡到另一个。要结束并绘制图元，您必须调用 draw_primitive_end()。

句法：

draw_vertex_colour(x, y, col, alpha)

争论描述
x顶点的x坐标。
y顶点的y坐标。
col绘制此顶点所用的颜色。
alpha用（0-1）绘制此顶点的Alpha。

返回值：

N/A

例子：

draw_primitive_begin(pr_trianglelist);
draw_vertex_colour(100, 100, c_blue, 0.1);
draw_vertex_colour(100, 200, c_red, 0.1);
draw_vertex_colour(150, 150, c_green, 1);
draw_primitive_end();

上面的代码将绘制一个半透明的三角形，每个顶点用不同的颜色上色。

返回：基元和顶点格式
