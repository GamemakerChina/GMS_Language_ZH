draw_vertex_texture

此函数定义图元的纹理顶点的位置。基本体的最终外观将取决于选择绘制的基本体类型（请参见 draw_primitive_begin()有关更多信息），请向其中添加顶点的顺序以及为纹理设置的起点的位置。要结束并绘制图元，您必须调用 draw_primitive_end()。

您还可以选择通过设置适当的xtex / ytex位置（在此坐标上也称为UV点）来设置在基元表面上仅绘制纹理的一部分。该点由0到1之间的值定义，要使用的图像的左上角为（0,0）点，右下角为（1,1）点。这样，您可以仅使用源图像资产的一部分来构造图元。

绘制带纹理的图元时，该纹理将与设置的颜色混合 draw_set_colour。这意味着要绘制没有混合的纹理，应将其设置为 c_white，如下例所示。

注意：要重复纹理，其大小必须是2的幂，即：32x32、128x128等。

句法：

draw_vertex_texture(x, y, xtex, ytex)

争论描述
X顶点的x坐标。
ÿ顶点的y坐标。
特克斯要使用的纹理样本的宽度。
tex要使用的纹理样本的高度。

返回值：

N/A

例子：

draw_set_colour(c_white);
var tex = sprite_get_texture(spr_Background, 0);
draw_primitive_begin_texture(pr_trianglestrip, tex);
draw_vertex_texture(0, 0, 0, 0);
draw_vertex_texture(640, 0, 1, 0);
draw_vertex_texture(0, 480, 0, 1);
draw_vertex_texture(640, 480, 1, 1);
draw_primitive_end();

上面的代码将绘制4个顶点三角带（制作一个矩形），并在“ tex”变量中保留纹理，整个纹理将用于覆盖完整的图元。

返回：基元和顶点格式
