绘制顶点纹理的颜色

此函数定义了一个基元的纹理顶点的位置。基元的最终外观取决于所选择的要绘制的基元类型（参见 draw_primitive_begin()获取更多信息）。) 向其添加顶点的顺序、为纹理样本提供的开始和结束点的位置以及所设置的颜色和阿尔法值。为了保持纹理的外观而只改变阿尔法值，一个数值为 值为-1（或c_white），可用于颜色参数。要结束并绘制基元，您必须调用 draw_primitive_end().

注意：对于一个纹理来说，它的大小必须是2的幂，即：32x32，128x128，等等。

语法。

draw_vertex_texture_colour(x, y, xtex, ytex, col, alpha)

争论 描述
x 顶点的X坐标。
y 顶点的Y坐标。
xtex 纹理中的X坐标。
ytex 纹理中的y坐标。
殖民地 在此顶点与纹理混合的颜色（-1或c_white表示不混合）。
阿尔法 绘制此顶点的α值（0-1）。

返回。

不适用

例子。

draw_set_colour(c_white);
var tex = sprite_get_texture(spr_Background, 0);
draw_primitive_begin_texture(pr_trianglestrip, tex);
draw_vertex_texture_colour(0, 0, 0, c_fuchsia, 1);
draw_vertex_texture_colour(640, 0, 1, 0, c_yellow, 1);
draw_vertex_texture_colour(0, 480, 0, 1, c_aqua, 1);
draw_vertex_texture_colour(640, 480, 1, 1, c_lime, 1);
draw_primitive_end() 。

上面的代码将绘制一个4个顶点的三角形条（构成一个矩形），用 "tex "变量中持有的纹理进行纹理处理，整个纹理将被用来覆盖已完成的基元，并将其与四种不同的颜色混合。

返回。基元和顶点格式
