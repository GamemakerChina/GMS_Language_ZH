draw_vertex_texture_colour

此函数定义图元的纹理顶点的位置。基本体的最终外观将取决于选择绘制的基本体类型（请参见 draw_primitive_begin()有关更多信息），请向其中添加顶点的顺序，为纹理样本指定的起点和终点的位置以及已设置的颜色和Alpha值。要在仅更改Alpha（值-1（或 c_white）可用于color参数。要结束并绘制图元，您必须调用 draw_primitive_end()。

注意：要重复纹理，其大小必须是2的幂，即：32x32、128x128等。

句法：

draw_vertex_texture_colour(x, y, xtex, ytex, col, alpha)

争论描述
x顶点的x坐标。
y顶点的y坐标。
xtex纹理内的x坐标。
ytex纹理内的y坐标。
col在此顶点与纹理融合的颜色（-1或 c_white（不混合）。
alpha用（0-1）绘制此顶点的Alpha。

返回值：

N/A

例子：

draw_set_colour(c_white);
var tex = sprite_get_texture(spr_Background, 0);
draw_primitive_begin_texture(pr_trianglestrip, tex);
draw_vertex_texture_colour(0, 0, 0, 0, c_fuchsia, 1);
draw_vertex_texture_colour(640, 0, 1, 0, c_yellow, 1);
draw_vertex_texture_colour(0, 480, 0, 1, c_aqua, 1);
draw_vertex_texture_colour(640, 480, 1, 1, c_lime, 1);
draw_primitive_end();

上面的代码将绘制4个顶点三角形带（制作一个矩形），并在“ tex”变量中保留纹理，整个纹理将用于覆盖完整的图元，并将其与四种不同的颜色混合。

返回：基元和顶点格式
