vertex_argb

此函数将为当前为自定义图元定义的顶点设置ARGB值。您提供缓冲区以将数据以及希望用作单个32位无符号整数的红色，绿色，蓝色和alpha值写入其中-最高8位的alpha采样，然后是红色采样，绿色采样，最后是最低8位的蓝色采样。您可以使用十六进制表示法（ $AA RRGGBB）实数或使用任何 make_colour_*()函数定义颜色值。

句法：

vertex_argb(buffer, argb);

争论描述
buffer将信息写入的缓冲区。
a颜色的Alpha值（0-255）。
r颜色的红色组成部分（0-255）。
g颜色的绿色组成部分（0-255）。
b颜色的蓝色组成部分（0-255）。

返回值：

N/A

例子：

vertex_argb(buff, $FFFFFFFF);

上面的代码会将定义的顶点的ARGB值设置为白色。

返回：基元和顶点格式
