vertex_delete_buffer

使用此功能，您可以开始定义自定义原语。您可以分配一个缓冲区来写入图元，并指定要使用的顶点格式（先前使用顶点格式功能定义）。然后，您将在调用之前为图元的每个顶点定义必要的点 vertex_end()完成原始创作。

句法：

vertex_begin(buffer, format);

争论描述
buffer要写入的缓冲区。
format要使用的顶点格式。

返回值：

N/A

例子：

vertex_format_begin();
vertex_format_add_position();
vertex_format_add_colour();
vertex_format_add_textcoord();
v_format = vertex_format_end();
v_buff = vertex_create_buffer();
vertex_begin(v_buff, v_format);

上面的代码将定义新的顶点格式，创建新的缓冲区并开始新图元的定义过程。

返回：基元和顶点格式
