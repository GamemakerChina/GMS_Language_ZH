vertex_colour

如果定义的顶点格式采用颜色值，则可以使用此函数将数据添加到为当前图元定义的顶点中。该函数需要一个缓冲区来存储数据，并将采用一个颜色常量或一个十六进制值（使用GameMaker Studio 2格式，例如： $FF0000表示蓝色）以及一个从0开始的alpha值（透明）到1（完全不透明）。

句法：

vertex_colour(buffer, colour, alpha);

争论描述
buffer将信息写入的缓冲区。
colour该顶点的颜色（可以是常数或十六进制值）。
alpha顶点的Alpha值（从0到1）。

返回值：

N/A

例子：

vertex_colour(b, c_white, 1);

上面的代码将定义的顶点设置为白色，alpha值为1。

返回：基元和顶点格式
