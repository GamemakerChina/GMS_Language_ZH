顶点颜色

如果您定义的顶点格式需要一个颜色值，则可使用此函数将该数据添加到正在为当前基元定义的顶点。该函数需要一个缓冲区来存储数据，并且会接受一个颜色常数。 或一个十六进制值（使用 BGR 的标准GameMaker Studio 2格式，例如：$FF0000 表示蓝色），以及一个从 0（透明）到 1（完全不透明）的 alpha 值。

语法。

vertex_colour(buffer, color, alpha)。

争论 描述
缓冲区 要写入信息的缓冲区。
颜色 这个顶点的颜色（可以是一个常数或一个十六进制值）。
阿尔法 顶点的alpha值（从0到1）。

返回。

不适用

例子。

vertex_colour(b, c_white, 1);

上面的代码将把被定义的顶点设置为白色，alpha值为1。
