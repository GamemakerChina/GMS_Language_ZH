顶点创建缓冲区（vertex_create_buffer

通过这个函数，你可以创建一个新的顶点缓冲区。这是由GameMaker Studio 2创建的一个特殊的增长缓冲区，它在构建基元时被预先格式化（例如，用于着色器）。该函数将返回一个 "指针"（索引），然后在所有对它的进一步调用中使用。

当使用用此函数创建的顶点缓冲区时，你只需调用 vertex_begin()来开始向其分配顶点数据，以开始定义你的自定义基元，然后将其保存在缓冲器中，准备提交给 提交给着色器。缓冲区在必要时可以被重复使用（除非你使用了 vertex_freeze()函数），每次调用 vertex_begin()都会擦除之前的 缓冲区数据准备好接受新的数据。

语法。

vertex_create_buffer()。

返回。

指针

例子。

v_buff = vertex_create_buffer（）。

上面的代码将创建一个新的顶点缓冲区，并将其句柄存储在变量 "v_buff "中。

返回。基元和顶点格式
