vertex_create_buffer

使用此功能，您可以创建一个新的顶点缓冲区。这是GameMaker Studio 2创建的特殊增长缓冲区，该缓冲区已预先格式化，可在构建基元时使用（例如， shaders该函数将为缓冲区返回一个“指针”（索引），然后在以后的所有调用中都应使用该指针。

使用此功能创建的顶点缓冲区时，只需调用 vertex_begin()开始向其分配顶点数据以开始定义您的自定义图元，然后将其保存在缓冲区中以准备提交给shader 。必要时可以重新使用缓冲区（除非您已使用 vertex_freeze()函数），每次调用 vertex_begin()擦除先前的缓冲区数据准备接受新数据。

句法：

vertex_create_buffer();

返回值：

Pointer

例子：

v_buff = vertex_create_buffer();

上面的代码将创建一个新的新顶点缓冲区，并将其句柄存储在变量“ v_buff”中。

返回：基元和顶点格式
