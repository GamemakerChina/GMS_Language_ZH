顶点_创建缓冲区_ext

和函数 vertex_create_buffer()一样，这个函数将创建一个新的顶点缓冲区。这是由GameMaker Studio 2创建的一个特殊的增长缓冲区，它是预先格式化的，以便在构建基元时用于着色器。 构建用于着色器的基元时使用。你可以为缓冲区指定一个初始的起始大小（字节），它将返回一个缓冲区的 "句柄"（索引），然后在所有对缓冲区的进一步调用中使用。

语法。

vertex_create_buffer_ext(size)。

争论 描述
尺寸 缓冲区的初始大小（以字节为单位）。

返回。

指针

例子。

v_buff = vertex_create_buffer_ext（1024 * 1024）。

上述代码将创建一个新的顶点缓冲区，最初大小为1MB，并在变量 "v_buff "中存储其句柄。

返回。基元和顶点格式
