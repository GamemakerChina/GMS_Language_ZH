vertex_create_buffer_ext

和功能一样 vertex_create_buffer()，此函数将创建一个新的顶点缓冲区。 GameMaker Studio 2创建的一个特殊的增长缓冲区，该缓冲区已预先格式化，可在构建与shaders一起使用的图元时使用。您可以为缓冲区指定一个初始起始大小（以字节为单位），它将为缓冲区返回一个“句柄”（索引），然后将在以后的所有缓冲区调用中使用该句柄。

句法：

vertex_create_buffer_ext(size);

争论描述
size缓冲区的初始大小（以字节为单位）。

返回值：

Pointer

例子：

v_buff = vertex_create_buffer_ext(1024 * 1024);

上面的代码将创建一个新的顶点缓冲区，最初的大小为1MB，并将其句柄存储在变量“ v_buff”中。

返回：基元和顶点格式
