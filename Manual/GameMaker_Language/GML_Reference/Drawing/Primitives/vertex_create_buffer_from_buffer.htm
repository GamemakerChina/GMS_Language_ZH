顶点创建缓冲区的方法

和函数 vertex_create_buffer()一样，这个函数将创建一个新的顶点缓冲区，只是现在它存储的顶点数据是从被指定为源的常规缓冲区中复制的。创建的缓冲区 是一个特殊的增长缓冲区，它被预先格式化为用于构建基元的顶点格式，以用于（例如）着色器。这个函数要求你提供一个指向先前创建的常规缓冲区的指针，以及应该应用于复制的顶点格式的顶点 格式，以便应用于复制的数据。

语法。

vertex_create_buffer_from_buffer(buffer, format)。

争论 描述
缓冲区 用于创建顶点缓冲区的缓冲区。
格式 要使用的原始顶点格式。

返回。

指针

例子。

vertex_format_begin();
vertex_format_add_position_3d();
vertex_format_add_colour();
vertex_format_add_textcoord();
var my_format = vertex_format_end();
v_buff = vertex_create_buffer_from_buffer(global.modelBuff, myFormat);

上面的代码将创建一个新的顶点格式，然后从先前创建的常规缓冲区创建一个新的顶点缓冲区，将自定义的顶点格式应用到它。

返回。基元和顶点格式
