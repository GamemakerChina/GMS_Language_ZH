vertex_create_buffer_from_buffer

和功能一样 vertex_create_buffer()，此函数将创建一个新的顶点缓冲区，仅现在从指定为源的常规缓冲区中复制它存储的顶点数据。创建的缓冲区是一种特殊的增长缓冲区，该缓冲区预先以顶点格式进行了格式化，以构建与（例如） shaders一起使用的图元。此功能要求您提供指向先前创建的常规缓冲区的指针，以及应应用于复制数据的顶点格式。

句法：

vertex_create_buffer_from_buffer(buffer, format);

争论描述
buffer从中创建顶点缓冲区的缓冲区。
format要使用的原始顶点格式。

返回值：

Pointer

例子：

vertex_format_begin();
vertex_format_add_position_3d();
vertex_format_add_colour();
vertex_format_add_textcoord();
var my_format = vertex_format_end();
v_buff = vertex_create_buffer_from_buffer(global.modelBuff, myFormat);

上面的代码将创建一个新的顶点格式，然后从以前创建的常规缓冲区中创建一个新的顶点缓冲区，并对其应用自定义顶点格式。

返回：基元和顶点格式
