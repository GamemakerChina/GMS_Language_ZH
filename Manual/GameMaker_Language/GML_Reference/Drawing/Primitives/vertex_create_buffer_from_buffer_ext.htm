vertex_create_buffer_from_buffer_ext

和功能一样 vertex_create_buffer()，此函数将创建一个新的顶点缓冲区，仅现在从指定为源的常规缓冲区中复制它存储的顶点数据。缓冲区已使用顶点格式进行了预格式化，以构建供（例如） shaders使用的图元，并且您还可以在源缓冲区内提供要从中进行复制的偏移量以及最终缓冲区应具有的顶点数量。请注意，如果顶点数量与要复制的顶点数量不匹配，则可能会损坏顶点数据。

句法：

vertex_create_buffer_from_buffer_ext(buffer, format, src_offset, vert_num);

争论描述
buffer从中创建顶点缓冲区的缓冲区。
format要使用的原始顶点格式。
src_offset源缓冲区中要复制的偏移量。
vert_num缓冲区应具有的顶点数。

返回值：

Pointer

例子：

vertex_format_begin();
vertex_format_add_position_3d();
vertex_format_add_colour();
vertex_format_add_textcoord();
var my_format = vertex_format_end();
v_buff = vertex_create_buffer_from_buffer(global.modelBuff, myFormat, 0, 512);

上面的代码将创建一个新的顶点格式，然后从以前创建的常规缓冲区中创建一个新的顶点缓冲区，并将自定义顶点格式应用于偏移量为0的位置。该函数告诉新的顶点缓冲区应该有512个顶点。

返回：基元和顶点格式
