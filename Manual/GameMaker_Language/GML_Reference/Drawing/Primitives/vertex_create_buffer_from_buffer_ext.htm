顶点_从缓冲区创建缓冲区_ext

和函数 vertex_create_buffer()一样，这个函数将创建一个新的顶点缓冲区，只是现在它存储的顶点数据是从被指定为源的常规缓冲区中复制的。该缓冲区 被预先格式化，用于构建基元以用于（例如）着色器，并且你还可以提供源缓冲区内的偏移量，以便从源缓冲区复制，以及最终缓冲区应该有的顶点数量。注意，如果 顶点数量与被复制的顶点数量不一致，你可能会得到损坏的顶点数据。

语法。

vertex_create_buffer_from_buffer_ext(buffer, format, src_offset, vert_num)。

争论 描述
缓冲区 用于创建顶点缓冲区的缓冲区。
格式 要使用的原始顶点格式。
src_offset 从源缓冲区中复制的偏移量。
vert_num 缓冲区应该有多少个顶点。

返回。

指针

例子。

vertex_format_begin();
vertex_format_add_position_3d();
vertex_format_add_colour();
vertex_format_add_textcoord();
var my_format = vertex_format_end();
v_buff = vertex_create_buffer_from_buffer( global.modelBuff, myFormat, 0, 512);

上面的代码将创建一个新的顶点格式，然后从先前创建的常规缓冲区创建一个新的顶点缓冲区，将自定义的顶点格式应用于它，偏移量为0。该函数告诉新的顶点缓冲区，它应该期待512个顶点。

返回。基元和顶点格式
