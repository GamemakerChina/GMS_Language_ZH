vertex_format_add_texcoord

告诉GameMaker Studio 2接受纹理位置数据（u和v）作为正在创建的新顶点格式的一部分。

句法：

vertex_format_add_textcoord();

返回值：

N/A

例子：

vertex_format_begin();
vertex_format_add_textcoord();
vertex_format_add_position_3d();
my_format = vertex_format_end();

上面的代码将创建仅具有纹理和位置值的新顶点格式，然后将格式ID存储在变量“ my_format”中。

返回：基元和顶点格式
