vertex_format_begin

在定义新的顶点格式之前，您必须使用此功能GameMaker Studio 2您必须首先调用它，然后使用适当的函数定义格式值，最后调用 vertex_format_end()完成定义并返回新格式“ handle”。

句法：

vertex_format_begin();

返回值：

N/A

例子：

vertex_format_begin();
vertex_format_add_colour();
vertex_format_add_position();
my_format = vertex_format_end();

上面的代码将创建仅具有颜色和位置值的新顶点格式，然后将格式ID存储在变量“ my_format”中。

返回：基元和顶点格式
