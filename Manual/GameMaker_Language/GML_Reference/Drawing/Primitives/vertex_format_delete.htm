vertex_format_delete

使用完所有创建的矢量格式后，都必须调用此函数。您提供格式ID值（由函数返回）vector_format_end() ），此功能将释放与其关联的内存。请注意，如果您在调用此函数后尝试再次使用此格式，则会收到错误消息。

句法：

vertex_format_delete(formatID);

返回值：

N/A

例子：

vertex_format_delete(myFormat);

上面的代码将从内存中删除在变量“ myFormat”中创建的顶点格式。

返回：基元和顶点格式
