vertex_format_end

在定义任何新的顶点格式之后，必须调用此函数。它返回新格式“ handle”（索引），该格式必须在所有其他引用此新格式的顶点函数中使用。

句法：

vertex_format_end();

返回值：

Real

例子：

vertex_format_begin();
vertex_format_add_colour();
vertex_format_add_position();
my_format = vertex_format_end();

上面的代码将创建仅具有颜色和位置值的新顶点格式，然后将格式ID存储在变量“ my_format”中。

返回：基元和顶点格式
