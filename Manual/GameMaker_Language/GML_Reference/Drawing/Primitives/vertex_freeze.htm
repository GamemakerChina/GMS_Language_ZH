顶点冻结（vertex_freeze

这个函数可以用来 "冻结 "一个顶点缓冲区。这个缓冲区变成只读，这意味着如果你需要改变它，你必须删除整个缓冲区并重新创建它。冻结的缓冲区可以被提交给着色器 比正常的动态缓冲区更快，推荐用于那些需要在关卡或游戏期间保持不变的自定义基元的效果。

语法。

vertex_freeze(buffer)。

争论 描述
缓冲区 要冻结的缓冲区。

返回。

不适用

例子。

vertex_freeze(buff)。

上述代码将冻结给定的顶点缓冲区。
