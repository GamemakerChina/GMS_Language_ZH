顶点_get_buffer_size

通过这个函数你可以得到给定的顶点缓冲区的大小，单位是字节。

语法。

vertex_get_buffer_size(buffer)。

争论 描述
缓冲区 要获得的缓冲区的大小。

返回。

真实的

例子。

bufferBytes = vertex_get_buffer_size（buff）。

上面的代码将获得给定的顶点缓冲器所使用的字节数，并将该值存储在变量 "bufferBytes "中。
