顶点_正常

此函数将向当前正在为自定义基元定义的顶点添加表面法线数据。您可以提供要写入数据的缓冲区，以及法线的x、y和z分量部分。

语法。

vertex_normal(buffer, nx, ny, nz)。

争论 描述
缓冲区 要写入信息的缓冲区。
nx 表面法线的X分量。
ny 表面法线的Y分量。
nz 表面法线的Z分量。

返回。

不适用

例子。

vertex_normal(buff, 0, 1, 1);

上面的代码将设置被定义的顶点的表面法线。
