vertex_normal

此功能会将表面法线数据添加到当前为自定义图元定义的顶点。您提供了缓冲区以将数据写入法线的x，y和z组成部分。

句法：

vertex_normal(buffer, nx, ny, nz);

争论描述
buffer将信息写入的缓冲区。
nx表面法线的x分量。
ny表面法线的y分量。
nz表面法线的z分量。

返回值：

N/A

例子：

vertex_normal(buff, 0, 1, 1);

上面的代码将设置要定义的顶点的表面法线。

返回：基元和顶点格式
