vertex_texcoord

此函数将设置纹理坐标以用于当前为自定义图元定义的顶点。您提供缓冲区以将数据写入纹理以及要使用的纹理中的UV位置。

句法：

vertex_texcoord(buffer, u, v);

争论描述
buffer将信息写入的缓冲区。
u要使用的第一个纹理坐标（0-1）。
v要使用的第二个纹理坐标（0-1）。

返回值：

N/A

例子：

vertex_texcoord(buff, 0, 0);

上面的代码将设置要定义的顶点的UV值。

返回：基元和顶点格式
