顶点坐标(vertex_texcoord)

此函数将为当前正在为自定义基元定义的顶点设置要使用的纹理坐标。您可以提供要写入数据的缓冲区，以及要使用的纹理中的 UV 位置。

语法。

vertex_texcoord(buffer, u, v)。

争论 描述
缓冲区 要写入信息的缓冲区。
u 要使用的第一个纹理坐标（0 - 1）。
v 要使用的第二个纹理坐标（0-1）。

返回。

不适用

例子。

vertex_texcoord(buff, 0, 0)。

上述代码将设置被定义的顶点的UV值。

返回。基元和顶点格式
