vertex_ubyte4

这个函数将向顶点数据添加四个无符号字节值（0 - 255）。顶点必须已经被正确地格式化，以接受这个使用 vertex_format_add_custom()函数来接受。

语法。

vertex_ubyte4(buffer, byte, byte, byte, byte)。

争论 描述
缓冲区 要写入信息的缓冲区。
字节 第一个输入值。
字节 第二个输入值。
字节 第三个输入值。
字节 第四个输入值。

返回。

不适用

例子。

vertex_ubyte4(buff, irandom(255), irandom(255), irandom(255), 127)。

上面的代码将为正在定义的顶点数据添加四个值。

返回。基元和顶点格式
