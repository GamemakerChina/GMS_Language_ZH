draw_enable_swf_aa

通过这个功能，你可以为SWF格式的矢量精灵启用或禁用抗锯齿（AA）。AA只是对矢量图像的边缘进行平滑处理，使其看起来更漂亮。使用的AA量将取决于使用该函数设置的值 draw_set_swf_aa_level(). 默认情况下这是禁用的。

语法。

draw_enable_swf_aa(启用)。

争论 描述
启用 对所有SWF精灵启用（真）或禁用（假）AA。

返回。

不适用

例子。

如果 draw_get_swf_aa_level() == 0
{
draw_enable_swf_aa(true);
draw_set_swf_aa_level(0.5);
}

上述代码将检查SWF格式精灵的AA值，如果它是0，它将启用AA，并将值设置为0.5。

返回。精灵和瓷砖
