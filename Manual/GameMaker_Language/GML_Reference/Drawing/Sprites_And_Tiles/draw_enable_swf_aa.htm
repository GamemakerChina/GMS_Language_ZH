draw_enable_swf_aa

使用此功能，可以启用或禁用SWF格式矢量精灵的抗锯齿（AA）。 AA只是平滑了矢量图像的边缘，使它们看起来更好。使用的AA量取决于使用功能设置的值 draw_set_swf_aa_level()。默认情况下是禁用的。

句法：

draw_enable_swf_aa(enable);

争论描述
enable使能够 （true ）或禁用（false ）所有SWF精灵的AA。

返回值：

N/A

例子：

if draw_get_swf_aa_level() == 0
    {
    draw_enable_swf_aa(true);
    draw_set_swf_aa_level(0.5);
    }

上面的代码将检查SWF格式精灵的AA值，如果为0，则启用AA并将其设置为0.5。
