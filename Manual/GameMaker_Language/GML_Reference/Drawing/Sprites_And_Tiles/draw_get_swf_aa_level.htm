draw_get_swf_aa_level

此功能可用于获取SWF格式矢量子画面的抗锯齿（AA）级别。返回值将在0到1之间，并显示将如何“平滑”绘制这些精灵的边缘。您可以使用功能设置AA等级 draw_set_swf_aa_level()。

句法：

draw_get_swf_aa_level();

返回值：

Real

例子：

if draw_get_swf_aa_level() == 0
    {
    draw_enable_swf_aa(true);
    draw_set_swf_aa_level(0.5);
    }

上面的代码将检查SWF格式精灵的AA值，如果为0，则启用AA并将其设置为0.5。
