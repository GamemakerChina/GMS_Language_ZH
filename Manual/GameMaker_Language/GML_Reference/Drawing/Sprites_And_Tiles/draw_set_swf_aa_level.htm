draw_set_swf_aa_level

此功能可用于设置SWF格式矢量精灵的抗锯齿（AA）级别。这可以是从0到1的实际值，并且可以“平滑”这些精灵的边缘。请注意，要查看此效果，您必须首先使用该功能启用AA draw_enable_swf_aa() 。

句法：

draw_set_swf_aa_level(AA);

争论描述
AA使用0到1的抗锯齿值。

返回值：

N/A

例子：

if draw_get_swf_aa_level() == 0
    {
    draw_enable_swf_aa(true);
    draw_set_swf_aa_level(0.5);
    }

上面的代码将检查SWF格式精灵的AA值，如果为0，则启用AA并将其设置为0.5。
