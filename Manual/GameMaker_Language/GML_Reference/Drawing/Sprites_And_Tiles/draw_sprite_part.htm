draw_sprite_part

使用此功能，您可以在房间内的给定位置绘制任何精灵的一部分。与 draw_sprite()您可以指定一个精灵和一个用于绘制的子图像，然后必须在要选择绘制的区域的精灵中指定相对坐标。这意味着左位置0和顶部位置0将是精灵的左上角，并且所有其他坐标都应从该位置获取。下图显示了如何工作的示例：

您应该注意，如果纹理页面允许自动裁剪，则此功能可能无法按预期方式工作，因为将删除小精灵周围多余的“空”空间以创建纹理页面。要解决此问题，您将需要设置纹理页面设置（在“纹理组编辑器”中）以禁用选项“自动裁剪” 。

注意：使用此功能进行绘制时，将忽略子图形x偏移和y偏移，并且将在房间中指定的x / y位置的左上角绘制子图形部件。

注意：此功能仅对位图精灵有用，不适用于SWF或JSON（Spine）精灵。

句法：

draw_sprite_part(sprite, subimg, left, top, width, height, x, y);

争论描述
sprite要绘制的精灵的索引。
subimg要绘制的精灵的子图像（帧）（image_index或-1与对象中的当前动画帧相关）。
left要绘制的区域左上角的子画面上的x位置。
top要绘制的区域左上角的sprite上的y位置。
width要绘制的区域的宽度。
height要绘制的区域的高度。
x绘制精灵的位置的x坐标。
y绘制子画面的y坐标。

返回值：

N/A

例子：

draw_sprite_part(sprite_index, image_index, 4, 0, sprite_width-16, sprite_height-16, x, y );

这将绘制实例指定的精灵（sprite_index）及其当前的动画帧（image_index），但是它将在两侧的宽度上减少4px的空白，从原始24x24像素的底部开始的高度减少8像素的空白雪碧。

背面：精灵和瓷砖
