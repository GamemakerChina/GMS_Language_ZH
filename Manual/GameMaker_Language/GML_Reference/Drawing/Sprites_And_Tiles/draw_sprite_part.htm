画笔部分

通过这个函数，你可以在房间里的一个给定的位置上绘制任何精灵的一部分。与 draw_sprite()一样，你可以指定一个精灵和一个要绘制的子图像，然后你必须给出要选择绘制的区域在精灵中的相对坐标。这意味着，左边的位置为0，上面的位置为0，就是精灵的左上角，所有进一步的坐标都应该从这个位置开始。下面的图片显示了一个例子 的例子。

你应该注意，如果纹理页允许自动裁剪，那么这个函数可能不会像预期那样工作。 因为在创建纹理页时，精灵周围的额外 "空白 "空间将被移除。要解决这个问题，您需要在纹理页设置中（在纹理组编辑器中）设定 禁用自动裁剪选项。

注意：用这个函数绘图时，精灵的x偏移和y偏移被忽略，精灵部分将以左上角在房间中指定的x/y位置绘制。

注意：这个函数只对位图精灵有用，对SWF或JSON（Spine）精灵不起作用。

语法。

draw_sprite_part(sprite, subimg, left, top, width, height, x, y)。

争论 描述
精灵 要绘制的精灵的索引。
亚img 要绘制的精灵的subimg（帧）（image_index或-1与对象中的当前动画帧相关）。
左边 要绘制的区域的左上角在精灵上的X位置。
顶部 要绘制的区域的左上角在精灵上的y位置。
宽度 要绘制的区域的宽度。
高度 要绘制的区域的高度。
x 绘制精灵的X坐标。
y 绘制精灵位置的y坐标。

返回。

不适用

例子。

draw_sprite_part(sprite_index, image_index, 4, 0, sprite_width-16, sprite_height-16, x, y ) 。

这将绘制实例指定的精灵（sprite_index）及其当前的动画帧（image_index），但是它将在两边的宽度上减去4像素的边距，在原24x24像素的精灵底部减去8像素的高度。 像素的精灵的底部减去8像素。
