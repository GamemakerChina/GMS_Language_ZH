draw_sprite_pos

使用此功能，您可以在四个角坐标定义的区域上绘制变形的精灵。前两个参数是要绘制的精灵和该精灵的子图像（与函数中的相同）draw_sprite() ），但接下来的是定义给定Sprite边界框的四个角中的每个角的位置的那些角。这些应按顺时针顺序给出，因此第一个坐标是左上角，然后是右上角，然后是右下角，最后是左下角。您还可以为精灵的Alpha设置一个值，以使其具有透明性。下图说明了此功能的工作原理：

警告！上面的图像仅用于说明目的，如果在sprite上使用此功能，您将获得不同的结果，并且由于sprite由四元图元构造的方式而可能会遇到纹理“剪切”。

注意：此功能仅对位图精灵有用，不适用于SWF或JSON（Spine）精灵。

注意：使用此功能时，应禁用纹理页面的“自动裁剪”设置。

句法：

draw_sprite_pos(sprite, subimg, x1, y1, x2, y2, x3, y3, x4, y4, alpha);

争论描述
sprite要绘制的精灵的索引。
subimg要绘制的精灵的子图像（帧）（image_index或-1与对象中的当前动画帧相关）。
x1第一个x坐标。
y1第一个y坐标。
x2第二个x坐标。
y2第二个y坐标。
x3第三个x坐标。
y3第三个y坐标。
x4第四个x坐标。
y4第四个y坐标。
alpha子画面的Alpha（从0到1，其中0是透明的，而1是不透明的）。

返回值：

N/A

例子：

draw_sprite_pos(sprite_index, image_index, x - 100, y - 50, x - 50, y +150, x + 100, y + 200, x + 100, y, 1);

上面的代码将绘制与实例关联的精灵，该实例运行的代码在实例的x / y位置周围变形并且具有完全不透明的alpha。
