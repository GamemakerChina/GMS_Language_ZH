draw_sprite_stretched

这个函数只是简单地获取一个精灵资源，并在给定的宽度和高度上拉伸它，使其占据该区域。与 draw_sprite()您可以指定一个精灵和一个子图像进行绘制，然后指定要在其上绘制精灵的空间中的x / y位置，最后指定宽度和高度（必须为像素值）。下图说明了此功能的工作原理：

注意：使用此功能进行绘制时，将忽略精灵的x偏移和y偏移，并且将以房间的指定x / y位置的左上角绘制精灵。

句法：

draw_sprite_stretched(sprite, subimg, x, y, w, h);

争论描述
sprite要绘制的精灵的索引。
subimg要绘制的精灵的子图像（帧）（image_index或-1与对象中的当前动画帧相关）。
x绘制精灵的位置的x坐标。
y绘制子画面的y坐标。
w拉伸的精灵将占据的区域的宽度。
h拉伸的精灵将占据的区域的高度。

返回值：

N/A

例子：

draw_sprite_stretched(sprite_index, image_index, x, y, sprite_width / 2, sprite_height / 2);

这将在实例x / y位置的左角绘制实例分配的精灵及其子图像。其宽度设置为与精灵相同，高度为精灵高度除以2。

背面：精灵和瓷砖
