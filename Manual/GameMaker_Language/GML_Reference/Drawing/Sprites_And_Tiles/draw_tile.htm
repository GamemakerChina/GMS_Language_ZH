draw_tile

使用此功能，您可以从图块集中绘制任何给定的图块-与旋转，翻转和反射镜竞争-设置框架（如果设置了动画）和房间内的位置。瓦片集ID值是在IDE中设置的瓦片集索引，瓦片数据是数据“ blob”，其中包含有关瓦片索引和已应用的转换的所有信息。您可以使用该功能 tilemap_get()以获得可以使用Tile函数处理的tile数据，或者您可以使用以下tile常量以及要创建的tile的索引值自己构造它：

持续的描述
tile_rotate用于设置/获取tile数据blob的旋转位。
tile_mirror用于设置/获取瓦片数据块的镜像位。
tile_flip用于设置/获取瓦片数据块的翻转位。
tile_index_mask一个特殊常量，用于与图块数据blob进行“与”运算以提取图块索引。


您可以使用图块索引值按位“或”这些常量，以获得最终的图块数据：

var tiledata = 8 | tile_rotate | tile_flip;

上面的代码只是创建一个局部变量来存储图块集上索引为8的图块的图块数据，并对其进行旋转和翻转。然后可以使用它从您指定的图块集以该索引处绘制图块。此图块未链接到任何tilemap图层，并且将在执行绘图的实例的图层深度处（在您在房间中指定的x / y位置处）绘制图块。请注意，您也可以将框架设置为绘制。如果索引的图块是动画集的一部分，则可以在此处设置帧号（从0到帧数-1），索引的图块将简单地指示您正在使用哪个动画图块集，同时是绘制的给定框架。如果图块不属于动画集，则应将frame设置为0。

句法：

draw_tile(tile set, tiledata, frame, x, y)

争论描述
tile set设置要使用的图块的ID
tiledata要使用的图块数据
frame动画磁贴要使用的帧号（非动画磁贴默认为0）
x在房间内绘制的x位置
y房间内的y位置

返回值：

Real

例子：

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var data = tilemap_get(map_id, 0, 0);
tile_set_flip(data, true);
draw_tile(set_Wall, data, 0, x, y);

上面的代码将从给定的tilemap中获取一个tile上的数据，然后设置“ flip”标志，并使用此数据绘制在实例位置翻转的tile。
