draw_tilemap

此功能可用于在房间的任何地方绘制给定的图块。您提供tilemap元素ID（使用以下代码创建tilemap元素时会获得此ID layer_tilemap_create()或使用该功能时 layer_tilemap_get_id()），然后给出绘图的x / y位置（在房间中）。请注意，这将仅使用调用该函数的实例的层深度在指定点绘制tilemap。它不动tilemap的-也无法改变它以任何方式-不，如果tilemap的被标记为可见或不可见其重要。

句法：

draw_tilemap(tilemap_element_id, x, y);

争论描述
tilemap_element_id tilemap元素的唯一ID值，用于获取的单元格y位置
x在房间内绘制的x位置
y房间内的y位置

返回值：

N/A

例子：

var lay_id = layer_get_id("Tiles_Effects");
var map_id = layer_tilemap_get_id(lay_id);
draw_tilemap(map_id, mouse_x, mouse_y);

上面的代码获取图层ID，然后使用该ID获取特定的tilemap ID，后者又用于在鼠标位置绘制tilemap。
