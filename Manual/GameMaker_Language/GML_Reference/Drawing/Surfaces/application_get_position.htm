application_get_position

当您在目标平台的“游戏选项”中勾选了“保持纵横比”时， GameMaker Studio 2将自动设置应用程序表面的绘制位置，以便在给定的显示中正确地居中显示并缩放比例。然而，如果你操纵这个表面，并希望自己绘制它，那么这个功能给你一个简单的方法来找到确切位置的显示或窗口内的表面被绘制，这样你就可以自己或对齐GUI绘制有图像或将绘图图像发布到其中。

该函数将返回具有四个键的数组，其中键0和1是曲面左上角的x和y位置，键2和3是曲面右下角的x和y，所有相对于显示器或窗口的大小。

句法：

application_get_position();

返回值：

Array

例子：

var a = application_get_position();
xx = a[0];
yy = a[1];
ww = a[2] - a[0];
hh = a[3] - a[1];

上面的代码将获取应用程序表面的位置以及相对于显示窗口的绝对宽度和高度，并将它们存储在四个变量中以备将来使用。
