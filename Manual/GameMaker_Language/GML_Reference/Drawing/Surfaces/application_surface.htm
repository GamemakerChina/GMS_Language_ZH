application_surface

可以使用任何表面函数来使用此全局范围的内置变量来访问应用程序表面。该表面是永久可用的， GameMaker Studio 2可以完成大部分绘图工作。

句法：

application_surface;

返回值：

Pointer

例子：

surface_resize(application_surface, display_get_gui_width(), display_get_gui_height())

上面的代码将调整应用程序表面的大小，使其与GUI层的比例为1：1。
