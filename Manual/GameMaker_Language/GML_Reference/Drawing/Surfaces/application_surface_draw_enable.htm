应用程序_表面处理_绘图_启用

你可以使用这个功能来启用或禁用应用程序表面的自动绘制。默认情况下是启用的，但在许多情况下，你会希望接管何时以及如何绘制表面（例如使用着色器时），在这种情况下 你可以使用这个函数将其设置为false，这样你就可以在你需要的时候自己绘制。值得注意的是，当你关闭自动绘制并自己绘制应用程序的表面时，你可能会看到某些问题 你可能会看到精灵的 alpha 分量和表面本身的某些问题。这是因为GameMaker Studio 2在开启自动绘图时，会在没有alpha混合的情况下绘制应用程序的表面。如果你关闭了自动绘图，你需要 自己处理这个问题，使用下面的代码（例如）。

gpu_set_blendenable(false);
draw_surface_ext(application_surface, 0, 0, 1, 1, 0, c_white, 1);
gpu_set_blendenable(true) 。

语法。

application_surface_draw_enable(flag)。

争论 描述
旗帜 该标志将是真（启用，默认值）或假（禁用）。

返回。

不适用

例子。

application_surface_draw_enable(false)。

上述代码将关闭应用程序表面的自动绘图。
