application_surface_draw_enable

您可以使用此功能启用或禁用自动绘制应用程序表面。默认情况下启用此功能，但是在许多情况下，您将需要接管何时以及如何绘制表面（ shaders时），在这种情况下，可以使用此功能将其设置为false，以便可以绘制它自己何时以及如何要求。重要的是要注意，当您关闭自动绘图并自己绘制应用程序曲面时，您可能会看到精灵的alpha分量和曲面本身的某些问题。这是因为当启用自动绘图功能时， GameMaker Studio 2会在不使用alpha混合的情况下绘制应用程序表面。如果关闭自动绘图，则需要使用以下代码（例如）自行处理：

gpu_set_blendenable(false);
draw_surface_ext(application_surface, 0, 0, 1, 1, 0, c_white, 1);
gpu_set_blendenable(true);

句法：

application_surface_draw_enable(flag);

争论描述
flag标志将是 true（启用，默认值）或 false（已禁用）。

返回值：

N/A

例子：

application_surface_draw_enable(false);

上面的代码将关闭应用程序表面的自动绘制。
