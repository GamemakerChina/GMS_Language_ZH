application_surface_enable

使用此功能可以启用或禁用应用程序界面。默认情况下，应用程序表面已启用，并且Draw事件的所有绘制都将完成，并将表面绘制到屏幕上。但是，在某些较旧的设备和特定芯片组上，这可能会导致性能下降。在这些情况下，您可以关闭绘图到应用程序表面，然后所有Draw事件绘图将直接在屏幕缓冲区中完成。

您可以在游戏中的任何时候使用此功能来启用或禁用应用程序界面，但应注意，它至少需要一帧（步骤）才能使更改生效。 Switch它会关闭所有纵横比校正，这是在目标平台的“游戏选项”中设置的。这意味着您应该有一些方法可以将游戏缩放到所需的设备，窗口或屏幕尺寸。如果再次将其打开，则“游戏选项”将在下一步再次生效。

建议您仔细使用此功能，因为它将影响所有内容在屏幕上的绘制方式。通常，您希望将其作为游戏菜单中的一个选项，或者只在游戏开始时调用一次，然后围绕启用或禁用它编写代码。

句法：

application_surface_enable(enable);

争论描述
enable设置为启用（true ）或禁用（false ）应用程序表面。

返回值：

N/A

例子：

if keyboard_check_pressed(vk_space)
    {
    if application_surface_is_enabled()
        {
        application_surface_enable(false);
        }
    else
        {
        application_surface_enable(true);
        }
    }

上面的代码检查是否有按键，并根据其状态（例如在选项菜单中）来打开或关闭应用程序表面。
