draw_surface

使用此功能，您可以在房间内的给定位置绘制表面，并在指定的x / y位置绘制图像的左上角。

注意：使用曲面时，由于它们存储在纹理内存中，因此它们可能随时不复存在。在直接引用表面之前，应始终使用surface_exists（）检查表面是否存在。

句法：

draw_surface(id, x, y);

争论描述
id要绘制的表面的唯一ID值。
x绘制表面位置的x位置。
y绘制表面的y位置。

返回值：

N/A

例子：

var _vx = camera_get_view_x(view_camera[0]);
var _vy = camera_get_view_y(view_camera[0]);
draw_surface(surf, _vx, _vy);

上面的代码在与相机view [0]相同的位置绘制在“ surf”中索引的表面。
