绘制表面

通过这个函数，你可以在房间内的指定位置画出一个表面，图像的左上角被画在指定的X/Y位置。

注意：在处理表面时，由于它们被储存在纹理内存中，它们有可能随时不存在。你应该在直接引用曲面之前，先用 surface_exists() 来检查曲面是否存在，然后再直接引用它们。

语法。

draw_surface(id, x, y);

争论 描述
身份证 要绘制的曲面的唯一ID值。
x 绘制曲面的X位置。
y 绘制曲面的y位置。

返回。

不适用

例子。

var _vx = camera_get_view_x(view_camera[0]);
var _vy = camera_get_view_y(view_camera[0]);
draw_surface(surface, _vx, _vy);

上面的代码在与摄像机视图[0]相同的位置绘制了索引为 "surf "的表面。
