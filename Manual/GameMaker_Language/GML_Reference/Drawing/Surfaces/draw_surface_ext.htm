draw_surface_ext

该函数将像函数中那样绘制给定的曲面 draw_surface()但是还有其他选项可以更改所绘制表面的比例，混合，旋转和Alpha。改变这些值不会以任何方式修改（它只是如何绘制）的资源。

注意：使用曲面时，由于它们存储在纹理内存中，因此它们可能随时不复存在。在直接引用表面之前，应始终使用surface_exists（）检查表面是否存在。

句法：

draw_surface_ext(id, x, y, xscale, yscale, rot, col, alpha);

争论描述
id要绘制的表面的唯一ID值。
x绘制表面位置的x位置。
y绘制表面的y位置。
xscale水平刻度。
yscale垂直比例尺。
rot绘制表面的旋转或角度。
col用来混合表面的颜色。
alpha用于绘制表面的Alpha透明度。

返回值：

N/A

例子：

draw_surface_ext(surf, 0, 0, 2, 2, 0, c_red, 0.5);

上面的代码将在房间的（0,0）位置绘制一个在变量“ surf”中索引的表面，其原始缩放比例为原始比例的两倍，混合为红色和半透明。
