绘画_表面_内容

这个函数会像在函数中一样绘制给定的曲面 draw_surface()中的做法，但会有额外的选项来改变被绘制的曲面的比例、混合、旋转和阿尔法。改变这些值并不 不会以任何方式修改资源（只修改它的绘制方式）。

注意：在处理表面时，由于它们被储存在纹理内存中，它们有可能在任何时候都不存在了。你应该在直接引用曲面之前，先用 surface_exists() 来检查曲面是否存在，然后再直接引用它们。

语法。

draw_surface_ext(id, x, y, xscale, yscale, rot, col, alpha)。

争论 描述
身份证 要绘制的曲面的唯一ID值。
x 绘制曲面的X位置。
y 绘制曲面的y位置。
xscale 水平刻度。
yscale 纵向比例。
腐烂 绘制曲面的旋转或角度。
殖民地 用来混合表面的颜色。
阿尔法 绘制曲面时的阿尔法透明度。

返回。

不适用

例子。

draw_surface_ext(surface, 0, 0, 2, 2, 0, c_red, 0.5)。

上面的代码将在房间的(0,0)位置绘制一个在变量 "surf "中索引的表面，其比例是原来的两倍，混合红色和半透明。
