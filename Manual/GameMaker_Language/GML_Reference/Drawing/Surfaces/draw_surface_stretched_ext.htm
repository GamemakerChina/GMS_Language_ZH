draw_surface_stretched_ext

此功能与 draw_surface_stretched()功能，并具有设置绘制表面时的颜色混合和Alpha值的功能（类似于该功能 draw_surface_ext()）。

注意：使用曲面时，由于它们存储在纹理内存中，因此它们可能随时不复存在。在直接引用表面之前，应始终使用surface_exists（）检查表面是否存在。

句法：

draw_surface_stretched_ext(id, x, y, w, h, col, alpha);

争论描述
id要绘制的表面的唯一ID值。
x绘制表面位置的x位置。
y绘制表面的y位置。
w绘制表面的宽度。
h绘制表面的高度。
colour为表面着色的颜色。
alpha用来混合表面的Alpha。

返回值：

N/A

例子：

draw_surface_stretched_ext(surf, x, y, 200, 200, c_white, 0.5);

这将绘制给定的表面，其左角位于实例x / y位置，并且将其拉伸以占据200x200像素的区域，而没有融合，但具有部分透明度。

背面：表面
