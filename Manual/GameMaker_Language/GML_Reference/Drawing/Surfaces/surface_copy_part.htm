surface_copy_part

此功能仅从一个表面获取图像并将其复制到该表面内指定的本地位置（其中（0,0）位置是目标表面的左上角）上的图像。您可以指定要复制的本地x和y位置以及该部分的宽度和高度。请注意，这些是基于表面尺寸的坐标，而不是基于在房间中绘制表面的位置。如果目标表面已经有了这个信息将被覆盖由副本，该函数不以任何方式更改源表面。

注意：使用曲面时，由于它们存储在纹理内存中，因此它们可能随时不复存在。在直接引用表面之前，应始终使用surface_exists（）检查表面是否存在。

句法：

surface_copy_part(destination, x, y, source, xs, ys, ws, hs);

争论描述
destination要将另一个表面复制到的表面的ID。
x要复制到的x位置。
y要复制到的y位置。
source要复制的表面的ID。
xs源曲面中要复制的x位置。
ys要从中复制的源曲面中的y位置。
ws源表面中要复制的区域的宽度。
hs要复制的源曲面中区域的高度。

返回值：

N/A

例子：

if view_current == 0
    {
    surface_copy_part(surf, 0, 0, temp_surf, 0, 0, view_xview[1] - mouse_x, view_yview[1] - mouse_y);
    }
else
    {
    draw_surface(surf, 0, 0);
    }

上面的代码将检查当前绘制的视图，如果它是view [0]，它将把在变量“ temp_surf”中索引的曲面复制到在变量“ surf”中索引的曲面。复制的区域对应于鼠标在表面内的相对位置所形成的矩形，就像在view [1]中绘制的那样。如果当前视图不是view [0]，则将“ surf”表面绘制到屏幕上。
