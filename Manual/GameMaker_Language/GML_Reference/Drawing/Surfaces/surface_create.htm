创建表面

这个函数用来创建一个曲面，并将返回该曲面的索引，该索引应保存在一个变量中，以便以后调用函数。当第一次创建曲面时，它可能包含 "噪音"，因为基本上它只是一个区域的 存储器的一个区域（而且这个存储器可能仍然包含信息），所以你可能想在使用类似以下函数之前清除这个表面 draw_clear_alpha().

强烈建议所有表面的大小都是2的幂，例如：16、128、512或1024像素大小。这在某些平台（如Windows和MacOS）上并非完全必要，但它肯定会提高 兼容性，而对于HTML5和设备来说，这一点是必不可少的，你一定要记住这一点，否则以后可能会遇到问题。

语法。

surface_create(w, h)。

争论 描述
w 要创建的表面的宽度。
h 要创建的表面的高度。

返回。

真实的

例子。

if !surface_exists(surface)
{
surf = surface_create(1024, 1024);
surface_set_target(surface);
draw_clear_alpha(c_black, 0);
surface_reset_target();
view_surface_id[0] = surf;
}

上面的代码检查是否存在一个曲面，如果不存在，它将创建一个1024像素宽、1024像素高的曲面，并将其索引分配给变量 "surf"。绘图目标将被设置为新的表面，并将其清除，使其透明。 在将绘图目标重设为显示器之前，它将被清空并变得透明。最后，这个曲面被分配到一个视图中。
