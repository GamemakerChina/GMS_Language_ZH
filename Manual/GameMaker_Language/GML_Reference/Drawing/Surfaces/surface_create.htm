surface_create

该函数用于创建表面，并将返回表面的索引，该索引应存储在变量中以供将来的函数调用。首次创建表面时，它可能包含“噪音”，因为基本上它只是为该目的而留出的内存区域（并且该内存可能仍包含信息），因此您可能需要在与像 draw_clear_alpha()。

强烈建议创建所有表面，其大小为2的幂，例如：16、128、512或1024像素。 Windows和MacOS）上，这并不是完全必要的，但肯定会提高这些目标的兼容性，而对于HTML5和设备，这是必不可少的，非常重要的是要记住这一点，否则以后可能会遇到问题。

句法：

surface_create(w, h);

争论描述
w要创建的表面的宽度。
h要创建的表面的高度。

返回值：

Real

例子：

if !surface_exists(surf)
    {
    surf = surface_create(1024, 1024);
    surface_set_target(surf);
    draw_clear_alpha(c_black, 0);
    surface_reset_target();
    view_surface_id[0] = surf;
    }

上面的代码检查表面是否存在，如果不存在，它将创建一个宽1024像素，高1024像素的表面，并将索引分配给变量“ surf”。然后将绘制目标设置为新表面，在将绘制目标重置为显示器之前将其清除并变得透明。最后，将表面分配给视图。
