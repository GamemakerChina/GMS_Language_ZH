surface_create_ext

此功能使您可以将表面附加到网页上已经存在的画布元素上，这意味着您可以有效地分割游戏的各个部分，以在页面中的不同位置进行绘制。为此，您必须在 *.html使用与您要创建的表面相对应的正确尺寸和名称来显示游戏页面。因此，您将拥有“主”画布，然后是辅助曲面画布元素，将使用此功能将其分配给正确的曲面。下图是如何设置包含三个画布元素的页面的示例：

页面布局可能很棘手，特别是如果您希望所有元素正确对齐，但是一旦完成了创建布局的艰巨任务后，就可以添加此页面 html使用“游戏选项”的“ HTML5”选项卡将文件作为游戏的默认页面文件。接下来要做的是设置房间和视图，因为每个表面都需要与特定的视图关联以“捕获”游戏图像（请参见view变量）。view_surface_id[0...7] ）。下图显示了如何设置以上画布示例的游戏室：

最后，您将使用此函数创建表面，其名称与画布元素所使用的名称相同，并且其大小与该画布的大小相对应。该函数将返回表面的索引，该索引应存储在变量中，以供将来的函数调用。首次创建表面时，它可能包含“噪音”，因为基本上它只是为该目的而留出的内存区域（并且该内存可能仍包含信息），因此您可能需要在与像 draw_clear_alpha()。

注意：此功能仅可用于HTML5模块。

句法：

surface_create_ext(name, w, h);

争论描述
name链接曲面的canvas元素的名称。
w要创建的表面的宽度。
h要创建的表面的高度。

返回值：

Real

例子：

s1 = surface_create_ext("surface1", 192, 550);
s2 = surface_create_ext("surface2", 608, 186);
view_surface_id[1] = s1;
view_surface_id[2] = s2;

上面的代码创建了两个大小不同的表面，将每个表面分配给一个不同的画布元素，然后将这些表面分配给两个视图，以便捕获房间的正确部分。
