surface_free

当你在处理表面时，每当你使用完它们时，你都应该使用这个功能。表面会占用内存空间，因此需要移除，通常是在一个房间结束时，但也可以在任何时候移除，这取决于你的 你对它们的使用情况。如果不这样做，会导致内存泄漏，最终会使你的游戏变慢并崩溃。

注意：在处理表面时，由于它们被储存在纹理内存中，它们有可能随时不存在。你应该在直接引用曲面之前，先用 surface_exists() 来检查曲面是否存在，然后再直接引用它们。

语法。

surface_free(surface_id)。

争论 描述
surface_id 要释放的表面的ID。

返回。

不适用

例子。

if keyboard_check_pressed(vk_escape)
{
surface_free(surface);
room_goto(rm_Menu)。
}

上面的代码检查是否有按键，如果检测到有按键，它将释放为变量 "surf "中索引的表面保留的内存，然后改变房间。
