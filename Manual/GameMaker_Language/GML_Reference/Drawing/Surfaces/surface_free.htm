surface_free

使用曲面时，无论何时使用完表面，都应始终使用此功能。表面占用了内存中的空间，因此通常需要在房间的末端将其删除，但可以根据您的用途随时在表面上将其删除。否则可能会导致内存泄漏，从而最终减慢速度并使游戏崩溃。

注意：使用曲面时，由于它们存储在纹理内存中，因此它们可能随时不复存在。在直接引用表面之前，应始终使用surface_exists（）检查表面是否存在。

句法：

surface_free(surface_id);

争论描述
surface_id要释放的表面的ID。

返回值：

N/A

例子：

if keyboard_check_pressed(vk_escape)
    {
    surface_free(surf);
    room_goto(rm_Menu);
    }

上面的代码检查是否有按键，如果检测到按键，则释放为变量“ surf”中索引的表面保留的内存，然后更改空间。
