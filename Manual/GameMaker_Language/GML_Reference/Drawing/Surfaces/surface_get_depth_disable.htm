surface_get_depth_disable

此功能检查是否启用了为曲面自动生成深度缓冲区。通常，所有表面都具有深度缓冲区，因此，如果向其绘制3D对象，则可以按深度对它们进行正确排序，但是分配深度缓冲区实际上会使表面的大小加倍，这可能是过多且不必要的开销，尤其是当您的游戏内存很大密集或主要是2D。在这些情况下，您可以使用此功能进行检查，并在需要时使用该功能禁用表面的深度缓冲区 surface_depth_disable()。

句法：

surface_get_depth_disable();

返回值：

Boolean

例子：

if (!surface_exists(surf))
    {
    if surface_get_depth_disable() == false
        {
        surface_depth_disable(true);
        }
    surf = surface_create(room_width, room_height);
    }

上面的代码将检查给定的曲面是否存在，如果不存在，则将检查曲面深度缓冲区的当前状态，如果已启用，则将其禁用，最后创建曲面。
