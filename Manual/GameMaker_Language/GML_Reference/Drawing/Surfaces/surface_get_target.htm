surface_get_target

使用此功能，您可以获得绘制目标所在的当前表面的ID。如果您使用创建表面 surface_create()，那么这将是正整数，它是该表面的唯一ID

如果未设置自定义图面，但仍启用默认的application_surface，则该函数通常将返回0。但是，如果您之前一直在操纵application_surface，则可能希望检查获取的ID与当前值不匹配。如果该值不再为0，则为application_surface变量。

如果未设置自定义表面，并且默认的application_surface也已禁用，则该函数将返回-1。

句法：

surface_get_target();

返回值：

Integer (-1 if no surface target exists, otherwise the target surface's ID)

例子：

surface_get_target() != -1
    {
    surface_reset_target();
    }
draw_surface(surf, 0, 0);

上面的代码将检查当前渲染目标是否为曲面，如果是，它将重置目标，然后绘制一个曲面。
