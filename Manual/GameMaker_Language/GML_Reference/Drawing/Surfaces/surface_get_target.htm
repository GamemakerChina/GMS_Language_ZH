表面_获取目标

通过这个函数，你可以得到当前要绘制的曲面的ID。如果你用 surface_create()创建的，那么这个ID将是一个正整数，是该曲面的唯一ID。 该表面的唯一ID

如果没有设置自定义表面，但默认的application_surface仍然被启用，那么该函数通常会返回0。然而，如果你之前一直在操作application_surface，你可能希望检查你得到的ID是否 与application_surface变量的当前值不一致，以防止它不再是0。

如果没有设置自定义表面，并且默认的application_surface也被禁用，那么该函数将返回-1。

语法。

surface_get_target()。

返回。

整数（如果不存在目标表面，则为-1，否则为目标表面的ID）。

例子。

surface_get_target() != -1
{
surface_reset_target()。
}
draw_surface(surface, 0, 0);

上面的代码会检查当前的渲染目标是否是一个曲面，如果是，就会重置目标，然后画出一个曲面。
