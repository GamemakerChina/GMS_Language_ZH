表面_获取目标内容

这个函数将检索分配给曲面的4个渲染目标之一的曲面ID。如果没有指定曲面，函数会返回-1，或者返回一个>=0的整数，表示 分配给曲面的ID（如函数所返回的 surface_create()).

语法。

surface_get_target_ext(index)。

争论 描述
指数 要检查的渲染目标索引（从0到3）。

返回。

整数（-1表示没有表面目标，或一个表面ID值）。

例子。

if surface_get_target_ext(0) == -1
{
surface_set_target_ext(0, global.Surf);
}

上面的代码首先会检查并查看着色器的渲染目标0是否已经被设置为一个曲面，如果没有，那么就会分配一个。
