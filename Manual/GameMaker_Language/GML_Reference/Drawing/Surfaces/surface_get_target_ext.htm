surface_get_target_ext

此函数将检索分配给曲面可用的4个渲染目标之一的曲面ID。您提供要检查的渲染目标的索引，如果未分配任何表面，则该函数将返回-1，或者>> 0的整数值，表示分配的表面的ID（由该函数返回）surface_create() ）。

句法：

surface_get_target_ext(index);

争论描述
index要检查的渲染目标索引（从0到3）。

返回值：

Integer (-1 for no surface target, or a surface ID value)

例子：

if surface_get_target_ext(0) == -1
    {
    surface_set_target_ext(0, global.Surf);
    }

上面的代码将首先检查并查看是否将shader渲染目标0设置为曲面，如果没有，则分配一个。
