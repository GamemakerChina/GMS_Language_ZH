表面_获取纹理

这个函数返回表面纹理的一个特殊指针。然后这个值可以在其他的绘制函数中使用，特别是在一般的三维和一些二维原始函数中。

注意：在处理表面时，由于它们被储存在纹理内存中，它们有可能在任何时候都不存在了。你应该在直接引用曲面之前，先用 surface_exists() 来检查曲面是否存在，然后再直接引用它们。

语法。

surface_get_texture(surface_id)。

争论 描述
surface_id 要获取宽度的曲面的ID。

返回。

指针

例子。

var tex;
tex = surface_get_texture(surface);
draw_primitive_begin_texture(pr_trianglestrip, tex);
draw_vertex_texture(0, 480, 0, 0);
draw_vertex_texture(640, 480, 1, 0);
draw_vertex_texture(640, 480, 1, 1);
draw_vertex_texture(0, 480, 0, 1);
draw_primitive_end()。

上面的代码将绘制一个4个顶点的三角形条状图，其纹理保存在 "tex "变量中，该变量本身来自先前创建的曲面。
