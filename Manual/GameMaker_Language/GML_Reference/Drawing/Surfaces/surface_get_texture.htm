surface_get_texture

此函数返回表面纹理的特殊指针。然后，可以在其他绘图功能中使用此值，特别是在通用3D和某些2D基本功能中。

注意：使用曲面时，由于它们存储在纹理内存中，因此它们可能随时不复存在。在直接引用表面之前，应始终使用surface_exists（）检查表面是否存在。

句法：

surface_get_texture(surface_id);

争论描述
surface_id要获得宽度的表面的ID。

返回值：

Pointer

例子：

var tex;
tex = surface_get_texture(surf);
draw_primitive_begin_texture(pr_trianglestrip, tex);
draw_vertex_texture(0, 480, 0, 0);
draw_vertex_texture(640, 480, 1, 0);
draw_vertex_texture(640, 480, 1, 1);
draw_vertex_texture(0, 480, 0, 1);
draw_primitive_end();

上面的代码将绘制一个4顶点三角带，其纹理保留在“ tex”变量中，该变量本身取自先前创建的曲面。
