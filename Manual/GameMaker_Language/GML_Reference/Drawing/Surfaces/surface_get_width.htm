surface_get_width

该函数仅返回索引曲面的宽度（以像素为单位）。请注意，如果您调用此命令来检查 application_surface更改大小后立即使用 surface_resize()它不会返回新值，因为更改需要一两个步骤才能完全处理。等待一步后，它应该正确返回新的大小。

注意：使用曲面时，由于它们存储在纹理内存中，因此它们可能随时不复存在。在直接引用表面之前，应始终使用surface_exists（）检查表面是否存在。

句法：

surface_get_width(surface_id);

争论描述
surface_id要获得宽度的表面的ID。

返回值：

Real

例子：

sw = surface_get_width(surf);

上面的代码将在变量“ sw”的变量“ surf”中存储索引的表面宽度。
