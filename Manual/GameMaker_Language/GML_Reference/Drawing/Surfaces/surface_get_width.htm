表面_获取宽度

这个函数简单地返回索引的表面的宽度，单位是像素。需要注意的是，如果你在改变了application_surface的尺寸后，立即调用这个函数来检查其 surface_resize() 后，它不会返回新的值，因为改变需要一两步才能完全处理。在等待一个步骤后，它应该正确地返回新的尺寸。

注意：在处理表面时，由于它们被存储在纹理内存中，它们有可能随时不存在。你应该在直接引用曲面之前，先用 surface_exists() 来检查曲面是否存在，然后再直接引用它们。

语法。

surface_get_width(surface_id)。

争论 描述
surface_id 要获取宽度的曲面的ID。

返回。

真实的

例子。

sw = surface_get_width(surface)。

上面的代码将把变量 "surf "中索引的表面的宽度存储到变量 "sw "中。
