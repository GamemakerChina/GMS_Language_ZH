surface_reset_target

使用此功能，您可以将所有进一步的绘制从目标表面重置回屏幕。请注意，要开始在表面上绘制，必须先调用函数 surface_set_target()然后在完成之后执行此操作，对于您设置的每个曲面目标，否则将不会在屏幕上绘制任何图形，因为所有进一步绘制（即使在其他情况下）也将在曲面上完成。您还应该意识到，如果在实例的绘制事件中未在屏幕上绘制表面本身，则什么也看不到。

注意：如果您以前未使用函数surface_set_target（）设置渲染目标，则使用此函数将以静默方式（即：没有任何错误消息）结束该事件的所有进一步代码执行。

句法：

surface_reset_target();

返回值：

N/A

例子：

if view_current = 0
    {
    surface_set_target(surf);
    with (obj_Effect)
        {
        draw_self();
        }
    surface_reset_target();
    }
else
    {
    draw_surface(surf, 0, 0);
    }

上面的代码将检查当前正在绘制哪个视图，如果是view [0]，则将绘制目标设置为表面并绘制对象“ obj_Effect”的所有实例，然后再次重置绘制目标。如果视图不是view [0]，则将表面绘制到屏幕上。
