surface_resize

此功能会将表面大小调整为给定的尺寸（以像素为单位）。 “ surface_id”是您先前创建的表面的表面，或 application_suface，该函数将调整该表面的大小。请注意，这既不会裁剪也不会拉伸表面的内容，而是会破坏当前表面，并使用与新尺寸相同的手柄（surface_id）重新创建它，这意味着需要清除并再次绘制（除非它是 application_surface在这种情况下， GameMaker Studio 2将自动执行此操作）。

注意：如果要调整应用程序表面的大小，则在下一个并条机开始之前将不会记录这些更改，这意味着在同一事件或步骤中调用surface_get_width（）或surface_get_height（）函数将返回先前的值。

句法：

surface_resize(surface_id, w, h);

争论描述
surface_id要更改的表面的ID。
w新表面的宽度。
h新表面的高度。

返回值：

N/A

例子：

if view_wport[0] != surface_get_width(application_surface) || view_hport[0] != surface_get_height(application_surface)
    {
    surface_resize(application_surface, view_wport[0],view_hport[0]);
    }

上面的代码将根据表面“ aplication_surface”的大小检查视口大小，如果不同，则调整表面大小。
