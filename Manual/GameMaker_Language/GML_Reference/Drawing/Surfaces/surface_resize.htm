表面_调整大小

这个函数将调整一个曲面的尺寸（单位：像素）。surface_id "是你之前创建的曲面的id，或者是 应用程序_表面，该函数将调整 那个曲面。请注意，这样做既不会裁剪也不会拉伸曲面的内容，而是破坏当前的曲面，用相同的句柄(surface_id)重新创建新的尺寸，这意味着它需要被清空 并重新绘制（除非它是application_surface，在这种情况下GameMaker Studio 2会自动这样做）。

注意：如果你正在调整应用程序表面的大小，这些变化将不会被注册，直到下一个绘图帧的开始，这意味着，调用 surface_get_width() 或 surface_get_height()函数会返回之前的值。

语法。

surface_resize(surface_id, w, h)。

争论 描述
surface_id 要改变的表面的ID。
w 新表面的宽度。
h 新表面的高度。

返回。

不适用

例子。

if view_wport[0] != surface_get_width(application_surface) || view_hport[0] != surface_get_height(application_surface)
{
surface_resize(application_surface, view_wport[0],view_hport[0]);
}

上面的代码将检查视图端口的大小和表面 "aplication_surface "的大小，如果不一样，表面将被重新调整大小。
