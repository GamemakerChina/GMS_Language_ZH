surface_save

此功能将使用给定的文件名将表面保存到光盘。表面必须另存为 *.png格式文件。

句法：

surface_save(surface_id, fname);

争论描述
surface_id要设置为绘制目标的表面的ID。
fname保存的图像文件的名称。

返回值：

N/A

例子：

if keyboard_check_pressed(ord("S")
    {
    surface_save(surf, "test.png");
    }

上面的代码将检查用户是否按了键盘上的“ S”键，如果这样做，则会将变量“ surf”中索引的表面保存到光盘中。
