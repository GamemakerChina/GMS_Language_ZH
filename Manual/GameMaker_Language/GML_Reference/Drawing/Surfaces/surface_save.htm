表面_保存

这个函数将使用给定的文件名将一个曲面保存到光盘中。曲面必须保存为*.png格式的文件。

语法。

surface_save(surface_id, fname)。

争论 描述
surface_id 要设置为绘图目标的曲面的ID。
fname 保存的图像文件的名称。

返回。

不适用

例子。

if keyboard_check_pressed(ord("S")
{
surface_save(surface, "test.png");
}

上面的代码将检查用户是否按了键盘上的 "S "键，如果按了，它将把变量 "surf "中索引的表面保存到光盘中。
