surface_save_part

此功能将使用给定的文件名将表面的一部分保存到光盘上。表面必须另存为 *.png格式文件，并且（x，y）位置必须作为该表面的局部坐标，并要记住表面的左上角始终为（0,0）。

句法：

surface_save_part(surface_id, fname, x, y, width, height);

争论描述
surface_id要设置为绘制目标的表面的ID。
fname保存的图像文件的名称。
x曲面内的起始x位置。
y曲面内的起始y位置。
width要保存的零件的宽度。
height要保存的零件的高度。

返回值：

N/A

例子：

if keyboard_check_pressed(ord("S")
    {
    surface_save_part(surf, "test.png", 0, 0, 100, 100);
    }

上面的代码将检查用户是否按了键盘上的“ S”键，如果这样做，会将在变量“ surf”中索引的部分表面保存到光盘中。
