surface_set_target

使用此功能，您可以将所有进一步绘制设置到目标表面而不是屏幕上，这样您就可以告诉GameMaker Studio 2仅在指定表面上绘制特定物体。请注意，如果您不拨打电话 surface_reset_target()完成后，屏幕上将不会绘制任何内容，因为所有进一步绘制（即使在其他情况下）也将在表面上完成。您还应该意识到，如果在实例的绘制事件中未在屏幕上绘制表面本身，则什么也看不到。您也可以作为调试工具来检查此函数的返回值，以检查是否设置了表面目标，返回值0表示设置目标失败，其他任何正值表示成功。

应该注意的一件事是，曲面是堆叠在一起的，因此您不能从目标跳到目标，然后最后重置为正常绘制目标，而必须打开和关闭渲染目标。例如，这将无法正常工作：

surface_set_target(surf1);
draw_text(32, 32, "surface1");
surface_set_target(surf2);
draw_text(32, 64, "surface2");
surface_reset_target();

取而代之的是，您必须为设置的每个表面重置目标，就像必须对代码块使用开括号{}一样。因此，以上内容应这样写：

surface_set_target(surf1);
draw_text(32, 32, "surface1");
surface_reset_target();
surface_set_target(surf2);
draw_text(32, 64, "surface2");
surface_reset_target();

或这样：

surface_set_target(surf1);
draw_text(32, 32, "surface1");
surface_set_target(surf2);
draw_text(32, 64, "surface2");
surface_reset_target();
surface_reset_target();

注意：使用曲面时，由于它们存储在纹理内存中，因此它们可能随时不复存在。在直接引用表面之前，应始终使用surface_exists（）检查表面是否存在。

句法：

surface_set_target(surface_id);

争论描述
surface_id要设置为绘制目标的表面的ID。

返回值：

Boolean

例子：

if view_current = 0
    {
    surface_set_target(surf);
    with (obj_Effect)
        {
        draw_self();
        }
    surface_reset_target();
    }
else
    {
    draw_surface(surf, 0, 0);
    }

上面的代码将检查当前正在绘制哪个视图，如果是view [0]，则将绘制目标设置为表面并绘制对象“ obj_Effect”的所有实例，然后再次重置绘制目标。如果视图不是view [0]，则将表面绘制到屏幕上。
