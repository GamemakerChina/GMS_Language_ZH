surface_set_target_ext

这个函数与着色器函数一起使用，并为本地级别的着色器（DX9、DX11、OpenGL）设置MRT（0 - 3）。

注意：在处理曲面时，由于它们被储存在纹理内存中，它们有可能在任何时候都不存在了。你应该在直接引用曲面之前，先用 surface_exists() 来检查曲面是否存在，然后再直接引用它们。

语法。

surface_set_target_ext(index, surface_id)。

争论 描述
指数 要使用的渲染目标索引（从0到3）。
surface_id 要使用的表面的ID。

返回。

布尔型

例子。

surface_set_target_ext(0, surf)。

上面的代码将把渲染目标0设置为变量 "surf "中索引的表面ID。
