surface_set_target_ext

此功能与“ Shader功能”一起使用，并为本机级shaders （DX9，DX11，OpenGL）设置MRT（0-3）。

注意：使用曲面时，由于它们存储在纹理内存中，因此它们可能随时不复存在。在直接引用表面之前，应始终使用surface_exists（）检查表面是否存在。

句法：

surface_set_target_ext(index, surface_id);

争论描述
index要使用的渲染目标索引（从0到3）。
surface_id要使用的表面的ID。

返回值：

Boolean

例子：

surface_set_target_ext(0, surf);

上面的代码会将渲染目标0设置为在变量“ surf”中索引的表面ID。
