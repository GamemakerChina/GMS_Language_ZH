文本

您可以使用多种功能来绘制各种形式的文本。应该注意的是，其中一些功能在绘制时会以动态方式更改文本的颜色和大小，但是当使用HTML5或设备时，这些功能可能会减慢游戏速度或使其外观变差，因此最好使用各种功能。字体大小的资源，而不是使用比例图，或具有大字体的资源，然后绘制它们按比例缩小而不是向上。另外，如果您需要为大块文本加上颜色，则最好使用 draw_set_colour()和 draw_set_alpha()功能首先与简单 draw_text函数，而不是具有颜色成分的那些扩展函数。

以下功能用于设置绘制时要使用的字体以及绘制时文本的位置：

  • draw_set_font
  • draw_set_halign
  • draw_set_valign
  • draw_get_font
  • draw_get_halign
  • draw_get_valign

这些功能可用于将文本绘制到屏幕上，既可以简单地也可以使用变换和彩色：

  • draw_text
  • draw_text_ext
  • draw_text_colour
  • draw_text_transformed
  • draw_text_ext_colour
  • draw_text_ext_transformed
  • draw_text_transformed_colour
  • draw_text_ext_transformed_colour

还存在以下特定于绘制使用内置highscore系统创建的highscore的功能：

  • draw_highscore
