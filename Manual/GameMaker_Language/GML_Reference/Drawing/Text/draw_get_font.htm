绘制字体

这个函数将获得当前为绘制文本所分配的字体。如果没有设置字体，该函数将返回-1，或者返回指定字体资源的ID值（一个正整数）。

语法。

draw_get_font()。

返回。

整数（-1或字体ID值）。

例子。

如果 draw_get_font() != fnt_MainMenu
{
draw_set_colour(c_blue);
draw_set_font(fnt_MainMenu)。
}
draw_text(room_width / 2, 200, "MENU" ) 。

上面的代码将检查哪种字体被用于绘制，如果不是fnt_MainMenu，那么在一些文本被绘制到屏幕上之前，将设置字体和绘制颜色。

返回。文本
