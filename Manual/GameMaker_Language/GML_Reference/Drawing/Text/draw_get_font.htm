draw_get_font

此功能将获取当前分配给工程图文本的字体。如果未设置字体，或者分配的字体资源的ID值（正整数），该函数将返回-1。

句法：

draw_get_font();

返回值：

Integer (-1 or font ID value)

例子：

if draw_get_font() != fnt_MainMenu
    {
    draw_set_colour(c_blue);
    draw_set_font(fnt_MainMenu);
    }
draw_text(room_width / 2, 200, "MENU" );

上面的代码将检查是否使用了哪种字体进行绘制，如果没有使用 fnt_MainMenu，然后在将某些文本绘制到屏幕之前设置字体和绘制颜色。
