draw_get_halign

此函数用于获取水平轴上的文本对齐设置，并将返回下面列出的常量之一。

句法：

draw_get_halign();

返回值：

Constant

持续的结盟
fa_left
fa_center
fa_right

例子：

if draw_get_halign() != fa_left
    {
    draw_set_halign(fa_left);
    }
draw_text(100, 32, "Score: " + string(score));

上面的代码将检查当前文本对齐方式是否为左对齐，如果不是，则将其设置，然后绘制一些文本。
