draw_get_valign

此函数用于获取沿垂直轴的文本对齐设置，并将返回下面列出的常量之一。

句法：

draw_get_valign();

返回值：

Constant

持续的结盟
fa_top
fa_middle
fa_bottom

例子：

if draw_get_valign() != fa_top
    {
    draw_set_valign(fa_top);
    }
draw_text(100, 32, "Score: " + string(score));

上面的代码将检查当前文本对齐方式是否为顶部对齐，如果不是，则将其设置，然后绘制一些文本。
