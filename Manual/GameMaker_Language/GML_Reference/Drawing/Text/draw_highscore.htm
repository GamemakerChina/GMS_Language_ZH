draw_highscore

这个简单的功能将使用指定矩形内的当前设置的字体，颜色和Alpha值来绘制当前内部存储的高分列表。您提供用于绘制文本的矩形区域的左上角和右下角的坐标， GameMaker Studio 2将负责其余部分，并自动完成间距和位置。

句法：

draw_highscore( x1, y1, x2, y2 );

争论描述
x1高分矩形左侧的x坐标。
y1高分矩形顶部的y坐标。
x2高分矩形右侧的x坐标。
y2高分矩形底部的y坐标。

返回值：

N/A

例子：

draw_highscore(100, 100, room_width - 100, room_height - 100);

这将在房间中间以100px边框的矩形中绘制高分表。
