draw_set_font

此功能将设置所有其他文本绘图使用的字体。该字体必须已添加到游戏的字体资源中，或已使用 font_add ()或者 font_add_sprite ()。

句法：

draw_set_font(font);

争论描述
font要使用的字体名称。

返回值：

N/A

例子：

draw_set_colour(c_blue);
draw_set_font(fnt_Game);
draw_text(200, 200, "Hello World" );

上面的代码将使用在变量“ fnt_Game”中索引并显示为蓝色的字体绘制给定的文本。
