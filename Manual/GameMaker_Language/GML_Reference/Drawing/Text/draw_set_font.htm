绘制字体

这个函数将设置所有进一步绘制文本时使用的字体。这个字体必须已经被添加到游戏的字体资产中，或者是用 font_add() 或 font_add_sprite().

语法。

draw_set_font(font)。

争论 描述
字体 要使用的字体的名称。

返回。

不适用

例子。

draw_set_colour(c_blue);
draw_set_font(fnt_Game);
draw_text(200, 200, "Hello World" )。

上述代码将使用变量 "fnt_Game "中索引的字体绘制给定的文本，并将其染成蓝色。
