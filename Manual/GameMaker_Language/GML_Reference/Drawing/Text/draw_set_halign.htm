draw_set_halign

此功能用于沿水平轴对齐文本，更改水平对齐方式将更改绘制所有其他文本的位置和方向，默认值为 fa_left。接受以下常量：

持续的结盟
fa_left
fa_center
fa_right

句法：

draw_set_halign(halign);

争论描述
halign水平对齐。

返回值：

N/A

例子：

draw_set_halign(fa_left);
draw_text(100, 32, "Score: " + string(score));
draw_set_halign(fa_right);
draw_text(room_width - 100, 32, "Health: " + string(health));

上面的代码将在同一行上绘制两个字符串，其中乐谱为左对齐，而运行状况为右对齐。
