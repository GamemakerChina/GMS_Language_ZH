draw_set_valign

此功能用于沿垂直轴对齐文本，更改垂直对齐方式将更改绘制所有其他文本的位置和方向，默认值为 fa_top。接受以下常量：

持续的结盟
fa_top
fa_middle
fa_bottom

句法：

draw_set_valign(valign);

争论描述
valign垂直对齐。

返回值：

N/A

例子：

draw_set_halign(fa_center);
draw_set_valign(fa_middle);
draw_text(100, 32, "Score: " + string(score));

上面的代码将以文本的正中心为中心绘制分数。
