draw_text

使用此功能，您可以在房间内的任何位置绘制任何字符串（要绘制实数，应使用 string ()函数将它们转换为文本）。要组合字符串，可以使用+ （请参见下面的示例），也可以在字符串中使用\ n添加换行符，以便在多行上绘制换行符（有关如何正确设置字符串格式以及可以使用哪些转义字符的信息使用，请参阅此处）。文字和Alpha的颜色由当前基本Alpha和颜色值控制，该值由 draw_set_alpha ()和 draw_set_colour ()。

注意：文本的实际位置将受设置的对齐值的影响 draw_set_halign ()和 draw_set_valign ()。

句法：

draw_text(x, y, string);

争论描述
x绘制的字符串的x坐标。
y绘制字符串的y坐标。
string要绘制的字符串。

返回值：

N/A

例子：

draw_text(x, y, "Hello, " + global.Name + "!\nI hope you are well!");

上面的代码将在实例x / y位置绘制一个字符串，该字符串将使用存储在全局变量“ Name”中的字符串并将其分成两行。
