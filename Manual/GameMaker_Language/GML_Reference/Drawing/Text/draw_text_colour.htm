draw_text_colour

此功能将以类似于 draw_text ()仅现在，您可以选择用于为文本着色的颜色以及alpha值，并且将使用这些新值代替基础图形颜色和alpha。

注意：除非启用了WebGL，否则渐变混合不适用于HTML5目标，尽管您仍然可以设置混合颜色，并且它将字体与第一种给定颜色混合。但是，以这种方式进行的所有混合都会创建一个重复的字体，然后将其存储在缓存中并在需要时使用，这并不是最佳选择，如果您使用多种颜色更改，将会降低游戏性能。您可以使用函数font_set_cache_size（）设置字体缓存大小，以尝试限制此大小。

句法：

draw_text_colour(x, y, string, c1, c2, c3, c4, alpha);

争论描述
x绘制的字符串的x坐标。
y绘制字符串的y坐标。
string要绘制的字符串。
c1所绘制文本左上角的颜色。
c2所绘制文本右上角的颜色。
c3所绘制文本右下角的颜色。
c4所绘制文本左下角的颜色。
alpha文字的Alpha。

返回值：

N/A

例子：

draw_set_colour(c_white);
draw_text(100, 100, "Health");
draw_text_colour(100, 200, string(health), c_lime, c_lime, c_green, c_green, 1);

上面的代码将在同一行上绘制文本的两个部分，第一个文本被绘制为白色（这是基本的绘制颜色），第二个文本则用柠檬绿到正常的绿色渐变绘制。
