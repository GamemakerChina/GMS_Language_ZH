draw_text_ext

此功能将以类似于 draw_text ()直到现在，您才可以设置文本每行之间的间隔-如果文本占用多于一行-并限制每行字符串的宽度（以像素为单位），以便任何行超过此值， GameMaker Studio 2都会自动拆分将文本移到最接近的可用空白处的下一行（如果文本没有空白，则它将超出此最大宽度值）。请注意，因此，在解析字符串以进行绘制之前，将删除字符串开头的所有空白。还要注意，行分隔参数的值-1将默认为基于所选字体中“ M”字符高度的分隔。

句法：

draw_text_ext(x, y, string, sep, w);

争论描述
x绘制的字符串的x坐标。
y绘制字符串的y坐标。
string要绘制的字符串。
sep文本行之间的距离（以像素为单位）。
w换行之前字符串的最大宽度（以像素为单位）。

返回值：

N/A

例子：

draw_text_ext(100, 50, keyboard_string, 3, 300);

上面的代码将在用户键入的任何文本上绘制到键盘上，每当该行的字符串长度超过300像素时，将其拆分为新行。如果发生这种情况，该代码还将在行之间保持3个像素的间隔。
