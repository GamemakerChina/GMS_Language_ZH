draw_text_ext_transformed

这个函数是由基础的 绘制文本()函数与 draw_text_ext()和 draw_text_transformed() 函数的组合，允许你在保持特定行距和每行最大宽度的情况下缩放和旋转文本。请注意，"宽度 "参数是以1的比例为基础的，所以如果比例不同，这个值应按比例改变 比例地改变。例如，如果断行的基本宽度是300，而你把比例设置为2，那么文本就会出现错误，超出了给定的宽度。相反，你应该将宽度设置为150，以补偿比例的变化。

语法。

draw_text_ext_transformed(x, y, string, sep, w, xscale, yscale, angle)。

争论 描述
x 画出的字符串的X坐标。
y 画出的字符串的y坐标。
绳子 要拉的绳子。
绵绵不绝 文本行之间的距离，以像素为单位。
w 换行前字符串的最大宽度（像素）。
xscale 水平刻度。
yscale 纵向比例。
角度 文字的角度。

返回。

不适用

例子。

draw_set_halign（fa_center）;
draw_set_valign（fa_middle）。
image_angle += 1;
draw_text_ext_transformed（room_width / 2, room_height / 2, keyboard_string, 10, 300, 2, 2, image_angle）。

上面的代码将在房间中间画出给定的文字，最大字符串长度为300像素，每行之间的间距为10像素，旋转一圈，缩放为原始尺寸的两倍。
