draw_text_ext_transformed

此功能是基础的组合 draw_text ()与 draw_text_ext ()和 draw_text_transformed ()功能，允许您缩放和旋转文本，同时保持特定的行距和每行的最大宽度。请注意，“ width”参数基于小数位1，因此，如果小数位不同，则应按比例更改此值。例如，如果换行符的基本宽度为300，并且将比例设置为2，则文本将显示为错误，超出给定宽度。相反，您应该将宽度设置为150以补偿缩放比例。

句法：

draw_text_ext_transformed(x, y, string, sep, w, xscale, yscale, angle);

争论描述
x绘制的字符串的x坐标。
y绘制字符串的y坐标。
string要绘制的字符串。
sep文本行之间的距离（以像素为单位）。
w换行之前字符串的最大宽度（以像素为单位）。
xscale水平刻度。
yscale垂直比例尺。
angle文本的角度。

返回值：

N/A

例子：

draw_set_halign(fa_center);
draw_set_valign(fa_middle);
image_angle += 1;
draw_text_ext_transformed(room_width / 2, room_height / 2, keyboard_string, 10, 300, 2, 2, image_angle);

上面的代码将在房间的中间绘制给定的文本，最大字符串长度为300像素，每行之间的间距为10像素，旋转并缩放到其原始大小的两倍。
