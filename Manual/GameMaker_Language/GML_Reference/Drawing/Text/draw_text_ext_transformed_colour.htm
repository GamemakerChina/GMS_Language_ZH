draw_text_ext_transformed_colour

此功能是基础的组合 draw_text ()与 draw_text_ext ()，draw_text_transformed ()和 draw_text_colour ()功能。因此，这是所有用于绘制文本的功能中最通用的功能，并且在绘制时具有很大的灵活性。请注意，“ width”参数基于小数位1，因此，如果小数位不同，则应按比例更改此值。例如，如果换行符的基本宽度为300，并且将比例设置为2，则文本将显示为错误，超出给定宽度。相反，您应该将宽度设置为150以补偿缩放比例。

注意：除非启用了WebGL，否则渐变混合不适用于HTML5目标，尽管您仍然可以设置混合颜色，并且它将字体与第一种给定颜色混合。但是，以这种方式进行的所有混合都会创建一个重复的字体，然后将其存储在缓存中并在需要时使用，这并不是最佳选择，如果您使用多种颜色更改，将会降低游戏性能。您可以使用函数font_set_cache_size（）设置字体缓存大小，以尝试限制此大小。

句法：

draw_text_ext_transformed_colour(x, y, string, sep, w, xscale, yscale, angle, c1, c2, c3, c4, alpha);

争论描述
x绘制的字符串的x坐标。
y绘制字符串的y坐标。
string要绘制的字符串。
sep文本行之间的距离（以像素为单位）。
w换行之前字符串的最大宽度（以像素为单位）。
xscale水平刻度。
yscale垂直比例尺。
angle文本的角度。
c1所绘制文本左上角的颜色。
c2所绘制文本右上角的颜色。
c3所绘制文本右下角的颜色。
c4所绘制文本左下角的颜色。
alpha文字的Alpha。

返回值：

N/A

例子：

draw_set_halign(fa_center);
draw_set_valign(fa_middle);
draw_text_ext_transformed_colour(200, 200, "My name is" + global.name +".", 20, 200, 0.5, 0.5, 180, c_blue, c_blue, c_navy, c_navy, 1);

这将以蓝色到深蓝色的向下渐变绘制给定的文本，该渐变在房间中的位置（200,200）处旋转，使其上下颠倒，并且为字体正常大小的一半。如果绘制了多条线，则每条线之间将有20个像素，并且每条线的最大宽度为200像素。
