draw_text_transformed

此功能将以类似于 draw_text ()直到现在，您可以选择沿水平或垂直轴缩放文本（有效地拉伸或缩小文本），并让GameMaker Studio 2以一定的角度绘制文本（其中0为法线，而度数大于0则逆时针旋转文本） 。

句法：

draw_text_transformed(x, y, string, xscale, yscale, angle);

争论描述
x绘制的字符串的x坐标。
y绘制字符串的y坐标。
string要绘制的字符串。
xscale水平刻度（默认为1）。
yscale垂直刻度（默认为1）。
angle文本的角度。

返回值：

N/A

例子：

draw_set_halign(fa_center);
draw_set_valign(fa_middle);
image_angle += 1;
draw_text_transformed(room_width / 2, room_height / 2, "GAME OVER!", 2, 2, image_angle);

上面的代码将在房间中间绘制给定的文本，旋转并缩放到其原始大小的两倍。
