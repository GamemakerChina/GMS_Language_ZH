绘图_文本_变换的

这个函数将以类似于 draw_text()只是现在你可以选择沿水平或垂直轴缩放文本（有效地拉伸或缩小它），也可以让 GameMaker Studio 2以某种角度绘制（0是正常的，每超过0度，文本就会逆时针旋转）。

语法。

draw_text_transformed(x, y, string, xscale, yscale, angle)。

争论 描述
x 画出的字符串的X坐标。
y 画出的字符串的y坐标。
绳子 要拉的绳子。
xscale 水平刻度（默认为1）。
yscale 垂直比例（默认为1）。
角度 文字的角度。

返回。

不适用

例子。

draw_set_halign（fa_center）;
draw_set_valign（fa_middle）;
image_angle += 1;
draw_text_transformed(room_width / 2, room_height / 2, "GAME OVER!", 2, 2, image_angle) 。

上面的代码将在房间的中间画出给定的文字，旋转一圈，并将其缩放为原来的两倍大小。
