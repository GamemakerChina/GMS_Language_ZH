draw_text_transformed_colour

此功能是基础的组合 draw_text()与 draw_text_transformed()和 draw_text_colour()功能，允许您缩放和旋转文本以及使用渐变填充为文本上色并更改其alpha值，而忽略绘图的基本alpha和颜色设置。

注意：除非启用了WebGL，否则渐变混合不适用于HTML5目标，尽管您仍然可以设置混合颜色，它将使字体与第一种给定颜色混合。但是，以这种方式进行的所有混合都会创建一个重复的字体，然后将其存储在缓存中并在需要时使用，这并不是最佳选择，如果您使用多种颜色更改，将会降低游戏性能。您可以使用函数font_set_cache_size（）设置字体缓存大小，以尝试限制此大小。

句法：

draw_text_transformed_colour(x, y, string, xscale, yscale, angle, c1, c2, c3, c4, alpha);

争论描述
x绘制的字符串的x坐标。
y绘制字符串的y坐标。
string要绘制的字符串。
xscale水平刻度。
yscale垂直比例尺。
angle文本的角度。
c1所绘制文本左上角的颜色。
c2所绘制文本右上角的颜色。
c3所绘制文本右下角的颜色。
c4所绘制文本左下角的颜色。
alpha文字的Alpha。

返回值：

N/A

例子：

draw_set_halign(fa_center);
draw_set_valign(fa_middle);
image_angle += 1;
draw_text_transformed_colour(room_width / 2, room_height / 2, keyboard_string, 2, 2, image_angle, c_red, c_red, c_yellow, c_yellow, 0.5);

上面的代码将在房间的中间绘制给定的文本，旋转并缩放到其原始大小的两倍，颜色渐变从黄色到红色，且alpha值为0.5。
