绘制_文本转换的颜色

这个函数是由基础的 draw_text()函数与 draw_text_transformed()和 draw_text_colour() 函数的组合，允许你缩放和旋转文本，以及用梯度填充为其着色，并改变其alpha值，忽略了绘图的基本alpha和颜色设置。

注意： 除非启用WebGL，否则梯度混合对HTML5目标是不可用的，尽管你仍然可以设置混合颜色，它将用第一个给定的颜色混合字体。然而，所有以这种方式进行的混合都会创建一个重复的 字体，然后存储在缓存中并在需要时使用，这远非最佳状态，而且如果你使用多种颜色变化，会降低你的游戏性能。如果有必要，你可以设置字体缓存的大小来尝试限制这种情况 使用函数 font_set_cache_size().

语法。

draw_text_transformed_colour(x, y, string, xscale, yscale, angle, c1, c2, c3, c4, alpha) 。

争论 描述
x 画出的字符串的X坐标。
y 画出的字符串的y坐标。
绳子 要拉的绳子。
xscale 水平刻度。
yscale 纵向比例。
角度 文字的角度。
c1 绘制文本左上方的颜色。
c2 绘制文本右上方的颜色。
c3 绘制文本右下方的颜色。
c4 绘制文本左下方的颜色。
阿尔法 文本的字母。

返回。

不适用

例子。

draw_set_halign（fa_center）;
draw_set_valign（fa_middle）。
image_angle += 1;
draw_text_transformed_colour(room_width / 2, room_height / 2, keyboard_string, 2, 2, image_angle, c_red, c_red, c_yellow, c_yellow, 0.5);

上面的代码将在房间的中间画出给定的文字，旋转的圆形，并按比例放大到原来的两倍，颜色渐变，从黄色到红色，还有一个0.5的alpha。

返回。文本
