以下功能均与“纹理”有关，即：存储在VRAM中的图像，这些图像可用作游戏中的精灵或其他东西。可用的第一组功能用于检索有关特定图像及其在纹理页面上的位置或大小的信息，以及设置某些纹理功能：

  • texture_get_width
  • texture_get_height
  • texture_get_texel_width
  • texture_get_texel_height
  • texture_get_uvs
  • texture_set_stage
  • texture_global_scale

GameMaker Studio 2还具有许多与管理与单个纹理以及整个纹理组关联的VRAM相关的功能：

  • draw_texture_flush
  • texture_prefetch
  • texture_flush
  • sprite_flush
  • sprite_flush_multi
  • sprite_prefetch
  • sprite_prefetch_multi
  • draw_flush

最后，我们具有以下功能，这些功能主要用于调试项目并确保有效使用纹理内存：

  • texture_debug_messages
  • texture_is_ready
  • texturegroup_get_textures
  • texturegroup_get_sprites
  • texturegroup_get_fonts
  • texturegroup_get_tilesets
