draw_texture_flush

使用此功能，您可以从视频内存中删除所有纹理，然后在首次使用时将其重新加载。当游戏中有多个纹理页面时，这是管理视频内存的唯一有效方法，因此应在游戏的各个级别之间刷新纹理内存，并使用纹理组功能组织图形，以确保最小数量的纹理用过的。

值得注意的是，默认情况下，所有目标上的纹理页面仅在需要时加载，并且您可以对sprite使用各种预取和刷新功能（在此处找到）来管理内容以及该功能。

句法：

draw_texture_flush();

返回值：

N/A

例子：

draw_texture_flush();

上面的代码刷新了纹理数据的视频存储器，可能会放置在放置在房间中的对象的第一个实例的create事件中。
