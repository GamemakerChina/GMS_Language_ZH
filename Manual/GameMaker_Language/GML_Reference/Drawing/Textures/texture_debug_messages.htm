texture_debug_messages

此功能可用于启用或禁用纹理调试消息。启用时（设置为 true），有关纹理页面使用的其他信息将发送到输出窗口。设置 false再次禁用此输出。

句法：

texture_debug_messages(enable);

争论描述
enable启用或禁用纹理调试消息

返回值：

N/A

例子：

if global.debug
    {
    texture_debug_messages(true);
    }

上面的代码将检查全局变量的值以及是否将其设置为 true然后启用纹理调试消息。
