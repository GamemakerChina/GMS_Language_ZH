texture_flush

此功能可用于“刷新”纹理页面或一组纹理页面，即：在不再需要时将它们从VRAM中删除。您提供了唯一的纹理页面ID （由 texturegroup_*函数）以刷新单个页面，或者您可以提供纹理组名称（在Texture Group Editor中定义）以刷新组中的所有纹理页面。

句法：

texture_flush(tex_id);

争论描述
tex_id纹理页面ID（整数）或纹理组ID（字符串）

返回值：

N/A

例子：

var _tex_array = texturegroup_get_textures( "MainMenu");
for (var i = 0; i < array_length(_tex_array); ++i;)
    {
    texture_flush(_tex_array[i]);
    }

上面的代码将刷新纹理组“ MainMenu”下的所有纹理页面。
