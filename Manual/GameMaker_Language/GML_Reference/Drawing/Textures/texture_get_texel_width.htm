texture_get_texel_width

纹理元素或纹理元素是计算机图形学中使用的纹理空间的基本单位。纹理由像素阵列表示，就像图片由像素阵列表示一样，此函数从使用的图像资产的纹理页面返回单个像素的宽度。

句法：

texture_get_texel_width(tex);

争论描述
tex要使用的纹理页面资产指针

返回值：

Real

例子：

var tex = sprite_get_texture(sprite_index, 0);
tex_w = texture_get_texel_width(tex);
tex_h = texture_get_texel_height(tex);

上面的代码将获取从Sprite资产获取的纹理的texel宽度和高度。
