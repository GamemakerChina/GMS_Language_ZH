texture_get_uvs

此函数返回一维数组，其中包含4个元素，代表纹理页面上图像的UV坐标，并使用以下值填充数组：

  • [0] =左
  • [1] =顶部
  • [2] =正确
  • [3] =底部
  • [4] =资产编译器从sprite左侧修剪的像素数量（仅sprite资产）
  • [5] =资产编译器从sprite顶部修剪的像素数量（仅sprite资产）
  • [6] =已保存到纹理页面的原始精灵宽度的标准化像素数据百分比（仅限精灵资产）
  • [7] =已保存到纹理页面的原始精灵高度的归一化像素数据百分比（仅精灵资产）

然后可以将此值用于其他绘图函数，特别是在通用3D和某些2D基本图元函数以及Shader函数中。如果您需要UVS来制作子画面，则可以使用 sprite_get_uvs()对于字体，font_get_uvs() 。

句法：

texture_get_uvs(texid)

争论描述
texid从中获取纹理的图像资产

返回值：

1D Array (4 - 8 elements)

例子：

var tex = texture_get_uvs(surf_back);
tex_left = tex[0];
tex_top = tex[1];
tex_right = tex[2];
tex_left = tex[3];

上面的代码将给定表面纹理的UV坐标存储在局部数组中，然后将这些值分配给实例变量。
