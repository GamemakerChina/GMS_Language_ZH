texture_global_scale

使用此功能，您可以控制作为最终游戏可执行文件一部分的WAD文件在加载时纹理页面的缩放比例。输入值必须是两个值的幂，并且必须满足以下条件：

  • 要将纹理页面设置为1：1，请使用 texture_global_scale(1)
  • 一直使用一半纹理 texture_global_scale(2)
  • 季度使用 texture_global_scale(4)， 等等...

这样，您可以在内存不足的平台上更好地控制纹理页面内存的使用。请注意，纹理页面是根据WAD的要求创建的，因此您可以在游戏开始时调用此页面以加载缩放后的所有纹理，也可以在特定时间调用它以加载特定的纹理，在这种情况下，您可以可能想打电话 draw_texture_flush()在调用此函数之前，请使用预取函数将正确的纹理页面存储到内存中。

句法：

texture_global_scale(pow2integer);

争论描述
pow2integer要使用的比例因子（1，无比例，2，半比例，4，四分之一比例等）

返回值：

N/A

例子：

draw_texture_flush();
texture_global_scale(2);
sprite_prefetch(spr_Trees);

上面的代码将从内存中清除所有纹理，然后将纹理缩放比例设置为2（因此纹理页面的大小为一半），然后最终预取其中包含精灵“ spr_Trees”的纹理页面。
