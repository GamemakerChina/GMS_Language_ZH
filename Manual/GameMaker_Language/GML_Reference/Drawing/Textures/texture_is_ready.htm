texture_is_ready

此功能可用于检查特定的纹理页面是否已打开包装并准备好使用，或一组纹理页面是否已打开包装并准备好使用。您提供了唯一的纹理页面ID （由 texturegroup_*函数）或纹理组ID字符串（在“纹理组编辑器”中定义），函数将返回 true如果已打开包装，或者 false否则。

句法：

texture_is_ready(tex_id);

争论描述
tex_id纹理页面ID（整数）或纹理组ID（字符串）

返回值：

Boolean

例子：

var _tex_array = texturegroup_get_textures( "MainMenu");
for (var i = 0; i < array_length(_tex_array); ++i;)
    {
    if !texture_is_ready(_tex_array[i])
        {
        texture_prefetch(_tex_array[i]);
        }
    }

上面的代码将检索纹理组“ MainMenu”的纹理页面ID，然后检查它们是否已解包，如果没有，则将它们预取到VRAM中。
