纹理_is_ready

这个函数可以用来检查一个特定的纹理页是否已经被解压并可以使用，或者一组纹理页是否已经被解压并可以使用。你提供唯一的纹理页ID（由texturegroup_*函数返回）或纹理组ID字符串（在Texture Group Editor中定义），如果它们已经被解压，该函数将返回true，否则返回false。

语法。

texture_is_ready(tex_id)。

争论 描述
tex_id 纹理页ID（一个整数）或纹理组ID（一个字符串）。

返回。

布尔型

例子。

var _tex_array = texturegroup_get_textures( "MainMenu");
for (var i = 0; i < array_length( _tex_array); ++i;)
{
if !texture_is_ready(_tex_array[i])
{
texture_prefetch(_tex_array[i])。
}
}

上面的代码将检索纹理组 "MainMenu "的纹理页ID，然后检查它们是否被解包，如果没有，它们将被预取到VRAM中。

返回。纹理
