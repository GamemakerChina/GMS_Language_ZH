纹理预取

这个函数可以用来 "预取 "一个纹理页或一组纹理页，即：在需要时将它们加载到 VRAM 中。你可以提供唯一的纹理页ID（由texturegroup_*函数返回）来预取一个单一的 页面，或者你可以提供一个纹理组名称（如在纹理组编辑器中定义的）来预取该组中的所有纹理页面。

语法。

texture_prefetch(tex_id)。

争论 描述
tex_id 纹理页ID（一个整数）或纹理组ID（一个字符串）。

返回。

不适用

例子。

var _tex_array = texturegroup_get_textures( "MainMenu");
for (var i = 0; i < array_length( _tex_array); ++i;)
{
texture_prefetch(_tex_array[i])。
}

上述代码将预取纹理组 "MainMenu "下的所有纹理页。

返回。纹理
