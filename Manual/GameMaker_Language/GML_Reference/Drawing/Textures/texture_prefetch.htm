texture_prefetch

此功能可用于“预取”一个纹理页面或一组纹理页面，即：在需要时将它们加载到VRAM中。您提供了唯一的纹理页面ID （由 texturegroup_*函数）以预取单个页面，或者可以提供纹理组名称（在Texture Group Editor中定义）以预取组中的所有纹理页面。

句法：

texture_prefetch(tex_id);

争论描述
tex_id纹理页面ID（整数）或纹理组ID（字符串）

返回值：

N/A

例子：

var _tex_array = texturegroup_get_textures( "MainMenu");
for (var i = 0; i < array_length(_tex_array); ++i;)
    {
    texture_prefetch(_tex_array[i]);
    }

上面的代码将预取纹理组“ MainMenu”下的所有纹理页面。
