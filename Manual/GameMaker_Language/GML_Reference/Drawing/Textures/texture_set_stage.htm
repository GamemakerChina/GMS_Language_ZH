纹理_set_stage

这个函数将为给定的舞台 "槽 "设置一个要使用的纹理。可用的阶段 "槽 "的数量取决于你所编译的平台，Windows、Mac和Linux最多有8个，但在低端的Android设备上（例如），这个数字可能低至2个。 值得注意的是，GameMakerStudio 2总是自动使用第一个阶段 "槽"。

注意：这个函数在运行中的着色器的上下文之外不会做任何事情参见 着色器 获取更多信息。

语法。

texture_set_stage(stage, tex)。

争论 描述
阶段 要使用的纹理 "槽"。
特克斯 要使用的纹理。

返回。

不适用

例子。

glassshader_bgsampler =shader_get_sampler_index(GlassShader, "s_BackgroundSampler")
spr = sprite_get_texture(sprite_index, 0);
shader_set(GlassShader) 。
texture_set_stage(glassshader_bgsampler, spr);
shader_reset()。

上面的代码将在索引为 "GlassShader "的着色器中获取采样器的句柄，然后将该着色器常量设置为给定的精灵纹理。

返回。纹理
