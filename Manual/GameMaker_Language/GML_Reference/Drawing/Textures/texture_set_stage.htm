texture_set_stage

此功能将在给定的舞台“插槽”中设置要使用的纹理。可用的阶段“插槽”数量取决于您要编译的平台， Windows ，Mac和Linux最多可以使用8个，但是在较低端的Android设备上（例如），该数量可以低至2个。 GameMaker Studio 2始终自动使用第一阶段的“插槽”。

注意：此功能在正在运行的shader的上下文之外不起作用！有关更多信息，请参见Shaders

句法：

texture_set_stage(stage, tex);

争论描述
stage要使用的纹理“插槽”。
tex使用的纹理。

返回值：

N/A

例子：

glassshader_bgsampler = shader_get_sampler_index(GlassShader, "s_BackgroundSampler")
spr = sprite_get_texture(sprite_index, 0);
shader_set(GlassShader);
texture_set_stage(glassshader_bgsampler, spr);
shader_reset();

上面的代码将在索引为“ Glass Shader shader获取采样器的句柄，然后将该shader常数设置为给定的sprite纹理。
