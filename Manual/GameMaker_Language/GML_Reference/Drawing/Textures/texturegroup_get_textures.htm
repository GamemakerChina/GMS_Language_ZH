texturegroup_get_textures

此功能可用于检索组成纹理组的各个页面的纹理页面ID。您提供纹理组ID字符串（在“纹理组编辑器”中定义），该函数将返回一维数组，其中数组中的每个条目都是单个纹理页面ID。如果函数失败-即：给了一个无效的组，或者该组没有分配纹理-那么该数组将为空（长度为0）。

句法：

texturegroup_get_textures(tex_id);

争论描述
tex_id要检查的纹理组的名称（字符串）

返回值：

1D array

例子：

var _tex_array = texturegroup_get_textures( "MainMenu");
for (var i = 0; i < array_length(_tex_array); ++i;)
    {
    if texture_is_ready(_tex_array[i])
        {
        texture_prefetch(_tex_array[i]);
        }
    }

上面的代码将检索纹理组“ MainMenu”的纹理页面ID，然后检查它们是否已解压缩，如果是，则将其放入VRAM。
