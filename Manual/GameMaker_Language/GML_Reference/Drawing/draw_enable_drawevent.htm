draw_enable_drawevent

使用此功能，您可以选择启用（true ）或禁用（false ）游戏中所有实例的绘制事件，从而使您可以控制事物绘制的方式和时间，如果您希望实现“跳帧”技术，则很有用。请注意，这不仅阻止实例在屏幕上绘制，而且还完全抑制了绘制事件，这意味着应格外小心，因为该事件中存在的任何游戏逻辑也不会运行。有关此功能的一件重要事情是，如果您在游戏开始时就调用它，在渲染初始帧之前（即：游戏第一个房间中第一个对象的Create Event），则该游戏窗口将不会呈现。这对于需要不需要渲染任何内容的服务器的在线项目很有用。

句法：

draw_enable_drawevent(enable);

争论描述
Enable设置 true或者 false。

返回值：

N/A

例子：

frame_skip ++;
if (frame_skip mod 5) == 0
    {
    draw_enable_drawevent(true);
    }
else
    {
    draw_enable_drawevent(false);
    }

上面的代码检查变量，如果返回 true，则启用绘图事件，否则将禁用绘图事件。
