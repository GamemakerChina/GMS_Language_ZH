二进制文件

在极少数情况下，您可能需要从二进制文件读取数据。这些函数可以用于此目的，尽管通常，我们建议您使用专用的缓冲区函数来创建，保存和加载二进制数据，而不是使用这些函数。

注：当目标模块是HTML5这些功能不起作用。

为此，存在以下低级例程：

  • file_bin_open
  • file_bin_rewrite
  • file_bin_close
  • file_bin_size
  • file_bin_position
  • file_bin_seek
  • file_bin_write_byte
  • file_bin_read_byte
