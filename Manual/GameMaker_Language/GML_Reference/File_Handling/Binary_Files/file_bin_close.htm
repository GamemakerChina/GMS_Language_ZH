file_bin_close

关闭一个先前打开的二进制文件。你提供文件的ID值，如函数所返回的 file_bin_open().

注意：当目标模块为HTML5时，这些功能不起作用。

语法。

file_bin_close(binfile)。

争论 描述
二进制文件 要关闭的文件的ID。

返回。

不适用

例子。

file = file_bin_open("myfile.bin", 2);
file_bin_rewrite(file);
file_bin_close(file)。

这将从与游戏相同的目录中打开一个文件，并将其索引分配给变量 "file"。然后，它将重写该文件（清除它的数据），并再次关闭它。

返回：二进制文件
