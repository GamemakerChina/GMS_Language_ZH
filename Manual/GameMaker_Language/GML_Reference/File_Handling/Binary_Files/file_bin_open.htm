file_bin_open

此功能将打开具有指定名称的二进制文件。该模式指示该文件可以做什么：

  • 0 =阅读
  • 1 =写作
  • 2 =读写

当文件不存在时，将创建该文件，并且该函数返回在其他函数中必须使用的文件的ID。您可以同时打开多个文件（最多32个），但一旦完成操作，别忘了关闭它们。

警告！ GameMaker Studio 2沙盒化，因此此功能可能无法按预期工作！请参阅文件系统部分以获取更多信息。

注：当目标模块是HTML5这些功能不起作用。

句法：

file_bin_open(fname, mode);

争论描述
fname要读取的文件的名称。
mode该文件可以做什么的指示符。

返回值：

Real

例子：

file = file_bin_open("myfile.bin", 2);

这将从与游戏相同的目录中打开文件，并将其索引分配给变量“ file”。

返回：二进制文件
