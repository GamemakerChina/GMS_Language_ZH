file_bin_position

此函数将返回具有给定文件ID的文件的当前位置（以字节为单位），其中0是第一个位置。您提供函数返回的文件ID值 file_bin_open()。

注：当目标模块是HTML5这些功能不起作用。

句法：

file_bin_position(binfile);

争论描述
binfile要获取位置的文件的ID。

返回值：

Real

例子：

pos = file_bin_position(file);

这会将当前位置存储在变量“ pos”中。

返回：二进制文件
