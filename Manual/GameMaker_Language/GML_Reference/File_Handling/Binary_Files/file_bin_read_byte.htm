file_bin_read_byte

该函数将从文件中具有给定文件ID的当前位置返回一个字节的数据。您提供函数返回的文件ID值 file_bin_open()。

注：当目标模块是HTML5这些功能不起作用。

句法：

file_bin_read_byte(binfile);

争论描述
binfile从中获取数据的文件的ID。

返回值：

Real

例子：

file = file_bin_open("myfile.bin", 2);
data = file_bin_read_byte(file);
file_bin_close(file);

这将从与游戏相同的目录中打开文件，并在再次关闭文件之前从文件中获取一个字节的数据。

返回：二进制文件
