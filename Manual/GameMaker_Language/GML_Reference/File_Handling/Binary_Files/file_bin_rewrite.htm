file_bin_rewrite

此函数采用函数返回的文件名句柄 file_bin_open()然后重写文件，清除所有先前的数据，以从文件开头开始写入。

注：当目标模块是HTML5这些功能不起作用。

句法：

file_bin_rewrite(binfile);

争论描述
binfile要重写的文件的ID。

返回值：

N/A

例子：

file = file_bin_open("myfile.bin", 2);
file_bin_rewrite(file);

这将从与游戏相同的目录中打开文件，并将其索引分配给变量“ file”。然后它将重新写入文件（清除数据）。

返回：二进制文件
