file_bin_rewrite

这个函数接收由函数返回的文件名句柄 file_bin_open()返回的文件名句柄，然后重写该文件，清除之前的所有数据，从文件的开头开始写。

注意：当目标模块为HTML5时，这些功能不起作用。

语法。

file_bin_rewrite(binfile)。

争论 描述
二进制文件 要重写的文件的ID。

返回。

不适用

例子。

file = file_bin_open("myfile.bin", 2);
file_bin_rewrite(file)。

这将从与游戏相同的目录中打开一个文件，并将其索引分配给变量 "file"。然后，它将重写该文件（清除它的数据）。

返回：二进制文件
