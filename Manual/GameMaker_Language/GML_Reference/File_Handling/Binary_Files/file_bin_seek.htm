file_bin_seek

此功能将文件中的当前读取位置移动到指示的位置。您提供函数返回的文件ID值 file_bin_open()，并要追加文件，请在写入之前将位置移动到文件的大小。

注：当目标模块是HTML5这些功能不起作用。

句法：

file_bin_seek(binfile, pos);

争论描述
binfile要读取的文件的ID。
pos文件中要移动到的位置。

返回值：

N/A

例子：

file = file_bin_open("myfile.bin", 2);
size = file_bin_size(file);
file_bin_seek(file, size);

这将从与游戏相同的目录中打开文件，并将其索引分配给变量“ file”，然后获取文件的大小并将下一个写入位置设置为该大小。

返回：二进制文件
