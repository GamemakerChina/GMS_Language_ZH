file_bin_size

这个函数将返回一个已经被打开进行读写的文件的大小（以字节为单位）。文件ID是由该函数返回的值 file_bin_open().

注意：当目标模块为HTML5时，这些功能不起作用。

语法。

file_bin_size(binfile)。

争论 描述
二进制文件 要获取文件大小的ID。

返回。

真实的

例子。

file = file_bin_open("myfile.bin", 2);
size = file_bin_size(file);
file_bin_close(file)。

这将从游戏的本地目录中打开一个文件，并将其索引分配给变量 "file"。然后，它将获得该文件的大小，并再次关闭它。

返回：二进制文件
