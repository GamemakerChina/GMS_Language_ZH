file_bin_size

此函数将返回已打开以进行读取和/或写入的文件的大小（以字节为单位）。文件ID是函数返回的值 file_bin_open()。

注：当目标模块是HTML5这些功能不起作用。

句法：

file_bin_size(binfile);

争论描述
binfile要获取大小的文件的ID。

返回值：

Real

例子：

file = file_bin_open("myfile.bin", 2);
size = file_bin_size(file);
file_bin_close(file);

这将从游戏的本地目录中打开一个文件，并将其索引分配给变量“ file”。然后，它将获取该文件的大小并再次关闭它。

返回：二进制文件
