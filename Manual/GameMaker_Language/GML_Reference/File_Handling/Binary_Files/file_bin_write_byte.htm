file_bin_write_byte

此函数将在当前写入位置将一个字节的数据写入由文件ID标识的文件。您提供函数返回的文件ID值 file_bin_open()以及要写入的数据字节。

注：当目标模块是HTML5这些功能不起作用。

句法：

file_bin_write_byte(binfile, byte);

争论描述
binfile要写入的文件的ID。
byte要写入的数据。

返回值：

real

例子：

file_bin_write_byte(file, data);

这将向所选文件写入一个字节。

返回：二进制文件
