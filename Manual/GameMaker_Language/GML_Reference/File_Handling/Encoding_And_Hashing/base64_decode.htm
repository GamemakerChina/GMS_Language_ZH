base64_decode

此函数会将以前使用base64格式编码的字符串转换为标准文本。 Base64是一种常用的编码方案，通常用于需要以文本形式在互联网上存储或传输的任何媒体，并使输出无法被人眼识别。

句法：

base64_decode(string)

争论描述
string要解码的字符串。

返回值：

String

例子：

var str, file;
str = base64_encode(game_data);
file = file_text_open_read("save.txt");
str = file_text_read_string(file); level_data = base64_decode(str);
file_text_close(file);

上面的代码将打开一个文本文件，并将其中的字符串读入局部变量“ str”。然后对该字符串进行解码，并将结果存储在实例变量“ level_data”中。
