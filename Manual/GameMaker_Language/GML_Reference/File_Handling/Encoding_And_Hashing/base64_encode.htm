base64_encode

此函数会将字符串转换为以base64格式编码的字符串。这是一种常用的编码方案，通常用于需要以文本形式在互联网上存储或传输的任何媒体，并使输出无法被人眼识别。

句法：

base64_encode(string)

争论描述
string要编码的字符串。

返回值：

String

例子：

var str, file;
str = base64_encode(game_data);
file = file_text_open_write("save.txt");
file_text_write_string(file, str);
file_text_close(file);

上面的代码会将存储在“ game_data”中的字符串转换为以base64编码的字符串，然后将其存储在外部文本文件中。
