load_csv

此函数将加载CSV格式的文件并将其转换为DS网格，并为创建的网格返回唯一的ID值。

句法：

load_csv(filename)

争论描述
filename要打开的文件名（以字符串形式）

返回值：

DS Grid Index (real)

例子：

file_grid = load_csv("spreadsheet.csv");
var ww = ds_grid_width(file_grid);
var hh = ds_grid_height(file_grid);
var xx = 32;
var yy = 32;
for (var i = 0; i < ww; i++;)
    {
    for (var j = 0; j < hh; j++;)
        {
        draw_text(xx, yy, string(file_grid[# i, j]));
        yy += 32;
        }
    yy = 32;
    xx += 32;
    }

上面的代码将打开给定的CSV文件，并将返回的DS网格存储在变量“ file_grid”中。然后将此网格解析为几个 for镜头，然后将内容绘制到屏幕上。
