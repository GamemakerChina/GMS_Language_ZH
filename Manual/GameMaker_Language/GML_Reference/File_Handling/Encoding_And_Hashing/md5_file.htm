md5_file

在密码学中，MD5（消息摘要算法5）是具有128位哈希值的广泛使用的密码哈希函数，已被广泛用于各种安全应用程序中。它也通常用于检查文件和字符串的完整性。此功能将获取给定的文件，并为该文件生成唯一的MD5，然后将其存储以供以后使用。

句法：

md5_file(filename)

争论描述
filename为其生成MD5哈希的文件。

返回值：

String

例子：

hash = md5_file(working_directory + "game_data.ini")

上面的代码将为指定的文件生成MD5哈希，并将返回的值存储在变量“哈希”中。
