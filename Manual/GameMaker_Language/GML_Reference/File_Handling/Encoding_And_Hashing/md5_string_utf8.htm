md5_string_utf8

在密码学中，MD5（消息摘要算法5）是具有128位哈希值的广泛使用的密码哈希函数，已被广泛用于各种安全应用程序中。它也通常用于检查文件和字符串的完整性。此函数将使用输入的UTF-8字符串（每个字符具有可变的字节数），并返回该字符串唯一的32个字符的十六进制MD5哈希。这样，您可以生成一个安全密钥，该密钥可以存储并用于检查发送到（例如）外部服务器（或从外部服务器接收）的信息的完整性。

注意：MD5编码有两种格式，即UTF-8和unicode。提供两者都是为了促进与不同服务器设置的通信，但是最常用的是unicode。

句法：

md5_string_utf8(string)

争论描述
string要哈希的字符串。

返回值：

String

例子：

var hash, str;
str = json_encode(hiscore_map);
hash = md5_string_utf8(str);
ini_open("local.ini");
ini_write_string("info", "0", hash);
ini_close();
get[0] = http_post_string("http://www.MacSweeney Games.com/CatchTheHaggis?game_hiscores=" + string(global.game_id), str)

上面的代码会将DS映射编码为JSON字符串。然后生成MD5哈希并将其存储在ini文件中，以便以后可从服务器接收到相同信息时，将其用于以后检查JSON的完整性。然后发送JSON。
