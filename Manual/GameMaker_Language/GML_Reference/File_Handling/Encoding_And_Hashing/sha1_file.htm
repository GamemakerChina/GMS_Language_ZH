sha1_file

在加密技术中，SHA-1是一种由美国国家安全局设计的加密哈希函数，并在几种广泛使用的流行Git应用程序中使用，在该应用程序中它用于检查文件更改以及协议TLS和SSL，PGP，SSH， S / MIME和IPsec。此函数将获取一个输入文件，并以该文件唯一的ASCII格式返回一个160位的消息摘要，以便以后用于完整性验证。

句法：

sha1_file(filename)

争论描述
filename为其生成sha1哈希的文件。

返回值：

String

例子：

hash = sha1_file(working_directory + "game_data.ini")

上面的代码将为指定文件生成一个sha1哈希，并将返回的值存储在变量“哈希”中。
