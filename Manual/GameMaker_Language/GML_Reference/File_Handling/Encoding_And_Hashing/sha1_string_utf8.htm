sha1_string_utf8

在加密技术中，SHA-1是由美国国家安全局设计的加密哈希函数，已在多种广泛使用的应用程序和协议中使用，例如流行的Git ，用于检查文件更改。此函数将采用输入的UTF-8字符串（每个字符具有可变的字节数），并以ASCII格式返回160位消息摘要，例如：

sha1 input string = "The quick brown fox jumps over the lazy dog"
sha1 output hash = "2fd4e1c6 7a2d28fc ed849ee1 bb76e739 1b93eb12"

这样，您可以生成一个安全密钥，该密钥可以存储并用于检查发送到（例如）外部服务器（或从外部服务器接收）的信息的完整性。

注意：sha1编码有两种格式，即UTF-8和unicode。提供两者都是为了促进与不同服务器设置的通信，但是最常用的是unicode。

句法：

sha1_string_utf8(string)

争论描述
string要哈希的字符串。

返回值：

String

例子：

var hash, str;
str = json_encode(hiscore_map);
hash = sha1_string_utf8(str);
ini_open("local.ini");
ini_write_string("info", "0", hash);
ini_close();
get[0] = http_post_string("http://www.MacSweeney Games.com/CatchTheHaggis?game_hiscores=" + string(global.game_id), str)

上面的代码会将DS映射编码为JSON字符串。然后会生成一个sha1哈希，并将其存储在ini文件中，以便以后在从服务器接收到相同信息时，可将其用于检查JSON的完整性。然后发送JSON。
