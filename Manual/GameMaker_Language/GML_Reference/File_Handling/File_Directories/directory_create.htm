directory_create

此函数将在保存区域中创建具有给定名称的目录。

注意：此功能不适用于HTML5，因为您无法在浏览器本地存储中创建或删除目录。

警告！ GameMaker Studio 2沙盒化，因此此功能可能无法按预期工作！请参阅文件系统部分以获取更多信息。

句法：

directory_create(dname)

争论描述
dname要创建的目录的名称。

返回值：

N/A

例子：

if !directory_exists("Games")
   {
   directory_create("Games");
   }

这将检查本地数据文件夹中是否存在指定的目录，如果不存在，则会为您创建该目录。

返回：文件目录
