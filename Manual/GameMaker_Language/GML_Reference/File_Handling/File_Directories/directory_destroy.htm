directory_destroy

此功能将在保存区域中删除具有给定名称的目录。

注意：此功能不适用于HTML5，因为您无法在浏览器本地存储中创建或删除目录。

警告！ GameMaker Studio 2沙盒化，因此此功能可能无法按预期工作！请参阅文件系统部分以获取更多信息。

句法：

directory_destroy(dname)

争论描述
dname要删除的目录的名称。

返回值：

N/A

例子：

if directory_exists("DLC")
   {
   directory_destroy("DLC");
   }

这将检查指定的目录是否存在于本地数据文件夹中，如果存在，则将其删除。

返回：文件目录
