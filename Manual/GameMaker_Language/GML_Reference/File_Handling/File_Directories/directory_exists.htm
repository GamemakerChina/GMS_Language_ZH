directory_exists

该函数将返回 true如果指示的目录存在或 false如果没有。指定的名称必须包含完整路径，而不是相对路径，并且默认情况下，由于所有游戏都被沙箱化，因此您无法从游戏捆绑包外部访问任何目录（有关更多信息，请参见文件系统部分）。

句法：

directory_exists(dname)

争论描述
dname要查找的目录的名称。

返回值：

Boolean

例子：

if directory_exists(working_directory + "Saves/")
   {
   file = file_find_first(working_directory + "Saves/*.doc", fa_readonly);
   }

这将检查指定的目录是否存在，如果存在，则转到该目录并返回第一个“只读”doc找到文件。

返回：文件目录
