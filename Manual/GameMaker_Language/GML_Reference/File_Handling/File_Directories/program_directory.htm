程序_目录

这将返回游戏可执行文件的存储目录。然而，这可能并不总是有用的，特别是一些设备从*.zip文件中运行exe，所以无论游戏从哪里实际运行，这都会返回相同的结果。

警告!由于GameMaker Studio 2是沙盒式的，这个功能可能不会像你预期的那样工作!请参阅文件系统部分以了解更多信息。

注意：如果你禁用了沙盒，那么你仍然不能写到这个目录，以防止任何可能的删除所需游戏文件的问题。

语法。

程序_目录

返回。

字符串

例子。

dir = program_directory。

这将把可执行文件所在的目录存储在一个变量中。
