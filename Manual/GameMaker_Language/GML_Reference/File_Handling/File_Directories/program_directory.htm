program_directory

这将返回游戏可执行文件的存储目录。但是，这可能并不总是有用的，尤其是当某些设备从*.zip文件运行exe时，因此无论游戏实际在何处运行，它都将返回相同的结果。

警告！ GameMaker Studio 2沙盒化，因此此功能可能无法按预期工作！请参阅文件系统部分以获取更多信息。

注意：如果禁用沙箱，则仍将无法写入此目录，以防止删除必需的游戏文件时出现任何可能的问题。

句法：

program_directory

返回值：

String

例子：

dir = program_directory;

这将在可执行文件中存储变量的目录。

返回：文件目录
