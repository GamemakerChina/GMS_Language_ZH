temp_directory

每次运行游戏时，都可以使用它返回为您的游戏创建的临时目录（根目录不包含最终的“ \”）。该目录将保存文件，并且在游戏运行时可以访问该目录，但是在游戏关闭时，该目录将被删除（以及包含的所有文件）。

警告！ GameMaker Studio 2沙盒化，因此此功能可能无法按预期工作！请参阅文件系统部分以获取更多信息。

句法：

temp_directory

返回值：

String

例子：

ini_open(temp_directory + "\temp_ini.ini");

这将在游戏的临时目录中打开一个ini文件（如果尚不存在，则创建该文件）。
