文件系统

在GameMaker Studio 2中存在一些功能，使你能够以不同的方式处理各种类型的文件。

  • 文件存在
  • 文件删除
  • 文件重命名
  • 文件拷贝
  • 文件_搜索_第一
  • 文件_查找下一个
  • 文件_找寻_关闭
  • 文件属性

以下功能可用于改变文件名。

注意：这些函数并不对实际的文件起作用！它们只处理与不同路径元素有关的字符串。它们只处理与不同路径元素有关的字符串，应该与其他文件函数一起使用。

  • 文件名_名称
  • 文件名_路径
  • 文件名_dir
  • 文件名_drive
  • 文件名_ext
  • 文件名_改变_文本

最后，有一些特殊的函数用于获取文件名，可以用文件类型过滤器加载和保存文件。这些函数将打开一个资源管理器窗口，并因此导致游戏失去焦点。这意味着，游戏所依赖的任何输入 因此，我们建议（如果在键盘事件中使用这些函数）使用向上键事件，而不是按下或放下的事件。

注意：这些功能是 窗口系统和 MacOS仅供参考。

警告!由于GameMaker Studio 2是沙盒式的，这些功能可能不会像你预期的那样工作!请参阅文件系统一节以了解 更多信息。

  • get_open_filename
  • get_open_filename_ext
  • get_save_filename
  • get_save_filename_ext
