文件系统

GameMaker Studio 2存在许多功能，可让您以不同的方式操作所有类型的文件：

  • file_exists
  • file_delete
  • file_rename
  • file_copy
  • file_find_first
  • file_find_next
  • file_find_close
  • file_attributes

以下功能可用于更改文件名：

注意：这些功能不适用于实际文件！它们仅处理与不同路径元素相关的字符串，应与其他文件功能一起使用。

  • filename_name
  • filename_path
  • filename_dir
  • filename_drive
  • filename_ext
  • filename_change_ext

最后，还有一些用于获取文件名的特殊功能，可用于通过文件类型过滤器加载和保存文件。这些功能将打开一个浏览器窗口，并因此导致游戏失去焦点。这意味着游戏所依赖的任何输入都可能无法正确注册，因此，我们建议（如果在键盘事件中使用了这些功能）则使用按键向上事件，而不要使用按下或按下事件。

注意：这些功能仅Windows和macOS 。

警告！ GameMaker Studio 2沙盒化，因此这些功能可能无法按预期工作！请参阅文件系统部分以获取更多信息。

  • get_open_filename
  • get_open_filename_ext
  • get_save_filename
  • get_save_filename_ext
