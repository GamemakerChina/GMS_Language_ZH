file_attributes

您可以使用此功能检查给定文件的属性。您可以添加以下常量以查看所需的文件类型：

持续的描述
fa_readonly只读文件
fa_hidden隐藏文件
fa_sysfile系统档案
fa_volumeid卷ID文件
fa_directory目录
fa_archive存档文件

注意：这是Windows功能。

句法：

file_attributes(fname, attr);

争论描述
fname要检查的文件名。
attr要检查的属性。

返回值：

Boolean

例子：

if !file_attributes(file, fa_hidden)
    {
    file_delete(file);
    }

这将检查文件以查看其是否隐藏，如果没有，则将其删除。

背面：文件系统
