file_copy

此功能将复制指定的文件，并为其提供您选择的新名称。如果您要从“包含的文件” Android上复制文件，它将首先复制到内存中，因此请注意，从APK复制非常大的文件可能会导致设备内存不足。在HTML5目标上，我们使用HTML5本地存储API进行客户端保存，并且该功能不支持二进制文件操作，因此假设您正在复制文本文件。

警告！ GameMaker Studio 2沙盒化，因此此功能可能无法按预期工作！请参阅文件系统限制部分以获取更多信息。

句法：

file_copy(fname, newname);

争论描述
fname要复制的文件名。
newname新的复制文件的名称。

返回值：

N/A

例子：

if file_exists("level1.txt")
    {
    file_copy("level1.txt", "level2.txt");
    }

这将检查文件，如果存在则将其复制。
