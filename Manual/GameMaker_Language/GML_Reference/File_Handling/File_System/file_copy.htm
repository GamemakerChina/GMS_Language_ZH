文件拷贝

这个函数将复制指定的文件，给它一个你选择的新名字。如果你是在你的包含文件区（将在你的APK内）复制一个文件 安卓从你的包含文件区（将在你的APK内）复制文件，它将首先复制到内存中，所以，请注意 从APK中复制非常大的文件可能会导致设备的内存耗尽。在HTML5目标上，我们使用HTML5本地存储API进行客户端保存，该功能不支持二进制文件操作，所以假设你是在 复制一个文本文件。

警告!由于GameMaker Studio 2是沙盒式的，这个功能可能不会像你所期望的那样工作!请参阅文件系统限制一节以了解更多信息。

语法。

file_copy(fname, newname)。

争论 描述
fname 要复制的文件的名称。
新名称 新的、复制的文件的名称。

返回。

不适用

例子。

if file_exists("level1.txt")
{
file_copy("level1.txt", "level2.txt");
}

这将检查一个文件，如果它存在，就复制它。

返回。文件系统
