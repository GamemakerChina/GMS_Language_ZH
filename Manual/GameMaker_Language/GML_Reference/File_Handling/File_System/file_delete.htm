file_delete

此功能将从系统中删除指定的文件。应该注意的是，此功能只会删除GameMaker Studio 2能够创建和解析的文件：ini文件，文本文件和二进制文件，或者用于存储游戏创建的资源（如精灵或曲面）的文件。但是，它不会删除任何其他文件。该函数还将返回 true如果文件已成功删除，或者 false在任何其他情况下。

警告！ GameMaker Studio 2沙盒化，因此此功能可能无法按预期工作！请参阅文件系统限制部分以获取更多信息。

句法：

file_delete(fname);

争论描述
fname要删除的文件名。

返回值：

Boolean

例子：

if file_exists("level.txt")
    {
    file_delete("level.txt");
    }

这将检查文件，如果存在则将其删除。
