file_exists

该函数将返回 true如果指定的文件存在，并且 false如果没有。请注意，该功能只能用于检查本地文件，而不能用于检查存储在远程服务器上的任何文件。

句法：

file_exists(fname);

争论描述
fname要检查的文件名。

返回值：

Boolean

例子：

if file_exists("level.txt")
    {
    file = file_text_open_read("level.txt");
    }

这将检查文件，如果存在则将其打开以进行读取。

背面：文件系统
