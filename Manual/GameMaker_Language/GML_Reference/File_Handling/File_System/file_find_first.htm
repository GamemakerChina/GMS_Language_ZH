文件_搜索_第一

这个函数将返回符合掩码和属性的第一个文件的名称。如果没有这样的文件存在，则返回一个空字符串。掩码可以包含一个路径，也可以包含通配符，例如'C:\temp\*.doc'。该 属性给出了你想看到的额外文件，所以正常的文件在满足掩码的情况下总是会被返回。你可以把下面的常数加起来，以看到你想要的文件类型（如果你不希望添加任何属性，就用0）。

恒定 描述
fa_readonly 只读文件
fa_hidden 隐藏的文件
fa_sysfile 系统文件
fa_volumeid 卷标文件
fa_directory 目录
基金会 归档文件

警告!由于GameMaker Studio 2是沙盒式的，这个功能可能不会像你预期的那样工作!请参阅文件系统部分以了解更多信息。

注意：这个功能在HTML5目标上完全不起作用，过滤标志只在Windows上起作用（其他平台都使用0）。

语法。

file_find_first(mask, attr)。

争论 描述
面罩 搜索时使用的面具。
符号（attr 要寻找的具体文件属性。

返回。

字符串

例子。

if directory_exists("/User Content")
{
file = file_find_first("/User Content/*.doc", fa_readonly);
}

这将检查指定的目录是否存在，如果存在，就去那里，并返回找到的第一个 "只读 "文档文件。
