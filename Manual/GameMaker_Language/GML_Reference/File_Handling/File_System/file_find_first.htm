file_find_first

此函数将返回满足掩码和属性的第一个文件的名称。如果不存在这样的文件，则返回一个空字符串。掩码可以包含路径，也可以包含通配符，例如'C：\ temp \ *。doc'。这些属性为您提供了您想要查看的其他文件，因此普通文件在满足遮罩时始终会返回。您可以添加以下常量以查看所需的文件类型（如果您不希望添加任何属性，请使用0）：

持续的描述
fa_readonly只读文件
fa_hidden隐藏文件
fa_sysfile系统档案
fa_volumeid卷ID文件
fa_directory目录
fa_archive存档文件

警告！ GameMaker Studio 2沙盒化，因此此功能可能无法按预期工作！请参阅文件系统部分以获取更多信息。

注意：此功能根本无法在HTML5目标上使用，并且过滤器标志仅在Windows （对于所有其他平台，请使用0）。

句法：

file_find_first(mask, attr);

争论描述
mask用于搜索的掩码。
attr要查找的特定文件属性。

返回值：

String

例子：

if directory_exists("\User Content")
    {
    file = file_find_first("/User Content/*.doc", fa_readonly);
    }

这将检查指定的目录是否存在，如果存在，则转到该目录并返回第一个“只读”doc找到文件。
