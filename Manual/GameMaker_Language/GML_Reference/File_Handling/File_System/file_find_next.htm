file_find_next

此函数返回满足先前给定掩码和属性（由定义的下一个文件）的下一个文件的名称。file_find_first() ）。如果不存在这样的文件，则返回空字符串。

警告！ GameMaker Studio 2沙盒化，因此此功能可能无法按预期工作！请参阅文件系统部分以获取更多信息。

注意：此功能在所有C ++目标平台（ Windows ，Mac， iOS ， Android和WinPhone）上均可使用，但过滤器标志仅在Windows上适用。

句法：

file_find_next();

返回值：

String

例子：

file = file_find_next();

这将返回与先前给定属性匹配的下一个文件。
