文件_查找下一个

该函数返回符合先前给定的掩码和属性（由 file_find_first()).如果不存在这样的文件，则返回空字符串。

警告!由于GameMaker Studio 2是沙盒式的，这个功能可能不会像你预期的那样工作!请参阅文件系统部分以了解更多信息。

注意：这个功能可以在所有的C++目标平台（Windows、Mac、iOS、Android和WinPhone）上使用，但过滤标志只在Windows上使用。

语法。

file_find_next()。

返回。

字符串

例子。

file = file_find_next()。

这将返回下一个与先前给定属性相匹配的文件。
