file_rename

此函数将使用指定的名称重命名指定的文件。该函数将返回 true如果文件已成功重命名，或者 false在任何其他情况下。

警告！ GameMaker Studio 2沙盒化，因此此功能可能无法按预期工作！请参阅文件系统部分以获取更多信息。

句法：

file_rename(oldname, newname);

争论描述
oldname要更改的文件名。
newname赋予文件的新名称。

返回值：

Boolean

例子：

if file_exists("level1.txt")
    {
    file_rename("level1.txt", "level.txt");
    }

这将检查文件，如果存在，则将其重命名。
