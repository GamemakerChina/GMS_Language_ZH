filename_change_ext

该函数返回指示的文件名，并将扩展名（包括点）更改为新的扩展名。通过使用空字符串作为新扩展名，您可以一起删除扩展名部分。

句法：

filename_change_ext(fname, newext);

争论描述
fname要使用的文件。
newext要使用的新扩展名。

返回值：

String

例子：

ext = filename_change_ext(file_find_first(working_directory + "*.*", 0), "");

上面的代码获取删除扩展部分的文件名（作为字符串）。
