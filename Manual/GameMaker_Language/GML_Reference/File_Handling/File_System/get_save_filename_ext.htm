get_save_filename_ext

此功能将打开一个对话框，并要求播放器提供要使用给定过滤器保存的文件名。过滤器的形式为“ name1 | mask1 | name2 | mask2 |... ”，其中，掩码可能包含不同的选项，并且选项之间以分号分隔，您也可以使用“ *”来表示任何字符串。例如：“ bitmaps | *.bmp; *。wmf ”，将要求用户从系统中选择位图文件的名称，但仅选择具有两个指定扩展名的文件（bmp和 png）。这一功能并不保存文件本身！它仅返回带有文件全名和路径的字符串。如果用户按下“取消”，则返回一个空字符串“”。您也可以添加目录以开始搜索，并在窗口顶部显示标题，而不是默认标题。

使用此功能时要注意的重要一点是，它会在游戏期间为您授予该文件某些权限。所以，一旦你从用户那里得到的文件路径，你可以一次又一次地访问它，而不必问，和权限适用于阅读和写作相等。

注意：此功能仅在Windows和macOS目标上有效。

句法：

get_save_filename_ext(filter, fname, directory, caption);

争论描述
filter文件类型过滤器和/或名称。
fname建议使用的文件名。
directory从中开始搜索的目录。
caption窗口的标题。

返回值：

String

例子：

var file;
file = get_save_filename_ext("screenshot|*.png", "", working_directory, "Save a level here");
if file != ""
    {
    screen_save(file);
    }

上面的代码将要求用户选择要保存在工作目录中的图像文件名文件，然后检查返回的文件名和路径。
