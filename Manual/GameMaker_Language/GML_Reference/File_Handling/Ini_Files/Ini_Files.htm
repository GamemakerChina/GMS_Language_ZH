Ini文件

INI文件是小型，轻便的文件，与大多数平台兼容。它们是存储小信息的理想选择，例如界面首选项，本地高分，关卡数据等。因此，这是您开始从游戏中保存信息的简便快捷的方法。但是，值得注意的是，它们确实对它们可以存储的数据有限制，最大为64KB，因此，如果要保存诸如数据结构之类的内容，则可能希望查看其他一些文件功能，如Text文件。

重要的！所有GameMaker Studio 2项目都有一个名为“ options.ini ”的文件，因此您不能使用该名称命名任何文件（尝试编译游戏项目时会出现uid错误）。

请注意，对于具有本地化版本的游戏，从外部文件中读取带重音的字母非常重要。这意味着您应该首先以UTF8格式创建ini文件，然后将其GameMaker Studio 2 ，以便在运行游戏时将其导出并代替由ini创建的默认ANSI格式ini文件使用。以前未提供任何功能时，该功能起作用。这样，您可以在保持所有重音和非罗马字母的情况下正确地对其进行读写。

ini文件存在以下功能：

  • ini_open
  • ini_close
  • ini_write_real
  • ini_write_string
  • ini_read_real
  • ini_read_string
  • ini_key_exists
  • ini_section_exists
  • ini_key_delete
  • ini_section_delete
  • ini_open_from_string
