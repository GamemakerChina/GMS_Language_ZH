ini_close

完成对任何打开的ini文件的读取或写入后，应立即调用此函数。如果在使用任何ini写入函数后不使用该函数，则不会将任何内容写入磁盘，因为文件信息会保留在内存中，直到调用此函数为止，这会强制执行写入操作。如果您尝试打开一个ini而不先关闭另一个ini（或同一个ini），您也会收到错误消息。

该函数还将返回一个字符串，其中包含编码为ini文件的ini文件。然后可以将该字符串保存到服务器和/或与功能一起再次使用 ini_open_from_string()重新创建ini。

句法：

ini_close();

返回值：

String

例子：

ini_open("savedata.ini");
score = ini_read_real("save1", "score", 0);
ini_close();

这将打开“ savedata.ini”并将分数设置为“ save1”部分下的值，并在其中带有键“ score”，然后再次关闭.ini。如果“ save1”，“ score”下没有任何值，或者不存在“ savedata.ini”文件，则将score设置为0，并创建一个新的ini_file。

返回： Ini文件
