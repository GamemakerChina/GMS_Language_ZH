ini_key_delete

使用此功能，您可以从ini文件中删除所选的键（及其对应的值）。

句法：

ini_key_delete(section, key);

争论描述
section要从中删除密钥的部分。
key删除键。

返回值：

N/A

例子：

ini_open("savedata.ini");
ini_write_real("save1","Score",score);
ini_key_delete("save1","Score");
ini_close();

本示例将打开“ savedata.ini”，并将值写入“ save1”>“ Score”。然后，它将删除“分数”键并关闭.ini文件。

返回： Ini文件
