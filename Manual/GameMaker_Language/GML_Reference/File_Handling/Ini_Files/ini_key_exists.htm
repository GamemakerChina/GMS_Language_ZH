ini_key_exists

这个函数检查一个键是否存在于当前打开的ini中，如果存在，将返回true，否则返回false。这不是防止错误的必要检查，因为当一个键不存在时，从一个不存在的键读取 将只返回一个默认值。然而，它对于查看一个ini文件是否保存了特定的数据和其他一些东西是有用的。

语法。

ini_key_exists(section, key)。

争论 描述
节 在打开的.ini文件中，要检查一个键的部分。
钥匙 要检查的关键。

返回。

布尔型

例子。

ini_open("saveata.ini");
if !ini_key_exists("save1", "name")
{
global.name = "Player1";
}
ini_close()。

如果不存在 "name "这样的键，这将把变量global.name设置为 "Player1"。
