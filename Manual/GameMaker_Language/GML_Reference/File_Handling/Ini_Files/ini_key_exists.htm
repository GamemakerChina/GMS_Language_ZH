ini_key_exists

此函数检查当前打开的ini中是否存在键，并将返回 true如果有或者 false否则。这不是防止错误的必要检查，因为当某个键不存在时，从不存在的键中读取将仅返回默认值。但是，查看ini文件是否保存了特定数据和其他一些信息可能会很有用。

句法：

ini_key_exists(section, key);

争论描述
section打开的.ini文件中用于签入的部分。
key要检查的钥匙。

返回值：

Boolean

例子：

ini_open("savedata.ini");
if !ini_key_exists("save1", "name")
    {
    global.name = "Player1";
    }
ini_close();

如果不存在诸如“名称”之类的键，这会将变量global.name设置为“ Player1”。

返回： Ini文件
