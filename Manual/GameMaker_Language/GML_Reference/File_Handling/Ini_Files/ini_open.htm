ini_open

这将打开一个ini_file进行读取和/或写入。如果您要检查的位置不存在GameMaker Studio 2可能会创建一个，但前提是您要向其中写入数据。如果你只读从ini文件中的信息，然后读取功能的默认值将被退回，但INI文件实际上不会被创建。

请注意，您一次只能打开一个ini文件，请记住使用 ini_close()一旦您完成了对.ini文件的读取/写入操作，因为该信息直到那时才真正存储到磁盘中（它也存储在内存中，直到该文件关闭）。

警告！ GameMaker Studio 2沙盒化，因此此功能可能无法按预期工作！请参阅文件系统部分以获取更多信息。

句法：

ini_open(name);

争论描述
name.ini文件的文件名。

返回值：

N/A

例子：

ini_open("Settings/savedata.ini");
score = ini_read_real("save1", "score", 0);
ini_close();

这将打开“ savedata.ini”并读取其中带有键“ score”的“ save1”部分下的得分值，然后再次关闭.ini。如果“ save1”，“ score”下没有任何值，或者不存在“ savedata.ini”文件，则得分将设置为0（默认值）。请注意，ini文件已放置在子目录“设置”中，该子目录是资产浏览器随附文件中包含ini文件的文件夹。
