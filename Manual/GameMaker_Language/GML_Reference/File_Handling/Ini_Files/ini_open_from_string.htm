ini_open_from_string

该函数将根据字符串创建一个ini文件，并打开它进行读取/写入。该字符串应正确设置为ini文件格式（即：带有节，换行键和值），否则将无法正确创建ini文件。请注意，此ini文件是临时文件，在关闭时将被从内存中删除，丢失了存储在其中的所有信息，但是 ini_close()函数返回完整ini文件的字符串，然后可以将其保存到服务器或磁盘中。

句法：

ini_open_from_string(string);

争论描述
string包含所有ini信息的字符串。

返回值：

N/A

例子：

ini_open_from_string(str)
global.sound = ini_read_string("Options", "Sound", true);
ini_close();

上面的代码设置了一个临时ini来保存字符串“ str”中的信息，然后从ini中读取信息，然后再次关闭它。

返回： Ini文件
