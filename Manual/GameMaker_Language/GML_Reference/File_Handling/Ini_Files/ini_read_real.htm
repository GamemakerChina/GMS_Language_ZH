ini_read_real

你可以用这个函数从一个ini数据文件中读取一个数字。Ini文件被分割成若干部分，然后每个部分随后被分割成键-值对。因此，一个典型的ini文件看起来会是这样的。

语法。

ini_read_real(section, key, default)。

争论 描述
节 从.ini中读取的部分。
钥匙 读取.ini中相关部分的键。
违约 如果在定义的地方没有找到一个值（或者.ini文件不存在），要返回的值。必须是一个实数。

返回。

真实的

例子。

ini_open("savedata.ini");
score = ini_read_real( "save1", "score", 0 );
ini_close();

这将打开 "savedata.ini"，并将分数设置为其中 "save1">"score "下的值，然后再次关闭.ini文件。如果 "save1">"score "下没有数值，或者没有 "savedata.ini "文件，score将被设置为0。 文件，分数将被设置为0。
