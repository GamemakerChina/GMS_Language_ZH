ini_read_real

您可以使用此功能从ini数据文件中读取数字。 Ini文件被拆分为多个部分，然后每个部分随后被拆分为键-值对。因此，典型的ini文件将如下所示：

句法：

ini_read_real(section, key, default);

争论描述
section要读取的.ini部分。
key要读取的.ini相关部分中的键。
default如果在定义的位置找不到值（或.ini文件不存在），则返回的值。必须是实数。

返回值：

Real

例子：

ini_open("savedata.ini");
score = ini_read_real("save1", "score", 0 );
ini_close();

这将打开“ savedata.ini”并将分数设置为其中“ save1”>“ score”下的值，然后再次关闭.ini。如果“ save1”>“ score”下没有任何值，或者不存在“ savedata.ini”文件，则得分将设置为0。

返回： Ini文件
