ini_read_string

您可以使用此功能从ini数据文件中读取字符串（文本）。 Ini文件被拆分为多个部分，然后每个部分随后被拆分为键-值对。因此，典型的ini文件将如下所示：

句法：

ini_read_string(section, key, default);

争论描述
section要读取的.ini部分。
key要读取的.ini相关部分中的键。
default如果在定义的位置（或.ini文件不存在）中找不到字符串，则返回的字符串。必须是一个字符串。

返回值：

String

例子：

ini_open("savedata.ini");
global.name = ini_read_string("player", "name', "Player1");
ini_close();

这将打开“ savedata.ini”，并将global.name设置为其中的“ save1”>“ Name”下的字符串，然后再次关闭.ini。如果“ save1”>“名称”下没有字符串，或者不存在“ savedata.ini”文件，则global.name将设置为“ Player1”。

返回： Ini文件
