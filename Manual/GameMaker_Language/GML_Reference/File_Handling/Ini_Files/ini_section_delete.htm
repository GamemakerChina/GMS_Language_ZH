ini_section_delete

使用此功能，您可以删除ini文件的整个部分，这还将删除与该文件关联的所有键值对。

句法：

ini_section_delete(section);

争论描述
section要删除的部分。

返回值：

N/A

例子：

ini_open("savedata.ini");
ini_write_real("save1", "Score", score );
ini_section_delete("save1");
ini_close();

本示例将打开“ savedata.ini”，并将值写入“ save1”>“ Score”。然后它将删除“ save1”部分并关闭.ini文件。

返回： Ini文件
