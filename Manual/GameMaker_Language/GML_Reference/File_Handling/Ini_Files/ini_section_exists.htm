ini_section_exists

此函数检查当前打开的ini中是否存在某个节并将返回 true如果有或者 false否则。这不是防止错误的必要检查，因为当某个部分不存在时，从不存在的部分中读取将仅返回默认值，但是查看ini文件是否已保存特定数据可能很有用。

句法：

ini_section_exists(section);

争论描述
section打开的.ini文件中要检查的部分。

返回值：

Boolean

例子：

ini_open("savedata.ini");
if !ini_section_exists("save1")
    {
    global.savegame = "1";
    }
ini_close();

如果未找到“ save1”，这会将变量global.savegame设置为“ 1”。有了for循环，这可以用作计算一个游戏有多少个保存游戏的基本方法（即：为一个游戏创建同时保存多个游戏的系统）。

返回： Ini文件
