ini_write_real

您可以使用此函数将值（数字）写入ini数据文件。 Ini文件被拆分为多个部分，然后每个部分随后被拆分为键-值对。因此，典型的ini文件将如下所示：

句法：

ini_write_real(section, key, value);

争论描述
section要写入的.ini部分。
key.ini相关部分内要写入的密钥。
value写入相关目标的实际值。

返回值：

N/A

例子：

score = 1000;
ini_open("savedata.ini");
ini_write_real("save1", "Score", score );
score2 = ini_read_real("save1", "Score", 0 );
ini_close();

这会将分数设置为1000，然后打开“ savedata.ini”，并将此值写入“ save1”>“ Score”。然后，将Score2设置为其中的“ save1”>“ score”下的实际值1000，然后再次关闭.ini。如果找不到该值，那么score2最终将被设置为0。

返回： Ini文件
