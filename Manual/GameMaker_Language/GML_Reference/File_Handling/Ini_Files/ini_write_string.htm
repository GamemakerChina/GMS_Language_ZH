ini_write_string

你可以使用这个函数将一个字符串（文本）写入一个ini数据文件中。Ini文件被分割成若干部分，然后每个部分随后被分割成键-值对。因此，一个典型的ini文件看起来会是这样的。

值得注意的是，在写ini字符串时使用引号和转义字符可能会在以后读回数据时造成问题。 时可能会出现问题，因为它可能被过早地截断。例如，写这个将是一个问题。

ini_write_string("test2", "section", "hello /"Fritz /")。

语法。

ini_write_string(section, key, value)。

争论 描述
节 写入.ini中的部分。
钥匙 写入.ini中相关部分的键。
价值 要写到相关目的地的字符串。

返回。

不适用

例子。

ini_open("saidata.ini");
ini_write_string("Save", "Player", global.Name);
ini_close();

上述代码打开一个ini文件进行读写，然后将存储在全局变量 "Name "中的字符串写入键为 "Player "的 "Save "部分，然后再次关闭该文件。
