ini_write_string

您可以使用此功能将字符串（文本）写入ini数据文件。 Ini文件被拆分为多个部分，然后每个部分随后被拆分为键-值对。因此，典型的ini文件将如下所示：

值得注意的是，在编写ini字符串时使用引号和转义符可能会在以后读取数据时引起问题，因为它可能会过早地被截断。例如，编写此文件将是一个问题：

ini_write_string("test2", "section", "hello \"Fritz\"");

句法：

ini_write_string(section, key, value);

争论描述
section要写入的.ini部分。
key.ini相关部分内要写入的密钥。
value写入相关目标的字符串。

返回值：

N/A

例子：

ini_open("savedata.ini");
ini_write_string("Save", "Player", global.Name);
ini_close();

上面的代码打开一个ini文件进行读写，然后使用键“ Player”将存储在全局变量“ Name”中的字符串写入“ Save”部分，然后再次关闭该文件。

返回： Ini文件
