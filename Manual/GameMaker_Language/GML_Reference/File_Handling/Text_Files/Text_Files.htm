文字档案

文本文件是从游戏外部存储大量数据的一种有用方法。与ini文件相反，您最多可以一次打开32个文本文件，也可以向其中读取和写入大块数据。但是，由于没有像 *.ini文件。

请注意，对于具有本地化版本的游戏，从外部文件中读取带重音的字母非常重要。这意味着您应该首先以UTF8格式创建txt文件，然后将其GameMaker Studio 2 ，以便在运行游戏时将其导出并使用，而不是由GameMaker Studio 2如果以前没有提供文件，则GameMaker Studio 2这样，您可以在保持所有重音和非罗马字母的情况下正确地对其进行读写。

存在以下处理文件的功能：

  • file_text_open_read
  • file_text_open_write
  • file_text_open_append
  • file_text_open_from_string
  • file_text_read_real
  • file_text_read_string
  • file_text_readln
  • file_text_write_real
  • file_text_write_string
  • file_text_writeln
  • file_text_eoln
  • file_text_eof
  • file_text_close
