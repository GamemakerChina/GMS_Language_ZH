file_text_close

一旦处理完给定文件（无论是读取文件还是写入文件），就必须再次关闭该文件，否则可能会丢失其中包含的信息。这还可以防止内存泄漏，并通过打开32个以上的文件来确保您永远不会超过文件限制。

句法：

file_text_close(fileid);

争论描述
fileid要关闭的文件的ID。

返回值：

N/A

例子：

var file = file_text_open_write(working_directory + "Game_Data.txt");
while (!file_text_eof(file))
    {
    file_text_readln(file);
    }
file_text_write_string(file, level_data);
file_text_close(file);

上面的代码打开一个要写入的文件，然后在已经写入该文件的文本行中循环，直到到达结尾为止。此时，它将写入一个字符串，然后再次关闭该文件。

返回：文本文件
