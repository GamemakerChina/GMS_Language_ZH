从字符串打开的文本

这个函数将从一个字符串中创建一个文本文件，并打开它进行阅读，返回文件 "句柄"，该句柄应在所有进一步的文件函数调用中使用，以从该文件中读取。请注意，这个文件是临时的，只读的。 因此，它将在关闭时从内存中删除。

注意：你在任何时候最多只能打开32个文件。你也应该在结束时总是关闭文件，因为这样可以写入信息并释放与文件相关的内存。

警告!由于GameMaker Studio 2是沙盒式的，这个功能可能不会像你预期的那样工作!请参阅文件系统部分以了解更多信息。

语法。

file_text_open_from_string(string)。

争论 描述
绳子 用于创建文件的字符串。

返回。

真实的

例子。

file = file_text_open_from_string（reset_str）。

上面的代码接收存储在变量 "reset_str "中的字符串，并使用它来创建一个只读的文本文件。然后，这个文件的 "句柄 "被存储在变量 "file "中，供所有其他文件函数使用。

返回。文本文件
