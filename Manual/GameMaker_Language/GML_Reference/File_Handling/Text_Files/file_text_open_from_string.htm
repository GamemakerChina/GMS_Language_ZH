file_text_open_from_string

此函数将从字符串创建一个文本文件，然后将其打开以进行读取，并返回“ handle”文件，该文件应在所有进一步的文件函数调用中使用以从该文件读取。请注意，此文件是临时文件，并且是只读文件，因此，它将在关闭时从内存中删除。

注意：一次最多只能打开32个文件。完成后，您还应该始终关闭文件，因为这将写入信息并释放与文件关联的内存。

警告！ GameMaker Studio 2沙盒化，因此此功能可能无法按预期工作！请参阅文件系统部分以获取更多信息。

句法：

file_text_open_from_string(string);

争论描述
string从中创建文件的字符串。

返回值：

Real

例子：

file = file_text_open_from_string(reset_str);

上面的代码获取存储在变量“ reset_str”中的字符串，并使用它创建一个只读文本文件。然后，此文件的“句柄”存储在变量“ file”中，供所有其他文件功能使用。

返回：文本文件
