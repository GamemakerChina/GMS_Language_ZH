文件_text_open_read

该函数打开指定文件名的文本文件，仅供阅读，并返回该文件的唯一ID，该ID应存储在一个变量中，因为它将用于所有与该文件有关的进一步操作。如果该文件不 存在，则该函数将返回值-1。

注意：你在任何时候最多只能打开32个文件。你也应该在结束时总是关闭文件，因为这样可以写入信息并释放与文件相关的内存。

警告!由于GameMaker Studio 2是沙盒式的，这个功能可能不会像你预期的那样工作!请参阅文件系统部分以了解更多信息。

语法。

file_text_open_read(fname)。

争论 描述
fname 要读取的文件的名称。

返回。

真实的

例子。

file = file_text_open_read（working_directory + "level.txt"）。

这将从与游戏相同的目录中打开 "level.txt"，并在变量 "file "中存储文件ID。

返回。文本文件
