file_text_open_read

此函数将打开具有指定文件名的文本文件以供只读，返回应存储在变量中的文件的唯一ID，因为它将用于对该文件执行的所有其他操作。如果该文件不存在，则该函数将返回值-1。

注意：一次最多只能打开32个文件。完成后，您还应该始终关闭文件，因为这将写入信息并释放与文件关联的内存。

警告！ GameMaker Studio 2沙盒化，因此此功能可能无法按预期工作！请参阅文件系统部分以获取更多信息。

句法：

file_text_open_read(fname);

争论描述
fname要读取的文件的名称。

返回值：

Real

例子：

file = file_text_open_read(working_directory + "level.txt");

这将从与游戏相同的目录中打开“ level.txt”，并将文件ID存储在变量“ file”中。

返回：文本文件
