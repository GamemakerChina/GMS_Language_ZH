file_text_open_write

此函数打开具有指定文件名的文本文件，仅用于写入（如果文件不存在，则创建该文件），并返回该文件的唯一ID ，该ID应存储在变量中，因为它将用于所有其他操作与该文件有关。请注意，如果无法创建文件（例如，由于文件名非法），该函数将返回-1。

注意：一次最多只能打开32个文件。完成后，您还应该始终关闭文件，因为这将写入信息并释放与文件关联的内存。

警告！ GameMaker Studio 2沙盒化，因此此功能可能无法按预期工作！请参阅文件系统部分以获取更多信息。

句法：

file_text_open_write(fname);

争论描述
fname要写入的文件名。

返回值：

Real

例子：

var file;
file = file_text_open_write(working_directory + "level.txt");
file_text_write_string(file, level_data);
file_text_close(file);

上面的代码将打开文件“ level.txt”进行写入，然后写入存储在变量“ level_data”中的字符串，然后最终再次关闭该文件。

返回：文本文件
