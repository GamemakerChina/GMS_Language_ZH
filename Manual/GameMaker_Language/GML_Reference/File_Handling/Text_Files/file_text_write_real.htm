文件_文本_写作_真实

通过这个函数，你可以把一个数字写到先前打开的文本文件中。注意，由于要写入的数值可以是一个实数，所有的小数都将以". "点作为分隔符写入。如果该文件已经包含了信息。 这些信息将被清除，而字符串将被写在文件的开头，除非你在打开文件时使用了 file_text_open_append().你也可以通过使用 file_text_readln()函数和 file_text_eof()函数来循环浏览文件的内容，直到最后再开始写入。

需要注意的是，当使用这个函数将非常大的数字写入文本文件时，它可能会被翻译成标准的简化格式，如 "6.6624e+003"，从而无法正确读回GameMakerStudio 2。 为了防止这样的问题，你应该把数值转换成字符串，然后使用函数 file_text_write_string()来代替。

语法。

file_text_write_real(fileid, val)。

争论 描述
文件名 要编辑的文件的ID。
价值 要写入文件的真实值。

返回。

真实的

例子。

var file = file_text_open_write(working_directory + "hiscore.txt");
for (var i = 0; i < 10; ++i;)
{
file_text_write_real(file, scr[i]);
file_text_writeln(file)。
file_text_write_string(file, scr_name[i]);
file_text_writeln(file);
}
file_text_close(file)。

上面的代码打开一个文件进行写入，然后在两个数组中循环，将每个数组的值写入文件的新行中。循环结束后，该文件被关闭。
