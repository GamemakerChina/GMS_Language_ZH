file_text_write_string

使用此功能，您可以将字符串写入以前打开的文本文件。如果文件已经包含信息，则除非您使用 file_text_open_append()。您也可以通过使用 file_text_readln()功能以及 file_text_eof()函数循环浏览文件的内容，直到结束，然后开始写入。

句法：

file_text_write_string(fileid, str);

争论描述
fileid要编辑的文件的ID。
str写入文件的字符串。

返回值：

Real

例子：

var file = file_text_open_write(working_directory + "hiscore.txt");
for (var i = 0; i < 10; ++i;)
    {
    file_text_write_real(file, scr[i]);
    file_text_writeln(file);
    file_text_write_string(file, scr_name[i]);
    file_text_writeln(file);
    }
file_text_close(file);

上面的代码打开一个要写入的文件，然后遍历两个数组，将每个数组的值写入文件的新行。循环结束后，文件将关闭。

返回：文本文件
