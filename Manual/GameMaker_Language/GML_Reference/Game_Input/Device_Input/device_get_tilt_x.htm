device_get_tilt_x

此函数根据设备的“倾斜”角度返回介于-1和1之间的值。倾斜度和返回值之间的实际相关性取决于所使用的设备和OS，但通常值为1或-1与+/- 90°相同。下图显示了每种可用功能与设备的关系：

句法：

device_get_tilt_x()

返回值：

Real

例子：

if display_get_orientation() = display_landscape
   {
   x += sign(device_get_tilt_y());
   }
else
   {
   x += sign(device_get_tilt_x());
   }

上面的代码检查显示器的方向，然后使用相应的倾斜值沿x轴移动播放器。
