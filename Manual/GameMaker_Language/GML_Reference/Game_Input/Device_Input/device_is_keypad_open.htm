device_is_keypad_open

这会检查设备是否有键盘，如果有键盘，它将返回 true否则返回 false。请注意，此功能主要用于Android设备

对于使用Sony Xperia Play的GameMaker Studio 2了一个设置按钮/键映射设置，因此您可以将键盘常数vk_up ， vk_down ， vk_left ， vk_right用作游戏手柄键，而Triangle是 ord("T")，广场是 ord("S")， Circle是vk_alt + vk_backspace，而后退按钮只是 vk_backspace， Cross是vk_space ， Select是vk_return且Start是vk_rshift ， L Trigger是 ord("L")并且R Trigger为 ord("R")。

句法：

device_is_keypad_open

返回值：

Boolean

例子：

if device_is_keypad_open()
   {
   global.Setting = 2;
   }
else
   {
   global.Setting = 1;
   }

上面的代码检查键盘，然后根据返回的值更改全局变量。
