设备_is_keypad_open

这个函数检查设备是否有键盘，如果有，则返回真，否则返回假。请注意，这个函数主要用于Android设备。

对于那些使用索尼Xperia Play的用户，在GameMaker Studio 2内有一套按钮/按键地图设置，所以你可以使用键盘常量 vk_up,vk_down,vk_left, vk_right用于操纵板按键，三角形是ord("T")，方形是ord("S")，圆形是vk_alt + vk_ backspace，而后退按钮只是vk_backspace。 十字是vk_space，选择是vk_return，开始是vk_rshift，L触发器是ord("L")，R触发器是ord("R")。

语法。

设备_is_keypad_open

返回。

布尔型

例子。

if device_is_keypad_open()
{
global.Setting = 2;
}
else
{
global.Setting = 1;
}

上面的代码检查一个键盘，然后根据返回的值改变一个全局变量。
