设备_鼠标_检查按钮

这个函数根据给定设备上给定的鼠标按钮是否被按住而返回真或假。对于鼠标按钮，你可以提供本页所列的常数之一。

这个函数可以用于触摸屏，设备 参数可以从0到n，其中n是可以同时触摸屏幕的最大 "手指 "数。 检测到的最大触摸数在很大程度上取决于所使用的设备和它所运行的操作系统，但大多数设备应该至少能检测到4个）。)请注意，只有在检测到双击触摸时，才会检测到mb_right（该行为可以用 函数 device_mouse_dbclick_enable()).

语法。

device_mouse_check_button(device, button)。

争论 描述
设备 被检查的设备（从0-n）。
按钮 被检查的设备的按钮。

返回。

布尔型

例子。

if device_mouse_check_button(0, mb_left) && device_mouse_check_button(1, mb_left)
{
room_goto(rm_Menu);
}

上面的代码检查是否同时按住了两个触摸点，如果是，那么它就会去另一个房间。
