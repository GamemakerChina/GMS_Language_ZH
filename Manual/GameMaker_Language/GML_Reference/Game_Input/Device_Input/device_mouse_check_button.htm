device_mouse_check_button

该函数返回 true（1）或 false（0），取决于是否正在使用您指定的设备。现在，设备引用的鼠标数可以为0到n，并且在触摸移动设备的触摸屏时触发此事件-其中n是在以下位置可以触摸屏幕的“手指”的最大数量一次（可以检测到的最大触摸次数在很大程度上取决于所使用的设备及其运行的操作系统，但是大多数设备至少应检测到4次）。此函数会不断更新每个步骤，以及 mb_left和 mb_right可用于检查任一按钮。注意 mb_right仅在检测到双击时才能检测到（可以使用功能禁用此行为 device_mouse_dbclick_enable()。

句法：

device_mouse_check_button(device, button);

争论描述
device正在检查的设备（从0- n）
button正在检查的设备的按钮。

返回值：

Boolean

例子：

if device_mouse_check_button(0, mb_left) && device_mouse_check_button(1, mb_left)
{
room_goto(rm_Menu);
}

上面的代码检查两个触摸是否同时按住，然后又转到另一个房间。
