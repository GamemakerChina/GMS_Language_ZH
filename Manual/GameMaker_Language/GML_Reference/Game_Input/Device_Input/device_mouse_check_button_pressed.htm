device_mouse_check_button_pressed

该函数返回true或false取决于您指定的设备是否被“触摸”（单击）。设备参数是指触摸次数，可以是0到n，可以检测到的最大触摸次数将在很大程度上取决于所使用的设备及其运行的操作系统（大多数设备将支持至少4个同时触摸）。该功能仅由实际的按动操作和标准常数来触发一次。mb_left和 mb_right可用于检查任一按钮。注意 mb_right仅在检测到双击并按住第二次点击的情况下才会检测到（可以使用功能禁用此行为 device_mouse_dbclick_enable()。

句法：

device_mouse_check_button_pressed(device, button);

争论描述
device正在检查的设备（从0- n）。
button正在检查的设备的按钮。

返回值：

Boolean

例子：

if device_mouse_check_button_pressed(0, mb_left)
{
press=true;
}

上面的代码检查是否已按下设备，如果已按下，则将变量设置为true。
