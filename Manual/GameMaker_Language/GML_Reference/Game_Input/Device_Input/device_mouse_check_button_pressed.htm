设备_鼠标_检查_按钮_按下

这个函数根据你指定的设备是否被 "触摸"（点击）而返回真或假。设备参数指的是触摸次数，可以从0到n，可以检测到的最大触摸次数在很大程度上取决于所使用的设备和所运行的操作系统。 可以检测到的最大触摸次数在很大程度上取决于正在使用的设备和它所运行的操作系统（大多数设备至少支持4次同时触摸）。这个函数只被实际的按压动作触发一次 ，本页所列的常数 列出的常量可以用来检查鼠标按钮。请注意，只有在检测到双击触摸并在第二次点击时保持住时，mb_right才会被检测到（这个行为可以用函数 来禁用 使用函数 device_mouse_dbclick_enable())。

语法。

device_mouse_check_button_pressed(device, button)。

争论 描述
设备 被检查的设备（从0-n）。
按钮 被检查的设备的按钮。

返回。

布尔型

例子。

if device_mouse_check_button_pressed(0, mb_left)
{
press=true;
}

上面的代码检查设备是否被按下，如果被按下则将一个变量设置为真。
