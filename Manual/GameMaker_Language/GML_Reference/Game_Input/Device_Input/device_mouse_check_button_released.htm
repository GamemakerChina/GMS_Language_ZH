device_mouse_check_button_released

该函数返回true或false取决于您指定的设备触摸是否已释放。设备参数指的是设备触摸次数，可以是0到n，并且请注意，可以检测到的最大触摸次数在很大程度上取决于所使用的设备及其运行的操作系统（大多数设备至少支持）。 4个同时触摸）。该功能仅由实际的释放动作和的标准常数触发一次。mb_left和 mb_right可用于检查任一按钮。注意 mb_right只有在释放之前检测到并按住第二个水龙头的情况下，才会检测到该水龙头（可以使用功能禁用此行为 device_mouse_dbclick_enable()。

句法：

device_mouse_check_button_released(device, button);

争论描述
device正在检查的设备（从0- n）。
button正在检查的设备的按钮。

返回值：

Boolean

例子：

if device_mouse_check_button_released(0, mb_left)
{
press=false;
}

上面的代码检查是否已按下设备，如果已按下则将变量设置为false。
