device_mouse_dbclick_enable

此功能可用于设置设备以检测双击 mb_left（鼠标左键）作为 mb_right（鼠标右键）是否点击。默认情况下，此设置为 true，这意味着每次用户连续两次快速单击设备屏幕时，返回值都与单击鼠标右键相同。启用此功能后，第一次点击将被检测为 mb_left，第二个为 mb_right，因此请确保您使用的任何代码都考虑到了这一点。

句法：

device_mouse_dbclick_enable(bool);

争论描述
bool将双击检测设置为（true ）或关闭（false ）。

返回值：

Boolean

例子：

if os_type == os_windows || os_type == os_mac
   {
   device_mouse_dbclick_enable(false);
   }

上面的代码检查运行游戏的设备是Windows PC还是Mac，如果两者都是，则禁用双击功能。
