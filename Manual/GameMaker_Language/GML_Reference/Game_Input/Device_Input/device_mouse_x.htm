device_mouse_x

此函数返回设备上触摸的x位置。如果您在HTML5或PC和Mac模块上运行此代码，则只要插入设备（通常是鼠标），此值就会不断更新，但是对于移动设备，此值仅在屏幕显示时才更新请注意，可以检测到的最大触摸次数在很大程度上取决于所使用的设备及其运行的操作系统。另请注意，在HTML5上，只有“设备0”才能正确运行，其他设备可能会给出错误的值。

句法：

device_mouse_x(device);

争论描述
device正在检查的设备（从0- n）。

返回值：

Real

例子：

if device_mouse_check_button(0, mb_left)
   {
   if device_mouse_x(0) > x-32 && device_mouse_x(0) < x+32
      {
      pressed = true;
      }
   else
      {
      pressed = false;
      }
   }

上面的代码检查是否按下了设备，如果是，则轮询设备的x位置以查看它是否在参数内。如果它是将变量“按下”设置为 true，否则将其设置为 false。
