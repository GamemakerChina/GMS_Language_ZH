device_mouse_y_to_gui

此函数返回相对于GUI层的设备上触摸的y位置。由于可以将GUI层设置为与屏幕分辨率和房间大小无关的其他大小，因此有时可能很难在GUI层上获得用户触摸屏幕的确切点。使用此功能，您可以根据实际的GUI尺寸获得该位置，从而使GUI按钮和HUD元素更易于创建和交互。如果您在HTML5或PC和Mac模块上运行此代码，则只要插入设备（通常是鼠标），此值就会不断更新，但是对于移动设备，仅在屏幕显示时才会更新触摸，请注意，可以检测到的最大触摸次数在很大程度上取决于所使用的设备及其运行的操作系统。另请注意，在HTML5上，只有“设备0”才能正确运行，其他设备可能会给出错误的值。

句法：

device_mouse_y_to_gui(device);

争论描述
device正在检查的设备（从0- n）。

返回值：

Real

例子：

if device_mouse_check_button(0, mb_left)
   {
   if device_mouse_y_to_gui(0) > 0 && device_mouse_y_to_gui(0) < 32
      {
      pressed = true;
      }
   else
      {
      pressed = false;
      }
   }

上面的代码检查是否按下了设备鼠标，如果是，则在GUI层上轮询设备x的位置以查看它是否在参数内。如果它是将变量“按下”设置为 true，否则将其设置为 false。
