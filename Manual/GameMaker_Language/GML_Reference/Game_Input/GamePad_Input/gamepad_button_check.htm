游戏手柄_按钮_检查

这个函数将返回真或假，取决于给定的游戏板按钮是否被检测到被按住。如果你检查的是一个模拟按钮，那么检查将考虑到阈值设置，只有当原始输入值超过给定的阈值时才会返回 true （你可以用函数 gamepad_button_value() 来得到这个原始值）。

语法。

gamepad_button_check(device, button)。

争论 描述
设备 要检查哪个游戏板设备 "插槽"。
按钮 要检查哪个游戏板按钮常数。

返回。

布尔型

例子。

if gamepad_button_check(0, gp_face1)
{
if canshoot = true
{
audio_play_sound(snd_Shoot, 0, false);
instance_create_layer(x, y, "Bullets", obj_Bullet)
canshoot = false;
alarm[0] = room_speed / 2;
}
}

上面的代码将检测顶面的底部按钮（ps3控制器上的 "X"）是否被按住，如果是，它将射出一个 "子弹 "实例并设置一个警报。

返回。游戏手柄输入
