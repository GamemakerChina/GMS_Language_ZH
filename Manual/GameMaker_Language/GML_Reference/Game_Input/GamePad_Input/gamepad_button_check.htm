gamepad_button_check

该函数将返回 true或者 false取决于是否检测到给定的游戏手柄按钮被按下。如果您正在检查模拟按钮，则检查将考虑阈值设置，仅返回 true当原始输入值超过给定阈值时（您可以使用函数获得该原始值 gamepad_button_value()）。

句法：

gamepad_button_check(device, button);

争论描述
device要检查哪个游戏手柄设备的“插槽”。
button要检查哪个游戏手柄按钮常数。

返回值：

Boolean

例子：

if gamepad_button_check(0, gp_face1)
   {
   if canshoot = true
      {
      audio_play_sound(snd_Shoot, 0, false);
      instance_create_layer(x, y, "Bullets", obj_Bullet)
      canshoot = false;
      alarm[0] = room_speed / 2;
      }
   }

上面的代码将检测是否按下了顶面的底部按钮（ps3控制器上的“ X”），如果是，则将射击“子弹”实例并设置警报。
