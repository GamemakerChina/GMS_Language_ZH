gamepad_button_check_pressed

该函数将返回 true或者 false取决于是否检测到给定的游戏手柄按钮。请注意，此功能仅在第一次按下按钮时触发一次。为了再次触发，必须先释放按钮，然后再按一次。如果需要检查连续按下按钮，则应使用该功能 gamepad_button_check()。

如果您正在检查模拟按钮，则检查将考虑阈值设置，仅返回 true当原始输入值超过给定阈值时（您可以使用函数获得该原始值 gamepad_button_value()）。

句法：

gamepad_button_check_pressed(device, button);

争论描述
device要检查哪个游戏手柄设备的“插槽”。
button要检查哪个游戏手柄按钮常数。

返回值：

Boolean

例子：

if gamepad_button_check_pressed(0, gp_start)
   {
   audio_play_sound(snd_Start, 0, false);
   room_goto(rm_Level_1);
   }

上面的代码将检测连接到设备“ slot” 0的游戏手柄的“开始”按钮是否已按下，如果有，则更改房间。
