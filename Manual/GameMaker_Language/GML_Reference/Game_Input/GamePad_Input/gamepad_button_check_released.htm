gamepad_button_check_released

该函数将返回 true或者 false取决于是否检测到给定的游戏手柄按钮已被释放。请注意，此功能在释放按钮时仅会为按钮触发一次。为了再次触发，必须先按下按钮，然后再释放一次。如果您正在检查模拟按钮，则检查将考虑阈值设置，仅返回 true当原始输入值低于给定阈值时（您可以使用函数获得该原始值 gamepad_button_value()）。

句法：

gamepad_button_check_released(device, button);

争论描述
device要检查哪个游戏手柄设备的“插槽”。
button要检查哪个游戏手柄按钮常数。

返回值：

Boolean

例子：

if gamepad_button_check_released(0, gp_select)
   {
   audio_play_sound(snd_Button, 0, false);
   global.Pause = !global.Pause;
   }

上面的代码将检测连接到设备“ slot” 0的游戏手柄的“选择”按钮是否被按下，并切换全局“ Pause”变量。

背面：游戏手柄输入
