gamepad_button_value

使用此功能，您可以获得模拟按钮的当前值，从0到1，其中0为无压力，而1为全压力。您提供的手柄时隙索引来检查，与任一个按钮恒定沿（如列出这里），或0和之间的整数值 gamepad_button_count()-1。

请注意，这将返回原始值的按钮，并没有考虑到的阈值设置（见这里关于阈值的详细信息）。

句法：

gamepad_button_value(device, button);

争论描述
device要检查哪个游戏手柄设备的“插槽”。
button要检查哪个游戏手柄按钮常数。

返回值：

Real

例子：

speed = gamepad_button_value(0, gp_shoulderrb) * 4;

上面的代码使用插入到设备“插槽” 0中的游戏手柄的模拟触发值来设置实例的速度。
