gamepad_get_axis_deadzone

此功能可用于获取操纵杆轴的“死区”值。您指定要获取的设备插槽，该函数将返回一个介于0到1之间的值，其中该值反映了将操纵杆轴视为在0以下的阈值。

句法：

gamepad_get_axis_deadzone(device);

争论描述
device要检查哪个游戏手柄设备的“插槽”。

返回值：

Real

例子：

if gamepad_get_axis_deadzone(global.PadId) != 0.5
    {
    gamepad_set_axis_deadzone(global.PadId, 0.5);
    }

上面的代码检查连接到存储在全局变量中的设备“插槽”上的游戏手柄的模拟轴阈值，如果它不是默认值0.5，则将其设置为该值。
