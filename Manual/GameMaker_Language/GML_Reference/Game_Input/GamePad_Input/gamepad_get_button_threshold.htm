gamepad_get_button_threshold

这个功能可以用来检测一个给定设备的模拟按钮的当前阈值设置。所有模拟按钮的默认阈值是0.5，范围是0到1。 被视为 "按下"，因为游戏需要它们作为一个数字按钮。

语法。

gamepad_get_button_threshold(device)。

争论 描述
设备 要检查哪个游戏板设备 "插槽"。

返回。

真实的

例子。

if gamepad_get_button_threshold(0) != 0.5
{
gamepad_set_button_threshold(0, 0.5);
}

上面的代码检查连接到设备 "插槽 "0的游戏板的模拟按钮阈值，如果它不是默认值0.5，它就被设置为这个值。

返回。游戏手柄输入
