gamepad_get_description

此函数将返回带有给定游戏手柄描述的字符串，例如，PS3控制器可能返回类似于“ PLAYSTATION（R）3控制器”的字符串，或者XBox360控制器可能返回“ Xbox 360控制器（XInput STANDARD GAMEPAD） ”。此字符串取决于硬件，返回值将取决于插入正在检查的设备“插槽”中的游戏手柄。

句法：

gamepad_get_description(device);

争论描述
device获得哪个游戏手柄的“插槽”名称。

返回值：

String

例子：

var gp_num = gamepad_get_device_count();
for (var i = 0; i < gp_num; i++;)
   {
   if gamepad_is_connected(i)
      {
      draw_text(32, 32 + (i * 32), gamepad_get_description(i));
      }
   else
      {
      draw_text(32, 32 + (i * 32), "No Gamepad Connected");
      }
   }

上面的代码将遍历所有游戏手柄插槽，以检查连接的设备，然后根据游戏手柄是否已连接到插槽将一些文本绘制到屏幕上。
