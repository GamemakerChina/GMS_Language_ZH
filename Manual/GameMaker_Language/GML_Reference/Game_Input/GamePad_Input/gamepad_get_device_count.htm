gamepad_get_device_count

此功能将告诉您以下两件事之一。连接的游戏手柄数量或要连接的游戏手柄的可用“插槽”数量。实际的返回值将取决于平台和该平台的内部配置，因此，此功能应与该功能结合使用 gamepad_is_connected()以确保随时连接的打击垫的确切数量。

句法：

gamepad_get_device_count();

返回值：

Real

例子：

var gp_num = gamepad_get_device_count();
for (var i = 0; i < gp_num; i++;)
    {
    if gamepad_is_connected(i)
        {
        global.gp[i] = true;
        }
    else
        {
        global.gp[i] = false;
        }
    }

上面的代码循环浏览可用的游戏手柄（或游戏手柄插槽），然后检查每个游戏手柄是否已连接。然后，将返回的值用于将全局数组设置为 true或者 false供将来检查时使用。
