gamepad_get_guid

使用此功能，您可以检索连接到给定插槽索引的游戏手柄的GUID。您提供游戏手柄插槽进行检查（从0到11），该函数将返回以下字符串之一：

  • 如果没有可用的GUID或未连接游戏手柄，则为“ none”
  • 如果游戏手柄插槽索引超出范围，则为“设备索引超出范围”
  • GUID字符串，类似于“ 050000005e040000fd020000ffff3f00”，如果插槽中分配了游戏手柄，则可以为其获取GUID

此功能通常与 gamepad_get_description()Windows ， Android和macOS上重新映射控制器所需的SDL字符串的部分。

句法：

gamepad_get_guid(index);

争论描述
index要检查哪个游戏手柄“插槽”索引

返回值：

String

例子：

var _guid = gamepad_get_guid(global.PadIndex);
var _desc = gamepad_get_description(global.PadIndex);
global.GamepadID = _guid + "," + _desc;

上面的代码获取GUID和名称描述字符串，然后将它们连接起来并将最终字符串存储在全局变量中，以备将来参考。
