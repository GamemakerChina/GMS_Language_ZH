gamepad_get_mapping

这个函数可以用来检索游戏手柄的映射字符串。这个字符串将是以下任何一种。

  • 当前游戏手柄的地图字符串，例如。

    "050000005e040000fd020000ffff3f00,Xbox Wireless Controller,a:b0,b:b1,start:b4,dpdown:h0.4,dpleft:h0.8,dpright:h0.2,dpup:h0.1,Guide:b6,leftshoulder:b9,leftstick:b7,lefttrigger:a4,leftx:a0,lefty:a1,righthoulder:b10,rightstick:b8,righttrigger:a5,rightx:a2,righty:a3,x:b2,y:b3,platform:android"

  • 如果没有设置映射，则为 "无映射 "的字符串（这与无映射时的情况不同）。
  • 如果插槽索引不是有效的游戏手柄索引，则会出现 "设备索引超出范围 "的字符串
  • 如果所连接的游戏手柄没有映射，则为空字符串""

关于返回的地图字符串格式的更多信息，请参见函数 gamepad_test_mapping().

语法。

gamepad_get_mapping(index)。

争论 描述
指数 从哪个游戏板索引 "槽 "获取映射。

返回。

字符串

例子。

var _gpMap = gamepad_get_mapping(global.PadIndex);
show_debug_message("Gamepad Mapping = " + _gpMap);

上面的代码为给定的游戏手柄插槽获取地图字符串，然后将其输出到控制台进行调试。

返回。游戏手柄输入
