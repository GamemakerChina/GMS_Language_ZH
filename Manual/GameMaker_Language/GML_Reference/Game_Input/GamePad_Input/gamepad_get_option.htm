gamepad_get_option

此函数可用于检索给定的选项值。您提供游戏手柄“插槽”以进行检查，以及选项字符串，该函数将返回关联的值。可用的选项字符串将取决于运行项目的平台，如下所示：

选项平台描述退货
“ allow_rotation” tvOS返回是否将Siri Remote旋转锁定（即：只能垂直使用）。解锁后，旋转控件将更改输入，例如向左旋转90º将使触摸板的“右侧”返回“向上”，依此类推...这样所有输入都将相对于方向设备的布尔型
“ dpad_absolute” tvOS返回Siri Remote触摸板是接受绝对输入还是相对输入。在绝对模式下，触摸板的中心被认为是（0,0）位置-即：模拟摇杆处于静止状态或dpad的中心所在位置-围绕此位置的移动将产生输入。当设置为相对时，则将（0,0）位置视为在打击垫上进行初始触摸的位置，并且所有输入都将从该位置生成，而不是打击垫的中心。布尔型

句法：

gamepad_get_option(device, option_name);

争论描述
device要检查哪个游戏手柄设备的“插槽”。
option_name从中获取值的选项的名称（字符串，请参见上表）

返回值：

Int, Bool, Real, String, N/A (depends on platform)

例子：

for(var i = 0; i < 12; ++i;)
    {
    if gamepad_is_connected(i) && gamepad_get_description(i) == "tvOS Siri Remote"
        {
        if gamepad_get_option(i, "allow_rotation") == false
            {
            gamepad_set_option(i, "allow_rotation", true);
            }
        }
    }

上面的代码遍历所有游戏手柄插槽，并在tvOS平台上检查“ Siri Remote”。如果检测到一个，则将遥控器设置为允许输入旋转。
