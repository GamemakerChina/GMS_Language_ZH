gamepad_hat_count

此功能将返回连接到给定设备“插槽”的游戏手柄可用的帽子总数。帽子通常指的是向上/向下/向左/向右按钮，请注意，在Windows目标上，帽子仅在DirectInput控制器上可用（因此，从插槽4向上）。

句法：

gamepad_hat_count(device);

争论描述
device要检查哪个游戏手柄设备的“插槽”。

返回值：

Real

例子：

h_num = gamepad_hat_count(4);

上面的代码将返回插入设备“插槽” 4的游戏板上可用的帽子数量，并将该值存储在变量“ h_num”中。
