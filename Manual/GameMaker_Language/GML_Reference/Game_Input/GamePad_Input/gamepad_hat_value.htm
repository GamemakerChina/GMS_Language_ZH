gamepad_hat_value

使用此功能，您可以获取游戏手柄“帽子”的当前值。每个帽子值是不同方向的位掩码，其中：

  • 上= 1
  • 右= 2
  • 下= 4
  • 左= 8

请注意，可以将它们组合（例如，将hat index设置为3可以向上和向右检查），但是一次最多只能按2位。该函数将返回一个介于0和1之间的实数值，其中未按下0且已完全按下1（并且可能在两者之间存在值，具体取决于游戏手柄是否支持帽子的模拟输入）。

句法：

gamepad_hat_value(device, hatindex);

争论描述
device要检查哪个游戏手柄设备的“插槽”。
hatindex要检查哪个游戏手柄帽子。

返回值：

Real

例子：

var _dir = gamepad_hat_value(global.PadIndex, 0);
switch (_dir)
    {
    case 1: y -= 3; break;
    case 2: x += 3; break;
    case 3: y -= 3; x += 3; break;
    case 4: x -= 3; break;
    case 6: y += 3; x += 3; break;
    case 8: y += 3; break;
    case 9: y -= 3; x -= 3; break;
    case 12: y += 3; x -= 3; break;
    }

上面的代码将帽子“ 0”的状态存储在本地变量中，然后检查返回值是什么并采取相应的措施。
