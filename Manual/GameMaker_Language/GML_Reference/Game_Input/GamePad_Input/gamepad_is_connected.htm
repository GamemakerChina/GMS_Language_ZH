gamepad_is_connected

此功能将返回是否将游戏手柄连接到给定的“插槽”（返回 true）否（返回 false）。通常，您可以将此功能与 gamepad_get_device_count()功能以获取正确数量的可用游戏手柄和/或游戏手柄“插槽”。请注意，连接游戏手柄的用户和GameMaker Studio 2检测到已连接用户之间可能会稍有延迟（在处理蓝牙连接的控制器时尤其如此）。

注意：在某些平台（尤其是控制台）上，连接/选择了游戏手柄后，此功能可能立即返回false，并且可能需要在几帧（步骤）后检查警报，以使其能够正确检测到游戏手柄。

句法：

gamepad_is_connected(numb);

争论描述
device要检查哪个游戏手柄“插槽”。

返回值：

Boolean

例子：

var gp_num = gamepad_get_device_count();
for (var i = 0; i < gp_num; i++;)
   {
   if gamepad_is_connected(i) global.gp[i] = true else global.gp[i] = false;
   }

上面的代码循环浏览可用的游戏手柄（或游戏手柄插槽），然后检查每个游戏手柄是否已连接。然后，将返回的值用于将全局数组设置为 true或者 false供将来检查时使用。
