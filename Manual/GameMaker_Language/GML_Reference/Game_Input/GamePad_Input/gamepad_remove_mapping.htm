gamepad_remove_mapping

此功能可用于从给定的游戏手柄插槽索引中删除当前设备映射。调用后，将需要使用来重新映射插槽索引 gamepad_test_mapping()函数，如果您希望能够使用游戏手柄常量正确检测输入（始终可以使用 gamepad_axis/button/hat_count()和 gamepad_axis/button/hat_value()一起使用）。

句法：

gamepad_remove_mapping(index);

争论描述
index从哪个游戏手柄索引“插槽”中删除映射。

返回值：

N/A

例子：

if remap == true
    {
    gamepad_remove_mapping(global.PadIndex);
    }

上面的代码将基于变量的值从给定的游戏手柄索引槽中删除映射。
