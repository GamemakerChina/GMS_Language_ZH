gamepad_set_axis_deadzone

此功能可用于设置操纵杆轴的“死区”。您指定要设置的设备插槽，然后将值设置为0到1，如果输入量小于给定值，则将操纵杆轴视为0。请注意，这是一个全局设置，将影响所有连接到指定设备插槽的所有操纵杆的轴线。

使用此功能并不意味着轴值将从死区值开始，而是该功能gamepad_axis_value()将始终返回介于-1和0或0和1之间的归一化值。例如，将死区设置为0.2将意味着向右推操纵杆仅在原始轴值超过0.2-时才开始返回0-1中的值-因此，当原始值为0.2时，返回值为0；当原始值为0.5时，返回值为0.375；如果原始值为0.9，则返回值为0.875。

句法：

gamepad_set_axis_deadzone(device, deadzone);

争论描述
device要检查哪个游戏手柄设备的“插槽”。
deadzone盲区值从0到1。

返回值：

N/A

例子：

if gamepad_is_connected(0) gamepad_set_axis_deadzone(0, 0.05);
if gamepad_is_connected(1) gamepad_set_axis_deadzone(1, 0.05);
if gamepad_is_connected(2) gamepad_set_axis_deadzone(2, 0.05);
if gamepad_is_connected(3) gamepad_set_axis_deadzone(3, 0.05);

上面的代码会将4台已连接设备中任何一个上的操纵杆轴的死区设置为0.05。
