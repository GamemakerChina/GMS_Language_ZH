gamepad_set_button_threshold

此功能可用于设置给定设备的模拟按钮的当前阈值设置。所有模拟按钮的默认阈值为0.5，范围从0到1。该阈值定义了要求将其用作数字按钮的游戏在什么时候认为该按钮被“按下”。请注意，此功能将影响模拟按钮的检查，按下和释放状态，但不会影响该功能返回的值gamepad_button_check()，它将始终返回按钮的原始值。

句法：

gamepad_set_button_threshold(device, threshold);

争论描述
device要检查哪个游戏手柄设备的“插槽”。
threshold新的阈值（从0-1，默认为0.5）。

返回值：

N/A

例子：

if gamepad_get_button_threshold(0) != 0.5
   {
   gamepad_set_button_threshold(0, 0.5);
   }

上面的代码检查连接到设备“插槽” 0的游戏手柄的模拟按钮阈值，如果它不是默认值0.5，则将其设置为该值。
