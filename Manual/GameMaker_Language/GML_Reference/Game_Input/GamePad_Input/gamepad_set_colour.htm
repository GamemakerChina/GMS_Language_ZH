gamepad_set_colour

此功能可用于设置PlayStation控制器内LED的颜色。您指定要设置的设备插槽，然后提供一种颜色，该颜色可以是任何颜色常量或使用特定颜色函数创建的颜色值或十六进制值（例如$FFFFFFF ）。

句法：

gamepad_set_colour(device, colour);

争论描述
device要设置哪个游戏手柄设备“插槽”。
colour使用的颜色。

返回值：

N/A

例子：

if health < 10
   {
   gamepad_set_colour(0, c_red);
   }

如果以下情况，上面的代码会将PlayStation控制器的LED设置为红色。health变量低于10。
