gamepad_set_option

此功能可用于设置任何可用的游戏手柄选项。您可以提供游戏板“插槽”来设置选项，以及要设置的选项字符串和要使用的值。可用的选项字符串将取决于运行项目的平台，如下所示：

选项平台描述价值
“ allow_rotation” tvOS设置是否将Siri Remote旋转锁定（即：只能垂直使用）。解锁时（值为 true），旋转控件将更改输入，例如，向左旋转90º将使触摸板的“右侧”返回“上”，依此类推……这样所有输入将相对于触摸板的方向设备。锁定时（值为 false），则输入将不会相对于控制器的位置进行更改，并且始终将其视为控制器处于垂直位置。布尔型
“ dpad_absolute” tvOS设置Siri Remote触摸板是接受绝对输入还是相对输入。在绝对模式下（值为 true）触摸板的中心被认为是（0,0）位置-即：模拟摇杆处于静止状态或dpad的中心所在位置-围绕此位置的移动将产生输入。设为相对值（false那么（0,0）位置将被认为是在打击垫上进行初次触摸的位置，并且所有输入都将从该位置生成，而不是在打击垫的中心。布尔型

句法：

gamepad_set_option(device, option_name, value);

争论描述
device要设置哪个游戏手柄设备“插槽”。
option_name要设置的选项的名称（字符串，请参见上表）
value将该选项设置为的值（可以是布尔值，实数，整数或字符串-参见上表）

返回值：

N/A

例子：

for(var i = 0; i < 12; ++i;)
    {
    if gamepad_is_connected(i) && gamepad_get_description(i) == "tvOS Siri Remote"
        {
        if gamepad_get_option(i, "allow_rotation") == false
            {
            gamepad_set_option(i, "allow_rotation", true);
            }
        }
    }

上面的代码遍历所有游戏手柄插槽，并在tvOS平台上检查“ Siri Remote”。如果检测到一个，则将遥控器设置为允许输入旋转。
