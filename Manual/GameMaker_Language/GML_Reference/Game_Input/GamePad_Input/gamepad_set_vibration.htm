gamepad_set_vibration

使用此功能，您可以设置游戏手柄电机的振动，其中任一电机的值都从0（无振动）到1（全振动）。请注意，此功能没有时间限制，因此您需要在给定时间过去后使用变量或警报来关闭振动（将电动机设置为0），否则游戏手柄将无限期地振动。

注意：此功能当前仅适用于标准Windows ， PS4和Xbox One目标模块。

句法：

gamepad_set_vibration(device, left_motor, right_motor);

争论描述
device要检查哪个游戏手柄设备的“插槽”。
left_motor左电机的振动量从0到1。
right_motor右马达从0到1的振动量。

返回值：

N/A

例子：

if lives = 0
   {
   gamepad_set_vibration(0, 1, 1);
   alarm[0] = room_speed / 2;
   }

上面的代码将（例如）在碰撞事件中使用，以使插入“ slot” 0的游戏手柄振动半秒钟，并使用设置的警报在该时间过后再次将其关闭。
