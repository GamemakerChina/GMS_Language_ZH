游戏输入

使游戏成为游戏的基本要素是与玩家的互动。这可以通过多种方式实现，包括计算机上的键盘和鼠标，触摸屏和移动设备上的倾斜设备，甚至通过所有平台上的游戏手柄！

首先要在GameMaker Studio 2使用PC上的专用鼠标或键盘事件，或在移动设备上使用手势事件，但有时您需要的控件比这些事件所提供的更多，或者您想要包括游戏手柄控件或多个不同的控件选项。为此，您可以使用不同的游戏输入功能来获取已按下或释放的按钮或键，以及获取屏幕上的触摸或移动了游戏手柄的数量。

游戏中有大量用于游戏输入和控制的功能，因此将它们分为与特定输入法有关的部分，以使查找所需内容变得更加容易。可用部分为：

  • 键盘输入
  • 鼠标输入
  • 设备输入
  • 手势输入
  • GamePad输入
  • 虚拟键和虚拟键盘
