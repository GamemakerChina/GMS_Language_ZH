event_data

此变量的作用域是全局变量，在“手势事件”中使用时用于保留DS地图，在其他所有时间都为-1。 DS地图的实际内容取决于触发它的手势事件的类型，因此请参阅这些事件的各个部分。

句法：

event_data;

返回值：

Real (integer value)

例子：

isFlick = event_data[?"isflick"];
if (isFlick)
    {
    flickVelX = event_data[?"diffX"];
    flickVelY = event_data[?"diffY"];
    }
else
    {
    flickVelX = 0;
    flickVelY = 0;
    }

上面的代码来自“拖动结束”手势事件，并检查通风口是否为“轻拂”事件，如果是，则从通风口中提取所需数据。event_data DS映射并使用它来设置一些变量。如果未检测到“轻拂”事件，则将相同的变量设置为0。
