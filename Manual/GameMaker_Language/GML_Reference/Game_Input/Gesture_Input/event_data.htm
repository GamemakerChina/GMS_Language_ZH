事件_data

这个变量是全局性的，在手势事件中使用时，用来保存DS Map，其他时候都是-1。 其他时候都是-1。DS地图的实际内容将取决于触发它的手势事件的类型，所以请参考这些事件的个别章节。

语法。

event_data;

返回。

实数（整数值

例子。

isFlick = event_data[? "isflick"];
if (isFlick)
{
flickVelX = event_data[? "diffX"];
flickVelY = event_data[? "diffY"];
}
else
{
flickVelX = 0;
flickVelY = 0;
}

上面的代码是从Drag Endgesture事件中提取的，它检查发泄物是否是一个 "flick "事件，如果是，它从event_dataDS map中提取所需的数据，并使用它来设置一些变量。如果没有检测到一个 "轻弹 "事件 事件，那么同样的变量被设置为0。
