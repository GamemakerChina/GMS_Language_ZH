gesture_double_tap_time

此功能用于设置两次触摸/单击之间触发“双击手势”事件所花费的时间。时间以秒为单位，默认值为0.16。

句法：

gesture_double_tap_time(time);

争论描述
time设置双击手势事件检测的时间（以秒为单位）。

返回值：

N/A

例子：

if gesture_get_double_tap_time() != 0.16
    {
    gesture_get_double_tap_time(0.16);
    }

上面的代码检查手势的双击时间是否设置为0.16秒，如果不是，则将其设置为该值。
