手势_拖曳时间

此功能用于设置拖动手势事件被触摸或点击触发的时间。这个时间也会影响到轻敲事件的触发方式 因为在这个时间结束前的触摸/点击和释放将被认为是一个轻敲。时间的单位是秒，默认值是0.16。

语法。

gesture_drag_time(time)。

争论 描述
时间 为拖动手势事件检测设置的时间（以秒为单位）。

返回。

不适用

例子。

if gesture_get_drag_time() != 0.16
{
gesture_drag_time(0.16);
}

上述代码检查手势的拖动时间是否设置为0.16秒，如果不是，则将其设置为该值。

返回。手势输入
