gesture_drag_time

此功能用于设置通过触摸或单击触发“拖动手势”事件所花费的时间。此时间还将影响在此时间结束之前通过触摸/单击和释放触发Tap事件的方式。时间以秒为单位，默认值为0.16。

句法：

gesture_drag_time(time);

争论描述
time设置拖动手势事件检测的时间（以秒为单位）。

返回值：

N/A

例子：

if gesture_get_drag_time() != 0.16
    {
    gesture_drag_time(0.16);
    }

上面的代码检查手势的拖动时间是否设置为0.16秒，如果不是，则将其设置为该值。
