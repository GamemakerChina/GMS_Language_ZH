gesture_get_drag_distance

此功能用于获取通过触摸或单击的移动触发“拖动手势”事件所花费的距离。距离以英寸为单位，默认值为0.1。

句法：

gesture_get_drag_distance();

返回值：

Real (inches)

例子：

if gesture_get_drag_distance() != 0.1
    {
    gesture_drag_distance(0.1);
    }

上面的代码检查手势的拖动距离是否设置为0.1英寸，如果不是，则将其设置为该值。
