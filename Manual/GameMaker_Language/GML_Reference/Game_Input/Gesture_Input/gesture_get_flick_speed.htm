gesture_get_flick_speed

此功能用于获取释放触摸或单击时触发“笔势手势”事件所需的速度。速度以每秒英寸数为单位，默认值为2.0。

句法：

gesture_get_flick_speed();

返回值：

Real (inches per second)

例子：

if gesture_get_flick_speed() != 2
    {
    gesture_flick_speed(2);
    }

上面的代码检查手势的滑动速度是否设置为每秒2英寸，如果不是，则将其设置为该值。
