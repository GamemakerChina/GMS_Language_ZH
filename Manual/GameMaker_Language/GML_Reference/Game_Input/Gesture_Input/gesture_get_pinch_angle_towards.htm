手势_获取夹角_朝向

此功能用于获得一个触摸必须向另一个触摸移动的角度，然后才有可能启动捏合手势。角度的测量单位是 度，默认值为45°。

语法。

gesture_get_pinch_angle_towards（）。

返回。

真实（英寸

例子。

if gesture_get_pinch_angle_towards() != 45
{
gesture_pinch_angle_towards(45);
}

上面的代码检查手势的捏合角度是否设置为45°，如果不是，则将其设置为该值。
