gesture_get_pinch_distance

此功能用于获取触发触摸手势之前必须触摸/单击屏幕并用两根手指移动的距离。距离以英寸为单位，默认值为0.1。

句法：

gesture_get_pinch_distance();

返回值：

Real (inches)

例子：

if gesture_get_pinch_distance() != 0.1
    {
    gesture_pinch_distance(0.1);
    }

上面的代码检查手势的捏紧距离是否设置为0.1英寸，如果不是，则将其设置为该值。
