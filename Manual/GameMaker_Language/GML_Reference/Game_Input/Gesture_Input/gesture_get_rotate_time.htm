gesture_get_rotate_time

此功能用于获取必须在一对触摸中沿一致的方向旋转才能触发“旋转开始手势”的时间。时间以秒为单位，默认值为0.16s。

重要的！当前，对于60fps的游戏，您只能将其设置为最大一秒，否则将不会检测到旋转。对于较低的帧率，这将增加，例如30fps游戏的最长时间为2秒。

句法：

gesture_get_rotate_time();

返回值：

Real (inches)

例子：

if gesture_get_rotate_time() != 0.1
    {
    gesture_rotate_time(0.1);
    }

上面的代码检查手势的旋转时间是否设置为0.1秒，如果不是，则将其设置为该值。
