gesture_get_tap_count

此功能用于检查是否启用了分接头计数。该函数将返回 true如果已启用，并且 false否则，您可以使用功能启用或禁用点击计数 gesture_tap_count()。启用后，每个点击事件将有一个附加的“ tapcount” DS映射条目，该条目将记录点击次数，并且启用后，这意味着将触发所有点击事件，即：两次点击将同时触发一次点击事件和双击事件，单击事件的点击次数为1，双击事件的点击次数为2。在经过两次双击检测设置的时间后，点击计数值将重置为0（请参见功能）。gesture_double_tap_time() ）。如果禁用了分接头计数，则直到经过两次分接头时间并且未检测到第二次分接头，才会注册初始分接头。请注意，默认情况下启用此功能。

句法：

gesture_get_tap_count();

返回值：

Bool

例子：

if !gesture_get_tap_count()
    {
    getsure_tap_count(true);
    }

上面的代码检查是否启用了分接头计数，如果未启用，则将其打开。
