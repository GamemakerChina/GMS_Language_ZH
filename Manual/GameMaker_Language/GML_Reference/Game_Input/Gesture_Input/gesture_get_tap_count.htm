手势_get_tap_count

该函数用于检查是否启用或禁用了分接计数。如果启用，该函数将返回 "true"，否则返回 "false"，你可以使用该函数启用或禁用分接计数。 gesture_tap_count(). 启用后，每个点触事件将有一个额外的 "tapcount "DS地图条目，该条目将记录点触次数，启用后意味着所有点触事件都将被触发，即：两次点触将同时触发单点触事件和双点触事件。 即：两次敲击将触发单次敲击事件和两次敲击事件，单次敲击事件的敲击次数为1，两次敲击事件的敲击次数为2。在为双击检测设定的时间过后，分接计数值将被重置为0（见函数 gesture_double_tap_time()). 如果分接计数被禁用，那么初始分接将不会被注册，直到双击时间过去且没有检测到第二个分接。请注意，这在默认情况下是启用的。

语法。

gesture_get_tap_count()。

返回。

Bool

例子。

if !gesture_get_tap_count()
{
getsure_tap_count(true)。
}

上面的代码检查是否启用了分接计数，如果没有则开启。
