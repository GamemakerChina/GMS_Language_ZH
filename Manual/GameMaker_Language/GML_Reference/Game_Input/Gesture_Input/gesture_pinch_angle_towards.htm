gesture_pinch_angle_towards

此功能用于设置在可能开始捏合手势之前触摸必须向另一触摸移动的角度。角度以度为单位，默认值为45°。

句法：

gesture_pinch_angle_towards();

返回值：

Real (inches)

例子：

if gesture_get_pinch_angle_towards() != 45
    {
    gesture_pinch_angle_towards(45);
    }

上面的代码检查手势的捏角度是否设置为45°，如果不是，则将其设置为该值。
