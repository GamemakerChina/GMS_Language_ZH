手势_旋转_时间

此功能用于设置时间，在此时间内，一对触手必须以一致的方向旋转，才能触发旋转开始手势。该时间是 以秒为单位，默认值为0.16秒。

重要的是!目前，对于60fps的游戏，你只能将其设置为最多一秒钟，否则将检测不到旋转。对于低帧率的游戏，这个时间会增加，例如，30帧的游戏可以有2秒的最大时间。

语法。

gesture_rotate_time()。

返回。

真实（英寸

例子。

if gesture_get_rotate_time() != 0.1
{
gesture_rotate_time(0.1);
}

上面的代码检查手势的旋转时间是否被设置为0.1秒，如果不是，则将其设置为该值。
