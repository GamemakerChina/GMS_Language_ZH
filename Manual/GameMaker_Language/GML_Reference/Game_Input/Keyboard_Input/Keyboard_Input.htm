键盘输入

GameMaker Studio 2处理键盘时，您可以使用多种功能来识别不同的键盘状态，例如按下或释放。还有一些将所有按键存储为字符串或可以告诉您最后一次按下的按键是什么，还有一些按键可以让您完全清除键盘状态。

注意：这些功能仅适用于Windows / Mac / Ubuntu桌面平台。您可能会发现某些内置变量和常量在其他平台上无效，并且许多功能在手机上不起作用。

一个键（或多个键）中的每个输入字符均由其UTF8代码定义，该代码是一个数值。可以使用 ord()功能，但GameMaker Studio 2还具有一系列 constants适用于最常用的键盘特殊键和特殊功能。通常，您会结合使用 ord()与 keyboard_check*()函数，如下所示：

if keyboard_check(ord("A"))
    {
    hspeed = -5;
    }

因此，上面的代码将检查“ A”键，如果按下该键，则会将对象的水平速度设置为-5。注意，使用 ord()仅当输入字符串的长度只有一个字符且为0到9的数字或从A到Z的大写罗马字符时，以这种方式才能正常工作。ord()将返回完整的UTF8值，但是 keyboard_check*()函数将仅检测A-Z和0-9 。但是，如果您想使用箭头键怎么办？还是要使用“ Shift”键修改动作？好吧，为此， GameMaker Studio 2具有一系列 vk_常数（vk_代表虚拟密钥），您可以用来代替 ord()：

以下是如何使用 vk_ 常数：

if keyboard_check_pressed(vk_tab)
    {
    instance_create_layer(x, y, "Controllers", obj_Menu);
    }

上面的代码将检测是否按下了“ Tab”键，并创建了一个对象“ obj_Menu”的实例。

如果您需要检查不是0-9，A-Z或VK常数之一的键字符，则应检查以下字符之一：keyboard_*变量，例如 keyboard_lastchar例如：

var _key = keyboard_lastchar;
if ord(_key) == ord("ç")
    {
    show_debug_message("ç key pressed");
    }
