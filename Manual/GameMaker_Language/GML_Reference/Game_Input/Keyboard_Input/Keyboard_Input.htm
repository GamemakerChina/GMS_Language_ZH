键盘输入

在GameMaker Studio 2中处理键盘时，你有各种函数可以用来识别不同的键盘状态，如按下或释放。也有一些函数将所有的按键存储为一个字符串，或者可以告诉你最后一个按键是什么。 告诉你最后一次按下的键是什么，还有一些允许你完全清除键盘状态。

注意：这些函数只为Windows/Mac/Ubuntu桌面平台设计。你可能会发现一些内置的变量和常量在其他平台上无效，许多函数在手机上无法使用。

一个键（或多个键）的每个输入字符都由其UTF8代码定义，这是一个数字值。这个值可以使用函数为任何字符检索 ord() 函数，但GameMaker Studio 2也有一系列的常量，用于最常用的键盘特殊键和特殊函数。通常，你会使用ord()与keyboard_check*()函数的组合。 像这样。

if keyboard_check(ord("A"))
{
hspeed = -5;
}

所以，上面将检查 "A "键，如果它被按下，那么它将把对象的水平速度设置为-5。注意，以这种方式使用ord()，只有在输入的字符串只有一个字符长度，并且是0到9的数字或大写的罗马字符的情况下才会正常工作。 函数ord()将返回一个完整的UTF8值，但keyboard_check*()函数只能检测A-Z和0-9。但是，如果 你想使用方向键呢？或者如果你想用 "shift "键来修改一个动作？那么，GameMaker Studio 2有一系列的vk_常量（vk_代表虚拟键），你可以用它来代替 ord()的位置。

下面是一个小例子，说明如何使用 vk_ 常量的小例子。

if keyboard_check_pressed(vk_tab)
{
instance_create_layer(x, y, "Controllers", obj_Menu);
}

上述代码将检测 "Tab "键是否被按下，如果被按下，将创建一个对象 "obj_Menu "的实例。

如果你需要检查一个不是0-9、A-Z或VK常数之一的按键字符，那么你应该检查键盘_*变量之一，例如键盘_lastchar。

var _key = keyboard_lastchar;
if ord(_key) == ord("ç")
{
show_debug_message("ç key pressed");
}

使用键盘工作时，可使用以下功能。

注意：在使用屏幕上的虚拟键盘时，这些功能将无法使用。

  • io_clear
  • 键盘_检查
  • 键盘_检查_按下
  • 键盘_检查_释放
  • keyboard_check_direct
  • 键盘_清空
  • keyboard_set_map
  • keyboard_get_map
  • 键盘_unset_map
  • keyboard_set_numlock
  • keyboard_get_numlock

还有一些功能可以用来模拟按键。

  • 键盘_按键_按压
  • 键盘_按键_释放
