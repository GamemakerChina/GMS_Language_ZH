io_clear

此功能将清除当前的键盘和鼠标状态。请注意，调用此函数后下一步将发生什么取决于所检查的内容和所检查的平台。使用鼠标输入时，必须释放并再次按下鼠标按钮，才能检测到任何其他输入，但是使用键盘并不是那么简单。通常，仅在该步骤清除键盘，但下一步将再次检测键，但是在所有平台和键盘上可能都不相同，并且在某些情况下，该功能的作用可能与鼠标和键盘相同。需要提起并再次按下键。

句法：

io_clear();

返回值：

N/A

例子：

io_clear();

这将清除所有键盘和鼠标状态。
