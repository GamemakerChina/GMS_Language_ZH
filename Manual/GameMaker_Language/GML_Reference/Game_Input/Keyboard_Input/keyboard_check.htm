keyboard_check

使用此功能，您可以检查按键是否被按下。不像 keyboard_check_pressed()或者 keyboard_check_released()这些功能仅在按下或释放键时触发一次，此功能在按下键的每一步都会触发。该函数将采用函数返回的键码值 ord()（仅限AZ的大写字母或0-9的数字），或任何 vk_*键盘输入主页面上列出的常量。

句法：

keyboard_check(key);

争论描述
key检查下降状态的关键。

返回值：

Boolean

例子：

if keyboard_check(vk_left)
   {
   x -= 5;
   }

上面的代码将检查是否已按下箭头键，并将实例返回true的每一步向左移动像素5个像素。
