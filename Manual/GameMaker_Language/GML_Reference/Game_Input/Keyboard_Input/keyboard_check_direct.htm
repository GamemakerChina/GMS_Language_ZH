keyboard_check_direct

该函数将返回 true如果按下具有特定键码的键，或者 false如果不是，则直接检查硬件。它允许进行更多检查，特别是您可以使用键码vk_lshift ， vk_lcontrol ， vk_lalt ， vk_rshift ， vk_rcontrol和vk_ralt来检查是否按下了左移键或右移键，控制键或alt键。该函数将采用函数返回的键码值 ord()（仅限AZ的大写字母或0-9的数字），或任何 vk_*键盘输入主页面上列出的常量。

注意：此功能仅适用于标准Windows目标，并且结果与哪个应用程序具有焦点无关。

句法：

keyboard_check_direct(key);

争论描述
key检查下降状态的关键。

返回值：

Boolean

例子：

if keyboard_check_direct(vk_ralt) || keyboard_check_direct(vk_lalt)
   {
   crouch = true;
   }

上面的代码将检查是否已按下向左或向右alt键，以及是否已将变量“ crouch”设置为true。
