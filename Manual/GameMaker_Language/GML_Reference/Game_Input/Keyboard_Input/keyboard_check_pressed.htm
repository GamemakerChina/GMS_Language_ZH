keyboard_check_pressed

使用此功能，您可以检查是否已按下某个键。不像 keyboard_check()功能，每次按下该键时该功能仅运行一次，因此要再次触发，必须先释放该键，然后再按一次。该函数将采用函数返回的键码值 ord()（仅限AZ的大写字母或0-9的数字），或任何 vk_*键盘输入主页面上列出的常量。

句法：

keyboard_check_pressed(key);

争论描述
key检查按下状态的键。

返回值：

Boolean

例子：

if keyboard_check_pressed(vk_anykey)
   {
   room_goto_next();
   }

如果玩家按下键盘上的任意键（类似于“按任意键继续”的提示），则上面的代码将前进到下一个房间。
