键盘_检查_释放

通过这个函数，你可以检查一个键是否已经被释放。不像 keyboard_check()函数，这个函数在每次按键被解除时只运行一次，所以要想再次触发它。 所以要再次触发，必须先按下钥匙，然后再释放。该函数将接受一个由函数返回的键码值 ord()(只有A-Z的大写字母或0-9的数字)，或任何一个 vk_*常量，这些常量列在键盘输入主页面上。

语法。

keyboard_check_released(key)。

争论 描述
钥匙 检查释放状态的关键。

返回。

布尔型

例子。

if keyboard_check_released(ord("P"))
{
instance_create_layer(0, 0, "Controllers", obj_Pause);
}

上面的代码将检查 "P "键是否被释放，如果是，就创建一个 "obj_Pause "的实例。
