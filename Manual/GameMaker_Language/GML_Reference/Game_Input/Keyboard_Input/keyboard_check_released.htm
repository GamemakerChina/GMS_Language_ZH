keyboard_check_released

使用此功能，您可以检查是否已释放密钥。不像 keyboard_check()功能，每次按下该键时该功能仅运行一次，因此要再次触发，必须先按下该键，然后再释放一次。该函数将采用函数返回的键码值 ord()（仅限AZ的大写字母或0-9的数字），或任何 vk_*键盘输入主页面上列出的常量。

句法：

keyboard_check_released(key);

争论描述
key检查释放状态的键。

返回值：

Boolean

例子：

if keyboard_check_released(ord("P"))
   {
   instance_create_layer(0, 0, "Controllers", obj_Pause);
   }

上面的代码将检查“ P”键是否已释放，如果已释放，则创建“ obj_Pause”的实例。
