keyboard_clear

使用此功能，您可以清除当前的键盘状态，这实质上意味着，如果按住该键，则在再次释放之前将不再识别该键（此时不会触发键盘键已释放事件）然后再按一次该函数将采用函数返回的键码值 ord()（仅限AZ的大写字母或0-9的数字），或任何 vk_*键盘输入主页面上列出的常量。

句法：

keyboard_clear(key);

争论描述
key清除的关键。

返回值：

N/A

例子：

keyboard_clear(vk_space);

上面的代码清除空格键的状态。
