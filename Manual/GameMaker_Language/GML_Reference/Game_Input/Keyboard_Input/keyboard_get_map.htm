keyboard_get_map

有时你可能希望得到一个被映射的键的ascii代码（例如，看它是否已经被映射），这就是这个函数的返回值。该函数将接受一个由函数返回的键码值 ord() (只有A-Z的大写字母或0-9的数字)，或主键盘输入页面上列出的任何vk_*常数。

语法。

keyboard_get_map(key)。

争论 描述
钥匙 这是你希望获得映射值的键。

返回。

真实的

例子。

if keyboard_get_map(ord("A")) = ord("A")
{
keyboard_set_map(ord("A"), keyboard_lastkey);
}

上面的例子代码将首先检查并查看 "A "是否已被映射到另一个键上，如果没有，它将把它映射到用户最后按过的键上。
