keyboard_get_map

有时您可能希望获取映射键的ascii代码（例如，查看它是否已被映射），这是此函数返回的内容。该函数将采用函数返回的键码值 ord()（仅限AZ的大写字母或0-9的数字），或任何 vk_*键盘输入主页面上列出的常量。

句法：

keyboard_get_map(key);

争论描述
key这是您希望从中获取映射值的关键

返回值：

Real

例子：

if keyboard_get_map(ord("A")) = ord("A")
   {
   keyboard_set_map(ord("A"), keyboard_lastkey);
   }

上面的示例代码将首先检查并查看“ A”是否已映射到另一个键，如果尚未映射，则将其映射到用户按下的最后一个键。
