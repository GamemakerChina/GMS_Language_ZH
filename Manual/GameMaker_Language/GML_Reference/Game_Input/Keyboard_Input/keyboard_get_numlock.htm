keyboard_get_numlock

您可以使用此功能查找键盘数字锁的状态，其中“ true”返回打开，“ false”返回关闭。

注意：此功能仅在Windows exe版本中可用，在其他任何设备上均不起作用。

句法：

keyboard_get_numlock();

返回值：

Boolean

例子：

if keyboard_get_numlock()
   {
   keyboard_set_numlock(false);
   }
else
   {
   keyboard_set_numlock(true);
   }

上面的示例代码将获取数字锁键的状态，如果该键处于打开状态（true），则将其设置为关闭（false），反之亦然。
