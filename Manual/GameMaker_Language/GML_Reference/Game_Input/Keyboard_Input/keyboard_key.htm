keyboard_key

使用此变量，您可以获得当前正在按下的键的键控代码，如果在完成检查时未按下任何键，它将返回0。

句法：

keyboard_key;

返回值：

Real

例子：

switch (keyboard_key)
   {
   case vk_numpad1: gun = weapon[0][0]; break;
   case vk_numpad2: gun = weapon[1][0]; break;
   case vk_numpad3: gun = weapon[2][0]; break;
   }

上面的代码使用了 keyboard_key变量以将变量设置为与数组相同的值。
