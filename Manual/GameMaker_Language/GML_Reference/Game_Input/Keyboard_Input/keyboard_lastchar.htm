keyboard_lastchar

此变量存储最后一次按下的字符串。该变量不是只读的，您可以更改它，例如，如果已经处理过，则将其设置为“”（空字符串）。

句法：

keyboard_lastchar;

返回值：

String

例子：

if keyboard_lastkey != -1
   {
   str += keyboard_lastchar;
   keyboard_lastkey = -1;
   }

上面的代码检查lastkey变量是否大于-1，如果是，则将最后一个键作为字符串添加到变量“ str”中。
