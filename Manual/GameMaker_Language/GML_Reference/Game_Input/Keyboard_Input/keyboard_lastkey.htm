keyboard_lastkey

此变量指的是 keyboard_key在上一帧中，返回该键的键码（返回所有标准键码常量）。该变量不是只读的，您可以更改它，例如，如果已经处理过，则将其设置为-1。

句法：

keyboard_lastkey;

返回值：

Real

例子：

if (keyboard_lastkey != -1)
   {
   str += keyboard_lastchar;
   keyboard_lastkey = -1;
   }

上面的代码检查lastkey变量是否不等于-1，如果是，则将最后一个键作为字符串添加到变量“ str”中，然后重置 keyboard_lastkey变量以接受进一步的输入。
