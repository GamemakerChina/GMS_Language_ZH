keyboard_set_numlock

您可以使用此功能来打开或关闭键盘数字锁定（将on设置为true ，将on设置为false ）。

注意：此功能仅在Windows目标版本中可用，在其他任何设备上均不起作用。

句法：

keyboard_set_numlock(value);

争论描述
value将其设置为true表示“开”，设置为false表示“关”

返回值：

N/A

例子：

if keyboard_get_numlock()
   {
   keyboard_set_numlock(false);
   }
else
   {
   keyboard_set_numlock(true);
   }

上面的示例代码将获取数字锁键的状态，如果该键处于打开状态（true），则将其设置为关闭（false），反之亦然。
