keyboard_string

此变量包含一个字符串，其中包含在键盘上键入的最后（最多）1024个字符。此字符串将只包含可打印字符类型，但它会正确地按压通过擦除的最后一个字符退格键响应。该变量不是只读的，您可以对其进行更改，例如，如果已经处理过，则将其设置为“”（空字符串），并且可以使用字符串函数来对其进行操作。请注意，在使用屏幕上的虚拟键盘时，只有此变量将使用键盘输入进行更新。

句法：

keyboard_string;

返回值：

String

例子：

if string_length(keyboard_string) > 15
   {
   keyboard_string = string_copy(keyboard_string, 1, 15);
   }

上面的代码将键盘字符串的长度限制为15个字符，通过将前15个字符复制回变量来删除超出该限制的字符。

背面：键盘输入
