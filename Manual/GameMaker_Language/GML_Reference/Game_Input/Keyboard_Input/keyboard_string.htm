键盘字符串

这个变量持有一个字符串，包含最后（最多）在键盘上输入的1024个字符。这个字符串只包含输入的可打印字符，但它会正确响应按退格键，删除最后一个字符。 这个变量不是只读的，你可以改变它，例如把它设置为""（一个空字符串），如果你已经处理了它，你可以使用字符串函数来操作它。需要注意的是 当使用屏幕上的虚拟键盘时，只有这个变量会随着键盘输入而被更新。

语法。

keyboard_string;

返回。

字符串

例子。

if string_length(keyboard_string) > 15
{
keyboard_string = string_copy(keyboard_string, 1, 15);
}

上述代码将限制键盘字符串的长度为15个字符，通过将前15个字符复制到变量中来删除那些超过该限制的字符。

返回。键盘输入
