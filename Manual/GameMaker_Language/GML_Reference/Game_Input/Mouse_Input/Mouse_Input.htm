鼠标输入

鼠标输入在所有平台上都被接受（在移动设备上，它被接受为单屏触摸。如果需要使用多点触摸，则应该使用设备特定的功能），并且具有一些用于指定按钮的常量被压。下表中显示了这些常量：

持续的描述
mb_left鼠标左键
mb_middle鼠标中键（可能不适用于所有目标平台）
mb_right鼠标右键
mb_none没有鼠标按键
mb_any任何鼠标按钮

标准鼠标按钮控件具有以下功能：

  • mouse_button
  • mouse_check_button
  • mouse_check_button_pressed
  • mouse_check_button_released
  • mouse_clear
  • mouse_lastbutton
  • mouse_wheel_up
  • mouse_wheel_down
  • mouse_x
  • mouse_y

请注意，还有一组窗口功能与在桌面目标上使用鼠标有关：

  • window_mouse_get_x
  • window_mouse_get_y
  • window_mouse_set
  • window_view_mouse_get_x
  • window_view_mouse_get_y
  • window_views_mouse_get_x
  • window_views_mouse_get_y
