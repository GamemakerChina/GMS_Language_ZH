鼠标输入

鼠标输入在所有平台上都被接受（在移动设备上，它被接受为单屏触摸--如果你需要使用多点触摸，你应该使用设备的特定功能），并有一些常数 用来指定被按下的按钮。这些常数在下表中显示。

恒定 描述
mb_left 鼠标左键
mb_middle 鼠标中键（这可能不是对所有目标平台都有效）。
mb_right 鼠标右键
mb_side1* 鼠标侧键1
mb_side2* 鼠标侧键2
mb_any 鼠标的任何一个按钮
mb_none 没有鼠标按钮


*注意：mb_side1和mb_side2 按钮只适用于Windows、macOS、Ubuntu和HTML5。

标准鼠标按钮控制存在以下功能。

  • 鼠标_按钮
  • 鼠标_检查按钮
  • 鼠标_检查_按钮_按下
  • 鼠标_检查_按钮_释放
  • 鼠标清空
  • 鼠标_lastbutton
  • 鼠标_轮子_上
  • 鼠标_滚轮_下
  • 鼠标_x
  • 鼠标_y

注意，还有一组与在桌面目标上使用鼠标有关的窗口功能。

  • 窗口_mouse_get_x
  • 窗口_mouse_get_y
  • 窗口_鼠标_设置
  • 窗口_view_mouse_get_x
  • 窗口_view_mouse_get_y
  • 窗口_views_mouse_get_x
  • 窗口_views_mouse_get_y
