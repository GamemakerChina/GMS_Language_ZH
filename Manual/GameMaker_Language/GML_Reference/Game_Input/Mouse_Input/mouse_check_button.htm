鼠标_检查按钮

如果被检查的鼠标按钮被按住，这个函数将返回 "true"，如果没有，则返回 "false"。你可以从以下常数中选择要检查的鼠标按钮。

恒定 描述
mb_left 鼠标左键
mb_middle 鼠标中键（这可能不是对所有目标平台都有效）。
mb_right 鼠标右键
mb_side1* 鼠标侧键1
mb_side2* 鼠标侧键2
mb_any 鼠标的任何一个按钮
mb_none 没有鼠标按钮


*注意：mb_side1和mb_side2 按钮只适用于Windows、macOS、Ubuntu和HTML5。

语法。

mouse_check_button(numb)。

争论 描述
哑巴 检查哪个鼠标按键常数。

返回。

布尔型

例子。

if mouse_check_button(mb_left)
{
instance_create_layer(mouse_x, mouse_y, "Effects", obj_Star);
}

上面的代码将检查鼠标左键，每按住一步就会创建一个以 "obj_Star "为索引的对象实例。
