mouse_check_button

该函数将返回 true如果按住选中的鼠标按钮，或者 false如果不是这样。您提供鼠标按钮以检查以下常量之一：

持续的描述
mb_left鼠标左键
mb_middle鼠标中键（可能不适用于所有目标平台）
mb_right鼠标右键
mb_none没有鼠标按键
mb_any任何鼠标按钮

句法：

mouse_check_button(numb);

争论描述
numb要检查的鼠标按键常数。

返回值：

Boolean

例子：

if mouse_check_button(mb_left)
   {
   instance_create_layer(mouse_x, mouse_y, "Effects", obj_Star);
   }

上面的代码将检查鼠标左键，按下该按钮的每一步都将创建在“ obj_Star”中索引的对象的实例。
