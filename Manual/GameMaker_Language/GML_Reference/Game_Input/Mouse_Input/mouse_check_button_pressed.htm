mouse_check_button_pressed

该函数将返回 true如果按下了选中的鼠标按钮，或者 false如果没有。第一次按下任何鼠标按钮时，此功能仅会触发一次，要再次触发，则需要释放并再次按下该按钮。请注意，它将在该步骤的持续时间内以及具有任何鼠标事件或使用此相同功能的所有实例被视为已触发。

您提供鼠标按钮以检查以下常量之一：

持续的描述
mb_left鼠标左键
mb_middle鼠标中键（可能不适用于所有目标平台）
mb_right鼠标右键
mb_none没有鼠标按键
mb_any任何鼠标按钮

句法：

mouse_check_button_pressed(numb);

争论描述
numb要检查的鼠标按键常数。

返回值：

Boolean

例子：

if mouse_check_button_pressed(mb_left)
   {
   score += 50;
   }

上面的代码将检查是否已按下鼠标左键，如果已按下，则得分将增加50。
