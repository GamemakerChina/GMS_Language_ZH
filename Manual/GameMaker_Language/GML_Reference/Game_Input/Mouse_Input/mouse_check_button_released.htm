mouse_check_button_released

该函数将返回 true如果被检查的鼠标按钮已经释放，或者 false如果没有。释放任何鼠标按钮时，仅会触发一次此功能，然后再次按下该按钮才能再次触发该功能。您提供鼠标按钮以检查以下常量之一：

持续的描述
mb_left鼠标左键
mb_middle鼠标中键（可能不适用于所有目标平台）
mb_right鼠标右键
mb_none没有鼠标按键
mb_any任何鼠标按钮

句法：

mouse_check_button_released(numb);

争论描述
numb要检查的鼠标按键常数。

返回值：

Boolean

例子：

if mouse_check_button_released(mb_right)
   {
   speed = point_distance(x, y, mouse_x, mouse_y) / 10;
   }

上面的代码将检查鼠标右键是否已释放，如果已释放，它将实例的速度设置为当前x / y位置和鼠标x / y位置之间距离的十分之一。
