mouse_clear

此功能将清除正在使用的任何鼠标按钮的当前状态。这意味着被压低的支票不会退还 true直到播放器释放并按下按钮为止（但如果在按住鼠标按钮的同时进行清除操作，仍会检测到释放状态）。您提供鼠标按钮以检查以下常量之一：

持续的描述
mb_left鼠标左键
mb_middle鼠标中键（可能不适用于所有目标平台）
mb_right鼠标右键
mb_none没有鼠标按键
mb_any任何鼠标按钮

句法：

mouse_clear(button);

争论描述
button要检查的鼠标按键常数。

返回值：

Boolean

例子：

mouse_clear(mb_any);

上面的代码将清除所有鼠标按钮的按下状态。
