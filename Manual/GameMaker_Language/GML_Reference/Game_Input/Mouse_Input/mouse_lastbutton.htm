mouse_lastbutton

该变量返回按下的最后一个鼠标按钮，并且可以返回任何特殊的鼠标常量（您也可以将此变量也设置为常量之一）。返回（或设置）的常数将是以下之一：

持续的描述
mb_left鼠标左键
mb_middle鼠标中键（可能不适用于所有目标平台）
mb_right鼠标右键
mb_none没有鼠标按键

句法：

mouse_lastbutton;

返回值：

Constant

例子：

if mouse_lastbutton = mb_left
   {
   x -= 1;
   }

如果最后一次按下的按钮是鼠标左键，则将当前实例向左移动。
