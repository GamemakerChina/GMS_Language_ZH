keyboard_virtual_height

此函数将根据显示器的大小返回虚拟键盘的当前高度（以像素为单位）。如果键盘不可见，将返回0。

句法

keyboard_virtual_height();

返回值：

Int

例子：

if keyboard_virtual_status() == true
    {
    key_h = keyboard_virtual_height();
    }

上面的代码将检查OS虚拟键盘的状态，如果可见，则将变量设置为键盘的高度。

返回：虚拟键和键盘
