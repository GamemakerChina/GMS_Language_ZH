keyboard_virtual_hide

此功能可用于在运行游戏的设备上隐藏虚拟键盘。调用此函数将生成一个系统异步事件，其中 async_loadDS映射将填充以下键/值对：

  • “ event_type ”-由虚拟键盘功能触发时将为“虚拟键盘状态”。
  • “ screen_height ”-虚拟键盘的高度（以像素为单位）。如果键盘不可见，则该值为0。
  • “ keyboard_status ”-键盘的当前状态，以下列字符串之一返回：
    • “隐藏”
    • “隐”
    • “正在显示”
    • “可见的”

句法

keyboard_virtual_hide();

返回值：

N/A

例子：

if input == true
    {
    input = false;
    keyboard_virtual_hide();
    }

如果给定变量未设置为，则上面的代码将隐藏OS虚拟键盘。false 。

返回：虚拟键和键盘
