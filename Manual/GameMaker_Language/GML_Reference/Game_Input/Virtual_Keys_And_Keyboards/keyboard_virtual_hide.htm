键盘_虚拟隐藏

这个函数可以用来隐藏运行游戏的设备上的虚拟键盘。调用这个函数将产生一个系统异步事件，其中的 async_loadDS map将被填充以下键/值对。

  • " event_type" -当被虚拟键盘功能触发时，这将是 " 虚拟键盘状态 "。
  • "screen_height" - 虚拟键盘的高度（像素）。如果键盘是不可见的，这将是0。
  • "keyboard_status" - 键盘的当前状态，以下列字符串之一返回。
    • "隐藏"
    • "隐藏"
    • "显示"
    • "可见"

语法

keyboard_virtual_hide()。

返回。

不适用

例子。

如果input == true
{
input = false;
keyboard_virtual_hide()。
}

如果给定的变量没有设置为false，上述代码将隐藏操作系统的虚拟键盘。
