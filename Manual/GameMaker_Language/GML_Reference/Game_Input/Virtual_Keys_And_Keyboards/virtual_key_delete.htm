virtual_key_delete

如果您的游戏具有不同的房间或实例来控制菜单，游戏玩法和其他事物，则可能需要在某个时候更改虚拟键的位置和键图。为此，您可以使用此功能在创建新旧功能之前删除旧旧功能（如有必要）。该功能要求您提供要删除的虚拟密钥的ID-由virtual_key_add()。

请注意，更改房间后，所有虚拟钥匙都会自动从房间中删除，因此仅当您希望在房间本身完成之前手动删除钥匙时，才需要使用此功能。

句法

virtual_key_delete(index);

争论描述
index要删除的虚拟键的索引。

返回值：

N/A

例子：

virtual_key_delete(global.Left);

上面的代码删除了在全局变量“ global.Left”中索引的虚拟键。

返回：虚拟键和键盘
