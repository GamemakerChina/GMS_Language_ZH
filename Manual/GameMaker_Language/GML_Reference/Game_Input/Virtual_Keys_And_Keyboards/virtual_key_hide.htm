virtual_key_hide

为设备创建虚拟密钥后，您可能需要调试和测试虚拟密钥的位置正确并可以正常工作，为此您将调用该函数 virtual_key_show()。但是，您可能需要打开或关闭此视图，因此也可以使用要隐藏的虚拟键的索引（以前使用来创建和存储该功能）来使用此功能隐藏虚拟键。virtual_key_add() ）。对事物感到满意后，建议您创建自己的图形并将其放置在其区域中。

句法

virtual_key_hide(index);

争论描述
index隐藏的虚拟键的索引。

返回值：

N/A

例子：

if global.Debug
   {
   virtual_key_show(global.Left);
   }
else
   {
   virtual_key_hide(global.Left);
   }

上面的代码检查全局变量“ global.Debug”，如果测试为true，则将在屏幕上绘制索引为变量“ global.Left”的虚拟键，如果为false，则该键将被隐藏。
