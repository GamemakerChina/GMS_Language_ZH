virtual_key_show

为设备创建虚拟密钥后，您可能需要调试和测试虚拟密钥的位置正确并且工作正常。为此，您可以使用要查看的虚拟键的索引来调用此函数（以前使用 virtual_key_add()），它将绘制一个彩色矩形以表示屏幕上的键。对事物感到满意后，建议您创建自己的图形并将其放置在其区域中。

注意：绘制的矩形将受到相应绘制功能设置的混合，颜色和Alpha选项的影响。

句法

virtual_key_show(index);

争论描述
index要显示的虚拟键的索引。

返回值：

N/A

例子：

if global.Debug
   {
   virtual_key_show(global.Left);
   }
else
   {
   virtual_key_hide(global.Left);
   }

上面的代码检查全局变量“ global.Debug”，如果测试为true，则将在屏幕上绘制索引为变量“ global.Left”的虚拟键，如果为false，则该键将被隐藏。
