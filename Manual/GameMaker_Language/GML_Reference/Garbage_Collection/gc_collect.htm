gc_collect

使用此功能，您可以触发垃圾收集器，强制其在当前帧的末尾运行（步骤）。值得一提的是，垃圾收集器并不需要为这是积极的工作。禁用垃圾收集器后调用此函数（使用该函数 gc_enable()）将启用一帧垃圾收集器，在该帧中，已标记为要收集的所有对象将在再次禁用垃圾收集器之前从内存中删除。

句法：

gc_collect();

返回值：

N/A

例子：

if (global.debug == true && keyboard_check_pressed(vk_f1))
    {
    gc_collect();
    }

上面的代码检查一个全局变量和一个按键是否被按下 true然后在帧结束时触发垃圾收集（步骤）。
