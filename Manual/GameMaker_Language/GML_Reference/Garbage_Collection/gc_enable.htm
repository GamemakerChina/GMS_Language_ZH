gc_enable

使用此功能，您可以启用或禁用垃圾收集器。用调用函数 true作为参数启用它并使用 false禁用它（不推荐）。默认情况下启用。

句法：

gc_enable(enable);

争论描述
enable使能够 （true ）或禁用（false ）垃圾收集器。

返回值：

N/A

例子：

if (global.debug == true)
    {
    gc_enable(false);
    }

如果给定的全局变量为，上面的代码将禁用垃圾回收。true 。
