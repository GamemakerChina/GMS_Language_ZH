gc_enable

通过这个函数，你可以启用或禁用垃圾收集器。以"true"作为参数调用该函数可以启用它，以 "false"作为参数可以禁用它（不推荐）。默认情况下，它是启用的。

语法。

gc_enable(启用)。

争论 描述
启用 启用（true）或禁用（false）垃圾收集器。

返回。

不适用

例子。

如果（global.debug == true）
{
gc_enable(false)。
}

如果给定的全局变量为"真"，上述代码将禁用垃圾回收。
