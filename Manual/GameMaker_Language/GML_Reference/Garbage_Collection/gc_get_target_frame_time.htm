gc_get_target_frame_time

使用此功能，您可以检索垃圾收集器的当前目标帧值。返回的值以微秒为单位（其中1,000,000微秒等于一秒），默认目标帧时间为100微秒。如果要更改此值，则应使用该功能gc_target_frame_time()。

句法：

gc_get_target_frame_time(time);

返回值：

Real

例子：

if (gc_get_target_frame_time() != 50)
    {
    gc_target_frame_time(50);
    }

上面的代码检查垃圾回收器的当前帧时间目标，如果不是50微秒，则将其设置为该值。
