gc_target_frame_time

使用此功能，您可以向垃圾收集器指示它的目标是花费不超过指定的运行每个帧的时间。该函数采用以微秒为单位指定的时间值（其中1,000,000微秒等于一秒），默认目标帧时间为100微秒。

请注意，这只是一个目标值，因为某些操作仍然需要完全在一帧中运行，有时可能会花费比目标时间更长的时间。还要注意，增加目标时间将使垃圾回收器对内存使用的快速变化更加敏感，并且将导致更快地释放内存，尽管实际上在大多数情况下不太可能需要这样做。另请注意，将目标帧时间设置为0不会导致收集器完全停止-它仍将在每帧中进行最少的工作。要完全禁用收集器，请使用 gc_enable()功能。

句法：

gc_target_frame_time(time);

争论描述
time垃圾收集器每帧工作的目标时间（以微秒为单位）

返回值：

N/A (undefined)

例子：

if (gc_get_target_frame_time() != 50)
    {
    gc_target_frame_time(50);
    }

上面的代码检查垃圾回收器的当前帧时间目标，如果不是50微秒，则将其设置为该值。
