weak_ref_create

使用此功能，您可以检查对结构的弱引用，以查看其是否仍然“有效”。您提供弱引用进行检查（由函数返回）weak_ref_create() ），该函数将返回 true如果该结构仍在某处被引用或 false如果不是，并且已经被垃圾回收了。请注意，如果您提供的值不是弱引用，则该函数将返回 undefined。

句法：

weak_ref_alive(weak_ref);

争论描述
weak_ref对要检查的结构的弱引用。

返回值：

Boolean (or undefined)

例子：

if weak_ref_alive(inventory_ref)
    {
    inventory = -1;
    }

上面的代码检查对结构的弱引用，如果它仍然存在，则将引用它的变量设置为-1，以取消对该结构的引用并使其能够被垃圾回收。

背面：垃圾回收
