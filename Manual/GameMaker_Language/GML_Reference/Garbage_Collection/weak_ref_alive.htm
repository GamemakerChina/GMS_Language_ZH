弱引用创建

通过这个函数，你可以检查一个结构的弱引用，看看它是否仍然 "活着"。您可以提供要检查的弱引用（如由函数 weak_ref_create())，如果该结构仍在某处被引用，该函数将返回 true ，如果没有被引用且已被垃圾回收，则返回 false 。 请注意，如果您提供的值不是弱引用，该函数将返回 undefined 。

语法。

weak_ref_alive(weak_ref)。

争论 描述
弱参考 您要检查的结构的弱引用。

返回。

布尔型（或未定义）。

例子。

if weak_ref_alive(inventory_ref)
{
inventory = -1;
}

上面的代码检查了一个结构的弱引用，如果它仍然存在，那么引用它的变量被设置为-1，取消对结构的引用，使其能够被垃圾回收。
