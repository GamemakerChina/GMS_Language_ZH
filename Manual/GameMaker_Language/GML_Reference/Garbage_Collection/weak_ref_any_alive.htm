weak_ref_create

使用此功能，您可以检查对各种结构的弱引用，以查看它们是否仍然“有效”。您提供了一组弱引用进行检查（每个弱引用都应该使用函数创建weak_ref_create()），该函数将返回 true如果任何结构仍在某处被引用或 false如果不是，并且已经被垃圾收集了。请注意，如果您提供的数组中的任何值都不是弱引用，则该函数将返回 undefined。

请注意，该函数还带有两个可选参数，其中第一个允许您向数组提供初始索引以进行检查，以及一个用于设置要检查的索引的位置数（长度）的参数。仅检查那些参数内的数组索引。

句法：

weak_ref_any_alive(weak_ref, [index], [length]);

争论描述
weak_ref对要检查的结构的弱引用。
[index]选修的！数组索引开始检查
[length]选修的！从给定索引到要检查的头数

返回值：

Boolean (or undefined)

例子：

if weak_ref_any_alive(inventory_ref_array)
    {
    instance_destroy(obj_Inventory_Control);
    }

上面的代码检查弱引用的数组，如果还有弱引用，则实例将被销毁。
