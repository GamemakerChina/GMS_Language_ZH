weak_ref_create

使用此功能，您可以创建对结构的弱引用，然后可以使用该弱引用来检查该结构在游戏中是否仍“处于活动状态”（被引用）。您提供对要跟踪的结构的引用，该函数将返回另一个结构，该结构是对该结构的弱引用。请注意，您可以使用函数检查引用是“强”还是“弱” instanceof() ，作为强引用将返回“struct “或创建该结构的构造函数的名称，或”weakref “如果它是弱引用。还请注意，弱引用结构将具有一个变量”ref ”，除非已对其进行了垃圾回收，否则可以访问该字符串以获取对该结构的字符串引用，在这种情况下它将返回 undefined。

句法：

weak_ref_create(struct_to_track);

争论描述
struct_to_track您要为其创建弱引用的结构

返回值：

Struct

例子：

inventory_ref = weak_ref_create(inventory);

上面的代码创建了一个对结构的弱引用，并将其存储在实例变量中以供以后使用。

背面：垃圾回收
