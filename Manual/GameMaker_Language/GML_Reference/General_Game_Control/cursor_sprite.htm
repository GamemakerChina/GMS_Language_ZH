cursor_sprite

设置此变量将指示GameMaker Studio 2使用指定的精灵作为光标（基本上将其每步设置为当前鼠标的x / y位置）。默认值为-1，它不是光标的精灵，但是您可以从游戏资产中分配任何精灵索引，也可以从外部资源中导入任何精灵索引。请注意，无法控制动画速度或image_index，因此，如果子画面具有子图像，则这些子图像将以与房间速度相同的速度循环播放。要删除光标精灵，可以再次将此变量设置为-1。

还值得一提的是，这个变量不会取代游戏窗口的光标，它仍然会被绘制成正常的。为了避免这种情况，您可以使用函数 window_set_cursor()并将其设置为常数 cr_none这将使标准光标不可见。

句法：

cursor_sprite;

返回值：

Real

例子：

cursor_sprite = spr_CustomCursor;

上面的代码将在变量“ spr_CustomCursor”中索引的精灵设置为游戏的光标精灵。
