game_end

使用此功能，您可以结束游戏（并且将触发游戏结束事件）。这不会立即发生，而是在当前步骤结束时发生，因此在调用此函数之后，您在同一步骤中拥有的任何代码仍将运行。请注意，此功能具有以下限制：

  • 在Android设备上，调用 game_end()会将应用推入后台，但不会关闭该应用。这必须由用户完成。
  • 在iOS它什么也不会做，只会默默地失败。
  • 在UWP，XBoxOne，PS4和Nintendo Switch它可能会使游戏崩溃（或至少会无声地失败），因此不能使用。
  • 在HTML5上，它将结束游戏，但会让用户盯着网页上的空白画布，因此，建议不要在目标平台上使用它。
  • 在Windows ， Linux和Mac上，该功能仅结束游戏并关闭游戏窗口（将触发游戏结束事件）。


句法：

game_end();

返回值：

N/A

例子：

if keyboard_check_pressed(vk_escape) game_end();

如果玩家按下“退出”键，将结束游戏。
