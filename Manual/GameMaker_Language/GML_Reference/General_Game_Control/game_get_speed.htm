game_get_speed

此功能可用于获取游戏速度，即每秒运行的游戏帧数或每个游戏帧的微秒数。请注意，这不是实际的运行速度FPS值（为此，请使用 fps_real变量），而是游戏每秒尝试维护的游戏帧数（FPS），或者游戏试图维持的每个游戏帧的长度（以微秒为单位）（MPF）。使用此函数时，需要提供以下常量之一，这些常量将确定返回值的类型：

持续的描述
gamespeed_fps使用每秒帧数获取游戏速度。
gamespeed_microseconds以每帧微秒为单位获取游戏速度。

因此，例如，如果在“游戏选项”中将游戏速度设置为30，并且您使用的是FPS类型，则该函数将返回30，但如果使用MPF，则该函数将返回33333。

句法：

game_get_speed(type);

争论描述
type用于获得游戏速度的方法类型（请参见上面的常数）。

返回值：

Real

例子：

if game_get_speed(gamespeed_fps) != 60
    {
    game_set_speed(60, gamespeed_fps);
    }

上面的代码检查游戏是否以60 FPS的速度运行，如果没有，则将其设置为60 FPS。
