game_id

此只读变量返回您创建的游戏的唯一标识符。如果您需要唯一的文件名，或者需要其他任何东西来仅标识您的游戏，则可以使用此名称。可以在“游戏选项”中设置。

句法：

game_id

返回值：

Real

例子：

ini_open("Score.ini");
ini_write_real("Scores","0",score + game_id);
ini_close();

上面的代码通过在保存之前将game_id添加到得分上来对得分进行非常基本的加密。将其读回游戏后，您将扣除game_id以再次获得正确的值。
