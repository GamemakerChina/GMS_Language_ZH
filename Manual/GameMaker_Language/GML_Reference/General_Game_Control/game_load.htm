game_load

使用此功能，您可以加载以前保存的游戏（使用 game_save()）。

句法：

game_load(filename);

争论描述
filename要加载的文件的名称。

返回值：

N/A

例子：

if keyboard_check_pressed(ord("L"))
    {
    if global.Save game_load("Save.dat");
    }

如果全局变量为，则上面的代码将加载以前保存的游戏 true当玩家按下“ L”键时。
