game_load_buffer

使用此功能，您可以加载以前保存的游戏状态。游戏是从先前创建的“增长”缓冲区加载的（请参见Buffers ），并且该缓冲区必须已使用来保存了游戏状态 game_save_buffer()功能。

句法：

game_load_buffer(buffer);

争论描述
buffer要从中加载的缓冲区ID。

返回值：

N/A

例子：

if keyboard_check_pressed(ord("L"))
    {
    if global.Checkpoint game_load_buffer(save_buff);
    }

上面的代码将从全局变量为“ save_buff”中索引的缓冲区中加载以前保存的游戏状态，true ，当播放器按“ L”键时。

背面：一般游戏控制
