game_project_name

此只读变量以目标平台的“保存友好”格式返回目标平台游戏的显示名称。如果显示名称包含文件名不允许的任何字符，则它们将自动替换为“ _”。显示名称可以在“游戏选项”中设置。请注意，由于对HTML5的文件名没有限制，因此该字符串可能与 game_display_name。

句法：

game_project_name

返回值：

String

例子：

var file = game_project_name + ".ini";
ini_open(file);
ini_write_real("Scores", "Highscore", score);
ini_close();

上面的代码获取项目名称，并使用它打开（或创建）ini文件，然后将其写入。
