游戏_项目名称

这个只读变量以目标平台的 "友好保存 "格式返回你的游戏在目标平台的显示名称。如果显示名称包含任何不允许用于文件名的字符，它们将被自动替换为"_"。 会被自动替换成"_"。显示名称可以在游戏选项中设置。请注意，由于HTML5对文件名没有限制，这个字符串可能会与 与由 游戏显示名称.
