重启游戏

通过这个功能，你可以重新启动游戏。这本质上与第一次运行游戏相同，因此将触发游戏开始事件，以及 游戏结束事件。但是应该注意的是，当这个函数被调用时，某些东西将不会被重置。

  • 全局变量不会被重新初始化，除非有明确的编码--例如，如果游戏中已经被修改过，那么内置的全局变量score在游戏重启后不会从0开始。
  • GPU的状态将不会被改变（因此，如果你已经设置了绘制颜色或alpha，例如，它将保持在改变后的值）。
  • 游戏速度将保持在你在游戏代码中设置的水平（如果你改变了它，这种改变将被延续）。
  • 资产浏览器中的任何资产如果在游戏中的运行时间被改变了--例如，如果你改变了精灵资源的原点或移动了路径资源的位置--将不会被重置。
  • 缓冲区、表面、数据结构或导入的精灵等动态资源也不会被清理或删除（尽管你可能会失去对它们的引用，所以在使用这个函数时要注意，要么为动态资源使用全局引用。 或者在调用该函数之前将它们清理掉）。)

语法。

game_restart()。

返回。

不适用

例子。

if keyboard_check_pressed(ord("R")) game_restart()。

这将在玩家按下 "R "键时重新启动游戏。
