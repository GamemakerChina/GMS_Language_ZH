game_restart

使用此功能，您可以重新启动游戏。这与第一次运行游戏基本相同，因此将触发“游戏开始事件”以及“游戏结束事件” 。但是应注意，调用此函数时某些事情不会重置：

  • 除非明确编码，否则不会重新初始化全局变量-例如，内置全局变量 score重新启动游戏后，如果已经在游戏中进行了修改，它将不会从零开始。
  • GPU状态不会更改（例如，如果您设置了绘制颜色或Alpha，则它将保持更改后的值）。
  • 游戏速度将保持为您在游戏代码中设置的速度（如果更改，则此更改将永久保留）。
  • 资产浏览器中在游戏运行时已更改的任何资产（例如，如果您更改了精灵资源的原点或移动了路径资源的位置）都不会被重置。
  • 诸如缓冲区，表面，数据结构或导入的精灵之类的动态资源也不会被清除或删除（尽管您可能会丢失对它们的引用，因此在使用此函数为动态资源使用全局引用或进行清理时应格外小心它们被调用之前）。

句法：

game_restart();

返回值：

N/A

例子：

if keyboard_check_pressed(ord("R")) game_restart();

当玩家按下“ R”键时，这将重新启动游戏。
