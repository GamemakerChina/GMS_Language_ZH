game_save

使用此功能，您可以将游戏基本保存到给定文件中。

注意：此功能非常有限，仅供初学者快速启动并运行保存系统，但由于游戏无法保存，因此更高级的用户可能更喜欢使用File功能编写自己的系统代码。任何动态资源，例如数据结构，表面，添加的精灵等。

句法：

game_save(filename);

争论描述
filename将游戏保存到的文件名。

返回值：

N/A

例子：

if keyboard_check_pressed(ord("S"))
    {
    global.Saved = true;
    game_save("Save.dat");
    }

上面的代码将全局变量设置为 true然后在按下“ S”键时将游戏保存到“ Save.dat”文件中。

背面：一般游戏控制
