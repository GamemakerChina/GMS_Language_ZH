game_save_buffer

使用此功能，您可以将游戏的当前状态保存到先前创建的“增长”缓冲区（请参阅Buffers），然后可以使用随时重新加载该缓冲区。game_load_buffer()功能。此功能旨在一次在单个房间中使用（即：不建议在一个房间中保存缓冲区，然后尝试从另一个房间中加载缓冲区），并且理想情况下，仅应将其用于检查点或重新启动级别。

注意：此功能非常有限，仅供初学者快速启动并运行检查点系统，但由于游戏无法保存，因此更高级的用户可能更喜欢使用“文件”功能编写自己的系统代码。任何动态资源，例如数据结构，表面，添加的精灵等。

句法：

game_save_buffer(buffer);

争论描述
buffer要保存到的缓冲区ID。

返回值：

N/A

例子：

if keyboard_check_pressed(ord("S"))
    {
    global.Checkpoint = true;
    game_save_buffer(save_buff);
    }

上面的代码将全局变量设置为 true然后在按下“ S”键时将当前游戏状态保存到在变量“ save_buff”中索引的缓冲区中。
