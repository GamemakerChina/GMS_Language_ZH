game_save_id

这个只读变量将返回你的游戏用来保存文件的目录的完整路径ID。这个目录对其他应用程序可能是可见的，也可能是不可见的，这取决于平台，而在HTML5目标上，它将返回 一个空字符串。

注意：这个变量纯粹是为了提供信息，不应该用来获取文件路径或其他类似的东西，因为它不会在所有平台上正确工作。

语法。

game_save_id

返回。

字符串

例子。

save_dir = game_save_id;

这将把保存文件的目录存储到一个变量中，以便将来使用。

返回。一般游戏控制
