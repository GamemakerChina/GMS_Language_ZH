game_set_speed

此功能可用于设置游戏速度。您可以使用以下两种常数之一，以两种方式之一设置此值-设置为每秒游戏帧（FPS）或设置为每游戏帧微秒（MPF）：

持续的描述
gamespeed_fps使用每秒帧数设置游戏速度。
gamespeed_microseconds使用每帧微秒设置游戏速度。

因此，例如，每秒30帧的游戏速度将是每个游戏游戏帧33333微秒，然后由该函数表示为：

game_set_speed(30, gamespeed_fps);

或者：

game_set_speed(33333, gamespeed_microseconds);

句法：

game_set_speed(speed, type);

争论描述
speed新游戏速度（以FPS或MPF表示）。
type用来设置游戏速度的方法类型（请参见上面的常数）。

返回值：

N/A

例子：

if os_browser == browser_not_a_browser
    {
    game_set_speed(60, gamespeed_fps);
    }
else
    {
    game_set_speed(30, gamespeed_fps);
    }

上面的代码检查游戏是否在浏览器中运行，并将游戏速度相应地设置为FPS值。
