游戏_设置速度

这个函数可以用来设置游戏速度。你可以用以下两种方式之一来设置--以每秒游戏帧数（FPS）或以每一游戏帧的微秒数（MPF）--使用以下两个常数之一。

恒定 描述
游戏速度_fps 用每秒的帧数设置游戏速度。
速率_microseconds 使用每帧微秒设置游戏速度。

因此，例如，游戏速度为每秒30帧，则每一游戏游戏帧为33333微秒，然后由这个函数表示为：或者。

game_set_speed(30, gamespeed_fps)。

或。

game_set_speed(33333, gamespeed_microseconds)。

语法。

game_set_speed(speed, type)。

争论 描述
速度 新的游戏速度（作为FPS或MPF）。
类型 用于设置游戏速度的方法的类型（见上面的常数）。

返回。

不适用

例子。

如果os_browser == browser_not_a_browser
{
game_set_speed(60, gamespeed_fps);
}
else
{
game_set_speed(30, gamespeed_fps);
}

上述代码检查游戏是否在浏览器中运行，并相应地设置游戏速度为FPS值。

返回。一般游戏控制
