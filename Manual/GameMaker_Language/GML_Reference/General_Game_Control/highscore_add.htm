highscore_add

使用此功能，您可以在内部全局高分列表中添加名称和分数。无需检查该值是否足够高以进入得分列表，因为GameMaker Studio 2将仅存储那些大于存储的第十位的值。

句法：

highscore_add(str, numb);

争论描述
str将分数归因于的字符串（名称）
numb要添加的数字（分数）。

返回值：

N/A

例子：

if lives = 0
    {
    highscore_add(global.Name, score);
    score = 0;
    room_goto(rm_MainMenu);
    }

上面的代码将检查生命，如果将它们设置为0，它将把当前分数和全局变量“名称”中保存的字符串添加到高分列表中，最后将分数设置为0并将玩家发送到在“ rm_MainMenu”中索引的房间。

背面：一般游戏控制
