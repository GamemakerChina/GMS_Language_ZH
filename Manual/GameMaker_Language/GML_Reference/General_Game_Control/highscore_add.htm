高分_添加

通过这个函数，你可以将一个名字和一个分数添加到内部全局高分列表中。不需要检查数值是否高到可以进入分数列表，因为GameMaker Studio 2只会存储那些大于第十位的数值。 大于第十个位置的值。

语法。

highscore_add(str, numb)。

争论 描述
弦外之音 赋予分数的字符串（名称）。
哑巴 要添加的数字（分数）。

返回。

不适用

例子。

如果生命=0
{
highscore_add(global.Name, score);
score = 0;
room_goto(rm_MainMenu) 。
}

上述代码将检查生命值，如果它们被设置为0，它将把当前分数和全局变量 "姓名 "中的字符串添加到高分列表中，最后将分数设置为0，并将玩家送入房间 rm_MainMenu "中的索引。

返回。一般游戏控制
