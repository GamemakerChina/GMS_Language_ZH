highscore_name

使用此功能，您可以检索在给定位置的高分列表中存储的名称字符串。如果未输入任何名称，则将返回字符串“ Unknown”。

句法：

highscore_name(place);

争论描述
place桌子上的位置（1-10）。

返回值：

String

例子：

var i = 9;
repeat(10)
    {
    name[i] = highscore_name(i + 1);
    i -= 1;
    }

上面的代码将遍历高分列表，并将所有名称存储在数组中。

背面：一般游戏控制
