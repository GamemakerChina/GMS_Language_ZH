highscore_value

使用此功能，您可以检索在给定位置的高得分列表中存储的得分值。如果未输入分数，该函数将返回0。

句法：

highscore_value(place);

争论描述
place桌子上的位置（1-10）。

返回值：

N/A

例子：

var i = 9;
repeat(10)
    {
    scr[i] = highscore_value(i + 1);
    i -= 1;
    }

上面的代码将遍历高分列表，并将所有得分存储在数组中。

背面：一般游戏控制
