degtorad

在GM中，所有三角函数均以弧度为单位，但是大多数人以度为单位，这意味着要将您的度数转换为弧度，您需要使用此函数。例如，degtorad(180) 返回3.14159265弧度。此函数使用以下公式将度数转换为弧度：

angle_radians = angle_degrees * pi / 180

句法：

degtorad(deg);

争论描述
deg要转换的度数。

返回值：

Real

例子：

val = degtorad(90);

这会将val设置为pi / 2。

背面：角度和距离
