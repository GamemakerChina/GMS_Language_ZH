到物体的距离

这个函数计算从调用实例的边界框的边缘到指定对象的最近实例的最近边缘的距离。该对象可以是一个对象索引或一个特定的实例ID，也可以是 关键字 另一个，并且距离以像素为单位返回。注意，如果任何一个对象没有定义精灵或者没有定义掩码，结果将是不正确的。

语法。

distance_to_object(obj)。

争论 描述
obj 要检查的对象。

返回。

真正的。

例子。

如果(distance_to_object(obj_Player) < range)
{
canshoot = true;
}

上面的代码将检查到玩家对象的距离，如果它小于存储在变量 "范围 "中的值，变量 "canshoot "将被设置为真。
