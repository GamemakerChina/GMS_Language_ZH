distance_to_point

此函数计算从调用实例的边界框的边缘到房间中指定的x / y位置的距离，返回值以像素为单位。请注意，如果调用对象没有定义精灵或掩码，则结果将不正确。

句法：

distance_to_point(x, y);

争论描述
x要检查的x位置。
y要检查的y位置。

返回值：

Real

例子：

if (distance_to_point(obj_Player.x, obj_Player.y) < range)
    {
    canshoot = true;
    }

上面的代码将检查到玩家对象x / y位置的距离，如果该距离小于存储在变量“ range”中的值，则变量“ canshoot”将设置为true。
