lengthdir_x

此函数用于从起点沿“ dir”方向获取位置“ len”个像素的x分量。如果您在实例周围想象一个圆，然后在该圆上的任意位置想象一个点，要移动到该点，我们需要在该方向上移动对象这么多的像素...因此，此功能（当与 lengthdir_y()）获取实例上要在代码中使用的圆上该点的位置。有关详细信息，请参见下图：

句法：

lengthdir_x(len, dir);

争论描述
len返回点的长度。
dir返回点的方向。

返回值：

Real

例子：

var _xx = x + lengthdir_x(64, image_angle);
var _yy = y + lengthdir_y(64, image_angle);
instance_create_layer(_xx, _yy, "Bullets", obj_bullet);

这将在（_xx ，_yy ），它将在图像角度方向上距离父实例64个像素。

背面：角度和距离
