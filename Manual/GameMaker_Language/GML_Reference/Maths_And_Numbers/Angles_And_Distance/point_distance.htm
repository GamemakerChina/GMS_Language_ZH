point_distance

此函数返回由指定分量[x1，y1]和[x2，y2]形成的向量的长度。例如，在下图中，如果我们想获取玩家飞船位置与敌人位置之间的距离，以便我们可以计算出敌人是否足够近，可以射击，那么我们将使用此功能（确切的代码在图片下方的示例）：

句法：

point_distance(x1, y1, x2, y2);

争论描述
x1向量第一个分量的x坐标
y1向量第一个分量的y坐标
x2向量第二个分量的x坐标
y2向量第二个分量的y坐标

返回值：

Real

例子：

var ex, ey;
ex = instance_nearest(x, y, enemy).x;
ey = instance_nearest(x, y, enemy).y;
if (point_distance(x, y, ex, ey) < 200)
    {
    instance_create_layer(x, y, "Bullets", obj_Missile)
    }

上面的代码将获取最近的敌人的x和y坐标，然后使用它们检查由它们形成的矢量和玩家坐标的距离（长度）。如果该值小于200，则播放器对象将创建“ obj_Missile”的实例。

背面：角度和距离
