point_distance_3d

此函数采用向量提供的分量，并返回向量的长度（距离）。它的工作方式与 point_distance()但增加了在3D空间中使用的z值（深度）的分解系数。

句法：

point_distance_3d(x1, y1, z1, x2, y2, z2);

争论描述
x1向量第一个分量的x坐标
y1向量第一个分量的y坐标
z1向量的第一个分量的z坐标
x2向量第二个分量的x坐标
y2向量第二个分量的y坐标
z2向量第二个分量的z坐标

返回值：

Real

例子：

var inst, ex, ey, ez;
inst = instance_nearest(x, y, enemy);
if inst
    {
    ex = inst.x;
    ey = inst.y;
    ez = inst.z;
    if point_distance_3d(x, y, z, ex, ey, ez) < 200
        {
        instance_create_layer(x, y, "Bullets", obj_Missile)
        }
    }

上面的代码将获取最近的敌人的x，y和z坐标，然后使用它们检查由它们形成的矢量与玩家坐标的距离（长度）。如果该值小于200，则播放器对象将创建“ obj_Missile”的实例。
