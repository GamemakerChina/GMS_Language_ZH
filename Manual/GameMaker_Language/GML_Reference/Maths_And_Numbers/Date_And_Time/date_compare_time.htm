date_compare_time

使用此功能，您可以检查两个日期和时间，以查看哪个日期早于另一个时间。如果datetime1较早，则该函数返回-1；如果datetime1较晚，则返回1；如果它们相同则返回0，并且它将忽略日期，因此从字面上看，哪一个时间比其给定的日期更远。

句法：

date_compare_time(datetime1, datetime2);

争论描述
datetime1第一个日期时间。
datetime2比较第一个的日期时间。

返回值：

Real

例子：

d = date_compare_time(date_create_datetime( 2011, 9, 15, 11, 4, 0 ), date_current_datetime());

这将根据最早的时间将“ d”设置为相应的值。基本上，如果当前时间晚于11:04 am，它将返回1。如果更早，则返回-1。如果时间是11:04，它将返回0。

返回：日期和时间
