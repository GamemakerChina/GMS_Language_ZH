date_datetime_string

使用此函数，您可以创建一个包含给定日期时间的字符串，该字符串的格式应为调用该函数时正在运行游戏的系统或设备的格式。

句法：

date_datetime_string(date);

争论描述
date使用日期。

返回值：

String

例子：

str = date_datetime_string(date_current_datetime());

根据日期/时间显示的系统设置以及当前日期和时间，这会将“ str”设置为“ September 15th 2011，11：33.00”。

返回：日期和时间
