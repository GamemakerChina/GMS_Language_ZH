date_get_timezone

获取所有其他日期和时间函数所使用的基准时区。此时区可以是本地时区（由系统设置）或UTC ，并且该函数将返回以下常量之一：

持续的描述
timezone_local使用系统设置的本地时区
timezone_utc使用世界标准时间

句法：

date_get_timezone();

返回值：

Constant

例子：

if (date_get_timezone() != timezone_utc)
    {
    date_set_timezone(timezone_utc);
    }

此代码检查游戏的基本时区设置，如果不是UTC，则将其更改。

返回：日期和时间
