日期_是今天

如果给定的日期值是被检查的那一天（即：今天），这个函数将返回true，否则返回false。对于游戏中的复活节彩蛋或解锁季节性内容来说，这是一个很方便的功能。 请注意，这个函数将受到所设置的时区的影响（默认是当地时间），你可以用 date_set_timezone()函数来改变。

语法。

date_is_today(date)。

争论 描述
日期 要使用的日期时间。

返回。

布尔型

例子。

如果date_is_today(global.Halloween)
{
global.Max_Levels = 200;
}

上面的代码将检查存储在全局变量 "Halloween "中的日期时间，看它是否是今天，如果是，它就将另一个全局变量设置为一个新值。
