date_is_today

该函数将返回 true如果给定的datetime值是正在检查的日期（即：今天），或者 false否则。对于游戏中的复活节彩蛋之类的东西，或用于解锁季节性内容，这可能是一个方便的功能。请注意，此功能会受到时区设置（默认为当地时间）的影响，您可以使用 date_set_timezone()功能。

句法：

date_is_today(date);

争论描述
date使用的日期时间。

返回值：

Boolean

例子：

if date_is_today(global.Halloween)
    {
    global.Max_Levels = 200;
    }

上面的代码将检查存储在全局变量“万圣节”中的日期时间，以查看是否为今天，是否将另一个全局变量设置为新值。

返回：日期和时间
