date_leap_year

该函数将返回 true如果给定日期时间值的年部分是a年，则 false否则。对于游戏中的“复活节彩蛋”之类的功能或解锁特殊内容，这可能是一种方便的功能。

句法：

date_leap_year(date);

争论描述
date使用的日期时间。

返回值：

Boolean

例子：

if date_leap_year(date_current_datetime())
    {
    if !global.ExtraContent
        {
        global.ExtraContent = true;
        }
    }

上面的代码将检查当前日期时间，以查看年份是否为a年。如果是，它将设置一个全局变量。

返回：日期和时间
