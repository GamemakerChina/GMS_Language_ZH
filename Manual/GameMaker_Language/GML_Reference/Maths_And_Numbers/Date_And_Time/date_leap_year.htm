闰年（date_leap_year

如果给定的日期时间值中的年份成分是闰年，该函数将返回true，否则返回false。这个函数对于游戏中的复活节彩蛋或解锁特殊内容来说是一个很方便的函数。

语法。

date_leap_year(date)。

争论 描述
日期 要使用的日期时间。

返回。

布尔型

例子。

if date_leap_year(date_current_datetime())
{
if !global.ExtraContent
{
global.ExtraContent = true;
}
}

上面的代码将检查当前的日期时间，看看这一年是否是闰年。如果是，它将设置一个全局变量。
