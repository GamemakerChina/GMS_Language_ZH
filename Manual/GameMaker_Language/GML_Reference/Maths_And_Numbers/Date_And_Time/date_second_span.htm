date_second_span

使用此功能，您可以获取两个日期之间的秒数。返回值始终为正，将为整数。

句法：

date_second_span(date1, date2);

争论描述
date1第一个日期时间。
date2进行比较的日期时间。

返回值：

Real

例子：

diff = date_second_span(date_create_datetime(2011, 9, 15, 11, 4, 0 ), date_current_datetime());

这会将“ diff”设置为2011年9月15日11:04到当前日期和时间之间的秒数。

返回：日期和时间
