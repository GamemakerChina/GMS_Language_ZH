日期_second_span

通过这个函数你可以得到两个日期之间的秒数。返回值总是正数，并且是一个整数。

语法。

date_second_span(date1, date2)。

争论 描述
日期1 第一个日期时间。
日期2 要与之比较的日期时间。

返回。

真实的

例子。

diff = date_second_span(date_create_datetime(2011, 9, 15, 11, 4, 0 ), date_current_datetime()) 。

这将把 "diff "设置为2011年9月15日11:04与当前日期和时间之间的秒数。

返回。日期和时间
