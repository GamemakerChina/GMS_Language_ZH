date_set_timezone

设置要用于所有其他日期和时间函数的基本时区。该时区可以是本地时区（由系统设置）或UTC ，您可以使用以下常量之一来定义正在使用的时区（默认情况下，这是本地时间）：

持续的描述
timezone_local使用系统设置的本地时区
timezone_utc使用世界标准时间

句法：

date_set_timezone(timezone);

争论描述
timezone基本时间使用的时区。

返回值：

N/A

例子：

if (date_get_timezone() != timezone_utc)
    {
    date_set_timezone(timezone_utc);
    }

此代码检查游戏的基本时区设置，如果不是UTC，则将其更改。

返回：日期和时间
