date_set_timezone

设置基本时区，用于所有其他的日期和时间功能。这个时区可以是本地时间（由系统设置），也可以是UTC，你可以使用下面的一个常数来定义使用哪一个时区（默认为 你可以使用下列常数之一来定义使用哪种时间（默认情况下，这是本地时间）。)

恒定 描述
时区_本地 使用系统设置的本地时区
时区_UTC 使用协调世界时

语法。

date_set_timezone(时区)。

争论 描述
时区 用于基准时间的时区。

返回。

不适用

例子。

如果(date_get_timezone() != timezone_utc)
{
date_set_timezone(timezone_utc)。
}

这段代码检查游戏的基本时区设置，如果它不是UTC，就改变它。
