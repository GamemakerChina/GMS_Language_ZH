date_time_string

使用此函数，您可以创建一个包含给定时间的字符串，该字符串的格式将针对调用该函数的运行游戏的系统或设备进行格式化。

句法：

date_time_string( date );

争论描述
date使用的日期时间。

返回值：

String

例子：

str = date_time_string(date_current_datetime());

这将根据日期/时间显示和当前日期和时间的系统设置将给定变量设置为“ 11：36.00”。

返回：日期和时间
