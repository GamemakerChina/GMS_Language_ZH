date_valid_datetime

使用此功能，您可以检查日期时间值是否有效（返回 true）否（返回 false）。请注意，此函数只会将有效的日期时间视为在1970年1月1日之后，并且在此之前的任何时间都将返回 false，因此最早可以查看的是：

date_valid_datetime(1970, 01, 01, 0, 0, 0);

句法：

date_valid_datetime(year, month, day, hour, minute, second);

争论描述
year要检查的年份。
month要检查的月份。
day当天检查。
hour需要检查的时间。
minute分钟检查。
second第二个检查。

返回值：

Boolean

例子：

if date_valid_datetime(2011, 9, 15, 10, 3, 30)
    {
    mydatetime = date_create_datetime(2011, 9, 15, 10, 3, 30);
    }

如果该值是有效值，则它将“ mydatetime”设置为2011年9月15日10：03.30。

返回：日期和时间
