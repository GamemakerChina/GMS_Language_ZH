矩阵函数

矩阵是一个数字的集合，排列成固定数量的行和列。通常这些数字都是实数，但也不一定是这样的。下图显示了一般情况下如何构建一个有四行和四列的矩阵 和四列。

最上面的行是第1行，最左边的列是第1列，这个矩阵是一个4x4矩阵 因为它有四行和四列（也可以构建其他大小的矩阵）。在描述矩阵时，格式总是行/列，构成矩阵的每个数字被称为矩阵的一个元素。矩阵中的元素 有特定的位置，由其行和列的位置来描述，矩阵的左上角是行1列1。在上面的矩阵例子中，第1行第1列的元素是数值 "1"。第2行第3列的元素是 值 "4.8"。

矩阵对很多事情都非常重要，但在处理三维空间时用得最多，例如，它们可以用来设置摄像机视图，或者用来平移或转换模型。在GameMaker Studio 2中，存在以下处理矩阵的功能。
