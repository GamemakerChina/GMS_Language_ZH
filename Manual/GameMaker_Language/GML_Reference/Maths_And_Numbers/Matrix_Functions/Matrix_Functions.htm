矩阵函数

矩阵是排列成固定数量的行和列的数字的集合。通常，数字是实数，但不一定总是这样。下图大体上显示了如何构造具有四行四列的矩阵：

最上面的行是第1行，最左边的列是第1列，该矩阵是4x4矩阵，因为它有四行四列（可以构造其他尺寸的矩阵）。在描述矩阵时，格式始终是行/列，组成矩阵的每个数字都称为矩阵的元素。矩阵中的元素具有特定的位置，由它们的行和列位置描述，矩阵的左上角为第1行第1列。在上述矩阵示例中，第1行col 1的元素为值“ 1”。第2行第3列的元素是值“ 4.8”。

矩阵在许多方面都很重要，但是最常用于处理3D空间，例如，它们可用于设置相机视图或转换或转换模型。 GameMaker Studio 2存在以下用于处理矩阵的功能：

  • matrix_get
  • matrix_set
  • matrix_build
  • matrix_multiply
  • matrix_build_identity
  • matrix_build_lookat
  • matrix_build_projection_ortho
  • matrix_build_projection_perspective
  • matrix_build_projection_perspective_fov
  • matrix_transform_vertex
  • is_matrix

以下功能用于使用矩阵堆栈，该矩阵堆栈类似于DS堆栈，但仅设计用于矩阵。这是一种在创建3D场景时一个接一个地应用多个矩阵运算（如变换）的便捷方法。

  • matrix_stack_is_empty
  • matrix_stack_clear
  • matrix_stack_set
  • matrix_stack_push
  • matrix_stack_pop
  • matrix_stack_top
