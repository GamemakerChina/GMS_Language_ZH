matrix_build

该函数可用于创建自己的自定义矩阵，并将为此新矩阵返回一个索引值，该索引值应存储在变量中，以备将来参考和使用。矩阵本身被创建为16值数组，其中前4个元素是第4行矩阵的第1行，后4个元素是第2行等。

注意：以这种方式构建新矩阵时，操作顺序为YXZ。

句法：

matrix_build(x, y, z, xrotation, yrotation, zrotation, xscale, yscale, zscale);

争论描述
x翻译向量的x分量。
y翻译向量的y分量。
z翻译向量的z分量。
xrotation绕x轴旋转的角度（以度为单位）。
yrotation绕y轴旋转的角度（以度为单位）。
xzotation绕z轴旋转的角度（以度为单位）。
xscale x比例尺数量。
yscale y标度量。
zscale z缩放量。

返回值：

Matrix index (4x4 Matrix)

例子：

t_matrix = matrix_build(x, y, 0, 0, 90, 0, 1, 2, 1);

上面的代码将构建一个新的矩阵变换，并将结果矩阵索引存储在变量“ t_matrix”中。

返回：矩阵函数
