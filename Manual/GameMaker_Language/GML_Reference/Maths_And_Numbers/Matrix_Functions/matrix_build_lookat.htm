matrix_build_lookat

此函数根据下面列出的指定参数构建“外观”（视图）矩阵。由于此函数修改视图矩阵而不是投影矩阵，因此您应该首先使用另一个矩阵函数初始化投影矩阵 matrix_build_projection_perspective()，然后使用此功能在投影内四处移动观察相机。

要设置视图，您首先需要定义您从中看的位置，该位置由参数（xfrom，yfrom，zfrom）指示。接下来，您必须指定你看的方向，这是通过给第二点把目光投向与参数（XTO，YTO，ZTO）完成。最后，您仍然可以围绕摄像机从视点到观察点的线旋转摄像机，并且要指定该值，我们必须给出“向上”矢量-摄像机中向上的方向。这由最后三个参数（xup，yup，zup）给出。

句法：

matrix_build_lookat(xfrom, yfrom, zfrom, xto, yto, zto, xup, yup, zup);

争论描述
xfrom要查看的位置的x坐标。
yfrom要查看的位置的y坐标。
zfrom要查看的位置的z坐标。
xto要寻找的位置的x坐标。
yto要查找的位置的y坐标。
zto要查找的位置的z坐标。
xup “向上”向量的x坐标。
yup “向上”向量的y坐标。
zup “上”向量的z坐标。

返回值：

Matrix index

例子：

viewmat = matrix_build_lookat(640, 240, -10, 640, 240, 0, 0, 1, 0);
projmat = matrix_build_projection_ortho(640, 480, 1.0, 32000.0);
camera_set_view_mat(view_camera[0], viewmat);
camera_set_proj_mat(view_camera[0], projmat);

上面的代码创建一个新的查看矩阵和正交矩阵，将其ID存储在局部变量中，然后使用它们为分配给查看端口[0]的摄像机设置查看矩阵和投影矩阵。
