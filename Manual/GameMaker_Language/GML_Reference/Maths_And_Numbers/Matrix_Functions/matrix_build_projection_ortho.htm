matrix_build_projection_ortho

此功能根据下面列出的指定参数构建正交投影矩阵（这是在GameMaker Studio 2创建房间而未进行任何更改时使用的默认投影方法）。有时，您需要从透视投影切换为正投影，这是该功能的作用。它通常用于绘制叠加层，例如显示得分或其他方面，因为这可以为3D游戏中绘制的元素提供“平坦”视图（即：无透视图）。请参见下图，以了解正交视图和透视图之间的区别。

注意：如果希望绘制信息而与当前深度值无关，则可能还需要暂时关闭隐藏表面去除。

句法：

matrix_build_projection_ortho(width, height, znear, zfar);

争论描述
w投影的宽度。
h投影的高度。
znear附近的剪切平面。
zfar远剪切平面。

返回值：

Matrix index

例子：

viewmat = matrix_build_lookat(640, 240, -10, 640, 240, 0, 0, 1, 0);
projmat = matrix_build_projection_ortho(640, 480, 1.0, 32000.0);
camera_set_view_mat(view_camera[0], viewmat);
camera_set_proj_mat(view_camera[0], projmat);

上面的代码创建一个新的查看矩阵和正交矩阵，将其ID存储在局部变量中，然后使用它们为分配给查看端口[0]的摄像机设置查看矩阵和投影矩阵。
