matrix_build_projection_perspective

此功能使用下面列出的指定参数，根据近平面的尺寸构建透视投影矩阵矩阵。

句法：

matrix_build_projection_perspective(width, height, znear, zfar);

争论描述
w投影的宽度。
h投影的高度。
znear附近的剪切平面。
zfar远剪切平面。

返回值：

Matrix index

例子：

var projmat = matrix_build_projection_perspective(640, 480, 1.0, 32000.0);
camera_set_proj_mat(view_camera[0], projmat);

上面的代码创建了透视投影，然后使用返回的矩阵来设置分配给视口[0]的摄像机。

返回：矩阵函数
