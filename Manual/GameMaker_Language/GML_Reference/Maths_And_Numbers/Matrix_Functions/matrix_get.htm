matrix_get

此函数返回一个16个值的数组，对应于给定的4x4矩阵类型，其中第1行是元素[0-3]，第2行是元素[4 -7]，依此类推...（请参见主页上的图像） ）。可用的矩阵是view ， projection和world ，可以使用以下常量之一：

持续的描述
matrix_view当前视图矩阵
matrix_projection当前投影矩阵
matrix_world当前世界矩阵

句法：

matrix_get(type);

争论描述
type要获取值的矩阵的类型（请参见上面列出的常数）

返回值：

Array (16 elements)

例子：

v_array = matrix_get(matrix_view);

上面的代码将获取当前视图矩阵的值，并使用它们填充数组“ v_array”。

返回：矩阵函数
