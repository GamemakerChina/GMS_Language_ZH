matrix_multiply

使用此功能，您可以将两个矩阵阵列相乘以创建一个新的变换矩阵。该函数将返回新的矩阵索引，该索引应存储在变量中以备将来使用。

注意：此函数不能将矩阵常数用作参数，因此，如果要将（例如）视图矩阵与已构建的自定义矩阵相乘，则必须首先调用matrix_get（）并分配视图将矩阵值转换为数组变量，然后将其乘以自定义矩阵，然后设置选定的矩阵（视图，投影或世界）。

句法：

matrix_multiply(matrix1, matrix2);

争论描述
matrix1要使用的第一个矩阵索引。
matrix2要使用的第二个矩阵索引。

返回值：

Matrix Index (4x4 Matrix)

例子：

var v_matrix = matrix_get(matrix_view);
var new_matrix = matrix_multiply(v_matrix, my_matrix);
matrix_set(matrix_view, new_matrix);

上面的代码将获取当前视图矩阵，然后将其与自定义矩阵相乘，然后使用生成的矩阵索引再次设置视图矩阵。

返回：矩阵函数
