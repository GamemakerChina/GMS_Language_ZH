矩陣_倍增

通过这个函数，你可以将两个矩阵数组相乘来创建一个新的变换矩阵。该函数将返回新的矩阵索引，该索引应存储在一个变量中供将来使用。

注意：你不能使用矩阵常数作为这个函数的参数，所以如果你想把视图矩阵（例如）与你建立的自定义矩阵相乘，你必须先调用 matrix_get() 并将视图矩阵的值赋给一个数组变量，然后将其与你的自定义矩阵相乘，再设置所选矩阵（视图、投影或世界）。

语法。

matrix_multiply(matrix1, matrix2)。

争论 描述
矩阵1 要使用的第一个矩阵索引。
矩阵2 要使用的第二个矩阵索引。

返回。

矩阵指数（4x4矩阵

例子。

var v_matrix = matrix_get(matrix_view);
var new_matrix = matrix_multiply(v_matrix, my_matrix);
matrix_set(matrix_view, new_matrix) 。

上面的代码将得到当前的视图矩阵，然后与自定义矩阵相乘，然后使用得到的矩阵索引再次设置视图矩阵。
