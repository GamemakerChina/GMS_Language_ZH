matrix_set

此函数采用16个值组成的数组，对应于给定的4x4矩阵类型，其中元素[0-3]将是第1行，元素[4 -7]将是第2行，依此类推...（请参见主页）。您可以使用 matrix_build()或者 matrix_get()函数或只是自己构建数组并将其传递给函数。可用的矩阵类型是view ， projection和world ，您可以使用以下常量之一：

持续的描述
matrix_view当前视图矩阵
matrix_projection当前投影矩阵
matrix_world当前世界矩阵

句法：

matrix_set(type, matrix);

争论描述
type要获取值的矩阵的类型（请参见上面列出的常数）
matrix矩阵数据作为数组

返回值：

NA

例子：

matrix_set(matrix_world, m_array);

上面的代码会将当前世界矩阵的值设置为存储在数组矩阵“ m_array”中的值。

返回：矩阵函数
