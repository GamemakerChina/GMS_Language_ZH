Matrix_stack_push

这个函数将指定的矩阵推到矩阵栈上。

语法。

matrix_stack_push(matrix)。

争论 描述
基体 要使用的矩阵索引。

返回。

不适用

例子。

var m1 = matrix_build(66, 145, 0, 0, 0, 1, 1, 1);
var m2 = matrix_build(0, 0, 0, 0, 0, image_angle * 6, 1, 1) ;
matrix_stack_push(m1);
matrix_stack_push(m2)。
matrix_set(matrix_world, matrix_stack_top());
draw_sprite(tyre, 0, 0, 0);
matrix_stack_pop()。
matrix_stack_pop()。

上面的代码将建立两个新的矩阵，然后把它们推到矩阵栈中。然后世界矩阵被设置到堆栈的顶部，绘制一个精灵，然后从堆栈中跳出变换。
