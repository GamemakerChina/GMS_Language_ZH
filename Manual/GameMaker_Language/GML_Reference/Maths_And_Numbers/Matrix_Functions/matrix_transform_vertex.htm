矩阵转化_顶点

这个函数用一个矩阵来转换一个向量。你提供转换矩阵的ID（如函数返回的 矩阵 build())，以及要转换的向量的x、y和z值。该函数将返回 一个包含3个元素的数组，其中。

  • 数组[0] = x
  • array[1] = y
  • array[2] = z.

语法。

matrix_transform_vertex(matrix, x, y, z)。

争论 描述
基体 要使用的矩阵
x 变换矢量的X分量
y 变换矢量的Y分量
z 变换矢量的Z分量

返回。

数组（3个元素

例子。

t_matrix = matrix_build(0, 0, 0, 0, 90, 0, 1, 2, 1);
verts = matrix_transform_vertex(t_matrix, x, y, z) 。

上述代码使用存储在变量 "t_matrix "中的矩阵对给定值进行转换，并将其存储在数组 "verts "中。
