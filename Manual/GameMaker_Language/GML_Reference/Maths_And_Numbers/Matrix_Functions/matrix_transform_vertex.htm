matrix_transform_vertex

此函数通过矩阵转换向量。您提供变换矩阵ID（由函数返回）matrix build() ），以及要转换的向量的x，y和z值。该函数将返回一个包含3个元素的数组，其中：

  • 数组[0] = x
  • 数组[1] = y
  • array [2] = z。

句法：

matrix_transform_vertex(matrix, x, y, z);

争论描述
matrix使用的矩阵
x变换向量的x分量
y变换向量的y分量
z变换向量的z分量

返回值：

Array (3 elements)

例子：

t_matrix = matrix_build(0, 0, 0, 0, 90, 0, 1, 2, 1);
verts = matrix_transform_vertex(t_matrix, x, y, z);

上面的代码使用存储在变量“ t_matrix”中的矩阵来转换给定值，并将其存储在数组“ verts”中。

返回：矩阵函数
