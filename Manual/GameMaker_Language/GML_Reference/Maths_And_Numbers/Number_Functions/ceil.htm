ceil

此函数采用任何实数并将其舍入到最接近的整数。应该小心使用此函数，因为一个常见的错误是使用它对一个随机值进行取整并期望其始终大于1，即：

int = ceil(random(5));

现在，您希望此代码始终提供1到5之间的整数，但这可能并非总是如此，因为随机函数返回0的可能性很小，而四舍五入仍会得到0。可能性很小，但在使用此功能时应予以考虑。

句法：

ceil(val);

争论描述
x要更改的数字。

返回值：

Integer

例子：

val = ceil( 3.4 );

这会将val设置为4。
