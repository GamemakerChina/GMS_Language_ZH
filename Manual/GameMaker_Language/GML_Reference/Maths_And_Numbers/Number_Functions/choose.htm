choose

有时您想要为随机选择指定数字以外的其他内容，或者您想要的数字没有任何真实顺序或在任何设置范围内。在这些情况下，您将使用 choose()产生随机结果。例如，假设您要在开始时创建一个带有随机子画面的对象，然后可以使用此功能将子画面索引设置为一组给定子画面中的一个。请注意，您可以根据需要进行任意设置（注意，更多的参数将意味着该函数的解析速度较慢）。

注意GameMaker Studio 2每次都会生成相同的初始随机种子，从而使调试代码变得容易得多，因此每次重新运行游戏时，此函数将返回相同的值。为了避免这种行为，请在游戏开始时使用randomise（）。

句法：

choose(val0, val1, val2... max_val);

争论描述
val0... max_val输入值可以是字符串，整数，变量或常量。

返回值：

One of the given arguments

例子：

sprite_index = choose(spr_Cactus, spr_Flower, spr_Tree, spr_Shrub);
hp = choose(5, 8, 15, 32, 40);
name = choose("John", "Steven", "Graham", "Jack", "Emily", "Tina", "Jill", "Helen");

上面的代码使用select来设置实例的许多属性。
