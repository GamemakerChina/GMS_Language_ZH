irandom

这个非常有用的函数仅返回整数（整数）。因此，例如，要获取0到9之间的随机数，您可以使用 irandom(9)它会返回数从0到9的包容性。请注意，也可以使用浮点数，但该点之后的上限值将被排除，因此 irandom(9.7)将仅返回0到9之间的值。该函数的上限为$7fffffffffffffff LL，因此如果使用非常大的数字，则应小心。

注意： GameMaker Studio 2每次都会生成相同的初始随机种子，因此使调试代码变得容易得多，因此每次重新运行游戏时，此函数将返回相同的值。为了避免这种行为，请在游戏开始时使用randomise（）。

句法：

irandom(n);

争论描述
n从中选择随机数的上限范围。

返回值：

Integer

例子：

if (irandom(9) == 1)
   {
   score += 100;
   }

这将产生十分之一的机会（因为包括了0），将分数加100。

背面：数字函数
