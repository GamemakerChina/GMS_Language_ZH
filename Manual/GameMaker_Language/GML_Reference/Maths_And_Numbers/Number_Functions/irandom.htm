随机

这个非常有用的函数只返回整数（整数）。因此，例如，要得到一个从0到9的随机数，你可以使用irandom(9)，它将返回一个从0到9的数字，包括在内。注意，也可以使用浮点数，但是 但点之后的最大值将被排除，所以irandom(9.7)将只返回一个从0到9的值。该函数的上限是$7fffffffffffffLL，所以如果使用非常大的数字，应该注意。

注意：由于GameMaker Studio 2每次都会生成相同的初始随机种子，使调试代码变得更加容易，所以每次运行游戏时，这个函数都会返回相同的值。为了避免这种行为，请使用 randomise()在你的游戏开始时使用。

语法。

irandom(n)。

争论 描述
n 将从中选择随机数的上限范围。

返回。

整数

例子。

如果（irandom(9) == 1）
{
score += 100;
}

这将产生十分之一（因为包括0）的机会，使分数增加100。
