irandom_range

该功能类似于 random_range()仅使用整数值作为输入。您提供范围的低值和最大值，函数将返回给定范围内（包括该范围）的随机整数值。例如，irandom_range(10, 35)将返回10和35之间的整数。

与 irandom()函数，可以使用实数，在这种情况下，它们将被四舍五入为最接近的整数EG：irandom_range（6.2,9.9）将给出6到9之间的值。

注意： GameMaker ：Studio每次都会生成相同的初始随机种子，因此使调试代码变得容易得多，因此每次重新运行游戏时，此函数将返回相同的值。为了避免这种行为，请在游戏开始时使用randomise（）。

句法：

irandom_range(n1, n2);

争论描述
n1从中选择随机数的范围的低端。
n2从中选择随机数的范围的高端。

返回值：

Integer

例子：

score += irandom_range(500, 600);

这会将整数值添加到总分的500到600（含）之间。

背面：数字函数
