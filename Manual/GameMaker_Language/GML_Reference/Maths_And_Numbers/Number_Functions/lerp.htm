lerp

使用此功能，您可以找到等于给定百分比的其他两个值之间位置的值。因此，例如，如果这样做：

lerp(0, 10, 0.5)

您将获得返回值5，即输入值的50％。您也可以使用此函数进行推断，方法是为插入量提供正值或负值，以便执行以下操作：

lerp(0, 10, 2)

将返回值20。

句法：

lerp(a, b, amt)

争论描述
a第一个值。
b第二个值。
amt要插值的数量。

返回值：

Real

例子：

xx = lerp(x, x + hspeed, room_speed);
yy = lerp(y, y + vspeed, room_speed);

上面的代码使用线性插值函数来预测实例在游戏时间一秒后将移动到的位置。
