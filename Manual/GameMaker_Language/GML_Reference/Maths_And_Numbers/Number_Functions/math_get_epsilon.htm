math_get_epsilon

此函数将返回目标平台的当前epsilon值。有关epsilon的更多信息，请参见功能 math_set_epsilon()。

句法：

math_get_epsilon();

返回值：

Real

例子：

var e = math_get_epsilon();
if e != 0.000001
    {
    math_set_epsilon(0.000001);
    }

这将检索当前的epsilon值并将其存储在本地变量中，然后将其检查并在需要时设置新的epsilon。
