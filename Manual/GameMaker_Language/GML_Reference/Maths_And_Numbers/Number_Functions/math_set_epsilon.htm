math_set_epsilon

Epsilon是一个值，用于确定发生舍入误差的两个数字是否足够接近以被视为“相等”。在处理浮点数学运算时，此方法很有用，因为它可以减少使某些运算返回的值看起来不正确或与我们期望相反的“舍入误差”。例如，我们可能已经向sprite的图像索引添加了一个值，并期望结果为单个整数，但是由于浮点数学的性质，实际最终值最终类似于5.0000002，所以当我们检查以下内容：

if image_index == 5
    {
    //do something
    }

该代码的行为不符合预期，并返回 false。但是，如果将epsilon值设置为0.000001，则 image_index值将四舍五入到最接近原始值的+/- 0.000001的实数，使上述比较返回true。

epsilon值将用于以下所有整数运算符：

  • <  ： 少于
  • > ： 比...更棒
  • == ： 等于
  • <= ：小于或等于
  • >= ：大于或等于
  • != ： 不等于

请注意，将epsilon值设置为0将禁用所有舍入，而将值设置为1将给出错误。

句法：

math_set_epsilon(epsilon);

争论描述
epsilon新的epsilon值（从0到0.999999999）。

返回值：

Real

例子：

math_set_epsilon(0.0001);

这将为所有其他浮点运算设置epsilon值。
