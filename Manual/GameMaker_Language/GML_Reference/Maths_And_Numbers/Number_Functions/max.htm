最大

这个函数返回输入值的最大值，其中最多可以有16个。例如max(12, 96, 32, 75)将返回96，因为那是所有输入值中最高的一个。

语法。

max(val1, val2, ... max_val)。

争论 描述
val0 ... max_val 要比较的数值。

返回。

真实的

例子。

x = max(x, 0)。

这将阻止玩家从房间的左边出来。它的工作原理是不断地将它的X设置为自身或0，以较大者为准。如果玩家离开了左边，它的X会小于0（即负数），所以它会被直接设置为 归位。
