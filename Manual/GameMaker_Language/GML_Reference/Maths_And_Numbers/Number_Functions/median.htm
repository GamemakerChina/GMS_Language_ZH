median

此函数返回输入值的中间值，即中间值。当参数数量为偶数时，将返回两个中间值中较小的一个，并且该函数可以具有所需的任意数量的参数（请注意，更多的参数将意味着该函数将较慢地解析），这些参数都必须是实数值。例如，这意味着 median(43, 12, 25, 19, 6)将返回19，因为它是所有其余值之间的中间值。

句法：

median(val1, val2, ... max_val);

争论描述
val0... max_val要比较的值。

返回值：

Real

例子：

x = median( 0, x, room_width );
y = median( 0, y, room_height );

通过使用中位数作为钳位，这将阻止玩家退出房间的任何一侧。例如，如果玩家移至房间边界的左侧，则其x会小于0。这意味着第一个中位数的中间数将为0，因此玩家将跳至（0， y）。
