这个函数返回输入值的最小值，你需要多少个就有多少个（注意，更多的参数将意味着函数的解析速度会更慢）。例如，min(12, 96, 32, 75)将返回12，因为这是所有输入值中最低的一个。 是所有输入值中最低的。

语法。

min(val1, val2, ... max_val)。

争论 描述
val0 ... max_val 要比较的数值。

返回。

真实的

例子。

x = min(x, room_width)。

这将阻止玩家从房间的右边出来。它的工作原理是不断地将它的x设置为自己或room_width变量，以较小者为准。如果玩家从右边出去，它的x会大于room_width，所以它会被直接设置回来。 会被直接设置回来。
