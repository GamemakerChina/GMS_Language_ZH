random

此函数适用于不需要返回整数（整数）的概率。例如，random(100)将返回一个从0到99的值，但是该值可以是22.56473！您也可以在此函数中使用实数而不是整数，如下所示：random(0.5) ，它将返回介于0和0.4999999之间的值。

注意GameMaker Studio 2每次都会生成相同的初始随机种子，从而使调试代码变得容易得多，因此每次重新运行游戏时，此函数将返回相同的值。为了避免这种行为，请在游戏开始时使用randomise（）。

句法：

random(n);

争论描述
n从中选择随机数的上限范围。

返回值：

Real

例子：

if (random(10) >= 9)
   {
   score += 100;
   }

这将产生大约十分之一的几率，使总分增加100。

背面：数字函数
