随机

这个函数适用于不需要返回整数（整数）的概率。例如，random(100)将返回一个从0到99的值，但这个值可能是22.56473！这时，你可以使用实数，而不是整数。你也可以在这个函数中使用实数而不是整数。 随机(0.5)，它将返回 0 到 0.4999999 之间的值。

注意：由于GameMaker Studio 2每次都会生成相同的初始随机种子，使调试代码变得更加容易，所以每次运行游戏时这个函数都会返回相同的值。为了避免这种行为，请使用 randomise()在你的游戏开始时使用。

语法。

随机(n)。

争论 描述
n 将从中选择随机数的上限范围。

返回。

真实的

例子。

如果（随机（10）>=9）
{
score += 100;
}

这将产生大约十分之一的机会，使分数增加100分。
