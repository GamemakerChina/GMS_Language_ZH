random_get_seed

为了生成一个随机数， GameMaker Studio 2从一个随机种子数开始。使用此功能，您可以检索该种子，然后可以将其存储以重现特定系列的随机事件。例如，假设您有一个程序生成的游戏，但希望玩家能够返回以前发现的区域。好吧，在每个区域的开头，您都可以使用此功能存储随机种子，因此每次需要返回该位置时，都可以使用 random_set_seed()将种子设置为以前的状态，从而迫使GameMaker Studio 2重现以前使用的同一组“随机”功能。

句法：

random_get_seed();

返回值：

Real

例子：

ini_open("Save.ini")
ini_write_real("Levels", string(level), random_get_seed());
ini_close();

上面的代码将获取随机种子并将其存储在ini文件中。
