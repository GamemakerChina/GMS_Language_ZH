random_range

此函数返回指定范围之间的随机数，并且此返回值不必为整数。例如，random_range(20,50)会返回20到50之间的随机数，但该值可能是真实数字，例如38.65265。实数也可以用作输入参数。

注意GameMaker Studio 2每次都会生成相同的初始随机种子，从而使调试代码变得容易得多，因此每次重新运行游戏时，此函数将返回相同的值。为了避免这种行为，请在游戏开始时使用randomise（）。

句法：

random_range(n1, n2);

争论描述
n1从中选择随机数的范围的低端。
n2从中选择随机数的范围的高端。

返回值：

Real

例子：

score += random_range(500, 600);

这将使总分增加500至600。

背面：数字函数
