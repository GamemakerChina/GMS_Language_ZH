random_set_seed

为了生成一个随机数， GameMaker Studio 2从一个随机种子数开始。使用此功能，您可以将该种子设置为一个已知值，然后在每次运行程序时“强制”所有随机事件的结果相同。例如，此功能可与 random_get_seed()创建过程生成的内容并保存结果而无需进行大量的保存游戏（您只保存种子，不需要任何其他操作）。如果您需要为所有事情提供真正随机的结果，则应使用 randomise()功能。

注意：尽管此种子将在每个目标平台上提供一致的结果，但由于每个目标的工作方式不同，结果在平台之间可能会有所不同。

句法：

random_set_seed(val);

争论描述
val要设定的种子。

返回值：

N/A

例子：

if debug
    {
    random_set_seed(1);
    }

上面的代码仅在变量“ debug”为true时将随机种子设置为1。
