randomise

此函数将种子设置为随机值。如果您需要在多次游戏中保持一致的价值，则应该使用 random_set_seed()。 GameMaker Studio 2使用随机数功能时，初始种子始终是相同的，因为这使跟踪错误和调试变得更加容易。如果您希望以真正的随机性进行测试，则应在游戏开始时调用此函数。该函数将返回新的随机种子值（无符号的32位整数）。

句法：

randomise();
// or
randomize();

返回值：

Integer (unsigned 32 bit value)

例子：

randomise();

上面的代码将随机化种子。
