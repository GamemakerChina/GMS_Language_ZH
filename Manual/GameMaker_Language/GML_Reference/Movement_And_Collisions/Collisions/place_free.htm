place_free

您可以使用此函数来检查并查看调用实例是否会与游戏中标记为实体的任何实例发生冲突。现在，应该注意的是，要使此功能正常运行，运行代码的实例必须具有有效的碰撞掩码（对于精灵本身或通过 mask_index），并且只会注册与也具有有效掩码的那些带有固体标记的实例的冲突。

该函数本身基本上通过以下方式工作：将实例放置在x / y参数指定的位置，并仅测试与实心标记的实例之间的冲突。碰撞检查可以是精确的，也可以基于实例的边界框，具体取决于选择了哪种碰撞遮罩。

注意，在执行检查之前，给定的x / y坐标将被取整为最接近的整数。

句法：

place_free(x, y);

争论描述
x要检查的x位置。
y要检查的y位置。

返回值：

Boolean

例子：

if place_free(mouse_x, mouse_y)
   {
   x = mouse_x;
   y = mouse_y;
   }

如果将调用实例放置在与鼠标相同的位置，则上面的代码将检查是否与任何带有固体标记的实例发生冲突。如果未检测到冲突，则实例的x / y坐标设置为鼠标的x / y坐标。
