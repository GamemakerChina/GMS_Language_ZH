place_meeting

使用此功能，您可以使用运行检查代码的实例的碰撞屏蔽来检查与另一个实例或对象的所有实例的碰撞位置。使用此功能时，您实际上是在要求GameMaker Studio 2将实例移动到新位置，检查是否有碰撞，然后移回并告诉您是否发现碰撞。这将适用于精确的碰撞，但是仅当实例和要检查的对象都选择了精确的碰撞蒙版时，否则仅应用边界碰撞。此函数还接受所有其他特殊关键字。请注意，如果您需要获取要与之碰撞的对象的唯一实例ID，则应使用 instance_place()。

注意，在执行检查之前，给定的x / y坐标将被取整为最接近的整数。

句法：

place_meeting(x, y, obj);

争论描述
x要检查的x位置。
y要检查的y位置。
obj要检查的实例或对象。

返回值：

Boolean

例子：

if keyboard_check(vk_left)
   {
   if !place_meeting(x - 5, y, obj_wall) x -=5;
   }

上面的代码检查实例左侧是否没有冲突，如果没有冲突，则移动实例。
