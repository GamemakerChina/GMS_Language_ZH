地点_会议

通过这个函数，你可以使用运行检查代码的实例的碰撞掩码来检查一个位置是否与另一个实例或一个物体的所有实例发生了碰撞。当你使用这个函数时，你实际上是要求GameMaker Studio 2将实例移动到新的位置，检查是否有碰撞，然后向后移动并告诉你是否发现了碰撞。这对精确的碰撞是有效的，但前提是实例和被检查的对象都有精确的碰撞掩码 选择，否则只应用边界框碰撞。这个函数也接受特殊关键字all和other。请注意，如果你需要获得被碰撞对象的唯一实例ID，你应该 应该使用 instance_place().

注意，在进行检查之前，给定的x/y坐标将被浮动到最接近的整数。 执行。

语法。

place_meeting(x, y, obj)。

争论 描述
x 要检查的X位置。
y 要检查的y位置。
obj 要检查的实例或对象。

返回。

布尔型

例子。

if keyboard_check(vk_left)
{
if !place_meeting(x - 5, y, obj_wall) x -=5;
}

上述代码检查实例左侧是否有碰撞，如果没有则移动实例。

返回。碰撞
