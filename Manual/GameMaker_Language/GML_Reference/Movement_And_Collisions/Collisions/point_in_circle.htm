point_in_circle

使用此功能时，您定义一个圆形区域， GameMaker Studio 2将计算出给定点是否在其范围内。如果点落在定义的圆内，函数将返回 true否则函数将返回 false。

句法：

point_in_circle(px, py, x1, y1, rad);

争论描述
px要检查点的x坐标。
py要检查点的y坐标。
x1圆心的x坐标。
y1圆心的y坐标。
rad圆的半径。

返回值：

Boolean

例子：

if point_in_circle(mouse_x, mouse_y, x, y, 16)
   {
   over = true;
   }
else
   {
   over = false;
   }

上面的代码使用 point_in_circle用于检查鼠标位置是否在定义的圆形区域内的函数，将变量“ over”设置为 true如果有，或者 false否则。
