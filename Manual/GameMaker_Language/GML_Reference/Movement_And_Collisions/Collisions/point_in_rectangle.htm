point_in_rectangle

使用此功能时，您定义一个矩形区域， GameMaker Studio 2将计算出给定点是否在其边界之内。如果该点落在定义的矩形内，则函数将返回 true否则函数将返回 false。

句法：

point_in_rectangle(px, py, x1, y1, x2, y2);

争论描述
px要检查点的x坐标。
py要检查点的y坐标。
x1要检查的矩形左侧的x坐标。
y1要检查的矩形顶部的y坐标。
x2要检查的矩形右侧的x坐标。
y2要检查的矩形底部的y坐标。

返回值：

Boolean

例子：

if point_in_rectangle(mouse_x, mouse_y, x -10, y - 10, x + 10, y + 10)
   {
   audio_play_sound(snd_click, 0, false);
   }

此短代码检查鼠标在定义的矩形区域上的位置，并在其落入边界时播放声音。
