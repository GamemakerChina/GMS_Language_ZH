point_in_triangle

使用此功能时，您定义一个三角形区域， GameMaker Studio 2将计算出给定点是否在其边界之内。如果该点落在定义的三角形内，函数将返回 true否则函数将返回 false。

句法：

point_in_triangle(px, py, x1, y1, x2, y2, x3, y3);

争论描述
px要检查点的x坐标。
py要检查点的y坐标。
x1要检查的三角形第一个角的x坐标。
y1要检查的三角形第一个角的y坐标。
x2要检查的三角形第二个角的x坐标。
y2要检查的三角形第二个角的y坐标。
x3要检查的三角形第三个角的x坐标。
y3要检查的三角形第三个角的y坐标。

返回值：

Boolean

例子：

var inst = instance_nearest(x, y, obj_Player);
if instance_exists(inst)
   {
   var x1 = x + lengthdir_x(100, image_angle - 45);
   var y1 = y + lengthdir_y(100, image_angle - 45);
   var x2 = x + lengthdir_x(100, image_angle + 45);
   var y2 = y + lengthdir_y(100, image_angle + 45);
   if point_in_triangle(inst.x, inst.y, x, y, x1, y1, x2, y2)
      {
      can_see = true;
      }
   }

上面的代码使用 point_in_triangle充当“视锥”检查“ obj_player”的实例，将变量设置为 true对象的x / y位置是否应位于定义的三角形内。
