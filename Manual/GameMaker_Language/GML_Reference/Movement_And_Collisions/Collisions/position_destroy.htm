position_destroy

此功能仅破坏发现与指定位置发生碰撞的所有实例。碰撞基于实例的蒙版，如果蒙版的任何部分与目标点重叠，则函数将销毁该实例。以这种方式销毁的实例将触发其销毁和清理事件。

句法：

position_destroy(x, y);

争论描述
x销毁碰撞实例的x坐标。
y销毁碰撞实例的y坐标。

返回值：

N/A

例子：

if mouse_check_button_pressed(mb_left)
   {
   position_destroy(mouse_x, mouse_y);
   }

当按下鼠标左键时，这将破坏所有与鼠标光标的位置发生冲突的实例。
