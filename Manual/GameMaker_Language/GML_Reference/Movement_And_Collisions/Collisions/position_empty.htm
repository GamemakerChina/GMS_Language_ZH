position_empty

此功能将检查给定位置是否与在给定位置具有有效碰撞蒙版的任何实例发生碰撞。

句法：

position_empty(x, y);

争论描述
x要检查的x位置。
y要检查的y位置。

返回值：

Boolean

例子：

var xx, yy;
xx = random(room_width);
yy = random(room_height);
if position_empty(xx, yy)
   {
   instance_create_layer(xx, yy, "Bullets", obj_Bomb);
   }

这将检查房间中某个位置的随机位置是否发生碰撞，如果没有碰撞，它将创建obj_Bomb的实例。
