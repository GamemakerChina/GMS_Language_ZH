空的位置

这个函数将检查一个给定的位置是否与给定位置上的任何具有有效碰撞掩码的实例进入碰撞。

语法。

position_empty(x, y)。

争论 描述
x 要检查的X位置。
y 要检查的y位置。

返回。

布尔型

例子。

var xx, yy;
xx = random（room_width）;
yy = random（room_height）;
if position_empty(xx, yy)
{
instance_create_layer(xx, yy, "Bullets", obj_Bomb);
}

这将检查房间里某个随机位置是否有碰撞，如果没有，它将创建一个obj_Bomb的实例。

返回。碰撞
