position_meeting

使用此功能，您可以检查与另一个实例或对象的所有实例发生碰撞的位置。使用此功能时，您正在检查房间中的单个实例或对象。如果实例的边界已检查精确，则将根据实例的边界框或实例的掩码进行此检查。如果需要与对象碰撞时获得唯一的实例ID，则应使用 instance_position()。此函数还接受所有其他特殊关键字。

句法：

position_meeting(x, y, obj);

争论描述
x要检查的x位置。
y要检查的y位置。
obj用于检查冲突的对象（或实例ID，或关键字“ all”或“ other”）。

返回值：

Boolean

例子：

if mouse_check_button(mb_left)
   {
   if !position_meeting(mouse_x, mouse_y, all) instance_create_layer(mouse_x, mouse_y, "Walls", obj_Wall);
   }

上面的代码检查鼠标左键，如果按下了它，则检查鼠标x / y位置是否与任何实例发生冲突。如果不存在，则创建“ obj_Wall”的实例。
