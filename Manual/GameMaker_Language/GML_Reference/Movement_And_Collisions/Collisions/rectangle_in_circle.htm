rectangle_in_circle

此功能将检查您定义的矩形区域，以查看它是否未发生碰撞，是否完全在目标范围内，或者是否只是触摸了定义的圆形区域。如果它们根本没有触摸，则该函数将返回0，如果源完全位于目标中，则该函数将返回1，如果它们只是重叠，则将返回2。

句法：

rectangle_in_circle(sx1, sy1, sx2, sy2, x, y, rad);

争论描述
sx1源矩形左侧的x坐标。
sy1源矩形顶部的y坐标。
sx2源矩形右侧的x坐标。
sy2源矩形底部的y坐标。
x圆心的x坐标
y圆心的y坐标。
rad用于检查碰撞的中心点周围的半径。

返回值：

Real

例子：

inst = instance_nearest(x, y, obj_Bullet);
if instance_exists(inst)
   {
   if rectangle_in_circle(inst.x - 5, inst.y - 5, inst.x + 5, inst.y + 5, x, y - 25, 20) > 0
      {
      hit = true;
      }
   }

上面的代码使用 rectangle_in_circle用于检查圆形区域和找到的实例周围的矩形内是否存在碰撞的函数。如果发生冲突（边缘重叠或被包围），则将变量设置为 true。
