rectangle_in_rectangle

此功能将检查您定义的两个矩形区域，以查看源矩形是否未发生碰撞，是否完全在目标矩形范围内或是否只是在触摸。如果它们根本没有触摸，则该函数将返回0，如果源完全位于目标中，则该函数将返回1，如果它们只是重叠，则将返回2。

句法：

rectangle_in_rectangle(sx1, sy1, sx2, sy2, dx1, dy1, dx2, dy2);

争论描述
sx1源矩形左侧的x坐标。
sy1源矩形顶部的y坐标。
sx2源矩形右侧的x坐标。
sy2源矩形底部的y坐标。
dx1要检查的目标矩形左侧的x坐标。
dy1要检查的目标矩形顶部的y坐标。
dx2要检查的目标矩形右侧的x坐标。
dy2要检查的目标矩形底部的y坐标。

返回值：

Real

例子：

switch(rectangle_in_rectangle(x - 50, y - 50, x + 50, y + 50, mouse_x - 50, mouse_y - 50, mouse_x + 50, mouse_y + 50))
   {
   case 0: image_blend = c_yellow; break;
   case 1: image_blend = c_red; break;
   case 2: image_blend = c_orange; break;
   }

上面的代码检查两个矩形区域是否发生碰撞，并设置实例 image_blend根据返回值。
