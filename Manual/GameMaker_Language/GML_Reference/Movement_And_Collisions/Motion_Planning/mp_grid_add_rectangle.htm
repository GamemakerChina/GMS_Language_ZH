mp_grid_add_rectangle

此功能要求您在房间内定义一个矩形，然后将所有MP网格单元标记为“触摸”该矩形，这表示禁止该路径查找功能。下图说明了它是如何工作的：

如您所见，由（50，90）到（200，180）定义的矩形将其触摸的所有等效MP网格单元格标记为禁止。

句法：

mp_grid_add_rectangle(id, x1, y1, x2, y2);

争论描述
id要使用的mp_grid的索引
x1要检查的矩形左侧的x坐标。
y1要检查的矩形顶部的y坐标。
x2要检查的矩形右侧的x坐标。
y2要检查的矩形底部的x坐标。

返回值：

N/A

例子：

mp_grid_add_rectangle(grid, 0, 0, 100, 200);

上面的代码会将在变量“ grid”中索引的mp_grid的所有单元都标记为禁止，这些单元在0,0到100,200范围内。

背面：运动计划
