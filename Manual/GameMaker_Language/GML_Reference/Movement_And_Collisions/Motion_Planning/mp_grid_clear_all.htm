mp_grid_clear_all

使用此功能，您可以清除所有“禁止”单元格的MP网格。

句法：

mp_grid_clear_all(id);

争论描述
id要使用的mp_grid的索引

返回值：

Boolean

例子：

if !instance_exists(obj_Player)
    {
    mp_grid_clear_all(grid);
    }

如果对象“ obj_Player”的实例在房间中不再存在，则上面的代码将清除变量“ grid”中索引的mp_grid，将所有单元格标记为空闲。
