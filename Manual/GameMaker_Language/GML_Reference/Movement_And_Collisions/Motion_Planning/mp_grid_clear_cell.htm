mp_grid_clear_cell

此功能可用于清除MP网格的特定“单元”。像元不是作为房间坐标计算的，而是作为网格坐标计算的，其中（0,0）是网格的顶部让入角。这意味着要清除房间中特定位置的像元，我们必须将x和y坐标更改为像元坐标，再将它们除以MP网格的分辨率。下面的代码示例显示了它是如何工作的。

句法：

mp_grid_clear_cell(id, h, v);

争论描述
id要使用的mp_grid的索引
h要清除的单元格的水平位置
v要清除的单元格的垂直位置

返回值：

Boolean

例子：

with (obj_Box)
    {
    mp_grid_clear_cell(grid, floor(x / 32), floor(y /32));
    instance_destroy();
    }

上面的代码将使所有“ obj_Box”销毁自己，并使它们将在变量“ grid”中索引的mp_grid中占用的单元格标记为空闲。在此示例中，我们通过获取对象的x / y坐标并将它们除以网格的分辨率（使用 floor将值保持为整数）。

背面：运动计划
