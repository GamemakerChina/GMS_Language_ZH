mp_grid_clear_cell

这个函数可以用来清除MP网格的一个特定 "单元"。单元不是作为房间坐标计算的，而是作为网格坐标，其中（0,0）是网格的顶角。 我们必须把x和y坐标改成单元格坐标，并把它们除以MP网格的分辨率。下面的代码例子显示了这是如何工作的。

语法。

mp_grid_clear_cell(id, h, v)。

争论 描述
身份证 将要使用的mp_grid的索引。
h 要清除的单元格的水平位置
v 要清除的单元格的垂直位置

返回。

布尔型

例子。

with (obj_Box)
{
mp_grid_clear_cell(grid, floor(x / 32), floor(y / 32));
instance_destroy()。
}

上面的代码将使所有的 "obj_Box "自我销毁，并让它们把它们在变量 "grid "中索引的mp_grid中占据的单元格标记为空闲。在这个例子中，我们通过取物体的x/y坐标并除以分辨率来找到合适的单元格。 并除以网格的分辨率（使用floor来保持数值为整数）来找到合适的单元。
