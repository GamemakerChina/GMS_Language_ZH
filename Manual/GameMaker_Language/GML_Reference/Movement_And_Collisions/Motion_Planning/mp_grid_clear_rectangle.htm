mp_grid_clear_rectangle

使用此功能，您可以在房间坐标中定义一个区域，然后将其清除指定MP网格中的相应单元格。即使单元格部分落在定义的矩形区域内，也会清除该单元格。

句法：

mp_grid_clear_rectangle(id, x1, y1, x2, y2);

争论描述
id要使用的mp_grid的索引
x1要检查的矩形左侧的x坐标。
y1要检查的矩形顶部的y坐标。
x2要检查的矩形右侧的x坐标。
y2要检查的矩形底部的x坐标。

返回值：

Boolean

例子：

mp_grid_clear_rectangle(grid, 0, 0, 100, 200);

上面的代码会将在变量“ grid”中索引的mp_grid的所有单元格标记为空闲，这些单元格在0,0到100,200范围内。
