mp_grid_destroy

您可以使用此函数销毁指示的mp_grid（由该函数定义）mp_grid_create() ）并释放分配给它的内存。你把这个每当MP电网完成了，或者你可能会得到一个内存泄漏，这意味着你的游戏速度会变慢随着时间的推移，最终崩溃是必要的。

注意：使用 mp_grid_*网格上的函数被销毁后将导致错误。

句法：

mp_grid_destroy(id);

争论描述
id待销毁的mp_grid的索引

返回值：

N/A

例子：

if timer = 0
    {
    mp_grid_destroy(grid);
    room_goto(rm_Menu);
    }

如果变量“ timer”等于0，则上面的代码将销毁在变量“ grid”中索引的mp_grid，然后转到另一个房间。
