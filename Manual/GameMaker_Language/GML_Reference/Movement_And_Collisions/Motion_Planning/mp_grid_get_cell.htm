mp_grid_get_cell

使用此功能，您可以检查mp_grid的任何给定单元，以查看是否已将其标记为已占用，并给出（先前创建的）mp_grid的索引以及要检查的单元的x和y坐标。如果已被占用或正在检查的位置不在网格范围内，则该函数将返回-1，否则将返回0。

句法：

mp_grid_get_cell(id, x , y);

争论描述
id要使用的mp_grid的索引
x1要检查的网格的x坐标。
y1要检查的网格的y坐标。

返回值：

Real

例子：

if mp_grid_get_cell(grid, mouse_x div 16, mouse_y div 16) == -1
    {
    image_blend = c_red;
    }
else
    {
    image_blend = c_lime;
    }

上面的代码将检查与鼠标位置相对应的mp_grid单元，如果它被占用，它将设置 image_blend变量设置为红色，如果未被占用，则将其设置为绿色。

背面：运动计划
