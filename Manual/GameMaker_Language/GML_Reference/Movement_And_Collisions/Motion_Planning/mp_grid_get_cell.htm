mp_grid_get_cell

通过这个函数，你可以检查mp_grid的任何给定的单元格，看它是否被标记为被占用，给出mp_grid的索引（先前创建的）和要检查的单元格的x和y坐标。如果它已经被占用或者被检查的位置 则该函数将返回-1，否则将返回0。

语法。

mp_grid_get_cell(id, x , y)。

争论 描述
身份证 将要使用的mp_grid的索引。
x1 要检查的网格的X坐标。
y1 要检查的网格的y坐标。

返回。

真实的

例子。

if mp_grid_get_cell(grid, mouse_x div 16, mouse_y div 16) == -1
{
image_blend = c_red;
}
else
{
image_blend = c_lime;
}

上面的代码将检查与鼠标位置相对应的mp_grid单元，如果它被占用，就将image_blend变量设置为红色，如果它没有被占用，就设置为绿色。
