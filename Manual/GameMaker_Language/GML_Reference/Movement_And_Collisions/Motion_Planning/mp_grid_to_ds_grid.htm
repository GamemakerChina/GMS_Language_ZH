mp_grid_to_ds_grid

使用此功能，您可以将给定的MP网格复制到DS网格。 DS网格的大小应该与MP网格相同，尽管不必如此（如果数据较小，则数据将丢失；如果数据较大，则所有额外的网格像元将为0）。如果将MP网格单元格标记为已占用，则DS网格单元格将包含值-1；否则，其将为0。

句法：

mp_grid_to_ds_grid(source, destination);

争论描述
source要使用的mp_grid的索引
destination要将网格数据复制到的ds_grid的索引。

返回值：

Boolean

例子：

motion_grid = ds_grid_create(room_width / 32, room_height / 32);
mp_grid_to_ds_grid(mp_grid, motion_grid);

上面的代码将创建一个新的DS网格，然后将变量“ mp_grid”中包含的MP网格数据复制到新的DS网格中。

背面：运动计划
