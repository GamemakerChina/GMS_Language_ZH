mp_grid_to_ds_grid

通过这个函数，你可以把给定的MP网格复制到一个DS网格中。DS网格应该和MP网格一样大，尽管它不一定是这样的（如果它更小，数据就会丢失，而如果它更大，所有额外的网格单元都会变成0。 如果它更大，所有额外的网格单元将是0）。)如果MP网格单元被标记为占用，DS网格单元将包含数值-1，如果没有，则为0。

语法。

mp_grid_to_ds_grid(source, destination)。

争论 描述
来源 将要使用的mp_grid的索引。
目的地 要用来复制网格数据的ds_grid的索引。

返回。

布尔型

例子。

motion_grid = ds_grid_create(room_width / 32, room_height / 32);
mp_grid_to_ds_grid（mp_grid, motion_grid）。

上述代码将创建一个新的DS网格，然后将变量 "mp_grid "中包含的MP网格数据复制到新的DS网格中。
