mp_linear_path

这个函数计算从当前实例位置到 "xgoal "和 "ygoal "值所指定的点的直线路径。它使用指定的步长，与函数中的 mp_linear_step(). 指定的路径必须已经存在，并将被新的路径覆盖，函数将返回是否找到了完整的路径（true）或没有（false）。如果返回false，那么仍然会产生一个路径，但它只会运行到 的位置，但它只会运行到路径被阻止的位置。

注意：这个函数不会移动实例。它只设置一个路径，你必须使用路径函数来移动。

语法。

mp_linear_path(path, xgoal, ygoal, stepsize, checkall)。

争论 描述
路 要使用的路径的索引。
xgoal 目标X位置。
宗旨 目标Y位置。
步长 实例每步移动的速度，单位是像素。
检查所有 是检查所有实例的碰撞（真）还是只检查实心的（假）。

返回。

布尔型

例子。

if mp_linear_path(path, obj_Player.x, obj_Player.y, 2, 0)
{
path_start(path, 2, 0, 0);
}

这将使对象检查并查看从其当前位置到 "obj_Player "的位置是否有一个线性路径。如果有，它就开始走这条路。
