mp_linear_path_object

此函数计算从当前实例位置到“ xgoal”和“ ygoal”值指定的点的直线路径。它使用指示的步长，与功能相同 mp_linear_step()。指示的路径必须已经存在，并且将被新路径覆盖，并且如果找到完整路径，则函数将返回（真），否则将返回（假）。仅找到完整路径，并且与指定对象或实例没有冲突，如果返回false，则仍会生成路径，但它只会运行到该路径被阻塞的位置。

注意：此功能不会移动实例。它仅设置路径，并且必须使用路径功能进行移动。

句法：

mp_linear_path_object(path, xgoal, ygoal, stepsize, obj);

争论描述
path要使用的路径的索引
xgoal目标x位置。
ygoal目标y位置。
stepsize实例每步移动的速度（以像素为单位）。
obj阻塞路径的对象。可以是对象索引，实例ID或特殊关键字，全部

返回值：

Boolean

例子：

if mp_linear_path_object(path, mouse_x, mouse_y, 4, obj_Wall)
    {
    path_start(path, 4, 0, 0);
    }

上面的代码检查在运行代码的对象与鼠标的x / y位置之间的路径上是否与“ obj_Wall”发生冲突。如果没有碰撞并且生成了完整路径，则它将沿着生成的路径启动对象。

背面：运动计划
