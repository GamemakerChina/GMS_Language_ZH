mp_linear_step

通过这个函数，你可以告诉一个实例向一个特定的点迈出 "一步"，这个点由xgoal和ygoal值指定。这一步的大小（即实例每一步应该移动多少像素）由stepsize表示，并且 如果实例已经到了该位置，它将停止，不再继续移动，这与其他更简单的函数相反，如 move_towards_point().步长也是 "前面 "的距离 你可以选择实例是在与任何实例发生碰撞时停止，还是只与那些被标记为实体的实例发生碰撞。该函数将返回它是否已经到达目标位置 真）或失败（假）。

注意：如果遇到障碍物，该功能不会试图进行绕行，它只是失败并停止移动。

语法。

mp_linear_step(xgoal, ygoal, stepsize, checkall)。

争论 描述
xgoal 目标X位置。
宗旨 目标Y位置。
步长 实例每步移动的速度，单位是像素。
检查所有 是检查所有实例（true）还是只检查可靠的实例（false）。

返回。

布尔型

例子。

if mp_linear_step(mouse_x, mouse_y, 5, 0)
{
instance_create_layer(x, y, "Effects", obj_Explosion);
instance_destroy();
}

上面的代码将使对象以每步5像素的速度向鼠标移动。一旦它到达鼠标的位置，它将创建一个对象 "obj_Explosion "并销毁自己。
