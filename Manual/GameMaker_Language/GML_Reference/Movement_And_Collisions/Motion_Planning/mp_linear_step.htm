mp_linear_step

使用此功能，您可以告诉实例朝由xgoal和ygoal值指定的特定点迈出“一步”。步骤的大小（实例应在每个步骤中移动多少像素）由步骤大小指示，如果实例已经在该位置，它将停止并且不会再移动，这与其他更简单的功能（例如 move_towards_point()。步长大小也是对象将检查碰撞的每个步骤的“提前”距离，您可以选择实例是在与任何实例的碰撞处停止还是仅在标记为实体的实例处停止。该函数将返回是否达到目标位置（真）或失败（假）。

注意：如果遇到障碍物，此功能不会尝试绕行，只会失败并停止移动。

句法：

mp_linear_step(xgoal, ygoal, stepsize, checkall);

争论描述
xgoal目标x位置。
ygoal目标y位置。
stepsize实例每步移动的速度（以像素为单位）。
checkall是检查所有实例（是）还是仅检查实体实例（否）。

返回值：

Boolean

例子：

if mp_linear_step(mouse_x, mouse_y, 5, 0)
    {
    instance_create_layer(x, y, "Effects", obj_Explosion);
    instance_destroy();
    }

上面的代码将使对象以每步5像素的速度向鼠标移动。一旦到达鼠标位置，它将创建一个对象“ obj_Explosion”并销毁自身。
