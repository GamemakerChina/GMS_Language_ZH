mp_potential_step

这个函数让实例向一个由xgoal/ygoal定义的特定位置迈出一步，同时试图避开障碍。当实例遇到一个实心的实例时（或者当checkall为真时的任何实例），它将改变运动方向，试图避开该实例并绕过它。 试图避开该实例并在其周围移动。这种方法不能保证有效，但在大多数简单的情况下，它将有效地将实例推向目标。该函数返回是否达到了目标。

语法。

mp_potential_step(xgoal, ygoal, stepsize, checkall)

争论 描述
xgoal 目标X位置。
宗旨 目标Y位置。
步长 实例每步移动的速度，单位是像素。
检查所有 是检查所有实例（true）还是只检查可靠的实例（false）。

返回。

布尔型

例子。

if instance_exists(obj_Enemy)
{
var inst;
inst = instance_nearest(x, y, obj_Enemy);
mp_potential_step(inst.x, inst.y, 5, false);
}

上面的代码首先检查房间里是否有任何 "obj_Enemy "的实例。如果有的话，它就会找到最近的一个，并将其ID存储在一个变量中。然后这个变量被用来告诉mp_potential_step把 实例，以每步5像素的速度向找到的物体的X/Y位置移动，同时避开标记为实体的实例。
