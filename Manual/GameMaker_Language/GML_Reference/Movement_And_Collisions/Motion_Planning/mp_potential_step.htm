mp_potential_step

使用此功能，实例可在尝试避开障碍物的同时，朝由xgoal / ygoal定义的特定位置迈出一步。当实例碰到实体实例（或checkall为true的任何实例）时，它将更改运动方向以尝试避开该实例并在其周围移动。这种方法不能保证能正常工作，但在大多数情况下，它将有效地将实例移向目标。该函数返回是否达到目标。

句法：

mp_potential_step(xgoal, ygoal, stepsize, checkall)

争论描述
xgoal目标x位置。
ygoal目标y位置。
stepsize实例每步移动的速度（以像素为单位）。
checkall是检查所有实例（是）还是仅检查实体实例（否）。

返回值：

Boolean

例子：

if instance_exists(obj_Enemy)
    {
    var inst;
    inst = instance_nearest(x, y, obj_Enemy);
    mp_potential_step(inst.x, inst.y, 5, false);
    }

上面的代码首先检查房间中是否有“ obj_Enemy”的实例。如果有，则找到最近的一个并将其ID存储在变量中。这个变量然后被用来告诉 mp_potential_step将带有代码的实例移向以每步5像素的速度找到的对象的x / y位置，同时避免仅将实例标记为实体。

背面：运动计划
