潜在的阶梯对象(mp_potential_step_object)

这个函数让实例向由xgoal/ygoal定义的特定位置迈出一步，同时试图避开障碍物。当实例遇到一个由 "obj "参数指定的物体实例时，它将改变 运动方向，试图避开该实例并绕过它。这种方法不能保证有效，但在大多数情况下，它将有效地使实例向目标移动。该函数返回是否达到了目标。

语法。

mp_potential_step_object(xgoal, ygoal, stepsize, obj)

争论 描述
xgoal 目标X位置。
宗旨 目标Y位置。
步长 实例每步移动的速度，单位是像素。
obj 阻止运行该函数的实例的路径的对象。可以是一个对象的索引，一个实例的ID或特殊的关键字,all。

返回。

布尔型

例子。

if instance_exists(obj_Enemy)
{
var inst;
inst = instance_nearest(x, y, obj_Enemy);
mp_potential_step_object(inst.x, inst.y, 5, obj_Wall);
}

上面的代码首先检查房间里是否有任何 "obj_Enemy "的实例。如果有的话，它会找到最近的一个，并把它的ID存储在一个变量中。然后这个变量被用来告诉mp_potential_step_object将 的实例以每步5像素的速度向找到的物体的X/Y位置移动，同时避开物体 "obj_Wall "的实例。
