运动_添加

这个函数将修改运行代码的实例的当前 direction 和 speed 。 运行代码的实例，将给定的值与当前值相结合。如果你想简单地改变这些值，你应该使用函数 motion_set() 。

语法。

motion_add(dir, speed)。

争论 描述
dir 增加的方向。
速度 增加的速度。

返回。

不适用

例子。

var pdir;
pdir = point_direction(other.x, other.y, x, y);
motion_add(pdir, other.speed);
if speed > 8 speed = 8;

上述代码将在与另一个物体的碰撞事件中被调用。它为实例的运动方向和速度添加一个基于参与碰撞的其他实例的位置和速度的矢量。然后它限制 速度，如果它超过了8（每步像素）。
