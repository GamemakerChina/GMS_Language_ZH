motion_add

此功能将修改当前 direction和 speed运行代码的实例的实例，将给定的值与当前值相结合。如果您只想更改这些值，则应使用该函数 motion_set()。

句法：

motion_add(dir, speed);

争论描述
dir增加的方向。
speed增加的速度。

返回值：

N/A

例子：

var pdir;
pdir = point_direction(other.x, other.y, x, y);
motion_add(pdir, other.speed);
if speed > 8 speed = 8;

上面的代码将在与另一个对象的碰撞事件中被调用。它基于碰撞中所涉及的其他实例的位置和速度，将矢量添加到实例的运动方向和速度。然后，如果速度超过8（每步像素），它将限制速度。
