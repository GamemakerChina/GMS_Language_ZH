motion_set

此函数为运行代码的实例设置新的移动方向和新的速度。请注意，这不会增加实例的当前速度和方向（为此您将使用 motion_add()），而是将其强制为新设置。

句法：

motion_set(dir, speed);

争论描述
dir新方向。
speed新的速度。

返回值：

N/A

例子：

if irandom(9) = 1
    {
    motion_set(random(360), 1 + random(3));
    }

上面的代码将使实例以随机的间隔更改速度和方向。
