move_bounce_all

使用此功能，您可以命令一个实例使房间内的所有实例弹起，唯一的例外是没有为其分配精灵或蒙版索引的实例。您还可以告诉它在启用时使用精确的碰撞检查，但是请注意，这要求所有实例都启用精确的遮罩，并且由于必须执行的处理量，当涉及许多实例时，这将大大降低您的游戏速度。通常，这应该在实例的step事件中进行，但也可以在碰撞事件中有选择地使用，如下面的代码示例所示。

句法：

move_bounce_all( adv );

争论描述
adv是否启用高级退信（true）（false）。

返回值：

N/A

例子：

if other.visible
    {
    move_bounce_all(false);
    }

上面的代码将与另一个对象发生碰撞事件。首先检查对象是否可见，然后执行 move_bounce_all()行动。请注意，在这种情况下，反弹是有选择的，并且只会针对该碰撞进行计算，而不是针对每个实例的每个步骤进行计算。
