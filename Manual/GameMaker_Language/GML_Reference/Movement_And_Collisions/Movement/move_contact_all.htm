move_contact_all

此功能会将运行代码的实例沿指定方向移动一定数量的像素，直到遇到带有有效掩码的其他任何实例为止。您可以将-1或0用作maxdist，默认值为1000px，即： GameMaker Studio 2会将实例连续向上移动1000个像素，直到脱离碰撞为止。

句法：

move_contact_all(dir, maxdist);

争论描述
dir进入的方向。
maxdist对象可以移动的最大距离（-1或0，默认值为1000像素）。

返回值：

N/A

例子：

if !place_meeting(x, y + 1, all)
    {
    move_contact_all(270, -1);
    }

上面的代码将在实例下方检查是否存在冲突，如果没有冲突，则它将向下移动该冲突，直到发生冲突或直到覆盖1000像素为止。
