move_contact_solid

此函数会将运行代码的实例在指定方向上移动一定数量的像素，直到遇到带有标记为true的实心实例为止。您可以将-1或0用作maxdist，默认值为1000px，即： GameMaker Studio 2会将实例连续向上移动1000个像素，直到脱离碰撞为止。

句法：

move_contact_solid( dir, maxdist );

争论描述
dir进入的方向。
maxdist对象可以移动的最大距离（-1或0，默认值为1000像素）。

返回值：

N/A

例子：

if !place_meeting(x, y + 1, obj_Floor)
    {
    move_contact_solid(270, -1);
    }

上面的代码将在实例下方检查与“ obj_Floor”的冲突，如果不存在冲突，则将其向下移动直到发生冲突或覆盖了1000个像素。
