Move_contact_solid

这个函数会将运行代码的实例在指定的方向上移动一定数量的像素，直到它遇到一个带有 实体标记为真的实例。你可以 使用-1或0，最大距离默认为1000像素，即：GameMaker Studio 2将不断地将实例向上移动1000像素，直到它不被碰撞。

语法。

move_contact_solid( dir, maxdist )。

争论 描述
dir 要前进的方向。
maxdist 对象可以移动的最大距离（-1或0，默认值为1000像素）。

返回。

不适用

例子。

if !place_meeting(x, y + 1, obj_Floor)
{
move_contact_solid(270, -1);
}

上述代码将检查实例下方是否与 "obj_Floor "发生碰撞，如果没有，那么它将向下移动，直到发生碰撞，或覆盖1000个像素。
