move_outside_all

使用此功能，您可以告诉实例从任何方向，每个步长以任意数量的像素移出冲突，maxdist的值-1或0为默认的1000px，即： GameMaker Studio 2将移动实例持续增加1000像素，直到没有碰撞为止。

句法：

move_outside_all(dir, maxdist);

争论描述
dir进入的方向。
maxdist对象可以移动的最大距离（-1或0，默认值为1000像素）。

返回值：

N/A

例子：

if place_meeting(x, y, all)
   {
   move_outside_all(90, 1);
   }

上面的代码会将实例与其他实例发生冲突的每一步向上移动1个像素。
