move_outside_solid

使用此功能，您可以告诉实例移出冲突，且该实例在任何方向都标记为实心，并且每个步长标记为任何像素，maxdist的值-1或0为默认的1000px，即： GameMaker Studio 2会将实例连续移动1000像素，直到脱离碰撞为止。

句法：

move_outside_solid(dir, maxdist);

争论描述
dir进入的方向。
maxdist对象可以移动的最大距离（-1或0，默认值为1000像素）。

返回值：

N/A

例子：

if other.solid
    {
    var pdir;
    pdir = point_direction(other.x, other.y, x, y);
    move_outside_solid(pdir, -1);
    }

上面的代码将在发生碰撞事件时进行检查，以查看碰撞中的“其他”实例是否标记为“实体”。如果是，它将使实例脱离碰撞。
