移动_随机

通过这个函数，你可以将实例设置到房间里的任何位置，但是要对准一个 "不可见 "的网格。因此，hsnap和vsnap的值为32，将把实例设置为一个随机的位置，并与32x32的方格对齐。 你可以把这些值设置为1，以获得房间里的任何位置。

注意：由于GameMaker Studio 2每次都会生成相同的初始随机种子，使调试代码变得更加容易，所以这个函数在每次重新运行游戏时都会捕捉到相同的位置。为了避免这种行为 使用 randomise()在你的游戏开始时使用。这只是在测试和调试游戏的时候，因为最终的 可执行包将不会显示这种行为，每次游戏都是随机的。

语法。

move_random(hsnap, vsnap)。

争论 描述
hsnap 横向捕捉（"单元格 "之间的像素大小）。
vsnap 竖直抢答（"单元格 "之间的像素大小）。

返回。

不适用

例子。

if keyboard_check(vk_space)
{
move_random(1,1);
}

当按下空格键时，这将把实例移动到房间的任何地方的随机位置。
