move_random

使用此功能，您可以将实例设置为房间中任何位置，但与“不可见”网格对齐。因此，hsnap和vsnap的值为32会将实例设置为与32x32正方形网格对齐的随机位置。您可以将这些值设置为1，以获取房间中任何位置的位置。

注意GameMaker Studio 2每次都会生成相同的初始随机种子，因此使调试代码变得容易得多，因此，每次重新运行游戏时，此功能都会捕捉到相同位置。为了避免这种行为，请在游戏开始时使用randomise（）。这仅在测试和调试游戏时适用，因为最终的可执行程序包不会显示此行为，并且每次播放都是随机的。

句法：

move_random(hsnap, vsnap);

争论描述
hsnap水平捕捉（“单元”之间的像素大小）。
vsnap垂直捕捉（“单元”之间的像素大小）。

返回值：

N/A

例子：

if keyboard_check(vk_space)
    {
    move_random(1,1);
    }

当按下空格键时，会将实例移动到房间中任意位置的随机位置。
