Move_snap

这个函数用来将实例 "抓取 "到一个给定大小的网格上。它将被捕捉到hsnap和vsnap值定义的 "不可见 "网格上最近的对应位置。

语法。

move_snap( hsnap, vsnap ) 。

争论 描述
hsnap 横向捕捉（"单元格 "之间的像素大小）。
vsnap 竖直抢答（"单元格 "之间的像素大小）。

返回。

不适用

例子。

with (obj_Pieces)
{
if !place_snapped(32, 32)
{
move_snap(32, 32);
}
}

上面的代码检查 "obj_Pieces "的所有实例，看它们是否被固定在一个32x32像素的网格上，如果不是，就把它们固定在该网格中最近的位置。
