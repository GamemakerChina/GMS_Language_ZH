move_snap

此功能用于将实例“快照”到给定大小的网格。它将被捕捉到hsnap和vsnap值定义的“不可见”网格上最接近的对应位置。

句法：

move_snap( hsnap, vsnap );

争论描述
hsnap水平捕捉（“单元”之间的像素大小）。
vsnap垂直捕捉（“单元”之间的像素大小）。

返回值：

N/A

例子：

with (obj_Pieces)
    {
    if !place_snapped(32, 32)
        {
        move_snap(32, 32);
        }
    }

上面的代码检查“ obj_Pieces”的所有实例，以查看它们是否已捕捉到32x32像素的网格，如果不是，则将其捕捉到该网格中的最近位置。
