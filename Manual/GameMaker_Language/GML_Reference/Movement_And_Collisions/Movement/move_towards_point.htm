move_towards_point

该函数只是告诉运行代码的实例以设定的速度向设定点移动。请记住，一旦到达目的地，您就必须告诉它要做什么，因为它不会自己停下来。这意味着您可以在“振动”实例中向前移动3个像素，将x / y目标位置超调2个，然后再向后移动3个像素，将目标点超调一个，依此类推。 。有多种方法可以避免这种情况，例如使用 distance_to_point()在移动之前查看实例距指定位置有多远。

注意：此函数设置实例的速度，因此即使您停止使用此代码，实例也将继续沿先前的方向移动，因此，如果希望实例速度停止，则必须将其设置为0。请参见下面的代码示例。

句法：

move_towards_point( x, y, sp );

争论描述
x要移向的点的x位置。
y要移动的点的y位置。
sp每步以像素为单位移动的速度。

返回值：

N/A

例子：

if point_distance(x, y, target.x, target.y) > 5
    {
    move_towards_point(target.x, target.y, 5);
    }
else speed = 0;

上面的代码将以每步5像素的速度将实例移向在变量“目标”中索引的实例的位置。到达该位置后，速度将设置为0。
