place_snapped

使用此功能，您可以检查实例的原点（其x和y位置）是否与您指定的hsnap和vsnap值对齐到网格。

句法：

place_snapped(hsnap, vsnap);

争论描述
hsnap水平捕捉要检查。
vsnap垂直对齐以进行检查。

返回值：

Boolean

例子：

with (obj_Pieces)
    {
    if !place_snapped(32, 32)
        {
        move_snap(32, 32);
        }
    }

上面的代码检查“ obj_Pieces”的所有实例，以查看它们是否已捕捉到32x32像素的网格，如果不是，则将其捕捉到该网格中的最近位置。
