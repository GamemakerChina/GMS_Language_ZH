网络_连接

通过这个函数，你可以发送一个连接到服务器的请求。该函数接收要连接的套接字ID（见 network_create_socket())，并要求你提供要连接的IP地址 的IP地址（一个字符串）以及连接的端口，如果连接失败，将返回一个小于0的值。连接使用一个特殊的协议，确保只有GameMaker Studio 2游戏可以相互连接，但是如果你需要 连接到一个不是GameMaker Studio 2游戏的服务器，你可以使用 network_connect_raw().请注意，在默认情况下，该功能是同步的，这意味着你的游戏可能会在连接时出现 "挂起"。 当连接建立时。你可以为连接设置一个超时值，或者使用函数使创建异步。 network_set_config()，或者使用函数 network_connect_async()来代替。

语法。

network_connect(socket, url, port)。

争论 描述
插座 要使用的插座的ID。
网址 要连接的URL或IP（一个字符串）。
港口 要连接的端口。

返回。

真实的

例子。

client = network_create_socket(network_socket_tcp);
network_connect(client, "192.134.0.1", 6510) 。

上述代码将创建一个新的TCP套接字，然后试图通过该套接字连接到6510端口的指定IP地址。
