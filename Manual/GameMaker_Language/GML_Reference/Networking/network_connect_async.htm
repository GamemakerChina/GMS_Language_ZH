网络_连接_async

通过这个函数，你可以发送一个连接到服务器的请求。该函数接收要连接的套接字ID（见 network_create_socket())，并要求你提供要连接的IP地址 如果连接失败，将返回一个小于0的值。连接使用一个特殊的协议，确保只有GameMaker Studio 2游戏可以相互连接，但是如果你需要 连接到一个不是GameMaker Studio 2游戏的服务器，你可以使用 network_connect_async_raw().注意，这个函数是异步的，生成一个类型为network_type_non_blocking_connect 的异步网络事件。

语法。

network_connect_async(socket, url, port)。

争论 描述
插座 要使用的插座的ID。
网址 要连接的URL或IP（一个字符串）。
港口 要连接的端口。

返回。

真实的

例子。

client = network_create_socket(network_socket_tcp);
network_connect_async(client, "192.134.0.1", 6510) 。

上面的代码将创建一个新的TCP套接字，然后试图通过该套接字建立一个异步连接到6510端口的指定IP地址。
