network_connect_async

使用此功能，您可以发送连接到服务器的请求。该函数采用套接字ID进行连接（请参见 network_create_socket()），并要求您提供要连接的IP地址（一个字符串）以及要通过其进行连接的端口，如果连接失败，将返回小于0的值。该连接使用特殊协议，以确保仅GameMaker Studio 2游戏可以相互连接，但是，如果您需要连接到非GameMaker Studio 2游戏的服务器，则可以使用 network_connect_async_raw()。请注意，此函数是异步的，生成类型为“异步网络”的事件network_type_non_blocking_connect。

句法：

network_connect_async(socket, url, port);

争论描述
socket要使用的套接字的ID。
url要连接的URL或IP（字符串）。
port要连接的端口。

返回值：

Real

例子：

client = network_create_socket(network_socket_tcp);
network_connect_async(client, "192.134.0.1", 6510);

上面的代码将创建一个新的TCP套接字，然后尝试通过该套接字与端口6510上的给定IP地址建立异步连接。
