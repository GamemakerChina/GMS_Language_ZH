network_connect_raw_async

使用此功能，您可以发送连接到服务器的请求。该函数采用套接字ID进行连接（请参见 network_create_socket()），并要求您提供要连接的IP地址（一个字符串）以及要通过其进行连接的端口，如果连接失败，将返回小于0的值。此功能与 network_connect_async()就是该功能可以连接到任何服务器，并且对原始数据不做任何事，这意味着您必须在服务器端自己实现协议。请注意，此函数是异步的，生成类型为“异步网络”的事件network_type_non_blocking_connect。

句法：

network_connect_raw_async(socket, url, port);

争论描述
socket要使用的套接字的ID。
url要连接的URL或IP（字符串）。
port要连接的端口。

返回值：

Real

例子：

client = network_create_socket(network_socket_tcp);
network_connect_raw_async(client, "www.macsweeneygames.com", 6510);

上面的代码将创建一个新的TCP套接字，然后尝试通过它建立与端口6510上给定URL的“原始”异步连接。
