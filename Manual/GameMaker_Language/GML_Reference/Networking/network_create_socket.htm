network_create_socket

此功能用于为游戏创建新的客户端套接字，以通过网络进行通信。您必须定义套接字类型（请参见下面的常量列表），并且该函数将返回该套接字的唯一ID，该ID将在该套接字的所有其他函数调用中使用，如果连接失败，则该值应小于0 。

注意：您可以使用IP“ 127.0.0.1”连接回运行游戏的同一设备。

持续的描述
network_socket_tcp使用TCP创建套接字。
network_socket_udp使用UDP创建套接字。
network_socket_ws使用TCP创建Web套接字（仅适用于在HTML5上运行的项目）。
network_socket_bluetooth创建一个蓝牙套接字（当前不可用！ ）。

句法：

network_create_socket(type);

争论描述
type要创建的套接字连接的类型（请参见下面列出的常量）。

返回值：

Real

例子：

if os_browser == browser_not_a_browser
    {
    client = network_create_socket(network_socket_tcp);
    network_connect( client, "192.134.0.1", 6510 );
    }
else
    {
    client = network_create_socket(network_socket_ws);
    network_connect( client, "192.134.0.1", 6520 );
    }

上面的代码将检查游戏是否正在浏览器中运行，并在尝试通过游戏连接到给定端口上的给定IP地址之前创建一个新的TCP或Web套接字。
