网络_创建插座

这个函数用来为你的游戏创建一个新的客户端套接字，以便通过网络进行通信。你必须定义套接字类型（见下面的常量列表），该函数将为该套接字返回一个唯一的ID，该ID应被用于该套接字的所有后续函数调用。 在该套接字的所有后续函数调用中使用，如果连接失败，则返回一个小于0的值。

注意：你可以使用IP "127.0.0.1 "来连接回正在运行游戏的同一设备。

恒定 描述
网络_socket_tcp 使用TCP创建一个套接字。
network_socket_udp 使用UDP创建一个套接字。
network_socket_ws 创建一个网络套接字（仅用于在HTML5上运行的项目），使用TCP。
网络_socket_bluetooth 创建一个蓝牙插座（目前不可用！）。

语法。

network_create_socket(type)。

争论 描述
类型 要创建的套接字连接的类型（见下面列出的常数）。

返回。

真实的

例子。

如果 os_browser == browser_not_a_browser
{
client = network_create_socket(network_socket_tcp);
network_connect( client, "192.134.0.1", 6510 ) ;
}
else
{
client = network_create_socket(network_socket_ws);
network_connect( client, "192.134.0.1", 6520 );
}

上述代码将检查游戏是否在浏览器中运行，并创建一个新的TCP或Web套接字，然后尝试通过该套接字连接到指定端口的指定IP地址。
