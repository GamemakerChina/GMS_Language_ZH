network_create_socket_ext

此功能用于为游戏创建新的客户端套接字，以通过网络进行通信。您必须定义套接字协议（请参见下面的常量列表）并指定要使用的端口，该函数将返回一个唯一的ID ，该ID将在该套接字的所有其他函数调用中使用，如果小于，则该值小于0连接失败。

持续的描述
network_socket_tcp使用TCP创建套接字。
network_socket_udp使用UDP创建套接字。
network_socket_ws使用TCP创建Web套接字（仅适用于在HTML5上运行的项目）。

句法：

network_create_socket_ext(protocol, port);

争论描述
protocol使用的网络协议
port使用的端口

返回值：

Real

例子：

client = network_create_socket_ext(network_socket_udp, 6510);

上面的代码将在端口6510上创建一个新的UDP套接字。

返回：联网
