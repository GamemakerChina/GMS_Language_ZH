network_destroy

使用此功能，您可以从游戏中删除网络套接字连接。

注意：在HTML5目标上运行的项目中使用此功能时将不起作用。

句法：

network_destroy(socket);

争论描述
socket要删除的套接字的ID。

返回值：

N/A

例子：

if !os_is_network_connected()
   {
   network_destroy(sock);
   }

上面的代码将检查是否存在数据连接，如果找不到，请销毁ID为“ sock”的套接字。
