network_send_broadcast

使用此功能，您可以将缓冲区中的数据本地广播到一系列IP地址（有关缓冲区的更多信息，请参见参考-缓冲区）。该范围仅限于运行服务器的设备的范围，因此，如果该设备的IP为92.168.11.130，则数据将在范围92.168.11。*中广播。该函数将返回已发送数据的字节数，如果发送失败，则返回小于0的数字。

注意：此功能仅在与UDP一起使用时才起作用-您的服务器需要是TCP，并且您的客户端需要具有用network_create_socket_ext（）创建的UDP客户端套接字，以便接收从服务器发送的任何广播。

注意：在HTML5目标上运行的项目中使用此功能时将不起作用。

句法：

network_send_broadcast(socket, port, buffer, size);

争论描述
socket要使用的套接字的ID。
port服务器将使用的端口。
buffer从中获取数据的缓冲区的ID。
size数据大小（以字节为单位）。

返回值：

Real

例子：

buffer_seek(broadcast_buffer, buffer_seek_start, 0);
buffer_write(broadcast_buffer, buffer_string, global.ServerName);
network_send_broadcast(server, 6511, broadcast_buffer, buffer_tell(broadcast_buffer));

上面的代码写入当前服务器的名称字符串（存储在“ global.ServerName”中），然后将其写入ID为“ broadcast_buffer”的二进制缓冲区中。然后，此数据在本地广播到端口6511的IP范围（当前隐含设备IP作为广播的基本范围）。
