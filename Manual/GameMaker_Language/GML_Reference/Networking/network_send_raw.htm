network_send_raw

使用此功能，您可以通过网络发送“原始”数据包。该函数使用套接字ID进行连接，然后必须提供包含要发送的数据的缓冲区ID （有关缓冲区的更多信息，请参见Reference-Buffers ），最后是数据包的大小（以字节为单位）。发送的数据没有通过GameMaker Studio 2进行任何格式化，接收设备将以流的形式获取数据，这意味着您必须自己处理它。该函数将返回已发送数据的字节数，如果发送失败，则返回小于0的数字。

句法：

network_send_raw(socket, buffer, size);

争论描述
socket要使用的套接字的ID。
buffer从中获取数据的缓冲区的ID。
size数据大小（以字节为单位）。

返回值：

Real

例子：

buff = buffer_load("player_save.dat");
network_send_raw(sock, buff, buffer_get_size(buff));

以上信息将先前保存的缓冲区数据加载到内存中，并返回要存储在变量“ buff”中的缓冲区id。然后，使用变量“ sock”标识的套接字将该完整的缓冲区作为原始数据包通过网络发送。
