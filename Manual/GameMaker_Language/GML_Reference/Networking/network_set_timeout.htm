network_set_timeout

使用此功能，您可以设置通过给定套接字向服务器读取数据或从服务器写入数据的超时时间。请注意，超时不会生成任何类型的事件，因此您需要自己使用警报来处理超时（例如）。请注意，此值仅影响数据的发送和接收，并且如果您希望更改连接超时值，则应使用该功能 network_set_config()。

句法：

network_set_timeout(socket, read_timeout, write_timeout);

争论描述
socket要使用的套接字的ID。
read_timeout从服务器传输超时的毫秒数。
write_timeout传输到服务器的超时时间（以毫秒为单位）。

返回值：

N/A

例子：

network_set_timeout(sock, 3000, 3000);

上面的代码会将通过变量“ sock”中索引的套接字读取和写入数据的超时设置为3秒。
