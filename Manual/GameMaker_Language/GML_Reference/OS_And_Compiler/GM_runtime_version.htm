GM_ runtime _version

该常量保存 在“首选项”中runtime版本号，作为用于构建项目的runtime该值存储为字符串。

句法：

GM_runtime_version;

返回值：

String

例子：

draw_text(32, 32, date_time_string(GM_build_date));
draw_text(32, 64, "v" + GM_version);
draw_text(32, 96, "Runtime " + GM_runtime_version);

上面的代码采用GM日期，版本和runtime时常量，并将它们绘制到屏幕上。

返回： OS和编译器
