environment_get_variable

此函数返回具有给定名称（也是字符串）的环境变量的值（字符串）。您可以通过在Linux macOS和Ubuntu （Linux）上输入“env “进入终端应用程序，有关Windows环境变量的信息，如果您使用的是命令提示符，请键入”echo %PATH% “，并且使用PowerShell，它是“ls env “。请注意，在macOS和Ubuntu （ Linux ）上，“ HOME”环境变量都将返回“~/ “映射到”的路径 /Users/<username>在macOS和 /home/<username>在Ubuntu （ Linux ）上。

注意：此功能仅适用于Windows ， macOS和Ubuntu （ Linux ）。

句法：

environment_get_variable(name);

争论描述
name要检查的环境变量的名称（字符串）。

返回值：

String

例子：

e_str = environment_get_variable("APPDATA");

上面的代码将返回环境变量“%appdata% “，通常是”C:\Users\{username}\AppData\Roaming ”。

返回： OS和编译器
