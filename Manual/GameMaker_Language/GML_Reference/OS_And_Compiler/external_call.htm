external_call

如果您创建了外部函数，请调用 dll或者 dylib使用 external_define()，您可以使用此函数来调用它。您提供先前定义的函数的名称以及它需要的每个参数（每个参数必须为正确的类型，实数或字符串），并且该函数返回外部调用的结果。

句法：

external_call(id, args[0...15]);

争论描述
id您要调用的函数的名称
args[0...10]您想要传递给外部函数的不同参数

返回值：

Value (the type of value returned will depend on the defined function)

例子：

my_function = external_define("MyDLL.dll", "MyMin", dll_cdecl, ty_real, 2, ty_real, ty_real);
var _a = external_call(my_function, x, y);

上面的示例代码调用先前定义的外部函数，并将返回的值存储在局部变量中。

返回： OS和编译器
