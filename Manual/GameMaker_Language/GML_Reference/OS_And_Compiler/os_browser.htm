os_browser

此只读GameMaker Studio 2必须告诉您当前在哪个浏览器上运行游戏的各种常量（如下所列）之一。

句法：

os_browser

返回值：

Constant

持续的描述
browser_not_a_browser没有在浏览器中玩游戏
browser_unknown未知的浏览器
browser_ie IE浏览器
browser_ie_mobile移动设备上的Internet Explorer
browser_firefox火狐浏览器
browser_chrome谷歌浏览器
browser_safari苹果浏览器
browser_safari_mobile行动装置上的Safari
browser_opera歌剧
browser_tizen Tizen移动设备浏览器
browser_windows_store Windows应用程式

例子：

if (os_browser == browser_not_a_browser)
   {
   global.Config = 0;
   }
else
   {
   global.Config = 1;
   }

上面的代码检查游戏是否在浏览器中运行，并根据检查结果将全局变量设置为一个值。

返回： OS和编译器
