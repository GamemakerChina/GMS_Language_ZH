os_browser

这个只读变量持有GameMaker Studio 2的各种常量之一（如下所列），用来告诉你当前在哪个浏览器中运行游戏（如果有的话）。

语法。

os_browser

返回。

恒定

恒定 描述
浏览器_不是一个浏览器 游戏不是在浏览器中进行的
浏览器_未知 未知的浏览器
浏览器_ie Internet Explorer
浏览器_ie_mobile 移动设备上的Internet Explorer
浏览器_firefox 火狐浏览器
浏览器_chrome 谷歌浏览器
浏览器_safari 野生动物园
浏览器_safari_mobile 移动设备上的Safari浏览器
浏览器 歌剧
浏览器 Tizen移动设备浏览器
浏览器_windows_store Windows应用程序

例子。

如果（os_browser == browser_not_a_browser）
{
global.Config = 0;
}
else
{
global.Config = 1;
}

上面的代码检查游戏是否在浏览器中运行，并根据检查结果将一个全局变量设置为一个值。
