os_device

此只读变量保存下面列出的各种常量值之一，以告诉您当前在哪个设备上运行游戏。请注意，不建议使用此变量，而推荐使用该函数os_get_info()在运行游戏的设备上返回更准确的信息。

句法：

os_device

返回值：

Constant

持续的描述
device_ios_ipad的iPad
device_ios_ipad_retina新型iPad，Retina显示屏尺寸为2048 x 1536
device_ios_iphone6尺寸为1334 x 750的iPhone6
device_ios_iphone6plus具有显示屏1920 x 1080的更大iPhone 6
device_ios_iphone5尺寸为640 x 1136的iPhone5）
device_ios_iphone较旧的iPhone / iPod Touch（480 x 320屏幕）或Android手机
device_ios_iphone_retina带有960 x 640视网膜显示屏的新型iPhone / iPod Touch
device_emulator该设备实际上是模拟器（ Windows Phone或Android ）
device_tablet Android平板电脑
device_ios_unknown iOS未知

例子：

if (os_browser = browser_not_a_browser)
    {
    switch (os_device)
        {
        case device_ios_ipad: global.Config = 2; break;
        case device_ios_iphone: global.Config = 3; break;
        case device_ios_iphone_retina: global.Config = 4; break;
        case device_ios_unknown: global.Config = 5; break;
        }
    }
else
    {
    global.Config = 1;
    }

上面的代码检查游戏是否在浏览器中运行，并根据检查结果将全局变量设置为一个值。

返回： OS和编译器
