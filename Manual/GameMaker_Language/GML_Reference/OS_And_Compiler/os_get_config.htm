os_get_config

此函数返回您当前为游戏选择的配置的名称（以字符串形式）。有关配置的更多信息，请参见“配置”部分。

句法：

os_get_config()

返回值：

String

例子：

if os_get_config() = "Free_Version"
   {
   global.Ads = true;
   }
else global.Ads = false;

上面的代码将检查正在使用的配置，如果该配置称为“ Free_Version”，则将在游戏中启用广告。

返回： OS和编译器
