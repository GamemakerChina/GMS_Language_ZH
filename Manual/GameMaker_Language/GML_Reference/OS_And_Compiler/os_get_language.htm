os_get_language

此函数返回一个字符串，该字符串包含ISO639标准设置的运行游戏的操作系统的两个字母的语言代码。如果该信息不可用，它将仅包含一个空字符串“”或“ en”（代表“英语”语言）。请注意，有些语言也具有相关的地区代码，因此要使用同一功能来区分同一国家的不同地区 os_get_region()。

下表显示了ISO 639定义的一些主要的两种字母语言代码的示例：

语言代码
阿拉伯AR
中国人zh
丹麦文DA
英语恩
法语fr
德语德
希腊语埃尔
义大利文它
日本人JA
挪威不
抛光PL
葡萄牙语pt
俄语RU
西班牙语es
瑞典sv

注意：这不是返回的位置国家/地区代码，而是操作系统的语言代码。

句法：

os_get_language()

返回值：

String

例子：

switch (os_get_language())
   {
   case "es": ini_open("spanish.ini"); break;
   case "fr": ini_open("french.ini"); break;
   case "it": ini_open("italian.ini"); break;
   default: ini_open("english.ini"); break;
   ;

上面的代码检查OS语言并打开另一个 *.ini文件取决于返回的值。

返回： OS和编译器
