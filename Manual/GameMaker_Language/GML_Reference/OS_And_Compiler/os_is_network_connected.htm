os_is_network_connected

使用此功能，您可以检查并查看您的设备当前是否已连接互联网，并且它将返回 true如果有，或者 false如果不是这样，并且取决于操作系统，它可能会尝试在返回值之前建立连接。

该函数具有一个可选参数来启用/禁用它在被调用时尝试建立连接的功能，并且该选项仅适用于Nintendo Switch目标。设置为 true该功能将尝试建立连接，当设置为 false它只限于检查连接。请注意，尝试建立连接可能会导致Switch OS挂起几秒钟。

注意：此功能检查用于控制连接的内部设备API，因此，如果存在蓝牙连接，Wi-Fi连接，甚至只是允许Internet访问的普通网络连接，它都可能返回true。

句法：

os_is_network_connected(attempt_connection);

争论描述
attempt_connection选修的！仅适用于Nintendo Switch Target。设置 true尝试在调用时进行OS级连接。

返回值：

Boolean

例子：

if os_is_network_connected()
   {
   global.connected = true;
   }

上面的代码检查设备是否可以连接到互联网，如果可以，请设置全局变量。

返回： OS和编译器
