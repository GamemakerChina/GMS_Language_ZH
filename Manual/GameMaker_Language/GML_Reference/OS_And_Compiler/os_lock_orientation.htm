锁定方向

有了这个功能，你可以把你的设备 "锁定 "在当前的方向上，直到你 "释放 "它，让所有游戏选项启用的方向再次适用于该目标平台。 请注意，在锁定之前，你可能要确认方向是否符合要求。

语法。

os_lock_orientation(flag)

争论 描述
旗帜 设置为真或假，以启用或禁用方向锁定。

返回。

不适用

例子。

if (os_type == os_android) || (os_type == os_ios)
{
os_lock_orientation(true);
}

上面的代码检查了操作系统的类型，如果是Android或iOS，那么方向锁定就被标记为真。
