os_lock_orientation

使用此功能，您可以将设备“锁定”到当前方向，直到您“释放”设备，以再次允许该目标平台启用所有启用了游戏选项的方向。请注意，您可能需要在锁定之前确认方向是否正确。

句法：

os_lock_orientation(flag)

争论描述
flag设置 true或者 false启用或禁用方向锁定。

返回值：

N/A

例子：

if (os_type == os_android) || (os_type == os_ios)
   {
   os_lock_orientation(true);
   }

上面的代码检查OS类型，如果是Android或iOS则将方向锁定标记为 true。

返回： OS和编译器
