os_powersave_enable

使用此功能，您可以打开或关闭设备的节能功能。这很重要，因为某些游戏（例如，使用倾斜功能的游戏）可能不会生成OS可以解释为用户输入的事件，因此会关闭屏幕或退出游戏。通过将此功能设置为 false您可以禁用省电功能，并确保屏幕（和游戏）始终正常运行。

注意：这仅限于iOS和Android目标。

句法：

os_powersave_enable(flag)

争论描述
flag设置 true或者 false启用或禁用节电功能。

返回值：

N/A

例子：

if (os_type == os_android) || (os_type == os_ios)
   {
   os_powersave_enable(false);
   }

上面的代码检查操作系统类型，如果它是Android或iOS则将禁用省电功能。

返回： OS和编译器
