启用OS_POWERSAVE_ENABLE

通过这个功能，你可以打开或关闭设备的省电功能。这一点很重要，因为某些游戏（例如那些使用倾斜功能的游戏）可能不会产生操作系统可以解释为用户输入的事件，从而关闭屏幕或退出游戏。 屏幕或退出游戏。通过将此功能设置为假，你可以禁用省电功能，并确保屏幕（和游戏）始终正常运行。

注意：这仅限于iOS和Android目标。

语法。

os_powersave_enable(flag)

争论 描述
旗帜 设置为真或假，以启用或禁用省电功能。

返回。

不适用

例子。

if (os_type == os_android) || (os_type == os_ios)
{
os_powersave_enable(false);
}

上述代码检查操作系统类型，如果是安卓或iOS，那么省电功能将被停用。
