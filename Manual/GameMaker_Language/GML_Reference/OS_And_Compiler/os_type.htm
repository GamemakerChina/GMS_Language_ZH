种类

这个只读变量持有GameMaker Studio 2的各种常量之一，告诉你游戏是为哪个操作系统创建的。请注意，这不一定与运行游戏的设备的操作系统相同，因为 因为--例如--你的游戏可能在Amazon Fire操作系统上运行，但它是为Android平台创建的（在这种情况下os_type将是os_android）。

可以返回以下常数。

语法。

os_type;

返回。

恒定

例子。

switch (os_type)
{
case os_windows: global.Config = 0; break;
case os_android: global.Config = 1; break;
case os_linux: global.Config = 2; break;
case os_macosx: global.Config = 3; break;
case os_ios: global.Config = 4; break;
case os_winphone: global.Config = 5; break;
}

上述代码检查运行游戏的操作系统并相应地设置一个全局变量。
