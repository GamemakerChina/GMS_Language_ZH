os_type

这个只读变量包含各种常量，其中GameMaker Studio 2必须告诉您为该游戏创建了哪个操作系统。请注意，这不一定是运行它，因为相同设备的操作系统-例如-你的游戏可以在一个运行Amazon Fire OS，但对于将已建成Android平台（在这种情况下，os_type将 os_android）。

可以返回以下常量：

持续的描述
os_windows Windows操作系统
os_uwp Windows 10通用Windows平台
os_linux Linux
os_macosx macOS X
os_ios iOS （iPhone，iPad，iPod Touch）
os_tvos苹果tvOS
os_android Android
os_ps4索尼PlayStation 4
os_xboxone微软XBox One
os_switch Nintendo Switch
os_unknown未知的操作系统

句法：

os_type;

返回值：

Constant

例子：

switch (os_type)
   {
   case os_windows: global.Config = 0; break;
   case os_android: global.Config = 1; break;
   case os_linux: global.Config = 2; break;
   case os_macosx: global.Config = 3; break;
   case os_ios: global.Config = 4; break;
   case os_winphone: global.Config = 5; break;
   }

上面的代码检查运行游戏的OS，并相应地设置全局变量。

返回： OS和编译器
