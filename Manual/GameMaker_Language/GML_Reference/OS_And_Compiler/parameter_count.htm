parameter_count

命令行参数是可以添加到exe来更改程序运行方式的那些额外命令。您可以使用此函数找到当前游戏的参数数量，其中第一个参数的索引为1，最后一个参数的索引为该函数返回的值（0是特殊值，因为它是游戏可执行文件的文件名，包括路径）。应该注意的是，该函数将在HTML5平台上工作，并检索url参数。

句法：

parameter_count();

返回值：

Real

例子：

p_num = parameter_count();

上面的代码将在变量“ p_num”中存储已用于游戏的命令行参数的数量。

返回： OS和编译器
