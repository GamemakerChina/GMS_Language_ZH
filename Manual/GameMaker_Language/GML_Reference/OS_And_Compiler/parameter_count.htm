参数

命令行参数是那些额外的命令，你可以添加到exe中，以改变程序的运行方式。你可以用这个函数找到当前游戏的参数数量，其中第一个参数的索引是1，最后一个参数的索引是 该函数返回的索引（值为0很特别，它是游戏可执行文件的文件名，包括路径）。应该注意的是，这个函数将在HTML5平台上工作，检索的是url参数。

语法。

parameter_count()。

返回。

真实的

例子。

p_num = parameter_count()。

上面的代码将在变量 "p_num "中存储已经用于游戏的命令行参数的数量。
