参数_字符串

命令行参数是那些额外的命令，你可以添加到exe中，以改变程序的运行方式，用这个函数你可以得到所选择的命令行参数，作为一个字符串。你可以通过这个函数找到当前游戏的参数数量 使用函数 parameter_count()，其中第一个参数的索引为1，最后一个参数的索引由函数返回（值为0是特殊的，因为它是游戏可执行文件的文件名。 包括路径）。)应该注意的是，这个函数将在HTML5平台上工作，检索url参数。

语法。

parameter_string(n)。

返回。

字符串

例子。

var p_num;
p_num = parameter_count();
if p_num > 0
{
var i;
for (i = 0; i < p_num; i += 1)
{
p_string[i] = parameter_string(i + 1);
}
}

上面的代码将获得命令行参数的数量，如果有1个或更多，它将循环浏览这些参数并将它们作为字符串存储在一个数组中。
