parameter_string

命令行参数是可以添加到exe来更改程序运行方式的那些额外命令，使用此功能，您可以将所选的命令行参数作为字符串获取。您可以使用函数找到当前游戏的参数数量 parameter_count()，其中第一个参数的索引为1，最后一个参数的索引为函数返回的值（0值是特殊的，因为它是游戏可执行文件的文件名，包括路径）。应该注意的是，该函数将在HTML5平台上工作，并检索url参数。

句法：

parameter_string(n);

返回值：

String

例子：

var p_num;
p_num = parameter_count();
if p_num > 0
   {
   var i;
   for (i = 0; i < p_num; i += 1)
      {
      p_string[i] = parameter_string(i + 1);
      }
   }

上面的代码将获取命令行参数的数量，如果有1个或多个，它将遍历它们并将它们作为字符串存储在数组中。

返回： OS和编译器
