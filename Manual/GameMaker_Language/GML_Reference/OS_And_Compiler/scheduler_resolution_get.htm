scheduler_resolution_get

此函数用于检索Windows线程调度程序的分辨率（以毫秒为单位）。如果调度程序的分辨率设置为默认值（由Windows设置），则该函数将返回-1。

有关线程调度程序的信息，请参见页面。scheduler_resolution_set() 。

注意：此功能仅适用于Windows 。

句法：

scheduler_resolution_get();

返回值：

Real (or -1 for default)

例子：

scheduler_resolution = scheduler_resolution_get();

本示例检索Windows线程调度程序的分辨率，并将其存储在"scheduler_resolution"多变的。

返回： OS和编译器
