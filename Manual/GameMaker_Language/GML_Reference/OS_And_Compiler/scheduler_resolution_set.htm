scheduler_resolution_set

此功能用于更改Windows线程计划程序的分辨率。参数中提供的分辨率值必须以毫秒为单位。

默认情况下， GameMaker Studio 2使用1毫秒的分辨率，这是允许的最小分辨率，并提供最佳性能。将分辨率更改为较高的值可能会对性能产生负面影响，但也可能会降低功耗。

您可以提供-1作为参数，以将分辨率恢复为其默认值（由Windows设置）。请注意，更改调度程序的分辨率将影响与游戏同时运行的所有进程。

注意：此功能仅适用于Windows 。

句法：

scheduler_resolution_set(milliseconds);

争论描述
milliseconds新的分辨率值（以毫秒为单位）或默认为-1

返回值：

N/A

例子：

scheduler_resolution_set(2);

本示例将Windows线程调度程序的分辨率设置为2毫秒。
