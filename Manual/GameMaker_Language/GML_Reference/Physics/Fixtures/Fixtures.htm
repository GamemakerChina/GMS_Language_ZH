治具

GameMaker Studio 2的实例可以具有物理属性之前，必须为其分配夹具。固定装置将形状或形式绑定到实例，这增加了材质属性，例如密度，摩擦力和复原力。夹具可以是盒子，圆形，多边形或您想要的任何其他简单形状，因为GameMaker Studio 2提供的功能使您可以逐点定义自己的形状，然后一旦有了形状，就可以继续定义其物理属性。这些属性决定着分配了夹具的实例与物理世界中其他实例的反应方式，下面列出了这些属性：

  • 密度：材料的密度定义为每单位体积的质量。从本质上讲，它是对物质拥挤在一起的紧密程度的度量。如果希望固定装置是静态的（即固定的），则应将密度设置为0，从根本上使其无限密集。还使用密度0定义了以恒定速度运动且不受碰撞物理影响的运动对象。
  • 摩擦力：摩擦力是抵抗固体表面，流体层和材料元素相互滑动的相对运动的力。
  • 线性阻尼：这是导致运动物体失去动量的广义力。在“现实世界”中，事情通常不会永远发生。例如，一架纸飞机会飞行一会儿，但最终会由于作用在其上的空气摩擦而坠落，从而使其坠落。在我们创建的物理世界中，很难建模，因此每个夹具都具有自己的线性阻尼来模拟这种“广义摩擦”。
  • 角阻尼：如上所述，这是施加到对象上以阻止其永久旋转的广义力。
  • 复原：在物理学中，复原被定义为“物体或系统在弹性变形后返回其原始状态”，但是由于GameMaker Studio 2中的固定装置实际上是刚体并且无法变形，因此可以将其视为基本上如何固定装置的“弹性”。

GameMaker Studio 2仅需几个简单功能，便可以帮助您创建复杂的系统和交互，从而为您的游戏带来前所未有的逼真感。

注意：您可以定义一个灯具并将其绑定到多个实例（想想一个台球桌，其中所有球都具有相同的物理属性，而每个球都是一个谨慎的实体）。您还可以将多个固定装置绑定到单个实例并组合物理属性（例如，使用两个三角形多边形固定装置创建星形）

以下功能用于创建，绑定，设置碰撞然后删除固定装置：

  • physics_fixture_create
  • physics_fixture_bind
  • physics_fixture_bind_ext
  • physics_fixture_set_collision_group
  • physics_fixture_delete
  • physics_remove_fixture

灯具必须具有一定的形状，否则物理世界将无法检测到该形状，并且可以通过以下功能定义该形状：

  • physics_fixture_set_box_shape
  • physics_fixture_set_circle_shape
  • physics_fixture_set_edge_shape
  • physics_fixture_set_chain_shape
  • physics_fixture_set_polygon_shape
  • physics_fixture_add_point

为了使启用了物理的实例对周围的环境做出正确的反应，您使用的夹具必须定义特定的属性，以赋予夹具反弹，摩擦，质量等作用。以下功能用于设置夹具的这些属性。治具：

  • physics_fixture_set_density
  • physics_fixture_set_friction
  • physics_fixture_set_linear_damping
  • physics_fixture_set_angular_damping
  • physics_fixture_set_restitution
  • physics_fixture_set_sensor
  • physics_fixture_set_kinematic
  • physics_fixture_set_awake

在将灯具绑定到实例之后，您还可以设置其某些属性。绑定“基本”灯具时，可以选择将绑定的物理属性的唯一索引存储在变量中。然后可以在以下函数中使用它来更改某些属性或获取它们的当前值：

  • physics_get_friction
  • physics_get_density
  • physics_get_restitution
  • physics_set_friction
  • physics_set_density
  • physics_set_restitution
  • physics_mass_properties
