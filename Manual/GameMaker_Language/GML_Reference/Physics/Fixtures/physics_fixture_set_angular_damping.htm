物理学_灯具_设置矩形阻尼

如果你想想 "现实世界 "中的任何旋转物体，除非它有一个马达或在太空中，否则由于外力的影响（如与周围空气的摩擦），它会随着时间的推移而变慢。我们可以使用GameMaker Studio 2中的physics_fixture_set_angular_damping()函数来模拟这种效果，降低物理世界中实例的旋转速度，因为如果没有它，任何旋转的实例都会继续无限地旋转。阻尼参数应该在0到无穷大之间。 0意味着没有阻尼，而无穷大意味着完全阻尼。通常你会使用0到1之间的阻尼值，但如果需要，你可以使用任何非负值。

语法。

physics_fixture_set_angular_damping(fixture, damping)

争论 描述
固定设备 夹具的索引
减震 夹具的角度阻尼，通常在0和1之间

返回。

不适用

例子。

physics_fixture_set_angular_damping(fix_Ball, 0.1)。

上面的代码将把 "fix_ball "中索引的夹具的角度阻尼设为0.1。
