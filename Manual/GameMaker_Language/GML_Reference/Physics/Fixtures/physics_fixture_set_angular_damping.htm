physics_fixture_set_angular_damping

如果考虑“现实世界”中的任何旋转物体，除非它具有马达或在太空中，否则它会由于外力的影响（例如与周围空气的摩擦）而随着时间的流逝而减慢速度。我们可以使用功能 physics_fixture_set_angular_damping()在GameMaker Studio 2可以模拟这种效果并降低物理世界中实例的旋转速度，因为没有它，任何旋转的实例都将继续无限旋转。阻尼参数应在0到无穷大之间，0表示无阻尼，无穷大表示完全阻尼。通常，您将使用介于0和1之间的阻尼值，但是如果需要，您可以使用任何非负值。

句法：

physics_fixture_set_angular_damping(fixture, damping)

争论描述
fixture夹具的索引
damping灯具的角度阻尼，通常在0到1之间

返回值：

N/A

例子：

physics_fixture_set_angular_damping(fix_Ball, 0.1);

上面的代码会将“ fix_ball”中索引的灯具的角阻尼设置为0.1。

返回：照明灯
