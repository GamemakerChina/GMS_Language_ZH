物理学_灯具_设置箱体形状

这个函数为你的灯具定义了一个盒子的形状。它需要一半的宽度和高度，因为物理学界使用这个值比整个宽度/高度值要多得多，以确定碰撞等事情。

语法。

physics_fixture_set_box_shape(fixture, halfWidth, halfHeight)

争论 描述
固定设备 夹具的索引
半宽 盒子的一半宽度
半高 盒子的一半高度

返回。

不适用

例子。

physics_fixture_set_box_shape(fix_Border, room_width/2, 10)。

上面的代码将在变量 "fix_Border "中索引的灯具上应用一个盒子形状，宽度为房间，高度为20像素。
