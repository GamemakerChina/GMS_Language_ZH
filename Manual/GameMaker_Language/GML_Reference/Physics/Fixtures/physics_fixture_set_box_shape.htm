physics_fixture_set_box_shape

此功能定义灯具的盒子形状。它需要一半的宽度和高度，因为物理世界使用此值远远超过整个宽度/高度值来确定诸如碰撞之类的事物。

句法：

physics_fixture_set_box_shape(fixture, halfWidth, halfHeight)

争论描述
fixture夹具的索引
halfWidth盒子的一半宽度
halfHeight盒子的一半高度

返回值：

N/A

例子：

physics_fixture_set_box_shape(fix_Border, room_width/2, 10);

上面的代码将把盒子形状应用到在变量“ fix_Border”中索引的灯具，该灯具的房间宽度为20像素。

返回：照明灯
