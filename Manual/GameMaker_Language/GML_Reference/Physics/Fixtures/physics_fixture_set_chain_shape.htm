physics_fixture_set_chain_shape

此功能定义了“链状”夹具形状。链夹具包括多个点，然后使用边缘形状将它们连接在一起以将它们连接在一起。函数本身获取灯具的索引（ID），并可以告诉灯具是否循环。设置为 true将最后一个点连接到第一个点并带有一条边，同时将其设置为 false将不会。请注意，此函数on将使夹具准备好接受制作链所需的点，并且应在使用调用此函数后添加这些点。physics_fixture_add_point() ，就像建立多边形固定装置时一样。

创建链式夹具时，必须给它至少两个点，但是可以添加以组成最终夹具的后续点数不受限制。

句法：

physics_fixture_set_chain_shape(fixture, loop)

争论描述
fixture夹具的索引
loop是否关闭连锁店（true ） 或不 （false ）。

返回值：

N/A

例子：

var xx = -100;
var yy = room_height / 2;
var fix = physics_fixture_create(); physics_fixture_set_chain_shape(fix, false);
for (var i = 0; i < 10; i++;)
    {
    physics_fixture_add_point(fix, xx, yy);
    xx += 50 + random(150);
    yy = (room_height / 2) - 250 + irandom(500);
    }
physics_fixture_bind(fix, id);

上面的代码将创建一行“边缘”固定装置，这些固定装置在房间的整个长度上具有各种高度。

返回：照明灯
