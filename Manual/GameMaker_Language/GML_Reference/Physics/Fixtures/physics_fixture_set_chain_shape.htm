物理学_夹具_设置链的形状

这个函数定义了一个 "链式 "灯具形状。一个链式灯具是由一些点组成的，然后用边缘形状把它们连接在一起。这个函数本身需要灯具的索引（ID），并且可以告诉灯具的 循环或不循环。设置为"true"，最后一个点将与第一个点用边缘连接，而设置为"false"，则不会。请注意，这个函数是为灯具准备的，以接受制作链条所需的点。 而这些点应该在调用此函数后添加，使用 physics_fixture_add_point()来添加这些点，就像你在建立一个多边形夹具时一样。

当创建一个链式灯具时，你必须给它至少两个点，但你不限制后续的点的数量，你可以添加组成成品的灯具。

语法。

physics_fixture_set_chain_shape(fixture, loop)

争论 描述
固定设备 夹具的索引
循环 是否关闭该链（真）或不关闭（假）。

返回。

不适用

例子。

var xx = -100;
var yy = room_height / 2;
var fix = physics_fixture_create(); physics_fixture_set_chain_shape(fix, false);
for (var i = 0; i < 10; i++; )
{
physics_fixture_add_point(fix, xx, yy);
xx += 50 + random(150);
yy = (room_height / 2) - 250 + irandom(500);
}
physics_fixture_bind(fix, id)。

上面的代码将在房间的长度上创建一排具有不同高度的 "边缘 "灯具。

返回。固定装置
