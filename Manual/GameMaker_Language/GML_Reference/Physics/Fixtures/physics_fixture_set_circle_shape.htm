物理学_夹具_设置圆形_形状

这个函数为你的灯具定义了一个圆形，其半径由参数 "rad "定义。

语法。

physics_fixture_set_circle_shape(fixture, rad)

争论 描述
固定设备 夹具的索引
辐射 圆的半径

返回。

不适用

例子。

physics_fixture_set_circle_shape(fix_Ball, sprite_get_width(spr_Ball) / 2) 。

上面的代码将对变量 "fix_Ball "中索引的夹具应用一个圆形，其半径与精灵 "spr_Ball "的宽度除以2相同。

返回。固定装置
