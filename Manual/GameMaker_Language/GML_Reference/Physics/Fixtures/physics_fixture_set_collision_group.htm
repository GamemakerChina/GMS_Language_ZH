physics_fixture_set_collision_group

创建夹具后，可以将其定义为碰撞组的一部分。这意味着您可以告诉具有该灯具的实例始终与其他实例发生碰撞或从不碰撞，这些实例绑定了具有相同碰撞组的灯具。可以分配的值基于正数或负数，默认值为0（这意味着具有此灯具设置的实例将被忽略，除非为其创建了碰撞事件）。这是一张有助于说明的图片：

如您所见，设置碰撞组对于创建与周围的物理世界（而不是彼此）交互的复杂物理实体非常有用。但是，应注意的是，由于基于这样的组计算碰撞的复杂性，建议您尝试将一个房间中正在使用的组的数量保持在绝对最低限度。

句法：

physics_fixture_set_collision_group(fixture, group)

争论描述
fixture要绑定的装置
group灯具分配给的组（正值，0或负值）

返回值：

N/A

例子：

physics_fixture_set_collision_group(fix_Wheel, -1);

上面的代码将在变量“ fix_Wheel”中索引的灯具设置为组“ -1”的一部分。

返回：照明灯
