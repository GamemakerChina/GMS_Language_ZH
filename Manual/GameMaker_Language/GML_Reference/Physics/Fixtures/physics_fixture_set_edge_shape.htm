物理学_灯具_设置边缘形状

这个函数定义了一个 "边缘 "灯具形状。边缘形状是一条简单的线，当其他灯具越过它时将会产生碰撞，对于生成（例如）地形或在房间周围创建边界非常有用。 边缘的位置是用局部空间定义的，即：实例的X/Y位置被认为是（0,0），所以在创建它们时应该考虑到这一点（在下面的代码例子中，实例将被放置在 (0,0)的位置，以避免复杂化）。)

语法。

physics_fixture_set_edge_shape(fixture, local_x1, local_y1, local_x2, local_y2)

争论 描述
固定设备 夹具的索引
本地_1 边缘的起始X位置
本地_y1 边缘的起始Y位置
本地_x2 边缘的X端位置
本地_y2 边缘的末端Y位置

返回。

不适用

例子。

var xx = 0;
var y1 = room_height - 100;
var y2 = room_height - 50 - irandom(100);
for (var i = 0; i < 10; i++;)
{
var fix = physics_fixture_create();
physics_fixture_set_edge_shape(fix, xx, y1, xx + 50, y2);
physics_fixture_bind(fix, id);
physics_fixture_delete(fix) 。
xx += 50;
y1 = y2;
y2 = room_height - 50 - irandom（100）;
}

上面的代码将在房间的长度上创建一排具有不同高度的 "边缘 "灯具。

返回。固定装置
