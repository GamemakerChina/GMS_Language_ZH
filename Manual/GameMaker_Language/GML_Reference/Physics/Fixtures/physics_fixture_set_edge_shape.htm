physics_fixture_set_edge_shape

此功能定义“边缘”夹具形状。边缘形状只是一条直线，当其他固定装置重叠在其上时会产生碰撞，对于生成（例如）地形或在房间周围创建边界非常有用。边缘的位置是使用局部空间定义的，即：实例的x / y位置被视为（0,0），因此在创建边缘时应将其考虑在内（在下面的代码示例中，实例将具有放置在房间的（0,0）以避免并发症）。

句法：

physics_fixture_set_edge_shape(fixture, local_x1, local_y1, local_x2, local_y2)

争论描述
fixture夹具的索引
local_x1边缘的起始x位置
local_y1边缘的y位置
local_x2边缘的x端位置
local_y2边缘的y位置

返回值：

N/A

例子：

var xx = 0;
var y1 = room_height - 100;
var y2 = room_height - 50 - irandom(100);
for (var i = 0; i < 10; i++;)
    {
    var fix = physics_fixture_create();
    physics_fixture_set_edge_shape(fix, xx, y1, xx + 50, y2);
    physics_fixture_bind(fix, id);
    physics_fixture_delete(fix);
    xx += 50;
    y1 = y2;
    y2 = room_height - 50 - irandom(100);
    }

上面的代码将创建一行“边缘”固定装置，这些固定装置在房间的整个长度上具有各种高度。

返回：照明灯
