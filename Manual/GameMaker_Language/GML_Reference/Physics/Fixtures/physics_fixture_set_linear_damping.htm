physics_fixture_set_linear_damping

阻尼用于降低实例的物理世界速度，它与摩擦的区别在于，仅当两个带有夹具的实例接触时才发生摩擦。模拟阻尼也要比摩擦便宜得多，但是请注意，阻尼不能代替摩擦。这两种效果可以并且应该一起使用。这是说明阻尼的图像：

阻尼参数应在0到无穷大之间，0表示无阻尼，无穷大表示完全阻尼。通常，您将使用介于0和1之间的阻尼值，但是如果需要，您可以使用任何非负值。

句法：

physics_fixture_set_linear_damping(fixture, damping)

争论描述
fixture夹具的索引
damping灯具的阻尼，通常在0到1之间

返回值：

N/A

例子：

physics_fixture_set_linear_damping(fix_Ball, 0.1);

上面的代码会将“ fix_ball”中索引的灯具的线性阻尼设置为0.1。

返回：照明灯
