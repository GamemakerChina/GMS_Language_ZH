physics_fixture_set_polygon_shape

该功能为您的灯具设置了多边形形状，但是您需要使用 physics_fixture_add_point()实际定义该多边形相对于灯具原点的形状。将装置绑定到实例时，多边形被关闭。您还应该注意，在定义任何点之前必须调用此函数，并且在将其绑定到实例之前还必须为多边形定义至少三个点，否则会出现错误。

句法：

physics_fixture_set_polygon_shape(fixture)

争论描述
fixture夹具的索引

返回值：

N/A

例子：

physics_fixture_set_polygon_shape(fix_Ship);
physics_fixture_add_point(fix_Ship, 0,0);
physics_fixture_add_point(fix_Ship, -40, 100);
physics_fixture_add_point(fix_Ship, 40, 100);

上面的代码将对变量“ fix_Ship”中索引的灯具应用多边形，然后定义三个点以创建三角形。

返回：照明灯
