物理学_夹具_设置恢复原状

在物理学中，恢复被定义为 "物体或系统在弹性变形后恢复到原来的状态"，但由于GameMakerStudio 2中的夹具是真正的刚性体，不能变形，恢复是指 实际上是说夹具的 "弹跳 "程度。这个设置将影响一个物体在与其他物体碰撞时的 "弹跳 "程度，并且与作用在实例上的其他力量（如重力和摩擦力）共同决定。 通常是一个介于0和1之间的值（可以使用更高的值，但可能会产生不可预知的结果）。下面是一个关于它如何工作的说明。

语法。

physics_fixture_set_restitution(fixture, restitution)

争论 描述
固定设备 夹具的索引
归还 夹具的恢复性（通常在0和1之间）。

返回。

不适用

例子。

physics_fixture_set_restitution(fix_Ball, 0.9)。

上面的代码将把 "fix_ball "中索引的夹具的还原度设为0.9。
