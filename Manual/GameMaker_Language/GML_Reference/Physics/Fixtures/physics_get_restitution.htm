physics_get_restitution

当您使用以下方式将灯具绑定到实例时 physics_fixture_bind()这将为绑定的灯具返回一个“ id”。您随时可以使用此ID来获取绑定的夹具而不是“基础”夹具的恢复值（“弹性”属性）。

句法：

physics_get_restitution(fixture)

争论描述
fixture绑定夹具的ID

返回值：

Real

例子：

var rest = physics_get_restitution(fix_id);
physics_set_restitution(fix_id, rest * 2);

上面的代码获取实例的绑定物理属性的当前恢复值，然后将其设置为其他值。

返回：照明灯
