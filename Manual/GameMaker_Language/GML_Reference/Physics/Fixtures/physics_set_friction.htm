physics_set_friction

当您使用以下方式将灯具绑定到实例时 physics_fixture_bind()这将为绑定的灯具返回一个“ id”。您可以随时使用此ID来设置绑定夹具而非“基础”夹具的摩擦值。请注意，摩擦通常设置为0到1之间的值，但是如果需要，您可以使用任何非负值。

句法：

physics_set_friction(fixture, friction)

争论描述
fixture绑定夹具的ID
friction要应用的新摩擦值

返回值：

N/A

例子：

var fric = physics_get_friction(fix_id);
physics_set_friction(fix_id, fric + 0.1);

上面的代码获取实例的绑定物理属性的当前摩擦值，然后将它们设置为其他值。

返回：照明灯
