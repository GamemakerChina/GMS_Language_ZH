physics_set_restitution

当您使用以下方式将灯具绑定到实例时 physics_fixture_bind()这将为绑定的灯具返回一个“ id”。您可以随时使用此ID来设置绑定的夹具（而不是“基础”夹具）的恢复值。恢复通常设置为0到1之间的值，但是如果需要，您可以使用更高的值，尽管结果可能无法预测。

句法：

physics_set_restitution(fixture, restitution)

争论描述
fixture绑定夹具的ID
restitution适用的新赔偿金价值

返回值：

N/A

例子：

var rest = physics_get_restitution(fix_id);
physics_set_restitution(fix_id, rest * 2);

上面的代码获取实例的绑定物理属性的当前恢复值，然后将其设置为其他值。

返回：照明灯
