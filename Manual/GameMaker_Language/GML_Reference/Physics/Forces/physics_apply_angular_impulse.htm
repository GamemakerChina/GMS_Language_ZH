physics_apply_angular_impulse

此功能将为启用了物理的实例提供角度脉冲。该脉冲将忽略给定的当前转矩来设置给定的角旋转量，从而实质上设置了灯具具有的“旋转”量。如果您希望使用扭矩将角力施加到实例，则应使用该功能 physics_apply_torque()。

句法：

physics_apply_angular_impulse(impulse)

争论描述
impulse施加的脉冲（以牛顿米为单位）

返回值：

N/A

例子：

if keyboard_check(vk_up)
    {
    physics_apply_angular_impulse(10);
    }

如果按下一个键，上面的代码将给物理启用的夹具一个角度脉冲。
