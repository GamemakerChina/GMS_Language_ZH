physics_apply_force

使用物理学时，您不仅需要将重力应用于对象，还需要将力应用于对象。力是导致物体发生速度变化，方向变化或形状变化的任何影响力，在GameMaker Studio 2物理世界中定义的作用力与我们定义重力的作用相同，即通过从两个xforce / yforce组件。应该注意的是，作用力不是作用在物体的质心上，而是作用在房间的某个点上，它们会根据作用的任何其他作用力来改变方向和速度以及转矩（旋转）。在物体上（如重力）。这是一个示例图像，以帮助说明：

在上面的示例中，底部的弹簧对象在球对象上产生了一个力，使它沿不同的方向移动。确切的力由我们从分量xforce / yforce相对于xpos / ypos坐标获得的矢量定义-简单地讲，该力的计算方式是从xpos / ypos到xforce / yforce的距离以牛顿为单位（牛顿为定义为以每秒一米的平方的速度加速一公斤质量所需的净力大小，方向是从xpos / ypos到xforce / yforce的角度。类似于上面的碰撞代码将在弹簧对象的碰撞事件中出现，并且看起来类似于以下示例中提供的代码。

注意：物理世界使用米来进行所有计算，并且在房间编辑器中或使用physics_world_create（）定义了每米的像素比率。

句法：

physics_apply_force(xpos, ypos, xforce, yforce)

争论描述
xpos施加力的房间中的x坐标
ypos施加力的房间中的y坐标
xforce力矢量的x分量
yforce力矢量的y分量

返回值：

N/A

例子：

with (other)
    {
    physics_apply_force(x, y, 0, -30);
    }

上面的代码将以30牛顿的力和一个方向将代码应用于碰撞中的另一个对象。
