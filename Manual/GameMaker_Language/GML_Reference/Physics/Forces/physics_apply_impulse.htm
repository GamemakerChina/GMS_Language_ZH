physics_apply_impulse

GameMaker Studio 2的物理将力和重力应用于对象，还可以施加脉冲。这与力略有不同，在施加力时会立即影响物体的速度，并可能会影响物体的扭矩（或“旋转”），特别是如果选择施加脉冲的点未与中心对齐质量（请注意：质心不一定与原点相同！）。这是一个例子：

如您所见，玩家单击了球，这给了它向右上方移动的冲动，它随其旋转。确切的脉冲是由我们从分量ximpulse / yimpulse相对于xpos / ypos坐标获得的矢量定义的-这简单地表示，脉冲的计算是从xpos / ypos到ximpulse / yimpulse的距离（以牛顿为单位）以及方向是我们从xpos / ypos到ximpulse / yimpulse的角度。

注意：物理世界使用米来进行所有计算，并且在房间编辑器中或使用physics_world_create（）定义了每米的像素比率。

句法：

physics_apply_impulse(xpos, ypos, ximpulse, yimpulse)

争论描述
xpos施加脉冲的房间中的x坐标
ypos施加脉冲的房间中的y坐标
ximpulse脉冲向量的x分量
yimpulse脉冲矢量的y分量

返回值：

N/A

例子：

if mouse_check_button_pressed(mb_left)
    {
    with (place_meeting(mouse_x, mouse_y, all))
        {
        physics_apply_impulse(mouse_x, mouse_y, -10 + irandom(20), -10 + irandom(20));
        }
    }

上面的代码将向按下鼠标左键的实例施加随机矢量脉冲。
