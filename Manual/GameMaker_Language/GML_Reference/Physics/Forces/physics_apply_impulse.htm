物理学_应用_冲动

在GameMaker Studio 2中，你不仅可以用物理学对物体施加力和重力，而且还可以施加冲力。这与力略有不同，因为当它被应用时，会立即 影响到速度，以及。 可能会影响物体的扭矩（或 "旋转"），特别是如果选择应用冲力的点不与质量中心对齐（注意：质量中心不一定与原点相同！）。下面是一个例子。

正如你所看到的，玩家已经点击了球，这给了它一个冲动 向右上方移动，边走边转。确切的冲力是由我们从与xpos/ypos坐标有关的组件ximpulse/yimpulse得到的矢量定义的--这意味着冲力被计算为从 xpos/ypos到ximpulse/yimpulse的距离，单位是牛顿，而方向是我们从xpos/ypos到ximpulse/yimpulse的角度。

注意：物理世界的所有计算都使用米，每米的像素比例在房间编辑器中定义，或者使用 physics_world_create().

语法。

physics_apply_impulse(xpos, ypos, ximpulse, yimpulse)

争论 描述
xpos 脉冲将被应用的房间的X坐标
ypos 脉冲将被应用的房间内的y坐标
ximpulse 脉冲向量的X分量
yimpulse 脉冲向量的Y分量

返回。

不适用

例子。

if mouse_check_button_pressed(mb_left)
{
with (place_meeting(mouse_x, mouse_y, all))
{
physics_apply_impulse(mouse_x, mouse_y, -10 + irandom(20), -10 + irandom(20));
}
}

上面的代码将对按下左键时处于鼠标位置的实例应用一个随机矢量的脉冲。

返回。力量
