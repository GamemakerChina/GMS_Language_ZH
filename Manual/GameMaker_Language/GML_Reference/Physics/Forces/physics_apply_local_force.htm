physics_apply_local_force

在物理世界中使用力量的另一种方法是将其局部应用于实例。这就是说，力的强度和方向是根据实例的原点（如果没有精灵，则是位置）来计算的，而没有考虑它在游戏室或物理世界中可能具有的方向或旋转。应该注意的是，使用此功能时，力不会施加到对象的质心，而是会在相对于实例的点上施加，并且由于它们依赖于正在作用的任何其他力，因此不会立即施加力对象（如重力）。下图说明了局部部队的工作方式：

如您所见，xlocal和ylocal坐标是根据始终被视为实例（0,0）的原点的轴位置来计算的，因此无论对象在房间中的哪个位置，xlocal和ylocal总是会是相同的。然后，您必须确定力的矢量分量（xforce和yforce）相对于xlocal / ylocal位置的位置。确切的力是由我们从分量xforce / yforce相对于这些坐标获得的矢量定义的-这仅表示力是以牛顿为单位从xlocal / ylocal到xforce / yforce的距离来计算的，方向是角度我们从xlocal / ylocal到xforce / yforce。这似乎有些令人困惑，但是您只需要看一下插图即可了解其工作原理，并且可以节省一些复杂的数学运算！

注意：物理世界使用米来进行所有计算，并且在房间编辑器中或使用physics_world_create（）定义了每米的像素比率。

句法：

physics_apply_local_force(xlocal, ylocal, xforce, yforce)

争论描述
xlocal相对于将施加力的原点的x坐标
ylocal相对于将施加力的原点的y坐标
xforce力矢量的x分量
yforce力矢量的y分量

返回值：

N/A

例子：

if keyboard_check(vk_right)
    {
    physics_apply_local_force(-25, 0, -50, 0);
    }

上面的代码将在按下右箭头键的同时，相对于原点从左到右对50牛顿的实例施加力。
