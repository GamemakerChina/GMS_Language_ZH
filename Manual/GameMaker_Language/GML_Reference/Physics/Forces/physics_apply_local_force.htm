物理学_应用_局部力

在物理世界中使用力的另一种方式是将它局部应用于一个实例。这意味着，力的强度和方向是根据实例的原点（如果没有精灵，则是位置）来计算的，而不考虑它在游戏室或物理世界中的方向或旋转。 不考虑它在游戏室或物理世界中可能有的方向或旋转。应该注意的是，在这个函数中，力不是施加在物体的质心上，而是在相对于实例的一个点上 而且它们不会被立即应用，因为它们依赖于任何其他作用在物体上的力（比如重力）。这个插图展示了局部力是如何工作的。

正如你所看到的，xlocal和ylocal坐标的计算基础是 轴的位置总是被认为是实例的原点（0,0），所以无论物体在房间的什么地方，xlocal和ylocal总是相同的。然后你必须决定力的矢量分量的位置 (xforce和yforce)相对于xlocal/ylocal的位置应该放在哪里。确切的力是由我们从xforce/yforce分量中得到的与这些坐标有关的矢量来定义的--这简单地意味着，力被计算为 从xlocal/ylocal到xforce/yforce的距离，单位是牛顿，而方向是我们从xlocal/ylocal到xforce/yforce的角度。这可能看起来有点混乱，但你只需看一下插图就能知道这是如何工作的，而且 它为你节省了一些复杂的数学运算!

注意：物理世界的所有计算都使用米，每米的像素比例在房间编辑器中定义，或者使用 physics_world_create().

语法。

physics_apply_local_force(xlocal, ylocal, xforce, yforce)

争论 描述
xlocal 相对于原点的X坐标，力将被施加在那里。
辽宁省 相对于原点的Y坐标，力将被施加于此。
xforce 力矢量的X分量
yforce 力矢量的y分量

返回。

不适用

例子。

if keyboard_check(vk_right)
{
physics_apply_local_force(-25, 0, -50, 0);
}

上面的代码将在按下右方向键时，对实例施加一个相对于原点从左到右的50牛顿的力。

返回。力量
