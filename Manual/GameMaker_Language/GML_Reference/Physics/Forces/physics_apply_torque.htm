physics_apply_torque

当谈论扭矩时，通常是指力或冲动使实例绕其质心旋转的趋势，但是在GameMaker Studio 2您也可以独立地将扭矩应用于实例，从而使实例在给定的范围内旋转方向。要获得所需的旋转，要施加多少扭矩全部取决于分配给实例的夹具的大小和密度，因为这两个属性都将定义其质量，从而定义其惯性，从而影响扭矩！一般的经验法则是，如果实例具有很大的惯性，则需要大量扭矩才能使其开始旋转。还应注意，其他因素（例如重力和碰撞）也会影响实例的扭矩。这是一张有助于说明的图片：

如您所见，旋转对象所需的扭矩大小取决于对象的基本质量和惯性。 GameMaker Studio 2根据创建实例时分配给实例的夹具的大小和密度为您计算的。另请注意，对于逆时针旋转，该值应为负。

句法：

physics_apply_torque(torque)

争论描述
torque要施加的扭矩（以牛顿米为单位）

返回值：

N/A

例子：

if keyboard_check(vk_left)
    {
    physics_apply_torque(-100);
    }

上面的代码将在按下左箭头键时以100Nm的扭矩将实例逆时针旋转（向左）。
