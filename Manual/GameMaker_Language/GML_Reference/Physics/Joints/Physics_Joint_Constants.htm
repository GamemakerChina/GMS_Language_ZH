物理联合常数

GameMaker Studio 2 GML中包含大量常量，这些常量特定于物理世界中夹具之间的连接。这些可以与许多不同的关节功能结合使用，以在物理模拟运行时实时设置或从中获取信息。但是，您应该知道，调用这些函数时会完成复杂的计算，因此应谨慎使用它们，并且仅在必要时使用，并请注意，许多计算对于特定类型的关节是唯一的。还要注意，尽管可以使用适当的功能获得所有这些值，但是只能设置标记为非只读的值。

通常，这些常量将与以下功能结合使用：

  • physics_joint_get_value
  • physics_joint_set_value

以下常量可以应用于任何可用的关节类型：

持续的描述只读
phy_joint_anchor_1_x房间中关节的第一个锚点的x坐标Yes
phy_joint_anchor_1_y房间中关节的第一个锚点的y坐标Yes
phy_joint_anchor_2_x房间中关节的第二个锚点的x坐标Yes
phy_joint_anchor_2_y房间中关节的第二个锚点的y坐标Yes
phy_joint_reaction_force_x这是在x锚点位置的关节中向第二个实例施加的反作用力Yes
phy_joint_reaction_force_y这是在y锚点位置的关节中向第二个实例施加的反作用力Yes
phy_joint_reaction_torque这是在锚点位置的关节中施加到第二个实例的扭矩Yes


这些常数适用于连接了电动机（旋转，棱柱形，车轮）的那些关节：

持续的描述只读
phy_joint_max_motor_force创建关节时为最大电动势指定的值No
phy_joint_max_motor_torque创建关节时为最大电机扭矩指定的值No
phy_joint_motor_force当前的马达力Yes
phy_joint_motor_speed当前电机转速No
phy_joint_motor_torque当前电机转矩Yes


对于旋转关节，可以使用以下常数（如果已添加，还可以使用电动机常数）：

持续的描述只读
phy_joint_angle关节的两个锚点之间的线所成的角度。这是使用物理世界坐标计算的
（不是GameMaker Studio 2房间坐标），以弧度为单位。		 Yes
phy_joint_angle_limits启用或禁用关节的角度限制。将值设置为 true启用或 false禁用。 No
phy_joint_upper_angle_limit关节的角度上限（以度为单位）。 No
phy_joint_lower_angle_limit关节的角度下限（以度为单位）。 No


对于棱柱形接头，可以使用以下常数：

持续的描述只读
phy_joint_translation获取锚点x / y坐标与局部x / y坐标之间的距离。 Yes
phy_joint_speed当前关节运动速度。 Yes


对于距离，焊接和车轮接头，可以使用以下常数（以及滑轮接头的常数）：

持续的描述只读
phy_joint_damping_ratio阻尼比是无量纲的，并且定义了关节的“弹性”。该常数的值通常在0到1之间，但可以更大，在1时，阻尼是至关重要的，这意味着所有振荡都应消失。 No
phy_joint_frequency这将返回（或设置）关节的振动频率（以赫兹为单位），通常该频率应小于该功能设置的时间步长的一半。physics_world_update_speed() 。 No
phy_joint_length_1这会将关节的长度从第一个局部x / y坐标返回到第一个锚点x / y坐标（仅距离关节，只能从中读取） Yes
phy_joint_length_2这会将关节的长度从第二个局部x / y坐标返回到第二个锚点x / y坐标（仅距离关节，只能写入） No


对于摩擦接头，可以使用以下常数：

持续的描述只读
phy_joint_max_torque关节的最大扭矩值。 No
phy_joint_max_force关节的最大力值。 No


对于绳接头，可以使用以下常量：

持续的描述只读
phy_joint_max_length两个锚点之间连接的最大扩展。 No
