physics_joint_get_value

通过使用一系列预定义的常量，您可以要求GameMaker Studio 2告诉您有关任何给定关节状态的许多信息。这非常有用，因为它使您能够根据当时的需要删除关节或更改实例行为。此函数中可以使用许多常量，可以在此处找到：物理联合常量，但是请注意，在调用这些常量时会进行复杂的计算，因此应谨慎使用它们，并且仅在必要时使用并注意有许多是特定类型的关节所特有的。

如果该属性不存在（例如，如果您检查滑轮接头是否为棱柱形平移），则返回值为0。

句法：

physics_joint_get_value(joint, value)

争论描述
joint您要测试的关节的索引
value您要测试的联合属性的常数

返回值：

Real

例子：

if shipJoint
    {
    var reactionForceX = physics_joint_get_value(shipJoint, phy_joint_reaction_force_x);
    var reactionForceY = physics_joint_get_value(shipJoint, phy_joint_reaction_force_y);
    var reactionForce = point_distance(0, 0, reactionForceX, reactionForceY);
    if reactionForce > 2
        {
        physics_joint_delete(shipJoint);
        shipJoint = -1;
        }
    }

上面的代码检查变量“ shipJoint”是否持有关节索引，如果确实存在，则使用两个常量计算施加到该关节的力。最后，如果总力大于2，则删除关节。
