phy_active

此变量控制实例当前是否处于“活动”状态。设置为 false将阻止实例参与物理世界，并将其设置为 true将再次参加。请注意，这与停用实例不同，因为该实例仍在屏幕上可见，并且仍可以通过代码进行更改，而此功能只是阻止它参与物理模拟

句法：

phy_active;

返回值：

Boolean (or undefined if the instance is not physics enabled)

例子：

if keyboard_check_pressed(ord"P")
    {
    global.Pause = !global.Pause
    with (obj_Parent)
        {
        phy_active = !global.Pause;
        }
    }

上面的代码将检测到字母“ P”的按键，然后从以下位置切换全局变量“ Pause”true到 false然后再回来。然后，使用该变量设置在变量“ obj_Parent”中索引的对象的子级实例中，物理学是否处于活动状态。

返回：物理变量
