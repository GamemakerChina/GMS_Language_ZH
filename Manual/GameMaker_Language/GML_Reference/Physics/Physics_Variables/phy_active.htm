激活

这个变量控制实例当前是否处于 "活动 "状态。将其设置为false将阻止实例参与物理世界，而将其设置为true将使其再次参与。请注意 请注意，这并不等同于停用该实例，因为该实例在屏幕上仍然是可见的，仍然可以通过代码来改变，而 这个函数只是阻止它参与物理模拟。

语法。

phy_active;

返回。

布尔型（如果实例没有启用物理学，则未定义）。

例子。

if keyboard_check_pressed(ord "P")
{
global.Pause = !global.Pause
with (obj_Parent)
{
phy_active = !global.Pause;
}
}

上面的代码将检测一个字母 "P "的按键，然后将全局变量 "Pause "从真到假再切换回来 。这个变量然后被用来设置物理学是否在子代的 变量 "obj_Parent "中索引的对象的实例中设置物理学是否激活。
