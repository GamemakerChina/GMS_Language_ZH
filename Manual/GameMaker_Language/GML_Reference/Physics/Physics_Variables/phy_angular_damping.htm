phy_angular_damping

该变量可用于设置实例的角度阻尼，也可用于获取当前的角度阻尼。阻尼是物理启用实例具有的对角旋转的“抵抗”量，较低的值允许实例在发生碰撞后旋转得更快（例如），而较高的值则使其需要更大的推动力并旋转得更慢

句法：

phy_angular_damping;

返回值：

Real (single precision floating point value, or undefined if the instance is not physics enabled)

例子：

if place_meeting(phy_position_x, phy_position_y, obj_Water)
    {
    phy_angular_damping = 10;
    }
else
    {
    phy_angular_damping = 3;
    }

上面的代码将检查调用实例与“ obj_Water”实例之间是否存在冲突，并相应地更改角度阻尼。

返回：物理变量
