phy_angular_velocity

此变量可用于设置实例的角速度，或者可用于获取当前角速度（以每秒度数为单位），并且所使用的值可以是正值（用于顺时针旋转）或负值（用于逆时针旋转） 。如果在以前是静态的实例上设置此值（即：密度为0），它将成为运动学对象并开始旋转

句法：

phy_angular_velocity;

返回值：

Real (single precision floating point value, or undefined if the instance is not physics enabled)

例子：

if abs(phy_angular_velocity) > 0
    {
    phy_angular_velocity -= sign(phy_angular_velocity) * 0.01;
    }
else
    {
    phy_angular_velocity = 0;
    }

上面的代码将检查实例的角速度，如果该角速度不为0，则它将每步增加（或减去）少量，直到该值为0。

返回：物理变量
