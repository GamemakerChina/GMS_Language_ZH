phy_collision_points

此只读变量返回碰撞中两个对象之间检测到的碰撞点数

注意：此变量仅在启用了物理的实例的碰撞事件中可用。

句法：

phy_collision_points;

返回值：

Real (integer value, or undefined if the instance is not physics enabled)

例子：

for(var i = 0; i < phy_collision_points; i += 1;)
    {
    part_particles_create(global.Sname, phy_collision_x[i], phy_collision_x[1], global.Spark, 5);
    }

上面的代码在两个启用了物理的实例之间的碰撞的所有定义点上创建了粒子。

返回：物理变量
