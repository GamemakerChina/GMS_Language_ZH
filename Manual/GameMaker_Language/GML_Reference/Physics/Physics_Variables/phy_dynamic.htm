phy_dynamic

动态实例是在物理世界中完全模拟的实例，该只读变量将返回 true如果被检查的实例是完全模拟的，或者 false如果不是

句法：

phy_dynamic;

返回值：

Boolean (or undefined if the instance is not physics enabled)

例子：

if other.phy_dynamic
    {
    with (other)
        {
        var dir;
        dir = point_direction(x, y, other.x, other.y);
        physics_apply_impulse(x, y, x + lengthdir_x(100, dir), y + lengthdir_y(100, dir));
        }
    }

上面的代码创建了一个新实例，然后将其定义为物理世界中的“子弹”，然后沿y轴施加了巨大的脉冲。

返回：物理变量
