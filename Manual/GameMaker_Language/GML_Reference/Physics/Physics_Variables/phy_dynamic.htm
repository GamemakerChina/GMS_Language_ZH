脉冲式的

一个动态的实例是一个在物理世界中完全模拟的实例，如果被检查的实例是完全模拟的，这个只读变量将返回真，如果不是，则返回假。

语法。

phy_dynamic。

返回。

布尔型（如果实例没有启用物理学，则未定义）。

例子。

if other.phy_dynamic
{
with (other)
{
var dir;
dir = point_direction(x, y, other.x, other.y);
physics_apply_impulse(x, y, x + lengthdir_x(100, dir), y + lengthdir_y(100, dir));
}
}

上面的代码创建了一个新的实例，然后将其定义为物理世界中的 "子弹"，然后沿y轴给它一个巨大的冲力。
