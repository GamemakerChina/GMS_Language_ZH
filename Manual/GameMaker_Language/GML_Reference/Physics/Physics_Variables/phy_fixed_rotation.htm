phy_fixed_rotation

此变量可用于设置实例是否受旋转力影响（默认为 false）。如果设置为 true，则没有任何外力（来自编码的脉冲或力，或来自碰撞）都不会影响实例的旋转值，因此必须使用 phy_rotation多变的

句法：

phy_fixed_rotation;

返回值：

Boolean (or undefined if the instance is not physics enabled)

例子：

phy_rotation = 0;
phy_fixed_rotation = true;

上面的代码将实例切换为固定旋转，然后设置旋转角度。

返回：物理变量
