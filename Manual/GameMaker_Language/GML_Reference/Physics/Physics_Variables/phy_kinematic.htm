phy_kinematic

此只读变量将返回 true如果实例被归类为运动学对象，或者 false如果不是这样。运动学实例是质量无限（密度为0）但可以移动的实例。因此，要使一个实例运动，首先要创建一个静态实例，然后设置一个或多个与运动有关的实例变量（即：phy_speed_x ，phy_speed_y ， 或者 phy_angular_velocity）

句法：

phy_kinematic;

返回值：

Boolean (or undefined if the instance is not physics enabled)

例子：

if !phy_kinematic
    {
    phy_speed_x = 5;
    }

上面的代码检查实例是否为运动学实例，如果不是，则将水平速度设置为5。

返回：物理变量
