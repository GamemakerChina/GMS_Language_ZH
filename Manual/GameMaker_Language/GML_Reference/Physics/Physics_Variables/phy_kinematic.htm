这个只读变量在实例被归类为运动物体时将返回true，如果不是则返回false。一个运动物体的实例是一个具有无限质量（密度为0）但可以移动的物体。因此，要使一个实例 运动性实例，你首先要创建一个静态实例，然后设置一个或多个与运动有关的实例变量（即。 phy_speed_x, phy_speed_y，或 phy_angular_velocity)

语法。

phy_kinematic;

返回。

布尔型（如果实例没有启用物理学，则未定义）。

例子。

if !phy_kinematic
{
phy_speed_x = 5;
}

上面的代码检查实例是否是运动学的，如果不是，则将水平速度设置为5。
