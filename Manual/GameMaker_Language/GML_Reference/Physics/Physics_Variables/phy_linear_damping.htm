phy_linear_damping

此变量可用于设置实例的线性阻尼，也可用于获取当前的线性阻尼。阻尼是启用了物理的实例具有的“抵抗”向前运动的量，较低的值允许实例更快地移动和加速，而较高的值则需要更强大的推动力

句法：

phy_linear_damping;

返回值：

Real (single precision floating point value, or undefined if the instance is not physics enabled)

例子：

if place_meeting(phy_position_x, phy_position_y, obj_Water)
    {
    phy_linear_damping = 10;
    }
else
    {
    phy_linear_damping = 3;
    }

上面的代码将检查调用实例与“ obj_Water”实例之间是否存在冲突，并相应地更改线性阻尼。

返回：物理变量
