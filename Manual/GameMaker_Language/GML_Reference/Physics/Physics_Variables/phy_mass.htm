湿度

这个只读的变量返回实例的质量，单位是公斤。这个值是根据分配的固定装置的表面积和它们的密度值自动计算的，但它可以用函数physics_mass_properties()来改变。

语法。

phy_mass。

返回。

实数（单精度浮点值，如果实例没有启用物理学，则未定义）。

例子。

如果（phy_mass < other.phy_mass）
{
instance_destroy()。
}

上面的代码来自于实例与另一个实例的碰撞事件，它比较了每个实例的质量，并摧毁了质量较小的那个。
