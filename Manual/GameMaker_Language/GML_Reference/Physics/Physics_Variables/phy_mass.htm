phy_mass

该只读变量返回实例质量（以千克为单位） 。该值是根据分配的灯具的表面积及其密度值自动计算的，但是可以使用以下功能进行更改 physics_mass_properties().

句法：

phy_mass;

返回值：

Real (single precision floating point value, or undefined if the instance is not physics enabled)

例子：

if (phy_mass < other.phy_mass)
    {
    instance_destroy();
    }

上面的代码来自实例与另一个实例的碰撞事件，它比较每个实例的质量并销毁质量较小的实例。

返回：物理变量
