phy_position_y

此变量可用于获取（或设置）实例在游戏室物理世界中的y位置。请注意，通过直接设置此变量来移动实例时，物理世界可能会出现错误，因为它将中断连续模拟。此变量是实例变量的物理等效项y。

句法：

phy_position_y;

返回值：

Real (single precision floating point value, or undefined if the instance is not physics enabled)

例子：

xx = phy_position_x;
yy = phy_position_y;

上面的代码将实例x和y的位置存储在两个变量中。

返回：物理变量
