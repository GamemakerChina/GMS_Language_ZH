phy_position_yprevious

此变量可用于获取（或设置）实例在游戏室物理世界中的先前y位置。这是实例到当前实例在物理世界中的位置

句法：

phy_position_yprevious;

返回值：

Real (single precision floating point value, or undefined if the instance is not physics enabled)

例子：

xx = phy_position_xprevious;
yy = phy_position_yprevious;

上面的代码将物理启用实例的先前x和y位置存储在两个变量中。

返回：物理变量
