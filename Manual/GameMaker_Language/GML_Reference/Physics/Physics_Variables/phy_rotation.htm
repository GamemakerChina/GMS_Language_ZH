phy_rotation

此变量可用于获取（或设置）实例灯具的角度（以度为单位），类似于设置或获取 image_angle。但是请注意，在物理世界中，旋转的计算方式与普通GameMaker Studio 2游戏世界相反，这意味着矢量函数 point_direction()应该修改其返回值（只需将正值设为负值即可解决此问题）。

句法：

phy_rotation;

返回值：

Real (single precision floating point value, or undefined if the instance is not physics enabled)

例子：

if (phy_speed_x > 0) || (phy_speed_y > 0)
    {
    phy_rotation += sqrt(sqr(phy_speed_x) + sqr(phy_speed_y)) /10;
    }

上面的代码检查线性速度，如果两个向量都不为0，则它计算实际速度，并使用该速度来设置旋转速度。

返回：物理变量
