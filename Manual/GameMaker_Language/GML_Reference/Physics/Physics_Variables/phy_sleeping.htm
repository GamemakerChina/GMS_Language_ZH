phy_sleeping

此只读变量返回实例当前是否在“睡眠中”（true ） 或不 （false ），“睡眠”实例是未积极参与任何物理模拟的实例。当一个实例处于静止状态且不与另一个实例发生碰撞时， GameMaker Studio 2

句法：

phy_sleeping;

返回值：

Boolean (or undefined if the instance is not physics enabled)

例子：

if phy_sleeping
    {
    instance_destroy();
    }

上面的代码检查对象是否正在被主动模拟，如果不是，则将其销毁。

返回：物理变量
