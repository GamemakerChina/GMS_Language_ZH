phy_speed_x

这个变量可以用来获取或改变实例的线性速度矢量的x分量，其定义单位为每步的像素数（关于每秒的像素数，见 phy_linear_velocity_x).改变这个 对于一个静态实例（即：一个密度为0的实例）来说，改变这个变量将把它变成一个运动学的实例。

语法。

phy_speed_x;

返回。

实数（单精度浮点值，如果实例没有启用物理学，则未定义）。

例子。

if phy_speed_x != 0
{
phy_speed_x = 0;
}

上述代码将检查线性速度矢量的X分量，如果它不是0，就将其设置为0。
