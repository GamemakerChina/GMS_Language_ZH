phy_speed_x

此变量可用于获取或更改实例线速度矢量的x分量，并以每步像素数为单位进行定义（有关每秒像素数，请参见 phy_linear_velocity_x）。将其更改为静态实例（即：密度为0的实例）会将其转变为运动学实例

句法：

phy_speed_x;

返回值：

Real (single precision floating point value, or undefined if the instance is not physics enabled)

例子：

if phy_speed_x != 0
    {
    phy_speed_x = 0;
    }

上面的代码将检查线性速度矢量的x分量，如果它不是0，则将其设置为0。

返回：物理变量
