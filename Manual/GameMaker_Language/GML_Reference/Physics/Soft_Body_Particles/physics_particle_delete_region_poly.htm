physics_particle_delete_region_poly

使用此功能，您可以从当前房间的物理模拟中删除（删除）所有位于定义的多边形区域范围内的粒子。该函数需要一个（先前创建的）ds_list()包含多边形每个点的x / y位置，列表中的偶数位置是x坐标，而奇数位置是y坐标，即：对于三角形，您的列表将有6个条目，其中条目0 ，2和4为x坐标，而1、3和5为y坐标。然后，该多边形将用于删除所有落在定义区域内的粒子。

请注意，所定义的多边形必须至少包含三个点，最多8个点。

句法：

physics_particle_delete_region_poly(pointList)

争论描述
pointList用于创建多边形的点的ds_list。

返回值：

N/A

例子：

var list = ds_list_create(p_list);
for (var i = 0; i < 5; i ++;)
    {
    ds_list_add(p_list, mx[i]);
    ds_list_add(p_list, my[i]);
    }
physics_particle_delete_region_poly(p_list);
ds_list_destroy(p_list);

上面的代码将删除在多边形区域中找到的所有粒子，这些多边形是由添加到该点的点定义的 DS List。
