physics_particle_draw_ext

此功能将在所有共享相同用户定义类别值且其标志与为蒙版设置的标志重合的粒子的位置绘制一个精灵。掩码值的定义方式与创建粒子时定义标志的方式相同（请参见函数 physics_particle_create()例如），即：使用按位或“ |”为每个标志屏蔽适当的位。

注意：如果用户类别相同，此功能将使用与蒙版的标记重合的标记绘制物理模拟中的所有粒子，无论它们是单个粒子还是组。

绘制的粒子将忽略创建时设置的颜色和Alpha值，而是使用此函数中指定的比例，角度，颜色和Alpha参数进行绘制。请注意，这纯粹是为了在粒子位置绘制精灵，绝不会影响粒子的行为。

句法：

physics_particle_draw_ext(typemask, category, sprite, subimg, xscale, yscale, ang, col, alpha)

争论描述
typemask用于比较粒子标志的掩码。
category用户定义的类别进行绘制（使用0绘制所有类别）。
sprite每个粒子要使用的精灵。
subimg这 image_index绘制的精灵。
xscale沿x轴应用于子画面的比例因子。
yscale沿y轴应用于子画面的比例因子。
ang绘制精灵的角度（以度为单位）。
col与精灵混合的颜色（默认为 c_white）。
alpha绘制时使用的Alpha值（0-1）。

返回值：

N/A

例子：

var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile;
physics_particle_draw_ext(flags, 2, sprBlob, 0, 2, 2, 0, c_white, 0.5);

上面的代码将为用户定义的类别“ 2”中的所有粒子绘制一个精灵，这些粒子具有与给定蒙版相同的标志，沿两个轴均缩放为2，alpha值为0.5。
