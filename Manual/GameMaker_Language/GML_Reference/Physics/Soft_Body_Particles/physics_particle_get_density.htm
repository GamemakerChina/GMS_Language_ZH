物理学_粒子_获取密度

通过这个函数，你可以知道物理模拟中的粒子的电流密度是多少（你可以用 physics_particle_set_density()).

语法。

physics_particle_get_damping()

返回。

真实的

例子。

如果(physics_particle_get_density() < 1)
{
physics_particle_set_density(physics_particle_get_density() + 0.01);
}

上述代码将检查系统中所有粒子的电流密度值，如果它小于1，那么它将添加0.01。
