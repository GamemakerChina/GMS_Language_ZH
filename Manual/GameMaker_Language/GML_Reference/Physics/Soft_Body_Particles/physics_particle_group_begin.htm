physics_particle_group_begin

使用此功能，您可以在房间中创建一组粒子。一组粒子是将创建特定形式所需的粒子作为一组同时创建的一个粒子，允许您创建具有通过所使用的标志设置的各种属性的模拟软体。 “标志”是粒子常数组合的返回值（请参见函数 physics_particle_create()（有关详细信息），而“ groupFlags”是以下组常量的组合（对于没有标志，则为0）：

持续的描述
phy_particle_group_flag_solid固体粒子组可防止其他灯具进入其内部。如果有任何东西穿透，则固体颗粒组会将有问题的固定装置推回其表面，从而使固体颗粒组具有特别强的排斥力。
phy_particle_group_flag_rigid刚性粒子组是形状不变的，即使它们与其他固定装置碰撞也是如此。

这些组标志使用位屏蔽来创建最终输出值，然后检查该最终输出值以设置组的不同基本属性，而不是由粒子标志设置的基本属性。您可以通过按位或“ |”来使用这些常数中的任何一个，两个或全部屏蔽适当的位（如下例所示）。

除标记外，还可以设置房间中的位置以在以下位置创建粒子组（给定的位置将是完成组的质心），初始水平和垂直速度以及角速度。组，允许您创建移动的，旋转的组。

您还可以设置组中所有粒子的初始颜色和Alpha，并使用函数绘制粒子时使用的值 physics_particle_draw()。之后，您可以在组中设置粒子的内聚强度，这是表示它们彼此“粘附”的强度的另一种方式。强度可以是介于0和1之间的值，其中0是无内聚力，而1是最大内聚力。

最后，您必须为您的粒子组指定类别。该值是一个任意整数值，可为每个粒子组提供要具有相似属性的属性，并且以后可用于为该特定类别以及许多其他事物设置标志。

注意：可以对类别使用任何整数值，但0除外，该值是模拟保留的，用于选择其他功能中的所有类别（例如用于绘图）。

需要注意的是调用这个函数并不在物理室创建组是很重要的。此功能只是开始该过程并设置组的基本粒子属性。要实际创建粒子组，您需要给它一个表格（请参见下面的示例），然后调用该函数 physics_particle_group_end()，此时将创建它。

注意：您设置的粒子类型标记也会影响粒子组的凝聚力，例如，如果您将粒子标记为水类型（例如），则当该组与另一个灯具碰撞时，它会破裂，而如果将粒子标记为弹簧类型，则它将在碰撞时保持其内聚力。

句法：

physics_particle_group_begin(flags, groupflags, x, y, ang, xv, yv, ang_velocity, col, alpha, strength, category)

争论描述
flags要在粒子上设置的标志。
groupflags要在粒子上设置的组标志（请参见下面的描述）。
x在其上创建粒子组的x位置。
y在其上创建粒子组的y位置。
ang创建时旋转组的角度（以度为单位）。
xv初始水平速度。
yv初始垂直速度。
ang_velocity初始角速度。
col用于粒子组的基本颜色。
alpha用于粒子组的基本Alpha。
strength设置组中粒子之间的内聚强度。
category组所属的用户定义的类别。

返回值：

N/A

例子：

var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile;
var groupflags = phy_particle_group_flag_solid | phy_particle_group_flag_rigid;
physics_particle_group_begin(flags, groupflags, mouse_x, mouse_y, 0, 0, 0, 0, c_white, 1, 1, 2);
physics_particle_group_circle(100);
mLastGroup = physics_particle_group_end();

上面的代码将粒子类型和粒子组属性的标记存储在变量中，然后使用这些标记在鼠标位置创建半径为100px的圆形粒子组。
