物理学_粒子组_盒子

这个函数将设置正在创建的粒子组的形状。你必须首先使用函数开始定义粒子组 physics_particle_group_begin()开始定义粒子组，然后你将 使用这个函数来设置粒子组的近似半宽和半高，单位是像素--近似是因为确切的宽度和高度将取决于基础粒子的大小，正如函数所定义的那样。 physics_particle_set_radius() 函数所定义的，因为物理模拟试图将尽可能多的粒子 "装入 "定义的形状中。最后，你需要调用 physics_particle_group_end()来创建房间里的一组 中的粒子组。

语法。

physics_particle_group_box(halfWidth, halfHeight)

争论 描述
半宽 盒子的一半宽度。
半高 盒子的一半高度。

返回。

不适用

例子。

var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile;
var groupflags = phy_particle_group_flag_solid;
physics_particle_group_begin(flags, groupflags, mouse_x, mouse_y, 0, 0, 0, c_white, 1, 1, 2);
physics_particle_group_box(100, 100);
mLastGroup = physics_particle_group_end() 。

上面的代码在变量中存储了粒子类型的标志和粒子组的属性，然后用这些来创建一个边长为200px的矩形粒子组，在鼠标位置。
