物理学_粒子组_结束

这个函数是用来结束一个粒子组形状的定义。调用这个函数将在给定的形状参数范围内创建粒子，并返回一个ID值，这个ID值可以被存储起来，并在以后的函数中用于与粒子组的交互作用 与粒子组的交互。

语法。

physics_particle_group_end()

返回。

真实的

例子。

var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile;
var groupflags = phy_particle_group_flag_solid;
physics_particle_group_begin(flags, groupflags, mouse_x, mouse_y, 0, 0, 0, c_white, 1, 1, 2);
physics_particle_group_circle(100);
mLastGroup = physics_particle_group_end();

上面的代码在变量中存储了粒子类型和粒子组属性的标志，然后使用这些标志在鼠标位置创建一个半径为100px的圆形粒子组。已经创建的组的ID被存储在 在变量 "mLastGroup "中。
