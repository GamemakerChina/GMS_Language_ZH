物理学_粒子组_多边形

这个函数将设置正在创建的粒子组的形状。你必须首先使用函数开始定义粒子组 physics_particle_group_begin()开始，然后你将 使用这个函数来告诉GameMaker Studio 2，你将添加用于定义多边形形状的近似点--近似点是因为确切的形状将取决于基础粒子的大小，如由 函数定义的基础粒子的大小。 physics_particle_set_radius()函数所定义的，因为物理模拟试图将尽可能多的粒子 "装入 "定义的形状中。

然后你将定义多边形的点，调用函数 physics_particle_group_add_point()的函数，为每个点设置房间里的X/Y位置（最小3个，最大8个，而且它们可以被定义为凹形，但形状内的点将被忽略）。 可以定义为凹面，但形状内的点将被忽略）。)最后，你需要调用 physics_particle_group_end()来创建房间里的粒子组。

语法。

physics_particle_group_polygon()

返回。

不适用

例子。

var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile;
var groupflags = phy_particle_group_flag_solid;
physics_particle_group_begin(flags, groupflags, mouse_x, mouse_y, 0, 0, 0, c_white, 1, 1, 2);
physics_particle_group_polygon() 。
physics_particle_group_add_point（200, 200）;
physics_particle_group_add_point（300, 300）;
physics_particle_group_add_point（100, 300）;
mLastGroup = physics_particle_group_end();

上面的代码在变量中存储了粒子类型的标志和粒子组的属性，然后用这些来在鼠标位置创建一个三边的多边形粒子组。
