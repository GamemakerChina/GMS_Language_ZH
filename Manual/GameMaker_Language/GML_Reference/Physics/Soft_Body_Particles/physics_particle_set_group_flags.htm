physics_particle_set_group_flags

使用此功能，您可以更改一组粒子的组标记。这 group值是使用函数创建粒子组时返回的值 physics_particle_group_end()，而标志是以下常量的组合的返回值：

持续的描述
phy_particle_group_flag_solid固体粒子组可防止其他灯具进入其内部。如果有任何东西穿透，则固体颗粒组会将有问题的固定装置推回其表面，从而使固体颗粒组具有特别强的排斥力。
phy_particle_group_flag_rigid刚性粒子组是形状不变的，即使它们与其他固定装置碰撞也是如此。

这些组标志使用位屏蔽来创建最终输出值，然后检查该最终输出值以设置组的不同基本属性，而不是由粒子标志设置的基本属性。您可以通过按位或“ |”来使用这些常数中的任何一个，两个或全部屏蔽适当的位（如下例所示）。

句法：

physics_particle_set_group_flags(group, flags)

争论描述
group要设置的粒子组。
flags要在粒子上设置的标志。

返回值：

N/A

例子：

var flags = phy_particle_group_flag_solid | phy_particle_group_flag_rigid;
physics_particle_set_group_flags(group, flags);

上面的代码将创建一个变量来存储标志值，然后使用它来设置在变量“ group”索引的组中创建的所有粒子的标志。
