physics_particle_set_max_count

此功能将设置物理模拟中允许的粒子总数。如果将此值设置为0，则创建的粒子没有上限，而当粒子总数等于该值时，将其设置为任何其他值将停止创建任何其他粒子。请注意，您可以在模拟中创建的最小粒子数为128，因此将此值设置为任何较低值均无效（除非设置为0）。

句法：

physics_particle_set_max_count(count)

争论描述
count允许的最大粒子数。

返回值：

N/A

例子：

physics_particle_set_max_count(500);

上面的代码将物理模拟的粒子上限设置为500。
