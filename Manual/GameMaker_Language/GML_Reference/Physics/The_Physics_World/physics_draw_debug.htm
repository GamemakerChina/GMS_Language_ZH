physics_draw_debug

GameMaker Studio 2创建物理系统时，直观地实例化实例通常很重要。这可以通过使用 physics_draw_debug()，它绘制了与运行代码的实例相关联的物理属性的示意图。这是它的外观图像：

应该注意的是，要使其生效，必须在实例的绘制事件中进行，并且将使用当前定义的绘制颜色对其进行绘制。

句法：

physics_draw_debug()

返回值：

N/A

例子：

draw_set_colour(c_red);
physics_draw_debug();

上面的代码将使用红色绘制与该实例关联的物理属性的表示。

返回：物理世界
