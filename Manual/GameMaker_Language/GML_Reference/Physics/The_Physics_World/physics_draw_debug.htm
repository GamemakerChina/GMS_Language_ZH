在GameMaker Studio 2中创建一个物理系统时，往往需要对一个实例实际发生的情况有一个可视化的表示。这可以通过使用physics_draw_debug()来实现，它可以画出与运行代码的实例相关的物理属性的图示轮廓 来实现的，它可以画出与运行代码的实例相关的物理属性的示意图。下面是它的外观图片。

值得注意的是，要使其发挥作用，必须在实例的绘制事件中进行，并且将使用当前定义的绘制颜色绘制。 实例中，并且它将使用当前定义的绘制颜色进行绘制。

语法。

physics_draw_debug()

返回。

不适用

例子。

draw_set_colour(c_red);
physics_draw_debug()。
