physics_pause_enable

通常，要使物理模拟正常工作，它必须是连续的并且不能停止和启动，或者实例突然从房间中的一个地方移到另一个地方。但是，有些时候您需要暂停仿真，例如（例如）设备操作系统已暂停，因此您可以使用此功能。如果将标志设置为，它将暂停模拟 true除非将该标志设置为，否则将不会进行进一步的物理计算。false再次。

注意：如果您希望停用房间中的所有实例，则此功能特别有用，即使停用的实体在物理世界中仍将继续计算和模拟。

句法：

physics_pause_enable(flag)

争论描述
flag可以设置为 true暂停模拟，或 false重新开始。

返回值：

N/A

例子：

if os_is_paused()
   {
   physics_pause_enable(true);
   instance_deactivate_all(true);
   instance_create_layer(x, y, "Controllers", obj_PauseMenu);
   }

上面的代码检查操作系统是否已暂停，如果已暂停，它将暂停物理世界，停用除自身之外的所有内容，然后创建一个暂停菜单实例。

返回：物理世界
