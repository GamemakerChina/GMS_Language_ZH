物理学_暂停_启用

通常情况下，为了使物理模拟发挥作用，它必须是连续的，不能停止和启动，或者让实例突然从房间的一个地方移到另一个地方。然而，在某些情况下，你需要暂停模拟，因为（例如）设备操作系统已经暂停，所以你会使用 例如）设备os暂停了，所以你会使用这个函数。如果标志被设置为"真"，它就会暂停模拟，在标志再次被设置为"假"之前，不会有进一步的物理学计算。

注意：如果你想停用一个房间里的所有实例，这是特别有用的，因为即使停用了一个物理体，仍然会继续在物理世界中被计算和模拟。

语法。

physics_pause_enable(flag)

争论 描述
旗帜 这可以被设置为"true "以暂停模拟，或 "false"以再次启动。

返回。

不适用

例子。

如果os_is_paused()
{
physics_pause_enable(true);
instance_deactivate_all(true);
instance_create_layer(x, y, "Controllers", obj_PauseMenu);
}

上面的代码检查操作系统是否已经暂停，如果已经暂停，那么它就暂停物理世界，停用除自己以外的所有东西，然后创建一个暂停菜单实例。
