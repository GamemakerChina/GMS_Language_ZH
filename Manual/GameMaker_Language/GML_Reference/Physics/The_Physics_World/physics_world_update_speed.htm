物理学_世界_更新_速度

GameMaker Studio 2根据房间的速度来更新东西，所以如果你把它设置为，比如说，30，GameMaker Studio 2将在一秒钟内运行30步。然而，对于物理功能来说，这可能是不够的，你可能希望事情 以稍快的速度更新，以提高稳定性或精确度。为此，我们使用了physics_world_update_speed()函数，它为物理系统设置了独立于房间速度的更新速度。这意味着 你可以有一个30的房间速度，但将物理系统设置为60，与步骤更新的速度相比，物理系统更新和执行计算的速度有效地增加了一倍。

注意：你目前不能将其设置为低于房间速度，但未来的更新可能会改变这一点。

语法。

physics_world_update_speed(speed)

争论 描述
速度 物理学系统每秒更新的次数

返回。

不适用

例子。

physics_world_update_speed(room_speed * 2);

上面的代码将物理系统的更新速度设定为房间速度的两倍。
