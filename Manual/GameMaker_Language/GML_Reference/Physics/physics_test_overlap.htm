physics_test_overlap

当旋转并将其放置在房间中的给定位置时，此功能可用于检查并查看物理物体（即实例的固定装置）是否重叠或将重叠。 “ angle”参数是调用实例在要检查的位置具有（或将具有）的旋转角度，“ obj”参数可以是单个实例ID，对象索引或关键字all或other 。

句法：

physics_test_overlap(xpos, ypos, angle, obj);

争论描述
xpos要检查的房间中的x位置
ypos房间中的y位置
angle检查（调用实例的）角度
obj要检查的对象

返回值：

Boolean

例子：

if physics_test_overlap(x+20, y-35, 0, obj_Bomb)
    {
    alarm[0] = room_speed;
    ignited = true;
    }

上面的代码将检查物理夹具重叠的位置，如果存在，则会设置一个变量和一个警报。
