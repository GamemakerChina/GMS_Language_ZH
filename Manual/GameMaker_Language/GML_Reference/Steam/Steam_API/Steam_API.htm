通用Steam API

以下功能集全部用于检查Steam客户端或服务器API某些方面的可用性，因此返回 true或者 false取决于服务器繁忙或用户登录等因素。这意味着应在Steam API函数之前使用这些功能，以确保客户端/服务器设置正确并与您的游戏进行通信：

  • steam初始化
  • steam _stats_ready
  • steam _get_app_id
  • steam _get_user_account_id
  • steam _get_user_ steam _id
  • steam _get_persona_name
  • steam _get_user_persona_name
  • steam _is_user_logged_on
  • steam _current_game_language
  • steam _available_languages
