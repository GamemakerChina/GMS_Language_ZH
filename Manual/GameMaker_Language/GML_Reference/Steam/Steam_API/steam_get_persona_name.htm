steam _get_persona_name

您可以使用此功能返回当前登录到Steam客户端的用户的用户名。这是可见的屏幕名称，而不是唯一的用户ID （可以使用函数找到 steam_get_user_steam_id()）。

句法：

steam_get_persona_name();

返回值：

String

例子：

if steam_initialised()
    {
    global.p_name = steam_get_persona_name();
    }

Steam客户端API，则以上代码将为当前用户的屏幕名称设置一个全局变量。

返回：通用Steam API
