steam _get_user_persona_name

此功能可用于检索任何特定用户ID的用户名（屏幕名称）。您提供用户ID，然后该函数将返回唯一的异步ID值，该值可用于在Steam Async事件中正确检查详细信息。该事件将具有 async_loadDS映射，其中将填充以下键/值对：

  • “ id ”-触发事件的函数返回的异步ID
  • “ event_type ”-字符串“ user_persona_name ”
  • “ steam id” “-用于获取用户名Steam
  • “ persona_name ”-给定Steam ID的用户名

句法：

steam_get_user_persona_name(steamID);

争论描述
steam ID用户的唯一Steam ID。

返回值：

Real

例子：

request = steam_get_user_persona_name(global.UGC_UserID);

上面的代码将请求存储在全局变量“ UGC_UserID”中的用户ID的用户名，并将返回值存储在变量中以在异步事件中进行解析。

返回：通用Steam API
