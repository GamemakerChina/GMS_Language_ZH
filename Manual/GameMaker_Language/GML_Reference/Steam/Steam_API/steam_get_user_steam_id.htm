steam _get_user_ steam _id

您可以使用此功能返回当前登录到Steam客户端Steam用户ID。这是一个唯一的64位值，可用于识别单个用户，而不是其屏幕上的用户名（可以使用函数找到 steam_get_persona_name()）。由于这是int64，因此在后续功能中使用 int64()功能。

句法：

steam_get_user_steam_id();

返回值：

Integer (int64)

例子：

if steam_initialised()
    {
    global.u_id = steam_get_user_steam_id();
    }

如果Steam客户端API，则以上代码将为当前用户唯一的Steam
