steam初始化

使用Steam API时，可以在调用游戏中的Steam Steam客户端API是否已正确初始化。

句法：

steam_initialised();

返回值：

Boolean

例子：

global.steam_api = false;
if steam_initialised()
    {
    if steam_stats_ready() && steam_is_overlay_enabled()
        {
        global.steam_api = true;
        }
    }

上面的代码将全局变量设置为 true如果Steam客户端API以及Steam统计信息和叠加功能已正确初始化，或者它将变量设置为 false否则。

返回：通用Steam API
