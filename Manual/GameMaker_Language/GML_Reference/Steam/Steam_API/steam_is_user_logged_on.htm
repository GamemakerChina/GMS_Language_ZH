蒸汽_用户是否已登录？

如果Steam客户端目前与Steam服务器有一个实时连接，这个函数将返回true。如果返回false，说明没有活动连接，原因是本地机器的网络问题，或者Steam 服务器关闭或繁忙。

语法。

steam_is_user_logged_on()。

返回。

字符串

例子。

ifsteam_is_user_logged_on()
{
global.user_id =steam_get_user_account_id();
}

上面的代码将检查用户是否登录了Steam服务器，并将用户的ID存储在一个全局变量中。
