steam _is_user_logged_on

该函数将返回 true如果Steam客户端当前与Steam服务器之间存在实时连接。如果返回 false，这意味着由于本地计算机上的网络问题或Steam服务器出现故障或繁忙而导致没有活动的连接。

句法：

steam_is_user_logged_on();

返回值：

String

例子：

if steam_is_user_logged_on()
    {
    global.user_id = steam_get_user_account_id();
    }

上面的代码将检查用户是否已登录Steam服务器以及是否将用户ID存储在全局变量中。

返回：通用Steam API
