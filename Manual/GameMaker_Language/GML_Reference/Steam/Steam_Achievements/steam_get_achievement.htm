蒸汽_获取成就

通过这个函数，你可以检查SteamAPI，看看某个特定的成就是否已经被授予。该成就应该已经在你的游戏的Steamworks控制面板账户页面上定义过，并且传递给函数的字符串 的字符串应该与控制面板上用作API名称的字符串一致。

语法。

steam_get_achievement(ach_name)。

争论 描述
ach_name 要获取的成就的名称（字符串）。

返回。

布尔型

例子。

如果 hp <= 0
{
global.Deaths += 1;
if global.Deaths == 10
{
if !steam_get_achievement("ach_Player_Dies_Ten_Times") steam_set_achievement("ach_Player_Dies_Ten_Times);
}
}

如果全局变量 "死亡人数 "等于10，并且成就还没有被授予，那么上述代码将奖励玩家一个成就。
