steam _get_achievement

使用此功能，您可以检查Steam API以查看是否已授予特定成就。该成就应该已经在您的Steam作品控制面板帐户页面上预先定义，并且传递给该函数的字符串应与控制面板上用作API名称的字符串匹配。

句法：

steam_get_achievement(ach_name);

争论描述
ach_name要获取的成就的名称（字符串）。

返回值：

Boolean

例子：

if hp <= 0
    {
    global.Deaths += 1;
    if global.Deaths == 10
        {
        if !steam_get_achievement("ach_Player_Dies_Ten_Times") steam_set_achievement("ach_Player_Dies_Ten_Times");
        }
    }

如果全局变量“死亡”等于10并且尚未授予成就，则上述代码将奖励玩家成就。

返回：成就与统计
