蒸汽_复位_所有统计信息

通过这个功能，你可以将当前用户的所有统计数据重置为它们的默认值（在你的游戏的Steamworks控制面板中定义）。建议你在开发游戏时，只把这个功能作为一个调试工具。 你的游戏。

语法。

steam_reset_all_stats()。

返回。

不适用

例子。

ini_open("Save.ini");
if global.Version != ini_read_real("Data", "Version", 0)
{
ini_write_real("Data", "Version", global.Version);
steam_reset_all_stats();
}
ini_close()。

上述代码检查ini文件中的存储值和全局变量的存储值，如果它们不同，就会重置游戏的统计数据。

返回。成就和统计数据
