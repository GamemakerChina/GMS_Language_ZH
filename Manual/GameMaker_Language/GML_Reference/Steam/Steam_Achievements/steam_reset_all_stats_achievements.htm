蒸汽_复位_所有统计数字_成就

通过这个功能，您可以将当前用户的所有统计数据和 成就重置为默认值（在您游戏的Steamworks控制面板中定义）。建议您在开发游戏时，只将此功能 作为开发游戏时的一个调试工具。

语法。

steam_reset_all_stats_achievements()。

返回。

不适用

例子。

ini_open("Save.ini");
if global.Version != ini_read_real("Data", "Version", 0)
{
ini_write_real("Data", "Version", global.Version);
steam_reset_all_stats_achievements()。
}
ini_close()。

上述代码将ini文件中的存储值与全局变量的存储值进行核对，如果两者不同，就会重置游戏的统计数据和成就。

返回。成就和统计数据
