steam _reset_all_stats_achievements

使用此功能，您可以将当前用户的所有统计数据和成就重置为其默认值（在Steam Works控制面板中定义）。建议您在开发游戏时仅将此功能用作调试工具。

句法：

steam_reset_all_stats_achievements();

返回值：

N/A

例子：

ini_open("Save.ini");
if global.Version != ini_read_real("Data", "Version", 0)
    {
    ini_write_real("Data", "Version", global.Version);
    steam_reset_all_stats_achievements();
    }
ini_close();

上面的代码将ini文件中的存储值与全局变量的存储值进行比较，如果它们不同，则会重置游戏的统计信息和成就。

返回：成就与统计
