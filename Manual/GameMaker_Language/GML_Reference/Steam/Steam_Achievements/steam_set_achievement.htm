蒸汽_设置_成就

通过这个函数，你可以告诉SteamAPI为玩家授予（"设置"）一个成就。这些成就应该在你的游戏的Steamworks控制面板账户页面中定义，并且传递给函数的字符串 的字符串应该与控制面板上用作API名称的字符串一致。Steam游戏覆盖将向用户显示一个通知面板，告知他们所获得的成就，除非该成就已经被授予，在这种情况下不会发生任何事情。 在这种情况下，什么都不会发生。

语法。

steam_set_achievement(ach_name)。

争论 描述
ach_name 要设置的成就的名称（字符串）。

返回。

不适用

例子。

如果 hp <= 0
{
global.Deaths += 1;
if global.Deaths == 10
{
if !steam_get_achievement("ach_Player_Dies_Ten_Times") steam_set_achievement("ach_Player_Dies_Ten_Times);
}
}

如果全局变量 "死亡人数 "等于10，并且成就还没有被授予，那么上述代码将奖励玩家一个成就。

返回。成就和统计数据
