steam成就

使用此功能，您可以告诉Steam API为玩家奖励（“设置”）一项成就。这些成就应该已经在Steam Works控制面板帐户页面上定义，并且传递给该函数的字符串应与控制面板上用作API名称的字符串匹配。 Steam Game Overlay将向用户显示一个通知面板，通知他们他们已经收到的成就，除非已经授予了成就，在这种情况下将不会发生任何事情。

句法：

steam_set_achievement(ach_name);

争论描述
ach_name要设置的成就名称（字符串）。

返回值：

N/A

例子：

if hp <= 0
    {
    global.Deaths += 1;
    if global.Deaths == 10
        {
        if !steam_get_achievement("ach_Player_Dies_Ten_Times") steam_set_achievement("ach_Player_Dies_Ten_Times");
        }
    }

如果全局变量“死亡”等于10并且尚未授予成就，则上述代码将奖励玩家成就。

返回：成就与统计
