steam统计浮点数

使用此功能，您可以将特定的统计信息设置为新的浮点值。该统计信息应事先在您的Steam作品控制面板帐户页面上定义，并且传递给该函数的字符串应与控制面板上用作API名称的字符串匹配。何时使用它的示例是跟踪玩家的行进距离或完成游戏的百分比。

句法：

steam_set_stat_float(stat_name, value);

争论描述
stat_name要设置的统计信息的名称（字符串）。
value将统计信息设置为的值（32位浮点数）。

返回值：

N/A

例子：

var dist_pc = (dist / dist_max) * 100;
steam_set_stat_float("Travelled", dist_pc);

上面的代码根据行进距离变量“ dist”和最大行进距离“ dist_max”计算百分比，然后将统计信息“ Travelled”设置为新值。

返回：成就与统计
