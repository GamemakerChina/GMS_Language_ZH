steam _set_stat_int

使用此功能，您可以将特定的统计信息设置为新的带符号的integer值。该统计信息应事先在您的Steam作品控制面板帐户页面上定义，并且传递给该函数的字符串应与控制面板上用作API名称的字符串匹配。何时使用它的示例是跟踪玩家死亡的次数或跟踪成就的进度。

句法：

steam_set_stat_int(stat_name, value);

争论描述
stat_name要设置的统计信息的名称（字符串）。
value将统计信息设置为的值（32位整数）。

返回值：

N/A

例子：

xp += 100;
steam_set_stat_int("Total_XP", steam_get_stat_int("Total_XP") + 100);
if steam_get_stat_int("Total_XP") > 1000
    {
    if !steam_get_achievement("Ach_1000XP") steam_set_achievement("Ach_1000XP");
    }

上面的代码设置了一个统计信息，然后检查其最终值，以决定是否授予成就。

返回：成就与统计
