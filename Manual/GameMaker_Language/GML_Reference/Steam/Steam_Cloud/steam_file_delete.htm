steam _file_delete

Steam云中删除给定的文件。如果该函数由于任何原因而失败，则将返回值0，如果成功则将返回大于0的值。

句法：

steam_file_delete(filename);

争论描述
filename文件名删除。

返回值：

Real

例子：

if steam_file_exists("Save.txt")
    {
    steam_file_delete("Save.txt");
    }

上面的代码将检查文件是否存在，如果存在，它将从Steam云中删除该文件。
