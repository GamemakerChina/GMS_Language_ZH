删掉文件

这个函数将从Steam云中删除给定的文件。如果由于某种原因失败，该函数将返回一个0的值，如果成功，则返回一个大于0的值。

语法。

steam_file_delete（文件名）。

争论 描述
文件名 文件删除的名称。

返回。

真实的

例子。

ifsteam_file_exists("Save.txt")
{
steam_file_delete("Save.txt")。
}

上面的代码会检查文件是否存在，如果存在，就会从Steam云中删除该文件。
