蒸汽文件的存在

通过这个函数，你可以检查Steam云中的文件是否存在，如果存在，返回值为true，否则为false。

语法。

steam_file_exists(filename)。

争论 描述
文件名 要检查的文件的名称。

返回。

布尔型

例子。

ifsteam_file_exists("Save.txt")
{
save_str =steam_file_read("Save.txt");
}

上面的代码会检查Steam云端是否存在一个文件，如果存在，就会打开它并将其内容读入变量 "save_str"。

返回。蒸汽云
