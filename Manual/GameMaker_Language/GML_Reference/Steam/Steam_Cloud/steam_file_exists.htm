steam _file_exists

使用此功能，您可以检查是否Steam Cloud的文件，返回值是 true如果存在，或者 false否则。

句法：

steam_file_exists(filename);

争论描述
filename要检查的文件名。

返回值：

Boolean

例子：

if steam_file_exists("Save.txt")
    {
    save_str = steam_file_read("Save.txt");
    }

上面的代码检查一下Steam云上是否存在文件，如果存在，则将其打开并将其内容读入变量“ save_str”。
