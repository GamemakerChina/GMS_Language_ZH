steam _file_persisted

使用此功能，您可以检查给定的文件以查看其是否已与Steam Cloud同步。返回值为 true表示是 false表示不是。

句法：

steam_file_persisted(filename);

争论描述
filename要检查的文件名。

返回值：

Boolean

例子：

if !steam_file_persisted("Save.txt")
    {
    steam_file_share("Save.txt");
    }

上面的代码将检查文件是否已存储到Steam Cloud中，如果没有，则将对其进行同步。
