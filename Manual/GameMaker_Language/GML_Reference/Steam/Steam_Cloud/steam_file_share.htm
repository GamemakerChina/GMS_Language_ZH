steam _file_share

使用此功能，您可以强制游戏将给定文件与Steam Cloud同步。由于游戏会在玩家会话结束时自动进行同步，因此通常不需要这样做， Steam也不建议这样做，但是这对于确保立即同步敏感信息很有用。如果该函数由于任何原因而失败，则将返回值0，如果成功则将返回大于0的值。

句法：

steam_file_share(filename);

争论描述
filename文件同步的名称。

返回值：

Real

例子：

if !steam_file_persisted("Save.txt")
    {
    steam_file_share("Save.txt");
    }

上面的代码将检查文件是否已存储到Steam Cloud中，如果没有，则将对其进行同步。

背面： Steam云
