蒸汽文件共享（steam_file_share

通过这个功能，你可以强制你的游戏将指定的文件与Steam云同步。这通常不是必要的，因为游戏会在玩家的会话结束后自动同步，而且Steam也不推荐这样做。 但对于确保敏感信息被立即同步来说，这是很有用的。如果因为某种原因失败，该函数将返回一个0的值，如果成功则返回一个大于0的值。

语法。

steam_file_share(文件名)。

争论 描述
文件名 同步的文件的名称。

返回。

真实的

例子。

if !steam_file_persisted("Save.txt")
{
steam_file_share("Save.txt")。
}

上述代码将检查一个文件是否已经存储到Steam云端，如果没有，它就会同步化。
