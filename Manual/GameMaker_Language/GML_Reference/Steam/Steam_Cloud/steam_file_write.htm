steam _file_write

您可以使用此功能将数据写入文件，然后在用户退出游戏时Steam如果文件不存在，则此函数将为您创建文件；如果文件已存在，则它将使用新的数据字符串覆盖文件中已存储的所有数据。如果该函数由于任何原因而失败，则将返回值0，如果成功则将返回大于0的值。

句法：

steam_file_write(filename, data, size);

争论描述
filename要写入的文件名。
data要写入的数据（字符串）。
size要写入的数据大小。

返回值：

Real

例子：

var fname="SaveData.txt";
var data = string(global.Level) + "|" + string(global.Points) + "|" + string(global.HP);
var len = string_length(data);
steam_file_write_file(fname, data, len);

上面的代码将准备一些局部变量，然后使用它们写入（或创建）文件，然后将其与Steam Cloud同步。
