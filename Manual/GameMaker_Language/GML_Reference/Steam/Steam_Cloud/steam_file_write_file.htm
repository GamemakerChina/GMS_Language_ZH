steam _file_write_file

使用此功能，您可以将本地保存的文件的内容复制到与Steam Cloud同步的文件。使用此功能之前，本地文件必须存在，如果由于某种原因失败，它将返回值0，如果成功则返回大于0的值。

句法：

steam_file_write_file(steam_filename, local_filename);

争论描述
steam _filename要复制的Steam Cloud文件。
local_filename用于复制的本地文件。

返回值：

real

例子：

steam_file_write_file("rm_koala.png", "Koala2.png");

上面的代码会将文件“ Koala2.png”的内容复制到Steam Cloud文件“ rm_koala.png”。

背面： Steam云
