蒸汽_获取配额_自由

通过这个函数，你可以了解到SteamCloud配额的用户还剩下多少可用空间。返回的值是以字节为单位。

语法。

steam_get_quota_free()。

返回。

真实的

例子。

ifsteam_is_cloud_enabled_for_app()
{
quota =steam_get_quota_free();
}

上面的代码检查游戏是否启用了steam云，如果是的话，它就会获得可用存储空间的大小并将其存储在一个变量中。

返回。蒸汽云
