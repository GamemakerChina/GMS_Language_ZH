steam _get_quota_free

使用此功能，您可以找出Steam云配额用户剩余的可用空间。返回的值以字节为单位。

句法：

steam_get_quota_free();

返回值：

Real

例子：

if steam_is_cloud_enabled_for_app()
    {
    quota = steam_get_quota_free();
    }

上面的代码检查是否steam云，如果启用了蒸汽云，它将获取可用存储空间的大小并将其存储在变量中。
