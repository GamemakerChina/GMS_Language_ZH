帐户是否启用了云计算？

通过这个函数，你可以检查确定用户在他们的Steam客户端设置中是否启用了Steam云服务。如果是，它将返回true，否则返回false。请注意，这并不自动意味着你 可以将数据存储到云端，因为它还必须在你的游戏中被启用（你可以使用函数 steam_is_cloud_enabled_for_app()).如果你的游戏启用了Steam云 但用户在本地关闭了它，你仍然可以使用云功能来存储和检索所有文件的本地副本的数据，只是在游戏结束时不会将它们上传到云端，也不会在游戏开始时进行同步。

语法。

steam_is_cloud_enabled_for_account（）。

返回。

布尔型

例子。

ifsteam_is_cloud_enabled_for_account
{
steam_file_share("Save.txt")。
}

上面的代码会检查用户是否启用了Steam云，如果返回为真，那么它就会同步给定的文件。
