steam _is_cloud_enabled_for_account

使用此功能，您可以检查以确保用户在其Steam Client设置中Steam它将返回 true如果是和 false否则。请注意，这并不意味着您可以自动将数据存储到云端，因为还必须为您的游戏启用该功能（您可以使用以下功能检查该功能：steam_is_cloud_enabled_for_app() ）。如果Steam云，但用户已在本地将其关闭，则您仍可以使用Cloud功能从所有文件的本地副本中存储和检索数据，而不会将其上传到桌面上的云中。游戏结束，也不会在游戏开始时进行同步。

句法：

steam_is_cloud_enabled_for_account();

返回值：

Boolean

例子：

if steam_is_cloud_enabled_for_account
    {
    steam_file_share("Save.txt");
    }

上面的代码检查用户是否已Steam Cloud以及是否返回 true，然后它将同步给定的文件。

背面： Steam云
