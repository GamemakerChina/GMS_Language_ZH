steam _is_cloud_enabled_for_app

使用此功能，您可以检查以确保为您的游戏启用Steam它将返回 true如果是和 false否则。请注意，这并不意味着您可以使用云功能，因为用户可以从其Steam客户端关闭云同步。您可以使用功能检查 steam_is_cloud_enabled_for_account()，但是，即使用户禁用了该功能（并为游戏启用了该功能），这些功能仍然可以从所有文件的本地副本中存储和检索数据，但不会将它们上传到游戏端的云中，也不会在游戏开始时进行同步。

句法：

steam_is_cloud_enabled_for_app();

返回值：

Boolean

例子：

if steam_is_cloud_enabled_for_app()
    {
    quota = steam_get_quota_total();
    }

上面的代码检查是否steam云，如果启用了蒸汽云，它将获取存储配额的大小并将其存储在变量中。
