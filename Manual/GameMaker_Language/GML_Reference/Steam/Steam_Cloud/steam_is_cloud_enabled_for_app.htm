蒸汽_是云启用的应用程序

通过这个函数，你可以检查是否为你的游戏启用了Steam云服务。如果是，它将返回true，否则返回false。请注意，这并不自动意味着你可以使用云计算功能 因为用户可以在他们的Steam客户端上关闭云端同步功能。你可以使用函数来检查这一点 steam_is_cloud_enabled_for_account()来检查，但是，即使它对用户来说是禁用的（对游戏来说是启用的）。 用户禁用（而游戏启用），这些功能仍然可以从所有文件的本地副本中存储和检索数据，只是不会在游戏结束时将它们上传到云端，也不会在游戏开始时进行同步。

语法。

steam_is_cloud_enabled_for_app()。

返回。

布尔型

例子。

ifsteam_is_cloud_enabled_for_app()
{
quota =steam_get_quota_total();
}

上面的代码检查游戏是否启用了steam云，如果是，它就会得到存储配额的大小，并将其存储在一个变量中。

返回。蒸汽云
