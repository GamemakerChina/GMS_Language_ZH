蒸汽用户安装_dlc

如果你的游戏有为其创建的DLC，你可以使用这个函数在访问与之相关的任何文件之前检查并查看用户是否已经安装了它。如果玩家安装了该内容，该函数返回true，如果用户没有安装，则返回false，但请注意，用户必须同时拥有该DLC，所以你应该在使用该函数之前使用 steam_user_owns_dlc()的附加函数来检查它是否被拥有，然后再使用它。请注意，Steam ID可能是一个很大的数字，所以你可能需要将你的ID值作为一个 int64()然后再提供给这个函数。

语法。

steam_user_installed_dlc(dlc_id)。

争论 描述
dlc_id 要检查的DLC的唯一标识符（一个整数）。

返回。

布尔型

例子。

global.Level_Max = 100;
ifsteam_user_owns_dlc(10354)
{
ifsteam_user_installed_dlc(10354)
{
global.Level_max = 200;
}
}

上述代码将检查用户是否购买并安装了ID为10354的DLC，如果是，则将一个全局变量设置为不同的值。

返回。DLC
