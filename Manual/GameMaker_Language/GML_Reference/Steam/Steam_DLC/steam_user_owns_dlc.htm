steam _user_owns_dlc

如果您的游戏已为此创建了DLC，则可以使用此功能在访问与其关联的任何文件之前检查并查看用户是否已购买了该产品。函数返回 true如果播放器拥有内容，并且 false如果用户没有，但请注意，即使用户拥有DLC，他们也可能未安装该DLC，因此您应该使用以下附加功能：steam_user_installed_dlc()在使用前检查其是否已安装。请注意， Steam ID可以是大数字，因此您可能需要将ID值转换为 int64()在将其提供给功能之前。

句法：

steam_user_owns_dlc(dlc_id);

争论描述
dlc_id要检查的DLC的唯一标识符（整数int64）。

返回值：

Boolean

例子：

global.Level_Max = 100;
if steam_user_owns_dlc(10354)
    {
    if steam_user_installed_dlc(10354)
        {
        global.Level_max = 200;
        }
    }

上面的代码将检查用户是否已购买并安装了ID为10354的DLC，如果已安装，则将全局变量设置为其他值。
