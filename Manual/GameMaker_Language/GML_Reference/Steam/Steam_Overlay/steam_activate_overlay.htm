steam _activate_overlay

Steam叠加层是Steam用户界面的一部分，可以在通过Steam发行的几乎所有游戏的顶部激活。它使用户可以访问他们的朋友列表，Web浏览器，聊天和游戏中DLC购买。用户在游戏中访问叠加层的默认键是SHIFT + TAB，但是您也可以使用此功能调出叠加层的任何页面。它采用下面列出的六个常数之一：

持续的描述
ov_friends当前用户的好友页面
ov_community您游戏的社区页面
ov_players该页面显示正在玩同一游戏或您最近玩过的其他游戏的页面
ov_settings Steam客户端叠加设置
ov_gamegroup将Steam社区的网络浏览器打开到该游戏的官方游戏组
ov_achievements游戏的成就页面

句法：

steam_activate_overlay(overlayindex);

争论描述
overlayindex Steam API UI的页面索引（请参见下面列出的常量）。

返回值：

N/A

例子：

var key = keyboard_lastkey;
switch (key)
    {
    case vk_f1: steam_activate_overlay(ov_friends); break;
    case vk_f2: steam_activate_overlay(ov_community); break;
    case vk_f3: steam_activate_overlay(ov_players); break;
    case vk_f4: steam_activate_overlay(ov_settings); break;
    case vk_f5: steam_activate_overlay(ov_gamegroup); break;
    case vk_f6: steam_activate_overlay(ov_achievements); break;
    }

上面的代码轮询最后按下的键盘键，如果它是1到6之间的任何功能键，它将打开Steam叠加层的相应页面。

背面： Steam覆盖
