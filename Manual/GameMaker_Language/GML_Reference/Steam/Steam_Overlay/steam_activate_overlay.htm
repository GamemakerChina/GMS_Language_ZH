激活覆盖层

Steam覆盖层是Steam用户界面的一部分，可以在通过Steam启动的几乎所有游戏的顶部激活。它允许用户访问他们的朋友列表、网络浏览器、聊天和游戏中的DLC购买。用户的默认键是 的默认键是SHIFT+TAB，但你也可以用这个功能调出覆盖层的任何页面。它需要以下六个常量中的一个。

恒定 描述
伙伴们 当前用户的好友页面
共同体 你的游戏的社区页面
运动员 显示正在玩同一游戏或最近与你一起玩的其他人的页面。
ov_settings Steam客户端的叠加设置
赌博集团 打开Steam社区网络浏览器，进入该游戏的官方游戏组。
绩效指标 你的游戏的成就页面

语法。

steam_activate_overlay(overlayindex)。

争论 描述
覆盖指数 要显示的SteamAPI用户界面的页面索引（见下面列出的常数）。

返回。

不适用

例子。

var key = keyboard_lastkey;
switch (key)
{
case vk_f1:steam_activate_overlay（ov_friends）; break;
case vk_f2:steam_activate_overlay（ov_community）; break;
case vk_f3:steam_activate_overlay（ov_players）; break;
case vk_f4:steam_activate_overlay（ov_settings）。 break;
case vk_f5:steam_activate_overlay（ov_gamegroup）; break;
case vk_f6:steam_activate_overlay（ov_achievements）; break;
}

上面的代码调查了最后一次按下的键盘键，如果它是1到6的任何一个功能键，它将打开Steam覆盖的相应页面。
