激活叠加浏览器

通过这个功能，你可以打开Steam游戏覆盖到它的网络浏览器，然后让它加载指定的URL。你需要使用完整的URL作为字符串，这样才能正确解决，例如："http://www.steamgames.com/"。

语法。

steam_activate_overlay(url)。

争论 描述
网址 要打开的覆盖层的（完整）URL。

返回。

不适用

例子。

if keyboard_check_pressed(vk_f1)
{
steam_activate_overlay_browser("http://www.steamgames.com/")。
}

上面的代码调查了键盘上的F1键，如果有的话，Steam覆盖将被打开并解析到给定的URL。
