steam _activate_overlay_browser

使用此功能，您可以将Steam游戏叠加层打开到其网络浏览器，然后使其加载指定的URL。你需要使用完整的URL作为一个字符串为这个正确解决，例如：“HTTP：// WWW steam games.com/”。

句法：

steam_activate_overlay(url);

争论描述
url打开叠加层的（完整）URL。

返回值：

N/A

例子：

if keyboard_check_pressed(vk_f1)
    {
    steam_activate_overlay_browser("http://www.steamgames.com/");
    }

上面的代码轮询键盘上的F1键，如果是，则将Steam覆盖并解析为给定的URL。
