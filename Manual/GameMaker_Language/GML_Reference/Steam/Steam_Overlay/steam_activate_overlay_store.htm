蒸汽_激活_覆盖层_商店

有了这个功能，你可以在游戏的商店页面上打开Steam覆盖，这样用户就可以购买或下载DLC（例如）。你需要提供游戏或DLC的唯一应用ID，当你设置它时，你会从Steam仪表板上得到这个ID。 了。

语法。

steam_activate_overlay_store(app_id)。

争论 描述
应用_id 你的游戏的唯一应用ID。

返回。

不适用

例子。

if keyboard_check_pressed(vk_f1)
{
steam_activate_overlay_store(global.DLC_id)。
}

上面的代码调查了键盘上的F1键，如果有的话，Steam覆盖层就会在页面上使用存储在全局变量中的应用程序ID打开游戏内容。
