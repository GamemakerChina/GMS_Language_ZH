steam _activate_overlay_store

使用此功能，您可以在Steam的商店页面上打开Steam叠加层，以便用户可以购买或下载DLC（例如）。您需要提供游戏或DLC的唯一应用ID，该ID会在您设置时Steam

句法：

steam_activate_overlay_store(app_id);

争论描述
app_id您游戏的唯一应用ID。

返回值：

N/A

例子：

if keyboard_check_pressed(vk_f1)
    {
    steam_activate_overlay_store(global.DLC_id);
    }

上面的代码轮询键盘上的F1键，如果是，则将使用存储在全局变量中的应用程序ID在页面上打开Steam
