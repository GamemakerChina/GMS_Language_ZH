steam _activate_overlay_user

此功能将打开Steam叠加层至与给定用户ID相关的所选对话框之一。可用的对话字符串为：

  • “ steam ID ”-将Steam社区Web浏览器打开到用户或组的页面
  • “ chat ”-为指定用户打开一个聊天窗口

请注意， Steam ID可以是大数字，因此您可能需要将ID值转换为 int64()在将其提供给功能之前。

句法：

steam_activate_overlay_user(dialog_name, steamid);

争论描述
dialog_name打开对话框的对话框（请参见下文）。
steam id要使用的Steam用户ID或组ID。

返回值：

N/A

例子：

var key = keyboard_lastkey;
switch (key)
    {
    case vk_f1: steam_activate_overlay_user("steamid", global.GameGroupID); break;
    case vk_f2: steam_activate_overlay_user("chat", global.FriendID); break;
    }

上面的代码轮询最后一个按下的键盘键，如果它是功能键1或功能键2，它将打开Steam叠加层以查看存储在全局变量“ GamegroupID”中Steam与存储在全局“ FriendID”变量中的用户聊天。
