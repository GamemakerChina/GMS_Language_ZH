steam _is_overlay_activated

此功能可用于确定用户是否Steam覆盖。如果覆盖层处于活动状态并且对用户可见，则该函数将返回 true，如果不是，它将返回 false。此功能可以使用的一个示例是轮询Steam API的叠加层，以便您可以在显示叠加层时暂停游戏。

句法：

steam_is_overlay_activated();

返回值：

Boolean

例子：

if steam_is_overlay_activated()
    {
    global.Pause = true;
    }

上面的代码将检查Steam叠加层是否处于活动状态，以及是否将全局变量“ Pause”设置为true。
