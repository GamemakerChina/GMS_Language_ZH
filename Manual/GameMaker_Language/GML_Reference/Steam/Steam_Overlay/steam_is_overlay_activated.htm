激活了吗？

这个函数可以用来发现用户是否激活了蒸汽覆盖层。如果覆盖层是激活的，并且对用户是可见的，该函数将返回true，如果不是，那么它将返回false。以下是一个例子 这个函数可以用来轮询SteamAPI的覆盖层，这样你就可以在显示覆盖层的时候暂停游戏了。

语法。

steam_is_overlay_activated()。

返回。

布尔型

例子。

ifsteam_is_overlay_activated()
{
global.Pause = true;
}

上面的代码将检查Steam叠加是否激活，如果是，它将把全局变量 "Pause "设置为true。
