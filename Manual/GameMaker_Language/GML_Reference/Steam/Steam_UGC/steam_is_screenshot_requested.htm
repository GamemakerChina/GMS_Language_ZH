steam _is_screenshot_requested

此功能将轮询Steam API，以查看是否已按下用于拍摄游戏屏幕截图的键。该函数只会返回 true如果已按下该键，则进行一步（游戏计时），然后将返回 false在其他所有时间。

注意：这不采取截图你。这仅表示已按下该键，您必须使用GameMaker Studio 2函数screen_save（）或screen_save_part（）来保存要上传的文件的本地副本。

句法：

steam_is_screenshot_requested();

返回值：

Boolean

例子：

if steam_is_screenshot_requested()
    {
    var file = "Catch_The_Haggis_" + string(global.scrn_num) + ".png");
    screen_save(file)
    steam_send_screenshot(file, window_get_width(), window_get_height());
    global.scrn_num += 1;
    }

上面的代码将轮询Steam API以获取屏幕截图请求，如果已经请求，则将生成图像文件的唯一名称，将获取屏幕截图，并将文件发送给用户的Steam社区页面。
