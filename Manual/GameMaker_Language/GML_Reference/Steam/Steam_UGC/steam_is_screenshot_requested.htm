蒸汽_is_screenshot_requested

该函数将轮询SteamAPI，查看用于拍摄游戏截图的按键是否被按下。如果按键被按下，该函数将只在一个步骤（游戏时间）中返回真，而在所有其他时间将返回假。 时都会返回false。

注意：这不会为你拍下屏幕。这只是提示你按键已被按下，你必须使用GameMaker Studio 2的功能 screen_save() 或 screen_save_part()来保存要上传的文件的本地副本。

语法。

steam_is_screenshot_requested()。

返回。

布尔型

例子。

ifsteam_is_screenshot_requested()
{
var file = "Catch_The_Haggis_" + string(global.scrn_num) + ".png");
screen_save(file)
steam_send_screenshot(file, window_get_width(), window_get_height());
global.scrn_num += 1;
}

上面的代码将轮询SteamAPI的截图请求，如果有的话，就会为图片文件生成一个唯一的名字，进行截图，并将文件发送到用户的Steam社区页面。

返回。UGC
