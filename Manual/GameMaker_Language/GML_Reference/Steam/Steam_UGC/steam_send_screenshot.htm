steam _send_screenshot

使用此功能，您可以将屏幕截图上传到Steam社区个人资料页面。 GameMaker Studio 2功能截图时创建的本地文件的名称。screen_save()或者 screen_save_part()。宽度和高度定义图像的大小，如果由于某种原因而失败，该函数将返回值0，如果成功则返回大于0的值。

句法：

steam_send_screenshot(filename, width, height);

争论描述
filename要上传的图像文件的名称（字符串）。
width图片的宽度（整数）。
height图片的高度（整数）。

返回值：

Real

例子：

if steam_is_screenshot_requested()
    {
    var file = "Catch_The_Haggis_" + string(global.scrn_num) + ".png");
    screen_save(file)
    steam_send_screenshot(file, window_get_width(), window_get_height());
    global.scrn_num += 1;
    }

上面的代码将轮询Steam API以获取屏幕截图请求，如果已经请求，则将生成图像文件的唯一名称，将获取屏幕截图，并将文件发送给用户的Steam社区页面。
