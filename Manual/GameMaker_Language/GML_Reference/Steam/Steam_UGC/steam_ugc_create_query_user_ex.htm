该函数返回唯一的查询句柄值，该值应存储在变量中，以供其他查询函数使用。请注意，此函数仅准备查询，但实际上不发送查询-为此，您必须调用函数steam _ugc_send_query.htm“>steam_ugc_send_query() -您可以进一步使用 steam_ugc_query_*()函数在搜索请求实际发送之前对其进行优化。

句法：

steam_ugc_create_query_user_ex(list_type, match_type, sort_order, page, account_id, creator_app_id, consumer_app_id);

争论描述
list_type要创建的数据列表的类型（请参见下面列出的常量）。
match_type要查询的UGC项目的类型（请参见下面列出的常量）。
sort_order数据的排序方式（请参见下面列出的常量）。
page要查询的页码。
account_id Steam帐户ID。
creator_app_id商品创建者应用ID。
consumer_app_id消费者应用程序ID。

返回值：

Query Handle

例子：

query_handle = steam_ugc_create_query_user_ex(ugc_list_Published, ugc_match_Items, ugc_sortorder_TitleAsc, page, global.AccountID, 0, global.GameID);

上面的代码创建一个查询请求，并将其句柄存储在变量中以备将来使用。
