steam _ugc_get_item_install_info

此功能可用于检索有关已订阅并下载到Steam本地存储区的任何给定已发布文件项的信息。您提供商品ID，并将索引提供给一个空的DS映射，如果该商品存在，则将使用以下键/值对填充该DS映射：

  • “ size_on_disk ”-磁盘上的文件大小（以字节为单位）
  • “ legacy_item ”-将 true或者 false取决于它是否是旧文件。
  • “文件夹”-这是安装内容的完整路径（请参阅Steam SDK文档中的“项目安装”，因为使用旧方法上传的“旧版”项目会有所不同）。

如果该项存在，则函数将返回 true并填充地图，否则它将返回 false并且地图将保持空白。

句法：

steam_ugc_get_item_install_info(published_file_id, info_map);

争论描述
published_file_id待更新的UGC的唯一句柄。
info_map （先前创建的）DS地图索引。

返回值：

Boolean

例子：

var item_map = ds_map_create();
steam_ugc_get_item_install_info(global.fileID, item_map);

上面的代码将使用以下命令查询全局变量“ fileID”中索引的项目的安装状态 DS Map存储信息。
