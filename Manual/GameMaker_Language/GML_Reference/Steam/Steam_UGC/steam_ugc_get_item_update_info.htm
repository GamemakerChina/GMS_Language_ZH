steam _ugc_get_item_update_info

此功能可用于检索有关给定文件ID的当前下载状态的信息。您提供商品ID，并将索引提供给一个空的DS映射，如果该商品存在，则将使用以下键/值对填充该DS映射：

  • “ needs_update ”-将返回 true或者 false取决于项目是否需要更新。
  • “ is_downloading ”-将返回 true或者 false取决于该项目当前是否正在下载。
  • “ bytes_downloaded ”-已下载的字节数。
  • “ bytes_total ”-磁盘上项目所需的总大小（字节数）。

如果该项存在，则函数将返回 true并填充地图，否则它将返回 false并且地图将保持空白。

句法：

steam_ugc_get_item_update_info(published_file_id, info_map);

争论描述
published_file_id要检查的UGC的唯一文件ID。
info_map （先前创建的）DS地图索引。

返回值：

Boolean

例子：

var info_map = ds_map_create();
var info = steam_ugc_get_item_update_info(global.fileID, info_map);
if info
    {
    draw_text(32, 15, "needs_update: " + string(info_map[? "needs_update"]));
    draw_text(32, 30, "is_downloading: " + string(info_map[? "is_downloading"]));
    draw_text(32, 45, "bytes_downloaded: " + string(info_map[? "bytes_downloaded"]));
    draw_text(32, 60, "bytes_total: " + string(info_map[? "bytes_total"]));
    }

上面的代码将使用 DS Map存储信息。
