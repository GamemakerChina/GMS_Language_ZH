steam _ugc_query_set_return_long_description

此函数可用于进一步过滤任何给定的UGC查询，特别是在发送查询时触发的回调事件中检索详细描述文本。查询句柄是您创建查询时返回的值（例如，使用 steam_ugc_create_query_user()），第二个参数是 true或者 false。该函数将返回 true如果查询过滤器设置正确，或者 false否则。

句法：

steam_ugc_query_set_return_long_description(ugc_query_handle , long_description);

争论描述
ugc_query_handle要使用的查询句柄。
long_description是否有查询返回详细描述文本。

返回值：

Boolean

例子：

var query_handle = steam_ugc_create_query_all(ugc_query_RankedByVote, ugc_match_Items, 1);
steam_ugc_query_set_return_long_description(query_handle, true);
steam_ugc_query_set_allow_cached_response(query_handle, true);
query_ID = steam_ugc_send_query(query_handle);

上面的代码创建一个查询请求，并将其句柄存储在一个局部变量中，以备将来在其余函数中使用，以便在发送查询之前进一步定义查询请求。
