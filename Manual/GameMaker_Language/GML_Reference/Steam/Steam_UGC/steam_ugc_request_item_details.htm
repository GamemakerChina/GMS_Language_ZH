steam _ugc_request_item_details

此功能可用于检索有关给定文件ID的信息。您提供文件ID并提供检查的最长期限（ Steam文档），然后该函数返回一个唯一的异步ID值，该值可用于在Steam Async事件中正确检查详细信息。该事件将具有 async_loadDS映射，其中将填充以下键/值对：

  • “ id ”-触发事件的函数返回的异步ID
  • “ result ”-运算结果（实数值）。这将是GML常数 ugc_result_success或其他一些实数。因此，您应该检查该常量，以确保调用成功，否则无法正常进行。返回的其余可能值显示为Steam “ EResult”值的结果，您应该看到 steamclientpublic.h在所有89个可能值的SDK标头中。
  • “ event_type ”-字符串“ ugc_item_details ”
  • “ cached_data ”-将 true如果返回的详细信息来自本地缓存，或者 false如果它们来自服务器
  • “ published_file_id” “-为项目保留唯一的已发布文件ID
  • “ file_type ”-使用的文件类型
  • “ creator_app_id ”-项目创建者Steam
  • “ consumer_app_id ”-商品消费者Steam
  • “ title ”-项目标题
  • “ description ”-商品说明
  • “ steam _id_owner ”-物品所有者Steam
  • “ time_created ”-首次创建项目的时间
  • “ time_updated ”-上次更新项目
  • “ time_added_to_user_list ”-订阅该项目的时间
  • “可见性”-项目的可见性（请参见 steam_ugc_set_item_visibility()对于返回的常量）
  • “被禁止”-该项目是否已被禁止（true ） 或不 （false ）
  • “ accepted_for_use ”-该项目是否已被接受使用（true ） 或不 （false ）
  • “ tags_truncated ”-标记的简短版本作为数组
  • “ tags ”-项目标签的数组
  • “ handle_file ”-项目的唯一文件句柄
  • “ handle_preview_file ”-该项目的图像预览的唯一句柄（可以用作 steam_ugc_download()下载预览图像）
  • “ filename ”-项目文件的名称
  • “ file_size ”-项目文件的大小
  • “ Preview_file_size ”-预览图像的大小
  • “ url ”-项目的完整URL
  • “投票数”-收到的投票数
  • “ voices_down ”-收到的否决票数
  • “得分”-项目的总得分
  • “account_id_owner” -从该帐户ID Steam ID所有者（这可以在函数中使用 steam_ugc_create_query_user_ex()）

句法：

steam_ugc_request_item_details(published_file_id, max_age_seconds);

争论描述
published_file_id要检查的UGC的唯一文件ID。
max_age_seconds要检查数据的“年龄”（建议30-60秒）。

返回值：

Async ID

扩展示例：

在此示例中，我们发送了对商品的详细信息请求，然后解析结果 async_loadDS映射设置一些变量。首先，我们发送请求：

steam_details = steam_ugc_request_item_details(global.fileID, 60);

上面的代码将请求有关具有存储在全局变量中的文件ID的项目的详细信息，并将使用返回的信息Steam在这种情况下，我们可以解析地图并将某些值存储在变量中，这些变量然后可以用于向用户显示信息：

var map_id = async_load[? "id"];
var result = async_load[? "result"];
if (map_id == steam_details) && (result == ugc_result_success)
    {
    mTitle = async_load[? "title"];
    mDesc = async_load[? "description"];
    mTags = async_load[? "tags"];
    m_hPreviewFile = async_load[? "handle_preview_file"];
    m_hOwnerSteamId = async_load[? "steam_id_owner"];
    mOwnerAccountId = async_load[? "account_id_owner"];
    mPubFileId = async_load[? "published_file_id"];
    mScore = async_load[? "score"];
    }
