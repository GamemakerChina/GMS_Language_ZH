steam _ugc_subscribe_item

此功能可用于订阅UGC项目。您提供已发布的文件ID，然后该函数返回唯一的异步ID值，该值可用于检查Steam Async事件中何时下载了已订阅的项目。该事件将具有 async_loadDS映射，其中将填充以下键/值对：

  • “ id ”-触发事件的函数返回的异步ID
  • “ result ”-运算结果（实数值）。这将是GML常数 ugc_result_success或其他一些实数。因此，您应该检查该常量，以确保调用成功，否则无法正常进行。返回的其余可能值显示为Steam “ EResult”值的结果，您应该看到 steamclientpublic.h在所有89个可能值的SDK标头中。
  • “ event_type ”-字符串“ ugc_subscribe_item ”。
  • “ published_file_id ”-发布项目的唯一ID。

句法：

steam_ugc_subscribe_item(published_file_id);

争论描述
published_file_id UGC要订阅的唯一文件ID。

返回值：

Async ID

例子：

steam_sub = steam_ugc_subscribe_item(global.pubFileID);

上面的代码将订阅（并下载）具有存储在全局变量“ pubFileID”中的文件ID的项目。
