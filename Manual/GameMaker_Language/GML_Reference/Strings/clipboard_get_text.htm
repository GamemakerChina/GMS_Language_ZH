clipboard_get_text

此函数将返回剪贴板中包含的文本字符串。如果没有文本存储，它将返回一个空字符串“”。

注意：此功能仅对Windows目标有效。

句法：

clipboard_get_text();

返回值：

String

例子：

if clipboard_has_text()
   {
   str = clipboard_get_text();
   clipboard_set_text("");
   }

上面的代码检查剪贴板中的文本，如果其中包含文本，则将其作为字符串读入变量“ str”。最后，通过将剪贴板设置为空字符串来清除剪贴板。
