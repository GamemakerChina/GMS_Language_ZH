clipboard_set_text

此函数将设置剪贴板以保存定义的字符串。您可以将其设置为空字符串“”，以有效清除剪贴板中的文本。

注意：此功能仅对Windows目标有效。

句法：

clipboard_set_text(string);

争论描述
string将剪贴板设置为保留的文本（作为字符串）。

返回值：

N/A

例子：

if clipboard_has_text()
    {
    str = clipboard_get_text();
    clipboard_set_text("");
    }

上面的代码检查剪贴板中的文本，如果其中包含文本，则将其作为字符串读入变量“ str”。最后，通过将剪贴板设置为空字符串来清除剪贴板。
