ord

此函数采用单个字符输入字符串，并返回该字符的Unicode（UTF8）值。请注意，当与 keyboard_check*函数，输入字符串的长度只能是一个字符，并且只能是0到9之间的数字，或者是A到Z之间的大写罗马字符。

句法：

ord(string);

争论描述
string查找Unicode代码的字符串。

返回值：

Real

例子：

if keyboard_check(ord("W"))
    {
    y -= 4;
    }

如果按住键W，这将使调用实例向上移动四个像素。

背面：琴弦
