real

此函数可用于将给定的字符串转换为实数。使用此功能时，将考虑字符串中的数字，减号，小数点和指数部分，而其他字符（例如字母）将引发错误。如果您知道或怀疑某个字符串可能包含其他字符，则可以使用 string_digits()删除所有非数字字符，然后再使用此功能将结果字符串转换为实数。

句法：

real(string);

争论描述
string要转换为实数值的字符串。

返回值：

Real

例子：

var t_str = string_digits(input_str);
age = real(t_str);

上面的代码将采用输入字符串，除去数字以外的所有其他字符，然后将变量“ age”设置为保留返回字符串的实数值。
