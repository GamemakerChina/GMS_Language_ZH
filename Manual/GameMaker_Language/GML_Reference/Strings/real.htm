真正的

这个函数可以用来把一个给定的字符串变成一个实数。当使用这个函数时，字符串中的数字、减号、小数点和指数部分会被考虑进去，而其他字符（如字母）会导致错误 被抛出。如果你知道或怀疑一个字符串可能有其他字符，那么你可以使用 string_digits()来删除所有非数字字符，然后再使用这个函数将得到的字符串 变成一个实数。

语法。

real(string)。

争论 描述
绳子 要转换为实值的字符串。

返回。

真实的

例子。

var t_str = string_digits(input_str);
age = real(t_str);

上面的代码将接收输入的字符串，剥离除数字以外的所有字符，然后设置变量 "age "来保存返回字符串的真实数字值。
