绳子

通过这个函数，你可以把任何实数变成一个字符串。如果该实数是一个整数，它将被保存为无小数位，否则，它将被保存为两个小数位。如果你需要更多的小数位，那么请使用函数 string_format().还要注意的是，在存储数组、数据结构的变量上使用这个函数。 或结构体的变量上使用这个函数，将把这些变量的内容转换成字符串，然后可以输出到控制台或保存到文件中进行调试。

语法。

string(val)。

争论 描述
绳子 要转为字符串的实数。

返回。

字符串

例子。

draw_text(100, 100, "Score: " + string(score) + " / Health: " + string(health));

上面的代码使用字符串函数来同时绘制实数和字符串（draw只接受字符串或实数，而不是两者）。
