string

使用此功能，您可以将任何实数转换为字符串。如果实数是整数，则将其保存为不带小数位，否则，将以两个小数位保存。如果需要更多小数位，请使用string_format()。还要注意，在存储数组，数据结构或结构的变量上使用此函数会将这些变量的内容转换为字符串，然后可以将其输出到控制台或保存到文件以进行调试。

句法：

string(val);

争论描述
string要转换为字符串的实数。

返回值：

String

例子：

draw_text(100, 100, "Score: " + string(score) + " / Health: " + string(health));

上面的代码使用 string函数画实数和字符串连接在一起（画只接受一个字符串或一个真实的，但不能同时）。
