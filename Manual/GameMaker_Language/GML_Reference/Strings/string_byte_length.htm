string_byte_length

这个函数返回一个字符串中的字节数，但你应该注意，由于它们被保持为UTF8，这将不等于它们的字符串长度。

语法。

string_byte_length(string)。

争论 描述
绳子 要测量字节数的字符串。

返回。

真实的

例子。

bytesize = string_byte_length("Hello World")。

这将把bytesize设置为 "Hello World "中的字节数。

返回。弦乐
