符号

你可以使用这个函数来返回一个字符串中特定位置的特定字符，索引从1开始为第一个字符。如果没有找到任何字符，或者字符串比给定的索引值短，则返回一个空字符串"" 将被返回。

语法。

string_char_at(str, index)。

争论 描述
弦外之音 要检查的字符串。
指数 从的位置来获取字符。

返回。

字符串

例子。

str1 = "Hello World";
str2 = string_char_at(str1, 7);

这将把str2设置为字符串str1中的第七个字符（"H "算作第一个），在这里是 "W"。

返回。弦乐
