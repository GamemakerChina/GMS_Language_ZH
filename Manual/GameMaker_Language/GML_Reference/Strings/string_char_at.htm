string_char_at

您可以使用此函数在字符串中的特定位置返回特定字符，第一个字符的索引从1开始。如果未找到任何字符或该字符串短于索引的值，则返回空字符串“”。

句法：

string_char_at(str, index);

争论描述
str要检查的字符串。
index从中获取角色的位置。

返回值：

String

例子：

str1 = "Hello World";
str2 = string_char_at(str1, 7);

这将设置 str2到字符串中的第七个字符（“ H”作为第一个字符）str1 ，在这种情况下为“ W”。
