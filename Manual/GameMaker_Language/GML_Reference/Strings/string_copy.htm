string_copy

使用此功能，您可以轻松地从字符串中选择多个字符以复制到另一个字符。字符串中的第一个字符总是索引为1，而不是您期望的0，因此要复制（例如）字符串的前五个字符 string_copy(str, 1, 5)。下面提供了另一个示例。

句法：

string_copy(str, index, count);

争论描述
str要复制的字符串。
index要从中复制的字符串中第一个字符的位置（从1编号）
count从第一个位置开始复制的字符数

返回值：

String

例子：

name = keyboard_string;
if (string_length(name) > 15)
      {
      keyboard_string = string_copy(name, 1, 15);
      }

上面的代码将允许玩家通过键盘输入一个字符串，然后将其存储在变量中 name。然后检查此变量以查看其是否超过了15个字符的最大长度，如果超过，则将keyboard_string替换为15个字符的副本。name细绳。这有效地将玩家输入限制为15个字符。
