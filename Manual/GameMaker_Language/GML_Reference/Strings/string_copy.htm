拷贝字符串

通过这个函数，你可以很容易地从一个字符串中选择一些字符复制到另一个字符串中。字符串中的第一个字符总是被索引为1，而不是你所期望的0，所以要复制（例如）字符串的前五个字符 的前五个字符，你可以使用string_copy(str, 1, 5)。下面提供一个进一步的例子。

语法。

string_copy(str, index, count)。

争论 描述
弦外之音 要复制的字符串。
指数 要复制的字符串中第一个字符的位置（从1开始编号）。
数 要复制的字符数，从第一个字符的位置开始。

返回。

字符串

例子。

name = keyboard_string;
if (string_length(name) > 15)
{
keyboard_string = string_copy(name, 1, 15);
}

上面的代码将允许玩家通过键盘输入一个字符串，然后存储在变量名中。然后检查这个变量是否超过了15个字符的最大长度，如果超过了，键盘字符串 会被一个15个字符的名字字符串的副本所取代。这就有效地将玩家的输入限制在15个字符。

返回。弦乐
