弦外之音_count

这个函数将返回给定子串在一个特定字符串中出现的次数。通过这种方式，你可以检查一个字母或一个短语在一段存储文本中重复出现了多少次。

语法。

string_count(substr, str)。

争论 描述
子层 要检查字符串的子串。
弦外之音 要检查的字符串。

返回。

真实的

例子。

str1 = "Hello World";
ocount = string_count( "o", str1 ) 。

这将把变量ocount设置为str1中的 "o "的数量，本例中为2。

返回。弦乐
