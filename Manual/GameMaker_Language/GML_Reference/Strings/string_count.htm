string_count

此函数将返回给定子字符串出现在特定字符串中的次数。这样，您可以检查一个字母或短语在存储的文本部分中重复多少次。

句法：

string_count(substr, str);

争论描述
substr要检查字符串的子字符串。
str要检查的字符串。

返回值：

Real

例子：

str1 = "Hello World";
ocount = string_count( "o", str1 );

这将设置变量 ocount到“ o”的数量 str1，在这种情况下为2。
