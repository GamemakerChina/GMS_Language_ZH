string_delete

您可以使用此功能删除字符串的特定部分。因此，您提供了输入字符串以及该字符串中的开始和结束位置以删除字符（索引从1开始），并且该函数将返回其中没有该节的新字符串。

句法：

string_delete(str, index, count);

争论描述
str要复制和删除的字符串。
index要删除的第一个字符的位置。
count要删除的字符数。

返回值：

String

例子：

str1 = "Helloo World";
str2 = string_delete(str1, 5, 1);

这将设置 str2到“ Hello World”，因为它从“ Hello”中删除了多余的“ o”，即：字符串从第一个字母起算5个位置，然后删除1个字符）。
