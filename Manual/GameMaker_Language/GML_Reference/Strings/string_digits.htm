string_digits

您可以使用此函数来解析给定的字符串并从中获取任何数字。例如，假设您有这样的文字-“我今年81岁”。使用此函数，您将获得返回字符串“ 81”。

句法：

string_digits(string);

争论描述
string从中获取数字的字符串。

返回值：

String

例子：

var t_str = string_digits(input_str);
age = real(t_str);

上面的代码将采用输入字符串，去除数字以外的所有字符，然后设置变量 age以保存返回字符串的实数值（因此，如果输入字符串为-例如-“ I am 18”，则该函数将返回“ 18”）。
