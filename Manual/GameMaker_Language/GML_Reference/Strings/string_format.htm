string_format

使用您自己的格式将实数转换为字符串，您可以在其中选择将多少“位”保存到字符串中以及还保存多少小数位。两者都很方便，有些游戏更喜欢将分数显示为设定的位数，而精确控制小数点后两位的小数点可能更好。string()无法提供。如果指定的位数大于要显示的值和/或指定的位数加小数位数小于总位数，则将在该值之前添加空格以弥补差异（请参见下面的示例）。如果给定的值小于总数和要包括的小数位数，则零将添加到小数点右边。默认格式是在左边没有多余的空格，在右边仅保留两位小数，例如“ 265.73”。

句法：

string_format(val, tot, dec);

争论描述
val要转换为字符串的实数。
tot要显示的主要号码的总位数。零或空格将被插入以与此匹配。
dec要包括的小数位数。

返回值：

String

例子：

str1 = string_format(1234, 8, 0);
str2 = string_format(pi, 1, 10);
str3 = string_format(pi, 5, 5);

这将设置 str1到“ 1234”，str2至“ 3.1415926535”，并 str3改为“ 3.14159”。
