string_hash_to_newline

此函数返回一个字符串，其中“＃”符号已转换为新行。因此，字符串格式为：

"Hello#World"

将以以下方式打印到屏幕上：

Hello
World

重要的！提供此功能是为了使GameMaker Studio 2和该软件的先前版本之间具有导入兼容性，因此，除非有必要复制过时的功能，否则不应使用此功能。相反，您应该使用转义字符标记新行。有关导入兼容性的更多信息，请参见此处。

句法：

string_hash_to_newline(string);

争论描述
string要在多行中转换的字符串

返回值：

String

例子：

var str = string_hash_to_newline("Hello#World");
draw_text(32, 32, str);

上面的代码将带有哈希符号的字符串转换为分为两行的字符串，然后将其打印到屏幕上。
