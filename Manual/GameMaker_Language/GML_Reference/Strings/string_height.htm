string_height

该函数将返回输入字符串的高度（以像素为单位），并考虑到行间距和文本可能具有的任何换行符。根据构成字符串的最高字母来计算文本元素之间的距离非常方便 draw_text()使用当前定义的字体。

句法：

string_height(string);

争论描述
string要测量高度的字符串。

返回值：

Real

例子：

var hh = string_height("ABCDEFGHIJKLMNOPQRSTUVWXYZ");
draw_text(32, 32, string(score));
draw_text(32, 32 + hh, string(lives);

上面的代码将获取给定字符串的高度，然后使用返回的字符串高度作为分隔符绘制两行文本。
