string_insert

使用此功能，您可以创建由两个字符串组成的新字符串，其中一个已在给定位置插入到另一个中。例如，将用户名添加到预定义的文本中可能很有用，这样可以使您的游戏玩家更多地参与到该操作中。计算插入字符串的位置时要记住，索引从1开始，因此字符串中的第一个字符为1，而不是您期望的0。

句法：

string_insert(substr, str, index);

争论描述
substr要插入的子字符串。
str要复制的字符串。
index插入子字符串的字符串的字符位置。

返回值：

String

例子：

str2 = string_insert(username, "Hello, , how are you?", 8);

这会将字符串“ username”变量中的字符串插入给定的短语中，结果字符串如下所示：“你好，NAME，你好吗？”。
