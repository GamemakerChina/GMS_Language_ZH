植入字符串

通过这个函数，你可以创建一个由两个字符串组成的新字符串，其中一个被插入到另一个的指定位置。例如，它可以用来在预定义的文本中加入一个用户名，从而使你的游戏的玩家感到 更多的参与到行动中。在计算插入的位置时要记住，字符串的索引是从1开始的，所以字符串中的第一个字符是1，而不是你可能期望的0。

语法。

string_insert(substr, str, index)。

争论 描述
子层 要插入的子串。
弦外之音 要复制的字符串。
指数 在字符串中插入子串的位置，即字符数。

返回。

字符串

例子。

str2 = string_insert(username, "Hello, , how are you?", 8);

这将把 "用户名 "变量中的字符串插入给定的短语中，产生的字符串看起来像这样："你好，NAME，你好吗？"。

返回。弦乐
