string_last_pos

此函数将返回字符串中子字符串实例的字符位置，从字符串的末尾开始搜索（因此，string_pos() 。如果找不到搜索字符串，则该函数将返回0，否则将返回搜索字符串的第一个字符的位置。请记住，对于旧的支持字符串，索引从1开始，因此1是字符串中的第一个位置，而不是您期望的0。

句法：

string_last_pos(substr, str);

争论描述
substr要在字符串中查找的子字符串。
str字符串。

返回值：

Integer

例子：

if (string_last_pos(",", text) != 0)
   {
   string_insert(name, text, string_last_pos(",", text));
   }

上面的代码从变量“ text”中存储的字符串中搜索一个逗号，从字符串的末尾开始搜索。如果找到一个，它将在该位置插入子字符串“ name”。
