string_last_pos_ext

此函数将返回字符串中子字符串实例的字符位置，从作为起始位置的位置向后搜索字符串。如果找不到搜索字符串，则该函数将返回0，否则将返回搜索字符串的第一个字符的位置。请记住，对于旧的支持字符串，索引从1开始，因此1是字符串中的第一个位置，而不是您期望的0。

句法：

string_last_pos_ext(substr, str, start_pos);

争论描述
substr要在字符串中查找的子字符串。
str字符串。
start_pos搜索的起始位置。

返回值：

Integer

例子：

if (string_last_pos_ext(",", text, 20) != 0)
   {
   string_insert(name, text, string_last_pos_ext(",", text, 20));
   }

上面的代码在存储在变量“文本”中的字符串中搜索第20个字符之前的逗号，如果找到一个逗号，则在该位置插入子字符串“名称”。
