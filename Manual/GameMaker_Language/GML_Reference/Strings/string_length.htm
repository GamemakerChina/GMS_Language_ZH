string_length

此函数返回组成给定字符串的字符数。它对于诸如确定何时限制自定义文本条目的字符长度（例如：将玩家的名字限制为10个字符）之类的事情很有用。请记住，这与 string_width()在于它测量的字符数串中，而不是它的宽度为吸入像素在屏幕上。

句法：

string_length(string);

争论描述
string要测量的字符数的字符串。

返回值：

Real

例子：

if (string_length(name) > 10)
   {
   name = string_copy(name, 1, 10);
   }

这将检查名称字符串是否大于十个字符，如果大于十个字符，它将仅复制和使用前十个字符。
