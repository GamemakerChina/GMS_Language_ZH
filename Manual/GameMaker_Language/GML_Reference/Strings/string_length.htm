字符串_length

这个函数返回组成一个给定字符串的字符数。它对于计算何时限制自定义文本条目的字符长度（例如：玩家的名字上限为10个字符）很有用。请记住，这与 不同于 string_width()不同的是，它测量的是字符串中的字符数，而不是在屏幕上显示的像素宽度。

语法。

string_length(string)。

争论 描述
绳子 要测量字符数的字符串。

返回。

真实的

例子。

如果（string_length(name) > 10）
{
name = string_copy(name, 1, 10);
}

这将检查名称字符串是否大于十个字符，如果是，它将只复制和使用前十个字符。

返回。弦乐
