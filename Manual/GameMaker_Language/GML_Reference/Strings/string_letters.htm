string_letters

使用此功能，您可以删除所有未归类为字母的字符。这对于防止人们在文本输入中输入不需要的字符非常方便（例如输入“＃”以强制换行）。

注意：此功能仅从A-Z（大写或小写）检测并返回26个字母的英文字母。

句法：

string_letters(string);

争论描述
string要转换为字母的字符串。

返回值：

String

例子：

username = string_letters(username);

上面的代码将“ username”变量设置为仅包含原始字符串的字母字符（id原始字符串为“ MacSw33ny”，返回的字符串为“ MacSwny”）。
