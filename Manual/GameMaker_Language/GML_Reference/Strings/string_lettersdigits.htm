字母数字

这个函数将返回一个给定字符串的副本，除字母和数字外的所有内容都被删除，这意味着它可以用来删除任何不需要的字符（如 "#"或"？"），例如，一个登录名或密码。

注意：该功能只返回A-Z（大写或小写）26个英文字母和数字0-9。

语法。

string_lettersdigits(string)。

争论 描述
绳子 要转换为字母和数字的字符串。

返回。

字符串

例子。

如果(string_length(password)>string_length(string_lettersdigits(password)))
{
draw_text(32,32, "无效密码！请只输入字母和数字！");
}

上面的代码将检查一个字符串的长度和同样的字符串的长度，但删除了所有的符号。如果它们不一样，就会在屏幕上画出一条信息。

返回。弦乐
