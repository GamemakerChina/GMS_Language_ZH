string_lettersdigits

此函数将返回给定字符串的副本，但除去其字母和数字外的所有内容，这意味着该字符串可用于从登录名或密码中删除所有不需要的字符（例如“＃”或“？”）密码。

注意：此功能仅返回A-Z（大写或小写）的26个字母的英文字母和数字0-9。

句法：

string_lettersdigits(string);

争论描述
string要转换为字母和数字的字符串。

返回值：

String

例子：

if (string_length(password) > string_length(string_lettersdigits(password)))
   {
   draw_text(32,32,"Invalid Password! Only Letters and numbers please!");
   }

上面的代码将对照相同字符串的长度来检查字符串的长度，但会删除所有符号。如果它们不同，则会在屏幕上显示一条消息。
