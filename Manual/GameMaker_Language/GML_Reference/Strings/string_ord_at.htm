弦外之音_at

你可以使用这个函数来返回一个字符串中特定位置的特定字符代码，索引从第一个字符的1开始。如果没有找到任何字符，或者字符串比给定的索引值短，则返回-1。

语法。

string_ord_at(str, index)。

争论 描述
弦外之音 要检查的字符串。
指数 获取字符代码的位置。

返回。

真实的

例子。

str = "Hello World";
char_code = string_ord_at(str, 7);

这将获得字符串 "str "中第七个字符（其中 "H "算作第一个）的字符代码，并将其存储在变量 "char_code "中。

返回。弦乐
