string_ord_at

您可以使用此函数在字符串中的特定位置返回特定的字符代码，第一个字符的索引从1开始。如果未找到字符或字符串短于索引的值，则返回-1。

句法：

string_ord_at(str, index);

争论描述
str要检查的字符串。
index从中获取字符代码的位置。

返回值：

Real

例子：

str = "Hello World";
char_code = string_ord_at(str, 7);

这将获取字符串“ str”中第七个字符的字符代码（其中“ H”为第一个字符），并将其存储在变量“ char_code”中。
