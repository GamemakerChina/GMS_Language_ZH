词条_位置

这个函数将返回一个字符串中子字符串实例的字符位置，从字符串的开头开始搜索（要从结尾开始搜索，使用函数 string_last_pos()).该 如果完全没有找到，该函数将返回0，如果找到了，则返回子串的第一个字符的位置。请记住，对于传统的支持，字符串的索引是从1开始的，所以1是字符串中的第一个位置，而不是你可能期望的0。 的预期。这方面的一个用途是过滤可能被认为具有攻击性的词语，或者寻找正确的位置将一些文字插入另一个字符串中。

语法。

string_pos(substr, str)。

争论 描述
子层 要在字符串中寻找的子串。
弦外之音 的字符串。

返回。

整数

例子。

if (string_pos(",", text) != 0)
{
string_insert(name, text, string_pos(",", text));
}

上面的代码在存储在变量 "text "中的字符串中搜索逗号，如果找到一个，就在该位置插入子串 "name"。
