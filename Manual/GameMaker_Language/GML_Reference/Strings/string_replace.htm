string_replace

您可以使用此函数来解析字符串以查找特定部分，然后可以将其替换为您指定的新字符串。

句法：

string_replace(str, substr, newstr);

争论描述
str要复制的字符串。
substr字符串中要替换的子字符串。
newstr新的子字符串将替换先前的子字符串。

返回值：

String

例子：

str1 = "Hello Earth";
str2 = string_replace(str1, "Earth", "World");

这将设置 str2到 str1，但将其实例的“地球”替换为“世界”，从而导致 str2被称为“ Hello World”。
