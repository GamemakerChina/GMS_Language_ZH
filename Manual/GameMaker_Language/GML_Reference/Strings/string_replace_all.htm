替换掉所有的字符串

你可以使用这个函数来解析一个字符串，寻找特定的部分，然后用你指定的新字符串替换所有出现的部分。

语法。

string_replace_all(str, substr, newstr)。

争论 描述
弦外之音 要复制的字符串。
子层 在所有情况下，要替换的字符串中的子串。
新的 新的子串将取代之前的子串。

返回。

字符串

例子。

str1 = "Hexxo Worxd";
str2 = string_replace_all(str1, "x", "l") 。

这将把str2设置为str1，但将其 "x "的实例替换为 "l"，从而使str2成为 "HelloWorld"。

返回。弦乐
