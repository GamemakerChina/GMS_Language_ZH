string_replace_all

您可以使用此函数来解析字符串以查找特定部分，然后将其替换为在出现的所有位置指定的新字符串。

句法：

string_replace_all(str, substr, newstr);

争论描述
str要复制的字符串。
substr在所有实例中要替换的字符串中的子字符串。
newstr新的子字符串替换以前的子字符串。

返回值：

String

例子：

str1 = "Hexxo Worxd";
str2 = string_replace_all(str1, "x", "l");

这将设置 str2到 str1，但将其实例“ x”替换为“ l”，结果是 str2被称为“ Hello World”。
