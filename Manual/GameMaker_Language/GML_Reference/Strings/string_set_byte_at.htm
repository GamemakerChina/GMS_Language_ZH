string_set_byte_at

这个函数在一个字符串中直接设置一个字节（基于UTF8格式），并返回一个带有变化的字符串的副本。

注意：这个功能非常慢，所以要仔细考虑是否有必要，以及在什么地方使用它。

语法。

string_set_byte_at(str, pos, byte)。

争论 描述
弦外之音 要改变字节的字符串。
姿势 在字符串中改变字节的位置（从1开始）。
字节 新的字节值。

返回。

字符串

例子。

str = string_set_byte_at("hello", 2, 97);

上面的代码将改变字符串中第二个字母的字节值，因此将变量str设置为持有 "hallo"。

返回。弦乐
