string_set_byte_at

此函数直接在字符串中设置一个字节（基于UTF8格式），并返回带有更改的字符串副本。

注意：此功能的运行速度非常慢，因此请仔细考虑是否必要以及在何处使用它。

句法：

string_set_byte_at(str, pos, byte);

争论描述
str更改字节的字符串。
pos字符串中要更改其字节的位置（从1开始）。
byte新的字节值。

返回值：

String

例子：

str = string_set_byte_at("hello", 2, 97);

上面的代码将更改字符串中第二个字母的字节值，因此设置变量 str举行“你好”。
