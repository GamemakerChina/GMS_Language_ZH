string_width_ext

此函数将返回输入字符串的最大宽度（以像素为单位），并考虑到行距和换行宽度（定义为在插入换行之前字符串可以占据的像素数）。这是非常方便的，它可以根据一个字符串的最大宽度来计算文本元素之间的距离，该最大宽度被分割成几行，用 draw_text_ext()使用当前定义的字体。间隔和宽度可以设置为-1，以获取默认间隔。

句法：

string_width_ext(string, sep, w);

争论描述
string要测量宽度的字符串。
sep文本行之间的距离（以像素为单位），就像正在绘制字符串一样。
w换行之前，字符串的最大宽度（以像素为单位），就像绘制字符串一样。

返回值：

Real

例子：

var ww = string_width_ext(str_Story_Text[1], -1, 100);
draw_text_ext(32, 32, str_Story_Text[1], -1, 100);
draw_text_ext(32 + ww, 32, str_Story_Text[2], -1, 100);

上面的代码将考虑给定的行距和换行符宽度，获得给定字符串的宽度，然后使用返回的总字符串宽度作为分隔符，绘制两行文本。
