弦外之音

这个函数将返回输入字符串的最大宽度（像素），同时考虑到行间距和换行宽度（定义为在插入换行之前字符串所能占用的像素数）。它非常 在计算文本元素之间的距离时，它非常方便，因为它是根据字符串的最大宽度分割成几行来绘制的。 draw_text_ext()使用当前定义的 字体来绘制的。分离度和宽度可以设置为-1以获得默认的间距。

语法。

string_width_ext(string, sep, w)。

争论 描述
绳子 要测量宽度的字符串。
绵绵不绝 以像素为单位的文本行之间的距离，就像正在绘制的字符串一样。
w 换行前字符串的最大宽度（单位：像素），就像字符串被画出来一样。

返回。

真实的

例子。

var ww = string_width_ext(str_Story_Text[1], -1, 100);
draw_text_ext(32, 32, str_Story_Text[1], -1, 100);
draw_text_ext(32 + ww, 32, str_Story_Text[2], -1, 100) 。

上面的代码将得到给定字符串的宽度，考虑到分行和换行宽度，然后画出两行文字，用返回的字符串总宽度作为分隔符。
