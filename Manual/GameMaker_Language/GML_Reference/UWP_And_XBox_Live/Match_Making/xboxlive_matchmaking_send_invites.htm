xboxlive_matchmaking_send_invites

此功能将打开Xbox Live系统对话框，以允许用户向其游戏发送邀请。您指定应该控制对话框的本地用户（对话框将使用谁的朋友列表），他们要邀请人们参加的会话的唯一会话ID值以及在对话框中显示的一些默认文本。您可以从 async_load该函数生成的DS地图 xboxlive_matchmaking_create()。

如果参数无效或用户未登录Xbox Live，则该函数将返回-1，否则返回大于0的值。

句法：

xboxlive_matchmaking_send_invites(from_user, session_to_invite_into, text_to_add_to_invite);

争论描述
from_user用于控制对话的本地用户ID。
session_to_invite_into要加入的会话的唯一会话ID。
text_to_add_to_invite要添加到邀请中的短文字。

返回值：

Integer

例子：

if gamepad_button_check_pressed(0, gp_face1)
    {
    var _uid = xboxlive_user_for_pad(0);
    xboxlive_matchmaking_send_invites(_uid, global.session_ID, "Come and join my game!");
    }

上面的代码检查是否按下了游戏板按钮，如果检测到按下该按钮，则将打开会话邀请的Xbox Live对话框。
