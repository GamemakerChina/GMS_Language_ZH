xboxlive_matchmaking_session_get_users

此函数将使用指定会话中的用户详细信息创建并填充DS映射，如果有错误，则返回-1。您可以从 async_load创建或找到会话时在异步社交事件中返回的DS地图（请参阅 xboxlive_matchmaking_create()更多细节）。返回的DS映射将具有以下键/值对：

  • “ num_results ”-包含会话中的用户数
  • “ userId < index > ”-包含用户的ID（< index >是从0到“ num_results”的值-1）
  • “ userIsOwner < index > ”-如果用户是主机，则包含1，否则为0（< index >是从0到“ num_results”的值-1）

句法：

xboxlive_matchmaking_session_get_users(session_id);

争论描述
session_id要使用的会话ID指针

返回值：

Request ID (integer)

例子：

var session_map = xboxlive_matchmaking_session_get_users(global.SessionID);
if (session_map != -1)
    {
    for (var i = 0; i < session_map[? "num_results"]; i++)
        {
        if (session_map[? "userIsOwner" + string(i)] == 1)
            {
            global.OwnerID = session_map[? "userId" + string(i)];
            }
        }
    }

上面的代码将检索参与配对会话的所有用户的用户数据，并将全局变量设置为会话所有者的ID。
