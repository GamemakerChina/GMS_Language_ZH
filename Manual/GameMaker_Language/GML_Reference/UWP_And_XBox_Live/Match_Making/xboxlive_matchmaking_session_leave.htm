xboxlive_matchmaking_session_leave

此功能使创建或找到指定会话的用户离开该会话。您可以从 async_load创建或找到会话时在异步社交事件中返回的DS地图（请参阅 xboxlive_matchmaking_create()更多细节）。

句法：

xboxlive_matchmaking_session_leave(session_id);

争论描述
session_id要离开的会话的会话ID指针。

返回值：

N/A

例子：

if gamepad_button_check_pressed(0, gp_start)
    {
    xboxlive_matchmaking_session_leave(global.SessionID);
    }

上面的代码检查是否按下了游戏手柄按钮，如果检测到按下按钮，则用户将退出给定的配对会话。
