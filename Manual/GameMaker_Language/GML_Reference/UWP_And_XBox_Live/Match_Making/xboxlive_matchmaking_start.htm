xboxlive_matchmaking_start

必须先调用此函数，然后才能为用户启动任何多人游戏会话。它采用了用户ID指针，您可以通过调用特定的游戏手柄来获取该ID。xboxlive_user_for_pad()功能，并初始化多人游戏API。

句法：

xboxlive_matchmaking_start(user_id);

争论描述
user_id要使用的用户ID指针

返回值：

N/A

例子：

global.UserID = xboxlive_user_for_pad(global.PadIndex);
xboxlive_matchmaking_start(global.UserID);

上面的代码获取给定键盘索引的用户ID，然后初始化多人游戏功能。
