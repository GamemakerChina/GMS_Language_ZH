xboxlive_get_savedata_user

此函数返回当前与文件保存关联的用户ID指针（或常量 pointer_null如果当前未使用用户ID）。看 xboxlive_set_savedata_user()有关更多详细信息。

句法：

xboxlive_get_savedata_user();

返回值：

Pointer or pointer_null

例子：

if (xboxlive_get_savedata_user() != user_id[0])
    {
    xboxlive_set_savedata_user(user_id[0]);
    }

上面的代码检查当前是否已将用户分配为保存目标，如果未分配用户，则将其分配。

返回：保存数据
