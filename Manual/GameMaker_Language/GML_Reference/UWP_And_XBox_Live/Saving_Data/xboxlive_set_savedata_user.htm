xboxlive_set_savedata_user

此函数指定将来在保存游戏区域中操作的文件操作（即，所有文件写入以及从捆绑区域以外的文件中读取文件）将与指定的用户ID指针相关联。可以根据需要多次调用此命令以将保存数据重定向到适当的用户，或者可以使用常量 pointer_null保存到通用计算机的存储区域。

句法：

xboxlive_set_savedata_user(user_id);

争论描述
user_id要保存的用户ID（指针），或 pointer_null

返回值：

N/A

例子：

if (xboxlive_get_savedata_user() != user_id[0])
    {
    xboxlive_set_savedata_user(user_id[0]);
    }

上面的代码检查当前是否已将用户分配为保存目标，如果未分配用户，则将其分配。

返回：保存数据
