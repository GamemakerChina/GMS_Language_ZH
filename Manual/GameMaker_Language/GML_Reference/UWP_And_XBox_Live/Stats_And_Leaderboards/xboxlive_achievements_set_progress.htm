xboxlive_achievements_set_progress

此功能可用于更新成就的进度。您提供了例如通过函数返回的用户标识 xboxlive_get_user()，然后是成就字符串（这是Windows Dev Center中分配的成就ID）。最后，您将进度值设置为0到100之间的一个值。请注意，成就系统将拒绝低于当前进度值的更新。

句法：

xboxlive_achievements_set_progress(user_id, achievement, progress);

争论描述
user_id要为其设置成就的用户的用户ID
achievement要设置的成就（一个字符串）
progress要设置的进度值（从0到100）

返回值：

N/A

扩展示例：

以下是如何使用此功能的扩展示例。首先，您需要在Room Start或Create之类的事件中调用它：

xboxlive_stats_get_social_leaderboard(user_id, "GlobalTime", 20, 1, false, false);

上面的代码将被调用以获取游戏的所有全球排行榜位置的列表，并将生成一个社交异步事件回调，我们将通过以下方式进行处理：

var _progress = (global.Level / global.LevelMax) * 100;
xboxlive_achievements_set_progress(user_id, "Game_Completed", _progress);

上面的代码基于一些全局变量创建一个百分比值，然后使用它来设置“ Game_Completed”成就的成就进度。
