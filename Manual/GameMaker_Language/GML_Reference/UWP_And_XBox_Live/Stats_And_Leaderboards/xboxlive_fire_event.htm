xboxlive_read_player_leaderboard

此功能可用于触发stat事件。 “ event_name”参数是要在游戏的XDP控制台中定义的要触发的事件的名称，以下其他参数也将取决于您为统计设置的内容。如果事件已发送（并且应由服务器接收/处理），则该函数将返回0；如果发生错误（即：由于项目中包含的事件清单文件显示了另一个数字，则未设置您的事件），该函数将返回-1。 。

句法：

xboxlive_fire_event(event_name, [stat_parameters]);

争论描述
event_name要触发的事件的名称。
stat_parameters [可选]要添加到事件的其他参数。

返回值：

String

例子：

var _user = xboxlive_get_user(0);
var _uid = xboxlive_user_id_for_user(_user);
result = xboxlive_fire_event("PlayerSessionStart", _uid, global.guid_str, 0, 42, 42);

上面的代码获取有关用户的一些信息，然后触发stat事件以及此（和其他）数据。
