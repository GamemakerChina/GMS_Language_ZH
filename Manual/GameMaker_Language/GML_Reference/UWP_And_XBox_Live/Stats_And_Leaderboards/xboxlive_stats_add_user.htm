xboxlive_stats_add_user

此功能可用于将给定的用户ID指针添加到统计信息管理器。必须先完成此操作，然后再使用其他任何统计功能将游戏与XBox Live服务器自动同步并获取最新值。您提供该函数返回的用户ID（例如）xboxlive_get_user() ，如果出现错误或用户ID无效，则该函数将返回-1；如果成功调用了该函数，则该函数将返回任何其他值。

该函数将生成一个回调，该回调将触发系统异步事件。此活动将具有特殊的DS地图 async_load然后应解析以下键：

  • “ id ”-将保持不变 achievement_stat_event
  • “ eventname ”-将包含字符串“ LocalUserAdded ”
  • “ userid ”-与请求关联的用户ID
  • “ error”-如果成功，则返回0；如果出现错误，则返回其他值
  • “ errormessage ”-带有错误信息的字符串，如果有的话

句法：

xboxlive_stats_add_user(user_id);

争论描述
user_id要添加的用户ID（指针）

返回值：

Real

例子：

for(var i = 0; i < xboxlive_get_user_count(); ++i;)
    {
    user_id[i] = xboxlive_get_user(i);
    xboxlive_stats_add_user(user_id[i]);
    }

上面的代码将获取可用的用户帐户数量，然后遍历它们，并将帐户ID分配给数组，以及在统计管理器中注册用户。

背面：统计数据和页首横幅
