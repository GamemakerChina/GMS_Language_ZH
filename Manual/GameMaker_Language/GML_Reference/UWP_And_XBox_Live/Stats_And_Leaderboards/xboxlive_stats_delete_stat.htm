xboxlive_stats_delete_stat

此功能可用于从统计信息管理器中删除给定用户ID的统计信息。您提供该函数返回的用户ID（例如）xboxlive_get_user() ，然后删除该统计信息字符串。这将清除统计值并将其从统计管理器中删除，这意味着函数将不再返回该值 xboxlive_stats_get_stat_names()或者 xboxlive_stats_get_stat()。

如果发生错误或用户ID无效，该函数将返回-1；如果成功调用该函数，则返回任何其他值

句法：

xboxlive_stats_delete_stat(user_id, stat);

争论描述
user_id用于删除统计信息的用户ID指针
stat要删除的统计信息（字符串）

返回值：

Real

例子：

for(var i = 0; i < xboxlive_get_user_count(); i++;)
    {
    user_id[i] = xboxlive_get_user(i);
    xboxlive_stats_delete_stat(user_id[i], "HighScore");
    }

上面的代码遍历连接的用户，并从统计管理器中为每个用户删除指定的统计。

背面：统计数据和页首横幅
