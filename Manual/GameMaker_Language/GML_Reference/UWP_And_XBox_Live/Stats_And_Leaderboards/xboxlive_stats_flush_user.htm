xboxlive_stats_flush_user

此功能可用于将给定用户的统计数据从统计信息管理器刷新到实时服务器，以确保该服务器是最新的当前值。根据XBox文档，开发人员应注意不要过于频繁地调用它，因为XBox将对请求进行速率限制，并且XBox OS还将自动大约每5分钟自动刷新一次统计信息。

注意：如果对用户设置的统计信息无效（例如，包含非字母数字字符的统计信息名称），则删除用户可能会返回错误。

要使用该功能，请提供（例如）该功能返回的用户ID xboxlive_get_user() ，以及一个属性值（高优先级将为1，低优先级将为0）。如果出现错误或用户ID无效，该函数将返回-1，如果成功调用该函数，则返回任何其他值。该函数还将生成将触发系统异步事件的回调。此活动将具有特殊的DS地图 async_load然后应解析以下键：

  • “ id ”-将保持不变 achievement_stat_event
  • “ eventname ”-将包含字符串“ StatisticUpdateComplete ”
  • “ userid ”-与请求关联的用户ID
  • “ error”-如果成功，则返回0；如果出现错误，则返回其他值
  • “ errormessage ”-带有错误信息的字符串，如果有的话

请注意，当您从统计信息管理器中删除用户时，统计信息将自动为您刷新。看 xboxlive_stats_remove_user()欲获得更多信息。

句法：

xboxlive_stats_flush_user(user_id, priority);

争论描述
user_id要使用的用户ID（指针）
priority属性值（0或1）

返回值：

Real

例子：

for(var i = 0; i < array_length(user_id); ++i;)
    {
    xboxlive_stats_flush_user(user_id[i], 0);
    }

上面的代码循环遍历存储的用户帐户ID，然后刷新每个用户的状态管理器数据。
