xboxlive_stats_get_stat

此功能可用于从统计信息管理器中检索给定用户的单个统计信息值。您提供该函数返回的用户ID（例如）xboxlive_get_user() ，然后是使用以下其中一个创建时定义的状态字符串 xboxlive_stats_set_stat_*功能。返回值可以是字符串，也可以是实数（取决于要检查的统计信息）或GML常量 undefined如果该统计信息不存在或出现错误。

句法：

xboxlive_stats_get_stat(user_id, stat);

争论描述
user_id用于获取统计信息名称的用户ID指针
stat要设置的统计信息（字符串）

返回值：

Real, string or undefined

例子：

if (game_over == true)
    {
    if (xboxlive_stats_get_stat(p_user_id, "PercentDone") < 100)
        {
        var _val = (global.LevelsFinished / global.LevelsTotal) * 100;
        xboxlive_stats_set_stat_real(p_user_id, "PercentDone", _val);
        }
    }

上面的代码检查一个变量，如果它是 true，那么它将检查“ PercentDone”统计信息的值。如果该值小于100，则将生成一个值，并将stat设置为该值。

背面：统计数据和页首横幅
