xboxlive_stats_get_stat_names

此功能可用于从统计信息管理器中检索给定用户的所有已定义统计信息。您提供该函数返回的用户ID（例如）xboxlive_get_user() ，该函数将返回一个包含用户统计信息的字符串数组。如果发生错误或用户没有统计信息，则仍将返回该数组，但元素计数为0。

句法：

xboxlive_stats_get_stat_names(user_id);

争论描述
user_id用于获取统计信息名称的用户ID指针

返回值：

Array

例子：

var _stat_str = xboxlive_stats_get_stat_names(user_id);
for (var i = 0; i < array_length(_stat_str); ++i;)
    {
    xboxlive_stats_delete_stat(user_id, _stat_str[i]);
    }

上面的代码检索存储在给定变量中的用户ID指针的所有已定义统计信息，然后循环遍历数组，并从统计信息管理器中删除指定的统计信息。
