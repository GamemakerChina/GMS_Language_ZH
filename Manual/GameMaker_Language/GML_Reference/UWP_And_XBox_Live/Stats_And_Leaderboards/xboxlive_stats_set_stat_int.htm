xboxlive_stats_set_stat_int

此功能可用于设置给定用户ID的统计值。您提供该函数返回的用户ID（例如）xboxlive_get_user() ，然后设置状态字符串（如果状态字符串尚不存在，则将创建一个新状态并将其设置为给定值）和一个值（一个整数）以将状态设置为。请注意，统计信息名称只能是字母数字字符，没有符号或空格。

设置统计值时，任何先前的值都将被覆盖，因此您可以通过与当前统计值进行比较来确定是否应更新统计值（即检查高分实际上是否最高）。与新的设置之前。

如果出现错误或用户ID无效，该函数将返回-1，如果成功调用该函数，则返回任何其他值。

句法：

xboxlive_stats_set_stat_int(user_id, stat, value);

争论描述
user_id用于设置统计信息的用户ID指针
stat要设置的统计信息（字符串）
value将统计信息设置为的值（整数）

返回值：

Real

例子：

var _val = xboxlive_stats_get_stat(p_user_id, "GamesPlayed");
xboxlive_stats_set_stat_int(p_user_id, "GamesPlayed", ++_val);

上面的代码检索统计信息“ GamesPlayed”的值，然后将相同的统计值设置为该值加1。
