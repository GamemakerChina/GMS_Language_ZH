xboxlive_stats_set_stat_real

此功能可用于设置给定用户ID的统计值。您提供该函数返回的用户ID（例如）xboxlive_get_user() ，然后设置要设置的stat字符串（如果还不存在stat字符串，则将创建一个新的stat并将其设置为给定值）和一个值（一个实数）以将stat设置为。请注意，统计信息名称只能是字母数字字符，没有符号或空格。

设置统计值时，任何先前的值都将被覆盖，因此您可以通过与当前统计值进行比较来确定是否应更新统计值（即检查高分实际上是否最高）。与新的设置之前。

如果出现错误或用户ID无效，该函数将返回-1，如果成功调用该函数，则返回任何其他值。

句法：

xboxlive_stats_set_stat_real(user_id, stat, value);

争论描述
user_id用于设置统计信息的用户ID指针
stat要设置的统计信息（字符串）
value将统计信息设置为（实数）的值

返回值：

Real

例子：

if (game_over == true)
    {
    if (xboxlive_stats_get_stat(p_user_id, "PercentDone") < 100)
        {
        var _val = (global.LevelsFinished / global.LevelsTotal) * 100;
        xboxlive_stats_set_stat_real(p_user_id, "PercentDone", _val);
        }
    }

上面的代码检查一个变量，如果它是 true，那么它将检查“ PercentDone”统计信息的值。如果该值小于100，则将生成一个值，并将stat设置为该值。

背面：统计数据和页首横幅
