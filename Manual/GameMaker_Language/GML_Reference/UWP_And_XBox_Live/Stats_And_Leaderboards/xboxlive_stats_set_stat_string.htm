xboxlive_stats_set_stat_string

此功能可用于将统计信息设置为给定用户ID的字符串。您提供该函数返回的用户ID（例如）xboxlive_get_user() ，然后设置状态字符串（如果该状态字符串尚不存在，则将创建一个新状态并将其设置为给定值），然后再设置该字符串。请注意，统计信息名称只能是字母数字字符，没有符号或空格，但是您设置的字符串值可以有空格和特殊字符。

设置统计字符串时，任何先前的字符串都将被覆盖，因此，由您决定是否应更新统计字符串，方法是在设置之前将其与新的统计字符串进行比较。

如果出现错误或用户ID无效，该函数将返回-1，如果成功调用该函数，则返回任何其他值。

句法：

xboxlive_stats_set_stat_string(user_id, stat, value);

争论描述
user_id用于设置统计信息的用户ID指针
stat要设置的统计信息（字符串）
value将统计信息设置为（字符串）的值

返回值：

Real

例子：

xboxlive_stats_set_stat_string(p_user_id, "LevelName", global.LevelName);

上面的代码将状态“ LevelName”设置为存储在给定全局变量中的字符串。
