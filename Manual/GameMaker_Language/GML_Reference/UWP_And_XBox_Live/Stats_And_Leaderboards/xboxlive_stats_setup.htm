xboxlive_stats_setup

您必须先调用此函数，然后才能使用任何其他Xbox Live stat函数，并且只需在系统上初始化所需的库即可。 “ user_id”参数是函数返回的原始用户ID xboxlive_get_user() ，而“ service_config”和“ title_id”是Xbox Live Dev Center上游戏的唯一ID。

句法：

xboxlive_stats_setup(user_id, service_config_id, title_id);

争论描述
user_id要检查的用户的ID指针。
service_config_id这是游戏的config_id字符串。
title_id Xbox Dev Center上您的游戏的唯一ID。

返回值：

N/A

例子：

var user = xboxlive_get_user(0); xboxlive_stats_setup(user, "4d61a1aa-61ac-4541-badd-31f91244fea6", $1244FEA6);

上面的代码初始化给定用户ID的统计信息系统。
