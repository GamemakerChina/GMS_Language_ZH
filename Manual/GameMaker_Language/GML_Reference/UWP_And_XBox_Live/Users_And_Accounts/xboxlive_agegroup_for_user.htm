xboxlive_agegroup_for_user

此函数将返回四个常量之一（如下所示），以指示分配给指定用户ID指针的年龄范围。

句法：

xboxlive_agegroup_for_user(user_id);

争论描述
user_id要检查的用户ID（指针）

返回值：

Constant

持续的描述
xboxlive_agegroup_unknown用户的年龄段未知
xboxlive_agegroup_child用户年龄段介于8到12（含）之间
xboxlive_agegroup_teen用户年龄段介于13到17岁之间（包括韩国），介于13到19岁之间
xboxlive_agegroup_adult用户年龄段为18岁以上（或韩国为20岁以上）

例子：

var _age = xboxlive_agegroup_for_user(xboxlive_get_activating_user());
if _age != xboxlive_agegroup_adult
    {
    global.ContentControl = true;
    }
else global.ContentControl = false;

上面的代码检查了激活用户的年龄段，并将全局变量设置为 true或者 false取决于返回的常数。

返回：用户和帐户
