xboxlive_get_activating_user

使用此功能，您可以从控制台检索启动游戏的用户的用户ID指针。

注意：如何使用此指针将取决于您要在游戏中做什么，但您不应依赖它作为当前正在播放的用户ID，因为激活用户可以在游戏运行时注销。

句法：

xboxlive_get_activating_user();

返回值：

Pointer

例子：

global.activUser = xboxlive_get_activating_user();

上面将激活用户ID指针存储在全局变量中

返回：用户和帐户
