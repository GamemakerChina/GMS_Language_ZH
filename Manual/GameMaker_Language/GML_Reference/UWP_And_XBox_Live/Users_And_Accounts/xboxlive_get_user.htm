xboxlive_get_user

使用此功能，您可以检索被索引用户的用户ID指针。如果用户不存在，该函数将返回常量 pointer_null反而。您可以使用该功能找到当前登录的用户数 xboxlive_get_user_count()。

重要说明：此功能仅应与 xboxlive_get_user_count()-请勿在游戏手柄上使用。使用 xboxlive_user_for_pad()反而。

句法：

xboxlive_get_user(index);

争论描述
index从中获取用户ID的索引（整数）。

返回值：

Pointer

例子：

for(var i = 0; i < xboxlive_get_user_count(); ++i;)
    {
    user_id[i] = xboxlive_get_user(i);
    }

上面的代码循环遍历所有已登录的用户，并将其唯一的ID指针存储在数组中。

返回：用户和帐户
