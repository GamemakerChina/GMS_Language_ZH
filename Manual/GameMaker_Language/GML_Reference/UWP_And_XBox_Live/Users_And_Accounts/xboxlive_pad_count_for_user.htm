xboxlive_pad_count_for_user

使用此功能，您可以找出当前用户有多少个打击板“插槽”处于活动状态（请参阅 xboxlive_pad_for_user()了解更多详情）。

句法：

xboxlive_pad_count_for_user(user_id, slot);

争论描述
user_id要检查的用户ID（指针）

返回值：

Integer

例子：

global.slots[0] = xboxlive_pad_count_for_user(user_id[0])

上面的代码存储与给定的用户ID指针关联的打击垫的数量。

返回：用户和帐户
