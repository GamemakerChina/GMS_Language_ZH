盒子里的小键盘

Xbox用户可以为他们的用户ID分配各种垫子，每一个垫子都充满一个 "槽"。这些槽从0开始编号，可以容纳任何垫子的索引，所以槽0可以分配给垫子3，槽1可以分配给垫子2。这个函数 可以用来获取分配给任何给定槽位的垫子，如果用户在一定时间内没有输入游戏板，它就会自动关闭，所以你可以用这个函数检查任何给定的用户ID 以确保他们有一个激活的游戏板，如果他们没有，就可以通过（例如）暂停游戏做出相应的反应。如果游戏板处于活动状态，它的索引值将被返回，但如果不是，该函数将返回-1。

请注意，如果任何槽中的垫子不活动，后面的槽会 "滑 "到空出的位置，所以如果你有三个活动的垫子分配给槽0、1和2，而槽0的垫子不活动，槽1和槽2现在会变成槽0和槽1。 1，所以槽0应该总是包含一个活动的垫子，除非所有的垫子都关闭了（在这种情况下，它将返回-1）。

语法。

xboxlive_pad_for_user(user_id, slot);

争论 描述
user_id 要检查的用户ID（一个指针）。
user_id 要检查的游戏板插槽的索引（一个整数）。

返回。

指针

例子。

if (xboxlive_pad_for_user(user_id, 0) == -1)
{
for(var i = 0; i < gamepad_get_device_count(); i++;)
{
var u_id = xboxlive_user_for_pad(i);
if (u_id == -1)
{
async_ID = xboxlive_pad_for_user(i, u_id);
}
}
}

上面的代码检查给定用户ID指针的0号槽，看是否有游戏板被分配，如果没有则分配一个。

返回。用户和账户
