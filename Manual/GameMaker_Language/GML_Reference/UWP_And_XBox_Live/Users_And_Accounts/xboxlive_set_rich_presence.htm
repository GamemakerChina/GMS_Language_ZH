xboxlive_set_rich_presence

此函数将为给定的用户设置丰富状态字符串。丰富状态字符串显示用户正在玩的游戏名称之后用户的游戏活动，并用连字符分隔。此字符串显示在“朋友和俱乐部”列表中的玩家的Gamertag下以及玩家的Xbox Live用户配置文件中。

使用此功能时，您需要为用户提供用户ID指针，然后可以将用户标记为当前在游戏中处于活动状态（使用 true/false ）。如果设置为 false，则丰富状态字符串将附加“ / afk”或适当的内容。下一个参数是要显示的丰富状态字符串ID，最后您可以（可选）提供一个 service_config_id细绳。请注意，这是一个可选参数，因为如果您调用了 xboxlive_stats_setup()您不需要通过 service_config_id这里。

有关丰富状态以及如何设置在合作伙伴中心使用的字符串的更多信息，请参见以下Microsoft文档：

  • 状态字符串概述

句法：

xboxlive_set_rich_presence(user_id, is_user_active, rich_presence_string, [service_config_id])

争论描述
user_id要检查的用户的ID指针。
is_user_active标记用户为活动状态。
rich_presence_string要使用的丰富状态字符串ID（在合作伙伴中心中定义-最多50个字符）
service_config_id [可选]这是游戏的config_id字符串。

返回值：

N/A

例子：

var pad_uid = xboxlive_user_for_pad(0);
xboxlive_set_rich_presence(pad_uid, true, "Playing_Challenge");

上面的代码获取分配给游戏手柄[0]插槽的用户的用户ID指针，然后为该用户设置丰富状态字符串。

返回：用户和帐户
