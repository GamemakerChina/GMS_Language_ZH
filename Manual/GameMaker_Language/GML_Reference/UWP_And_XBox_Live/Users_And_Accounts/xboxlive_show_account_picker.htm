xboxlive_show_account_picker

该函数将打开XBox账户选择器屏幕，如果用户还没有登录，他们可以登录（如果他们已经登录，那么该函数将不做任何事情）。注意，这个功能只适用于在PC上使用UWP构建，而不是Xbox。 因为要在Xbox上启动一个游戏，用户必须已经登录了。

语法。

xboxlive_show_account_picker()。

返回。

不适用

例子。

if !xboxlive_user_is_signed_in()
{
if !xboxlive_user_is_signing_in()
{
xboxlive_show_account_picker();
}
}
else
{
global.GamerTag = xboxlive_gamertag_for_user();
}

上面的代码检查用户是否登录了XBox Live，如果他们登录了，代码将获得用户的gamertag并存储在一个全局变量中供将来使用。如果他们没有登录，那么代码会检查他们是否正在 在签入的过程中，在这种情况下不会发生进一步的事情，如果他们没有，那么它将打开账户选择器，提示他们签入。
