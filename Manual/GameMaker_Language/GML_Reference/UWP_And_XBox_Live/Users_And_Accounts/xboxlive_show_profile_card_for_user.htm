xboxlive_show_profile_card_for_user

使用此功能，您可以请求XBox显示目标用户ID指针的配置文件卡。该功能既需要请求信息的用户的用户ID指针，也要以用户的用户ID指针为目标并获取其个人资料卡。

句法：

xboxlive_show_profile_card_for_user(requesting_user_id, target_user_id);

争论描述
requesting_user_id请求用户的用户ID（指针）
target_user_id要获取的个人资料卡的用户的用户ID（指针）

返回值：

N/A

例子：

if (gamepad(0, gp_start))
    {
    xboxlive_show_profile_card_for_user(user[0], user[1]);
    }

上面的代码检查是否按下了游戏板按钮，如果检测到按下按钮，则显示给定用户的个人资料卡。

返回：用户和帐户
