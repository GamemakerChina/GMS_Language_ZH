淘宝网上卖的是什么？

该函数检索给定用户赞助人的ID。

语法。

xboxlive_sponsor_for_user（user_id）。

争论 描述
user_id 要检查的用户的ID指针。

返回。

真实的

例子。

var _uid = xboxlive_user_for_pad(0);
sponsor = xboxlive_sponsor_for_user(_uid);

上面的代码获得分配给游戏手柄[0]槽的用户ID指针，然后使用该ID检索用户的赞助商ID。

返回。用户和账户
