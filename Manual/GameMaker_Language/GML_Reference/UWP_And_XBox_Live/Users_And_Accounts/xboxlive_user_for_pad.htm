xboxlive_user_for_pad

此函数将返回与给定游戏手柄索引值关联的用户ID指针，或者 pointer_null如果没有用户与之关联。请注意，此功能只能与游戏手柄索引一起使用，而不能与 xboxlive_get_user_count()。

句法：

xboxlive_user_for_pad(pad_index);

争论描述
pad_index要检查的游戏手柄插槽的索引（整数）

返回值：

Pointer

例子：

for(var i = 0; i < 11; ++i;)
    {
    var u_id = xboxlive_user_for_pad(i);
    if u_id != pointer_null
        {
        pad_name[i] = xboxlive_gamedisplayname_for_user(u_id);
        }
    else
        {
        pad_name[i] = "Press Play";
        }
    }

上面的代码遍历所有可用的填充插槽，并检查是否有与之关联的用户ID指针。如果存在，则使用用户显示名称设置一个数组，否则将该数组设置为某些默认文本。

返回：用户和帐户
