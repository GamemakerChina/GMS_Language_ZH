访客是谁？

如果给定的用户ID指针是一个访客用户，这个函数将返回true，如果不是，则返回false。你可以通过以下函数获得一个用户ID指针 xboxlive_get_user().

语法。

xboxlive_user_is_guest(user_id)。

争论 描述
user_id 要检查的用户的ID指针。

返回。

布尔型

例子。

if xboxlive_user_is_guest(user_id[0])
{
global.user_name[0] = "GUEST";
}
else
{
global.user_name[0] = xboxlive_gamedisplayname_for_user(user_id[0]) ;
}

上述内容将激活的用户ID指针存储在一个全局变量中。

返回。用户和账户
