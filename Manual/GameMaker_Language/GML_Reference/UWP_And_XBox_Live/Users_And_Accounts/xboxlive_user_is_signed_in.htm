xboxlive_user_is_signed_in

该功能将检查用户是否已登录并返回 true如果有和 false否则。如果没有用户登录，则可以调用该函数 xboxlive_show_account_picker()打开帐户选择器并提示他们登录。

句法：

xboxlive_user_is_signed_in();

返回值：

Boolean

例子：

if !xboxlive_user_is_signed_in()
    {
    if !xboxlive_user_is_signing_in()
        {
        xboxlive_show_account_picker();
        }
    }
else
    {
    global.GamerTag = xboxlive_gamertag_for_user();
    }

上面的代码检查用户是否已登录XBox Live，以及是否已登录，该代码将获取用户的gamertag并将其存储在全局变量中以备将来使用。如果未登录，则代码将检查它们是否正在登录，在这种情况下，将不会再发生任何事情；如果未登录，则它将打开帐户选择器，提示他们进行登录。 -在。

返回：用户和帐户
