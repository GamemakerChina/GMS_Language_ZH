uwp_is_constrained

该函数将返回 true如果游戏在资源受限模式下运行，或者 false如果正常运行。当游戏没有焦点时，可以激活资源受限模式，并且在这种模式下，游戏减少了GPU资源。对此的响应通常应该是暂停游戏（因为用户不再拥有控制权）。同样，如果在此模式下观察到速度变慢，则禁用任何特别的税收影响可能是一个好主意。请注意，与 uwp_is_suspending()函数，此函数将返回 true只要游戏处于受限模式（并且在第一次发生时不只是一次）就可以连续进行。

句法：

uwp_is_constrained();

返回值：

Boolean

例子：

if xboxliveis_constrained()
    {
    if !instance_exists(obj_PauseMenu)
        {
        instance_create(0,0,obj_PauseMenu);
        }
    }

上面的代码检查该应用程序是否受约束以及是否受约束，它创建了一个对象实例（用于暂停游戏）。

返回： UWP和XBox Live
