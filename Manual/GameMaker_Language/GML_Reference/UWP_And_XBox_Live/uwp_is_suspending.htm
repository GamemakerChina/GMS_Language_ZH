uwp_is_suspending

该函数将返回 true单帧（和 false否则）以类似于 os_is_paused()。然后，游戏可能有一秒钟的时间来保存游戏状态，然后系统可能会终止该状态。游戏完成数据保存后，应调用 uwp_suspend()干净地悬挂。游戏可能还应该进入暂停状态，这样，如果系统将其恢复（如果尚未完全终止），则玩家不会直接回到游戏状态。

句法：

uwp_is_suspending();

返回值：

Boolean

例子：

if uwp_is_suspending()
    {
    scr_Save_Game_Data();
    uwp_suspend();
    }

上面的代码检查应用程序是否将要暂停，以及是否在暂停游戏之前调用脚本来保存游戏数据。
