uwp_license_trial_user

该函数将返回当前正在使用其试用许可证的用户ID指针，否则它将返回 pointer_null如果用户当前未处于活动状态，或者游戏当前未处于试用模式。

句法：

uwp_license_trial_user();

返回值：

Pointer

例子：

if gamepad_button_check_pressed(0, gp_start)
    {
    if uwp_license_trial_version()
        {
        var _uid = uwp_license_trial_user();
        uwp_show_help(_uid);
        }
    }

上面的代码检查是否按下了按钮，然后检查应用程序是否正在运行并具有试用许可证。如果是，则它将获取许可证的用户ID并打开该用户的帮助文件。

返回： UWP和XBox Live
