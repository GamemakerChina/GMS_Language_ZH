uwp_license_trial_version

此功能可用于检查游戏是否已获得试用许可证。如果是，函数将返回 true， 或者 false否则。

句法：

uwp_license_trial_version();

返回值：

Boolean

例子：

if uwp_license_trial_version()
    {
    global.LevelCap = 10;
    }

上面的代码检查该应用是否受试用许可，如果有，则设置一个全局变量。

返回： UWP和XBox Live
