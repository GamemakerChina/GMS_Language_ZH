uwp_show_help

此功能将启动Xbox One帮助应用程序，该应用程序将依次显示游戏的帮助文件（该文件是您先前上传到XBox开发人员门户的HTML格式文件）。请注意，在调用此函数时指定了用户ID，以便在显示该用户的语言环境信息时可以使用该用户的语言环境信息。您可以使用以下功能检索用户ID xboxlive_get_user() ，xboxlive_get_activating_user() ， 或者 uwp_license_trial_user()。

句法：

uwp_show_help(user_id);

争论描述
user_id用于为其打开帮助文件的用户ID指针。

返回值：

Boolean

例子：

if gamepad_button_check_pressed(0, gp_start)
    {
    if uwp_license_trial_version()
        {
        var _uid = uwp_license_trial_user;
        uwp_show_help(_uid);
        }
    }

上面的代码检查是否按下了按钮，然后检查应用程序是否正在运行并具有试用许可证。如果是，则它将获取许可证的用户ID并打开该用户的帮助文件。

返回： UWP和XBox Live
