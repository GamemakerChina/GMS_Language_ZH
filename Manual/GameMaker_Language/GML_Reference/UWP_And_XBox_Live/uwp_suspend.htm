uwp_suspend

此功能向系统指示该应用在检测到暂停请求后已完成保存数据。 Xbox One提交需要调用此函数，并且应始终与该函数结合使用 uwp_is_suspending()，如下例所示。

句法：

uwp_suspend();

返回值：

N/A

例子：

if uwp_is_suspending()
    {
    scr_Save_Game_Data();
    uwp_suspend();
    }

上面的代码检查应用程序是否将要暂停，以及是否在暂停游戏之前调用脚本来保存游戏数据。

返回： UWP和XBox Live
