uwp_was_closed_by_user

通过这个函数，你可以发现游戏在玩的时候是否被用户关闭过（也就是说，它被正确地终止，而不是被系统终止）。如果是的话，该函数将返回真，否则将返回假。

语法。

uwp_was_closed_by_user()。

返回。

布尔型

例子。

如果 uwp_was_closed_by_user()
{
global.GameLoaded = false;
}

上面的代码检查应用程序是否以前被用户终止过，如果是，它将设置一个全局变量。
