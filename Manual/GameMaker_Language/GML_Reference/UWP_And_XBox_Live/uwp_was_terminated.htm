uwp_was_terminated

使用此功能，您可以确定游戏之前是否已被系统关闭（即不是游戏的第一次运行）。如果已经存在，则该函数将返回 true，否则它将返回 false。使用的一个示例是检查它，并查看游戏是否先前已被系统终止，然后（如果返回 true）以加载玩家在游戏中所处的上一个阶段。如果针对XBox One则在不同版本的认证文档之间，如果检测到游戏所需的行为将有所不同，因此值得检查最新版本以了解游戏应该做什么。

句法：

uwp_was_terminated();

返回值：

Boolean

例子：

if xboxlivewas_terminated()
    {
    buffer_load("Player_Save.sav");
    global.GameLoaded = true;
    }

上面的代码检查该应用程序是否先前已被系统终止，如果已终止，则它将加载缓冲区并设置全局变量。
