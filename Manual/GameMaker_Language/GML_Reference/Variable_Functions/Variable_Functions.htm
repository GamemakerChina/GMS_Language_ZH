可变功能

此页面上的功能旨在在游戏中GameMaker Language时处理所有不同的变量和变量类型。有关变量的更多信息，请参见“变量和变量范围”页面。

以下变量功能可用：

  • variable_instance_exists
  • variable_instance_get_names
  • variable_instance_names_count
  • variable_instance_get
  • variable_instance_set
  • variable_global_exists
  • variable_global_get
  • variable_global_set
  • variable_struct_exists
  • variable_struct_get
  • variable_struct_set
  • variable_struct_remove
  • variable_struct_get_names
  • variable_struct_names_count
  • instanceof

数组的使用也有一些功能。这些设计旨在使您的代码更具灵活性，并允许您在制作游戏时创建更多功能和动态数组。这些功能是：

  • array_create
  • array_copy
  • array_equals
  • array_get
  • array_set
  • array_push
  • array_pop
  • array_insert
  • array_delete
  • array_sort
  • array_length
  • array_resize
  • array_length_1d（已弃用）
  • array_length_2d（已弃用）
  • array_height_2d（已弃用）

接下来的这些功能都与使用方法变量有关：

  • method
  • method_get_self
  • method_get_index

最后，您还具有确定变量类型的以下功能（有关数据类型的更多信息，请参见此处，有关类型表，请参见此处）：

  • is_string
  • is_real
  • is_numeric
  • is_bool
  • is_array
  • is_struct
  • is_method
  • is_ptr
  • is_int32
  • is_int64
  • is_vec3
  • is_vec4
  • is_matrix
  • is_undefined
  • is_nan
  • is_infinity
  • typeof
  • bool
  • ptr
  • int64
