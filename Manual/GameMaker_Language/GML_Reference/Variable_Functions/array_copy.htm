array_copy

使用此功能，您可以将阵列的全部或部分复制到任何位置的另一个阵列中。您需要同时提供源阵列和目标阵列（都需要先前创建），以及源阵列中要复制的位置和目标阵列中要复制到的位置。最后，您需要指定要复制的数组的长度（或所需部分的长度）。如果要复制的数据超出了目标数组的长度，则该数组将被扩展以接受数据。

此函数也可用于多维数组，只要您按照以下模式指定在提供数组索引时要复制的维度：

// Copy from the first dimension of an array
// to the second dimension of an array
array_copy(item_array, 0, inventory_array[0], 0, len);

// Copy from the third dimension of an array
// to the first dimension of an array
array_copy(item_array[0][0], 0, inventory_array, 0, len);

// etc...

句法：

array_copy(dest, dest_index, src, src_index, length);

争论描述
dest要复制到的阵列的ID。
dest_index数组中要复制到的索引。
src要复制的阵列的ID。
src_index包含要开始复制的数组的索引。
length要复制的长度（数组索引的数量）。

返回值：

N/A

例子：

if !array_equals(inventory_array, item_array)
   {
   var len = array_length(inventory_array);
   array_copy(item_array, 0, inventory_array, 0, len);
   }

上面的代码将检查两个数组，看它们是否具有相等的值，如果不相等，则代码会将数组“ inventory_array”的整个内容复制到数组“ item_array”。

返回：变量函数
