array_create

使用此功能，您可以创建给定大小的一维数组。您告诉函数要创建的数组的长度，它将返回数组的“句柄”，然后可以将其分配给变量。除非您指定（可选）初始化值，否则以此方式创建的数组会将每个条目初始化为0。如果确实为初始化数组提供了额外的值，则新数组中的所有索引都将设置为该值而不是0，但是请注意，在这种情况下，该函数的性能开销会更大。

句法：

array_create(size, [value]);

争论描述
size要创建的数组的大小。
value用于初始化所有数组索引的值（可选）。

返回值：

Array Reference

例子：

instance_array = array_create(100, noone);

上面的代码将创建一个包含100个条目的新数组，每个条目都设置为关键字 noone，然后将其分配给变量“ instance_array”。

返回：变量函数
