array_delete

使用此功能，您可以在任何给定位置从数组中删除一个或多个值。该函数要求您提供一个变量，该变量保存数组，要从中删除的数组中的索引（位置）以及要删除的值数。

句法：

array_delete(variable, index, number);

争论描述
variable保存数组的变量。
index数组中要从中删除值的索引（位置）。
number要删除的值数。

返回值：

N/A

例子：

array_delete(score_array, 0, 10);

上面的代码将删除给定数组中从索引0到索引9的所有值。

返回：变量函数
