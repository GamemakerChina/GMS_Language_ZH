array_equals

使用此功能，您可以检查两个数组是否相等（相等或相同）。您提供两个数组进行检查，函数将返回 true如果它们相等（相等或相同）或 false如果不是。请注意，这与使用==检查两个数组是否相同是不一样的，后者将不检查两个数组是否具有相等的值，而仅检查两个数组是否引用相同的初始数组。例如：

var a = [1,2,3,4];
var b = [1,2,3,4];
if (a == b)
   {
   show_debug_message( "This will never fire, as a and b do not reference the SAME array" );
   }
if (array_equals(a, b))
   {
   show_debug_message( "This will fire now" );
   }

句法：

array_equals(var1, var2);

争论描述
var1第一个数组的索引。
var2第二个数组的索引。

返回值：

Boolean

例子：

if !array_equals(inventory_array, item_array)
   {
   var len = array_length(inventory_array);
   array_copy(item_array, 0, inventory_array, 0, len);
   }

上面的代码将检查两个数组，看它们是否具有相等的值，如果不相等，则代码会将数组“ inventory_array”的整个内容复制到数组“ item_array”。

返回：变量函数
