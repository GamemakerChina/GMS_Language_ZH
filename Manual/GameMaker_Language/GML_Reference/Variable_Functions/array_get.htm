array_get

通过这个函数，你可以从一个数组中的一个索引中获取数值。该函数要求你提供一个存放数组的变量和从该数组中获取数值的索引。注意，如果给定的数组索引超出了范围 那么游戏将崩溃并出现错误。这个函数也可以用于多维数组，只要你在提供数组索引时指定你想得到的维度，按照这个模式。

// 一维数组
array_get(my_array, 0);
// 二维数组
array_get(my_array[0], 0);
// 三维数组
array_get(my_array[0][0], 0);
// 等等。

语法。

array_get(variable, index)。

争论 描述
变量 保存数组的变量。
指数 从数组中获取数值的索引。

返回。

任何数组可以容纳的有效数据类型。

例子。

for (var i = 0; i < 10; ++i;)
{
show_debug_message(array_get(my_array, i));
}

上面的代码将把给定的前10项输出到控制台。

返回。可变的功能
