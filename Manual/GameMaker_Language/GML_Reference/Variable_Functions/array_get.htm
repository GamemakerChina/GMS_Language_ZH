array_get

使用此功能，您可以从数组的索引中检索值。该函数要求您提供一个包含数组和索引的变量，以从该数组中获取值。请注意，如果给定的数组索引超出范围，则游戏将因错误而崩溃。此函数还可以用于多维数组，只要您按照以下模式指定在提供数组索引时要获取的维数即可：

// 1D array
array_get(my_array, 0);
// 2D array
array_get(my_array[0], 0);
// 3D array
array_get(my_array[0][0], 0);
// etc...

句法：

array_get(variable, index);

争论描述
variable保存数组的变量。
index要从中获取值的数组的索引。

返回值：

Any valid data type that an array can hold.

例子：

for (var i = 0; i < 10; ++i;)
    {
    show_debug_message(array_get(my_array, i));
    }

上面的代码会将给定的前10个项目输出到控制台。

返回：变量函数
