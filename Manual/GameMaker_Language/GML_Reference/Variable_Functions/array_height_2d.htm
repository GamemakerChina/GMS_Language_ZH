array_height_2d（已弃用）

警告！不推荐使用此功能，因为数组不再仅限于1或2维，因此仅提供此功能用于支持旧项目。所有新项目都应使用当前的方式来创建和访问多维数组（有关更多信息，请参见此处）。

使用此功能，您可以获取2D数组第一维的高度（条目数）。您提供要检查的数组，函数的输出将告诉您它包含多少个初始条目。您可以使用函数获取数组第二维的条目数 array_length_2d。

句法：

array_height_2d(array_index);

争论描述
array_index要检查的数组的索引。

返回值：

Real

例子：

for (var i = 0; i < array_height_2d(a); ++i;)
   {
   for (var j = 0; j < array_length_2d(a, i); ++j;)
      {
      a[i, j] = -1;
      }
   }

上面的代码将遍历2D数组并将每个条目设置为-1。

返回：变量函数
