array_height_2d(deprecated)

警告!由于数组不再局限于1或2维，该函数已被废弃，因此该函数仅用于支持传统项目。所有的新项目应该使用当前的方式来创建和访问多维数组。 数组的方式（更多信息请看这里）。

通过这个函数你可以得到一个二维数组的第一维的高度（条目数）。你提供要检查的数组，该函数的输出告诉你它包含多少个初始条目。你可以用 使用函数获得数组第二维的条目数 array_length_2d.

语法。

array_height_2d(array_index);

争论 描述
阵列_索引 要检查的数组的索引。

返回。

真实的

例子。

for (var i = 0; i < array_height_2d(a); ++i;)
{
for (var j = 0; j < array_length_2d(a, i); ++j;)
{
a[i, j] = -1;
}
}

上面的代码将循环浏览一个二维数组，并将每个条目设置为-1。

返回。可变的功能
