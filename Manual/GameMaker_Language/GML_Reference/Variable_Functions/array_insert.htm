array_insert

通过这个函数，你可以在一个数组的任何给定位置插入一个（或多个）值。这个函数要求你提供一个存放数组的变量，要插入的数组中的索引（位置），以及至少一个要插入的值。 插入，尽管你可以选择提供更多的参数，它们都将从给定的索引开始按顺序插入数组中。

注意，如果给定的索引在数组的长度之外，这些值将被添加到给定的索引处，数组的最后一个值和插入的值之间的任何空槽将被设置为默认值0。

语法。

array_insert(variable, index, value, [value], [value], [etc...]) 。

争论 描述
变量 保存数组的变量。
指数 在数组中插入值的索引（位置）。
价值 要插入的值
[价值]，[价值]，[等...] 可选）要插入到数组中的其他值

返回。

不适用

例子。

array_insert(score_array, score_rank, current_score)。

上面的代码将把当前的分数插入score数组，在score_rank变量所定义的索引处。

返回。可变的功能
