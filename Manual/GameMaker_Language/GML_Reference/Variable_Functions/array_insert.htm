array_insert

使用此功能，您可以在数组的任何给定位置插入一个或多个值。该函数要求您提供一个变量，用于保存数组，数组中要插入的索引（位置）以及至少一个要插入的值，尽管您可以选择提供其他参数，并且它们都将插入到从给定索引开始按连续顺序排列数组。请注意，如果给定索引在数组长度之外，则将简单地将值添加到给定索引，并且数组中最后一个值和插入值之间的任何“空格”将设置为默认值0。

句法：

array_insert(variable, index, value, [value], [value], [etc...]);

争论描述
variable保存数组的变量。
index数组中要插入值的索引（位置）
value要插入的值
[value], [value], [etc...]选修的！进一步的值将被推到数组上

返回值：

N/A

例子：

array_insert(score_array, score_rank, current_core);

上面的代码将在变量“ score_rank”中保留的索引处将值插入给定数组。

返回：变量函数
