array_length_1d（已弃用）

警告！不推荐使用此功能，因为数组不再仅限于1或2维，因此仅提供此功能用于支持旧项目。所有新项目都应使用当前的方式来创建和访问多维数组（有关更多信息，请参见此处）。

使用此功能，您可以获取一维数组的长度（条目数）。对于2D阵列，您应该使用 array_height_2d和 array_length_2d功能。

重要提示：如果数组有32,000个条目，则此函数将返回错误值，因此不应使用。

句法：

array_length_1d(array_index);

争论描述
array_index要检查的数组的索引。

返回值：

Real

例子：

for (var i = array_length_1d(a) - 1; i > -1; i--;)
   {
   a[i] = -1;
   }

上面的代码将遍历数组并将每个条目设置为-1。

返回：变量函数
