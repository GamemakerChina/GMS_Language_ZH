array_length_2d(已弃用)

警告!由于数组不再局限于1或2维，该函数已被废弃，因此该函数仅用于支持传统项目。所有的新项目应该使用当前的方式来创建和访问多维数组。 数组的方式（更多信息请看这里）。

通过这个函数你可以得到一个数组的第二维的长度（条目数）。你提供第一维的条目数，该函数将返回数组的第二维条目数。 第一维的长度，请使用函数 array_height_2D()).如果给定的变量不是一个数组，函数将返回0；如果变量是一个一维数组（因为仍然有1行），函数将返回1。

语法。

array_length_2d(array_index, n);

争论 描述
阵列_索引 要检查的数组的索引。
n 数组的条目，以获得其长度。

返回。

真实的

例子。

for (var i = 0; i < array_height_2d(a); ++i;)
{
for (var j = 0; j < array_length_2d(a, i); ++j;)
{
a[i, j] = -1;
}
}

上面的代码将循环浏览一个二维数组，并将每个条目设置为-1。

返回。可变的功能
