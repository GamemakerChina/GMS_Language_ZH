array_pop

通过这个函数，你可以从一个数组的末端检索一个值，而不需要知道数组的长度。你提供存放数组的变量，该函数将返回数组中最后一项的内容。如果该数组是 空，该函数将返回 undefined 。

语法。

array_pop(变量)。

争论 描述
变量 保存数组的变量。

返回。

值（任何数据类型）或未定义

例子。

var _lastscore = array_pop(score_array);
draw_text(32, 32, "Last Score = " + string(_lastscore));

上面的代码将从给定的数组中弹出最后一个值，然后把它和一些文字一起画到屏幕上。

返回。可变的功能
