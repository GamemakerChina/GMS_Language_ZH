array_pop

使用此功能，您可以从数组的末尾检索值，而不必知道数组的长度。您提供保存数组的变量，该函数将返回数组中最后一个条目的内容。如果数组为空，函数将返回 undefined。

句法：

array_pop(variable);

争论描述
variable保存数组的变量。

返回值：

Value (any data type) or undefined

例子：

var _lastscore = array_pop(score_array);
draw_text(32, 32, "Last Score = " + string(_lastscore));

上面的代码将从给定数组中弹出最后一个值，然后将其与一些文本一起绘制到屏幕上。

返回：变量函数
