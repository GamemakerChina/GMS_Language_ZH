array_push

使用此功能，您可以将一个（或多个）值推入数组的末尾而不必知道数组的长度。该函数要求您提供一个变量，该变量保存数组以及至少一个要推送的值，尽管您可以选择提供其他参数，并且它们都将以连续顺序推送到数组中。

句法：

array_push(variable, value, [value], [value], [etc...]);

争论描述
variable保存数组的变量。
value要推到数组末尾的值
[value], [value], [etc...]选修的！进一步的值将被推到数组上

返回值：

N/A

例子：

array_push(score_array, obj_Player1.scr, obj_Player2.scr, obj_Player3.scr, obj_Player4.scr);

上面的代码会将四个值压入给定数组的末尾。

返回：变量函数
