array_resize

使用此功能，您可以将现有数组尺寸调整为新尺寸。您提供要调整大小的数组以及该数组的新索引数，该函数将调整该数组的大小。请注意，此功能是专为调整大小的数组到一个更小的长度，你可以通过简单的设置阵列在一个新的指数调整了。就是说，如果您确实使用它来增加数组大小，那么任何新索引都将设置为默认值0。

此函数也可以用于多维数组，只要您按照以下模式指定在提供数组索引时要调整尺寸的维即可：

// Resize the first dimension of the array
array_resize(my_array, 10);

// Resize the second dimension of the array
array_resize(my_array[0], 10);

// Resize the third dimension of the array
array_resize(my_array[0][0], 10);

// etc...

句法：

array_resize(array_index, new_size);

争论描述
array_index要调整大小的数组的索引。
new_size数组的新大小（整数，从0开始）。

返回值：

N/A

例子：

if array_length(a) > 10
    {
    array_resize(a, 10);
    }

上面的代码检查数组的长度，如果它的索引超过10个，则将其调整大小。

返回：变量函数
