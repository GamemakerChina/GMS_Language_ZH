Array_resize

通过这个函数，你可以将一个现有的数组尺寸调整为一个新的尺寸。你提供要调整的数组以及数组的新索引数，该函数将调整该数组的大小。请注意，这个函数是为调整数组的大小而设计的 缩小到一个较小的长度，因为你可以通过简单地在数组中设置一个新索引来调整大小。也就是说，如果你真的用它来调整一个数组的大小，那么任何新的索引将被设置为默认值0。

这个函数也可以用于多维数组，只要你在提供数组索引时指定你要调整的维度，按照这个模式。


array_resize(my_array, 10);

// 调整数组的第一维大小
array_resize(my_array[0], 10);

// 调整数组的第三维大小
array_resize(my_array[0][0], 10);

//等等。

语法。

array_resize(array_index, new_size);

争论 描述
阵列_索引 要调整大小的数组的索引。
新尺寸 数组的新大小（一个整数，从0开始）。

返回。

不适用

例子。

if array_length(a) > 10
{
array_resize(a, 10);
}

上面的代码检查了一个数组的长度，如果它有超过10个索引，那么它就会被调整大小。
