array_set

使用此功能，您可以将数组中的索引值设置为一个值。该函数要求您提供一个变量，用于保存数组以及要设置的索引和要设置的值。此函数还可以用于多维数组，只要您按照以下模式指定在提供数组索引时要设置的维数即可：

// 1D array
array_set(my_array, 0, 100);
// 2D array
array_set(my_array[0], 0, 100);
// 3D array
array_set(my_array[0][0], 0, 100);
// etc...

句法：

array_set(variable, index, value);

争论描述
variable保存数组的变量。
index要为其设置值的数组的索引。
value要设置的值。

返回值：

N/A

例子：

for (var i = 0; i < 10; ++i;)
    {
    array_set(score_array, i, i*100));
    }

上面的代码将给定数组中的前10个项目设置为一个值。

返回：变量函数
