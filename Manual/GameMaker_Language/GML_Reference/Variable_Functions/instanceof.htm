instanceof

使用构造函数创建结构时，可以使用该函数获取用于创建结构的函数的名称。new操作员。为变量提供结构引用以进行检查，该函数将返回带有函数名称的字符串或 undefined。请注意，如果您向函数传递struct文字（即，不使用构造函数创建的struct），则它将仅返回字符串“ struct”。此函数还可用于检查结构引用是否为弱引用，在这种情况下，该函数将返回字符串“weakref而不是创建该结构的函数的名称以获取更多信息，请参见该函数 weak_ref_create()）。

句法：

instanceof(struct_id);

争论描述
struct要使用的结构参考。

返回值：

String or undefined

扩展示例：

在此示例中，我们必须首先定义将用作struct的构造函数的函数：

function init_struct(_a, _b, _c) constructor
    {
    a = _a;
    b = _b;
    c = _c;
    }

然后可以将此功能与 new运算符以创建一个结构，并使用设置为函数中使用的参数值的变量填充该结构：

mystruct = new init_struct(10, 100, "Hello World");

然后，我们可以获取像这样使用的函数的名称：

var _name = instanceof(mystruct);
if is_string(_name)
    {
    show_debug_message(_name);
    }

返回：变量函数
