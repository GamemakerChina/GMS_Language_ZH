这个函数将尝试把一个给定的值转换成一个64位的整数，这个值必须是一个实数，一个字符串，一个 int64 ，一个 int32 ，或者一个 ptr 。任何 否则将导致游戏崩溃并出现错误信息。你可以使用函数来检查一个变量是否持有int64。 is_int64().

语法。

int64(val)。

争论 描述
价值 要转换的值。

返回。

64位整数

例子。

steam_handle = int64(global.fileReadString)。

上面的代码将全局变量中的值转换为64位整数。

返回。可变的功能
