int64

此函数将尝试将给定值转换为64位整数，其中该值必须是实数，字符串， int64 ， 一个 int32或 ptr。任何其他情况都会导致游戏崩溃并显示错误消息。您可以使用函数检查变量是否包含int64 is_int64() 。

句法：

int64(val);

争论描述
val要转换的值。

返回值：

64bit integer

例子：

steam_handle = int64(global.fileReadString);

上面的代码将全局变量中保存的值转换为64位整数。

返回：变量函数
