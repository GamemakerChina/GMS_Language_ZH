is_bool

这个函数返回一个给定的变量是否是布尔值（真或假）。在某些情况下，你想检查GameMaker Studio 2中的一个变量是否持有布尔值 值，这时你就会使用这个函数。

语法。

is_bool(n)。

争论 描述
n 要检查的参数。

返回。

布尔型

例子。

如果is_bool(val)
{
global.Sound = val;
}
else
{
global.Sound = true;
}

上面的代码检查变量 "val "是否是一个实数，如果是，就用它来设置一个全局变量。

返回。可变的功能
