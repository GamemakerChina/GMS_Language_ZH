is_int32

此函数返回给定变量是否为323位整数。在某些情况下，您需要检查并查看变量在GameMaker Studio 2保留的数据类型，而这正是您使用此功能的时间。它返回 true或者 false取决于该值是否为int 32。

句法：

is_int32(n);

争论描述
n要检查的参数。

返回值：

Boolean

例子：

if !is_int32(val)
    {
    show_debug_message("Not a 32 bit integer!");
    }

上面的代码检查变量“ val”以查看它是否包含int32，如果不包含int32，则它在调试控制台中显示一条消息。

返回：变量函数
