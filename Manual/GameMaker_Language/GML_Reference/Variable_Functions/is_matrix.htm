is_matrix

这个函数返回一个给定的变量是否是指一个矩阵。在某些情况下，你想检查并查看一个变量在GameMaker Studio 2中持有什么数据类型，这时你就会使用这个函数。 它返回真或假，取决于该值是否是一个矩阵。

语法。

is_matrix(n)。

争论 描述
n 要检查的参数。

返回。

布尔型

例子。

if !is_matrix(val)
{
show_debug_message("不是一个有效的矩阵！");
}

上述代码检查变量 "val "是否包含矩阵，如果不包含则在调试控制台上显示一个信息。

返回。可变的功能
