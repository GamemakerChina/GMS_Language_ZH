is_numeric

此函数返回给定变量是否为数字值（实数，int32，int64或布尔值）。在某些情况下，您需要检查GameMaker Studio 2的变量是否包含任何数值，而这正是您使用此功能的时间。该函数将返回 true如果给定的输入是数字，则 false否则。

句法：

is_numeric(n);

争论描述
n要检查的输入。

返回值：

Boolean

例子：

if is_numeric(val)
    {
    current_val += val;
    }

上面的代码检查变量“ val”以查看它是否为数值，然后将其添加到给定变量中。

返回：变量函数
