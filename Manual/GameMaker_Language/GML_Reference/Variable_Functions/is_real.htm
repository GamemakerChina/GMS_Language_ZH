is_real

此函数返回给定变量是否为实数（单，双或整数）。在某些情况下，您需要检查GameMaker Studio 2的变量是否包含实数，这就是您使用此功能的时间。它不返回实数，而是 true或者 false，因此，例如，值为“ fish”n将返回 false，但值200 n将返回 true。

句法：

is_real(n);

争论描述
n要检查的参数。

返回值：

Boolean

例子：

if is_real(val)
    {
    score += val;
    }

上面的代码检查变量“ val”是否为实数，如果为实数，则将其添加到分数中。

返回：变量函数
