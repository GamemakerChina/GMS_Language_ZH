is_string

这个函数返回一个给定的变量是否是一个字符串数字。在某些情况下，你想检查GameMaker Studio 2中的一个变量是否持有字符串而不是实数，这时你会使用这个 函数。它不返回字符串，而是返回真或假，所以一个值，例如，"鱼 "的n将返回真。 但如果n的值为200，则会返回false。

语法。

is_string(n)。

争论 描述
n 要检查的参数。

返回。

布尔型

例子。

if is_string(val)
{
name = "Player: " + val;
}

上述代码检查变量 "val "是否包含一个字符串，如果包含，则将其添加到另一个字符串中，然后分配给变量 "name"。

返回。可变的功能
