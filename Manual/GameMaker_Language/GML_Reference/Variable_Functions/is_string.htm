is_string

此函数返回给定变量是否为字符串号。在某些情况下，您需要检查GameMaker Studio 2的变量是否包含字符串而不是实数，这就是您使用此功能的时间。它不返回字符串，而是 true或者 false，因此n的“ fish”值将返回 true，但n的值200将返回 false。

句法：

is_string(n);

争论描述
n要检查的参数。

返回值：

Boolean

例子：

if is_string(val)
    {
    name = "Player: " + val;
    }

上面的代码检查变量“ val”是否包含一个字符串，如果包含，则将其添加到另一个字符串中，然后将其分配给变量“ name”。

返回：变量函数
