is_undefined

此函数返回是否定义了给定变量。在某些情况下，您需要检查并查看变量在GameMaker Studio 2保留的数据类型，而这正是您使用此功能的时间。它返回 true或者 false取决于是否定义了值。

注意：此函数不能用于验证变量的存在。请改用variable_instance_exists（）或variable_global_exists（） 。

句法：

is_undefined(n);

争论描述
n要检查的参数。

返回值：

Boolean

例子：

var val = ds_map_find_value(map, 13);
if is_undefined(val)
    {
    show_debug_message("Map entry does not exist!");
    }

上面的代码检查变量“ val”以查看是否未定义，并显示调试消息。

返回：变量函数
