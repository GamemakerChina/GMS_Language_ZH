is_vec3

在某些情况下，你想检查并查看一个变量在GameMaker Studio 2中持有什么数据类型，这时你会使用这个函数。它返回真或假，取决于 取决于该值是否是一个3值向量。

语法。

is_vec3(n)。

争论 描述
n 要检查的参数。

返回。

布尔型

例子。

if !is_vec3(val)
{
show_debug_message("不是3值向量！");
}

上述代码检查变量 "val "是否包含vec3，如果不包含，则在调试控制台显示一条信息。

返回。可变的功能
