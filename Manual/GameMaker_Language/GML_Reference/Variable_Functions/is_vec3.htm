is_vec3

在某些情况下，您需要检查并查看变量在GameMaker Studio 2保留的数据类型，而这正是您使用此功能的时间。它返回 true或者 false取决于该值是否为3值向量。

句法：

is_vec3(n);

争论描述
n要检查的参数。

返回值：

Boolean

例子：

if !is_vec3(val)
    {
    show_debug_message("Not a 3 value vector!");
    }

上面的代码检查变量“ val”以查看其是否包含vec3，如果不包含，则在调试控制台中显示一条消息。

返回：变量函数
