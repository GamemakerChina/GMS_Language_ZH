method_get_index

使用此功能，可以检索定义方法的脚本的脚本索引。如果未在脚本中定义该方法，则该函数将返回-1，否则它将返回该脚本的索引值。

句法：

method_get_index(method);

争论描述
method要检查的方法变量

返回值：

Integer

例子：

var _index = method_get_index(light_setup);
if _index != -1
    {
    show_debug_message(script_get_name(_index));
    }

上面的代码将检查方法“ light_setup”的脚本索引，然后如果它不是-1，它将脚本名称输出到控制台。

返回：变量函数
