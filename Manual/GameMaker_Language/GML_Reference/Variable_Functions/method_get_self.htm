method_get_self

使用此功能，您可以检索实例ID或结构引用，即 self调用方法时使用的上下文。如果变量不是方法，则函数将返回 undefined。请注意，该函数还可能返回常量 pointer_null，在这种情况下，通话时会使用当前的自我。

句法：

method_get_self(method);

争论描述
method要检查的方法变量。

返回值：

Integer, undefined, pointer_null

例子：

var _self = method_get_self(light_properties);
show_debug_message(string(_self));

上面的代码获取给定方法变量的self上下文，并将其输出到控制台。

返回：变量函数
