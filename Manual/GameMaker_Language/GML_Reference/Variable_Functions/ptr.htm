ptr

此函数将尝试将给定值转换为指针数据类型，其中该值必须是 real， 一种 string， 一个 int64， 一个 int32或 ptr。任何其他情况都会导致游戏崩溃并显示错误消息。

句法：

ptr(n);

争论描述
n要转换的值。

返回值：

Pointer

例子：

if !is_ptr(val)
   {
   val = ptr(application_surface);
   }

上面的代码检查变量“ val”，以查看它是否包含一个指针，如果不包含，则为其分配一个指针。

返回：变量函数
