变量_global_exists

通过这个函数，你可以检查一个全局范围的变量是否存在。你提供要检查的全局变量名称为一个字符串（见下面的示例代码），该函数将返回true，如果 则返回 "true"，否则返回 "false"。

语法。

variable_global_exists(name)。

争论 描述
名称 要检查的全局变量的名称（作为一个字符串）。

返回。

布尔型

例子。

if !variable_global_exists("enemy_num"))
{
global.enemy_num = instance_number(obj_Enemey_Parent);
}

上面的代码将检查名为 "enemy_num "的全局变量是否存在，如果不存在，将创建它并设置一个值。

返回。可变的功能
