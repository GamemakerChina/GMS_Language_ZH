variable_global_exists

使用此功能，您可以检查是否存在全局作用域变量。您提供全局变量名称以作为字符串进行检查（请参见下面的示例代码），该函数将返回 true如果存在具有给定名称的全局变量，或者 false否则。

句法：

variable_global_exists(name);

争论描述
name要检查的全局变量的名称（作为字符串）

返回值：

Boolean

例子：

if !variable_global_exists("enemy_num"))
{
global.enemy_num = instance_number(obj_Enemey_Parent);
}

上面的代码将检查是否存在名为“ enemy_num”的全局变量，如果不存在则创建并设置一个值。

返回：变量函数
