variable_global_get

使用此函数，您可以从给定的命名全局变量中获取值。您提供全局变量的名称以获取字符串的值（请参见下面的示例代码），该函数将返回全局变量或 undefined如果变量不存在。

重要的！如果获取的全局变量不存在，则该函数将返回关键字 undefined并且您可能会收到错误消息，这将使游戏无法正常运行，因此如果有疑问，请使用该功能 variable_global_exists第一的。

句法：

variable_global_get(name);

争论描述
name要获取的全局变量的名称（作为字符串）

返回值：

Value (any data type) or undefined (if the named variable does not exist)

例子：

if variable_global_exists("enemy_num")
   {
   show_debug_message("enemy_num = " + string(variable_global_get("enemy_num"));)
   }

上面的代码将检查是否存在全局变量，如果存在，则将其输出到控制台。

返回：变量函数
