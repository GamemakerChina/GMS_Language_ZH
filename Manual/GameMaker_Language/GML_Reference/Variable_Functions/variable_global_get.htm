变量_global_get

通过这个函数，你可以从一个给定的全局变量中获取数值。你提供一个字符串的全局变量名称来获取其值（见下面的示例代码），该函数将返回全局变量的值。 如果该变量不存在，该函数将返回该全局变量所持有的值，或者 undefined 。

重要的是!如果你得到的全局变量不存在，那么该函数将返回关键字undefined，你可能会得到错误，使游戏停止运行，所以如果有疑问的话 先使用该函数 变量_全局_存在首先。

语法。

variable_global_get(name)。

争论 描述
名称 要获取的全局变量的名称（作为一个字符串）。

返回。

值（任何数据类型）或未定义（如果命名的变量不存在）。

例子。

如果variable_global_exists("enemy_num")
{
show_debug_message("enemy_num = " + string(variable_global_get("enemy_num")); )
}

上面的代码将检查一个全局变量是否存在，如果存在则输出到控制台。

返回。可变的功能
