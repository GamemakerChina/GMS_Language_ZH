variable_global_set

使用此功能，您可以设置给定全局变量的值。您提供全局变量的名称以将其值设置为字符串（请参见下面的示例代码），然后提供要设置的值（可以是任何有效的数据类型）。如果游戏中尚不存在全局变量，则会创建该全局变量，然后为其分配值。

句法：

variable_global_set(name, val);

争论描述
name要设置的全局变量的名称（作为字符串）
val将全局变量设置为的值

返回值：

N/A

例子：

if !variable_global_exists("enemy_num")
   {
   variable_global_set("enemy_num", 0);
   }

上面的代码将检查全局变量是否存在，如果不存在，则将其创建并设置为0。

返回：变量函数
