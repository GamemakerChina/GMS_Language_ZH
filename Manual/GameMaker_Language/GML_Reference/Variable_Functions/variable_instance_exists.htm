变量_实例_存在

通过这个函数你可以检查一个实例范围变量是否存在。你提供唯一的实例ID值（可以从房间编辑器的实例属性中找到，或者在你调用该函数时返回 instance_create_layer())以及要检查的变量名称作为一个字符串（见下面的示例代码）。 如果实例中存在一个给定名称的变量，该函数将返回true，否则返回false。

语法。

variable_instance_exists(instance_id, name)。

争论 描述
instance_id 要检查的实例的唯一ID值
名称 要检查的变量的名称（作为一个字符串

返回。

布尔型

例子。

if !variable_instance_exists(id, "shields"))
{
shields = 0;
}

上面的代码将检查运行代码的实例中是否存在名为 "shields "的变量，如果不存在，那么它将被创建并初始化为0。

返回。可变的功能
