variable_instance_exists

使用此功能，您可以检查实例范围变量是否存在。您提供唯一的实例ID值（可以在房间编辑器的“实例属性”中找到，或者在调用函数时返回）instance_create_layer() ）以及要作为字符串进行检查的变量名称（请参见下面的示例代码）。该函数将返回 true如果实例具有给定名称的变量，并且 false否则。

句法：

variable_instance_exists(instance_id, name);

争论描述
instance_id要检查的实例的唯一ID值
name要检查的变量名称（以字符串形式）

返回值：

Boolean

例子：

if !variable_instance_exists(id, "shields"))
{
shields = 0;
}

上面的代码将检查运行代码的实例中是否存在名为“ shields”的变量，如果不存在，则将其创建并初始化为0。

返回：变量函数
