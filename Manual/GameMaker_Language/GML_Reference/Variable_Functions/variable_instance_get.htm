variable_instance_get

使用此函数，您可以从给定的命名变量中获取值。您提供唯一的实例ID值（可以在房间编辑器的“实例属性”中找到，或者在调用函数时返回）instance_create_layer() ）以及要获取字符串值的变量名称（请参见下面的示例代码）。该函数将返回变量保存的值，或者 undefined如果变量不存在。

重要的！如果获取的变量不存在，则该函数将返回关键字 undefined并且您可能会收到错误消息，这将使游戏无法正常运行，因此如果有疑问，请使用该功能 variable_instance_exists第一的。

句法：

variable_instance_get(instance_id, name);

争论描述
instance_id要使用的实例的唯一ID值
name要获取的变量的名称（作为字符串）

返回值：

Value (any data type) or undefined (if the named variable does not exist)

例子：

if variable_instance_exists(id, "shields")
   {
   var ss = variable_instance_get(id, "shields");
   }
else
   }
   var ss = -1;
   }

上面的代码将检查变量是否存在，如果存在，则将其保存的值检索并存储在局部变量中。如果不存在，则将局部变量设置为-1。

返回：变量函数
