变量_实例_获取

通过这个函数，你可以从一个给定的命名变量中获取数值。你提供唯一的实例ID值（可以在房间的实例属性中找到，或者在你调用函数时返回）。 编辑器中找到，或者在你调用该函数时返回 instance_create_layer()以及变量的名称来获取字符串的值（见下面的示例代码 下面的示例代码）。)该函数将返回该变量所持有的值，如果该变量不存在，则返回 undefined 。

重要的是!如果你得到的变量不存在，那么该函数将返回关键字undefined，你可能会得到错误，从而使游戏停止运行，所以如果有疑问，请使用函数 variable_instance_exists 首先。

语法。

variable_instance_get(instance_id, name)。

争论 描述
instance_id 要使用的实例的唯一ID值
名称 要获取的变量的名称（作为一个字符串

返回。

值（任何数据类型）或未定义（如果命名的变量不存在）。

例子。

if variable_instance_exists(id, "shields")
{
var ss = variable_instance_get(id, "shields");
}
else
}
var ss = -1;
}

上面的代码将检查一个变量是否存在，如果存在，那么它的值将被检索并存储在一个局部变量中。如果它不存在，本地变量将被设置为-1。
