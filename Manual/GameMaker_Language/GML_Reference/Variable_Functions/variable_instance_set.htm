variable_instance_set

使用此功能，您可以在实例中设置给定变量的值。您提供唯一的实例ID值（可以在房间编辑器的“实例属性”中找到，或者在调用函数时返回）instance_create_layer() ）以及将值设置为字符串的变量名称（请参见下面的示例代码），最后是要设置的值（可以是任何有效的数据类型）。如果实例中尚不存在该变量，则将创建该变量，然后为其分配值。

句法：

variable_instance_set(instance_id, name, val);

争论描述
instance_id要使用的实例的唯一ID值
name要设置的变量名称（以字符串形式）
val将变量设置为的值

返回值：

N/A

例子：

if !variable_instance_exists(id, "shields")
   {
   variable_instance_set(id, "shields", 0);
   }

上面的代码将检查调用实例中是否存在实例变量，如果不存在，则将其创建并设置为0。

返回：变量函数
