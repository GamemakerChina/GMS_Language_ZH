variable_struct_exists

使用此功能，您可以检查给定结构中是否存在变量。您提供要使用的结构引用以及要作为字符串检查的变量名称（请参见下面的示例代码）。该函数将返回 true如果该结构存在具有给定名称的变量，并且 false否则。

句法：

variable_struct_exists(struct, name);

争论描述
struct要检查的结构参考
name要检查的struct变量的名称（以字符串形式）

返回值：

Boolean

例子：

if !variable_struct_exists(mystruct, "shields"))
{
mystruct.shields = 0;
}

上面的代码将检查给定结构中是否存在名为“ shields”的变量，如果不存在，则将其创建并初始化为0。

返回：变量函数
