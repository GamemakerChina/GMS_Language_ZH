variable_struct_get

使用此函数，您可以从结构中给定的命名变量中获取值。您提供结构引用以及变量的名称，以获取字符串的值（请参见下面的示例代码）。该函数将返回变量所保存的值或 undefined如果命名变量不存在。

重要的！如果获取的变量不存在，则该函数将返回关键字 undefined并且您可能会收到错误消息，这将使游戏无法正常运行，因此如果有疑问，请使用该功能 variable_struct_exists()第一的。

句法：

variable_instance_get(struct_id, name);

争论描述
struct使用的结构参考
name要获取的变量的名称（作为字符串）

返回值：

Value (any data type) or undefined

例子：

if variable_struct_exists(mystruct, "shields")
    {
    var ss = variable_struct_get(mystruct, "shields");
    }
else
    }
    var ss = -1;
    }

上面的代码将检查给定结构中是否存在变量，如果存在，则将其保存的值检索并存储在局部变量中。如果不存在，则将局部变量设置为-1。

返回：变量函数
