变量_结构_获取

通过这个函数，您可以从一个结构中给定的命名变量中获取数值。您可以提供结构引用以及要获取字符串值的变量名称（见以下示例代码）。该函数将返回 如果指定变量不存在，则返回未定义值。

重要的是!如果你得到的变量不存在，那么该函数将返回关键字undefined，你可能会得到错误，从而使游戏停止运行，所以如果有疑问，请使用函数 variable_struct_exists() 首先。

语法。

variable_struct_get(struct, name)。

争论 描述
结构 要使用的结构参考
名称 要获取的变量的名称（作为一个字符串

返回。

值（任何数据类型）或未定义

例子。

if variable_struct_exists(mystruct, "shields")
{
var ss = variable_struct_get(mystruct, "shields");
}
else
}
var ss = -1;
}

上述代码将检查给定结构中是否存在一个变量，如果存在，则检索该变量的值并将其存储在一个局部变量中。如果不存在，则将本地变量设置为-1。
