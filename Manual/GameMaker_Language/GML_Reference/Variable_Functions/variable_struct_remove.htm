variable_struct_remove

使用此功能，您可以从结构中删除变量。您提供了从中删除变量的结构ID和要删除的变量的字符串名称。

句法：

variable_struct_set(struct, name);

争论描述
struct从中删除变量的结构引用
name要删除的变量的名称（作为字符串）

返回值：

N/A

例子：

if variable_struct_exists(mystruct, "shields")
   {
   variable_struct_remove(mystruct, "shields");
   }

上面的代码将检查给定结构中是否存在给定变量，如果存在，则将其删除。

返回：变量函数
