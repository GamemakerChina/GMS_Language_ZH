variable_struct_set

使用此功能，您可以在结构中设置给定变量的值。您提供结构引用以及变量的名称，以将其值设置为字符串（请参见下面的示例代码），最后提供要设置的值（可以是任何有效的数据类型）。如果该变量在结构中尚不存在，则将创建该变量，然后为其分配值。

句法：

variable_struct_set(struct, name, val);

争论描述
struct要设置的结构参考
name要设置的变量名称（以字符串形式）
val将变量设置为的值

返回值：

N/A

例子：

if !variable_struct_exists(mystruct, "shields")
   {
   variable_struct_set(mystruct, "shields", 0);
   }

上面的代码将检查给定结构中是否存在给定变量，如果不存在，则将其创建并设置为0。

返回：变量函数
