analytics_event

此功能会将指定的文本发送到您通过HTML5游戏选项设置的分析提供程序。此功能可用于创建自定义分析，以跟踪超出所使用提供程序范围的内容。

句法：

analytics_event(string);

争论描述
string要发送给提供者的字符串

返回值：

N/A

例子：

if hs_new
    {
    analytics_event("New hiscore of " + string (score));
    }

上面的代码将检查一个变量，看是否 true如果是，则将使用指定的字符串触发特殊的分析事件。

返回： Web和HTML5
