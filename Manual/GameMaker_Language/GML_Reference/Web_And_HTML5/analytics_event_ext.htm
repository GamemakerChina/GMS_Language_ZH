analytics_event_ext

此功能会将指定的文本发送到您通过HTML5游戏选项设置的分析提供程序。此函数可用于创建自定义分析以跟踪所使用的提供程序范围之外的内容，并且还将接受自定义参数/值对，其中参数是字符串，而值是实数。对于Google Analytics（分析），您只能添加一对，而Flurry最多可以接受7对。

句法：

analytics_event_ext(string, string_param, value);

争论描述
string要发送给提供者的字符串。
string_param[0 -9]要发送的参数（字符串）。
value[0 - 9]参数的值（实数）。

返回值：

N/A

例子：

var num, time;
ini_open("Options.ini");
num = ini_read_real("Data", "Plays", 0);
num += 1;
time = current_time;
analytics_event_ext(GAME_NAME, "Plays", num, "Time", time);
ini_write_real("Data", "Plays", num);
ini_close();

上面的代码将从ini文件中获取播放信息以及当前时间，然后将这些详细信息发送给分析提供商。

返回： Web和HTML5
