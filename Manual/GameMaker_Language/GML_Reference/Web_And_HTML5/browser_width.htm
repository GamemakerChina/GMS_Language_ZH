浏览器宽度

这个变量表示游戏所运行的浏览器的宽度（单位：像素）。如果没有浏览器，则返回窗口大小。下面的图片说明了这个变量与其他可用的定位函数的关系 窗口功能的关系。

语法。

浏览器宽度

返回。

真实的

例子。

如果(browser_height > window_get_height())|| (browser_width > window_get_width())
{
var xx, yy;
if (browser_width > window_get_width())
{
xx = (browser_width - window_get_width() )/ 2;
}
else
{
xx = 0;
}
if (browser_height > window_get_height())
{
yy = (browser_height - window_get_height() )/ 2;
}
else
{
yy = 0;
}
window_set_position(xx, yy);
}

上面的代码检查浏览器的宽度和高度，看是否比当前窗口的宽度和高度大。如果是的话，它就为游戏窗口在浏览器中计算一个新的位置。
