clickable_add_ext

使用此功能，您可以创建一个自定义的，可单击的图标，该图标充当游戏画布外部但位于浏览器窗口本身内的“按钮”。该函数将返回一个id值，该id值必须存储在变量中，以便在引用该新图标的所有其他按钮函数中使用。此功能对于防止浏览器单击按钮时创建弹出窗口非常有用，因为这都是通过GameMaker Studio 2本身完成的。

此功能与 clickable_add()函数，必须设置相对于浏览器窗口（0,0）的位置，为您希望按钮图标具有的精灵提供纹理页面条目（请参见：sprite_get_tpe() ），请指定要打开的URL或要打开或运行的脚本（来自GameMaker Studio 2的资源浏览器），然后设置要使用的目标和参数。但是，此功能还有两个附加选项，允许您将图标alpha从0（完全透明）设置为1（完全不透明），并设置图标的比例，比例是应用于所使用精灵的乘数。因此，值1是默认比例，而（例如）0.5将是一半，而2将是两倍。

还值得注意的是，如果您传入的URL参数是脚本函数的名称（即：它不是以“ http：//”开头），并且脚本被调用 gmcallback_*那么您可以将直接供稿添加到html5事件中。 GameMaker Studio 2脚本功能名称以开头，则不会混淆 gmcallback_。

句法：

clickable_add_ext(x, y, tpe, url, target, params, alpha, scale);

争论描述
x窗口内的x位置。
y窗口内的y位置。
tpe要使用的精灵的纹理页面条目。
url链接到的URL（网站地址）或要运行的脚本。
target这是在其中打开URL的目标区域（请参阅描述）。
params图标的各种参数（请参阅说明）。
alpha图标的图像字母（默认为1）。
scale图标的比例（默认为1）。

返回值：

Real

例子：

home_but = clickable_add_ext(32, 32, sprite_get_tpe(spr_MS_Home, 0), "http://macsweeney_games.com", "_blank", "width=700, height=500, menubar=0, toolbar=0, scrollbars=0", 1, 2);

上面的代码在运行游戏画布的页面的位置（32、32）处创建了一个可单击的DOM图标。该图标使用从纹理页面引用的精灵作为“ spr_MS_Home”，单击该图标将为指定的URL打开一个新窗口，并为该窗口定义属性。该按钮也将缩放为原始精灵大小的两倍，图像alpha为1的一半。
