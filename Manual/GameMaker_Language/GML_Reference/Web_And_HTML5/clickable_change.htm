clickable_change

使用此功能，您可以更改以前使用以下命令创建的可点击图标的精灵和位置 clickable_add()。请注意，该位置基于窗口而不是画布的（0,0）位置，并且必须直接从纹理页面引用该sprite（请参阅：sprite_get_tpe() ）。

句法：

clickable_change(index, tpe, x, y)

争论描述
index要更改的可点击图标的索引。
tpe要使用的精灵的纹理页面条目。
x窗口中新的x位置。
y窗口中的新y位置。

返回值：

N/A

例子：

switch (room)
    {
    case rm_Menu: clickable_change(global.Help_Icon, sprite_get_tpe(spr_MS_Help, 1), 32, 32); break;
    case rm_Game: clickable_change(global.Help_Icon, sprite_get_tpe(spr_MS_Help, 0), 200, 32); break;
    }

上面的代码将根据当前房间更改图像索引和在变量“ global.Help”中索引的可点击图标的位置。

返回： Web和HTML5
