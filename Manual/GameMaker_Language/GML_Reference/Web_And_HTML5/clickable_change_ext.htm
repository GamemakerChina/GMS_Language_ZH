clickable_change_ext

使用此功能，您可以更改以前使用以下命令创建的可点击图标的精灵和位置 clickable_add()。请记住，该位置基于窗口而不是画布的（0,0）位置，并且必须直接从纹理页面引用该sprite（请参阅：sprite_get_tpe() ）。此功能还允许您更改所使用的精灵的比例（作为乘数，以便默认为1，0.5将为一半，而2将为两倍），并将alpha值从0（完全透明）更改为1（完全不透明）屏幕上的最终图标。

句法：

clickable_change_ext(index, tpe, x, y, alpha, scale)

争论描述
index要更改的可点击图标的索引。
tpe要使用的精灵的纹理页面条目
x窗口中新的x位置。
y窗口中的新y位置。
scale图标的比例（默认为1）。
alpha图标的图像字母（默认为1）。

返回值：

N/A

例子：

switch (room)
    {
    case rm_Menu: clickable_change_ext(global.Help_Icon, sprite_get_tpe(spr_MS_Help, 1), 32, 32, 2, 1); break;
    case rm_Game: clickable_change_ext(global.Help_Icon, sprite_get_tpe(spr_MS_Help, 0), 200, 32, 1, 0.5); break;
    }

上面的代码将根据当前房间更改图像索引和在变量“ global.Help”中索引的可点击图标的位置，同时也会更改图标的比例和alpha。

返回： Web和HTML5
