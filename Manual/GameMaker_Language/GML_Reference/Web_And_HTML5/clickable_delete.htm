clickable_delete

必须使用此功能来删除以前使用创建的可点击图标 clickable_add()从游戏窗口。

句法：

clickable_delete(index);

争论描述
index要删除的可点击图标的索引。

返回值：

N/A

例子：

if clickable_exists(global.Help_Icon)
    {
    clickable_delete(global.Help_Icon);
    }

上面的代码从游戏窗口中删除在全局变量“ Help_Icon”中索引的可单击图标（如果存在）。

返回： Web和HTML5
