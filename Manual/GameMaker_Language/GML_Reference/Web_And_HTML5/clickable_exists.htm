可点击_存在

这个函数返回一个可点击的DOM图标是否已经创建，指定的索引是否存在。请注意，用于检查的值必须在之前被初始化，否则你会得到一个错误，导致GameMaker Studio 2关闭。

语法。

clickable_exists(index)。

争论 描述
指数 要检查的可点击的索引。

返回。

布尔型

例子。

if !clickable_exists(home_but)
{
home_but = clickable_add(32, 32, sprite_get_tpe(spr_MS_Home, 0), "http://macsweeney_games.com", "_blank", "width=700, height=500, menubar=0, toolbar=0, scrollbars=0") ;
}

上面的代码检查变量 "home_but "是否已经存在，如果不存在，就在游戏画布运行的页面的位置（32，32）创建一个可点击的DOM图标。该图标使用从纹理页面引用的精灵 纹理页面引用的精灵 "spr_MS_Home"，当点击该图标时，将为指定的URL打开一个新窗口，并为该窗口定义属性。
