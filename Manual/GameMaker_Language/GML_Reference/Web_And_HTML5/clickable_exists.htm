clickable_exists

此函数返回是否存在使用指定索引创建的可单击DOM图标。请注意，用于检查的值必须事先已初始化，否则您将收到错误消息，导致GameMaker Studio 2关闭。

句法：

clickable_exists(index);

争论描述
index要检查的可点击索引。

返回值：

Boolean

例子：

if !clickable_exists(home_but)
    {
    home_but = clickable_add(32, 32, sprite_get_tpe(spr_MS_Home, 0), "http://macsweeney_games.com", "_blank", "width=700, height=500, menubar=0, toolbar=0, scrollbars=0");
    }

上面的代码检查变量“ home_but”是否已经存在，如果不存在，它将在运行游戏画布的页面的位置（32，32）处创建一个可单击的DOM图标。该图标使用从纹理页面引用的精灵作为“ spr_MS_Home”，单击该图标将为指定的URL打开一个新窗口，并为该窗口定义属性。
