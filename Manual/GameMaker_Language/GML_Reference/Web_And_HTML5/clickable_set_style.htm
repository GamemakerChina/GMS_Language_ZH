clickable_set_style

通过此功能，您可以通过提供的键/值对为给定按钮设置CSS样式属性 DS Map。您需要事先创建两个button元素（使用 clickable_add()）和 DS Map并将存储的索引作为参数提供给每个索引。

句法：

clickable_set_style(index, map)

争论描述
index可单击图标的样式索引。
map的索引 DS Map设置样式。

返回值：

N/A

例子：

var map = ds_map_create();
ds_map_add(map, "opacity", "0.5");
clickable_set_style(button, map);
ds_map_destroy();

上面的代码将使用存储在变量“按钮”中的索引来更改可单击按钮的样式。

返回： Web和HTML5
