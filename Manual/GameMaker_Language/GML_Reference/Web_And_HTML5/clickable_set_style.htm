可点击_设置风格

这个函数可以让你通过提供的键/值对来设置给定按钮的CSS样式属性。 DS地图.你需要在之前创建按钮元素（使用 clickable_add())和DS Map，并提供每个元素的存储索引作为参数。

语法。

clickable_set_style(index, map)

争论 描述
指数 样式的可点击图标的索引。
地图 要设置样式的DS地图的索引。

返回。

不适用

例子。

var map = ds_map_create();
ds_map_add(map, "opacity", "0.5");
clickable_set_style(button, map);
ds_map_destroy() 。

上面的代码将改变可点击按钮的样式，其索引存储在变量 "button "中。
