url_open_ext

这将在所选目标设备的浏览器上打开指定的URL，或者如果您正在使用HTML5模块，则在当前打开的浏览器中打开指定的URL。当您使用以下命令时，您指定的“目标”参数与标准JavaScript“名称”值相同 open()方法，您应该注意，除了“ _self ”以外，其他所有内容都可能导致浏览器阻塞，或者询问用户是否希望允许它。

有效的目标是：

目标描述
_空白的在新窗口或新标签页中打开链接的文档（如果弹出窗口被用户阻止，则此功能将无法使用，在这种情况下，您可以使用 clickable_*功能）。
_自己在与单击相同的框架中打开链接的文档（这是默认设置）。
_父母在父框架中打开链接的文档。
_最佳在窗口的整个正文中打开链接的文档。

句法：

url_open_ext(url, target);

争论描述
url链接到的URL（网站地址）。
target这是打开URL的目标区域。

返回值：

N/A

例子：

url_open_ext("http://yoyogames.com", "_blank");

这将在新窗口中YoYo Games

返回： Web和HTML5
