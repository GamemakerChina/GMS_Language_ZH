url_open_full

这将在所选目标设备的浏览器上打开指定的URL，或者如果您正在使用HTML5模块，则在当前打开的浏览器中打开指定的URL。当您使用以下命令时，您指定的“目标”参数与标准JavaScript“名称”值相同 open()方法（请注意，除“ _self ”以外的所有内容都可能导致浏览器阻塞，或者询问用户是否希望允许它），并且“ options”与JavaScript“ specs”参数相同，用于控制新窗口的属性/ tab应该显示（并非所有浏览器都支持所有功能）。

有效的目标是：

目标描述
_空白的在新窗口或新标签页中打开链接的文档（如果弹出窗口被用户阻止，则此功能将无法使用，在这种情况下，您可以使用 clickable_*功能）。
_自己在与单击相同的框架中打开链接的文档（这是默认设置）。
_父母在父框架中打开链接的文档。
_最佳在窗口的整个正文中打开链接的文档。

有效选项包括：

参数描述
“高度= [ px ]”窗口的高度，最小值为100。
'宽度= [ px ]'窗口的宽度，最小值为100。
“左= [ px ]”窗口的左侧位置。
'top = [ px ]'窗口的顶部位置（仅适用于IE）。
“位置= [布尔值]”是否显示地址字段（默认为1）。
'menubar = [布尔值]'是否显示菜单栏（默认为1）。
'resizable = [ boolean ]'窗口是否可调整大小（默认为1）。
'scrollbars = [布尔值]'是否显示滚动条（默认为1）。
'状态= [布尔]'是否添加状态栏（默认为1）。
'titlebar = [ boolean ]'是否显示标题栏。除非调用应用程序是HTML Application或受信任的对话框（默认值为1），否则将忽略此设置。
'工具栏= [布尔]'是否显示浏览器工具栏（默认为是）。

句法：

url_open_full(url, target, options);

争论描述
url链接到的URL（网站地址）。
target这是在其中打开URL的目标区域（请参阅描述）。
options标准浏览器选项（请参阅说明）。

返回值：

N/A

例子：

url_open_full("http://yoyogames.com", "_blank", "resizable=0, height=200, scrollbars=0");

这将在无法调整大小，高度为200像素且没有滚动条的新窗口中YoYo Games
