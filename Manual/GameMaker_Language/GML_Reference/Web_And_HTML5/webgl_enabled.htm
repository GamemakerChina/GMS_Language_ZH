webgl_enabled

此只读变量将返回是否启用了WebGL（true ） 或不 （false ）为您的游戏。它仅适用于通过浏览器运行的游戏（例如HTML5），对于所有其他平台，它将返回true。

句法：

webgl_enabled

返回值：

Boolean

例子：

if webgl_enabled
    {
    global.quality = 1;
    }
else global.quality = 0;

上面的代码检查WebGL标志，然后相应地设置全局变量“ quality”。

返回： Web和HTML5
