启用webgl_enabled

这个只读变量将返回你的游戏是否启用了WebGL（true）或没有（false）。它只对那些通过浏览器运行的游戏起作用（即：HTML5），对于所有其他平台，它将返回true。

语法。

启用webgl_enabled

返回。

布尔型

例子。

如果webgl_enabled
{
global.quality = 1;
}
else global.quality = 0;

上面的代码检查WebGL标志，然后相应地设置全局变量 "quality"。
