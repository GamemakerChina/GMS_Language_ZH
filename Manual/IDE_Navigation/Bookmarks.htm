书签

另一种导航工作区的方法是使用书签。你可以在IDE窗口或脚本或对象事件中的代码/DnD™部分添加多达9个 "书签"，然后你可以使用正确的 组合键快速跳过。

要添加一个书签，只需选择你想标记的窗口或代码行，然后在Windows上 + + NumberKey[1 - 9] ，或在Mac上 + +NumberKey[1 - 9] ，这将设置书签，在IDE中标记该项目，如下所示。

在上图中，我们已经为IDE的不同部分指定了编号为1、2和4的书签，现在我们可以离开这些窗口，甚至完全关闭它们，因为书签将允许我们在任何时候直接回到它的位置上 以后。这可以通过按 / + NumberKey[1 - 9] 来实现，如果书签上的项目已经关闭，它就会被打开，否则工作区就会平移到选定的书签上。

重要的是!书签不会随项目或IDE关闭时一起保存。它们只在它们创建的会话中有效。

如果你想删除一个书签，你只需要选择有书签的窗口或代码行，然后重复你用来创建它的 + / + NumberKey[1 - 9] 。
