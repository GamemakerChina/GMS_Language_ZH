IDE输入和导航

GameMaker Studio 2 IDE接受鼠标和键盘输入，并且可以使用一个或两个或多个来执行许多操作。通常，您可以单击鼠标左键选择任何东西，使用/ +选择多个项目，按住将项目拖到不同的停靠区或工作区中，以及打开上下文相关菜单。请注意，如果您在macOS GameMaker Studio 2 ，并且使用的是单键鼠标，请使用+获得鼠标右键。

还有许多键盘快捷键可用于浏览不同的工作区元素和资源编辑器，我们将快速浏览一些最重要的快捷键（您可以在此处找到完整的列表）：

  • / +“ Z”：这将撤消上一个操作，并且在大多数编辑器中均可使用。您也有多个级别的撤消，因此可以多次按此按钮以“回滚”更改。
  • / +“ Y”：这将重做以前撤消的操作，并且在大多数编辑器中均可使用。像撤消一样，您也有多个级别的重做。
  • / +“ S”：这将迫使GameMaker Studio 2保存您的项目。
  • / +“ T”：这将打开“转到”窗口，使您可以快速搜索并转到游戏中的任何资产，以及搜索“首选项”和“游戏选项” 。系统会为您显示所有内容的列表，并且在您键入搜索词时，该列表将被删除以适合您的情况，因此，例如，如果您所有房间的名称都以“ rm_”为前缀，则键入将剔除该列表以仅显示那些以该名称开头。

  • / + ：这将打开“工作区概述”窗口，该窗口可用于在不同工作区中打开的项目之间快速导航。

  • ：这将打开手册。请注意，在使用DnD™或GML代码时，您还可以选择一个动作或一个函数（或关键字或其他内容），然后按或单击鼠标中键在相关页面上打开手册。
  • ：可以用来折叠或展开IDE中所有不同的停靠窗口。

笔记本电脑模式

如果您GameMaker Studio 2 ，则IDE顶部将在笔记本电脑模式下提供另一个选项：

默认情况下此功能为打开状态，但如果您愿意通过切换此按钮可以将其关闭。笔记本电脑模式与IDE中的一些工具相结合，可以在使用触摸板时提供更好的体验，从而简化了三种主要的鼠标交互：平移，缩放和滚动。它使用两个修饰键来执行此操作：左/离开。笔记本电脑模式打开时，向左/和左被“保留”用于鼠标交互，例如平移房间编辑器，其中“向左” /是缩放，向左滚动和平移。在笔记本电脑模式下滚动的行为就像平移一样，因此应该足够直观和精确，以使您不需要使用滚动条。请注意，在按住这些按钮时，您只需要移动鼠标-而不是单击或拖动，而只是移动-并且，如果存在通常需要使用/或者键，您可以改用键盘的向右键。



触摸屏支持

我们在这里要提到的最后一件事是GameMaker Studio 2的IDE对触摸屏的支持也很少。在所有操作系统上，您都可以使用触摸屏在主工作区中单击和拖动项目，并且我们支持2个同时指针，第二次点击将在其中单击鼠标右键。请注意，在Windows 8及更高版本上， GameMaker Studio 2 IDE也将支持手写笔设备。
