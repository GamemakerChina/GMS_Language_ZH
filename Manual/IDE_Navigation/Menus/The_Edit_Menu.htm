编辑菜单

在编辑菜单中，您可以使用GameMaker Studio 2界面的常规版本工具。可用的选项有：

  • 撤消-撤消操作。
  • 重做-重做上一个动作。
  • 搜索并替换-这将打开一个搜索窗口，允许您搜索整个代码（包括会议室创建代码，脚本和事件）以查找给定的输入-有关更多信息，请参见下文。

当您从“编辑”菜单中选择“搜索并替换”时，它将打开以下窗口：

您可以使用它来搜索游戏中所有脚本化资产，包括房间代码，事件代码，脚本和shaders 。搜索时，可以在顶部的框中键入新的搜索词，也可以单击右侧的箭头图标打开以前搜索的词的列表。您还可以选择提供将用于替换初始搜索词的替换词，并且还可以选择使用右侧箭头按钮中的先前使用的替换字符串。

搜索可以是常规搜索，也可以使用搜索词输入下方的选项进行优化：

  • 您可以选择使搜索大小写敏感，在这种情况下，它仅包含与搜索字符串大小写完全匹配的单词，即：不使用此选项搜索“ My_Var”，则结果将显示为“ my_var”，“ mY_VAR”，或“ My_Var”，但选中此选项后，搜索结果中将仅包含“ My_Var”。
  • 您可以选择只包括仅包含整个单词的搜索结果，在这种情况下，将忽略所有包含搜索词但不完全匹配的结果，即：未选中此选项的“游戏”搜索将显示诸如函数“ game_end（）”之类的变量或诸如“ game_state”之类的变量，在进行检查并进行搜索时只会显示“ game”本身的结果。
  • 您可以选择让搜索忽略注释。选中此选项后，代码中包含搜索词的所有注释都将被忽略，并且不会显示在输出结果中。
  • 您可以根据资产浏览器中的不同资产类型对搜索结果应用过滤器。默认情况下，此选项设置为“所有”，但是单击此选项将打开以下菜单，您可以在其中检查或取消选中要包括在搜索中的资产：

给出搜索词（以及可选的替换词）并设置搜索选项后，您可以使用底部的不同按钮来选择执行搜索的方式以及要执行的任何操作。这些按钮是：

  • 全部查找：查找所有搜索词并在输出窗口中显示它们。
  • 查找上一个：查找搜索词的上一个实例。这将在工作空间中打开包含引用的资产（如果尚未打开）并平移到该资源。
  • 查找下一个：查找搜索词的下一个实例。这将在工作空间中打开包含引用的资产（如果尚未打开）并平移到该资源。
  • 全部替换：这将使用选项集搜索给定术语，并用您指定的新替换术语将其替换在资产中。
  • 替换：这将用所选替换词替换当前编辑器中当前选中的词，并在当前编辑器中突出显示该词。
  • 停止搜索：如果正在搜索，则停止当前搜索。

使用“搜索/替换”窗口获得的所有结果将显示在IDE底部的“搜索输出”窗口中。每个结果可以是双击以在当前工作空间中打开带有术语的资产，然后平移至该资产，以进行编辑。
