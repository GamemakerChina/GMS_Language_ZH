文件菜单

您可以在文件菜单中选择更改，保存或导入项目，以及设置GameMaker Studio 2 IDE首选项。可用的选项有：

  • 新建项目GameMaker Studio 2从头开始创建新项目。它将打开“保存”对话框，并提示您选择要将新项目保存到的位置。为项目指定名称和保存位置后，您将被带到一个新的空白工作区开始对其进行处理。项目文件与 *.yyp文件扩展名。
  • 打开项目-这将打开一个文件浏览器，供您选择以前创建的文件 *.yyz用于加载的项目文件。请注意，在导入时，如果使用低于2.3.0的版本创建项目，那么您将收到以下警告消息：如果选择继续，则将打开文件资源管理器，并且可以为项目提供更新的名称和位置以保存它，因此原始项目不会被覆盖。将项目转换为新的2.3.0+格式后，它将在IDE中打开，并且您会收到一条消息，说明转换成功。您应该继续检查转换过程所生成的兼容性报告，并在尝试运行或编译项目之前查看是否需要修复任何内容。请注意，如果在尝试打开使用2.3.0之前的版本制作的项目时遇到重复的错误，则应联系YoYo Games （使用“帮助”菜单中的“报告错误”选项），并提供原始内容。 *.yyp您要转换的文件。
  • 导入项目-在这里您可以导入以前导出为项目的项目 *.yyz存档（请参阅下面的“导出项目”），或者-如果您有使用YoYo Games GameMaker ：Studio 1.4软件制作的项目，则可以选择此选项以导入 *.gmx或者 *.gmz该产品系列中的文件。与打开项目一样，如果 *.yyzGameMaker Studio 2版本创建的，因此随着该更新所更改的项目文件格式，它需要进行转换。

注意：要将先前的GameMaker项目导入GameMaker Studio 2 ，必须使用1.4版本创建该项目，即使如此，它也可能无法直接用于导入。 GameMaker Studio 2和GameMaker ：Studio 1.4之间有很多不同之处，因此较旧的项目可能需要进行一些调整才能起作用。请参阅YoYo Games知识库上的将GM：S 1.4游戏移植到GameMaker Studio 2的文章，以及手册的包含过时功能和兼容性脚本的部分。

重要的！导入一个 *.yyz包含字体资源的项目可能需要您通过删除/重新添加字体或在字体编辑器中更改字体然后再次将其重新设置来“重新生成”字体。这是因为现有字体信息被保留。 *.yyz导出时，以及以后导入或在另一台计算机上导入时，项目可能尚未安装所需的字体（或可能具有相同名称的其他字体）。仅当导入后字体呈现方式出现问题时，才应执行此操作。

  • 最近的项目-您可以使用此选项查看最近使用的项目。选择它会重新打开“起始页” （如果您已将其关闭），则可以从那里找到的列表中选择一个最近使用的项目。
  • 保存项目-这会将您的游戏保存到创建项目时（或从中打开项目的位置）定义的位置。
  • 将项目另存为-与上面相同，只有您指定项目的位置和新名称。
  • 导出项目-这会将项目导出为一个 *.yyz文件。将副本发送给他人或保存备份非常有用。
  • 首选项-选择此项将转到“首选项”窗口，您可以在其中设置GameMaker Studio 2外观和感觉。
  • 新建IDE-选择此选项后，将打开一个新的IDE实例供您使用（它将在“起始页”上打开）。
  • 我的帐户-此选项将打开您的默认Web浏览器，并带您进入“您的YoYo帐户”页面。
  • 注销-选择此项将使您退出GameMaker Studio 2 。您将需要使用相同的许可证（或任何其他许可证）再次登录，以便以后可以使用该产品。

重要的！如果您一直在使用GameMaker Studio 2的试用许可证，然后购买了其他许可证之一（台式机，移动，UWP，Web或控制台），则可能需要注销IDE，然后重新启动GameMaker Studio 2并重新登录再次输入以确保正确应用了新许可证（尽管这应该自动发生）。

  • 退出-将GameMaker Studio 2退出到桌面。
