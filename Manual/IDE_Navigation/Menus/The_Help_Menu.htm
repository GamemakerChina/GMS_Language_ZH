帮助菜单

此菜单与使用GameMaker Studio 2时获得帮助有关。它包含以下菜单项：

  • 起始页-这将打开一个新的“起始页”工作区。
  • 欢迎页面-这将在新工作区中打开初始欢迎页面幻灯片放映。
  • 打开手册-打开帮助文档。请注意，单击帮助按钮菜单栏中的也会打开文档。
  • 知识库-这将带您进入GameMaker Studio 2帮助中心，您可以在其中找到有关平台设置，已知问题，使用GameMaker Studio 2和许多其他内容的文章。
  • 报告错误GameMaker Studio 2时，如果发现任何错误或有任何问题，可以在此处报告。
  • YoYo Games.com-打开主要的YoYo Games主页。
  • GameMaker社区-这将在您的浏览器中GameMaker社区论坛。请注意，这是一个社区运行的论坛，因此，如果您有错误报告或需要技术帮助，则应首先访问官方支持网站（请参见上文）。
  • 发行说明-这将在您的浏览器中GameMaker Studio 2的发行说明。如果需要回滚更新，请单击此页面上的版本号链接，以获取以前的版本安装程序。
  • 当前的Runtime发行说明-由于Runtime可以独立于IDE进行更新，因此它具有自己的一组Runtime发行说明。选择此选项将在浏览器中打开它们，但请注意，当前runtime可能不是您使用的运行时。要了解runtime ，或有关如何添加，删除或激活runtimes ，请参阅“ Runtime首选项”部分。
  • 所需GameMaker Studio 2的跨平台性质，因此需要许多第三方SDK。单击此按钮将带您到支持论坛文章，该文章详细介绍了每个平台的需求。
  • 在资源管理器中打开项目-这将在OS文件浏览器中打开当前加载的项目的保存位置。
  • 关于-打开“关于GameMaker Studio 2 ”窗口，您可以在其中获取产品的版本信息（报告任何错误或问题所必需的）。
