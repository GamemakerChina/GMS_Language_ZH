Windows菜单

此菜单用于打开常规的停靠的IDE窗口。由于可以重新排列坞站，并且可以将其内容作为独立的窗口删除，甚至可以完全关闭，因此，如果您发现需要使用它们，则可以使用此窗口重新打开它们。可以从此处打开的窗口是“资源浏览器”窗口和所有不同的“输出”窗口（“输出”，“搜索”，“源代码控制”，“断点”和“编译/语法错误”）。

接下来，您可以打开“检查器”窗口。该窗口用于检查当前选定的IDE元素的属性，然后可以根据需要编辑这些属性。

在“检查器”选项之后，您具有“最近使用的Windows列表，还可以选择打开“环境变量”窗口，您可以在其中查看“首选项”中使用的各种环境变量指向的不同路径。这些变量无法编辑（它们的名称或路径/设置都不能），但是您可以选择并复制/粘贴它们中的任何一个以在其他地方使用。

最后，您可以选择打开Tutorial Windows 。这将重新打开与您已打开的教程项目相关的所有窗口，但前提是该项目是作为教程的一部分创建的（如果没有，则此选项将不可用）。有关创建自己的教程项目的信息，请参阅Marketplace上的部分（因为所有教程都是从那里创建，托管和下载的）。
