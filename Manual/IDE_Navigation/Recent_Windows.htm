最近的窗口

最近的窗口列表可以从菜单上打开。 窗口菜单中打开，它将显示一个简短的列表（默认情况下，它最多有10个项目），其中包括你之前打开的窗口。 默认情况下，它最多有10个项目），显示你以前打开过的窗口。开始时这个列表是空白的，但当你在IDE中打开不同的窗口（如精灵、脚本或对象等）时，这个列表就会被填满。如果 如果你需要回到这些以前打开的窗口，那么你只需在列表中的项目上点击鼠标左键 ，工作区就会转到该窗口。 如果你在一个项目上点击鼠标右键 ，那么你将打开以下菜单：在这里你可以选择从列表中删除一个项目（或多个项目，因为你可以用 + 或/ + 来选择多个），或者你可以把它们移到列表的顶部。人民币菜单也有一个额外的选项，允许你完全清除列表。值得注意的是 最近的Windows列表可以停靠在IDE的两侧，你也可以调整某些偏好来改变它的行为方式。
