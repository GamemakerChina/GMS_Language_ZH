本地资产包

在“工具”菜单下，有两个用于创建和导入本地资产包的选项。本地资产包是通过以下方式保存的一组资源： *.yymp可以轻松安装到现有项目中的文件格式。这意味着您可以为常用资源创建自己的软件包库，例如脚本库，启动画面介绍，菜单系统等，而无需使用YoYo Games Marketplace 。

创建资产包

要创建资产包，必须首先创建了要保存在项目中的资源，然后从“工具”菜单中选择“创建本地包”选项。这将打开以下窗口：

在此窗口中，您可以提供可选的发布者名称以及必需的显示名称和包ID 。软件包ID将用于生成与软件包一起保存的文件名，并自动附加到 *.yymp创建文件扩展名。您还可以设置软件包的版本号，然后选择是否签名（默认情况下处于关闭状态）。对软件包进行签名需要您创建一个签名证书（可以从Marketplace首选项中进行此操作），这意味着如果已签名的资产在创建后被篡改，则将无法将其导入。

填写完软件包的详细信息后，就可以继续进行操作，然后从右侧的“项目”部分中选择要包含的资产（使用鼠标左键）或/和以及鼠标左键）。可以使用“添加”或“全部添加”按钮将选定的资源移至“资产包”视图，并可以使用“删除”和“全部删除”按钮再次将其删除。

对资源选择满意并填写了所需的详细信息后，单击“确定”按钮将打开一个文件浏览器，您可以在其中选择将包保存到的位置。选择后，将创建软件包并显示确认窗口，并带有在OS文件浏览器中打开软件包位置的选项。

导入资产包

有两种导入本地资产包的方法：

  • 首先只是拖动 *.yymp文件从资源管理器窗口转到GameMaker Studio 2 IDE
  • 第二种是从“工具”菜单中选择“导入本地软件包”选项。

使用“工具”菜单时，将打开一个资源管理器窗口，您可以浏览到程序包位置，然后选择要导入的位置。无论哪种方式，都将在GameMaker Studio 2打开以下窗口：

软件包的内容在左侧，而您要导入的部分则在右侧。您可以从包中选择一种或多种资源（使用鼠标左键或/和以及鼠标左键），然后使用“添加”按钮添加它们以进行导入，或使用“添加全部”按钮添加所有内容以进行导入。

准备就绪后，您可以单击“导入”按钮以将所选资源添加到项目中。请注意，您还可以选择将所有资源导入到新项目中，这将为您创建一个新项目（并提示您提供一个保存位置），然后将包资源添加到该项目中。

GameMaker Studio 2.3之前的版本创建的资产包将需要转换，因为项目文件格式随2.3更新而更改。这意味着，如果资产经常被导入其他项目，则最好导入整个项目并转换所有内容，然后将其作为新包导出到新位置。
