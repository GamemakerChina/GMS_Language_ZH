项目图像生成器

可从GameMaker Studio 2 IDE顶部的“工具”菜单访问“项目图像生成器”。该工具旨在从一组图像中快速创建所有不同目标平台所需的所有不同初始屏幕，促销图像和图标。您提供基本图像和基本图标，然后单击“生成”按钮，每个平台的“游戏选项”将使用新图像进行更新。

为了使其正常工作，您需要提供高分辨率的初始屏幕和高分辨率的图标图像。推荐的尺寸为：

  • 开机画面：纵向至少为1920x1080px或1080x1920px，并以24bit创作 *.png没有透明胶片。
  • 图标：至少1024x1024px（这是所需的最大图标的大小），并被创作为24bit *.png可以有透明胶片

提供这些图像后，它们将显示在预览Windows ，然后您可以设置将它们应用于不同平台时要使用的“适合”类型。您可以从以下选项中进行选择：

  • 保持宽高比-可以调整图像以适应宽高比，并在不适合1：1的情况下添加背景色以填充侧面
  • 拉伸以适合-这将拉伸图像以适合图像/图标的整个尺寸。
  • 裁剪以适合-这将获取基本图像并使其沿水平或垂直轴最适合，然后裁剪其余部分。例如，如果你有一个具有被用作一个初始屏幕480×800 640×480的图像，则该图像将被放大到适合垂直轴（480 - > 800）和图像的其余部分将被裁剪掉。同样，如果您的1080x1920图像必须适合800x800，则图像将按比例缩小以适合水平轴，其余部分将被裁剪掉。

注意：以上声明的例外是Android自适应图标。由于它们的结构方式，它们将无法使用此工具生成。

重要的！尽管此工具将生成您需要的所有图像，但这并不意味着它是完美的！它将尽最大努力使图像适合每个平台的不同大小和分辨率，但是有时这可能看起来并不像您真正想要的那样，因为实际上并没有“一刀切”的解决方案。因此，运行此工具后，建议您修改每个平台游戏选项中的所有图像，并调整那些可能看起来与手动图像不完全相同的图像。
