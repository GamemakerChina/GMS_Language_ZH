混音器

在创建任何项目时，检查并设置多组声音的音量很有用，这样可以确保在游戏中它们都听起来正确。当您必须打开工作区中的每个声音编辑器时，这可能会很棘手，因此GameMaker Studio 2允许您打开“混音器”窗口，您可以在其中播放来自同一位置的多种声音并相应地调整其音量。可以从“声音编辑器” （将所选声音添加到混音器视图）或“工具菜单”中打开此窗口。

要将声音添加到调音台，您可以使用/ +在资源浏览器中选择声音，然后右键单击菜单，然后选择在混音器中打开。或者，您可以单击将其拖动到调音台中。

混音器本身分为以下几部分：

  • 列控件：此部分包含声音列的控件。您可以选择播放列中的所有声音，也可以将其设置为循环播放或倒回开始。您还可以设置应播放的音量，更改此值将实时更改组中所有声音的音量。
  • 音频组：如果您已将声音资源分配给不同的音频组，则可以从此下拉菜单中选择组，所有声音将添加到当前的混音器列中。
  • 声音资源：列视图的这一部分显示了已添加到混音器中的每种声音。您可以在混音器顶部的列中设置所有声音的整体音量，然后使用各个音量控制器按需更改它，还可以播放和循环各个声音。要从列视图中删除声音，只需单击关闭按钮声音，也可以扩展/折叠单个声音的播放和音量控件，以使视图更紧凑。
  • 链接列：“链接列”按钮将链接相邻的列，这意味着它们的音量和播放控件现在将被链接。因此，任一链接列中的任何声音的音量都会受到任一列控件更改的影响。
  • Extra Column / Add Column ：您可以通过单击混音器右侧的大按钮，根据需要向混音器添加任意多的附加列。每个额外的列将添加到最右列，然后可以链接到前一列，并且您可以像往常一样添加声音或音频组。如果需要从混音器中删除音柱，则可以找到“删除声音组”按钮在“音频组”下拉菜单旁边找到。
