存储库选项

在为你的项目设置了GIT凭证并启用了源码控制后，你现在需要将GameMaker Studio 2IDE链接到一个资源库，这可以通过三种方式之一完成。

  • 创建一个项目库
  • 将当前项目推送到外部资源库中
  • 克隆一个现有的存储库

在第一种情况下，创建一个项目库，这将直接在你的项目中创建一个GIT仓库。在GIT的术语中，这是一个 "非裸库"，所以推/拉在这种情况下没有意义。 在这种情况下没有意义，但你仍然可以提交和跟踪项目的变化，以及在需要时恢复变化和修订。要设置这个，你只需要在主菜单上点击Source Control>Create Project Repository。

然而，你可能已经有了一个现有的项目，并想把它放到一个外部资源库中，以便在你的团队中跟踪和分享 在你的团队中共享。这需要一个已经设置好的外部资源库，以及在你的身份偏好中设置正确的用户名和密码来访问它。如果你已经准备好了，那么请点击Source Control>Import Project into Repository，并填写URL。

最后，作为一个方便的功能，你可以通过 IDE克隆 一个外部资源库。同样，你需要在你的身份偏好中设置你要连接的版本库的用户名和密码，然后就是 只需点击Source Control>Clone Repository。填写URL（使用HTTPS变量，而不是SSH），并说明你想把它放到哪里。

一旦克隆完成，它将自动打开一个文件浏览器，如果你愿意，可以打开该项目。
