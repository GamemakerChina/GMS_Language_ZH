储存库选项

设置好GIT凭据并为您的项目启用了Source Control之后，您现在需要将GameMaker Studio 2 IDE链接到存储库，这可以通过以下三种方式之一完成：

  • 创建一个项目存储库
  • 将当前项目推送到外部存储库
  • 克隆现有存储库

在第一种情况下，创建项目存储库，这将直接在您的项目所在的位置GIT用GIT话来说，这是一个“非裸仓库”，因此在这种情况下，“推/拉”毫无意义，但是您仍然可以提交并跟踪项目更改，并在需要时还原和修订。要进行此设置，您只需在主菜单上单击“源代码控制” >“创建项目存储库”。

但是，您可能已经有一个现有项目，并希望将其放入外部存储库中以在团队之间进行跟踪和共享。这需要已经建立了一个外部存储库，以及在您的身份首选项中设置的正确用户名和密码才能访问它。如果准备好了，请单击“源代码控制” >“将项目导入到存储库” ，然后填写URL：
