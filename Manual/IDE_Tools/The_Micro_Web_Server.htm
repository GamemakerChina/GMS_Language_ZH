微型Web服务器

这个小工具的主要功能是连接到GameMaker Studio 2某些可用目标平台并为其提供文件（例如HTML5目标）。当您在需要Micro Web Server的目标上运行游戏时，随着将游戏提供给所选设备，您将获得一个新窗口：

注意：第一次在需要Micro Web Server的目标上运行游戏时，可能会收到防火墙警报。如果发生这种情况，则应允许Web服务器通过防火墙进行通信。

Web服务器将显示与已提供的文件有关的一定数量的调试信息，并在底部具有清除该信息的按钮。 “清除”按钮旁边是项目的根JS文件的路径，而在其下方则具有“网络速度”选项。默认情况下，它设置为“无限制”，但是单击它会打开一个不同网络选项的列表：

更改此设置还将在右侧更改“包装尺寸”选项。您也可以手动编辑，在这种情况下，网络速度将设置为“自定义”。将其设置为最大值以外的任何值将有效地限制网络传输速度，并允许您在低带宽环境中测试游戏。

您还可以看到允许使用的IP列表以及正在使用的端口号，但是这些不能在Micro Web Server中进行编辑，而应在给定设备目标的“设备管理器”设置中进行设置。
