GameMaker Studio 2简介

GameMaker Studio 2是一种工具，旨在使您和您的团队能够在多个目标平台上以最快，最直观的方式制作新颖和创新的游戏以及原型创意。尽管3D游戏是完全可行的，但它主要是用作制作2D游戏的工具，并且附带了许多工具和编辑器来帮助您实现梦想和想法，最终项目从同一初始版本移植到多个平台上基础资源。

对于不熟悉编程领域的人或以前从未使用过任何游戏创建工具的用户， GameMaker Studio 2提供了直观易用Drag and Drop （ DnD™ ）界面，使您可以开始创建自己的游戏图标。使用视觉脚本非常快速地进行游戏：

您甚至可以将GML功能与DnD™动作结合在一起使用，并为您在编写内容时提供很大的灵活性。所有这些都以直观的方式进行布局，在IDE中将游戏的不同方面“链接”在一起，从而为您提供工作流的独特且非常直观的表示。

GameMaker Studio 2附带了许多强大的工具，例如用于创建自己的精灵的完整图像编辑器以及使用图块集的功能，您还可以导入在任何外部编辑器中制作的图像，包括Spine骨架动画和SWF格式矢量文件。拥有图像后，只需将它们分配给对象编辑器中的对象，添加一些代码或DnD™动作，然后将这些对象的实例放入“房间编辑器”中的游戏室即可。还有其他工具可以创建Paths ， Sequences ， Shaders等。您会发现IDE也非常灵活，如果默认的IDE设置不能满足您的要求，可以使用不同的Preferences对其进行自定义。

该帮助文件将告诉您所有有关GameMaker Studio 2以及如何使用它创建自己的游戏。但是，请意识到，即使使用GameMaker Studio 2类的程序，设计计算机游戏也不是完全不费力的-因此，这里没有“使我成为MMORPG”按钮！也要注意，制作游戏有很多方面，而且它们同等重要：游戏玩法，图形，声音，用户交互等。但是，最重要的方面是耐心。尽管GameMaker Studio 2是一个功能强大的工具，但只要按一下按钮，它就无法为您做所有事情，因此，从简单的示例开始，您会发现不仅创造游戏非常有趣，而且学习使用GameMaker Studio 2做游戏这也是非常有益的！
