创建发布者帐户

您必须先创建一个YoYo帐户，然后才能使用YoYo Games Marketplace购买或出售资产，并且如果您想出售资产包，还需要购买GameMaker Studio 2许可证（Creator Studio 2许可证）（Creator，Desktop，Web，Mobile或控制台）并注册了Marketplace发布者帐户。

注意：GMS2的30天试用许可证使您可以通过Marketplace获得购买和下载免费或付费资产的权限，但是您无法使用它创建发布者帐户。

注册为发布者所需遵循的步骤如下：

  • 登录到您的YoYo帐户。
  • 进入帐户信息中心后，请访问发布商注册页面。可以在“您的YoYo帐户”仪表板发布者部分中找到，您需要在该部分中点击出售资产：
  • 在下一页上，单击“注册”按钮，然后输入有关发布者身份的一些基本信息，例如发布者名称，电子邮件和您计划上传的资产类型。

您必须先阅读并接受《 Marketplace发布商协议》，然后才能完成发布商帐户注册。完成后，您的提交将完成，但是请注意，所有注册都必须由YoYo Games进行验证，因此您将无法立即使用新的发布者。验证您的注册后，您将通过您提供的电子邮件地址收到通知。 （请注意，我们按英国标准营业时间营业，因此周末申请等将导致您的批准时间延迟。）

提交注册后，您将被带到您的发布者帐户，在这里您可以看到已经给出的详细信息并进行编辑，以及上传头像：

现在，您应该转到“付款细节”部分，并在此处填写表格：

请注意：

  • 目前， Marketplace仅通过PayPal提供付款，因此在创建发布者帐户时，将要求您为PayPal帐户提供有效的电子邮件，否则您将无法获得付款。
  • 对于来自英国/欧盟的用户，如果适用于您的组织，则应设置有效的增值税号（对于英国以外的用户，您应在增值税号前加上您所在国家/地区的两位数国家/地区代码）。

如果您打算仅将免费资产上传到Marketplace ，则可以忽略PayPal和VAT字段。然后，您可以单击“提交”按钮继续。

验证注册成功后，即可使用发布商控制台中的其余选项。这是所有资产发布操作的“主目录”，您可以在页面左侧看到一个菜单项。每个人都会向您显示有关发布活动的不同详细信息：

  • 资产：此页面为您提供资产（精灵，音频，项目等）的快速概述，并允许您查看和编辑资产的详细信息，查看销售情况，下载统计信息以及发布资产的新版本。您还可以添加为发布而创建的新资产。有关发布资产的更多信息，请参见页面：列出资产。
  • 服务：在这里您可以查看向其他Marketplace用户提供的服务，以及编辑他们的详细信息并更新或添加新的用户。有关服务的更多信息，请参见页面：列出服务。
  • 帐户详细信息：更新商店上发布者的公开资料，并通过“帐户详细信息”添加付款详细信息（有关设置付款帐户的更多信息，请参见上文）。
  • 用户访问权限：如果您在团队中工作，则可以将多个用户添加到您的发布者帐户，并设置他们可以访问发布者控制台的哪些部分。注册发布者帐户的帐户是所有者，拥有对开发者控制台所有部分的完全访问权限。然后，所有者可以添加用户帐户并管理他们有权访问开发者控制台的哪些部分。只需提供您想要添加的人的电子邮件地址，然后检查您要求他们有权访问的发布者身份的各个部分。
  • 分析：开发人员控制台可让您访问每日销售额并下载报告和图表，从而可以跟踪资产绩效。您可以按多个维度过滤报告，包括“付费”或“免费” ，“类别” ，“国家/地区”和“日期” 。
  • 购买：您可以在此处看到所有资产购买的清单（将显示免费购买，但价格为$0.00）
  • 作者条款：这是当前发布到Marketplace作者条款和条件。您必须先接受这些条款，然后才能发布任何内容，甚至是免费资产。

请注意，一旦设置并验证了一个发布者帐户，便可以返回“ YoYo帐户”信息中心并根据需要创建其他帐户。这样，您可以使用一个销售艺术品资产和另一个销售代码资产（例如）创建两个单独的帐户。
