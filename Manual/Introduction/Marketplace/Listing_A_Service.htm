Marketplace上拥有资产外，您还可以创建服务。在继续之前，请确保您已阅读下面列出的伴随页面，因为在提交服务要约之前，您将需要完成它们中描述的步骤：

  • 创建发布者帐户

YoYo Games Marketplace一个子部分，个人或公司可以在其中提供第三方服务。例如，如果您是一家拥有广告API或您希望向GameMaker Studio 2用户提供类似服务的公司，则可以在此处提供服务。同样，如果您是音乐家，艺术家或程序员，有很多作品可以向人们展示，并希望自己献身于工作，那么您也可以在此处提供服务。

重要的！ YoYo Games对承包商与承包方之间的任何问题概不负责，并且如果有任何争议，将不进行调解或以其他方式介入。

您可以从Marketplace主页，顶部的“类别”菜单或从右侧的菜单中选择服务面板：

单击绿色箭头将打开另一个子菜单，显示可用服务的类别（例如货币化或广告），或者您可以选择“服务本身”以查看所有服务。

从提供的服务列表中，您可以单击任何一个以带您进入服务概述，其中将包含有关所提供服务的所有信息以及与提供商联系或注册其服务的链接等。..您可能还具有指向该服务所需的任何其他资源的其他链接，例如，指向提供排行榜和成就服务的网站的SDK资产的链接。

Marketplace上宣布的任何服务提供商之前，请确保检查其他用户提供的评论（如果有）并阅读提供的EULA ，并确保您了解所指定的任何服务条款。 YoYo Games对最终用户与任何给定服务之间的任何交易不承担任何责任，仅为人们提供广告宣传的便利场所。

要设置自己的服务列表，您必须首先完成以下操作：

  • 设置Marketplace发布者帐户页面（请注意，如果您仅拥有30天免费试用许可证，则无法创建发布者帐户）。
  • 填写有关发布者身份的基本信息，例如发布者名称和电子邮件，您应该从显示的资产类型列表中选择“其他”。

完成后，您将收到一封确认电子邮件，其中包含用于激活新帐户的链接。收到确认电子邮件后，您就可以准备建立服务列表了。您可以通过单击发布者窗口左侧的“服务”菜单项来执行此操作，此后，系统将向您显示必须同意的服务条款和条件，然后才能继续。一旦被接受，您将被带到列出您当前所有服务优惠的页面。要创建一个新的，只需单击标记为“添加新服务”的按钮。

在此页面中，您可以定义要添加的服务的类型，提供有关所提供产品的详细信息以及任何其他相关信息。您所需的详细信息如下：

  • 主要类别：这是您提供的服务类型。设置好主要类别后，您就可以继续选择其他子类别（如果有），这将有助于人们在搜索Marketplace时找到合适的服务。
  • 显示名称：将在服务页面上显示的服务名称（最大255字节）。
  • Description ：简短描述（少于2000字），描述您的服务，其功能和可能的用途，以及指向更多文档的链接等。
  • 关键字：一个或多个单词，可用于缩小搜索范围以在Marketplace上找到您的服务。这些关键字应与所提供的服务有关！
  • 相关资产：如果您的服务需要任何其他资产（GMZ，SDK等），则可以为其创建常规的Marketplace资产清单并在此处链接到该资产。您需要提供资产名称，然后从将出现的列表中选择它。该资产必须先前已发布。
  • 支持电子邮件：人们希望与您联系提供的服务时，可以与您联系以获得支持的电子邮件帐户。
  • 支持网站：用户可以在其中获得对服务支持的URL（即：常见问题解答，论坛，联系人等）。
  • 自定义EULA ：用户可以在其中读取为所提供服务编写的特定EULA的URL（如果未指定，则您的服务将在此处找到YoYo Games EULA之下）。
  • 隐私权政策网址：如果您的服务需要设备或用户数据，或者该服务是为儿童设计的，则您必须提供隐私权政策的链接。
  • 市场营销URL ：一个URL，用户可以在其中获得有关服务和发布者的更多信息，以及观看视频，屏幕截图和任何其他相关的市场营销材料。

最后，您将需要提供大量图像，以在Marketplace上正确营销您的服务。图标（200x200 PNG ）和功能图片（1140x360 PNG ），然后至少拍摄一个屏幕截图（应该是 PNG但可以使用以下尺寸创作：1280×720、1366×768、1600×900或1920×1080），最后您还可以选择从YouTube或Vimeo添加视频。有关如何设置商家信息资产的更多详细信息和建议，请参阅《 Marketplace风格指南》 。

填写完所有这些详细信息后，您现在可以单击“保存”，这将带您回到发布者仪表板的常规服务列表。现在，您应该看到列出的新服务的状态为“正在审核”。

如果您的服务具有SDK，或者您希望为其创建扩展，则可以从创建Marketplace资产一文中找到解决方法。完成此操作后，您便可以将资产包的URL添加到服务列表的“相关资产”部分。

Marketplace提交任何服务报价时YoYo Games将需要对其进行审核，以确保其符合最低入门级标准。这意味着：

  • 您的屏幕截图，图标和功能图片应清晰明了，并且与所提供的服务有关。
  • 您对服务的描述必须与您联系后提供的服务相匹配。
  • 您对服务的描述应该连贯，详细且写得很好。
  • 说明或列表页面中的所有链接均正确且相关。
  • 您提供的服务属于适用材料的规定和准则之内（包括屏幕截图和图标）

审核过程可能需要一两天，但是一旦完成，您的服务就会在“ Marketplace服务”页面上公开显示。如果审查失败，您会收到一封电子邮件通知您，如果不确定为什么失败或无法完成审查过程，请联系帮助中心，有人会帮助您解决问题。
