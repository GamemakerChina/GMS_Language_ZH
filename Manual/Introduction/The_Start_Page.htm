起始页

首次打开GameMaker Studio 2 ，系统将要求您登录您的YoYo帐户（如果尚未注册，请注册一个-参见此处）。注册并登录后，主GameMaker Studio 2 IDE将在“起始页”上打开：

在顶部可以找到常规的IDE菜单，这些手册的以下各节对此进行了说明：

  • 文件菜单
  • 编辑菜单
  • 构建菜单
  • Windows菜单
  • 工具菜单
  • Marketplace菜单
  • 布局菜单
  • 帮助菜单

请注意，取决于您当前关注的窗口，顶部菜单栏中还会显示特定于上下文的菜单选项。例如，如果您具有“图像编辑器”窗口的焦点，则此处将有“图像”，“视图”和“效果”的额外菜单项。这些额外的菜单项在手册的相关部分中进行了说明，该菜单项用于生成它们的工作区或窗口。

起始页的其余部分如下：

  • 主页-导航“起始页”的不同部分时，单击鼠标左键只需将您带回到主页。
  • 版本详细信息-本节提供有关当前使用的IDE版本以及当前Runtime版本的详细信息。您还将在本节中Runtime可用的任何更改的通知。
  • 最近的项目-在开始页面的左侧，您可以看到可以打开的先前项目的列表。如果将鼠标悬停在它们上方，则可以看到完整的项目名称和路径，然后单击鼠标左键将打开项目。
  • 社交-起始页的顶部栏包含许多链接，以使导航更轻松。它们在下面列出：
    • YoYo Games -带您进入YoYo Games的主要网站
    • 社交媒体图标-将您带到各种YoYo Games社交媒体网站，您可以在其中随时了解有关GameMaker Studio 2
  • 项目选项-您可以在“项目”部分中创建，打开或导入项目。这将在此页的后面详细说明。
  • 信息-起始页的信息部分允许您浏览YoYo Games Marketplace YoYo Games团队制作的教程和演示，以展示GameMaker Studio 2特定方面（有关更多信息，请参见下文）。

开始一个新项目

您可以单击“新建”按钮创建一个新项目，单击“打开”按钮打开一个现有项目，或者单击“导入”按钮打开GameMaker ：Studio 1.4 GMX项目文件或打开压缩文件 YYZGameMaker Studio 2项目文件。打开和导入都将打开文件浏览器，供您浏览所需的项目文件。

注意： GameMaker Studio 2 GameMaker ：Studio 1.4项目并不完全向后兼容，但是导入的1.4项目仍应运行，因为已经使用兼容性脚本为您自动创建了过时的功能。 GML所做的更改的完整详细信息，请参阅帮助中心文章将GMS 1.4项目移植到GameMaker Studio 2 ，以及手册中有关过时功能的部分。

要创建一个新项目，只需单击标有“新建”的按钮，即可进入以下屏幕：

GameMaker Studio 2允许您指定要创建的项目类型，并且可以根据自己的编码知识或习惯进行选择。可用的选项有：

  • 新建Drag and Drop项目–这将创建一个新项目，其中将设置对象编辑器（以及允许代码的任何其他编辑器）以使用GameMaker Studio 2 Drag and Drop接口。 Drag and Drop （ DnD™）的更多信息，请参见此处。请注意，这不是DnD™专有的界面，您可以使用鼠标右键DnD™和GML 。
  • 新的GameMaker Language项目：这将创建一个项目，在其中使用GameMaker Language （ GML ）完成默认编程。您可以在此处GML更多信息。请注意，这不是专有的GML界面，您可以使用鼠标右键GML代码和DnD 。

无论您选择什么，都将提示您选择项目的保存位置，然后在GameMaker Studio 2中将其打开。

教程和演示

单击“教程”或“演示”按钮时，将转到一个屏幕，其中显示您拥有的教程/演示以及可供下载的官方教程/演示：

要下载任何官方教程或演示，只需单击您要浏览的官方教程或演示的图标，它将添加到顶部的“拥有的”部分（请注意，在此之前Marketplace可以做到）。一旦教程或演示位于“拥有的”部分中，您可以将鼠标悬停在其上并安装或从Marketplace上查看其信息：

一旦安装了教程或演示，只需单击一下即可将其打开，然后系统会提示您提供一个位置，以将教程项目保存到IDE打开之前即可开始。请注意，已安装的教程和演示将带有一个附加图标可以用来从您的计算机上卸载项目（这不会从“拥有的”列表中删除该教程或演示）。

“起始页”的教程/演示部分还具有几个按钮，可帮助您更好地使用和浏览资产：

单击此按钮刷新从项目列表Marketplace ，所以你可以检查是否有任何已更新，并获得最新的版本。
单击此按钮将在您拥有的教程或演示与已安装的教程或演示之间切换视图。

还值得注意的是，您也可以从Marketplace找到第三方教程和演示，一旦购买并将它们添加到您的帐户中，它们也将出现在相应页面的“拥有”部分中。
