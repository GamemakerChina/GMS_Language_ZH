创建声音效果

所有游戏都必须具有声音效果，并且大多数游戏还必须具有音乐，这意味着您需要能够将它们添加到GameMaker Studio 2游戏中。这是通过声音编辑器完成的，只要您在资产浏览器中创建新的声音资产，就会打开该声音编辑器：

GameMaker Studio 2声音可以解压缩，也可以压缩，压缩后的声音具有在运行时如何处理声音的其他选项。未压缩的声音是已保存为 WAV格式化文件，而压缩声音可以是 MP3或者 OGG格式。通常，您会希望使用较小的声音效果 WAV-像是拍摄声音或按钮声音等-以及更大的效果或音乐 MP3或者 OGG。与其他资源一样，建议您使用适当的前缀来命名声音，例如“ snd_ ”，这样您以后就可以轻松识别它。

注意：未压缩的声音仅表示未对其应用任何类型的压缩算法。对声音应用压缩意味着它将需要较少的存储空间，但同时也意味着在播放之前需要对其进行解压缩，并且由于小声音效果通常会非常快速地被使用-并且不会占用太多的存储空间-您不希望在播放它时解压缩它会花费额外的CPU成本。但是，由于诸如音乐文件之类的东西通常可以是非常大的文件，因此您确实希望对其进行压缩，因为它们将占用较少的存储空间，并且解压缩的额外CPU成本可以忽略不计，因为它们通常只是在播放一次一个。

添加声音就像单击添加声音按钮一样简单然后在打开您要使用的文件的文件资源管理器中浏览（您也可以将声音文件拖放到GameMaker Studio 2 IDE上，然后将它们作为新的声音资产添加）。选择并加载一个文件后，便可以使用显示的播放控件预览该文件，并更改文件导出方式的某些属性。如果您对此不确定，则可以直接保留所有内容，即使您要添加音乐或 MP3/ OGG格式的声音，您可能应该更改为“压缩-不流式”格式。

您可以在以下链接的页面上的声音编辑器上找到更多信息：

  • 编辑器：声音编辑器

请注意，声音会占用大量磁盘空间，因此在为游戏项目创建声音时应格外小心。您真的需要48KHz的声音效果只播放一秒钟的时间吗？可能不是！因此，考虑将效果的采样率降低到11025或22050，对于音乐或循环效果等较长的事物，还应考虑使用压缩格式。还要注意，如果您打算使用3D位置音频效果，则初始声音文件必须为mono 。
