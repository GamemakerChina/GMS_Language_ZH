创建精灵

既然您知道什么是程序了，是时候来看看可以在GameMaker Studio 2用来制作游戏的各种资产。首先，我们将看一下sprites ，它们通常是制作任何项目时首先需要的东西之一。如“资产浏览器”部分中所述，子画面是可以进行动画处理（尽管不一定要进行动画处理）然后绘制到屏幕上的图像。一般而言，精灵将与对象相关联的，但你可以自己绘制的精灵，无论是从代码或在房间编辑资产层（我们谈谈这个在稍后详细，在部分房间）。您还可以创建一个Sprite用作图块集，但我们也会在后面的部分中探讨该选项。

从“起始页”创建新项目时，资产浏览器将仅包含您可能需要的资源的空文件夹，因此需要右键单击在Sprite资产文件夹中，然后选择选项Create （如果您没有Sprite资产文件夹，则可以从右键菜单中创建一个，然后创建Sprite，也可以在资产浏览器中的任意位置右键单击并创建它）。这将创建一个新的Sprite资产并为您打开Sprite编辑器（如果Sprite编辑器没有打开，只需双击在新资产上）：

如您所见，窗口的左上角有一个精灵名称的字段。所有子画面（和所有其他资产）都必须具有一个名称，以便您（和GameMaker Studio 2 ）可以轻松识别它们，尽管您应注意，该名称实际上只是一个变量，其ID值“指向”资源，在这种情况下是精灵。最好为每个精灵指定一个描述性名称，以便您一眼就能识别出某个特定资产是精灵还是对象还是其他东西，许多人通过在资源上添加前缀或后缀“ spr “ - 例如， ” spr_Ball请注意，资源名称仅限于使用字母，数字和下划线符号“ _”，并且所有资产的名称必须以字母开头，而不是数字。

我们将在本节末尾讨论Sprite编辑器的其他功能，但是首先我们需要解释如何绘制Sprite。这是在“图像编辑器”中完成的。图像编辑器是用于在游戏中创建图形的非常强大的工具，可通过单击“编辑图像”按钮打开。

我们不会对这里可用的工具进行过多的介绍-因为我们在手册的“图像编辑器”部分中进行了介绍-但简要地讲，您有：

  • 框架视图的左上角-显示了精灵的框架，可以使用放大镜图标调整其大小。
  • 在中心，我们有一个绘图画布-显示当前选择的框架以准备绘图，您可以使用鼠标中键平移和鼠标滚轮放大和缩小。
  • 右上角有基本的笔刷-在这里您可以选择要绘制的笔刷，任何自定义笔刷也将添加到此处。
  • 在右中角有一个颜色选择器-在这里我们可以选择用于鼠标左键和/或鼠标右键的颜色，因为您可以同时使用一个或另一个按钮在画布上绘制。
  • 在颜色选择器的下面，我们有我们的工具-这些是不同的绘图工具，包括画笔，填充，蒙版和形状工具。请注意，某些工具是沿对角线分割的，它们将根据您单击以选择它们的线条的哪一侧绘制轮廓或填充形状。
  • 在右下角，我们具有“图层”视图-像那里的许多其他绘图工具一样，您可以创建不同的图层以进行绘制，从而允许您试验位置和颜色等...而不必担心破坏其下方的内容。

在图像编辑器中绘制一些内容，并花一些时间来使用这些选项，然后在准备好关闭图像编辑器工作区（将保存图像）后，将您带回到带有精灵编辑器的主工作区。我们将讨论Sprite编辑器的其他一些功能，这些功能对于刚入门的用户来说非常重要。

您需要知道的第一件事是如何设置精灵的原点。原点只是用于在房间内“固定”精灵的点，在这种情况下，我们要将其设置为中心。

所以我们点击在从下菜单中设置原点，然后选择“中间中心”：

您将在预览图像中看到蓝色的原点标记移动到精灵的中心。请注意，只需单击即可将原点放置在所需的任何位置在预览图像上，您还可以通过在顶部的x / y字段中添加值来手动输入原点位置。

要做的另一件事是如何为Sprite设置碰撞蒙版。碰撞遮罩是GameMaker Studio 2将检查的区域，以查看是否与游戏中的任何其他对象实例发生碰撞，并且默认情况下将其设置为矩形（这也是最快解决的问题）。请注意，您可以将边界框的检测模式设置为自动（默认值）或手动。如果选择“手动”，则可以在左侧，右侧，顶部和底部设置边界框值。

除矩形以外的任何类型的碰撞遮罩都将较慢地解决，但是在制作小型，简单的游戏时通常不会出现问题。请注意，您还可以通过单击Sprite编辑器中的“导入”按钮来导入使用其他软件制作的Sprite，还可以将图像文件拖到GameMaker Studio 2 IDE上，以将它们作为Sprite添加到Asset Browser中。您还可以从资源浏览器中拖动精灵以将其直接添加到事物中，在这种情况下，IDE会突出显示可以将精灵添加到的位置：
