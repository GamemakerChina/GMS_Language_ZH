创建图块集

拼贴集GameMaker Studio 2处理Sprite和对象的方式有些不同，它们也被归类为单独的资产类型。因此，图块集是在一个精灵中一起创建的图块的集合，而图块只是作为图块地图的GameMaker Studio 2房间的图形资源。然后，只需很少的CPU / GPU处理即可将这些平铺图绘制到屏幕上，使其成为设计关卡中任何静态项目的理想选择。

为了使处理图块如此之快，它们确实有一定的限制，最基本的限制是它们只能是矩形或正方形（如果您想要除此以外的任何东西，则需要创建一个常规的精灵并使用房间编辑器中的“资产层”，但我们将在关于房间的下一部分中进行讨论。其他限制包括以下事实：它们只能旋转90°或镜像或翻转，并且只能使用2的幂的帧（即：2、4、8、16等）进行动画处理。 。

注意，第一个图块在该图像中为空白。 GameMaker Studio 2存储房间的图块地图的方式意味着，第一个图块将始终需要为空白，因为它实际上是“删除”图块或“空白空间”图块（此“空白”图块将具有索引）值0-使用从0开始并从左到右，从上到下递增的值来索引图块。因此，在创建图块集时，始终需要以这种方式将左上角的图块留空。还要注意，尽管上图显示了一个全墙瓷砖，其中包含47个瓷砖，但稍后将它们放置在房间编辑器中时，可能不需要旋转和翻转瓷砖的数量。

创建了精灵之后，我们可以继续定义图块集本身，这意味着您必须先通过右键单击来创建新的图块集资源在图块集资产文件夹中，然后选择创建。这将打开Tile Set Editor ，然后您可以在其中添加精灵并设置其属性：

我们已将图块集大小设置为64px，并且将其余选项保留为默认值，因为我们不需要偏移量或类似的东西（偏移值对于任何在周围有空格的图块集子图很有用每个磁贴）。您需要使用适当的前缀（例如“ ts_因此您可以在以后轻松地识别它，并注意与精灵相同的限制，即：名称不能以数字开头，只能使用下划线“ _”符号和字母数字字符。
