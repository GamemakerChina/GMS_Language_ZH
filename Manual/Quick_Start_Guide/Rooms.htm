房间数

GameMaker Studio 2制作的所有游戏都需要至少一个房间来运行（但可以有更多房间），而房间只是一个空间，您可以在其中放置组成游戏对象的实例以及图块和任何其他图形资源。首次在资产浏览器中创建房间时，将为您提供一个新的“房间编辑器”工作空间，可用于编辑其属性，如下所示：

默认情况下，房间工具将位于左侧，主要分为三个部分：

  • 图层编辑器：房间中的所有内容都放置在图层上，并且可以选择不同的图层类型（稍后会对此进行更多介绍）。图层按深度排序，该深度定义了游戏运行时图层内容在屏幕上呈现的顺序。深度顺序是从最高到最低，因此深度越低，“相机”越近，深度越高，距离越远。例如，深度为-300的图层将在深度为-100的图层上渲染，深度为1000的图层将在所有深度小于此值的图层下渲染。请注意，默认情况下， GameMaker Studio 2会根据“图层编辑器”中的位置为您排序图层深度，并且您可以使用鼠标在图层周围拖动以对图层进行重新排序，而GameMaker Studio 2将对深度进行重新排序以进行匹配。
  • 图层属性：添加到房间中的每个图层将具有其自己的属性，并且这些属性将根据图层类型而变化。该窗口允许您更改这些属性并编辑将如何渲染图层。
  • 房间属性：房间也有许多属性，可以在此处设置。可以在此处设置房间大小，活动的摄像机视口以及其他一些内容。

我们已经提到了不同的图层类型，所以让我们来看看它们是什么以及在构建游戏时如何使用它们：

背景层背景层是可填充有单一颜色或单个图像的层。它可以在房间内移动和定位，并且可以有多个背景层。顾名思义，通常将其用于为房间中所有其他图层生成恒定的背景。默认情况下，新房间将始终包含背景图层，但是如果不需要它可以将其删除。
实例层实例层是放置游戏所需对象的所有实例的地方。要将实例添加到图层，只需单击在资产浏览器中的对象上，然后将其拖到房间编辑器工作区中。当您释放鼠标按钮时，该对象的实例将被添加到房间中。默认情况下，新房间将始终包含一个实例层，但是如果不需要它可以将其删除。
平铺地图图层创建图块集后，您需要将图块添加到您的房间，这是通过创建图块地图图层来完成的。磁贴图图层是允许您从已创建的任何磁贴组资产中添加磁贴的层，该图层将自动设置为使用与磁贴组像元大小相同的网格。您只能向每个网格单元添加一个图块，因此，如果需要多个图块占用相同的空间，则应使用多个图块地图图层。
路径层路径层是运行游戏时唯一未真正呈现的层。该层更像是“便利”层，可帮助您创建或编辑路径资产，因为它允许您在实际房间空间内添加或编辑路径。
资产层有时，您想要一个不错的图形效果，但又不想使用实例的开销，或者除了绘制自身之外，不需要实例做任何其他事情。您可以使用图块地图，但是鉴于图块地图仅限于使用网格，并且每个网格单元仅包含一张图像，因此它们可能会有些限制。这是您要使用资产层的时候。资产层只是获取一个精灵资源，并使用将其添加到房间中时设置的参数对其进行绘制。可以将精灵和序列添加到此层，就像将对象的实例添加到实例层一样，即：单击在资产浏览器中的精灵上，然后将其拖动到房间编辑器工作区中。

如果要在房间中放置多个实例，则可以从资源树中选择一个对象，然后按住预览资源，如果您另外单击鼠标左键然后可以将其“绘制”到房间层中。
