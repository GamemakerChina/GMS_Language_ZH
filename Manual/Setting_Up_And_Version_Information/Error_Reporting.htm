错误报告

尽管在创建GameMaker Studio 2要格外小心，以确保它是稳定可靠的工具，但鉴于软件及其运行的操作系统的性质，总有可能发生问题。在大多数情况下， GameMaker Studio 2会捕获任何错误，并且可能会提示您保存项目，然后关闭并重新启动程序。如果发生这种情况，那么我们将要求您向YoYo Games服务台提交错误。您可以使用“帮助”菜单中的“报告错误”选项来执行此操作，我们要求这样做时提供尽可能多的信息，并提供指向GameMaker Studio 2 ui.log文件，可以在以下位置找到，具体取决于操作系统：

  • 在Windows - C:\ProgramData\GameMaker Studio2
  • ，O n macOS - /Users/Shared/GameMaker Studio2 /（Macintosh HD>用户>共享> GameMaker Studio2）

在极少数情况下，如果错误很严重，则产品可能崩溃而未给出上述警告，并且如果发生这种情况，则会显示以下窗口：

该窗口将向您显示一些由GameMaker Studio 2编译的崩溃报告数据，并提供将其发送到YoYo Games的选项，以便他们可以查看发生的情况并在将来的更新中解决问题。如果您想确切知道要发送的数据，请在YoYo Games帮助台上查看此文章。
