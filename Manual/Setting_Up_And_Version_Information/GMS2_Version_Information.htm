桌面

使用 桌面版 的GameMaker Studio 2取消了试用许可证的所有限制（见上文），你可以将项目编译到以下桌面目标。

所有目标都有两个编译选项，一个是 VM ，它使用一个运行器和解释代码，另一个是 YYC (YoYoCompiler)，它创建了本地代码和一个编译的可执行文件。请注意，YYC选项将需要额外的设置。欲了解更多信息，请参见以下章节 所需的SDKs或 桌面目标。
