GMS2设置和版本信息

GameMaker Studio 2获得不同版本的许可，这些许可的选择将在很大程度上取决于您要定位的受众以及您是否只是想玩一些有趣的游戏或希望出售它们并分发它们。您可以在下面找到每种许可证类型的详细信息：

审判

注册YoYo帐户会授予您30天的免费试用许可证，以启动GameMaker Studio 2 ，然后您可以根据需要购买不同的升级。试用许可证为您提供了很多GameMaker Studio 2体验，可以在macOS或Windows PC上运行，但使用期限不得超过30天。在最初的30天用完后，您需要升级到其中一个付费许可证才能继续使用该产品。请注意，试用版许可证确实具有以下限制：

  • 从您通过确认电子邮件YoYo帐户之日起，它仅允许30天访问GMS2。
  • 您不能为任何目标平台编译最终的可执行文件。
  • 您无权测试性能更高的“ YYC”目标出口。
  • 您无法禁用有关试用版许可证的IDE通知即将过期（但是，如果您在“偏好设置”中关闭了通知，则其他通知将被禁用）。
  • 您无法创建Marketplace Publisher帐户（但您的YoYo帐户可以访问Marketplace并购买付费/免费资产）。


    创作者

    使用GameMaker Studio 2的创建者版本时，将删除30天试用许可证的限制（请参见上文），并且可以将游戏导出到所选平台，该平台可以是macOS或Windows 。可用的目标平台将取决于您购买的IDE的版本（ macOS Creator或Windows Creator），并将使用VM （运行程序+解释代码）运行和构建游戏。有关设置信息，请参阅“必需的SDK”上的“帮助台”部分或“桌面目标”上的部分。请注意，创建者许可证有两个较小的限制：

    • 您无法在加载游戏时更改默认的初始屏幕。
    • 您无权使用性能更高的“ YYC”（ YoYo编译器）目标导出来构建项目。


      桌面

      使用GameMaker Studio 2的桌面版时，将删除试用许可证（请参见上文）的所有限制，您可以将项目编译为以下桌面目标：

      • Windows
      • macOS
      • Ubuntu （ Linux ）

      所有目标都带有两个编译选项，一个是使用运行程序和解释代码的VM ，另一个是创建本机代码和已编译的可执行文件的YYC （ YoYo请注意，YYC选项将需要额外的设置。有关更多信息，请参见“ 必需的SDK ”部分或“ 桌面目标”部分。


      网页

      使用GameMaker Studio 2的Web版本时，试用许可证的所有限制（请参见上文）均已删除，您可以将项目编译为HTML5目标。游戏使用JavaScript进行编译。您可以在以下YoYo Games帮助台找到更多信息：设置网络（HTML5） 。


      移动的

      使用GameMaker Studio 2的移动版时，将删除试用许可证的所有限制（请参见上文），并且您可以将项目编译为以下目标：

      • Android
      • iOS
      • tvOS

      所有目标都有两个编译选项，一个是使用运行程序和解释代码的VM ，另一个是创建本机代码和已编译的可执行文件的YYC （ YoYo请注意，YYC选项将需要额外的设置。有关设置的更多信息，请参见“ 必需的SDK ”部分或“移动目标”的帮助台部分。


      超宽工作计划

      使用GameMaker Studio 2的UWP版本时，试用许可证的所有限制（请参见上文）均已删除，您可以将项目编译为兼容UWP的可执行文件。可以将这些可执行文件配置为在两个UWP平台之一上运行：

      • Windows 10
      • XBox One

      UWP目标带有两个编译选项，一个是使用运行程序和解释代码的VM ，另一个是创建本机代码和已编译的可执行文件的YYC （ YoYo请注意，YYC选项将需要额外的设置。有关设置的更多信息，请参见“必需的SDK ”部分或“ UWP目标”的帮助台部分。


      游戏机

      使用GameMaker Studio 2的控制台版本时，将删除试用许可证的所有限制（请参见上文），并且您可以为以下一个或多个目标平台编译项目：

      • XBox One
      • 的PlayStation 4
      • Nintendo Switch

      每个控制台目标都带有两个编译选项，一个是使用运行程序和解释代码的VM ，另一个是创建本地代码和已编译的可执行文件的YYC （ YoYo请注意，YYC选项将需要额外的设置。有关设置的更多信息，请参见“必需的SDK ”部分。

      重要的！由于与控制台导出相关的各种NDA的限制， GameMaker Studio 2手册中没有有关为这些目标进行设置或编程的信息。一旦您成为所选平台的注册开发人员，请通过您的YoYo帐户联系帮助台，并请求访问相应的文档。


      教育

      GameMaker Studio 2也可以在学校或大学环境中使用。有关更多信息，请通过YoYo Games Education Portal YoYo Games联系。
