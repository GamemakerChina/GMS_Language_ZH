字体编辑器首选项

字体编辑器首选项用于定义字体编辑器的行为。下面列出了可用的不同选项以及每个选项的功能说明。

  • GameMaker Studio 1项目导入的字体：选中此选项后， GameMaker Studio 2将尝试更新字体并重新生成其纹理。仅在以下情况下会发生这种情况：
    • 此首选项已启用
    • 字体在机器上可用
    • GameMaker Studio 2可以在字体系列中找到匹配的面孔（这很可能是导入不一致的原因）
    请注意，如果GameMaker Studio 1字体没有缓存的字体纹理，也会发生这种情况。如果旧字体无法更新，则GameMaker Studio 2将默认使用缓存的字体纹理，并且字体编辑器将不再允许您编辑字体，并显示备份和更新到另一个选项：

    单击此按钮将使您可以编辑字体，添加丢失的字体或设置另一种字体，然后将备份的字体存储在字体资产文件夹中，如下所示： <font name>.yy.gms1和 <font name>.png.gms1。要还原这些文件，请手动删除 <font name>.yy文件，然后通过删除这些文件替换它们 .gms1文件名的扩展名。重要的！所有资产都将其名称存储在 .yy文件，因此您必须编辑这些文件并将正确的资源名称放入 <name>字段（如果不同）。
  • 自动选择失败的内容重新生成对话框：当您重新生成字体资产纹理时，将显示警告。将此选项设置为“确定”将意味着在不显示警告的情况下重新生成了字体。
  • 自动选择失败的字体重新生成对话框：当字体资产纹理的重新生成失败时，将显示警告。将此选项设置为“确定”将跳过显示消息。
  • 升级导入的旧字体的自动选择：导入旧项目时，您可能会收到有关旧字体和升级提议的警告。将此选项设置为“确定”将自动完成更新，而不会显示此消息。
