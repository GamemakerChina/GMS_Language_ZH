文件监视程序首选项

File Watcher是IDE的一部分的名称，该部分监视与项目相关的文件，以查看由于外部软件（例如：在外部编辑器中编辑图像）或由于内部更改（如编辑脚本资产）而导致的任何更改。通常，这些设置不需要进行编辑，但是在某些系统上（例如使用SSD作为项目文件的保存驱动器的系统），当您执行诸如简单地保存文件时，文件观察器可能会通知您更改。项目或编辑任何项目资产时，在这种情况下，您可能需要更改这些设置。

可用的设置为：

  • 启用文件监视程序：可用于启用或禁用文件监视程序系统。如果禁用此功能，则IDE将不再拾取外部源更改的任何文件，因此在禁用此功能时应格外小心。默认情况下启用文件监视程序。
  • 在IDE中保存后在MS中设置File Watcher容限：这将调整保存文件和检查文件更改之间的时间。如果要保存项目，并且File Watcher错误地显示某些内容已更改，则可以调整此值，在这种情况下，您将增加延迟，直到消息不再出现。默认值为2000ms，但是范围可以从1ms到10,000ms。
