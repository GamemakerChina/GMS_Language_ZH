输入偏好

输入 偏好控制着IDE中鼠标和键盘输入的某些方面。可用的选项有。

  • 双击时间慢（ms）。集成开发环境的某些区域需要较慢的双击速度才能正常工作，例如在重命名资源时（快速双击将打开资源编辑器，而慢速双击将允许你 重命名它）。)该选项允许你设置检测速度，默认值为500ms。
  • 滚动锁定时间（ms）。这个选项涉及到GameMaker Studio 2在滚动窗口重叠时应该等待的时间。基本上，当你在窗口 "A "中滚动时，光标可能会悬停在窗口 "B "中的另一个可滚动区域。 窗口 "B "中的另一个可滚动区域，所以这个首选项定义了在你停止滚动 "A "之后要等待多长时间才可以滚动 "B"，这样你就不会意外地开始滚动不同的面板。 默认值是500ms。
  • 平移时将光标环绕在屏幕上。启用后，当平移工作区时，这将使光标绕着屏幕边界移动。默认为启用。
  • 滚动速度：这控制了不同工作空间内的滚动速度。默认值是1。
  • 标签拖动距离取消最大化桌面（px）。当你将工作区标签从IDE的默认位置拖走时，如果当时只有一个工作区处于活动状态，那么GameMaker Studio 2窗口将取消最大化，并随着鼠标移动（如果你有多个工作区处于活动状态，那么将创建一个新的IDE实例来容纳工作区）。 移动（如果你有一个以上的工作区处于活动状态，将创建一个新的IDE实例来容纳工作区）。这个设置控制了在工作区识别标签之前你必须拖动它的距离，默认值是 50px。
  • 与窗口内容互动的最小缩放水平（%）。这个选项控制缩放的截止点，在这个截止点上，鼠标将不再能够与工作区的窗口进行互动（即：添加节点、拖动窗口等...）。默认的 默认值是40。
  • 鼠标滚轮缩放需要Ctrl键。当这个选项被选中时，所有的编辑器（和主工作区）都将通过 / + 鼠标滚轮 启用缩放，而不选中它将启用 鼠标滚轮缩放的MMB图标。
  • Ctrl + MMB来打开继承事件，而不是手动。当这个被选中时，在一个对象中按住 / + 鼠标中键 ，将打开对象的继承事件（如果它有 如果它有的话）而不是手册。
  • 为文本选择添加偏置，防止跳行。当这个被选中时，在任何文本窗口中选择单行时设想的行高将被填充，使其不容易出现垂直不精确的情况。默认情况下，这个功能是启用的。 默认启用。
  • 在所有设备上启用笔记本模式。勾选此项将显示IDE上的笔记本模式按钮，使你能够根据需要打开或关闭它，无论运行的设备是否是 GameMaker Studio 2是否是一台笔记本电脑。默认情况下，这个选项是不被选中的。
