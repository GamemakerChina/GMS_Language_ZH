输入首选项

输入首选项控制IDE中鼠标和键盘输入的某些方面。可用的选项有：

  • 缓慢的双击时间（ms） ：IDE的某些区域需要较慢的双击才能正常工作，例如在重命名资源时（快速双击将打开资源编辑器，而缓慢的双击将允许您重命名它）。此选项允许您设置检测速度，默认值为500ms。
  • 滚动锁定时间（毫秒） ：此选项用于处理GameMaker Studio 2在重叠的滚动窗口之间应等待的时间。本质上，当您在窗口“ A”中滚动时，光标可能会悬停在窗口“ B”中的另一个可滚动区域上，因此此首选项定义了停止滚动“ A”后必须等待多长时间才能再滚动“ B”，因此您不会意外开始滚动其他面板。默认值为500ms。
  • 平移时将光标环绕在屏幕周围：启用后，将在平移工作区时将光标环绕在屏幕边界周围。默认启用。
  • 滚动速度：控制不同工作区中的滚动速度。预设值是1。
  • 选项卡拖动到未最大化桌面的距离（px） ：当将工作区选项卡拖离IDE中其默认位置时，如果当时只有一个工作区处于活动状态，则GameMaker Studio 2窗口将取消最大化并随其移动鼠标（如果您有多个活动的工作空间，则将创建一个IDE的新实例来容纳该工作空间）。此设置控制在工作空间识别该选项卡之前必须将其拖到多远，默认值为50px。
  • 与窗口内容进行交互的最小缩放级别（％） ：此选项控制鼠标将不再与工作区窗口进行交互的缩放截止点（即：添加节点，拖动窗口等）。默认值为40。
  • 鼠标滚轮缩放需要Ctrl键：选中此选项后，所有编辑器（和主工作区）将启用/ +鼠标滚轮，取消选中该选项将仅使用鼠标滚轮MMB图标启用缩放。
  • Ctrl + MMB打开继承的事件，而不是手册：选中后，按住/ +鼠标中键在一个对象中将打开该对象的继承事件（如果有的话）而不是手动事件。
  • 为文本选择添加偏倚以防止行跳动：选中此选项时，将填充在任何文本窗口中选择单个行时所设想的行高，从而减少了垂直不精确的可能性。默认情况下启用。
  • 在所有设备上启用笔记本电脑模式：选中此选项将在IDE上显示“笔记本电脑模式” GameMaker Studio 2的设备是否是笔记本电脑，您都可以根据需要将其打开或关闭。默认情况下未选中。
