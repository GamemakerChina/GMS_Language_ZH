一般首选项

常规首选项GameMaker Studio 2 IDE的常规行为。主页上列出了最常用的子菜单，而其他许多子菜单又分为了多个子类别，在此页面中进一步进行了说明。

  • IDE语言：此选项允许您从提供的选项列表中选择IDE的语言。默认为英语。
  • IDE外观：在这里，您可以从IDE可以使用的不同外观中进行选择。默认情况下， GameMaker Studio 2带有两种皮肤，一种是浅色，一种是深色，初始设置使用深色皮肤。
  • GameMaker Studio 2编辑器的最大帧速率：此选项允许您设置GameMaker Studio 2 IDE将更新的最大帧速率，允许您设置帧速率以匹配显示器的更新速度。预设值为60。
  • 默认音频设备：您可以在此处选择GameMaker Studio 2 IDE使用的默认音频设备。您可以通过单击刷新按钮来刷新列表。 。
  • 对退出确认的自动响应：此选项允许您设置退出GameMaker Studio 2时是否应显示确认消息。默认情况下，它设置为“显示消息”，但可以设置为跳过消息，就像单击“确定”一样。
  • 对系统字体更改对话框的自动响应：此选项允许您设置在添加，删除或编辑系统字体时是否应显示确认消息。默认情况下，它设置为“显示消息”，但可以设置为跳过消息，就像单击“确定”一样。
  • 启用通知：选中此选项将使IDE能够显示有关新runtimes通知，等等。默认情况下处于启用状态。
  • 通知超时（毫秒） ：设置显示通知的时间长度（毫秒）。默认值为6000。
  • 退出时注销：如果选中此选项，则当GameMaker Studio 2关闭时，IDE将自动注销用户会话。这主要针对教育用户，因为-选中时-这意味着每当程序关闭时，用户都将注销，因此释放了会议席位供其他用户使用。默认情况下是关闭的。
  • 启用DPI覆盖：通过此选项，您可以使用自己的覆盖默认DPI设置。您必须先启用该选项，然后设置Raw DPi Override值（默认值为96）或设置本机DPi的百分比（默认值100％），更改后将提示您重新启动IDE，可能需要您保存在当前打开的项目上完成的所有工作。请注意，DPi的最小百分比为50％，最大百分比为500％，如果将鼠标悬停在信息图标上，则可以看到所使用显示器的建议最大设置：

    请注意，如果您超过显示器的建议DPi设置，将显示一条通知，允许您重置DPi。
  • 启动时显示欢迎屏幕：默认情况下GameMaker Studio 2在启动程序时显示简短的启动屏幕。取消选中此选项将阻止显示初始屏幕。
  • 允许通过快捷方式关闭空白标签页：您可以使用键盘快捷键关闭所有打开的标签页/ + W ，但如果禁用此选项，则该快捷方式将不再起作用。默认情况下启用。
  • 打开新资源进行编辑，以将文件拖到IDE中：可以将某些资源（如精灵或声音）拖到GameMaker Studio 2 IDE中的主工作区上，并将被检测并作为新资源添加到资源浏览器中。默认情况下，也会在发生这种情况时打开相应的资源编辑器进行编辑，但是取消选中此选项将阻止打开编辑器，并且需要您手动打开它们。
  • 打开新资源以进行创建时编辑：如果选中此选项，则每次在资源浏览器中创建新资源时，都会自动打开相应的资源编辑器。默认情况下已选中此选项，取消选中它意味着您必须手动打开所有新创建的资源。
  • 自动重新加载已更改的文件：如果选中此选项，则IDE将自动扫描并检查文件，以查看是否有任何文件在GameMaker Studio 2之外进行了更改，如果有，请重新加载。
  • 禁用IDE过渡动画： GameMaker Studio 2 IDE拥有许多用于打开窗口等功能的动画过渡，但是它们并不符合所有人的喜好，如果在较旧或动力不足的计算机上禁用它们，则可能会提高性能如果它们被禁用。您可以在此处禁用它们，并且默认情况下它们处于启用状态。
  • 选择加入反馈调查：选中此选项时，您是在告诉GameMaker Studio 2您希望参加反馈调查，这意味着在工作时偶尔会显示一条有关您正在研究的领域的消息，并要求评分和回馈。默认情况下，此选项处于启用状态。

上面的选项是“常规”首选项中可用的主要选项，但是在下面列出的子类别中，您可以使用许多其他选项：

  • 背景
  • 编译中
  • 确认对话框
  • 调试
  • 文件观察者
  • 去
  • 帮助
  • 输入
  • 输出Windows
  • 路径
  • 省电
  • 最近的Windows
  • 资源
  • 工作空间
