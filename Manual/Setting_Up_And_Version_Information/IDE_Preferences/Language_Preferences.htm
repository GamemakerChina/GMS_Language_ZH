语言偏好

语言首选项用于定义如何在GameMaker Studio 2的IDE中显示一些编程的语言功能。GML的选项如下。

  • 在状态栏中显示参数类型。这个选项与你使用自己的自定义脚本并使用JSDoc符号来识别不同的参数等有关......（见这里的 更多信息）。)当这个选项被选中，并且你已经在脚本中指定了参数类型，那么它们将显示在编辑器底部的状态栏中，用 {} 。这在默认情况下是关闭的。
  • 在状态栏中显示参数描述。如果勾选了这个选项，那么你用 JSDoc 符号给你的自定义脚本的任何描述都会显示在编辑器窗口底部的状态栏中。这在默认情况下是关闭的。
  • 显示系统功能的美式拼写。默认情况下，GameMaker Studio 2使用GB英文拼写的函数，例如： draw_text_colour() ，但这可能与你的习惯不同，所以你可以使用这个选项切换到美式拼写，这样函数就变成了 。 你可以使用这个选项切换到美式拼写，比如说函数变成 draw_text_color() 。这在默认情况下是关闭的。
  • 显示的最大语法错误数。该值设置在任何时候显示的最大语法错误数。如果你发现IDE很慢，那么你可以试着减少这个值。默认情况下，该值被设置为50。
  • 启用变量引用警告。当启用时，这将标志着任何潜在的变量问题，比如当一个变量被定义在某处，但实际上在代码中没有被使用。如果禁用，这些警告将不再被 显示。默认情况下，这个功能是启用的。
