语言首选项

语言首选项GameMaker Studio 2的IDE中显示一些用于编程的语言功能。 GML的选项如下：

  • 在状态栏中显示参数类型：此选项与您使用自己的自定义脚本并且使用JSDoc表示法标识不同的参数等有关（请参阅此处以了解更多信息）。选中此选项并在脚本中指定了参数类型后，它们将显示在编辑器底部的状态栏中，并用 {}。默认情况下是关闭的。
  • 在状态栏中显示参数说明：如果选中此选项，则使用JSDoc标记为您的自定义脚本提供的任何说明都将显示在编辑器窗口底部的状态栏中。默认情况下是关闭的。
  • 对系统功能显示美国拼写：默认情况下， GameMaker Studio 2对功能使用GB英文拼写 draw_text_colour()，但这可能不完全是您所习惯的，因此您可以使用此选项切换到美国拼写，以便功能变为 draw_text_color()， 例如。默认情况下是关闭的。
  • 要显示的最大语法错误数：此值设置一次将显示的最大语法错误数。如果发现IDE落后，则可以尝试减小此值。默认情况下设置为50。
  • 启用变量引用警告：启用后，它将标记变量的任何潜在问题，例如已在某个地方定义了变量但未在代码中实际使用的情况。禁用后，这些警告将不再显示。默认为启用。
