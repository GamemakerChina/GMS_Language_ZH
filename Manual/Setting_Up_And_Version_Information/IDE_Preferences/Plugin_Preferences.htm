插件首选项

插件首选项用于控制已添加到GameMaker Studio 2 IDE中的所有其他插件。默认情况下，该部分专门用于源代码控制插件，用于将GameMaker Studio 2项目与Git集成。

源代码控制（ Git ）

本节介绍您需要设置的首选项，以使Git SCM插件与GameMaker Studio 2 。首先，你必须设定G的ENERAL选项：

  • 自动打开源代码控制输出窗口：选中此选项时，发生某些更改时，将自动显示源代码控制输出窗口。默认情况下处于启用状态。
  • 详细日志记录Git输出日志启用详细日志记录。默认情况下是关闭的。

接下来是“用户身份”部分，其中包含以下必填字段：

  • 提交为的用户名：在这里，您必须输入用于对Git进行任何提交的用户名（该名称必须与设置Git存储库时指定的用户名相同）。
  • 提交为的电子邮件：与该Git身份相关联的电子邮件。

设置用户身份后，您可以通过单击标记为添加新用户/密码身份验证的按钮来添加新的用户名/密码组合以进行身份验证，这将打开以下窗口，其中包含不同的字段供您填写：

您也可以通过单击带有该标签的按钮来添加新的SSH密钥对身份验证。这将打开以下窗口供您填写：

在这些按钮下方，您可以使用以下选项设置SCM的合并工具和差异工具：

  • 合并工具的路径：在此，您可以指定Git合并工具的完整文件路径。
  • 合并工具选项：在此字段中，您可以添加任何在使用合并工具时要运行的其他命令。
  • 路径比较工具：在这里，你指定完整的文件路径Git比较工具。
  • 差异工具选项：在此字段中，您可以添加任何其他命令，以在您使用差异工具时运行。

最后，设置完这些后，您还有最后一个选择：

  • 自动启用源代码管理：选中此选项后，将自动为所有新项目启用源代码管理。默认情况下是关闭的。
