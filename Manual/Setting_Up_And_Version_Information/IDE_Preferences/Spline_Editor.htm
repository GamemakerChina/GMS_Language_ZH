样条线编辑器首选项

样条曲线编辑器首选项主要用于定义“路径编辑器”中的平滑路径外观，尽管它们会影响IDE的某些其他方面（例如“物理形状编辑器”）。存在以下选项：

  • 插入路径点精度（px） ：设置在路径上插入点的精度值（以像素为单位），即：鼠标必须在此范围内才能插入点，而不是添加新点。默认值为20px。
  • 锚点宽度：以像素为单位设置锚点的宽度。默认是8px;
  • 锚点高度：设置锚点的高度，以像素为单位。默认是8px;
  • 路径颜色：编辑时用于显示路径的颜色。默认值为（ ARGB ）#FFFFFFFF。
  • 高亮的路径段颜色：用于突出显示任何选定的路径段的颜色。默认值为（ARGB）#FFFFFFFF。
  • 无效的路径颜色：用于突出显示无效路径的颜色。默认值为（ARGB）＃FF0000FF。
  • 锚点颜色：用于绘制路径锚点的颜色。默认值为（ARGB）＃FFC0C0C0。
  • 无效的锚点颜色：用于绘制路径锚点的颜色。默认值为（ARGB）＃FF800000。
  • 选定的锚点颜色：用于绘制已突出显示（选定）的路径锚点的颜色。默认值为（ARGB）＃FF00A4FF。
  • 选定的无效锚点颜色：用于绘制突出显示但无效的锚点的颜色。默认值为（ARGB）＃FFA400FF。
  • 鼠标悬停在锚点颜色上：用于绘制处于活动状态的锚点的颜色。默认值为（ARGB）#FFFFFFFF。
  • 鼠标悬停在无效的锚点颜色上：用于绘制处于活动状态但无效的锚点的颜色。默认值为（ARGB）＃FF0000FF。
  • 蒙版覆盖色：如果启用了蒙版，则设置蒙版的覆盖色（请参见上文）。默认值为（ARGB）＃FF404040。
