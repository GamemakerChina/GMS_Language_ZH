雪碧编辑器首选项

Sprite Editor首选项用于定义Sprite Editor的外观和功能。有以下选项。

  • 默认精灵原点。这个选项允许你为你创建的任何新的精灵资源设置默认原点位置。默认值是左上角。
  • 默认的精灵碰撞掩码种类。这个选项允许你设置新精灵使用的默认碰撞掩码种类。选择是矩形、旋转矩形、椭圆、钻石或精确/每帧精确。默认是矩形。
  • 默认的精灵动画模式。这个选项允许你在精灵编辑器中设置精灵的默认动画模式。你可以选择 "每秒帧数 "或"每游戏帧数"。 默认设置是 "每秒一帧"。
  • 默认精灵动画速度：这个选项允许你设置创建新精灵资产时的默认动画速度值。请注意，这个值与动画模式有关，例如，每秒3帧的动画速度与每一帧30帧的动画速度不同。 并不等同于每游戏帧30帧，所以一定要参照首选模式正确设置这个值。默认值是30。
  • 默认精灵宽度：该选项允许你为所有新创建的精灵设置默认宽度。默认值是64。
  • 默认的精灵高度。这个选项允许你为所有新的精灵设置默认高度。默认值是64。
  • 对精灵原点应用调整大小和缩放。勾选该选项后，当精灵被调整大小或缩放时，将改变精灵的原点，新的原点被移动/缩放以匹配精灵的新值。需要注意的是 这只适用于未在精灵编辑器中锁定的自定义精灵原点。如果原点被锁定或者精灵使用了一个默认的位置--比如："中心 "或 "左上角"，等等--那么这个选项将 就会变成灰色，无法改变。默认为启用。

Sprite Editor还存在以下子类别。

确认对话

这里的选项可以控制在需要确认一个动作时弹出的信息。你可以改变这些设置，使其始终显示信息，或通过特定的操作（是或否）来取消它。

  • 对文件导入的自动响应。当你导入一个新的图像并且已经存在一个图像时，默认情况下，你会被提示继续导入并覆盖现有的图像或取消该操作。这个选项可以让你抑制这个信息，并且 始终通过选择 "No "取消导入，或者通过选择 "Yes "覆盖当前精灵。默认值是 "显示信息"。
  • 对图像加载失败的自动响应。当你导入一个新的图像，并且导入失败时，默认情况下，你会看到一个警告信息，告知你有一个失败。这个选项让你通过选择 "好 "来抑制这个信息。 默认值是 "显示信息"。
