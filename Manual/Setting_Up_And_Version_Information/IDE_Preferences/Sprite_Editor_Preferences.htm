精灵编辑器偏好

Sprite Editor首选项用于定义Sprite Editor的外观和功能。存在以下选项：

  • 默认Sprite原点：此选项允许您为创建的任何新Sprite资源设置默认原点位置。默认值为左上角。
  • 默认的Sprite碰撞遮罩种类：此选项允许您设置要用于新Sprite的默认的碰撞遮罩种类。选择是矩形，旋转矩形，椭圆，菱形或每帧精确/精确。默认值为矩形。
  • 默认的sprite动画模式：此选项允许您在sprite编辑器中为sprite设置默认的动画模式。您可以选择“每秒帧数”或“每游戏帧数”，默认设置为“每秒帧数”。
  • 默认的Sprite动画速度：此选项允许您设置创建新的Sprite资产时的默认动画速度值。请注意，此值链接到动画模式，例如-每秒3帧与每个游戏帧30帧不同，因此请确保参考首选模式正确设置此值。预设值为30。
  • 默认精灵宽度：此选项允许您为所有创建的新精灵设置默认宽度。默认值为64。
  • 默认精灵高度：此选项允许您为所有新精灵设置默认高度。默认值为64。
  • 将调整大小和缩放比例应用于子画面原点：选中此选项将在调整大小或缩放子画面时更改子画面的原点，并移动/缩放新原点以匹配子画面的新值。请注意，这仅适用于未在Sprite编辑器中锁定的自定义Sprite原点。如果原点被锁定，或者精灵使用默认位置之一（例如：“中心”或“左上角”等），则此选项将显示为灰色，并且不可更改。默认启用。

Sprite编辑器还存在以下子类别：

确认对话框

此处可用的选项控制在需要确认操作时弹出的消息。您可以更改以下设置以始终显示该消息，或通过特定操作（是或否）将其关闭：

  • 对文件导入的自动响应：导入新映像时，默认情况下会提示您继续导入并覆盖现有映像或取消操作。此选项使您可以禁止显示此消息，并且始终可以通过选择“否”来取消导入，或者通过选择“是”来覆盖当前的精灵。默认值为“显示消息”。
  • 对映像加载失败的自动响应：当您导入新映像并且导入失败时，默认情况下会显示一条警告消息，通知您发生了故障。此选项使您可以通过选择“确定”来禁止显示此消息。默认值为“显示消息”。
