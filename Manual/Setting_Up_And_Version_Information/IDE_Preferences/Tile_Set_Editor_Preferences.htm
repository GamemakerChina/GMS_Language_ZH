拼贴集编辑器首选项

“图块集编辑器首选项”具有以下用于自动平铺的选项：

自动平铺背景颜色：此选项可在创建自动平铺部分中的平铺集编辑器中设置背景颜色。默认颜色是（rgba）＃808080FF。

“图块集编辑器”首选项还具有“确认对话框”子部分：

此处可用的选项控制在更改图块集的像元大小时弹出的消息，这将影响房间图块地图图层的呈现方式（导致裁剪的图层）。默认情况下，系统将提示您继续或取消该操作，并且此选项使您可以通过选择“确定”来取消显示此消息，该操作将继续进行图块集单元格更改，而不考虑对房间图块地图图层的影响。默认值为“显示消息”。
