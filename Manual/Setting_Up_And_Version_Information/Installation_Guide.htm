安装指南

GameMaker Studio 2 macOS和Windows台式计算机上使用而设计的，此处具有最低要求。

您也可以下载GameMaker Studio 2从您的安装YoYo的账户页面YoYo Games网站。安装完成后，首次运行GameMaker Studio 2 ，系统可能会提示您进行更新，然后才能使用该产品。安装任何更新后，将需要您提供YoYo Games帐户的许可详细信息（请参阅此处以获取更多信息），然后才能使用该产品，这意味着您首次使用GameMaker Studio 2时必须具有互联网连接。初始登录后，您应该可以脱机使用该产品，但是它会定期要求您进行连接以验证登录帐户。

值得注意的是-根据您拥有的许可证类型-您可能需要安装其他组件或按照其他步骤来针对特定平台GameMaker Studio 2有关每个平台要求的完整信息，请参见YoYo Games帮助台上的本文：GMS2必需的SDK 。

稳定版和Beta版

YoYo Games网页上下载的版本GameMaker Studio 2的当前稳定版本，但也可能有另一个“ Beta ”版本可用，该版本可用于展示或测试新功能，然后再进行最终稳定的构建。完整版本详细信息如下：

  • 稳定：此版本是您将用于一般开发的版本，建议与任何商业产品一起使用。它可能缺少Beta版本中可用的某些功能，但这是因为它们尚未经过全面测试或需要其他工作。
  • Beta GameMaker Studio 2的Beta版本适用于那些希望在将新功能添加到Stable版本之前尝试新功能的用户，或需要解决此问题的用户。不建议您将Beta版本用于最终游戏制作，除非它包含一个重要的修复程序，该错误修复了项目的负担。您可以从您的YoYo GameMaker Studio 2的Beta版，并且可以将其安装并与Stable版同时运行。请注意，如果您通过Steam GameMaker Studio 2 ，则尽管仍可以下载并使用桌面版本，但是您Steam本身访问该产品的Beta版本。
