许可证信息

一旦你下载并安装了GameMaker Studio 2，第一次使用它时，你将被要求输入你的许可证详细信息。

你的许可证与一个YoYo账户（YYA）绑定，你必须建立一个账户才能使用GameMaker Studio 2。这个帐户保存着您拥有的YoYo Games产品的所有许可信息，同时也用于 YoYo Games 市场.你可以在以下页面找到更多关于YYA的信息 的更多信息，请访问以下页面。

  • 创建一个悠游账户

重要的是!在购买GameMaker Studio 2的任何新许可证后，我们建议你退出IDE，然后重新启动GameMaker Studio 2 ，并再次登录，以确保新许可证 正确应用。你可以从文件菜单中注销。

代理人支持

如果你在安装GameMaker Studio 2时需要代理支持，可以通过点击代理来添加。这将弹出以下窗口。

你现在可以添加你的代理细节。上面显示的一些字段是可选的，如果有一个你不需要的字段 则只需将该字段留空。在继续之前，值得注意的是，如果您在Windows互联网选项中设置了一个代理，并且 它不需要用户名/密码，那么它就应该 "按原样"，你可能不需要在GameMakerStudio 2中设置代理。

如果你的代理需要一个用户名和/或密码，那么你需要将它们添加到上面窗口中的字段中。请注意，在这些情况下，用户名/密码/代理详细信息将始终是必需的，并且代理IP也应添加端口。 添加端口，例如： xx.xx.xx.xx:port 。

一旦你添加了相关信息，点击 "好"，然后像平常一样登录GameMaker Studio 2。
