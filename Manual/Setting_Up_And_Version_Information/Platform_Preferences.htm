平台偏好

手册的此部分介绍了在将游戏编译为特定目标之前需要设置的不同平台特定的首选项。这些通常只需要设置一次即可开始（尽管将来对runtimes更新可能需要您进行某些更改，但是这些将在发行说明中提到），并且用于指向所有必需的SDK并设置为基本您希望GameMaker Studio 2如何为任何给定平台编译游戏项目的编译选项。请注意，可用的平台将取决于您拥有的许可证，并且可能并非全部可用。

以下是标准许可的所有可用目标平台的列表：

  • Android
  • HTML5
  • iOS
  • tvOS
  • macOS
  • Steam
  • Windows
  • Windows UWP

请注意，除了设置这些首选项外，您还需要设置某些平台针对的设备，否则您可能无法编译或测试项目。有关更多信息，请参见“编译”部分和“设备管理器”部分。
