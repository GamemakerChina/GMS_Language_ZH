Windows首选项

Windows首选项具有以下选项：

  • Visual Studio路径：此选项用于将GameMaker Studio 2指向构建PC上的MS Visual Studio
  • 默认包装类型：构建最终的Windows可执行程序包时，可以选择创建安装程序（使用NSIS ）还是压缩文件 ZIP 该文件将包含游戏以及其中的所有其他文件。默认情况下，您可以选择在编译时要选择的两个选项中的一个，但是可以将此首选项设置为始终使用一个或另一个。请注意，当此选项设置为Show Dialogue时，在编译时弹出的实际对话框窗口中，选择“ Remember Packaging Option”会将此首选项设置为您选择的选项。
