发布说明

GameMaker Studio 2的每个版本都有一套自己的发行说明。这些说明包含了关于所做修改的详细信息，以及所有已修复的错误的完整列表。这是一个宝贵的工具，用于决定 是否更新你的当前版本，或者当你在发布后发现一些可能是错误的东西时，这是一个宝贵的工具。
