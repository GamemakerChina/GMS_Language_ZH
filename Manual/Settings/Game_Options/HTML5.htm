HTML5游戏选项

本节概述了可用于控制如何编译HTML5游戏项目的不同选项。不同的部分是：

一般的

在此部分中，您将具有用于填写有关游戏的以下信息的字段：

  • 浏览器标题：您可以在此处输入要用于浏览器中游戏选项卡标题栏的文本。
  • 版本：您项目的版本号。
  • 文件夹名称：将存储HTML5游戏文件的文件夹的名称（默认为“html5game ”）。
  • 输出名称：这是输出HTML文件名，可以更改为默认的“index.html ”。

在游戏信息字段下方，您也具有以下选项：

  • 输出调试到控制台：这会将所有调试输出发送到浏览器的JS控制台。默认情况下是关闭的。
  • Display Cursor（显示光标） ：当鼠标光标悬停在游戏画布上时，它将禁用鼠标光标。默认情况下处于启用状态。
  • 显示“正在服务器外部运行”警报：如果您正在本地测试游戏（即：从计算机而不是服务器上单击索引文件-不推荐），那么通常会收到一条警告消息，通知您事情将无法进行您可能会因为浏览器对本地文件的安全性而感到期待。您可以通过取消选中此选项来禁止显示此警告。默认情况下它是打开的。

在常规设置下，您具有高级选项，它们是：

  • 包含的文件为index.html ：用于添加自定义的“ index.html”文件，以代替GameMaker Studio 2默认情况下生成的预定义文件。如果您已实施网站设计或要在所有HTML5版本中包含某些功能，则此功能将非常有用。要使用的文件必须作为包含文件包含在游戏项目中。
  • 加载栏扩展：这将更改默认扩展栏，该默认加载栏已作为扩展GameMaker Studio 2项目中。
  • 前置输出.js ：您可以在此处前置 *.js输出游戏所需的任何其他JavaScript。例如，Chrome应用商店应用可能要求您添加 window.localStorage = undefined;到输出JS，您可以使用此选项。

分析工具

本部分允许您向游戏中添加分析，从而允许您使用某些第三方产品来跟踪您的游戏及其销售或游戏。您可以指定使用Flurry Analytics还是Google Analytics（分析）来跟踪您的游戏。由于这些功能的性质，您一次只能激活一个，并且标记了Enable Flurry Analytics或Enable Google Analytics（分析）后，还应该将分配给您的唯一ID添加到适当的区域。

有关可用于发送分析数据的功能的信息，请参阅此处。

图形

在这里，您可以更改与游戏显示方式相关的以下详细信息（请注意，并非所有浏览器都以相同的方式显示游戏，因此请在尽可能多的浏览器上测试这些选项，以确保它们具有理想的效果） ：

  • 允许切换到全屏模式：选中此选项将允许用户使用以下命令将游戏从窗口模式切换到全屏模式/ +“F在Mac和 F10在Windows 。默认情况下处于启用状态。
  • 在像素之间插值颜色：打开线性插值，基本上可以“平滑”像素。对于清晰的像素图形，应该将其关闭，但是如果您具有不错的alpha混合和平滑的边缘图形，则最好保留它。默认情况下处于启用状态。
  • 在浏览器中居中游戏：选中此项将自动在运行游戏的浏览器中使游戏屏幕居中。默认情况下是关闭的。
  • 使用内置粒子：此选项将导出内置粒子系统所需的精灵以及您的游戏。这些精灵存储在单独的纹理页面中，这意味着它们不是为此平台生成粒子的最有效方法，因此建议（除非使用WebGL）取消选中此选项并使用自己的粒子精灵，除非一起存储在主纹理页面上，供您的游戏使用。默认情况下处于启用状态。
  • 使用内置字体：此选项将导出GameMaker Studio 2内置字体以及其他游戏文件。这会稍微增加纹理页面的大小，并需要少量的额外内存和存储空间，因此，如果您的游戏不使用内置的Arial字体，则应将其禁用。默认情况下它是打开的。
  • WebGL ：选择是关闭WebGL支持，使其成为可选（自动检测）还是使其成为游戏的要求。应当注意，要在所有情况下测试性能，都应将其禁用，然后通常将其设置为自动检测以进行发布。使其成为一项要求可能会大大降低您的游戏与不同浏览器类型的兼容性。默认设置为自动检测。
  • 缩放：您可以将游戏配置为自动缩放绘图画布，以在浏览器中保持宽高比，也可以选择使其全屏运行。完全缩放选项不会在浏览器中全屏显示游戏，而是拉伸所绘制的内容以适合游戏第一个房间所定义的画布大小。默认情况下设置为保持宽高比。

最后，可以选择设置纹理页面的大小。默认（也是最兼容）大小是2048x2048，但是您可以选择256x256到8192x8192之间的任意值。还有一个标记为“预览”的按钮，它将为该平台生成纹理页面，然后打开一个窗口，以便您查看它们的外观。如果希望查看纹理页面的结构并防止纹理页面大于（或小于）所需的页面，这将非常有用。有关纹理页面的更多信息，请参见此处。

注意：请注意，纹理页面的大小越大，游戏与不同浏览器和设备的兼容性越差。

图片

在此部分中，您可以定义Favicon和启动画面。该图标应被创作为 *.ico文件，大小可以为16x16、32x32、48x48或64x64像素，颜色深度为8位，24位或32位。初始屏幕应为将容纳游戏的画布大小，如果您选中“使用初始屏幕”选项，则初始屏幕将覆盖所有加载栏（即：您可以具有加载栏，也可以具有初始屏幕，但不能同时具有两者） 。允许使用较小或较大的图像，但会进行缩放以适合需要的图像。初始屏幕应为 *.png图像。

值得注意的是， GameMaker Studio 2具有项目图像生成器工具，该工具可用于自动创建将游戏编译到的所有不同目标平台所需的所有图像。如果使用此工具，则应修改创建的图像，以确保它们是您所需要的。

社会的

社交标签用于在游戏中设置Facebook通信。如果需要Facebook功能，则需要单击“使用Facebook”复选框，然后提供Facebook App ID和Facebook App显示名称。请注意，所有Facebook功能都可以通过扩展程序访问，并且当您首次选中“使用Facebook”框时，系统会提示您从Marketplace上下载并安装Facebook扩展程序（可能需要登录到YoYo帐户才能下载）。 。
