最后，你可以选择禁用文件系统沙盒。勾选这个选项将禁用GameMaker Studio 2对保存和加载文件的沙盒控制，允许你的游戏从运行游戏的系统的任何地方访问文件。 这个选项在默认情况下是不被选中的，因为我们建议你在所有的系统上保持GameMaker Studio 2的文件沙盒，当被选中时，一些保存/加载区域可能仍然不可用，这取决于操作系统级别的权限。关于沙盒的更多信息 关于沙盒的更多信息，请参见GameMaker Studio 2 文件系统的页面。

图形

图形选项是那些你应该配置的选项，以决定你的游戏如何使用目标Ubuntu机器的显卡。 包括以下选项供你修改。

  • 开始全屏。如果选中这个选项，游戏将以全屏模式启动，否则将以窗口模式启动。默认情况下这是关闭的。
  • 允许全屏切换：勾选此项后，用户可以使用标准的Ubuntu操作系统快捷键从全屏切换到窗口，然后再返回。默认情况下这是关闭的。
  • 在像素之间进行颜色插值。打开线性插值，这基本上是 "平滑 "像素。对于清晰的像素图形，它应该是关闭的，但如果你有漂亮的阿尔法混合和平滑边缘的图形，它最好是打开。 如果你有很好的阿尔法混合和平滑边缘的图形，最好打开它。
  • 显示光标。当这个选项被启用时，默认的操作系统光标将显示在你的游戏中。如果这个选项被禁用，那么当鼠标在游戏窗口上时，将没有可见的光标，除非你在游戏项目中创建了一个光标。 的一部分（默认情况下是打开的）。
  • 使用同步来避免撕裂。这将切换v-sync的开启或关闭。注意，如果你有一个房间速度为120的游戏，而玩家有一个刷新率为60的显示器，把这个选项打开也会把你的游戏速度锁定为60。 60，打开这个选项将把你的游戏速度也锁定为60。这在默认情况下是关闭的。
  • 允许窗口调整大小。勾选此选项允许用户改变游戏窗口的大小（无边界窗口选项需要关闭才能发挥作用）。该选项默认为关闭。
  • 缩放。在这里你可以选择保持长宽比（所以4:3的房间在16:9的屏幕上会被 "字母框住"）或完全缩放（拉伸图像以适应全屏）。

    警告! 关闭应用程序表面将禁用Windows游戏选项中设置的所有缩放选项，直到它被重新打开。更多的细节请看应用程序表面。

最后是设置纹理页的大小的选项。默认（也是最兼容的）尺寸是2048x2048，但你可以从256x256到8192x8192之间的任何地方进行选择。在这里 也有一个按钮标明 预览 的按钮，它将为这个平台生成纹理页，然后打开一个窗口，以便你能看到它们的样子。如果你想看看纹理页是如何结构的，这可能非常有用。 结构，并防止纹理页大于（或小于）所需。关于纹理页的更多信息，请看这里。

注意：请注意，纹理页的尺寸越大，你的游戏在规格较低的PC上的兼容性就越差。

图片

这里你可以为你的游戏添加一个闪屏，并设置将使用的游戏图标 。该图标必须是 一个64x64像素的图片，格式为 .png ，并且闪屏应该至少与你游戏中的第一个房间（或它的视图端口）相同大小，也是 .png 。闪屏 将在游戏加载时显示，可以通过勾选使用闪屏选项（默认为关闭）来启用或禁用。

值得注意的是，GameMaker Studio 2有一个项目图像生成器工具，可以用来自动创建所有不同的目标平台所需的图像。 被编译到的所有不同的目标平台上。如果你使用这个工具，你应该修改创建的图像以确保它们是你所需要的。
