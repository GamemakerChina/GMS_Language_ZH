Windows游戏选项

本节概述了可用于控制Windows游戏项目的编译方式的不同选项。不同的部分是：

一般的

Windows游戏的“常规”页面上，您应在其中提供必要的信息，以便Windows在安装并运行游戏时显示。此信息在“产品信息”部分中，应完整填写。

通过此窗口的“选项”部分，您可以选择要将游戏包安装到的位置，%localappdata% “ 或者 ”%appdata% ”文件夹，以及是否将“复制exe”设置为输出文件夹。通常，您永远不需要此选项，但它是为具有各种DLL的用户提供的，这些DLL彼此之间可以与您的游戏进行通信，并且在启用后它将保持测试时将游戏EXE放在输出文件夹中，而不是将其放在Temp文件夹中。

接下来，您可以设置Windows Sleep Margin 。此选项与减少在特定系统上运行游戏时的卡顿有关。基本上，如果您的游戏运行速度快于您的房间速度， GameMaker Studio 2将在剩余时间内“休眠”，但是这种休眠可能会非常不准确，并且您通常可能会结束不必要的长时间睡眠，从而导致帧花费更长的时间。导致口吃。为了解决这个问题，我们可以睡更少的时间，然后在剩下的时间里紧紧地坐一圈以使其更加准确-尽管坐在一个循环中的问题是会导致CPU使用率增加，这反过来又可能导致CPU温度升高，风扇运行速度加快。默认情况下，此值设置为10，在99.99％的情况下，您可以使用此值，但对于低端计算机或运行大量后台进程的计算机，这可能不是理想的解决方案（或高端计算机也是如此），并且可能需要1或15的值。请注意，尽管这很大程度上是系统特定的配置，并且您的构建机器上可以正常运行的配置可能不适用于其他用户，因此，如果有疑问，请将其设置为10。

注意：除了“睡眠裕量”之外，您还可以通过在runtime调整线程调度程序的分辨率Windows上的性能和功耗。

下一个选项是为您的游戏启用Steam如果启用此选项，请确保已Steam Works SDK（请参阅此处），并且已将Steam App ID添加到游戏的“常规选项”中。请注意，如果您正在Windows并从Steam GameMaker Studio 2 ，并且正在为Steam开发游戏，那么您可能还需要选中“ Steam应用程序使用替代启动器”选项。该选项的作用是在测试时使用备用启动程序启动游戏，以防止Steam使用GameMaker Studio 2的应用程序ID（而不是游戏自己的应用程序ID）标记游戏。 Steam开发时，并且发现启动游戏时遇到问题时，才应启用此功能。

倒数第二个选项是“禁用文件系统沙箱”选项。选中此选项将禁用GameMaker Studio 2沙箱来保存和加载文件，从而允许您的游戏从运行游戏的系统上的任何位置访问文件。默认情况下，此选项为未选中状态，因为我们建议您GameMaker Studio 2沙箱，并且选中该选项后，根据操作系统级别的权限，某些保存/加载区域可能仍然不可用。有关沙盒的更多信息，请参阅GameMaker Studio 2文件系统上的页面。

最后，您可以选择让GameMaker Studio 2 Windows项目创建一个x64可执行文件。启用后，此选项将在编译时为您的游戏创建一个64位可执行文件，我们强烈建议所有新项目都使用此选项。从长远来看，随着32位运行器逐渐被淘汰， Windows可执行文件的默认（唯一）格式。请注意，启用后，项目使用的任何扩展dll也将需要为64位（有关更多信息，请参见此处）。

图形

图形选项是您应该配置的选项，以确定您的游戏如何使用目标Windows PC的图形卡。包含以下选项供您修改：

  • 显示光标：选中此选项后，将显示常规的Windows光标，而取消选中将意味着不显示任何光标，除非您在游戏代码中创建了一个。默认情况下处于启用状态。
  • 启动全屏：如果选中此选项，则游戏将以全屏模式启动，否则将启动窗口显示。默认情况下是关闭的。
  • 允许全屏Switch Windows快捷方式从全屏切换到窗口并再次返回。默认情况下是关闭的。
  • 在像素之间插值颜色：打开线性插值，基本上可以“平滑”像素。对于清晰的像素图形，应该将其关闭，但是如果您具有不错的alpha混合和平滑的边缘图形，则最好保留为默认状态。
  • 使用同步避免撕裂：这将打开或关闭垂直同步。请注意，如果您的游戏的室内速度为120，而玩家的监视器的刷新率为60，则打开此选项也将您的游戏速度也锁定为60。默认情况下是关闭的。
  • 允许窗口大小调整：选中此选项将允许用户更改游戏窗口的大小（“无边界窗口”选项必须关闭才能起作用）。默认情况下，此选项是关闭的。
  • 无边距窗口：选中此选项将使您的游戏在无边距窗口中运行，删除常规的最小化，最大化和关闭按钮以及游戏标题。默认情况下是关闭的。
  • 缩放：您可以在此处选择保持宽高比（这样4：3的房间将在16：9处“装箱”）或完全缩放（拉伸图像以适合全屏）。

    警告！ Switch荷兰国际集团关闭应用面将禁止在设置所有缩放选项Windows游戏选项，直到它已经切换回再次打开。有关更多详细信息，请参见应用程序表面。

最后，可以选择设置纹理页面的大小。默认（也是最兼容）大小是2048x2048，但是您可以选择256x256到8192x8192之间的任意值。还有一个标记为“预览”的按钮，它将为该平台生成纹理页面，然后打开一个窗口，以便您查看它们的外观。如果希望查看纹理页面的结构并防止纹理页面大于（或小于）所需的页面，这将非常有用。有关纹理页面的更多信息，请参见此处。

注意：请注意，纹理页面的尺寸越大，游戏在较低规格的PC上的兼容性越差。

图片

图像部分是您提供游戏所需图像的地方。对于Windows ，您需要提供一个Icon文件（该文件必须位于 .ico格式）和启动画面（可以是 .png， .bmp ， .jpg或者 .gif格式）。启动屏幕将在游戏加载时显示，并且可以通过选中“使用启动屏幕”选项（默认情况下处于禁用状态）来启用或禁用。

值得注意的是， GameMaker Studio 2具有项目图像生成器工具，该工具可用于自动创建将游戏编译到的所有不同目标平台所需的所有图像。如果使用此工具，则应修改创建的图像，以确保它们是您所需要的。

安装程序

可以使用安装程序Windows游戏，在这里您可以定义GameMaker Studio 2在创建它时将使用的各种设置和图形。首先，我们具有图形选项，这些选项将定义安装程序本身的外观：

  • 完成：出现在安装程序“完成”页面上的图形。必须在 *.bmp格式和尺寸为164x314px。
  • 标题：安装程序运行时显示的图形。必须在 *.bmp格式和尺寸为150x57px。

您还可以指向特定的许可协议GameMaker Studio 2提供了一个示例）和自定义的NSI脚本文件（在此处进行详细说明）（如果需要），或单击适当的按钮以返回GameMaker Studio 2 。
