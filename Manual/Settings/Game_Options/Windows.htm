接下来你可以设置Windows睡眠保证金。这个选项与减少在特定系统上运行游戏时的卡顿有关。基本上，如果你的游戏运行速度比你的房间速度快，GameMaker Studio 2将 "睡眠 "剩余时间。 剩余的时间，但是这种睡眠可能是相当不准确的，你经常会因为睡眠时间过长而导致你的画面耗时过长，从而导致卡顿现象。为了解决这个问题，我们可以减少睡眠时间，然后在余下的时间里进行严格的循环。 循环，以使其更加准确--尽管循环的问题是它会导致CPU使用率增加，这反过来又会导致你的CPU温度上升和风扇运行速度加快。默认情况下，这个值 设置为10，在99.99%的情况下，你会觉得很好，但对于低端机器或有大量后台进程运行的机器，这可能不是一个理想的解决方案（或对于非常高端的机器也是如此），一个1或15的值 的值可能是必要的。但请注意，这在很大程度上是一个系统特定的配置，在你的构建机器上有效的配置可能不适合其他用户，因此，如果有疑问，请将其设置为10。

注意：除了睡眠保证金外，你还可以通过在运行时调整线程调度器的分辨率来进一步控制Windows上的性能和功耗。

你拥有的倒数第二个选项是禁用文件系统沙盒的选项。勾选这个选项将禁用GameMaker Studio 2的保存和加载文件的沙盒功能，允许你的游戏从系统的任何地方访问文件。 运行游戏。这个选项默认是不被选中的，因为我们建议你在所有系统上保持GameMaker Studio 2的文件沙盒，当被选中时，一些保存/加载区域可能仍然不可用，这取决于操作系统级别的权限。 关于沙盒的更多信息，请参见GameMaker Studio 2 文件系统的页面。

图形

图形选项是那些你应该配置的选项，以确定你的游戏将如何使用你的目标Windows PC的显卡。 目标Windows电脑的显卡。包括以下选项供你修改。

  • 显示光标。当这个选项被选中时，常规的窗口光标将被显示出来，不选中它将意味着没有光标被显示，除非你在游戏代码中创建了一个光标。默认情况下，该选项是打开的。
  • 开始全屏。如果选中这个选项，游戏将以全屏模式启动，否则将以窗口模式启动。默认情况下这是关闭的。
  • 允许全屏切换：如果勾选了这个选项，用户可以使用标准的Windows快捷键从全屏切换到窗口，然后再回到全屏。这在默认情况下是关闭的。
  • 在像素之间进行颜色插值。打开线性插值，这基本上是 "平滑 "像素。对于清晰的像素图形，它应该是关闭的，但如果你有漂亮的阿尔法混合和平滑的图形，它最好是打开的。 如果你有很好的阿尔法混合和平滑边缘的图形，最好保持开启状态。
  • 使用同步来避免撕裂。这将切换v-sync的开启或关闭。注意，如果你有一个房间速度为120的游戏，而玩家的显示器刷新率为60，打开这个选项也会把你的游戏速度锁定在60。 60，打开这个选项将把你的游戏速度也锁定为60。这在默认情况下是关闭的。
  • 允许窗口调整大小。勾选此选项允许用户改变游戏窗口的大小（无边界窗口选项需要关闭才能发挥作用）。该选项默认为关闭。
  • 无边界窗口。勾选此选项将使你的游戏以无边界窗口运行，移除正常的最小化、最大化和关闭按钮以及游戏标题。这在默认情况下是关闭的。
  • 缩放。在这里你可以选择保持长宽比（所以4:3的房间在16:9的屏幕上会有 "字母框"）或完全缩放（拉伸图像以适应全屏）。

    警告! 关闭应用程序表面将禁用Windows游戏选项中设置的所有缩放选项，直到它被重新打开。更多的细节请参见应用程序表面。

最后是设置纹理页的大小的选项。默认（也是最兼容的）尺寸是2048x2048，但你可以从256x256到8192x8192之间的任何地方进行选择。在这里 也有一个按钮标明 预览 的按钮，它将为这个平台生成纹理页，然后打开一个窗口，以便你能看到它们的样子。如果你想看看纹理页是如何结构的，这可能非常有用。 结构，并防止纹理页大于（或小于）所需。关于纹理页的更多信息，请看这里。

注意：请注意，纹理页的尺寸越大，你的游戏在规格较低的PC上的兼容性就越差。

图片

图像部分是你提供游戏所需图像的地方。对于Windows，你需要提供一个图标文件（必须是 .ico 格式）和一个闪屏（可以是 .png, .bmp, .jpg 或 .gif 格式）。 闪屏将在游戏加载时显示，可以通过勾选使用闪屏选项（默认为关闭）来启用或禁用。

值得注意的是，GameMaker Studio 2有一个项目图像生成器工具，可以用来自动创建所有不同的目标平台所需的图像。 被编译到的所有不同的目标平台上。如果你使用这个工具，你应该修改创建的图像以确保它们是你所需要的。
