iOS游戏选项

本节概述了可用于控制如何iOS游戏项目的不同选项。不同的部分是：

一般的

“常规”部分的第一部分介绍了游戏的“构建设置”。首先，请设置可以运行游戏的方向，以获得最佳的用户体验。可用的选项有：

  • 肖像
  • 翻转的肖像
  • 风景
  • 翻转的风景

然后，您可以转到“设备支持”部分，其中提供了以下选项，用于设置要定位的设备类型：

  • iPhone或iPod Touch ：iPhone 6及更高版本或iPod Touch设备
  • iPad ：iPad设备
  • 两者：iPhone / iPod和iPad

第二件事是填写有关您游戏的产品信息，包括其显示名称，捆绑ID和版本信息。请注意，Bundle ID必须采用正确的反向URL格式，才能使游戏的最终版本正确运行，例如： com.[Company].[GameName] 。在这里，您还可以提供GameMaker Studio 2为您创建的最终应用程序的团队标识符。在此处进行设置将覆盖您在iOS “偏好设置”中提供的团队ID。

接下来，您可以选中或取消选中Defer Home Indicator Output的选项。选中此选项后，主菜单栏将始终以低字母显示，并且第一次滑动会唤醒该菜单栏，第二次滑动会将您带回家。如果未选中该选项，则游戏中不会显示该条，但是第一次边缘滑动会将您带回家。默认为未选中。

最后，您可以选择生成Podfile和Podfile.lock 。如果要使用扩展编辑器CocoaPods添加到iOS扩展，则需要这些文件。 GameMaker Studio 2所需的特定宏，以在构建游戏时设置任何列出的CocaoPod单击按钮将创建具有预先生成的宏的文件，并且如果您希望手动编辑它们，还将在文件的位置打开文件浏览器。还有一个按钮，将带您进入以下指南：

  • iOS和tvOS ：使用可可豆荚

图形

您可以在此处更改以下有关游戏显示方式的详细信息，并提供以下选项：

  • 在像素之间插值颜色：打开线性插值，基本上可以“平滑”像素。对于清晰的像素图形，应该将其关闭，但是如果您具有不错的alpha混合和平滑的边缘图形，则最好保留它。默认情况下是关闭的。
  • 一半的iPad 1纹理： iPad 1的纹理内存比最近的设备少得多，因此，如果要定位此设备或获得最大的兼容性，则可以启用此选项以将纹理大小减小一半。
  • 缩放：在这里，您可以选择缩放绘图画布，以在设备屏幕上保持宽高比-在边缘周围添加“填充”以适合屏幕-或者您可以选择拉伸画布以适合设备屏幕。

最后，可以选择设置纹理页面的大小。默认（也是最兼容）大小是2048x2048，但是您可以选择256x256到8192x8192之间的任意值。还有一个标记为“预览”的按钮，它将为该平台生成纹理页面，然后打开一个窗口，以便您查看它们的外观。如果希望查看纹理页面的结构并防止纹理页面大于（或小于）所需的页面，这将非常有用。有关纹理页面的更多信息，请参见此处。

注意：请注意，纹理页面的大小越大，游戏与不同浏览器和设备的兼容性越差。

图示

本部分允许您添加各种iPhone和iPad设备以及iOS App Store的不同部分所需的各种图标。这些图标应被创作为24bit *.png图像以每个指定的尺寸显示。

值得注意的是， GameMaker Studio 2具有项目图像生成器工具，该工具可用于自动创建将游戏编译到的所有不同目标平台所需的所有图像。如果使用此工具，则应修改创建的图像，以确保它们是您所需要的。该工具将生成图标以及图像（请参阅以下部分）。

注意：导入较旧的项目时，可能会出现以下警告：这是因为iOS目标版本已更改了所需图标的格式。您将需要重新创建所有图标，因为它们已被通用的GameMaker Studio 2图标所取代（您可以使用上述Project Image Generator工具来快速完成此操作）。

图片

本部分允许您为每个设备方向添加单独的图形作为启动屏幕（在应用程序加载时将短暂显示的图像），以及将用作非启动屏幕背景的边框颜色设置为全屏显示。这些图像必须作为单独的24位*.png图像创作，且没有透明胶片且尺寸适当。请注意，这不被视为初始屏幕，并且Apple对于应如何设置和使用它有一些相当严格的指导原则。有关更多信息， 请参见《人机界面指南》上的开发人员页面。

您还可以通过以下选项设置启动屏幕的显示方式：

  • 保持纵横比：与设备屏幕相比，这将保持图像的纵横比，并在需要时在边缘周围添加“填充”。
  • 裁剪以适合：这将使图像在设备屏幕上保持1：1，并根据需要裁剪（或填充）边缘。
  • 拉伸：这将拉伸图像以填满设备屏幕（并可能使显示的图像变形）。

社会的

本部分用于在iOS游戏中启用不同的社交功能。可以选择以下选项：

  • Facebook ：如果需要Facebook功能，则需要选中此复选框，然后提供Facebook App ID和Facebook App Name 。请注意，所有Facebook功能都可以通过扩展程序访问，并且当您首次选中“使用Facebook”框时，系统将提示您从Marketplace下载和安装Facebook扩展程序。
  • 启用推送通知：为了让您的游戏使用推送通知，必须首先在此处启用它们，否则将无法使用。
  • 启用“使用Apple登录” ：此选项用于在iOS上使用Apple登录扩展。您可以从YoYo Games Marketplace获得此扩展，该扩展本身包含完整的使用说明。如果您不使用此扩展名，则不应勾选此选项。
