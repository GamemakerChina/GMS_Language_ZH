tvOS游戏选项

本节概述了可用于控制如何tvOS游戏项目的不同选项。不同的部分是：

一般的

在这里，您可以填写有关游戏的产品信息，包括游戏的显示名称，游戏包ID和版本信息。请注意，Bundle ID必须采用正确的反向URL格式，才能使游戏的最终版本正确运行，例如： com.[Company].[GameName] 。您还可以在构建Mac上设置“应用程序输出”位置，并且可以提供希望用于签署GameMaker Studio 2为您创建的最终应用程序的团队标识符。在此处进行设置将覆盖您在tvOS首选项中提供的团队ID。

在这里，您还可以选择生成Podfile和Podfile.lock 。如果要使用扩展编辑器CocoaPods添加到tvOS扩展，则需要这些文件。 GameMaker Studio 2所需的特定宏，以在构建游戏时设置任何列出的CocaoPod单击按钮将创建具有预先生成的宏的文件，并且如果您希望手动编辑它们，还将在文件的位置打开文件浏览器。还有一个按钮，将带您进入以下指南：

  • iOS和tvOS ：使用可可豆荚

图形

您可以在此处更改以下有关游戏显示方式的详细信息，并提供以下选项：

  • 在像素之间插值颜色：打开线性插值，基本上可以“平滑”像素。对于清晰的像素图形，应该将其关闭，但是如果您具有不错的alpha混合和平滑的边缘图形，则最好保留它。默认情况下是关闭的。
  • 缩放：在这里，您可以选择缩放绘图画布，以在设备屏幕上保持宽高比-在边缘周围添加“填充”以适合屏幕-或者您可以选择拉伸画布以适合设备屏幕。

接下来是用于设置纹理页面大小的选项。默认（也是最兼容）大小是2048x2048，但是您可以选择256x256到8192x8192之间的任意值。还有一个标记为“预览”的按钮，它将为该平台生成纹理页面，然后打开一个窗口，以便您查看它们的外观。如果希望查看纹理页面的结构并防止纹理页面大于（或小于）所需的页面，这将非常有用。有关纹理页面的更多信息，请参见此处。

注意：请注意，纹理页面的大小越大，游戏与不同浏览器和设备的兼容性越差。

最后，选择是否显示光标。如果选中此选项，则每当鼠标移动（在鼠标位置处）时，操作系统都会显示一个光标，并且在一段时间内不移动鼠标时，光标将淡出并消失。如果未选中，则不会显示任何光标。

图示

本部分允许您添加各种设备和App Store不同部分所需的各种图标。这些图标应被创作为24bit *.png图像以每个指定的尺寸显示。

值得注意的是， GameMaker Studio 2具有项目图像生成器工具，该工具可用于自动创建将游戏编译到的所有不同目标平台所需的所有图像。如果使用此工具，则应修改创建的图像以确保它们符合您的要求，尤其是对于此平台，因为所需的图标是矩形而不是正方形。该工具还将生成图像（如下所述）以及图标。

图片

tvOS目标大小添加单独的图形作为“顶层架子”图像和启动屏幕。这些图像必须被创作为单独的24bit *.png没有透明胶片且尺寸正确的图像。您还可以从底部的下拉菜单中设置显示初始屏幕的最短时间（0到10秒）。

如图标部分所述，您可以使用Project Image Generator创建这些图像。

社会的

本部分用于在tvOS游戏中启用不同的社交功能。可以选择以下选项：

  • 启用推送通知：为了让您的游戏使用推送通知，必须首先在此处启用它们，否则将无法使用。
  • 启用“使用Apple登录” ：此选项用于在iOS上使用Apple登录扩展。您可以从YoYo Games Marketplace获得此扩展，该扩展本身包含完整的使用说明。如果您不使用此扩展名，则不应勾选此选项。
