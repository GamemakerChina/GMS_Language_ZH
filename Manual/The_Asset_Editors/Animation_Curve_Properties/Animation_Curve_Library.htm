动画曲线库

动画曲线库窗口允许您更改动画曲线的曲线类型，并将预设应用于所选通道。可以通过以下方式打开左键单击动画曲线编辑器或“序列Dope Sheet中的“按钮”，或在“通道编辑器”中的任意位置单击鼠标右键。

曲线类型

曲线类型控制通道从点到点的流动方式，可以将其设置为以下选项之一：

  • 线性：这将在通道上的点之间创建线性级数
  • 平滑：这将使用catmull-rom插值法在通道上的点之间创建平滑的级进
  • Bezier ：这将使用Bezier曲线数学在通道上的点之间创建平滑的渐变，您可以在该点上通过操纵点上的“句柄”来控制曲线的外观

请注意，更改“曲线类型”会将该类型应用于“动画曲线”中的所有通道。

曲线预设

A C urve预设改变你的曲线的形状。左键单击“预设”菜单中的预设，该预设将应用于您选择的通道，然后单击“重置”会将通道恢复到打开“动画曲线库”窗口之前的状态。

注意：仅当“曲线类型”设置为“贝塞尔曲线”时才能应用预设。

默认情况下，预设应用于整个通道。您可以通过拖动来选择频道的一部分在“通道编辑器”中单击鼠标左键，然后通过按钮或按右键单击，这会将所有预设仅应用于所选部分。

请注意，您的曲线通道（或该通道内的选择）需要具有一定的斜率才能使预设生效，因为任何应用的插值均基于起点和终点之间的垂直差。平坦曲线在两点之间没有这种差异，因此将预设应用于此类曲线后，您将看不到任何结果。

您可以选择两种应用模式，这些模式会影响如何将预设应用到您的频道（或您在该频道中的选择）：


  • 覆盖：这是通过窗口打开窗口时的默认设置在“动画曲线编辑器”或“序列Dope Sheet单击“确定”按钮。它会覆盖通道或所选内容的起点和终点之间的所有点。
  • 之间：这是通过以下方式打开窗口时的默认设置在“通道编辑器”中的任意位置单击鼠标右键。它在每对选定的点之间应用预设，并在不删除任何现有点的情况下添加所需的任何新点。

如您在上面的视觉效果中所看到的，“之间”将曲线预设应用于每对点，并插入所需的任何新点。覆盖将删除起点和终点之间的所有点，然后应用预设。

您还可以选择三种缓动模式，它们控制曲线如何加速或减速。默认的缓动模式是“缓出” 。

  • 缓入：曲线以更快的加速度开始，然后减速
  • 缓和：曲线以较慢的加速度开始，然后随时间加速
  • 缓入缓出：曲线以较慢的加速度开始，随时间加速，然后减速
