代码段

使用代码段时，您可以使用的一种非常方便的工具是使用代码段。使用脚本编辑器时，可以按 F4打开代码片段弹出窗口，该弹出窗口允许您从列表中选择一种常用的代码方法：

在此窗口中，您可以用鼠标选择要使用的代码片段，也可以按右侧列出的相关热键。这会将代码段添加到您的代码中进行编辑。

如果需要，您还可以定义自己的代码段。在执行此操作之前，您需要创建一个名为“snippets.txt在以下目录之一中：

  • Windows ： %ProgramData%\GameMaker Studio2 \ User \
  • macOS ： /Users/Shared/GameMaker Studio2 /用户/

GameMaker Studio 2进行任何更新期间，都不会修改此文件夹，因此，您编辑的文件将保持不变（但是将还原对安装目录中基本片段文件的所有编辑）。

在“用户”目录中创建文件后，您可以按照以下规则使用任何文本编辑器对其进行编辑：

  • 每个代码段必须位于单独的行中（不应有空白行）
  • 每个代码段都必须以要使用的热键开头，后跟“- “，然后是代码段名称（菜单中显示的名称），后跟冒号”: ”，例如：

我-实例创建：

在冒号之后，添加代码段

I-实例Create：instance_create_layer（x，y，| layer |，object）;

您添加的代码还必须遵循特定的格式，其中：

  • 所有代码都在一行上
  • 你用 ”$ “换行
  • 您将需要选择的文本放在“| “符号（添加摘要后，光标会选择此文本进行编辑）

请参阅基本摘要文件中已有的示例，以了解如何按照上述规则进行设置。您可以在GameMaker Studio 2安装目录中找到基本文件：

  • \GameMaker Studio 2 \ TextEditor \ snippets.txt
