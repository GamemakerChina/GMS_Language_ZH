扩展编辑器

一个扩展包可以用于许多附加功能，并添加常数S到GameMaker Language （ GML ），或作为一种传输游戏资产，甚至整个项目容易人与人之间。如果扩展是添加额外的功能，那么可以使用您的游戏链接到外部的功能SDK S或源代码写在其他语言（如Java进行Android ，例如），以及原生GML 。

通常，第3方扩展程序将来自GameMaker Marketplace ，而您已订阅的Marketplace库窗口中，并可从那里进行安装。如果您没有从Marketplace获得扩展程序包，仍然可以通过再次右键单击将其添加到项目中在资源浏览器中的任意位置，然后选择添加现有，然后浏览至扩展程序的位置以将其选中。

如果你有一个 *.GMEZGameMaker Studio的先前版本创建的游戏，您可以通过选择鼠标右键来添加这些游戏菜单选项导入扩展名。

注意：旧版格式 *.GEX扩展程序不再受支持，无法导入到GameMaker Studio 2项目中。

将扩展添加到资源浏览器后，您应该为其命名和版本号，并设置要将其导出到的平台。通过取消选中所选平台，您可以告诉GameMaker Studio 2不要将扩展导出为这些平台的可执行文件的一部分。您还应确保扩展属性与目标平台匹配，并根据以下内容编辑扩展文件的导出目标：

  • 所有目标： GML扩展。这些是使用GameMaker Language （ GML ）进行的扩展，并且与所有目标平台兼容。
  • Windows （标准，UWP，XBoxOne） ：可以使用Windows DLL文件（动态链接库）。请注意，如果您已经在Windows游戏选项中Windows Runtime ”，则该项目的所有DLL必须与64位兼容。 32位DLL无法与64位可执行文件一起使用。
  • macOS ： macOS目标可以使用标准 DyLib扩展文件。
  • Ubuntu （ Linux ） ：对于Ubuntu目标，您需要确保扩展使用 SO文件。
  • HTML5 ：这些平台可以使用 JavaScript扩展名。
  • Android ：此平台需要扩展才能编写 Java或 Android框架SDK。
  • iOS / tvOS ：此平台的扩展是用Objective C编写的，并且需要 *.mm和一个 *.h文件，或者如果使用任何框架，则是预编译的 *.a本机文件。
  • PlayStation ：对于PlayStation 3或4，您的扩展程序应为 PRX文件，每个目标要求不同 PRX以符合目标的规格（PS3或PS4）。如果使用PSVita，则扩展名应为 SUPRX反而。

注意：需要某些扩展程序才能遵循GameMaker特定命名约定，以正确识别它们设计的平台和CPU体系结构。请参阅“创建扩展”部分以获取更多信息。

由于单个扩展可以具有多个相关文件，因此可以为多个平台创建单个扩展，在这种情况下，您需要在主扩展编辑器中指定所有兼容的平台（即：选中它们旁边的复选框），然后在扩展的每个部分的单独文件属性，设置应将这些文件导出到的平台。

要更改特定扩展文件或占位符的目标平台，必须首先打开扩展属性（双击资产浏览器中的扩展名或使用并选择Open Properties ），然后在扩展名属性中选择要编辑的右侧资源文件：

注意：如果要导出到macOS或Ubuntu目标，则可以设置“解压缩为zip文件”选项以保留扩展名的文件权限。

安装扩展程序后，就可以使用其功能和常量，就像使用GameMaker Language使用的内置函数和常量一样。如果您的游戏是跨平台的（例如，它被设计为可以在Windows和Android目标上运行），那么在调用扩展特定的函数以防止出现问题之前，应使用OS常量和函数检查平台，例如：

switch(os_type)
    {
    case os_android:
        call_android_extension_function(x, y, z);
        break;
    case os_windows:
        call_windows_extension_function(x, y, z);
        break;
    }

请注意，对于不同的平台，您的扩展名可以共享功能名称，因此，如果您的扩展名带有不同的文件，以（例如）弹出Windows和Android消息，则两个扩展名文件可以共享相同的功能名，只要它们两者都设置为仅导出到各自的平台。

YoYo Games 3rd Party Extensions

传统版本的GameMaker具有许多内置的第三方功能，但这效率低下且难以维护，因为如果第三方更改了任何特定功能，则必须更新GameMaker IDE和Runtime因此，当前版本已采用扩展包的使用来为游戏添加第三方支持。这意味着您只能添加所需的那些额外功能，甚至可以修改和编辑它们以满足您的特定需求。
