字体编辑器

当你想在你的游戏中绘制文本时，这个文本是以标准的Arial 12点字体绘制的，但为了制作更有趣或看起来更独特的文本，你可能想使用不同的字体。 因此，要使用你电脑上的不同字体，你必须在GameMaker Studio 2中创建一个字体资产。这可以通过在资产浏览器上右击 ，然后选择"创建" -> "字体 "来完成。对于你创建的每一个字体资产，你要从你的电脑中指定一个特定类型的字体，然后可以在你的游戏中使用。 然后在你的游戏中使用动作或代码来设置绘制的字体。

当你第一次创建一个字体资源时，字体编辑器窗口将打开，有以下选项。

详细内容

在这里，你可以给你的字体起一个独特的名字（仅限字母数字字符和"_"下的符号），以及选择要使用的字体。字体必须安装在你的机器上，由于许可问题，GameMaker Studio 2默认不将实际的字体文件与项目文件一起存储。 默认情况下，GameMaker Studio 2不会将实际的字体文件与项目文件一起存储，而是将字体作为一系列位图字形呈现在纹理页面上。 的位图字形。纹理页将被分配到一个纹理组，你也可以在这里的组选项中选择。

如果你想在制作市场资产时或出于其他原因，将实际的字体文件（ *.ttf 或*.otf）与项目一起分发，你可以勾选复制到项目这一选项。这 将打开文件资源管理器，你可以选择字体文件，然后它将被添加到与项目捆绑的文件中。

重要的是!你应该只使用你被合法允许和许可在你的项目中重新分配或使用的字体。

预设

在这一节中，你可以设置字体的大小（以点为单位--默认为12pt），以及字体是否应被绘制为粗体或斜体，或应用抗锯齿。右边的预览窗口 将向你展示文字在你的游戏中的大致样子，以及你所指定的大小和变换。


范围控制

当创建一个字体资源时，你很少会想包括字体的每一个字形。这就是为什么GameMaker Studio 2默认只使用所选字体中从32到127的字符，因为这给了我们数字和 小写和大写字母以及某些标点符号。然而，你可以改变使用的字符范围，以帮助优化你的游戏，即：如果你只需要一个特定字体的数字，那么只 选择数字。要做到这一点 要做到这一点，你应该首先点击删除 按钮来清除当前的范围（你可能需要在预览窗口中点击 来选择这个范围），然后点击删除 按钮。 预览窗口），然后点击 "添加 "按钮，添加一个新的范围。这将打开以下链式窗口。

这个窗口有一些按钮可以帮助你为你的字体建立一个标准范围。

-正常 范围从32到127

-ASCII 范围从0到255

-数字 范围只包含10位数字

-字母 范围只包含所有大写和小写的字母

其他范围可以通过在字符范围输入框中输入你想设置的范围的第一个和最后一个字符的索引来使用（如果一个字符不在范围内，则用空格代替）。除了这些按钮和输入框外，你还有两个选项。 你还有两个特别有用的选项。

  • 从代码：如果你点击这个按钮，然后点击确定，GameMaker Studio 2将自动解析你的游戏代码中的字符串，然后创建不同的字符范围来覆盖你游戏中的所有文本。注意，它寻找的是 所有的字符串（无论是在""或"'"内），因此也可能在字符范围内包括文件名。
  • 从文件：这与上面解释的 "从代码 "选项的作用类似。如果你点击它，你将被要求提供一个文件，然后一旦完成，你应该点击确定。然后GameMaker Studio 2将解析该文件并为文本创建字符范围。 文件，并为其中的文本创建字符范围。

注意，你可以通过直接在预览窗口中添加或删除文本字形来手动编辑文本范围。

纹理控制

如上所述（在字体细节部分），默认情况下，所有字体字形将作为纹理组的一部分被渲染到一个纹理页中，以将它们转换为图形资源。 当游戏被编译时，字体将被添加到一个临时纹理中，以创建一个包含所有所需字体字形的单一图像，然后这将被添加到主纹理组中进行导出。这里的按钮是用来调整 字体纹理的生成方式或重新生成一个新的字体纹理。

选项 "按钮将打开高级字体纹理选项。

这些选项有时可以从根本上改变字体在游戏中的显示方式，而且并非所有字体都需要相同的设置。 以下选项旨在为你的字体提供最大程度的定制服务。

  • 暗示。字体提示（也被称为指示）是使用数学指令来调整轮廓字体的显示，使其与光栅化的网格相一致。在低屏幕分辨率下，提示对于产生清晰可读的文本至关重要。 在这里，你可以从不同的提示算法中进行选择，以获得最好的结果。默认是 "正常"。
  • 字形选项。这些选项可以用来通过启用/禁用不同的操作来改变字形的加载方式。这些选项是
    1. 禁用提示。这将禁用字体的本机提示和自动提示。
    2. 禁用自动跳线。这将禁用自动烧结，而是使用字体的本地烧结（如果它有的话）。请注意，这将覆盖 "首选自动烧结 "选项。
    3. Prefer Auto-Hinter:这将告诉GameMaker Studio 2优先使用自动烧结器，而不是字体可能具有的任何本地提示。请注意，这将被 "禁用自动烧结 "和 "禁用提示 "选项所覆盖。 选项覆盖。
    4. 变形。启用该选项将使字形略微弯曲以改善网格对齐。请注意，这需要自动烧结器和 "正常 "提示模式。
    5. 无比例。当这个选项被选中时，它将保持字形的字体单位，而忽略要求的尺寸。这意味着提示功能被禁用。
    上述所有选项的默认状态是禁用。
  • 解释器：在这里你可以选择GameMaker Studio 2应该用于字体渲染的TrueType版本。默认是TrueType v40。
  • 四舍五入。该选项用于设置从定点小数转换为整数时的舍入方法。默认为 "floor"。

重新生成 " 按钮可以用来重新生成字体纹理。如果你想更新你对现有字体所做的任何改动，或者你已经创建了一个新的字体，并且你想在预览窗口中看到它的样子，那么就需要这样做。 窗口中的样子。请注意，你也可以从资产浏览器中的字体（或字体）的右键 菜单中重新生成字体。
