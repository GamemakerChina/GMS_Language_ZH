字体编辑器

当您要在游戏中绘制文本时，会以标准Arial 12磅字体绘制该文本，但是要使文本更有趣或更独特，您可能需要使用其他字体。因此，要使用计算机上使用的不同字体，必须在GameMaker Studio 2创建字体资源。通过右键单击来完成在资源浏览器上，然后选择“创建字体”。对于您创建的每种字体资源，您都可以从计算机中指定一种特定的字体类型，然后可以使用该动作或代码为绘画设置字体，然后将其用于游戏中。

首次创建字体资源时，将打开带有以下选项的字体编辑器窗口：

细节

在这里，您可以为字体指定唯一的名称（仅字母数字字符和下划线符号“ _”），并选择要使用的字体。字体必须安装在您的计算机上，并且由于许可问题， GameMaker Studio 2不会将实际的字体文件与项目文件一起存储，而是将字体作为一系列位图字形呈现给纹理页面。纹理页面将被分配给一个纹理组，您也可以从此处的“组”选项中进行选择。

如果要分发实际的字体文件（ *.ttf Marketplace资产时或由于其他任何原因而仅与项目一起使用时，可以选中选项复制到项目。这将打开文件浏览器，您可以在其中选择字体文件，然后将其添加到与项目捆绑在一起的文件中。

重要的！您仅应使用法律许可并许可在项目中重新分发或使用的字体。

预设值

在本部分中，您可以设置大小（以磅为单位-默认为12pt），以及字体是应以粗体还是斜体绘制还是应用抗锯齿功能。右侧的预览窗口将向您显示与您在游戏中显示的文本大致相同的文本，并带有您指定的大小和变换。


范围控制

创建字体资源时，很少会包含字体的每个字形。这就是为什么GameMaker Studio 2默认只使用所选字体的32到127个字符的原因，因为这会为我们提供数字，小写和大写字母以及某些标点符号。但是，您可以更改用于帮助优化游戏的字符范围，即：如果只需要特定字体中的数字，则只需选择数字。为此，您应该首先单击“删除”按钮以清除当前范围（您可能需要选择范围才能通过单击在预览窗口中对其进行选择），然后点击添加按钮以添加新范围。这将打开以下链接的窗口：

此窗口中有一些按钮可帮助您建立字体的标准范围：

-正常范围是32至127

-ASCII范围从0到255

-位数范围仅包含10位数字

-字母范围仅包含所有大写和小写字母

通过在字符范围输入框中输入要设置的范围的第一个和最后一个字符索引，可以使用其他范围（如果一个字符不在该范围内，则用空格代替）。除了这些按钮和输入框之外，您还有两个其他选项特别有用：

  • 从代码：如果您单击此按钮，然后单击确定， GameMaker Studio 2将自动分析您的游戏代码中的字符串，然后创建不同的字符范围以覆盖游戏中的所有文本。请注意，它会查找所有字符串（在“”或“之内），因此也可能在字符范围内包含文件名。
  • 从文件：与上面说明的“从代码”选项相似。如果单击它，将要求您提供一个文件，然后一旦完成，您应该单击“确定”。 GameMaker Studio 2将解析该文件并为其中找到的文本创建字符范围。

请注意，您可以通过直接在预览窗口中添加或删除文本字形来手动编辑文本范围。

纹理控制

正如上面提到的（在“字体详细信息”部分中），默认情况下，所有字体字形都将作为纹理组的一部分呈现到纹理页面，以将其转换为图形资源，并且在编译游戏时，字体将添加到临时纹理以创建具有所有必需字体字形的单个图像，然后将其添加到主纹理组中以进行导出。此处的按钮用于调整字体纹理的生成方式或重新生成新的字体纹理。

“选项”按钮将打开高级字体纹理选项：

这些选项有时可以从根本上改变字体在游戏中的显示方式，并且并非所有字体都需要相同的设置。以下选项旨在为您提供最大的字体自定义设置：

  • 暗示：字体提示（也称为指令）是用数学说明来调整轮廓字体的显示，使得其线与光栅化网格。在低屏幕分辨率下，提示对于产生清晰，清晰的文本至关重要，在这里您可以从不同的提示算法中进行选择，以尽可能获得最佳效果。默认值为“正常”。
  • 字形选项：这些选项可用于通过启用/禁用不同的操作来更改字形的加载方式。这些是：
    1. 禁用提示：这将禁用字体本机提示和自动插入。
    2. 禁用自动隐藏：这将禁用自动隐藏，而是使用字体本机提示（如果有的话）。请注意，这将覆盖“首选自动插入”选项。
    3. 首选自动预设置：这将告诉GameMaker Studio 2首选使用自动预设置，而不是字体可能具有的任何本机提示。注意，这被“禁用自动插入”和“禁用提示”选项所覆盖。
    4. 变形：启用此选项将稍微变形字形以改善网格对齐。请注意，这需要自动暂停和“正常”提示模式。
    5. 无缩放：选中此选项时，它将使字形保持字体单位，并忽略请求的大小。这意味着提示被禁用。
    上述所有选项的默认状态为禁用。
  • 口译员：在这里，您可以选择GameMaker Studio 2用于字体渲染的TrueType版本。默认值为TrueType v40 。
  • 舍入：此选项用于设置从定点十进制大小转换为整数时的舍入方法。默认值为“ floor”。

“重新生成”按钮可用于重新生成字体纹理。如果要更新对现有字体所做的任何更改，或者如果您创建了新字体并且想要在预览窗口中查看其外观，则这是必需的。请注意，您还可以通过右键重新生成字体资产浏览器中字体的菜单。

重要的是要注意，当您第一次创建新字体时，如果不在编辑器中重新生成纹理，则纹理将在您首次运行或测试游戏时创建，但这是唯一的一次将自动生成。这意味着一旦字体具有生成的纹理，则对字体资源进行的任何更改都将要求重新生成该字体，以使更改在游戏中可见，否则仍将使用“旧”字体纹理。

GameMaker Studio 2会将生成的字体纹理渲染为 PNG 文件以及其他项目文件，并且应该加载项目并且找不到基本字体，则游戏仍将使用存储的纹理正确显示文本。但是，缺少字体意味着您将无法再更改其任何属性，除了其名称，字体系列和纹理组（来自“纹理组编辑器” ）。


预览模式

在此部分中，您可以选择让“预览窗口”显示所选字体的一些示例文本，或者显示为该字体生成的渲染纹理图像。

预览

在预览窗口中，您可以在其中看到字体的预览，既可以作为文本的示例，也可以作为纹理的预览。当“预览模式”设置为Sample时，您可以在预览窗口中直接输入要测试的任何文本以查看其外观，同时选择“游戏预览”将显示为字体生成的渲染纹理图像。如果没有图像，则应单击“重新生成”按钮以生成纹理。

请注意，纹理图像可能与样本不同，因为除非明确告知，否则纹理不会重新生成。这意味着在最初生成字体纹理之后，除非重新生成纹理，否则对字体资源所做的更改将不会被带到游戏中。

范围预览

此处将列出为字体添加的不同字符范围。范围值显示在左侧，而与这些值相对应的范围字形显示在右侧。
