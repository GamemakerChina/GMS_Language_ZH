异步事件

异步事件是当GameMaker Studio 2收到来自某些外部来源的回叫时触发的事件，该来源可以是来自网络或运行游戏的设备。基本上，您告诉GameMaker Studio 2做一些事情-例如加载图像-它会开始执行此操作，但同时继续执行其必须执行的所有其他工作。然后，当请求完成时，回调将被发送到GameMaker Studio 2，并且将触发为该回调类型定义的任何异步事件。

请注意，与键盘事件非常类似，会为所有具有异步事件的实例触发异步事件，因此您可以-例如- http_get()在一个实例中调用，而在另一个实例中使用异步HTTP事件来处理回调。

注意：您应该意识到，由于浏览器中的XSS保护，跨域的请求和加载资源的尝试均被阻止，并且在使用以下任何事件时，可能会返回空白结果。
