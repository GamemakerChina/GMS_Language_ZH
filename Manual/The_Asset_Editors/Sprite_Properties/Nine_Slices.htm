九片

九切片（也称为9切片）是一种用于缩放矩形图像以保留细节的技术，使矩形图像在缩放后仍保持其原始形式。

这是通过将图像分为九个单独的部分或“切片”来实现的。缩放时会移动角切片，并以相同的比例绘制，然后拉伸或重复边缘和中央切片（取决于您的设置）以填充空间。

在子画面上放置四个向导，以将图像分为九个切片。下图显示了使用九分切片缩放精灵后指南如何保持在原位：

切片的“平铺模式”确定缩放精灵时如何显示该切片。只能为边缘和中心切片设置该角度（因为不对角切片进行缩放）。

下图演示了应用于中心切片时的每种“平铺模式”：如果为精灵启用了“九分切片” ，它将用于调整精灵的大小，无论它出现在什么地方，并且将在使用该精灵的任何现有位置工作，而无需进行任何其他更改。

注意：九分切片法不适用于仅绘制部分精灵或使其变形的函数，例如draw_sprite_part（）和GameMaker _Language / GML _Reference / Drawing / Sprites_And_Tiles / draw_sprite_pos.htm“> draw_sprite_pos（） 。

与Sequences结合使用，此功能可用于构建用户界面，并可用于例如在游戏级别中创建矩形墙。纹理内存也可以使用Nine Slicing进行保存，因为较小的图像可以用来覆盖屏幕上较大的区域。

可以在IDE中的Sprite上启用九个切片。有关更多信息，请参见Sprite Editor 。
