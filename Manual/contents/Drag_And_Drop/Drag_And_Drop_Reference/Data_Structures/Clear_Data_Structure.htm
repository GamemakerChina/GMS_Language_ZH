清除数据结构

这个动作将清除给定数据结构中的所有值。你必须首先选择要清除的数据结构类型，然后给出数据结构的索引（索引是你创建数据结构时返回并存储在变量中的值 索引是您最初创建数据结构时返回并存储在变量中的值）。)一个被清除的数据结构被认为是 "空 "的（即：它没有条目），网格除外。对于网格，这个动作将简单地把所有的网格单元清除为0。

注意：这并不能将数据结构从内存中释放出来，只是清除了它的任何信息。要释放分配给该数据结构的内存，你必须使用Free Data Structure动作。
