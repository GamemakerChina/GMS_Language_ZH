如果数据结构存在

通过这个动作，你可以检查一个变量，看它是否索引了一个数据结构。你首先需要选择要检查的数据结构类型（列表、地图、网格或堆栈），然后你提供持有数据结构的变量 索引的变量（如创建列表、创建地图、创建网格和创建堆栈等操作所返回的）。如果数据结构存在，该动作将返回true，如果不存在则返回false。注意，你可以在动作中标记 "not "来检查一个变量是否持有数据结构索引。

请注意，要把动作添加到 "如果 "块中，应该把它们放到动作的一侧，如下图所示。
