String To Number

这个动作可以将一个字符串转换成一个数字。你提供要改变的字符串，然后给出一个目标变量，将真实的数字返回给它（这个变量可以被标记为一个临时的局部变量）。请注意，如果字符串有任何字符 如果字符串中有数字和小数点以外的任何字符，它将不会被转换，尽管如果字符串以数字开头，那么这些字符将被转换，字符串的其他部分将被排除。值得注意的是，当把小数点传入这个 动作时，返回的字符串将被四舍五入到两个小数位。下面是一个输入值和你期望的返回值的表格

字符串 返回的值
"123" 123
"123你好，世界" 123
"Hello World" 0（默认情况下，如果字符串不能被转换，将返回0。
"Hello World 123" 0
"123.45678" 123.46
