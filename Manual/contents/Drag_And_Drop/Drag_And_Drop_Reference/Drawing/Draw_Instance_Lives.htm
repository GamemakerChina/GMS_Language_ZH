绘图实例的寿命

这个动作将绘制一个精灵来代表实例当前所拥有的每一条生命。要绘制的数量取自本地实例变量lives的当前值，这是一个特殊的变量，如果你使用任何需要它的动作，它就会在实例中自动创建。 如果你使用任何需要它的操作，该变量会在实例中自动创建。你要给这些生命绘制的精灵和堆栈顺序，可以是行（水平方向，从左到右），也可以是列（垂直方向，从上到下），还有位置。 位置可以是房间内的绝对位置，也可以是相对于进行绘制的实例的位置，而图像之间的间隔将基于精灵的宽度或高度。请注意，这只是绘制一个静态图像 - 绘制给定精灵的初始单幅图像（第0帧），任何其他帧都将被忽略，任何通过改变实例变量（如image_xscale或image_blend）添加到实例的变换也将被忽略。 标题文本将被画在生命值的前面，如果是文本，应该用""包裹，两者都将使用当前绘制的字体、颜色和对齐方式进行格式化。

注意：这个动作只在各种抽奖活动中使用，如果在其他地方使用，则不会抽到任何东西。
