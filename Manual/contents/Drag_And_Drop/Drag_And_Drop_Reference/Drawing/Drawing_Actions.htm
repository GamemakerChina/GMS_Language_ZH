绘画动作库

在绘图动作库中，你可以找到绘制精灵、文本或形状所需的动作，以及设置某些绘图属性。 形状以及设置某些绘图属性所需的动作。这些动作大多只在对象的各种绘图事件中使用，如果在绘图事件之外使用，可能无法工作。 绘制事件外使用可能无法工作。这方面的例外是设置动作，它可以被添加到任何事件中，并会影响之后所有实例的所有绘制。

需要注意的是，如果你在一个对象的主要绘图事件中添加了任何动作，那么它将不会绘制分配给该实例的精灵，除非你明确告诉GameMaker Studio 2来绘制它，使用一个动作 绘制自我。基本上，GameMaker Studio 2将默认绘制分配给一个实例的任何精灵，只有在Draw Event中没有 其他东西的情况下。
