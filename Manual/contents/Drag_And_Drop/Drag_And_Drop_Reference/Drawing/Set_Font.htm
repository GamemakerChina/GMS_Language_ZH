设置字体

通过这个动作，你可以设置所有后续绘制文本动作要使用的字体。点击对话将打开以下窗口，然后你可以选择你希望使用的字体。

注意，这将为所有后续的绘制文本调用设置字体，因此可以在游戏开始时调用一次--在任何情况下，不仅仅是绘制事件--所有绘制文本的动作都将使用设置的字体。 所以可以在游戏开始时调用一次--在任何情况下，而不仅仅是在绘制事件中--所有绘制文本的动作都将使用设定的字体。然而，如果你在游戏中需要多种字体，那么你就需要在绘制任何所需字体的文本之前调用这个动作。 绘制任何所需字体的文本之前调用这个动作。
