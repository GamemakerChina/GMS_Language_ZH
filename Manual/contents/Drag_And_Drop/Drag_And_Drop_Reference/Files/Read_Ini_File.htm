读取Ini文件

通过这个动作，你可以从一个Ini文件中读取一段数据。你必须事先用打开Ini文件的动作打开Ini文件进行读/写，然后用这个动作给出要读的数据类型 实数或用""括起来的字符串），以及要获取数据的章节标题和该章节中的键。你也可以提供一个默认值，它可以是一个实数，也可以是一个字符串，并且这个类型应该是 与动作中设置的 "类型 "相吻合。如果出现以下情况，这个默认动作将由动作返回。

  • 你所打开的Ini文件不存在
  • 你提到的章节标题不存在
  • 你试图获取数据的钥匙不存在

在上述任何情况下，你都会得到返回的默认值，而不是来自Ini文件的值。无论返回的是什么值，都将存储在你指定的目标变量中，如果标记为临时局部变量，也可以在这时创建 作为一个临时的局部变量。

重要的是！试图从一个以前没有用打开Ini文件动作打开的Ini文件中读取数据，将导致一个严重的错误并使你的游戏崩溃。
