保存游戏

这个动作将做一个基本的游戏保存到一个给定的文件。你给一个文件名（字符串，包括扩展名），该动作将对当前游戏状态进行有限的保存。重要的是要理解这个动作是 被设计为对当前游戏状态进行 "快照"，更多的是作为检查点系统使用，而不是一个通用的保存系统，因为它不保存任何动态资源（如数据结构或粒子）。因此，这个动作应该非常小心地使用，以免出现错误或内存泄漏。关于保存游戏数据的更全面的方法，请看文件动作。
