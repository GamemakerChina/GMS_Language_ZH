如果按了游戏手柄按钮

通过这个动作，你可以轮询游戏板，看看当前是否有任何按钮被按下。该动作将在游戏板按钮被按下（并保持）的每一步返回真，否则返回假，不过如果你点击 not修改器，你可以检查按钮是否没有被按下，即：如果没有按钮被按下，动作将返回true，如果有按钮，则返回false。如果你只需要检查是否有单个按钮被按下，那么 使用动作if gamepad button down。

如果被检查的按钮是一个模拟按钮，那么只有在按钮值超过了用操作Set Gamepad Button Threshold设置的阈值时，才会检测到按下。

请注意，要把动作添加到 "如果 "块中，应该把它们放到动作的一侧，如下图所示。

这些动作现在将被运行，如果 "if "评估为真时，这些动作将被运行，而在其他地方放弃的任何动作将在 "if "块之后执行。
