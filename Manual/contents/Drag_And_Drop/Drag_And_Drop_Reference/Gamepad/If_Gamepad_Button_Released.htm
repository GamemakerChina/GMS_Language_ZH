如果释放游戏手柄按钮

这个动作检查给定的游戏手柄索引的游戏手柄按钮何时被释放。当按钮被释放时，它将返回 "true"，而在所有其他时间则返回"false"。如果你启用not修改器。 这个动作将被反转，并检查按钮是否被释放，在它被释放的游戏刻度线上返回false，在所有其他时间返回true。

如果被检查的按钮是一个模拟按钮，那么只有在按钮值低于使用操作Set Gamepad Button Threshold设置的阈值时，才能检测到释放。

请注意，要把动作添加到 "如果 "块中，应该把它们放到动作的一侧，如下图所示。

现在，如果 "if "评估为 true时，这些动作将被运行，而在其他地方放弃的任何动作将在 "if "块之后执行。
