Set Instance Alpha

这个动作块设置实例的image_alpha值。图像alpha是绘制分配给该实例的任何精灵的 "透明度 "值，其值在0和1之间，其中0为完全透明，1为 是完全不透明的。请注意，如果精灵有透明或半透明的像素，那么它们的透明度将被按比例放大，所以如果你在精灵中有一些50%的透明像素，并将实例的alpha设置为0.5，那么最终的alpha 的最终alpha值将是25%。你可以把这个值设置为一个相对量

请注意，要使这个变量的变化可见，实例应该没有绘制事件（因此GameMaker Studio 2将默认绘制精灵），或者使用Draw Self 动作绘制。
