设置实例颜色

这个动作块设置image_blend颜色，以便与实例精灵 "混合"。默认值是十六进制代码$FFFFFFFF（用ARGB格式表示白色），这将使用无混合和全alpha绘制精灵。任何其他值（包括内部颜色常量，如c_red或c_aqua）将把指定的颜色与原始精灵混合，并为实例设置 image_alpha ，但只有在勾选了 "使用阿尔法"复选框的情况下。这将覆盖之前使用 "设置实例阿尔法"动作设置的任何阿尔法值。

请注意，为了使这个动作的变化可见，这个实例应该没有绘制事件（因此GameMaker Studio 2将默认绘制精灵），或者使用Draw Self动作绘制。值得注意的是 也要注意，你应该尽量限制HTML5平台上的混合（除非使用WebGL），因为每个混合的精灵都要单独缓存，所以有很多混合的精灵可能会对性能产生不利影响。
