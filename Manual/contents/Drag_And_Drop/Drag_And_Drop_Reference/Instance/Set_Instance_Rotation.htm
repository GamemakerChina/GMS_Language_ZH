设置实例旋转

这个动作块设置实例的image_angle（旋转）。角度的单位是度，右边是0°，向上是90°，左边是180°，向下是270°。将这个变量设置为0可以重置实例，这意味着它将 绘制任何指定的精灵，就像它在精灵编辑器中定义的那样。请注意，为了使这个动作中的变化可见，实例应该没有绘制事件（因此GameMaker Studio 2将默认绘制精灵），或者使用绘制自我动作来绘制 使用Draw Self动作。
