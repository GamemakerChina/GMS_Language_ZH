这个动作块将实例的sprite_index设置为资产浏览器中的另一个sprite。你选择要转换的精灵，然后给出要显示的图像（动画帧）。请注意，如果你不希望精灵在设置框架后出现动画，你就需要将其设置为 注意，如果你不想让精灵在设置帧后继续动画，你需要使用设置动画速度动作，并将其设置为0，或者如果你想让精灵继续动画，那么就将值设置为使用image_index内置变量变量 来代替。

改变精灵也会改变实例的碰撞掩码，除非你在对象编辑器中还指定了一个单独的掩码_索引。请注意，为了使这个动作中的变化 的变化是可见的，实例应该没有绘制事件（因此GameMaker Studio 2将默认绘制精灵），或者使用Draw Self动作绘制。
