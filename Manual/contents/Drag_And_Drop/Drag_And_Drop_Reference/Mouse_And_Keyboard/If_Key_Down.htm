如果键值下降

通过这个动作，你可以轮询键盘，看是否有任何键正在被按住。该动作将在键盘按键被按下的每一步返回真，否则返回假，不过如果你点击not修改器，你可以检查按键是否没有被按下，即：如果没有按键被按下，该动作将返回真，如果有则返回假。如果你只需要检查单次按键，那么就使用If Key Pressed动作。

请注意，要把动作添加到 "如果 "块中，应该把它们放到动作的一侧，如下图所示。
