如果按了键

这个动作检查键盘按键的初始按下情况。它将在键盘最初被按下的单一游戏步骤（帧）中返回 "真"，而在所有其他时间返回 "假"。如果你启用not修改器，这个动作 将被反转，并检查该键是否被按下，在该键被按下的游戏步骤（帧）中返回false，在所有其他时间中返回true。如果你需要检查一个键是否被按住，而不仅仅是 初次按下的事件，那么就使用If Key Down动作。

请注意，要把动作添加到 "如果 "块中，应该把它们放到动作的一侧，如下图所示。

在其他地方放弃的任何行动将在 "if "块之后执行。
