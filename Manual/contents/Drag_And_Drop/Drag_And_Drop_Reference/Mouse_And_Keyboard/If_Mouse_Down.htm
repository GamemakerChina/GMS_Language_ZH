如果鼠标向下

通过这个动作，你可以对鼠标进行轮询，看看所选的按钮目前是否被按住。这个动作在按钮被按下的每一步都会返回 "true"，否则就会返回 "false"，不过如果你点击not修改器，就可以检查按钮是否被按下，即：当没有按钮被按下时，这个动作会返回 "true"，如果有的话，则返回 "false"。如果你只需要检查一个按钮是否被按下，那么可以使用动作 If Mouse Pressed.

请注意，要把动作添加到 "如果 "块中，应该把它们放到动作的一侧，如下图所示。

如果 "if "评估为真，这些动作现在将被运行。 而在其他地方放弃的任何行动将在 "if "块之后执行。
