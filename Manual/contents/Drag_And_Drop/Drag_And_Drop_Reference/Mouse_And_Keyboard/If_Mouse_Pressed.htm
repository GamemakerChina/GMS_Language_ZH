如果鼠标被按下

这个动作检查鼠标按钮的初始按下情况。它将在按钮最初被按下的单一游戏步骤（帧）中返回 "true"，在所有其他时间返回 "false"。如果你启用not修改器，这个动作 将被反转，并检查按钮是否被按下，在按钮被按下的游戏步骤（帧）中返回false，而在所有其他时间返回true。如果你需要检查一个按钮是否被按住，而不仅仅是最初的按下事件，那么 不仅仅是最初的按下事件，那么使用If Mouse Down动作。

请注意，要把动作添加到 "如果 "块中，应该把它们放到动作的一侧，如下图所示。

如果 "if "评估为真，这些动作现在将被运行。 而在其他地方放弃的任何行动将在 "if "块之后执行。
