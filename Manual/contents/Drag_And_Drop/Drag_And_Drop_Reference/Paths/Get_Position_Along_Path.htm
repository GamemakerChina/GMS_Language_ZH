沿着路径获取位置

这个动作将获得一个实例在路径上的位置。返回的值在0-1之间，所以如果你得到的结果是，例如0.5，那么这个实例正好在路径的中间。注意，你必须调用Start Following Path动作为实例分配一个路径，这个动作才会返回一个0以外的值。 返回的值将存储在你给的目标变量中，或者你可以将该变量标记为一个临时的局部变量来创建它并分配值。
