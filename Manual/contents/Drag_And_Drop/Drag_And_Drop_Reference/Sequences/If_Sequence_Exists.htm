如果序列存在

这个动作可以用来检查给定的序列元素是否存在于游戏室中。你给出元素的ID，就像创建序列的动作所返回的那样，如果这个序列元素存在于房间里，那么 该动作将返回真，否则将返回假。

如果你标记了 "不是 "参数，那么该动作将检查是否有这样的序列元素存在，如果没有发现，它将返回true。

请注意，要把动作添加到 "if" 块中，应该把它们放到动作的一侧，如下图所示。

如果 "if" 评估为真，这些操作现在将被运行。 评估为"真"，而在其他地方放弃的任何行动将在 "if" 块之后执行。
