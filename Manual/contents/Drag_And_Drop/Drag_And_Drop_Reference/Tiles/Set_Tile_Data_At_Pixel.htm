在像素处设置瓦片数据

通过这个动作，你可以为瓦片地图层上特定位置的瓦片单元设置瓦片数据。瓦片数据是一个简单的值，反映了瓦片的索引以及 翻转、镜像和旋转值，以及你所使用的任何自定义瓦片掩码。在这个动作中，你首先使用房间编辑器中定义的图层名称（一个字符串）提供目标图层，然后使用房间中的X和Y轴 来设置瓷砖数据。注意，瓦片数据将被设置为给定位置所在的瓦片地图 "单元"。例如，如果你的瓷砖设置是16x16，那么瓷砖地图层将在16x16像素的单元中保存瓷砖，所以 如果你使用这个动作针对房间的位置（8，8），你将改变瓷砖地图单元格（0，0）中的瓷砖。

注意：所选的图层必须在房间编辑器中被定义为瓦片地图层，否则你可能会出错。

这个动作的最后一个参数是瓦片数据本身。你通常会使用动作Get Tile Data At Pixel来检索瓦片地图单元的瓦片数据，然后使用动作Set Tile Data Transform来操作它，然后用这个动作再次设置它。关于瓦片数据的更多信息，请参见GML中的瓦片地图功能部分。

Y 沿着垂直（Y）轴的位置来设置瓦片数据
数据 用来设置单元格位置的瓦片数据。
