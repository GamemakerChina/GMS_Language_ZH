在单元格中设置瓦片数据

通过这个动作，你可以为瓦片图层上的一个瓦片单元设置瓦片数据。瓦片数据是一个简单的值，反映了瓦片的索引以及翻转、镜像和旋转的值。 以及你所使用的任何自定义瓦片掩码。在这个动作中，你首先使用房间编辑器中定义的图层名称（一个字符串）来提供目标图层，然后提供要设置瓷砖数据的瓷砖贴图层的列和行。 的列和行。单元 "是瓦片地图网格上的区域，用来放置你想设置数据的瓦片，所以如果你的瓦片是16x16，例如，房间是1024x768，瓦片地图将有64列和48行。

注意：所选的图层必须在房间编辑器中被定义为瓦片地图层，否则你可能会出错。

这个动作的最后一个参数是瓦片数据本身。你通常会使用动作Get Tile Data In Cell来检索瓦片地图单元的瓦片数据，然后使用动作Set Tile Data Transform来操作它，然后再用这个动作来设置它。关于瓦片数据的更多信息，请参见GML中的瓦片地图功能部分。
