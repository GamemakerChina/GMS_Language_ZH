在单元格中设置瓦片索引

通过这个动作，你可以将一个给定图层上的瓷砖设置为一个新的索引。图层是由给定的图层名称（一个字符串，在房间编辑器中用来命名图层）指定的，然后你给出图层中的单元格的行和列，以获得要设置的瓷砖。 要设置的瓦片。最后，你提供要设置的瓦片索引（瓦片的索引从左到右，从上到下，0在左上角）。注意，你可以随时将瓷砖索引设置为0来清除瓷砖。
