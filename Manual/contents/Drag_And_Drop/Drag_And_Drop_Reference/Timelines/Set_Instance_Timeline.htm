集体实例时间线

这个动作将资产浏览器中的一条时间线分配给实例。你给出要分配的时间线，然后你可以选择让时间线在到达最后一刻时循环（最后一刻运行后的下一帧将循环回到 时间线的第一时刻），以及是否立即启动时间线。如果你不立即启动时间线，你可以使用动作Set Timeline State来启动它，在另一个 事件或脚本中启动。
