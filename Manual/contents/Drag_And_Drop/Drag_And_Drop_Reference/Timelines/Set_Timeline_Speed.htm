设置时间线速度

这个动作可以用来设置一个时间线在它所包含的时刻中的移动速度。通常情况下，在每个游戏帧中，你希望时间线的位置增加1，然而你可以通过设置这个 动作设置为一个不同的值来改变这个量。你选择的值将被视为一个乘数，因此，例如，一个2的值将使时间线以两倍的速度运行，而0.5的值将使它以一半的速度运行。 这对时刻来说意味着，如果数值大于1，那么在同一游戏帧中可能会出现几个时刻（没有时刻被跳过），而如果速度小于1，那么在一个时刻运行后，会有一些帧 如果你把时间线速度设置为0.1，例如，你会在一个游戏帧上得到一个时刻，然后有9个游戏帧没有时刻，而在第10个游戏帧上，下一个时刻将被运行。

如果你用设置实例时间线或 设置时间线状态的操作来设置时间线的开始，但没有设置 速度，那么时间线将默认为每游戏帧1分钟。
