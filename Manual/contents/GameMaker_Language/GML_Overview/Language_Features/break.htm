突破

break 语句是用来提前结束一个 for, 重复, while或 do / until 循环，或者告诉一个 转换语句在该点结束，或者提前结束一个 与调用。请看这些不同函数的单独页面，以获得更多的 深入解释它在每种情况下的使用方法。请注意，如果在这些情况之外使用break，就会产生错误。

下面你可以看到几个基本的例子，说明如何使用 break ，其语法很简单。

突破。

在一个for循环中休息。

for (var i = 0; i < 10; i += 1)
{
if (array[i] == 234)
{
break;
}
}
num = i;

在一个重复的循环中打破。

var i = 0;
var temp = 0;
repeat (10)
{
temp += array[i];
if (temp > max_total)
{
break;
}
else
{
i += 1;
}
}

在一个while循环中断裂。

var i = 0;
while (!place_free(x, y))
{
x = random(room_width);
y = random(room_height);
if (i > 50)
{
break;
}
else
{
i+=1。
}
}

break 在一个 do / until 循环中。

var _id = noone;
do
{
_id = list[| 0];
if instance_exists(_id)
{
_break;
}
ds_list_delete(list, 0);
}
until (ds_list_empty(list));
target = _id;

使用时断裂。

var count = 0;
with (obj_Enemy)
{
count++;
if (count > 10)
{
break;
}
hp = 100;
}

break in a switch:

switch (keyboard_key)
{
case vk_left:
case ord("A"):
x -= 4;
break;
case vk_right:
case ord("D"):
x += 4;
break;
case vk_up:
case ord("W"):
y -= 4;
break;
case vk_down:
case ord("S"):
y += 4;
break;
}
