删除

delete 操作符用于取消对结构的引用，其语法如下。

删除<变量>。

这意味着您在调用删除操作符的同时，还可以调用保存结构的变量，它将删除存储在给定变量中的结构的特定引用 （引用只是一个指向存储结构和内容的内存区域的值 存储结构及其内容的内存区域）。)如果该引用是您游戏中对该结构的最后一次引用，那么垃圾收集器将在接下来的迭代中从内存中删除该结构。 迭代，通常是在下一步的开始。

注意：删除只能与变量一起使用，而不是表达式。

默认情况下，代码中没有进一步引用的结构将在使用后的步骤中自动进行垃圾收集，但使用该操作符明确标记它们，以便垃圾收集器在下一步骤开始时从内存中删除，这是一个好的做法。 从内存中删除，从而快速、高效地释放内存。

下面的例子显示了一个正在创建的结构--例如在一个实例的创建事件中。

mystruct =
{
a : 0,
b : 100,
c : "Hello World"
}

然后这个结构将像往常一样在实例事件中使用，最后在清理事件中被标记为垃圾收集，就像这样。

删除神秘结构。
