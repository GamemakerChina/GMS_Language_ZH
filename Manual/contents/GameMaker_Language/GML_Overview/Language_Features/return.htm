返回

返回语句的语法如下。

返回 (<表达式>)

你只在 脚本函数和方法中使用返回语句，它被用来从函数中返回一个值，以便在进一步的代码或函数调用中使用。 调用。应该注意的是，函数的执行在 return 语句处结束，也就是说，任何在 return 被调用之后的代码都不会被运行。下面是一个简短的 脚本的例子，它计算传递给它的任何数值的平方，并且它包括在传递的参数不是实数的情况下的错误捕捉。

/// @function sqr_calc(val);
/// @param {real} val 要计算的数值的平方
/// @description 计算给定数值的平方

function sqr_calc(value)
{
if !is_real(value)
{
return 0;
}
else
{
return (value * value);
}
}

要在一段代码中调用一个函数，只需使用与调用运行时函数相同的方法--即把函数名称和参数值写在括号里。因此，上述函数将被这样调用。

if keyboard_check_pressed(vk_enter)
{
val = scr_sqr(amount);
}

这里的变量 val ，要么是0（因为变量 amount 不是实数），要么是 value * value 的总数。
