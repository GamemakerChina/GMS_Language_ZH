结构禁忌变量

在创建结构时，可以使用某些内置变量作为成员变量名称，例如。

mystruct = {
object_index : obj_Player,
speed : 3,
image_blend : c_red
}

然而，只有局部和实例范围的内置变量可以这样使用，并像常规结构成员变量一样赋值。使用全局范围的内置变量是被禁止的，并且会给你的游戏带来问题。 下面你可以找到这些变量的完整列表，以便你知道哪些变量需要避免。
