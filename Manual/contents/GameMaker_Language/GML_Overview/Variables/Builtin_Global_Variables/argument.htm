争论

这个变量持有一个数组，与脚本函数或方法中的只读变量argument_count一起使用。 每个数组的位置都有一个函数的输入值，专门用于变量 参数函数。

请注意，还有一系列独立的全局范围变量，这些变量也可以在用户定义的函数中用来引用不同的输入参数： argument0, argument1, argument2...等等。 等等......直到argument15。

注意：这不再是严格意义上的要求，更多的是为了支持传统的东西而提供。所有用户定义的函数将接受变量参数，任何传入函数的参数如果不属于命名的 输入的任何参数都将被默认初始化为未定义，这可以通过is_undefined()函数来检查。

语法。

参数[n]
参数1
argument2
...
argument15

返回。

值（可以是提供给函数的任何数据类型）。

例子。

函数 my_array()
{
var i, arg;
for (i = 0; i < 5; i += 1;)
{
if argument_count > i
{
arg[i] = argument[i]
}
else
{
arg[i] = -1;
}
}
}

上面的代码使用argument_count变量以及 argument 数组来初始化变量，是一个允许用户定义的函数接受可变数量参数的例子。

返回。全局变量
