得分

这个变量是全局范围的，用来保存一个数字值，通常用于玩家的分数。这个变量只被设计用来支持以前版本的 游戏制作者的遗留项目，并且应该 不在新项目中使用 因为它在未来可能会被废弃。

语法。

分。

返回。

实数（单精度浮点值

例子


switch (object_index)
{
case obj_Enemy_Fighter:
score += 10;
break;
case obj_Enemy_Mage:
score += 25;
break;
case obj_Enemy_Boss:
score += 100;
} break.
instance_destroy()。

上面的代码使用开关检查运行代码的实例的对象索引，然后根据是什么对象，向 score 变量添加不同的金额。

返回。全局变量
