animcurve_channel_new

这个函数可以用来创建一个新的动画曲线通道结构。一个通道结构有以下变量。

变量名称 数据类型 描述
名称 绳子 这是频道的名称。
类型 常数 这将是一个常数 曲线类型_线性- 用于点与点之间的线性内插 - 或 animcurvety_catmullrom- 用于使用catmull-rom插值的点之间的 "平滑 "插值。
迭代 整数 如果通道使用catmull-rom("平滑")插值，这个值表示为曲线的每一段生成了多少个点(注意，这些额外的点是函数内部的，只用于运行时的计算)。 如果通道使用的是线性插值，这个值仍然存在，但可以忽略，因为它对曲线的插值方式没有影响。
点 阵列指针 这是一个数组，其中数组中的每一项都是一个点结构。


默认情况下，当您创建一个新的通道结构时，"name "变量将是一个空字符串，"type "将是animcurvetype_linear，"iterations "将是16。所有这些变量都可以被设置为 值（注意 "迭代次数 "的值对线性曲线类型没有影响）。创建后，您需要为曲线添加点，这可以通过在 "点 "阵列中添加不同的点结构来实现。 这些点的结构可以用函数创建 animcurve_point_new()创建，并保存以下变量。

变量名称 数据类型 描述
姿势 真正的 该点在时间上的位置（从0到1的正常化）。
价值 真正的 点的价值。

语法。

animcurve_channel_new()。

返回。

结构参考

例子。

my_curve = animcurve_create();
my_curve.name = "My_Curve";
var _channels = array_create(1);
_channels[0] = animcurve_channel_new();
_channels[0] .name = "alpha";
_channels[0] .type = animcurvetype_catmullrom;
_channels[0] .iterations = 8;
var _points = array_create(3);
_points[0] = animcurve_point_new();
_points[0].posx = 0;
_points[0].value = 0;
_points[1] = animcurve_point_new();
_points[1].posx = 0.5;
_points[1].value = 1;
_points[2] = animcurve_point_new();
_points[2].posx = 1;
_points[2].value = 0;
_channels[0].point = _points;
my_curve.channels = _channels;

上面的代码创建了一个新的动画曲线结构，并将其存储在变量 "my_curve "中。然后这个结构被填充了一个名称和一个通道数组。通道数组包含一个有三个点的单一通道。
