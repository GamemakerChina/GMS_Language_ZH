创建动画曲线

这个函数将创建一个空的动画曲线结构，准备让你填充通道数据。该函数将返回一个指向该结构的指针，然后用它来为动画曲线添加通道 和其他数据添加到动画曲线中。你也可以把它提供给像 animcurve_get()等函数，以便以后从曲线上获取数据。

当你使用这个函数时，创建的结构中没有数据，要使用它，你应该至少添加一个通道，并且该通道应该添加点。要添加一个通道，请参见函数 animcurve_channel_new(), 要向通道中添加点，请参见函数 animcurve_point_new().此外，您可以通过设置 "name "变量为动画曲线结构命名（如下图所示）。 下面的示例代码）。)添加到新的动画曲线中的通道应该以数组的形式添加，数组中的每一项都引用一个通道结构，通道中的点也应该被添加为 作为一个数组，其中每个数组项都引用一个点结构。

请注意，以这种方式创建的动画曲线必须在不再需要时删除，否则会占用内存空间，这可能会导致内存泄漏，使游戏速度变慢并最终崩溃。 你可以在不再需要的时候删除一条动画曲线，使用函数 animcurve_destroy().你不需要清理曲线中的任何通道或点的数据，因为这将会在 因为当曲线本身被销毁时，垃圾收集器会自动完成。

语法。

animcurve_create()。

返回。

结构参考

例子。

my_curve = animcurve_create();
my_curve.name = "My_Curve";
var _channels = array_create(1);
_channels[0] = animcurve_channel_new();
_channels[0] .name = "alpha";
_channels[0] .type = animcurvetype_catmullrom;
_channels[0] .iterations = 8;
var _points = array_create(3);
_points[0] = animcurve_point_new();
_points[0].posx = 0;
_points[0].value = 0;
_points[1] = animcurve_point_new();
_points[1].posx = 0.5;
_points[1].value = 1;
_points[2] = animcurve_point_new();
_points[2].posx = 1;
_points[2].value = 0;
_channels[0].point = _points;
my_curve.channels = _channels;

上面的代码创建了一个新的动画曲线结构，并将其存储在变量 "my_curve "中。然后这个结构被填充了一个名称和一个通道数组。通道数组包含一个有三个点的单一通道。
