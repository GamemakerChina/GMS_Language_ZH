动画curve_get

这个函数将返回一个包含给定动画曲线的所有数据的结构体。你提供了动画曲线的资产ID（如在资产浏览器中定义的），该函数将返回一个结构体 包含以下变量。

变量名称 数据类型 描述
名称 绳子 这是该动画曲线的名称。
渠道 阵列指针 这是一个数组，数组中的每一项都是一个通道结构。


结构中的通道变量是一个数组，每个数组项是一个结构，包含与曲线中的通道有关的数据。通道结构将有以下变量。

变量名称 数据类型 描述
名称 绳子 这是频道的名称。
类型 常数 这将是一个常数 曲线类型_线性用于点与点之间的线性插值，或 animcurvety_catmullrom用于使用catmull-rom插值在各点之间进行 "平滑 "插值。
迭代 整数 如果通道使用catmull-rom("平滑")插值，这个值表示为曲线的每一段生成了多少个点(注意，这些额外的点是函数内部的，只用于运行时的计算)。 如果通道使用的是线性插值，这个值仍然存在，但可以忽略，因为它对曲线的插值方式没有影响。
点 阵列指针 这是一个数组，其中数组中的每一项都是一个点结构。


与通道一样，单个通道上的点以结构形式存储在一个数组中，数组中的每一项都是单个点结构。点结构有以下变量。

变量名称 数据类型 描述
姿势 真正的 该点在时间上的位置（从0到1的正常化）。
价值 真正的 点的价值。


注意，如果该函数失败（即：给定的动画曲线ID不存在），则该函数将返回-1。

语法。

animcurve_get(curve_id)。

争论 描述
曲线_id 要获取的动画曲线的资产浏览器ID（索引）。

返回。

结构或-1

例子。

var _curve = animcurve_get(ac_ButtonTween);
var _channel = _curve.channels[0];
if _channel.type != animcurvetype_linear
{
_channel.type = animcurvetype_linear;
}

上面的代码检索了曲线资产 "ac_ButtonTween "的动画曲线结构，然后检查第一个通道的曲线类型，如果不是线性的，就把它设置为线性。
