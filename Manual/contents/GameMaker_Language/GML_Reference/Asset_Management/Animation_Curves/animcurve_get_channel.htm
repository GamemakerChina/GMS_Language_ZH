animcurve_get_channel

这个函数将返回包含动画曲线资产中指定通道数据的结构。你可以提供动画曲线的ID或结构，以及通道名称或索引。 函数将返回一个格式如下的结构。

变量名称 数据类型 描述
名称 绳子 这是频道的名称。
类型 常数 这将是一个常数 曲线类型_线性用于点与点之间的线性插值，或 animcurvety_catmullrom用于使用catmull-rom插值在各点之间进行 "平滑 "插值。
迭代 整数 如果通道使用catmull-rom("平滑")插值，这个值表示为曲线的每一段生成了多少个点(注意，这些额外的点是函数内部的，只用于运行时的计算)。 如果通道使用的是线性插值，这个值仍然存在，但可以忽略，因为它对曲线的插值方式没有影响。
点 阵列指针 这是一个数组，其中数组中的每一项都是一个点结构。


动画曲线ID是在资产浏览器中定义的动画曲线的名称，或由函数返回的结构指针。 animcurve_create().通道名称是一个字符串 指的是动画曲线资产中定义的通道，或者你可以提供一个索引值，从0到n，其中n是曲线资产中的最后一个通道（例如：如果一个动画曲线有4个通道，它们将被索引为 从0到3）。)请注意，传递一个索引值比传递一个通道名称需要更少的处理。如果函数失败（即：没有给定名称或索引的通道存在），你将得到一个错误。

单个通道上的点以结构形式存储在一个数组中，数组中的每一项都是一个点结构。点结构有以下变量。

变量名称 数据类型 描述
姿势 真正的 该点在时间上的位置（从0到1的正常化）。
价值 真正的 点的价值。

语法。

animcurve_get_channel(curve_struct_or_id, channel_name_or_index)。

争论 描述
曲线_结构_或id 目标动画曲线的ID或结构指针
channel_name_or_index 通道名称（一个字符串）或通道索引（一个整数）。

返回。

结构

例子。

var _channeldata = animcurve_get_channel(ac_ButtonTween, 0);
var _points = _channeldata.point;
for (var i = 0; i < array_length(_points); ++i;)
{
_points[i].value += 1;
}

上面的代码检索了曲线资产 "ac_ButtonTween "中通道0的通道结构，然后循环浏览通道曲线上的点，并在它们的值上加一。
