资产_获取_索引

你可以使用这个函数从一个游戏资产的名称中获取其唯一的识别索引。如果没有找到该资产，该函数将返回一个值-1，否则它将返回被检查资产的唯一索引ID。然后这个ID可以 可以在其他函数中使用，就像使用其他的索引值一样，比如 精灵指数(sprite_index)或 路径_index, 例如。请注意，尽管这个函数可以用来从字符串中引用资产（见下面的例子），但在使用它之前，你应该确保资产是存在的，否则你可能会得到错误，使你的游戏崩溃。

语法。

asset_get_index(name)。

争论 描述
名称 要获得索引的游戏资产的名称（一个字符串）。

返回。

资产指数（整数）。

例子。

var obj = asset_get_index("obj_Enemy_" + string(global.Level));
if obj > -1
{
instance_create_layer(random(room_width), random(room_height), obj, "Enemy_Layer");
}

上面的代码将从一个字符串中获得一个对象索引，如果该索引存在，就在游戏中创建一个对象的实例。

返回。资产和标签
