资产_获取类型

通过这个函数，你可以从资产的名称（一个字符串）中获得被引用的资产的类型。下面列出的常数之一将被返回。

语法。

asset_get_type(name)。

争论 描述
名称 要获得的游戏资产类型的名称（一个字符串）。

返回。

恒定

恒定 描述
资产_对象 给定的名称指的是一个对象。
资产_石膏 给定的名称指的是一个精灵。
资产_声音 这个名字指的是一种声音。
资产室 这个名字指的是一个房间。
资产 给定的名称指的是一个瓦片组。
资产路径 给定的名称指的是一个路径。
资产脚本 给定的名称指的是一个脚本。
资产字体 该名称指的是一种字体。
资产时间线 这个名字指的是一条时间线。
资产着色器 这个名字指的是一个 着色器.
资产_动画曲线 该名称指的是一个动画曲线。
资产序列 给定的名称指的是一个Sequence。
资产_不详 该名称指的是一种不存在的资产，或者不是上述列出的资产之一。

例子。

if asset_get_type("pth_Path_" + string(global.Game)) == asset_unknown
{
show_debug_message("路径不存在!!");
}
else
{
path_index = asset_get_index("pth_Path_" + string(global.Game));
}

上面的代码检查一个动态创建的资产名称，看该资产的类型是否正确。如果不是，那么将显示一条调试信息，否则，资产名称将被用于将资产分配给该实例。
