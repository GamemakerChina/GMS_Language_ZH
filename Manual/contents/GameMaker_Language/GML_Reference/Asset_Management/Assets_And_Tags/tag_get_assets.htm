tag_get_assets

通过这个函数，你可以检索到所有被分配了给定标签的资产的名称。你可以提供一个单一的标签字符串或一个数组，数组中的每一项都是一个标签字符串。 该函数将返回一个数组，其中每个条目是具有给定标签的资产的名称（字符串）。如果你需要资产的唯一索引，那么你可以使用这个函数 asset_get_index()和 以及返回的名称。如果没有给定标签的资产，或者出现错误（例如：给定标签不存在），那么将返回一个空数组。

语法。

tag_get_assets(tags)。

争论 描述
标签 一个单一的资产标签字符串或一个包含各种资产标签的数组。

返回。

阵列

例子。

backgrounds = ds_list_create();
var _assets = tag_get_assets("background");
for (var i = 0; i< array_length(_assets); ++i;)
{
if asset_get_type(_assets[i]) == asset_sprite
{
ds_list_add(backgrounds, asset_get_index(_assets[i]) );
}
}

上面的代码创建了一个列表，然后检索所有带有 "background "标签的资产的名称。它在返回的名称数组中循环，检查其中是否有精灵资产，如果有，则将该资产的唯一索引值 的唯一索引值被添加到列表中，以供将来使用。

返回。资产和标签
