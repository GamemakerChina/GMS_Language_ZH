音频_get_recorder_count

这个函数将返回你的游戏当前可用的音频记录源（如麦克风等......）的数量。因此，如果你的返回值是4，那么你将有0、1、2和3个音频输入源，其中一个是你使用的。 其中一个值是你使用函数开始录音的值。 audio_start_recording().这个值可以在任何时候改变，因为人们在设备上插入/拔出麦克风或其他 这个值可以随时改变，因为人们在设备上插拔麦克风或其他输入源。请注意，你可以使用功能 audio_get_recorder_info()来获取所连接的每个设备的信息。

重要的是!在HTML5目标上使用该函数时，将始终返回0。

语法。

audio_get_recorder_count()。

返回。

整数

例子。

如果 audio_get_recorder_count() > 0
{
channel_index = audio_start_recording(0);
}

上述代码检查可用的录音机，如果有1个或更多，就从第一个索引的录音机（源0）开始录音。
