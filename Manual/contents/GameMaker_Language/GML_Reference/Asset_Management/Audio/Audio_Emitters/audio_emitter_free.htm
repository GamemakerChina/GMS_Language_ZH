语音识别器

通过这个函数，你可以从内存中删除指定的发射器。这应该在房间或游戏中不再使用发射器时进行，例如：在实例的Destroy事件或房间结束事件中。 实例的销毁事件或房间结束事件中，否则你可能会出现内存泄漏，从而导致游戏速度减慢并最终崩溃。

语法。

audio_emitter_free(emitter)。

争论 描述
发射器 要释放的发射器的索引。

返回。

不适用

例子。

if lives = 0
{
audio_emitter_free(s_emit);
room_goto(rm_Menu);
}

上述代码检查全局变量 "lives "的值，如果它返回0，它将销毁变量 "s_emit "中索引的发射器，然后去变量 "rm_Menu "中索引的房间。

返回。音频发射器
