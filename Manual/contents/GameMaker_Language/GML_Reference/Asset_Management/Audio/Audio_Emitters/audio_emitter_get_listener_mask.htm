音频发射器_get_listener_mask

这个函数将返回定义发射器应该从哪些音频监听器中播放声音的比特掩码数据。更多信息请参见音频监听器一节。

语法。

audio_emitter_get_listener_mask(emitterID)。

争论 描述
emitterID 要获取掩码的发射器的唯一ID。

返回。

真实的

例子。

s_emit = audio_emitter_create();
if audio_emitter_get_listener_mask(s_emit) != global.PlayerMask
{
audio_emitter_set_listener_mask(snd, global.PlayerMask);
}

上面的代码创建了一个发射器，然后检查它的监听器掩码数据，如果它和存储在全局变量中的数据不一样，它就用存储在全局变量中的掩码数据来设置监听器的播放。
