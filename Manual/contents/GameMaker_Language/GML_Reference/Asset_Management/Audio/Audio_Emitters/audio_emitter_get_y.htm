音频_发射器_get_y

该函数返回给定音频发射器的当前Y位置。

语法。

audio_emitter_get_y(emitter)。

争论 描述
发射器 要使用的发射器的索引。

返回。

真实的

例子。

if audio_emitter_get_y(emitter_player) != x
{
audio_emitter_position(emitter_player, x, y, 0);
}

上述代码检查给定发射器的当前y位置，如果它不等于实例的y位置，则将其设置为实例的位置。

返回。音频发射器
