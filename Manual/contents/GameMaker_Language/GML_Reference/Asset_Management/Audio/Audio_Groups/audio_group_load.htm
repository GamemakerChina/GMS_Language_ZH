音频组加载

这个函数将把所有被标记为属于给定音频组的声音加载到内存中。如果加载开始了，该函数将返回true，如果组的 ID是无效的，或者已经被标记为加载。该函数是异步的，所以当音频在后台加载时，你的游戏将继续运行。这意味着它也会触发一个异步加载/保存事件，以通知你整个组已经被加载。 通知你整个组已经被加载到内存中，现在可以使用这些声音了。

语法。

audio_group_load(groupID)。

争论 描述
groupID 要加载的音频组的索引（如在音频组窗口中定义）。

返回。

布尔型

例子。

if !audio_group_is_loaded(audiogroup_level1)
{
audio_group_load(audiogroup_level1)。
}

上面的代码检查是否已经加载了一个音频组，如果没有，就加载它。
