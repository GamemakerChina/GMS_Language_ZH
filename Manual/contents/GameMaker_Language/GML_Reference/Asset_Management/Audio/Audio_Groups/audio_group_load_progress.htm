音频组加载进度

这个函数将检查一个音频组的加载进度，并返回一个0到100之间的（近似）值。

语法。

audio_group_load_progress(groupID)。

争论 描述
groupID 要检查的音频组的索引（如在音频组窗口中定义）。

返回。

真实的

例子。

如果加载
{
var pc = audio_group_load_progress(audiogroup_level1);
draw_text(32, 32, "Loading: " + string(pc));
}

上面的代码检查一个变量，如果它返回真，那么就会画出一些文本，显示正在加载音频的进度。
