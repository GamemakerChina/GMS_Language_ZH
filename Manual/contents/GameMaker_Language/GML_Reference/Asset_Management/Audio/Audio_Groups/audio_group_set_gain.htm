音频组_设置增益

用这个功能，你可以在给定的时间长度内淡化一组声音，也可以用来即时设置组的增益。时间是以毫秒为单位的，该功能要求你输入一个最终的增益水平，以便在该时间结束时使该组声音达到。 该功能要求你输入一个最终的增益水平，以便在该时间结束时群组能够达到。这个增益可以在0（无声）和1（全音量）之间，而且刻度是线性的，比如说0.5的值就是一半的音量。要立即改变增益，只需将时间参数设置为0。 在一些平台上，你可以有一个大于1的增益，尽管1的值被认为是 "全音量"，任何更大的值都可能引入音频削波。

语法。

audio_group_set_gain(groupID, volume, time)。

争论 描述
groupID 要停止的音频组的索引（如在音频组窗口中定义）。
量 组卷的最终值。
时间 增益变化的长度，以毫秒计。

返回。

不适用

例子。

if keyboard_check_pressed(vk_space)
{
audio_group_set_gain(audiogroup1, 0, 5000);
}

上面的代码检查 "空格 "键，然后在5秒内将 "audiogroup1 "的所有音频淡化为0。
