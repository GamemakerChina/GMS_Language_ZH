音频组_停止所有

这个函数将停止当前正在播放的给定音频组中的所有声音。

语法。

audio_group_stop_all(groupID)。

争论 描述
groupID 要停止的音频组的索引（如在音频组窗口中定义）。

返回。

不适用

例子。

if keyboard_check_pressed(vk_space)
{
audio_group_stop_all(audiogroup_level1);
}

上面的代码检查 "空格 "键，然后停止所有来自给定组的音频播放。

返回。音频组
