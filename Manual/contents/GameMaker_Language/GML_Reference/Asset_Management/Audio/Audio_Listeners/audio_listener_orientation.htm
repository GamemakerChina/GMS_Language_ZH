音频听众的方向

通过这个功能，你可以改变听众在三维音频空间中的方向。观察方向和上升方向是基于从给定的相对X、Y和Z位置解析出来的矢量。 来解决的，默认为（0, 0, 1000）为观察方向，（0, 1, 0）为上升方向，如下面的图所示。

注意： 如果你有多个监听器，你应该使用函数 audio_listener_set_orientation().

用这个函数改变听众的方向将改变游戏室周围的音频发射器所产生的声音被你的游戏玩家感知的方式。在下面的例子中，当听众处于默认位置时，由发射器产生的声音会从听众的下方和右侧传来。 位置时，声音会从听众的下面和右边传来，但在听众的新位置和方向上，它们现在会被认为是从上面和右边来的。

语法。

audio_listener_orientation(lookat_x, lookat_y, lookat_z, up_x, up_y, up_z)。

争论 描述
观察_x x看向量（默认为0）。
看一看_y y看向量（默认为0）。
lookat_z z看向量（默认为1000）。
up_x x向上的向量（默认为0）。
上一页_y y向上的向量（默认为1）。
up_z z向上的向量（默认为0）。

返回。

不适用

例子。

xt = x + dcos(direction);
yt = y - dsin(direction);
zt = z - dsin(zdirection);
audio_listener_position(x, y, z)
audio_listener_orientation(xt, yt, zt, 0, 0, 1)

上面的代码使用三个变量来设置3D音频监听器的位置和方向。
