聆听者设置位置

通过这个函数，你可以改变一个给定的听众在三维音频空间中的位置。默认监听器的索引是0，但你可以用这个函数 audio_get_listener_info()来 获得目标平台上的不同指数。下面的示例图显示了听众在音频空间中的默认位置。

如你所见，默认位置是（0, 0, 0），但你通常会使用这个函数 来移动监听器和游戏中的玩家对象，从而改变玩家听到发射器产生的音频的方式，例如，在下面这个俯视游戏的图片中，玩家实例设置了监听器，这将导致 当玩家在关卡中移动时，来自不同发射器的音频会发生 "变化"。

语法。

audio_listener_set_position(index, x, y, z)。

争论 描述
指数 要获取数据的监听器（默认为0）。
x 听众的X位置（默认为0）。
y 倾听者的y位置（默认为0）。
z 倾听者的z位置（默认为0）。

返回。

不适用

例子。

var _vmat = camera_get_view_mat(view_camera[0]);
audio_listener_set_position(global.Player_Listener, _vmat[0], _vmat[1], _vmat[2]);
audio_listener_set_orientation(info[? "index"], _vmat[3], _vmat[4], _vmat[5], _vmat[6], _vmat[7], _vmat[8]）。)

上面的代码检索了摄像机视图[0]的视图矩阵，然后用它来设置音频监听器的位置和方向，监听器的ID存储在全局变量 "Player_Listener "中。
