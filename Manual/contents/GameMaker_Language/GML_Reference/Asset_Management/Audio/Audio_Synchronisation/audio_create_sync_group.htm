音频创建同步组

创建一个同步组，并返回一个唯一的ID值，这个ID值应该在这个组的所有后续音频函数调用中使用。如果你想让这个组循环，那么就传入true，否则就传入false。 想让它们循环播放，那么后面添加的所有音轨都需要是相同的长度。注意，当你创建一个同步组时，你将需要释放内存和与之相关的声音，在不使用时使用 audio_destroy_sync_group() 函数释放与之相关的内存和声音--例如，在Room End或Destroy事件中。

注意：该功能不适用于HTML5目标平台。

语法。

audio_create_sync_group(loop)。

争论 描述
循环 该组中的轨道是否应该循环（true）或不循环（false）。

返回。

指数

例子。

sg = audio_create_sync_group(true);
audio_play_in_sync_group(sg, sound1);
audio_play_in_sync_group(sg, sound2);
audio_sound_gain(sound2, 0, 0);
audio_play_in_sync_group(sg, sound3);
audio_sound_gain(sound3, 0, 0);
audio_play_in_sync_group(sg, sound4）;
audio_sound_gain(sound4, 0, 0);
audio_start_sync_group(sg);

上面创建了一个新的同步组，并将该组的索引分配给变量 "sg"。然后，四个声音被添加到组中，其中三个声音的增益被设置为0，最后，同步组被播放。
