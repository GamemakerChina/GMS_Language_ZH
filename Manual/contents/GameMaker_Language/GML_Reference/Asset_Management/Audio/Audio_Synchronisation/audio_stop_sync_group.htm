音频停止_同步组

这个函数将停止给定的同步组，如果它正在播放，组的索引是你使用函数创建组时返回的值。 audio_create_sync_group().

注意：该功能不适用于HTML5目标平台。

语法。

audio_stop_sync_group(group_index);

争论 描述
组_索引 这群人指数停止播放。

返回。

不适用

例子。

if mouse_check_button_pressed(mb_left)
{
audio_stop_sync_group(sg);
}

上面的代码检查鼠标点击，如果检测到鼠标点击，就停止变量 "sg "中索引的同步组。
