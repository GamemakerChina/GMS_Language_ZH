音频同步组正在播放

这个函数可以用来检查一个同步组中的任何音频是否正在播放。你需要提供由该函数返回的同步组ID。 audio_create_sync_group().

注意：此功能不适用于HTML5目标平台。

语法。

audio_sync_group_is_playing(group_index)。

争论 描述
组_索引 要检查的组别索引。

返回。

不适用

例子。

if audio_sync_group_is_playing(group_one)
{
audio_stop_sync_group(group_one);
}

上面的代码检查同步组group_one是否正在播放，如果是，则停止该组。
