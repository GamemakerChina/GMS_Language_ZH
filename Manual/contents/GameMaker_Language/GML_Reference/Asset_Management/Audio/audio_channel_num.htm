音频_通道_编号

通过这个功能，你可以设置在GameMaker Studio 2中，有多少个音频通道可以用来播放音频。这基本上意味着，你给出了在任何时候可以同时播放的声音的数量，如果 数目超过这个数量，那些优先级较低的声音就会被停止，以腾出一个通道给优先级较高的声音。你可以使用这个功能来优化你的游戏设备 因为音频通道的数量越少，性能就越好，但请记住，如果一次播放很多声音，这也可能导致某些声音的明显切断。

语法。

audio_channel_num(num)。

争论 描述
数值 可用的音频通道的数量（默认为128）。

返回。

不适用

例子。

switch (os_browser)
{
case browser_not_a_browser:
switch (os_type)
{
case os_windows:
case os_macos:
audio_channel_num(200);
break;
默认。
audio_channel_num(64);
break;
}
break;
默认。
audio_channel_num(16);
break;
}

上述代码检查游戏运行的平台，并改变可用的声音通道的数量以提高性能。
