音频_存在

这个函数返回一个声音是否存在（真）或不存在（假）。请注意，如果你搜索的索引之前没有被初始化，这个函数将导致一个错误，因为它正在搜索不存在的资产索引。要检查的 要检查的声音可以是一个声音的单一实例（当使用 "或 "时，正在播放的单个声音的索引可以存储在一个变量中）。 audio_play_sound()或 audio_play_sound_at()函数时，可以将单个声音的索引存储在一个变量中）。) 或者是一个声音资产。

语法。

audio_exists(index)。

争论 描述
指数 要检查存在的声音的索引。

返回。

布尔型

例子。

if audio_exists(global.Music)
{
audio_play_sound(global.Music, 0, true);
}

上面的代码检查是否存在一个声音。如果有的话，它就被设置为循环播放。
