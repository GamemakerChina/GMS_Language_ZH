音频_set_master_gain

通过这个函数，您可以为一个特定的听众设置所有声音和音乐的全局音量的绝对值。默认的听众索引是0，但你可以用这个函数 audio_get_listener_info()来获取目标平台的不同指数。 获得目标平台上的不同指数。增益值是基于从0（无声）到1（全音量）的线性尺度，并将影响所有通过该监听器从你的游戏内播放的声音和音乐的相对音量。

语法。

audio_set_master_gain(listenerIndex, gain)。

争论 描述
听众索引 要设置增益的听众的索引。
收益 全局音量的值（0到1）。

返回。

不适用

例子。

var num = audio_get_listener_count();
for( var i = 0; i < num; i++;)
{
var info = audio_get_listener_info(i);
audio_set_master_gain(info[? "index"], 0.75);
ds_map_destroy(info);
}

上面的代码在可用的监听器中循环，然后将它们的主增益设置为0.75。
