音频_声音_是可播放的

这个函数可以用来检查当前是否可以播放给定的声音索引。由于HTML5目标平台上处理流式和非流式声音播放的方式不同，所以需要这个函数。 如果声音可以被播放，则返回true，如果不能，则返回false。请注意，在除HTML5以外的所有其他平台上，该函数将总是返回true。

语法。

audio_sound_is_playable(index)。

争论 描述
指数 要检查的声音的索引。

返回。

真实的

例子。

if audio_sound_get_pitch(snd_Explode) != 1
{
audio_sound_pitch(snd_Explode, 1);
}

上面的代码将改变从索引为 "snd_Explode "的声音中播放的音频的音调，如果它的音调不等于1。
