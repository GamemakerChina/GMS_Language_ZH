音频_声音_设置音轨_位置

这个函数将设置声音文件中声音播放的位置（以秒为单位）。声音可以是一个单独的声音实例（当使用 "Audio_play_sound_at "函数时，被播放的单个声音的索引可以存储在一个变量中。 audio_play_sound()或 audio_play_sound_at()函数时，可以将单个被播放的声音的索引存储在一个变量中）或一个声音资产。如果你使用的是一个独特的声音ID，那么位置将立即改变 到给定的新时间，如果是资产浏览器的资产，那么所有给定的声音的进一步播放将从新时间开始。该函数将忽略任何比 "索引 "所给的声音长度长的值。 值。

语法。

audio_sound_set_track_position(index, time);

争论 描述
指数 要改变的声音的索引。
时间 将起始点设置为的时间（以秒为单位）。

返回。

不适用

例子。

var snd = audio_play_sound(snd_MainTrack, 0, false);
audio_sound_set_track_position(snd, 32);

上面的代码播放了一个声音，然后使用返回的ID值，将声音的开始位置设置为32秒。

返回。音频
