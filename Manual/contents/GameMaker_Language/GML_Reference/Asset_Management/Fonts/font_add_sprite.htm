通过这个函数，你可以使用 "精灵条 "来创建一个新的字体资产，其中每个子图像将是一个单独的符号或字母。索引是要使用的第一个字符的UTF8地图编号，使用的范围与普通字体相同。 使用的范围与普通字体相同，以"！"开始，但请记住，你应该只包括你需要的内容，而且你应该使用 ord()函数来获取起始字母的正确UTF8值。 字母。创建的精灵字体将把子图像设置为字体中的单个字形，其中 "第一个 "字形的image_index为0。 你可以将 "空格 "定义为你想要的任何字符，比如说 你可以将 "空格 "定义为你想要的任何字符，例如：你想要的空格大小的单行，文本中的所有空格都将以该宽度呈现（但所选择的图像将不会被呈现），或者如果你没有为空格提供一个精灵，那么将使用精灵字体中最宽的字符的宽度 则将使用精灵字体中最宽的字符的宽度来代替。

你还可以将字体的间距设置为比例（true）或非比例（false），其中比例字体的间距是基于字母的实际宽度（所以字母 "i "比字母 "w "占用的空间要小。 例如，字母 "i "比字母 "w "占用的空间要小），而非比例字体将根据子图像的宽度来确定间距，就像单行线字体一样。最后，你可以定义书写时每个字母之间的空间，这个值可以是任何整数，0表示没有空间（如果是比例字体，字母会 "接触"）。 字母将按比例 "接触"）。)该函数返回一个索引值，这个索引值应该保存在一个变量中，因为在以后所有引用该字体的代码中都需要这个索引值，如果该函数因任何原因而失败，它将返回-1。 则返回-1。

还值得注意的是，字符图中不能有任何空格，因此，如果你，例如，使用'.'和'0'，但不使用'/'，你需要包括'/'，或者在两个使用的字符之间留下一个空白子图像。 或者在两个使用的字符之间留下一个空白的子图像。

该精灵本身可以是资产浏览器中的精灵资产，也可以是从文件中加载的精灵，首先使用sprite_add来获取加载的精灵的索引（然后在函数中使用）。值得注意的是，对于字体对齐的 函数（比如 draw_set_halign())要想在使用精灵字体时如期工作，精灵的原点应该设置在左上角。如果你使用其他值，那么你将 需要在使用该字体绘制文本时考虑到原点偏移。

注意： 当你把一个字体加载到GameMaker Studio 2中时，你必须记得在不再需要它时再把它删除（用 font_delete())，否则会有 有内存泄漏的风险，这将使你的游戏变慢并最终崩溃。

例子。

global.Font = font_add_sprite(spr_HUD_Font, ord("!"), true, 2);

上面的代码将从一个精灵资产中创建一个新的字体资产，并将新字体的索引存储在变量 "global.Font "中，供以后使用。
