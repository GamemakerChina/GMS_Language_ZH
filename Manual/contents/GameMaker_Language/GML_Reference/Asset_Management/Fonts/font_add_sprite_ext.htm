font_add_sprite_ext

通过这个函数，你可以使用一个 "精灵条"（精灵本身必须是资产浏览器中的一个精灵资产，或者是你用精灵_添加到游戏资源中的一个精灵）来创建一个新的字体资产，其中每个子图像 将是一个单独的符号或字母。不像普通的 font_add_sprite()有着精灵子图像的特定顺序，这个函数将根据函数的参数 "string_map "来映射精灵子图像。 "string_map"。这个参数是一个字符串，你可以用它来告诉GameMaker Studio 2sprite字体的子图像是什么顺序，它将在编写文本时自动映射这些图像。因此，举例来说，如果你有一个字符串映射 为 "AaBbCcDdEeFfGgHh"，你的精灵字体的子图像必须以这种方式排列。你可以将 "空格 "定义为你想要的任何字符，例如，你想要的空格大小的单行，而所有的空格 将以该宽度呈现（但所选择的图像将永远不会被呈现），或者如果你没有为空格提供一个精灵，那么将使用精灵字体中最宽的字符的宽度来代替。

你还可以将字体的间距设置为比例（true）或非比例（false），其中比例字体的间距是基于字母的实际宽度（所以字母 "i "比字母 "w "占用的空间要小。 例如，字母 "i "比字母 "w "占用的空间要小），而非比例字体将根据子图像的宽度来确定间距，就像单行线字体一样。最后，你可以定义书写时每个字母之间的空间，这个值可以是任何整数，0表示没有空间（如果是比例字体，字母会 "接触"）。 字母将按比例 "接触"）。)该函数返回一个索引值，这个索引值应该保存在一个变量中，因为在以后所有引用该字体的代码中都需要这个索引值，如果该函数因任何原因而失败，它将返回-1。 则返回-1。

值得注意的是，为了让字体对齐功能发挥作用（如 draw_set_halign())，那么字体精灵的原点应该设置在左上角。如果你使用其他值 那么在使用该字体绘制文本时，你将需要考虑原点偏移的问题。

注意： 当你把一个字体加载到GameMaker Studio 2中时，你必须记得在不再需要它时再把它删除（用 font_delete())，否则会有 有内存泄漏的风险，这将使你的游戏变慢并最终崩溃。

语法。
