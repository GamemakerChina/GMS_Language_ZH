font_get_texture

这个函数为字体纹理页返回一个特殊的指针。然后，这个值可以在其他绘图函数中使用，特别是在使用基元的一般绘图中，以及在使用 以及 着色器函数。你可以使用这里找到的不同的texture_函数来获得更多关于返回纹理页的信息。

语法。

font_get_texture(font)。

争论 描述
字体 要使用的字体的索引。

返回。

指针（纹理页ID）。

例子。

tex = font_get_texture(fnt_Main)。

上述代码将获得索引为 "fnt_Main "的字体的纹理指针。

返回。字体
