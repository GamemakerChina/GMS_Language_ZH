font_get_uvs

这个函数返回一个数组，其中包含纹理页上字体纹理的UV坐标，在数组中填充以下数值。

  • [0] = 左
  • [1] = 顶部
  • [2] = 对
  • [3]=底部

然后，这个值可以在其他绘图函数中使用，特别是在一般的绘图中，当使用基元以及 着色器函数。

语法。

font_get_uvs(font)。

争论 描述
字体 要使用的字体的索引。

返回。

阵列

例子。

var tex = font_get_uvs(fnt_Main);
tex_left = tex[0];
tex_top = tex[1];
tex_right = tex[2];
tex_left = tex[3];

上面的代码将把给定背景的UV坐标存储在一个本地数组中，然后把这些值分配给实例变量。

返回。字体
