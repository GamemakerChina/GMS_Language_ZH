实例_停用_层

通过这个函数，你可以停用分配给某个特定层的所有实例。你需要提供图层ID，它可以是代码编辑器中写的图层名称（作为一个字符串），也可以是实际的图层ID值，如图所示 的实际层ID值。 layer_create()，并注意你只能用这个函数停用实例层。请注意，如果你已经停用了一个有对象实例的图层 标记为持久性的对象，那么你将需要在改变房间之前用函数重新激活该层。 instance_activate_layer()否则，该层上的任何持久化的实例将不会被转移，而会被丢弃。 否则，该层上的任何持久化实例将不会被带入，并将被丢弃。还要注意的是，停用并不是即时的，以这种方式停用的实例在调用该函数的事件结束前不会被认为是不活跃的。

注意：如果你在房间启动时停用一个实例（即：从房间创建代码，或从房间内一个实例的创建事件），所有从房间编辑器放置在房间内的实例在被停用之前仍然会运行它们的创建事件。

警告：停用启用了物理的实例不会停止其灯具在物理模拟中的互动。为此，你应该设置它们的 phy_active变量 为真或假，因为你激活/停用了这些实例。

语法。

instance_deactivate_layer(obj)。

争论 描述
层 要使用的层ID字符串（或值）。

返回。

不适用

例子。

instance_deactivate_layer("Enemy Layer");
var _vx = camera_get_view_x(view_camera[0]);
var _vy = camera_get_view_y(view_camera[0]) 。
var _vw = camera_get_view_width(view_camera[0]);
var _vh = camera_get_view_height(view_camera[0]);
instance_activate_region(_vx - 64, _vy - 64, _vw + 128, _vh + 128, false) 。

上面的代码停用了分配给 "Enemy_Layer "层的所有实例，然后激活了房间内的一个区域。
