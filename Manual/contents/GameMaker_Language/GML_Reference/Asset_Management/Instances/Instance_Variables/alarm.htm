警报

这个一维数组用于获取该实例可能拥有的任何警报的当前值，或者用于设置这些警报。每个对象的实例中都有12个报警器。 每一个都有自己的事件，当这个变量达到0时就会运行。

警报在每个步骤开始时进行倒计时，在Begin Step事件之后但在主Step事件之前，在为该步骤呈现任何东西之前。应该注意的是，当警报达到0时，并没有结束（尽管该事件已经被触发），因为下一步它将下降到0。 事件已经被触发），因为下一步它将下降到-1，所以如果你需要因为任何原因停止一个警报，你应该把它的数组值设置为-1而不是0。报警时间是以游戏步骤计算的，30的值是30步。 60就是60步，等等。

注意：一个没有动作或代码的警报不会运行。然而，即使只有注释和没有代码，警报也会倒计时。

你可以直接设置报警数组（也可以通过使用数组直接检索当前的报警值），但是在某些情况下这不是最合适的方法，所以你也可以使用函数 alarm_set()来 设置一个警报，并使用函数 alarm_get()来获取一个警报的值。

语法。

警报[val]。

返回。

整数（如果警报没有运行，则为-1）。

例子。

if keyboard_check_pressed(vk_space)

canshoot = false;
alarm[0] = room_speed;
