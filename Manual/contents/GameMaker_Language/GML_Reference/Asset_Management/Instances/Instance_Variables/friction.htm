摩擦力

摩擦是实例可以拥有的 "内置 "属性之一，可以用来使实例随着时间的推移而减速，当 速度为零以外的时候。它的工作原理很简单，就是每一步都要从速度中减去一个量 直到物体的速度为0，所以如果摩擦力被设置为，例如，0.1，而实例的速度为1（每步1像素），它将在10步过后减速并停止。也请注意，摩擦力是 也要注意，摩擦力对正的和负的速度的作用是相同的，最终的结果总是物体在给定的时间后有一个0的速度。

语法。

摩擦。

返回。

实数（单精度浮点值

例子。

如果abs(speed)>0
{
摩擦力 = 0.05;
}
else
{
摩擦力 = 0;
}

上述代码只有在实例的绝对速度高于0时才会应用摩擦力。
