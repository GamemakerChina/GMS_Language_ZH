重力

重力是所有实例都有的 "内置 "变量之一，当它被设置时，将在实例中施加一个恒定的力。 重力方向中施加一个恒定的力，同时影响到实例的 速度 和 direction 。请注意，重力是一个累积的力，如果你不选择限制最终速度，它将加速物体，通常你会将这个变量设置为小数点后的 值，比如0.01。如果你把 gravity 设置为0，那么就不会对实例施加重力（这是默认值）。

语法。

重力。

返回。

实数（单精度浮点值

例子。

if !place_meeting(x, y + 1, obj_Ground)
{
重力 = 0.01;
}
else
{
重力=0;
}

上面的代码只有在实例下面没有找到任何 "obj_Ground "的实例时才会应用重力。
