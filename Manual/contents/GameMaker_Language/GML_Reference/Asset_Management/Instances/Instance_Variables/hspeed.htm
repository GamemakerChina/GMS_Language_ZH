迅捷

hspeed是所有实例都有的 "内置 "属性之一，它定义了实例的水平移动速度（沿X轴），单位为每步像素。所以，hspeed为3意味着每一步向右移动3个像素 (+x)每一步，而hspeed为-3则意味着每一步向左(-x)移动3个像素。

请注意，如果你设置了 speed 和/或 direction 变量，那么 hspeed 的值就会自动更新以反映这些变化。 反映这些变化，同样，改变 hspeed 的值也会影响 speed 和 direction 的值。还要注意的是 gravity, gravity_direction和 friction都可以在你的游戏中使用时修改这个变量的值。

语法。

hspeed。

返回。

实数（单精度浮点值

例子。

if keyboard_check(vk_left) hspeed = -5;
if keyboard_check(vk_right) hspeed = 5;

上面的代码将根据哪个键被按下而改变水平速度。
