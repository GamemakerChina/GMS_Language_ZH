固体

一个实例可以通过对象编辑器中的对象属性，或者通过改变这个内置变量的值，被标记为实体。如果solid被设置为true，那么。 当检测到碰撞时，碰撞的实例会自动返回到它在碰撞前的步骤中的位置（然后在碰撞事件中运行任何代码或动作）。如果它被设置为false，所有的定位 必须通过碰撞事件来处理。

语法。

固体。

返回。

布尔型

例子。

如果other.solid = true
{
x = xprevious;
y = yprevious;
}

上面的代码将检查碰撞中的 "另一个 "实例的实体标志，如果它被设置为真，则将该实例移回其先前的位置。
