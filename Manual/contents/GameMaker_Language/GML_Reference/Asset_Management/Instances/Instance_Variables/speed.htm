速度

GameMaker Studio 2中的所有实例都有某些 "内置 "属性，你可以使用和设置这些属性来管理它们的外观和行为。 不同于 hspeed和 vspeed不同的是，速度没有与之相关的方向，因为这是由实例的值决定的。 方向的值，但它 可以有一个负值，在这种情况下，实例将以与方向函数设置相反的方向移动（即：方向-180°）。请注意，设置 speed 和/或 direction 。 也会修改 hspeed 和 vspeed 变量的值，并且 gravity, gravity_direction 和 friction当它们在你的游戏中使用时，都可以修改这个变量的值。

语法。

速度。

返回。

实数（单精度浮点值

例子。

if keyboard_check(vk_up) speed = 2;
if keyboard_check(vk_left) direction += 5;
if keyboard_check(vk_right) direction -= 5;

上面的代码将使用方向键来设置实例的速度和方向。
