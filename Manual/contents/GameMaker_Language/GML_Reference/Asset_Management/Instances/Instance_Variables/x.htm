x

一个实例的x值是当前房间的水平位置，以像素为单位。这个值可以是0，也可以是正值或负值，其中0是房间的左边，向右移动增加x，向左移动减少x（一个负的 负值意味着实例已经离开了房间的左侧）。)你甚至可以给x一个真实的值，比如12.345，但是由于你不能得到0.345的像素，实例可能看起来没有移动，尽管这些值会被保持下来 并由GameMaker Studio 2内部使用，而且完全有效。

通过设置x（和y）值，你可以让物体在房间里跳到你选择的位置，或者你可以对它们进行较小量的加减，以产生运动的错觉，而不需要实际使用内置的 速度 和 方向功能。在制作游戏时应考虑到这一点，因为通常一种类型的运动比另一种类型的运动更适合某种特定的游戏风格。

语法。

x;

返回。

实数（单精度浮点值

例子。

if keyboard_check(vk_left) x += 5;
if keyboard_check(vk_right) x -= 5;

上面的代码将使用方向键来增加或减少X的位置，因此创造了运动的错觉。
