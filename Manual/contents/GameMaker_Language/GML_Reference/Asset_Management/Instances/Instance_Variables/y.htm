y

实例的y值是当前房间的垂直位置，单位是像素。这个值可以是0，也可以是正数或负数，其中0是房间的顶部，向下移动会增加Y，向上移动会减少Y（Y的负值意味着 负值意味着实例已经离开了房间的顶部）。)你甚至可以给y一个真实的值，比如12.345，但由于你不能得到0.345的像素，实例可能看起来没有移动，尽管这些值将被维护并被内部使用 GameMaker Studio 2维护和使用，并且完全有效。

通过设置y（和x）值，你可以让物体在房间里跳到你选择的位置，或者你可以对它们进行较小量的加减，以产生运动的错觉，而不需要实际使用内置的 速度 和 方向功能。在制作游戏时应考虑到这一点，因为通常一种类型的运动比另一种类型的运动更适合某种特定的游戏风格。

语法。

y;

返回。

实数（单精度浮点值

例子。

if keyboard_check(vk_down) y += 5;
if keyboard_check(vk_up) y -= 5;

上面的代码将使用方向键来增加或减少Y的位置，因此创造了运动的错觉。
