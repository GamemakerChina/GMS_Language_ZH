实例_count

通过这个只读变量，你可以得到房间里所有活动实例的数量。这将包括运行代码的实例，但不包括那些已经使用实例停用函数停用的实例。注意，这个变量只会给你在步骤开始时的实例数量，所以在步骤开始后对房间里的实例的任何改变都不会被考虑在内。

语法。

instance_count。

返回。

真实的

例子。

如果（instance_count < 100）
{
var dif = 100 - instance_count;
while (--dif > 0)
{
instance_create_layer（随机（room_width）, 随机（room_height）, "Effects", obj_Star）;
}
}

上述代码将创建对象 "obj_Star "的多个实例，直到实例总数达到100。
