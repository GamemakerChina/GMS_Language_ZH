实例_创建层

通过这个函数，你可以在房间的任何给定点和指定的层上创建一个指定对象的新实例。该层可以通过层的ID值来识别（由函数返回 layer_create()) 或者通过房间编辑器中定义的层的名称（字符串，例如 "instance_layer"）来识别。

这个函数返回 id，然后可以将其存储在一个变量中或用于访问该实例。请注意，在继续执行调用该函数的事件的代码或动作之前，该函数也将调用被创建的实例的创建事件。

重要的是!有一个最小和最大的层深度为-16000到16000。任何放置在该范围之外的图层都不会被画出来，尽管所有的事件仍将正常运行。

语法。

instance_create_layer(x, y, layer_id, obj)。

争论 描述
x 该对象将被创建的X位置
y 该对象将被创建在的y位置
layer_id 将创建的实例分配给的层ID（或名称）。
obj 要创建一个实例的对象的索引

返回。

真实（实例ID值

例子。

var inst = instance_create_layer(x, y, "instance_layer", obj_Bullet);
with (inst)
{
speed = other.shoot_speed;
direction = other.image_angle;
}

上面的代码创建了一个新的对象 "obj_Bullet "的实例，并将实例ID存储在一个变量中。然后这个变量被用来给新的实例分配速度和方向。
