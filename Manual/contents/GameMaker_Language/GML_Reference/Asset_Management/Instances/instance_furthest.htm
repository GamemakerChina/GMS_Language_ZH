实例_最远

这个函数将检查给定对象的所有实例，看哪个实例离给定的x/y点最远。所有检查将从给定的x/y位置到指定对象实例的原点（x/y位置）。如果没有该对象的实例 的实例，该函数将返回关键字noone，但如果有实例，它将返回找到的 id的实例。 请注意，如果运行代码的实例与被检查的对象具有相同的对象索引，那么它将被包括在检查中（这包括对父对象的检查，如果调用的实例也是父对象的一个孩子）。

语法。

instance_furthest(x, y, obj)。

争论 描述
x 检查远处的实例的X位置。
y 检查远离的实例的y位置。
obj 要检查的对象的实例。

返回。

真实（要么是一个实例ID值，要么是关键词noone）。

例子。

var inst;
inst = instance_furthest(x, y, object_index);
if inst != id
{
draw_line(x, y, inst.x, inst.y);
}

上述代码将找到与运行代码的实例共享相同对象索引的最远的实例，并将其id存储在一个变量中。然后检查这个变量是否与调用实例的id相同，如果不是，就在两个实例之间画一条线。 如果不是，就在两个实例之间画一条线。
