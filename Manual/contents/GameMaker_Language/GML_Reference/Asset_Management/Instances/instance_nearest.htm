实例_最近的

这个函数将检查给定对象的所有实例，看哪个实例离给定的x/y点最近。所有的检查将从给定的x/y位置到指定对象的实例的原点（x/y位置）。如果没有对象的实例 的实例，该函数将返回关键字 noone，但如果存在实例，那么它将返回找到的实例的 id 的实例。请注意，如果运行代码的实例与被检查的对象具有相同的对象索引，那么它将被包括在检查中（这包括对父对象的检查，如果调用的实例也是父对象的一个孩子 的孩子）。)

语法。

instance_nearest(x, y, obj)。

争论 描述
x 要检查的X位置。
y 要检查的Y位置。
obj 要检查的对象的实例。

返回。

真实（要么是一个实例ID值，要么是关键词noone）。

例子。

var inst, xx;
xx = x;
x -= 10000;
inst = instance_nearest(xx, y, object_index);
if inst != id
{
draw_line(x, y, inst.x, inst.y);
}
x += 10000;

上面的代码将当前实例移动10000像素，然后检查它之前的位置，找到最近的相同对象类型的实例。如果这个实例是它自己，它就会做任何事情，只是移动到它原来的位置，但如果发现的 实例是不同的，它将在两者之间画一条线。
