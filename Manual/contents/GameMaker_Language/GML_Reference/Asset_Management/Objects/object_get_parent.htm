对象_获取母体

这个函数将得到任何已经分配给指定对象的父类的object_index，否则返回-100，表明该对象没有分配给它的父类，如果被检查的对象不存在，则返回-1。对于更多的信息 关于父类的更多信息，请参见对象编辑器部分。

语法。

object_get_parent(obj)。

争论 描述
obj 要检查的对象的索引

返回。

真实的

例子。

par = object_get_parent(object_index);

上面的代码将获得运行代码的实例的对象索引的父级，并将返回值存储在变量 "par "中。
