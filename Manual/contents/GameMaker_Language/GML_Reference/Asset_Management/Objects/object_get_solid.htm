object_get_solid

这个函数将告诉你，你正在检查的对象是否已经被标记为 "实体"。当使用传统的碰撞系统（即：物理世界是关闭的）时，一个实体物体会产生一个特殊的碰撞事件。请注意 这不是一个实例函数!所以，你可以有一个实体对象和同一个对象的一个普通实例，反之亦然。你可以使用 "实例 "来设置单个实例的实体标志。 固体实例 变量来设置单个实例的实体标志。

语法。

object_get_solid(obj)。

争论 描述
obj 要检查的对象的索引。

返回。

布尔型

例子。

if (!solid) && (object_get_solid(object_index))
{
solid = true;
}

上面的代码将检查运行中的实例，看它是否是实体，以及检查实例的对象索引，看它是否被标记为实体。如果该实例不是实体，但其对象索引被标记为实体，则 将为该实例设置 "solid "为true。

返回。物体
