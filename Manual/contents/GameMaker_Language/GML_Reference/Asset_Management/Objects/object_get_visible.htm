对象_获取可见性

这个函数将告诉你，你正在检查的对象是否被标记为 "可见"（运行其绘制事件）或不（不运行其绘制事件）。请注意，这不是一个实例函数!所以，你可以有一个可见的对象 和同一对象的不可见实例，反之亦然。你可以使用 "实例 "变量来设置单个实例的可见性。 可见实例变量来设置单个实例的可见性。

语法。

object_get_visible(obj)。

争论 描述
obj 要检查的对象的索引。

返回。

布尔型

例子。

if (!visible) && (object_get_visible(object_index))
{
visible = true;
}

上面的代码将检查运行中的实例，看它是否可见，以及检查实例的对象索引，看它是否被标记为可见。如果实例还不可见，那么对象索引就被标记为on。 它将为该实例设置 "可见 "为真。

返回。物体
