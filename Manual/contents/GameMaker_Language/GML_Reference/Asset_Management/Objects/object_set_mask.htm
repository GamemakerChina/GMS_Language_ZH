对象_set_mask

通过这个函数，你可以设置一个特定对象的mask_index（关于掩码的更多信息，请看对象编辑器）。这意味着这个对象的所有实例在面具被改变后被创建时都将使用这个新的面具，而已经在房间里的实例可能不受影响。你可以把这个设置为-1来移除遮罩的精灵，因此默认为该对象的精灵定义的遮罩（如果没有选择精灵，则没有遮罩 遮罩）。)

请注意，这不是一个实例函数!你可以使用变量来设置各个实例的掩码索引，这样就可以有十个实例都有不同的掩码。 mask_index变量来设置单个实例的掩码索引，因此有十个实例的掩码都与它们创建的对象不同。 即使你用这个函数改变了对象的掩码索引，目前在房间里的所有实例将保持原样，只有在调用这个函数后创建的实例才会从 以新的掩码开始。

语法。

object_set_mask(index, spr)。

争论 描述
指数 要改变的对象的索引。
洒水 用来作为对象的遮罩的新精灵。

返回。

不适用

例子。

如果（global.level == 10）
{
object_set_mask(obj_Platform, spr_mask_10);
}

上面的代码检查了全局变量的值，然后改变对象 "obj_Platform "的掩码索引，如果它等于10。
