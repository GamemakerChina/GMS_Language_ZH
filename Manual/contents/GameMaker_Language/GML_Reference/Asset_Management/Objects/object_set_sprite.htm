对象_set_sprite

通过这个函数，你可以设置一个特定对象的精灵索引。这意味着该对象的所有实例在精灵指数被改变后，将以新的精灵指数创建，而已经在房间里的实例可能不受影响。 的实例可能不会受到影响。

请注意，这不是一个实例函数!你可以使用变量来设置单个实例的精灵索引，这样就有了十个实例，它们都有自己的精灵。 魍魉指数变量来设置单个实例的精灵索引，因此有十个实例的精灵都与它们所创建的对象不同。 即使你用这个函数改变了对象的精灵索引，所有当前在房间里的实例都将保持原样，只有在调用这个函数后创建的实例才会以新的精灵开始。 后创建的实例才会以新的精灵开始。

语法。

object_set_sprite ( index, spr ) 。

争论 描述
指数 要改变的对象的索引。
洒水 要分配给对象的精灵。

返回。

不适用

例子。

switch (room)
{
case rm_start: object_set_sprite(obj_Player, spr_uniform); break;
case rm_middle: object_set_sprite(obj_Player, spr_swimsuit); break;
case rm_end: object_set_sprite(obj_Player, spr_casual) ; break;
}
instance_create_layer(32, 32, "Instances", obj_Player);

上面的代码将根据运行代码的实例当前所处的房间，将对象 "obj_Player "的精灵索引设置为不同的值。然后它创建一个 "obj_Player "的实例。

返回。物体
