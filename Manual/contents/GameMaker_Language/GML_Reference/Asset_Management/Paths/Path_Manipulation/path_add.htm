路径_添加

通过这个函数，你可以在GameMaker Studio 2中创建一个路径，而不需要使用路径编辑器。这个函数将返回路径的索引，该索引应该存储在一个变量中，并从那时起作为该路径的参考。请注意 创建的路径是空的，即：它没有定义任何点，所以你必须使用其他可用的函数来为路径添加点，或者使用MP网格来 生成路径。一旦你完成了对路径的使用，或者希望创建一个新的路径并将其索引存储在同一变量中，你应该先用以下命令删除旧的路径 path_delete以防止内存泄漏，因为内存泄漏 最终会使你的游戏崩溃。

语法。

path_add()。

返回。

真实的

例子。

global.newpath = path_add()。

这将创建一个新的路径并将其索引分配给global.newpath。
