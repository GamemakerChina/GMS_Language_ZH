路径_附加

通过这段代码，你可以将一个路径追加到另一个路径上，有效地将它们连接在一起，要做到这一点，两个路径必须事先存在。一般来说，你想把这个函数用在用 path_add() 。 因为如果你在路径资产上使用这个函数，那么从使用这个函数的那一刻起到游戏结束，它将永久地影响游戏中的实例的路径。

注意：这不会从游戏中删除被附加的路径。它仍然存在，只是它现在没有点，如果你不想再使用它，你应该用函数删除它。 path_delete().

语法。

path_append(index, path)。

争论 描述
指数 路径的新索引。
路 将被追加到'index'的路径的索引。

返回。

不适用

例子。

path_append(mypath, mppath);
path_delete(mppath)。

上述代码将变量 "mppath "中索引的路径数据追加到变量 "mypath "中索引的路径，然后从内存中删除 "mppath"。
