路径_指定

通过这个函数，你可以将路径数据从一个路径复制到另一个路径。被复制的路径将首先被清除（如果它有任何路径点的话），并被被复制的路径完全覆盖。 被复制的路径完全覆盖。在这个过程中，两个路径都不会被删除，结果是两个路径，有两个不同的索引，但形式和点都完全一样。一般来说，你想在用 path_add() 创建的路径上使用这个函数。 因为如果你在路径资产上使用这个函数，从使用这个函数的那一刻起到游戏结束，它将永久影响游戏中所有实例的路径。

语法。

path_assign(index, path)。

争论 描述
指数 要被覆盖的路径的索引。
路 将覆盖'index'的路径的索引。

返回。

不适用

例子。

mypath = path_add();
path_assign(mypath, choose(path_1, path_2, path_3)) 。

上面的代码将创建一个新的、在变量 "mypath "中索引的空路径，然后从三个可用的路径资源中复制过来的路径数据。
