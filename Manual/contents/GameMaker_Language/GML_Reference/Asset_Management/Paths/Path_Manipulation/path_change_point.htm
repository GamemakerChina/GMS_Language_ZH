路径_改变点

通过这个函数，你可以改变之前为一个路径定义的任何点的位置和/或速度系数（该路径可以在路径编辑器中创建，也可以通过代码使用 path_add()).如果在一个 路径，请注意该函数将改变实际的资产，因此将永久地影响游戏中所有实例对该路径的使用，从使用该函数的那一刻起直到游戏结束。如果 这不是你所需要的，那么你应该使用一个类似于 path_duplicate() 的函数，首先创建一个路径的副本，然后在复制的资产上调用这个函数（不要忘记在资产上调用 当资产不再需要时，调用 path_delete() ）。)

语法。

path_change_point(index, n, x, y, speed)。

争论 描述
指数 要改变一个点的路径的索引。
n 改变属性的定义点。
x 点的新X坐标（相对于路径）。
y 该点的新Y坐标（相对于路径）。
速度 点的新速度系数。

返回。

不适用

例子。

for (var i = 0; i < path_get_number(path0); ++i;)
{
var px = path_get_point_x(pth_Patrol, i) + 64 - random(128);
var py = path_get_point_y(pth_Patrol, i) + 64 - random(128);
path_change_point(pth_Patrol, i, px, py, 100);
}

上面的代码循环浏览索引为 "path0 "的路径中的所有点，并将所有点重新定位到128x128像素区域内的一个随机位置。
