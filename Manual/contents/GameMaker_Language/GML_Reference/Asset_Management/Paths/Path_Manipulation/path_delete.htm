路径_删除

你可以用这段代码从内存中删除一个路径。如果这个路径是用动态方式创建的 path_add()创建的，保存路径索引的变量将不再适用于访问该路径，因为它已经不存在了。 如果这个路径是用路径编辑器创建的，那么在整个游戏中就不能再访问这个路径了，因为你是在永久性地删除它。

语法。

path_delete(index)。

争论 描述
指数 要删除的路径的索引。

返回。

不适用

例子。

var t_path = path_add();
if mp_grid_path(grid, t_path, x, y, obj_Player.x, obj_Player.y, 1)
{
path_assign(mypath, t_path);
}
path_delete(t_path)。

上述代码将创建一个路径并将其索引存储在一个局部变量中。然后这个路径被用来存储一个 mp_grid_path()生成的路径，如果它能成功 如果它成功地找到了目标，就会被分配到 "mypath "中索引的路径。最后 "t_path "被删除。
