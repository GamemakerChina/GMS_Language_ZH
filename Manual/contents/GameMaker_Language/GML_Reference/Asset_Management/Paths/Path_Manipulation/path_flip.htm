路径_翻转

这个函数将所有的路径点沿横轴进行翻转。这个函数改变了实际的路径资产，因此将永久地影响游戏中所有实例对路径的使用，从使用该函数的那一刻起直到游戏结束。 如果这不是你所需要的，那么你应该使用一个类似于 path_duplicate() 的函数先创建一个路径的副本，然后在这个副本上调用这个函数 然后在复制的资产上调用这个函数（不要忘记在资产不再需要时调用 path_delete() ）。

语法。

path_flip(index)。

争论 描述
指数 要翻转的路径的索引。

返回。

不适用

例子。

path_flip(mypath)。

这将使变量 "mypath "中索引的路径沿水平轴翻转。
