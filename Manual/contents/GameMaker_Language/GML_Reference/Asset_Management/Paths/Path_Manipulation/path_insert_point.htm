路径_插入点

通过这个函数，你可以在一个路径中插入一个新的点（这个路径可以在路径编辑器中创建，也可以通过代码使用 path_add()).该点将被添加到路径中的 "n "点之前。 之前。

语法。

path_insert_point(index, n, x, y, speed)。

争论 描述
指数 要插入点的路径的索引。
n 插入新点的定义点，即BEFORE。
x 新点的X坐标（相对于路径）。
y 新点的Y坐标（相对于路径）。
速度 点的速度系数。

返回。

不适用

例子。

path_insert_point(mypath, 0, 50, 50, 100)。

这将在变量 "mypath "中索引的路径的最开始插入一个点，位于(50,50)，速度系数为100%。
