路径_封闭

这个函数可以用来标记一个给定的路径是开放（假）还是封闭（真）。一个封闭的路径的起点与终点相连，形成一个循环，而一个开放的路径有一个明确的、不相连的起点和终点。这个函数改变了实际的路径资产。 如果这不是你所需要的，那么你应该使用一个类似 path_duplicate() 的函数，先创建一个路径的副本，然后在复制的资产上调用这个函数（不要忘记在不再需要的时候调用 path_delete() ）。

语法。

path_set_closed(index, closed)。

争论 描述
指数 要改变的路径的索引。
关闭的 路径是否关闭（真）或不关闭（假）。

返回。

不适用

例子。

path_set_closed(pth_Patrol, true)。

这将把给定的路径设置为封闭路径。
