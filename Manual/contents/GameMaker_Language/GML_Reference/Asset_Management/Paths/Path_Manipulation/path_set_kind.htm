路径_设置类型

这个函数可以用来设置你希望指定的资源是哪种路径。这可以是一条直线路径（设置为0），也可以是一条平滑的路径（设置为1），在这种情况下，路径的精度也必须被考虑在内（精度 也可以用 path_set_precision()).这个函数改变了实际的路径资源，因此将永久地影响游戏中所有实例对路径的使用，从使用该函数的那一刻起直到游戏结束。 如果这不是你所需要的，那么你应该使用一个类似 path_duplicate() 的函数先创建一个路径的副本，然后在复制的资产上调用这个函数 然后在复制的资产上调用这个函数（别忘了在不再需要的时候在资产上调用 path_delete() ）。

语法。

path_set_kind(index, val)。

争论 描述
指数 要改变的路径的索引。
价值 路径的种类，0代表直线，1代表平滑。

返回。

不适用

例子。

path_set_kind(pth_Patrol, true)。

这将把给定的路径设定为 "平滑 "路径。
