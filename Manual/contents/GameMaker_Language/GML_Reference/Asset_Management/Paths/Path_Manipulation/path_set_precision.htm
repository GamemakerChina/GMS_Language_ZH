路径_设置精度

通过这个函数，你可以改变路径的 "平滑度"。这个值必须在1到8之间，低的值会产生更直的边缘，点与点之间的曲线更尖锐，而高的值会使点变圆，使路径更 更加 "曲折"。注意，如果在路径编辑器中没有将路径设置为平滑，或者使用函数 path_set_kind().这个函数改变了实际的路径资产。 因此，从使用该函数的那一刻起到游戏结束，它将永久地影响游戏中所有实例对路径的使用方式。如果这不是你所需要的，那么你应该使用一个类似 path_duplicate() 的函数，先创建一个路径的副本，然后在复制的资产上调用这个函数（不要忘记在不再需要的时候在资产上调用 path_delete() ）。

语法。

path_set_precision(index, prec)。

争论 描述
指数 要改变的路径的索引。
沉淀 路径的精度。必须是1到8之间的整数。

返回。

不适用

例子。

path_set_precision(pth_Patrol, 2)。

这将把给定路径的精度设置为2。
