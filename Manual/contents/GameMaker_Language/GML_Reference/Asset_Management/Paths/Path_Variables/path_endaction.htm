路径_行动

这个变量可以用来获取或改变一个实例到达当前路径终点时的反应。通常情况下，你会在启动路径时设置这个变量，使用 path_start()但 你可能希望根据你游戏中的任何事件来改变这一行为。可用的值用以下常量表示。

恒定 描述
路径_行动_停止 拦截道路
路径_行动_重启 从起始位置继续，如果路径没有关闭，则跳到起始位置
路径_行动_继续 从当前位置重新开始路径
逆转的路径_行动 逆转路径的速度（反向运行路径）。

语法。

path_endaction。

返回。

常数(见描述)

例子。

如果（path_endaction == path_action_stop）
{
path_endaction = path_action_reverse;
}

上面的代码将检查路径的结束动作，如果它被设置为停止，那么结束动作将被改为反向。

返回。路径变量
