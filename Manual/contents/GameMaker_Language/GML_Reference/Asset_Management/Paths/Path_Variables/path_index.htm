路径_索引

GameMaker Studio 2中的所有资源都有一个唯一的识别号码，path_index是一个只读变量，它保存了一个给定的 路径资源的索引值，该资源已通过 path_start() 函数分配给一个实例。如果该实例没有分配路径，该变量将被设置为-1。

语法。

path_index;

返回。

整数（路径ID）。

例子。

如果(path_index == -1)
{
path_start(pth_enemy3, 4, path_action_reverse, 0);
}

上面的代码检查是否有一个路径被分配给了实例，如果没有，就启动一个新的路径（同时将其分配给path_index）。

返回。路径变量
