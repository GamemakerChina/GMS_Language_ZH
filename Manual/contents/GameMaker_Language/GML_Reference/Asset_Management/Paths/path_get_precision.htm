路径_获取精度

这个函数返回给定路径被 "平滑 "的精度，它将是一个从1到8的整数。尽管你可以为一条直线路径获取（和设置）这个值，但它不会影响到一个实例如何使用 因为只有当路径类型被设置为 "平滑 "时，它才是相关的。

语法。

path_get_precision(index)。

争论 描述
指数 要检查的路径的索引。

返回。

真实的

例子。

if (path_get_kind(pth_Patrol))
{
if (path_get_precision(pth_Patrol)) != 8
{
path_set_precision(pth_Patrol, 8) ;
}
}

上面的代码检查 "pth_Patrol "中索引的路径，看它是什么类型的，如果它是平滑的，那么它就检查平滑的精确程度。如果它小于8，那么它就被设置为8的值。

返回。路径
