路径_获取速度

这个函数返回路径上任何给定位置的速度系数。这个位置应该是一个介于0和1之间的值（如果你需要当前位置，你可以使用 路径_位置，例如，如果你需要当前的 位置），0是起始位置，1是终点位置。返回的值将是一个系数，实际的 path_speed 被设置的系数。例如，在一条路径上 只有两个点，0和1，其中0点的速度系数是100，1点的速度系数是0，函数path_get_speed(path, 0.5)将返回50。

语法。

path_get_speed(ind, pos);

争论 描述
指数 要检查的路径的索引。
姿势 要检查多远的路径。在0（开始）和1（结束）之间，可以是小数。

返回。

真实的

例子。

speed = path_get_speed(pth_Patrol, 0.5) / 20;

这将把调用实例的速度设置为正好在给定路径一半的点的速度系数，除以20。因此，如果找到的速度因子是默认的100，它将把调用实例的速度设置为5。

返回。路径
