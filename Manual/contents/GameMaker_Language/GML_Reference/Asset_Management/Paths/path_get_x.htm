path_get_x

通过这个函数，你可以得到任何给定路径上某个位置的X坐标。0是路径的起点，1是路径的终点，而在这两者之间的任何位置都相当于路径的终点。这不需要是路径的定义点，它可以是路径上的任何地方。 可以是路径上的任何地方。

语法。

path_get_x(ind, pos)。

争论 描述
指数 要检查的路径的索引。
姿势 要检查多远的路径。在0（开始）和1（结束）之间。

返回。

真实的

例子。

x = path_get_x(pth_WallWalk, 0.5)。

这将把调用实例的x设置为正好在给定路径的一半处的点的x坐标。

返回。路径
