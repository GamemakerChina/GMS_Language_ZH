层_背景_获取_alpha

这个函数可以用来获取背景元素的alpha值。你给背景元素的ID（当你用以下方法创建一个背景元素时，你会得到 layer_background_create()或 函数时得到的），该函数将返回 layer_background_get_id())，该函数将返回一个在0(完全透明)和1(完全不透明)之间的值。

语法。

layer_background_get_alpha(background_element_id)。

争论 描述
背景_元素_id 获取信息的背景元素的唯一ID值。

返回。

实数（从0到1

例子。

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
if layer_background_get_alpha(back_id) < 0.1
{
layer_background_destroy(back_id);
}

上面的代码将得到名为 "Background_sky "的层的ID，然后用它来得到该层上的背景元素的ID。这个ID然后被用来检查元素的alpha，如果它小于0.1，那么该层元素 就会被销毁。
