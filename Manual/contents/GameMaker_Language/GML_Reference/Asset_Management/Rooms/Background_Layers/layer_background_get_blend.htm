图层_Background_get_blend

这个函数可以用来获取背景元素的混合颜色。你给背景元素的ID（当你用以下方法创建一个背景元素时，你会得到 layer_background_create()或 时得到的），该函数将返回实数。 layer_background_get_id())，该函数将返回代表指定颜色的真实值。

语法。

layer_background_get_blend(background_element_id)。

争论 描述
背景_元素_id 获取信息的背景元素的唯一ID值。

返回。

色值（实数

例子。

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
if layer_background_get_blend(back_id) == c_white
{
layer_background_blend(back_id, make_colour_rgb(random(255), random(255), 255));
}

上面的代码将获得名为 "Background_sky "的层的ID，然后用它来获得该层上的背景元素的ID。这个ID然后被用来检查元素的混合颜色，如果它相当于常数 c_white，那么该层的混合颜色就会被设置为随机颜色。
