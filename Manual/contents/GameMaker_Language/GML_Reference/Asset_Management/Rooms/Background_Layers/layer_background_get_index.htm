层_背景_获取_索引

这个函数可以用来获取背景元素的当前图像索引值。你给背景元素的ID（当你用 "创建背景元素 "时，你会得到这个ID）。 layer_background_create() 或当你使用函数 layer_background_get_id()时得到的），该函数将返回代表该精灵正在显示的图像索引的实值。该函数将返回-1，如果 该函数将返回-1，如果背景元素不存在，或者该元素没有分配给它的有效精灵。

语法。

layer_background_get_index(background_element_id)。

争论 描述
背景_元素_id 获取信息的背景元素的唯一ID值。

返回。

实数（当前精灵图像的索引或-1）。

例子。

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
if layer_background_get_index(back_id) < 4
{
layer_background_index(back_id, 4);
}

上面的代码将获得名为 "Background_sky "的图层的ID，然后用它来获得该图层上背景元素的ID。然后，这个ID被用来检查该元素的图像索引是否小于4，如果是，它就被设置为4。 则被设置为4。
