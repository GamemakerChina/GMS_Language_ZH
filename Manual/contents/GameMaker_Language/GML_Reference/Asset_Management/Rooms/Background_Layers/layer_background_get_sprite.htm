层_背景_获取_斯普雷特

这个函数可以用来获取背景元素的当前精灵索引值。你给背景元素的ID（你在使用 "创建背景元素 "时得到的ID）。 layer_background_create() 或使用函数 layer_background_get_id()时得到的），该函数将返回一个代表正在显示的精灵索引的实值。如果该元素没有指定精灵，该函数 将返回-1。

语法。

layer_background_get_sprite(background_element_id)。

争论 描述
背景_元素_id 获取信息的背景元素的唯一ID值。

返回。

Sprite指数（实数）或-1

例子。

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
if layer_background_get_sprite(back_id) != spr_Clouds
{
layer_background_sprite(back_id, spr_Clouds);
}

上面的代码将获得名为 "Background_sky "的图层的ID，然后用它来获得该图层上背景元素的ID。然后，这个ID被用来检查分配给该元素的精灵，如果不是的话，就将其设置为精灵 "spr_Clouds" 如果它还不是的话。
