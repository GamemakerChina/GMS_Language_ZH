层_背景_获取_拉伸

这个函数可以用来获取背景元素精灵的拉伸状态。你给背景元素的ID（当你用 "创建 "背景元素时，你会得到这个ID。 layer_background_create() 或使用函数 layer_background_get_id()时得到的），如果该元素的精灵目前被拉伸到适合该房间，该函数将返回true，如果不是，则返回false。 不是。

语法。

layer_background_get_stretch(background_element_id)。

争论 描述
背景_元素_id 获取信息的背景元素的唯一ID值。

返回。

布尔型

例子。

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
if layer_background_get_stretch(back_id)
{
layer_background_stretch(back_id, false);
}

上面的代码将获得名为 "Background_sky "的层的ID，然后用它来获得该层上背景元素的ID。这个ID然后被用来检查元素精灵是否会被拉伸以适应房间的大小。 如果它被拉伸了，那么这个属性就被设置为false。
