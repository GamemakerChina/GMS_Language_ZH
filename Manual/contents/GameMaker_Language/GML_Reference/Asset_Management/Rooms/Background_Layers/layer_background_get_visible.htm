层_背景_获取_可见性

这个函数可以用来获取背景元素的可见状态。你给背景元素的ID（当你用以下方法创建一个背景元素时，你会得到 layer_background_create()或 函数时得到的），该函数将返回 layer_background_get_id())，如果该元素当前是可见的，该函数将返回true，如果不是，则返回false。请注意，这个返回值 不受该元素所在的层是否可见的影响。

语法。

layer_background_get_visible(background_element_id)。

争论 描述
背景_元素_id 获取信息的背景元素的唯一ID值。

返回。

布尔型

例子。

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
if layer_background_get_visible(back_id)
{
layer_background_visible(back_id, false);
}

上面的代码将获得名为 "Background_sky "的层的ID，然后用它来获得该层上的背景元素的ID。这个ID然后被用来检查元素的可见性，如果它是可见的，那么这个属性 被设置为false。
