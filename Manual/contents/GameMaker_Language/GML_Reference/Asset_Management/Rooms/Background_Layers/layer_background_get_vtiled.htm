层_背景_获取Vtiled

这个函数可以用来查找背景元素是否是垂直平铺的。你给背景元素的ID（当你用 "创建 "一个背景元素时，你会得到这个ID。 layer_background_create() 或使用函数 layer_background_get_id())，如果该元素目前是平铺的，该函数将返回true，如果不是，则返回false。

语法。

layer_background_get_vtiled(background_element_id)。

争论 描述
背景_元素_id 获取信息的背景元素的唯一ID值。

返回。

布尔型

例子。

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
if !lay_background_get_vtiled(back_id)
{
layer_background_vtiled(back_id, true);
}

上面的代码将获得名为 "Background_sky "的层的ID，然后用它来获得该层上的背景元素的ID。然后，这个ID被用来检查该元素是否在垂直轴上平铺，如果不是，则被设置为平铺。 不是的话，它就被设置为横跨纵轴。
