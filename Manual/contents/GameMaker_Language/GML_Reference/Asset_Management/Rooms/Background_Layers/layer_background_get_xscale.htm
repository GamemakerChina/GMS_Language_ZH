图层_背景_获取_xscale

这个函数可以用来获取背景元素的当前比例乘数值。你给背景元素的ID（当你用 "创建 "背景元素时，你会得到这个ID。 layer_background_create() 或使用函数 layer_background_get_id()时得到的），该函数将返回代表用于绘制精灵的比例乘数的实际值。默认值是1。

语法。

layer_background_get_xscale(background_element_id)。

争论 描述
背景_元素_id 获取信息的背景元素的唯一ID值。

返回。

真实的

例子。

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
if layer_background_get_xscale(back_id) != -1
{
layer_background_xscale(back_id, -1);
}

上面的代码将获得名为 "Background_sky "的层的ID，然后用它来获得该层上的背景元素的ID。这个ID然后被用来检查元素的X比例，如果它不是-1，就被设置为 -1.
