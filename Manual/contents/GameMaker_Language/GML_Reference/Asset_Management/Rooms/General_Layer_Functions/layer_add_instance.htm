层_add_instance

这个函数可以用来把一个给定的实例从它当前所在的层移到另一个层。你可以提供图层的ID（当你使用 layer_create())或者层的名称（作为 一个字符串--这将对性能产生影响），以及要在层之间移动的实例的实例ID。例如，假设你的播放器在一个比另一个层的深度低的层上，你想让它出现在两个层之间的层后面。 之间的层后面。你可以用这个函数把播放器从前景层切换到背景层，使它出现在正在绘制的其他层后面。

注意：这个函数不能用来给一个图层添加一个新的实例。你必须事先创建实例并将其ID存储到一个变量中。

注意：这个功能只在当前房间的范围内有效，当使用功能layer_set_target_room() 更改范围时，不能使用。

语法。

layer_add_instance(layer_id, instance_id)

争论 描述
layer_id 目标实例层的唯一ID值（或作为字符串的层名）。
instance_id 要转移到目标层的实例的唯一实例ID值

返回。

不适用

例子。

var near = instance_nearest(x, y, obj_Tree);
var layer_id = layer_get_id("Instances Front");
layer_add_instance(layer_id, near) 。

上面的代码首先会得到离给定X/Y位置最近的实例的索引，并将其存储在一个局部变量中。然后获取名为 "Instances Front "的层的唯一实例层ID，并将找到的实例移动到 该层上。

返回。一般层的功能
