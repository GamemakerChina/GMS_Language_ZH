层数_强制绘制_深度

这个函数强制所有图层以指定的Z深度绘制。这并不改变图层的渲染顺序，它们仍然会按深度顺序绘制，它只是改变了使用的Z值。一般来说，你不 需要担心这个问题，但是如果你的图层的深度超出了法定范围（-16000到16000），那么它们就不会被渲染，所以你可以强制Z深度为一个合理的值--比如说0--它们都会被正常渲染。 请注意，这通常只适用于以前的版本的传统项目。 GameMaker的传统项目中使用，在这些项目中，你的绘制深度可能高于或低于允许的层范围。

语法。

layer_force_draw_depth(force, depth)

争论 描述
力量 是否启用（设置为真）或禁用（设置为假）Z深度强制
深度 新的Z深度

返回。

不适用

例子。

if !layer_is_draw_depth_forced()
{
layer_force_draw_depth(true, 0);
}

上面的代码检查图层的Z深度是否被强制，如果不是，就将Z深度设置为0并启用。
