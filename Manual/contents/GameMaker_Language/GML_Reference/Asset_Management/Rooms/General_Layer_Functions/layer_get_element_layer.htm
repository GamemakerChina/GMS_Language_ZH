层_获取元素_层

你可以使用这个函数来获取给定元素所处的层ID。你提供唯一的元素ID值（例如，由创建元素的函数或从房间编辑器返回），该函数将返回该元素所在的唯一的 的唯一ID，该函数将返回该元素所在的层。如果给定的元素ID不是一个有效的ID，那么该函数将返回-1。

语法。

layer_get_element_layer(element_id)

争论 描述
元素_id 元素的唯一ID值，以获得其层数。

返回。

图层ID，如果该元素无效（即：不存在），则为-1。

例子。

element = layer_get_element_layer(asset_1);

上面的代码得到了元素的层ID，其ID存储在变量 "asset_1 "中，并将其存储在一个变量中。

返回。一般层的功能
