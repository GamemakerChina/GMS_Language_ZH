层_获取速度

你可以使用这个函数来检索当前范围内房间内的图层的垂直速度（以每游戏帧的像素为单位）。你提供图层的ID（当你使用 layer_create()) 或图层名称（作为一个字符串--这将对性能产生影响），该函数返回一个垂直速度的实数，其中正值是向下，负值是向上。默认为0（除非在房间编辑器中设置）。

语法。

layer_get_vspeed(layer_id)

争论 描述
layer_id 要获得垂直速度的层的唯一ID值

返回。

不适用

例子。

var lay_id = layer_get_id("Sprites");
if layer_get_hspeed(lay_id) != 0 || layer_get_vspeed(lay_id) != 0
{
layer_hspeed(lay_id, 0);
layer_vspeed(lay_id, 0);
}

上面的代码检查给定的层的水平和垂直速度，如果它们不是都设置为0，那么它就将它们设置为0。

返回。一般层的功能
