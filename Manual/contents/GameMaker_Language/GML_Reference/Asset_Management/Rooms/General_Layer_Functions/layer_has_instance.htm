层_有实例

这个函数可以用来检查一个给定的实例是否当前被分配到给定的层。你可以提供图层的ID（当你创建图层时，你可以用 layer_create())或者层的名字（作为一个字符串 - 这将会对性能产生影响）和要检查的实例的实例ID。你也可以给一个object_index（即：资产浏览器中的对象名称），该函数将检查该对象的任何实例是否存在于 给定的层上是否有该对象的实例。如果实例在该层上，该函数将返回真，如果不在，则返回假。

语法。

layer_has_instance(layer_id, instance_id)

争论 描述
layer_id 目标层的唯一ID值（或作为字符串的层名）。
instance_id 要检查的唯一实例ID或实例的对象索引

返回。

布尔型

例子。

if !layer_has_instance(global.Bullet_Layer, obj_Bullet_Parent)
{
layer_destroy(global.Bullet_Layer);
}

上面的代码将检查给定的图层是否包含任何对象 "obj_Bullet_Parent "的实例，如果没有，它将销毁该图层。

返回。一般层的功能
