层_hspeed

你可以使用这个函数来设置当前范围内房间内的图层的水平速度（以每游戏帧的像素为单位）。你提供图层的ID（你在创建图层时使用 layer_create()) 或图层名称（作为一个字符串--这将对性能产生影响）和要设置的速度值，其中正值是向右，负值是向左。

语法。

layer_hspeed(layer_id, hspd)

争论 描述
layer_id 设置水平速度的层的唯一ID值。
hspd 要设置的水平速度（以每游戏帧的像素为单位）。

返回。

不适用

例子。

var lay_id = layer_get_id("Sprites");
if layer_get_hspeed(lay_id) != 0 || layer_get_vspeed(lay_id) != 0
{
layer_hspeed(lay_id, 0);
layer_vspeed(lay_id, 0);
}

上面的代码检查给定的层的水平和垂直速度，如果它们不是都设置为0，那么它就将它们设置为0。
