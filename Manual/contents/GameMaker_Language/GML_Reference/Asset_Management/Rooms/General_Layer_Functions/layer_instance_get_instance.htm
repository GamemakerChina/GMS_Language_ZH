层_instance_get_instance

这个函数可以用来获取给定实例元素的唯一实例ID。你给出实例元素的ID（关于如何获得这个ID，请看下面的代码示例），该函数将返回一个实值，代表该元素的唯一的 元素的唯一实例ID。如果该元素不是一个实例，该函数将返回-1。

语法。

layer_instance_get_instance(element_id)

争论 描述
元素_id 实例元素的唯一ID值，以获得其ID。

返回。

真实（一个实例的唯一ID值

例子。

elements = layer_get_all_elements("Instances");
for (var i = 0; i < array_length(elements); i++)
{
if (layer_get_element_type(elements[i]) == layerelementtype_instance)
{
var layerelement = elements[i];
var inst = layer_instance_get_instance(layerelement);
inst.x = inst.x + 10;
}
}

上面的代码将检查得到一个图层上的所有实例元素，然后得到它们唯一的ID值，并使用它将它们向右移动10px。

返回。一般层的功能
