层_获取目标房间

该函数用于将图层目标重置为当前房间。参见函数 layer_set_target_room()获取更多信息。

语法。

layer_reset_target_room()

返回。

不适用

例子。

layer_set_target_room(rm_Game);
var l = layer_get_id("SpriteAssets");
repeat(50)
{
layer_sprite_create(l, random(room_get_width(rm_Game)), random(room_get_height(rm_Game)), spr_Trees) ;
}
layer_reset_target_room()。

上面的代码将目标房间设置为 "rm_Game "房间，然后获得该房间中名为 "SpriteAssets "的层的ID。这个图层ID然后被用来向该图层添加50个随机的精灵资产，然后图层目标 被重置为当前房间。

返回。一般层的功能
