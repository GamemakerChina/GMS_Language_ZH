层_set_visible

通过这个函数你可以切换一个图层的可见性。你可以提供图层的ID（你在创建图层时使用 layer_create())或图层名称（作为一个字符串--这将对性能产生影响 影响）以及该层的切换值，可见是true ，不可见是false 。

当一个图层是不可见的，分配给该图层的任何东西都不会被绘制，如果有任何实例被分配给该图层，那么它们甚至不会运行它们的绘制事件（与你将实例变量visible设置为false的情况相同）。 请注意，任何已经被标记为不可见的实例，如果它们所在的图层被设置为可见，将不会被标记为可见，因为这是两个独立的设置，尽管它们的行为是一样的。

语法。

layer_set_visible(layer_id, visible)。

争论 描述
layer_id 目标层的唯一ID值（或作为字符串的层名）。
可见 该层是否应该可见（true）或不可见（false）。

返回。

不适用

例子。

var lay_id = layer_get_id("Instances");
if layer_get_visible(lay_id)
{
layer_set_visible(lay_id, false);
}
else
{
layer_set_visible(lay_id, true);
}

上面的代码在房间编辑器中获得名为 "Instances "的图层的ID值，然后使用该ID检查该图层是否可见，根据返回值切换该图层的可见性。

返回。一般层的功能
