层_序列_角度

通过这个函数，你可以设置给定序列元素的旋转角度。你提供的序列元素ID是由 layer_sequence_create()或某个层元素函数返回的序列元素ID，以及要设置的新角度，序列将围绕其原点旋转到新的位置。GameMaker中的角度是以0º向右计算的，并且是逆时针方向的--所以90º是向上的，180º是向左的，270º是向下的--而序列的默认角度是 为一个序列的默认角度是0º。

注意，你不应该改变 image_xscale \ image_yscale \ image_angle \ x \ y 变量。 的变量，因为在调用该函数后，一旦序列更新，它们就会被覆盖。

语法。

layer_sequence_angle(sequence_element_id, angle)

争论 描述
sequence_element_id 目标序列元素的唯一ID值
角度 将序列元素旋转到的新角度

返回。

不适用

例子。

如果current_angle < 90
{
current_angle += 1;
layer_sequence_angle(my_seq, current_angle);
}

上面的代码检查了 current_angle 变量中的值，如果它小于90，那么它就把它加上去，然后用这个值来设置变量 my_seq 中引用的序列元素的角度。

返回。序列层数
