层_序列_创建

通过这个函数，你可以在给定的层上创建一个序列资产的实例。你提供的图层ID可以是在房间编辑器中定义的图层名称的字符串，也可以是唯一的图层ID，由函数返回。 layer_get_id(), 以及要创建序列的房间中的X和Y位置，最后是要创建的序列的ID。序列ID是你在资产浏览器中为序列定义的名称常数。该函数将返回序列元素的唯一ID 该函数将返回序列元素的唯一ID，该ID可用于序列的所有后续图层函数，或者可用于使用函数检索序列实例结构 layer_sequence_get_instance().

语法。

layer_sequence_create(layer_id, x, y, sequence_id)。

争论 描述
layer_id 目标实例层的唯一ID值
x 在房间里创建序列的X位置
y 在房间里创建序列的Y位置
sequence_id 要创建的序列ID，如资产浏览器中定义的那样

返回。

整数（元素ID）。

例子。

var _s = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
layer_sequence_pause(_s);

上面的代码在 "背景 "层上创建了一个新的序列，然后暂停了它。

返回。序列层数
