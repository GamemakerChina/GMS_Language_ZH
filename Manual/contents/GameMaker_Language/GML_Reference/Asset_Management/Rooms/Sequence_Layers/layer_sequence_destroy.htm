层_序列_销毁

通过这个函数，你可以销毁（从房间中移除）一个序列元素。你提供的序列元素ID是由 layer_sequence_create()或某个层元素函数返回的序列元素ID，该序列将被销毁。

语法。

layer_sequence_destroy(sequence_element_id)。

争论 描述
sequence_element_id 目标序列元素的唯一ID值

返回。

不适用

例子。

var a = layer_get_all_elements(layer);
for (var i = 0; i < array_length(a); i++; )
{
if layer_get_element_type(a[i]) == layerelementtype_sequence
{
layer_sequence_destroy(a[i])
}
}

上面的代码得到了分配给运行代码的实例层的所有元素的ID。然后，该代码检查是否有返回的元素是序列资产，如果是，则将其销毁。

返回。序列层数
