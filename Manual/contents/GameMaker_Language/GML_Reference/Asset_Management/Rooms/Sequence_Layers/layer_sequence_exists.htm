层_序列_存在

通过这个函数，你可以检查在给定的层上是否存在一个序列元素。你提供的图层ID可以是一个图层名称的字符串--如在房间编辑器中定义的那样--或者是唯一的图层ID--如函数所返回的那样 layer_get_id(), 以及序列元素的ID--如由 layer_sequence_create()或某个层元素函数返回的序列元素ID，并且它将返回 如果给定的元素存在，它将返回true，否则返回false。

语法。

layer_sequence_exists(layer_id, sequence_element_id)。

争论 描述
图层_ID 要检查的层的唯一ID或名称
sequence_element_id 目标序列元素的唯一ID值

返回。

布尔型

例子。

if !layer_sequence_exists("Background", my_seq)
{
my_seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
layer_sequence_pause(my_seq);
}

上面的代码检查给定的序列元素是否存在，如果不存在则在给定的层上创建一个新的序列，然后暂停它。

返回。序列层数
