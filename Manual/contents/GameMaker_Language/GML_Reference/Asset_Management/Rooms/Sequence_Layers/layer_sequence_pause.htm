层_序列_暂停

通过这个函数，你可以暂停给定序列的播放。你提供的序列元素ID是由 layer_sequence_create()返回的序列元素ID，该函数将暂停该序列，直到你再次使用函数开始播放 layer_sequence_play().

语法。

layer_sequence_pause(sequence_element_id)。

争论 描述
sequence_element_id 目标序列元素的唯一ID值

返回。

不适用

例子。

var _seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
if global.Pause
{
layer_sequence_pause(_seq);
}

上面的代码创建了一个新的序列，并将其ID存储在一个局部变量中。然后，它检查游戏是否暂停，如果是，它就暂停序列的播放。

返回。序列层数
