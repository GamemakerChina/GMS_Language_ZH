层数_亚细亚_混合

这个函数控制精灵sprite的混合（或 "着色"），默认值是-1（代表常数c_white，也可以使用）。任何其他值（包括内部颜色常数，如c_red, 或c_aqua）将把指定的颜色与原始精灵混合。你给精灵元素的ID（当你用 "C "创建一个精灵元素时，你会得到它。 layer_sprite_create()或在使用 函数时得到的），然后设置混合颜色。 layer_sprite_get_id())，然后设置要使用的混合颜色。下面你可以看到一个用不同颜色混合的精灵的例子。

请注意，在HTML5平台上，你应该尽量限制混合（除非使用WebGL）。 因为每一个混合的精灵都必须被单独缓存，所以有许多混合的精灵可能会对性能产生不利影响（你也可以使用函数设置缓存大小 sprite_set_cache_size()).

语法。

layer_sprite_blend(sprite_element_id, blend)。

争论 描述
萌芽_元素_ID 要改变的精灵元素的唯一ID值
融合 与精灵sprite混合的颜色（默认为c_white）。

返回。

不适用

例子。

var lay_id = layer_get_id("Asset_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
layer_sprite_blend( spr_id, c_gray);

上面的代码得到了分配给 "Asset_sky "层的名为 "Clouds "的精灵的ID值，然后将其染成一种颜色。

返回。精灵层
