层数的改变

使用这个函数，你可以改变分配给一个图层上的某个精灵元素的精灵资源。你给精灵元素的ID（你在用 "创建 "精灵元素时得到的ID），或者你在用 "创建 "精灵元素时得到的ID。 layer_sprite_create() 或当你使用函数 layer_sprite_get_id())，然后提供一个将被改变的精灵索引元素资产。注意，如果该精灵有子图像，那么它也会有动画（这可以用 使用 layer_sprite_index()和 layer_sprite_speed()函数来控制）。)请注意，你可以指定一个-1的值作为新的精灵索引，这样就不会有精灵被显示。 尽管这个元素仍然存在，而且以后还可以再修改。

语法。

layer_sprite_change(sprite_element_id, sprite_index)。

争论 描述
萌芽_元素_ID 要改变的精灵元素的唯一ID值
魉_指数 要使用的新精灵索引

返回。

不适用

例子。

var s = layer_sprite_get_id(layer, global.Asset_sprite);
if layer_sprite_get_sprite(s) != spr_Nighttime
{
layer_sprite_change(s, spr_nighttime);
}

上述代码用存储在全局变量global.Asset_sprite中的ID检查元素的sprite索引，如果它不是 "spr_Nighttime"，那么该sprite将被分配给该元素。
