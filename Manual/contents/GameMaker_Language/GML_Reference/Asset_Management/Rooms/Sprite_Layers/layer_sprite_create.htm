创建层数

通过这个函数，你可以将一个精灵资源分配给你的项目中使用的层。你要提供图层的ID（你在使用 "创建 "图层时得到的ID）。 layer_create()或当 使用图层名称和 layer_get_id())、房间内的一个位置和一个精灵索引（这将是资产浏览器中显示的精灵的名字），然后它将被添加到层中。 被添加到该层。该函数返回该元素的唯一ID值，然后可以在进一步的层函数中用于精灵。

语法。

layer_sprite_create(layer_id, x, y, sprite)。

争论 描述
layer_id 目标层的唯一ID值
x 要使用的X位置
y 要使用的y位置
精灵 要使用的精灵索引

返回。

雪碧元素ID（实数）。

例子。

global.asset_layer = layer_create(10000);
for (var i = 0; i< 10; i++;)
{
var _x = random(room_width);
var _y = room_height - 100;
global.asset_spr_trees[i] = layer_sprite_create(global.asset_layer, _x, _y, spr_Trees);
}

上面的代码创建了一个新的图层，然后向其添加了10个新的精灵元素，将每个元素的ID存储到一个数组中。

返回。精灵层
