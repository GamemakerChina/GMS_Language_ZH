层数_sprite_exists

你可以使用这个函数来检查并查看任何给定层上是否存在一个精灵元素。你可以提供图层的ID（当你使用 layer_create())或 层的名称（作为一个字符串--这将对性能产生影响）和精灵元素的ID（当你使用 layer_sprite_create()或当你使用函数 layer_sprite_get_id()) 如果该元素存在，该函数将返回true，如果不存在，则返回false。

注意：该函数在当前目标房间的范围内工作--默认为调用该函数的房间--可以使用函数设置 layer_set_target_room().

语法。

layer_sprite_exists(layer_id, sprite_element_id)。

争论 描述
layer_id 目标层的唯一ID值（或作为字符串的层名）。
萌芽_元素_ID 要检查的精灵元素的唯一ID值

返回。

布尔型

例子。

var lay_id = layer_get_id("Asset_Trees");
if layer_sprite_exists(lay_id, global.TreesSprites)
{
layer_sprite_destroy(lay_id, global.TreesSprites);
}

上述代码检查图层 "Asset_trees"，看给定的精灵元素是否存在，如果存在，那么它就会被销毁（但不是图层）。

返回。精灵层
