层_sprite_get_angle

这个函数可以用来获取精灵元素的当前角度。你可以给精灵元素的ID（当你用 "创建 "精灵元素时，你会得到这个ID。 layer_sprite_create()或在使用 函数时得到的），该函数将返回实数。 layer_sprite_get_id()时得到的），该函数将返回角度的实际值。这个值将在0到360之间，其中0代表右，90代表上，180代表左，270代表下。

语法。

layer_sprite_get_angle(sprite_element_id)。

争论 描述
萌芽_元素_ID 要获取信息的精灵元素的唯一ID值。

返回。

真实的

例子。

var lay_id = layer_get_id("sprite_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
if layer_sprite_get_angle( spr_id) != 0
{
layer_sprite_yscale(spr_id, 0);
}

上面的代码将获得名为 "sprite_sky "的图层的ID，然后用它来获得该图层上精灵元素的ID。这个ID然后被用来检查角度，如果它不是0，就被设置为0。

返回。精灵层
