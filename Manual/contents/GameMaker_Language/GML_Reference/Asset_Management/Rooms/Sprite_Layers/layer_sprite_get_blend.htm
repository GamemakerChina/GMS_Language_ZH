层级_栅栏_获取混合

这个函数可以用来获取精灵元素的混合颜色。你可以给精灵元素的ID（当你创建一个精灵元素时，你可以用 layer_sprite_create()或使用函数 layer_sprite_get_id())，该函数将返回代表指定颜色的实值。

语法。

layer_sprite_get_blend(sprite_element_id)。

争论 描述
萌芽_元素_ID 要获取信息的精灵元素的唯一ID值。

返回。

色值（实数

例子。

var lay_id = layer_get_id("sprite_sky");
var back_id = layer_sprite_get_id(lay_id, "Clouds");
if layer_sprite_get_blend(back_id) == c_white
{
layer_sprite_blend(back_id, make_colour_rgb(random(255), random(255), 255));
}

上面的代码将获得名为 "sprite_sky "的图层的ID，然后用它来获得该图层上精灵元素的ID。这个ID然后被用来检查元素的混合颜色，如果它等同于常数c_white。 则该元素的混合颜色将被设置为一个随机颜色。

返回。精灵层
