层数_栅栏_获取_ID

这个函数可以用来检索一个层上的精灵元素的唯一ID值。你要提供图层的ID（你在使用 "创建 "图层时得到的ID）。 layer_create()或当 使用图层名称和 layer_get_id()时得到的）和在房间编辑器中定义的精灵元素的名称。该函数将返回与该层的精灵元素相关的ID值。 元素相关的ID值。

注意，这个函数是专门为在IDE中添加在资产层上的精灵而设计的，如果你通过代码添加精灵到一个层上，使用函数 layer_sprite_create(), 那么该函数返回的ID应被用于所有未来的参考（因为该精灵元素将没有名字可以传入该函数）。

如果指定的层不存在，或者找不到指定的精灵元素，该函数将返回-1。

语法。

layer_sprite_get_id(layer_id, sprite_element_name)。

争论 描述
layer_id 目标层的唯一ID值
精灵元素名称 在房间编辑器中定义的该层上的精灵元素的唯一名称。

返回。

Sprite元素的ID值（Real

例子。

var lay_id = layer_get_id("Assets_trees");
var back_id = layer_sprite_get_id(lay_id, "graphic_254367CB");
layer_sprite_change(back_id, spr_Trees_Winter) 。

上面的代码将获得名为 "Assets_trees "的层的ID，然后用它来检索该层上的精灵元素 "graphic_254367CB "的ID。检索到的精灵元素ID然后被用来改变该元素的 精灵。

返回。精灵层
