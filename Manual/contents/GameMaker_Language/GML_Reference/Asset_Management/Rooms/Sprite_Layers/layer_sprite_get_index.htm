层数_栅栏_获取索引

这个函数可以用来获取精灵元素的当前图像索引值。你给精灵元素的ID（当你用 "创建 "精灵元素时，你会得到这个ID。 layer_sprite_create()或当你 使用该函数时得到的 layer_sprite_get_id()时得到的），该函数将返回代表该精灵正在显示的图像索引的实值。如果精灵元素不存在，或者该元素没有显示，该函数将返回-1。 或者该元素没有被分配到一个有效的精灵。

语法。

layer_sprite_get_index(sprite_element_id)。

争论 描述
萌芽_元素_ID 要获取信息的精灵元素的唯一ID值。

返回。

实数（当前精灵图像的索引或-1）。

例子。

var lay_id = layer_get_id("sprite_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
if layer_sprite_get_index( spr_id) < 4
{
layer_sprite_index(spr_id, 4);
}

上面的代码将获得名为 "sprite_sky "的图层的ID，然后用它来获得该图层上精灵元素的ID。然后，这个ID被用来检查该元素的图像索引是否小于4，如果是，它就被设置为 为4。

返回。精灵层
