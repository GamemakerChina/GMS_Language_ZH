层_栅栏_获取_x

这个函数可以用来获取精灵元素在房间里的X位置。你可以给精灵元素的ID（当你用 "创建 "精灵元素时，你会得到这个ID。 layer_sprite_create()或当你 使用该函数时得到的 layer_sprite_get_id()时得到的），该函数将返回x位置的值。

语法。

layer_sprite_get_x(sprite_element_id)。

争论 描述
萌芽_元素_ID 要获取信息的精灵元素的唯一ID值。

返回。

真实的

例子。

var lay_id = layer_get_id("sprite_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
if layer_sprite_get_x( spr_id) < 0
{
layer_sprite_x(spr_id, room_width);
}

上面的代码将获得名为 "sprite_sky "的图层的ID，然后用它来获得该图层上精灵元素的ID。这个ID然后被用来检查元素的X位置，如果它小于0，那么该层元素 被移到一个不同的X位置。

返回。精灵层
