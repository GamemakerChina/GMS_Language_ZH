层数_sprite_get_y

这个函数可以用来获得精灵元素在房间里的y位置。你给出精灵元素的ID（你在用 "创建 "精灵元素时得到的ID）。 layer_sprite_create()或当你 使用该函数时得到的），该函数将返回精灵元素的y位置。 layer_sprite_get_id()时得到的），该函数将返回y位置值。

语法。

layer_sprite_get_y(sprite_element_id)。

争论 描述
萌芽_元素_ID 要获取信息的精灵元素的唯一ID值。

返回。

真实的

例子。

var lay_id = layer_get_id("sprite_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
if layer_sprite_get_y( spr_id) < 0
{
layer_sprite_y(spr_id, room_height);
}

上面的代码将获得名为 "sprite_sky "的图层的ID，然后用它来获得该图层上精灵元素的ID。这个ID然后被用来检查元素的Y位置，如果它小于0，那么该层元素 被移到一个不同的Y位置。

返回。精灵层
