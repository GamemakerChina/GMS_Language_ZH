层数_栅栏_x刻度

使用这个函数，你可以改变一个图层上给定的精灵元素是否应该沿X轴缩放。你给精灵元素的ID（当你用 "X "创建一个精灵元素时，你会得到它。 layer_sprite_create() 或使用函数 layer_sprite_get_id()时得到的），然后设置缩放值。1表示不缩放（1:1），较小的值将会缩小（例如0.5，将是所使用的精灵的一半宽度 的一半），较大的值将向上缩放，而负值将翻转精灵并缩放，除非使用的值正好是-1（在这种情况下，使用的精灵只是围绕其原点位置从右向左翻转，没有缩放）。

语法。

layer_sprite_xscale(sprite_element_id, xscale)。

争论 描述
萌芽_元素_ID 要改变的精灵元素的唯一ID值
xscale X刻度值（默认为1）。

返回。

不适用

例子。

var asset_sprite = layer_sprite_get_id(layer, "gfc_Trees");
if layer_sprite_get_xscale(asset_sprite) != 1 || layer_sprite_get_yscale(asset_sprite) != 1
{
layer_sprite_xscale(asset_sprite, 1);
layer_sprite_yscale(asset_sprite, 1);
}

上述代码将检查分配给运行该代码的实例所在层的精灵元素，如果它在任何一个方向上被缩放，它将把X轴刻度和Y轴刻度都设置为1。

返回。精灵层
