层数_sprite_y

这个函数控制层上资产精灵元素的房间沿Y轴的位置。你给精灵元素的ID（当你用 "创建 "一个精灵元素时，你会得到这个ID。 layer_sprite_create() 或当你使用函数 layer_sprite_get_id()时得到的），然后设置要使用的Y值（基于房间坐标）。

语法。

layer_sprite_y(sprite_element_id, y)。

争论 描述
萌芽_元素_ID 要改变的精灵元素的唯一ID值
y 魍魉的y位置

返回。

不适用

例子。

var lay_id = layer_get_id("Asset_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
layer_sprite_y(spr_id, 50 + irandom(room_height - 200));

上面的代码得到了分配给 "Asset_sky "层的精灵资产 "Clouds "的ID值，然后将它的Y位置设置为50和房间高度减去200之间的随机值。

返回。精灵层
