瓦片地图层

GameMaker Studio 2 房间编辑器允许你使用图层将瓷砖添加到任何指定的房间。顾名思义，瓦片图层只是一个瓦片集瓦片的集合，这些瓦片都被认为在房间的同一深度。 通过堆叠图层，你可以让一些东西画在其他东西上面或下面。你也可以从代码中控制图层的某些方面，添加或删除一些东西，或者在运行时改变图层的某些属性或其包含的内容。 层的某些属性或它在运行时包含的内容。请注意，在房间编辑器中，你被限制在每层一个瓷砖图（the 瓷砖层)，但当在代码中处理它们时，你可以有多个瓷砖图。 代码中，你可以为一个图层分配多个瓷砖地图，每个地图都有自己的唯一ID和属性，所以。

  • 瓦片集是你用来创建瓦片层和瓦片地图的资源
  • 瓷砖层是你在房间编辑器中创建的，用来添加瓷砖组的。
  • 瓷砖地图就是你所说的瓷砖的集合，它被添加到一个图层中，无论是在房间编辑器中还是通过代码，都是该图层上的一个元素

除了上面列出的设置瓦片地图的函数外，你还有一些函数可以用来操作单个瓦片的数据。一个瓦片数据的 "blob "是由瓦片索引和一些额外的位组成的，以显示是否 瓷砖已经被旋转、翻转或镜像。你也可以向瓷砖数据添加你自己的位掩码，尽管这只适用于高级用户，因为你将掩码保留给瓷砖索引的位 - 基本上 有19个位保留给你的磁贴索引，但是如果你只使用其中的8个位，那么你还有11个位可以用来创建一个自定义掩码。然后，这个掩码可以在代码中使用和检查，以创建自定义的瓷砖碰撞或其他。 然而，你通常不需要创建你自己的瓦片数据块，而是使用函数 tilemap_get()来获取瓦片数据，然后用下面的函数来处理它，最后用 你会再次使用 tilemap_set().
