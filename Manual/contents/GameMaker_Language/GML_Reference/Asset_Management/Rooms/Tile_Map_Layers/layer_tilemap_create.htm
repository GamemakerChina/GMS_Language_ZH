图层_悬臂图_创建

通过这个函数，你可以给一个图层分配一个瓦片集资源，在你的项目中作为瓦片图使用。你提供图层的ID（你在使用 "创建 "图层时得到的ID）。 layer_create()) 或图层名称（作为一个字符串--这将对性能产生影响），然后提供一个初始（x, y）位置来将瓷砖贴图添加到房间里，提供要使用的瓷砖集资源，然后提供瓷砖贴图的宽度和高度（单元格）（例如：宽度为 即：宽度为20，高度为10将创建一个有200个单元的瓷砖地图，宽度为20个单元，高度为10个单元），单元本身的大小由选择的瓷砖集定义。值得注意的是，你不能在瓷砖地图中的负数位置放置瓷砖。 所以所有的瓷砖必须放在0到宽-1，0到高-1的单元格空间内。

语法。

layer_tilemap_create(layer_id, x, y, tile set, width, height)

争论 描述
layer_id 目标层的唯一ID值（或作为字符串的层名）。
x 瓦片地图在房间里的X位置
y 瓦片图在房间里的Y位置
瓷砖套装 要使用的瓦片组索引
宽度 宽度瓦片图（以单元格为单位）。
高度 瓦片图的高度（以单元格为单位）。

返回。

瓦片地图元素ID（Real

例子。

global.back_layer = layer_create(10000);
global.back_tiles = layer_tilemap_create(global.back_layer, 0, 0, tmap_Trees, 16, 32) 。

上面的代码创建了一个新的图层，然后给它添加了一个新的瓷砖贴图元素，将瓷砖贴图的位置设置为（0,0），以及要使用的瓷砖集和瓷砖贴图的宽度和高度。
