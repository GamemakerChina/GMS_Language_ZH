销毁图层

这个函数将销毁给定的瓦片贴图元素。你提供瓷砖贴图的ID（你在使用 "创建 "瓷砖贴图时得到的ID）。 layer_tilemap_create()或在使用图层ID和 layer_get_tilemap_id()) 时得到的），这将会删除它。注意，这不会删除图层，只是删除其中的瓷砖贴图，如果瓷砖贴图是在房间编辑器中添加的，那么下次你离开房间再回来时，瓷砖贴图将被重新创建 再次创建。然而，如果房间是持久的，瓷砖地图将被删除，除非房间的持久性被再次关闭。

语法。

layer_tilemap_destroy(tilemap_element_id)。

争论 描述
tilemap_element_id 要销毁的瓦片图的唯一ID值

返回。

不适用

例子。

var lay_id = layer_get_id("Tiles_trees");
var tile_id = layer_tilemap_get_id(lay_id);
if layer_tilemap_exists(lay_id, tile_id)
{
layer_tilemap_destroy(tile_id)。
}

上面的代码检查图层 "Tiles_trees"，看给定的瓷砖地图元素是否存在，如果存在，那么它就被销毁（但不包括图层）。
