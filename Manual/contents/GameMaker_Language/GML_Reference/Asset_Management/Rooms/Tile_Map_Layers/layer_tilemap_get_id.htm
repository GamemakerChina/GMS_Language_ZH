图层_tilemap_get_id

这个函数可以用来检索一个图层上的瓦片地图元素的唯一ID值。你可以提供图层的ID（当你使用 layer_create())或 图层名称（作为一个字符串--这将对性能产生影响），该函数将返回与该图层上的瓷砖地图元素相关的ID值。请注意，这个函数是专门为在IDE中添加的瓷砖贴图而设计的。 因为如果你通过代码使用该函数将瓷砖地图添加到一个图层上 layer_tilemap_create()，那么它将返回添加的瓦片地图元素的唯一ID。如果给定的瓦片图ID 不正确或者瓦片图不存在，该函数将返回-1。

语法。

layer_tilemap_get_id(layer_id)

争论 描述
layer_id 目标层的唯一ID值（或作为字符串的层名）。

返回。

瓦片图元素ID值（Real）或-1

例子。

var lay_id = layer_get_id("tilemap_trees");
var tile_id = layer_tilemap_get_id(lay_id);
layer_tilemap_destroy(tile_id) 。

上面的代码将获得名为 "tilemap_trees "的图层的ID，然后用它来获得该图层上的瓦片地图元素的ID。这个ID然后被用来从该层移除瓷砖地图。
