瓷砖图案_清空

使用这个函数，你可以清除/设置一个给定的瓦片地图上的所有瓦片。你给瓦片贴图元素ID（当你创建一个瓦片贴图元素时，你会得到它，使用 layer_tilemap_create()或当你使用 函数 layer_tilemap_get_id())，然后提供你希望清除该层的瓦片数据。默认值为0将清除图层中的所有瓦片（本质上是使所有瓦片 "空"）。 而你可以使用专用的 tile_* 函数来创建你自己的瓦片数据来清除瓦片图。

语法。

tilemap_clear(tilemap_element_id, tiledata)。

争论 描述
tilemap_element_id 要改变的瓦片地图元素的唯一ID值
平铺数据 用来清除图层的瓦片数据

返回。

不适用

例子。

var layer_id = layer_get_id("Forest");
var tile_id = layer_tilemap_get_id(layer_id);
tilemap_clear(tile_id, 0) 。

上面的代码得到了在房间编辑器中创建的瓷砖地图的ID值，然后用 "空 "瓷砖来清除它。
