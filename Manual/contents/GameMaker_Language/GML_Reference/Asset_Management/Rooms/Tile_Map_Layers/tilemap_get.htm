瓷砖地图_获取

使用这个函数，你可以从瓷砖贴图元素的一个单元中检索瓷砖数据。你给瓦片贴图元素的ID（你在用以下方法创建一个瓦片贴图元素时得到的 layer_tilemap_create()或 时得到的），以及单元格的x layer_tilemap_get_id())，以及单元格的x和y位置，该函数将返回瓦片数据 "blob"。这个数据本质上是一个位掩码，包含了 瓦片索引、翻转/旋转/镜像布尔运算和任何已经应用的掩码数据（详见 tilemap_set_mask()详见），然后产生的数据值可以在 tile_* 函数中使用。 来改变一个瓷砖的属性。注意，我们使用的是基于瓦片地图网格大小的x/y单元位置，而不是房间里的位置，所以 "cell_x "是一个从0到瓦片地图宽度的值。 和 "cell_y "是一个从0到瓷砖贴图高度的值。如果你需要在一个特定的房间位置获得一个瓷砖的数据，那么你需要使用函数 tilemap_get_at_pixel().

重要的是!如果瓷砖地图中的瓷砖没有变化（即：它们没有旋转或翻转等......），那么瓷砖集数据的返回值将完全等于瓷砖集上瓷砖的索引。因此，你可以用一块瓷砖创建 "碰撞 瓷砖的 "碰撞图"，使用瓷砖集中索引为1的瓷砖--例如--然后使用这个函数来检查1或0（空瓷砖）来计算碰撞。

语法。

tilemap_get(tilemap_element_id, cell_x, cell_y)。

争论 描述
tilemap_element_id 获取瓦片数据的瓦片地图元素的唯一ID值。
x_cell 沿着X轴的单元格，以获得瓦片的数据
y_cell 沿着Y轴的单元格，以获得瓦片的数据

返回。

实数（如果有错误则为-1

例子。

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var data = tilemap_get(map_id, 0, 0);
data = tile_set_flip(data, true);
tilemap_set(map_id, data, 0, 0);

上面的代码得到了 "Tiles_Sky "层上的瓷砖地图的ID，然后用它来获取单元格（0，0）处的瓷砖数据。这个瓦片数据在被用于再次设置瓦片地图上的瓦片之前被翻转。
