瓷砖地图_获取单元格_y_at_pixel

使用这个函数，你可以通过给出房间内的相对Y轴位置来检索单个瓦片地图单元的Y轴位置。你可以给瓦片贴图元素ID（当你用 "创建 "瓦片贴图元素时，你会得到这个ID。 layer_tilemap_create() 或者当你使用函数 layer_tilemap_get_id())，以及房间内的X和Y的位置，该函数将返回该点的瓷砖贴图内的单元的Y的位置。 注意，如果该值在瓷砖贴图区域之外，并且没有单元格可用，它将返回-1。

语法。

tilemap_get_cell_y_at_pixel(tilemap_element_id, x, y)。

争论 描述
tilemap_element_id 瓷砖地图元素的唯一ID值，以获得单元格的Y位置。
x 用于获取单元格的房间内的X位置
y 用于获取单元格的房间内的Y位置

返回。

整数（Y轴单元的位置，如果有错误则为-1）。

例子。

var lay_id = layer_get_id("Tiles_Walls");
var map_id = layer_tilemap_get_id(lay_id);
var _x = tilemap_get_cell_x_at_pixel(map_id, mouse_x, mouse_y);
var _y = tilemap_get_cell_y_at_pixel(map_id, mouse_x, mouse_y);
tiledata = tilemap_get(map_id, _x, _y);

上述代码使用检索到的瓦片地图ID从一组房间坐标中获取单元格位置，然后将在那里发现的任何瓦片的数据存储在一个实例变量中。
