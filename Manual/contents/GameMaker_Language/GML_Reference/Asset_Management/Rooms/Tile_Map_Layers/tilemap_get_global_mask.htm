瓷砖地图_获取全局掩码

这个函数可以用来检索所有瓷砖贴图的位掩码值，返回当前掩码值，如果有错误则返回-1，如果没有指定掩码则返回0。关于全局瓦片图位掩码的进一步信息，请参见函数 tilemap_set_global_mask().

语法。

tilemap_get_global_mask()。

返回。

实数（0表示没有屏蔽，-1表示错误，否则为屏蔽值）。

例子。

var mask = tilemap_get_global_mask(map_id);
var new_mask = tile_mirror | tile_flip | tile_rotate | 255;
if mask != new_mask
{
tilemap_set_global_mask（new_mask）。
}

上面的代码获取与所有瓷砖贴图相关的全局掩码值。如果它与变量 "new_mask "中定义的值不一样，那么它就被设置为该值。
