瓷砖图谱_获取掩码

这个函数可以用来检索给定瓦片贴图的位掩码值。你给瓦片贴图元素ID（当你使用 layer_tilemap_create()或当 使用该函数时得到的 layer_tilemap_get_id())，该函数将返回当前的掩码值，如果有错误则返回-1，如果没有指定掩码则返回0。关于瓷砖贴图位掩码的进一步信息。 请参阅函数 tilemap_set_mask().

语法。

tilemap_get_mask(tilemap_element_id)。

争论 描述
tilemap_element_id 瓷砖贴图元素的唯一ID值，以获得其掩码。

返回。

实数（0表示没有掩码，-1表示错误）。

例子。

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var mask = tilemap_get_mask(map_id);
var new_mask = tile_mirror | tile_flip | tile_rotate | 255;
if mask != new_mask
{
tilemap_set_mask(map_id, new_mask);
}

上面的代码从给定的图层中获取瓦片贴图ID，然后检查与之相关的掩码值。如果它与变量 "new_mask "中定义的值不一样，那么它就被设置为该值。
