瓷砖地图_获取瓷砖高度

使用这个函数你可以检索到瓷砖贴图元素的每个瓷砖单元的高度（像素）。你给瓦片贴图元素的ID（你在使用 "创建 "瓦片贴图元素时得到的ID）。 layer_tilemap_create() 或使用函数 layer_tilemap_get_id())，该函数将返回瓦片集单元的高度。

语法。

tilemap_get_tile_height(tilemap_element_id)。

争论 描述
tilemap_element_id 瓷砖地图元素的唯一ID值，以获得瓷砖单元的高度。

返回。

真实的

例子。

var lay_id = layer_get_id("Tiles_Walls");
var map_id = layer_tilemap_get_id(lay_id);
global.snap_x = tilemap_get_tile_width(map_id) ;
global.snap_y = tilemap_get_tile_height(map_id);

上面的代码使用检索到的瓦片地图ID来获得瓦片单元的宽度和高度，并使用它们来设置两个全局变量。
