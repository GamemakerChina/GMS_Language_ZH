瓷砖图的宽度

使用这个函数，你可以检索到瓷砖地图元素的宽度（以单元格为单位）。你给瓦片贴图元素的ID（当你用以下方法创建一个瓦片贴图元素时得到的 layer_tilemap_create()或当 使用函数 layer_tilemap_get_id()时得到的），该函数将返回宽度（单元格的数量）。

语法。

tilemap_get_width(tilemap_element_id)。

争论 描述
tilemap_element_id 瓷砖地图元素的唯一ID值，以获得其宽度。

返回。

整数

例子。

var lay_id = layer_get_id("Tiles_Walls");
var map_id = layer_tilemap_get_id(lay_id);
var _w = tilemap_get_width(map_id);
var _h = tilemap_get_height(map_id);
total_tiles = (_w*_h);

上面的代码使用检索到的瓦片地图ID来获得瓦片地图的宽度和高度，然后使用这些值来设置一个实例变量。
