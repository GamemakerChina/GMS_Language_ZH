瓷砖图案_set_at_pixel

这个函数可以用来将图层上的瓦片贴图元素中的一个单元设置为新的瓦片，使用瓦片在房间内的实际位置来改变。你给瓦片贴图元素的ID（当你用 "D "创建一个瓦片贴图元素时，你会得到这个ID）。 layer_tilemap_create()或当你使用函数 layer_tilemap_get_id())，然后是要设置的瓷砖数据以及在房间里的位置。你可以 可以使用函数检索瓷砖数据 tilemap_get_at_pixel()检索瓷砖数据，然后在使用这个函数再次设置之前，使用tile_函数来改变它。如果瓷砖被成功设置，该函数将返回 "true"；如果有问题而没有被设置，则返回 "false"。

语法。

tilemap_set_at_pixel(tilemap_element_id, tiledata, x, y)

争论 描述
tilemap_element_id 要改变的瓦片地图元素的唯一ID值
平铺数据 要设置的瓦片组数据
xcell X位置（在房间里
ycell y的位置（在房间里

返回。

布尔型

例子。

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var data = tilemap_get_at_pixel(map_id, 4, 4);
data = tile_set_flip(data, true);
tilemap_set_at_pixel(map_id, data, 4, 4) 。

上面的代码得到了 "Tiles_Sky "层上的瓷砖地图的ID，然后用它来获取位置为（4, 4）的瓷砖数据。这个瓦片数据在被用来再次设置瓦片地图上的瓦片之前会被翻转。
