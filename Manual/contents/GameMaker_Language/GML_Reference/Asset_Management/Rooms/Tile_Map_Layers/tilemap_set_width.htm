tilemap_set_width

这个函数可以用来调整一个瓦片贴图元素的大小。你给瓦片贴图元素的ID（当你创建一个瓦片贴图元素时，使用 layer_tilemap_create()或使用函数 layer_tilemap_get_id()), 和瓦片地图的新宽度（以瓦片单元为单位）。

语法。

tilemap_set_width(tilemap_element_id, width)。

争论 描述
tilemap_element_id 要设置宽度的瓦片图元素的唯一ID值。
宽度 宽度值（以 "单元格 "为单位）。

返回。

不适用

例子。

var lay_id = layer_get_id("Tiles_Walls");
var map_id = layer_tilemap_get_id(lay_id);
if tilemap_get_width(map_id) != room_width div 16
{
tilemap_set_width(map_id, room_width div 16);
}

上面的代码检查特定瓷砖地图的宽度，如果它不是正确的尺寸，那么宽度就会被设置。
