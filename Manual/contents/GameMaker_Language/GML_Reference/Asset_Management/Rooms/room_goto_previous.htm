房间_转到_上一个

有了这个功能，你可以让你的游戏进入到游戏编译时房间管理器中列出的前一个房间。如果这个房间不存在，就会出现一个错误，游戏将被强制关闭。 将被强制关闭。请注意，在调用该函数的事件结束前，房间不会改变，所以如果在同一事件中，调用该函数后的任何代码仍将运行。需要注意的是，任何创建实例的代码，如果在这之后调用 的代码将不会创建实例，所以应该在改变房间之前调用。这个函数也会触发房间结束事件。

语法。

room_goto_previous()。

返回。

不适用

例子。

if room_exists(room_previous(room))
{
room_goto_previous();
}

上面的代码将检查在当前房间之前是否有另一个房间，如果有，它将去那个房间。

返回。房间
