房间高度

这个变量持有当前房间的高度，单位是像素。你可以在任何时候改变这个变量来改变房间的高度，而改变将应用于房间的底部，因为原点被认为是左上角。所以。 例如，如果房间的高度是480px，而你把它设置为640px，那么房间就会向下扩展，底部会多出180px。

语法。

room_height;

返回。

真实的

例子。

如果 bbox_bottom > room_height
{
y += room_height - bbox_bottom;
}

上面的代码检查当前实例的精灵边界框是否大于房间的高度，如果是，则将实例向上移动。
