房间_instance_add

通过这个函数，你可以将一个实例添加到当前房间以外的任何房间，并在该房间的任何位置添加。该函数返回实例的唯一 id实例的唯一性，然后可以 可以在进一步的函数调用中使用，以设置该实例的属性等，但只有在游戏进入指定的房间后才能使用。如果你想在当前房间创建一个实例，你应该使用函数 instance_create_layer().注意 在资产浏览器中创建的房间资产上调用这个函数将永久地把这个实例添加到房间中，即使调用 game_restart() ，也不会把房间恢复到它的原始状态 (只有结束游戏并再次打开游戏时，房间才会再次处于原始状态)。

语法。

room_instance_add(index, x, y, obj)。

争论 描述
指数 要添加一个对象实例的房间的索引。
x 新实例的X位置。
y 新实例的y位置。
obj 要添加一个实例的对象。

返回。

实例ID

例子。

global.rm = room_add();
room_assign(rm_Base, global.rm);
room_instance_add( global.rm, 100, 100, obj_Player);

上面的代码将在游戏中添加一个新的房间，然后将索引为 "rm_Base "的房间的内容复制到其中。然后，它将在这个新房间的100,100位置添加一个对象 "obj_player "的实例。

返回。房间
