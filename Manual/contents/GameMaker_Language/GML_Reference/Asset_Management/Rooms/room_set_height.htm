房间_设置高度

通过这个功能，你可以改变（或设置）你游戏中任何房间的高度，除了当前房间。

语法。

room_set_height(index, h);

争论 描述
指数 要设置高度的房间的索引。
h 房间的新高度，单位是像素。

返回。

不适用

例子。

global.myroom = room_add();
room_set_width( global.myroom, 640);
room_set_height( global.myroom, 480);
room_set_persistent( global.myroom, false);

这将创建一个新的房间，并将其索引存储在变量 "global.myroom "中。然后，它将设置其宽度为640像素，高度为480像素，标题为 "游戏室"，其持久性为 "false"。

返回。房间
