持续性的房间_设置

通过这个函数，你可以改变（或设置）你游戏中任何房间的持久性，除了当前的房间。如果一个房间的持久性被标记为"true"，它将保持该房间内所有实例的状态，如果玩家离开并 然后返回，而如果持久性被标记为假，那么每次都会被重置为初始状态。你应该注意，一个持久化的房间比一个普通的房间使用更多的内存，所以不建议在你的游戏中拥有太多的 不建议在你的游戏中拥有太多这样的房间。

注意：如果房间已经被访问过，这个函数将无法关闭持久性!一个持久化的房间，一旦被访问过，就会被保存在内存中，要关闭持久化，你应该去那个房间并将 room_persistent 变量为false，然后再次退出该房间。

语法。

room_set_persistent(index, val)。

争论 描述
指数 要设置持久性的房间的索引。
价值 房间是否应是持久的（真）或不（假）。

返回。

不适用

例子。

global.myroom = room_add();
room_set_width( global.myroom, 640);
room_set_height( global.myroom, 480);
room_set_persistent( global.myroom, true);

这将创建一个新的房间，并将其索引存储在变量 "global.myroom "中。然后，它将设置其宽度为640像素，高度为480像素，标题为 "游戏室"，其持久性为 "true"。

返回。房间
