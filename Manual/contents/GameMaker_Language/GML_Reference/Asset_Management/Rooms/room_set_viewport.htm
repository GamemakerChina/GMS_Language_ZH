视图端口的设置

通过这个函数，你可以为你游戏中的任何房间设置视口属性，除了当前的房间。你要提供要设置的房间索引、视口索引（从0到7）以及视口在房间开始时是否可见（设置为 "true "表示视口可见，否则为 "false"）。 视图端口在房间开始时是否可见（设置为"true"使端口可见，否则为"false"）。之后你设置x和y坐标（对应于视口的左上角位置），然后设置宽度和高度（像素）。

语法。

room_set_viewport(rm, vind, vis, xport, yport, wport, hport)。

争论 描述
rm 要设置的房间的索引
vind 要设置的视图端口的索引
视觉 视图端口的可见性（true为可见，false为不可见）。
出口 房间内视口的X位置
舷窗 视口在房间里的Y位置
仓库 视口的宽度（单位：像素）。
报道 视口的高度（像素）。

返回。

不适用

例子。

global.myroom = room_add();
room_set_width(global.myroom, 640);
room_set_height(global.myroom, 480);
room_set_viewport(global.myroom, 0, true, 0, 0, 640, 480) 。

这将在变量 "global.myroom "中索引的房间中设置视图端口[0]属性。

返回。房间
