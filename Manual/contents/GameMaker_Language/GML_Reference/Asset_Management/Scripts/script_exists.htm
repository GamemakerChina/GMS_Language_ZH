脚本_存在

这个函数将根据给定索引的脚本或脚本函数是否存在而返回真或假。注意。 这不是一个字符串，而是为每个脚本保留唯一索引的资产名称（正如它在IDE中显示的那样）或脚本函数名称变量，正如在脚本资产中定义的那样（注意 这对方法变量不起作用）。)关于脚本的更多信息，请参见《脚本编辑器》。

语法。

script_exists(scr)。

争论 描述
擦洗 你想检查的脚本的名称。

返回。

布尔型

例子。

script[0] = -1;
script[1] = AI_Left;
script[2] = AI_Right;
var script_num = choose(0, 1, 2);
if script_exists(script[script_num] )
{
script_execute(script[script_num])。
}

上面的例子将两个脚本函数和一个值添加到一个数组中，然后继续得到一个随机数，用它来选择一个脚本函数来运行，除非选择-1，否则什么也不会发生。
