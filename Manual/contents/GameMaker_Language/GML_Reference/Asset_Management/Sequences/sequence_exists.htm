序列_存在

通过这个函数，你可以检查一个序列对象是否存在。你可以提供序列对象的结构（如函数返回的 sequence_create()或 sequence_get()) 或者序列ID（由函数返回 layer_sequence_get_sequence()或从序列实例结构属性序列中返回，或从资产 浏览器），如果序列对象存在，该函数将返回真，如果不存在，则返回假。

语法。

sequence_exists(sequence_struct_or_id)。

争论 描述
序列结构_or_id 要检查的序列对象结构或ID

返回。

Bool

例子。

if sequence_exists(my_seq)
{
sequence_destroy(my_seq)。
}

上面的代码检查给定的序列对象是否存在，如果存在，就销毁它。

返回。序列
