序列_获取对象

通过这个函数，你可以检索一个数组，其中包含所有在给定序列中被创建的实例的对象索引。你可以提供序列对象的结构（如由函数 sequence_create()或 sequence_get())或序列ID(由函数 layer_sequence_get_sequence()或 从序列实例结构属性序列中返回），该函数将返回一个数组，其中数组中的每一项是一个 object_index为序列所使用的不同对象 来创建实例。

语法。

sequence_get_objects(sequence_struct_or_id)。

争论 描述
序列结构_or_id 用于获取对象的序列对象结构或ID。

返回。

阵列

例子。

var _seq = sequence_get(seq_Logo);
obj_array = sequence_get_objects(_seq) 。

上面的代码得到了一个序列对象的结构，然后检索了它所使用的对象，并将数组存储在一个变量中。
