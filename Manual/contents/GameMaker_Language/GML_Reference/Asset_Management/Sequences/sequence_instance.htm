序列_instance

这是一个内置变量，是为游戏中每个对象实例创建的实例变量的一部分。如果该实例是由一个序列控制的。 这个变量将持有控制该实例的序列实例结构，否则它将是未定义的。这是一个只读变量 不能被改变。

请注意，在控制序列结束后，这个变量将变得未定义，即使其序列元素仍然存在，如果该序列元素被重放，将再次获得结构。

语法。

序列_instance

返回。

布尔型

例子。

如果（in_sequence）
{
sequence_instance.speedScale = 2;
}

上面的代码检查 in_sequence变量，如果它是 true （意味着该实例被一个序列控制），那么它将把该序列的速度比例改为 序列为2。

返回。序列
