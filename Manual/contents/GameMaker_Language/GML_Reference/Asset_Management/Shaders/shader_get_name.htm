shader_get_name

通过这个函数，你可以检索到一个着色器资源的名称。你提供要获取着色器名称的唯一ID值，该函数将以字符串形式返回该资源的名称。

语法。

shader_get_name(shader)。

争论 描述
着色器 要获取着色器名称的索引。

返回。

字符串

例子。

var _shader=shader_current();
var _name =shader_get_name(_shader);
draw_text(32, 32, "Debug - Currently Rendering = " + _name);

上面的代码将获得给定着色器的名称并将其绘制到屏幕上。
