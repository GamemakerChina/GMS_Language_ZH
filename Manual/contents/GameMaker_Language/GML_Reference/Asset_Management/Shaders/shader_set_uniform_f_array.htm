shader_set_uniform_f_array

通过这个函数，你可以设置一个着色器常量来保存一个数组的值。你必须事先用函数获得常量的 "句柄"，并且你将有机会获得常量。 shader_get_uniform()得到常量的 "句柄"，并且你必须 并且你必须事先初始化该数组。

注意：所有制服必须在调用函数后设置 shader_set()之后，并在调用 shader_reset().

语法。

shader_set_uniform_f_array(handle, array)。

争论 描述
处理 要设置的着色器常量的句柄。
阵列 一个先前初始化的浮点数值的数组。

返回。

不适用

例子。

shader_set(shader_Glass);
tex_array[0] = 0.5;
tex_array[1] = 0.1;
tex_array[2] = 0.25;
shader_params =shader_get_uniform(shader_glass, "u_vParams") ;
shader_set_uniform_f_array(shader_params, tex_array);
draw_self()
shader_reset()。

上面的代码将获得着色器常量 "u_vParams "的句柄，然后将该常量设置为给定的数组。
