shader_set_uniform_i

通过这个函数，你可以设置一个着色器常量的值（或数值）。你必须在之前通过函数获得该常量的 "句柄"，并在此基础上对其进行设置。 shader_get_uniform()获取常量的 "句柄"，并且 你必须知道它是什么类型的常量，以便向它传递正确数量的整数值，即：如果你有一个vec2，你将需要向该函数传递两个值。

注意：所有的制服都必须在调用函数 shader_set()之后，并在调用 shader_reset().

语法。

shader_set_uniform_i(handle, value1 [, value2, value3, value4])。

争论 描述
处理 要设置的着色器常量的句柄。
价值1 ... 价值4 将着色器常量设置为的整数值（或数值）。

返回。

不适用

例子。

shader_set(shader_Glass);
shader_params =shader_get_uniform(shader_glass, "u_vParams");
shader_set_uniform_i(shader_params, 0, 65, 255);
draw_self() 。
shader_reset()。

上面的代码将获得着色器常量 "u_vParams "的句柄（一个vec3），然后将该常量设置为给定的整数值。
