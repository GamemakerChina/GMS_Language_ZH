shader_set_uniform_i

通过这个函数，你可以设置一个着色器常量来保存一个数组的值。你必须事先用函数获得该常量的 "句柄"。 shader_get_uniform()得到常数的 "句柄"，并且 你必须事先初始化该数组。

注意：所有制服必须在调用函数后设置 shader_set()之后，并在调用 shader_reset().

语法。

shader_set_uniform_i_array(handle, array)。

争论 描述
处理 要设置的着色器常量的句柄。
阵列 一个先前初始化的整数值数组。

返回。

不适用

例子。

shader_set(shader_Glass);
col_array[0] = 255;
col_array[2] = 255;
col_array[3] = 64;
col_array[4] = 128;
shader_params =shader_get_uniform(shader_tint, "cColourArray") ;
shader_set_uniform_i_array(shader_params, col_array);
draw_self();
shader_reset()。

上面的代码将获得着色器常量 "cColourArray "的句柄，然后将该常量设置为给定的阵列。
