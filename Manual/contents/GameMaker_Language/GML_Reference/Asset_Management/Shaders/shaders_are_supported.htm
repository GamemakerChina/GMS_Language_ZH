着色器是否支持

这个函数将进行检查，看所选的目标平台是否支持着色器，如果支持则返回true，如果不支持则返回false。值得注意的是，在 安卓系统, 如果该项目没有定义任何着色器资源，那么该函数将总是返回false，而不管设备是否支持着色器。

语法。

shaders_are_supported()。

返回。

布尔型

例子。

global.GFX = false;
ifshaders_are_supported()
{
ifshader_is_compiled(sh_glass) && shader_is_compiled(sh_warp)
{
global.GFX = true;
}
}

上面的代码将把一个全局变量设置为false，然后如果平台支持着色器，并且被检查的两个着色器都已经正确编译，那么它将被设置为true。
