骨架_动画_获取

通过这个函数你可以得到你的骨骼动画精灵正在使用的动画集。返回值是一个字符串，它将是你在你的骨骼动画程序中定义的动画集的名称。

语法。

skeleton_animation_get()。

返回。

字符串

例子。

if keyboard_check_pressed(vk_space)
{
if skeleton_animation_get() != "jump"
{
skeleton_animation_set("jump");
}
}

上面的代码会在按下空格键时将正在使用的动画集改为 "跳跃 "集，但前提是当前正在使用的动画集不是 "跳跃"。

返回。动画
