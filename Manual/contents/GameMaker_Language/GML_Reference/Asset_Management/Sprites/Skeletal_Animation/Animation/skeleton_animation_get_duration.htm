骨架_动画_获取时间

这个函数将返回给定的动画集在循环回到开始之前所需的时间。返回值的单位是秒。

语法。

skeleton_animation_get_duration(animname)。

争论 描述
动画名称 要使用的动画集的名称（一个字符串）。

返回。

真实的

例1:

time += delta_time / 1000000;
var duration = skeleton_animation_get_duration(skeleton_animation_get() );
var frame = floor((image_number * (mTime / duration))+ 0.5) % image_number;
image_index = frame;
draw_self();

上述代码将把image_index设置为当前分配的骨骼动画精灵的正确值。

例2:

time += delta_time / 1000000;
var d = skeleton_animation_get_duration("walk");
if time > d time -= d;
draw_skeleton_time(sprite_index, "walk", "skin1", time, x, y, image_xscale, image_ysscale, image_angle, c_white）。)

上面的代码将使用delta-time设置被绘制的帧来绘制给定的骨架动画精灵。

返回。动画
