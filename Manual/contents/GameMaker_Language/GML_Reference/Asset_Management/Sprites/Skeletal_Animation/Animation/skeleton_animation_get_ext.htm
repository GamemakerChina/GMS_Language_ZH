骨架_动画_获取文本

一个骨架动画精灵可以有各种动画集，这些动画集可以被分配到不同的轨道上，这样你就可以 "混合搭配 "动画集。这个函数将返回当前使用的动画集的名称。 给定的轨道号（如函数设定的 skeleton_animation_set_ext).

语法。

skeleton_animation_get_ext(track)。

争论 描述
追踪 要获取动画集名称的轨道编号。

返回。

字符串

例子。

if skeleton_animation_get_ext(1) != "bodyfight"
{
skeleton_animation_set_ext("bodyfight", 1);
}

上面的代码将把轨道1使用的动画集改为 "bodyfight"，如果它还不是的话。

返回。动画
