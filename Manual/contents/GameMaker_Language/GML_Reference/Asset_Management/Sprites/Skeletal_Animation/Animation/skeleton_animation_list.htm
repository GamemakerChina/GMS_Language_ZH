骨架_动画_列表

通过这个函数，你可以用作为骨架动画精灵的一部分的所有动画的名字填充一个（预先创建的）DS列表。这些名称将是字符串，然后可以 被用于这些类型的精灵的其他动画函数中。

语法。

skeleton_animation_list(sprite, list)。

争论 描述
精灵 要获取列表的Spine骨骼动画的精灵索引。
列表 要用动画名称填充的DS列表的ID。

返回。

不适用

例子。

var list = ds_list_create();
skeleton_animation_list(sprite_index, list);
for (var i = 1; i < ds_list_size(list); i++; )
{
skeleton_animation_mix(list[| 0], list[| i], 0.5);
}
ds_list_destroy(list)。

上面的代码为实例所使用的精灵创建了一个所有动画名称的DS列表。然后它循环浏览这些动画，并将所有动画的混合值与第一个动画的混合值设置为0.5。

返回。动画
