骨架_动画_设置文本

当你为你的骨架动画精灵设置一个动画集时，它在内部被指定为0轨。但是，你可以将更多的动画集分配到更多的轨道（1和更多），这样你的精灵就可以同时播放两个（或更多）动画。 多）动画。请注意，你的动画程序在设计精灵时应该考虑到这一点，比如说0轨道的 "基本 "动画只有某些骨骼的关键帧，而接下来的轨道有其他骨骼的关键帧 的关键帧。例如，如果你有一个 "走 "的动画，它的关键帧只影响精灵的腿，那么你可以有一组头部动画和手臂动画，你可以分别在轨道1和2中设置，以便 你可以有一个基本的动作，并在游戏进行中实时调整它。

语法。

skeleton_animation_set_ext(animname, track)。

争论 描述
动画名称 要使用的动画集的名称（一个字符串）。
追踪 要使用的动画集的轨道。

返回。

不适用

例子。

if mouse_check_button(mb_left)
{
skeleton_animation_set_ext("bodyfight", 1);
}
else
{
skeleton_animation_set_ext("bodywalk", 1);
}

当按下鼠标左键时，上述代码将把轨道1正在使用的动画集改为 "bodyfight "集或 "bodywalk "集。

返回。动画
