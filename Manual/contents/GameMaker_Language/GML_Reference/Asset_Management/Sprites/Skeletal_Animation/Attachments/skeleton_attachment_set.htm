骨架_附件_设置

一个骨架动画精灵可以添加其他图像作为附件，这些图像被添加到一个命名的槽中（名称是在你的动画程序中创建附件槽时给出的），它们将与当前精灵的动画一起被绘制。 当前精灵的动画。通过这个函数，你可以将一个附件设置到一个给定的槽中，你需要给出槽和附件的名称（字符串）。这些名字是由所使用的动画程序定义的，或者（在附件的情况下）在你调用 当你调用 skeleton_attachment_create().

注意，你也可以传入一个sprite_index作为附件，该精灵将被使用，或者你可以用-1来从槽中移除附件。当你把一个精灵的索引作为参数传入时，它将以精灵的名字作为字符串创建 一个附件槽，使用精灵的名字作为字符串来命名该槽（例如，使用spr_sword，将创建一个附件槽 "spr_sword"），该槽将使用该精灵的第一个图像索引（0），以及它的x。 槽将使用该精灵的第一个图像索引（0），以及它的x和y原点偏移，比例为（1，1），旋转为0。

语法。

skeleton_attachment_set(slot, attachment)。

争论 描述
槽 要获取附件的槽名（字符串）。
附件 附件图像的名称（以字符串或 sprite_index的形式）。

返回。

字符串

例子。

if skeleton_attachment_get("slot_leftHand") == ""
{
skeleton_attachment_set("slot_leftHand", choose("sword", "sparar", "killer"));
}

上述代码将检查当前分配给名为 "slot_leftHand "的插槽的附件名称，如果返回一个空字符串，就会附加一个新的精灵。

返回。附件
