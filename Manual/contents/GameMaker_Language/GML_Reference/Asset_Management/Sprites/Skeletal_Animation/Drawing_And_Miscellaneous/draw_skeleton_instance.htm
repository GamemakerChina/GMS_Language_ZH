绘制骨架实例

这个函数只适用于那些用骨架动画程序Spine创建的精灵分配给它们的实例。虽然你可以使用普通的绘制函数来绘制这些精灵（如 draw_self()), 有些时候，你可能想画一个单帧或显示皮肤纹理的变化，而不需要实际改变实例的精灵属性。在这些情况下，你可以用这个函数绘制已经分配给实例的精灵。 在这个函数中，你给出唯一的实例ID，然后给出动画的名称以获取帧（一个字符串，在用于制作精灵的程序中设置）。接下来，你给出要使用的皮肤的名称（同样，在制作精灵时设置的），以及一个 帧值。帧值是用来告诉GameMaker Studio 2要显示动画的哪一部分，是一个介于0和 图像_编号.其余的参数是 其余参数与普通精灵绘制所用的参数相同，将改变最终绘制的动画帧的比例、角度、混合颜色和阿尔法。

警告!由于骨架动画在不同的 "关键 "帧之间插值的方式，这个调用会产生额外的开销，因此建议你避免使用这个方法来绘制基于骨架的精灵。 绘制基于骨骼的精灵，除非绝对必要。

语法。

draw_skeleton_instance(instance, animname, skinname, frame, x, y, xscale, yscale, rot, color, alpha)。

争论 描述
实例 要绘制精灵的实例的索引。
动画名称 获取动画帧的名称（一个字符串）。
皮肤名称 要使用的皮肤的名称（一个字符串）。
框架 要绘制的动画帧（从0到image_number - 1）。
x 绘制精灵的X坐标。
y 绘制精灵位置的y坐标。
xscale 精灵的水平比例，作为一个乘数。1=正常缩放，0.5是一半等等...
yscale 精灵的垂直缩放比例，是一个乘数。1=正常缩放，0.5是一半等等...
腐烂 该精灵的旋转。0=正常，90=逆时针旋转90度等等。
颜色 用于混合精灵的颜色。
阿尔法 精灵的alpha（从0到1，其中0是透明的，1是不透明的）。

返回。

不适用

例子。

draw_skeleton_instance(obj_Player, "jump", "skin1", 0, x, y, 1, 1, 0, c_white, 1);

上述代码将绘制分配给所给实例的骨架动画精灵的第一帧，使用 "跳跃 "动画集和定义为 "skin1 "的皮肤。
