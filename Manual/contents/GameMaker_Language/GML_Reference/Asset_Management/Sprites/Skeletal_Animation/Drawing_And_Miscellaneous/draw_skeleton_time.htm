draw_skeleton_time

这个函数将使用给定的皮肤在动画的特定时间内绘制给定的动画。时间值应该在0（开始）和动画的结束时间之间，你可以用函数找到 skeleton_animation_get_duration(). 你可以将时间值设置为高于动画总持续时间的值，这样动画就会循环到开始，但是你有可能随着累计时间的增加而失去浮点精度 变得更大。

语法。

draw_skeleton_time(sprite, animname, skinname, time, x, y, xscale, yscale, rot, color)

争论 描述
精灵 要绘制的精灵的索引。
动画名称 获取动画帧的名称（一个字符串）。
皮肤名称 用于绘图的皮肤的名称（一个字符串）。
时间 绘制动画的时间段（从0到结束时间，单位为秒）。
x 绘制精灵的X坐标。
y 绘制精灵位置的y坐标。
xscale 精灵的水平比例，作为一个乘数。1=正常缩放，0.5是一半等等...
yscale 精灵的垂直缩放比例，是一个乘数。1=正常缩放，0.5是一半等等...
腐烂 该精灵的旋转。0=正常，90=逆时针旋转90度等等。
颜色 用于混合精灵的颜色。

返回。

不适用

例子。

var time += delta_time / 1000000;
var d = skeleton_animation_get_duration("walk");
if time > d time -= d;
draw_skeleton_time(sprite_index, "walk", "skin1", time, x, y, image_xscale, image_ysscale, image_angle, c_white）。)

上面的代码将使用delta-time设置被绘制的帧来绘制给定的骨架动画精灵。
