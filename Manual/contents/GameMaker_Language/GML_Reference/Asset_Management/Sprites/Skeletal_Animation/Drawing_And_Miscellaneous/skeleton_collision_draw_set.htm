骨架_碰撞_绘制_设置

通过这个函数，你可以切换开启（true）或关闭（false）绘制实例正在使用的当前骨架动画精灵的碰撞数据。如果这个功能被打开，边界框和精确的碰撞掩码将被画成精灵周围的轮廓。 被绘制成精灵周围的轮廓线。

语法。

skeleton_collision_draw_set(flag)。

争论 描述
旗帜 设置为"true"可以打开绘图，设置为 "false"可以关闭绘图。

返回。

不适用

例子。

如果debug_mode == true
{
skeleton_collision_draw_set(true);
}

上面的代码检查游戏是否在调试模式下运行，如果是，它将切换实例的骨架碰撞数据来显示。
