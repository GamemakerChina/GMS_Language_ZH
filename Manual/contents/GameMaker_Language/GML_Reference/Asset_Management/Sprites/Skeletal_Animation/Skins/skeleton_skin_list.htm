骷髅头_皮肤_列表

通过这个函数，你可以在一个（预先创建的）DS列表中填入作为骨架动画精灵一部分的所有皮肤的名称。这些名字将是字符串，然后可以 用于这些类型的精灵的其他骨架动画皮肤函数中。

语法。

skeleton_skin_list(sprite, list)。

争论 描述
精灵 要获取列表的Spine骨骼动画的精灵索引。
列表 要用动画名称填充的DS列表的ID。

返回。

不适用

例子。

var list = ds_list_create();
skeleton_skin_list(sprite_index, list);
var num = ds_list_size(list);
skeleton_skin_set(list[| irandom(num - 1))) ;
ds_list_destroy(list);

上面的代码创建了一个DS列表，然后用皮肤名称来填充它。然后随机选择一个，并在列表被销毁前应用于实例。
