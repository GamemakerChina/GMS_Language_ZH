骷髅头_皮肤_设置

一个骨架动画精灵可以用各种纹理（称为 "皮肤"）来绘制，允许你将一个单一的动画用于多个不同的目的。例如，你可能有一个通用的行走动画，它可以被你的 在游戏中的NPC角色、敌人等等，只需改变它所使用的皮肤。这些皮肤需要事先在你的动画程序中创建并命名，这个名称被传递到函数中（作为一个字符串），以设置精灵 在被绘制时使用该皮肤。

语法。

skeleton_skin_set(skinname)。

争论 描述
皮肤名称 要使用的皮肤的名称（一个字符串）。

返回。

不适用

例子。

if skeleton_skin_get() == "skin_Enemy1"
{
skeleton_skin_set(choose("skin_Enemy1", "skin_Enemy2", "skin_Enemy3");
}

上述代码将检查当前分配给动画的皮肤，如果它是 "skin_Enemy1"，它将从三个选项中选择并设置一个新的皮肤。
