骨架_插槽_颜色_设置

通过这个函数，你可以设置一个附件槽的颜色和阿尔法，这样所有附着在它身上的精灵都会用这些混合值来绘制。请记住，实例精灵可以有一个混合颜色和阿尔法设置(图像混合和 图像_alpha)，附件也可以(见函数 skeleton_attachment_create_colour())，所以 因此，为槽指定的附件精灵的最终颜色和阿尔法将是所有这些值的组合。

语法。

skeleton_slot_colour_set(slot, color, alpha)。

争论 描述
槽 要设置的插槽名称，一个字符串
颜色 要设置的颜色，可以是一个整数、一个常数或一个十六进制值。
阿尔法 要设置的阿尔法从0到1。

返回。

不适用

例子。

slot_list = ds_list_create();
skeleton_find_slot( mouse_x, mouse_y, slot_list);
if !ds_list_empty(slot_list)
{
for (var i = 0; i < ds_list_size(slot_list); ++i;)
{
if skeleton_slot_colour_get(slot_list[| i]) != c_white
{
skeleton_slot_colour_set(slot_list[| i], c_white, 1);
}
}
}

上面的代码创建了一个DS列表，然后用鼠标在房间里的位置发现的插槽名称来填充它。然后循环浏览槽位列表，并将找到的槽位的颜色重置为白色，如果该颜色不是 已经是白色的。

返回。槽位
