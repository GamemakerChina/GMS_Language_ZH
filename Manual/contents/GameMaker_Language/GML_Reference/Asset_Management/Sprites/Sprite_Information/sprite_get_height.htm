幽灵的高度

通过这个函数，你可以找到基本精灵资产的高度，没有变换，单位是像素。

语法。

sprite_get_height(index)。

争论 描述
指数 要寻找精灵的高度的索引。

返回

真实的

例子。

if sprite_height != sprite_get_height(sprite_index)
{
image_yscale = 1;
}

上面的代码检查精灵在当前实例中的高度，如果与原来的基础精灵有差异，它就会重置Y轴比例。
