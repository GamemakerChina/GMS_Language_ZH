幽灵_获取_九宫格

该函数返回精灵的九宫格结构，包含在精灵编辑器中或使用sprite_set_nineslice() 设置的所有九宫格属性。该结构的内容详见本页面。

如果提供的精灵没有分配给它一个Nine Slice结构，我们将为该精灵创建一个带有默认Nine Slice属性的新结构并返回。

改变这个结构中的任何值都会 修改原始精灵的九宫格属性，影响今后对该精灵进行的任何绘制调用。

语法。

sprite_get_nineslice(ind)。

争论 描述
ind 检索Nine Slice结构的精灵的索引。

返回。

结构（如果精灵不存在，则为-1

例子。

var _box_nineslice =sprite_get_nineslice(spr_box_0);

_box_nineslice.enabled = true;
_box_nineslice.left = 10;
_box_nineslice.right = 10;
_box_nineslice.top = 10;
_box_nineslice.bottom = 10。

上面的代码从精灵中检索九段结构，为其启用九段，并设置引导偏移。
