幽灵的数量

一个精灵必须至少有一个子图像，这个函数将返回它有多少个。记住，一个精灵的子图像从0开始编号，所以如果这个函数返回3（被检查的精灵有3个子图像），它们将被编号为 0、1和2。

语法。

sprite_get_number(index)。

争论 描述
指数 该精灵的索引。

返回

整数

例子。

image_index = floor(随机(sprite_get_number(sprite_index)))。

上述代码将根据精灵的子图像数量，将图像索引设置为一个随机帧。
