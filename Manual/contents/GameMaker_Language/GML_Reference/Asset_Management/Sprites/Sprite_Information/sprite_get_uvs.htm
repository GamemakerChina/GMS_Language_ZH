幽灵_获取_uvs

该函数返回一个数组，其中包含纹理页上精灵子图像的纹理的UV坐标和其他数据。该函数返回一个包含以下8个元素的数组。

  • [0] = 左
  • [1] = 顶部
  • [2] = 对
  • [3]=底部
  • [4] = 资产编译器从精灵左侧修剪的像素数量
  • [5] = 资产编译器从精灵顶部修剪的像素数量
  • [6] = 已保存到纹理页的原始精灵宽度的像素数据的正常化百分比
  • [7] = 已保存到纹理页的原始精灵高度的像素数据的正常化百分比

然后，这个数组可以被用于其他的绘制函数，特别是在3D工作或使用2D原始函数时，以及在使用 着色器函数时。

注意：这个函数对矢量精灵或骨架动画精灵不起作用。

语法。

sprite_get_uvs(sprite, subimage)。

争论 描述
精灵 要使用的精灵的索引。
子图像 要使用的精灵的子图像。

返回

数组 (1D, 8个元素)

例子。

var tex = sprite_get_uvs(sprite, 0);
tex_left = tex[0];
tex_top = tex[1];
tex_right = tex[2];
tex_bottom = tex[3];

上述代码将把给定精灵的UV坐标存储在一个本地数组中，然后把这些值分配给实例变量。
