幽灵的宽度

通过这个函数，你可以找到基本资源精灵的宽度，没有变换，单位是像素。

语法。

sprite_get_width(index)。

争论 描述
指数 要查找宽度的精灵的索引。

返回

真实的

例子。

if sprite_width != sprite_get_width(sprite_index)
{
image_xscale = 1;
}

上面的代码检查精灵的宽度，因为它在当前实例中是这样的，如果它与原始基础精灵之间有差异，它就会重置X轴的比例。
