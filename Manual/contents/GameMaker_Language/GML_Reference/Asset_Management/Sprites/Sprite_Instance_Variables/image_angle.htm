图像_角度

这个值设置精灵的角度（旋转），单位是度，右边是0度，向上是90度，左边是180度，向下是270度。将这个变量设置为0，可以将精灵重置为在精灵编辑器中定义的那样绘制。请注意 请注意，为了使这个变量的变化可见，这个实例应该没有绘制事件（因此GameMaker Studio 2将默认绘制精灵），或者使用一个扩展的绘制函数，如 draw_self()或 draw_sprite_ext().

语法。

图像_角度。

返回。

实数（单精度浮点值

例子。

image_angle = point_direction(x, y, mouse_x, mouse_y)。

上述代码将旋转实例的精灵，使其始终指向鼠标的位置。
