图像编号

这个只读变量可以用来获取已经分配给一个实例的精灵的子图像数量（如果你需要一个精灵的子图像数量，而不是分配给实例的那个，你应该使用 应该使用 sprite_get_number()).请注意，当有（例如）1个子图像时，该变量将返回 "1"，但该子图像的image_index 是0。

语法。

Image_number;

返回。

实数（单精度浮点值

例子。

if image_speed > 0
{
if image_index > image_number - 1 instance_destroy();
}

上面的代码检查精灵是否在做动画，如果是，则检查当前image_index是否大于子图像的数量，如果是，则销毁该实例。
