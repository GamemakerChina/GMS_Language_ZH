图像_速度

这个变量决定了GameMaker Studio 2在当前实例精灵的子图像中的循环速度。给出的速度值是一个乘数，1是默认值，将其设置为0.5将是动画速度的一半--如在Sprite Editor或Image Editor中设置的--而将其设置为2将是两倍。 动画速度的一半--如在Sprite Editor或Image Editor 中设置的那样--而将其设置为 2 将使其加倍。如果使用的精灵没有 没有子图像，这个变量就没有影响。

语法。

image_speed;

返回。

实数（单精度浮点值

例子。

with (instance_create_layer(x, y, "Effects", obj_Explosion))
{
image_speed = 0.5;
}

上述代码创建了一个对象 "obj_Explosion "的实例，并将其image_speed设置为0.5（有效地将运行速度减半）。
