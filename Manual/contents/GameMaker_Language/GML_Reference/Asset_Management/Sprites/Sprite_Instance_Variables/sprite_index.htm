魉_指数

这个变量返回实例的当前精灵的索引，如果实例没有与之相关的精灵，则返回-1。你可以改变它，给实例一个不同的精灵，方法是给它一个资源树上的精灵的名字，或者通过 使用一个变量，其中有一个外部加载的精灵索引。改变精灵并不改变当前可见帧的索引，所以如果你改变了第3帧的精灵，新的精灵将在该帧可见时被绘制出来 假设它有相同的帧数）。

语法。

萌芽指数。

返回。

实数（整数 - 资源索引值）。

例子。

with (obj_Check)
{
if !collision_line(x, y, other.x, other.y, obj_Wall, false, true)
{
sprite_index = spr_spotted;
}
else
{
sprite_index = spr_clear;
}
}

上述代码将循环浏览所有 "obj_Check "的实例，检查它们与运行代码的实例之间是否有碰撞线。这些实例的精灵将被改变，这取决于碰撞线的返回值（真或假）。 碰撞线。
