从表面添加精灵

这个函数的工作方式与 sprite_create_from_surface()一样，只是它不是从你选择的索引表面的区域创建一个新的精灵，而是 将曲面的定义区域作为一个新的子图像添加到先前创建的精灵中（这意味着你不能将其直接添加到资源树中的精灵中，而只能添加到先前从曲面创建的精灵中，或者已经复制的精灵中 的精灵，或者用 sprite_duplicate()).

应该注意的是，以这种方式添加精灵时使用的内存会比你预期的大。这是因为GameMaker Studio 2将把精灵作为一个纹理页来存储，它也将被存储在GPU内存中。 也将被存储在GPU内存中，所以总的内存将比相同精灵的图像文件的预期大。

语法。

sprite_add_from_surface(index, surface, x, y, w, h, removeback, smooth)。

争论 描述
指数 要添加新图像的精灵的索引。
表面 获取图像的表面的索引。
x 要复制的X位置。
y 要复制的y位置。
w 要复制的区域的宽度（从x位置开始）。
h 要复制的区域的高度（从y位置开始）。
撤回 表示是否使所有具有背景颜色的像素（左下角的像素）透明。
顺利 表示是否要平滑边缘。

返回。

不适用

例子。

spr_custom = sprite_create_from_surface(surface, 0, 0, 32, 32, true, true, 16, 16);
var i;
for (i = 1; i < 8; i +=1)
{
sprite_add_from_surface(spr_Custom, surf, i, 0, 32, 32, true, true, 16, 16) ;
}

上面的代码从变量 "surf "中索引的表面创建了一个精灵，并将其索引分配给变量 "spr_Custom"，然后使用for循环在表面上移动并捕获不同的部分作为子图像加入精灵。 各个部分，并将其作为子图像添加到精灵中。
