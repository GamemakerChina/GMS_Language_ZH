萌芽_保存_条状物

该函数将从一个精灵的所有子图像中创建一个条状图像，以指定的文件名保存到光盘中。该精灵必须是在运行时 添加的（你不能保存通过IDE添加的精灵），并且 文件必须以*.png为扩展名保存。

注意：根据所选择的目标平台，你可以从哪里保存和加载文件是有限制的。更多信息请参见文件处理。

语法。

sprite_save_strip(ind, filename)。

争论 描述
ind 要改变透明度贴图的精灵的索引。
文件名 保存的精灵条的文件名。

返回

不适用

例子。

var spr, i;
spr = sprite_create_from_surface(0, 0, 32, 32, true, true, 16, 16);
for (i = 1; i < 8; i +=1)
{
sprite_add_from_surface(spr, i, 0, 32, 32, true, true, 16, 16);
}
sprite_save_strip(spr, "Custom_Player_Sprite.png");
sprite_delete(spr);

上面的代码从变量 "surf "中索引的表面创建了一个精灵，并将其索引分配给本地变量 "spr_Custom"，然后使用for循环在表面上移动并捕捉不同的部分，并将其作为子图像加入精灵。 这个新的精灵在被从内存中删除之前被保存为一个png条。
