萌芽_集束

这个函数可以用来设置一个精灵的边界框值。你提供要使用的精灵索引，然后提供围框位置的左、上、右和下的值。这些位置是绝对值，其中（0，0）位置 对应于精灵的左上角，而不考虑精灵的偏移量、精灵可能有的任何 "空 "像素，或者它在房间里的位置。

注意：这个函数会影响精灵资产 ，因此所有带有这个精灵的进一步实例都会有相同的边界框。

语法。

sprite_set_box(ind, left, top, right, bottom)。

争论 描述
ind 要设置包围盒的精灵的索引。
左边 界限盒的左侧
顶部 界定框的顶部。
对 界限盒的右侧
底部 界定框的底部。

返回

不适用

例子。

如果 sprite_get_box_mode(sprite_index) == 2
{
left = irandom(sprite_width / 2);
right = irandom((sprite_width / 2) + irandom(sprite_width / 2));
top = irandom(sprite_height / 2);
bottom = irandom((sprite_height / 2) + irandom(sprite_height / 2));
sprite_set_box(sprite_index, left, top, right, bottom);
}

上述代码将检查分配给sprite_index的精灵的边界框模式，如果它被设置为手动，那么它的边界框将被改变。
