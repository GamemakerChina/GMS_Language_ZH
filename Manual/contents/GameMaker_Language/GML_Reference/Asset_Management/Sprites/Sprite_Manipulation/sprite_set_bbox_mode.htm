萌芽_set_box_mode

这个函数可以用来设置一个精灵的边界框模式。你提供精灵的索引和要使用的模式，它应该是下列常数之一。

恒定 描述
bboxmode_automatic 自动--将根据精灵的公差设置，自动计算边界框。
bboxmode_fullimage 全图--包围盒将被设置为使用精灵的全部宽度和高度，而不考虑公差和 "空 "像素。
bboxmode_manual 手动 - 边界框被手动设置为用户定义的值，这需要使用函数 sprite_set_box()

注意：这个函数会影响精灵资产 ，所以所有带有这个精灵的进一步实例都会有相同的边界盒模式。

语法。

sprite_set_box_mode(ind, mode)。

争论 描述
ind 要改变模式的精灵的索引。
模式 要设置的模式（一个常数）。

返回

不适用

例子。

if sprite_get_box_mode(sprite_index) != 0
{
sprite_set_box_mode(sprite_index, bboxmode_automatic);
}

上面检查了当前精灵的bbox模式，如果它不是自动的，那么它就被设置为这个值。
