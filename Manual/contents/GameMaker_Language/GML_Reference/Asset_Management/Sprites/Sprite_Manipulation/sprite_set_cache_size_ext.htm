绘图_设置缓存尺寸_ext

在执行图像混合时，HTML5 无法以可执行文件的方式动态地执行它。因此，GameMaker Studio 2必须临时保存图像的混合副本，并将其载入。这个函数设置了多少个混合的副本 在旧的拷贝被覆盖之前，可以缓存多少个给定精灵的混合拷贝。默认为4。这适用于精灵的一个单一的给定子图像。

语法。

sprite_set_cache_size_ext(ind, index, max)。

争论 描述
ind 要改变缓存大小的精灵的索引。
指数 要改变缓存大小的ind的子图像。
最大 可以存储的精灵的最大缓存份数。

返回

不适用

例子。

sprite_set_cache_size_ext(sprite0, 0, 2)。

这将把 sprite0 的第一个子图像的精灵缓存设置为 2 份。
