魉的故事

该函数用于将一个九宫格结构应用于一个精灵。您需要提供精灵的索引和要应用的九宫格结构，可以使用 sprite_nineslice_create()或 sprite_get_nineslice().

注意： 这个函数会影响精灵资产，所以你用这个函数所做的任何改变都会影响所有正在使用这个精灵的实例，以及所有未来的实例。

语法。

sprite_set_nineslice(ind, nineslice)。

争论 描述
ind 要修改的精灵的索引
ǞǞǞ 应用九段式结构

返回。

不适用

例子。

var _nineslice = sprite_nineslice_create();

_nineslice.enabled = true;
_nineslice.left = 10;
_nineslice.right = 10;
_nineslice.top = 10;
_nineslice.bottom = 10;

sprite_set_nineslice(spr_box_0, _nineslice)。

上面的代码创建了一个新的Nine Slice结构，将其启用并为导引器设置了偏移量。然后将该结构应用于精灵，改变其Nine Slice属性。
