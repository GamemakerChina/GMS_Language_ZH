时间线_最大值_时刻

这个函数将返回时间线执行最后动作的时刻值。因此，如果你有一条有8个不同动作的时间线，相隔20分钟，这个函数将返回该时间线中最后一个动作的时刻值。 也就是160。这个函数是测试一条时间线在运行时是否已经过了最后的活动时刻的好方法，因为无论是否有进一步的行动，时间线的位置在游戏的每一步都会继续递增。

语法。

Timeline_max_moment(ind)。

争论 描述
ind 要获得最后时刻的时间线的索引。

返回。

不适用

例子。

If timeline_position > timeline_max_moment(timeline_index)
{
timeline_running = false;
}

上面的代码将检查当前时间线的位置与最大活动时刻，如果大于这个时刻，时间线将被停止。

返回。时间轴
