Timeline_moment_add_script

通过这个函数，你可以在时间线中的任何给定的 "时刻 "向时间线动态添加脚本函数，这里的 "时刻 "相当于一个游戏刻度（或 步骤）。)通过这种方式，你可以使用该函数创建一个新的时间线，并在任何时候添加不同的行为。 Timeline_add()函数创建一个新的时间线，并在任何时候添加不同的行为，或者简单地用新的行为修改先前创建的时间线资源。注意 如果你用它来修改资产浏览器中的时间线资产，那么所有使用这个时间线的实例都会受到影响，并且这种改变会持续到 游戏结束（调用 game_restart() ，也不会重置该变化）。

语法。

Timeline_moment_add_script(ind, step, script)。

争论 描述
ind 时间线的索引，以增加一个时刻。
阶梯 要添加到的时刻（步骤）。
脚本 要添加到时刻中的脚本函数的索引。

返回。

不适用

例子。

global.tl = timeline_add();
var i = room_speed * 60;
repeat(3)
{
timeline_moment_add_script(global.tl, i, choose(attack_1, Attack_2, Attack_3);
i += room_speed * 60;
}

上述代码将创建一个新的时间线，并将其索引存储在变量 "global.tl "中。然后，它将在时间线上添加三个脚本，以一分钟的间隔随机选择。
