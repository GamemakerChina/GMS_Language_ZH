时间轴_运行中

这个变量保持指定的时间线的当前状态，如果它正在运行，将返回true，如果它没有运行，将返回false。你也可以把这个变量设置为真或假，以便随时启动和停止时间线。 应该注意的是，停止的时间线不会被重置，因此在以后的时间里再次启动它将从它被停止的确切时刻开始。

语法。

Timeline_running;

返回。

布尔型

例子。

if !timeline_running
{
timeline_position = 0;
timeline_running = true;
}

上面的代码将检查实例是否正在运行时间线，如果没有，那么它将重置指定的时间线，在第一时间启动，然后启动它。
