云存储

在制作游戏时，通常需要将游戏状态的信息存储在某种类型的文件中，但将这些信息存储在设备上并不总是最好的选择，因为如果玩家删除游戏后再重新安装，这些信息 可能会丢失。为了防止这种情况，你可以使用各种不同的云服务，这些服务通过互联网提供数据存储，以便在任何时候检索和修改。你应该注意，这个功能仅限于每个游戏的一个单一的 数据 "blob"，所以每次你向云服务发送一个新的保存，无论是一个字符串还是一个文件，它都会覆盖任何以前存储的数据。

注意： 目前只有 安卓目标使用这些功能，你需要在安卓游戏选项的社交部分勾选启用谷歌云存储的复选框。 中的 "社交 "部分的复选框。这将提示你下载谷歌服务扩展。 这是在安卓系统上进行云存储的必要条件。关于云存储的更多信息，请参见以下帮助台文章。安卓云存储

GameMaker Studio 2通过一些简单的函数来支持这一点，这些函数与异步云事件一起工作。这些功能 在下面几页中进行了解释。

  • 云_同步
  • 云端字符串保存
  • 云文件保存
