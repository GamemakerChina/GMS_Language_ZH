显示_消息_async

这个函数打开一个窗口，向用户显示你在该函数中定义的信息。这是一个异步函数，因此GameMaker Studio 2在等待回答时不会阻止它所运行的设备，而是 继续正常地运行事件。一旦用户按下 "Okay "按钮，一个异步的 对话框事件被触发，在该事件的持续时间内 该事件的持续时间，将有一个ds_map存储在变量 async_load.

这个映射将包含两个键，"id "和 "status"，其中 "id "是函数调用时返回的值，"status "是 "Okay "按钮被按下时的真值，或者是被取消时的假值。 如果消息被取消，则为 "true"；如果消息被取消，则为 "false"（因为并非所有目标平台都允许取消对话）。

注意。这个功能 只用于调试和测试，不应该在已发布的游戏中使用。为此，你应该创建一个自定义的用户界面来接收玩家的输入，这样你就可以完全控制对话框的外观和行为。

语法。

show_message_async(string)。
