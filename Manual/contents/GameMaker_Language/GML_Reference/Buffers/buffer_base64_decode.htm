buffer_base64_decode

通过这个函数，你可以将一个base64编码的字符串（用 buffer_base64_encode()函数创建的）到一个缓冲区。这个函数将创建一个缓冲区（作为一个1字节对齐的 "增长 "缓冲区 并返回该缓冲区的唯一索引，该索引应在所有进一步的函数调用中使用。

注意：当你不再需要这样的动态创建的资源时，必须将其从内存中删除，以防止内存泄漏，所以当你用完你所创建的缓冲区后，你应该再次释放它，这一点很重要 使用 buffer_delete().

语法。

buffer_base64_decode(string)。

争论 描述

真实的

例子。

ini_open("Save.ini");
buff = buffer_base64_decode(ini_read_string("Save", "Slot1", "") );
ini_close();

上面的代码将打开一个ini文件，然后从里面读取一个字符串到解码函数。该函数将返回一个缓冲区索引，该索引存储在变量 "buff "中，包含先前编码和保存的数据。该ini文件 然后被关闭。
