buffer_base64_decode_ext

通过这个函数，你可以将一个base64编码的字符串（用 buffer_base64_encode()函数创建的）到一个缓冲区。与函数不同的是 buffer_base64_decode(), 这个函数不会为你创建一个缓冲区，而是你应该已经创建了缓冲区（见 buffer_create())，然后你可以用这个函数来使用它的ID。偏移量 "是指 在缓冲区中解码给定字符串的位置（以字节为单位）。

语法。

buffer_base64_decode_ext(buffer, string, offset)。

争论 描述
缓冲区 要将字符串解码到的缓冲区的索引。
绳子 要解码的base64编码的字符串。
补偿 数据偏移值。

返回。

不适用

例子。

buff = buffer_create(16384, buffer_grow, 2);
ini_open("Save.ini");
var str = ini_read_string("Save", "Slot1", "");
buffer_base64_decode_ext(buff, str, 0);
ini_close();

上面的代码将创建一个缓冲区，并在变量 "buff "中存储其唯一的ID，然后打开一个ini文件，从其中读取一个字符串到本地变量 "str "中。然后这个字符串被解码到新创建的缓冲区中，然后再关闭ini文件。 再次关闭ini文件。
