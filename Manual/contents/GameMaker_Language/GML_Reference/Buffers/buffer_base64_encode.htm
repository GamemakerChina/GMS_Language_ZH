buffer_base64_encode

这个函数将把给定缓冲区的数据转换成base64格式的编码字符串。这是一个常用的编码方案，通常用于任何需要在互联网上以文本形式存储或传输的媒体，并使其 输出，使人眼无法阅读。要使用这个，你需要指定一个已经创建的缓冲区，偏移值（这是缓冲区内你希望开始编码的点）以及缓冲区内存的大小，以字节为单位，进行编码。

语法。

buffer_base64_encode(buffer, offset, size)。

争论 描述
缓冲区 要使用的缓冲区的索引。
补偿 数据偏移值。
尺寸 缓冲区的大小。

返回。

字符串

例子。

var b_str = buffer_base64_encode(buff, 0, buffer_get_size(buff) )

上述代码将对存储在由变量 "buff "索引的缓冲区中的全部数据进行创建编码，并将返回的字符串存储在本地变量 "b_str"。
