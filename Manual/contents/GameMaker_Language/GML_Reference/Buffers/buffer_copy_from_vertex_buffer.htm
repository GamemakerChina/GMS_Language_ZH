buffer_copy_from_vertex_buffer

这个函数可以用来将存储在一个顶点缓冲区中的部分（或全部）顶点数据复制到一个先前创建的普通缓冲区中。当用这个函数从一个顶点缓冲区复制到一个普通缓冲区时，这两个缓冲区必须在之前已经被 创建（使用 vertex_create_buffer()和 buffer_create()函数，例如）。)你可以指定你想复制到缓冲区的顶点数据的范围。 你可以指定要复制到缓冲区的顶点数据范围，其中起始顶点可以是0到顶点数-1之间的任何地方，你可以给出从该点开始要复制的顶点数。你可以使用函数 vertex_get_number() 来获取存储的顶点总数。最后，你可以给出要拷贝顶点数据的缓冲区索引，以及定义拷贝顶点数据在目标缓冲区中的位置的数据偏移。

语法。

buffer_copy_from_vertex_buffer(vertex_buffer, start_vertex, num_vertices, dest_buffer, dest_offset)。

争论 描述
顶点缓冲区（vertex_buffer 要复制的顶点缓冲区的索引。
开始时_顶点 开始的顶点。
num_vertices 要使用的顶点总数。
缓冲区（Dest_buffer） 要复制到的缓冲区的索引。
目的地_offset 要复制数据的偏移位置（以字节为单位）。

返回。

不适用

例子。

var v_num = vertex_get_number(model_buff); buffer_copy_from_vertex_buffer(model_buffer, 0, v_num - 1, player_buffer, 0) 。

上述代码将复制存储在变量 "model_buffer "中索引的顶点缓冲区的顶点数据，然后将其粘贴到变量 "player_buffer "中索引的缓冲区。
