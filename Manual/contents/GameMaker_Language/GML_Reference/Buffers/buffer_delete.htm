緩衝區_刪除

通过这个函数，你可以从内存中删除一个以前用函数创建的缓冲区 buffer_create()创建的缓冲区，释放用于创建该缓冲区的资源，并删除其当前可能包含的任何数据。

注意：当你不再需要任何动态创建的资源时，一定要将其从内存中删除，以防止内存泄漏，这一点很重要。

重要的是!当你创建一个缓冲区时，用来识别它的索引值是一个从0开始的整数。这些索引被GameMaker重复使用，所以一个被破坏的缓冲区索引值可能会被之后新创建的缓冲区使用。 我们建议你在销毁缓冲区后，将持有缓冲区索引的任何变量设置为-1。

语法。

buffer_delete(buffer)

争论 描述
缓冲区 要删除的缓冲区的索引。

返回。

不适用

例子。

buffer_delete(player_buffer);
player_buffer = -1;

上面的代码将删除先前创建的缓冲区，其id存储在变量 "player_buffer "中，然后将该变量设置为-1。
