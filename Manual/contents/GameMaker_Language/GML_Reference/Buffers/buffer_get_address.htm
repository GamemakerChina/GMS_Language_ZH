buffer_get_address

通过这个函数你可以得到一个指向原始的、对齐的缓冲区地址的指针。这主要是用于扩展，因为你可以把这个值传递给他们，允许他们访问缓冲区的数据。请注意，有几件事 当从扩展访问时，你不能对缓冲区进行操作。

  • 你不能释放记忆。
  • 你不能调整内存的大小。
  • 你不能写超过缓冲区地址和缓冲区大小（你可以使用 buffer_get_size()函数来实现），否则你会得到一个越界的错误。

语法。

buffer_get_address(buffer)。

争论 描述
缓冲区 要使用的缓冲区的索引。

返回。

指针

例子。

var b_address = buffer_get_address(buff_model);
var b_end_address = b_address + buffer_get_size(buff_model) 。

上面的代码将获得在变量 "buff_model "中索引的缓冲区的内存地址，然后使用buffer_get_size()函数获得缓冲区末端的内存地址，将这两个值存储在本地变量中，以便进一步使用。 进一步使用。
