缓冲区_peek

使用该 buffer_read()函数，你可以在当前的 "寻道 "位置从给定的缓冲区中读取数据，每一块数据的读取都是通过正在读取或写入的字节来推进这个位置。 然而，你可能有必要在不改变当前寻址位置的情况下读取一个给定的数据，这时你就会使用这个函数。你只需向该函数提供一个缓冲区的ID，然后提供该缓冲区内的一个偏移位置（从 缓冲区内的偏移位置（从缓冲区开始），以及你想要读取的数据类型。

注意：对正在读取的数据使用不正确的数据类型，将导致错误的数值!

如果函数成功，那么将根据数据类型返回一个值（Real, string, int, etc...），而失败将简单地返回0。

语法。

buffer_peek(buffer, offset, type)。

争论 描述
缓冲区 要使用的缓冲区的索引。
补偿 缓冲区内的偏移位置（以字节为单位），从这里读取给定的数据。
类型 要从缓冲区读取的数据类型（见这里的常量列表）。

返回。

实数/字符串，如果失败则为0。

例子。

var red = buffer_peek(buff, 1, buffer_u8);
var green = buffer_peek(buff, 2, buffer_u8);
var blue = buffer_peek(buff, 3, buffer_u8);
image_blend = make_colour_rgb(red, green, blue) 。

上述代码将从变量 "buff "中索引的缓冲区内的三个不同位置获得三个值，然后使用这些值来设置实例的图像混合。
