缓冲区大小调整

通过这个函数，你可以将一个给定的缓冲区调整为你指定的大小（以字节为单位）。

语法。

buffer_resize(buffer, newsize)。

争论 描述
缓冲区 要改变大小的缓冲区的索引。
新尺寸 缓冲区的新大小（以字节为单位）。

返回。

不适用

例子。

if (buffer_get_size(buff) < 16384)
{
buffer_resize(buff, 16384);
}

上面的代码将检查变量 "buff "中索引的缓冲区的大小，如果它小于给定的值，缓冲区将被调整大小。
