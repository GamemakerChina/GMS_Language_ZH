buffer_save_ext

通过这个函数，你可以将缓冲区的部分内容保存到一个文件中，准备用下面的函数读回内存。 buffer_load()函数读回内存。偏移量 "定义了缓冲区内的起始位置 的起始位置（以字节为单位），"大小 "是指从该偏移开始要保存的缓冲区的大小（也以字节为单位）。

语法。

buffer_save_ext(buffer, filename, offset, size)。

争论 描述
缓冲区 要保存的缓冲区的索引。
文件名 要保存为的文件的名称。
补偿 要保存的缓冲区内的偏移量（以字节为单位）。
尺寸 要保存的缓冲区的大小（以字节为单位）。

返回。

不适用

例子。

buffer_save_ext(buff, "Player_Save.sav", 0, 16384)。

将当前缓冲区的部分内容与变量 "buff "中存储的id一起保存到一个文件中。
