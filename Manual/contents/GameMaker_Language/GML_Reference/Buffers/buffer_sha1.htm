buffer_sha1

在密码学中，SHA-1是一个由美国国家安全局设计的密码散列函数，在一些广泛使用的应用程序和协议中被采用，如流行的 编码在那里它被用来检查文件的变化。这个函数将从一个缓冲区获取输入数据，并以ASCII格式返回一个160位的信息摘要。通过这种方式，你可以生成一个安全的密钥，它可以被存储并用于检查 被发送到（或从）外部服务器（例如）接收的信息的完整性。当使用这个函数将其应用于缓冲区时，你必须指定要使用的缓冲区ID，然后指定一个偏移值（以字节为单位）作为开始的位置，然后是 被散列的区域的大小（同样以字节为单位）。

注意：SHA-1不是完全安全的，可以被破解。更多信息请见本页面。

语法。

buffer_sha1(buffer, offset, size)。

争论 描述
缓冲区 要使用的缓冲区的索引。
补偿 数据偏移值。
尺寸 缓冲区的大小。

返回。

字符串

例子。

check_string = buffer_sha1(buff, 0, buffer_get_size(buff))。

上面的代码将为存储在由变量 "buff "索引的缓冲区中的全部数据创建一个sha1哈希值，并将返回的哈希值存储在变量 "check_string "中。
