buffer_sizeof

这个函数将返回任何给定的数据常量（这里列出）的大小（以字节为单位）。

语法。

buffer_sizeof(type)。

争论 描述
类型 要检查的数据类型（见这里的常数列表）。

返回。

不适用

例子。

var b = 12 * buffer_sizeof(buffer_u8);
buff = buffer_create(b, buffer_fixed, 1);

上面的代码首先通过无符号8位数据类型乘以12来计算要创建的缓冲区的大小（因为我们将用缓冲区来容纳12个数据），然后用这个值来设置一个固定的缓冲区，使之达到正确的大小。
