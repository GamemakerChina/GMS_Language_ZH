摄制组_获取结束脚本

此函数可用于检索分配给给定摄像机的作为结束函数的脚本函数的 ID。如果没有指定脚本函数，则该函数将返回-1。

语法。

camera_get_end_script(camera_id)

争论 描述
摄像机_ID 创建摄像机时返回的唯一摄像机ID值

返回。

脚本功能ID，如果没有指定功能，则为-1。

例子。

var scr = camera_get_end_script(camera_view[0]);
if scr != cutScene
{
camera_set_update_script(camera_view[0], cutScene);
}

上面的代码检查分配给view port[0]的摄像机的开始脚本函数，如果它不是 "cutScene"，则被设置为该函数。
