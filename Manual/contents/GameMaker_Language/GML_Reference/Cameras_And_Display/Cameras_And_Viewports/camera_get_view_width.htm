摄像机的宽度

此函数可用于检索给定摄像机的宽度（单位：像素）。请注意，此函数仅适用于使用 Camera_create_view()创建的摄像机或在房间 编辑器创建的摄像机。

语法。

camera_get_view_width(camera_id)

争论 描述
摄像机_ID 创建摄像机时返回的唯一摄像机ID值

返回。

真实的

例子。

var vw = camera_get_view_width(view_camera[0]) / 2;
var vh = camera_get_view_height(view_camera[0]) / 2;
camera_set_view_pos(view_camera[0], x - vw, y - vh) 。

上面的代码检索分配给view port[0]的摄像机的宽度和高度，然后设置其相对于中心的位置。
