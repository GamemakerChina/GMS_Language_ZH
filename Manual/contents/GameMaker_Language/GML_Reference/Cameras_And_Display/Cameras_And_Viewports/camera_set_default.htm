摄像机_设置默认值

此函数可用于将默认摄像机设置为使用你先前用某个 Camera_create()函数创建的自定义摄像机。当你创建的房间没有活动的视图端口 或摄像机时，GameMaker Studio 2仍然使用一个摄像机来显示游戏中的动作。这个摄像机被称为默认摄像机，可以像其他摄像机一样进行设置和操作（甚至是销毁）。

语法。

camera_set_default(camera_id)

争论 描述
摄像机_ID 创建摄像机时返回的唯一摄像机ID值

返回。

不适用

例子。

var new_cam = camera_create_view（0, 0, 640, 480, 0, obj_Player, 5, 5, -1, -1）;
camera_set_default（new_cam）。

上面的代码创建了一个新的摄像机，然后把它指定为要使用的默认摄像机。
