摄像机_set_proj_mat

这个函数将为一个给定的摄像机设置投影矩阵。你要给出唯一的相机ID值（如不同的函数所返回的 摄像机创建()函数返回的）和一个要应用的视图矩阵。你可以找到 了解更多关于创建投影矩阵的信息，特别是在矩阵函数一节中。 matrix_build_projection_perspective()和 matrix_build_projection_ortho().请注意，如果你的摄像机能够自动追踪物体，即：它是用 camera_create_view() 创建的物体索引/实例ID不是-1，或者你正在设置房间编辑器中定义的摄像机，或者你正在设置默认摄像机--那么这个矩阵将在下一个游戏帧中被覆盖。

语法。

camera_set_proj_mat(camera_id, matrix)。

争论 描述
摄像机_ID 创建摄像机时返回的唯一摄像机ID值。
基体 你创建矩阵时返回的唯一的矩阵ID。

返回。

不适用

例子。

view_camera[0] = camera_create();
var viewmat = matrix_build_lookat(640, 240, -10, 640, 240, 0, 0, 1, 0);
var projmat = matrix_build_projection_ortho(640, 480, 1.0, 32000.0);
camera_set_view_mat(view_camera[0], viewmat);
camera_set_proj_mat(view_camera[0], projmat）。)

上面的代码创建了一个新的摄像机并将其分配给view port[0]。然后这个相机的视图和投影矩阵被设置。
