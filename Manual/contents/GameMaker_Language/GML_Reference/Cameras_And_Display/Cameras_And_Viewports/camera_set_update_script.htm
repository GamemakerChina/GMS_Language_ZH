摄像机_设置更新_脚本

这个函数可以用来设置一个脚本函数，在摄像机被分配到一个可见和活动的视口时，每一个游戏帧都将被调用。你可以给出唯一的摄像机ID值（如不同的摄像机创建函数所返回的 不同的摄像机_创建函数返回的）和要调用的函数的ID。连接到摄像机的函数和摄像机视图的实际渲染的顺序如下。

  • 所有可见和活动视图端口的摄像机的更新函数 都被称为
  • 然后对于每个视图。
    • 调用该视图的相机的开始函数
    • 为该视图执行绘制事件
    • 摄像机的结束函数 被调用
  • 移动到下一个活动的可见视图并重复...

语法。

camera_set_update_script(camera_id, script)

争论 描述
摄像机_ID 创建摄像机时返回的唯一摄像机ID值。
脚本 运行每个游戏帧的脚本函数

返回。

不适用

例子。

camera_set_update_script(view_camera[0], updateCamera)。

上面的代码设置了分配给视口[0]的摄像机的更新功能。
