摄像机_设置视图角度

你可以用这个函数来更新房间内的视图摄像机的角度。你给出唯一的摄像机ID值（如由不同的 摄像机创建()函数返回的），然后给出你希望设置的摄像机角度。 然后给出你希望设置摄像机视角的角度。默认值是0°，正值会使摄像机逆时针旋转，即：将值设为90会使摄像机向左旋转90°。

语法。

camera_set_view_angle(camera_id, angle)。

争论 描述
摄像机_ID 创建摄像机时返回的唯一摄像机ID值
角度 将摄像机视图设置为新的角度

返回。

不适用

例子。

camera_set_view_angle(view_camera[0], point_direction(x, y, mouse_x, mouse_y))。

上面的代码将为分配给view port[0]的摄像机设置视图摄像机角度。
