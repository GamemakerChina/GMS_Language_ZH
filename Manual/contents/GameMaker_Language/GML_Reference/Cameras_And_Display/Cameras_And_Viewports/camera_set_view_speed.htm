摄像机_设置视图速度

你可以用这个函数来更新房间内的视图摄像机的速度。你给出唯一的摄像机ID值（由不同的 Camera_create()函数所返回的），然后给出X和Y（水平的 和垂直）速度，当设置为跟随一个给定的实例时，它应该移动。速度以每步像素计算，可以设置为"-1 "以使摄像机立即移动，但如果摄像机没有被设置为跟随任何实例，那么 这里设置的值不会有明显的效果。

语法。

camera_set_view_speed(camera_id, xspeed, yspeed)

争论 描述
摄像机_ID 创建摄像机时返回的唯一摄像机ID值。
xspeed 视图在水平（X）轴上移动的速度（每游戏帧的像素数）。
yspeed 视图在垂直（Y）轴上移动的速度（每游戏帧的像素数）。

返回。

不适用

例子。

camera_set_view_speed(view_camera[0], 5, 5)。

上面的代码将为分配给view port[0]的摄像机设置视图摄像机的速度。
