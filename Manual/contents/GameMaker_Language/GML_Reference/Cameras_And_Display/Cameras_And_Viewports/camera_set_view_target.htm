摄像机_set_view_target

你可以使用这个函数来设置房间内的观测摄像机的跟踪目标。你给出唯一的摄像机ID值（由不同的 摄像头创建()函数返回的），然后给出你想设置的目标实例 或对象ID，以便将摄像机视图设置为该对象。注意，如果你选择了一个对象ID，而房间里有不止一个该对象的实例，GameMaker Studio 2没有办法知道你想跟踪哪个实例，所以可能是任何一个。 所以它可能是其中的任何一个。

语法。

camera_set_view_target(camera_id, instance_id/object_id)

争论 描述
摄像机_ID 创建摄像机时返回的唯一摄像机ID值
instance_id/object_id 摄像机的目标实例或对象，以便进行跟踪。

返回。

不适用

例子。

camera_set_view_target(view_camera[0], id)。

上述代码将把分配给view port[0]的摄像机的视图摄像机目标设置为运行代码的实例的目标。
