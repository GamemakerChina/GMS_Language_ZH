查看_get_surface_id

这个函数可以用来检索分配给给定视图端口的表面的唯一ID值（如果没有分配表面，将返回-1）。

语法。

view_get_surface_id(view_port)

争论 描述
视图_端口 目标视图端口（0-7）。

返回。

真实的

例子。

如果view_get_surface_id(0) == -1
{
view_set_surface_id(0, global.vSurf);
}

上面的代码将检查是否有一个表面被分配到视图端口[0]，如果没有，就会分配一个。
