查看_获取_可见性

这个函数可以用来检查给定的视图端口的可见性。如果它是可见的，该函数将返回true，如果它是不可见的，则返回false。

语法。

view_get_visible(view_port)

争论 描述
视图_端口 目标视图端口（0-7）。

返回。

布尔型

例子。

if !view_get_visible(0)
{
view_set_visible(0, true);
}

上面的代码将检查视图端口[0]是否可见，如果不可见，则设置为可见。
