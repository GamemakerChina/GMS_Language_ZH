查看_set_hport

这个函数可以用来设置给定视口的高度。你给出视图端口的索引（从0到7）和要使用的新高度。

语法。

view_set_hport(view_port, h)

争论 描述
视图_端口 目标视图端口（0-7）。
h 新的高度（单位：像素

返回。

真实的

例子。

if view_get_hport(0) != (display_get_height () / 2)
{
view_set_hport(0, display_get_height() / 2);
}

上面的代码将检查view port[0]的高度，如果它不是显示高度的一半，就会被设置为这个值。
