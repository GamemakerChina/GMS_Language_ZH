查看_set_visible

这个函数可以用来设置给定视口的可见性。该函数接收视图端口的索引（从0到7）和一个布尔值，如果它是可见的，则为true，如果它不是，则为false。

语法。

view_set_visible(view_port, visible)

争论 描述
视图_端口 目标视图端口（0-7）。
可见 可见性切换（true为可见，false为不可见）。

返回。

不适用

例子。

if !view_get_visible(0)
{
view_set_visible(0, true);
}

上面的代码将检查视图端口[0]是否可见，如果不可见，则设置为可见。
