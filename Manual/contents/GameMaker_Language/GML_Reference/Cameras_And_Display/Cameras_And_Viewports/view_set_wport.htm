查看_set_wport

这个函数可以用来设置给定视口的宽度。你给出视图端口的索引（从0到7）和要使用的新宽度。

语法。

view_set_wport(view_port, w)

争论 描述
视图_端口 目标视图端口（0-7）。
w 新的宽度（单位：像素

返回。

真实的

例子。

if view_get_wport(0) != (display_get_width () / 2)
{
view_set_wport(0, display_get_width() / 2);
}

上面的代码将检查view port[0]的宽度，如果它不是显示宽度的一半，就会被设置为这个值。
