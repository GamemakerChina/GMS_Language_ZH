窗口_设置位置

有了这个功能，你可以将游戏窗口设置在显示器（在macOS、Linux（Ubuntu）和Windows）或浏览器（HTML5）中的特定位置。

注意：如果你的HTML5游戏使用自定义的index.html，并将画布设置为固定位置，那么这个函数将对窗口位置没有影响。

语法。

window_set_position(x, y)。

争论 描述
x 窗口定位的X坐标。
y 窗口定位的Y坐标。

返回。

不适用

例子。

window_set_position(0, 0)。

上述代码将把游戏窗口定位在浏览器或显示器的左上角（取决于正在使用的目标模块）。
