窗口_设置大小

用这个函数你可以改变游戏窗口的大小。如果你使用这个函数来改变游戏窗口的大小，窗口的内容将被缩放以适应背景画布的1：1比例，这将保持图像的清晰 和清晰（尽管如果窗口的大小与房间或视图的大小不同，也会被缩放）。例如，如果你有一个200 x 150像素的窗口，而玩家通过拖动边框拉伸窗口，最终的窗口尺寸可能是400 x 300像素，但绘图 画布仍然是200 x 150，这就产生了一个模糊的图像，因为现在的像素比是1：2。然而，如果你使用这个函数来设置窗口大小，1:1的比例将被保持，即使图像被缩放以适应新的窗口大小。

语法。

window_set_size ( w, h ) 。

争论 描述
w 窗口的新宽度。
h 窗户的新高度。

返回。

不适用

例子。

if mouse_check_button_pressed(mb_left)
{
global.scale += 1;
window_set_size(200 * global.scale, 150 * global.scale);
}

上面的代码将在按下鼠标按钮时根据变量 "global.scale "的值改变窗口的大小。
