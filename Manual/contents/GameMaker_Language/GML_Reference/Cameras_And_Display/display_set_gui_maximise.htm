display_set_gui_maximise

这个函数可以用来最大化GUI层，并将其设置为与屏幕尺寸有关的缩放和偏移，而不是默认的应用程序表面位置和尺寸。默认情况下，GUI层与应用程序表面的分辨率为1:1。 分辨率，并在表面的(0,0)位置绘制。然而这并不总是你想要的，所以你可以使用这个函数将其设置为相对于显示器或游戏窗口的绝对（0,0）位置绘制。你提供给这个函数的参数是什么？ 的参数将取决于你希望它对GUI层产生的效果。简单地调用没有参数的函数将设置GUI层在屏幕或游戏窗口的(0,0)位置绘制，其宽度和高度 被缩放以适应整个区域。

display_set_gui_maximise()。

然而，你可以为GUI层设置缩放系数，宽度和高度将按该量进行缩放。请记住，GUI层总是被制作成适合显示器或游戏窗口或应用程序表面的大小，所以将这个值设置为1以外的任何值，基本上都是沿着宽度和高度缩放像素。 所以把这个值设置为1以外的任何值，基本上都是按照宽度和高度来缩放像素数的。如果你的显示器是1024x768，而你把缩放系数设置为0.5，那么你的GUI层将是显示器的一半大小，有效地加倍了 像素的大小。

display_set_gui_maximise(0.5, 0.5)。

以这种方式设置数值也会将绘制位置设置为显示器或游戏窗口的（0,0），因此你可以提供偏移值来 "移动"（0,0）的位置。

var pos = application_get_position();
display_set_gui_maximise(0.5, 0.5, pos[0], pos[1]) 。

最后，你可以使用这个函数重置GUI层，把缩放系数设置为-1。这将把GUI层重新设置为1:1的比例，并把绘制位置设置为应用程序表面的(0,0)位置，而不是显示器或 窗口。

display_set_gui_maximise(-1, -1)。

语法。

display_set_gui_maximise(xscale, yscale, xoffset, yoffset)。

争论 描述
xscale <可选>水平缩放系数（使用-1重置为默认值）。
yscale <可选>垂直缩放系数（使用-1重置为默认值）。
xoffset <可选>绘图时的X偏移位置。
yoffset <可选>绘图时的y偏移位置。

返回。

不适用

例子。

display_set_gui_size(display_get_width() / 2, display_get_height() / 2);
display_set_gui_maximise(2, 2, 0, 0)。

上述代码将把绘制GUI事件锁定在给定的宽度和高度上，使用该比例缩放所有组件以适应显示器。
