gif_save

用这个函数你可以保存出一个GIF动画。你提供GIF的索引（由函数 gif_open())以及一个文件名来保存它。请注意，GameMaker Studio 2不会自动 附加.gif文件扩展名，所以如果你希望保存的文件被识别为GIF，你应该把它作为文件名字符串的一部分。创建的GIF将使用通用884调色板进行调色（更多信息见这里）。

请注意，如果该函数是成功的，GIF被正确保存，那么它将返回0，否则它将返回-1。

语法。

gif_save(gif_index, fname)。

争论 描述
gif_index 要保存的GIF的ID
fname 用于gif文件的文件名

返回。

整数

例子。

if save_gif == true
{
if count == 0
{
gif_image = gif_open(room_width, room_height);
}
else if count < 30
{
gif_add_surface(gif_image, application_surface, 6/100);
}
else
{
gif_save(gif_image, "GameCapture.gif");
count = 0;
save_gif = false;
}
count++;
}

上面的代码将创建一个GIF图像文件，其中有30个从应用程序表面拍摄的帧，然后将其保存。
