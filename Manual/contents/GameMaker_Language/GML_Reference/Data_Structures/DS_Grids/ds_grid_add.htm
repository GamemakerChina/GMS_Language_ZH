ds_grid_add

这个函数可以用来将一个给定的值（实数或字符串）添加到网格内给定的单元格的值中。要添加的值必须与网格单元中的值类型相同，即：你不能将字符串添加到实数中，反之亦然，而对于 对于字符串来说，这相当于串联。

语法。

ds_grid_add(index, x, y, val)。

争论 描述
指数 网格的索引。
x 该单元格在网格中的X位置。
y 该单元格在网格中的y位置。
价值 要添加到单元格的值。

返回。

不适用

例子。

ds_grid_add(grid, 5, 5, 6)

这将在变量 "grid "所引用的DS网格中给定的单元格添加6。
