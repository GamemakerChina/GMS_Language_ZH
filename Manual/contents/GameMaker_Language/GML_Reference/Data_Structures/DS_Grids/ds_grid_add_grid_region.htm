ds_grid_add_grid_region

这个函数可以用来将一个网格的源区域内的所有单元格的值加到目标网格内的值上，如下图所示。

注意： 你也可以在同一个网格上使用这个函数，将网格中一个区域的数值添加到另一个区域的数值中（见下面的代码示例）。

语法。

ds_grid_add_grid_region（index, source, x1, y1, x2, y2, xpos, ypos）。

争论 描述
指数 目标网格的索引。
来源 源网格的索引。
x1 要从源网格复制的单元格区域的左边位置。
y1 要从源网格复制的单元格区域的顶部位置。
x2 要从源网格复制的单元格区域的右边位置。
y2 要从源网格复制的单元格区域的底部位置。
xpos 目标网格上的X位置，用于添加源区域。
ypos 目标网格上的y位置，用于添加源区域。

返回。

不适用

例子。

ds_grid_add_grid_region（grid, grid, 0, 0, 1, 5, 2, 0）。

上述代码将复制变量 "grid "中索引的DS网格的(0,0)到(1,5)的单元格区域，并将它们添加到同一DS网格的(2,0)位置的单元格中。
