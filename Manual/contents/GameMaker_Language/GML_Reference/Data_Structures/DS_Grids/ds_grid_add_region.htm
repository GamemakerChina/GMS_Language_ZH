ds_grid_add_region

这个函数可以用来将一个给定的值（实数或字符串）添加到一个网格的定义区域内的所有单元格的值中。要添加的值必须与网格单元中的值具有相同的类型，即：你不能将一个字符串添加到一个实数中，反之亦然。 实数，反之亦然，对于字符串来说，这相当于串联。

语法。

ds_grid_add_region（index, x1, y1, x2, y2, val）。

争论 描述
指数 网格的索引。
x1 该区域在网格中的左边的X位置。
y1 该区域顶部在网格中的y位置。
x2 该区域的右侧在网格中的X位置。
y2 该区域底部在网格中的Y位置。
价值 要添加到区域单元中的值。

返回。

不适用

例子。

ds_grid_add_region(grid, 2, 4, 5, 5, ".")

这将在变量 "grid "所引用的DS网格的定义区域内的单元格中的所有字符串中添加"."。
