ds_grid_create

通过这个函数，你可以创建一个新的DS网格数据结构，并指定单元格的宽度和高度。这个函数返回一个id，这个id必须在所有处理这个DS网格的后续函数中使用。

重要的是!当你创建一个数据结构时，识别它的索引值是一个从0开始的整数值。 这意味着不同类型的数据结构可以有相同的索引值，所以如果有疑问，你应该使用 ds_exists()函数来访问它们。还要注意的是，索引是可以重复使用的，所以一个被破坏的数据结构索引值可能会被之后新创建的数据结构使用。

语法。

ds_grid_create(w, h)。

争论 描述
w 要创建的网格的宽度。
h 要创建的网格的高度。

返回。

真实的

例子。

mygrid = ds_grid_create(10, 10)

这将创建一个10个单元格高和10个单元格宽的网格。
