ds_grid_get_disk_max

这个函数可以用来寻找在一个网格的定义盘区中发现的所有单元格的最大值，如下图所示。

语法。

ds_grid_get_disk_max（index, xm, ym, r）。

争论 描述
指数 网格的索引。
xm 磁盘中心在网格中的X位置。
ym 磁盘中心在网格中的y位置。
r 网格中圆盘的半径。

返回。

实数或字符串

例子。

val = ds_grid_get_disk_max(grid, 5, 5, 2)

上述代码将把变量 "val "设置为变量 "grid "中索引的DS网格的给定磁盘中包含的最大值。
