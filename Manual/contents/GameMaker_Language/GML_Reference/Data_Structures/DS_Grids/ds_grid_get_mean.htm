ds_grid_get_mean

这个函数可以用来寻找在网格的定义区域内发现的所有单元格的平均值（所有单元格的值加在一起，然后除以构成该区域的单元格总数），如下图所示。

语法。

ds_grid_get_mean（index, x1, y1, x2, y2）。

争论 描述
指数 网格的索引。
x1 区域的左边单元格列。
y1 区域的顶部单元格行。
x2 该区域的右侧单元格列。
y2 该区域的底部单元格行。

返回。

实数或字符串

例子。

val = ds_grid_get_mean(grid, 0, 0, 5, 5)

上述代码将把变量 "val "设置为包含在变量 "grid "中索引的DS网格的给定区域内的平均值。
