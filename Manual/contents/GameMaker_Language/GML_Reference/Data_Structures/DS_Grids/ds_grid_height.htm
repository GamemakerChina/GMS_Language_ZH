ds_grid_height

这个函数将返回给定网格的高度。这个值是网格沿y轴的单元数，总是一个整数，如下图所示。

语法。

ds_grid_height(index)。

争论 描述
指数 这个网格的索引来寻找高度。

返回。

真实的

例子。

for (var i = 0; i < ds_grid_width(grid); ++i;)
{
for (var j = 0; j < ds_grid_height(grid); ++j;)
{
if (ds_grid_get(grid, i, j) == 1)
{
instance_create_Layer(i * 32, j * 32, "Walls", obj_Wwall);
}
}
}

上面的代码将循环浏览变量 "grid "中索引的DS网格，如果在任何特定的单元格中发现的值等于1，那么它将在房间的适当位置创建一个 "obj_Wall "的实例。
