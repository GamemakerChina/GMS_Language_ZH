ds_grid_multiply_grid_region

通过这个函数，你可以在一个给定的DS网格内定义一个区域，然后将这些数值与同一个DS网格或另一个网格（之前已经创建）的单独区域中的数值相乘。原来的区域将保持 保持不变，而与之相乘的区域现在将存储每个单元的新值。

语法。

ds_grid_multiply_grid_region（index, source, x1, y1, x2, y2, xpos, ypos）。

争论 描述
指数 目标网格的索引。
来源 源网格的索引。
x1 要从源网格复制的单元格区域的左边位置。
y1 要从源网格复制的单元格区域的顶部位置。
x2 要从源网格复制的单元格区域的右边位置。
y2 要从源网格复制的单元格区域的底部位置。
xpos 目标网格上的X位置，与源区域相乘。
ypos 目标网格上的y位置，与源区域相乘。

返回。

不适用

例子。

ds_grid_multiply_grid_region（mygrid, mygrid, 0, 0, 5, 5, 0, 0）。

这将从DS网格 "mygrid "的(0,0)到(5,5)的单元格区域，与同一DS网格的(0,0)位置的单元格相乘。
