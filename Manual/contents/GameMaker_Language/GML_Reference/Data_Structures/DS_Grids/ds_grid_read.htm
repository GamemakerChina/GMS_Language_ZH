ds_grid_read

这个函数可以用来将之前由函数创建的字符串转换成 ds_grid_write()创建的字符串转换为一个DS网格。该DS网格必须是之前创建的（见下面的例子）。注意 注意，如果数据结构是用以前的 游戏制作者你应该添加可选的参数 "legacy"，将其设置为true，因为这个版本的字符串格式已经改变。

语法。

ds_grid_read(index, string [, legacy])。

争论 描述
指数 要读取的网格的索引。
绳子 要读入DS网格的字符串。
遗产（可选 可以是真或假，也可以完全省略。

返回。

不适用

例子。

grid = ds_grid_create(room_width div 32, room_height div 32);
ini_open(" Save.ini");
ds_grid_read(grid, ini_read_string("Save", "0", "") );
ini_close()

上面的代码根据房间的大小创建了一个DS网格（每个32x32的正方形像素代表一个网格单元），然后从ini文件中读取先前保存的一组网格数据到新的DS网格中。
