ds_grid_set_disk

通过这个函数，你可以将一个网格的圆形区域设置为某个值。你需要提供一个起始网格单元（作为x和y轴坐标），以及要设置的圆盘半径和你希望设置的圆盘值，如下图所示 下面的图示。

语法。

ds_grid_set_disk（index, xm, ym, r, val）。

争论 描述
指数 网格的索引。
xm 磁盘在网格上的X位置。
ym 磁盘在网格上的y位置。
r 网格上的圆盘半径。
价值 要在磁盘内的单元格中设置的值。

返回。

不适用

例子。

ds_grid_set_disk(grid, ds_grid_width(grid) div 2, ds_grid_height(grid) div 2, 5, -4)

上述代码将在变量 "grid "中索引的DS网格中设置一个半径为5个单元的圆形区域，其值为-4。
