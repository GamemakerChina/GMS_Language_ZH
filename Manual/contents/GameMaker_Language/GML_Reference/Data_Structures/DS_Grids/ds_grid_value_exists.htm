ds_grid_value_exists

通过这个函数，你可以检查一个特定的值（实数或字符串）是否存在于一个给定的DS网格的矩形区域内。如果它是存在的，该函数将返回真，否则将返回假。

语法。

ds_grid_value_exists（index, x1, y1, x2, y2, val）。

争论 描述
指数 网格的索引。
x1 该区域在网格中的左边的X位置。
y1 该区域顶部在网格中的y位置。
x2 该区域的右侧在网格中的X位置。
y2 区域底部在网格中的y位置。
价值 要找到的价值。

返回。

布尔型

例子。

if ds_grid_value_exists(grid, 0, 1, 5, 6, val)
{
xpos = ds_grid_value_x(grid, 0, 1, 5, 6, val);
ypos = ds_grid_value_y(grid, 0, 1, 5, 6, val);
}

上面的代码在一个矩形区域内检查DS网格的特定值。如果找到了，它就将该值的x和y位置存储在两个变量中，供以后使用。
