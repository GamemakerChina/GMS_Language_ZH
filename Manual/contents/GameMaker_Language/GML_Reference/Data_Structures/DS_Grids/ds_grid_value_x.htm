ds_grid_value_x

通过这个函数，你可以得到被搜索的值的X坐标（在给定的矩形网格区域内）。你给出DS网格的索引（如由 ds_grid_create()返回的），以及左上角和右下角的x/y位置 以及要搜索的网格区域的左上角和右下角的x/y位置，然后再提供要搜索的值。如果被搜索的值不存在，那么该函数将返回-1。

语法。

ds_grid_value_x（index, x1, y1, x2, y2, val）。

争论 描述
指数 网格的索引。
x1 区域左侧在网格中的X位置，从0到（网格宽度-1）。
y1 区域顶部在网格中的y位置，从0到（网格高度-1）。
x2 区域右侧在网格中的X位置，从0到（网格宽度-1）。
y2 区域底部在网格中的y位置，从0到（网格高度-1）。
价值 要找到的价值。

返回。

真实的

例子。

if ds_grid_value_exists(grid, 0, 1, 5, 6, val)
{
xpos = ds_grid_value_x(grid, 0, 1, 5, 6, val);
ypos = ds_grid_value_y(grid, 0, 1, 5, 6, val);
}

上面的代码在一个矩形区域内检查一个ds_grid的特定值。如果找到了，它就把这个值的x和y位置储存在两个变量中，以便以后使用。
