ds_list_find_index

通过这个函数，你可以检查给定列表中的一个值，并返回该值在列表中的位置。请注意，如果列表中存在多个具有相同值的条目，可以返回其中任何一个的位置。 如果该值不存在，那么将返回-1。请注意，该值可以是一个数组，你可以通过函数 is_array().

语法。

ds_list_find_index(id, val)。

争论 描述
身份证 要使用的列表的ID。
价值 要找到的价值。

返回。

真实的

例子。

pos = ds_list_find_index(list, "Player1");

上述代码检查变量 "list "中索引的列表的值 "Player1"，并将返回的位置存储在变量 "pos "中。
