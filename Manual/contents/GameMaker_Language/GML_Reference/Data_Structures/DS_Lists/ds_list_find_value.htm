ds_list_find_value

通过这个函数，你可以检查给定的列表位置，并返回该位置的列表中的值。请注意，如果你给出的位置超出了给定的列表大小（例如：10个值的列表中的第11个位置），那么这个函数 这是因为当您创建列表时，在内部将列表中的前几个条目设置为 0，以减少最初向列表添加项目时的性能问题。 ds_list_size() 函数仍然会在一个新创建的列表中返回 0）。)如果你想确保列表在创建时是 "真正的 "空的，那么你应该在创建列表后调用 ds_list_clear()然后再创建列表，这将 这意味着任何未填充的列表槽的返回值都是未定义的，然后您可以使用 is_undefined()函数来检查。

语法。

ds_list_find_value(id, pos)。

争论 描述
身份证 要使用的列表的ID。
姿势 要查看的位置，其中0对应于列表的最开始，最终位置是ds_list_size(id)-1。

返回。

实数、字符串或未定义

例子。

val = ds_list_find_value(list, ds_list_size(list) - 1);

上面的代码检查变量 "list "中索引的列表中的最后一个位置，并将返回值存储在变量 "val "中。
