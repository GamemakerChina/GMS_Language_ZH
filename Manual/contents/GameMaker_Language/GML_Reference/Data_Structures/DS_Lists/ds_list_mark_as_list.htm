ds_list_mark_as_list

这个函数将 "标记"（或 "标志"）先前创建的DS列表中的一个给定位置，作为另一个DS列表。在对JSON字符串进行编码时需要这个功能（你可以使用 json_encode()), 但在嵌套数据结构时也很有用，因为当父DS列表被销毁时，以这种方式标记的项目将自动被垃圾收集（销毁）。这意味着你不需要在销毁数据结构之前手动浏览列表内容并 在销毁 "父 "列表之前单独销毁标记的数据结构。然而，如果你单独删除列表的位置，它所包含的DS列表将不会被垃圾收集，你应该在删除DS列表之前调用适当的 DS 列表销毁函数，然后再从父列表位置删除项目。还要注意，如果你在一个列表上调用函数 ds_list_clear()的时候，任何标记为列表的项目也会被销毁。 列表被清除时也会被销毁。

值得注意的是，如果被标记的位置在列表中发生了变化，"标记 "也会随之移动，所以如果您在列表中标记了位置3（例如），然后在它下面再插入2个项目，使它上升到位置 5，它仍然会被标记为一个列表。

语法。

ds_list_mark_as_list(id, pos)。

争论 描述
身份证 要标记的列表的id。
姿势 列表中要标记的位置。

返回。

Real (整数 - 如果成功，DS列表的ID，如果失败，则为-1)

例子。

var j_list = ds_list_create();
var sub_list = ds_list_create();
ds_list_add(sub_list, health);
ds_list_add(sub_list, lives);
ds_list_add(sub_list, score);
ds_list_add(j_list, sub_list) ;
ds_list_mark_as_list(j_list, 0);

上面的代码创建了两个DS列表，然后用全局变量的各种值填充其中一个。然后将这个列表添加到第二个列表中，并将这个位置 "标记 "出来，以便以后能够正确地进行编码。

返回。DS列表
