ds_list_mark_as_map

这个函数将 "标记"（或 "标志"）先前创建的DS列表中的一个给定位置，作为持有DS地图的位置。在对JSON字符串进行编码时需要这个功能（你可以用 使用 json_encode())，但在嵌套数据结构时也很有用，因为以这种方式标记的项目会在父级的 DS列表被销毁。这意味着你不必在销毁 "父 "级列表之前手动浏览列表内容并单独销毁被标记的数据结构。然而，如果你单独删除了列表的位置。 它所包含的数据结构将不会被垃圾回收，你应该在删除父列表位置之前调用适当的DS map destroy函数。还要注意，如果你在一个列表上调用函数 ds_list_clear() 时，任何标记为地图的项目都会在清除列表时被销毁。

还值得注意的是，如果被标记的位置在列表中发生了变化，"标记 "也会随之移动，所以如果你在列表中标记了位置3（例如），然后在它下面再插入2个项目，使它上升到位置 5，它仍然会被标记为一张地图。

语法。

ds_list_mark_as_map(id, pos)。

争论 描述
身份证 要标记的列表的id。
姿势 列表中要标记的位置。

返回。

Real (整数 - DS地图的ID，如果成功，或-1，如果失败)

例子。

var sub_map = ds_map_create();
ds_map_add(sub_map, "player", player_array);
ds_map_add(sub_map, "enemy", enemy_array);
ds_list_add(j_list, sub_map);
ds_list_mark_ as_map(j_list, 0);

上面的代码创建了一个DS地图，然后用两个键填充它，每个键都包含一个数组的值。然后，这个地图被添加到给定的DS列表中，并将其位置 "标记 "出来，以便以后可以正确编码。

返回。DS列表
