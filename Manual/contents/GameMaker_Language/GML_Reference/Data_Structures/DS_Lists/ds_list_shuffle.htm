ds_list_shuffle

通过这个函数，你可以对一个列表进行洗牌，这将把所有的组件值重新排序到与它们最初被添加到列表的位置不同的随机位置。

注意：由于GameMaker Studio 2每次都会生成相同的初始随机种子，使调试代码变得更加容易，所以这个函数会在每次重新运行游戏时将列表项洗到相同的位置。为了避免 这种行为，请使用 randomise()在你的游戏开始时使用。这只是在测试和调试游戏的时候。 因为最终的可执行包不会显示这种行为，每次游戏都是随机的。

语法。

ds_list_shuffle(id)。

争论 描述
身份证 要洗牌的列表的ID。

返回。

不适用

例子。

如果重新开始
{
ds_list_shuffle(card_list)。
}

如果变量 "重启 "被标记为 "true"，上述代码将对变量 "card_list "中索引的列表进行洗牌。
