ds_list_sort

通过这个函数，你可以对一个列表中的所有数值进行排序，可以按升序或降序排序。如果列表中包含字符串，这些字符串将按照英文26个字母的字母表进行排序。

语法。

ds_list_sort(id, ascend)。

争论 描述
身份证 要排序的列表的ID。
升腾 数值应该是升序（真）还是降序（假）。

返回。

不适用

例子。

如果newgame
{
ds_list_sort(name_list, true);
}

如果变量 "newgame "被标记为 "true"，上述代码将对变量 "name_list "中索引的列表进行排序。
