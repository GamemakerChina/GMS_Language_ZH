ds_map_clear

这个函数将清除（先前创建的）DS地图的所有键/值对，但它不会从内存中删除DS地图本身。为此，你应该使用函数 ds_map_destroy().请注意，清除 会取消对存储在地图中的任何数据结构的引用，从而导致内存泄漏，所以你需要在清除地图之前，手动销毁所有的数据结构项，以防止这种情况。唯一不需要这样做的时候是在 你将地图中的任何项目标记为DS列表或另一个DS地图，在这种情况下，这些项目将被销毁（而不是清除！），当父地图被清除时，它们的内存将自动被清理。 父地图被清除时，这些项目将被销毁（而不是清除！），它们的内存将被自动清理。

语法。

ds_map_clear(id)。

争论 描述
身份证 要清除的数据结构的ID。

返回。

不适用

例子。

如果 global.new_game
{
ds_map_clear(inventory)。
}

上述函数将检查一个全局变量是否为真，如果为真，那么实例变量 "库存 "中索引的DS地图将被清除所有键/值对（但它不会被销毁）。

返回。DS地图
