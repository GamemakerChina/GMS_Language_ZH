ds_map_exists

如果指定的键存在于（先前创建的）DS地图 中，这个函数将返回true，如果不存在则返回false。

语法。

ds_map_exists(id, key)。

争论 描述
身份证 要检查的数据结构的id
钥匙 检查的关键是

返回。

布尔型

例子。

if !ds_map_exists(inventory, "potions")
{
ds_map_add(inventory, "potions", 1);
}

上述代码将检查变量 "库存 "中索引的DS地图中的 "药水 "键，如果它不存在，它将把它添加到地图中。
