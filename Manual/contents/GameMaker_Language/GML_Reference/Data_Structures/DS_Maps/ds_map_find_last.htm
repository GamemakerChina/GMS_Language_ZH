ds_map_find_last

这个函数返回存储在DS地图中的最后一个键。这并不是你所添加的顺序中的最后一个键! DS地图不是以线性形式存储的，为此要使用 DS列表，所以这个函数所做的就是找到计算机存储的最后一个键。如果你必须在DS地图中反复寻找某个东西，这可能很有用，但应该尽可能避免，因为它可能很慢。

注意，如果给定的DS地图是空的，这个函数将返回未定义。

语法。

ds_map_find_last(id)。

争论 描述
身份证 要使用的地图的ID。

返回。

实数、字符串或未定义

例子。

var size = ds_map_size(inventory);
var key = ds_map_find_last(inventory);
for (var i = size; i > 0; i--; )
{
if (key != "gold")
{
key = ds_map_find_previous(inventory, key);
}
else break;
}

上面的代码创建了一些临时变量，然后得到DS地图的大小，并找到计算机存储在地图中的最后一个键。然后，它使用for循环在DS地图中回溯，寻找键值 "gold"。 如果它找到了，它就跳出循环。

返回。DS地图
