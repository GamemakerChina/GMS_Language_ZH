ds_map_find_next

这个函数返回在函数中指定的键之后存储在DS地图中的下一个键。如果你必须在DS地图中反复寻找，这可能很有用，但应该尽可能避免，因为它可能很慢。如果没有这样的 键存在，那么该函数将返回未定义。你应该总是用 is_undefined()函数来检查。

语法。

ds_map_find_next(id, key)。

争论 描述
身份证 要使用的地图的ID。
钥匙 关键是要找到下一个到。

返回。

实数、字符串或未定义

例子。

var size = ds_map_size(inventory);
var key = ds_map_find_first(inventory);
for (var i = 0; i < size; i++;)
{
if (key != "gold")
{
key = ds_map_find_next(inventory, key);
}
else break;
}

上面的代码创建了一些临时变量，然后得到DS地图的大小，并找到计算机存储在地图中的第一个键。然后，它使用一个for循环遍历DS地图，寻找键值 "gold"。如果 找到它，它就会跳出循环。

返回。DS地图
