ds_map_replace_list

通过这个函数，你可以用之前创建的另一个列表替换已经存储在给定 "key "中的DS列表。这个函数是为创建JSON兼容地图而设计的。 然后你会用 json_encode()编码，所以只能与该功能一起使用。

语法。

ds_map_replace_list(id, key, value)

争论 描述
身份证 要使用的地图的ID。
钥匙 替换的关键。
价值 用来替换先前存储在给定键中的ds_list的id。

返回。

不适用

例子。

var j_list = ds_list_create();
ds_list_add(j_list, health);
ds_list_add(j_list, lives);
ds_list_add(j_list, score);
ds_map_replace_list(j_map, "list", j_list);
var j = json_encode(j_map);
ds_list_destroy(j_list) 。

上面的代码将创建一个DS列表，并在替换DS地图"j_map "中先前存储的列表之前，用各种全局变量的值来填充它。

返回。DS地图
