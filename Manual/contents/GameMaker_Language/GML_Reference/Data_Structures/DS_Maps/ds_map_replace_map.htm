ds_map_replace_map

通过这个函数，你可以用之前创建的另一个地图替换一个已经存储在给定 "key "中的DS地图。这个函数是为创建JSON兼容的地图而设计的，然后你会用 json_encode()编码，并且只能与该功能一起使用。

语法。

ds_map_replace_map(id, key, value)

争论 描述
身份证 要使用的ds_map的id。
钥匙 替换的关键。
价值 用来替换先前存储在给定键中的ds_map的id。

返回。

不适用

例子。

var temp_map = ds_map_create();
ds_map_add_list(temp_map, "list", j_list);
ds_map_add(temp_map, "array", j_array) 。
ds_map_replace_map(j_map, "maps", temp_map);
var j = json_encode(j_map);
ds_map_destroy(temp_map);

上面的代码将创建一个DS地图，并在替换DS地图"j_map "中先前存储的地图之前，用一个数组和一个DS列表来填充它。

返回。DS地图
