ds_map_secure_load_buffer

这个函数将从一个缓冲区加载一个安全保存的DS地图。你必须事先将缓冲区加载到内存中（使用 buffer_load())，然后把它传到这个函数中，将返回一个由缓冲区内容填充的DS 缓冲区的内容填充的DS地图。注意，缓冲区必须是用函数 ds_map_secure_save_buffer()创建的缓冲区才能正常工作，还要注意，如果被加载的DS 地图包含一个数组，那么在加载时，它将被转换为一个DS列表。

重要的是!安全保存的文件的一个特点是，它被锁定在创建它的设备上，所以你不能把保存在一个设备上的文件加载到另一个设备上运行的项目中。

语法。

ds_map_secure_load_buffer(buffer)。

争论 描述
文件名 要加载地图数据的缓冲区的ID。

返回。

整数（DS地图索引

例子。

var buff = buffer_load("save.dat");
map = ds_map_secure_load_buffer(buff);
buffer_delete(buff);

上面的代码将从缓冲区加载一个安全保存的DS地图，并将其索引值存储在一个变量中供将来使用。

返回。DS地图
