ds_map_set

通过这个函数，你可以在一个给定的DS地图中设置一个键的值。你提供DS地图的ID值（由函数返回 ds_map_create())，然后给出你想设置的键和要设置的值 的值。键值可以是整数或字符串，如果给定的键值不存在，那么将为你创建并设置为该值。这个函数与使用DS map访问器来设置/创建一个 地图的键/值对。这个函数不返回任何东西，所以如果你需要检查键值是否已经被替换或者一个新的键已经被创建，那么你应该使用函数 ds_map_replace() 。

语法。

ds_map_set(id, key, value)

争论 描述
身份证 要使用的地图的ID。
钥匙 钥匙的设置。
价值 将键设置为的值。

返回。

不适用

例子。

if is_undefined(ds_map_find_value(map, "score"))
{
ds_map_set(map, "score", 0);
}

上面的代码将检查给定的键是否存在，如果不存在，那么它将被创建和设置。

返回。DS地图
