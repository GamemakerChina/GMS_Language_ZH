ds_map_values_to_array

通过这个函数，你可以检索一个DS地图中每个键所包含的所有值。你提供DS地图的ID来获取键（如由 ds_map_create()返回的），然后该 函数将返回一个数组，数组中的每个条目都是DS地图中的一个值（例如，如果地图有10个键，那么将返回一个由10个条目组成的数组，每个条目对应一个键的值 对应于一个键的值）。)该函数有一个可选的第二个参数，你可以提供一个你已经创建的数组，在这种情况下，地图的键值数据将被附加到数组中的任何现有数据上。请注意，该函数将直接修改 直接修改所提供的数组，但也会返回它的一个引用（如果没有提供，则返回一个新数组的引用）。

语法。

ds_map_values_to_array(id, [array])

争论 描述
身份证 要使用的地图的ID。
[数组] OPTIONAL!用于追加DS地图值的数组。

返回。

var _values =ds_map_values_to_array(lvl_score);
var _total = 0;
