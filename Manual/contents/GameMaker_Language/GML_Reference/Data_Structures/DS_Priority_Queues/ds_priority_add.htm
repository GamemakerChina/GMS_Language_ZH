ds_priority_add

通过这个函数，你可以向一个优先级队列添加一个值（实数或字符串），同时给它分配一个优先级值。

语法。

ds_priority_add(id, val, priority)。

争论 描述
身份证 要添加到的优先级队列的ID。
价值 要添加到优先级队列的值。
优先权 要增加的价值的优先级。

返回。

不适用

例子。

ds_priority_add(ai_priority, AI_Search, 5)。

上面的代码给变量 "ai_priority "中索引的优先级队列添加了一个脚本函数，并给它分配了5的优先级。
