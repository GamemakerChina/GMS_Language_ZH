ds_priority_find_min

通过这个函数，你可以找到存储在优先级队列中优先级最低的值，如果有多个值具有相同的优先级，那么它们中的任何一个都可以按任何顺序被返回。然而，与之不同的是 ds_priority_delete_min(), 这个函数不会从队列中删除该值。

语法。

ds_priority_find_min(id)。

争论 描述
身份证 要使用的优先级队列的ID。

返回。

实数或字符串

例子。

if ai_move
{
script_execute(ds_priority_find_min(ai_priority)) 。
}

上面的代码检查了一个实例变量，如果它返回真，它将执行一个在优先级队列中以最低优先级值为索引的脚本函数。
