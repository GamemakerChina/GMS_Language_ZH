ds_queue_copy

这个函数可以用来把一个队列的内容复制到另一个队列中。注意，这不会从原队列中删除内容，也不会破坏原队列。当使用这个函数时，被复制到的队列必须是 如果它在复制之前包含任何项目，那么这些项目将首先被清除（意味着这些信息将被丢失）。

语法。

ds_queue_copy(id, source)。

争论 描述
身份证 新队列的ID。
来源 要复制到id的原始队列。

返回。

不适用

例子。

with (instance_create_layer(x, y, "Enemies", obj_Enemy))
{
queue = ds_queue_create();
ds_queue_copy(queue, other.queue);
}

上述函数创建了一个新的实例，然后在该实例中创建了一个新的DS队列，并将运行代码块的实例中的队列内容，复制到新创建的实例队列中。
