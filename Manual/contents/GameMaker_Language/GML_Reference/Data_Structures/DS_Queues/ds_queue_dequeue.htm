ds_queue_dequeue

这个函数将从DS队列中去掉头部的值，将其从队列中移除，并返回存储在一个变量中的值。如果队列是空的，那么这个函数将返回常数 未定义，否则它将返回队列中包含的实数或字符串值。

语法。

ds_queue_dequeue(id)。

争论 描述
身份证 要取消排队的队列的ID。

返回。

存储在队列中的数据类型值或未定义的值

例子。

if !ds_queue_empty(move_queue)
{
var xx = ds_queue_dequeue(move_queue);
var yy = ds_queue_dequeue(move_queue);
move_towards_point(xx, yy, 4);
}

上述代码检查变量 "move_queue "中索引的DS队列是否为空，如果不是，它就从队列的头部去排两个值，用它们来设置移动的方向。
