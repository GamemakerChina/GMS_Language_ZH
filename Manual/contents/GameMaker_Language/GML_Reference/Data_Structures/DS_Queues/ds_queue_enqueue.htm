ds_queue_enqueue

这个函数将在DS队列的尾部添加一个值（实数或字符串）。该函数可以接受另外14个可选的参数（总共有15个可能的添加值），允许你在一次调用中向队列尾部连续添加多个值 允许你在一次调用中连续向队列尾部添加多个值。

语法。

ds_queue_enqueue(id, val [, val2, ... val15])。

争论 描述
身份证 要添加到的队列的ID。
价值 要添加到队列中的值。
[vale2, ... val15] 。 要添加到队列中的可选值。

返回。

不适用

例子。

move_queue = ds_queue_create();
ds_queue_enqueue(move_queue, x + 200);
ds_queue_enqueue(move_queue, y) ;
ds_queue_enqueue(move_queue, x + 200);
ds_queue_enqueue(move_queue, y + 200);
ds_queue_enqueue(move_queue, x) ;
ds_queue_enqueue(move_queue, y + 200);
ds_queue_enqueue(move_queue, x);
ds_queue_enqueue(move_queue, y) 。

上面的代码创建了一个新的DS队列，并将其索引存储在变量 "move_queue "中。然后，它把一些值推到队列中，以供将来使用。
