ds_queue_head

这个函数将只读取队列中的第一个值（即 "头部 "的值）。它不会对该值进行去queue，这意味着它仍然可以在将来被这个函数或 ds_queue_dequeue(). 如果队列是空的，那么该函数将返回常数undefined，否则它将返回队列中包含的实数或字符串值。

语法。

ds_queue_head(id)。

争论 描述
身份证 要读取的数据结构的ID。

返回。

存储在队列中的数据类型值

例子。

num = ds_queue_head(control_queue)。

上述代码将从变量 "control_queue "中索引的队列中读取数值，并将返回值存储在变量 "num "中。
