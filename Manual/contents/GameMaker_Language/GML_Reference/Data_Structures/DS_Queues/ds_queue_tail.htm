ds_queue_tail

这个函数只读取队列的最后一个值（即 "在尾部 "的值）。它不会对该值进行去queue，这意味着它在未来仍然可以被这个函数或 ds_queue_dequeue().

语法。

ds_queue_tail(id)。

争论 描述
身份证 要读取的数据结构的ID。

返回。

存储在队列中的数据类型值

例子。

num = ds_queue_tail(control_queue)。

上述代码将从变量 "control_queue "中索引的队列中读取数值，并将返回值存储在变量 "num "中。
