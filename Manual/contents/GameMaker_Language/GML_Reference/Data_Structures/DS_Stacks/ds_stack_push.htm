ds_stack_push

这个函数将把一个可以是任何数据类型的值推到栈顶。这个函数可以接受另外14个可选的参数（总共有15个可能的增加值），允许你在一次调用中连续推送多个值到堆栈中。 到堆栈中。

语法。

ds_stack_push(id, val [, val2, ... val15])。

争论 描述
身份证 要推送到的数据结构的ID。
价值 要推入堆栈的值。
[val2, ... val13]。 要添加到堆栈的可选值。

返回。

不适用

例子。

move_stack = ds_stack_create();
ds_stack_push(move_stack, x, y, x, y + 200, x + 200, y + 200, x +200, y);

上面的代码创建了一个新的DS堆栈，并将其索引存储在变量 "move_stack "中。然后，它把一些数值推到堆栈上，以供将来使用。
