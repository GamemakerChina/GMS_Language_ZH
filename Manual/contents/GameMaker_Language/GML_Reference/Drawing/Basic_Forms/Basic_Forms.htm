基本形式

GameMaker Studio 2有一大堆函数可用于绘制不同的形式和形状。这些函数只能在一个实例的绘制事件中使用（你可以在其他地方使用它们，但你不会看到任何东西！），并创建纯粹的图形效果。 在其他地方使用，但你不会看到任何东西！），并创建纯粹的图形效果，这意味着它们不会产生碰撞事件，因为那些只能通过分配给实例的精灵或碰撞掩码来完成。所有这些函数 都会受到当前绘制颜色、阿尔法和混合模式的影响（见 "颜色 "部分）。

重要的是!线条、点和表格总是以1:1的屏幕分辨率绘制，无论它们是在什么情况下绘制的。

重要的是!这些函数会破坏顶点批处理，并增加你游戏中纹理交换的数量，所以在一个绘图帧中多次调用其中的任何一个函数都可能导致性能问题。

以下是用于绘制基本形状的绘图函数。请注意，这些函数的调用速度很慢，只作为基本绘图工具提供。

  • 绘制_箭头
  • 绘制圆圈
  • 画圆_颜色
  • 绘制椭圆
  • 绘制椭圆的颜色
  • 画线
  • 画线_颜色
  • 绘制线宽
  • 绘制线宽_颜色
  • 绘制点
  • 绘画_点_颜色
  • 绘制矩形
  • 绘制_矩形_颜色
  • 绘制圆角
  • 绘画_圆角_颜色
  • 绘制圆弧曲线
  • 绘画_圆角_颜色_文本
  • 绘制三角形
  • 绘制三角形的颜色

GameMaker Studio 2中还包括以下函数，用于设置绘图精度和绘制一些非常特殊的东西。

  • draw_set_circle_precision
  • 绘制按钮
  • 绘制健康栏
  • 绘制路径
