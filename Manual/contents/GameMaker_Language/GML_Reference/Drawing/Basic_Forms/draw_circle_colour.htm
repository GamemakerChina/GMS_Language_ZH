画圆_颜色

通过这个函数，你可以画一个圆的轮廓或一个填充的圆，如果是填充的，你可以定义内部和外部的填充颜色。如果这些颜色不一样，你会得到一个从一个到另一个的渐变效果，并且 的颜色设置将优先于用函数设置的基本颜色。 draw_set_colour().你可以用函数定义绘图的精确程度 draw_set_circle_precision().

注意：如果你想用着色器来画一个形状，你应该知道，大多数着色器都希望有以下输入：顶点、纹理、颜色。然而，当使用这个函数时，只有顶点和颜色数据被传入，所以 着色器可能不会画出任何东西（或者画出一些东西但不正确）。如果你需要以这种方式绘制形状，那么着色器的定制应该考虑到这一点。

语法。

draw_circle_colour(x, y, r, col1, col2, outline)。

争论 描述
x 圆心的X坐标。
y 圆心的Y坐标。
r 圆的半径（从中心到边缘的距离），单位是像素。
col1 圆心处的颜色。
col2 圆的边缘的颜色。
纲要 圆是否是一个轮廓（true）或不是（false）。如果是真的，col1是不相关的。

返回。

不适用

例子。

draw_circle_colour(x, y, 100, c_white, c_black, false)。

这将画一个填充的圆，其中心位于执行实例的x和y位置，半径为100像素，从中心的白色到外部的黑色。
