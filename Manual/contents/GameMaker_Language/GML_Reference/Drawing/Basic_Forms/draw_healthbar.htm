绘制健康栏

通过这个函数，你可以画一个彩色条来显示一个常量值。虽然这个函数使用了 "健康条 "这个词，但你可以用它来显示任何你想要的东西，只要显示的量是一个介于0和100之间的百分比值（任何值的百分比可以用公式（CURRENT_Value / MAXIMUM_value）*100计算）。 100（任何数值的百分比都可以用公式（CURRENT_Value / MAXIMUM_value）*100计算），因此，例如，你可以用它来显示力量、健康、法力、时间或任何其他可以从条形显示中受益的东西。

你可以设置各种不同的东西来改变条形图的视觉效果，如下图所示（注意，这里使用的颜色是c_green代表最大值，c_red代表最小值，但你可以使用 但你可以使用你希望的颜色）。)

注意：如果你想用着色器来画一个形状，你应该知道，大多数着色器都希望有以下输入：顶点、纹理、颜色。然而，当使用这个函数时，只有顶点和颜色数据被传入，所以 着色器可能不会画出任何东西（或者画出一些东西但不正确）。如果你需要以这种方式绘制形状，那么着色器的定制应该考虑到这一点。

语法。

draw_healthbar(x1, y1, x2, y2, amount, backcol, mincol, maxcol, direction, showback, showborder)。

争论 描述
x1 健康条左边的X坐标
y1 健康条顶部的y坐标
x2 健康条右侧的X坐标
y2 健康条底部的y坐标
数量 定义总健康状况的变量（在0和100之间）。
溯源 栏的背景颜色
岷江 无健康状态下的条形图的颜色 (0)
ǞǞǞ 全身健康时的颜色(100)
方向 条形图被 "锚定 "的位置（0=左，1=右，2=上，3=下）。
回放 是否绘制酒吧的背景（true）或不绘制（false）。如果是false，backcol参数就被忽略了。
显示边界 栏的元素是否有1px宽的黑边（真）或没有（假）。

返回。

不适用

例子。

var pc;
pc = (hp / max_hp) * 100;
draw_healthbar(100, 100, 500, 200, pc, c_black, c_red, c_lime, 0, true, true)

上面的代码使用你从变量 "hp "和 "hp_max "得到的百分比值来绘制一个标准的红/绿健康条。
