画线

通过这个函数，你可以在游戏室的任何两点之间画一条1像素宽的线。请注意，由于不同平台的差异，所画的线可能需要不同的值（x，y上的+/-1）才能画出所需的尺寸。 支持的平台不同。

注意：如果你想用着色器来画一个形状，你应该知道，大多数着色器都希望有以下输入：顶点、纹理、颜色。然而，当使用这个函数时，只有顶点和颜色数据被传入，所以 着色器可能不会画出任何东西（或者画出一些东西但不正确）。如果你需要以这种方式绘制形状，那么着色器的定制应该考虑到这一点。

语法。

draw_line(x1, y1, x2, y2);

争论 描述
x1 线条起点的X坐标。
y1 线条起点的y坐标。
x2 线条末端的X坐标。
y2 线条末端的y坐标。

返回。

不适用

例子。

draw_set_colour(c_lime);
draw_line(50,50,150,50);

这将从点（50,50）到点（150,50）画一条浅绿色的水平线。
