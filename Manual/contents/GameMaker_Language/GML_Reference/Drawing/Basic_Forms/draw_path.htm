绘制路径

通过这个函数，你可以让GameMaker Studio 2在屏幕上绘制一个路径。路径将被画成一条简单的线，可以是相对于调用实例的，也可以是在路径编辑器中创建的绝对位置或通过 代码创建的绝对位置。这个功能在调试动态路径时非常有用（例如，那些为实例创建的 mp_grid_path()函数创建的）。)

注意：如果你想用着色器来画一个形状，你应该知道，大多数着色器都希望有以下输入：顶点、纹理、颜色。然而，当使用这个函数时，只有顶点和颜色数据被传入，所以 着色器可能不会画出任何东西（或者画出一些东西但不正确）。如果你需要以这种方式绘制形状，那么着色器的定制应该考虑到这一点。

语法。

draw_path(path, x, y, absolute)。

争论 描述
路 绘制的路径
x 绘制路径的X坐标
y 绘制路径的y坐标
绝对的 是在绝对位置（true）还是在相对位置（false）绘制路径。

返回。

不适用

例子。

if mp_grid_path(grid, path, x, y, obj_Player.x, obj_Player.y, 1)
{
draw_path(path, x, y, false);
}

上面的代码将使用mp_grid_path函数来生成一个路径，并将其存储在变量 "path "中。如果路径被成功创建，它就会在屏幕上以相对于运行代码的实例的位置绘制出来。
