绘画_点_颜色

通过这个函数，你可以在屏幕的任何地方用你定义的颜色绘制一个单一的像素。该颜色设置将覆盖用函数设置的基本颜色 draw_set_colour().

注意：如果你想用着色器来画一个形状，你应该知道，大多数着色器都希望有以下输入：顶点、纹理、颜色。然而，当使用这个函数时，只有顶点和颜色数据被传入，所以 着色器可能不会画出任何东西（或者画出一些东西但不正确）。如果你需要以这种方式绘制形状，那么着色器的定制应该考虑到这一点。

语法。

draw_point_colour(x, y, col1);

争论 描述
x 该点的X坐标。
y 该点的Y坐标。
col1 点的颜色。

返回。

不适用

例子。

draw_point_colour(50, 50, c_red)。

这将在(50,50)处画一个红色像素。
