绘制矩形

通过这个函数，你可以画出一个矩形的轮廓或一个填充的矩形，其中(x1,y1)位置是左上角，(x2,y2)位置是右下角。请注意，被绘制的矩形可能需要不同的 值（在x、y或宽度或高度上的+/-1）来绘制所需的尺寸，因为不同的支持平台之间存在差异。

注意：如果你想用着色器来画一个形状，你应该知道，大多数着色器都希望有以下输入：顶点、纹理、颜色。然而，当使用这个函数时，只有顶点和颜色数据被传入，所以 着色器可能不会画出任何东西（或者画出一些东西但不正确）。如果你需要以这种方式绘制形状，那么着色器的定制应该考虑到这一点。

语法。

draw_rectangle(x1, y1, x2, y2, outline);

争论 描述
x1 矩形的左上角的X坐标。
y1 矩形的左上角的y坐标。
x2 矩形的右下角的X坐标。
y2 矩形右下角的y坐标。
纲要 矩形是否被填充（false）或作为一个像素宽的轮廓（true）绘制。

返回。

不适用

例子。

draw_set_colour（c_yellow）;
draw_rectangle（100, 100, 300, 200, true）。

这将画出一个矩形轮廓，其左上角在（100,100），右下角在（300,200）。
