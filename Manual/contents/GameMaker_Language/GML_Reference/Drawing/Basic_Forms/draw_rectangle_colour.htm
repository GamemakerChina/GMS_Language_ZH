绘制_矩形_颜色

通过这个函数，你可以画一个矩形的轮廓，也可以通过定义一个区域来画一个填充的矩形，其中(x1,y1)位置是左上角，(x2,y2)位置是右下角。如果它是填充的，你可以定义每个角点的单独 如果这些颜色不一样，你将得到一个从一个到另一个的渐变效果（颜色设置将覆盖用函数设置的基本颜色 draw_set_colour()). 请注意，由于各种支持的平台之间的差异，被绘制的矩形可能需要不同的值（在x、y、或宽度或高度上的+/-1）来绘制所需的尺寸。

注意：如果你想用着色器来画一个形状，你应该知道，大多数着色器都希望有以下输入：顶点、纹理、颜色。然而，当使用这个函数时，只有顶点和颜色数据被传入，所以 着色器可能不会画出任何东西（或者画出一些东西但不正确）。如果你需要以这种方式绘制形状，那么着色器的定制应该考虑到这一点。

语法。

draw_rectangle_colour(x1, y1, x2, y2, col1, col2, col3, col4, outline) 。

争论 描述
x1 矩形左边的X坐标。
y1 矩形顶部的y坐标。
x2 矩形右侧的X坐标。
y2 矩形底部的y坐标。
col1 左上角的颜色。
col2 右上角的颜色。
col3 右下角的颜色。
col4 左下角的颜色。
纲要 矩形是否为轮廓（真）或填充（假）。

返回。

不适用

例子。

draw_rectangle_colour(50, 50, 200, 200, c_black, c_white, c_black, c_white, false) 。

这将画出一个填充的正方形，四角之间有一个平滑的黑/白渐变。
