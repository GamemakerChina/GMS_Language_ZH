绘画_圆角_颜色_文本

通过这个函数，你可以画一个圆角矩形的轮廓，或者画一个填充的圆角矩形，其中（x1,y1）的位置是左上角，（x2,y2）的位置是右下角。如果该矩形是填充的，那么颜色 参数将被用来生成一个从中心到边缘的颜色梯度，其中颜色1是中心色，颜色2是边缘色。你还必须提供X和Y轴的半径值（单位：像素），角将被四舍五入。 这些数值。你可以用函数定义拐角的精确程度。 draw_set_circle_precision().请注意，被绘制的矩形可能需要不同的值（+/-1 在x、y或宽度或高度上），才能画出所需的尺寸，因为各种支持的平台之间存在差异。

注意：如果你想用着色器来画一个形状，你应该知道，大多数着色器都希望有以下输入：顶点、纹理、颜色。然而，当使用这个函数时，只有顶点和颜色数据被传入，所以 着色器可能不会画出任何东西（或者画出一些东西但不正确）。如果你需要以这种方式绘制形状，那么着色器的定制应该考虑到这一点。

语法。

draw_roundrect_colour_ext(x1, y1, x2, y2, xrad, yrad, col1, col2, outline)。

争论 描述
x1 圆角矩形左边的X坐标。
y1 圆角矩形顶部的y坐标。
x2 圆角矩形右边的X坐标。
y2 圆角矩形底部的y坐标。
xrad 从矩形角沿x轴的曲线半径。
yrad 从矩形角沿Y轴的曲线半径。
col1 中间的颜色。
col2 外边缘的颜色。
纲要 矩形是否为轮廓（真）或填充（假）。

返回。

不适用

例子。

var dist = point_distance(x, y, mouse_x, mouse_y) / 10;
var col = make_colour_hsv(clamp(dist, 0, 255), 255, 255);
draw_roundrect_colour_ext(x - 50, y - 50, x + 50, y + 50, dist, dist, col, c_white, 0) 。

这将绘制一个带有圆角的填充方块，其半径由进行绘制的实例到鼠标的距离定义。这个值也被用来计算中心混合颜色。
