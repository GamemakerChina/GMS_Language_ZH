draw_set_circle_precision

为了优化绘制圆圈时的性能，GameMaker Studio 2基本上绘制了一个多边形，有足够的边，使其看起来是圆形的。然而，根据你需要的圆圈的大小，你可能会发现改变 这个值可以帮助提高游戏的性能，或者使圆圈看起来更好。你可以输入这个函数的精度值是圆形多边形的边数，这个数字可以是4到64之间的任何数字。 但必须是一个能被4整除的数字，默认值为24。

注意：如果你想用着色器来画一个形状，你应该知道，大多数着色器都希望有以下输入：顶点、纹理、颜色。然而，当使用这个函数时，只有顶点和颜色数据被传入，所以 着色器可能不会画出任何东西（或者画出一些东西但不正确）。如果你需要以这种方式绘制形状，那么着色器的定制应该考虑到这一点。

语法。

draw_set_circle_precision( precision)。

争论 描述
精度 调用此函数后绘制的所有圆的精度。

返回。

不适用

例子。

draw_set_circle_precision(12)。

上面的代码将使用12边的多边形绘制所有的圆。
