绘制三角形

用这个函数，你可以画一个三角形的轮廓，也可以画一个填充的三角形。

注意：如果你想用着色器来画一个形状，你应该知道，大多数着色器都希望有以下输入：顶点、纹理、颜色。然而，当使用这个函数时，只有顶点和颜色数据被传入，所以 着色器可能不会画出任何东西（或者画出一些东西但不正确）。如果你需要以这种方式绘制形状，那么在定制着色器的时候就应该考虑到这一点。

语法。

draw_triangle(x1, y1, x2, y2, x3, y3, outline)。

争论 描述
x1 三角形的第一个角的X坐标。
y1 三角形的第一个角的y坐标。
x2 三角形第二个角的X坐标。
y2 三角形第二个角的y坐标。
x3 三角形第三个角的X坐标。
y3 三角形第三个角的y坐标。
纲要 三角形是否被填充（false）或作为一个像素宽的轮廓（true）绘制。

返回。

不适用

例子。

draw_set_colour(c_aqua);
draw_triangle(50, 50, 200, 50, 200, 0);

这将画出一个充满水蓝色的等腰直角三角形，它的第一个角在（50,50），第二个角在（200,50），第三个角在（50,200）。
