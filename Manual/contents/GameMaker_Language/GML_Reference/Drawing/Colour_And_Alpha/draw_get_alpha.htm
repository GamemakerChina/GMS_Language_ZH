绘画_获取阿尔法（draw_get_alpha

该函数返回绘制alpha的当前值，其范围在0（完全透明）和1（完全不透明）之间。绘制alpha会影响所有绘制函数的透明度，可以用 draw_set_alpha()函数来设置。

语法。

draw_get_alpha()

返回。

真实的

例子。

如果 draw_get_alpha() != 1
{
draw_set_alpha(1);
}

上面的代码首先检查当前绘制的alpha，然后如果它没有被设置为1，就将其设置为1。
