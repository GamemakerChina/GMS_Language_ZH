draw_set_alpha

通过这个函数，你可以设置游戏的基本绘制alpha。这个值可以从0到1设置，0是完全透明的，1是完全不透明的（默认值），它将影响所有进一步的绘制，包括背景、精灵、字体。 基元和3D。

请注意，如果你正在绘制任何资产，并作为函数的一部分提供了一个alpha参数（例如，当使用 draw_sprite_ext())，那么这里设置的alpha值将被 覆盖，所以使用draw_sprite_ext()绘制一个alpha参数为1的精灵时，不管用draw_set_alpha()设置的alpha值是多少，都会使精灵完全不透明（这在使用 draw_self() 和设置 image_alpha的时候也是如此）。)还要注意的是，当一个实例是默认绘制时（即：它有一个精灵但没有绘制事件），那么 image_alpha变量将被钳制在draw_set_alpha()设置的alpha值上，这样，image_alpha值为1，draw alpha值为0.5，将以alpha值0.5绘制精灵，而image_alpha值为0.25，draw alpha值为 将以0.25的阿尔法值绘制精灵，而image_alpha为0.25，draw alpha为0.5，将以0.25的阿尔法值绘制精灵。

语法。

draw_set_alpha(alpha)。

争论 描述
阿尔法 要设置的alpha（在0和1之间）。

返回。

不适用

例子。

draw_set_alpha(0.5);
draw_set_colour(c_black);
draw_text(x+5, y+5, "LEVEL 1");
draw_set_alpha(1);
draw_set_colour(c_white);
draw_text(x, y, "LEVEL 1");

上面的代码将在指定的位置绘制一些文字，并通过修改绘制的alpha和颜色来创建阴影效果。
