绘画_设置颜色

通过这个函数，你可以设置游戏的基本绘图颜色。这个值会影响到所有合适的进一步绘制，包括字体、表格、基元和3D。如果任何这些资产在绘制时改变了它们自己的颜色值，这个值 将被忽略。

语法。

draw_set_colour(col)。

争论 描述
殖民地 绘制时要设置的颜色。

返回。

不适用

例子。

draw_set_alpha(0.5);
draw_set_colour(c_black);
draw_text(x+5, y+5, "LEVEL 1");
draw_set_alpha(1);
draw_set_colour(c_white);
draw_text(x, y, "LEVEL 1");

上面的代码将在指定的位置绘制一些文字，并通过修改绘制的alpha和颜色创建阴影效果。
