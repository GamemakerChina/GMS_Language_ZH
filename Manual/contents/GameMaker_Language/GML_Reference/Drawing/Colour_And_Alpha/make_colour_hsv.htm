make_colour_hsv

GameMaker Studio 2提供了这个功能（以及其他功能），允许用户制作自己的颜色。这个特殊的功能需要三个组成部分：色调、饱和度和值（也称为 "亮度"）。 来创造所需的颜色。这些值在0和255之间，所以你可以用它来制作16,777,216(256*256*256)种颜色!下面你可以看到这些组件分离后的图像。

左边的图片是功能的各个组成部分的分解，然后右边是一个说明 改变这些组件如何影响最终的颜色。正如你所看到的，即使不改变色相（它保持在0，红色），我们也可以通过其他两个设置极大地改变颜色。 到纯色(255)，而数值则使它从较深的(0)到较浅的(255)。

语法。

make_colour_hsv(hue, sat, val)。

争论 描述
色调 颜色的色调
坐 颜色的饱和度如何
价值 颜色有多深

返回。

整数

例子。

col = make_colour_hsv(100, 145, 255);

上面的代码使用该函数来创建一个颜色，并将其值存储在变量 "col "中，以便以后使用。
