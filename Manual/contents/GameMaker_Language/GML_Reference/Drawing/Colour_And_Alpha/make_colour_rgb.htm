make_colour_rgb

GameMaker Studio 2提供了这个功能（以及其他功能），允许用户制作自己的颜色。这个特殊的函数需要三个组成部分，即红色、绿色和蓝色的颜色组成部分。 你想制作的颜色。这些值在0和255之间，所以你可以用它来制作16,777,216(256*256*256)种颜色!下面你可以看到这些成分分离后的图像。

左边的图片是功能的各个组成部分的分解，然后右边是一个说明 改变这些组件如何影响最终的颜色。

语法。

make_colour_rgb(red, green, blue)。

争论 描述
红色 颜色中的红色成分
绿色 颜色中的绿色成分
蓝色的 颜色中的蓝色成分

返回。

沟通

例子。

col = make_colour_rgb(100, 145, 255);

上面的代码使用该函数来创建一个颜色，并将其值存储在变量 "col "中，以便以后使用。
