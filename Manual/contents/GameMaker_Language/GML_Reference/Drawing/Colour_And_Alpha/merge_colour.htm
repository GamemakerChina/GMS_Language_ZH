合并颜色

通过这个函数，你可以取两种颜色，然后把它们合并在一起，形成一种新的颜色。可以通过改变 "量 "参数来定义每种颜色的分量，值为0将返回第一种颜色（col1）。 值为1将返回第二种颜色（col2），介于两者之间的值将返回相应的混合物。例如，一个0.5的值将平均混合两种颜色。下面的图片说明了如何将红色和蓝色合并在一起。 蓝色合并在一起。

语法。

merge_colour(col1, col2, amount)。

争论 描述
col1 第一个合并的颜色
col2 第二种颜色的合并
数量 每种颜色应该有多少被合并。例如，0将返回col1，1将返回col2，而0.5将返回两种颜色的平均合并。

返回。

沟通

例子。

col = merge_colour(c_lime, c_orange, 0.5);

上面的代码使用该函数通过合并石灰和橙色50/50来创建一个颜色，然后将其值存储在变量 "col "中供以后使用。
