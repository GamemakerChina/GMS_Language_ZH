gpu_get_alphatestenable

通过这个函数，你可以检查是否启用了alpha测试（返回true）或没有（返回false）。关于阿尔法测试的更多信息，见函数 gpu_set_alphatestref().

语法。

gpu_get_alphatestenable()。

返回。

布尔型

例子。

if !gpu_get_alphatestenable()
{
gpu_set_alphatestenable(true);
gpu_set_alphatestref(128);
}

上面的代码将检查是否启用了alpha测试，如果没有，它将开启alpha测试，并将测试阈值设置为128（只有alpha超过0.5的像素才会被绘制）。
