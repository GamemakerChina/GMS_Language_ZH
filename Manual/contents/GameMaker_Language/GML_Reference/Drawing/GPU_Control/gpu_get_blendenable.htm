gpu_get_blendenable

这个函数可以用来检索alpha-blending的状态。如果它返回true，那么alpha-blending是启用的，如果它返回false，则是禁用的。默认情况下，它是打开的，因此该函数将返回true。

语法。

gpu_get_blendenable()。

返回。

布尔型

例子。

如果gpu_get_blendenable() == false
{
gpu_set_blendenable(true);
}

上面的代码检查阿尔法混合的状态，如果它被禁用，就会再次启用。
