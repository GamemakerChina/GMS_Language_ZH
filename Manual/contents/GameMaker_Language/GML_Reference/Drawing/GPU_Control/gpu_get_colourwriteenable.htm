gpu_get_colourwriteenable

这个函数可以用来检索当前颜色的可写值。该函数返回一个4个元素的一维数组，其中有以下元素，这些元素将是真（启用）或假（禁用）。在默认情况下，所有的颜色 写入都被设置为真。

  • [0] = 红色通道启用/禁用
  • [1] = 绿色通道启用/禁用
  • [2] = 启用/停用蓝色通道
  • [3] = 启用/停用Alpha通道

语法。

gpu_get_colorwriteenable()。

返回。

4个元素的一维阵列（详见上文）。

例子。

var cw = gpu_get_colorwriteenable();
cw[3] = false;
gpu_set_colorwriteenable(cw);

上面的代码得到了当前的颜色写入值，然后将alpha组件设置为false。
