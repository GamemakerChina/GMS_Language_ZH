gpu_get_texfilter

通过这个函数，你可以检查是否启用了纹理过滤（线性插值）（返回true）或没有（返回false）。关于纹理过滤的更多信息，见函数 gpu_set_texfilter().

注意：在HTML5目标上，这个功能只有在启用WebGL的情况下才能发挥作用。

语法。

gpu_get_texfilter()。

返回。

布尔型

例子。

如果gpu_get_texfilter()
{
gpu_set_texfilter(false);
}
else
{
gpu_set_texfilter(true);
}

上述代码检查纹理过滤是否开启或关闭，并进行相应的切换。
