gpu_get_zwriteenable

该函数可用于检索是否启用了z-writing（该函数返回true）或未启用（该函数返回false）。默认值是启用了z-writing，所以该函数将返回true。

语法。

gpu_get_zwriteenable()。

返回。

布尔型

例子。

如果gpu_get_zwriteenable() == false
{
gpu_set_zwriteenable(true)。
}

上面的代码检查是否启用了z-writing，如果禁用了，则再次启用。
