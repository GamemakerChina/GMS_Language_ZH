gpu_set_alphatestenable

这个功能将为你的游戏启用或禁用阿尔法测试（默认情况下这是禁用的）。通过开启阿尔法测试，你可以使用配套的功能 gpu_set_alphatestref()来设置 "截止 "值。 下面的图片显示了开启或关闭阿尔法测试时可以看到的差异。

注意：该功能可能对iOS和Android设备的性能产生负面影响。

语法。

gpu_set_alphatestenable(enable)。

争论 描述
启用 启用或禁用alpha测试（true/false）。

返回。

不适用

例子。

if !gpu_get_alphatestenable()
{
gpu_set_alphatestenable(true);
gpu_set_alphatestref(128);
}

上面的代码将检查是否启用了alpha测试，如果没有，它将开启alpha测试，并将测试阈值设置为128（只有alpha超过0.5的像素才会被绘制）。
