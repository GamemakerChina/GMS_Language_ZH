gpu_set_alphatestref

这个函数将设置阿尔法测试的参考值（当它已经用 gpu_set_alphatestenable()).这是一个 "截止 "阈值，在这个阈值上，带有阿尔法的像素 的像素将不会被绘制。

基本上，如果你把这个值设置为0，并且你打开了线性插值（无论是在游戏选项中还是用 gpu_set_texfilter())，你仍然可以在重叠对象的边缘得到视觉错误。 对象的边缘出现视觉错误。这是因为你可以有几乎完全透明的区域，阻挡在它们后面绘制的东西，但如果你把这个参考值设置为，例如254（即比最大值低一个），这保证只有100%不透明的像素会被绘制出来 将被绘制。你也可以关闭纹理插值，这通常允许你使用默认值，没有任何问题，根本不需要使用这个函数，然而这在很大程度上取决于源艺术作品。

注意：这是一个高级绘图功能，所以如果你不确定你是否需要它，或者不知道它的具体作用，你可能最好不要使用它，因为它可能会严重破坏你的游戏的绘图方式。

语法。

gpu_set_alphatestref(val)。

争论 描述
价值 在0到255之间设置阿尔法测试的参考值（默认值为0

返回。

不适用

例子。

if !gpu_get_alphatestenable()
{
gpu_set_alphatestenable(true);
gpu_set_alphatestref(128);
}

上面的代码将检查是否启用了alpha测试，如果没有，它将开启alpha测试，并将测试阈值设置为128（只有alpha超过0.5的像素才会被绘制）。
