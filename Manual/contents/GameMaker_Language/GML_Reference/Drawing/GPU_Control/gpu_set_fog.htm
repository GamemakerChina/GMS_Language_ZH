gpu_set_fog

这个功能可以用来启用或禁用绘制雾气。在3D游戏中，雾可以用来使远处的实例看起来模糊甚至消失，这有助于营造气氛，也可以掩盖你没有绘制远处的实例的事实 远处的实例。你可以设置它是启用（true）还是禁用（false），雾应该使用什么颜色来混合，以及开始和结束的绘制距离。请注意，雾的起始点表示相对于 视图摄像机的半径，而雾的终点则表示在哪个半径（也是相对的）上实例将完全不可见。

该函数可以接受四个单独的参数（上面给出的）或一个具有以下结构的数组（下面的示例代码显示了这种方法）。

  • [0] = 启用的切换（一个布尔值，要么是真，要么是假），默认为假。
  • [1] = 颜色（实际），默认为c_black
  • [2] = 启动距离（实数），默认为0
  • [3] = 末端距离（实数），默认为1

语法。

gpu_set_fog(enable, color, start, end)。

争论 描述
启用 启用或禁用雾气
颜色 雾的颜色
开始 开始应用雾化的距离
结束 结束雾霾的距离

返回。

不适用

例子。

var fog_a = gpu_get_fog();
fog_a[1] = c_red;
gpu_set_fog(fog_a);

上面的代码得到了当前的雾气设置，然后将数组中的颜色元素设置为c_red，然后使用改变后的数组再次设置雾气。
