gpu_set_texrepeat

这个函数可以用来向GameMaker Studio 2指示，当用于模型和基元时，纹理是否应该重复（true）或不重复（false）。通常，纹理的宽度和高度在0-1的范围内，但如果你指定的 大于 1 的值，其余部分就不会被画出来（整个纹理将被画出一个 "空 "的部分，与使用的 1 的百分比相对应。因此，纹理宽度或高度的值为2时，将在一半的空间内绘制纹理，而留下另一半空间。 纹理宽度或高度的值为2，就会在一半的空间内绘制纹理，而另一半则是空的）。)然而，通过使用这个函数将重复设置为真，纹理会在所有模型和基元上自动重复。这些函数可能的用例是在三维中重复一个纹理，但是 为了让它发挥作用，并且不从纹理页的其他部分拉取图像，需要在 Sprite Editor 中将所使用的精灵标记为在 "独立的纹理页"。

注意：精灵和背景的绘制总是不重复的，所以一旦你绘制了一个精灵或背景，这个值就会重置为false。这个默认行为不会影响纹理阶段的设置（即：调用该函数会，但是 内部设置为false则不会）。)

注意：这个设置将覆盖在使用函数时为着色器设置的任何纹理阶段重复。 gpu_set_texrepeat_ext().

语法。

gpu_set_texrepeat(enable)。

争论 描述
启用 启用或禁用纹理过滤（true/false）。

返回。

不适用

例子。

如果gpu_get_texrepeat()
{
gpu_set_texrepeat(false);
}
else
{
gpu_set_texrepeat(true);
}

上面的代码检查纹理重复是否打开或关闭，并进行相应的切换。
