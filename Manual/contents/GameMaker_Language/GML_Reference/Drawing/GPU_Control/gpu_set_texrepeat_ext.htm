gpu_set_texrepeat_ext

当在GameMaker Studio 2中使用这个函数时，可以用来设置单个采样器 "槽 "是否重复给定的纹理。 着色器在GameMaker Studio 2中，可以用来设置单个采样器 "槽 "是否重复给定的纹理。设置为"true"将重复 如果uv坐标不在0-1范围内，设置为 "true "将重复纹理，而设置为 "false"将意味着不重复。这些函数可能的用途是在3D中重复纹理，但为了让它发挥作用，而不是从纹理页面的其他部分拉出图像 纹理页中的图像，所使用的精灵将需要在精灵编辑器中被标记为 "独立的纹理页"。

注意: 这个设置将被调用函数时设置的值所覆盖 gpu_set_texrepeat().

语法。

gpu_set_texrepeat_ext(sampler_id, enable)。

争论 描述
sampler_id 着色器的取样器ID。
启用 启用或禁用纹理过滤（true/false）。

返回。

不适用

例子。

var s_tex =shader_get_sampler_index(shader_glass, "s_NoiseSampler");
if gpu_get_filter_ext(s_tex)
{
gpu_set_texfilter_ext(s_tex, false);
}
else
{
gpu_set_texfilter_ext(s_tex, true);
}

上述代码检查特定采样器ID（存储在一个局部变量中）的纹理过滤是否开启或关闭，并相应地进行切换。
