gpu_set_zfunc

这个函数可以用来设置z-buffer测试比较模式（见 gpu_set_ztestenable()获取更多信息）。)可使用的值是以下任何一个常数（默认值为 是cmpfunc_lessequal）。)

  • cmpfunc_never
  • cmpfunc_less
  • cmpfunc_equal
  • cmpfunc_lessequal
  • cmpfunc_greater
  • cmpfunc_notequal
  • cmpfunc_greaterequal
  • cmpfunc_always

语法。

gpu_set_zfunc(cmp_func)。

争论 描述
cmp_func 使用的比较模式（见上面的列表）。

返回。

不适用

例子。

gpu_set_ztestenable（true）;
gpu_set_zfunc（cmpfunc_always）。
draw_sprite(spr_Background, 0, 0, 0);
gpu_set_ztestenable(false);

上面的代码在绘制背景精灵之前开启了z-buffer测试，并设置了它的比较模式，然后又将其关闭，继续绘制。
