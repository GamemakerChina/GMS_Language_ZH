gpu_get_tex_max_mip_ext

通过这个函数，你可以得到当前设置的最大mipmap级别，该级别将用于给定的着色器采样器。你需要提供该着色器采样器的索引值（由函数 shader_get_sampler_index(), 该函数将返回当前的最大mipmap级别，其中0是最高分辨率，1是使用第一个mipmap，2是第二个等等。请注意，这对于避免渲染纹理时的出血伪影是非常有用的。 例如，将纹理页边框设置为8px，然后将最大mipmap级别设置为3，将确保你在更大的渲染距离上不会出现任何出血问题。

语法。

gpu_get_tex_max_mip_ext(sampler_index)。

争论 描述
采样器_索引 要获得的着色器采样器的索引

返回。

实数（默认：16

例子。

var _sampleIndex =shader_get_sampler_index(shd_Glass, "s_Background");
var _spriteTex = sprite_get_texture(sprite_index, 0);
shader_set(shd_Glass) 。
if gpu_get_tex_max_mip_ext(_sampleIndex) != 4
{
gpu_set_tex_max_mip_ext(_sampleIndex, 4);
}
texture_set_stage(_sampleIndex , _spriteTex);
shader_reset()。

上面的代码为给定的着色器纹理采样器设置最大mipmap级别为4，如果它还没有被设置为4的话。

返回。Mipmapping
