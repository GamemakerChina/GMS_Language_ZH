gpu_get_tex_mip_bias

用这个函数你可以检索mipmap的偏置值，0等于没有偏置，1等于第一个mipmap，2等于第二个mipmap等等。这控制了mip map交换的速度，通常会使渲染的纹理 一般来说，数值越高，观看的 "距离 "越大，就越模糊。注意，它也可以返回负值，在这种情况下，值越低，渲染的纹理在更大的距离上就越清晰。

语法。

gpu_get_tex_mip_bias()。

返回。

实数（默认：0

例子。

if gpu_get_tex_mip_bias() != 0
{
gpu_set_tex_mip_bias(0);
}

上面的代码将检查当前的mipmap偏差，如果它不是0，就设置为0。

返回。Mipmapping
