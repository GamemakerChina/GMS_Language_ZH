gpu_get_tex_mip_enable_ext

通过这个函数，你可以得到mipmapping是否被关闭，是否对所有东西都打开，或者只对着色器采样器上的Texture Group Manager中选择的纹理组打开。你可以 提供着色器采样器的索引值（如函数所返回的）。 shader_get_sampler_index()而该函数将返回下面列出的常数之一。

语法。

gpu_get_tex_mip_enable_ext(sampler_index)。

争论 描述
采样器_索引 要获得的着色器采样器的索引

返回。

恒定。

恒定 描述
ǞǞǞ Mipmapping被禁用。
姆普_安 所有纹理的Mipmapping都已启用。
mip_markedonly 对于在纹理组选项中启用了Mipmapping的纹理，会启用Mipmapping。

例子。

var _sampleIndex =shader_get_sampler_index(shd_Glass, "s_Background");
var _spriteTex = sprite_get_texture(sprite_index, 0);
shader_set(shd_Glass) 。
if gpu_get_tex_mip_enable_ext(_sampleIndex) != mip_on
{
gpu_set_tex_mip_enable_ext(_sampleIndex, mip_on);
}
texture_set_stage(_sampleIndex , _spriteTex);
shader_reset()。

上面的代码为给定的着色器纹理采样器启用了mipmapping，如果它还没有被启用的话。

返回。Mipmapping
