创作的部分

这个函数必须用来创建一个新的发射器，并将其分配给一个给定的粒子系统。该函数将返回新发射器的索引值，该索引值必须存储在一个变量中，并在所有引用该发射器的其他函数中使用。 当不再使用时，发射器本身必须被销毁，以防止内存泄漏（这可以通过特定的发射器销毁函数或通过销毁发射器所属的整个粒子系统来实现）。

语法。

part_emitter_create(ps)。

争论 描述
诗句 用于创建发射器的粒子系统。

返回。

真实的

例子。

p_emit1 = part_emitter_create(Sname)。

这将创建一个新的粒子发射器，并在变量 "p_emit "中存储其索引。
