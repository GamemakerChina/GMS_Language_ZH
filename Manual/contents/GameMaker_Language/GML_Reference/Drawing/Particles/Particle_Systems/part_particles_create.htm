栏目组_创建

这个功能是那些不需要粒子发射器提供的任何功能的效果的理想选择（例如，从导弹中产生烟雾，或一个简单的爆炸效果），因为 它允许你在游戏室的任何位置快速而轻松地创建粒子。注意，你必须在之前创建了粒子系统和粒子类型，才能使用这个功能。

语法。

part_particles_create(ind, x, y, parttype, number)。

争论 描述
ind 粒子系统的索引。
x 创建粒子的X坐标。
y 创建粒子位置的y坐标。
零件类型 要创建的粒子的索引（类型）。
数 要创建的粒子的数量。

返回。

不适用

例子。

if mouse_check_button(mb_left)
{
part_particles_create(sname, mouse_x, mouse_y, p_CursorEffect, 5);
}

上面的代码检查鼠标按钮是否被按下，如果返回为真，则在鼠标位置生成5个粒子。
