栏目组_创建

这个功能是那些不需要粒子发射器提供的任何功能的效果的理想选择（例如，从导弹中产生烟雾，或一个简单的爆炸效果），因为 它允许你在游戏室的任何位置快速而轻松地创建粒子。你还可以在用这个功能创建粒子时 "即时 "为其上色，这个颜色将超越粒子的预定义颜色。 但它不会在粒子的生命周期内混合这种颜色。注意，你必须事先创建好粒子系统和粒子类型，才能使用这个函数。

语法。

part_particles_create_colour(ind, x, y, parttype, color, number)。

争论 描述
ind 粒子系统的索引。
x 创建粒子的X坐标。
y 创建粒子位置的y坐标。
零件类型 要创建的粒子的索引（类型）。
颜色 颗粒的颜色混合。
数 要创建的粒子的数量。

返回。

不适用

例子。

如果（速度>1）
{
var _c = choose(c_aqua, c_lime, c_fuschia, c_orange);
part_particles_create_colour(name, x, y, p_Smoke, _c, 1);
}

上述代码将在具有该代码的实例的速度大于1的每一步产生一个粒子。这些粒子将是一个随机的颜色。
