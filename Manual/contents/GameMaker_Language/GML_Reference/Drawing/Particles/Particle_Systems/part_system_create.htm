创建系统

这个函数用于创建一个新的粒子系统，并将返回一个唯一的索引号，该索引号应该被储存起来，并用于与该系统有关的所有后续函数。该系统将被分配到一个托管层，并且将被设置为深度为0。 被管理的图层是用户无法访问的，当使用深度而不是图层时，只用于内部管理。通常情况下，你会使用函数 part_system_create_layer()而不是 而不是这个函数。

语法。

part_system_create()。

返回。

真实的

例子。

global.Sname = part_system_create()。

这将创建一个新的粒子系统，并在全局变量 "Sname "中存储索引。
