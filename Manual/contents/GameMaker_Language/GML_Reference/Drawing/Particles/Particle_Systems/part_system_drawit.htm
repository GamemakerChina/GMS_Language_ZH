部分系统_更新

通过这个函数，你可以强制粒子系统在屏幕上绘制当前所有的粒子。如果 part_system_automatic_draw()是关闭的，那么这个函数将显示粒子，当 在一个实例的绘制事件中使用，或者当绘制目标被设置为一个表面时（使用 surface_set_target())来将系统中的粒子绘制到该表面。

语法。

part_system_drawit(ind)。

争论 描述
ind 要绘制的粒子系统的索引。

返回。

不适用

例子。

if surface_exists(surface)
{
surface_set_target(surface);
part_system_drawit(global.psys);
surface_reset_target()。
}

上面的代码检查变量 "surf "中索引的表面是否存在，如果存在，则将绘图目标设置为该表面，以全局变量中存储的ID绘制粒子系统，然后重置绘图 目标，这样所有正常的绘图就会再次出现在屏幕上。
