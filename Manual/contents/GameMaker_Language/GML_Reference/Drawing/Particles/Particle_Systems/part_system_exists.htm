部分系统存在

通过这个函数，你可以检查给定的粒子系统的索引是否存在于游戏中。请注意，如果被检查的变量是一个未初始化的变量（否则粒子系统将被分配其索引），这将会 抛出一个错误。

语法。

part_system_exists(ind)。

争论 描述
ind 要检查的粒子系统的索引。

返回。

布尔型

例子。

if !part_system_exists(global.Sname)
{
global.Sname = part_system_create();
}

上面的代码检查全局变量中引用的粒子系统是否存在，如果不存在就会被创建。
