部分_系统_层

这个函数可以用来将一个粒子系统从它的当前层切换到一个新层。你需要提供粒子系统的ID值（如函数所返回的 part_system_create_layer())，然后再提供 唯一的层ID（由函数返回 layer_create()或作为字符串使用的层的名称--例如 "instance_layer" --在房间编辑器中定义）。 在房间编辑器中定义的），然后系统将被移到新的层。

语法。

part_system_layer(ps, layer)。

争论 描述
诗句 粒子系统ID
层 新的图层ID

返回。

不适用

例子。

如果（part_system_get_layer(global.p_sys) != "effects_layer"）
{
part_system_layer(global.p_sys, "effects_layer";
}

上面的代码将检查一个粒子系统，看它在哪个层上，如果它不在给定的层上，它将被改变。
