部分系统_更新

这个函数将在游戏时间内将给定的粒子系统推进一步，可以用来模拟自动更新关闭时的粒子系统行为，也可以用来将一个粒子系统一步推进到一个特定的点。 例如，如果你有一个雪的粒子系统，你希望从玩家进入房间的那一刻起，雪就开始飘落，那么你可能希望在一个实例的创建事件中，将你的雪粒子提前几秒钟 以便在第一次绘制时，屏幕上到处都是颗粒（下面的代码示例显示了这一点）。当系统的自动更新被函数关闭时，你也可以使用这个函数 part_system_automatic_update() 来推动系统的发展。

语法。

part_system_update(ind)。

争论 描述
ind 要更新的粒子系统的索引。

返回。

不适用

例子。

repeat (room_speed * 3)
{
part_system_update(global.RainSys);
}

上述代码将使全局变量 "RainSys "中索引的粒子系统提前3秒。
