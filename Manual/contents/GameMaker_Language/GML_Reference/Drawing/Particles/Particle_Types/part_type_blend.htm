部分_类型_混合

通过这个函数，你可以让你的粒子以加法混合模式（true）或不加法（false）绘制。加法混合是一种特殊的混合模式，它在每个粒子重叠时增加它们的亮度值，因此，浅色 会逐渐变亮（直到它们看起来像白色），而深色会变得越来越透明，黑色几乎看不见。这个功能可以带来一些漂亮的粒子效果，但不一定需要。比如说 例如，一个烟雾跟踪将关闭加法混合以保持灰色效果，但一个火焰效果可能会打开加法混合以使粒子更透明和更明亮。

语法。

part_type_blend(ind, additive)。

争论 描述
ind 要改变的粒子类型的索引。
添加剂 颗粒是否应进行加法混合（真）或正常混合（假）。

返回。

不适用

例子。

part_type_blend(part_Fire, true)。

上述代码将把所有以 "part_Fire "变量为索引的粒子类型创建的粒子设置为具有加法混合模式。
