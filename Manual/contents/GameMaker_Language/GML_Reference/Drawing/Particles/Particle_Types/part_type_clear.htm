栏目组_type_clear

通过这个函数，你可以 "重置 "一个粒子，将与该粒子有关的每个函数（生命、颜色、阿尔法、方向等......）的所有值返回到它们的默认值。请注意，这个函数不会从屏幕上删除任何 当前在房间里可见的粒子，为此你应该使用 part_particles_clear().

语法。

part_type_clear(ind)。

争论 描述
ind 要清除的粒子类型的索引。

返回。

不适用

例子。

part_type_clear(global.explode_part)。

上述代码将把全局变量 "explode_part "中索引的粒子类型清除为其默认值。
