部分_类型_颜色_混合

通过这个函数，你可以将给定的粒子类型设定为两种颜色的随机混合。

语法。

part_type_colour_mix(ind, color1, color2)。

争论 描述
ind 要改变的粒子类型的索引。
颜色1 混合物的第一种颜色将取自。
颜色2 第二种颜色的混合将取自。

返回。

不适用

例子。

part_type_colour_mix(global.P_Damage, c_red, c_yellow)。

上述代码将把全局变量 "P_Damage "中索引的粒子类型的每个粒子发射的颜色设置为红色和黄色之间的随机混合色。
