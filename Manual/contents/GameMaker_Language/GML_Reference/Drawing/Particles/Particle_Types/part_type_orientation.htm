零件类型方向

这个函数用于确定粒子精灵创建时的方向，也可以用来使粒子的方向在其生命周期内增加或减少。最小和最大的方向值默认为0，但这些 值可以按照GameMaker Studio 2的标准方向进行随机调整，0度为右，90度为上，180度为左，270度为下。如果你把它们设置成相同的值，那么 粒子都会以相同的方向被创建。

你也可以设置一个增量值，它将在方向的有效期内增加（如果是正数）或减少（如果是负数）一定的度数。这个值可以是最小的（+/-）0.01。

你也可以设置 "摆动 "因子，和其他粒子函数一样。这是一个值，它将在粒子寿命的每一步中随机加入或减去其方向。很明显，大的数值比小的数值更明显。 这个值甚至可以是一个负值，最大范围在-20和20之间。

最后，你可以选择是否有相对的方向，这意味着当粒子有一个方向（和速度）时，粒子精灵将围绕这个矢量定向。请记住，如果你将粒子的速度设置为降低 并达到0，缺乏速度会将方向设置为默认值0°，因此相对方向会导致粒子精灵 "跳 "到一个不同的角度。

语法。

part_type_orientation ( ind, ang_min, ang_max, ang_incr, ang_wiggle, ang_relative ) 。

争论 描述
ind 要改变的粒子类型的索引。
ang_min 粒子的最小起始角度。
角度_最大 颗粒的最大起始角度。
ang_incr 角度的增加（如果设置为负数，则减少），以每步的度数计算。
ang_wiggle 角度每一步顺时针/逆时针摆动的量。
角度_相对应的 是否设置其相对于粒子运动方向的角度（真）或不设置（假）。

返回。

不适用

例子。

part_type_shape(particle2, pt_shape_spark);
part_type_size(particle2, 0.10, 0.50, 0.01, 0);
part_type_scale(particle2, 0.30, 0.30);
part_type_colour1(particle2, 8454143);
part_type_alpha1(particle2, 0.50);
part_type_speed(particle2。 4, 4, -0.07, 1);
part_type_direction(particle2, 0, 359, 0, 20);
part_type_orientation(particle2, 0, 359, 0, 20, 1);
part_type_blend(particle2, 1);
part_type_life(particle2, 1, 5);

上述代码将设置各种粒子值，包括方向，这将是0º和359º之间的随机值。每一步也会有一个0到20之间的随机量加入其中，而方向是相对于 运动的方向。
