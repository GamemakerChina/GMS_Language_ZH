零件类型大小

这个函数用来确定粒子在创建时的大小，也可以用来使粒子在其生命周期中增加或减少大小。最小和最大的尺寸值默认为1（意味着尺寸与基本粒子的比例是相同的）。 与基本粒子的比例相同。 part_type_scale()如果它们被设置为相同的值，那么所有粒子的初始尺寸将是相同的，但是如果设置为不同的值，每个 粒子将从该范围内的一个随机大小开始，例如，如果你的最小值是0.5，最大值是2，那么所有的粒子将以基本比例的一半到两倍的大小被创建。

你也可以设置一个增量值，它将使粒子精灵在粒子的生命周期内增长（如果是正数）或缩小（如果是负数），每一步都会将给定的数量加入到粒子的尺寸中。这个值可以是 最小为（+/-）0.01。

最后，你可以设置 "摇摆 "因子。这是一个值，它将在粒子寿命的每一步中随机地添加或减去大小。很明显，大的数值比小的数值更明显，这个数值甚至可以是 负值，最大范围在-20和20之间。

语法。

part_type_size(ind, size_min, size_max, size_incr, size_wiggle)。

争论 描述
ind 要改变的粒子类型的索引。
尺寸_最小 粒子可以开始的最小尺寸。
最大尺寸 粒子可以开始的最大尺寸。
尺寸_incr 粒子每一步应该增加或减少多少。
尺寸_扭动 每一步应该从颗粒大小中随机加入或减去多少。

返回。

不适用

例子。

part_type_shape(particle1, pt_shape_explosion);
part_type_size(particle1, 2, 2, 0.10, 0.01);
part_type_scale(particle1, 1, 1);
part_type_colour2(particle1, 16777215, 12632256);
part_type_alpha3(particle1, 0, 0.20, 0);
part_type_speed(particle1, 0.20, 0.20, 0, 0);
part_type_direction(particle1, 0, 359, 0, 20);
part_type_orientation(particle1, 0, 359, 0, 0, 1);
part_type_blend(particle1, 0);
part_type_life(particle1, 40, 60) 。

上面的代码将设置各种粒子的值，包括在每个粒子被创建时被设置为两倍的大小，然后每一步将大小增加0.01，也有0.1的摇摆。
