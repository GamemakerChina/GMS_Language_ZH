零件类型_速度

这个函数用来确定粒子在创建时的速度，也可以用来使粒子在其生命周期内增加或减少其速度。最小和最大的速度值默认为1，这意味着粒子 将每步移动1个像素，如果它们被设置为相同的值，所有粒子的初始速度将是相同的，但如果设置为不同的值，每个粒子将从给定范围内的随机速度开始。例如，如果你 最小为3，最大为6，那么所有的粒子将以每步3到6个像素的速度被创建。

你还可以设置一个增量值，使粒子在其生命周期内移动得更快（如果是正数）或更慢（如果是负数），每一步都会在粒子速度上增加给定的数量。这个值可以是一个最小的 (+/-)0.01。

最后，你可以设置 "摇摆 "因子。这是一个值，它将在粒子寿命的每一步中随机加入或减去速度。很明显，大的数值比小的数值更明显，这个数值甚至可以是 负值，最大范围在-20和20之间。

语法。

part_type_speed(ind, speed_min, speed_max, speed_incr, speed_wiggle)。

争论 描述
ind 要改变的粒子类型的索引。
速度_min 粒子可以启动的最小速度。
速度_最大值 粒子可以启动的最大速度。
速度_incr 粒子的速度在每一步应该增加或减少多少。
速度_踌躇 每一步应该从粒子的速度中随机添加或减去多少。

返回。

不适用

例子。

part_type_shape(particle2, pt_shape_flare);
part_type_size(particle2, 0.01, 0.50, 0.01, 20);
part_type_scale(particle2, 0.10, 0.10);
part_type_colour3(particle2, 4210816, 4227327, 8240895);
part_type_alpha3(particle2, 0.20, 0.80, 0);
part_type_speed(particle2, 5, 10, -0.20, 20);
part_type_direction(particle2, 0, 359, -1, 20);
part_type_gravity(particle2, 0, 270);
part_type_orientation(particle2, 0, 359, 0, 20, 1);
part_type_blend(particle2, 1);
part_type_life(particle2, 1, 20)。

上述代码将设置各种粒子值，包括设置最小速度为5，最大速度为10的速度，每一步的速度都会以-0.2的数值减慢。速度也将以一个0到20之间的随机值进行摆动 之间的随机值。
