绘制顶点

此函数为一个基元定义了一个顶点的位置。基元的最终外观取决于选择的要绘制的基元类型和向其添加顶点的顺序。参见 draw_primitive_begin()获取更多信息。 更多信息。要结束并绘制基元，您必须调用 draw_primitive_end().

语法。

draw_vertex(x, y)

争论 描述
x 顶点的X坐标。
y 顶点的Y坐标。

返回。

不适用

例子。

draw_primitive_begin(pr_trianglelist);
draw_vertex(100, 100);
draw_vertex(100, 200);
draw_vertex(150, 150) ;
draw_primitive_end();

上面的代码将绘制一个简单的三角形基元。
