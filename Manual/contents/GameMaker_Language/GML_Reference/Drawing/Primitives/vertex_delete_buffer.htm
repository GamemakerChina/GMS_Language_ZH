顶点删除缓冲区（vertex_delete_buffer

这个函数可以用来从系统内存中删除一个先前创建的顶点缓冲区（见 vertex_create_buffer()）从系统内存中删除。这也将删除它所包含的任何自定义基元的所有顶点数据。

语法。

vertex_delete_buffer(buffer)。

争论 描述
缓冲区 要删除的缓冲区。

返回。

不适用

例子。

vertex_delete_buffer(v_buff)。

上述代码将从内存中删除变量 "v_buff "中索引的缓冲区。

返回。基元和顶点格式
