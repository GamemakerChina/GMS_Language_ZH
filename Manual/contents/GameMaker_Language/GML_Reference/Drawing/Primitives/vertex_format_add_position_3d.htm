顶点格式_添加位置_三维

告诉GameMaker Studio 2接受三维位置数据（x、y和z）作为正在创建的新顶点格式的一部分。

语法。

vertex_format_add_position_3d（）。

返回。

不适用

例子。

vertex_format_begin();
vertex_format_add_colour();
vertex_format_add_position_3d();
my_format = vertex_format_end();

上面的代码将创建一个只有颜色和位置值的新顶点格式，然后将格式ID存储在变量 "my_format "中。

返回。基元和顶点格式
