顶点格式的删除

当你使用完任何创建的矢量格式时，必须调用这个函数。你提供格式的ID值（如由函数 vector_format_end())，这个函数将释放 与它相关的内存。注意，如果你在调用这个函数后试图再次使用这个格式，你会得到一个错误。

语法。

vertex_format_delete(formatID)。

返回。

不适用

例子。

vertex_format_delete(myFormat)。

上述代码将从内存中删除在变量 "myFormat "中创建的顶点格式。

返回。基元和顶点格式
