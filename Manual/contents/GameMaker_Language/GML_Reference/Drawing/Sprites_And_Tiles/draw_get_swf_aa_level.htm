draw_get_swf_aa_level

这个函数可以用来获取SWF格式矢量精灵的抗锯齿（AA）水平。返回值介于0和1之间，显示这些精灵的边缘将被绘制的 "平滑 "程度。你可以用这个函数来设置AA级别 draw_set_swf_aa_level().

语法。

draw_get_swf_aa_level()。

返回。

真实的

例子。

如果 draw_get_swf_aa_level() == 0
{
draw_enable_swf_aa(true);
draw_set_swf_aa_level(0.5);
}

上述代码将检查SWF格式精灵的AA值，如果它是0，它将启用AA，并将值设置为0.5。
