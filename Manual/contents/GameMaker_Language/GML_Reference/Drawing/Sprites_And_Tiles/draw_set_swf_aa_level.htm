draw_set_swf_aa_level

这个函数可以用来设置SWF格式矢量精灵的抗锯齿（AA）水平。这可以是一个从0到1的实值，将 "平滑 "这些精灵的边缘。请注意，要看到这种效果，你必须先用 函数 draw_enable_swf_aa().

语法。

draw_set_swf_aa_level（AA）。

争论 描述
AA 要使用的抗锯齿值，从0到1。

返回。

不适用

例子。

如果 draw_get_swf_aa_level() == 0
{
draw_enable_swf_aa(true);
draw_set_swf_aa_level(0.5);
}

上述代码将检查SWF格式精灵的AA值，如果它是0，它将启用AA，并将值设置为0.5。

返回。精灵和瓷砖
