画笔的位置

通过这个函数，你可以在四个角坐标定义的区域内绘制一个变形的精灵。前两个参数是要画的精灵和该精灵的子图像（与函数中的相同 draw_sprite()) 但接下来的参数是定义给定精灵的边界框的四个角的位置的参数。这些坐标应按顺时针顺序给出，所以第一个坐标是左上角，然后是右上角，然后是下角 然后是右下角，最后是左下角。你还可以为精灵的阿尔法值设置一个值，以使其具有透明度。下面的图片说明了这个函数是如何工作的。

警告!上面的图片只是为了说明问题，如果你在精灵上使用这个函数，你会得到不同的结果，而且可能会遇到纹理 "剪切 "的情况，这是因为精灵是由四元组构造而成的。

注意：这个函数只对位图精灵有用，对SWF或JSON（Spine）精灵不起作用。

注意：使用这个功能时，你应该禁用纹理页的自动裁剪设置。

语法。

draw_sprite_pos(sprite, subimg, x1, y1, x2, y2, x3, y3, x4, y4, alpha)。

争论 描述
精灵 要绘制的精灵的索引。
亚img 要绘制的精灵的subimg（帧）（image_index或-1与对象中的当前动画帧相关）。
x1 第一个X坐标。
y1 第一个y坐标。
x2 第二个X坐标。
y2 第二个y坐标。
x3 第三个X坐标。
y3 第三个y坐标。
x4 第四个X坐标。
y4 第四个y坐标。
阿尔法 精灵的alpha（从0到1，其中0是透明的，1是不透明的）。

返回。

不适用

例子。

draw_sprite_pos(sprite_index, image_index, x - 100, y - 50, x - 50, y +150, x + 100, y + 200, x + 100, y, 1)。

上述代码将围绕实例的x/y位置扭曲地绘制与运行代码的实例相关的精灵，并且具有完全不透明的alpha。
