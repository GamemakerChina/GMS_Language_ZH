这个函数将获取一个精灵，然后在整个视图（如果没有定义视图，则为房间）中重复铺设它，从你在函数中给出的坐标开始，每个铺设的精灵都被缩放、调色，并使用你定义的alpha。 (这些属性与函数中使用的属性相同)。 draw_sprite_ext()).这个函数只适用于二维（正交）投影，当使用三维摄像机投影时，将不能正确工作。

注意：只有在启用WebGL时，才建议对HTML5目标进行颜色混合，尽管在未启用WebGL的情况下，你仍然可以设置混合颜色，它将像正常一样混合精灵。然而，所有以这种方式进行的混合都会创建一个重复的 然后存储在缓存中并在需要时使用。这远非最佳状态，如果你使用多种颜色变化，除非你激活WebGL，否则会降低你的游戏性能。 字体缓存的大小，以便在必要时使用该函数来限制它。 sprite_set_cache_size().

语法。

draw_sprite_tiled_ext(sprite, subimg, x, y, xscale, yscale, color, alpha)。

争论 描述
精灵 要绘制的精灵的索引。
亚img 要绘制的精灵的subimg（帧）（image_index或-1与对象中的当前动画帧相关）。
x 绘制精灵的X坐标。
y 绘制精灵位置的y坐标。
xscale 精灵的水平缩放比例。一个乘数 ' 1 = 正常缩放，0.5是一半等等。
yscale 精灵的垂直缩放。一个乘数 ' 1 = 正常缩放，0.5是一半等等。
颜色 混合精灵的颜色。c_white是正常显示。
阿尔法 精灵的alpha（从0到1，其中0是透明的，1是不透明的）。

返回。

不适用

例子。

draw_sprite_tiled_ext(sprite_index, image_index, x, y, 2, 2, c_red, 0.5)。

这将在实例自己的X和Y位置上画出实例指定的精灵（sprite_index）和它当前的动画帧（image_index），但缩放为正常尺寸的两倍，混合为红色，并有正常alpha的一半。该精灵将被平铺在 水平和垂直地铺满整个视图。
