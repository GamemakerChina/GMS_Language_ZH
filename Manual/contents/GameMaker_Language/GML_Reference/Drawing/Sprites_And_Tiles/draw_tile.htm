绘制瓷砖

通过这个函数，你可以从一个瓷砖集中画出任何给定的瓷砖--与旋转、翻转和镜像相竞争--设置框架（如果是动画）和房间内的位置。瓷砖集的ID值是IDE中设置的瓷砖集索引，瓷砖 数据是数据 "blob"，包含所有关于瓷砖索引和已应用的变换的信息。你可以使用函数 tilemap_get() 来获取瓦片数据，然后用瓦片函数来操作，或者你可以用以下瓦片常量和要创建的瓦片的索引值来自己构建瓦片数据 来创建。

恒定 描述
瓷砖_旋转 用于设置/获取瓦片数据blob的旋转位。
瓷砖_镜面 用于设置/获取瓦片数据blob的镜像位。
瓷砖_翻转 用于设置/获取瓦片数据blob的翻转位。
瓷砖_index_mask 一个特殊的常数，用于与瓦片数据blob进行 "和"，以提取瓦片索引。


你可以将这些常数与瓦片索引值进行比特 "或"，以获得最终的瓦片数据。

var tiledata = 8 | tile_rotate | tile_flip;

上面的代码只是创建了一个局部变量来存储瓷砖集上索引为8的瓷砖数据，并对它进行了旋转和翻转。然后可以用这个变量从你指定的瓦片组中绘制该索引的瓦片。 与任何瓦片图层无关，瓦片将在进行绘制的实例的图层深度上，在你在房间中指定的X/Y位置上绘制。请注意，你也可以设置框架来绘制。如果被索引的瓦片是一个动画的一部分 的一部分，那么你可以在这里设置帧数（从0到帧数-1），索引瓦片将简单地成为你正在使用的动画瓦片集的指示器，而它将是被绘制的给定帧。如果这块瓷砖不属于 的一部分，那么帧应该被设置为0。

语法。

draw_tile(tile set, tiledata, frame, x, y)

争论 描述
瓷砖套装 要使用的瓦片组的ID
平铺数据 要使用的瓦片数据
框架 要使用的动画瓦片的帧数（非动画瓦片默认为0）。
x 在房间内的X位置绘制
y 在房间内绘制的Y位置

返回。

真实的

例子。

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var data = tilemap_get(map_id, 0, 0);
tile_set_flip(data, true);
draw_tile(set_Wall, data, 0, x, y) 。

上述代码将从给定的瓦片图中获取瓦片上的数据，然后设置 "翻转 "标志，并使用这些数据来绘制在实例位置上翻转的瓦片。
