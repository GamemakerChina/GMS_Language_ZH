绘制瓷砖图

这个函数可以用来在房间的任何地方绘制一个给定的瓷砖图。你给瓦片图元素ID（当你用 "创建 "瓦片图元素时，你会得到这个ID。 layer_tilemap_create() 或当你使用函数 layer_tilemap_get_id()时得到的），然后给出绘制的X/Y位置（在房间里）。请注意，这将只是在指定的点上绘制瓷砖图。 指定的点，使用调用该函数的实例的层深度。它不会移动瓦片图--也不会以任何方式改变它--而且瓦片图是否被标记为可见也不重要。

语法。

draw_tilemap(tilemap_element_id, x, y)。

争论 描述
tilemap_element_id 瓷砖图元素的唯一ID值，以获得单元格的Y位置。
x 在房间内的X位置绘制
y 在房间内绘制的Y位置

返回。

不适用

例子。

var lay_id = layer_get_id("Tiles_Effects");
var map_id = layer_tilemap_get_id(lay_id);
draw_tilemap(map_id, mouse_x, mouse_y) 。

上面的代码获得了图层ID，然后用它来获得一个特定的瓷砖图ID，这又被用来在鼠标位置上绘制瓷砖图。
