应用程序_表面

这个全局范围的内置变量可以用来使用任何表面函数访问应用程序的表面。这个表面是永久可用的，GameMaker Studio 2的大部分绘图都是在这里完成的。

语法。

应用程序_表面。

返回。

指针

例子。

surface_resize(application_surface, display_get_gui_width(), display_get_gui_height())

上述代码将调整应用程序表面的大小，使其与GUI层的比例为1：1。
