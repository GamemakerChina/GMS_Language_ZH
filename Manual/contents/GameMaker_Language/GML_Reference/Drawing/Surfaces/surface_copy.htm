表面_复制

这个函数只是把一个曲面的图像复制到另一个曲面的指定局部位置（其中（0,0）位置是目标曲面的左上角）。如果目标表面已经 的信息会被拷贝覆盖，这个函数不会以任何方式改变源表面。

注意：在处理表面时，由于它们被储存在纹理内存中，它们有可能随时不存在。你应该在直接引用曲面之前，先用 surface_exists() 来检查曲面是否存在，然后再直接引用它们。

语法。

surface_copy(destination, x, y, source)。

争论 描述
目的地 要复制到其他表面的ID。
x 要复制到的X位置。
y 要复制到的y位置。
来源 要复制的表面的ID。

返回。

不适用

例子。

如果view_current == 0
{
surface_copy(surface, 0, 0, temp_surf);
}
else
{
draw_surface(surface, 0, 0);
}

上面的代码将检查当前正在绘制的视图，如果是view[0]，它将把变量 "temp_surf "中索引的表面复制到变量 "surf "中索引的表面。如果当前视图不是view[0]，则会将 "surf "复制到变量 "surf "中。 surf "就会被绘制到屏幕上。
