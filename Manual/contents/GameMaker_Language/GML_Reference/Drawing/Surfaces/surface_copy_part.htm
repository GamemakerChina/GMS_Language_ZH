表面_复制_部分

这个函数只是把一个表面的图像复制到另一个表面的指定局部位置上（其中（0,0）位置是目标表面的左上角）。你可以指定一个局部的x和y位置 以及该部分的宽度和高度。请注意，这些是基于曲面尺寸的坐标，而不是基于曲面在房间里的位置。如果目标表面已经有信息 将会被复制覆盖，而且这个函数不会以任何方式改变源表面。

注意：在处理表面时，由于它们被储存在纹理内存中，它们有可能随时不存在。你应该在直接引用曲面之前，先用 surface_exists() 来检查曲面是否存在，然后再直接引用它们。

语法。

surface_copy_part( destination, x, y, source, xs, ys, ws, hs)。

争论 描述
目的地 要复制到其他表面的ID。
x 要复制到的X位置。
y 要复制到的y位置。
来源 要复制的表面的ID。
xs 要复制的源面中的X位置。
ys 要复制的源表面的y位置。
ws 要复制的源表面区域的宽度。
hs 要复制的源表面区域的高度。

返回。

不适用

例子。

如果view_current == 0
{
surface_copy_part(surface, 0, 0, temp_surf, 0, 0, view_xview[1] - mouse_x, view_yview[1] - mouse_y);
}
else
{
draw_surface(surface, 0, 0);
}

上面的代码将检查当前正在绘制的视图，如果它是view[0]，它将把变量 "temp_surf "中索引的表面复制到变量 "surf "中索引的表面。复制的区域对应于一个矩形 在视图[1]中，鼠标在曲面中的相对位置所形成的矩形。如果当前的视图不是view[0]，那么曲面 "surf "将被绘制到屏幕上。
