表面_获取深度_禁用

这个函数检查是否启用了为曲面自动生成深度缓冲区的功能。通常情况下，所有的表面都有深度缓冲区，所以如果你在上面绘制3D物体，它就会按照深度正确排序。 这可能是一个过度的、不必要的开销，特别是当你的游戏非常耗费内存或以2D为主的时候。在这种情况下，你可以用这个函数来检查，并禁用表面的深度缓冲区 如果需要的话，可以使用函数 surface_depth_disable().

语法。

surface_get_depth_disable()。

返回。

布尔型

例子。

if (!surface_exists(surface))
{
if surface_get_depth_disable() == false
{
surface_depth_disable(true);
}
surf = surface_create(room_width, room_height);
}

上面的代码会检查给定的曲面是否存在，如果不存在，就会检查曲面深度缓冲区的当前状态，如果它是启用的，就会改成禁用，最后才会创建这个曲面。
