surface_getpixel

这个函数可以用来从曲面上获取特定像素的颜色，使用曲面的局部坐标，其中（0,0）是左上角。这个函数不应该经常使用，因为它非常慢，可能会导致游戏暂停。 在你的游戏中暂停。

注意：在处理曲面时，由于它们被储存在纹理内存中，它们有可能在任何时候都不存在了。你应该在直接引用曲面之前，先用 surface_exists() 来检查曲面是否存在，然后再直接引用它们。

语法。

surface_getpixel(surface_id, x, y)。

争论 描述
surface_id 表面的ID。
x 在表面上的X位置，从该位置获取像素。
y 在表面上的y位置，用来获取像素。

返回。

真实的

例子。

col = surface_getpixel(surface, 56, 78 );

这将返回变量 "surf "中索引的表面的坐标（56,78）处的像素的颜色。
