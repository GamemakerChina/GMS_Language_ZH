表面_重设目标

通过这个函数，你可以将所有进一步的绘图从目标表面重置到屏幕上。请注意，要开始在一个曲面上绘图，你必须先调用函数 surface_set_target()和 然后再调用这个函数，否则屏幕上将不会有任何东西被绘制，因为所有进一步的绘制（甚至在其他情况下）都将在曲面上完成。你也应该意识到 如果曲面本身没有在实例的绘制事件中被绘制到屏幕上，就不会被看到。

注意：如果你之前没有用函数设置渲染目标 surface_set_target()来设置渲染目标，那么使用这个函数会悄悄地（即：没有任何错误信息）结束事件的所有后续代码执行。 没有任何错误信息）结束该事件的所有后续代码执行。

语法。

surface_reset_target()。

返回。

不适用

例子。

如果view_current = 0
{
surface_set_target(surface)。
with (obj_Effect)
{
draw_self();
}
surface_reset_target()。
}
else
{
draw_surface(surface, 0, 0);
}

上面的代码将检查当前正在绘制的视图，如果是view[0]，它将把绘制目标设置为一个表面，并在再次重置绘制目标之前绘制所有对象 "obj_Effect "的实例。如果视图不是view[0] 则将曲面绘制到屏幕上。
