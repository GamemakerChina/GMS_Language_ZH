表面_设置目标

通过这个函数，你把所有进一步的绘制都设置为目标表面，而不是屏幕，通过这种方式，你可以告诉GameMaker Studio 2只在指定的表面绘制特定的东西。请注意，如果你不在完成后调用 surface_reset_target() 后，屏幕上将不会有任何东西被绘制，因为所有进一步的绘制（甚至在其他情况下）都将在表面上完成。你也应该意识到，如果曲面本身没有被绘制在屏幕上的draw 事件中，如果曲面本身没有被绘制到屏幕上，那么就不会看到任何东西。你也可以检查这个函数的返回值，作为一个调试工具，检查曲面的目标是否被设置，返回值为0表示目标设置失败，其他正值表示成功。

有一点需要注意的是，表面是叠加的，所以你不能从一个目标跳到另一个目标，然后在最后重置到正常的绘制目标，而是必须打开和关闭渲染目标。例如，这样就不能正常工作。

surface_set_target(surface1);
draw_text(32, 32, "surface1");
surface_set_target(surface2);
draw_text(32, 64, "surface2");
surface_reset_target()。

相反，你必须为你设置的每一个面重置目标，就像你必须为代码块使用开括号和闭括号{}一样。所以上面的内容应该写成这样。

surface_set_target(surface1);
draw_text(32, 32, "surface1");
surface_reset_target() 。
surface_set_target(surface2)。
draw_text(32, 64, "surface2");
surface_reset_target()。

或者像这样。

surface_set_target(surface1);
draw_text(32, 32, "surface1");
surface_set_target(surface2);
draw_text(32, 64, "surface2");
surface_reset_target()。
surface_reset_target()。

注意：在处理曲面时，由于它们被储存在纹理内存中，它们有可能在任何时候都不存在了。你应该在直接引用曲面之前，先用 surface_exists() 来检查曲面是否存在，然后再直接引用它们。

语法。

surface_set_target(surface_id)。

争论 描述
surface_id 要设置为绘图目标的曲面的ID。

返回。

布尔型

例子。

如果view_current = 0
{
surface_set_target(surface)。
with (obj_Effect)
{
draw_self();
}
surface_reset_target()。
}
else
{
draw_surface(surface, 0, 0);
}

上面的代码将检查当前正在绘制的视图，如果是view[0]，它将把绘制目标设置为一个表面，并在再次重置绘制目标之前绘制所有对象 "obj_Effect "的实例。如果视图不是view[0] 则将曲面绘制到屏幕上。
