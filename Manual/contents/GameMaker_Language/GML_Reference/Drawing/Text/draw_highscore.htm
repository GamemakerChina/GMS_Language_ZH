绘制高分图

这个简单的函数将使用当前设置的字体、颜色和阿尔法值在指定的矩形内绘制当前内部存储的高分列表。你提供矩形区域的左上角和右下角的坐标来绘制文本。 矩形区域的左上角和右下角的坐标来绘制文本，GameMaker Studio 2会处理剩下的部分，间距和位置会自动完成。

语法。

draw_highscore( x1, y1, x2, y2 )。

争论 描述
x1 高分矩形左边的X坐标。
y1 高分矩形顶部的y坐标。
x2 高分矩形右侧的X坐标。
y2 高分矩形底部的y坐标。

返回。

不适用

例子。

draw_highscore(100, 100, room_width - 100, room_height - 100)。

这将把高分表画在房间中间的一个矩形里，边框为100px。
