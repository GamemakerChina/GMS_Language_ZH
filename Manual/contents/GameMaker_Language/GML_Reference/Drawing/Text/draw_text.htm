绘图_文本

通过这个函数，你可以在房间的任何位置画出任何字符串（对于画实数，你应该使用 字符串()函数将其转换为文本）。)要组合字符串，你可以 使用"+"（见下面的例子），你也可以在一个字符串中使用 "n"来增加一个换行符，这样它就会在多行上画出来（关于如何正确格式化一个字符串以及你可以使用哪些换行符的信息，请看这里）。 文本的颜色和阿尔法是由当前的基础阿尔法和颜色值决定的，由 绘制_set_alpha()和 draw_set_colour().

注意：文本的实际位置将受到由 "A "和 "B "设置的对齐值的影响。 绘制_set_halign()和 draw_set_valign().

语法。

draw_text(x, y, string)。

争论 描述
x 画出的字符串的X坐标。
y 画出的字符串的y坐标。
绳子 要拉的绳子。

返回。

不适用

例子。

draw_text(x, y, "Hello, " + global.Name + "!/nI hope you are well!")。

上面的代码将在实例的x/y位置画一个字符串，它将使用存储在全局变量 "Name "中的字符串，并将其分成两行。
