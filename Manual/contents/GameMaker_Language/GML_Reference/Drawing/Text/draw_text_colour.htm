绘制文本颜色

这个函数将以类似于 绘制文本()只是现在你可以选择为文本着色的颜色以及alpha值，这些新值将被用来代替 这些新值将被用来代替基本的绘图颜色和阿尔法值。

注意：除非启用了WebGL，否则梯度混合对HTML5目标是不可用的，尽管你仍然可以设置混合颜色，它将把字体与第一个给定的颜色混合。然而，所有以这种方式进行的混合都会创建一个重复的 字体，然后存储在缓存中并在需要时使用，这远非最佳状态，而且如果你使用多种颜色变化，会降低你的游戏性能。如果有必要，你可以设置字体缓存的大小来尝试限制这种情况 使用函数 font_set_cache_size().

语法。

draw_text_colour(x, y, string, c1, c2, c3, c4, alpha)。

争论 描述
x 画出的字符串的X坐标。
y 画出的字符串的y坐标。
绳子 要拉的绳子。
c1 绘制文本左上方的颜色。
c2 绘制文本右上方的颜色。
c3 绘制文本右下方的颜色。
c4 绘制文本左下方的颜色。
阿尔法 文本的字母。

返回。

不适用

例子。

draw_set_colour(c_white);
draw_text(100, 100, "Health");
draw_text_colour(100, 200, string(health), c_lime, c_lime, c_green, c_green, 1) 。

上面的代码将在同一条线上画出两段文字，第一段文字被画成白色（因为那是基本的绘图颜色），第二段文字被画成青绿色到正常绿色的渐变。
