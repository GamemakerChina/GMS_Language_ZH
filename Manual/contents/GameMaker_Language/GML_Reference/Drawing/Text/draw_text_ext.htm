绘图_文本_文本

这个函数将以类似于 绘制文本()只是现在你可以设置每行文本之间的空间--如果文本占用超过一行的话--并限制每行字符串的宽度（像素）。 这样，如果任何一行超过这个值，GameMaker Studio 2将自动将文本分割到下一行最近的可用白位（如果文本没有白位，它将超过这个最大的 宽度值）。)请注意，由于这个原因，任何放置在字符串开头的空白将在被解析绘制之前被剥离出来。还要注意的是，如果行间距参数的值为-1，则默认为基于 "M "字符高度的间距。 所选字体中 "M "字符的高度。

语法。

draw_text_ext(x, y, string, sep, w);

争论 描述
x 画出的字符串的X坐标。
y 画出的字符串的y坐标。
绳子 要拉的绳子。
绵绵不绝 文本行之间的距离，以像素为单位。
w 换行前字符串的最大宽度（像素）。

返回。

不适用

例子。

draw_text_ext(100, 50, keyboard_string, 3, 300)。

上述代码将绘制用户在键盘上输入的任何文本，每当该行的字符串长度超过300像素时，就将其分割到新的一行。

返回。文本
