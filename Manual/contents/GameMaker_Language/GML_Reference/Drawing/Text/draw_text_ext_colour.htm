绘制_文本_文字_颜色

这个函数是由基础 绘制文本()函数与 draw_text_ext()和 draw_text_colour() 函数的组合，允许你为文本定义渐变色，以及定义行距和每行最大宽度。

注意： 除非启用了WebGL，否则梯度混合对HTML5目标是不可用的，尽管你仍然可以设置混合颜色，它将把字体与第一个给定的颜色混合。然而，所有以这种方式进行的混合都会创建一个重复的 字体，然后存储在缓存中并在需要时使用，这远非最佳状态，而且如果你使用多种颜色变化，会降低你的游戏性能。如果有必要，你可以设置字体缓存的大小来尝试限制这种情况 使用函数 font_set_cache_size().

语法。

draw_text_ext_colour(x, y, string, sep, w, c1, c2, c3, c4, alpha) 。

争论 描述
x 画出的字符串的X坐标。
y 画出的字符串的y坐标。
绳子 要拉的绳子。
绵绵不绝 文本行之间的距离，以像素为单位。
w 换行前字符串的最大宽度（像素）。
c1 绘制文本左上方的颜色。
c2 绘制文本右上方的颜色。
c3 绘制文本右下方的颜色。
c4 绘制文本左下方的颜色。
阿尔法 文本的字母。

返回。

不适用

例子。

draw_text_ext_colour(200, 200, keyboard_string, 5, 300, c_blue, c_blue, c_navy, c_navy, 1) 。

上述代码将绘制用户在键盘上输入的任何文本，每当该行的字符串长度超过300像素时，就将其分割成新的一行。如果出现这种情况，代码还将在各行之间保持5像素的间隔。 每行文字将使用蓝色渐变色，顶部是浅蓝色，底部是深蓝色。
