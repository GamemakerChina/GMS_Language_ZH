绘画_文字_文本_变形的颜色

这个函数是由基础的 绘制文本()函数与 draw_text_ext(), draw_text_transformed() 和 绘制文本颜色()函数的组合。因此，这是所有绘制文本的函数中最通用的，并在绘制时提供了大量的灵活性。请注意，"宽度" 参数是基于1的比例，所以如果比例不同，这个值应该按比例改变。例如，如果一个断行的基本宽度是300，而你把比例设置为2，那么文本就会出现错误，超过了给定的 宽度。相反，你应该将宽度设置为150，以补偿比例的变化。

注意：除非启用WebGL，否则梯度混合对HTML5目标是不可用的，尽管你仍然可以设置混合颜色，它将用第一个给定的颜色混合字体。然而，所有以这种方式进行的混合都会创建一个重复的 字体，然后存储在缓存中并在需要时使用，这远非最佳状态，而且如果你使用多种颜色变化，会降低你的游戏性能。如果有必要，你可以设置字体缓存的大小来尝试限制这种情况 使用函数 font_set_cache_size().

语法。

draw_text_ext_transformed_colour(x, y, string, sep, w, xscale, yscale, angle, c1, c2, c3, c4, alpha) 。

争论 描述
x 画出的字符串的X坐标。
y 画出的字符串的y坐标。
绳子 要拉的绳子。
绵绵不绝 文本行之间的距离，以像素为单位。
w 换行前字符串的最大宽度（像素）。
xscale 水平刻度。
yscale 纵向比例。
角度 文字的角度。
c1 绘制文本左上方的颜色。
c2 绘制文本右上方的颜色。
c3 绘制文本右下方的颜色。
c4 绘制文本左下方的颜色。
阿尔法 文本的字母。

返回。

不适用

例子。

draw_set_halign（fa_center）;
draw_set_valign（fa_middle）。
draw_text_ext_transformed_colour(200, 200, "My name is" + global.name +". ", 20, 200, 0.5, 0.5, 180, c_blue, c_blue, c_navy, c_navy, 1);

这将在房间的(200,200)的位置上，用蓝色到深蓝色的渐变画出给定的文字，旋转为倒立，字体大小为正常点的一半。每行之间将有20个像素，如果有超过 每条线之间将有20个像素，每条线的最大宽度为200像素。

返回。文本
