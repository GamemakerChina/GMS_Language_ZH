纹理

下面的函数都与 "纹理 "有关，即：存储在VRAM中的图像，可作为游戏中的精灵或其他东西使用。第一组可用的函数 用于检索特定图像的信息和它们在纹理页上的位置或大小，以及设置某些纹理特征。

  • 纹理_获取宽度
  • 纹理_get_height
  • 纹理_get_texel_width
  • 纹理_get_texel_height
  • 纹理_get_uvs
  • 纹理_set_stage
  • 纹理_global_scale

GameMaker Studio 2也有一些与管理单个纹理和整个纹理组相关的 VRAM 有关的功能。

  • 绘画_纹理_刷新
  • 纹理预取
  • 纹理冲洗
  • 幽灵_刷新
  • 多重化精灵
  • 萌芽_预取
  • 萌芽_预取_多
  • draw_flush

最后，我们有以下函数，主要用于调试项目和确保有效使用纹理内存。

  • 纹理_调试_消息
  • 纹理_is_ready
  • 纹理组_get_textures
  • 纹理组_get_sprites
  • 纹理组_get_fonts
  • 纹理组_get_tilesets
