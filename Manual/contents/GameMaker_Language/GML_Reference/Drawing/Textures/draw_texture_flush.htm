绘画_纹理_刷新

通过这个函数，你可以从视频内存中删除所有的纹理，然后在第一次使用时重新加载它们。当你在游戏中有多个纹理页时，这是管理视频内存的唯一有效方法。 你应该在你的游戏关卡之间冲洗纹理内存，并使用纹理组功能来组织图形，以确保使用最少的纹理数量。

值得注意的是，在所有目标上，默认情况下，纹理页只有在需要时才会被加载，你可以使用精灵的各种预取和刷新函数（在这里找到 来管理，也可以使用这个函数。
