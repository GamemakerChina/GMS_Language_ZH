纹理_调试_消息

这个函数可以用来启用或禁用纹理调试信息。当启用时（设置为true），关于纹理页使用的额外信息将被发送到输出窗口。设置为false则再次禁用这种输出。

语法。

texture_debug_messages(enable)。

争论 描述
启用 启用或禁用纹理调试信息

返回。

不适用

例子。

如果 global.debug
{
texture_debug_messages(true)。
}

上面的代码将检查一个全局变量的值，如果它被设置为"真"，那么纹理调试信息将被启用。

返回。纹理
