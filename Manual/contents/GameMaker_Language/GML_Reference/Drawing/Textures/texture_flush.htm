纹理冲洗

这个函数可以用来 "刷新 "一个纹理页或一组纹理页，即：在不再需要时从 VRAM 中删除它们。你可以提供唯一的纹理页ID（由texturegroup_*函数返回）来刷新一个纹理页。 一个单独的页面，或者你可以提供一个纹理组的名称（如在纹理组编辑器中定义的）来刷新该组中的所有纹理页面。

语法。

texture_flush(tex_id)。

争论 描述
tex_id 纹理页ID（一个整数）或纹理组ID（一个字符串）。

返回。

不适用

例子。

var _tex_array = texturegroup_get_textures( "MainMenu");
for (var i = 0; i < array_length( _tex_array); ++i;)
{
texture_flush(_tex_array[i])。
}

上述代码将刷新 "MainMenu "纹理组下的所有纹理页。

返回。纹理
