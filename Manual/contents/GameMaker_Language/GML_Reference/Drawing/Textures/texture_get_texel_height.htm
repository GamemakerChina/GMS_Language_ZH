纹理_get_texel_height

texel或纹理元素是计算机图形中使用的纹理空间的基本单位。纹理是由纹理数组表示的，就像图片是由像素数组表示的一样，这个函数返回所使用的图像资产的纹理页中的单个 这个函数返回所使用的图像资产的纹理页中的单个 texel 的高度。

语法。

texture_get_texel_height(tex)。

争论 描述
特克斯 要使用的纹理页资产指针

返回。

真实的

例子。

var tex = sprite_get_texture(sprite_index, 0);
tex_w = texture_get_texel_width(tex);
tex_h = texture_get_texel_height(tex);

上面的代码将获得来自精灵资产的纹理的宽度和高度。
