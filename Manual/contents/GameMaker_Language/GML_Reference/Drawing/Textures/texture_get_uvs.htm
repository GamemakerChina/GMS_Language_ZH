纹理_get_uvs

这个函数返回一个有4个元素的一维数组，代表纹理页上图像的UV坐标，在数组中填充以下数值。

  • [0] = 左
  • [1] = 顶部
  • [2] = 对
  • [3]=底部
  • [4] = 资产编译器从精灵左侧修剪的像素数量（仅精灵资产）。
  • [5] = 资产编译器从精灵顶部修剪的像素数量（仅精灵资产）。
  • [6] = 被保存到纹理页的原始精灵宽度的像素数据的正常化百分比（仅精灵资产）。
  • [7] = 已保存到纹理页的原始精灵高度的像素数据的标准化百分比（仅精灵资产）。

这个值可以在其他绘图函数中使用，特别是在一般的3D和一些2D原始函数中，以及Shader函数中。如果你需要一个精灵的UVS，那么你可以使用 sprite_get_uvs()和 用于一个字体。 font_get_uvs().

语法。

text_get_uvs(texid)

例子。

var tex = texture_get_uvs(surf_back);
tex_left = tex[0];
tex_top = tex[1];
tex_right = tex[2];
tex_left = tex[3];

上面的代码将把给定的表面纹理的UV坐标存储在一个本地数组中，然后把这些值分配给实例变量。
