纹理组_get_textures

这个函数可以用来检索组成一个纹理组的各个页面的纹理页ID。你提供纹理组ID字符串（如在纹理组编辑器中定义的），该函数将返回一个一维数组，其中数组中的每个 数组中的每个条目都是一个单独的纹理页ID。如果函数失败--即：给出了一个无效的组，或者该组没有分配到纹理--那么该数组将是空的（0长度）。

语法。

texturegroup_get_textures(tex_id)。

争论 描述
tex_id 要检查的纹理组的名称（一个字符串）。

返回。

1D阵列

例子。

var _tex_array = texturegroup_get_textures( "MainMenu");
for (var i = 0; i < array_length( _tex_array); ++i;)
{
if texture_is_ready(_tex_array[i])
{
texture_prefetch(_tex_array[i])。
}
}

上面的代码将检索纹理组 "MainMenu "的纹理页ID，然后检查它们是否被解压，如果被解压，它们将被放置到VRAM中。

返回。纹理
