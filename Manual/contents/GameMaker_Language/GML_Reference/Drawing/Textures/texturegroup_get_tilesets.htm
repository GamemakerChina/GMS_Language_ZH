纹理组_get_tilesets

通过这个函数，您可以检索分配给给定纹理组中的纹理页的每一个瓷砖组的索引。你提供纹理组ID字符串（如在纹理组编辑器中定义的），该函数将返回一个1D的 数组，其中每个条目包含一个纹理组资源的瓦片组索引。如果函数失败了--即：给出了一个无效的组，或者该组没有分配给它的纹理--那么这个数组将是空的（0 length）。

语法。

texturegroup_get_tilesets(tex_id)。

争论 描述
特克斯 要检查的纹理组的名称（一个字符串）。

返回。

1D阵列

例子。

var _tex_array = texturegroup_get_tilesets( "MainMenu");
for (var i = 0; i < array_length( _tex_array); ++i;)
{
show_debug_message("Tileset" + string(i) + " Index:" + string(tex_array[i]));
}

上面的代码将检索纹理组 "MainMenu "的瓷砖组索引，然后在控制台输出窗口显示这些ID。

返回。纹理
