draw_enable_drawevent

通过这个函数，你可以选择启用（true）或禁用（false）游戏中所有实例的绘制事件，从而让你控制事物的绘制方式和时间，如果你想实现一个 "跳帧 "技术。请注意，这不仅仅是阻止实例绘制到屏幕上，它完全抑制了绘制事件，这意味着应该小心，因为任何存在于该事件中的游戏逻辑都不会被运行 也不会运行。关于这个函数，需要理解的一件事是，如果你在游戏开始时，在初始帧被渲染之前（即：游戏中第一个房间的第一个对象的创建事件）调用它，那么游戏窗口将不会被渲染。 这对于需要服务器不需要渲染任何东西的在线项目可能很有用。

语法。

draw_enable_drawevent(启用)。

争论 描述
启用 设置为真或假。

返回。

不适用

例子。

frame_skip ++;
if (frame_skip mod 5) == 0
{
draw_enable_drawevent(true);
}
else
{
draw_enable_drawevent(false);
}

上面的代码检查了一个变量，如果它返回真，它就启用了绘制事件，否则绘制事件就被禁用。
