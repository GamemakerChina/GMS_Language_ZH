sha1_string_unicode

在密码学中，SHA-1是一个由美国国家安全局设计的密码学哈希函数，在一些广泛使用的应用程序和协议中被采用，如流行的 指南针在那里，它被用来检查文件的变化。 这个函数将接受一个输入的unicode字符串，并返回一个ASCII格式的160位消息摘要，例如。

sha1输入字符串 = "快速的棕色狐狸跳过懒惰的狗"
sha1输出哈希 = "2fd4e1c6 7a2d28fc ed849ee1 bb76e739 1b93eb12"

通过这种方式，你可以生成一个安全的密钥，它可以被存储并用于检查被发送到（或从）外部服务器（例如）的信息的完整性。

注意：sha1编码有两种格式，UTF-8和unicode。提供这两种格式是为了方便与不同的服务器设置进行通信，但最常用的是unicode。

注意：SHA-1不是完全安全的，可以被破解。更多信息请见本页面。

语法。

sha1_string_unicode(string)

例子。

var hash, str;
str = base64_encode(game_data);
hash = sha1_string_unicode(str);
http_get("http://www.MacSweeneyGames.com/CatchTheHaggis/gamedata?hash=" + hash); http_get("http://www.MacSweeneyGames.com/CatchTheHaggis/gamedata?data=" + str）。)

上述代码将对一个字符串进行base64编码，然后生成一个sha1哈希值。最后，哈希值和编码后的字符串都被发送到服务器上。
