zip_unzip

这个函数将打开一个存储的ZIP文件，并将其内容提取到给定的目录中。注意，如果你没有提供ZIP目录的完整路径，那么将使用当前的驱动器根目录，如果你想把它放在一个相对的路径中 到游戏捆绑工作目录的相对路径，那么你应该使用 working_directory变量作为路径的一部分（使用"... "或"... "的相对路径将不会像预期的那样工作 所以应该避免）。)还要注意的是，压缩包必须是游戏包的一部分（即：包含的文件），或者已经用http_get_file()下载到存储区。

该函数将返回一个值，表明被提取的文件数量，如果提取失败，它将返回0或更少。

语法。

zip_unzip(zip_file, target_directory)

争论 描述
压缩文件 要打开的压缩文件
目标目录 将文件提取到的目标目录

返回。

真实的

例子。

var num = zip_unzip("/downloads/level_data.zip", working_directory + "extracted/");
if num <= 0
{
show_debug_message("Extraction Failed!");
}

上述代码将打开存储在 "downloads "目录下的压缩文件，并将其内容提取到 "extracted "目录下（如果该目录不存在，则创建该目录），然后检查提取是否正确。 是正确的，如果失败则显示一个调试信息。
