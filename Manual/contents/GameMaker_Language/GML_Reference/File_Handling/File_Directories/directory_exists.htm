directory_exists

如果指定的目录存在，该函数将返回true，如果不存在则返回false。指定的名称必须包括完整的路径，而不是相对路径，默认情况下，你不能从游戏包以外的地方访问任何目录 因为所有的游戏都是沙盒的（更多信息见文件系统部分）。

语法。

directory_exists(dname)

争论 描述
命名 要寻找的目录的名称。

返回。

布尔型

例子。

if directory_exists(working_directory + "Saves/")
{
file = file_find_first(working_directory + "Saves/*.doc", fa_readonly);
}

这将检查指定的目录是否存在，如果存在，就去那里，并返回找到的第一个 "只读 "文档文件。
