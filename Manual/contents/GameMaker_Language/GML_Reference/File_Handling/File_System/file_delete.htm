文件删除

这个函数将从系统中删除指定的文件。需要注意的是，这个函数只删除GameMaker Studio 2能够创建和解析的文件：ini文件、文本文件和二进制文件，或者那些用于存储游戏创建的资源（如精灵或表面）的文件。 储存游戏创建的资源，如精灵或表面。然而，它不会删除任何其他文件。如果文件被成功删除，该函数也将返回true，或者在任何其他情况下返回false。

警告!由于GameMaker Studio 2是沙盒式的，这个功能可能不会像你所期望的那样工作!请参阅文件系统限制一节以了解更多信息。

语法。

file_delete(fname)。

争论 描述
fname 要删除的文件的名称。

返回。

布尔型

例子。

if file_exists("level.txt")
{
file_delete("level.txt")。
}

这将检查一个文件，如果它存在，就将其删除。

返回。文件系统
