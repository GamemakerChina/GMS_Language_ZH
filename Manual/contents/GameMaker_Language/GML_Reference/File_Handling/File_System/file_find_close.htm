文件_找寻_关闭

必须在处理完用以下方法打开的文件后调用 file_find_first()和 file_find_next()函数打开的文件后调用，以释放内存。文件查找函数打开了进入文件目录的句柄 目录的句柄，这些句柄占用了极少的内存，而这些内存（随着时间的推移）会越来越多。因此，你应该总是在找到你需要的文件后调用这个函数来 "关闭 "这些句柄。

语法。

file_find_close()。

返回

不适用

例子。

file_find_close()。

这将关闭并释放与文件查找函数句柄相关的内存。
