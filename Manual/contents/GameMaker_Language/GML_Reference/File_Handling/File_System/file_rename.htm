文件重命名

这个函数将用指定的名称重命名指定的文件。如果文件被成功重命名，该函数将返回 "true"，或者在其他情况下返回 "false"。

警告!由于GameMaker Studio 2是沙盒式的，这个功能可能不会像你预期的那样工作!请参阅文件系统部分以了解更多信息。

语法。

file_rename(oldname, newname)。

争论 描述
旧名 要改变的文件的名称。
新名称 给予该文件的新名称。

返回。

布尔型

例子。

if file_exists("level1.txt")
{
file_rename("level1.txt", "level.txt");
}

这将检查一个文件，如果它存在，就会被重新命名。

返回。文件系统
