文件名_修改_文本

该函数返回指定的文件名，并将扩展名（包括点）改为新的扩展名。通过使用一个空字符串作为新的扩展名，你可以将扩展名部分全部删除。

语法。

filename_change_ext(fname, newext)。

争论 描述
fname 要使用的文件。
新文本 要使用的新扩展。

返回。

字符串

例子。

ext = filename_change_ext(file_find_first(working_directory + "*.*", 0), " ")。

上面的代码得到的是去掉扩展名部分的文件名（作为一个字符串）。

返回。文件系统
