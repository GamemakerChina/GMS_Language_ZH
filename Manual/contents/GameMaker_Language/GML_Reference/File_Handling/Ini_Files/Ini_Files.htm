伊尼文件

INI文件是小而轻的文件，与大多数平台兼容。它们是存储小型信息的理想选择，如界面偏好、本地高分、关卡数据等...，因此是 因此，它是你开始保存游戏信息的一个伟大而简单的方法。然而，值得注意的是，它们所能存储的数据是有限制的，最大为64KB，所以如果你要保存数据结构之类的东西，你可能希望 看一下其他的文件功能，如文本文件。

重要的是!所有GameMaker Studio 2项目都有一个名为""的文件。选项.ini"，所以你不能用这个名字命名 所以你不能用这个名字命名任何文件（当你试图编译你的游戏项目时，你会得到一个UID失败）。

请注意，对于有本地化的游戏来说，可以从外部文件读取重音字母是非常重要的。这意味着你应该首先创建UTF8格式的ini文件，然后将其作为一个包含文件添加到GameMaker Studio 2中。 然后将其添加到GameMaker Studio 2中，以便在运行游戏时将其导出，并用于替代默认的ANSI格式ini文件，该文件由ini函数创建，如果先前没有提供 提供的默认ANSI格式ini文件。通过这种方式，你可以正确地读和写它，所有的重音和非罗马字母都被保留。

ini文件有以下功能。

  • ini_open
  • ini_close
  • ini_write_real
  • ini_write_string
  • ini_read_real
  • ini_read_string
  • ini_key_exists
  • ini_section_exists
  • ini_key_delete
  • ini_section_delete
  • ini_open_from_string
