ini_close

这个函数应该在你完成对任何打开的ini文件的读写时被调用。如果你在使用了任何ini写函数之后没有使用这个函数，那么就不会有任何东西被写入磁盘，因为文件信息被保留在内存中，直到这个函数被调用，强制写入。 在内存中，直到这个函数被调用，强制写入。如果你试图打开一个ini，而之前没有关闭另一个（或同一个），你也会得到一个错误。

该函数也将返回一个将ini文件编码成的字符串。这个字符串可以被保存到服务器上和/或与该函数一起再次用于 ini_open_from_string()来重新创建 ini。

语法。

ini_close()。

返回。

字符串

例子。

ini_open("savedata.ini");
score = ini_read_real("save1", "score", 0);
ini_close();

这将打开 "savedata.ini"，并将分数设置为 "save1 "部分的数值，其中包含 "score "键，然后再次关闭.ini文件。如果 "save1 "下没有 "score "的值，或者没有 "savedata.ini "文件，分数将被设置为0，并创建一个新的ini_文件。
