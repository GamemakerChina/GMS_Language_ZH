ini_open_from_string

这个函数将从一个字符串创建一个ini文件，并打开它进行读/写。字符串应该被正确地格式化为ini文件（即：有章节、换行键和值），否则ini文件将不能被正确创建。注意 注意，这个ini文件是临时性的，一旦关闭就会从内存中删除，失去任何存储在其中的信息。 ini_close()函数返回一个完整的ini文件的字符串。 然后可以保存到服务器或磁盘上。

语法。

ini_open_from_string(string)。

争论 描述
绳子 包含所有ini信息的字符串。

返回。

不适用

例子。

ini_open_from_string(str)
global.sound = ini_read_string("Options", "Sound", true);
ini_close();

上面的代码设置了一个临时的ini来保存字符串 "str "的信息，然后在再次关闭它之前从ini中读取信息。
