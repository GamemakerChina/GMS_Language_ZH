ini_read_string

你可以用这个函数从一个ini数据文件中读取一个字符串（文本）。Ini文件被分割成若干部分，然后每个部分随后被分割成键-值对。因此，一个典型的ini文件看起来会是这样的。

语法。

ini_read_string(section, key, default)。

争论 描述
节 从.ini中读取的部分。
钥匙 读取.ini中相关部分的键。
违约 如果在定义的地方没有找到一个字符串（或者.ini文件不存在），要返回的字符串。必须是一个字符串。

返回。

字符串

例子。

ini_open("savedata.ini");
global.name = ini_read_string("player", "name', "Player1");
ini_close();

这将打开 "savedata.ini "并将global.name设置为其中 "save1">"Name "下的字符串，然后再次关闭.ini。如果 "save1">"Name "下没有字符串，或者没有 "savedata.ini "文件 文件，global.name将被设置为 "Player1"。
