ini_section_exists

这个函数检查当前打开的ini中是否存在一个部分，如果存在则返回true，否则返回false。这不是一个防止错误的必要检查，因为当一个部分不存在时，从一个不存在的 部分将只是返回一个默认值，然而它对于查看一个ini文件是否保存了特定的数据是很有用的。

语法。

ini_section_exists(section)。

争论 描述
节 打开的.ini文件中要检查的部分。

返回。

布尔型

例子。

ini_open("saidata.ini");
if !ini_section_exists("save1")
{
global.savegame = "1";
}
ini_close()。

如果没有找到 "save1"，这将把变量global.savegame设置为 "1"。通过for循环，这可以作为计算一个游戏有多少个保存游戏的基本方法（即：为一个游戏的多个同步保存游戏创建一个系统）。 游戏的系统）。)
