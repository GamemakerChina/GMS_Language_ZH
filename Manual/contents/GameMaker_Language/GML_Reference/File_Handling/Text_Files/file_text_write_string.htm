文件_text_write_string

通过这个函数，你可以将一个字符串写到一个先前打开的文本文件中。如果该文件已经包含了信息，这些信息将被清除，而字符串将被写在文件的开头，除非你已经用 file_text_open_append().你也可以通过使用 file_text_readln()函数和 file_text_eof() 函数循环浏览文件的内容，直到最后才开始写入。

语法。

file_text_write_string(fileid, str)。

争论 描述
文件名 要编辑的文件的ID。
弦外之音 要写到文件中的字符串。

返回。

真实的

例子。

var file = file_text_open_write(working_directory + "hiscore.txt");
for (var i = 0; i < 10; ++i;)
{
file_text_write_real(file, scr[i]);
file_text_writeln(file)。
file_text_write_string(file, scr_name[i]);
file_text_writeln(file);
}
file_text_close(file)。

上面的代码打开一个文件进行写入，然后在两个数组中循环，将每个数组的值写入文件的新行中。循环结束后，该文件被关闭。

返回。文本文件
