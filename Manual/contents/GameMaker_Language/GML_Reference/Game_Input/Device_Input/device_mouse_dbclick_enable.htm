device_mouse_dbclick_enable

这个函数可以用来设置设备检测mb_left（鼠标左键）的双击是否为mb_right（鼠标右键）的点击。默认情况下，它被设置为"true"，这意味着每次用户快速连续点击设备屏幕 设备屏幕时，返回值与鼠标右键被点击时相同。当这一点被打开时，第一次点击将被检测为mb_left，第二次被检测为mb_right，所以请确保你使用的任何代码都考虑到这一点。 你使用的任何代码都要考虑到这一点。

语法。

device_mouse_dbclick_enable(bool)。

争论 描述
bool 设置双击检测开启（真）或关闭（假）。

返回。

布尔型

例子。

if os_type == os_windows || os_type == os_mac
{
device_mouse_dbclick_enable(false)。
}

上述代码检查运行游戏的设备是WindowsPC还是Mac，如果是其中之一，它就会禁用双击功能。

返回。设备输入
