设备_mouse_x

这个函数返回设备上一个触摸的X位置。如果你在HTML5或PC和Mac模块上运行这个函数，那么只要设备（通常是鼠标）被插入，这个值就会持续更新。 将只在屏幕被触摸时更新，并且注意，可以检测到的最大触摸次数在很大程度上取决于所使用的设备和其运行的操作系统。还要注意的是，在HTML5上，只有 "设备0 "可以正常工作。 正确，其他设备可能会给出错误的值。

语法。

device_mouse_x(device)。

争论 描述
设备 被检查的设备（从0-n）。

返回。

真实的

例子。

if device_mouse_check_button(0, mb_left)
{
if device_mouse_x(0) > x-32 && device_mouse_x(0) < x+32
{
pressed = true;
}
else
{
pressed = false;
}
}

上面的代码检查设备是否被按下，如果被按下，它就轮询设备的X位置，看它是否在参数之内。如果是的话，它将变量 "pressed "设置为真，否则就设置为假。
