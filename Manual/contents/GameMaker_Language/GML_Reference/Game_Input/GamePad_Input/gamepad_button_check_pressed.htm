游戏手柄_按钮_检查_压制

这个函数将根据给定的游戏板按钮是否被检测到被按下而返回真或假。注意，这个函数只在按钮第一次被按下时触发一次。要想 要再次触发，必须先释放按钮，然后再重新按下。如果你需要检查连续按下的按钮，你应该使用函数 gamepad_button_check().

如果你正在检查一个模拟按钮，那么检查将考虑到阈值的设置，只有当原始输入值超过给定的阈值时才会返回 true 。 (你可以用函数 gamepad_button_value() 得到这个原始值)。

语法。

gamepad_button_check_pressed（设备，按钮）。

争论 描述
设备 要检查哪个游戏板设备 "插槽"。
按钮 要检查哪个游戏手柄的按钮常数。

返回。

布尔型

例子。

if gamepad_button_check_pressed(0, gp_start)
{
audio_play_sound(snd_Start, 0, false);
room_goto(rm_Level_1);
}

上述代码将检测连接到设备 "插槽 "0的游戏板的 "开始 "按钮是否被按下，如果被按下，则改变房间。

返回。游戏手柄输入
