游戏手柄_按钮_检查_释放

这个函数将根据给定的游戏手柄按钮是否被检测到已经释放而返回真或假。请注意，这个函数只在按钮被释放的瞬间触发一次。 要想再次触发，必须先按下按钮，然后再释放一次。如果你检查的是一个模拟按钮，那么检查会考虑到阈值的设置，只有在 当原始输入值低于给定的阈值时，才会返回 true （你可以用函数 gamepad_button_value() ，得到这个原始值）。

语法。

gamepad_button_check_released（设备，按钮）。

争论 描述
设备 要检查哪个游戏板设备 "插槽"。
按钮 要检查哪个游戏手柄的按钮常数。

返回。

布尔型

例子。

if gamepad_button_check_released(0, gp_select)
{
audio_play_sound(snd_Button, 0, false);
global.Pause = !global.Pause;
}

上述代码将检测连接到设备 "插槽 "0的游戏手柄的 "选择 "按钮是否被按下，并切换全局 "暂停 "变量。

返回。游戏手柄输入
