gamepad_get_description

这个函数将返回一个描述给定游戏板的字符串，例如，PS3控制器可能返回一个类似于 "PLAYSTATION(R)3控制器 "的字符串，或者XBox360控制器可能返回 "Xbox 360控制器（XInput 标准游戏手柄）"。)这个字符串与硬件有关，返回值取决于插入正在检查的设备 "插槽 "中的游戏板。

语法。

gamepad_get_description(device)。

争论 描述
设备 哪一个游戏板 "槽 "的名字，以获得。

返回。

字符串

例子。

var gp_num = gamepad_get_device_count();
for (var i = 0; i < gp_num; i++;)
{
if gamepad_is_connected(i)
{
draw_text(32, 32 + (i * 32), gamepad_get_description(i));
}
else
{
draw_text(32, 32 + (i * 32), "No Gamepad Connected");
}
}

上面的代码将循环浏览所有的游戏板插槽，检查是否有连接的设备，然后根据游戏板是否连接到插槽上，在屏幕上画一些文字。

返回。游戏手柄输入
