gamepad_get_device_count

这个函数将告诉你两件事中的一件。要么是已连接的游戏板的数量，要么是可供游戏板连接的 "槽 "的数量。实际的返回值将取决于平台和该平台的内部配置。 的内部配置，因此该函数应与该函数一起使用 gamepad_is_connected()一起使用，以确保任何时候都有准确的连接焊盘数量。

语法。

gamepad_get_device_count()。

返回。

真实的

例子。

var gp_num = gamepad_get_device_count();
for (var i = 0; i < gp_num; i++;)
{
if gamepad_is_connected(i)
{
global.gp[i] = true;
}
else
{
global.gp[i] = false;
}
}

上面的代码在可用的游戏板（或游戏板插槽）中循环，然后检查每个游戏板的连接情况。然后用返回值将一个全局数组设置为真或假，以便在以后的检查中使用。

返回。游戏手柄输入
