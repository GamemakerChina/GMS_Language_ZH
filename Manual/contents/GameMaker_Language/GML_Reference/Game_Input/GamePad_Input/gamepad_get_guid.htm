gamepad_get_guid

通过这个函数，你可以检索连接到给定槽位索引的游戏手柄的GUID。你提供要检查的游戏板插槽（从0-11），该函数将返回以下字符串之一。

  • 如果没有GUID或者没有连接游戏手柄，则为 "无"。
  • 如果游戏板插槽索引超出范围，则"设备索引超出范围"。
  • GUID字符串，类似于 "050000005e040000fd020000ff3f00"，如果插槽有一个分配的游戏手柄，并且可以得到它的GUID。

这个函数通常会和 gamepad_get_description()来生成Windows、Android和macOS上重新映射控制器所需的SDL字符串的一部分。

语法。

gamepad_get_guid(index)。

争论 描述
指数 检查哪个游戏板 "插槽 "索引

返回。

字符串

例子。

var _guid = gamepad_get_guid(global.PadIndex);
var _desc = gamepad_get_description(global.PadIndex);
global.GamepadID = _guid + ", " + _desc;

上述代码获得GUID和名称描述字符串，然后将它们连接起来，并将最终的字符串存储在一个全局变量中，以供将来参考。
