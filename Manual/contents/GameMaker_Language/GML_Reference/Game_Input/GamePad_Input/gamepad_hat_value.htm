游戏手柄_帽子_值

通过这个函数你可以得到一个游戏板 "帽子 "的当前值。每个帽子的值都是不同方向的位掩码，其中。

  • 上=1
  • 右边=2
  • 下降=4
  • 左边=8

请注意，这些可以结合起来（例如，提供一个帽子索引为3，将是向上和向右检查），但最多只能同时按下2位。该函数将返回一个介于0和1之间的实值，其中0表示没有按下，1表示完全按下 之间的值，这取决于游戏手柄是否支持帽子的模拟输入）。

语法。

gamepad_hat_value(device, hatindex)。

争论 描述
设备 要检查哪个游戏板设备 "插槽"。
帽子指数 要检查哪一顶游戏板的帽子（或几顶）。

返回。

真实的

例子。

var _dir = gamepad_hat_value(global.PadIndex, 0);
switch (_dir)
{
case 1: y -= 3; break;
case 2: x += 3; break;
case 3: y -= 3; x += 3; break;
case 4: x -= 3; break;
case 6: y += 3; x += 3; break;
case 8: y += 3; break;
case 9: y -= 3; x -= 3; break;
case 12: y += 3; x -= 3; break;
}

上面的代码将帽子的状态 "0 "存储在一个局部变量中，然后检查返回值是什么，并采取相应的行动。

返回。游戏手柄输入
