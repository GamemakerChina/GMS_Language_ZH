游戏手柄_删除映射

这个函数可以用来从给定的游戏手柄插槽索引中移除当前设备映射。一旦被调用，该槽位索引将需要被重新映射，使用 gamepad_test_mapping()函数重新映射，如果你希望 能够正确地使用游戏手柄常量来检测输入（直接输入总是可以使用gamepad_axis/button/hat_count()和 gamepad_axis/button/hat_value()函数一起检索）。)

语法。

gamepad_remove_mapping(index)。

争论 描述
指数 从哪个游戏板索引 "槽 "中删除映射。

返回。

不适用

例子。

如果remap == true
{
gamepad_remove_mapping（global.PadIndex）。
}

上面的代码将根据一个变量的值从给定的游戏手柄索引槽中删除映射。

返回。游戏手柄输入
