gamepad_set_colour

这个函数可以用来设置PlayStation控制器中的LEDs的颜色。你指定要设置的设备插槽，然后给一个颜色，可以是任何颜色常数或使用特定颜色函数创建的颜色值或HEX值（如$FFFFFFF）。

语法。

gamepad_set_colour(device, color)。

争论 描述
设备 设置哪个游戏手柄设备 "插槽"。
颜色 要使用的颜色。

返回。

不适用

例子。

如果健康状况<10
{
gamepad_set_colour(0, c_red);
}

上面的代码将在健康变量低于10时将PlayStation控制器的LED灯设置为红色。

返回。游戏手柄输入
