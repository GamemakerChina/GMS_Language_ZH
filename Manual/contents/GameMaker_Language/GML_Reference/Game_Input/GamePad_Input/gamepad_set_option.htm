gamepad_set_option

这个函数可以用来设置任何可用的游戏板选项。你提供要设置选项的游戏板 "插槽"，以及要设置的选项字符串和要使用的值。可用的选项字符串将取决于项目运行的平台。 项目运行的平台而定，如下所列。

选项 平台 描述 价值
"allow_rotation" tvOS 设置Siri遥控器是否会被锁定旋转（即：只能垂直使用）。当解锁时（值为true），旋转控制器将改变输入，例如向左旋转90º将使触摸板的 "右 "边返回 "上"，等等。 这样，所有的输入都是相对于设备的方向的。当锁定时（值为false），那么输入将不会相对于控制器的位置而改变。 并且将始终被视为控制器处于垂直位置。 布尔型
"dpad_absolute" tvOS 设置Siri Remote触摸板是接受绝对输入，还是接受相对输入。在绝对模式下（值为true），触摸板的中心被认为是(0,0)的位置--即模拟摇杆静止时的位置或数字键盘中心的位置。 触摸板的中心位置，围绕这个位置的移动将产生输入。当设置为相对（值为false）时，(0,0)位置将被认为是在触摸板上初始触摸的位置，所有的输入将从该位置产生，而不是从该位置。 从这个位置产生，而不是垫子的中心。 布尔型

语法。

gamepad_set_option(device, option_name, value)。

争论 描述
设备 设置哪个游戏手柄设备 "插槽"。
选项_名称 要设置的选项的名称（一个字符串，见上表）。
价值 将选项设置为的值（可以是布尔值、实数、整数或字符串，见上表）。

返回。

不适用

例子。

for(var i = 0; i < 12; ++i;)
{
if gamepad_is_connected(i) && gamepad_get_description(i) == " tvOS SiriRemote"
{
if gamepad_get_option(i, "allow_rotation") == false
{
gamepad_set_option(i, "allow_rotation", true);
}
}
}

上面的代码在所有的游戏板插槽中循环，并检查tvOS平台上的 "Siri遥控器"。如果检测到一个，它就会将遥控器设置为允许输入旋转。
