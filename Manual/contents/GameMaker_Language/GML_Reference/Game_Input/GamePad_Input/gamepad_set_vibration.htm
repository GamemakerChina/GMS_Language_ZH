游戏手柄_设置振动

通过这个功能，你可以设置游戏手柄电机的振动，其中任何一个电机都使用0（无振动）到1（完全振动）的数值。请注意，这个功能没有时间限制，所以你需要使用一个变量或一个报警器来关闭振动（将电机设置为0），否则就会在一定的时间内关闭。 关闭振动（将电机设置为0），否则游戏板将继续无限期地振动。

注意：这个功能目前只适用于标准的 窗口、PS4和Xbox One目标模块。

语法。

gamepad_set_vibration(device, left_motor, right_motor)。

争论 描述
设备 要检查哪个游戏板设备 "插槽"。
左边马达 左边电机的振动量从0到1。
右边的马达 右边电机的振动量从0到1。

返回。

不适用

例子。

如果生命=0
{
gamepad_set_vibration(0, 1, 1);
alarm[0] = room_speed / 2;
}

上述代码将被用于（例如）碰撞事件中，使插入 "槽 "0的游戏板振动半秒，一旦时间过了，设置的警报将被用来再次关闭它。
