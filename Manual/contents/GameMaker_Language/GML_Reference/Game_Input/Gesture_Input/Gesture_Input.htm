手势输入

设备输入功能对于处理简单的屏幕触摸或获取设备方向都是很好的，但有时你会希望你的项目能检测到 更复杂的东西，比如能够在房间里拖动或 "弹动 "一个实例。这些事情可以用设备功能来完成，但为了使事情更简单，GameMaker Studio 2有 一系列完整的手势事件，可以在实例和全局层面上检测双击、拖动、轻击、捏住和旋转，以及返回事件的信息。 以及返回关于该事件的信息。这里列出的功能都是为了让你更好地控制这些事件的触发方式和时间。

重要的是!这些函数在HTML5目标平台上无法工作。如果使用它们，不会出现错误，而是直接被忽略。

有以下不同的手势功能。

  • 手势_获取拖动时间
  • 手势_获取拖曳距离
  • 手势_get_flick_speed
  • 手势_get_double_tap_time
  • 手势_get_double_tap_distance
  • 手势_get_tap_count
  • 手势_获取夹角距离
  • 手势_获取夹角_朝向
  • 手势_获取夹子的角度_远离
  • 手势_get_rotate_time
  • 手势_get_rotate_angle
  • 手势_拖曳时间
  • 手势_拖曳距离
  • 手势_轻弹_速度
  • 手势_double_tap_time
  • 手势_double_tap_distance
  • 手势_轻拍_计数
  • 手势_夹角_距离
  • 手势_夹角_朝向
  • 手势_夹角_远处
  • 手势_旋转_时间
  • 手势_旋转_角度
