手势_double_tap_distance

该功能用于设置在单次点击后必须再次触摸/点击屏幕的距离，以触发双击手势。该距离 以英寸为单位，默认值为0.1。

语法。

gesture_double_tap_distance(distance)。

争论 描述
距离 为双击手势事件检测设置的距离（单位：英寸）。

返回。

不适用

例子。

if gesture_get_double_tap_distance() != 0.1
{
gesture_get_double_tap_distance(0.1);
}

上述代码检查手势的双击距离是否设置为0.1英寸，如果不是，则将其设置为该值。
