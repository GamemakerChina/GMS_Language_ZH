手势_double_tap_time

此功能用于设置两次触摸/点击之间触发双击手势事件的时间。该时间以秒为单位，默认值为 为0.16。

语法。

gesture_double_tap_time(time)。

争论 描述
时间 为双击手势事件检测设置的时间（秒）。

返回。

不适用

例子。

if gesture_get_double_tap_time() != 0.16
{
gesture_get_double_tap_time(0.16);
}

上面的代码检查手势的双击时间是否被设置为0.16秒，如果不是，则将其设置为该值。
