手势_拖曳距离

此功能用于设置拖动手势事件被触摸或点击的移动所触发的距离。该距离以英寸为单位 为单位，默认值为0.1。

语法。

gesture_drag_distance(distance)。

争论 描述
距离 为拖动手势事件检测设置的距离（单位：英寸）。

返回。

不适用

例子。

if gesture_get_drag_distance() != 0.1
{
gesture_drag_distance(0.1);
}

上面的代码检查手势的拖动距离是否被设置为0.1英寸，如果不是，则将其设置为该值。
