手势_获取夹子的角度_远离

此功能用于获得一个触摸必须远离另一个触摸的角度，然后才有可能开始捏出手势。角度的测量单位是 度，默认值为45°。

语法。

gesture_get_pinch_angle_away（）。

返回。

真实（英寸

例子。

if gesture_get_pinch_angle_away() != 45
{
gesture_pinch_angle_away(45);
}

上面的代码检查手势的捏出角度是否设置为45°，如果不是，则将其设置为该值。
