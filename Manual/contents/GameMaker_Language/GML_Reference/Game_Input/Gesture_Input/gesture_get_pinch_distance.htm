手势_获取夹角距离

这个功能是用来获取在触发捏手势之前，你必须用两个手指触摸/点击屏幕并移动的距离。这个距离是 以英寸为单位，默认值为0.1。

语法。

gesture_get_pinch_distance()。

返回。

真实（英寸

例子。

if gesture_get_pinch_distance() != 0.1
{
gesture_pinch_distance(0.1);
}

上面的代码检查手势的捏合距离是否设置为0.1英寸，如果不是，则将其设置为该值。
