键盘_检查

通过这个函数，你可以检查一个键是否被按住。不像 keyboard_check_pressed()或 keyboard_check_released()函数，后者 只在按键被按下或释放时被触发一次，而这个函数在按键被按下的每一步都被触发。该函数将接受一个由函数返回的键码值 ord() (只有A-Z的大写字母或0-9的数字)，或主键盘输入页面上列出的任何vk_*常数。

语法。

keyboard_check(key)。

争论 描述
钥匙 的关键是要检查下来的状态。

返回。

Boolean

例子。

if keyboard_check(vk_left)
{
x -= 5;
}

上面的代码将检查是否有方向键被按下，并在每一步返回真时向左移动实例5像素。
