键盘_检查_按下

通过这个函数，你可以检查一个键是否被按下。不像 keyboard_check()函数，这个函数在每次按键被按下时只运行一次，所以要想再次触发它。 所以要想再次触发，必须先释放该键，然后再按下。该函数将接受一个由函数返回的键码值 ord()(只有A-Z的大写字母或0-9的数字)，或任何一个 vk_*常量，这些常量列在键盘输入主页面上。

语法。

keyboard_check_pressed(key)。

争论 描述
钥匙 要检查的键的按下状态。

返回。

布尔型

例子。

if keyboard_check_pressed(vk_anykey)
{
room_goto_next()。
}

如果玩家按下键盘上的任何一个键，上述代码将推进到下一个房间（工作方式类似于 "按任何键继续 "的提示）。

返回。键盘输入
