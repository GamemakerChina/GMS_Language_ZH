键盘_清空

通过这个函数，你可以清除当前的键盘状态，这基本上意味着，如果按键被按住，它将不再被识别，直到它再次被释放（这也不会触发键盘按键释放事件，在这个 这时也不会触发键盘按键释放事件）并再次按下。该函数将接受一个由函数返回的键码值 ord()(只有A-Z的大写字母或0-9的数字)，或主键盘输入页面中列出的任何vk_*常量 键盘输入页面上列出的任何vk_*常量。

语法。

keyboard_clear(key)。

争论 描述
钥匙 清楚的关键。

返回。

不适用

例子。

keyboard_clear(vk_space)。

上面的代码清除了空格键的状态。
