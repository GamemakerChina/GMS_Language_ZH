鼠标_检查_按钮_按下

如果被检查的鼠标按钮被按下，该函数将返回真，如果没有，则返回假。对于任何一个鼠标按钮，当它第一次被按下时，这个函数将只被触发一次，要再次触发这个函数，该按钮 将需要被释放并再次按下。请注意，在该步骤的持续时间内，它将被认为是被触发的，对于所有有任何鼠标事件或使用该相同函数的实例，也是如此。

你提供的鼠标按钮要从以下常数之一检查。

恒定 描述
mb_left 鼠标左键
mb_middle 鼠标中键（这可能不是对所有目标平台都有效）。
mb_right 鼠标右键
mb_side1* 鼠标侧键1
mb_side2* 鼠标侧键2
mb_any 鼠标的任何一个按钮
mb_none 没有鼠标按钮


* 注意：mb_side1和mb_side2 按钮仅适用于Windows、macOS、Ubuntu和HTML5。

语法。

mouse_check_button_pressed(numb)。

争论 描述
哑巴 检查哪个鼠标按键常数。

返回。

布尔型

例子。

如果 mouse_check_button_pressed(mb_left)
{
score += 50;
}

上面的代码将检查鼠标左键是否被按下，如果被按下，它将在分数上增加50分。
