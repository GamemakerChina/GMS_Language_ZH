鼠标_检查_按钮_释放

如果被检查的鼠标按钮已经被释放，该函数将返回 "true"，如果没有，则返回 "false"。对于任何一个鼠标按钮，当它被释放时，这个函数将只被触发一次，要再次触发它，该按钮 需要再次被按下并释放。你可以从以下常数中选择要检查的鼠标按钮。

恒定 描述
mb_left 鼠标左键
mb_middle 鼠标中键（这可能不是对所有目标平台都有效）。
mb_right 鼠标右键
mb_side1* 鼠标侧键1
mb_side2* 鼠标侧键2
mb_any 鼠标的任何一个按钮
mb_none 没有鼠标按钮


* 注意：mb_side1和mb_side2 按钮仅适用于Windows、macOS、Ubuntu和HTML5。

语法。

mouse_check_button_released(numb)。

争论 描述
哑巴 检查哪个鼠标按键常数。

返回。

布尔型

例子。

if mouse_check_button_released(mb_right)
{
speed = point_distance(x, y, mouse_x, mouse_y) / 10;
}

上面的代码将检查鼠标右键是否被释放，如果被释放，它将把实例的速度设置为当前x/y位置与鼠标x/y位置之间距离的十分之一。

返回。鼠标输入
