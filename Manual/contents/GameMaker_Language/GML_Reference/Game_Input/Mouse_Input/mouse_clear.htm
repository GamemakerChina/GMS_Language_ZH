鼠标清空

这个函数将清除给定的鼠标按钮的当前状态。这意味着在玩家释放按钮并再次按下之前，对其是否被按下的检查将不会返回真值（但如果在按下鼠标按钮时进行清除，则仍会检测到释放状态 但如果在鼠标按钮被按下时进行清空，则仍会检测到释放状态）。)

你可以提供本页所列的一个鼠标按键常数。

语法。

mouse_clear(button)。

争论 描述
按钮 检查哪个鼠标按键常数。

返回。

布尔型

例子。

mouse_clear(mb_any);

上面的代码将清除所有鼠标按钮的下降状态。

返回。鼠标输入
