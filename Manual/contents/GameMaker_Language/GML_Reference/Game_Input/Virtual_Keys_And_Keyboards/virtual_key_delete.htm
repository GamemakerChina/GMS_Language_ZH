虚拟键删除

如果你的游戏有不同的房间或实例来控制菜单和游戏玩法以及其他东西，那么你很可能需要在某些时候改变虚拟按键的位置和按键图。为此，你可以使用这个功能删除 删除旧的，然后再创建新的（如果需要的话）。该函数要求你提供要删除的虚拟键的ID--如通过以下方式返回的 virtual_key_add().

请注意，当房间被改变时，任何虚拟钥匙将被自动从房间中删除，所以这个功能只有在你希望在房间本身结束之前手动删除钥匙时才有必要。

语法

virtual_key_delete(index)。

争论 描述
指数 要删除的虚拟键的索引。

返回。

不适用

例子。

virtual_key_delete(global.Left)。

上述代码删除了全局变量 "global.Left "中索引的虚拟键。
