弱引用创建

通过这个函数，你可以检查各种结构的弱引用，看看它们是否仍然 "活着"。您提供一个要检查的弱引用数组（每个弱 引用都应该是用函数 weak_ref_create())，该函数将返回 true ，如果 任何 的结构仍然 被引用，则该函数将返回 false ；如果不被引用且已被垃圾回收，则返回 。请注意，如果你提供的数组中的任何一个值不是弱引用，该函数将返回 undefined 。

请注意，该函数还需要两个可选参数，第一个参数允许你提供一个要检查的数组的初始索引，还有一个参数用于设置从该索引开始要检查的位置数（长度）。只有在这些参数内的数组索引 在这些参数内将被检查。

语法。

weak_ref_any_alive(weak_ref, [index], [length])。

争论 描述
弱参考 您要检查的结构的弱引用。
[索引] OPTIONAL!阵列中的索引，以开始检查
[长度] OPTIONAL!要检查的从给定索引开始的位置数

返回。

布尔型（或未定义）。

例子。

if weak_ref_any_alive(inventory_ref_array)
{
instance_destroy(obj_Inventory_Control);
}

上面的代码检查了一个弱引用的数组，如果有任何弱引用还活着，那么就会销毁一个实例。
