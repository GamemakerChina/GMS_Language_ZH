弱引用创建

通过这个函数，您可以创建一个结构的弱引用，然后可以用来检查该结构在游戏中是否仍然 "活着"（被引用）。您可以提供您想要追踪的结构的 的引用，该函数将返回另一个结构，即该结构的弱引用。请注意，您可以通过使用函数 instanceof() 来检查一个引用是 "强 "还是 "弱"。 因为强引用将返回 "struct" 或创建该结构的构造函数的名称，如果是弱引用，则返回 "weakref" 。还要注意的是 弱引用结构会有一个变量 "ref"，可以通过访问该变量来获取相关结构的字符串引用，除非它已经被收集了垃圾，在这种情况下，它将返回 undefined 。

语法。

weak_ref_create(struct_to_track)。

争论 描述
结构_to_track 你想为其创建一个弱引用的结构

返回。

结构

例子。

inventory_ref = weak_ref_create(inventory)。

上面的代码创建了一个结构的弱引用，并将其存储在一个实例变量中供以后使用。
