游标_斯普利特

设置这个变量将指示GameMaker Studio 2使用指定的精灵作为光标（基本上每一步都将其设置为当前鼠标的X/Y位置）。默认值是-1，也就是没有光标的精灵。 光标，但你可以从游戏资产中指定任何精灵的索引，或者从外部资源中导入。请注意，没有办法控制动画速度或图像索引，所以如果精灵有子图像，这些图像将以与房间速度相同的速度循环播放。 以与房间速度相同的速度循环。要删除光标精灵，你可以把这个变量再次设置为-1。

同样值得注意的是，这个变量并不能取代游戏窗口的光标，它仍然会被正常绘制。为了避免这一点，你可以使用函数 window_set_cursor()和 将其设置为常数cr_none，这将使标准光标不可见。

语法。

cursor_sprite。

返回。

真实的

例子。

cursor_sprite = spr_CustomCursor;

上述代码将把变量 "spr_CustomCursor "中索引的精灵设定为游戏的光标精灵。
