游戏加载

有了这个功能，你就可以加载一个以前保存过的游戏，用 game_save().请注意，它将恢复用于创建保存的游戏版本，因此在它之后进行的任何更新都将不可见。 看不见。欲了解更多信息，请阅读关于 game_save().

语法。

game_load(filename)。

争论 描述
文件名 要加载的文件的名称。

返回。

不适用

例子。

if keyboard_check_pressed(ord("L") )
{
if global.Save game_load("Save.dat");
}

当玩家按下 "L "键时，如果全局变量为真，上述代码将加载一个先前保存的游戏。
