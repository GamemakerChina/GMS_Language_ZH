game_load_buffer

通过这个函数，你可以加载一个之前保存的游戏状态。游戏从先前创建的 "增长 "缓冲区加载（见缓冲区），该缓冲区必须有一个游戏状态保存在里面。 使用 game_save_buffer()函数。

语法。

game_load_buffer(buffer)。

争论 描述
缓冲区 要加载的缓冲区ID。

返回。

不适用

例子。

if keyboard_check_pressed(ord("L"))
{
if global.Checkpoint game_load_buffer(save_buff);
}

上述代码将从变量 "save_buff "中索引的缓冲区加载先前保存的游戏状态，只有当全局变量为真时，当玩家按下 "L "键。

返回。一般游戏控制
