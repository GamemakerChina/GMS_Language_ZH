角度_差异

这个函数将返回两个角度之间的最小角差，为-180度和180度之间的数值（其中正角为逆时针，负角为顺时针）。

语法。

angle_difference(ang1, ang2)

争论 描述
ang1 第一个使用的角度。
ang2 第二个使用的角度。

返回。

真实的

例子。

var pd = point_direction(x, y, mouse_x, mouse_y);
var dd = angle_difference(image_angle, pd);
image_angle -= min(abs(dd), 10)* sign(dd)。

上面的代码将得到从实例到鼠标光标的方向角度，然后得到该角度与当前角度的差。 图像_角度, 使用这个值来慢慢地朝鼠标方向旋转。
