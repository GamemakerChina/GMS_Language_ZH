点_方向

这个函数返回由指定分量[x1,y1]和[x2,y2]组成的矢量的方向，与房间的固定x/y坐标有关。例如，在下面的图片中，如果我们想得到从玩家飞船位置到敌人位置的方向 到敌人位置的方向，这样我们就可以向敌人发射导弹，那么我们就可以使用这个函数（具体代码见图片下面的例子）。

语法。

point_direction(x1, y1, x2, y2)

争论 描述
x1 矢量的第一个分量的X坐标
y1 矢量的第一个分量的y坐标
x2 矢量第二分量的X坐标
y2 矢量第二分量的y坐标

返回。

真实的

例子。

var ex, ey;
ex = instance_nearest(x, y, enemy).x;
ey = instance_nearest(x, y, enemy).y;
with (instance_create_layer(x, y, "Bullets", obj_Missile)
{
direction = point_direction(x, y, ex, ey);
}

上面的代码将获得最近的敌人的x和y坐标，然后将它们传递给子弹对象，在point_direction函数中使用，以正确设置其行进方向。

返回。角度和距离
