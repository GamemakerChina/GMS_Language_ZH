点距离

该函数返回由指定分量[x1,y1]和[x2,y2]组成的矢量的长度。例如，在下面的图片中，如果我们想得到玩家飞船位置和敌人位置之间的距离，这样我们就可以计算出如果 敌人是否近到可以射击，那么我们将使用这个函数（具体代码在图片下面的例子中）。

语法。

point_distance(x1, y1, x2, y2)。

争论 描述
x1 矢量的第一个分量的X坐标
y1 矢量的第一个分量的y坐标
x2 矢量第二分量的X坐标
y2 矢量第二分量的y坐标

返回。

真实的

例子。

var ex, ey;
ex = instance_nearest(x, y, enemy).x;
ey = instance_nearest(x, y, enemy).y;
if (point_distance(x, y, ex, ey) < 200)
{
instance_create_layer(x, y, "Bullets", obj_Missile)
}

上面的代码将得到最近的敌人的x和y坐标，然后用它们来检查它们和玩家坐标形成的向量的距离（长度）。如果该值小于200，玩家对象将创建一个 "obj_Missile "的实例。
