点距离_3d

这个函数接收提供的矢量分量，并返回矢量的长度（距离）。它的工作方式与 point_distance()的方式完全相同，但增加了对Z值的因子 (深度)，以便在三维空间中使用。

语法。

point_distance_3d(x1, y1, z1, x2, y2, z2)。

争论 描述
x1 矢量的第一个分量的X坐标
y1 矢量的第一个分量的y坐标
z1 矢量的第一个分量的Z坐标
x2 矢量第二分量的X坐标
y2 矢量第二分量的y坐标
z2 矢量的第二个分量的Z坐标

返回。

真实的

例子。

var inst, ex, ey, ez;
inst = instance_nearest(x, y, enemy);
if inst
{
ex = inst.x;
ey = inst.y;
ez = inst.z;
if point_distance_3d(x, y, z, ex, ey, ez) < 200
{
instance_create_layer(x, y, "Bullets", obj_Missile)
}
}

上面的代码将得到最近的敌人的x和y以及z坐标，然后用它们来检查它们和玩家坐标形成的向量的距离（长度）。如果该值小于200，玩家对象将创建一个 "obj_Missile "的实例。
