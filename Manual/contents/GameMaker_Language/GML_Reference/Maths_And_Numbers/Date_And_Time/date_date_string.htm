日期_date_string

通过这个函数，你可以创建一个包含给定日期的字符串，格式为日/月/年。

语法。

date_date_string(date)。

争论 描述
日期 要使用的日期。

返回。

字符串

例子。

str = date_date_string(date_current_datetime());
draw_text(32, 32, str);

这将设置 "str "来保存系统显示的当前日期和时间的格式化字符串。

返回。日期和时间
