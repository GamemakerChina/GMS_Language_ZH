Date_datetime_string

通过这个函数，你可以创建一个包含给定日期时间的字符串，格式化为函数调用时正在运行游戏的系统或设备。

语法。

date_datetime_string(date)。

争论 描述
日期 要使用的日期。

返回。

字符串

例子。

str = date_datetime_string(date_current_datetime())。

这将把 "str "设置为类似 "2011年9月15日，11:33.00"，这取决于系统对日期/时间显示的设置以及当前的日期和时间。

返回。日期和时间
