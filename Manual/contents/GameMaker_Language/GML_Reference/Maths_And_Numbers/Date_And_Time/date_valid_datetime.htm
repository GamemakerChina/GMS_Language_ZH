日期_valid_datetime

通过这个函数，你可以检查一个日期时间值是否有效（返回true）或无效（返回false）。请注意，这个函数只考虑1970年1月1日之后的有效日期时间，在此之前的任何日期时间都会 返回false，所以你最早可以检查的是。

date_valid_datetime(1970, 01, 01, 0, 0, 0)。

语法。

date_valid_datetime(year, month, day, hour, minute, second)。

争论 描述
年 要检查的年份。
月 要检查的月份。
天 要检查的日子。
小时 要检查的时间。
分 要检查的一分钟。
第二 第二个要检查的。

返回。

布尔型

例子。

如果date_valid_datetime(2011, 9, 15, 10, 3, 30)
{
mydatetime = date_create_datetime(2011, 9, 15, 10, 3, 30);
}

这将把 "mydatetime "设置为2011年9月15日，10:03.30，如果它是一个有效的值（确实如此）。
