矩阵构建

这个函数可以用来创建你自己的自定义矩阵，并将返回这个新矩阵的索引值，这个索引值应该存储在一个变量中，以便将来参考和使用。矩阵本身被创建为16个值的数组，其中前4个元素是第1行，后4个元素是第2行，等等...的4x4矩阵。

注意：当你以这种方式建立一个新的矩阵时，操作的顺序是YXZ。

语法。

matrix_build(x, y, z, xrotation, yrotation, zrotation, xscale, yscale, zscale)。

争论 描述
x 平移向量的X分量。
y 平移向量的Y分量。
z 平移矢量的Z分量。
摆动 围绕X轴旋转的角度（单位：°）。
yrotation 围绕Y轴旋转的角度（单位：°）。
xzotation 围绕Z轴旋转的角度（单位：°）。
xscale X刻度的数量。
yscale Y刻度的数量。
规模 Z刻度的数量。

返回。

矩阵指数（4x4矩阵

例子。

t_matrix = matrix_build(x, y, 0, 0, 90, 0, 1, 2, 1);

上面的代码将建立一个新的矩阵变换，并将产生的矩阵索引存储在变量 "t_matrix "中。
