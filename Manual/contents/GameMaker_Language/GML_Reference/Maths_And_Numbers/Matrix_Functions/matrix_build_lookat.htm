矩阵_构建_观察

这个函数根据下面列出的指定参数建立了一个 "看"（视图）矩阵。由于这个函数修改的是视图矩阵，而不是投影矩阵，所以你应该首先使用其他矩阵来初始化投影矩阵 函数来初始化投影矩阵。 matrix_build_projection_perspective()来初始化投影矩阵，然后用这个函数在投影矩阵中移动摄像机。

要设置视图，你首先需要定义你看的位置，这由参数（xfrom, yfrom, zfrom）来表示。接下来你必须指定你看的方向，这是通过给出第二个看向的点来完成的 参数（xto, yto, zto）。最后，你仍然可以围绕从视角到观察点的直线旋转摄像机，为了指定这一点，我们必须给出一个 "向上 "矢量--在摄像机中向上的方向。这就是 由最后三个参数（xup, yup, zup）给出。

语法。

matrix_build_lookat(xfrom, yfrom, zfrom, xto, yto, zto, xup, yup, zup) 。

争论 描述
xfrom 要看的位置的X坐标。
从 要看的位置的y坐标。
兹从 要看的位置的Z坐标。
xto 要看的位置的X坐标。
yto 要看的位置的y坐标。
zto 要看的位置的Z坐标。
xup 上升 "矢量的X坐标。
是的 上升 "矢量的y坐标。
zup 上升 "矢量的Z坐标。

返回。

矩阵指数

例子。

viewmat = matrix_build_lookat(640, 240, -10, 640, 240, 0, 0, 1, 0);
projmat = matrix_build_projection_ortho(640, 480, 1.0, 32000.0);
camera_set_view_mat(view_camera[0], viewmat) ;
camera_set_proj_mat(view_camera[0], projmat);

上面的代码创建了一个新的俯视矩阵和正视矩阵，将它们的ID存储在本地变量中，然后用它们来设置分配给view port[0]的摄像机的视图和投影矩阵。
