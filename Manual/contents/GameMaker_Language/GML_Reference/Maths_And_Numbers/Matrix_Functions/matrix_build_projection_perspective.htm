矩阵_构建_投影_视角

这个函数根据近平面的尺寸，使用下面列出的指定参数，建立一个透视投影矩阵矩阵。

语法。

matrix_build_projection_perspective(width, height, znear, zfar)。

争论 描述
w 投影的宽度。
h 投影的高度。
znear 靠近剪接平面。
zfar 远处的剪接平面。

返回。

矩阵指数

例子。

var projmat = matrix_build_projection_perspective（640, 480, 1.0, 32000.0）;
camera_set_proj_mat（view_camera[0], projmat）。

上面的代码创建了一个透视投影，然后使用返回的矩阵来设置分配给view port[0]的相机。
