矩阵_build_projection_perspective_fov

该函数使用下面列出的指定参数，基于视场建立一个透视投影矩阵矩阵。

语法。

matrix_build_projection_perspective_fov(fov_y, aspect, znear, zfar)。

争论 描述
fov 视野的角度。
方面 视野的长宽比。
znear 靠近剪接平面。
zfar 远处的剪接平面。

返回。

矩阵指数

例子。

projmat = matrix_build_projection_perspective_fov(60, 320/240, 1.0, 32000.0);
camera_set_proj_mat(view_camera[0], projmat) 。

上面的代码创建了一个视场投影矩阵，然后存储在一个变量中。然后这个矩阵被用来设置分配给view port[0]的摄像机。
