矩阵_获取

该函数返回一个包含16个值的数组，对应于给定的4x4矩阵类型，其中第1行是元素[0-3]，第2行是元素[4-7]，等等。(见主页上的图片)。 可用的矩阵有视图、投影和世界，对于这些矩阵，你可以使用以下常数之一。

恒定 描述
矩阵_视图 当前视图矩阵
矩阵投影 当前的投影矩阵
矩阵_世界 当前的世界矩阵

语法。

matrix_get(type)。

争论 描述
类型 获取矩阵值的类型（见上面列出的常数）。

返回。

数组（16个元素

例子。

v_array = matrix_get(matrix_view)。

上述代码将获得当前视图矩阵的值，并将其填充到数组 "v_array "中。
