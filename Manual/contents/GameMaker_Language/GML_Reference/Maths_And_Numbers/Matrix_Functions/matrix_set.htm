Matrix_set

这个函数接收一个由16个值组成的数组，对应于给定的4x4矩阵类型，其中元素[0-3]是第1行，元素[4-7]是第2行，等等。(见主页上的图片)。 你可以用以下方法创建这样的矩阵 matrix_build()或 matrix_get()函数来创建这样的矩阵，或者干脆自己建立一个数组并将其传入函数。可用的矩阵 类型是视图、投影和世界，对于这些类型，你可以使用以下常数之一。

恒定 描述
矩阵_视图 当前视图矩阵
矩阵投影 当前的投影矩阵
矩阵_世界 当前的世界矩阵

语法。

matrix_set(type, matrix)。

争论 描述
类型 获取矩阵值的类型（见上面列出的常数）。
基体 矩阵数据是一个数组

返回。

这是对的。

例子。

matrix_set(matrix_world, m_array)。

上述代码将把当前世界矩阵的值设置为存储在数组矩阵 "m_array "中的值。
