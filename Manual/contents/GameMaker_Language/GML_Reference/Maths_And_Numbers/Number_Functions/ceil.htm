偃旗息鼓

这个函数接收任何实数并将其四舍五入到最接近的整数。使用这个函数时要小心，因为一个常见的错误是用它将一个随机值四舍五入，并期望它总是大于1，即。

int = ceil(random(5))。

现在，你会期望这段代码总是给出一个介于1和5之间的整数，但情况可能并不总是这样，因为随机函数有一个非常小的可能性会返回0，而对0进行四舍五入仍然会得到0。这只是一种很小的 但在使用这个函数时应考虑到这种可能性。

语法。

ceil(val)。
