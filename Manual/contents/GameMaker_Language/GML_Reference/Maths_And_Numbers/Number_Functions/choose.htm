选择

有时你想为随机选择指定一些数字以外的东西，或者你想要的数字不按任何实际顺序或在任何设定的范围内。在这些情况下，你会使用choose()来生成一个随机结果。例如，比如说 你想创建一个开头有随机精灵的对象，那么你可以使用这个函数将精灵索引设置为一组给定的精灵中的一个。请注意，你可以有你需要的任意数量的参数（注意，更多的参数将意味着该函数 的解析速度会变慢）。)

注意：由于GameMaker Studio 2每次都会生成相同的初始随机种子，使调试代码变得更加容易，所以每次运行游戏时，这个函数都会返回相同的值。为了避免这种行为，请使用 randomise()在你的游戏开始时使用。

语法。

choose(val0, val1, val2... max_val)。

争论 描述
val0... max_val 一个输入值，可以是字符串、整数、变量或常数。

返回。

给定的参数之一

例子。

sprite_index = choose(spr_Cactus, spr_Flower, spr_Tree, spr_Shrub);
hp = choose(5, 8, 15, 32, 40);
name = choose("John", "Steven", "Graham", "Jack", "Emily", "Tina", "Jill" 。 "Helen"）。)
