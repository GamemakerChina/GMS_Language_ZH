lerp

通过这个函数，你可以找到相当于给定百分比的两个其他数值之间的位置的数值。因此，如果你这样做，比如说

lerp(0, 10, 0.5)

你会得到5的返回值，也就是输入值的50%。你也可以用这个函数进行外推，方法是为插值量提供一个正值或负值，这样做就像。

lerp(0, 10, 2)

将返回一个20的值。

语法。

lerp(a, b, amt)

争论 描述
a 第一个值。
b 第二个值。
amt 要插值的数量。

返回。

真实的

例子。

xx = lerp(x, x + hspeed, room_speed);
yy = lerp(y, y + vspeed, room_speed) 。

上述代码使用线性插值函数来预测一个实例在一秒钟的游戏时间后会移动到哪里。
