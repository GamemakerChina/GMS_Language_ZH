place_free

你可以使用这个函数来检查，看看调用的实例是否会与你游戏中任何标记为实体的实例发生碰撞。现在，应该注意的是，要让这个函数发挥作用，运行代码的实例必须有一个有效的碰撞 掩码（无论是对精灵本身，还是通过 mask_index)，而且它只会登记与那些有实体标记的实例的碰撞，这些实例也有一个 有效的掩码。

该函数本身的工作原理是，当把实例放在x/y参数指定的位置时，只测试与实体标记的实例的碰撞。碰撞检查可以是精确的，也可以是基于实例的边界框的。 碰撞检查可以是精确的，也可以基于实例的边界，这取决于选择了什么样的碰撞掩码。

注意，在进行检查之前，给定的x/y坐标将被浮动到最接近的整数。

语法。

place_free(x, y)。

争论 描述
x 要检查的X位置。
y 要检查的y位置。

返回。

布尔型

例子。

if place_free(mouse_x, mouse_y)
{
x = mouse_x;
y = mouse_y;
}

如果调用的实例被放置在与鼠标相同的位置，上述代码将检查是否与任何实体标记的实例发生碰撞。如果没有检测到碰撞，那么该实例的x/y坐标就会被设置为鼠标的坐标。

返回。碰撞
