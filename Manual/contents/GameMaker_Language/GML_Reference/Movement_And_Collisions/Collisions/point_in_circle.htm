点_圆

当使用这个函数时，你定义一个圆形区域，GameMaker Studio 2将计算出给定的点是否在其范围内。如果该点在定义的圆圈内，该函数将返回true，否则该 函数将返回false。

语法。

point_in_circle(px, py, x1, y1, rad)。

争论 描述
px 要检查的点的X坐标。
蟒蛇 要检查的点的y坐标。
x1 圆心的X坐标。
y1 圆心的Y坐标。
辐射 圆的半径。

返回。

布尔型

例子。

如果point_in_circle(mouse_x, mouse_y, x, y, 16)
{
over = true;
}
else
{
over = false;
}

上面的代码使用point_in_circle函数来检查鼠标位置是否落在定义的圆形区域内，如果是，则将变量 "over "设置为真，否则为假。
