点_直角形

当使用这个函数时，你定义了一个矩形区域，GameMaker Studio 2将计算出给定的点是否落在其范围内。如果该点在定义的矩形区域内，该函数将返回true，否则 该函数将返回false。

语法。

point_in_rectangle(px, py, x1, y1, x2, y2)。

争论 描述
px 要检查的点的X坐标。
蟒蛇 要检查的点的y坐标。
x1 要检查的矩形左边的X坐标。
y1 要检查的矩形顶面的y坐标。
x2 要检查的矩形右侧的X坐标。
y2 要检查的矩形底边的y坐标。

返回。

布尔型

例子。

if point_in_rectangle(mouse_x, mouse_y, x -10, y -10, x + 10, y + 10)
{
audio_play_sound(snd_click, 0, false);
}

这段简短的代码根据定义的矩形区域检查鼠标的位置，如果它落在边界内，就播放声音。
