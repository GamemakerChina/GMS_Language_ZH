销毁位置

这个函数简单地销毁所有被发现与指定位置发生碰撞的实例。碰撞是基于实例的掩码，如果掩码的任何部分与目标点重叠，那么这个函数将销毁 该实例。以这种方式销毁的实例将触发它们的Destroy和Clean Up事件。

语法。

position_destroy(x, y)。

争论 描述
x 销毁碰撞实例的X坐标。
y 销毁碰撞实例的Y坐标。

返回。

不适用

例子。

if mouse_check_button_pressed(mb_left)
{
position_destroy(mouse_x, mouse_y);
}

这将销毁所有在按下鼠标左键时与鼠标光标位置相碰撞的实例。

返回。碰撞
