职位_会议

通过这个函数，你可以检查一个位置是否与另一个实例或一个物体的所有实例发生碰撞。当你使用这个函数时，你是在检查房间里的一个实例或一个物体的单个点。这个检查将针对实例的边界 框进行检查，如果该实例有精确的碰撞检查，则针对该实例的掩码。如果你需要获得唯一的实例 id的唯一实例，如果该对象被 碰撞的对象，你应该使用 instance_position().这个函数也接受特殊关键字 all和 其他。

语法。

position_meeting(x, y, obj)。

争论 描述
x 要检查的X位置。
y 要检查的y位置。
obj 要检查碰撞的对象（或实例ID，或关键词 "所有 "或 "其他"）。

返回。

布尔型

例子。

if mouse_check_button(mb_left)
{
if !position_meeting(mouse_x, mouse_y, all) instance_create_layer(mouse_x, mouse_y, "Walls", obj_Wwall);
}

上面的代码检查鼠标左键，如果它被按下，它就检查鼠标的X/Y位置是否与任何实例发生碰撞。如果没有，那么就会创建一个 "obj_Wall "的实例。

返回。碰撞
