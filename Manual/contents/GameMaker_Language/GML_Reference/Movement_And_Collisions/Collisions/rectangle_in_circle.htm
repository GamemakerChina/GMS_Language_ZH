矩形_in_circle

这个函数将检查你定义的一个矩形区域，看它是否没有碰撞，是否完全在目标边界内，或者如果它只是接触，是否在一个定义的圆形区域。如果它们完全没有接触，该函数将返回 0，如果源点完全在目的地范围内，它将返回1，如果它们只是重叠，它将返回2。下面的图片说明了这一点。

语法。

rectangle_in_circle(sx1, sy1, sx2, sy2, x, y, rad)。

争论 描述
sx1 源矩形左边的X坐标。
sy1 源矩形顶面的Y坐标。
sx2 源矩形右侧的X坐标。
sy2 源矩形的底边的Y坐标。
x 圆心的X坐标
y 圆心的Y坐标。
辐射 检查碰撞的中心点周围的半径。

返回。

真实的

例子。

inst = instance_nearest(x, y, obj_Bullet);
if instance_exists(inst)
{
if rectangle_in_circle(inst.x - 5, inst.y - 5, inst.x + 5, inst.y + 5, x, y - 25, 20) > 0
{
hit = true;
}
}

上面的代码使用rectangle_in_circle函数来检查圆形区域内和发现的实例周围的矩形是否有碰撞。如果有碰撞（无论是边缘重叠还是被包围），那么一个变量将被设置为真。

返回。碰撞
