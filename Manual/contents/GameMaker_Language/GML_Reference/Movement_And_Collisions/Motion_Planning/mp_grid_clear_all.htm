mp_grid_clear_all

通过这个功能，你可以清除一个MP网格中所有 "禁止 "的单元。

语法。

mp_grid_clear_all(id)。

争论 描述
身份证 将要使用的mp_grid的索引。

返回。

布尔型

例子。

if !instance_exists(obj_Player)
{
mp_grid_clear_all(grid);
}

如果房间里不再有对象 "obj_Player "的实例存在，上述代码将清除变量 "grid "中索引的mp_grid，将所有的单元格标记为空闲。
