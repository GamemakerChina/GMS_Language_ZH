mp_grid_destroy

你可以使用这个函数来销毁指定的mp_grid（如函数所定义的 mp_grid_create()) 并释放分配给它的内存。你必须在MP网格结束时调用这个函数 否则你有可能得到一个内存泄漏，这意味着你的游戏会随着时间的推移而变慢，并最终崩溃。

注意：在网格被销毁后在其上使用mp_grid_*函数会导致错误。

语法。

mp_grid_destroy(id)。

争论 描述
身份证 将被销毁的mp_grid的索引。

返回。

不适用

例子。

if timer = 0
{
mp_grid_destroy(grid);
room_goto(rm_Menu)。
}

如果变量 "计时器 "等于0，上述代码将破坏变量 "grid "中索引的mp_grid，然后转到另一个房间。
