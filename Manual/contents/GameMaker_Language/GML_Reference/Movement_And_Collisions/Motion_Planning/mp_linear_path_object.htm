mp_linear_path_object

这个函数计算从当前实例位置到 "xgoal "和 "ygoal "值所指定的点的直线路径。它使用指定的步长，与函数中的 mp_linear_step(). 指定的路径必须已经存在，并将被新的路径覆盖，函数将返回是否找到了完整的路径（真）或没有（假）。只有在与指定的对象或实例没有碰撞的情况下，才会找到完整的路径，如果返回false，那么就会有一个路径被发现。 如果返回false，那么仍然会生成一个路径，但它只会运行到路径被阻挡的位置。

注意：这个函数不会移动实例。它只设置一个路径，你必须使用路径函数来移动。

语法。

mp_linear_path_object(path, xgoal, ygoal, stepsize, obj)。

争论 描述
路 要使用的路径的索引
xgoal 目标X位置。
宗旨 目标Y位置。
步长 实例每步移动的速度，单位是像素。
obj 阻止路径的对象。可以是一个对象的索引，一个实例的ID或特殊的关键字,all

返回。

布尔型

例子。

if mp_linear_path_object(path, mouse_x, mouse_y, 4, obj_Wall)
{
path_start(path, 4, 0, 0);
}

上面的代码沿着运行代码的对象和鼠标的X/Y位置之间的路径检查是否与 "obj_Wall "发生碰撞。如果没有碰撞，并且生成了完整的路径，那么它将沿着生成的路径启动对象。 生成的路径。
