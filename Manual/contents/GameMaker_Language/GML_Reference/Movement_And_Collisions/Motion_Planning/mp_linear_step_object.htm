阶梯对象(mp_linear_step_object)

通过这个函数，你可以告诉一个实例向一个特定的点迈出 "一步"，这个点由xgoal和ygoal值指定。这一步的大小（即实例每一步应该移动多少像素）由stepsize表示，并且 如果实例已经到了该位置，它将停止，不再继续移动，这与其他更简单的函数相反，如 move_towards_point().步长也是 "前面 "的距离 你可以选择实例是否在与你选择的对象或实例发生碰撞时停止。

语法。

mp_linear_step_object(xgoal, ygoal, stepsize, obj)。

争论 描述
xgoal 目标X位置。
宗旨 目标Y位置。
步长 实例每步移动的速度，单位是像素。
obj 要阻止路径的对象。可以是一个对象的索引，一个实例的ID或特殊的关键字，都是

返回。

布尔型

例子。

if mp_linear_step_object(mouse_x, mouse_y, 5, obj_Wall)
{
instance_create_layer(x, y, "Effects", obj_Explosion);
instance_destroy()。
}

上面的代码将使对象以每步5像素的速度向鼠标移动，只检查与对象 "obj_Wall "的碰撞情况。一旦它到达鼠标的位置，它将创建一个对象 "obj_Explosion"，并且 销毁自己。
