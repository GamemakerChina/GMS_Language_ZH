运动_设置

这个函数为运行代码的实例设置一个新的运动方向和一个新的速度。请注意，这不会增加实例当前的速度和方向（为此，你可以使用 motion_add()) 而是强制它进入新的设置。

语法。

motion_set(dir, speed)。

争论 描述
dir 新的方向。
速度 新的速度。

返回。

不适用

例子。

如果irandom(9) = 1
{
motion_set(random(360), 1 + random(3));
}

上述代码将使实例以随机的时间间隔改变速度和方向。
