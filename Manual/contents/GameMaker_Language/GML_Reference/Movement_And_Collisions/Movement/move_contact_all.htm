Move_contact_all

这个函数将把运行代码的实例在指定的方向上移动一定数量的像素，直到它遇到任何其他具有有效掩码的实例。你可以使用-1或0作为maxdist，默认为1000像素，即：GameMaker Studio 2将不断地将实例向上移动1000像素，直到它脱离碰撞。

语法。

move_contact_all(dir, maxdist)。

争论 描述
dir 要前进的方向。
maxdist 对象可以移动的最大距离（-1或0，默认值为1000像素）。

返回。

不适用

例子。

if !place_meeting(x, y + 1, all)
{
move_contact_all(270, -1);
}

上面的代码将检查实例下面是否有碰撞，如果没有，那么它将向下移动，直到有或直到覆盖了1000pixels。
