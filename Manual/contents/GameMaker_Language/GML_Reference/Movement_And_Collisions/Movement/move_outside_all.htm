移动_外部_所有

有了这个函数，你可以告诉一个实例向任何方向和任何数量的像素移出碰撞，maxdist的值为-1或0，默认为1000像素，即：GameMaker Studio 2将持续移动实例，直到它离开碰撞。 上升1000像素，直到它脱离碰撞。

语法。

move_outside_all(dir, maxdist)。

争论 描述
dir 要前进的方向。
maxdist 对象可以移动的最大距离（-1或0，默认值为1000像素）。

返回。

不适用

例子。

if place_meeting(x, y, all)
{
move_outside_all(90, 1);
}

上述代码将在实例与任何其他实例发生碰撞时，每一步都向上移动1像素。
