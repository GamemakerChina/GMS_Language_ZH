移动_外侧_实体

通过这个函数，你可以告诉一个实例从与任何实例的碰撞中移动出来，标记为 实体在任何方向和每一步的任何像素数，用一个 值为-1或0的maxdist默认为1000像素，也就是说：GameMaker Studio 2会将实例不断地向上移动1000像素，直到它脱离碰撞。

语法。

move_outside_solid(dir, maxdist)。

争论 描述
dir 要前进的方向。
maxdist 对象可以移动的最大距离（-1或0，默认值为1000像素）。

返回。

不适用

例子。

如果other.solid
{
var pdir;
pdir = point_direction（other.x, other.y, x, y）;
move_outside_solid(pdir, -1);
}

上述代码将进入一个碰撞事件，并检查碰撞中的 "其他 "实例是否被标记为 "实体"。如果是的话，它就会把这个实例移出碰撞范围。
