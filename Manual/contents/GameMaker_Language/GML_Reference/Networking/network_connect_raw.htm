网络_连接_raw

通过这个函数，你可以发送一个连接到服务器的请求。该函数接收要连接的套接字ID（见 network_create_socket())，并要求你提供要连接的IP地址 连接的IP地址（一个字符串）以及连接的端口，如果连接失败，将返回一个小于0的值。这个函数和 network_connect()的区别是，这个函数 可以连接到任何服务器，对原始数据不做任何处理，这意味着你必须在服务器端自己实现这些协议。注意，在默认情况下，该函数是同步的，这意味着你的游戏可能在连接时出现 "挂起"。 连接的时候，你的游戏可能会出现 "挂起"。你可以为连接设置一个超时值，或者使用函数使创建过程异步化。 network_set_config()或者使用函数 network_connect_async_raw() 来代替。

语法。

network_connect_raw(socket, url, port)。

争论 描述
插座 要使用的插座的ID。
网址 要连接的URL或IP（一个字符串）。
港口 要连接的端口。

返回。

真实的

例子。

client = network_create_socket(network_socket_tcp);
network_connect_raw(client, "www.macsweeneygames.com", 6510);

上述代码将创建一个新的TCP套接字，然后试图通过该套接字创建一个 "原始 "连接到6510端口的指定URL。
