网络_创建_socket_ext

这个函数用来为你的游戏创建一个新的客户端套接字，以便通过网络进行通信。你必须定义套接字协议（见下面的常量列表），并给出一个要使用的端口，该函数将返回一个唯一的ID，该ID应该用于该套接字的所有后续函数调用。 在该套接字的所有后续函数调用中使用，如果连接失败，则返回一个小于0的值。

恒定 描述
网络_socket_tcp 使用TCP创建一个套接字。
network_socket_udp 使用UDP创建一个套接字。
network_socket_ws 创建一个网络套接字（仅用于在HTML5上运行的项目），使用TCP。

语法。

network_create_socket_ext(protocol, port)。

争论 描述
协议 要使用的网络协议
港口 要使用的端口

返回。

真实的

例子。

client = network_create_socket_ext(network_socket_udp, 6510);

上面的代码将在6510端口创建一个新的UDP套接字。
