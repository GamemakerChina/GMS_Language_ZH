network_send_packet

通过这个函数，你可以通过网络发送一个数据 "包"。该函数接收要连接的套接字ID，然后你必须提供包含要发送的数据的缓冲区ID（关于缓冲区的更多信息 参考 - 缓冲区），最后是数据包的大小（字节数）。用这个函数发送的数据包的格式是这样的，即GameMaker Studio 2游戏在接收数据时可以 "分割 "数据包。 数据包，该函数将返回发送的数据字节数，如果发送失败，则返回一个小于0的数字。值得注意的是，这个函数返回的正在发送的数据的最终大小也将包括 GameMaker头信息，这是一个额外的12字节。

语法。

network_send_packet(socket, buffer, size)。

争论 描述
插座 要使用的插座的ID。
缓冲区 要获取数据的缓冲区的ID。
尺寸 数据的大小（以字节为单位）。

返回。

真实的

例子。

buff = buffer_load("player_save.dat");
network_send_packet(sock, buff, buffer_get_size(buff)) 。

上述信息将先前保存的缓冲区数据加载到内存中，并返回缓冲区的ID，存储在变量 "buff "中。然后，这个完整的缓冲区将作为一个数据包通过网络发送，使用变量 "sock "所标识的套接字。 "sock "标识的套接字，将这个完整的缓冲区作为数据包发送到网络上。
