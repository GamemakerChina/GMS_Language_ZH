Network_send_raw

通过这个函数，你可以通过网络发送一个 "原始 "数据包。该函数接收要连接的套接字ID，然后你必须提供包含要发送的数据的缓冲区ID（关于缓冲区的更多信息 参考 - 缓冲区），最后是数据包的大小（字节数）。发送的数据不会被GameMaker Studio 2以任何方式格式化，接收设备将以流的形式获得数据，这意味着你必须自己处理。 这意味着你必须自己处理它。该函数将返回发送的数据字节数，如果发送失败，则返回一个小于0的数字。

语法。

network_send_raw(socket, buffer, size)。

争论 描述
插座 要使用的插座的ID。
缓冲区 要获取数据的缓冲区的ID。
尺寸 数据的大小（以字节为单位）。

返回。

真实的

例子。

buff = buffer_load("player_save.dat");
network_send_raw（sock, buff, buffer_get_size(buff)）。

上述信息将先前保存的缓冲区数据加载到内存中，并返回缓冲区的ID，存储在变量 "buff "中。然后，这个完整的缓冲区将作为一个原始数据包通过网络发送，使用的套接字是 变量 "sock "识别的套接字，将这个完整的缓冲区作为原始数据包发送到网络上。
