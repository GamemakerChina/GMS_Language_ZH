network_send_udp_raw

通过这个函数，你可以使用UDP通过网络向服务器发送数据。该函数接收要连接的套接字ID、要连接的URL和要使用的端口。然后，你必须提供包含要发送的数据的缓冲区ID。 要发送的数据（关于缓冲区的更多信息见参考资料-缓冲区），最后是数据的大小（以字节为单位）。UDP是 "无连接 "的，因为你实际上不做连接，你只是发送一个数据包 而服务器从一个IP地址获得传入的数据，并必须 "按原样 "处理它。发送的数据不会被GameMaker Studio 2以任何方式格式化，接收设备将以流的形式获得数据，这意味着你必须自己处理。 这意味着你必须自己处理它。该函数将返回发送的数据字节数，如果发送失败，则返回一个小于0的数字。

注意：这个功能在运行在HTML5目标上的项目中使用时将不会起作用，HTML5项目也不能接收UDP。

语法。

network_send_udp_raw(socket, url, port, buffer, size)。

争论 描述
插座 要使用的插座的ID。
网址 要连接的网址或IP（一个字符串）。
港口 要连接的端口。
缓冲区 要获取数据的缓冲区的ID。
尺寸 数据的大小（以字节为单位）。

返回。

真实的

例子。

network_send_udp_raw(sock, "www.macsweeneygames.com", 6510, buff, buffer_tell(buff))。

上面的代码将发送一个原始的UDP数据包到6510端口上由URL定义的服务器。数据取自变量 "buff "中索引的缓冲区。
