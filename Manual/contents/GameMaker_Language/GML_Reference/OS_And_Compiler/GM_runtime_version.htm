GM_runtime_version

这个常数用于保存在首选项中定义的运行时版本号，即用于构建项目的运行时。该值被存储为 一个字符串。

语法。

GM_runtime_version。

返回。

字符串

例子。

draw_text(32, 32, date_time_string(GM_build_date));
draw_text(32, 64, "v" + GM_version);
draw_text(32, 96, " Runtime " + GM_runtime_version) 。

上面的代码获取了GM的日期、版本和运行时间常量，并将它们绘制到屏幕上。
