环境_获取变量

这个函数返回给定名称（也是一个字符串）的环境变量的值（一个字符串）。你可以通过在终端程序中输入 "env "来获得macOS和Ubuntu（Linux）上的可用环境变量。 关于Windows环境变量的信息，如果你使用的是命令提示符，那么输入 "echo%PATH%"，而使用PowerShell则是 "ls env"。请注意，在macOS和Ubuntu（Linux）上，"HOME "环境变量将返回"~//"。 环境变量将返回"~/"路径，在macOS上映射到"/Users/<username>"，在Ubuntu（Linux）上映射到"/home/<username>"。

注意：该功能仅适用于Windows、macOS和Ubuntu（Linux）。

语法。

environment_get_variable(name)。

争论 描述
名称 要检查的环境变量的名称（一个字符串）。

返回。

字符串

例子。

e_str = environment_get_variable("APPDATA")。

上述代码将返回环境变量"%appdata%"的完整路径，通常为 "C:\Users\{username}AppData\Roaming"。
