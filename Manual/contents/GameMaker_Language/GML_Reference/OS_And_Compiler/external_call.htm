外部呼叫

如果你已经创建了一个外部函数调用到一个dll或dylib，使用 external_define()来创建一个外部函数，你可以使用这个函数来调用它。你提供先前定义的函数的名称以及它所需要的每个参数（每个参数必须是正确的）。 以及它所需要的每个参数（每个参数必须是正确的类型，要么是实数，要么是字符串），该函数将返回外部调用的结果。

语法。

external_call(id, args[0...15])。

争论 描述
身份证 你想调用的函数的名称
args[0...10] 你想传递给外部函数的不同参数

返回。

值（返回的值的类型将取决于定义的函数）。

例子。

my_function = external_define("MyDLL.dll", "MyMin", dll_cdecl, ty_real, 2, ty_real, ty_real);
var _a = external_call(my_function, x, y) 。

上面的示例代码调用了一个先前定义的外部函数，并将返回值存储在一个局部变量中。
