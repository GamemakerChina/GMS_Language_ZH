os_device

这个只读变量持有下面列出的各种常量值之一，告诉你当前在哪个设备上运行游戏。请注意，这个变量已被弃用，因为函数 os_get_info() 会返回更准确的关于运行游戏的设备的信息。

语法。

os_device

返回。

恒定

恒定 描述
设备_ios_ipad 笔记本电脑
设备_ios_ipad_retina 较新的iPad，其视网膜显示屏尺寸为2048 x 1536
设备_ios_iphone6 显示屏尺寸为1334 x 750的iPhone6
设备_ios_iphone6plus 更大的iPhone 6，显示屏为1920 x 1080
设备_ios_iphone5 iPhone5的显示屏尺寸为640 x 1136)
设备_ios_iphone 较旧的iPhone/iPod Touch（480 x 320屏幕）或安卓手机
设备_ios_iphone_retina 较新的iPhone/iPod Touch带有960 x 640的视网膜显示屏
装置_仿真器 该设备实际上是一个仿真器（Windows Phone或Android）。
device_tablet 安卓平板电脑
设备_ios_unknown 未知或不是iOS

例子。

如果（os_browser = browser_not_a_browser）
{
switch (os_device)
{
case device_ios_ipad: global.Config = 2; break;
case device_ios_iphone: global.Config = 3; break;
case device_ios_iphone_retina: global.Config = 4; break;
case device_ios_unknown: global.Config = 5; break;
}
}
else
{
global.Config = 1;
}

上面的代码检查游戏是否在浏览器中运行，并根据检查结果将一个全局变量设置为一个值。
