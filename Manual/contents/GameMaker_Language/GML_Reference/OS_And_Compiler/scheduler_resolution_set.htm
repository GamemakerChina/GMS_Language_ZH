调度器_分辨率_设置

这个函数用来改变Windows线程调度器的分辨率。参数中提供的分辨率值需要以毫秒为单位。

在默认情况下。 GameMaker Studio 2使用1毫秒的分辨率，这是允许的最小的分辨率，并提供最佳的性能。把分辨率改成更高的值可能会对性能产生负面影响。 但它也可能降低功耗。

你可以提供-1作为参数，将分辨率恢复到其默认值（由Windows设置）。注意，改变调度器的分辨率会影响所有与你的游戏同时运行的进程。

注意：此功能仅适用于 窗口 仅限。

语法。

scheduler_resolution_set(milliseconds)。

争论 描述
毫秒 新的分辨率值（以毫秒为单位）或-1为默认值。

返回。

不适用

例子。

scheduler_resolution_set(2)。

这个例子将Windows线程调度器的分辨率设置为2毫秒。
