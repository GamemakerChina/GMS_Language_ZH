物理学_灯具_设置碰撞组

一旦你创建了你的灯具，你可以把它定义为碰撞组的一部分。这意味着你可以告诉具有该灯具的实例总是与其他具有绑定在它们身上的灯具的实例发生碰撞或永不发生碰撞。 相同的碰撞组的其他实例发生碰撞。可以分配的值是基于正数或负数，0是默认值（意味着具有这个灯具设置的实例会被忽略，除非为它创建一个碰撞事件）。下面是 一张图片来帮助解释。

正如你所看到的，设置碰撞组对于创建复杂的 物理体，它们与周围的物理世界相互作用，但不是相互作用。然而，应该注意的是，由于基于这样的组来计算碰撞的复杂性，建议你尽量将任何一个房间中使用的组的数量保持在绝对的最小值。 在任何一个房间里使用的组的数量保持在绝对的最低限度。

语法。

physics_fixture_set_collision_group(fixture, group)

争论 描述
固定设备 要绑定的灯具
组 该灯具被分配到的组（正值，0，或负值）。

返回。

不适用

例子。

physics_fixture_set_collision_group(fix_Wheel, -1)。

上面的代码将设置变量 "fix_Wheel "中索引的灯具为"-1 "组的一部分。

返回。固定装置
