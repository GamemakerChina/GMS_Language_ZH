物理学_灯具_设置线性阻尼

阻尼用于降低物理世界中实例的速度，与摩擦不同的是，摩擦只在两个具有固定装置的实例接触时发生。阻尼的模拟也比摩擦便宜得多，但是请注意，阻尼 并不是摩擦力的替代品；这两种效果可以而且应该一起使用。这里有一张图片来说明阻尼。

阻尼参数应该在0到无穷大之间，0意味着没有阻尼。 和无穷大意味着完全阻尼。通常你会使用0到1之间的阻尼值，但如果需要，你可以使用任何非负值。

语法。

physics_fixture_set_linear_damping(fixture, damping)

争论 描述
固定设备 夹具的索引
减震 夹具的阻尼，通常在0和1之间

返回。

不适用

例子。

physics_fixture_set_linear_damping(fix_Ball, 0.1)。

上面的代码将把 "fix_ball "中索引的夹具的线性阻尼设置为0.1。
