物理学_set_density

当你把一个灯具绑定到一个实例上时，使用 physics_fixture_bind()返回一个绑定灯具的 "id"。你可以使用这个id来设置绑定的灯具的密度值，而不是"基本 "灯具。 "基本 "灯具，在任何时候都可以使用这个函数。

语法。

physics_set_density(fixture, density)

争论 描述
固定设备 绑定灯具的ID
密度 要应用的新密度值

返回。

不适用

例子。

var density = physics_get_density(fix_id);
physics_set_density(fix_id, density - 0.1);

上面的代码得到了实例的绑定物理属性的电流密度值，然后将它们设置为不同的值。

返回。固定装置
