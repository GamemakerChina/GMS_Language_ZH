物理学_set_restitution

当你把一个灯具绑定到一个实例上时，使用 physics_fixture_bind()返回一个绑定灯具的 "id"。你可以使用这个id来设置绑定的灯具的恢复值，而不是"基本 "灯具，在任何时候都可以使用这个函数。恢复值通常被设置为0到1之间的值，但是如果需要，你可以使用更高的值，尽管结果可能是不可预测的。

语法。

physics_set_restitution(fixture, restitution)。

争论 描述
固定设备 绑定灯具的ID
归还 适用新的归还价值

返回。

不适用

例子。

var rest = physics_get_restitution(fix_id);
physics_set_restitution(fix_id, rest * 2);

上面的代码为实例的绑定物理属性获取当前的恢复值，然后将它们设置为不同的值。

返回。固定装置
