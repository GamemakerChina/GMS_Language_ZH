物理学_应用_局部冲力

在GameMaker Studio 2中，你不仅可以用物理学对物体施加力和重力，而且还可以施加冲力。这与力略有不同，因为当它被应用时，会立即影响到速度，以及。 可能影响物体的扭矩（或 "旋转"），特别是如果选择应用冲力的点的矢量不与质心对齐（注意：质心不一定与原点相同！）。 这个函数对一个实例局部施加一个脉冲。这意味着冲力的强度和方向是根据实例的原点（如果没有精灵，则是位置）来计算的，而没有考虑到 它在游戏室或物理世界中可能具有的方向或旋转。下面是一个例子。

正如你所看到的，图像中的青蛙从下面得到了一个冲动，使它 "跳 "起来。 使其 "跳跃"。确切的冲力是由我们从与xpos/ypos坐标相关的组件ximpulse/yimpulse得到的矢量定义的--这简单地意味着冲力被计算为从xpos/ypos到ximpulse/yimpulse的距离 牛顿，而方向是我们从xpos/ypos到ximpulse/yimpulse的角度。

注意：物理世界的所有计算都使用米，每米的像素比例在房间编辑器中定义，或者使用 physics_world_create().

语法。

physics_apply_local_impulse(xpos, ypos, ximpulse, yimpulse)

争论 描述
xpos 相对于原点的X坐标，在那里脉冲将被应用。
ypos 相对于原点的y坐标，在那里脉冲将被应用。
ximpulse 脉冲矢量的X分量（相对于xpos坐标）。
yimpulse 脉冲矢量的y分量（相对于ypos坐标）。

返回。

不适用

例子。

如果跳跃
{
physics_apply_local_impulse(0, 20, 0, 50);
}

如果变量 "跳跃 "为真，上面的代码将应用一个冲力为30牛顿的脉冲，其方向是相对于实例原点从下往上。

返回。力量
