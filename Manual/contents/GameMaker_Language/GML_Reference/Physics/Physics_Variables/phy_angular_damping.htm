phy_angular_damping

这个变量可以用来设置实例的角度阻尼，也可以用来获取当前的角度阻尼。阻尼是启用了物理学的实例对角度旋转的 "阻力 "大小，数值越小 允许实例在碰撞后更快地旋转（例如），而较高的值则使其需要更多的力量推动，旋转得更慢。

语法。

phy_angular_damping;

返回。

实数（单精度浮点值，如果实例没有启用物理学，则未定义）。

例子。

if place_meeting(phy_position_x, phy_position_y, obj_Water)
{
phy_angular_damping = 10;
}
else
{
phy_angular_damping = 3;
}

上述代码将检查调用实例和 "obj_Water "的实例之间的碰撞，并相应地改变角度阻尼。
