锁定旋转

这个变量可以用来设置实例是否可以受到旋转力的影响（默认为false）。如果这个变量被设置为"true"，那么任何外力（无论是来自编码的脉冲或力，还是来自碰撞）都不会影响到 实例的旋转值，这将不得不使用 phy_rotation变量

语法。

phy_fixed_rotation。

返回。

布尔型（如果实例没有启用物理学，则未定义）。

例子。

phy_rotation = 0;
phy_fixed_rotation = true;

上面的代码将把实例切换为有固定的旋转，然后设置旋转角度。
