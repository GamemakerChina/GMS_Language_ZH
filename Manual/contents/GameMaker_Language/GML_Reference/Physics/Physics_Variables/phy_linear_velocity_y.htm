phy_linear_velocity_y

这个变量可以用来获取或改变实例的线速度矢量的Y分量，其定义单位是每秒的像素（关于每步的像素，见 phy_speed_y).对于一个静态实例来说，改变这个变量 即：一个密度为0的实例），会将其变成一个运动学的实例

语法。

phy_linear_velocity_y。

返回。

实数（单精度浮点值，如果实例没有启用物理学，则未定义）。

例子。

if (phy_linear_velocity_y != 0)
{
phy_linear_velocity_y = 0;
}

上面的代码将检查线速度矢量的Y分量，如果它不是0，它将把它设置为0。
