phy_sleeping

这个只读变量返回实例当前是否在 "睡眠"（true）或不在（false），一个"睡眠 "的实例是指没有主动参与任何物理模拟的实例。当一个实例处于休息状态并且没有与另一个实例发生碰撞时，GameMaker Studio 2将使对象进入睡眠状态以节省模拟周期。

语法。

phy_sleeping;

返回。

布尔型（如果实例没有启用物理学，则未定义）。

例子。

if phy_sleeping
{
instance_destroy()。
}

上面的代码检查对象是否正在被主动模拟，如果不是，则被销毁。
