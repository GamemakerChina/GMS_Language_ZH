物理学_粒子数

这个函数将返回在一个启用了物理学的房间里活动的粒子的数量。

语法。

物理学粒子数(physics_particle_count)

返回。

真实的

例子。

如果(physics_particle_count() < physics_particle_get_max_count() )
{
physics_particle_create(0, x, y, 0, 0, c_white, 1, 1)
}

上面的代码将检查房间里的颗粒是否少于允许的最大数量，如果是，就多创建一个。
