物理学_粒子_删除_区域_盒子

通过这个函数，你可以从当前房间的物理模拟中删除（移除）所有落在定义的矩形区域边界内的粒子。该函数接收要删除的区域中心的x和y位置，以及矩形的一半宽度和高度（像素）。 以及定义该区域的矩形的一半宽度和高度（单位：像素）。

语法。

physics_particle_delete_region_box(x, y, halfWidth, halfHeight)

争论 描述
x 要删除的区域的中心的X位置。
y 要删除的区域的中心的y位置。
半宽 矩形的一半宽度。
半高 矩形的一半高度。

返回。

不适用

例子。

physics_particle_delete_region_box（mouse_x, mouse_y, 32, 32）。

上面的代码将删除在鼠标位置周围的矩形区域内发现的所有粒子。
