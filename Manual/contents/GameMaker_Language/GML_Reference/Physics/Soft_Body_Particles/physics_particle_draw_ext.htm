物理学_粒子_绘图_文本

这个函数将在所有共享相同的用户定义的类别值的粒子的位置上绘制一个精灵，这些粒子的标志与为掩码设置的标志相吻合。遮罩值的定义方式与你在创建粒子时的定义方式相同。 掩码值的定义方式与你在创建粒子时定义标志的方式相同（见函数 physics_particle_create()为例），即：使用位法或"|"来屏蔽每个标志的适当位。

注意：如果用户类别相同，该函数将绘制物理模拟中的所有粒子，其标志与掩码的标志一致，无论它们是单个粒子还是组。

绘制的粒子将忽略你创建它时设置的颜色和阿尔法值，而是使用这个函数中给出的比例、角度、颜色和阿尔法参数来绘制。请注意，这纯粹是为了在粒子的位置上绘制一个精灵 而不会影响到粒子的行为。

语法。

physics_particle_draw_ext(typemask, category, sprite, subimg, xscale, yscale, ang, col, alpha)

争论 描述
打字员 用来比较颗粒标志的掩码。
类别 要绘制的用户定义的类别（使用0来绘制所有类别）。
精灵 每个粒子要使用的精灵。
亚img 要绘制的精灵的图像_索引。
xscale 沿着X轴应用于精灵的比例因子。
yscale 沿着Y轴应用于精灵的比例因子。
愤怒 绘制精灵的角度（单位：度）。
殖民地 与精灵混合的颜色（默认为c_white）。
阿尔法 绘图时使用的alpha值（0-1）。

返回。

不适用

例子。

var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile;
physics_particle_draw_ext（flags, 2, sprBlob, 0, 2, 2, 0, c_white, 0.5）。

上述代码将为用户定义的类别 "2 "中的所有粒子绘制一个精灵，这些粒子具有与给定掩码相同的标志，沿两个轴的比例为2，α值为0.5。
