物理学_粒子组_begin

通过这个功能，你可以在一个房间里创建一个粒子组。粒子组是指创建一个特定形态所需的粒子被一次性创建为一个组，允许你创建具有各种属性的模拟软体 由使用的标志设置。标志 "是粒子常数的组合的返回值（详见函数 physics_particle_create()的细节），而 "组标志 "则是以下组常数的组合 是以下组常数的组合（或者0表示没有标志）。

恒定 描述
phy_particle_group_flag_solid 固体粒子组可以防止其他固定装置在其内部停留。如果有什么东西穿透了它，固体颗粒组就会把违规的夹具推回它的表面，使固体颗粒组具有特别强的 排斥力。
phy_particle_group_flag_rigid 刚性粒子组是指其形状不会改变的粒子，即使它们与其他固定装置相撞。

这些组标志使用位屏蔽来创建一个最终的输出值，然后通过检查来设置组的不同基本属性，除了那些由粒子标志设置的属性之外。你可以通过使用位法或"|"来屏蔽相应的位，来使用这些常量中的任何一个、一个或两个（如下面的例子所示）。

除了标志之外，你还可以设置在房间里创建粒子组的位置（给出的位置将是完成的粒子组的质量中心），初始水平和垂直速度，以及粒子组的角速度。 组的角速度，允许你创建移动、旋转的组。

你还可以设置组中所有粒子的初始颜色和阿尔法，这些值在使用函数绘制粒子时被使用。 physics_particle_draw().之后，你可以设置组内粒子的内聚力 之后，你可以设置组中粒子的内聚强度，也就是它们相互 "粘 "在一起的程度。强度可以是一个介于0和1之间的值，其中0是没有内聚力，1是最大内聚力。

最后，你必须为你的粒子组给出类别。这个值是一个任意的整数，你可以给每一个你想拥有类似属性的粒子组，它可以用来为这个特定的类别设置标志以及 以及一些其他的东西。

注意：你可以使用任何整数值作为类别，但0除外，因为0被模拟保留，用于选择其他功能中的所有类别（如用于绘图）。

需要注意的是，调用这个函数并不会在物理室中创建组。这个函数只是开始了这个过程，并为该组设置了基本的粒子属性。要真正创建粒子组，你需要给它一个 它的形式（见下面的例子），然后调用这个函数 physics_particle_group_end()，这时它将被创建。

注意：你设置的粒子类型标志也会影响粒子组的内聚力，例如，如果你将粒子标志为水类型（例如），当粒子组与另一个夹具碰撞时，它就会散开。 而如果你将粒子标记为弹簧类型，那么它将在碰撞时保持其内聚力。

语法。

physics_particle_group_begin(flags, groupflags, x, y, ang, xv, yv, ang_velocity, col, alpha, strength, category)

争论 描述
旗帜 要在粒子上设置的标志。
集团标志 要在粒子上设置的组标志（见下面的描述）。
x 创建粒子组的X位置。
y 创建粒子组的y位置。
愤怒 创建时要旋转组的角度（单位：度）。
xv 初始水平速度。
yv 初始垂直速度。
角度_速度 初始角速度。
殖民地 粒子组要使用的基本颜色。
阿尔法 粒子组要使用的基础α。
强度 设置组内颗粒间的内聚强度。
类别 用户定义的该组所属的类别。

返回。

不适用

例子。

var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile;
var groupflags = phy_particle_group_flag_solid | phy_particle_group_flag_rigid;
physics_particle_group_begin（flags, groupflags, mouse_x, mouse_y, 0, 0, 0, 0, c_white, 1, 1, 2);
physics_particle_group_circle(100);
mLastGroup = physics_particle_group_end();

上面的代码在变量中存储了粒子类型和粒子组属性的标志，然后使用这些标志在鼠标位置创建一个半径为100px的圆形粒子组。
