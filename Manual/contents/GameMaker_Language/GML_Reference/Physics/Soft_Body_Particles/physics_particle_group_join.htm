物理学_粒子组_连接

通过这个函数，你可以将两个粒子组连接在一起，然后被连接的粒子组的行为就像它们都是一个单一实体的一部分。这些组在创建时应该有重叠的边缘，因为如果它们还没有接触到，它们就不会被连接起来。 将不会被连接。该函数使用要加入的组的唯一组ID，正如函数所返回的那样 physics_particle_group_end()返回的，你可以对一个组使用该函数任意次数。 你可以对一个组使用任何次数的函数来将各种软体连接在一起。

语法。

physics_particle_group_join(to, from)

争论 描述
至 第一个加入的粒子组。
从 要加入的第二个粒子组。

返回。

不适用

例子。

var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile;
var groupflags = phy_particle_group_flag_solid;
physics_particle_group_begin(flags, groupflags, mouse_x- 45, mouse_y, 0, 0, 0, C_white, 1, 1, 2);
physics_particle_group_circle(50);
var g1 = physics_particle_group_end();
physics_particle_group_begin(flags, groupflags, mouse_x + 45, mouse_y, 0, 0, 0, c_white, 1, 1, 2);
physics_particle_group_circle(50);
var g2 = physics_particle_group_end();
physics_particle_group_join(g1, g2);

上面的代码创建了两个圆形粒子组，并将它们连接在一起。
