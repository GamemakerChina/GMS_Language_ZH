物理学_测试_重叠

这个函数可以用来检查和查看一个物理体（即：一个实例的夹具）在旋转和放置在房间的一个给定位置时是否重叠，或将重叠。 角度 "参数是调用的实例在要检查的位置所具有（或将具有）的旋转角度，"obj "参数可以是一个单一的实例id和对象索引，或者是 关键字 全部或其他。

语法。

physics_test_overlap(xpos, ypos, angle, obj)。

争论 描述
xpos 检查房间里的X位置
ypos 检查房间里的Y位置
角度 要检查的角度（调用实例的）。
obj 要检查的对象

返回。

布尔型

例子。

if physics_test_overlap(x+20, y-35, 0, obj_Bomb)
{
alarm[0] = room_speed;
ignited = true;
}

上面的代码将检查一个物理夹具重叠的位置，如果有的话，它会设置一个变量和一个警报。
