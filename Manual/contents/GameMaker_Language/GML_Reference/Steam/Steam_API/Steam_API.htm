通用蒸汽API

下面这组函数都是用来检查Steam客户端或服务器API的某些方面的可用性，并根据以下因素返回true或false 如服务器是否繁忙或用户是否登录等。这意味着这些函数应该在任何其他的SteamAPI函数调用之前使用，以确保客户端/服务器的设置是正确的，并与你的游戏进行通信。

  • 蒸汽_initialised
  • 蒸汽统计资料
  • steam_get_app_id
  • steam_get_user_account_id
  • steam_get_user_steam_id
  • 蒸馏器的名称（steam_get_persona_name
  • steam_get_user_persona_name
  • 蒸汽_用户是否已登录？
  • 蒸汽_当前游戏语言
  • 可用的语言
