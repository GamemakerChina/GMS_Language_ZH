蒸馏器的名称（steam_get_persona_name

你可以使用这个函数来返回当前登录到Steam客户端的用户的用户名。这是可见的屏幕名称，而不是唯一的用户ID（这可以通过函数找到 steam_get_user_steam_id()).

语法。

steam_get_persona_name()。

返回。

字符串

例子。

ifsteam_initialised()
{
global.p_name =steam_get_persona_name();
}

如果Steam客户端API被正确初始化，上述代码将设置一个全局变量为当前用户的屏幕名称。

返回。通用蒸汽API
