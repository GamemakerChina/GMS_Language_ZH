steam_get_user_steam_id

你可以使用这个函数来返回当前登录到Steam客户端的用户的唯一Steam 用户ID。这是一个独特的64位值，可以用来识别个别用户，而不是他们在屏幕上的用户名（这可以通过 使用函数找到 steam_get_persona_name()).由于这是一个int64，你可能需要在随后的函数中使用它，并使用 int64() 函数。

语法。

steam_get_user_steam_id（）。

返回。

整数 (int64)

例子。

ifsteam_initialised()
{
global.u_id =steam_get_user_steam_id();
}

如果正确地初始化了Steam客户端API，上面的代码将设置一个全局变量为当前用户唯一的SteamID。

返回。通用蒸汽API
