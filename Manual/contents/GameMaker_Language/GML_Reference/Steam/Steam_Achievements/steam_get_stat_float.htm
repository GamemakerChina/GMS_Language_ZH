蒸汽数据统计表

通过这个函数，你可以得到一个特定的浮点统计量的值。这个统计值应该已经在你的游戏的Steamworks控制面板账户页面上定义过了，并且传递给这个函数的字符串应该是 匹配作为控制面板上的API名称。

语法。

steam_get_stat_float(stat_name)。

争论 描述
统计数据_名称 要获得的统计资料的名称（一个字符串）。

返回。

真实的

例子。

var dist_pc = (dist / dist_max) * 100;
ifsteam_get_stat_float("Travelled") < dist_pc
{
steam_set_stat_int("Travelled", dist_pc);
}

上面的代码根据旅行的距离变量 "dist "和你可以旅行的最大距离 "dist_max "计算出一个百分比。然后，它调查统计数字 "Travelled "的当前值，如果它少于 它就会再次设置该统计数据。

返回。成就和统计数据
