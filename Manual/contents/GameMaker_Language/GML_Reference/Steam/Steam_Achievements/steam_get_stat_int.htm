蒸汽统计表

通过这个函数，你可以得到一个特定的带符号的整数统计值。这个统计值应该已经在你的游戏的Steamworks控制面板账户页面上定义过了，并且传递给这个函数的字符串应该是 匹配作为控制面板上的API名称。

语法。

steam_get_stat_int(stat_name)。

争论 描述
统计数据_名称 要获得的统计资料的名称（一个字符串）。

返回。

真实的

例子。

xp += 100;
steam_set_stat_int("Total_XP", steam_get_stat_int("Total_XP") + 100);
if steam_get_stat_int("Total_XP") > 1000
{
if !steam_get_achievement("Ach_1000XP") steam_set_achievement("Ach_1000XP");
}

上面的代码设置了一个统计量，然后检查它的最终值，以决定是否授予成就。

返回。成就和统计数据
