蒸汽云

Steam云为你的游戏提供了一个简单而透明的远程文件存储系统。所有使用GameMaker Studio 2云功能（如下所示）写入磁盘的文件都会在游戏退出后被复制到Steam服务器上。如果 用户更换了电脑，这些文件会在游戏启动前被下载到新的电脑上，这意味着游戏可以通过使用适当的Steam功能读取这些文件。Steam客户端的工作是 确保这些文件在用户可能访问的所有电脑上保持同步。

注意： 默认情况下，Steamworks上的游戏不会启用云计算。它必须事先从Steamworks游戏管理的 "云 "选项卡中启用，在那里你应该设置字节和文件配额。下次你发布你的 游戏的配置，云存储就可以使用了。
