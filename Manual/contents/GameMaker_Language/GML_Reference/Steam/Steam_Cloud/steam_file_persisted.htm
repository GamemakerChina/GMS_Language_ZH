蒸汽文件的使用情况

通过这个函数，你可以检查给定的文件，看它是否已经与Steam云同步。返回值为 "true"表示已经同步，而"false"表示没有同步。

语法。

steam_file_persisted(文件名)。

争论 描述
文件名 要检查的文件的名称。

返回。

布尔型

例子。

if !steam_file_persisted("Save.txt")
{
steam_file_share("Save.txt")。
}

上述代码将检查一个文件是否已经存储到Steam云端，如果没有，它就会同步化。

返回。蒸汽云
