写文件

你可以使用这个函数将数据写入一个文件，当用户退出游戏时，这个文件就会与Steam云同步。 如果文件已经存在，它将用新的数据字符串覆盖已经存储在文件中的任何数据。如果由于某种原因失败，该函数将返回一个0的值，如果成功则返回一个大于0的值。

语法。

steam_file_write(filename, data, size)。

争论 描述
文件名 要写入的文件的名称。
数据 要写入的数据（一个字符串）。
尺寸 要写入的数据的大小。

返回。

真实的

例子。

var fname="SaveData.txt";
var data = string(global.Level) + "|" + string(global.Points) + "|" + string(global.HP) ;
var len = string_length(data);
steam_file_write_file(fname,data, len);

上面的代码将准备一些本地变量，然后用它们来写入（或创建）一个文件，然后将其与Steam云同步。
