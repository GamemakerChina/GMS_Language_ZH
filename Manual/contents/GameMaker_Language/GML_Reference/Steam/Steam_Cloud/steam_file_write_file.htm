蒸汽_文件_写文件

通过这个函数，您可以将本地保存的文件内容复制到与Steam云同步的文件中。在使用这个函数之前，本地文件必须存在，如果因为某种原因失败，它将返回一个0的值，如果成功，则返回一个大于0的值。 如果成功，则返回一个大于0的值。

语法。

steam_file_write_file(steam_filename, local_filename)。

争论 描述
steam_filename 要复制过来的Steam云文件。
本地_文件名 用来复制的本地文件。

返回。

真正的

例子。

steam_file_write_file("rm_koala.png", "Koala2.png") 。

上面的代码将复制文件 "Koala2.png "的内容到Steam云文件 "rm_koala.png "中。

返回。蒸汽云
