蒸汽用户拥有dlc

如果你的游戏有为其创建的DLC，你可以使用这个函数在访问与之相关的任何文件之前检查并查看用户是否已经购买了它。如果玩家拥有该内容，该函数返回true，如果用户没有，则返回false。 但请注意，即使用户拥有该DLC，他们也可能没有安装，所以你应该使用附加的函数 steam_user_installed_dlc()来检查它是否已经安装 在使用它之前，你应该使用附加功能of检查它是否已经安装。请注意，Steam的ID可能是一个很大的数字，所以你可能需要把你的ID值作为一个 int64()然后再提供给这个函数。

语法。

steam_user_owns_dlc(dlc_id)。

争论 描述
dlc_id 要检查的DLC的唯一标识（一个整数int64）。

返回。

布尔型

例子。

global.Level_Max = 100;
ifsteam_user_owns_dlc(10354)
{
ifsteam_user_installed_dlc(10354)
{
global.Level_max = 200;
}
}

上述代码将检查用户是否购买并安装了ID为10354的DLC，如果是，则将一个全局变量设置为不同的值。

返回。DLC
