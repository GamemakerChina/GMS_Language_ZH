激活覆盖的用户

这个函数将打开Steam覆盖层中与所给用户ID相关的一个选定的对话。可能的对话字符串有：。

  • "steamid" - 打开Steam社区的网页浏览器，进入该用户或团体的页面。
  • "聊天"- 向指定的用户打开一个聊天窗口。

请注意，Steam的ID可能是很大的数字，所以你可能需要把你的ID值转换成一个 int64()然后再提供给这个函数。

语法。

steam_activate_overlay_user(dialog_name,steamid)。

争论 描述
dialog_name 打开覆盖层的对话（见下文）。
湿度 要使用的Steam用户ID或组ID。

返回。

不适用

例子。

var key = keyboard_lastkey;
switch (key)
{
case vk_f1:steam_activate_overlay_user("steamid",global.GameGroupID); break;
case vk_f2: steam_activate_overlay_user(" chat", global.FriendID); break;
}

上面的代码调查了最后按下的键盘键，如果是功能键1或功能键2，它将打开Steam覆盖层，查看存储在全局变量 "GamegroupID "中的Steam组，或者它将打开聊天窗口 与存储在全局变量 "FriendID "中的用户聊天。
