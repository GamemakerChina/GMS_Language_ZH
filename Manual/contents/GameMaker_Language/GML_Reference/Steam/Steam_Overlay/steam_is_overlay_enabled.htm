启用了蒸汽覆盖功能

当使用SteamAPI时，可以调用这个函数来检查Steam客户端API是否启用了覆盖功能。

语法。

steam_is_overlay_enabled()。

返回。

布尔型

例子。

global.steam_api = false;
ifsteam_initialised()
{
ifsteam_stats_ready() && steam_is_overlay_enabled()
{
global.steamapi= true;
}
}

如果Steam客户端API被正确初始化，上述代码将把一个全局变量设置为"true"，同时还有Steam统计和叠加功能，否则将把该变量设置为"false"。
