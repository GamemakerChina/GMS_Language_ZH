发送截图

通过这个功能，你可以将截图上传到当前登录的用户的Steam社区资料页面。你提供的文件名是你使用GameMaker Studio 2功能拍摄屏幕截图时创建的本地文件的名称 screen_save()或 screen_save_part().宽度和高度定义了图片的大小，该函数将 如果它由于某种原因失败了，将返回一个0的值，如果它成功了，将返回一个大于0的值。

语法。

steam_send_screenshot(filename, width, height)。

争论 描述
文件名 要上传的图像文件的名称（一个字符串）。
宽度 图像的宽度（一个整数）。
高度 图像的高度（一个整数）。

返回。

真实的

例子。

ifsteam_is_screenshot_requested()
{
var file = "Catch_The_Haggis_" + string(global.scrn_num) + ".png");
screen_save(file)
steam_send_screenshot(file, window_get_width(), window_get_height());
global.scrn_num += 1;
}

上面的代码将轮询SteamAPI的截图请求，如果有的话，就会为图片文件生成一个唯一的名字，进行截图，并将文件发送到用户的Steam社区页面。
