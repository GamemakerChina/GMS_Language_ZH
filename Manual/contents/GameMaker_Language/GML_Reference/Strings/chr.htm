计时

该函数返回一个包含与输入的Unicode代码有关的字符的字符串，用于显示。这个字符取决于当前绘图字体的字符集代码页，如果没有设置字体，它将使用机器的默认代码页。 机器的默认代码页。

语法。

chr(val)。

争论 描述
价值 用于获取字符串的Unicode代码值。

返回。

字符串

例子。

mystring = chr(53) + chr(48)。

这将把 mystring 设为 "50"（一个字符串，而不是一个整数），因为CHR(53)是 "5"，CHR(48)是 "0"。

返回。弦乐
