剪贴板_获取文本

这个函数将返回剪贴板上包含的文本的字符串。如果没有存储文本，它将返回一个空字符串""。

注意：这个功能只对 窗口目标。

语法。

clipboard_get_text()。

返回。

字符串

例子。

if clipboard_has_text()
{
str = clipboard_get_text();
clipboard_set_text("")。
}

上面的代码检查剪贴板上的文本，如果它包含任何文本，就将其作为字符串读入变量 "str"。最后，通过将剪贴板设置为空字符串来清除它。
