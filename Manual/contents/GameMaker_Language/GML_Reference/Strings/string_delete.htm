删掉的字符串

你可以使用这个函数来删除一个字符串的特定部分。因此，你提供输入的字符串和该字符串中要删除的字符的开始和结束位置（索引从1开始），该函数将返回一个没有该部分的新字符串 的新字符串。

语法。

string_delete(str, index, count)。

争论 描述
弦外之音 要复制和删除的字符串。
指数 第一个要删除的字符的位置。
数 要删除的字符数。

返回。

字符串

例子。

str1 = "Helloo World";
str2 = string_delete(str1, 5, 1);

这将把str2设置为 "Hello World"，因为它从 "Hello "中删除了额外的 "o"，即：字符串从第一个字母开始数5位，然后删除一个字符）。)
