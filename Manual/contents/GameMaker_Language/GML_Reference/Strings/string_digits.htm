字符串_digits

你可以用这个函数来解析一个给定的字符串，并从中获得任何数字。例如，假设你有这样的文本--"我已经81岁了"。使用这个函数，你会得到一个 "81 "的返回字符串。

语法。

string_digits(string)。

争论 描述
绳子 要获取数字的字符串。

返回。

字符串

例子。

var t_str = string_digits(input_str);
age = real(t_str);

上面的代码将接收输入的字符串，去除除数字以外的所有字符，然后设置变量age来保存返回字符串的实际数字值（因此，如果输入的字符串是--例如--"我是18岁"，那么该 函数将返回 "18"）。)

返回。弦乐
