字符串_format

将一个实数变成一个使用你自己的格式的字符串，你可以选择多少个 "位 "被保存到字符串中，以及多少个小数位也被保存。这两种方法都很方便，有些游戏喜欢将分数显示为一个固定的数字 的数字，而控制小数点后的位数对提高精确度有好处，因为小数点后的两位数不能提供。 string()的两位小数不能提供。如果指定的位数大于要显示的数值和/或数字 加上指定的小数位数小于总位数，那么将在数值前添加空格以弥补差额（见下面的例子）。如果给定的数值小于总和，小数点右边将加零。 少于总数和要包括的小数位数，小数点右边将加零。默认的格式是左边没有额外的空格，右边只有两个小数位，例如 "265.73"。

语法。

string_format(val, tot, dec)。

争论 描述
价值 要转为字符串的实数。
总数 要显示的主数字的总位数。零或空格将被插入，以符合这个要求。
颓废 要包括的小数位数。

返回。

字符串

例子。

str1 = string_format(1234, 8, 0);
str2 = string_format(pi, 1, 10);
str3 = string_format(pi, 5, 5);

这将设置str1为 "1234"，str2为 "3.1415926535"，str3为 "3.14159"。

返回。弦乐
