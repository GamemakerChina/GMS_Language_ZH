string_hash_to_newline

该函数返回一个字符串，其中 "#"符号已被转换为一个新行。因此，一个字符串的格式为

"Hello#World"

将被打印到屏幕上，成为。

你好
世界

重要的是!这个功能是为了在GameMaker Studio 2和以前版本的软件之间提供导入兼容性，因此，除非有必要复制过时的功能，否则不应使用这个功能。相反 你应该使用转义字符来标记一个新行。关于导入兼容性的更多信息，请看这里。

语法。

string_hash_to_newline(string)。

争论 描述
绳子 要在多行中转换的字符串

返回。

字符串

例子。

var str = string_hash_to_newline("Hello#World");
draw_text(32, 32, str);

上述代码将带有哈希符号的字符串转换为分成两行的字符串，然后将其打印到屏幕上。
