弦乐_高鹰生殖中心

这个函数将返回输入字符串的高度（像素），同时考虑到文本可能有的行间距和任何断行。它对计算文本元素之间的距离非常方便，基于组成字符串的字母中最高的那个 构成字符串的字母中最高的，因为它将用 draw_text()使用当前定义的字体，计算文本元素之间的距离。

语法。

string_height(string)。

争论 描述
绳子 要测量高度的字符串。

返回。

真实的

例子。

var hh = string_height("ABCDEFGHIJKLMNOPQRSTUVWXYZ");
draw_text(32, 32, string(score));
draw_text(32, 32 + hh, string(lives);

上面的代码将获得给定字符串的高度，然后绘制两行文本，使用返回的字符串高度作为分隔符。

返回。弦乐
