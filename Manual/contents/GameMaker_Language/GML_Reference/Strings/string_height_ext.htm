弦乐器_高度_文本

这个函数将返回输入字符串的高度（像素），同时考虑到行间距和换行宽度（定义为字符串在插入换行之前可以占据的像素数）。它非常方便地 计算文本元素之间的距离，其依据是组成字符串的字母中最高的一个，以及字符串本身的实际长度，因为它将被分散到几行中，就像用 draw_text_ext() 使用当前定义的字体绘制。分离度和宽度可以被设置为-1，以获得默认的间距。

语法。

string_height_ext(string, sep, w)。

争论 描述
绳子 要测量高度的字符串。
绵绵不绝 以像素为单位的文本行之间的距离，就像正在绘制的字符串一样。
w 换行前字符串的最大宽度（单位：像素），就像字符串被画出来一样。

返回。

真实的

例子。

var hh = string_height_ext(str_Story_Text[1], -1, 100);
draw_text_ext(32, 32, str_Story_Text[1], -1, 100);
draw_text_ext(32, 32 + hh, str_Story_Text[2], -1, 100) 。

上面的代码将获得给定字符串的高度，考虑到分行和换行宽度，然后绘制两行文本，用返回的字符串总高度作为分隔符。
