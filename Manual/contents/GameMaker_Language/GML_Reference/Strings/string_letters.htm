字符串

通过这个功能，你可以删除所有不属于字母的字符。这对于防止人们在文本输入中输入不需要的字符（如输入 "#"强制换行）是很方便的。

注意：该功能只检测并返回A-Z（大写或小写）26个字母的英文字母。

语法。

string_letters(字符串)。

争论 描述
绳子 要转换为字母的字符串。

返回。

字符串

例子。

username = string_letters(username)。

上面的代码将设置 "username "变量，只保留原始字符串的字母字符（如果原始字符串是 "MacSw33ny"，返回的字符串将是 "MacSwny"）。
