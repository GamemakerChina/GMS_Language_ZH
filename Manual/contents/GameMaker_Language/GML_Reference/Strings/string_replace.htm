替换字符串

你可以用这个函数来解析一个字符串，寻找一个特定的部分，然后用你指定的新字符串来替换。

语法。

string_replace(str, substr, newstr)。

争论 描述
弦外之音 要复制的字符串。
子层 要替换的字符串中的子串。
新的 替换前一个子串的新子串。

返回。

字符串

例子。

str1 = "Hello Earth";
str2 = string_replace(str1, "Earth", "World");

这将把str2设置为str1，但把它的 "Earth "实例替换为 "World"，结果str2变成了 "HelloWorld"。

返回。弦乐
