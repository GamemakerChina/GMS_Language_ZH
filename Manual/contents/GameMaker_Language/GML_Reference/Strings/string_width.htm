弦的宽度

这个函数将返回输入字符串的宽度（单位：像素），同时考虑到文本可能有的断行。它对于根据组成字符串的字母的总宽度来计算文本元素之间的距离是非常方便的 因为它是用 draw_text()使用当前定义的字体绘制的字符串。

语法。

string_width(string)。

争论 描述
绳子 要测量宽度的字符串。

返回。

真实的

例子。

var ww = string_width(str_Name + " ");
draw_text(32, 32, str_Name));
draw_text(32 + ww, 32, "已赢得比赛！")。

上面的代码将获得给定字符串的宽度，然后画两行文字，用返回的字符串宽度作为分隔符。
