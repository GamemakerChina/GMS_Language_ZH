xboxlive_matchmaking_send_invites

这个函数带来了Xbox Live的系统对话框，允许用户向他们的游戏发送邀请。你指定控制对话框的本地用户（以及对话框将使用谁的好友列表），他们想邀请的会话的唯一会话ID值 的唯一会话ID值，以及一些要在对话框中显示的默认文本。你可以从 async_load生成的DS地图中获得会话ID。 函数 xboxlive_matchmaking_create() 。

如果参数无效或用户没有登录Xbox Live，该函数将返回-1，否则将返回一个大于0的值。

语法。

xboxlive_matchmaking_send_invites(from_user, session_to_invite_into, text_to_add_to_invite) 。

争论 描述
from_user 用于控制对话的本地用户ID。
启用的会话 要加入的会话的唯一会话ID。
文本_to_add_to_invite 在邀请函上添加简短的文字。

返回。

整数

例子。

if gamepad_button_check_pressed(0, gp_face1)
{
var _uid = xboxlive_user_for_pad(0);
xboxlive_matchmaking_send_invites(_uid, global.session_ID, "Come and join my game!");
}

上面的代码检查游戏手柄按钮的按下情况，如果检测到有按下，那么将打开Xbox Live的会话邀请的对话框。
