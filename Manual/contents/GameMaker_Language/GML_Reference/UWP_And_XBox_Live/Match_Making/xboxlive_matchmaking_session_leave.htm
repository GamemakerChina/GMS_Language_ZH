xboxlive_matchmaking_session_leave

这个函数使创建或找到指定会话的用户离开该会话。你可以从 敏捷加载(async_load)DS地图中获取会话ID指针，该地图是 当你创建或找到一个会话时，你可以从异步社交事件中返回的DS地图中获得会话ID指针（见 xboxlive_matchmaking_create() 了解更多细节）。)

语法。

xboxlive_matchmaking_session_leave(session_id)。

争论 描述
session_id 要离开的会话的会话 ID 指针。

返回。

不适用

例子。

如果gamepad_button_check_pressed(0, gp_start)
{
xboxlive_matchmaking_session_leave（global.SessionID）;
}

上述代码检查是否有游戏手柄按钮被按下，如果检测到，那么用户将退出指定的匹配会话。
