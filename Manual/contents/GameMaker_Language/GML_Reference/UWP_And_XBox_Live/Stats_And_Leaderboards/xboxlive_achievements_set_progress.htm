xboxlive_achievements_set_progress

这个函数可以用来更新一个成就的进度。你可以提供用户ID，例如，由函数 xboxlive_get_user()返回的用户ID，然后是成就字符串 这是在XDP/Windows开发中心创建成就时分配的成就ID）。最后将进度值设置为0到100的数值。注意，成就系统会拒绝低于当前进度值的更新。 值的更新。

语法。

xboxlive_achievements_set_progress(user_id, achievement, progress)。

争论 描述
user_id 要为其设置成就的用户ID。
成就 要设置的成就（一个字符串）。
进展 要设置的进度值（从0到100）。

返回。

不适用

扩展实例。

下面是一个关于如何使用这个函数的扩展例子。首先，你要在一些事件中调用它，如房间开始或 创建。

xboxlive_stats_get_social_leaderboard（user_id, "GlobalTime", 20, 1, false, false）。

上述代码将被调用，以获得所有游戏的全球排行榜位置列表，并将产生一个社交异步事件回调，我们将以如下方式处理。

var _progress = (global.Level / global.LevelMax) * 100;
xboxlive_achievements_set_progress（user_id, "Game_Completed", _progress）。

上面的代码根据一些全局变量创建了一个百分比值，然后用它来设置 "Game_Completed "成就的进度。

返回。统计资料和排行榜
