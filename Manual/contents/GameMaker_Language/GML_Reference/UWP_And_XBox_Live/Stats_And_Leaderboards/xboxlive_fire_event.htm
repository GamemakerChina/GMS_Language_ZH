xboxlive_read_player_leaderboard

这个函数可以用来触发一个统计事件。"event_name "参数是你的游戏在XDP控制台中定义的要触发的事件的名称，下面的附加参数也取决于你为状态设置的内容 状态的设置。如果事件被发送（并且应该被服务器接收/处理），该函数将返回0；如果有错误（即：你的事件没有被设置，因为项目中包含的事件清单文件说的是另一个数字），则返回1。

语法。

xboxlive_fire_event(event_name, [stat_parameters])。

争论 描述
事件_名称 要触发的事件的名称。
统计参数 [OPTIONAL]要添加到事件的额外参数。

返回。

字符串

例子。

var _user = xboxlive_get_user(0);
var _uid = xboxlive_user_id_for_user(_user);
result = xboxlive_fire_event("PlayerSessionStart", _uid, global.guid_str, 0, 42, 42) 。

上面的代码获得了一些关于用户的信息，然后连同这些（和其他）数据一起触发了一个统计事件。
