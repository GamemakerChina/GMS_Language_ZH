xboxlive_stats_delete_stat

这个函数可以用来从统计管理器中删除一个给定用户ID的统计信息。你提供的用户ID是由（例如）函数返回的 xboxlive_get_user()返回的用户ID，然后是要删除的 要删除的状态字符串。这将清除状态值并将其从状态管理器中删除，这意味着它将不再被函数 xboxlive_stats_get_stat_names()或 xboxlive_stats_get_stat().

如果出现错误或用户ID无效，该函数将返回-1，如果函数被成功调用，则返回任何其他值。

语法。

xboxlive_stats_delete_stat(user_id, stat)。

争论 描述
user_id 要删除的用户ID指针。
统计数据 要删除的统计数字（一个字符串

返回。

真实的

例子。

for(var i = 0; i < xboxlive_get_user_count(); i++;)
{
user_id[i] = xboxlive_get_user(i);
xboxlive_stats_delete_stat（user_id[i], "HighScore"）;
}

上面的代码在连接的用户中循环，并为每个用户从统计管理器中删除指定的统计。
