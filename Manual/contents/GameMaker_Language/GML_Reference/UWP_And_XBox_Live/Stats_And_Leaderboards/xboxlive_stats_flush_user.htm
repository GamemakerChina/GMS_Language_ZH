xboxlive_stats_flush_user

这个函数可以用来将某个用户的统计数据从统计管理器中冲到实时服务器上，确保服务器上的数据是最新的。根据XBox的文档，开发者应该注意不要 因为XBox会限制请求的速度，而且XBox操作系统也会自动刷新统计数据，大约每5分钟自动刷新一次。

注意：如果在该用户上设置的统计信息无效（比如统计信息名称包含非阿尔法数字字符），那么删除该用户会返回一个错误。

要使用该函数，你需要提供由（例如）函数返回的用户ID xboxlive_get_user()返回的用户ID，以及一个优先级值（高优先级为1，低优先级为0）。 如果出现错误或用户ID无效，该函数将返回-1，如果该函数被成功调用，则将返回任何其他值。该函数还将产生一个回调，它将触发一个系统异步事件。 这个事件将有特殊的DS地图 async_load ，然后应该对以下键进行解析。

  • " id" - 将持有常数achievement_stat_event。
  • "eventname" - 将持有字符串 "StatisticUpdateComplete" 。
  • "userid" - 与请求相关的用户ID。
  • "error" - 如果成功则为0，如果出现错误则为其他值
  • "errormessage" - 一个带有错误信息的字符串，如果有的话。

注意，当你从统计管理器中删除一个用户时，统计信息将自动为你刷新。参见 xboxlive_stats_remove_user()获取更多信息。

语法。

xboxlive_stats_flush_user(user_id, priority)。

争论 描述
user_id 要使用的用户ID（一个指针）。
优先权 优先权值（0或1）。

返回。

真实的

例子。

for(var i = 0; i < array_length(user_id); ++i;)
{
xboxlive_stats_flush_user（user_id[i], 0）;
}

上面的代码循环浏览存储的用户账户ID，然后为每个用户刷新统计管理器的数据。
