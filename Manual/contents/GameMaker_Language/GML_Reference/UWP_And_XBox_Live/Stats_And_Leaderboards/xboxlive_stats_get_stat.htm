统计数据

这个函数可以用来从状态管理器中为一个给定的用户检索一个单一的状态值。你提供的用户ID是由函数（例如）返回的 xboxlive_get_user(), 然后提供你在使用xboxlive_stats_set_stat_*函数创建它时定义的状态字符串。返回值可以是一个字符串或一个实数（取决于正在检查的状态），如果状态不存在或出现错误，则为GML常数undefined。

语法。

xboxlive_stats_get_stat(user_id, stat)。

争论 描述
user_id 用户ID指针，用于获取用户的状态名称。
统计数据 要设置的统计数字（一个字符串

返回。

实数、字符串或未定义

例子。

如果（game_over == true）
{
if (xboxlive_stats_get_stat(p_user_id, "PercentDone") < 100)
{
var _val = (global.LevelsFinished / global.LevelsTotal) * 100;
xboxlive_stats_set_stat_real(p_user_id, "PercentDone", _val);
}
}

上面的代码检查一个变量，如果它是真的，那么它将检查 "PercentDone "状态的值。如果这个值小于100，就会产生一个值，并将统计值设置为这个值。

返回。统计资料和排行榜
