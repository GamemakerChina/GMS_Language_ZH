xboxlive_stats_get_stat_names

这个函数可以用来从统计管理器中检索一个给定用户的所有定义的统计信息。你提供的用户ID是由（例如）函数返回的 xboxlive_get_user(), 而该函数将返回一个包含该用户统计信息的字符串数组。如果发生错误或者用户没有统计信息，数组仍然会被返回，但其元素数为0。

语法。

xboxlive_stats_get_stat_names(user_id)。

争论 描述
user_id 用户ID指针，用于获取用户的状态名称。

返回。

阵列

例子。

var _stat_str = xboxlive_stats_get_stat_names(user_id);
for （var i = 0; i < array_length(_stat_str); ++i;)
{
xboxlive_stats_delete_stat（user_id, _stat_str[i]）;
}

上面的代码为存储在给定变量中的用户ID指针检索所有定义的统计信息，然后循环浏览数组，从统计管理器中删除指定的统计信息。
