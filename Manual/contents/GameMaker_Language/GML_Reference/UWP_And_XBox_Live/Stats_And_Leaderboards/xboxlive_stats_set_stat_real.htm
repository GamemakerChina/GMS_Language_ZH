xboxlive_stats_set_stat_real

这个函数可以用来为给定的用户ID设置一个状态的值。你提供的用户ID是由（例如）函数返回的 xboxlive_get_user()返回的用户ID，然后是要设置的状态字符串 (如果stat string不存在，那么将创建一个新的stat并设置为给定的值)和一个要设置的值(一个实数)。注意，状态名称必须是字母数字，不能有符号或空格。

当设置状态值时，任何以前的值都会被覆盖，因此，在设置之前，你要通过比较当前的状态值和新的状态值来确定状态值是否应该被更新（即检查高分是否真的是最高分）。 新的状态值，然后再设置它。

如果出现错误或用户ID无效，该函数将返回-1，如果该函数被成功调用，则返回任何其他值。

语法。

xboxlive_stats_set_stat_real(user_id, stat, value)。

争论 描述
user_id 用于设置状态的用户ID指针。
统计数据 要设置的统计数字（一个字符串
价值 将状态设置为的值（一个实数）。

返回。

真实的

例子。

如果（game_over == true）
{
if (xboxlive_stats_get_stat(p_user_id, "PercentDone") < 100)
{
var _val = (global.LevelsFinished / global.LevelsTotal) * 100;
xboxlive_stats_set_stat_real(p_user_id, "PercentDone", _val);
}
}

上面的代码检查一个变量，如果它是真的，那么它将检查 "PercentDone "状态的值。如果这个值小于100，就会产生一个值，并将统计值设置为这个值。
