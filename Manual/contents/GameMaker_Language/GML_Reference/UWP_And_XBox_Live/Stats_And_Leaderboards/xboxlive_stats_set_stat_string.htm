xboxlive_stats_set_stat_string

这个函数可以用来为给定的用户ID设置一个字符串的状态。你提供的用户ID是由（例如）函数返回的 xboxlive_get_user()返回的用户ID，然后提供要设置的状态字符串 来设置（如果状态字符串不存在，那么将创建一个新的状态并设置为给定的值），然后是要设置状态的字符串。注意，状态名称必须是字母数字，没有符号或空格，但你要设置的字符串 值可以有空格和特殊字符。

当设置状态字符串时，以前的任何字符串都会被覆盖，因此，在设置之前，你要通过与当前的状态字符串和新的字符串进行比较来确定是否应该更新状态字符串。

如果出现错误或用户ID无效，该函数将返回-1，如果该函数被成功调用，则返回任何其他值。

语法。

xboxlive_stats_set_stat_string(user_id, stat, value)。

争论 描述
user_id 用于设置状态的用户ID指针。
统计数据 要设置的统计数字（一个字符串
价值 要设置的状态值（一个字符串）。

返回。

真实的

例子。

xboxlive_stats_set_stat_string（p_user_id, "LevelName", global.LevelName）。

上面的代码将状态 "LevelName "设置为存储在给定全局变量中的字符串。
