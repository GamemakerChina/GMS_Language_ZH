xboxlive_get_user

通过这个函数，你可以检索到被索引用户的用户ID指针。如果该用户不存在，该函数将返回常数pointer_null来代替。你可以用这个函数找到当前登录的用户数量 xboxlive_get_user_count().

重要提示：这个函数只能与xboxlive_get_user_count()一起使用，不要与游戏手柄一起使用。使用 xboxlive_user_for_pad() 来代替。

语法。

xboxlive_get_user(index);

争论 描述
指数 获取用户ID的索引（一个整数）。

返回。

指针

例子。

for(var i = 0; i < xboxlive_get_user_count(); ++i;)
{
user_id[i] = xboxlive_get_user(i);
}

上面的程序循环浏览所有签到的用户，并将他们唯一的ID指针存储在一个数组中。

返回。用户和账户
