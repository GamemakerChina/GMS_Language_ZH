xboxlive_pad_count_for_user

通过这个功能，你可以了解到当前用户有多少个垫子 "槽 "在使用中（详见 xboxlive_pad_for_user()以了解更多细节）。)

语法。

xboxlive_pad_count_for_user(user_id, slot)。

争论 描述
user_id 要检查的用户ID（一个指针）。

返回。

整数

例子。

global.slots[0] = xboxlive_pad_count_for_user(user_id[0] )

上面的代码存储了与给定的用户ID指针相关的垫子的数量。

返回。用户和账户
