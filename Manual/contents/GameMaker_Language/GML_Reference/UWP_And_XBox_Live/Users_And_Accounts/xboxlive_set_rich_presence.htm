xboxlive_set_rich_presence

这个函数将为给定的用户设置丰富的存在字符串。丰富的存在感字符串在用户正在玩的游戏名称之后显示用户的游戏内活动，用连字符分开。这个字符串会显示在玩家的 在 "朋友和俱乐部 "列表中的游戏名下，以及玩家的Xbox Live用户资料中。

当使用这个函数时，你需要为用户提供用户ID指针，然后你可以将用户标记为当前在游戏中是否活跃（使用真/假）。如果设置为false，那么丰富的存在字符串将被加上 附加上"/afk "或其他适当的东西。下一个参数是要显示的rich presence字符串的ID，最后你可以（选择性地）提供一个service_config_id字符串。注意，这是一个可选的参数，因为如果你 已经调用了 xboxlive_stats_setup()你不需要在这里传递service_config_id。

关于丰富的存在以及如何设置在合作伙伴中心使用的字符串的更多信息，请参见以下微软文档。

  • 存在感字符串的概述

语法。

xboxlive_set_rich_presence(user_id, is_user_active, rich_presence_string, [service_config_id] )

争论 描述
user_id 要检查的用户的ID指针。
is_user_active 将用户标记为活跃或不活跃。
丰富的存在感_字符串 要使用的丰富的存在字符串ID（如在合作伙伴中心定义的--最多50个字符
service_config_id [OPTIONAL] 这是游戏的config_id字符串。

返回。

不适用

例子。

var pad_uid = xboxlive_user_for_pad(0);
xboxlive_set_rich_presence(pad_uid, true, "Playing_Challenge") 。

上面的代码得到分配给游戏手柄[0]槽的用户的用户ID指针，然后为该用户设置丰富的存在字符串。
