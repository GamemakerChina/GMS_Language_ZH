xboxlive_show_profile_card_for_user

通过这个功能，你可以要求XBox显示目标用户ID指针的资料卡。该功能需要请求信息的用户的用户ID指针，以及目标用户的用户ID指针。 的用户ID指针，以获得目标用户的资料卡。

语法。

xboxlive_show_profile_card_for_user(requesting_user_id, target_user_id) 。

争论 描述
requesting_user_id 请求用户的用户ID（一个指针）。
target_user_id 用户的ID（一个指针），用于获取用户的资料卡。

返回。

不适用

例子。

如果(gamepad(0, gp_start))
{
xboxlive_show_profile_card_for_user（user[0], user[1]）;
}

上面的代码检查游戏手柄按钮的按下情况，如果检测到了，就显示给定用户的资料卡。

返回。用户和账户
