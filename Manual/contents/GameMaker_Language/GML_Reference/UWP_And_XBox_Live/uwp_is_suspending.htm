uwp_is_suspending

这个函数对于一个单帧会返回true（否则返回false），其方式类似于 os_is_paused().然后游戏有一秒钟的时间来保存游戏状态，然后它就可能被系统终止。 在它可能被系统终止之前，游戏有一秒钟的时间保存游戏状态。一旦游戏完成了数据保存，它应该调用 uwp_suspend()来干净利落地暂停游戏。游戏可能也应该进入暂停状态，这样如果它被系统恢复（如果它没有被完全终止的话），它就会进入暂停状态。 系统恢复时（如果还没有完全终止的话），玩家就不会被直接扔回游戏状态。

语法。

uwp_is_suspending()。

返回。

布尔型

例子。

如果 uwp_is_suspending()
{
scr_Save_Game_Data()。
uwp_suspend()。
}

上面的代码检查应用程序是否进入暂停状态，如果是，则调用一个脚本，在暂停游戏之前保存游戏数据。
