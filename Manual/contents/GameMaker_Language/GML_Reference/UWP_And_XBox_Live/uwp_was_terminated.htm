uwp_was_terminated

通过这个函数，你可以发现游戏之前是否被系统关闭过（也就是说，这不是游戏的第一次运行）。如果它已经被关闭，那么该函数将返回true，否则将返回false。一个例子是 使用的例子是检查它，看游戏是否先前被系统终止，然后（如果它返回true）加载玩家在游戏中的前一个阶段。如果以XBox One为目标，那么在检测到这种情况时，游戏所需的行为 如果检测到这种情况，不同版本的认证文件会有所不同，所以值得检查最新版本以了解游戏应该做什么。

语法。

uwp_was_terminated()。

返回。

布尔型

例子。

如果xboxlivewas_terminated()
{
buffer_load("Player_Save.sav")。
global.GameLoaded = true;
}

上面的代码检查应用程序之前是否被系统终止，如果是，它将加载一个缓冲区并设置一个全局变量。
