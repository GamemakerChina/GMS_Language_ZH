创建数组

通过这个函数，你可以创建一个指定大小的一维数组。你告诉函数要创建的数组的长度，它将返回数组的 "句柄"，然后你可以把它赋给一个变量。以这种方式创建的数组将有 每个条目都被初始化为0，除非你指定一个（可选）初始化值。如果你为初始化数组提供了额外的值，那么新数组中的所有索引都将被设置为这个值，而不是0，但是请注意，在这种情况下，该函数的性能开销会更大。 在这种情况下会有更大的性能开销。

语法。

array_create(size, [value]);

争论 描述
尺寸 要创建的数组的大小。
价值 用来初始化所有数组索引的值（可选）。

返回。

阵列参考

例子。

instance_array = array_create(100, noone);

上述代码将创建一个包含100个条目的新数组，每个条目都设置为关键字noone，然后将其分配给变量 "instance_array"。

返回。可变的功能
