Array_equals

通过这个函数，你可以检查两个数组是否相等（相等或相同）。你给出要检查的两个数组，如果它们相等（要么相等，要么相同），该函数将返回真，如果不相等，则返回假。 则返回 "true"。注意，这与使用==检查两个数组是否相同不同，后者不会检查两个数组是否持有相等的值，而只是检查数组是否引用了 相同的初始数组。比如说

var a = [1,2,3,4];
var b = [1,2,3,4];
if (a == b)
{
show_debug_message( "这将永远不会发生，因为a和b没有引用相同的数组");
}
if (array_equals(a, b))
{
show_debug_message( "这将启动");
}

语法。

array_equals(var1, var2)。

争论 描述
var1 第一个数组的索引。
变化2 第二个数组的索引。

返回。

布尔型

例子。

if !array_equals(inventory_array, item_array)
{
var len = array_length(inventory_array);
array_copy(item_array, 0, inventory_array, 0, len);
}

上面的代码将检查两个数组，看它们是否持有相等的值，如果没有，那么代码将把数组 "inventory_array "的全部内容复制到数组 "item_array "中。

返回。可变的功能
