数组长度

通过这个函数你可以得到一个数组维度的长度（条目数）。你提供数组索引值，该函数将返回一个整数值，代表数组包含的条目数。这个函数也可以用于 多维数组，只要你在提供数组索引时指定你想获得哪一维的长度，按照这个模式。

// 返回数组的第一维
val = array_length(my_array);

// 返回数组的第二维
val = array_length(my_array[0]);

// 返回数组的第三维
val = array_length(my_array[0][0]);

// 等等。

语法。

array_length(array_index);

争论 描述
阵列_索引 要检查的数组的索引。

返回。

整数

例子。

for (var i = 0; i < array_length(a); ++i;)
{
a[i] = -1;
}

上面的代码将循环浏览一个数组，并将每个条目设置为-1。

返回。可变的功能
