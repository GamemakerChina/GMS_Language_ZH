array_push

通过这个函数，你可以把一个（或多个）值推到一个数组的末端，而不必知道数组的长度。该函数要求你提供一个存放数组的变量，以及至少一个要推送的值，尽管 你可以选择提供更多的参数，这些参数将以连续的顺序被推送到数组中。

语法。

array_push(variable, value, [value], [value], [etc...])。

争论 描述
变量 保存数组的变量。
价值 要推到数组末端的值
[价值]，[价值]，[等...] OPTIONAL!要推送到数组上的更多数值

返回。

不适用

例子。

array_push(score_array, obj_Player1.scr, obj_Player2.scr, obj_Player3.scr, obj_Player4.scr) 。

上面的代码将把四个值推到给定数组的末端。

返回。可变的功能
