的实例

当使用构造函数创建结构时，该函数可用于获取用于创建该结构的函数名称，并且 新的操作符创建结构时，可使用该函数获取用于创建结构的函数名称。您提供带有要检查的结构引用的变量，该函数将返回带有该函数名称的字符串或未定义。请注意，如果您 传递给函数一个结构字（即：没有使用构造函数创建的结构），那么它将简单地返回字符串 "struct"。这个函数也可以用来检查一个结构引用是否是一个 弱引用还是 不是，在这种情况下，该函数将返回字符串 "weakref"，而不是创建该结构的函数的名称，更多信息请参见函数 weak_ref_create())。

语法。

instanceof(struct_id)。

扩展实例。

在这个例子中，我们必须首先定义将作为结构的构造函数的函数。

function init_struct(_a, _b, _c) constructor
{
a = _a;
b = _b;
c = _c;
}

然后，这个函数可以与new操作符一起使用，以创建一个结构，并将变量设置为函数中使用的参数值来填充它。

mystruct = new init_struct(10, 100, "Hello World")。

然后我们可以像这样获得所使用的函数的名称。

var _name = instanceof(mystruct);
if is_string(_name)
{
show_debug_message(_name)。
}
