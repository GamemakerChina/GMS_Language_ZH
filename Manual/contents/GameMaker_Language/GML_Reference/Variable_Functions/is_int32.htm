is_int32

这个函数返回一个给定的变量是否是323位的整数。在某些情况下，你想在GameMaker Studio 2中检查并查看一个变量的数据类型，这时你会使用这个函数。 它返回真或假，取决于该值是否是32位的int。

语法。

is_int32(n)。

争论 描述
n 要检查的参数。

返回。

布尔型

例子。

if !is_int32(val)
{
show_debug_message("不是32位整数！");
}

上面的代码检查变量 "val "是否包含int32，如果不包含则在调试控制台上显示一条信息。

返回。可变的功能
