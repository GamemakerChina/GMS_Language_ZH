is_numeric

这个函数返回一个给定的变量是否是一个数值（实数、int32、int64或布尔值）。在某些情况下，你想检查GameMaker Studio 2中的一个变量是否持有任何数字值，这时你会使用这个函数。 这时你会使用这个函数。如果给定的输入是数字，该函数将返回真，否则返回假。

语法。

is_numeric(n)。

争论 描述
n 要检查的输入。

返回。

布尔型

例子。

if is_numeric(val)
{
current_val += val;
}

上面的代码检查变量 "val"，看它是否是一个数值，如果是，就把它添加到给定的变量中。

返回。可变的功能
