is_real

这个函数返回一个给定的变量是否是一个实数（单数、双数或整数）。在某些情况下，你想检查并查看一个变量是否持有一个实数，这时你就会使用这个函数。它并不返回 实数，而是返回真或假，所以一个值，例如，"fish "将返回假，然而一个值为200将返回真。

注意：这个函数对于任何枚举值都会返回false，因为它们被存储为int64。检查该数据类型的正确函数是 is_int64().

语法。

is_real(n)。

争论 描述
n 要检查的参数。

布尔型

例子。

if is_real(val)
{
score += val;
}

上面的代码检查变量 "val "是否是一个实数，如果是，就把它加到分数上。
