is_undefined

这个函数返回一个给定的变量是否被定义。在某些情况下，你想检查并查看一个变量在GameMaker Studio 2中持有什么数据类型，这时你就会使用这个函数。它返回真或假，取决于 取决于该值是否被定义。

注意：这个函数不能用来验证一个变量的存在。使用 variable_instance_exists()或 variable_global_exists()来代替。

语法。

is_undefined(n)。

争论 描述
n 要检查的参数。

返回。

布尔型

例子。

var val = ds_map_find_value(map, 13);
if is_undefined(val)
{
show_debug_message("地图条目不存在！");
}

上面的代码检查变量 "val "是否是未定义的，如果是，则显示一个调试信息。

返回。可变的功能
