方法

通过这个函数，你可以将以前定义的任何函数绑定到一个给定的实例或结构上，创建一个方法变量，以后可以使用。 以后使用。你提供要使用的实例ID（它必须是一个活跃的、在房间里的实例，不能是一个对象ID）或一个结构引用，以及你想绑定的函数的ID。 该函数将返回一个方法，可以从它被分配到的变量中调用（见下面的代码例子）。值得注意的是，你可以绑定内置函数和用户定义的函数。 你也可以提供undefined作为实例/结构的参数，这意味着当前的 自我范围将 被用于绑定。

语法。

method(struct_ref_or_instance_id, function)。

争论 描述
结构_引用_或实例_ID 要使用的结构或实例的唯一引用或ID值（或未定义的自我）。
功能 要使用的函数的ID

返回。

方法

例子。

var _inst = instance_position(mouse_x, mouse_y, obj_Enemy);
if instance_exists(_inst)
{
enemy_func = method(_inst, enemy_ai);
}

上面的代码将检查在给定的位置是否存在一个实例，如果存在，则将一个函数与该实例绑定，并作为方法变量 "enemy_func "返回。

返回。可变的功能
