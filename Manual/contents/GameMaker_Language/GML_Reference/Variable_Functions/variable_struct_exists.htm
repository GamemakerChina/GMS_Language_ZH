变量结构存在

通过这个函数，您可以检查一个变量是否存在于给定的结构中。您可以提供要使用的结构参考，以及要检查的字符串变量名称（见下面的示例代码）。该函数 将在结构中存在给定名称的变量时返回true，否则返回false。

语法。

variable_struct_exists(struct, name)。

争论 描述
结构 要检查的结构参考
名称 要检查的结构变量的名称（作为一个字符串）。

返回。

布尔型

例子。

if !variable_struct_exists(mystruct, "shields"))
{
mystruct.shields = 0;
}

上述代码将检查名为 "shields "的变量是否存在于给定的结构中，如果不存在，则将其创建并初始化为0。

返回。可变的功能
