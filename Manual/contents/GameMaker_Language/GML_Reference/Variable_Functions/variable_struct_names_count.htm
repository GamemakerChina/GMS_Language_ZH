变量结构名称计数

通过这个函数，您可以找到为一个结构定义的变量总数。您提供要检查的结构ID，该函数将返回遇到的变量数量的整数，或者（如果不存在给定ID的结构）。 -1.

语法。

variable_struct_names_count(struct_id)。

争论 描述
结构_id 要检查的结构的唯一ID值。

返回。

整数

例子。

var _num = variable_struct_names_count(mystruct);
show_debug_message("Struct Variables = " + string(_num));

上述代码将检索给定结构中的变量数量，并在控制台输出中显示该值的调试信息。

返回。可变的功能
