变量_结构_删除

通过这个函数，您可以从一个结构中删除一个变量。您可以提供要移除变量的结构ID和要移除的变量的字符串名称。

语法。

variable_struct_set(struct, name)。

争论 描述
结构 移除该变量的结构参考
名称 要删除的变量的名称（作为一个字符串）。

返回。

不适用

例子。

if variable_struct_exists(mystruct, "shields")
{
variable_struct_remove(mystruct, "shields");
}

上述代码将检查给定的变量是否存在于给定的结构中，如果存在，则将其删除。

返回。可变的功能
