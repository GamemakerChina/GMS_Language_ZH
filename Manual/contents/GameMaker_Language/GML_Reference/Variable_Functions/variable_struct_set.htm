变量结构设置

通过这个函数，您可以设置结构中给定变量的值。您可以提供结构引用以及要设置的字符串变量的名称（见下面的示例代码），最后是要设置的值（可以是 任何有效的数据类型）。)如果该变量在结构中不存在，将被创建，然后赋值。

语法。

variable_struct_set(struct, name, val)。

争论 描述
结构 要设置的结构参考
名称 要设置的变量的名称（作为一个字符串
价值 将变量设置为的值

返回。

不适用

例子。

if !variable_struct_exists(mystruct, "shields")
{
variable_struct_set(mystruct, "shields", 0);
}

上述代码将检查给定结构中是否存在给定的变量，如果不存在，则创建并设置为0。

返回。可变的功能
