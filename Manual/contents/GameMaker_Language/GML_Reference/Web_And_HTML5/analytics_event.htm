分析_事件

这个函数将发送指定的文本到你通过HTML5游戏选项设置的分析提供者那里。这个函数可以用来创建一个自定义的分析，以跟踪所使用的供应商范围之外的东西 在所使用的提供商的范围之外。

语法。

analytics_event(string)。

争论 描述
绳子 发送给提供者的一个字符串

返回。

不适用

例子。

如果 hs_new
{
analytics_event("New hiscore of " + string (score));
}

上面的代码将检查一个变量是否为真，如果为真，那么将用指定的字符串触发一个特殊的分析事件。
