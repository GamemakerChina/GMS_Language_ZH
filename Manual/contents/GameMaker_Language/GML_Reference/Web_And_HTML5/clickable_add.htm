可点击_添加

通过这个函数，你可以创建一个自定义的、可点击的图标，作为游戏画布之外的 "按钮"，但在浏览器窗口本身。该函数将返回一个ID值，该值必须存储在一个变量中，以便在所有引用新图标的 的所有按钮函数中使用。这个函数非常有用，可以防止浏览器在点击按钮时创建一个弹出式窗口，因为这一切都通过GameMaker Studio 2本身完成。

你必须首先设置位置，这是相对于浏览器窗口的（0,0）（左上角），然后给你希望按钮图标的精灵的纹理页条目（见。 sprite_get_tpe()). 然后你可以指定一个要打开的URL或者一个要打开或运行的脚本（来自GameMaker Studio 2的资产浏览器）。你指定的 "目标 "参数与你使用open()方法时的标准JavaScript "名称 "值相同。 使用open()方法时，你指定的 "目标 "参数与标准的JavaScript "名称 "值相同（请注意，除了"_self "以外的所有参数都可能导致浏览器阻塞，或者询问用户是否愿意允许），"参数 "参数与JavaScript "规格 "参数相同，用于 控制新窗口/标签应该显示哪些属性（不是所有的浏览器都支持所有的功能）。如果你使用该图标来运行一个内部脚本，这最后两个参数可以留作空字符串""。

有效的目标是。

目标 描述
_blank 在一个新的窗口或标签中打开链接的文件。
亲自出马 在被点击的同一框架内打开链接的文件（这是默认的）。
父/母 打开父框架中的链接文档。
_top 在整个窗口中打开链接的文件。

有效参数为：。

参数 描述
'高度=[px]' 窗口的高度，最小值为100。
'宽度=[px]' 窗口的宽度，最小值为100。
'left=[px]' 窗口的左边位置。
'top=[px]' 窗口的顶部位置（仅限IE）。
'location=[boolean]' 是否显示地址栏（默认为1）。
'menubar=[boolean]' 是否显示菜单栏（默认为1）。
'resizable=[boolean]' 窗口是否可以调整大小（默认为1）。
'scrollbars=[boolean]' 是否显示滚动条（默认为1）。
'status=[boolean]' 是否添加状态栏（默认为1）。
'titlebar=[boolean]' 是否显示标题栏。除非调用的应用程序是一个HTML应用程序或一个受信任的对话框，否则这一点会被忽略（默认为1）。
'toolbar=[boolean]' 是否显示浏览器工具条（默认为是）。


还值得注意的是，如果你传入的URL参数是一个脚本函数的名称（即：不是以 "http://"开头），并且该脚本被称为 gmcallback_* 那么你就可以直接获得html5事件的反馈。GameMaker Studio 2脚本函数名称如果以gmcallback_ 开头，则不会被混淆。

语法。

clickable_add(x, y, tpe, url, target, params)。

争论 描述
x 窗口中的X位置。
y 窗口中的y位置。
帐目 要使用的精灵的纹理页条目。
网址 要链接的URL（网站地址），或要运行的脚本。
目标 这是打开URL的目标区域（见描述）。
参数 图标的各种参数（见描述）。

返回。

真实的

例子。

home_but = clickable_add(32, 32, sprite_get_tpe(spr_MS_Home, 0), "http://macsweeney_games.com", "_blank", "width=700, height=500, menubar=0, toolbar=0, scrollbars=0") 。

上述代码在游戏画布运行的页面的位置（32，32）创建了一个可点击的DOM图标。该图标使用从纹理页面引用的精灵 "spr_MS_Home"，当点击该图标时将打开一个新窗口 为指定的URL打开一个新窗口，并为该窗口定义属性。
