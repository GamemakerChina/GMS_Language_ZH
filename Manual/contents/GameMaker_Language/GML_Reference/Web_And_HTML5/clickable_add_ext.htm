可点击_添加_文本

通过这个函数，你可以创建一个自定义的、可点击的图标，作为游戏画布之外的 "按钮"，但在浏览器窗口本身。该函数将返回一个ID值，该值必须存储在一个变量中，以便在所有引用新图标的 的所有按钮函数中使用。这个函数非常有用，可以防止浏览器在点击按钮时创建一个弹出式窗口，因为这一切都通过GameMaker Studio 2本身完成。

这个函数的工作原理与 clickable_add()函数，你必须设置相对于浏览器窗口（0,0）的位置，给出你希望按钮的精灵的纹理页面条目 图标的纹理页条目（见。 sprite_get_tpe())，指定一个要打开的URL或一个要打开或运行的脚本（来自GameMaker Studio 2的资产浏览器），然后 设置要使用的目标和参数。然而，这个函数还有两个额外的选项，允许你设置图标的alpha，从0（完全透明）到1（完全不透明），以及图标的比例，这是一个乘数值，应用于 使用的精灵的乘数。因此，1的值是默认的比例，而（例如）0.5将是一半的比例，2将是两倍。

还值得注意的是，如果你传入的URL参数是一个脚本函数的名称（即：不是以 "http://"开头），并且该脚本被称为 gmcallback_* 那么你就可以直接获得html5事件的反馈。GameMaker Studio 2脚本函数名称如果以gmcallback_ 开头，则不会被混淆。

语法。

clickable_add_ext(x, y, tpe, url, target, params, alpha, scale)。

争论 描述
x 窗口中的X位置。
y 窗口中的y位置。
帐目 要使用的精灵的纹理页条目。
网址 要链接的URL（网站地址），或要运行的脚本。
目标 这是打开URL的目标区域（见描述）。
参数 图标的各种参数（见描述）。
阿尔法 图标的图像alpha（默认为1）。
规模 图标的比例（默认为1）。

返回。

真实的

例子。

home_but = clickable_add_ext(32, 32, sprite_get_tpe(spr_MS_Home, 0), "http://macsweeney_games.com", "_blank", "width=700, height=500, menubar=0, toolbar=0, scrollbars=0", 1, 2）;)

上述代码在游戏画布运行的页面的位置（32，32）创建了一个可点击的DOM图标。该图标使用从纹理页面引用的精灵 "spr_MS_Home"，当点击该图标时，将打开一个新的窗口，该窗口为指定的URL，并具有该窗口定义的属性。 该按钮也将被缩放为原始精灵大小的两倍，图像alpha为1的一半。
