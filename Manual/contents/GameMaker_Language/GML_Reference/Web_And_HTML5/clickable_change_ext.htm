可点击的变化_文本

通过这个函数，你可以改变之前用以下函数创建的可点击图标的精灵和位置 clickable_add().请记住，位置是基于窗口的，而不是画布的，(0,0)位置 而且精灵必须直接从纹理页中引用（见。 sprite_get_tpe()).这个函数也允许你改变精灵的比例 的比例（作为一个乘数，1是默认值，0.5是一半，2是两倍），以及屏幕上最终图标的阿尔法值从0（完全透明）到1（完全不透明）。

语法。

clickable_change_ext(index, tpe, x, y, alpha, scale)

争论 描述
指数 要改变的可点击图标的索引。
帐目 要使用的精灵的纹理页条目
x 窗口内新的X位置。
y 窗口内新的Y位置。
规模 图标的比例（默认为1）。
阿尔法 图标的图像alpha（默认为1）。

返回。

不适用

例子。

switch (room)
{
case rm_Menu: clickable_change_ext(global.Help_Icon, sprite_get_tpe(spr_MS_Help, 1), 32, 32, 2, 1); break;
case rm_Game: clickable_change_ext(global.Help_Icon, sprite_get_tpe(spr_MS_Help, 0), 200, 32, 1, 0.5); break;
}

上述代码将根据当前房间的情况，改变变量 "global.Help "中索引的可点击图标的图像索引和位置，同时改变图标的比例和alpha。
