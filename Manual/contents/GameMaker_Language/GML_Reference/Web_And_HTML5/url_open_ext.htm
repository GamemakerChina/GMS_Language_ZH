url_open_ext

这将在选定的目标设备的浏览器上打开指定的URL，如果你使用的是HTML5模块，则在当前打开的浏览器中。你指定的 "目标 "参数与你使用open()方法时的标准JavaScript "名称 "相同。 值相同，而且你应该知道，除了"_self"以外的所有参数都可能导致浏览器阻断，或者询问用户是否愿意允许。

有效的目标是。

目标 描述
_blank 在一个新的窗口或标签页中打开链接的文件（如果用户阻止了弹出式窗口，这将不起作用，在这种情况下，你可以使用 可点击_*函数代替）。)
亲自出马 在被点击的同一框架内打开链接的文件（这是默认的）。
父/母 打开父框架中的链接文档。
_top 在整个窗口中打开链接的文件。

语法。

url_open_ext(url, target)。

争论 描述
网址 要链接的URL（网站地址）。
目标 这是打开URL的目标区域。

返回。

不适用

例子。

url_open_ext("http://yoyogames.com", "_blank")。

这将在一个新窗口中打开YoYo Games的主页。
