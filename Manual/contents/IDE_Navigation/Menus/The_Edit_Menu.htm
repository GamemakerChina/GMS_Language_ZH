编辑菜单

编辑菜单是你可以使用GameMaker Studio 2界面的一般编辑工具的地方。可用的选项有。

  • 撤消 - 撤消一个动作。
  • 重做 - 重做之前的一个动作。
  • 搜索和替换 - 这将打开一个搜索窗口，允许你在整个代码（包括房间创建代码、脚本和事件）中搜索一个给定的输入 - 进一步信息见下文。

当你从 "编辑 "菜单中选择 "搜索和替换 "时，它将打开以下窗口。

你可以用它来搜索你游戏中所有的脚本资产，包括房间代码、事件代码、脚本和着色器。搜索时，你可以 可以在顶部的方框中输入一个新的搜索词，也可以点击右边的箭头图标，打开一个以前搜索过的词的列表。你还可以选择提供一个替代词，用来替代最初的搜索词。 词，这也有一个选项，可以使用右边的箭头按钮中以前使用过的替换字符串。

搜索可以是一般性的，或者你可以使用搜索词输入下面的选项来细化它们。

  • 你可以选择使搜索区分大小写，在这种情况下，它将只包括与搜索字符串大小写完全匹配的词，即：搜索 "My_Var"，如果没有这个选项，你会得到 "my_var "的结果。 "mY_VAR "或 "My_Var"，但如果勾选了这个选项，搜索结果中只包括 "My_Var"。
  • 你可以选择只包括包含整个单词的搜索结果，在这种情况下，任何包含搜索词但不是完全匹配的结果都会被忽略，例如：搜索 "game "而不勾选此项 将会显示诸如函数 "game_end() "或变量 "game_state "等内容，而勾选此项并进行搜索将只显示 "游戏 "本身的结果。
  • 你可以选择让搜索忽略注释。当勾选这个选项时，你的代码中任何包含搜索词的注释都会被忽略，不会显示在输出结果中。
  • 你可以根据资产浏览器中不同的资产类型对搜索结果应用一个过滤器 。默认情况下，这被设置为 "所有"，但点击它将打开以下菜单，你可以选中或取消选中 资产包括在搜索中。

一旦你给出了一个搜索（和可选的替换）术语并设置了搜索选项，你可以使用底部的不同按钮 来选择如何执行搜索和采取任何行动。这些按钮是

  • 查找所有。查找所有的搜索词，并在输出窗口中显示它们。
  • 查找上一个：查找搜索词的上一个实例。这将在工作空间中打开包含参考文献的资产（如果它还没有打开的话），并平移到它。
  • 查找下一个。查找搜索词的下一个实例。这将在工作空间中打开包含参考文献的资产（如果它还没有打开的话），并平移到它。
  • 全部替换：这将在设定的选项下对给定的术语进行搜索，同时在资产中用你指定的新的替换术语来替换它。
  • 替换。这将在当前编辑器中高亮显示的地方用选定的替换术语替换当前选定的术语。
  • 停止搜索。如果正在进行搜索，则停止当前的搜索。

在这个窗口打开时按下Escape 键将清除所选字段，但是如果该字段已经是空的，那么它将关闭该窗口。

使用 "搜索与替换 "窗口的所有结果将显示在IDE底部的 "搜索输出窗口"中。每个结果都可以 ，双击打开当前工作区中带有该术语的资产，并平移到该资产，准备进行编辑。
