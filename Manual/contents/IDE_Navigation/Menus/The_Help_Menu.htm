帮助菜单

这个菜单与使用GameMaker Studio 2时获得帮助有关。它包含以下菜单项。

  • 开始页- 这将打开一个新的开始页工作区。
  • 欢迎页--这将在一个新的工作区打开初始欢迎页的幻灯片。
  • 打开手册- 打开帮助文件。注意，点击菜单栏中的帮助按钮 ，也会打开该文件。 正如按下 。
  • 知识库- 这将带你到GameMaker Studio 2帮助中心，在那里你可以找到关于平台设置、已知问题、使用GameMaker Studio 2和其他一些文章。 以及其他一些内容。
  • 报告错误- 如果你在使用GameMaker Studio 2时发现任何错误或有任何问题，你可以在这里报告它们。
  • YoYoGames.com - 打开YoYo Games的主页。
  • GameMaker社区 - 这将在你的浏览器中打开GameMaker社区论坛。请注意，这是一个由社区 管理的论坛，所以如果你有一个错误报告或需要技术 你应该先去官方支持网站（见上文）。
  • 发行说明 - 这将在你的浏览器中打开 GameMaker Studio 2的发行说明。如果你需要回滚更新，你可以通过点击本页的版本号来获得以前的版本安装程序。 点击本页中的版本号链接，可以获得以前的安装程序。
  • 当前的运行时发布说明 - 由于 运行时间可以独立于IDE进行更新，它有自己的一套运行时发布说明。 选择该选项将在你的浏览器中打开它们，但请注意，当前的运行时可能不是你正在使用的那个。要了解你所使用的运行时，或要了解如何添加、删除或激活运行时的信息，请参见运行时偏好部分。
  • 所需的SDK- 由于GameMaker Studio 2的跨平台特性，需要一些第三方的SDK。点击这里将带你到支持论坛的文章，详细说明你在每个平台上需要什么。
  • 在资源管理器中打开项目- 这将在操作系统文件资源管理器中打开当前加载项目的保存位置。
  • 关于 - 打开 "关于GameMaker Studio 2 "窗口，你可以获得产品的版本信息（对于报告任何错误或问题至关重要）。
