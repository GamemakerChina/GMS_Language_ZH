地方资产包

在工具菜单下，你有两个选项用于创建和导入本地资产包。本地资产包是一组使用 *.yymp 文件格式保存的资源，可以很容易地安装到现有项目中。这意味着你可以为常用的资源创建自己的包库，如脚本库、闪屏介绍、菜单系统等...而不必使用 YoYo Games 市场.

创建一个资产包

要创建一个资产包，你必须首先创建你想在项目中保存的资源，然后从工具菜单中选择创建本地包。 这将打开以下窗口。

在这个窗口中，你可以提供一个可选的出版商名称和一个必需的显示名称和 包的ID。包的ID将用于生成包将被保存的文件名，并在创建时自动附加上 *.yymp 文件扩展名。你还可以设置 包的版本号，并选择是否签名（默认情况下是关闭的）。签署一个软件包需要你创建一个签署证书（你可以在市场偏好中 你可以从市场首选项中完成），这意味着任何已签署的资产如果在创建后被篡改，将不能被导入。 创建。

一旦你填写了资产包的细节，你就可以继续从右边的"项目 "部分选择你想包括的资产（使用鼠标左键 ，或者结合 / 和 ，同时使用鼠标左键）。选定的资源可以被移到 "资产包 "视图中，使用 使用 "添加 "或 " 全部添加"按钮，可以再次使用 "移除 "和 "全部移除"按钮移除所选资源。

一旦你对资源的选择感到满意并填写了所需的细节，点击 "确定 "按钮将打开一个文件浏览器，你可以选择保存包的位置。一旦选定，软件包将被创建 并显示一个确认窗口，其中有一个选项是在操作系统文件资源管理器中打开软件包的位置。

导入一个资产包

有两种方法来导入本地资产包。

  • 首先是简单地将 *.yymp 文件从资源管理器窗口拖到GameMaker Studio 2IDE上。
  • 第二种是选择工具菜单中的"导入本地软件包 "选项。

当使用工具菜单时，将打开一个资源管理器窗口，你可以浏览到软件包的位置，然后选择它进行导入。无论哪种方式都会在GameMaker Studio 2中打开以下窗口。

左边是软件包的内容，右边是你要导入的部分。你可以选择一个 你可以从包中选择一个或多个资源（使用鼠标左键 ，或结合 / 和 ，同时使用鼠标左键 按钮），然后使用 "添加 "按钮来添加它们进行导入，或者使用 "全部添加 "按钮来添加所有资源进行导入。

当你准备好一切后，你可以点击导入 按钮，将选定的资源添加到你的项目中。请注意，你也可以选择将所有资源导入到一个新的项目，这将为你创建一个新的 这将为你创建一个新的项目（并提示你给出一个保存位置），然后将包中的资源添加到该项目中。

请注意，使用GameMaker Studio 2.3之前的版本创建的资产包将需要转换，因为项目文件格式随着2.3的更新而改变。这意味着，最好是导入整个项目并转换所有内容 然后将其作为一个新的包导出到一个新的位置，如果资产经常被导入其他项目的话。
