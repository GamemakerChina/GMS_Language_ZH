微型网络服务器

这个小工具的主要功能是连接到GameMaker Studio 2的一些可用的目标平台并为其提供文件（例如，像HTML5目标）。当你在一个需要Micro Web服务器的目标上运行一个游戏时，你会看到 一个新的窗口打开，因为游戏被提供给所选择的设备。

注意：当你第一次在一个需要Micro Web服务器的目标上运行游戏时，你有可能会得到一个防火墙警报。如果发生这种情况，你应该允许网络服务器通过防火墙进行通信。

网络服务器将显示一定数量的与已提供的文件有关的调试信息，并在底部有一个按钮可以清除这些信息。在 "清除 "按钮的旁边是项目的JS根文件的路径，而在它的下面是网络速度选项。 你有网络速度选项。这个选项默认设置为 "无限制"，但点击它将打开一个不同网络选项的列表。

改变这个设置也会将包的大小 选项改变到右边。你也可以手动编辑这个。 在这种情况下，网络速度将被设置为 "自定义"。将其设置为除最大值以外的任何其他值，将有效地节制网络传输速度，并允许你在低带宽环境下测试游戏。

你也可以看到允许的IP列表和正在使用的端口号 ，但这些不能在Micro Web服务器中编辑，而应该在设备管理器设置中设置。 中设置。
