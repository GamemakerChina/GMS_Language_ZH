开始一个新项目

你可以点击 "新建 "按钮来创建一个新的项目，点击 "打开 "按钮来打开一个现有的项目，或者点击 "导入 "按钮来打开一个GameMaker:Studio 1.4 GMX 项目文件或打开一个压缩的 YYZ GameMaker Studio 2项目文件。打开和导入都会打开文件资源管理器 让你浏览到你需要的项目文件。

注意：GameMaker Studio 2并不完全向后兼容GameMaker。Studio 1.4 项目，但导入的 1.4 项目仍然可以运行，因为过时的功能已经使用兼容性脚本为你自动重新创建。 关于可能出现的问题和对GML所做的修改的全部细节，请参见帮助中心文章《将GMS 1.4项目移植到GameMaker Studio 2》。 以及手册中关于过时的功能的部分。

要创建一个新的项目，只需点击标有 "新建 "的按钮，即可进入以下屏幕。

GameMaker Studio 2允许你指定你希望创建的项目类型，你可以根据你的编码知识或习惯进行挑选。可用的选项有

  • NewDrag and Drop™ Project - 这将创建一个新的项目，其中的对象编辑器（和任何其他允许代码的编辑器）将被设置为使用GameMaker Studio 2 拖放™ 界面。关于拖放（DnD™）的更多信息，请看这里。请注意，这不是一个专属的DnD™界面，你可以使用鼠标右键在DnD™和GML代码之间切换。 按钮 。
  • 新的GameMaker语言项目。这将创建一个项目，其中默认的编程是使用GameMaker语言（GML）完成的。你可以在这里找到更多关于GML的信息。 请注意，这不是一个专属的GML界面，你可以使用鼠标右键在GML代码和DnD™之间切换 。

无论你选择什么，你都会被提示为项目选择一个保存位置，然后在GameMaker Studio 2中打开它。

教程和演示

当你点击 "教程 "或 "演示 "按钮时，你将进入一个屏幕，显示你所拥有的教程/演示以及可供下载的官方教程/演示。

要下载任何官方教程或演示，只需点击你想看的那个图标，它就会被添加到你顶部的 "拥有"部分（注意，你可能需要登录到市场的 才可以完成）。)一旦教程或演示在 "拥有 "部分，你可以把鼠标放在它上面，然后安装它或从市场上查看它的信息。

一旦安装了教程或演示，只需点击它就可以打开，在IDE为你打开之前，你会被提示提供一个保存教程项目的位置。请注意，已安装的教程和演示程序 会有一个额外的图标 ，可以用来从你的电脑上卸载项目（这不会从你的 "拥有 "列表中删除该教程或演示）。 这不会从你的 "拥有 "列表中删除该教程或演示）。)
