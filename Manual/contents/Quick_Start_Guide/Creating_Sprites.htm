如你所见，窗口的左上方有一个精灵的名称 字段。所有的精灵（和所有其他资产）都必须有一个名称，以便你（和GameMaker Studio 2）能够轻松地识别它们。 一个名字，这样你（和GameMaker Studio 2）就可以很容易地识别它们，不过你应该注意，这个名字实际上只是一个 变量它持有一个ID值 来 "指向 "资源，这里是指精灵。最好给每个精灵起一个描述性的名字，这样你就能一眼看出某个特定的资产是精灵还是物体或其他东西，很多人都是这样做的 许多人通过在资源前缀或后缀中加入字母 " spr"来做到这一点--例如，" spr_Ball"。请注意，资源名称只限于使用字母、数字和下划线 符号"_"，而且所有资产的名称必须以字母而不是数字开头。

  • 在左上角，是框架视图--它显示了你的精灵的框架，可以使用放大镜图标调整大小。
  • 在中央，我们有一个绘图画布--它显示了当前选定的准备绘图的框架，你可以用鼠标中键 来平移，用鼠标滚轮 来放大和缩小。
  • 在右上方，我们有基本的画笔--在这里你可以挑选画笔，任何自定义的画笔也会被添加到这里。
  • 在右边的中间位置，我们有一个颜色选择器--在这里我们可以为鼠标左键和/或右键选择颜色，因为你可以同时用一个或另一个按钮在画布上作画。
  • 就在颜色选择器的下面，我们有我们的工具 --这些是不同的绘图工具，包括画笔、泛滥填充、遮蔽和形状工具。请注意，有些工具是对角线分割的，它们会根据你点击的那条线的哪一边来绘制轮廓或填充的形状。 形状，这取决于你点击哪边的线来选择它们。

你需要知道的第一件事是如何为精灵设置原点 。原点只是用来将我们的精灵 "锚定 "在房间里的一个点，在这种情况下，我们要将它设置在中心位置。

因此，我们在设置原点的下拉菜单上点击 ，并选择 "居中"。

你会看到预览图中的蓝色原点标记移动到精灵的中心。请注意，你可以把原点放在你希望的任何地方，只需点击预览图上的 。 预览图，你可以把原点放在你想放的任何地方，你也可以通过在顶部的X/Y字段中添加数值来手动输入原点位置。

另一件重要的事情是如何设置精灵的碰撞掩码。碰撞掩码是GameMaker Studio 2将检查的区域，以查看是否与游戏中的任何其他对象实例发生碰撞。 默认情况下，它被设置为矩形（这也是最快解决的）。请注意，你可以将矩形的检测模式设置为 界限盒的检测模式可以是自动 (默认值)或手动。如果你选择手动，你将能够设置左、右、顶部和底部的边界框值。

除矩形以外的任何类型的碰撞掩码的解决速度都会比较慢，但当制作一个简单的小游戏时，这通常不是一个问题。请注意，你也可以通过点击精灵编辑器中的导入 按钮来导入其他软件中制作的精灵。 你也可以把图像文件拖到GameMaker Studio 2IDE中，把它们作为精灵添加到资产浏览器中。你也可以从资产浏览器中拖动精灵，将它们直接添加到事物中，在这种情况下，IDE会突出显示你可以将精灵添加到哪些地方。 你可以添加精灵的地方。
