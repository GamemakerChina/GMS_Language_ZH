瓷砖集是由精灵创建的，但它们也被归为独立的资产类型，因为GameMaker Studio 2处理它们的方式与精灵和物体都有些不同。所以，瓦片组是在一个精灵中创建的瓦片的集合。 的集合，而瓷砖只是一种图形资源，它被添加到GameMaker Studio 2的房间中，作为房间的一部分。 瓦片图.这些瓦片地图然后被绘制到 这些瓦片地图然后被绘制到屏幕上，只需很少的CPU/GPU处理，使它们成为设计关卡中任何静态项目的理想选择。

创建了精灵后，我们可以继续定义瓷砖组本身，这意味着你必须首先创建一个新的瓷砖组资源，右键点击 Tile Set asset 文件夹并选择创建。这将打开瓷砖组编辑器，在那里你可以添加精灵并设置其属性。
