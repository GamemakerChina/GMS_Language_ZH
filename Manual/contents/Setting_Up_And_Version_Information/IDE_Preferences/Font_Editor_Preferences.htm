字体编辑器首选项

字体编辑器首选项用于定义字体编辑器的行为方式。 下面列出了不同的选项，以及每个选项的作用描述。

  • 更新从GameMaker Studio 1项目导入的字体。当勾选此项时，GameMaker Studio 2将尝试更新字体并重新生成它们的纹理。这只有在以下情况下才会发生。
    • 启用该首选项
    • 该字体在机器上是可用的
    • GameMaker Studio 2可以在字体家族中找到匹配的字体（这是最有可能导致导入时不一致的原因）。
    请注意，如果GameMaker Studio1字体没有缓存的字体纹理，它也会发生。如果一个传统字体不能被更新，那么GameMaker Studio 2将默认使用缓存的 字体纹理，并且字体编辑器将不再允许你编辑字体，显示一个备份和更新到另一个的选项：

    点击它将使你能够编辑字体，要么添加缺少的字体，要么设置另一个，并且 备份的字体被存储在字体资产文件夹中，如 <font name>.yy.gms1 和 <font name>.png.gms1 。要恢复这些文件，请手动删除 <font name>.yy 文件，并从文件名中去掉 .gms1 的扩展名，用这些文件取代。 重要的是!所有的资产都在 .yy 文件中存储它们的名字，所以你必须编辑这些文件。 并把正确的资源名称也放在 <name> 字段中（如果不同）。
  • 内容再生失败对话框的自动选择。当你去重新生成一个字体资产纹理时，你会看到一个警告。把这个选项设置为 "OK "意味着字体的再生不会显示警告。 显示。
  • 字体再生失败对话框的自动选择。当字体资产纹理的再生失败时，你会看到一个警告。将此选项设置为 "确定 "将跳过显示该信息。
  • 自动选择升级已导入的遗留字体。当你导入一个遗留项目时，你可能会得到一个关于遗留字体的警告和升级的提议。把这个选项设置为 "确定"，就可以自动升级而不显示这个消息。 显示这个消息。
