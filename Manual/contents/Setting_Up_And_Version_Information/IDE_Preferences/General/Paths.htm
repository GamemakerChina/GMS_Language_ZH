路径偏好

路径 偏好控制GameMaker Studio 2IDE中的元素以及你需要的任何外部编辑器所需要的不同文件路径。在IDE中存在以下选项。

  • 临时目录。保存所有临时文件夹的位置。默认情况下，在Windows上这是

    Users\<Username>AppData\Local\Temp\GameMakerStudio2

    而在macOS上，它是

    /var/folders/<Path_Hash>/GameMakerStudio2

  • IDE 缓存目录。用于保存IDE缓存的位置。在Windows上默认为

    Users\<Username>AppData\Roaming\GameMakerStudio2

    \Cache 而在macOS上，它是

    /Users/<Username>/.config/GameMakerStudio2

    /Cache

  • 资产缓存目录。为每个项目保存资产缓存的位置。默认情况下，在Windows上这是

    Users\<Username>AppData\Roaming\GameMakerStudio2

    \Cache 而在macOS上，它是

    /Users/<Username>/.config/GameMakerStudio2

    /Cache

  • "我的项目 "位置。 GameMaker Studio 2最初创建新项目的位置。默认情况下，在Windows上这是

    Users\<Username>Documents\GameMakerStudio2

    而在macOS上，它是

    /Users/<Username>/GameMakerStudio2

  • 关闭时自动删除临时目录。勾选此项将迫使GameMaker Studio 2自动删除它为每个项目创建的用于编译等的临时文件夹。这个设置是默认开启的，取消勾选它将会关闭 关闭它（意味着你将不得不在以后手动删除任何临时文件）。
  • 关闭时自动删除资产缓存。如果勾选此项，那么当你退出GameMaker Studio 2时，资产编译器缓存文件夹将被删除。这在默认情况下是关闭的，启用它将意味着每次你加载和运行 任何项目都需要重建缓存（这可能需要时间，取决于游戏的大小）。
  • 关闭时自动删除 IDE 缓存。如果选中此项，那么当你退出GameMaker Studio 2时，IDE编译器缓存文件夹将被删除。默认情况下这是关闭的，启用它意味着每次启动GameMaker Studio 2时，都需要重新建立缓存。
  • 删除临时文件夹。点击这个按钮将删除项目的临时文件夹。
  • 删除资产缓存。点击这个按钮将删除项目的编译器资产缓存。
  • 删除IDE缓存。点击该按钮将删除IDE缓存。

在设置外部编辑器的路径方面有以下选项。

  • SWF文件的外部编辑器/查看器的路径。如果你正在处理SWF格式的精灵，你可以把它设置为你喜欢的查看器/编辑器的路径，当你点击精灵编辑器中的编辑图像按钮时，它会 就会打开给定的程序。这里的默认值是没有路径。
  • Spine文件的外部编辑器/查看器的路径。如果你正在处理Spine格式的精灵，你可以将其设置为你喜欢的查看器/编辑器的路径，当你点击精灵编辑器中的编辑图像按钮时，就会打开指定的程序。 它就会打开所给的程序。这里的默认值是没有路径。
