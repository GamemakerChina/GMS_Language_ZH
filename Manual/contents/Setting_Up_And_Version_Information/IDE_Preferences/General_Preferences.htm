一般偏好

常规首选项用于定义GameMaker Studio 2 IDE的一般行为。最普通的选项列在主页面上，而其他许多选项则被分成多个子类别，在本页面的下方有进一步解释。

  • IDE语言。该选项允许你从提供的选项列表中选择IDE的语言。默认为英语。
  • IDE皮肤。你可以在这里选择IDE可以使用的不同皮肤。默认情况下，GameMaker Studio 2有两种皮肤，一种是浅色，一种是深色，初始设置使用深色皮肤。
  • GameMaker Studio 2编辑器的最大帧率。该选项允许你设置GameMaker Studio 2IDE更新的最大帧率，允许你设置帧率以匹配显示器的更新速度。默认值 值为60。
  • 默认音频设备。在这里你可以选择GameMaker Studio 2IDE要使用的默认音频设备。你可以通过点击Refresh Button 来刷新列表。
  • 自动响应退出确认信息。这个选项允许你设置当你退出GameMaker Studio 2时是否应该显示确认信息。默认情况下，它被设置为 "显示消息"，但也可以设置为跳过消息，就像你点击确定一样。 信息，就像你点击了确定一样。
  • 对系统字体更改对话框的自动响应。该选项允许您设置当系统字体被添加、删除或编辑时，是否应显示确认信息。默认情况下，它被设置为 "显示消息"。 但也可以设置为跳过该信息，就像你点击了确定一样。
  • 启用通知。勾选此项将使IDE显示关于新运行时的通知，等等。默认情况下，该选项是打开的。
  • 通知超时（毫秒）。这设置了通知显示的时间长度（以毫秒为单位）。默认值是6000。
  • 退出时注销：如果勾选这个选项，当GameMaker Studio 2关闭时，IDE将自动注销用户会话。这主要是针对教育用户的，因为--如果选中--这意味着每当程序关闭时 关闭时，用户将被注销，从而腾出会话席位供其他用户使用。这在默认情况下是关闭的。
  • 启用DPI覆盖。这个选项可以让你用你自己的DPI设置来覆盖默认的DPI设置。你必须首先启用该选项，然后设置原始DPi覆盖值（默认值为96）或设置本地DPi的百分比（默认值为100%），一旦改变，你将被提示重新启动IDE，这可能需要你保存当前打开的项目上正在进行的任何工作。请注意，DPi的最小百分比为50%，最大百分比为500%，如果你 如果你把鼠标放在信息图标上，你可以看到正在使用的显示器的建议最大设置：

    注意，如果你超过了显示器的建议DPi设置，你将会看到一个通知，允许你重新设置。 通知，允许你重置DPi。
  • 启动时显示欢迎屏幕。默认情况下，该选项是打开的，只是告诉GameMaker Studio 2在你启动程序时显示一个简短的闪屏。取消勾选这个选项将防止显示闪屏。
  • 允许通过快捷键关闭空标签：你可以使用键盘快捷键 / +W 关闭任何打开的标签，但如果你禁用此选项，该快捷键将不再起作用。这在默认情况下是启用的。
  • 在将文件拖入IDE时打开新资源进行编辑。某些资源（如精灵或声音）可以拖到GameMaker Studio 2IDE的主工作区，并将被检测到并作为新资源添加到资产浏览器中。 浏览器中检测并添加为新资源。默认情况下，当这种情况发生时，相应的资源编辑器也会被打开进行编辑，但取消勾选该选项将阻止编辑器被打开，你将需要手动打开它们。
  • 创建时打开新资源进行编辑。如果勾选此选项，每次你在资产浏览器中创建一个新的资源，相应的资源编辑器将被自动打开。默认情况下勾选该选项，取消勾选则意味着您必须手动打开所有新创建的资源。 将意味着你必须手动打开所有新创建的资源。
  • 自动重新加载已更改的文件。如果勾选此项，那么IDE将自动扫描和检查文件，看是否有任何文件在GameMaker Studio 2之外被修改过，如果有，则重新加载。
  • 禁用IDE的过渡动画。GameMaker Studio 2IDE有许多动画过渡，比如打开窗口等......但是它们并不符合每个人的口味，如果禁用它们，你可能会遇到性能提升的问题。 如果禁用它们，你可能会体验到性能的提升，如果在旧的或动力不足的机器上禁用它们。你可以在这里禁用它们，而且它们在默认情况下是打开的。
  • 选择加入反馈调查。当这个选项被选中时，你将告诉GameMaker Studio 2你希望参加反馈调查，这意味着在工作时，偶尔会显示一条关于你正在工作的领域的信息 并要求进行评级和反馈。这个选项默认是打开的。
