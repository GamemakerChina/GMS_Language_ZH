房间编辑器首选项

房间编辑器首选项用于定义房间编辑器的外观和行为。一般设置如下。

  • 链接淡出的起始距离（px）。设置链式链接淡出时与父窗口的距离。默认为10px。
  • 链接淡出停止距离（px）。设置链式链接结束淡出时与父窗口的距离。默认为100px。
  • 链式最小阿尔法。设置链式阿尔法的淡出淡化点后的最终阿尔法值。默认是0.3。
  • 默认的图层深度间隔。当你在房间编辑器中创建多个图层时，它们会根据它们在图层列表中的位置自动分配一个深度。这个深度会以这里给出的数量递增。默认的 默认值是100，所以每个层将是前一层+100的深度。
  • 默认房间宽度：设置新房间默认的初始宽度。默认为1024px。
  • 默认的房间高度。设置初始高度，该高度将被选为新房间的默认高度。默认为768px。
  • 添加默认的房间图层。当你创建任何新的房间资源时，GameMaker Studio 2将为你生成几个 "基础 "层--一个背景层和一个实例层。你可以在这里关闭这一行为，这样 新的房间将没有预制层。这在默认情况下是打开的。
  • 显示房间的视图。如果你在一个房间里有视图活动，这将由房间里的视图边界的轮廓来显示，这个区域的可见性可以通过编辑器本身的专用按钮来切换。 然而，你可以选择在打开房间的时候不勾选这个选项，让视图边界不可见（仍然可以用这个按钮来再次显示）。默认值是开，当你打开一个房间时，视图边界是可见的。
  • 一般的颜色选项。这些与整个房间编辑器界面使用的不同颜色有关，如果你双击任何一个色板，就可以编辑该设置的颜色。
  • 主音量设置。房间编辑器中预览播放的主音量设置，一个在0和1之间的值。 默认为1。

更多的选项可从下面列出的子类别中进行编辑。

确认对话

"确认对话 " 首选项用于改变当你执行某些操作时在房间编辑器中显示的信息。你可以改变的信息有

  • 对更新图层深度的自动响应：当你改变图层深度时，会显示一条信息，警告你这样做也会自动更新所有其他图层深度。有了这个选项，你可以选择让信息显示 显示，让它更新图层，就像你点击 "OK "一样，或者让它不更新任何东西，就像你点击 "No "一样。
  • 对图层锁定警告的自动响应。如果你试图删除房间里的一个图层，而它恰好被锁定，默认情况下你将会看到一条信息。但是把这个选项设置为 "OK "将不显示信息，只是不让你删除图层。 只是不允许你删除该层。
  • 对项目锁定警告的自动反应: 如果你试图删除一个项目，它恰好被锁定，你将默认显示一条信息。但是把这个选项设置为 "OK "就不会显示信息，只是不让你删除项目。 只是不允许你删除该项目。
  • 对项目所有者层锁定警告的自动响应。如果你试图删除一个图层上的项目，而该图层恰好被锁定（即使该项目没有被锁定），那么默认情况下，你将会看到一条信息，通知你这是不可能的 无法完成。但是把这个选项设置为 "OK "就不会显示这个信息，只是不让你删除这个项目。
  • 当把序列/精灵拖到不合适的层上时，创建新的资产层。如果你试图将一个精灵（或序列）添加到一个不属于资产层的图层，那么你将默认显示一条信息，告知你这是不可能的 并询问你是否要为被拖动的项目创建一个新的资产层。但是将这个选项设置为 "OK "就不会显示这个信息，并自动为精灵（和序列）创建一个新的图层。
  • 对父房间项目删除的自动反应。在这里你可以选择当你删除一个属于父房间的项目时（因此在所有子房间都存在）会发生什么。默认情况下，会向你显示一条警告信息，让你决定如何处理。 但你可以把它设置为 "全部删除"，以抑制该信息并继续从父房间和所有子房间删除项目，或者你可以选择 "保留全部"，以抑制该信息并中止删除。
  • 对父房间图层删除的自动响应。在这里你可以选择当你删除一个属于父房间的图层（所以在所有子房间都存在）时，会发生什么。默认情况下，会显示一个警告信息，让你决定如何处理。 但你可以把它设置为 "全部删除"，以压制该信息并继续从父房间和所有子房间删除图层，或者你可以选择 "保留全部 "以压制该信息，中止删除。
  • 对无效的资产拖动错误的自动响应。如果你试图添加一个资产到一个没有被指定为该资产类型的图层（比如添加一个路径到一个实例图层），那么默认情况下，你会看到一个信息，告知你无法完成。 通知你这是不可能的。但是把这个选项设置为 "OK "就不会显示这个信息，只是不让你添加资产。注意，这个选项并不适用于实例、精灵或序列。
  • 当把对象拖到不合适的图层上时创建新的实例图层。如果你试图将一个对象实例添加到一个不属于实例层的图层中，那么默认情况下你会看到一条信息，告诉你这是不可能的。 并询问你是否要为被拖动的项目创建一个新的实例层。然而，将此选项设置为 "OK "将不会显示该信息并自动为该实例创建一个新的图层。
  • 对无效的图层选择拖动错误的自动响应。这里您可以选择当您在图层编辑器中选择了多个图层（或根本没有选择任何图层）时，将资源拖到房间编辑器中会发生什么情况。 时，你可以选择发生什么情况。)默认是显示一个警告信息，但你可以把这个选项设置为 "OK"，以抑制该信息并继续，而不添加任何资源。
  • 对锁定或不可见图层拖动错误的自动响应。在这里你可以选择当你拖动一个资源到一个被锁定或被标记为不可见的图层上时会发生什么。默认情况是显示一个警告信息，但你可以把这个选项设置为 "确定"，以抑制信息。 可以把这个选项设置为 "OK"，以抑制该信息并继续，而不添加任何资源。

网格设置

网格设置 网格设置首选项用于改变网格覆盖的工作方式，有以下选项。

  • 网格水平间距（px）。这将设置房间网格的水平空间，单位为像素。默认值是32px。
  • 网格垂直间距（px）。这将设置房间网格单元的垂直空间，单位为像素。默认值是32px。
  • 显示网格。当你第一次打开一个房间时，这个选项将切换为打开或关闭房间网格。它在默认情况下是打开的，但即使它被关闭了，你也可以在房间编辑器中使用适当的按钮再次启用它。
  • Snap to grid:通过这个选项，你可以设置一个房间在最初打开时，是否默认有资产扣到房间网格上。默认情况下它是打开的，但即使它被关闭了，你也可以通过房间编辑器中的相应按钮再次启用它。 房间编辑器中的相应按钮再次启用。
  • 网格线颜色：这个选项允许你设置房间网格线的颜色（包括阿尔法成分）。默认值是 ( argb）#30ffffff。



智能指南

本节详细介绍了 房间编辑器的智能向导首选项。可用的选项有。

  • 启用智能导引器。这用于在房间编辑器中启用/禁用智能导引。默认是启用。

  • 启用标尺。这个选项将启用/禁用房间编辑器中的侧边标尺。默认是启用。

  • 捕捉标尺到边缘。启用后，该选项将把从标尺拖入房间编辑器的导引器与放置在房间里的任何资产绑定。默认是启用。

  • 捕捉距离。这个设置告诉房间编辑器，导引器要离资产多远才会 "捕捉 "到资产的侧面。默认值是6px。
