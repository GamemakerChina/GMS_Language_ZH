你可以在YoYoGames网站上的YoYo账户页面下载GameMaker Studio 2安装程序。安装完成后，在第一次运行GameMaker Studio 2时，可能会提示您在使用产品前进行更新。 使用该产品。在安装任何更新后，您需要提供您的YoYo Games账户的许可证详细信息（见这里的更多信息），然后才能使用该产品。 这意味着你第一次使用GameMaker Studio 2时，必须有互联网连接。初次登录后，您应该能够离线使用该产品，但它会定期要求您连接以验证您的 登录账户。
